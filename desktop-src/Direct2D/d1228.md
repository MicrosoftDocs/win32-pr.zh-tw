# <a name="d1228-not-original-target"></a>D1228：不是原始目標

ID2D1HwndRenderTarget：： Resize 或 ID2D1DCRenderTarget：： BindDC 的呼叫是在裝置內容上設定了原始目標以外的目標時進行。
