# <a name="d1159-invalid-commandlist-topology"></a>D1159：不正確 Commandlist 拓撲

命令清單無法參考屬於效果圖形（使用命令清單）的效果。





 

## <a name="possible-causes"></a>可能的原因

在填入命令清單時，應用程式可能會包含對 ID2D1DeviceCoNtext：:D rawImage 的呼叫。 ID2D1DeviceCoNtext：:D rawImage 可以用來繪製點陣圖、效果或命令清單。 如此一來，ID2D1DeviceCoNtext 的語法可能是：:D rawImage 呼叫嘗試繪製命令清單，而該相同的命令清單設為目標。 但以這種方式自我參考的命令清單是不正確。 Direct2D 將會找出發生這種情況並回報錯誤的情況，而且 debug 層會輸出此訊息。

## <a name="possible-fixes"></a>可能的修正

填入命令清單時，請確定裝置內容不會發出任何使用相同命令清單的作業。 發生這種情況最明顯的方法，就是使用設定為目標的相同命令清單來呼叫 ID2D1DeviceCoNtext：:D rawImage。 不過，這種情況可能也比較不明顯： ID2D1DeviceCoNtext：:D rawImage 可以參考效果圖形，而效果圖形中的一個節點可能會嘗試取用目標命令清單。 請確定已正確填入命令清單，再使用，且對 ID2D1DeviceCoNtext：： SetTarget 的呼叫會參考預定的目標。
