# <a name="d1231-unknown-dxgi-format-support"></a>D1231：未知的 DXGI 格式支援

對 IsDxgiFormatSupported 的呼叫是以 DXGI \_ 格式 \_ 未知。 此 API 會傳回 false，但 \_ 未知的 DXGI 格式 \_ 可能會是對某些 api 的輸入有效。






 

## <a name="possible-causes"></a>可能的原因

對 IsDxgiFormatSupported 的呼叫是以 DXGI \_ 格式 \_ 未知。

## <a name="possible-fixes"></a>可能的修正

藉 \_ \_ 由閱讀該 api 的檔，而非嘗試以程式設計方式判斷，以判斷 DXGI 格式不明是否為指定 api 的有效緩衝區有效位數。
