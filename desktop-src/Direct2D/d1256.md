# <a name="d1256-invalid-analysis-effect-usage"></a>D1256：不正確分析效果使用方式

使用了不正確的分析效果。 當流量分析效果做為另一個效果或影像筆刷的輸入、將其繪製至命令清單，或快取其輸出時，就可能發生這種情況。
