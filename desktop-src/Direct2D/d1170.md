# <a name="d1170-invalid-flags-with-dynamic-texture-bitmap"></a>D1170：使用動態材質點陣圖的旗標無效

\_ \_ \_ \| \_ \_ \_ 從動態材質建立點陣圖對應時，需要 D2D1 對應選項 WRITE D2D1 map options 捨棄。






 

## <a name="possible-causes"></a>可能的原因

如果應用程式呼叫 ID2D1Bitmap1：： Map 的材質，其基礎 Direct3D 資源以 D3D11 使用方式進行屬性化 \_ \_ ，但未同時指定 D2D1 \_ 對應 \_ 選項 \_ WRITE 和 D2D1 \_ 對應 \_ 選項 \_ 捨棄旗標，Direct2D 可能會輸出此訊息。

## <a name="possible-fixes"></a>可能的修正

如果應用程式需要使用 ID2D1Bitmap1：： Map 來存取點陣圖，而其基礎 Direct3D 資源是以動態 D3D11 使用方式屬性化 \_ \_ ，請指定 D2D1 \_ 對應 \_ 選項 \_ WRITE 和 D2D1 \_ Map \_ options \_ 捨棄旗標。 如需 \_ 對應選項的描述，請參閱 D2D1 對應 \_ 選項。
