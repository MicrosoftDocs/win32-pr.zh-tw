# <a name="d1150-unsupported-mip-level-selection"></a>D1150：不支援的 Mip 層級選取

提供的 IDXGISurface2 有多個 mip 層級，但此裝置層級不支援 mip 層級的選取範圍 (D3D9) 。






 

## <a name="possible-causes"></a>可能的原因

Interopped Direct3D 紋理可以是 mip 對應;Direct2D 一次只會在一個 mip 層級上運作。 Direct2D 不會影響或轉譯父紋理的其他 mip 層級。 但是，功能層級9裝置不支援 mip 層級選取，也無法使用此互通性功能。

## <a name="possible-fixes"></a>可能的修正

針對預期在功能層級9裝置上正常執行的應用程式，請考慮修改轉譯邏輯，以避免需要交互操作 mipmapped 紋理。 可能的話，請改為與中繼互動。 為了將解決這項限制的影響降到最低，應用程式可能會選擇將其實作為功能層級9特定的回溯路徑。
