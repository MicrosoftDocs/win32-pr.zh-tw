# <a name="d2d1_rect_u"></a>D2D1 \_ RECT \_ U

代表由左上角座標所定義的矩形， (左、上) 和右下角的座標 (右邊、下) 。


```C++
typedef D2D_RECT_U D2D1_RECT_U;
```



## <a name="remarks"></a>備註

**D2D1 \_RECT \_ u** 是已定義的 [**D2D \_ RECT \_ U**](/windows/desktop/api/dcommon/ns-dcommon-d2d_rect_u) 結構的新名稱。
