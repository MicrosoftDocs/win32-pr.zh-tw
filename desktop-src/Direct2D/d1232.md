# <a name="d1232-32bpp-rgba-causes-conversion"></a>D1232： 32bpp RGBA 會導致轉換

請考慮針對32bpp 格式使用 BGR 而不是 RGB，讓 BitmapSource 效果可以避免進行其他格式轉換。
