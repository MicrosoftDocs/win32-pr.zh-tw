# <a name="d1239-source-transform-bitmap-invalid-call"></a>D1239：來源轉換點陣圖不正確呼叫

傳遞至 ID2D1SourceTransform 的點陣圖（:D raw）不支援這個方法。






 

## <a name="possible-causes"></a>可能的原因

應用程式在傳送至 ID2D1SourceTransform 的點陣圖上呼叫了不支援的方法：:D raw。

## <a name="possible-fixes"></a>可能的修正

若要更新此類型的點陣圖，請使用 ID2D1Bitmap1：： Map。
