# <a name="d1140-stroke-transform-type-uses-default-dpi"></a>D1140：筆劃轉換類型使用預設 DPI

使用固定或細線筆劃來擴展 ID2D1Geometry 時，Direct2D 會假設 DPI 為96。 轉譯加寬的幾何時，筆劃寬度可能會變更。






 

## <a name="possible-causes"></a>可能的原因

當 Direct2D 將幾何繪製至目標時，它會加寬幾何。 不過，應用程式也可以使用 ID2D1Geometry 的方法（例如加寬、StrokeContainsPoint 和 GetWidenedBounds）來建立沒有目標的 ID2D1Geometry 物件，並將它們擴大。 在這些作業期間，應用程式可能會使用 D2D1 \_ 筆觸 \_ 轉換 \_ 類型 \_ 的細項或固定。 這些屬性通常會考慮 DPI，但如果沒有目標，則不會指定 DPI。 這項參考用訊息會警告 Direct2D 會考慮預設的 DPI 96.0。

## <a name="possible-fixes"></a>可能的修正

此訊息僅供參考，而且不一定表示應用程式發生問題。
