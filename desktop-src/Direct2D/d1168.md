# <a name="d1168-write-map-option-requires-cpu-writable-bitmap"></a>D1168： Write Map 選項需要 CPU 可寫入點陣圖

D2D1 \_ 對應 \_ 選項 \_ 寫入只在 CPU 可寫入點陣圖上有效






 

## <a name="possible-causes"></a>可能的原因

如果應用程式使用 D2D1 對應選項 WRITE 屬性來呼叫 ID2D1Bitmap1：： Map \_ \_ \_ ，但點陣圖不是可寫入 CPU 的，Direct2D 可能會輸出此訊息。

## <a name="possible-fixes"></a>可能的修正

如果應用程式需要使用 ID2D1Bitmap1：： Map 的寫入存取權，請確定基礎 Direct3D 資源具有 D3D11 \_ CPU \_ 存取 \_ 寫入屬性。
