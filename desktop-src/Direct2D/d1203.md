# <a name="d1203-copyfrombitmap-with-multisampled-source"></a>D1203： CopyFromBitmap 與多重取樣來源

CopyFromBitmap 用於多重取樣來源點陣圖。 仍會發出呼叫，但是複製的結果可能不會代表預期的多重取樣模式。






 

## <a name="possible-causes"></a>可能的原因

應用程式會將多重取樣 Direct3D 材質 interoperated 為 Direct2D 點陣圖，並在點陣圖上呼叫 CopyFromBitmap。

## <a name="possible-fixes"></a>可能的修正

如果 CopyFromBitmap 功能很重要，請避免將多重取樣紋理作為點陣圖進行交互操作。 複製的結果可能不代表預期的多重取樣模式。
