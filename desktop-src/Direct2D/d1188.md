# <a name="d1188-cpu-read-and-target-bitmap-options-are-incompatible"></a>D1188： CPU 讀取和目標點陣圖選項不相容

D2D1 \_ 點陣圖 \_ 選項 \_ CPU \_ 讀取與 D2D1 \_ 點陣圖 \_ 選項目標不相容 \_ 。






 

## <a name="possible-causes"></a>可能的原因

應用程式指定了不相容的 D2D1 \_ 點陣圖 \_ 選項旗標組合。

## <a name="possible-fixes"></a>可能的修正

\_ \_ \_ 從旗標中移除 D2D1 點陣圖選項 CPU \_ 讀取或 D2D1 \_ 點陣圖 \_ 選項 \_ 目標。 如果應用程式需要這兩個選項，它應該會建立兩個不同的點陣圖。
