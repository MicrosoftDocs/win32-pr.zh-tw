# <a name="d1169-discard-map-option-requires-dynamic-texture"></a>D1169：捨棄對應選項需要動態材質

D2D1 \_ 對應 \_ 選項 \_ 捨棄只適用于從動態材質建立的點陣圖。






 

## <a name="possible-causes"></a>可能的原因

如果應用程式使用 D2D1 \_ 對應選項捨棄屬性來呼叫 ID2D1Bitmap1：： Map \_ \_ ，但點陣圖的基礎 Direct3D 資源未使用 D3D11 使用方式動態進行屬性化，Direct2D \_ 可能會輸出此訊息 \_ 。

## <a name="possible-fixes"></a>可能的修正

如果應用程式需要使用 ID2D1Bitmap1：： Map 存取點陣圖，請確定基礎 Direct3D 資源具有 D3D11 \_ USAGE \_ 動態屬性。
