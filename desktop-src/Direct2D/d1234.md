# <a name="d1234-null-resource-brush"></a>D1234： Null 資源筆刷

點陣圖筆刷或影像筆刷的關聯點陣圖/影像設為 Null 時，可能不會用來進行繪製。






 

## <a name="possible-causes"></a>可能的原因

應用程式嘗試使用 ID2D1BitmapBrush 或 ID2D1ImageBrush （其點陣圖或影像為 Null）來呈現基本類型。

## <a name="possible-fixes"></a>可能的修正

請先確定點陣圖或影像筆刷有指派的有效點陣圖或影像，再加以使用。 應用程式可能會在建立時或之後，使用 ID2D1BitmapBrush：： SetBitmap 或 ID2D1ImageBrush：： SetImage 來執行此作業。 Direct2D 允許 Null 筆刷資源，如此一來，應用程式就可以在使用這些資源轉譯之後自由釋放筆刷資源。 如果點陣圖或影像資源的成本特別高，應用程式可能會選擇這樣做，而且案例會要求它立即釋放。 將筆刷資源設定為 Null，可防止 Direct2D 保留其未完成的參考。 但是，如果應用程式將筆刷資源指派給 Null，則在指派給新的有效資源之前，它不應該嘗試使用筆刷。
