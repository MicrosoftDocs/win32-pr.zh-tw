# <a name="d1144-effect-is-not-a-target"></a>D1144：效果不是目標

SetTarget 失敗，因為效果以目標形式傳遞。






 

## <a name="possible-causes"></a>可能的原因

ID2D1DeviceCoNtext：： SetTarget 接受 ID2D1Image \* 參數。 Direct2D 影像可能是點陣圖、命令清單或效果。 Direct2D 只會將點陣圖和命令清單視為有效的目標。

## <a name="possible-fixes"></a>可能的修正

修改應用程式，以使用點陣圖或命令清單作為其裝置內容的目標。 Direct2D 不會使用 SetTarget 來執行效果的初始化。 相反地，若要選擇效果的內容，應用程式會使用 ID2D1DeviceCoNtext：： CreateEffect 來建立效果，並使用對應的效果 GUID。
