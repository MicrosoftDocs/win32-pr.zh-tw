# <a name="direct2d-helper-classes"></a>Direct2D helper 類別

D2D1 命名空間會定義下列類別。

## <a name="in-this-section"></a>本節內容



| 主題                                       | 描述                                                                                                                              |
|---------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| [**ColorF**](/windows/win32/api/d2d1helper/nl-d2d1helper-colorf)<br/>         | 定義一組表示已知色彩的常數，並提供可具現化不同色彩的便利方法。<br/>  |
| [**Matrix3x2F**](/windows/win32/api/d2d1helper/nl-d2d1helper-matrix3x2f)<br/> | [**Matrix3x2F**](/windows/win32/api/d2d1helper/nl-d2d1helper-matrix3x2f)類別代表3到2個矩陣，並提供建立矩陣的便利方法。<br/> |
| [**Matrix4x3F**](matrix4x3f.md)<br/> | [**Matrix4x3F**](matrix4x3f.md)類別代表 4 x 3 的矩陣，並提供建立矩陣的便利方法。<br/> |
| [**Matrix4x4F**](/windows/desktop/api/d2d1_1helper/nl-d2d1_1helper-matrix4x4f)<br/> | [**Matrix4x4F**](/windows/desktop/api/d2d1_1helper/nl-d2d1_1helper-matrix4x4f)類別代表4到4的矩陣，並提供建立矩陣的便利方法。<br/> |
| [**Matrix5x4F**](matrix5x4f.md)<br/> | [**Matrix5x4F**](matrix5x4f.md)類別代表一系列的4個矩陣，並提供建立矩陣的便利方法。<br/> |
