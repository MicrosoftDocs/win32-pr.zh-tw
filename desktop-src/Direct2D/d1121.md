# <a name="d1121-wrong-resource-domain"></a>D1121：資源網域錯誤

作業失敗，因為裝置相依的資源與錯誤的 ID2D1Device (資源網域) 相關聯。






 

## <a name="possible-causes"></a>可能的原因

當應用程式嘗試從不同資源網域混合資源的繪圖作業時，就會出現此訊息。 例如，裝置內容、筆刷和點陣圖都是裝置特定的。 執行諸如 ID2D1DeviceCoNtext：:D rawBitmap 這類作業時，引數點陣圖應屬於與裝置內容相同的資源網域。 在此範例中，您可以使用該裝置內容的 CreateBitmap 成員函式（而不是某些其他裝置內容）來建立點陣圖。

## <a name="possible-fixes"></a>可能的修正

當您應同時使用資源群組時，請務必從相同 ID2D1Device 建立資源群組。 從 ID2D1Factory （例如狀態欄塊和幾何）直接建立的非目標資源，會被視為與裝置無關，而且沒有這類限制。
