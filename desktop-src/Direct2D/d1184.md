# <a name="d1184-createwicbitmaprendertarget-incompatible-with-hw-target-type"></a>D1184： CreateWicBitmapRenderTarget 與 HW 目標型別不相容

將旗標 D2D1 轉譯 \_ \_ 目標 \_ 類型 \_ 硬體傳遞給 CreateWicBitmapRenderTarget 是不正確。






 

## <a name="possible-causes"></a>可能的原因

當名為 ID2D1Factory：： CreateWicBitmapRenderTarget 的應用程式時，D2D1 轉譯 \_ \_ 目標 \_ 屬性引數的類型欄位是 D2D1 \_ D2D1 轉譯 \_ \_ 目標 \_ 類型 \_ 硬體。

## <a name="possible-fixes"></a>可能的修正

確認建立 WIC interopped 轉譯目標時所使用的目標型別是 [軟體] 或 [預設]。 Direct2D 通常會保留硬體語義來描述 Direct3D interopability。
