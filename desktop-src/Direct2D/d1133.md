# <a name="d1133-buffer-precision-unknown-not-allowed"></a>D1133：不允許緩衝區精確度未知

D2D1 \_ 緩衝區有效 \_ 位數 \_ 未知，不允許作為此 API 的參數。






 

## <a name="possible-causes"></a>可能的原因

Direct2D 可讓應用程式針對某些作業指定所需的緩衝區精確度。 例如，使用 ID2D1DeviceCoNtext：： SetRenderingControls 來指定用於映射管線的精確度。 緩衝區 \_ 精確度 \_ 未知規範可讓應用程式放棄這項設定，而 Direct2D 會使用預設值，但某些作業不允許此規範。 尤其是在建立漸層停駐點時，可能不允許此情況。

## <a name="possible-fixes"></a>可能的修正

請確定緩衝區 \_ 精確度 \_ 未知的規範不會搭配 ID2D1DeviceCoNtext：： CreateGradientStopCollection 使用。 使用已知的有效位數值;請考慮為作業使用目標介面格式的有效位數。
