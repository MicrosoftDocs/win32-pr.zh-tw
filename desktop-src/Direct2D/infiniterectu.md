# <a name="infiniterectu-function"></a>InfiniteRectU 函式

建立矩形，其左上角設為 (0u，0u) ，其右下角設為 (無限大、無限大) 。

## <a name="syntax"></a>語法


```C++
D2D1_RECT_U InfiniteRectU(void);
```



## <a name="parameters"></a>參數

此函式沒有參數。

## <a name="return-value"></a>傳回值

類型： **[ **D2D1 \_ RECT \_ U**](d2d1-rect-u.md)**

新的矩形。
