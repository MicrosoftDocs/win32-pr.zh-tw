# <a name="d1130-dxgi-format-disallowed-by-caps"></a>D1130： Caps 不允許的 DXGI 格式

裝置沒有支援指定 DXGI 格式的功能。






 

## <a name="possible-causes"></a>可能的原因

針對在硬體上執行的 Direct2D 作業，以及使用介面格式，硬體的功能層級必須支援介面格式。 例如，如果硬體是功能層級9，它可能不支援16bpp 格式或 SRGB 格式，例如 DXGI \_ 格式 \_ B8G8R8A8 \_ UNORM \_ SRGB。

## <a name="possible-fixes"></a>可能的修正

請參閱 [Direct3D 功能等級](/previous-versions/windows/apps/hh994923(v=win.10)) ，以取得有關功能層級與其支援格式之間差異的指引。 請考慮使用 [**ID3D11Device：： CheckFormatSupport 方法**](/windows/desktop/api/d3d11/nf-d3d11-id3d11device-checkformatsupport) 來確認硬體支援所需的格式，再將它與 Direct2D 搭配使用。
