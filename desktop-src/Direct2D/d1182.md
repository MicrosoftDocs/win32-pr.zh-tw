# <a name="d1182-operation-on-unclosed-path-geometry"></a>D1182：未封閉路徑幾何的運算

未關閉的路徑幾何發生作業。






 

## <a name="possible-causes"></a>可能的原因

應用程式開始填入 ID2D1PathGeometry，但未在其填入接收上呼叫 ID2D1SimplifiedGeometrySink：： Close。 或者，應用程式呼叫 Close，但傳回錯誤值。

## <a name="possible-fixes"></a>可能的修正

確定已正確填入所有路徑幾何，方法是在其填入接收上呼叫 Close，並確保關閉不會傳回錯誤。 如需如何建立和填入路徑幾何物件的範例，請參閱 ID2D1PathGeometry 的參考。
