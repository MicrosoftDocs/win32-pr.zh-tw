# <a name="d1119-bitmap-bound-as-target"></a>D1119：點陣圖系結為目標

繪圖作業失敗，因為目標點陣圖和來源點陣圖相同。






 

## <a name="possible-causes"></a>可能的原因

此訊息的可能原因是應用程式嘗試使用相同點陣圖同時用於來源和目標的繪圖作業。 就語義而言，interoperated 具有系結 \_ 著色器資源和系結轉譯目標的 Direct3D 紋理可能是可行的 \_ \_ \_ 。

## <a name="possible-fixes"></a>可能的修正

確定應用程式不會嘗試使用其目標點陣圖作為繪圖作業的來源，包括效果。 在某些情況下，此訊息可能表示應用程式將其裝置內容的目標設定為與預期不同的點陣圖。
