# <a name="d1186-dc-render-target-requires-binddc"></a>D1186： DC 轉譯目標需要 BindDC

在呼叫其 BindDC 方法之前，使用 DC 轉譯目標是不正確。






 

## <a name="possible-causes"></a>可能的原因

應用程式已建立 ID2D1DCRenderTarget，並使用它來呼叫 BeginDraw，但尚未呼叫其 BindDC 方法。 或者，BindDC 造成錯誤。

## <a name="possible-fixes"></a>可能的修正

使用 DC interopped 轉譯目標之前，應用程式應該呼叫 BindDC，將目標與 GDI 裝置內容產生關聯。 在指派 GDI 裝置內容之前，目標不會被視為有效。 如需範例，請參閱 ID2D1DCRenderTarget：： BindDC。 如果應用程式已呼叫 BindDC，請確定 BindDC 未傳回錯誤。
