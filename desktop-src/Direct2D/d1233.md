# <a name="d1233-begindraw-invalid-from-previous-error"></a>D1233： BeginDraw 與先前的錯誤無效

已呼叫 BeginDraw，但因為裝置內容處於無效狀態，所以會予以忽略。






 

## <a name="possible-causes"></a>可能的原因

名為 BeginDraw 的應用程式，但裝置內容在先前的作業中處於無效狀態。

## <a name="possible-fixes"></a>可能的修正

請確定應用程式會適當地回應 Flush 或 EndDraw 所傳回的錯誤，並視需要重新建立其裝置。 如果命令造成裝置內容處於無效狀態，對應的 Flush 或 EndDraw 會傳回錯誤。
