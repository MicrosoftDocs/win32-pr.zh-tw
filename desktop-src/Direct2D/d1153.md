# <a name="d1153-closed-command-lists-cannot-be-set-as-targets"></a>D1153：關閉的命令清單無法設為目標

嘗試在目標上設定關閉的命令清單。






 

## <a name="possible-causes"></a>可能的原因

在關閉命令清單之後，應用程式不應該嘗試將它設為目標。 Direct2D 會將對 ID2D1DeviceCoNtext：： SetTarget 的呼叫視為錯誤，因為命令清單的內容在關閉後即無法變更。 將命令清單設定為目標，表示要呈現給它的意圖。

## <a name="possible-fixes"></a>可能的修正

若要解決此訊息，請勿使用已關閉的命令清單來呼叫 ID2D1DeviceCoNtext：： SetTarget。 如果需要使用具有新的不同內容的命令清單，請建立新的命令清單。
