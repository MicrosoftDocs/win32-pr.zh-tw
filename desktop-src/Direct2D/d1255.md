# <a name="d1255-tile-opacity-metadata-conflict"></a>D1255：磚不透明度中繼資料衝突

針對效果圖形計算的不透明矩形會與標示為透明的區域重迭。 這可能是因為使用的塔效果或界限調整轉換不當所造成。
