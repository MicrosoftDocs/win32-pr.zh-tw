# <a name="d1176-negative-dpi"></a>D1176：負 DPI

建立 rendertarget 時，指定負值 DPI 是錯誤的。






 

## <a name="possible-causes"></a>可能的原因

應用程式嘗試透過使用 ID2D1RenderTarget：： SetDpi 或建立具有負值 DPI 屬性的目標，來指定負 DPI。

## <a name="possible-fixes"></a>可能的修正

建立或修改轉譯目標時，請選擇有效的非負值值（DPI）。 如果可以接受預設 DPI，請避免在建立目標時呼叫 SetDpi 或指定 DPI 選項。 指定 DPI 0 相當於使用預設的 DPI。
