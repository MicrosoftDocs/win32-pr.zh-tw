# <a name="d1240-source-tranform-still-mapped"></a>D1240：來源轉換仍對應

傳遞給 ID2D1SourceTransform：:D raw 的點陣圖絕對不會對應。






 

## <a name="possible-causes"></a>可能的原因

名為 ID2D1Bitmap1：： Map 的應用程式，但未在傳遞給 ID2D1SourceTransform 的點陣圖上呼叫 ID2D1Bitmap1：：取消對應：:D raw。

## <a name="possible-fixes"></a>可能的修正

請務必在呼叫 Map 時呼叫取消對應。
