# <a name="d1163-print-bitmap-too-big"></a>D1163：列印點陣圖太大

點陣圖太大而無法列印，請考慮將它分割成較小的磚。






 

## <a name="possible-causes"></a>可能的原因

Direct2D 列印會限制可列印點陣圖的大小。 這項限制為65535裝置相依的單位。 如果應用程式在可列印的內容中包含超出此大小的點陣圖，Direct2D 將會報告錯誤，而且 debug 圖層會輸出此訊息。

## <a name="possible-fixes"></a>可能的修正

確定列印工作中使用的點陣圖不超過 Direct2D 列印的大小限制。 考慮將點陣圖分割成多個磚。
