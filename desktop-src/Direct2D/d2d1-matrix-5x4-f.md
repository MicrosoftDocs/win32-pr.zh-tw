# <a name="d2d1_matrix_5x4_f"></a>D2D1 \_ 矩陣 \_ 5X4 \_ F

表示5乘以4的矩陣。


```C++
typedef D2D_MATRIX_5X4_F D2D1_MATRIX_5X4_F;
```



## <a name="remarks"></a>備註

**D2D1 \_矩陣 \_ 5X4** 是 [**D2D \_ 矩陣 \_ 5X4 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_5x4_f) 結構的新名稱。 如需矩陣所提供的欄位清單，請參閱 [**D2D \_ 矩陣 \_ 5X4 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_5x4_f)。
