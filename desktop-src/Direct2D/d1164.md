# <a name="d1164-invalid-map-option"></a>D1164：不正確對應選項

D2D1 \_ 對應 \_ 選項 \_ NONE 不是對應的有效參數






 

## <a name="possible-causes"></a>可能的原因

如果應用程式使用 D2D1 \_ Map \_ OPTIONS NONE 選項呼叫 ID2D1Bitmap1：： Map，Direct2D 可能會輸出此訊息 \_ 。

## <a name="possible-fixes"></a>可能的修正

呼叫 ID2D1Bitmap1：： Map 時，請使用有效的對應選項。 \_如需對應選項的描述，請參閱 D2D1 對應 \_ 選項; 請考慮應用程式是否需要讀取或寫入存取權。
