# <a name="d1166-discard-map-option-requires-write-map-option"></a>D1166：捨棄對應選項需要 Write Map 選項

D2D1 \_ 對應 \_ 選項 \_ 捨棄必須結合 D2D1 \_ 對應 \_ 選項 \_ WRITE






 

## <a name="possible-causes"></a>可能的原因

如果應用程式呼叫 ID2D1Bitmap1：： Map 與 D2D1 \_ 對應 \_ 選項 \_ 捨棄但不是 D2D1 \_ 對應 \_ 選項 \_ 寫入，Direct2D 可能會輸出此訊息。

## <a name="possible-fixes"></a>可能的修正

捨棄選項需要 WRITE 才能成為應用程式很有用的作業。 如需 \_ 對應選項的描述，請參閱 D2D1 對應 \_ 選項。 呼叫 ID2D1Bitmap1：： Map 時，請考慮應用程式所需的存取類型。
