# <a name="id2d1devicecontextfillopacitymask-methods"></a>ID2D1DeviceCoNtext：： FillOpacityMask 方法

使用提供的不透明度遮罩點陣圖的 Alpha 色板填滿。 筆刷不透明度將由遮罩來進行調製。 轉譯目標消除鋸齒模式必須設定為別名。
