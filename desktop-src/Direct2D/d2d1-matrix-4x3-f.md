# <a name="d2d1_matrix_4x3_f"></a>D2D1 \_ 矩陣 \_ 4X3 \_ F

代表 4 x 3 的矩陣。


```C++
typedef D2D_MATRIX_4X3_F D2D1_MATRIX_4X3_F;
```



## <a name="remarks"></a>備註

**D2D1 \_矩陣 \_ 4X3** 是 [**D2D \_ 矩陣 \_ 4X3 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_4x3_f) 結構的新名稱。 如需矩陣所提供的欄位清單，請參閱 [**D2D \_ 矩陣 \_ 4X3 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_4x3_f)。
