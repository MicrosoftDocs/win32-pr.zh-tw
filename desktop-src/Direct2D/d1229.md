# <a name="d1229-d1229-immersive-call-to-getdesktopdpi"></a>D1229： D1229：沉浸式的 GetDesktopDpi 呼叫

ID2D1Factory：： GetDesktopDpi 是在 Windows Store 應用程式中呼叫。 不建議如此使用。 相反地，應該使用 DisplayProperties：： LogicalDpi。






 

## <a name="possible-causes"></a>可能的原因

ID2D1Factory：： GetDesktopDpi 是在 Windows Store 應用程式中呼叫。

## <a name="possible-fixes"></a>可能的修正

DisplayProperties：： LogicalDpi 應該用來取得 Windows Store 應用程式中的 DPI。
