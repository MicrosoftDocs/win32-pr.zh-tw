# <a name="d1179-gradient-stop-collection-zero-stops"></a>D1179：漸層停駐點集合零停止

建立 ID2D1GradientStopCollection 需要至少一個漸層停駐點。






 

## <a name="possible-causes"></a>可能的原因

名為 ID2D1RenderTarget：： CreateGradientStopCollection 的應用程式，但指定的漸層停駐點計數為0。

## <a name="possible-fixes"></a>可能的修正

建立 ID2D1GradientStopCollection 時，應用程式至少應以其色彩和位置指定一個漸層停駐點。 如需範例，請參閱 ID2D1RenderTarget：： CreateGradientStopCollection。
