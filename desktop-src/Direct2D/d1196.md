# <a name="d1196-a8-shared-bitmap"></a>D1196： A8 共用點陣圖

\_ \_ 此 API 不支援格式為 dxgi 格式 A8 UNORM 的 dxgi 介面 \_ 。





 

## <a name="possible-causes"></a>可能的原因

\_ \_ \_ 不支援使用 dxgi 格式 A8 UNORM 作為 dxgi interop 點陣圖格式，即使在其他 api 中支援它也是一樣。

## <a name="possible-fixes"></a>可能的修正

建立不同格式的 DXGI interop 點陣圖。
