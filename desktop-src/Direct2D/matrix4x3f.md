# <a name="matrix4x3f-class"></a>Matrix4x3F 類別

**Matrix4x3F** 類別代表 4 x 3 的矩陣，並提供建立矩陣的便利方法。

## <a name="members"></a>成員

**Matrix4x3F** 類別繼承自 [**D2D1 \_ 矩陣 \_ 4X3 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_4x3_f)。 **Matrix4x3F** 也有下列類型的成員：

-   [建構函式](#constructors)

### <a name="constructors"></a>建構函式

**Matrix4x3F** 類別具有這些函式。



| 建構函式                                                                                                                                                                                           | 描述                                                                                                                        |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------|
| [**Matrix4x3F ()**](matrix4x3f-matrix4x3f--.md)                                                                                                                                                       | 將未初始化之 **Matrix4x3F** 類別的新實例具現化。<br/>                                                   |
| [**Matrix4x3F (FLOAT，FLOAT，FLOAT，FLOAT，FLOAT，FLOAT，FLOAT，FLOAT，FLOAT，FLOAT，FLOAT，FLOAT)  (FLOAT，FLOAT，float，float，float，float，float，float，float，float，float，FLOAT)**](matrix4x3f-matrix4x3f-floats-.md) | 具現化 **Matrix4x3F** 類別的新實例，這個實例會使用所有浮點數矩陣值進行初始化。<br/> |
