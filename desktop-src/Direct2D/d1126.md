# <a name="d1126-alpha-mode-unknown-not-allowed"></a>D1126：不允許未知的 Alpha 模式

未知的 D2D1 \_ ALPHA \_ 模式 \_ 不允許作為此 API 的參數。





 

## <a name="possible-causes"></a>可能的原因

某些內容 \_ 中允許使用 ALPHA 模式 \_ 未知的規範，例如 ID2D1RenderTarget：： CreateCompatibleRenderTarget 的格式描述。 不過，類似于 DXGI \_ 格式 \_ 不明，Direct2D 通常需要完整指定格式。

## <a name="possible-fixes"></a>可能的修正

當您決定格式規範時，請考慮表面的 Alpha 處理方式，以使用 ALPHA 模式以外的 Alpha \_ 模式 \_ \_ 。 \_ \_ \_ 如果介面未使用 Alpha，請使用 D2D1 的 ALPHA 模式 [略過]。 請參閱 [**D2D1 \_ ALPHA \_ 模式**](/windows/desktop/api/dcommon/ne-dcommon-d2d1_alpha_mode)中的備註。
