# <a name="d1178-custom-dash-style-requires-dash-array"></a>D1178：自訂虛線樣式需要虛線陣列

以虛線 \_ 樣式自訂建立筆觸樣式 \_ 需要有虛線陣列。






 

## <a name="possible-causes"></a>可能的原因

當應用程式嘗試建立 ID2D1StrokeStyle 時，它會使用 D2D1 \_ 虛線 \_ 樣式的 \_ 自訂屬性，但提供的是 Null 虛線陣列。

## <a name="possible-fixes"></a>可能的修正

如果應用程式需要自訂的連字號，請確定已指定有效的虛線陣列。 如需範例，請參閱 CreateStrokeStyle。 如果可接受內建的虛線樣式，請將虛線陣列保留為 Null，並使用 D2D1 \_ 虛線樣式虛線之類的規範 \_ \_ 。
