# <a name="d1251-d3d-state-issue"></a>D1251： D3D 狀態問題

當 Direct2D 仍在使用時，Direct3D 裝置的狀態已變更。 這可能表示執行緒違規，因為一次只有一個執行緒可以使用裝置。






 

## <a name="possible-causes"></a>可能的原因

應用程式可能會同時在兩個不同的執行緒上使用單一執行緒的 D2D Factory。

## <a name="possible-fixes"></a>可能的修正

請確定應用程式已正確遵循 Direct2d 執行緒原則。
