# <a name="d1175-dxgi-target-must-not-use-software-flag"></a>D1175： DXGI 目標不得使用軟體旗標

這種類型的 rendertarget 必須標示為預設或硬體。






 

## <a name="possible-causes"></a>可能的原因

名為 ID2D1Factory：： CreateDxgiSurfaceRenderTarget 的應用程式， \_ \_ \_ 其類型欄位設定為 D2D1 轉譯 \_ \_ 目標 \_ 類型 \_ 軟體的 D2D1 呈現目標屬性物件。 Direct2D 會將此規範組合視為不正確，因為軟體旗標是針對使用 CreateWicBitmapRenderTarget 和 CreateHwndRenderTarget 建立的目標保留。

## <a name="possible-fixes"></a>可能的修正

使用 CreateDxgiSurfaceRenderTarget 時，應用程式應指定 D2D1 \_ 轉譯 \_ 目標 \_ 類型 \_ 硬體。
