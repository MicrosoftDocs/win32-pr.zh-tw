# <a name="d1171-redundant-call-to-map-bitmap"></a>D1171：對應點陣圖的多餘呼叫

嘗試對應已對應的點陣圖。






 

## <a name="possible-causes"></a>可能的原因

Direct2D 可能會輸出這則訊息，並在應用程式嘗試對應已經使用 ID2D1Bitmap1：： Map 對應的點陣圖時回報錯誤。

## <a name="possible-fixes"></a>可能的修正

如同 ID2D1RenderTarget：： EndDraw 應遵循對 ID2D1RenderTarget：： BeginDraw 的呼叫，請確定 ID2D1Bitmap1：：取消對應遵循對 ID2D1Bitmap1：： Map 的呼叫。 如果應用程式使用回呼或遞迴程式碼路徑來對應點陣圖，請確定其行為正確。
