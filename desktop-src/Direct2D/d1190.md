# <a name="d1190-cannotdraw-bitmap-flag-was-used-incorrectly"></a>D1190： CannotDraw 點陣圖旗標使用錯誤

D2D1 \_ 點陣圖 \_ 選項 \_ 無法 \_ 繪圖必須與 D2D1 \_ BITMAP \_ options \_ CPU \_ READ 或 D2D1 \_ 點陣圖 \_ 選項 \_ 目標結合





 

## <a name="possible-causes"></a>可能的原因

應用程式指定了不相容的 D2D1 \_ 點陣圖 \_ 選項旗標組合。

## <a name="possible-fixes"></a>可能的修正

將 D2D1 \_ 點陣圖 \_ 選項 \_ CPU \_ READ 或 D2D1 \_ BITMAP \_ 選項 \_ 目標新增至 D2D1 \_ 點陣圖 \_ 選項
