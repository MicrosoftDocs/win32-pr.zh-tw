# <a name="d1250-inaccessible-source"></a>D1250：無法存取的來源

偵測到嘗試從無法存取的點陣圖進行繪製。






 

## <a name="possible-causes"></a>可能的原因

偵測到嘗試從無法存取的來源進行繪製。 可能 inacessible 來源的原因之一，是因為它是從共用材質建立的，而且未取得紋理。

## <a name="possible-fixes"></a>可能的修正

請先確定所有點陣圖都可供存取，然後再使用 Direct2D 進行繪製。
