# <a name="d1146-invalid-render-controls"></a>D1146：不正確轉譯控制項

指定的轉譯控制項有不正確參數，將會予以忽略。






 

## <a name="possible-causes"></a>可能的原因

如果應用程式指定不正確緩衝區有效位數值，或圖格大小為零，則在 ID2D1DeviceCoNtext：： SetRenderingControls 期間可能會出現此訊息。

## <a name="possible-fixes"></a>可能的修正

請確定應用程式使用有效的緩衝區有效位數值。 請參閱 [**D2D1 \_ 緩衝區 \_ 精確度**](/windows/desktop/api/D2d1_1/ne-d2d1_1-d2d1_buffer_precision)。 確定應用程式指定非零的磚大小。
