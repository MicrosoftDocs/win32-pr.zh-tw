# <a name="d1161-copyfrommemory-requires-cpu-readable-bitmap"></a>D1161： CopyFromMemory 需要 CPU 可讀取點陣圖

使用 [**D2D1 \_ 點陣圖 \_ 選項 \_ CPU \_ READ**](/windows/desktop/api/D2d1_1/ne-d2d1_1-d2d1_bitmap_options)建立的點陣圖上不允許使用這個方法。






 

## <a name="possible-causes"></a>可能的原因

此訊息可能會在呼叫 [**ID2D1Bitmap：： CopyFromMemory**](/windows/win32/api/d2d1/nf-d2d1-id2d1bitmap-copyfrommemory)時發生。 如果點陣圖是使用 D2D1 \_ 點陣圖 \_ 選項 \_ CPU 讀取所建立 \_ ，Direct2D 將會報告錯誤，而且 debug 層會輸出此訊息。

## <a name="possible-fixes"></a>可能的修正

如果應用程式需要呼叫 [**ID2D1Bitmap：： CopyFromMemory**](/windows/win32/api/d2d1/nf-d2d1-id2d1bitmap-copyfrommemory)，請確定點陣圖未建立為 CPU 可讀取。 如果無法變更點陣圖的介面屬性，請考慮是否有對等的轉譯作業需要結果。
