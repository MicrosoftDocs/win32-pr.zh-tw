# <a name="d1249-inaccessible-target"></a>D1249：無法存取的目標

偵測到嘗試對無法存取的目標進行繪製。






 

## <a name="possible-causes"></a>可能的原因

偵測到嘗試對無法存取的目標進行繪製。 可能無法存取目標的原因之一，是因為它是從共用材質建立的，而且未取得紋理。

## <a name="possible-fixes"></a>可能的修正

請先確定所有目標都可供存取，然後再使用 Direct2D 來呈現它們。
