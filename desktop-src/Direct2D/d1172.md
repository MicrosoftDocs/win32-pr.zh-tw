# <a name="d1172-invalid-to-unmap-nonmapped-bitmap"></a>D1172：不正確對應 Nonmapped 點陣圖

嘗試取消對應未對應的點陣圖。






 

## <a name="possible-causes"></a>可能的原因

如果應用程式嘗試取消對應尚未使用 ID2D1Bitmap1：： Map 對應的點陣圖，Direct2D 可能會輸出此訊息並回報錯誤。

## <a name="possible-fixes"></a>可能的修正

如同 ID2D1RenderTarget：： EndDraw 應遵循對 ID2D1RenderTarget：： BeginDraw 的呼叫，請確定 ID2D1Bitmap1：：取消對應遵循對 ID2D1Bitmap1：： Map 的呼叫。 請勿呼叫 ID2D1Bitmap1：：取消對應多次以取消點陣圖的對應。
