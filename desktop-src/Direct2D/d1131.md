# <a name="d1131-dxgi-format-not-gdi-compatible"></a>D1131： DXGI 格式與 GDI 不相容

GDI 相容選項已傳遞至此 API，但 DXGI 格式與 GDI 不相容。






 

## <a name="possible-causes"></a>可能的原因

\_ \_ 點陣圖建立期間使用 D2D1 點陣圖選項 \_ GDI \_ 相容旗標，可讓點陣圖與 GDI 轉譯相容，但是會對點陣圖的介面格式引進額外的限制。 一般而言，GDI 轉譯與 DXGI \_ 格式相容 \_ B8G8R8A8 \_ UNORM 和 dxgi \_ format \_ B8G8R8A8 \_ UNORM SRGB 格式 \_ 。

## <a name="possible-fixes"></a>可能的修正

若要解決這個問題，請確認為 GDI 相容點陣圖指定的格式為四通道 BGRA 格式，如上面所指定。 如果點陣圖不需要與 GDI 之間的互通性，請省略與 GDI \_ 相容的規範。
