# <a name="d1238-source-transform-single-channel-srgb"></a>D1238：來源轉換單一通道 SRGB

ID2D1SourceTransform 不允許將 D2D1 \_ 緩衝區 \_ 精確度 \_ 的 SRGB 與 D2D1 \_ 通道 \_ 深度1結合 \_ 。






 

## <a name="possible-causes"></a>可能的原因

應用程式 \_ 在 ID2D1SourceTransform 上指定了 D2D1 緩衝區 \_ 精確度 \_ SRGB 和 D2D1 \_ 通道 \_ 深度 \_ 1。

## <a name="possible-fixes"></a>可能的修正

\_ \_ \_ 應改為使用 D2D1 通道深度預設或 D2D1 \_ 通道 \_ 深度 \_ 4。
