# <a name="d1174-pushlayer-to-command-list-will-ignore-layer-object"></a>D1174： PushLayer 至命令清單會忽略圖層物件

將 (非 Null) 層推送至命令清單時，將會忽略 layer 物件。



|             |         |
|-------------|---------|
| 錯誤層級 | 警告 |



 

## <a name="possible-causes"></a>可能的原因

當 PushLayer 命令使用 ID2D1Layer 物件，且目標是命令清單時，Direct2D 的行為會如同未指定任何層級，而且如果稍後轉譯命令清單，將不會在圖層上強制固定 boundries。

## <a name="possible-fixes"></a>可能的修正

預期對命令清單進行串流命令的應用程式應該考慮使用 Null 層物件，否則請留意 ID2D1Layer 物件中指定的大小限制設定將不會被接受。
