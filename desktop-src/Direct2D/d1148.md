# <a name="d1148-invalid-operation-on-closed-print-control"></a>D1148：關閉的列印控制項上有不正確操作

Direct2D Print：在關閉 [**ID2D1PrintControl**](/windows/win32/api/d2d1_1/nn-d2d1_1-id2d1printcontrol)之後呼叫 [**AddPage**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1printcontrol-addpage)或 [**Close**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1printcontrol-close)






 

## <a name="possible-causes"></a>可能的原因

如果應用程式在已關閉的列印控制項上呼叫 ID2D1PrintControl：： AddPage，就可能會出現此訊息。 Direct2D 會在關閉時將列印控制項物件視為無法變更，其方式與路徑幾何和命令清單類似。

## <a name="possible-fixes"></a>可能的修正

確定應用程式不會嘗試將頁面加入至關閉的列印控制項。 修改應用程式邏輯，以便在關閉之前將所有必要的頁面傳送至列印控制項。
