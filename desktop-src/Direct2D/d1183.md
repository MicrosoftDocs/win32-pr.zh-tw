# <a name="d1183-createwicbitmaprendertarget-with-incompatible-feature-level"></a>D1183：具有不相容功能層級的 CreateWicBitmapRenderTarget

將 D2D1 功能層級以外的 Direct2D 功能層級傳遞 \_ 至 CreateWicBitmapRenderTarget 是不正確 \_ \_ 。






 

## <a name="possible-causes"></a>可能的原因

當名為 ID2D1Factory：： CreateWicBitmapRenderTarget 的應用程式時，D2D1 轉譯 \_ \_ 目標 \_ 屬性引數的針對 minlevel 欄位不是 D2D1 \_ 功能 \_ 等級的 \_ 預設值。

## <a name="possible-fixes"></a>可能的修正

確認建立 WIC interopped 轉譯目標時所使用的功能層級是預設值。 與 WIC 的互通性牽涉到僅限軟體的功能;功能層級的語法與硬體行為有關。
