# <a name="d1199-changed-command-list-resource"></a>D1199：已變更命令清單資源

繪製到命令清單的影像剛剛已經變更。 這可能會產生非預期的結果。 命令清單現在會包含已變更的影像，而不是繪製時影像的內容。






 

## <a name="possible-causes"></a>可能的原因

應用程式已將影像新增至命令清單，之後變更了映射。

## <a name="possible-fixes"></a>可能的修正

確認產生的行為符合預期。 在此情況下，如果命令清單是目標，其行為將會產生不同的結果，而非點陣圖已設定為目標。 如果行為不是預期的行為，應用程式可能會想要在更新映射之前先複製映射。
