# <a name="d1191-gdi-compatible-bitmap-option-was-used-incorrectly"></a>D1191： GDI 相容點陣圖選項的使用方式不正確

D2D1 \_ 點陣圖 \_ 選項 \_ GDI \_ 相容必須結合 D2D1 \_ 點陣圖 \_ 選項 \_ 目標






 

## <a name="possible-causes"></a>可能的原因

應用程式指定了不相容的 D2D1 \_ 點陣圖 \_ 選項旗標組合。

## <a name="possible-fixes"></a>可能的修正

將 D2D1 \_ 點陣圖 \_ 選項 \_ 的目標新增至 D2D1 \_ 點陣圖 \_ 選項旗標。
