# <a name="d1145-print-control-dpi-incorrect"></a>D1145：列印控制項 DPI 不正確

Direct2D Print： PRINT \_ CONTROL \_ 屬性。 rasterDPI 值無效。






 

## <a name="possible-causes"></a>可能的原因

在建立列印控制項期間，此訊息表示應用程式指定了不正確 DPI;尤其是小於1.0 的 DPI。 雖然 Direct2D 不會將這視為錯誤，Direct2D Print 的行為就如同應用程式指定了150.0 的 DPI 一樣。

## <a name="possible-fixes"></a>可能的修正

若要解決此訊息，請在建立列印控制項時使用1.0 或更高的 DPI。
