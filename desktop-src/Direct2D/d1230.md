# <a name="d1230-unknown-buffer-precision-support"></a>D1230：未知的緩衝區精確度支援

呼叫 IsBufferPrecisionSupported 時，D2D1 \_ 緩衝區有效 \_ 位數 \_ 不明。 此 API 會傳回 false，但 \_ D2D1 \_ 緩衝區 \_ 有效位數 UNKNOWN 可能是對某些 api 的輸入有效。






 

## <a name="possible-causes"></a>可能的原因

呼叫 IsBufferPrecisionSupported 時，D2D1 \_ 緩衝區有效 \_ 位數 \_ 不明。

## <a name="possible-fixes"></a>可能的修正

藉 \_ \_ \_ 由閱讀該 api 的檔，而非嘗試以程式設計方式判斷，判斷 D2D1 緩衝區有效位數不明是否為指定 api 的有效緩衝區有效位數。
