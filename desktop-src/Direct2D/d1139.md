# <a name="d1139-unused-stroke-width"></a>D1139：未使用的筆觸寬度

已指定細項的筆劃轉換類型，但提供的筆觸寬度不是1。 筆劃寬度將會被忽略。






 

## <a name="possible-causes"></a>可能的原因

應用程式使用 ID2D1StrokeStyle 搭配 D2D1 \_ 筆觸轉換類型來繪製 \_ 基本 \_ 類型 \_ ，但該繪製呼叫中使用的筆觸寬度不是1。 指定最小的屬性時，它實際上會覆寫筆劃寬度，並以1.0 裝置相依的單位來修正它，而不論縮放轉換或 DPI 選擇為何。

## <a name="possible-fixes"></a>可能的修正

在 ID2D1StrokeStyle 上設定細項屬性時，應用程式應該使用1.0 單位的筆觸寬度。
