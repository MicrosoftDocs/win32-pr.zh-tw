# <a name="d1162-fillmesh-requires-aliased-rendering"></a>D1162： FillMesh 需要別名呈現

FillMesh 需要將消除鋸齒模式設定為 D2D1 的鋸齒 \_ 消除 \_ 模式 \_ 。






 

## <a name="possible-causes"></a>可能的原因

當應用程式呼叫 ID2D1RenderTarget：： FillMesh 時，相關 rendertarget 的消除鋸齒模式應該是 D2D1 的 \_ 消除鋸齒 \_ 模式 \_ 。 轉譯目標的預設行為是使用消除鋸齒，因此在填滿網格之前，必須先呼叫 ID2D1RenderTarget：： SetAntialiasMode。

## <a name="possible-fixes"></a>可能的修正

使用 ID2D1RenderTarget：： SetAntialiasMode，以確保 rendertarget 在填滿網格時使用別名呈現。 如果後續的作業需要消除鋸齒，請將目標的消除鋸齒模式設定為 \_ \_ \_ 每個基本 D2D1 的消除鋸齒模式 \_ 。 確定對 ID2D1RenderTarget：： SetAntialiasMode 和 ID2D1RenderTarget：： FillMesh 發出的呼叫都是在相同的目標上發出。
