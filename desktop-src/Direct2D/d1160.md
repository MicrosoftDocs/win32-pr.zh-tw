# <a name="d1160-cycle-in-graph"></a>D1160：迴圈 Graph

嘗試繪製具有迴圈的影像圖形。






 

## <a name="possible-causes"></a>可能的原因

Direct2D 可讓應用程式在效果的行為中，建立效果圖形的彈性。 但是，應用程式不應該建立包含迴圈的效果圖形。 例如，假設有一個效果圖形包含節點 A、B 和 C：將 A、b 和 c 連結到 B，然後將 C 連結至 A 也是不正確。 Direct2D 會識別迴圈效果圖形並回報錯誤;debug 層會輸出此訊息。

## <a name="possible-fixes"></a>可能的修正

若要避免這則訊息，請避免建立具有迴圈的效果圖形。 有效設計效果圖形，包括至少一個作為輸出的節點。
