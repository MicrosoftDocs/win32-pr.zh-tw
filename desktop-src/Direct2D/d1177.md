# <a name="d1177-unsupported-wic-pixel-format"></a>D1177：不支援的 WIC 像素格式

Direct2D 不支援所提供的 WIC 像素格式。 尤其是，Direct2D 不支援使用直接 Alpha 的 WIC 點陣圖，除非深度為8bpp。






 

## <a name="possible-causes"></a>可能的原因

應用程式嘗試交互操作 Direct2D 不支援其格式的 WIC 資源。 這些格式通常是具有直線的格式，且深度不是每個平面的8位。

## <a name="possible-fixes"></a>可能的修正

請考慮使用替代格式來建立 WIC 資源，以與 Direct2D 互通或進行轉換。
