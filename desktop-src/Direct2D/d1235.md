# <a name="d1235-target-rendered-to-from-multiple-dcs"></a>D1235：從多個 Dc 轉譯的目標

嘗試在影像仍由先前的裝置內容轉譯時，轉譯為影像。 在使用目前的裝置內容轉譯之前，必須先在該裝置內容上呼叫 EndDraw。 使用 debug 層時，此裝置內容的位址也會輸出。
