# <a name="d1197-wrong-d3d-device"></a>D1197：錯誤的 D3D 裝置

傳遞給此 API 的 DXGI 介面來自錯誤的裝置。



|             |         |
|-------------|---------|
| 錯誤層級 | 警告 |



 

## <a name="possible-causes"></a>可能的原因

傳遞給此 API 的 DXGI 介面來自錯誤的裝置。

## <a name="possible-fixes"></a>可能的修正
