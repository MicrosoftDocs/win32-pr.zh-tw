## <a name="possible-causes"></a>可能的原因

當 Direct2D 無法產生具有應用程式所指定之參數的硬體目標資源時，就會出現此訊息。 可能的原因是應用程式要求的硬體目標是 Direct2D 必須取得配置目標材質的擁有權，例如透過 [**ID2D1Factory：： CreateHwndRenderTarget**](/windows/win32/api/d2d1/nf-d2d1-id2d1factory-createhwndrendertarget(constd2d1_render_target_properties__constd2d1_hwnd_render_target_properties__id2d1hwndrendertarget))，且要求的大小太大。 每個 Direct3D 功能層級都有紋理大小的限制。 請參閱 [Direct3D 功能等級](/previous-versions/windows/apps/hh994923(v=win.10))。

## <a name="possible-fixes"></a>可能的修正

配置目標時，請確定目標大小未超過功能層級所允許的大小上限。 若要更精確地針對目標配置問題進行疑難排解，請考慮修改應用程式以配置目標紋理本身，並確認配置成功。 若要直接使用該材質做為目標，請將它與 [**ID2D1Bitmap**](/windows/win32/api/d2d1/nn-d2d1-id2d1bitmap) 交互操作，並使用 **ID2D1Bitmap** 做為 [**ID2D1DeviceCoNtext：： SetTarget**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1devicecontext-settarget)的引數。
