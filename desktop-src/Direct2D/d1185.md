# <a name="d1185-fillgeometry-opacity-mask-brush-restriction"></a>D1185： FillGeometry 不透明度遮罩筆刷限制

如果使用 FillGeometry 搭配不透明度遮罩，則主要筆刷必須是壓制點陣圖筆刷。






 

## <a name="possible-causes"></a>可能的原因

使用 ID2D1RenderTarget：： FillGeometry 時，主要筆刷是 ID2D1BitmapBrush 以外的類型。 或者，主要筆刷的 x 或 y 擴充模式不是 D2D1 \_ 延伸 \_ 模式的 \_ 夾具。

## <a name="possible-fixes"></a>可能的修正

如果應用程式使用具有不透明度遮罩的 FillGeometry，請確定主要筆刷 (不會與不透明度筆刷混淆) 是具有壓制延伸模式的 ID2D1BitmapBrush。 如果不透明度遮罩行為很重要，請將不透明度筆刷引數保留為 Null，以移除這些限制對主要筆刷的限制。
