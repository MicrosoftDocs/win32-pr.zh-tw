## <a name="remarks"></a><span data-ttu-id="6b255-117">備註</span><span class="sxs-lookup"><span data-stu-id="6b255-117">Remarks</span></span>

<span data-ttu-id="6b255-118">[**大綱**](/windows/win32/api/d2d1/nf-d2d1-id2d1geometry-outline(constd2d1_matrix_3x2_f__id2d1simplifiedgeometrysink))方法可讓呼叫者產生具有對輸入幾何之對等填滿的幾何，並具有下列額外屬性：</span><span class="sxs-lookup"><span data-stu-id="6b255-118">The [**Outline**](/windows/win32/api/d2d1/nf-d2d1-id2d1geometry-outline(constd2d1_matrix_3x2_f__id2d1simplifiedgeometrysink)) method allows the caller to produce a geometry with an equivalent fill to the input geometry, with the following additional properties:</span></span>

-   <span data-ttu-id="6b255-119">輸出幾何不包含任何橫向交集;也就是說，區段可以接觸，但永遠不會交叉。</span><span class="sxs-lookup"><span data-stu-id="6b255-119">The output geometry contains no transverse intersections; that is, segments may touch, but they never cross.</span></span>
-   <span data-ttu-id="6b255-120">輸出幾何中的最外層圖形都是逆時針方向的。</span><span class="sxs-lookup"><span data-stu-id="6b255-120">The outermost figures in the output geometry are all oriented counterclockwise.</span></span>
-   <span data-ttu-id="6b255-121">輸出幾何為填滿模式不變;也就是說，幾何的填滿不依賴填滿模式的選擇。</span><span class="sxs-lookup"><span data-stu-id="6b255-121">The output geometry is fill-mode invariant; that is, the fill of the geometry does not depend on the choice of the fill mode.</span></span> <span data-ttu-id="6b255-122">如需填滿模式的詳細資訊，請參閱 [**D2D1 \_ 填滿 \_ 模式**](/windows/desktop/api/d2d1/ne-d2d1-d2d1_fill_mode)。</span><span class="sxs-lookup"><span data-stu-id="6b255-122">For more information about the fill mode, see [**D2D1\_FILL\_MODE**](/windows/desktop/api/d2d1/ne-d2d1-d2d1_fill_mode).</span></span>

<span data-ttu-id="6b255-123">此外， [**外框**](/windows/win32/api/d2d1/nf-d2d1-id2d1geometry-outline(constd2d1_matrix_3x2_f__id2d1simplifiedgeometrysink)) 方法可能有助於移除已被提到幾何的多餘部分，以簡化複雜的幾何。</span><span class="sxs-lookup"><span data-stu-id="6b255-123">Additionally, the [**Outline**](/windows/win32/api/d2d1/nf-d2d1-id2d1geometry-outline(constd2d1_matrix_3x2_f__id2d1simplifiedgeometrysink)) method can be useful in removing redundant portions of said geometries to simplify complex geometries.</span></span> <span data-ttu-id="6b255-124">它也可以與 [**ID2D1GeometryGroup**](/windows/win32/api/d2d1/nn-d2d1-id2d1geometrygroup) 搭配使用，以同時在多個幾何之間建立聯集。</span><span class="sxs-lookup"><span data-stu-id="6b255-124">It can also be useful in combination with [**ID2D1GeometryGroup**](/windows/win32/api/d2d1/nn-d2d1-id2d1geometrygroup) to create unions among several geometries simultaneously.</span></span>

## <a name="examples"></a><span data-ttu-id="6b255-125">範例</span><span class="sxs-lookup"><span data-stu-id="6b255-125">Examples</span></span>

<span data-ttu-id="6b255-126">下列程式碼示範如何使用 **外框** 來建立沒有自我交集的相等幾何。</span><span class="sxs-lookup"><span data-stu-id="6b255-126">The following code shows how to use **Outline** to construct an equivalent geometry with no self-intersections.</span></span> <span data-ttu-id="6b255-127">它會使用預設的簡維容錯功能，因此不應該與非常小的幾何一起使用。</span><span class="sxs-lookup"><span data-stu-id="6b255-127">It uses the default flattening tolerance and hence should not be used with very small geometries.</span></span>


```C++
HRESULT D2DOutline(
    ID2D1Geometry *pGeometry,
    ID2D1Geometry **ppGeometry
    )
{
    HRESULT hr;
    ID2D1Factory *pFactory = NULL;
    pGeometry->GetFactory(&pFactory);

    ID2D1PathGeometry *pPathGeometry = NULL;
    hr = pFactory->CreatePathGeometry(&pPathGeometry);

    if (SUCCEEDED(hr))
    {
        ID2D1GeometrySink *pSink = NULL;
        hr = pPathGeometry->Open(&pSink);

        if (SUCCEEDED(hr))
        {
            hr = pGeometry->Outline(NULL, pSink);

            if (SUCCEEDED(hr))
            {
                hr = pSink->Close();

                if (SUCCEEDED(hr))
                {
                    *ppGeometry = pPathGeometry;
                    (*ppGeometry)->AddRef();
                }
            }
            pSink->Release();
        }
        pPathGeometry->Release();
    }

    pFactory->Release();

    return hr;
}
```
