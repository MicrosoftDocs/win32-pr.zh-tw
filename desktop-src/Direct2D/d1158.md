# <a name="d1158-pixels-unit-mode-unsupported-with-print"></a>D1158：不支援使用列印的圖元單位模式

Direct2D Print：傳送給列印的命令清單中出現不支援的 SetUnitMode (圖元) 命令。



|             |             |
|-------------|-------------|
| 錯誤層級 | 資訊 |



 

## <a name="possible-causes"></a>可能的原因

Direct2D 列印不允許圖元的單位模式。 預設單位模式為 DIP。

## <a name="possible-fixes"></a>可能的修正

若要解決此訊息，請避免在建立可列印的內容時，將裝置內容的單位模式設定為圖元。 如果必須套用會影響所有基本繪圖作業的全域調整轉換，請考慮對 ID2D1RenderTarget：： SetDpi 或 ID2D1RenderTarget：： SetTransform 的對等呼叫是否足夠。
