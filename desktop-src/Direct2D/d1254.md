# <a name="d1254-operation-on-single-threaded-domain"></a>D1254：單一執行緒網域上的運算

對單一執行緒處理站網域呼叫 [**ID2D1Multithread：： Enter**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1multithread-enter) 或 [**ID2D1Multithread：： Leave**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1multithread-leave) 。

如果應用程式不慎建立了具有 [**D2D1 \_ factory \_ 類型 \_ 多 \_ 執行緒**](/windows/desktop/api/d2d1/ne-d2d1-d2d1_factory_type)的 D2D factory，則可能會發生這種情況。 在此情況下，將會忽略呼叫。
