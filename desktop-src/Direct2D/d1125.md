傳遞給這個 API 的像素格式與介面的像素格式不相容 \[ \] 。

## <a name="possible-causes"></a>可能的原因

ID2D1RenderTarget：： CreateBitmapFromWicBitmap 或 ID2D1RenderTarget：： CreateDxgiSurfaceRendertarget 等作業都需要介面和指定的介面格式。 Direct2D 會驗證指定的介面類別型是否與介面相容。 如果不相容，Direct2D 會輸出此訊息。

## <a name="possible-fixes"></a>可能的修正

請確定指定的 DXGI \_ 格式符合或相容于基礎介面的類型。 針對應用程式所建立的 Direct3D 介面資源，請使用 \_ 用來建立資源的相同 DXGI 格式。 如需其他指引，請參閱 [支援的像素格式和 Alpha 模式](supported-pixel-formats-and-alpha-modes.md) 。
