# <a name="d1154-invalid-bitmap-input-format"></a>D1154：不正確點陣圖輸入格式

這項作業與點陣圖的像素格式不相容。






 

## <a name="possible-causes"></a>可能的原因

Direct2D 支援僅限 Alpha 的表面格式，例如 DXGI \_ 格式 \_ A8 \_ UNORM，但只能用來做為不透明度的筆刷或遮罩。 如果應用程式建立點陣圖筆刷，其點陣圖具有僅限英數位元的格式，則不應嘗試搭配需要色彩筆刷的作業使用筆刷。 ID2D1RenderTarget：:D rawLine 和 ID2D1RenderTarget：： Graphicswindow.fillrectangle，例如使用色彩筆刷。

## <a name="possible-fixes"></a>可能的修正

若要解決此訊息，請確定使用僅限英文格式的點陣圖僅用於不透明度的筆刷或遮罩。 如果點陣圖應該包含色彩資訊，請使用包含色彩通道的表面格式來建立它。
