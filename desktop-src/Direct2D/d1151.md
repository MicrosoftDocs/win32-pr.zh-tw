# <a name="d1151-unsupported-array-selection"></a>D1151：不支援的陣列選取範圍

提供的 IDXGISurface2 屬於陣列，但此裝置層級不支援陣列配量 (D3D9) 。






 

## <a name="possible-causes"></a>可能的原因

Interopped Direct3D 材質可能是陣列;Direct2D 一次只會在一個陣列配量上運作。 父紋理的其他陣列配量不會受到 Direct2D 的影響或轉譯。 但是，功能層級9裝置不支援陣列配量選取，也無法使用此互通性功能。

## <a name="possible-fixes"></a>可能的修正

針對預期在功能層級9裝置上正常執行的應用程式，請考慮修改轉譯邏輯，以避免需要互運算元組紋理。 可能的話，請改為與中繼互動。 為了將解決這項限制的影響降到最低，應用程式可能會選擇將其實作為功能層級9特定的回溯路徑。
