# <a name="d1192-print-font-subsetting-skipped"></a>D1192：已略過列印字型子集

Direct2D 列印：字型子集常式 [**CreateFontPackage**](/windows/desktop/api/fontsub/nf-fontsub-createfontpackage)失敗，錯誤 \[ *錯誤碼* \] 。 使用原始字型資料時，可能會增加輸出大小。
