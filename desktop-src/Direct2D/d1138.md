# <a name="d1138-invalid-command-list"></a>D1138：不正確命令清單

參考的命令清單無效 (在) 的命令清單擴展期間發生錯誤。






 

## <a name="possible-causes"></a>可能的原因

如果在命令清單擴展期間發生錯誤，應用程式將無法使用它來繪製作業或串流處理。 當命令清單關閉時，ID2D1CommandList：： Close 會傳回錯誤碼，指出未正確填入命令清單。 通常，在擴展期間使用的 ID2D1RenderTarget：： EndDraw 或 Flush 作業也會傳回錯誤碼，並提供其他資訊。 在 ID2D1RenderTarget：： EndDraw 期間造成錯誤的問題（例如嘗試使用另一個網域的資源或未封閉路徑幾何的繪圖作業），通常會導致命令清單無法成功關閉。

## <a name="possible-fixes"></a>可能的修正

若要確定已正確填入命令清單，請針對 ID2D1CommandList：： Close 和 ID2D1RenderTarget：： EndDraw 所發生的錯誤進行疑難排解，並在命令清單擴展期間進行清除。 確定應用程式不會忽略 ID2D1CommandList：： Close 的傳回碼。
