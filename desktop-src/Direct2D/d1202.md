# <a name="d1202-copyfromrendertarget-with-multisampled-source"></a>D1202： CopyfromRenderTarget 與多重取樣來源

CopyFromRenderTarget 與多重取樣 rendertarget 搭配使用。 仍會發出呼叫，但是複製的結果可能不會代表預期的多重取樣模式。






 

## <a name="possible-causes"></a>可能的原因

應用程式會 interoperated 多重取樣 Direct3D 材質作為 Direct2D 目標，並在目標上呼叫 CopyFromRenderTarget

## <a name="possible-fixes"></a>可能的修正

如果 CopyFromRenderTarget 功能很重要，請避免將多重取樣材質作為目標來進行交互操作。 複製的結果可能不代表預期的多重取樣模式。
