# <a name="d1165-read-and-discard-map-options-are-incompatible"></a>D1165：讀取和捨棄對應選項不相容

D2D1 \_ 對應 \_ 選項 \_ 讀取與 D2D1 \_ 對應 \_ 選項 \_ 捨棄不相容



|             |         |
|-------------|---------|
| 錯誤層級 | 警告 |



 

## <a name="possible-causes"></a>可能的原因

如果應用程式呼叫 ID2D1Bitmap1：： Map 與 D2D1 \_ 對應 \_ 選項 \_ READ 和 D2D1 \_ map \_ options \_ 捨棄選項，Direct2D 可能會輸出此訊息。

## <a name="possible-fixes"></a>可能的修正

[讀取] 和 [捨棄] 選項不會彼此同意，因為允許未定義資料的讀取存取權可用於應用程式。 如需 \_ 對應選項的描述，請參閱 D2D1 對應 \_ 選項。 呼叫 ID2D1Bitmap1：： Map 時，請考慮應用程式所需的存取類型。
