# <a name="d1117-bitmap-is-not-a-target"></a>D1117：點陣圖不是目標

SetTarget 失敗，因為點陣圖沒有目標選項。






 

## <a name="possible-causes"></a>可能的原因

為了將 ID2D1Bitmap 設為目標，您必須使用 D2D1 \_ 點陣圖 \_ 選項目標旗標來建立它 \_ 。 此旗標可作為 \_ \_ 與 ID2D1DeviceCoNtext：： CreateBitmap 搭配使用的 D2D1 BITMAP PROPERTIES1 欄位。

## <a name="possible-fixes"></a>可能的修正

確定應用程式使用 D2D1 \_ 點陣圖選項目標建立其目標 \_ 點陣圖 \_ 。 如果使用 ID2D1RenderTarget：： CreateBitmap 建立目標點陣圖，請將它修改為使用 ID2D1DeviceCoNtext：： CreateBitmap。 當指定目標旗標時，Direct2D 也會驗證材質的 Direct3D 屬性是否與做為目標的使用方式相容。
