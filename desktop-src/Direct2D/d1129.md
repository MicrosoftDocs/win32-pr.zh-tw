# <a name="d1129-dxgi-format-incompatible-with-alpha-mode"></a>D1129： DXGI 格式與 Alpha 模式不相容

提供的 Alpha 模式與 DXGI 格式的組合彼此不相容。



|             |         |
|-------------|---------|
| 錯誤層級 | 警告 |



 

## <a name="possible-causes"></a>可能的原因

若為使用表面格式和 Alpha 模式的 Direct2D 作業，則會在某些情況下，表面格式和 Alpha 模式不同意。 例如，像是 DXGI 格式 A8 UNORM 這類的表面格式 \_ \_ ，與 [略過 Alpha] 搭配使用並 \_ 不合理，因為它只會轉譯成 Alpha 色板。 此外，某些 DXGI 格式不支援直接 Alpha \_ ，但預乘的 Alpha 為。

## <a name="possible-fixes"></a>可能的修正

請參閱 [支援的像素格式和 Alpha 模式](supported-pixel-formats-and-alpha-modes.md) ，以取得有關支援及不支援哪些格式組合的指引。 修改應用程式以使用支援的格式組合。
