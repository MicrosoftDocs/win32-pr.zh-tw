# <a name="d1198-copy-to-read-only-bitmap"></a>D1198：複製到 Read-Only 點陣圖

已嘗試將資料複製到唯讀或不可變的點陣圖。






 

## <a name="possible-causes"></a>可能的原因

已嘗試將資料複製到唯讀或不可變的點陣圖。

## <a name="possible-fixes"></a>可能的修正

確認任何寫入至的點陣圖都不會建立為不可變。
