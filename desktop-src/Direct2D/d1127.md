# <a name="d1127-alpha-mode-mismatch"></a>D1127： Alpha 模式不相符

傳入此 API 的 Alpha 模式與介面的像素格式不相容 \[  \] 。

<dl> <dt>

</dd> </dl> 



 

## <a name="possible-causes"></a>可能的原因

ID2D1RenderTarget：： CreateBitmapFromWicBitmap 或 ID2D1RenderTarget：： CreateDxgiSurfaceRendertarget 等作業都需要介面和指定的 surface Alpha 模式。 Direct2D 會確認指定的 Alpha 模式與介面相容。 如果不相容，Direct2D 會輸出此訊息。

## <a name="possible-fixes"></a>可能的修正

修改應用程式，使其使用 \_ 符合或相容于基礎介面之 Alpha 模式的 Alpha 模式。 如需 Direct2D 所公開之 Alpha 模式的其他指引，請參閱 [支援的像素格式和 Alpha 模式](supported-pixel-formats-and-alpha-modes.md) 。
