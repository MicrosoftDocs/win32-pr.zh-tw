# <a name="matrix5x4f-class"></a>Matrix5x4F 類別

**Matrix5x4F** 類別代表一系列的4個矩陣，並提供建立矩陣的便利方法。

## <a name="members"></a>成員

**Matrix5x4F** 類別繼承自 [**D2D1 \_ 矩陣 \_ 5X4 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_5x4_f)。 **Matrix5x4F** 也有下列類型的成員：

-   [建構函式](#constructors)

### <a name="constructors"></a>建構函式

**Matrix5x4F** 類別具有這些函式。



| 建構函式                                                                                                                                                                                                                                                                                           | 描述                                                                                                                        |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------|
| [**Matrix5x4F ()**](matrix5x4f-matrix5x4f--.md)                                                                                                                                                                                                                                                       | 將未初始化之 **Matrix5x4F** 類別的新實例具現化。<br/>                                                   |
| [**Matrix5x4F (FLOAT，FLOAT，FLOAT，FLOAT，FLOAT，FLOAT，FLOAT，FLOAT，FLOAT，FLOAT，FLOAT，FLOAT，FLOAT，FLOAT，float，float，float，float，float，float)  (FLOAT，float，float，float，float，float，float，float，float，float，float，float，float，float，float)**](matrix5x4f-matrix5x4f-floats-.md) | 具現化 **Matrix5x4F** 類別的新實例，這個實例會使用所有浮點數矩陣值進行初始化。<br/> |
