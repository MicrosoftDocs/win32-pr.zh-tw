# <a name="d1167-read-map-option-requires-cpu-readable-bitmap"></a>D1167： Read Map 選項需要 CPU 可讀取點陣圖

D2D1 \_ 對應 \_ 選項 \_ 讀取只在 CPU 可讀取點陣圖上有效






 

## <a name="possible-causes"></a>可能的原因

如果應用程式使用 D2D1 Map OPTIONS READ 屬性呼叫 ID2D1Bitmap1：： Map \_ \_ \_ ，但點陣圖並非可讀取的 CPU，Direct2D 可能會輸出此訊息。

## <a name="possible-fixes"></a>可能的修正

如果應用程式需要使用 ID2D1Bitmap1：： Map 進行讀取存取，請使用 [D2D1 \_ bitmap \_ OPTIONS CPU \_ ] \_ 讀取選項建立點陣圖。
