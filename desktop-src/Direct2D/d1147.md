# <a name="d1147-print-control-closed-without-addpage"></a>D1147：未 AddPage 的列印控制項已關閉

Direct2D Print：不使用先前的 AddPage 呼叫來呼叫 Close






 

## <a name="possible-causes"></a>可能的原因

如果在沒有任何先前的 [**ID2D1PrintControl：： AddPage**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1printcontrol-addpage)調用的情況下關閉 [**ID2D1PrintControl**](/windows/win32/api/d2d1_1/nn-d2d1_1-id2d1printcontrol) ，就可能會發生此訊息。 Direct2D 列印會將沒有頁面的列印控制項視為錯誤，而對應的 [**ID2D1PrintControl：： Close**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1printcontrol-close) 命令將會傳回錯誤。

## <a name="possible-fixes"></a>可能的修正

若要處理訊息，請先將頁面加入至列印控制項，再將它關閉。 如果應用程式不需要列印任何內容，請避免關閉列印控制項或完全建立。
