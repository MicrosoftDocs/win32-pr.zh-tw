# <a name="d1173-texture-cube-interop-unsupported"></a>D1173：不支援材質 Cube Interop

標記為 D3D11RESOURCE \_ 其他 \_ TEXTURECUBE 的材質可能無法由 Direct2D 使用。






 

## <a name="possible-causes"></a>可能的原因

應用程式嘗試使用指定的 D3D11RESOURCE \_ 其他 TEXTURECUBE 選項來 Interop Direct3D 材質 \_ 。

## <a name="possible-fixes"></a>可能的修正

避免使用具有 TEXTURECUBE 規範的交互操作紋理。 請改為重新組織應用程式的轉譯設計，或考慮與中繼進行交互操作。
