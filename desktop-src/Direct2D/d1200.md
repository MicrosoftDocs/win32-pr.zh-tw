# <a name="d1200-pushlayer-with-multisampled-target"></a>D1200： PushLayer 與多重取樣目標

Pushlayer/Poplayer 與多重取樣目標搭配使用。 仍會發出呼叫，但轉譯的結果可能不會代表預期的多重取樣模式。






 

## <a name="possible-causes"></a>可能的原因

應用程式會 interoperated 多重取樣 Direct3D 材質作為 Direct2D 目標，並在目標上呼叫 PushLayer。

## <a name="possible-fixes"></a>可能的修正

如果圖層功能很重要，請避免交互操作多重取樣紋理。 使用具有這些資源類型的圖層或剪輯可能會導致效能下降或非預期的轉譯結果。 可能的話，請使用控制不透明度或遮罩的替代方法，而不是圖層。 或者，使用中繼而非多重取樣資源。 如果多重取樣資源適用于消除鋸齒2D 內容，請考慮 Direct2d 內建的每個基本消除鋸齒是否已足夠。
