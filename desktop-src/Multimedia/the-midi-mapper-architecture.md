# <a name="the-midi-mapper-architecture"></a>MIDI 對應程式架構

MIDI 對應工具會使用 MIDI 安裝對應來判斷如何轉譯和重新導向接收的訊息。 MIDI 安裝對應包含下列類型的對應。

-   [通道對應](the-channel-map.md)
-   [修補程式對應](patch-maps.md)
-   [金鑰組應](key-maps.md)
