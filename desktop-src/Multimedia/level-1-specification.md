# <a name="level-1-specification"></a>層級1規格

層級1規格的開發目的是為了鼓勵採用基本的多媒體功能（最低效能層級）。 當層級1規格發出時，可供 marketplace 使用的最常見多媒體元件包括數個光碟片 (CD-ROM) 磁片磁碟機，這些磁片磁碟機提供的持續傳輸速率為每秒90到 150 kb、8位音效卡，以及16色和256色 (SVGA) 視訊卡。
