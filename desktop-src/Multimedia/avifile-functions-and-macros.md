# <a name="avifile-functions-and-macros"></a>AVIFile 函式和宏

AVIFile 函式和宏提供以時間為基礎之檔案的存取權，這些檔案使用資源資訊檔案格式 (RIFF) ，例如波形音訊和音訊影片交錯 (AVI) 檔。 這些函式和宏會管理 RIFF 檔案，讓您不需要管理和流覽 RIFF 架構。
