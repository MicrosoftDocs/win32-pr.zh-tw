# <a name="embedding-information-chunks-in-an-avi-file"></a>將資訊區塊嵌入 AVI 檔案

您可以使用 [**WM \_ CAP \_ file \_ SET \_ INFOCHUNK**](wm-cap-file-set-infochunk.md) message (或 [**capFileSetInfoChunk**](/windows/desktop/api/Vfw/nf-vfw-capfilesetinfochunk) 宏) ，在 AVI 檔案中插入資訊區塊（例如文字或自訂資料）。 您也可以使用此訊息來清除 AVI 檔案中的資訊區塊。
