# <a name="single-image-compression"></a>Single-Image 壓縮

您可以使用 [**ICImageCompress**](/windows/desktop/api/Vfw/nf-vfw-icimagecompress) 函式來壓縮單一映射。 此函式會傳回壓縮的裝置獨立點陣圖 (DIB) 的控制碼。 壓縮的 DIB 會使用 CF \_ DIB 格式來封裝。
