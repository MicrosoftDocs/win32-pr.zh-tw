# <a name="creating-an-avicap-capture-window"></a>建立 AVICap Capture 視窗

您可以使用 [**capCreateCaptureWindow**](/windows/desktop/api/Vfw/nf-vfw-capcreatecapturewindowa) 函數來建立 AVICap 視窗類別的捕獲視窗。 此函式會傳回一個視窗控制碼，此控制碼會識別「捕獲」視窗，並由應用程式用來將後續訊息傳送至視窗。

您可以在應用程式中建立一或多個捕獲視窗，並將每個捕獲視窗連接至不同的捕獲裝置。
