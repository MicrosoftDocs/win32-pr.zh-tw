# <a name="frame-windows-multimedia"></a>Windows 多媒體)  (框架

當有新的影片畫面可供使用時，「捕獲」視窗會使用畫面格回呼通知訊息來通知您的應用程式。 只有當預覽率為非零且串流捕捉未進行時，[捕獲] 視窗才會啟用這些回呼通知。
