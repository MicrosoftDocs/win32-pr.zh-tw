# <a name="multimedia-input"></a>多媒體輸入

Windows 提供可讓應用程式接收多媒體輸入的功能。 多媒體輸入函式是在16位 Windows 中引進的。 其中有許多功能已由 DirectX 中的功能所取代。 如需詳細資訊，請參閱 DirectX 檔。

下列主題將在多媒體輸入檔中討論。

-   [多媒體檔案 i/o](multimedia-file-i-o.md)
-   [操縱 杆](joysticks.md)
-   [多媒體計時器](multimedia-timers.md)
