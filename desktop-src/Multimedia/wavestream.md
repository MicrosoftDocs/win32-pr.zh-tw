# <a name="wavestream"></a>Wavestream

應用程式會在串流捕捉期間使用 wavestream 回呼函式來處理音訊緩衝區。 捕捉視窗會先呼叫 wavestream 回呼函式，再將每個音訊緩衝區寫入磁片。
