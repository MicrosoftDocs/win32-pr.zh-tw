# <a name="capture-windows"></a>捕獲 Windows

「捕獲視窗」在概念上類似于標準控制項 (例如按鈕、清單方塊或捲軸) 。 一般而言，「捕獲視窗」會使用 WS \_ 子系和 ws \_ 可見的視窗樣式。
