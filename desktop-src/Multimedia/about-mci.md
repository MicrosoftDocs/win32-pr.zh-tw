# <a name="about-mci"></a>關於 MCI

MCI 為應用程式提供了與裝置無關的功能，可控制音訊和視覺設備。 您的應用程式可以使用 MCI 來控制任何支援的多媒體裝置，包括波形音訊裝置、MIDI sequencers、CD 音訊裝置，以及數位影片 (影片播放) 裝置。
