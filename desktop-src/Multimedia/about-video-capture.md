# <a name="about-video-capture"></a>關於影片捕獲

AVICap 支援即時串流影片捕獲和單一畫面格捕獲。 此外，AVICap 也提供影片來源的控制權， (MCI) 裝置，讓使用者可以透過應用程式來控制 (，) 影片來源的開始和停止位置，並增強捕捉作業以包含步驟框架抓取。

您使用 [AVICap] 視窗類別建立的視窗可以執行下列工作：

-   將音訊和影片串流捕獲到 audo (AVI) 檔案的影片。
-   動態連接及中斷影片和音訊輸入裝置的連線。
-   使用重迭或預覽方法來觀看即時連入的視訊訊號。
-   指定在捕獲檔案並將其內容複寫到另一個檔案時，所要使用的檔案。
-   設定 [捕獲速度]。
-   顯示控制影片來源和格式的對話方塊。
-   建立、儲存和載入調色板。
-   將影像和調色板複製到剪貼簿。
-   以與裝置無關的點陣圖形式來捕捉並儲存單一映射 (DIB) 。
