# <a name="video-for-windows"></a>適用于 Windows 的影片

Windows (VFW) 的影片提供可讓應用程式處理影片資料的功能。 VFW 是在16位 Windows 中引進。 其中許多功能已由 DirectX 中的功能所取代。 如需詳細資訊，請參閱 DirectX 檔。

適用于 Windows 的影片中會討論下列主題。

-   [AVIFile 函式和宏](avifile-functions-and-macros.md)
-   [影片壓縮管理員](video-compression-manager.md)
-   [影片捕獲](video-capture.md)
-   [自訂檔案和串流處理常式](custom-file-and-stream-handlers.md)
-   [DrawDib](drawdib.md)
