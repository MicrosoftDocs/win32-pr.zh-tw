# <a name="manufacturer-and-product-identifiers"></a>製造商和產品識別碼

本附錄說明針對多媒體應用程式所定義的製造商和產品識別碼。 當應用程式發出有關電腦上已安裝裝置的查詢時，就會使用這些識別碼。 若要在您的應用程式中使用這些識別碼，請包含 MMREG。H file。

-   [製造商識別碼](manufacturer-identifiers.md)
-   [Microsoft Corporation 產品識別碼](microsoft-corporation-product-identifiers.md)
-   [Microsoft Windows 音效系統驅動程式](microsoft-windows-sound-system-drivers.md)
-   [產品識別碼](product-identifiers.md)

> [!Note]  
> Microsoft 不再針對產品識別碼維護中央登錄。
