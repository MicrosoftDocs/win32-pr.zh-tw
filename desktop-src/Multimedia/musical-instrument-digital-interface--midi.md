# <a name="musical-instrument-digital-interface-midi"></a>樂器數位介面 (MIDI)

「音樂檢測數位介面」 (MIDI) 是一種通訊協定和一組命令，用於儲存和傳輸音樂的相關資訊。 MIDI 輸出裝置會解讀此資訊，並用它來合成音樂。

-   [使用 MIDI](using-midi.md)
-   [MIDI 參考](midi-reference.md)
