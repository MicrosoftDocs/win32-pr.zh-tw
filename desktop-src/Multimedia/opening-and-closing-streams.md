<span data-ttu-id="7a078-112">若要在檔案中存取一個以上的資料流程，請使用 **AVIFileOpen** 一次，然後再呼叫 **AVIFileGetStream** 的多個。</span><span class="sxs-lookup"><span data-stu-id="7a078-112">To access more than one stream in a file, use **AVIFileOpen** once followed by multiple calls to **AVIFileGetStream**.</span></span>
