# <a name="multimedia-pc-specifications"></a>多媒體電腦規格

多媒體電腦 (MPC) 行銷委員會已開發兩種規格，以鼓勵採用多媒體功能。 層級1規格（以1990開發）提供功能、硬體元件和軟體元件中多媒體運算的基準定義。 在1993中發行的層級2規格是以第一個規格為基礎，並著重于自第一次規格發出以來增強的多媒體功能。

本附錄摘要說明規格。

-   [層級1規格](level-1-specification.md)
-   [層級2規格](level-2-specification.md)
