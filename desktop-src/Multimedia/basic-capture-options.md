# <a name="basic-capture-options"></a>基本的捕獲選項

藉由修改 [**CAPTUREPARMS**](/windows/win32/api/vfw/ns-vfw-captureparms) 結構中所定義的一或多個 capture 參數，您可以執行下列工作：

-   變更畫面格捕獲速率。
-   指定用來結束捕捉會話的鍵盤或滑鼠控制項。
-   指定捕捉會話的持續時間。
