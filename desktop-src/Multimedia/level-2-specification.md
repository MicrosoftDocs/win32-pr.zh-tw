# <a name="level-2-specification"></a>層級2規格

層級2規格的開發目的是為了鼓勵採用增強的多媒體功能。 此規格是以層級1規格中設定的需求為基礎，而且是其超集合。 層級2規格會定義增強多媒體功能的最小系統功能。 這不是系統組態的建議。
