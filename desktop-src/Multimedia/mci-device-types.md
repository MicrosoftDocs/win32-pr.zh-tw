# <a name="mci-device-types"></a>MCI 裝置類型

下列值會識別 MCI 訊息和結構中的裝置：



| 值                         | 意義                       |
|-------------------------------|-------------------------------|
| MCI \_ 所有 \_ 裝置 \_ 識別碼          | 任何裝置                    |
| MCI \_ DEVTYPE \_ 動畫       | 動畫-播放裝置     |
| MCI \_ DEVTYPE \_ CD \_ 音訊       | CD 音訊裝置               |
| MCI \_ DEVTYPE \_ DAT             | 數位音訊磁帶裝置     |
| MCI \_ DEVTYPE \_ 數位 \_ 影片  | 數位視訊播放裝置 |
| MCI \_ DEVTYPE \_ 其他           | 未定義的裝置              |
| MCI \_ DEVTYPE 重迭 \_         | 影片-重迭裝置          |
| MCI \_ DEVTYPE \_ 掃描器         | 掃描器裝置                |
| MCI \_ DEVTYPE \_ SEQUENCER       | MIDI sequencer 裝置         |
| MCI \_ DEVTYPE \_ VCR             | 影片-卡帶錄製器       |
| MCI \_ DEVTYPE \_ VIDEODISC       | Videodisc player              |
| MCI \_ DEVTYPE \_ 波形 \_ 音訊 | 波形-音訊裝置         |
