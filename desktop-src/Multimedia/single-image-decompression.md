# <a name="single-image-decompression"></a>Single-Image 解壓縮

您可以使用 [**ICImageDecompress**](/windows/desktop/api/Vfw/nf-vfw-icimagedecompress) 函式來解壓縮單一映射。 此函式會傳回解壓縮的 DIB 的控制碼。 解壓縮的 DIB 會以 CF \_ dib 格式儲存。
