# <a name="level-1-system-resources"></a>層級1系統資源

符合層級1規格的電腦最小設定包含下列專案：

-   具有 2 mb (MB) 隨機存取記憶體 (RAM 的386SX 微處理器) 
-   3.5-英寸高密度磁片磁碟機
-   硬碟空間至少為 30 MB 的硬碟磁片磁碟機
-   顯示器解析度為 480 640、16圖元、16色的色彩監視器
-   提供與 Microsoft Windows 作業系統3.0 版的二進位相容性的系統軟體
