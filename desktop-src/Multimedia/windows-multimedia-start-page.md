# <a name="windows-multimedia"></a>Windows多媒體

## <a name="purpose"></a>目的

Microsoft Windows 多媒體支援可讓應用程式使用音效和影片。

## <a name="where-applicable"></a>適用時

Windows 多媒體可以用於所有 Windows 型應用程式。

## <a name="developer-audience"></a>開發人員對象

Windows 多媒體是設計來供 C/c + + 程式設計人員使用。 需要熟悉 Windows 的圖形化使用者介面和訊息驅動架構。
