# <a name="video-capture-windows-multimedia"></a> (Windows 多媒體) 的影片捕獲

您可以使用 AVICap 視窗類別，輕鬆地將影片捕獲功能併入您的應用程式。 AVICap 為應用程式提供簡單的訊息型介面來存取影片和波形音訊取得硬體，以及控制將影片捕獲串流處理至磁片的程式。
