# <a name="mciwnd-window-class"></a>MCIWnd 視窗類別

MCIWnd 是控制多媒體裝置的視窗類別。 與 MCIWnd 相關聯的函式、訊息和宏程式庫，可提供簡單的方法，將多媒體播放或記錄功能新增至您的應用程式。
