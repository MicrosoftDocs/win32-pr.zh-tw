# <a name="paste-command"></a>貼上命令

[貼上] 命令會將剪貼簿的內容複寫到工作區。 數位影片裝置辨識此命令。

若要傳送此命令，請使用 *lpszCommand* 參數設定來呼叫 [**mciSendString**](/previous-versions//dd757161(v=vs.85))函式，如下所示。

``` syntax
_stprintf_s(
  lpszCommand, 
  TEXT("paste %s %s %s"), 
  lpszDeviceID, 
  lpszItem, 
  lpszFlags
); 
```

## <a name="parameters"></a>參數

<dl> <dt>

<span id="lpszDeviceID"></span><span id="lpszdeviceid"></span><span id="LPSZDEVICEID"></span>*lpszDeviceID*
</dt> <dd>

MCI 裝置的識別碼。 開啟裝置時，會指派此識別碼或別名。

</dd> <dt>

<span id="lpszItem"></span><span id="lpszitem"></span><span id="LPSZITEM"></span>*lpszItem*
</dt> <dd>

下列一或多個旗標。



| 值                 | 意義                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
|-----------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 在 *矩形*        | 指定要貼上資料的框架內的位置。 *矩形* 的左上角會對應至已加入資料的左上角。 如果矩形在 X 或 Y 中的大小為非零，則在將剪貼簿的內容貼到框架時，剪貼簿的內容會在這些維度中縮放。 如果省略， *矩形* 就會預設為整個框架。 如果這個旗標是在 [插入] 模式中指定 (預設) ，則矩形之外的任何區域都是以純色繪製。                       |
| 音訊串流 *串流* | 指定工作區中受命令影響的音訊串流。 如果剪貼簿中只有一個音訊串流，則會將音訊資料貼入指定的 *資料流程*。 如果剪貼簿中有一個以上的音訊串流， *資料流程* 就會指出資料流程順序的起始數位。 如果您使用此旗標，而且也想要貼上影片，您也必須使用「影片串流」旗標。  (如果未指定任何旗標，則會貼上所有音訊和影片串流，並保留原始的串流號碼。 )  |
| insert                | 指定將資料插入工作區中。 插入點之後的任何資料會在工作空間中向前移動以騰出空間。 這是預設值。                                                                                                                                                                                                                                                                                                                                                    |
| overwrite             | 指定在插入點之後寫入任何現有資料，以將資料複製到工作區。 「插入」和「覆寫」旗標會影響在貼上作業期間是否終結或移動框架，而不是在每個框架中貼上資料的方式。                                                                                                                                                                                                                                              |
| *定位*         | 指定工作區中貼上資料的位置。 如果省略，則會預設為目前的位置。                                                                                                                                                                                                                                                                                                                                                                                                    |
| 影片串流 *串流* | 在工作區中指定受命令影響的影片串流。 如果剪貼簿中只有一個影片串流，則會將影片資料貼入指定的 *資料流程*。 如果剪貼簿中有一個以上的影片串流， *資料流程* 就會指出資料流程順序的起始數位。 如果您使用此旗標，而且也想要貼上音訊，您也必須使用「音訊串流」旗標。  (如果未指定任何旗標，則會貼上所有音訊和影片串流，並保留原始的串流號碼。 )  |



 

</dd> <dt>

<span id="lpszFlags"></span><span id="lpszflags"></span><span id="LPSZFLAGS"></span>*lpszFlags*
</dt> <dd>

可以是「等候」、「通知」、「測試」或它們的組合。 如需這些旗標的詳細資訊，請參閱 [Wait、Notify 和 Test 旗標](the-wait-notify-and-test-flags.md)。

</dd> </dl>

## <a name="return-value"></a>傳回值

如果成功則傳回零，否則會傳回錯誤。

## <a name="remarks"></a>備註

從剪貼簿複製的資料中不會出現任何信號。 只有在明確儲存資料時，變更才會變成永久;不過，播放的行為就如同已新增資料一樣。
