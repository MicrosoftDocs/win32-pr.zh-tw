# <a name="custom-file-and-stream-handlers"></a>自訂檔案和串流處理常式

檔案和串流處理常式是為控制多媒體資料的應用程式提供一致介面的驅動程式。 作業系統中所含的檔案和串流處理常式會使用影片和波形音訊資料，並以音訊顯示 (AVI) 和波形音訊檔案。

您可以撰寫處理常式，以允許您的應用程式寫入或存取另一個來源的多媒體資料，例如使用專屬格式的檔案、已擴充為包含其他資料流程的 AVI 檔案，或是產生它自己的多媒體資料的處理常式。 如果您有想要搭配 AVIFile 函式 [和宏](avifile-functions-and-macros.md)使用之 AVI 資料的自訂檔案格式，則需要撰寫自訂處理常式。
