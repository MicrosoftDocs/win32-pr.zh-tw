# <a name="about-audio-mixers"></a>關於音訊 Mixers

音訊混音器服務會控制音訊行到目的地裝置的路由播放或錄製。 這些服務也可以控制音量和其他效果。 使用這些服務所需的許多技巧，與其他多媒體簡介中所討論的音訊裝置類似。
