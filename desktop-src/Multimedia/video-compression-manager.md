# <a name="video-compression-manager"></a>影片壓縮管理員

影片壓縮管理員 (BC-VCM-LVM-HYPERV) 可讓您存取可安裝壓縮機用來處理即時資料的介面。 應用程式可以使用可安裝的壓縮機來執行下列工作：

-   壓縮和解壓縮影片資料。
-   傳送轉譯器壓縮的影片資料，並讓它將它繪製到顯示畫面。
-   使用應用程式定義的轉譯器來壓縮、解壓縮或繪製資料。
-   使用轉譯器來處理文字和自訂資料。
