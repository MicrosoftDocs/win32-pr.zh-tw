# <a name="user-data-messages"></a>使用者資料訊息

您可以使用 [**wm \_ cap \_ 取得 \_ 使用者 \_ 資料**](wm-cap-get-user-data.md) 和 [**Wm \_ \_ 設定 \_ 使用者 \_ 資料**](wm-cap-set-user-data.md) 訊息 (或 [**capGetUserData**](/windows/desktop/api/Vfw/nf-vfw-capgetuserdata) 和 [**capSetUserData**](/windows/desktop/api/Vfw/nf-vfw-capsetuserdata) 宏) ，將資料與捕捉視窗建立關聯。 您可以使用 WM  \_ cap 取得使用者資料訊息來取出長資料值 \_ \_ \_ ，並使用 wm  \_ cap \_ 設定 \_ 使用者 \_ 資料訊息來設定長資料值。
