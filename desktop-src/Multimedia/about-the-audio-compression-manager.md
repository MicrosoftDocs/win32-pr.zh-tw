# <a name="about-the-audio-compression-manager"></a>關於音訊壓縮管理員

音訊壓縮管理員新增了下列服務的系統層級支援：

-   透明執行時間音訊壓縮和解壓縮
-   波形-音訊資料格式選取範圍
-   波形-音訊資料篩選器選取範圍
-   波形-音訊資料格式轉換
-   波形-音訊資料篩選
