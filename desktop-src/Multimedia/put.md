# <a name="put-command"></a>put 命令

Put 命令定義來源影像的區域和用於顯示的目的地視窗。 數位影片和影片重迭裝置會辨識此命令。

若要傳送此命令，請使用 *lpszCommand* 參數設定來呼叫 [**mciSendString**](/previous-versions//dd757161(v=vs.85))函式，如下所示。

``` syntax
_stprintf_s(
  lpszCommand, 
  TEXT("put %s %s %s"), 
  lpszDeviceID, 
  lpszRegions, 
  lpszFlags
); 
```

## <a name="return-value"></a>傳回值

如果成功則傳回零，否則會傳回錯誤。

## <a name="remarks"></a>備註

Put 命令會在使用影片重迭裝置時，定義下列一或多個矩形：

-   影片矩形會定義要捕捉之內送影片影像的區域。
-   框架矩形會定義接收傳入影片影像的框架緩衝區區域。
-   來源矩形會定義要將框架緩衝區的哪個區域複製到目的地矩形。
-   目的地矩形會定義接收影片影像之顯示視窗工作區的區域。

影片矩形與框架矩形相關，與來源矩形與目的地矩形的關聯方式相同。 從影片矩形到框架矩形，以及從來源矩形到目的矩形，都可以進行延展。 並非所有裝置都支援延展，而且必須使用 [set](set.md) 命令) 來啟用延展 (。

下列命令會定義影片、畫面格和來源的三個區域。

``` syntax
put vboard video 120 120 200 200 frame 0 0 200 200 source 0 0 200 200
```

此範例中的區域定義如下：

-   傳入影片資料的200到200圖元區域（從左上角開始，從原點120圖元開始）將會被捕獲到框架緩衝區。
-   影片資料會放在框架緩衝區左上角的200到200圖元區域中。
-   從框架緩衝區左上角的200到 200-圖元區域傳輸到目的地視窗。
