# <a name="time-controls"></a>時間控制項

時間控制項可讓使用者輸入時間相關資料，例如 echo delay 或殘響。 時間資料會以正整數表示。 時間控制項的類型包括下列各項：



| 控制     | 描述                                                                                            |
|-------------|--------------------------------------------------------------------------------------------------------|
| 微秒 | 支援以微碼錶示的計時資料。 可接受的值範圍為0到 (232 – 1) 。 |
| Millisecond | 支援以毫碼錶示的計時資料。 可接受的值範圍為0到 (232 – 1) 。 |
