# <a name="video-format"></a>影片格式

您可以藉由將 [**WM \_ CAP \_ GET \_ VIDEOFORMAT**](wm-cap-get-videoformat.md) Message (或 [**capGetVideoFormat**](/windows/desktop/api/Vfw/nf-vfw-capgetvideoformat) 和 [**capGetVideoFormatSize**](/windows/desktop/api/Vfw/nf-vfw-capgetvideoformatsize) 宏) 傳送至 capture 視窗，來抓取指定影片格式或該結構大小的結構。 您可以藉由將 [**WM \_ CAP \_ set \_ VIDEOFORMAT**](wm-cap-set-videoformat.md) message (或 [**capSetVideoFormat**](/windows/desktop/api/Vfw/nf-vfw-capsetvideoformat) 宏) 傳送至 capture 視窗，來設定已捕捉的影片資料格式。
