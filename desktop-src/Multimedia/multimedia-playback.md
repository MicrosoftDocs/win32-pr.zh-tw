# <a name="multimedia-playback"></a>多媒體播放

[**MCIWndCreate**](/windows/desktop/api/Vfw/nf-vfw-mciwndcreatea)函式會提供方法來控制 MCIWnd 視窗與其相關聯的裝置。 一般而言，此函式會註冊 MCIWnd 視窗類別，並建立使用 MCI 服務的 MCIWnd 視窗。 本節說明如何使用 **MCIWndCreate** 來執行下列工作：

-   [新增 User-Controlled 播放](adding-user-controlled-playback.md)
-   [自動播放](automating-playback.md)
-   [使用視窗樣式變更 MCIWnd 視窗](using-window-styles-to-change-the-mciwnd-window.md)
-   [允許使用者指定裝置和檔案](allowing-the-user-to-specify-devices-and-files.md)
