# <a name="capture-driver-capabilities"></a>Capture 驅動程式功能

您可以使用 [ [**WM \_ cap \_ 驅動程式 \_ 取得 \_ cap**](wm-cap-driver-get-caps.md) 訊息 (] 或 [ [**capDriverGetCaps**](/windows/desktop/api/Vfw/nf-vfw-capdrivergetcaps) 宏) ，來取得目前連線的捕獲驅動程式的硬體功能。 這則訊息會傳回 [**CAPDRIVERCAPS**](/windows/win32/api/vfw/ns-vfw-capdrivercaps) 結構中的 capture 驅動程式和基礎硬體的功能。
