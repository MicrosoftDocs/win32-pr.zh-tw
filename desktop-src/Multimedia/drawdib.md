# <a name="drawdib"></a>DrawDib

DrawDib 函式會針對裝置獨立點陣圖提供高效能的影像繪製功能， (Dib) 。 DrawDib 函式支援8位、16位、24位和32位影像深度的 Dib。

DrawDib 函式會直接寫入視訊記憶體。 它們不依賴圖形裝置介面的函式 (GDI) 。
