# <a name="level-1-optical-storage"></a>層級1光學儲存體

最小效能光學儲存體裝置是符合下列準則的 CD-ROM 光碟機：

-   每秒 150 kb 的持續性資料傳輸速率
-   維持每秒 150 kb 的持續性資料傳輸速率時，CPU 頻寬使用量為40% 或更低
-   平均搜尋時間為1秒或更少
