# <a name="interface-hierarchy"></a>介面階層架構

此圖顯示介面繼承階層。

大部分的應用程式只會使用階層最上方的介面，不會直接使用繼承的介面; [**ID3D12Pageable**](/windows/win32/api/d3d12/nn-d3d12-id3d12pageable)、 [**ID3D12DeviceChild**](/windows/desktop/api/d3d12/nn-d3d12-id3d12devicechild)、 [**ID3D12Object**](/windows/desktop/api/d3d12/nn-d3d12-id3d12object) 和 [**ID3D12CommandList**](/windows/desktop/api/d3d12/nn-d3d12-id3d12commandlist)。

新介面會顯示在綠色方塊中。
