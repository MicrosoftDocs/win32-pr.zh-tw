# <a name="call-parameter-structure"></a>呼叫參數結構

以 inout 引數形式提供給 [**CallShader**](callshader-function.md) 呼叫的使用者定義結構，以及可呼叫 [著色器](callable-shader.md)的 inout 參數。 可呼叫著色器中使用的結構類型必須符合提供給對應 **CallShader** 呼叫的結構。
