# <a name="instanceindex-function"></a>InstanceIndex 函式

最上層 raytracing 加速結構中目前實例的自動產生索引。

## <a name="syntax"></a>語法

```
uint InstanceIndex();

```


## <a name="remarks"></a>備註

您可以從下列 raytracing 著色器類型呼叫這個函數：

* [**任何命中著色器**](any-hit-shader.md)
* [**最接近的點擊著色器**](closest-hit-shader.md)
* [**交集著色器**](intersection-shader.md)
