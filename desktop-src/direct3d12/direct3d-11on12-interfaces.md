# <a name="11on12-interfaces"></a>11on12 介面
下列介面是在 d3d11on12 中宣告。

## <a name="in-this-section"></a>本節內容
| 主題 | 描述 |
|-|-|
| [**ID3D11On12Device**](/windows/win32/api/d3d11on12/nn-d3d11on12-id3d11on12device) | 處理 Direct3D 11on12 的 D3D11 資源建立、包裝和釋放。 |
| [**ID3D11On12Device1**](/windows/win32/api/d3d11on12/nn-d3d11on12-id3d11on12device1) | 能夠與可能會獲得 Direct3D 11 裝置的元件進行更好的互通性，但想要改用 Direct3D 12。 |
| [**ID3D11On12Device2**](/windows/win32/api/d3d11on12/nn-d3d11on12-id3d11on12device2) | 可讓您取得透過 Direct3D 11 Api 建立的資源，並在 Direct3D 12 中使用它們。 這個介面會擴充 [ID3D11On12Device1](/windows/win32/api/d3d11on12/nn-d3d11on12-id3d11on12device1)。 |
