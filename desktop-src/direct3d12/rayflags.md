# <a name="rayflags"></a>RayFlags

包含目前 [**ray_flag**](ray_flag.md) 旗標的不帶正負號的整數。 

## <a name="syntax"></a>語法

```
uint RayFlags();

```

## <a name="remarks"></a>備註

您可以從下列 raytracing 著色器類型呼叫這個函數：

* [**任何命中著色器**](any-hit-shader.md)
* [**最接近的點擊著色器**](closest-hit-shader.md)
* [**交集著色器**](intersection-shader.md)
* [**遺漏著色器**](miss-shader.md)
