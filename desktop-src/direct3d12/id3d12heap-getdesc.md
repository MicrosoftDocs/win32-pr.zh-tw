# <a name="id3d12heapgetdesc-method"></a>ID3D12Heap：： GetDesc 方法

取得堆積描述。

## <a name="syntax"></a>語法


```C++
D3D12_HEAP_DESC GetDesc();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **[ **D3D12 \_ 堆積 \_ DESC**](/windows/desktop/api/d3d12/ns-d3d12-d3d12_heap_desc)**

傳回描述堆積的 [**D3D12 \_ 堆積 \_ DESC**](/windows/desktop/api/d3d12/ns-d3d12-d3d12_heap_desc) 結構。
