# <a name="raytracing-hlsl-structures-direct3d-12"></a>Raytracing HLSL 結構 (Direct3D 12) 

下列 HLSL 結構支援 Direct3D 12 raytracing 管線。

## <a name="in-this-section"></a>本節內容



| 主題                                                                                                       | 描述                                                                                                                                                                                                                                                                                             |
|-------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**呼叫參數結構**](call-parameter-structure.md)<br/>                              | 以 inout 引數形式提供給 CallShader 呼叫的使用者定義結構，以及可呼叫著色器的 inout 參數。<br/>                                                                                                                                                                                                                                              |
| [**交集屬性結構**](intersection-attributes.md)<br/>                              | 在 [**TraceRay**](traceray-function.md) 呼叫中提供作為 inout 引數的使用者定義結構，以及可存取光線承載之著色器類型中的 inout 參數。<br/>                                                                                                                                                                                                                                              |
| [**光線承載結構**](ray-payload.md)<br/>                              | 在 [**TraceRay**](traceray-function.md) 呼叫中提供作為 inout 引數的使用者定義結構，以及可存取光線承載之著色器類型中的 inout 參數。<br/>                                                                                                                                                                                                                                              |
| [**RayDesc 結構**](raydesc.md)<br/>                              | 傳遞至 [**TraceRay**](traceray-function.md) 函式的旗標，可覆寫透明度、剔除和早期輸出行為。<br/>                                                                                                                                                                                                                                              |
