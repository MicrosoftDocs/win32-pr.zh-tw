# <a name="d3d12_gpu_virtual_address"></a>D3D12 \_ GPU \_ 虛擬 \_ 位址

D3D12_GPU_VIRTUAL_ADDRESS 代表 GPU 記憶體中的虛擬位址。 資料類型定義于 d3d12 中，如下所示：


```
typedef UINT64 D3D12_GPU_VIRTUAL_ADDRESS;
```
