# <a name="windows-connect-now-interfaces"></a>Windows Connect Now 介面

[WINDOWS CONNECT NOW API](portal.md) (WCNAPI) 會定義並利用下列介面。



| 介面                                      | 描述                                                                                                |
|------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**IWCNConnectNotify**](/windows/desktop/api/WcnDevice/nn-wcndevice-iwcnconnectnotify) | 當 Windows Connect Now Connect 會話完成時，收到成功或失敗的通知。<br/> |
| [**IWCNDevice**](/windows/desktop/api/WcnDevice/nn-wcndevice-iwcndevice)               | 設定裝置並起始會話。<br/>                                                |
