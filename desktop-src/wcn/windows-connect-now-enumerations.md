# <a name="windows-connect-now-enumerations"></a>Windows Connect Now枚舉

Windows Connect Now 定義下列列舉類型。



| 列舉型別                                                                                             | 描述                                                                                                                   |
|---------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------|
| [**WCN \_ 屬性 \_ 類型**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_attribute_type)                                                      | 定義針對 Wi-Fi 受保護的設定所定義的各種屬性緩衝區類型。<br/>                                      |
| [**WCN \_ 密碼 \_ 類型**](/windows/win32/api/wcndevice/ne-wcndevice-wcn_password_type)                                                        | 定義將在 WPS 會話中使用的驗證。 <br/>                                                    |
| [**WCN \_ 會話 \_ 狀態**](/windows/win32/api/wcndevice/ne-wcndevice-wcn_session_status)                                                      | 定義 WPS 會話的狀態。<br/>                                                                               |
| [**WCN \_ 值 \_ 類型 \_ 關聯 \_ 狀態**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_association_state)                        | 定義在探索要求期間，無線工作站可能的關聯狀態。<br/>                          |
| [**WCN \_ 值 \_ 類型 \_ 布林值**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_boolean)                                             | 定義用來表示在裝置設定和關聯期間遇到之 true/false 條件的值。<br/>            |
| [**WCN \_ 值 \_ 類型 \_ 驗證 \_ 類型**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_authentication_type)                    | 定義「註冊人」 (存取點或工作站) 所支援的驗證方法。<br/>                            |
| [**WCN \_ 值 \_ 類型 \_ CONFIG \_ 方法**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_config_methods)                              | 定義登記者或註冊機構所支援的設定方法。<br/>                                          |
| [**WCN \_ 值 \_ 類型 \_ 設定 \_ 錯誤**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_configuration_error)                    | 定義在嘗試設定和與 WLAN 相關聯的情況下，可能會傳回給裝置的錯誤值。<br/> |
| [**WCN \_ 值 \_ 型別 \_ 裝置 \_ 密碼 \_ 識別碼**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_device_password_id)                     | 定義可能的密碼值。<br/>                                                                              |
| [**WCN \_ 值 \_ 類型 \_ 加密 \_ 類型**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_encryption_type)                            | 定義支援的 WLAN 加密類型。<br/>                                                                       |
| [**WCN \_ 值 \_ 類型 \_ RF \_ 區段**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_rf_bands)                                          | 定義登錄器傳送探索要求的無線頻率頻外。<br/>                                    |
| [**WCN \_ 值 \_ 類型 \_ 版本**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_version)                                             | 定義受支援的 Wi-Fi 受保護安裝程式 (WPS) 版本。<br/>                                                      |
| [**WCN \_ 值 \_ 類型 \_ WI-FI \_ \_ 保護的 \_ 設定 \_ 狀態**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_wi_fi_protected_setup_state) | 定義指出是否已設定裝置的值。<br/>                                                            |
