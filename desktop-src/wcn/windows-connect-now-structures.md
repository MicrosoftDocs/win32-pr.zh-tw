# <a name="windows-connect-now-structures"></a>Windows Connect Now 結構

下列結構支援 Windows Connect Now API 函數。



| 結構                                                                             | Description                                                                                                                                            |
|---------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**WCN \_ 值 \_ 類型 \_ 主要 \_ 裝置 \_ 類型**](/windows/win32/api/wcntypes/ns-wcntypes-wcn_value_type_primary_device_type) | 包含的列舉資料會依類別、子類別和製造商專屬的 OUI (組織識別碼) 來識別裝置的類型。<br/> |
| [**WCN \_ 廠商 \_ 延伸模組 \_ 規格**](/windows/win32/api/wcndevice/ns-wcndevice-wcn_vendor_extension_spec)                     | 包含定義廠商延伸模組的資料。 <br/>                                                                                             |
