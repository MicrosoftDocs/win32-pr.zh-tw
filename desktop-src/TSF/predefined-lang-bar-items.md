# <a name="predefined-language-bar-items"></a>預先定義的語言橫條圖專案

下列 [**GUID**](/windows/desktop/TSF/predefined-lang-bar-items) 值會識別預先定義的語言列專案。 這些值會與 [**GetItem**](/windows/desktop/api/Ctfutb/nf-ctfutb-itflangbaritemmgr-getitem) 搭配使用，以取得特定的語言列專案。

<dl> <dt>

<span id="_______________GUID_LBI_SAPILAYR_CFGMENUBUTTON_____________"></span><span id="_______________guid_lbi_sapilayr_cfgmenubutton_____________"></span> GUID \_ LBI \_ SAPILAYR \_ CFGMENUBUTTON 
</dt> <dd>

語音工具語言列功能表項目。

</dd> <dt>

<span id="_______________GUID_TFCAT_TIP_HANDWRITING_____________"></span><span id="_______________guid_tfcat_tip_handwriting_____________"></span> GUID \_ TFCAT \_ 提示 \_ 手寫 
</dt> <dd>

手寫類型專案。

</dd> <dt>

<span id="_______________GUID_TFCAT_TIP_KEYBOARD_____________"></span><span id="_______________guid_tfcat_tip_keyboard_____________"></span> GUID \_ TFCAT \_ 提示 \_ 鍵盤 
</dt> <dd>

鍵盤類型專案。

</dd> </dl>
