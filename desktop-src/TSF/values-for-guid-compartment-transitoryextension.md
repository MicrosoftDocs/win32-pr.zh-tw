# <a name="values-for-guid_compartment_transitoryextension"></a>GUID 區間 TRANSITORYEXTENSION 的值 \_ \_

下列 GUID 區間 TRANSITORYEXTENSION 區間的 \_ 值 \_ 可用來控制暫時性擴充功能的行為。

<dl> <dt>

<span id="TF_TRANSITORYEXTENSION_NONE"></span><span id="tf_transitoryextension_none"></span>**TF \_ TRANSITORYEXTENSION \_ NONE**
</dt> <dd>

此值會停止暫時性延伸模組。

</dd> <dt>

<span id="TF_TRANSITORYEXTENSION_FLOATING"></span><span id="tf_transitoryextension_floating"></span>**TF \_ TRANSITORYEXTENSION \_ 浮動**
</dt> <dd>

此值會以浮動 UI 啟動暫時性延伸模組。

</dd> <dt>

<span id="TF_TRANSITORYEXTENSION_ATSELECTION"></span><span id="tf_transitoryextension_atselection"></span>**TF \_ TRANSITORYEXTENSION \_ ATSELECTION**
</dt> <dd>

這個值會使用位於 IP 或父檔管理員選取範圍的快顯視窗 UI 來啟動暫時性延伸。

</dd> </dl>
