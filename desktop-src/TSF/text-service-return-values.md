# <a name="text-service-return-values"></a>文字服務傳回值

文字服務會產生 0x *hhhh hhhh* 0600 的傳回值。 下表列出文字服務傳回值。



| 傳回碼/值                                                                                                                                                                                                                          | Description                                                           |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------|
| <span id="TF_E_NOCONVERSION"></span><span id="tf_e_noconversion"></span><dl> <dt>**TF \_E \_ NOCONVERSION**</dt> <dt>0x80040600</dt> </dl> | 選取的文字範圍沒有任何可用的轉換。<br/> |
