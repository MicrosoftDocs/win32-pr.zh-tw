# <a name="text-services-framework-return-values"></a>文字服務架構傳回值

文字服務架構的介面方法會產生從 0x *hhhh hhhh* 0200 到 0x *hhhh hhhh* 0600 範圍的傳回值，其中 *hhhh hhhh* 是十六進位值，取決於特定傳回值的嚴重性代碼和設備碼。

傳回值會分組為下列類別目錄。

-   [文字服務傳回值](text-service-return-values.md)
-   [管理員傳回值](manager-return-values.md)
-   [文字存放區傳回值](text-store-return-values.md)
