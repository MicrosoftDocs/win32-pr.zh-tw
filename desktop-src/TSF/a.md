# <a name="a-text-services-framework"></a> (文字服務架構) 

<dl> <dt>

<span id="_tsf_anchor_gly"></span><span id="_TSF_ANCHOR_GLY"></span>**錨**
</dt> <dd>

標記文字範圍開頭和結尾的一組元素。 請參閱 [錨點](ranges.md)。

</dd> <dt>

<span id="_tsf_application_character_position_gly"></span><span id="_TSF_APPLICATION_CHARACTER_POSITION_GLY"></span>**應用程式字元位置 (ACP)**
</dt> <dd>

連續文字資料流程內以零為起始的字元位置，相對於資料流程的開頭。 ACP 會以長整數值表示。 請參閱 [文字存放區](text-stores.md)。

</dd> </dl>
