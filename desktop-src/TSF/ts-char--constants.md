# <a name="ts_char_-constants"></a>TS \_ CHAR \_ \* 常數

TS \_ CHAR \_ \* 常數描述字元的類型。



| 常數/值                                                                                                                                                                                                                                   | Description                                                                                         |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------|
| <span id="TS_CHAR_EMBEDDED"></span><span id="ts_char_embedded"></span><dl> <dt>**TS \_CHAR \_ EMBEDDED**</dt> <dt> ( 0xfffc )</dt> </dl>          | 字元是 Unicode 2.1 物件取代字元。<br/>                             |
| <span id="TS_CHAR_REGION"></span><span id="ts_char_region"></span><dl> <dt>**TS \_CHAR \_ 區域**</dt> <dt> ( 0 )</dt> </dl>                     | 字元是區域界限。<br/>                                                      |
| <span id="TS_CHAR_REPLACEMENT"></span><span id="ts_char_replacement"></span><dl> <dt>**TS \_字元 \_ 取代**</dt> <dt> ( 0xfffd )</dt> </dl> | 字元是隱藏文字預留位置字元或 Unicode 取代字元。<br/> |
