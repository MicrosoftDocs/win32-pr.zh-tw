# <a name="about-text-services-framework"></a>關於文字服務架構

Microsoft Windows 文字服務架構 (TSF) 是 (API) 的應用程式開發介面，可啟用與來源無關的 advanced Text 輸入。 TSF 會透過一組 COM 介面，向應用程式和文字服務公開其架構。

下列主題摘要說明 TSF 功能。

-   [為何要使用文字服務架構？](why-use-text-services-framework.md)
-   [架構](architecture.md)
-   [如何設定文字服務架構](how-to-set-up-tsf.md)
