# <a name="r-text-services-framework"></a>R (文字服務架構) 

<dl> <dt>

<span id="_tsf_range_gly"></span><span id="_TSF_RANGE_GLY"></span>**範圍**
</dt> <dd>

物件，描述存在於文字資料流程內的文字範圍。 範圍用來讀取、寫入和操作文字。 也稱為 *文字範圍*。

另請參閱 *region*。

</dd> <dt>

<span id="_tsf_reconversion_gly"></span><span id="_TSF_RECONVERSION_GLY"></span>**漢字**
</dt> <dd>

允許使用者針對插入的文字選取替代文字的程式。

</dd> <dt>

<span id="_tsf_region_gly"></span><span id="_TSF_REGION_GLY"></span>**地區**
</dt> <dd>

根據格式，以唯一編輯範圍的形式來處理的文字範圍。

</dd> </dl>
