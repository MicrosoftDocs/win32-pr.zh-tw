# <a name="ienumtfrenderingmarkupreset-method"></a>IEnumTfRenderingMarkup：： Reset 方法

**IEnumTfRenderingMarkup：： Reset** 方法會將目前的位置移至列舉序列的開頭，以重設列舉值物件。

## <a name="syntax"></a>語法


```C++
HRESULT Reset();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

<dl></dl>

## <a name="return-value"></a>傳回值

這個方法可以傳回其中一個值。



| 值                                                                                | 描述                           |
|--------------------------------------------------------------------------------------|---------------------------------------|
| <dl> <dt>**S \_ 確定**</dt> </dl> | 此方法成功。<br/> |
