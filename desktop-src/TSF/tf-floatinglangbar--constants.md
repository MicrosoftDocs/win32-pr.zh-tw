# <a name="tf_floatinglangbar_-constants"></a>TF \_ FLOATINGLANGBAR \_ \* 常數

\_針對語言列視窗的標題定義 TF FLOATINGLANGBAR \_ \* 常數位符串。



| 常數/值                                                                                                                                                                                                                                                                                  | Description                                                                                                                                                                  |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="TF_FLOATINGLANGBAR_WNDTITLEW"></span><span id="tf_floatinglangbar_wndtitlew"></span><dl> <dt>**TF \_FLOATINGLANGBAR \_ WNDTITLEW**</dt> <dt>LTF \_ FLOATINGLANGBAR \_ WndTitle</dt> </dl> | 標題的 Unicode 字元 L "TF \_ FloatingLangBar \_ WndTitle" 字串。<br/>                                                                                  |
| <span id="TF_FLOATINGLANGBAR_WNDTITLEA"></span><span id="tf_floatinglangbar_wndtitlea"></span><dl> <dt>**TF \_FLOATINGLANGBAR \_ WNDTITLEA**</dt> <dt>TF \_ FLOATINGLANGBAR \_ WndTitle</dt> </dl>  | 標題的 ANSI 字元 "TF \_ FloatingLangBar \_ WndTitle" 字串。<br/>                                                                                      |
| <span id="TF_FLOATINGLANGBAR_WNDTITLE"></span><span id="tf_floatinglangbar_wndtitle"></span><dl> <dt>**TF \_FLOATINGLANGBAR \_ WNDTITLE**</dt> <dt>TF \_ FLOATINGLANGBAR \_ WNDTITLEA</dt> </dl>    | 如果已定義 Unicode，則會接受 TF \_ FLOATINGLANGBAR \_ WNDTITLEW 常數的值。 否則，會取得 TF \_ FLOATINGLANGBAR \_ WNDTITLEA 常數的值。<br/> |



## <a name="remarks"></a>備註

雖然此群組中有三個常數，但主要一個是 TF \_ FLOATINGLANGBAR \_ WNDTITLE，視需要採用 ANSI 或 Unicode 值。
