# <a name="what-is-text-services-framework"></a>什麼是文字服務架構？

文字服務架構 (TSF) 是與裝置無關、非語言相關，而且可延伸的系統，適用于文字輸入和語言技術。

啟用 TSF 的應用程式可以從任何支援 TSF 的文字服務接收文字輸入，而不必留意文字來源的任何詳細資料。

文字服務提供文字給應用程式的方式，不需要對產生或接收文字的應用程式進行任何詳細的知識。 例如，文字服務可以提供語音或手寫輸入裝置的文字輸入。
