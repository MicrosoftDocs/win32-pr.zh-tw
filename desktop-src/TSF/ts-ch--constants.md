# <a name="ts_ch_-constants"></a>TS \_ CH \_ \* 常數

\_ \_ \* TSF 會在內部使用 TS CH 常數，以表示已發生文字刪除。



| 常數/值                                                                                                                                                                                                                               | Description                                                        |
|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------|
| <span id="TS_CH_PRECEDING_DEL"></span><span id="ts_ch_preceding_del"></span><dl> <dt>**TS \_CH \_ 之前 \_ 的 CH**</dt> <dt> ( 1 )</dt> </dl> | 已刪除字元位置之前的文字。<br/> |
| <span id="TS_CH_FOLLOWING_DEL"></span><span id="ts_ch_following_del"></span><dl> <dt>**TS \_\_以下是 \_ DEL**</dt> <dt> ( 2 )</dt> </dl> | 字元位置之後的文字已刪除。<br/> |
