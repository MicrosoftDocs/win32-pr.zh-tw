# <a name="other-predefined-attributes"></a>其他預先定義的屬性

下列值會識別以 [ITfCoNtext：： GetAppProperty](/windows/desktop/api/msctf/nf-msctf-itfcontext-getappproperty) 方法取得的其他屬性。 包含每個屬性類型的資料格式和內容。

## <a name="properties"></a>屬性



| 屬性                             | 描述                                                                       |
|--------------------------------------|-----------------------------------------------------------------------------------|
| **TSATTRID \_ 應用程式**                    | 未使用。                                                                         |
| **TSATTRID \_ 應用程式 \_ IncorrectGrammar**  | 如果文字包含文法錯誤，則包含非零值，否則為零。  |
| **TSATTRID \_ 應用程式 \_ IncorrectSpelling** | 如果文字包含拼寫錯誤，則包含非零值，否則為零。 |
| **TSATTRID \_ 其他人**                 | 未使用。                                                                         |
