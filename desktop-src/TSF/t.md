# <a name="t-text-services-framework"></a>T (文字服務架構) 

<dl> <dt>

<span id="_tsf_text_service_gly"></span><span id="_TSF_TEXT_SERVICE_GLY"></span>**文字服務**
</dt> <dd>

一種應用程式類型，可透過語音、多語系鍵盤或畫筆裝置來輸入文字。 文字服務也可以啟用文字編輯工具，例如拼寫和文法檢查。

另請參閱 *文字處理器*。

</dd> <dt>

<span id="_tsf_text_services_framework_gly"></span><span id="_TSF_TEXT_SERVICES_FRAMEWORK_GLY"></span>**文字服務架構 (TSF)**
</dt> <dd>

與裝置無關、非語言相關，以及適用于文字輸入和自然語言技術的可擴充系統。

</dd> </dl>
