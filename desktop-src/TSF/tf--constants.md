# <a name="tf_-constants"></a>TF \_ \* 常數

TSF atom 函式呼叫會使用這些常數來管理執行緒輸入進程。



| 常數/值                                                                                                                                                                                                                                                                       | Description                                                                                          |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------|
| <span id="TF_PROCESS_ATOM"></span><span id="tf_process_atom"></span><dl> <dt>**TF \_處理 \_ atom**</dt> <dt>文字 (\_ CTF \_ process \_ atom \_)</dt> </dl>                               | 用來通知 atom 函式，表示執行緒輸入進程已起始。<br/>                  |
| <span id="TF_ENABLE_PROCESS_ATOM"></span><span id="tf_enable_process_atom"></span><dl> <dt>**TF \_啟用 \_ 處理 \_ atom**</dt> <dt>文字 (\_ CTF \_ 啟用 \_ process \_ atom \_)</dt> </dl> | 用來透過 atom 函式通知 TSF，表示已啟用執行緒輸入進程。<br/> |
