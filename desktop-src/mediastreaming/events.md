# <a name="events-media-streaming-api"></a> (媒體串流 API) 的事件

[媒體串流 API](media-streaming-api-portal.md)會產生下列事件。

## <a name="in-this-section"></a>本節內容



| 主題                                                                           | 描述                                                                                                                                              |
|---------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ConnectionStatusChanged 事件**](connectionstatuschanged.md)<br/>     | 發生于裝置的網路連接狀態變更時。<br/>                                                                                   |
| [**DeviceArrival 事件**](devicearrival.md)<br/>                         | 當新的裝置新增至 [**CachedDevices**](idevicecontroller-cacheddevices.md) 方法所傳回的裝置清單時發生。<br/>     |
| [**DeviceDeparture 事件**](devicedeparture.md)<br/>                     | 從 [**CachedDevices**](idevicecontroller-cacheddevices.md) 方法傳回的裝置清單中移除新的裝置時發生。<br/> |
| [**RenderingParametersUpdate 事件**](renderingparametersupdate.md)<br/> | 每當 DMR 上有任何一組轉譯控制項參數進行更新時發生。<br/>                                                          |
| [**TransportParametersUpdate 事件**](transportparametersupdate.md)<br/> | 當 DMR 上有任何一組傳輸參數更新時發生。<br/>                                                                  |
