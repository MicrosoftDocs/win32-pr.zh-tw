# <a name="basicdevicepresentationurl-property"></a>BasicDevice. PresentationUrl 屬性

取得裝置的呈現 URL。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```C++
HRESULT get_PresentationUrl(
  [out] HSTRING *value
);
```



## <a name="property-value"></a>屬性值

裝置展示 URL 的指標。
