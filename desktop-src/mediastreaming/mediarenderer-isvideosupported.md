# <a name="mediarendererisvideosupported-property"></a>MediaRenderer. IsVideoSupported 屬性

取得值，這個值會指出 DMR 是否能夠播放影片內容。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```C++
HRESULT get_IsVideoSupported(
  [out] boolean *value
);
```



## <a name="property-value"></a>屬性值

布林值，如果 DMR 能夠播放影片內容則為 **True** ，否則為 **False** 。
