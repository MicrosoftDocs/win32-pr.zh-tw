# <a name="getpositioninformationoperation-class"></a>GetPositionInformationOperation 類別

註冊當 [**GetPositionInformationAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getpositioninformationasync) 啟動的非同步作業完成時叫用的事件處理常式，並提供方法來傳回作業的結果。

**GetPositionInformationOperation** 具有下列類型的成員：

-   [方法](#methods)
-   [屬性](#properties)

### <a name="methods"></a>方法

**GetPositionInformationOperation** 類別有這些方法。



| 方法                                                           | 描述                                                                                                                                                 |
|:-----------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetResults**](getpositioninformationoperation-getresults.md) | 傳回 [**GetPositionInformationAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getpositioninformationasync)啟動的非同步作業結果。 <br/> |



 

### <a name="properties"></a>屬性

**GetPositionInformationOperation** 類別具有這些屬性。



| 屬性                                                                  | 存取類型           | Description                                                                                                                                                                                          |
|:--------------------------------------------------------------------------|:----------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Completed**](getpositioninformationoperation-completed.md)<br/> | 讀取/寫入<br/> | 取得或設定當 [**GetPositionInformationAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getpositioninformationasync) 啟動的非同步作業完成時叫用的事件處理常式。 <br/> |
