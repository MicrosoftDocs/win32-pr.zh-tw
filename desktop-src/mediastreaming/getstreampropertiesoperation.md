# <a name="getstreampropertiesoperation-class"></a>GetStreamPropertiesOperation 類別

註冊當 [**GetStreamPropertiesAsync**](/previous-versions/windows/desktop/legacy/hh829001(v=vs.85)) 啟動的非同步作業完成時叫用的事件處理常式，並提供方法來傳回作業的結果。

**GetStreamPropertiesOperation** 具有下列類型的成員：

-   [方法](#methods)
-   [屬性](#getstreampropertiesoperation-class)

### <a name="methods"></a>方法

**GetStreamPropertiesOperation** 類別有這些方法。



| 方法                                                        | 描述                                                                                                                                          |
|:--------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetResults**](getstreampropertiesoperation-getresults.md) | 傳回 [**GetStreamPropertiesAsync**](/previous-versions/windows/desktop/legacy/hh829001(v=vs.85))啟動的非同步作業結果。<br/> |



 

### <a name="properties"></a>屬性

**GetStreamPropertiesOperation** 類別具有這些屬性。



| 屬性                                                               | 存取類型           | Description                                                                                                                                                                                   |
|:-----------------------------------------------------------------------|:----------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Completed**](getstreampropertiesoperation-completed.md)<br/> | 讀取/寫入<br/> | 取得或設定當 [**GetStreamPropertiesAsync**](/previous-versions/windows/desktop/legacy/hh829001(v=vs.85)) 啟動的非同步作業完成時叫用的事件處理常式。<br/> |
