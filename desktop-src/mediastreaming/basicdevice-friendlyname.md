# <a name="basicdevicefriendlyname-property"></a>BasicDevice FriendlyName 屬性

取得裝置的易記名稱。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```C++
HRESULT get_FriendlyName(
  [out] HSTRING *value
);
```



## <a name="property-value"></a>屬性值

裝置易記名稱的指標。
