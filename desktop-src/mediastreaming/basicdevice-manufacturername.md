# <a name="basicdevicemanufacturername-property"></a>BasicDevice. ManufacturerName 屬性

取得裝置的製造商名稱。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```C++
HRESULT get_ManufacturerName(
  [out] HSTRING *value
);
```



## <a name="property-value"></a>屬性值

裝置製造商名稱的指標。
