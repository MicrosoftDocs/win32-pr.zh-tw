# <a name="basicdeviceserialnumber-property"></a>BasicDevice. SerialNumber 屬性

取得裝置的序號。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```C++
HRESULT get_SerialNumber(
  [out] HSTRING *value
);
```



## <a name="property-value"></a>屬性值

裝置序號的指標。
