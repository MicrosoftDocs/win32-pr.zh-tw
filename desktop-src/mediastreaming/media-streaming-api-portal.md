# <a name="windows-media-streaming-api"></a>Windows媒體串流 API

## <a name="purpose"></a>目的

媒體串流 API 為數字媒體轉譯器 (DMR) 和數位媒體伺服器 (DMS) 裝置，如使用簡單的程式設計方式所定義，可讓開發人員建立具有數位媒體控制器的應用程式 (DMC) 、數位媒體播放機 (DMP) 和推播控制器 (+ PU +) 功能（如 DLNA 指導方針所定義）。

## <a name="developer-audience"></a>開發人員對象

媒體串流 API 是設計來供 c + + 開發人員使用，以建立與 DLNA 裝置互動的數位媒體應用程式。
