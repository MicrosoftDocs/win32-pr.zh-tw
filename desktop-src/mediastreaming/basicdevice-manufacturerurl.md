# <a name="basicdevicemanufacturerurl-property"></a>BasicDevice. ManufacturerUrl 屬性

取得裝置的製造商 URL。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```C++
HRESULT get_ManufacturerUrl(
  [out] HSTRING *value
);
```



## <a name="property-value"></a>屬性值

裝置製造商 URL 的指標。
