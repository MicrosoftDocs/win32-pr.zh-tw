# <a name="getmuteoperation-class"></a>GetMuteOperation 類別

註冊當 [**GetMuteAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getmuteasync) 啟動的非同步作業完成時叫用的事件處理常式，並提供方法來傳回作業的結果。

**GetMuteOperation** 具有下列類型的成員：

-   [方法](#methods)
-   [屬性](#properties)

### <a name="methods"></a>方法

**GetMuteOperation** 類別有這些方法。



| 方法                                            | 描述                                                                                                                   |
|:--------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------|
| [**GetResults**](getmuteoperation-getresults.md) | 傳回 [**GetMuteAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getmuteasync)啟動的非同步作業結果。 <br/> |



 

### <a name="properties"></a>屬性

**GetMuteOperation** 類別具有這些屬性。



| 屬性                                                   | 存取類型           | 描述                                                                                                                                                            |
|:-----------------------------------------------------------|:----------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Completed**](getmuteoperation-completed.md)<br/> | 讀取/寫入<br/> | 取得或設定當 [**GetMuteAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getmuteasync) 啟動的非同步作業完成時叫用的事件處理常式。 <br/> |
