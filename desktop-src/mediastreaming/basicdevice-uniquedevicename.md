# <a name="basicdeviceuniquedevicename-property"></a>BasicDevice. UniqueDeviceName 屬性

取得裝置 (UDN) 的唯一裝置名稱。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```C++
HRESULT get_UniqueDeviceName(
  [out] HSTRING *value
);
```



## <a name="property-value"></a>屬性值

裝置模型 UDN 的指標。
