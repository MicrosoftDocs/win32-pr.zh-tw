# <a name="basicdeviceremotestreamingurls-property"></a>BasicDevice. RemoteStreamingUrls 屬性

取得遠端串流 Url 的向量。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```C++
HRESULT get_RemoteStreamingUrls(
  [out] IVector< HSTRING > **value
);
```



## <a name="property-value"></a>屬性值

指向遠端串流 Url 之指標的可列舉集合。
