# <a name="basicdevicetype-property"></a>BasicDevice。 Type 屬性

取得列舉值，指出 DLNA 裝置的裝置類型。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```C++
HRESULT get_Type(
  [out] DeviceTypes *value
);
```



## <a name="property-value"></a>屬性值

[**DeviceTypes**](devicetypes.md)值的指標。
