# <a name="devicepairrenderer-property"></a>DevicePair：：轉譯器屬性

取得 active basic 裝置配對的轉譯器。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```C++
HRESULT get_Renderer(
  [out] ActiveBasicDevice **value
);
```



## <a name="property-value"></a>屬性值

接收代表轉譯器裝置的 [**ActiveBasicDevice**](/previous-versions/windows/desktop/legacy/dn385755(v=vs.85)) 物件。
