# <a name="basicdeviceipaddresses-property"></a>BasicDevice. IpAddresses 屬性

取得 IP 位址的向量。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```C++
HRESULT get_IpAddresses(
  [out] IVector< HSTRING > **value
);
```



## <a name="property-value"></a>屬性值

IP 位址指標的可列舉集合。
