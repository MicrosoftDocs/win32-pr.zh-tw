# <a name="mediarendererisaudiosupported-property"></a>MediaRenderer. IsAudioSupported 屬性

取得值，這個值會指出 DMR 是否能夠播放音訊內容。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```C++
HRESULT get_IsAudioSupported(
  [out] boolean *value
);
```



## <a name="property-value"></a>屬性值

布林值，如果 DMR 能夠播放音訊內容，則為 **True** ，否則為 **False** 。
