# <a name="media-streaming-structures"></a>媒體串流結構

[媒體串流 API](media-streaming-api-portal.md)提供下列結構。

## <a name="in-this-section"></a>本節內容



| 主題                                                           | 描述                                                                                                                                        |
|-----------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**PlaySpeed**](/previous-versions/windows/desktop/api/windows.media.streaming/ns-windows-media-streaming-playspeed)<br/>                       | 以有理數表示播放速度。 <br/>                                                                                      |
| [**PositionInformation**](/previous-versions/windows/desktop/api/windows.media.streaming/ns-windows-media-streaming-positioninformation)<br/>   | 包含從 DMR 取得之媒體播放位置資訊的目前值。<br/>                                               |
| [**RenderingParameters**](/previous-versions/windows/desktop/api/windows.media.streaming/ns-windows-media-streaming-renderingparameters)<br/>   | 包含 DMR 上呈現參數的目前值。<br/>                                                                         |
| [**TrackInformation**](/previous-versions/windows/desktop/api/windows.media.streaming/ns-windows-media-streaming-trackinformation)<br/>         | 包含目前的曲目數目和持續時間，作為從 DMR 取得之 [**PositionInformation**](/previous-versions/windows/desktop/api/windows.media.streaming/ns-windows-media-streaming-positioninformation) 的一部分。<br/> |
| [**TransportInformation**](/previous-versions/windows/desktop/api/windows.media.streaming/ns-windows-media-streaming-transportinformation)<br/> | 包含從 DMR 取得的 media 播放傳輸相關資訊目前的值。<br/>                                      |
