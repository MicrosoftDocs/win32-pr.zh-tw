# <a name="basicdevicephysicaladdresses-property"></a>BasicDevice. PhysicalAddresses 屬性

取得實體位址的向量。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```C++
HRESULT get_PhysicalAddresses(
  [out] IVector< HSTRING > **value
);
```



## <a name="property-value"></a>屬性值

實體位址指標的可列舉集合。
