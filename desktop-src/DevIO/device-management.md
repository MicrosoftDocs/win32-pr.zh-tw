# <a name="device-management"></a>裝置管理

裝置管理可讓您以一致的方式通知所有變更的應用程式和系統功能，這些變更可能會影響其作業或資源的存取權。 應用程式和系統會使用和處理裝置事件，以在新資源可用時利用這些資源，並在現有資源變成無法使用時防止資料遺失。
