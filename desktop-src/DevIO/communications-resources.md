# <a name="communications-resources"></a>通訊資源

*通訊資源* 是一種實體或邏輯裝置，可提供單一雙向的非同步資料流程。 序列埠、平行埠、傳真電腦和數據機是通訊資源的範例。 每個通訊資源都有一個服務提供者（由程式庫或驅動程式組成），可讓應用程式存取資源。

-   [使用通訊資源](using-communications-resources.md)
-   [通訊參考](communications-reference.md)
