# <a name="device_data_management_set_action"></a>裝置 \_ 資料 \_ 管理 \_ 設定 \_ 動作

下列常數值是 **裝置 \_ 資料 \_ 管理 \_ 集合 \_ 動作** 類型的可能值集合，其定義為 **DWORD** 類型。

<dl> <dt>
