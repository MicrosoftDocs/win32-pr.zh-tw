# <a name="importing-and-exporting-drm-protected-content"></a>匯入和匯出受 DRM 保護的內容

新 Windows 媒體 DRM 用戶端擴充 api 的主要角色之一，是要將受 DRM 保護的內容匯入和匯出至協力廠商 rights management 系統。

在您的應用程式中匯入和匯出受保護資料的主要挑戰，是要在您的系統與 DRM 子系統之間妥善共用 ASF 資料範例。 當您的應用程式處理資料範例時，必須確保符合特定的穩定性標準，以避免可能被規避防護機制。 這些標準會在您同意 DRM 時同意的健全狀況和合規性規則中加以概述。
