# <a name="description-windows-media-format-11-sdk"></a>描述 (Windows Media Format 11 SDK) 

**Description** 屬性包含檔案內容的描述。

## <a name="global-constant"></a>全域常數

g \_ wszWMDescription

## <a name="data-type"></a>資料類型

**WMT \_ 類型 \_ 字串**

## <a name="remarks"></a>備註

這是檔案層級的屬性。

若為 ID3v1 MP3 檔案，此屬性的限制為30個字元。
