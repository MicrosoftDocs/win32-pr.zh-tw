# <a name="about-the-windows-media-format-sdk"></a>關於 Windows 媒體格式 SDK

Microsoft® Windows 媒體®格式軟體發展工具組 (SDK) 可讓開發人員建立應用程式，以管理儲存在檔案中的數位媒體，這些檔案符合 Advanced Systems 格式 (ASF) 檔結構。 ASF 結構是適用于任意 [*串流*](wmformat-glossary.md) 媒體的多功能容器格式。 如需有關 ASF 檔案結構的詳細資訊，請參閱 [Asf 格式的總覽](overview-of-the-asf-format.md)。

此 SDK 也支援使用 Windows 媒體編解碼器來編碼和解碼音訊和影片內容。 每個編解碼器 (壓縮程式/解壓縮程式) 都會經過優化，以取得特定類型內容的最佳品質。 如需有關支援之編解碼器的詳細資訊，請參閱 [編解碼器功能](codec-features.md)。

除了一般檔案格式支援和編解碼器整合以外，Windows 媒體格式 SDK 也提供物件來編輯數位媒體的描述性資訊。 MP3 檔案也包含檔案讀取和中繼資料支援，可讓您建立更有彈性的應用程式。
