# <a name="audio-device-conformance-templates"></a>音訊裝置一致性範本

下表列出 Windows Media 音訊9編解碼器或更新版本的裝置一致性範本和相關聯的參數。



| 範本字串 | 位元速率範圍     | 備註                                                                                                                                                                                                                                |
|-----------------|--------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| L1            | 64 kbps 160 Kbps | 此範本適用于受限制的僅限音訊裝置。                                                                                                                                                                        |
| L2            | <= 160 Kbps     | 此範本會對應至 Windows Media 音訊移植套件中的類別4，可支援整個 Windows Media 音訊解碼器的執行。                                                                                   |
| L3            | <= 384 Kbps     | 此範本會對應至 Windows Media 音訊移植套件中的類別4，可支援整個 Windows Media 音訊解碼器的執行。位元速率範圍是此範本和 L2 之間的唯一差異。<br/> |
| L             | 所有位元速率      | 此範本僅供個人電腦使用，通常用來展示編解碼器的完整功能。                                                                                                           |



 

下表列出 Windows Media 音訊9語音編解碼器的裝置一致性範本和相關聯的參數。



| 範本字串 | 位元速率範圍 | 備註                                                                                                                      |
|-----------------|----------------|----------------------------------------------------------------------------------------------------------------------------|
| S1            | <= 20 Kbps  | 此範本僅適用于非常低複雜性的裝置。此範本僅限語音。<br/>                    |
| S2            | <= 20 Kbps  | 這是大部分應用程式的建議範本。此範本支援語音和音樂的組合。<br/> |



 

下表列出 Windows Media 音訊 9 Professional 編解碼器或更新版本的裝置一致性範本和相關聯的參數。



| 範本字串 | 屬性                                                                                   | 備註                                                                                                                                                                                                                                           |
|-----------------|----------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| M1            | 位元速率 <= 384 KbpsSampling 速率 <= 48 kHz<br/> 通道 <= 5。1<br/>   | 此範本建議用於具有環繞音效的標準電影。                                                                                                                                                                           |
| M2            | 位元速率 <= 768 KbpsSampling 速率 <= 96 kHz<br/> 通道 <= 5。1<br/>   | 此範本建議用於具有環繞音效的高定義電影。                                                                                                                                                                    |
| M3            | 位元速率 <= 1500 KbpsSampling 速率 <= 96 kHz<br/> 通道 <= 7。1<br/> | 此範本建議用於具有8聲道環繞音效的高定義影片，例如在劇院中編碼以呈現的內容。                                                                                                    |
| "M"             | 所有位 ratesAll 取樣率<br/> 所有通道<br/>                           | 此範本僅供個人電腦使用，通常用來展示編解碼器的完整功能。這也是使用 Windows Media 音訊9無失真編解碼器編碼的所有音訊的範本指定。<br/> |
