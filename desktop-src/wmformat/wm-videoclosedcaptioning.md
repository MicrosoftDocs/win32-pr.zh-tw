# <a name="wmvideoclosedcaptioning"></a>WM/VideoClosedCaptioning

**WM/VideoClosedCaptioning** 屬性會指出內容是否包含隱藏式輔助字幕。

## <a name="global-constant"></a>全域常數

g \_ wszWMVideoClosedCaptioning

## <a name="data-type"></a>資料類型

**WMT \_ 類型 \_ BOOL**

## <a name="remarks"></a>備註

此屬性用於廣播電視和其他影片內容。
