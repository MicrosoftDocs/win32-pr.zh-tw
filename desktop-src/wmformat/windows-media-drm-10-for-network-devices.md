# <a name="windows-media-drm-10-for-network-devices"></a>適用于網路裝置的 Windows Media DRM 10

您的應用程式可以使用適用于網路裝置的 Windows Media DRM 10，將受保護的內容串流至網路上的數位媒體接收者。 這項功能是安全地與網路裝置通訊的通訊協定。

Windows Media Format SDK 藉由維護網路裝置的資料庫、驗證這些裝置的網路鄰近性，以及將受 DRM 保護的內容轉換成安全的傳輸格式，來支援網路裝置的 Windows Media DRM 10。

> [!Note]  
> 此 SDK 的 x64 版本不支援 DRM。
