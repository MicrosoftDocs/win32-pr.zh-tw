# <a name="drm_crypto_type-enumeration"></a>DRM \_ 加密 \_ 類型列舉

**DRM \_ 加密 \_ 類型** 列舉會定義可用來加密內容的密碼編譯演算法類型。

## <a name="syntax"></a>Syntax


```C++
typedef enum DRM_CRYPTO_TYPE { 
  CRYPTO_TYPE_MCE  = 0
} ;
```



## <a name="constants"></a>常數

<dl> <dt>

<span id="CRYPTO_TYPE_MCE"></span><span id="crypto_type_mce"></span>**加密 \_ 類型 \_ MCE**
</dt> <dd>

指定 (AES) 加密進階加密標準128位的計數器模式。

</dd> </dl>
