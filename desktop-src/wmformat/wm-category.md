# <a name="wmcategory"></a>WM/類別

**WM/Category** 屬性包含內容的分類。 沒有分類的規格。 建議使用者輸入自己的類別來組織其內容。

## <a name="global-constant"></a>全域常數

g \_ wszWMCategory

## <a name="data-type"></a>資料類型

**WMT \_ 類型 \_ 字串**

## <a name="example"></a>範例



| 檔案類型 | 範例值   |
|-----------|-----------------|
| 音訊     | 「駕駛音樂」 |
| 影片     | 「首頁影片」   |
