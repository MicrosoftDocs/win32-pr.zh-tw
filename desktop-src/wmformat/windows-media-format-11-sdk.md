# <a name="windows-media-format-11-sdk"></a><span data-ttu-id="37035-110">Windows Media Format 11 SDK</span><span class="sxs-lookup"><span data-stu-id="37035-110">Windows Media Format 11 SDK</span></span>

<span data-ttu-id="37035-111">本檔說明 Microsoft Windows Media Format 軟體發展工具組 (SDK) 並適用于32位和 x64 版本的 SDK。</span><span class="sxs-lookup"><span data-stu-id="37035-111">This documentation describes the Microsoft Windows Media Format Software Development Kit (SDK) and applies to the 32-bit and x64-based versions of the SDK.</span></span>

<span data-ttu-id="37035-112">Windows Media Format SDK 是 Microsoft Windows Media 軟體發展工具組的元件， (SDK) 。</span><span class="sxs-lookup"><span data-stu-id="37035-112">The Windows Media Format SDK is a component of the Microsoft Windows Media Software Development Kit (SDK).</span></span> <span data-ttu-id="37035-113">其他元件包含 Windows Media Services SDK、Windows Media 編碼器 SDK、Windows Media Rights Manager SDK、Windows Media 裝置管理員 SDK 和 Windows Media Player SDK。</span><span class="sxs-lookup"><span data-stu-id="37035-113">Other components include the Windows Media Services SDK, Windows Media Encoder SDK, Windows Media Rights Manager SDK, Windows Media Device Manager SDK, and Windows Media Player SDK.</span></span>

<span data-ttu-id="37035-114">Windows Media Format SDK 可讓應用程式開發人員存取 Windows Media 格式的元件。</span><span class="sxs-lookup"><span data-stu-id="37035-114">The Windows Media Format SDK provides application developers with access to the components of the Windows Media Format.</span></span> <span data-ttu-id="37035-115">這些元件包括 Advanced Systems 格式 (ASF) 檔案容器、Windows Media 音訊和影片編解碼器、基本網路串流功能，以及數位版權管理。</span><span class="sxs-lookup"><span data-stu-id="37035-115">These components include the Advanced Systems Format (ASF) file container, the Windows Media Audio and Video codecs, basic network streaming capability, and digital rights management.</span></span> <span data-ttu-id="37035-116">Windows Media Format SDK 的物件會以較低的層級來操作 Windows Media 的元件;Windows Media SDK 的其他元件包含在較高層級上運作的物件。</span><span class="sxs-lookup"><span data-stu-id="37035-116">The objects of the Windows Media Format SDK manipulate the components of Windows Media at a low level; the other components of the Windows Media SDK include objects that work on a higher level.</span></span>

<span data-ttu-id="37035-117">Windows Media Format SDK 的主要目的是讓開發人員建立應用程式，以播放、寫入、編輯、加密和傳遞 Advanced Systems 格式 (ASF) 檔和網路串流。</span><span class="sxs-lookup"><span data-stu-id="37035-117">The primary purpose of the Windows Media Format SDK is to enable developers to create applications that play, write, edit, encrypt, and deliver Advanced Systems Format (ASF) files and network streams.</span></span> <span data-ttu-id="37035-118">這些檔案和資料流程通常包含使用 Windows Media 音訊和影片編解碼器編碼的音訊和影片內容。</span><span class="sxs-lookup"><span data-stu-id="37035-118">These files and streams commonly contain audio and video content encoded using the Windows Media Audio and Video codecs.</span></span> <span data-ttu-id="37035-119">不過，ASF 可以包含任何類型的資料。</span><span class="sxs-lookup"><span data-stu-id="37035-119">However, ASF can contain any type of data.</span></span> <span data-ttu-id="37035-120">如需 Advanced Systems Format 容器結構的詳細資訊，請參閱 [ASF 格式的總覽](overview-of-the-asf-format.md)。</span><span class="sxs-lookup"><span data-stu-id="37035-120">For more information about the Advanced Systems Format container structure, see [Overview of the ASF Format](overview-of-the-asf-format.md).</span></span>

<span data-ttu-id="37035-121">Windows Media Format SDK 的主要功能如下：</span><span class="sxs-lookup"><span data-stu-id="37035-121">The key features of the Windows Media Format SDK are:</span></span>

-   <span data-ttu-id="37035-122">支援領先業界的編解碼器。</span><span class="sxs-lookup"><span data-stu-id="37035-122">Support for industry-leading codecs.</span></span> <span data-ttu-id="37035-123">Windows Media Format 11 SDK 包含 Microsoft Windows Media 視訊9編解碼器和 Microsoft Windows Media 音訊9.1 編解碼器。</span><span class="sxs-lookup"><span data-stu-id="37035-123">The Windows Media Format 11 SDK includes the Microsoft Windows Media Video 9 codec and the Microsoft Windows Media Audio 9.1 codec.</span></span> <span data-ttu-id="37035-124">這兩種編解碼器都提供數位媒體內容的例外編碼。</span><span class="sxs-lookup"><span data-stu-id="37035-124">Both of these codecs provide exceptional encoding of digital media content.</span></span> <span data-ttu-id="37035-125">此版本的新功能是 Windows Media 視訊 9 Advanced Profile 編解碼器，提供廣播影片的優化。</span><span class="sxs-lookup"><span data-stu-id="37035-125">New for this release is the Windows Media Video 9 Advanced Profile codec, which provides optimizations for broadcast video.</span></span> <span data-ttu-id="37035-126">此 SDK 也包含 Microsoft Windows Media 視訊9畫面編解碼器，可在使用者應用程式會話期間壓縮電腦螢幕活動，以及 Windows Media 音訊9.1 語音編解碼器，它會編碼低複雜性的音訊（例如語音），並以智慧方式適應更複雜的音訊（例如音樂），以提供更佳的組合語音音樂案例。</span><span class="sxs-lookup"><span data-stu-id="37035-126">This SDK also includes the Microsoft Windows Media Video 9 Screen codec for compressing computer-screen activity during sessions of user applications, and the Windows Media Audio 9.1 Voice codec, which encodes low-complexity audio such as speech and intelligently adapts to more complex audio such as music, for superior representation of combined voice-music scenarios.</span></span>
-   <span data-ttu-id="37035-127">支援寫入 ASF 檔。</span><span class="sxs-lookup"><span data-stu-id="37035-127">Support for writing ASF files.</span></span> <span data-ttu-id="37035-128">檔案是根據可自訂的設定檔所建立，可讓您輕鬆地設定和標準化檔案。</span><span class="sxs-lookup"><span data-stu-id="37035-128">Files are created based on customizable profiles, enabling easy configuration and standardization of files.</span></span> <span data-ttu-id="37035-129">您可以使用此 SDK 來寫入超過 2 gb 的檔案，以啟用更長、更高品質的連續檔案。</span><span class="sxs-lookup"><span data-stu-id="37035-129">This SDK can be used to write files in excess of 2 gigabytes, enabling longer, better-quality, continuous files.</span></span>
-   <span data-ttu-id="37035-130">支援讀取 ASF 檔。</span><span class="sxs-lookup"><span data-stu-id="37035-130">Support for reading ASF files.</span></span> <span data-ttu-id="37035-131">此 SDK 提供讀取本機 ASF 檔案，以及讀取透過網路串流處理之 ASF 資料的支援。</span><span class="sxs-lookup"><span data-stu-id="37035-131">This SDK provides support for reading local ASF files as well as reading ASF data being streamed over a network.</span></span> <span data-ttu-id="37035-132">此外，也提供許多先進閱讀功能的支援，例如 (MBR) 檔案的原生支援，其中包含多個串流具有以不同位元速率編碼的相同內容。</span><span class="sxs-lookup"><span data-stu-id="37035-132">Support is also provided for many advanced reading features, such as native support for multiple bit rate (MBR) files, which contain multiple streams with the same content encoded at different bit rates.</span></span> <span data-ttu-id="37035-133">讀取器會根據播放時的可用頻寬，自動選取要使用的 MBR 串流。</span><span class="sxs-lookup"><span data-stu-id="37035-133">The reader automatically selects which MBR stream to use, depending upon available bandwidth at the time of playback.</span></span>
-   <span data-ttu-id="37035-134">透過網路傳遞 ASF 資料流程的支援。</span><span class="sxs-lookup"><span data-stu-id="37035-134">Support for delivering ASF streams over a network.</span></span> <span data-ttu-id="37035-135">此 SDK 支援透過 HTTP 將 ASF 資料傳遞到網路上的遠端電腦，也支援將資料直接傳遞至遠端 Windows Media 伺服器。</span><span class="sxs-lookup"><span data-stu-id="37035-135">This SDK provides support for delivering ASF data through HTTP to remote computers on a network, and also for delivering data directly to a remote Windows Media server.</span></span>
-   <span data-ttu-id="37035-136">支援在 ASF 檔案中編輯中繼資料。</span><span class="sxs-lookup"><span data-stu-id="37035-136">Support for editing metadata in ASF files.</span></span> <span data-ttu-id="37035-137">您可以使用此 SDK 輕鬆地操作檔案及其內容的相關資訊。</span><span class="sxs-lookup"><span data-stu-id="37035-137">Information about a file and its content is easily manipulated with this SDK.</span></span> <span data-ttu-id="37035-138">開發人員可以使用 SDK 中所包含之中繼資料屬性的健全系統，或建立自訂屬性以符合其需求。</span><span class="sxs-lookup"><span data-stu-id="37035-138">Developers can use the robust system of metadata attributes included in the SDK, or create custom attributes to suit their needs.</span></span>
-   <span data-ttu-id="37035-139">支援內容編輯應用程式。</span><span class="sxs-lookup"><span data-stu-id="37035-139">Support for content editing applications.</span></span> <span data-ttu-id="37035-140">此 SDK 可讓應用程式依簡報時間和影片畫面來搜尋檔案中的點。</span><span class="sxs-lookup"><span data-stu-id="37035-140">This SDK enables applications to seek to points within a file by presentation time and by video frame.</span></span> <span data-ttu-id="37035-141">此外，使用 Windows Media 格式 SDK 所建立的檔案，可以使用電影和電視生產環境中所使用的格式來維護時間戳記。</span><span class="sxs-lookup"><span data-stu-id="37035-141">In addition, files created by using the Windows Media Format SDK can maintain timestamps in formats used in film and television production.</span></span>
-   <span data-ttu-id="37035-142">支援在 MP3 檔中讀取和編輯中繼資料。</span><span class="sxs-lookup"><span data-stu-id="37035-142">Support for reading and editing metadata in MP3 files.</span></span> <span data-ttu-id="37035-143">此 SDK 提供使用讀取 ASF 檔案的相同方法來讀取 MP3 檔案的整合式支援。</span><span class="sxs-lookup"><span data-stu-id="37035-143">This SDK provides integrated support for reading MP3 files with the same methods used to read ASF files.</span></span> <span data-ttu-id="37035-144">以 Windows Media Format SDK 建立的應用程式也可以使用內容建立者所使用最常見的 ID3 標記內建的支援，編輯 MP3 檔案中的中繼資料屬性。</span><span class="sxs-lookup"><span data-stu-id="37035-144">Applications built with the Windows Media Format SDK can also edit metadata attributes in MP3 files using built-in support for the most common ID3 tags used by content creators.</span></span>
-   <span data-ttu-id="37035-145">支援數位 Rights Management 保護。</span><span class="sxs-lookup"><span data-stu-id="37035-145">Support for Digital Rights Management protection.</span></span> <span data-ttu-id="37035-146">此 SDK 提供的方法可用來讀取和寫入由數位 Rights Management 保護的 ASF 檔案和網路串流，以防止未經授權播放或複製內容。</span><span class="sxs-lookup"><span data-stu-id="37035-146">This SDK provides methods for reading and writing ASF files and network streams that are protected by Digital Rights Management to prevent unauthorized playback or copying of the content.</span></span>
