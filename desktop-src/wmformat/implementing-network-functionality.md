# <a name="implementing-network-functionality"></a>執行網路功能

本節說明如何使用可透過 Windows 媒體格式 SDK 取得的網路功能。 應用程式可以使用這些功能，從網路讀取 ASF 串流、將 ASF 串流廣播到網路，或將 ASF 串流推送至執行 Windows Media Services 之伺服器上的發行端點。

下列各節說明此 SDK 的網路功能。



| 區段                                                                    | 描述                                                                                                          |
|----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [網路介面的概觀](overview-of-networking-interfaces.md) | 命名並描述支援網路方法的每個介面。                                                 |
| [透過網路讀取 ASF 資料](reading-asf-data-over-a-network.md)     | 說明如何從網路來源播放檔案，以及如何設定讀取器物件上的網路設定。    |
| [透過網路傳送 ASF 資料](sending-asf-data-over-a-network.md)     | 說明如何透過網路廣播 asf 資料，或將 asf 資料傳送至 Windows 媒體伺服器上的發行端點。 |
| [預設網路設定](default-networking-settings.md)             | 提供 SDK 所使用之預設設定的快速參考。                                                 |
