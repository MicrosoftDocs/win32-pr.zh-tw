# <a name="windows-media-format-95-sdk-x64-edition"></a>Windows Media Format 9.5 SDK x64 版

本檔適用于新的 Windows Media Format 9.5 SDK x64 Edition，以及舊版的 Windows Media 格式 SDK 32 位版本。

下列清單摘要說明以 x64 為基礎之版本的 SDK 變更：

-   不支援 DRM。 已從 SDK 安裝中移除 DRM 相關的檔案 (包括) 的範例。
-   Windows Media 裝置管理員不是 x64 SDK 安裝的一部分。
-   支援 Windows Media 音訊和影片編解碼器介面的範例和標頭不是 x64 SDK 安裝的一部分。
-   當地語系化的系統設定檔不是 x64 SDK 安裝的一部分。

本檔的整個程式設計參考一節中的附注，會呼叫 Windows Media Format 9.5 SDK x64 Edition 不支援的功能。
