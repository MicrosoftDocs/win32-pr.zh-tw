# <a name="file-writing-features"></a>檔案寫入功能

Windows 媒體格式 SDK 的主要功能之一，是能夠以 Advanced Systems 格式寫入檔案 (ASF) 。 寫入器物件是用來寫入 ASF 檔。 如需詳細資訊，請參閱 [寫入器物件](writer-object.md)。

在最基本的檔案撰寫案例中，您可以指派要使用的設定檔，以及要建立之檔案的名稱。 您可以一次將範例傳遞給寫入器。 當您完成將範例傳遞給寫入器時，它會完成其作業並完成 ASF 檔案。 如需基本檔案寫入的詳細資訊，請參閱 [寫入 ASF](writing-asf-files.md)檔。

寫入器支援數個先進的功能，這些功能將在下列各節中討論。

-   [影片調整大小](video-resizing.md)
-   [色彩空間轉換](color-space-conversion.md)
-   [音訊重新取樣](audio-resampling.md)
-   [匯](sinks.md)
-   [浮水印支援](watermarking-support.md)
-   [輸入格式、輸入設定和資料單位延伸](input-formats-input-settings-and-data-unit-extensions.md)
-   [輸入格式列舉](input-format-enumeration.md)
