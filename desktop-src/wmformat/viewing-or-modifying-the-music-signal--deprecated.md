# <a name="viewing-or-modifying-the-music-signal-deprecated"></a> (淘汰的) 觀看或修改音樂信號

本頁面記載了在未來的 Windows 版本中，將會受到不同技術解決方案支援的功能。

在 Microsoft 安全音訊路徑 (SAP) 模型中，應用程式無法以任何方式修改受保護的音樂。 例如，當應用程式嘗試攔截音樂信號時，信號聽起來像是隨機雜訊。 如此一來，通常會修改信號 (的應用程式（例如等化器) ）無法變更音樂的音效。

有些應用程式只會看到音樂信號。 例如，有些應用程式會以音樂信號顯示時間閃爍的燈，但不會加以修改。 為了容納可查看信號的應用程式，會以純文字格式解密音樂的一小部分，並以加密的內容傳遞。 產生的信號很差 (比電話品質更糟) 但可滿足用來查看信號的應用程式。
