# <a name="high-resolution-audio-support"></a>High-Resolution 音訊支援

Windows Media 音訊 9 Professional 編解碼器和 Windows Media 音訊9無失真編解碼器都支援高解析度音訊。 高解析度音訊定義為音訊格式，每個樣本超過16個位。 個別音訊範例所使用的位數，會影響編碼音效的精確度。

因為已在 Windows 作業系統中使用 PCM 編碼的音效格式很長一段時間，所以某些 Windows 函式是舊的，且無法處理高解析度 PCM 音訊。 因此，高解析度 PCM 格式的表示方式是使用與標準音頻不同的結構。 如需詳細資訊，請參閱 [使用 High-Resolution PCM 音訊](working-with-high-resolution-pcm-audio.md)。
