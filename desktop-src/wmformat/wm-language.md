# <a name="wmlanguage"></a>WM/語言

**WM/Language** 屬性包含資料流程的語言。

## <a name="global-constant"></a>全域常數

g \_ wszWMLanguage

## <a name="data-type"></a>資料類型

**WMT \_ 類型 \_ 字串**

## <a name="remarks"></a>備註

使用的語言字串應該符合 RFC 1766。 如需詳細資訊，請參閱 [語言字串](language-strings.md)。
