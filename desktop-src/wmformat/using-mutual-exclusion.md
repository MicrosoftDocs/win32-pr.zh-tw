# <a name="using-mutual-exclusion"></a>使用相互排除

您可以使用相互排除物件來指定資料流程群組，一次只會傳遞一個資料流程群組。 相互排除可以用來處理各種不同的案例，這些案例都是 SDK 和自訂的原生。

下列各節說明如何使用不同類型的相互排除。



| 區段                                                                                  | 描述                                                                                                       |
|------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------|
| [使用多個位速率互斥](using-multiple-bit-rate-mutual-exclusion.md) | 描述如何使用相互排除，將一個輸入編碼為具有不同位速率設定的多個資料流程。 |
| [使用自訂相互排除類型](using-custom-mutual-exclusion-types.md)           | 討論自訂相互排除。                                                                                |
