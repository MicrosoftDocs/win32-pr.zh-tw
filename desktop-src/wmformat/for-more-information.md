# <a name="for-more-information-windows-media-format-sdk"></a>如需詳細資訊 (Windows Media Format SDK) 

-   如需有關 Windows Media Format SDK 已知問題的最新資訊，請參閱 [MSDN Library](/documentation/?url=%2flibrary%2fdnwmt%2fhtml%2fwmfsdk11_b2_release_notes_rzxb.asp%3fframe%3dtrue)網站上張貼的版本資訊。
-   如需有關一般串流媒體和 Windows Media Sdk 常見問題的功能、文章和解答，請參閱 [Microsoft](https://msdn.microsoft.com/windowsmedia/default.aspx)網站上的 Windows Media 開發人員中心。
-   如需取得 Windows Media Format SDK 授權的詳細資訊，請參閱 [Microsoft](https://msdn.microsoft.com/library/ms867178.aspx)網站上的 Windows media 授權頁面。
-   如需有關新聞群組的詳細資訊，請造訪 [Microsoft](https://windows.microsoft.com/windows/products/windows-media)網站上的 Windows Media 新聞群組和電子郵件別名頁面。
-   如需組成 ASF 檔案之資料物件的詳細、低層級描述，請從 [Microsoft 網站](https://download.microsoft.com/download/7/9/0/790fecaa-f64a-4a5e-a430-0bccdab3f1b4/ASF_Specification.doc)下載 ASF 規格。
-   如需有關數位版權管理 (DRM) 和 Windows Media Rights Manager 常見問題的功能、文章、最佳作法和解答，請參閱 Microsoft 網站上的 [數位 Rights Management](/documentation/?url=%2flibrary%2fdnwmt%2fhtml%2fdigitalrightsmanagement.asp) 頁面上的技術文章。
