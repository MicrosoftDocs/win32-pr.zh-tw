# <a name="wmaudiofileurl"></a>WM/AudioFileURL

**WM/AudioFileURL** 屬性包含官方網頁的位址，其中包含這個屬性所出現之檔案的相關資訊。 例如，歌曲可能會有一個連結回到演出者網站上的 [專輯] 頁面。
