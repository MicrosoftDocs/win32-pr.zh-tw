# <a name="wmt_videoimage_transition_cross_fade"></a>WMT \_ VIDEOIMAGE \_ 轉換 \_ 交叉 \_ 淡化

交叉淡出轉換沒有特殊效果;淡化 (溶解) 的特性取決於 [**WMT \_ VIDEOIMAGE \_ Sample2.xml**](/previous-versions/windows/desktop/api/Wmsdkidl/ns-wmsdkidl-wmt_videoimage_sample2)結構的 **fPrevBlendCoef** 和 **fCurrBlendCoef** 成員。

## <a name="parameters"></a>參數

這種轉換不接受任何參數。

## <a name="remarks"></a>備註

設定這項轉換與將 **WMT \_ VIDEOIMAGE \_ Sample2.xml** 結構的 **dwEffectType** 成員設定為0一樣。
