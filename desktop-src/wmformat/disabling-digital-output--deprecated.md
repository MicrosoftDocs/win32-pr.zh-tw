# <a name="disabling-digital-output-deprecated"></a>停用數位輸出 (已淘汰) 

本頁面記載了在未來的 Windows 版本中，將會受到不同技術解決方案支援的功能。

Microsoft 安全音訊路徑的另一項功能是停用數位輸出的能力。 某些音效卡提供從硬體裝置傳送數位輸出的方式;這些裝置可能已通過認證，且將從 DRM 核心元件通過授權。 因此，如果合法的音效卡已啟用數位輸出，則人們仍然可以捕獲一份完美的受保護音樂數位錄製。

這項功能可讓內容擁有者或授權簽發者藉由設定其音樂授權中的參數來停用數位輸出。 如果設定此參數，音效卡會在播放受保護的音樂時停用其數位輸出功能。 使用者可以接聽解密的音樂，但無法制作複本。
