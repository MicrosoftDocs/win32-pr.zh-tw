# <a name="getting-information-from-licenses-in-the-local-license-store"></a>從本機授權存放中的授權取得資訊

Windows 媒體 DRM 用戶端擴充 api 提供多種方式，可取得本機授權存放區中授權所授與的許可權資訊。 下列主題將說明如何取得這些不同類型的資訊。



| 主題                                                                                                  | 描述                                                                                                                                                                     |
|--------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [查詢簡單的許可權資訊](querying-for-simple-rights-information.md)                   | 描述如何取得受保護內容是否允許特定動作的簡單資訊。 例如，應用程式可以播放媒體檔案。                    |
| [查詢詳細的許可權資訊](querying-for-detailed-rights-information.md)               | 描述如何使用金鑰識別碼，取得受保護內容的本機授權存放區中，所有授權所授與之許可權的詳細匯總資訊。 |
| [列舉本機授權存放區中的授權](enumerating-licenses-in-the-local-license-store.md) | 說明如何從本機授權商店取得個別授權。                                                                                                          |
