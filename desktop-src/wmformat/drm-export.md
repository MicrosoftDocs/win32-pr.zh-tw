# <a name="drm-export"></a>DRM 匯出

Windows Media DRM 匯出功能可讓您建立授權的應用程式，以解密受 Windows Media DRM 保護的內容，並在 (CPS 的輸出內容保護設定中重新加密，) 由相關聯的授權簽發者明確指定。 授權簽發者會透過包含清單，明確地授權將內容匯出至特定 CPS。

> [!Note]  
> 若要存取匯出功能，您必須申請來自 Microsoft 的 [匯出應用程式憑證](export-application-certificate.md) 。

 

下列各節將說明 Windows Media DRM 匯出功能。



| 區段                                                                                      | 描述                                                                                         |
|----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [匯出應用程式憑證](export-application-certificate.md)                         | 描述匯出應用程式憑證。                                                        |
| [明確授權和包含清單](explicit-authorization-and-inclusion-lists.md) | 說明明確授權的程式，以及包含清單的運作方式。                      |
| [匯出壓縮的內容](exporting-compressed-content.md)                             | 說明如何從 Windows Media DRM 受保護的音訊或影片內容匯出已壓縮的內容。 |
