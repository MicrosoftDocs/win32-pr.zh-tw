# <a name="wmgenre"></a>WM/內容類型

**WM/** 內容屬性包含內容的內容類型。

## <a name="global-constant"></a>全域常數

g \_ wszWMGenre

## <a name="data-type"></a>資料類型

**WMT \_ 類型 \_ 字串**

## <a name="remarks"></a>備註

這是指定內容類型的慣用屬性。

如果您變更了 MP3 檔中的 **wm/** 內容類型或 **wm/GenreID** ，其他屬性將會變更為 [符合]。

### <a name="example"></a>範例



| 檔案類型 | 範例值 |
|-----------|---------------|
| 音訊     | 搖動        |
| 影片     | 戲劇       |
