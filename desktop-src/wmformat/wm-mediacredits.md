# <a name="wmmediacredits"></a>WM/MediaCredits

**WM/MediaCredits** 屬性包含廣播媒體的信用額度。

## <a name="global-constant"></a>全域常數

g \_ wszWMMediaCredits

## <a name="data-type"></a>資料類型

**WMT \_ 類型 \_ 字串**

## <a name="remarks"></a>備註

這個屬性會用於廣播電視或無線電。
