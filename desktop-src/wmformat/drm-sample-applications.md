# <a name="microsoft-windows-media-drm-client-sample-applications"></a>Microsoft Windows Media DRM 用戶端範例應用程式

使用 Windows Media DRM 用戶端擴充 Api 的範例應用程式是以 Microsoft Visual Studio 2005 的專案形式提供。

下表摘要說明這些範例。 您可以在每個範例目錄的 readme.txt 檔案中找到每個範例的其他相關資訊，包括使用方式資訊。



| 範例      | 描述                                                                                                                                 |
|-------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| QueryRights | 報告用戶端電腦上授權授與的許可權，以取得孩子的清單。 示範快速查詢和授權狀態查詢。 |
| DRMUpdate   | 會執行「個人化」，並建立在處理過程中引發之所有事件的記錄檔。                                                   |
