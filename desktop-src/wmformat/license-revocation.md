# <a name="license-revocation-windows-media-format-sdk"></a> (Windows Media Format SDK) 的授權撤銷

Windows Media Format SDK 提供的方法，可讓應用程式與授權簽發者的網站互動以撤銷內容授權。

授權撤銷的一般案例是當使用者提供舊電腦給他人時。 在此情況下，即使該使用者不再使用該電腦，原始使用者也可能會對儲存在電腦上的受保護內容有許多授權。 授權撤銷可讓原始使用者移除電腦的授權。

授權撤銷程式涉及用戶端應用程式與授權簽發者所建立的 Web 應用程式之間來回傳遞的訊息。

**注意** 此 SDK 的 x64 版本不支援 DRM。
