# <a name="low-delay-audio"></a>Low-Delay 音訊

「低延遲音訊」是一種編碼模式，會產生比其他模式更小之緩衝區視窗設定的壓縮音訊。 這適用于需要在播放期間快速切換的資料流程。 這項功能的一般案例是資料流程處理的簡報，其中包含了任意切換內容的能力，例如變更電視的頻道。

當您使用低延遲的音訊格式時，相較于其他音訊格式，切換內容的延遲會大幅降低。 當您使用低延遲格式時，即時廣播的延遲也會降低。

Windows Media 音訊9.1 和 Windows Media 音訊 9.1 Professional 編解碼器支援這項功能。 低延遲格式僅適用于一次傳遞和兩次傳遞)  (的常數位元速率編碼。
