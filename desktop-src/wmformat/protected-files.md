# <a name="protected-files"></a>受保護的檔案

使用 Windows Media DRM 保護檔案時，會加密檔案的資料區段。 保護檔案的處理常式也稱為封裝檔案。 Windows Media DRM 用戶端擴充 Api 無法讓您封裝檔案。 您可以使用 Windows Media Rights Manager SDK 或 Windows Media Format SDK 的物件來封裝檔案。

使用者若未先取得該檔案的授權，就無法存取受保護檔案中的資料。
