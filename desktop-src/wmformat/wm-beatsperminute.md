# <a name="wmbeatsperminute"></a>WM/BeatsPerMinute

**WM/BeatsPerMinute** 屬性包含內容每分鐘的節拍。

## <a name="global-constant"></a>全域常數

g \_ wszWMBeatsPerMinute

## <a name="data-type"></a>資料類型

**WMT \_ 類型 \_ 字串**

## <a name="remarks"></a>備註

**BeatsPerMinute** 屬性的值是儲存為字串的整數。

### <a name="example"></a>範例



| 檔案類型 | 範例值 |
|-----------|---------------|
| 音訊     | "120"         |
