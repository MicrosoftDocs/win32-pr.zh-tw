## <a name="remarks"></a>備註

此 SDK 中每個介面的介面識別碼都是前面加上 "IID" 的介面名稱 \_ 。 例如， **IWMReaderAdvanced** 介面的識別碼是 IID \_ IWMReaderAdvanced。 若要取得讀取器物件之 **IWMReaderAdvanced** 介面的指標，請使用下列程式碼，其中 *pReader* 是 **IWMReader** 介面的指標：


```C++
IWMReaderAdvanced *pReaderAdvanced;
hr = pReader->QueryInterface(IID_IWMReaderAdvanced,
                             (LPVOID*) &pReaderAdvanced);

```
