# <a name="wmcontentgroupdescription"></a>WM/ContentGroupDescription

**WM/ContentGroupDescription** 屬性包含內容群組的描述。

## <a name="global-constant"></a>全域常數

g \_ wszWMContentGroupDescription

## <a name="data-type"></a>資料類型

**WMT \_ 類型 \_ 字串**

## <a name="remarks"></a>備註

內容群組是個別檔案可能所屬的媒體集合。 例如，音訊檔案可能是一首歌曲，也就是一張一張光碟片上的一張光碟。 同樣地，影片剪輯可以是數列的一部分。
