# <a name="aspectratioy"></a>AspectRatioY

**AspectRatioY** 屬性包含影片串流之圖元外觀比例的高度元件。

## <a name="global-constant"></a>全域常數

g \_ wszWMAspectRatioY

## <a name="data-type"></a>資料類型

**WMT \_ 類型 \_ DWORD**

## <a name="remarks"></a>備註

存取寫入器物件的 **IWMHeaderInfo3** 介面時，您可以加入或變更此值。 在其他物件 (中繼資料編輯器、讀取器和同步讀取器) ，此值是唯讀的。
