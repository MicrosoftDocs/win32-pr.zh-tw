# <a name="wmwmshadowfilesourcefiletype-windows-media-format-11-sdk"></a>WM/WMShadowFileSourceFileType (Windows Media Format 11 SDK) 

**WM/WMShadowFileSourceFileType** 包含 ASF 檔案衍生來源原始檔案的檔案類型。

## <a name="global-constant"></a>全域常數

g \_ wszWMWMShadowFileSourceFileType

## <a name="data-type"></a>資料類型

**WMT \_ 類型 \_ 字串**

## <a name="remarks"></a>備註

存在於 ASF 以外之檔案格式的內容，可以透過稱為「授權匯入」的程式，轉換成受 Windows Media DRM 保護的 Windows Media 檔案。 這類 ASF 檔案應包含此屬性以及 WM/WMShadowFileSourceDRMType，以追蹤內容的來源。
