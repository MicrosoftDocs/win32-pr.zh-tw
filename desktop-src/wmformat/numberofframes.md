# <a name="numberofframes"></a>NumberOfFrames

**NumberOfFrames** 屬性包含影片資料流程中的框架數目。

## <a name="global-constant"></a>全域常數

g \_ wszWMNumberOfFrames

## <a name="data-type"></a>資料類型

**WMT \_ 類型 \_ QWORD**

## <a name="remarks"></a>備註

寫入器會在編碼時新增影片資料流程的這個值。 **NumberOfFrames** 不是唯讀的。 不過，您應該只在編輯檔案時變更值，變更影片資料流程中的框架數目。
