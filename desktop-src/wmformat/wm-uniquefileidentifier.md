# <a name="wmuniquefileidentifier"></a>WM/UniqueFileIdentifier

**WM/UniqueFileIdentifier** 屬性包含內容的唯一檔案識別碼。

## <a name="global-constant"></a>全域常數

g \_ wszWMUniqueFileIdentifier

## <a name="data-type"></a>資料類型

**WM \_ 類型 \_ 字串**

## <a name="remarks"></a>備註

唯一的檔案識別碼是可供應用程式和服務用來唯一識別檔案的一般字串。 字串包含以分號分隔的任意值。 您永遠不應清除此屬性。 您可以附加值並移除您自己的值，但所有其他值都應該保持不變。
