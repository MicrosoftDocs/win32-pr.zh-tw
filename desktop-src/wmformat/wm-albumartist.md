# <a name="wmalbumartist"></a>WM/AlbumArtist

**WM/AlbumArtist** 屬性包含專輯主要演出者的名稱。

## <a name="global-constant"></a>全域常數

g \_ wszWMAlbumArtist

## <a name="data-type"></a>資料類型

**WMT \_ 類型 \_ 字串**

## <a name="remarks"></a>備註

對於曲目可能具有不同演出者的專輯（例如 tribute 專輯或共同作業專輯），此屬性可以用來指定專輯的主要演出者。 最好使用此屬性，而不是使用「不同演出者」的作者值。
