# <a name="wmencodingsettings"></a>WM/EncodingSettings

**WM/EncodingSettings** 屬性包含用來編碼內容的設定。

## <a name="global-constant"></a>全域常數

g \_ wszWMEncodingSettings

## <a name="data-type"></a>資料類型

**WMT \_ 類型 \_ 字串**

## <a name="remarks"></a>備註

沒有標準格式可指出編碼設定。 不過，最好是以英文描述編碼設定，而不是包含設定的傾印。
