# <a name="drm_drmheader"></a>DRM \_ DRMHeader

**DRM \_DRMHeader** 是 DRMHeader 屬性的基底字串，例如 DRM \_ HeaderKeyID。

## <a name="global-constant"></a>全域常數

g \_ wszWMDRM \_ DRM \_ DRMHeader

## <a name="data-type"></a>資料類型

**WMT \_ 類型 \_ 字串**

## <a name="remarks"></a>備註

這個字串可以用來建立自訂的 DRMHeader 屬性。 它本身不是屬性。
