# <a name="windows-media-drm-client-extended-apis"></a>Windows媒體 DRM 用戶端擴充 Api

\[Windows 媒體 DRM 功能已被取代，不應該使用。 請改用[Microsoft PlayReady](/windows/uwp/audio-video-camera/playready-client-sdk) 。\]

本檔說明 Microsoft Windows 媒體數位 Rights Management (DRM) 用戶端擴充的應用程式開發介面 (api) 。 Windows 媒體 DRM 用戶端擴充 api 包括可用來管理用戶端電腦上 Windows 媒體數位 Rights Management (DRM) 作業的物件。

這些 Api 的主要焦點在於管理受保護數位媒體內容的授權。 此外，您也可以使用 api 來更新用戶端電腦上的 DRM 元件，並建立應用程式，以使用網路裝置 Windows 媒體 DRM 來傳輸內容。

這些 api 會構成 Windows 媒體版權管理員 SDK 的用戶端對應項。 使用 Windows 媒體版權管理員來建立可保護檔案和發出授權的線上服務時，會使用 Windows Media DRM 用戶端擴充 api 來建立使用該內容的應用程式。
