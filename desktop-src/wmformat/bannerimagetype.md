# <a name="bannerimagetype"></a>BannerImageType

**BannerImageType** 屬性會指定 [**BannerImageData**](bannerimagedata.md)屬性中所儲存的影像類型。 這個屬性是 [**WMT \_ 影像 \_ 類型**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_image_type) 列舉類型中的值。

## <a name="global-constant"></a>全域常數

g \_ wszWMBannerImageType

## <a name="data-type"></a>資料類型

**WMT \_ 類型 \_ DWORD**
