# <a name="file-streams"></a>檔案資料流程

檔案資料流程包含一或多個資料檔案。 當您以資料流程的形式傳送檔案資訊時，您應該將檔案名附加至資料流程中的每個範例。 如需將相關資訊附加至範例的詳細資訊，請參閱 [資料單位延伸](data-unit-extensions.md)。
