# <a name="idrmstatuscallback-interface"></a>IDRMStatusCallback 介面

**IDRMStatusCallback** 介面會提供一種方法，來接收非同步 DRM 呼叫的狀態。

## <a name="members"></a>成員

**IDRMStatusCallback** 介面繼承自 [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown)介面。 **IDRMStatusCallback** 也有下列類型的成員：

-   [方法](#methods)

### <a name="methods"></a>方法

**IDRMStatusCallback** 介面具有這些方法。



| 方法                                          | 描述                                                      |
|:------------------------------------------------|:-----------------------------------------------------------------|
| [**OnStatus**](idrmstatuscallback-onstatus.md) | 接收來自非同步進程的狀態訊息。<br/> |
