# <a name="rights-constants"></a>許可權常數

下表所列的常數可用來識別 DRM 動作，以及建立動作清單。



| 常數                                        | 描述                                                                                                                                                                                                                                                    |
|-------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| g \_ wszWMDRM \_ 時 \_ 標記                    | 定義動作清單的 XML 元素名稱。                                                                                                                                                                                                               |
| g \_ wszWMDRM \_ 動作 \_ 標記                        | 定義動作清單中專案的 XML 元素名稱。                                                                                                                                                                                                   |
| g \_ wszWMDRM \_ 右 \_ 播放                    | 定義播放內容之許可權的字串。                                                                                                                                                                                                              |
| g \_ wszWMDRM \_ RIGHT \_ COPY                        | 定義複製內容之許可權的字串。                                                                                                                                                                                                              |
| g \_ wszWMDRM \_ 右 \_ 播放清單 \_ 燒錄              | 定義在播放清單中將內容燒錄至 CD 之許可權的字串。                                                                                                                                                                                  |
| g \_ wszWMDRM \_ RIGHT \_ 建立 \_ 縮圖 \_ 影像    | 定義右邊的字串，以從影片內容建立縮圖影像。                                                                                                                                                                               |
| g \_ wszWMDRM \_ RIGHT \_ 複製 \_ 到 \_ CD                | 定義將內容複寫到 CD 之許可權的字串。 新的授權不應使用此許可權。 相反地，所有授與複製內容許可權的許可權都應該包含在複製許可權的上方，且播放清單會正確地進行燒錄。                                     |
| g \_ wszWMDRM \_ RIGHT \_ 複製 \_ 到 \_ SDMI \_ 裝置      | 定義將內容複寫到符合「安全數位音樂」方案 (SDMI) 之裝置許可權的字串。 新的授權不應使用此許可權。 相反地，複製內容的擁有權限都應該包含在複製許可權的涵蓋範圍內。 |
| g \_ wszWMDRM \_ RIGHT \_ COPY \_ 至 \_ 非 \_ SDMI \_ 裝置 | 定義要複製到不符合安全數位音樂方案 (SDMI) 之裝置的字串。 新的授權不應使用此許可權。 相反地，複製內容的擁有權限都應該包含在複製許可權的涵蓋範圍內。 |
| g \_ wszWMDRM \_ 右 \_ 備份                      | 定義備份授權之許可權的字串。                                                                                                                                                                                                        |
| g \_ wszWMDRM \_ RIGHT \_ 協同作業 \_ 播放         | 定義在共用播放清單中以網路播放內容之許可權的字串。                                                                                                                                                                  |
