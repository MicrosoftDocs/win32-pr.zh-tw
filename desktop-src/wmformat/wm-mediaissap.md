# <a name="wmmediaissap"></a>WM/MediaIsSAP

**WM/MediaIsSAP** 屬性會指出內容是否包含第二個音訊程式 (SAP) 。

## <a name="global-constant"></a>全域常數

g \_ wszWMMediaIsSAP

## <a name="data-type"></a>資料類型

**WMT \_ 類型 \_ BOOL**

## <a name="remarks"></a>備註

這個屬性是用於廣播電視。

SAP 通常會識別包含不同語言之替代聲軌的電視節目。
