# <a name="live-drm"></a>即時 DRM

從 Windows Media 9 系列開始，Windows Media Format SDK 可以用來撰寫受 DRM 保護的檔案。 這會啟用像是即時 DRM 的案例，在此情況下，DRM 保護會在編碼時套用至檔案或資料流程，而不是在編碼完成之後做為後置處理步驟。 Live DRM 讓內容擁有者能夠保護即時廣播，例如按演唱會或體育活動。 Live DRM 也代表一種簡化或簡化的方式來建立受保護的檔案。 您可以使用即時 DRM 來套用任何版本的 DRM 保護。 使用即時 DRM 建立的受保護檔案，與做為後置處理步驟保護的檔案完全相同。

**注意** 此 SDK 的 x64 版本不支援 DRM。
