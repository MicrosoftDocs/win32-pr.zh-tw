# <a name="revocation-type-identifiers"></a>撤銷類型識別碼

下表所列的常數會定義可識別不同類型之撤銷的 GUID 值。



| 常數                         | 描述                                                                                         |
|----------------------------------|-----------------------------------------------------------------------------------------------------|
| WMDRM \_ 撤銷 \_ 類型 \_ 應用程式     | 識別與應用程式憑證有關的撤銷資訊。                           |
| WMDRM \_ 撤銷 \_ 類型 \_ 裝置  | 識別有關裝置憑證的撤銷資訊。                                |
| WMDRM \_ 撤銷 \_ 類型 \_ CARDEA  | 識別與網路裝置憑證 Windows 媒體 DRM 相關的撤銷資訊。 |
| WMDRM \_ 撤銷 \_ 類型 \_ REVINFO | 未記載于此版本中。                                                                     |
| WMDRM \_ 撤銷 \_ 類型 \_ GRL     | 未記載于此版本中。                                                                     |
| WMDRM \_ 撤銷 \_ 類型 \_ HDCP    | 未記載于此版本中。                                                                     |
