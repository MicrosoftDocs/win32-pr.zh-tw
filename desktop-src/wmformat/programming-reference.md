# <a name="windows-media-format-sdk-programming-reference"></a>Windows媒體格式 SDK 程式設計參考

Microsoft® Windows 媒體®格式軟體發展工具組 (SDK) 支援一系列的物件、介面、獨立函式、結構、列舉類型、屬性等等。 下列各節將詳細說明這些詳細資料。



| 區段                                                                              | 描述                                                                                                  |
|--------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------|
| [物件](objects.md)                                                               | 描述 Windows 媒體格式 SDK 中每個物件所支援的物件和介面。           |
| [函數](functions.md)                                                           | 描述所有的獨立函式，通常是用來建立和初始化各種物件的函式。      |
| [介面](interfaces.md)                                                         | 描述此 SDK 中的所有介面和方法。                                                        |
| [結構](structures.md)                                                         | 描述此 SDK 所支援的結構。                                                              |
| [列舉類型](enumeration-types.md)                                           | 描述此 SDK 所支援的列舉類型。                                                       |
| [屬性](attributes.md)                                                         | 描述可在 ASF 檔案的標頭中指定的屬性。                                  |
| [媒體類型](media-types.md)                                                       | 描述此 SDK 所使用的媒體類型識別碼。                                                       |
| [輸出設定](output-settings.md)                                               | 描述用來識別讀取器輸出設定的全域常數。                                      |
| [語言字串](language-strings.md)                                             | 描述常用於 RFC 1766 相容的語言識別項。                                         |
| [裝置一致性範本參數](device-conformance-template-parameters.md) | 描述裝置一致性範本，其描述適用于各種裝置的值範圍。 |
| [系統設定檔](system-profiles.md)                                               | 列出所有支援的系統設定檔。                                                                         |
| [當地語系化的系統設定檔](localized-system-profiles.md)                           | 列出 SDK 隨附的當地語系化系統設定檔檔案，以及與每個檔案相關聯的語言。       |
| [GUID 值](guid-values.md)                                                       | 描述此 SDK 所使用的 GUID 值。                                                                  |
| [錯誤碼](error-codes.md)                                                       | 描述 SDK 中方法和函式所傳回的常見錯誤碼。                                   |
| [來源外掛程式](source-plug-ins.md)                                               | 描述使用者實作為來源的需求。                                                     |
