# <a name="wmplaylistdelay"></a>WM/PlaylistDelay

**WM/PlaylistDelay** 屬性包含在播放清單中的檔案播放之前應等待的毫秒數。

## <a name="global-constant"></a>全域常數

g \_ wszWMPlaylistDelay

## <a name="data-type"></a>資料類型

**WMT \_ 類型 \_ 字串**

## <a name="remarks"></a>備註

並非所有媒體播放機都能接受播放清單延遲。
