# <a name="wmproviderstyle"></a>WM/ProviderStyle

**WM/ProviderStyle** 屬性包含中繼資料內容提供者所指派之檔案的樣式。

## <a name="global-constant"></a>全域常數

g \_ wszWMProviderStyle

## <a name="data-type"></a>資料類型

**WMT \_ 類型 \_ 字串**

## <a name="remarks"></a>備註

此屬性基本上是第二個類型指定。 因為不同的分類器具有與內容相關聯之內容的不同概念，所以提供者樣式可能不同于內容建立者所指派的內容類型。
