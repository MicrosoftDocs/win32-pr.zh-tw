# <a name="local-license-store"></a>本機授權商店

當 DRM 授權傳遞至用戶端電腦時，會將它放入本機授權存放區。 這是硬碟上受保護的檔案，其中包含對使用者發出的所有授權，以及其他 DRM 資訊。

您可以使用 Windows Media DRM 用戶端擴充 Api，以幾個有限的方式操作本機授權存放區中的資料。

除了本機授權存放區之外，某些物件也會使用暫時的授權存放。 暫時存放區中的授權只會在短時間記憶體在。 不過，某些在暫存存放區中開始的授權可以儲存到一般本機授權存放區。
