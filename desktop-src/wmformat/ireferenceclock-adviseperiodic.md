# <a name="ireferenceclockadviseperiodic-method"></a>IReferenceClock：： AdvisePeriodic 方法

這個方法尚未實作。

在其他 **IReferenceClock** 介面的執行中（例如在 Microsoft® DirectX®的 DirectShow®元件）中， **AdvisePeriodic** 方法是用來建立週期性建議要求，以便在每次指定的時間間隔過後，發出信號給指定的事件物件。 此 SDK 不會執行這個方法的功能，對它所做的任何呼叫都會產生 E >notimpl 的傳回值 \_ 。

## <a name="syntax"></a>語法


```C++
 AdvisePeriodic();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。
