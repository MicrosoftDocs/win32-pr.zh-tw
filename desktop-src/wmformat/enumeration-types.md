# <a name="windows-media-format-sdk-enumeration-types"></a>Windows Media 格式 SDK 列舉類型

Windows Media 格式 SDK 會執行下列列舉類型。



| 列舉類型                                                               | Description                                                                                                                      |
|--------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------|
| [**DRM \_ HTTP \_ 狀態**](drm-http-status.md)                                   | 定義 DRM 要求的狀態範圍。                                                                                     |
| [**DRM \_ 授權 \_ 狀態 \_ 類別**](drm-license-state-category.md)            | 定義 DRM 授權字串的分類。                                                                                  |
| [**DRM 的 \_ 個人化 \_ 狀態**](drm-individualization-status.md)         | 定義 DRM 個人化的有效狀態。                                                                              |
| [**NETSOURCE \_ URLCREDPOLICY \_ 設定**](/previous-versions/windows/desktop/api/wmsinternaladminnetsource/ne-wmsinternaladminnetsource-netsource_urlcredpolicy_settings) | 定義用戶端電腦上可能存在的可能安全性原則設定。                                                   |
| [**WM \_ AETYPE**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wm_aetype)                                                | 在 IP 位址存取清單中指定專案的許可權。                                                             |
| [**WMT \_ ATTR \_ 資料類型**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_attr_datatype)                               | 定義基本資料類型的範圍。 這些值用來識別此 SDK 中各種方法所傳回的資料類型。 |
| [**WMT \_ ATTR \_ IMAGETYPE**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_attr_imagetype)                             | 定義可以儲存在 ASF 檔案標頭中的影像類型範圍。                                                         |
| [**WMT \_ 編解碼器 \_ 資訊 \_ 類型**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_codec_info_type)                          | 定義編解碼器所使用的資料大小範圍。                                                                                   |
| [**WMT \_ 認證 \_ 旗標**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_credential_flags)                         | 定義取得認證時可使用的旗標。                                                                   |
| [**WMT \_ DRMLA \_ 信任**](/previous-versions/windows/desktop/api/wmsdkidl/ne-wmsdkidl-wmt_drmla_trust)                                   | 定義 DRM 授權取得 URL 的信任狀態。                                                                        |
| [**WMT \_ FILESINK \_ 模式**](/previous-versions/windows/desktop/api/wmsdkidl/ne-wmsdkidl-wmt_filesink_mode)                               | 定義 file 接收所接受的輸入類型。                                                                            |
| [**WMT \_ 映射 \_ 類型**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_image_type)                                     | 定義用於橫幅廣告的可能影像類型。                                                                            |
| [**WMT \_ 索引 \_ 類型**](/previous-versions/windows/desktop/api/wmsdkidl/ne-wmsdkidl-wmt_index_type)                                     | 定義可以關聯以時間為基礎之索引的物件。                                                              |
| [**WMT \_ 索引子 \_ 類型**](/previous-versions/windows/desktop/api/wmsdkidl/ne-wmsdkidl-wmt_indexer_type)                                 | 定義索引子支援的索引類型。                                                                          |
| [**WMT \_ NET \_ 通訊協定**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_net_protocol)                                 | 定義網路接收器所支援的通訊協定類型。                                                                   |
| [**WMT \_ MUSICSPEECH \_ 類別 \_ 模式**](/previous-versions/windows/desktop/api/wmsdkidl/ne-wmsdkidl-wmt_musicspeech_class_mode)            | 定義可用 Windows Media 音訊9語音編解碼器的壓縮模式。                                                |
| [**WMT \_ 位移 \_ 格式**](/previous-versions/windows/desktop/api/wmsdkidl/ne-wmsdkidl-wmt_offset_format)                               | 定義此 SDK 中使用的位移類型。                                                                                   |
| [**WMT \_ 播放 \_ 模式**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_play_mode)                                       | 定義讀取器的播放選項。                                                                                      |
| [**WMT \_ PROXY \_ 設定**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_proxy_settings)                             | 定義用於讀取器物件的網路 proxy 設定。                                                                 |
| [**WMT \_ 許可權**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_rights)                                              | 定義可在 DRM 授權中指定的許可權。                                                                       |
| [**WMT \_ 狀態**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_status)                                              | 定義狀態旗標的範圍。                                                                                                 |
| [**WMT \_ 儲存體 \_ 格式**](/previous-versions/windows/desktop/api/wmsdkidl/ne-wmsdkidl-wmt_storage_format)                             | 定義可以使用此 SDK 操作的檔案類型。                                                                    |
| [**WMT \_ 資料流程 \_ 選取專案**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_stream_selection)                         | 定義資料流程的狀態。                                                                                                  |
| [**WMT \_ 傳輸 \_ 類型**](/previous-versions/windows/desktop/api/wmsdkidl/ne-wmsdkidl-wmt_transport_type)                             | 定義此 SDK 所支援的傳輸類型。                                                                               |
| [**WMT \_ 版本**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_version)                                            | 定義 Windows Media 格式 SDK 的版本號碼。                                                                     |
| [**WMT \_ 水位線 \_ 專案 \_ 類型**](/previous-versions/windows/desktop/api/wmsdkidl/ne-wmsdkidl-wmt_watermark_entry_type)                | 定義支援的浮水印類型。                                                                                       |
