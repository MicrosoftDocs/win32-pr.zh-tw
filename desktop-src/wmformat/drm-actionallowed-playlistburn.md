# <a name="drm_actionallowed_playlistburn"></a>DRM \_ ActionAllowed \_ PlaylistBurn

**DRM \_ ActionAllowed \_ PlaylistBurn** 屬性會指出是否允許將內容複寫到 CD 作為播放清單的一部分。

## <a name="global-constant"></a>全域常數

g \_ wszWMDRM \_ ActionAllowed \_ PlaylistBurn

## <a name="data-type"></a>資料類型

**WMT \_ 類型 \_ BOOL**

## <a name="remarks"></a>備註

這是使用 [**IWMDRMReader：： GetDRMProperty**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmreader-getdrmproperty)取出的唯讀屬性。
