# <a name="sdk-versions"></a>SDK 版本

本檔涵蓋 Windows 媒體格式 7 sdk （透過 Windows 媒體格式 11 sdk）。 個別方法、結構、函數和列舉都包含其參考主題中的版本需求。

只有 Windows Vista 和 Windows XP 支援 Windows 媒體格式 11 SDK。

只有 Windows XP 支援 Windows 媒體格式 9.5 SDK。 如果您撰寫的應用程式是以 Windows 媒體格式 9.5 SDK 為基礎，您的應用程式將無法在任何其他作業系統上執行。

Windows 媒體格式9系列 SDK 的一些功能也需要 Windows XP。 與這些功能相關的個別參考主題會呼叫該需求。
