# <a name="spdif-audio-output"></a>S/PDIF 音訊輸出

某些音訊裝置會使用索尼/Philips 數位互連格式 (S/PDIF) 來傳輸音訊資料。 您現在可以從使用 Windows 媒體 Professional 編解碼器編碼的音訊，將音訊解碼成 S/PDIF。
