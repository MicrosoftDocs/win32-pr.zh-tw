# <a name="export-application-certificate"></a>匯出應用程式憑證

若要匯出 Windows 媒體 drm 受保護的內容，您必須取得 Windows 媒體 drm 匯出應用程式憑證，您會在向 Microsoft 輸入 DRM 匯出合約之後收到此憑證。
