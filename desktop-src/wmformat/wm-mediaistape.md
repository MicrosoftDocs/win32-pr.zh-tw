# <a name="wmmediaistape"></a>WM/MediaIsTape

**WM/MediaIsTape** 屬性會指出內容是否是從先前錄製的磁帶廣播。

## <a name="global-constant"></a>全域常數

g \_ wszWMMediaIsTape

## <a name="data-type"></a>資料類型

**WMT \_ 類型 \_ BOOL**

## <a name="remarks"></a>備註

這個屬性是用於廣播電視。
