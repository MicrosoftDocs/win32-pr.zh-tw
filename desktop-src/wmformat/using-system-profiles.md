# <a name="using-system-profiles"></a>使用系統設定檔

Windows Media Format SDK 隨附的系統設定檔是設計來涵蓋數位媒體內容（由音訊和影片組成）的基本需求。 由於現在提供了各式各樣的功能，而且可能的數位媒體案例有相關的增加，因此已 deemphasized 系統設定檔。 最新的一組系統設定檔會使用 Windows XP 所發行的第8版 Windows Media 編解碼器。 雖然不支援最新版本的編解碼器，但系統設定檔也是建立設定檔以符合您需求的絕佳起點。

下列主題描述如何使用系統設定檔做為 ASF 檔案的基礎。

| [載入系統設定檔](to-load-a-system-profile.md)                             | 說明如何將系統設定檔載入至設定檔物件。    |
