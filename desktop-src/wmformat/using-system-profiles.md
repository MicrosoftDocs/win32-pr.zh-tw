# <a name="using-system-profiles"></a>使用系統設定檔

Windows 媒體格式 SDK 隨附的系統設定檔的設計目的，是要涵蓋音訊和影片所組成之數位媒體內容的基本需求。 由於現在提供了各式各樣的功能，而且可能的數位媒體案例有相關的增加，因此已 deemphasized 系統設定檔。 最新的一組系統設定檔會使用第8版 Windows 媒體編解碼器，與 Windows XP 一起發行。 雖然不支援最新版本的編解碼器，但系統設定檔也是建立設定檔以符合您需求的絕佳起點。

下列主題描述如何使用系統設定檔做為 ASF 檔案的基礎。

| [載入系統設定檔](to-load-a-system-profile.md)                             | 說明如何將系統設定檔載入至設定檔物件。    |
