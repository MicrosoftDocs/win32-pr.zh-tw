# <a name="windows-media-format-sdk-programming-guide"></a>Windows Media Format SDK 程式設計指南

程式設計指南旨在協助您瞭解如何使用 Microsoft Windows Media Format 軟體發展工具組 (SDK) 來建立應用程式，以使用 Advanced Systems 格式的檔案 (ASF) 。

您可以使用 Windows Media Format SDK 來建立應用程式，以將數位媒體串流寫入及讀取來自 ASF 檔案的資料流程。 此 SDK 也支援在 ASF 檔案中編輯中繼資料。 此外，此 SDK 可以用來讀取和操作 MP3 檔案中的中繼資料。
