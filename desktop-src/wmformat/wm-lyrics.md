# <a name="wmlyrics"></a>WM/歌詞

**WM/歌詞** 屬性以簡單字串的形式包含歌詞。

## <a name="global-constant"></a>全域常數

g \_ wszWMLyrics

## <a name="data-type"></a>資料類型

**WMT \_ 類型 \_ 字串**

## <a name="remarks"></a>備註

如需更多的歌詞儲存方式，請使用 [**WM/歌詞 \_ 內容**](wm-lyrics-synchronised.md) 屬性。
