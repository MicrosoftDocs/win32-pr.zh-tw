# <a name="wmmediaisfinale"></a>WM/MediaIsFinale

**WM/MediaIsFinale** 屬性會指出內容是否為數列的最後一個。

## <a name="global-constant"></a>全域常數

g \_ wszWMMediaIsFinale

## <a name="data-type"></a>資料類型

**WMT \_ 類型 \_ BOOL**

## <a name="remarks"></a>備註

這個屬性是用於廣播電視。
