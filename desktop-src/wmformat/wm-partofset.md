# <a name="wmpartofset"></a>WM/PartOfSet

**WM/PartOfSet** 屬性包含檔案所屬集合中的元件編號和部分總數。 這兩個值會以斜線分隔。

## <a name="global-constant"></a>全域常數

g \_ wszWMPartOfSet

## <a name="data-type"></a>資料類型

**WMT \_ 類型 \_ 字串**

## <a name="example"></a>範例



| 檔案類型 | 範例值 |
|-----------|---------------|
| 全部       | "2/3"         |
