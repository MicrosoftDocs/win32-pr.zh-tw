# <a name="wmoriginalartist"></a>WM/OriginalArtist

**WM/OriginalArtist** 屬性包含最初產生內容之演出者的名稱。

## <a name="global-constant"></a>全域常數

g \_ wszWMOriginalArtist

## <a name="data-type"></a>資料類型

**WMT \_ 類型 \_ 字串**

## <a name="remarks"></a>備註

原始演出者通常用於 remakes 歌曲。 適用于這類檔案。 使用原本錄製歌曲之寬線的 [**作者**](author.md) 名稱。
