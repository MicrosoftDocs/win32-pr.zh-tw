# <a name="video-complexity-settings"></a>影片複雜性設定

部分視頻編碼器支援多個 *複雜性層級*。 用來編碼資料流程的複雜性層級不會直接影響該資料流程的位元速率，但可能會影響品質。 位元速率是在 ASF 檔案中壓縮樣本大小的量值，複雜性層級是重建壓縮資料所需的處理能力量值。 因此，複雜度層級是由播放平臺的處理能力所決定，而不是可用的頻寬。

如需有關使用複雜性層級的詳細資訊，請參閱設定 [影片串流](configuring-video-streams.md)。
