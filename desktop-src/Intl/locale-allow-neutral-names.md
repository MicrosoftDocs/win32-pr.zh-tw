# <a name="locale_allow_neutral_names"></a>地區設定 \_ 允許 \_ 中性 \_ 名稱

**Windows 7 和更新版本：** 在地區設定名稱和地區設定識別碼之間進行轉換時，允許傳回中性名稱或 Lcid。
