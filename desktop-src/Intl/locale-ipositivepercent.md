# <a name="locale_ipositivepercent"></a>地區設定 \_ IPOSITIVEPERCENT

**Windows 7 和更新版本：** 地區設定的正百分比格式模式。 只能指定一個模式。



| 值 | 格式                                    |
|-------|-------------------------------------------|
| 0     | 數位、空間、百分比;例如， \# % |
| 1     | Number、percent、例如， \#%         |
| 2     | 百分比、數位;例如，%\#         |
| 3     | 百分比、空格、數位;例如，% \# |
