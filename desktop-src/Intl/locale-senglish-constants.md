# <a name="locale_senglish-constants"></a>地區設定 \_ SENGLISH \* 常數

本主題定義 NLS 所使用的地區設定 \_ SENGLISH \* 常數。



| 值                        | 意義                                                                                                                                                                                                                                                  |
|------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 地區設定 \_ SENGLISHCOUNTRYNAME  | **Windows 7 和更新版本：** 國家/地區的英文名稱，例如，德國代表 Deutschland。 這個名稱一律會限制為可以對應到 ASCII 127 字元子集的字元。 <br/>                                      |
| 地區設定 \_ SENGLISHDISPLAYNAME  | **Windows 7 和更新版本：** 以英文顯示地區設定的名稱。 顯示名稱通常是由語言和國家/地區所組成，例如德文 (德國) Deutsch (Deutschland) 。 <br/>                                           |
| 地區設定 \_ SENGLISHLANGUAGENAME | **Windows 7 和更新版本：** 英文版語言的英文名稱，例如德文（Deutsch），來自國際 ISO 標準639。 這個名稱一律會限制為可以對應到 ASCII 127 字元子集的字元。 <br/> |
