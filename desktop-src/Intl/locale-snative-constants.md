# <a name="locale_snative-constants"></a>地區設定 \_ SNATIVE \* 常數

本主題定義 \_ \* NLS 用來代表原生語言名稱的地區設定 SNATIVE 常數。



| 值                       | 意義                                                                                                                                                                                                                                                            |
|-----------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 地區設定 \_ SNATIVECOUNTRYNAME  | **Windows 7 和更新版本：** 國家/地區的原生名稱，例如，España，西班牙。 此字串允許的最大字元數為80，包括結束的 null 字元。                                                                 |
| 地區設定 \_ SNATIVECTRYNAME     | 適用于 Windows 7 和更新版本的已淘汰。 國家/地區的原生名稱。 請參閱地區設定 \_ SNATIVECOUNTRYNAME。                                                                                                                                                             |
| 地區設定 \_ SNATIVECURRNAME     | **Windows Me/98、windows 2000：** 與地區設定相關聯之貨幣的原生名稱（以地區設定的原生語言）。 此字串允許的字元數沒有限制。                                                          |
| 地區設定 \_ SNATIVEDIGITS       | ASCII 0 到9的原生對等專案。 此字串允許的最大字元數為十個字元，包括結束的 null 字元。 例如，阿拉伯文會使用 "012345 6789"。 另請參閱 [地區設定 \_ IDIGITSUBSTITUTION](locale-idigitsubstitution.md)。 |
| 地區設定 \_ SNATIVEDISPLAYNAME  | **Windows 7 和更新版本：** 以原生語言顯示地區設定的名稱，例如，Deutsch (Deutschland) 適用于地區設定德文 (德國) 。 <br/>                                                                                                        |
| 地區設定 \_ SNATIVELANGNAME     | 適用于 Windows 7 和更新版本的已淘汰。 語言的原生名稱。 請參閱地區設定 \_ SNATIVELANGUAGENAME。                                                                                                                                                                  |
| 地區設定 \_ SNATIVELANGUAGENAME | **Windows 7 和更新版本：** 語言的原生名稱，例如 (亞美尼亞) 的亞美尼亞文Հայերեն。 此字串允許的最大字元數為80，包括結束的 null 字元。                                                         |
