# <a name="availability-of-mui"></a>MUI 的可用性

下列版本的 Windows 包含 Api 和結構，以提供使用 MUI 技術開發多語系應用程式的原生支援：

-   Windows 7 和更新版本。
-   Windows Vista。

提供下層程式庫 [**LoadMUILibrary**](/windows/desktop/api/Muiload/nf-muiload-loadmuilibrarya)，可讓您針對 windows vista 之前的 windows 版本撰寫多語系應用程式，同時提供與針對 windows vista 和更新版本的 windows 寫入的 MUI 應用程式相容的行為。
