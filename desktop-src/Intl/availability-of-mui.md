# <a name="availability-of-mui"></a>MUI 的可用性

下列 Windows 版本包含 api 和結構，以提供使用 MUI 技術開發多語系應用程式的原生支援：

-   Windows 7 和更新版本。
-   Windows Vista。

提供下層程式庫 [**LoadMUILibrary**](/windows/desktop/api/Muiload/nf-muiload-loadmuilibrarya)，可讓您針對 Windows Vista 之前的 Windows 版本撰寫多語系應用程式，同時提供與 Windows Vista 和更新版本的 Windows 所寫入的 MUI 應用程式相容的行為。
