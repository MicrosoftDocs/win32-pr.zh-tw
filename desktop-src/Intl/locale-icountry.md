# <a name="locale_idialingcode-and-locale_icountry"></a>地區設定 \_ IDIALINGCODE 和地區設定 \_ ICOUNTRY

國家/地區代碼，以國際電話代碼為基礎，也稱為 IBM 國家/地區代碼。 此字串允許的最大字元數為六個字元，包括結束的 null 字元。

Windows 10 將地區設定 \_ IDIALINGCODE 新增為地區設定 ICOUNTRY 的更精確名稱的同義字 \_ 。 建議您使用新的名稱，以改善程式碼的可讀性。

> [!Note]  
> 國際撥號代碼對特定國家/地區而言並不是唯一的。 例如，包含美國和加拿大的數個國家/地區，請使用 "1" 作為國際電話號碼。
