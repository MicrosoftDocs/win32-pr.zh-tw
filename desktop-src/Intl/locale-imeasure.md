# <a name="locale_imeasure"></a>地區設定 \_ IMEASURE

測量系統。 此字串允許的最大字元數為二，包括結束的 null 字元。 如果計量系統 (Systéme 國際 d'Units 或使用 S.I. ) ，此值為0，如果使用美國系統，則此值為1。
