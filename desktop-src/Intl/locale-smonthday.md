# <a name="locale_smonthday"></a>地區設定 \_ SMONTHDAY

**Windows 7 和更新版本：** 只顯示月份和日的格式字串。 格式與針對 [地區設定 \_ SLONGDATE](locale-slongdate.md)所定義的格式類似。 例如，如果 [月/日] 模式是完整月份名稱，後面加上前置零的日號（如同 "九月 03"），則格式字串為 "MMMM dd"。 字串可以包含 day 和 month 格式圖片的組合，以及以單引號括住的任何字元字串。 以單引號括住的字元會維持指定的狀態，例如，「de」表示西班牙文 (西班牙) 。 地區設定只能指定一個月/日格式。
