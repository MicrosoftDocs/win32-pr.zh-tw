# <a name="using-nonspacing-characters-and-diacritics"></a>使用非間距字元和變音符號

當您的 Unicode 或分隔文字時，您的 Unicode 應用程式必須將非空白字元與其基底字元保持不中斷。 許多腳本包含的字元會以視覺化方式結合輸出上的其他字元，例如，以變音符號 ( 「浮動重音」 ) 。
