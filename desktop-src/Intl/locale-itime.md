# <a name="locale_itime"></a>地區設定 \_ ITIME

時間格式規格。 規格是下列其中一個值。 最好讓您的應用程式使用地區設定 [ \_ STIMEFORMAT](locale-stime-constants.md) 常數，而不是地區設定 \_ ITIME。



| 值 | 描述          |
|-------|----------------------|
| 0     | 上午/下午12小時格式 |
| 1     | 24小時制       |
