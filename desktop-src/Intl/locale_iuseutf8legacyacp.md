# <a name="locale_iuseutf8legacyacp"></a>地區設定 \_ IUSEUTF8LEGACYACP

**從下一版的 Windows 開始：** 建議應用程式改為使用 Unicode。 在使用者已設定為使用 utf-8 字碼頁的電腦上，LOCALE_IUSEUTF8LEGACYACP 是在非 utf-8 舊版 Windows 電腦上使用的舊版 Windows 字碼頁。
