# <a name="locale_iuseutf8legacyoemcp"></a>地區設定 \_ IUSEUTF8LEGACYOEMCP

**從下一版的 Windows 開始：** 建議應用程式改為使用 Unicode。 在使用者已設定為使用 utf-8 字碼頁的電腦上，LOCALE_IUSEUTF8LEGACYOEMCP 是在非 utf-8 舊版 Windows 電腦上使用的舊版 OEM 字碼頁。
