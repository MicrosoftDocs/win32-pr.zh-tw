# <a name="locale_sortname"></a>地區設定 \_ GETMEMBERS

**Windows 7 和更新版本：** 要用於排序的地區設定名稱（如果不同于 [地區設定 \_ SNAME](locale-sname.md)所指定的地區設定）。 自訂地區設定會使用這個常數。 另請參閱 [地區設定 \_ SSORTNAME](locale-ssort-constants.md)。
