# <a name="locale_sshortdate"></a>地區設定 \_ SSHORTDATE

地區設定的簡短日期格式字串。 此字串允許的最大字元數為80，包括結束的 null 字元。 此字串可以包含 [day、month、year 和 era 格式圖片](day--month--year--and-era-format-pictures.md)的組合。 例如，"M/d/yyyy" 表示2004年9月3日寫入9/3/2004。

地區設定可以定義多個簡短的日期格式。 若要取得此地區設定的所有簡短日期格式，請使用 [EnumDateFormats](/windows/desktop/api/Winnls/nf-winnls-enumdateformatsa)、 [EnumDateFormatsEx](/windows/desktop/api/Winnls/nf-winnls-enumdateformatsexa)或 [EnumDateFormatsExEx](/windows/desktop/api/Winnls/nf-winnls-enumdateformatsexex)。
