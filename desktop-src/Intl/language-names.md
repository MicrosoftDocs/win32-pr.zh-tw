# <a name="language-names"></a>語言名稱

語言名稱基本上會使用與 [地區設定名稱](locale-names.md)相同的格式。 例如，英文 (貝里斯) 地區設定的語言名稱是 "依"。 每個語言名稱都對應至 [語言識別項](language-identifiers.md)。

語言名稱和地區設定格式設定格式的主要差異如下：

-   語言名稱絕不會包含排序次序。 因此，對應至地區設定名稱 "hu-HU \_ technl" (匈牙利文語言、匈牙利和技術排序) 的語言名稱就是「hu-hu」。
-   語言名稱可以是與地區設定相關的「中性」。 例如，"en" (英文) 是一種有效的中性語言名稱，與 "依" (英文、貝里斯) 不同。
