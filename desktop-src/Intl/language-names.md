# <a name="language-names"></a><span data-ttu-id="8c2e1-105">語言名稱</span><span class="sxs-lookup"><span data-stu-id="8c2e1-105">Language Names</span></span>

<span data-ttu-id="8c2e1-106">語言名稱基本上會使用與 [地區設定名稱](locale-names.md)相同的格式。</span><span class="sxs-lookup"><span data-stu-id="8c2e1-106">A language name uses essentially the same formatting as a [locale name](locale-names.md).</span></span> <span data-ttu-id="8c2e1-107">例如，英文 (貝里斯) 地區設定的語言名稱是 "依"。</span><span class="sxs-lookup"><span data-stu-id="8c2e1-107">For example, the language name for the English (Belize) locale is "en-BZ".</span></span> <span data-ttu-id="8c2e1-108">每個語言名稱都對應至 [語言識別項](language-identifiers.md)。</span><span class="sxs-lookup"><span data-stu-id="8c2e1-108">Each language name corresponds to a [language identifier](language-identifiers.md).</span></span>

<span data-ttu-id="8c2e1-109">語言名稱和地區設定格式設定格式的主要差異如下：</span><span class="sxs-lookup"><span data-stu-id="8c2e1-109">The main differences between formatting for a language name and formatting for a locale name are:</span></span>

-   <span data-ttu-id="8c2e1-110">語言名稱絕不會包含排序次序。</span><span class="sxs-lookup"><span data-stu-id="8c2e1-110">A language name never includes a sort order.</span></span> <span data-ttu-id="8c2e1-111">因此，對應至地區設定名稱 "hu-HU \_ technl" (匈牙利文語言、匈牙利和技術排序) 的語言名稱就是「hu-hu」。</span><span class="sxs-lookup"><span data-stu-id="8c2e1-111">Thus, the language name corresponding to the locale name "hu-HU\_technl" (Hungarian language, Hungary, with technical sort) is just "hu-HU".</span></span>
-   <span data-ttu-id="8c2e1-112">語言名稱可以是與地區設定相關的「中性」。</span><span class="sxs-lookup"><span data-stu-id="8c2e1-112">A language name can be "neutral" with respect to locale.</span></span> <span data-ttu-id="8c2e1-113">例如，"en" (英文) 是一種有效的中性語言名稱，與 "依" (英文、貝里斯) 不同。</span><span class="sxs-lookup"><span data-stu-id="8c2e1-113">For example, "en" (English) is a valid neutral language name, as distinct from "en-BZ" (English, Belize).</span></span>
