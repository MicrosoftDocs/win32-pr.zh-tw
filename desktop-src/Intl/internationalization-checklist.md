# <a name="internationalization-checklist"></a>國際化檢查清單

本主題提供建立支援多個市場之程式碼所採取的動作。 當您評估組建時，請將這些語句視為程式碼設計和計量的指南。

-   建立可考慮從一開始就進行國際考慮的程式規格。
    -   將圖示和點陣圖設計成有意義且不具冒犯性的目標市場，且不包含文字。
    -   設計功能表和對話方塊，以留出空間供文字展開之用。 例如，英文字串在轉譯成德文或荷蘭文時，通常會擴展40%。
    -   請勿在 UI 元素或訊息中使用俚語或特定文化特性的參考。
    -   建立可在國際鍵盤上存取的快捷方式按鍵組合。 例如，請避免使用標點符號字元索引鍵做為快速鍵，因為它們不一定會出現在國際鍵盤上或由使用者輕鬆產生。 如需鍵盤配置的範例，請參閱[Windows 鍵盤佈局](https://msdn.microsoft.com/goglobal/bb964651.aspx)。
    -   考慮影響功能設計的當地法律，例如政府機構購買軟體以支援多種官方語言的需求。
    -   開發可支援貴組織國際 UI 標準和設計決策的協力廠商協定。
    -   在必須翻譯的使用者介面字串中使用一致的術語。

<!-- -->

-   建立與地區設定無關的程式碼。
    -   請勿將字串串連成表單句子。
    -   請勿在多個內容中使用指定的字串變數，例如在不同的訊息和提示中重複使用句子片段，因為這類字串可能不容易轉譯。
    -   使用批註的檔字串來提供翻譯人員的內容，並清楚地標示不應當地語系化的字串或字元。
    -   請勿使用採用特定語言的硬式編碼字元常數、數值常數、螢幕位置、檔案名或路徑。
    -   讓緩衝區夠大以容納轉譯的字串。
    -   允許資料的輸入格式會因地區設定而異，例如日期、時間和貨幣。
    -   使用適用于特定地區設定的紙張大小、信封大小和其他預設值。
    -   確定每個語言版本都可以讀取其他版本所建立的檔。
    -   如有必要，請提供地區設定特定硬體的支援。
    -   將不會套用至國際市場的功能設定為可輕鬆停用的實作為選項。

<!-- -->

-   建立利用 Microsoft Windows 所提供之國際功能的程式碼。
    -   使用系統所攜帶的國際資訊 (國語言支援) 。
    -   使用系統函數進行排序、案例轉換和字串對應。
    -   使用一般文字版面配置函數。
    -   在主控台中回應國際設定的變更。
    -   處理 WM \_ INPUTLANGCHANGEREQUEST 訊息。
    -   在東亞版中支援輸入方法編輯器、垂直文字和分行規則。

<!-- -->

-   從一組來源檔案編譯器的所有國際版本。
    -   最小化或消除需要針對不同語言版本重新編譯程式碼的機制。
    -   在 Windows 資源檔中儲存可當地語系化的專案，例如字串和圖示。
    -   使用相同的檔案格式，將檔儲存在所有語言版本中。

<!-- -->

-   支援不同的字元集，而不只是拉丁字母1字碼頁，數位1252。
    -   程式支援網路環境，在此環境中，電腦會使用不同的預設字碼頁來執行作業系統。
    -   使用 GetCPInfoEx 來取出東亞字碼頁的前導位元組範圍。
    -   剖析東亞語言應用程式中的雙位元組字元，除非程式碼是以 Unicode 為基礎。
    -   支援 Unicode 或 Unicode 和本機字碼頁之間的轉換。
    -   請勿假設所有字元都是8位或16位。
    -   使用泛型資料類型和泛型函數原型。
    -   使用 [字型字元集] 屬性，它會呼叫 EnumFontFamiliesEx 和 ChooseFont 通用對話方塊函式。
    -   使用適用于地區設定的字型來顯示和列印文字。

<!-- -->

-   測試程式的國際功能。
    -   翻譯文字應符合原生喇叭的標準。
    -   對話方塊應適當調整大小，並在顯示不同的語言時，適當地將文字放在適當的字元上。
    -   針對所有翻譯語言，對話方塊、狀態列、工具列和功能表都應該放在畫面上，並在顯示的解析度設定不同的解析度時讀取 legibly。
    -   功能表和對話方塊加速器應該是唯一的。
    -   使用者應該能夠將非歐洲腳本的字元輸入檔、對話方塊和檔案名。
    -   使用者應該能夠順利剪下、貼上、儲存和列印非歐洲腳本的字元。
    -   針對不同的地區設定，排序和大小寫轉換應該是正確的。
    -   應用程式應該在 Windows 的當地語系化版本上正確運作。
