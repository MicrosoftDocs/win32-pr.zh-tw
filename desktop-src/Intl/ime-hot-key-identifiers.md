# <a name="ime-hot-key-identifiers"></a>輸入碼熱鍵識別碼

這些識別碼會搭配 [**ImmSimulateHotKey**](/windows/desktop/api/Imm/nf-imm-immsimulatehotkey) 函式使用。



| 識別碼                       | 意義                                                                                                                                                                                                 |
|-----------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 輸入法 \_ CHOTKEY \_ IME \_ NONIME \_ 切換 | 當語言為簡體中文時， (簡體中文) 在輸入法和非 IME 作業之間切換。                                                                                                  |
| IME \_ CHOTKEY \_ 圖形 \_ 切換       |  (簡體中文) 切換 IME 的圖形轉換模式。                                                                                                                                           |
| IME \_ CHOTKEY \_ 符號 \_ 切換      |  (簡體中文) 切換 IME 的符號轉換模式。 符號模式表示使用者可以藉由對應至鍵盤上的標點符號和符號來輸入中文標點符號和符號。 |
| IME \_ ITHOTKEY \_ RECONVERTSTRING    | **Windows Me/98、windows 2000、WINDOWS XP：** (的繁體中文) 觸發程式重組。                                                                                                                |
| 輸入法 \_ JHOTKEY \_ 關閉 \_ 開啟         |  (日文) 也開啟並關閉 IME。                                                                                                                                                          |
| IME \_ KHOTKEY \_ 英文             |  (韓文) 切換為英文。                                                                                                                                                                             |
| IME \_ KHOTKEY \_ 圖形 \_ 切換       |  (韓文) 切換 IME 的圖形轉換模式。                                                                                                                                                       |
| IME \_ KHOTKEY \_ HANJACONVERT        |  (韓文) 切換至 [漢字轉換]。                                                                                                                                                                    |
| 輸入法 \_ THOTKEY \_ IME \_ NONIME \_ 切換 | 當語言是繁體中文時， (繁體中文) 在輸入法和非 IME 作業之間切換。                                                                                                |
| IME \_ THOTKEY \_ 圖形 \_ 切換       |  (繁體中文) 切換 IME 的圖形轉換模式。                                                                                                                                          |
| IME \_ THOTKEY \_ 符號 \_ 切換      |  (繁體中文) 切換 IME 的符號轉換模式。                                                                                                                                         |
