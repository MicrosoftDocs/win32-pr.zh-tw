# <a name="locale_siso-constants"></a>地區設定 \_ SISO \* 常數

本主題定義 NLS 所使用的地區設定 \_ SISO \* 常數。



| 值                     | 意義                                                                                                                                                                                                                                                                                                                                                                              |
|---------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 地區設定 \_ SISO3166CTRYNAME  | **Windows Me/98，Windows NT 4.0：** 國家/地區名稱，根據 ISO 標準3166（例如美國的 "US"）。 這也會傳回數位，例如 "029" 代表加勒比海。 此字串允許的最大字元數為九個字元，包括結束的 null 字元。                                                                                        |
| 地區設定 \_ SISO3166CTRYNAME2 | **Windows Vista 和更新版本：** 三個字母的 ISO 區功能變數名稱稱 (ISO 3166 3-國家/地區) 的字母代碼，例如美國的 "USA"。 這也會傳回數位，例如 "029" 代表加勒比海。 此字串允許的最大字元數為九個字元，包括結束的 null 字元。                                                            |
| 地區設定 \_ SISO639LANGNAME   | **Windows Me/98，Windows NT 4.0：** 完全以 ISO 標準639值為基礎之語言的縮寫名稱（以小寫形式表示，例如 "en" 代表英文）。 針對沒有2個字母代碼的語言，這可以是3個字母的代碼，例如夏威夷的 "haw"。 此字串允許的最大字元數為九個字元，包括結束的 null 字元。 |
| 地區設定 \_ SISO639LANGNAME2  | **Windows Vista 和更新版本：** 三個字母的 ISO 語言名稱（小寫形式） (ISO 639-2 三個字母代碼（適用于 language) ，例如 "eng" 代表英文）。 此字串允許的最大字元數為九個字元，包括結束的 null 字元。                                                                                                                  |
