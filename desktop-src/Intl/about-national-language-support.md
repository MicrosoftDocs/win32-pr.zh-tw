# <a name="about-national-language-support"></a>關於國家語言支援

 (NLS) 功能的國家語言支援可讓應用程式：

-   設定使用者的地區設定
-   識別使用者工作的語言
-   抓取表示時間、日期和其他已針對指定語言及地區設定正確格式的資訊的字串

NLS 也包含鍵盤配置和特定語言字型的支援。 如需詳細資訊，請參閱 [輸入方法管理員](input-method-manager.md) 和 [國際字型和文字顯示](international-fonts-and-text-display.md)。

-   [排序](sorting.md)
