# <a name="about-unicode-and-character-sets"></a>關於 Unicode 和字元集

全球的字元型資料是使用 [Unicode](unicode.md) 和傳統字元集所開發。 因此，Windows 提供了字元集函式，可協助應用程式將以字元為基礎的資料從其原始字元集轉換成 Unicode 或其他傳統字元集。 這些字元集函式也可協助應用程式建立以字元為基礎的資料，以便在任何作業系統上傳送和使用，包括不支援 Unicode 的資料。

如需有關此總覽範圍之外之 Unicode 的詳細資訊，請參閱 [Unicode 標準](https://www.unicode.org/standard/standard.html)。
