# <a name="locale_smonthname-constants"></a>地區設定 \_ SMONTHNAME \* 常數

本主題定義 NLS 所使用的地區設定 \_ SMONTHNAME \* 常數。




| 值 | 意義 | 
|-------|---------|
| LOCALE_SMONTHNAME1 | 1月的原生完整名稱。 此字串允許的最大字元數為80，包括結束的 null 字元。<blockquote>[!Note]<br />使用 LOCALE_SMONTHNAME * 常數來呼叫 <a href="/windows/desktop/api/Winnls/nf-winnls-getlocaleinfoa"><strong>GetLocaleInfo</strong></a> 或 <a href="/windows/desktop/api/Winnls/nf-winnls-getlocaleinfoex"><strong>GetLocaleInfoEx</strong></a> 函式會傳回月份名稱的獨立或 nominative 形式。 若要取得 month 格式的月份名稱，應用程式會使用 ddMMMM 的日期圖片來呼叫 <a href="/windows/desktop/api/datetimeapi/nf-datetimeapi-getdateformata">GetDateFormat</a> 或 <a href="/windows/desktop/api/datetimeapi/nf-datetimeapi-getdateformatex">GetDateFormatEx</a> ，並從抓取的字串開頭移除兩個數字。</blockquote><br /> | 
| LOCALE_SMONTHNAME2 | 二月的原生完整名稱。 此字串允許的最大字元數為80，包括結束的 null 字元。 請參閱 LOCALE_SMONTHNAME1 的注意事項。 | 
| LOCALE_SMONTHNAME3 | 三月的原生完整名稱。 此字串允許的最大字元數為80，包括結束的 null 字元。 請參閱 LOCALE_SMONTHNAME1 的注意事項。 | 
| LOCALE_SMONTHNAME4 | 四月的原生完整名稱。 此字串允許的最大字元數為80，包括結束的 null 字元。 請參閱 LOCALE_SMONTHNAME1 的注意事項。 | 
| LOCALE_SMONTHNAME5 | 可能的原生完整名稱。 此字串允許的最大字元數為80，包括結束的 null 字元。 請參閱 LOCALE_SMONTHNAME1 的注意事項。 | 
| LOCALE_SMONTHNAME6 | 六月的原生完整名稱。 此字串允許的最大字元數為80，包括結束的 null 字元。 請參閱 LOCALE_SMONTHNAME1 的注意事項。 | 
| LOCALE_SMONTHNAME7 | 七月的原生完整名稱。 此字串允許的最大字元數為80，包括結束的 null 字元。 請參閱 LOCALE_SMONTHNAME1 的注意事項。 | 
| LOCALE_SMONTHNAME8 | 八月的原生完整名稱。 此字串允許的最大字元數為80，包括結束的 null 字元。 請參閱 LOCALE_SMONTHNAME1 的注意事項。 | 
| LOCALE_SMONTHNAME9 | 9月的原生完整名稱。 此字串允許的最大字元數為80，包括結束的 null 字元。 請參閱 LOCALE_SMONTHNAME1 的注意事項。 | 
| LOCALE_SMONTHNAME10 | 十月的原生完整名稱。 此字串允許的最大字元數為80，包括結束的 null 字元。 請參閱 LOCALE_SMONTHNAME1 的注意事項。 | 
| LOCALE_SMONTHNAME11 | 11月的原生完整名稱。 此字串允許的最大字元數為80，包括結束的 null 字元。 請參閱 LOCALE_SMONTHNAME1 的注意事項。 | 
| LOCALE_SMONTHNAME12 | 12月的原生完整名稱。 此字串允許的最大字元數為80，包括結束的 null 字元。 請參閱 LOCALE_SMONTHNAME1 的注意事項。 | 
| LOCALE_SMONTHNAME13 | 第13個月的原生名稱（如果有的話）。 此字串允許的最大字元數為80，包括結束的 null 字元。 請參閱 LOCALE_SMONTHNAME1 的注意事項。 |
