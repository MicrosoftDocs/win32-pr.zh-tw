# <a name="locale-information-constants"></a>地區設定資訊常數

此區段會定義 NLS 用來指定特定地區設定資訊部分的常數。 所有名稱都以 "LOCALE \_ " 開頭。 代表整數值的常數名稱是以 "LOCALE \_ I" 開頭。 這些常數的所有值都是以 null 結束的 Unicode 字串。 最大字串長度包含終止的 null 字元。 如果沒有指定常數的最大長度，其值的長度可能會有所不同。

## <a name="combining-constants"></a>組合常數

下列常數可以與使用 binary 或運算子的其他常數合併。 其餘的常數是互斥的，因此無法彼此結合。

-   地區設定 \_ NOUSEROVERRIDE
-   地區設定傳回 \_ \_ 編號
-   地區設定 \_ 使用 \_ CP \_ ACP

## <a name="closely-related-locale-information-constants"></a>緊密相關的地區設定資訊常數

先前列出的許多地區設定資訊常數都是密切相關的，因此變更一個會影響其他的值。 下表顯示這些常數之間的關聯性。



| 值               | 受影響的常數                                                                                                 |
|---------------------|--------------------------------------------------------------------------------------------------------------------|
| 地區設定 \_ ICURRENCY   | 地區設定 \_ IPOSSEPBYSPACE，地區設定 \_ IPOSSYMPRECEDES                                                                    |
| 地區設定 \_ INEGCURR    | LOCALE \_ SPOSITIVESIGN、LOCALE \_ INEGSEPBYSPACE、LOCALE \_ INEGSYMPRECEDES、LOCALE \_ INEGSIGNPOSN、LOCALE \_ IPOSSIGNPOSN |
| 地區設定 \_ SSHORTDATE  | LOCALE \_ SDATE、LOCALE \_ IDATE、LOCALE \_ ICENTURY、LOCALE \_ IDAYLZERO、LOCALE \_ IMONLZERO                               |
| 地區設定 \_ SLONGDATE   | 地區設定 \_ ILDATE                                                                                                     |
| 地區設定 \_ STIMEFORMAT | LOCALE \_ STIME、LOCALE \_ ITIME、LOCALE \_ ITLZERO、LOCALE \_ ITIMEMARKPOSN                                               |
