# <a name="locale_sdecimal"></a>地區設定 \_ SDECIMAL

字元 (s) 用於小數分隔符號，例如 "3.14" 中的 "." 或 "3，14" 中的 "，"。 此字串允許的最大字元數為四個字元，包括結束的 null 字元。
