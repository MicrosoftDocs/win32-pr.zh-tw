# <a name="locale_use_cp_acp"></a>地區設定 \_ 使用 \_ CP \_ ACP

**Windows Me/98，Windows 2000：** 系統預設 Windows ANSI 字碼頁 (ACP) ，而不是用於字串轉譯的地區設定字碼頁。 如需 ANSI 和其他字碼頁的清單，請參閱 [字碼頁識別碼](code-page-identifiers.md) 。
