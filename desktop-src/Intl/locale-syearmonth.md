# <a name="locale_syearmonth"></a>地區設定 \_ SYEARMONTH

**Windows Me/98、windows 2000：** 地區設定的年月格式字串。 此字串允許的最大字元數為80，包括結束的 null 字元。 這個字串會針對只包含年份和月份的日期字串顯示適當的格式，並使用 [ [日]、[月]、[年] 和 [紀元格式圖片](day--month--year--and-era-format-pictures.md)] 中定義的格式圖片。
