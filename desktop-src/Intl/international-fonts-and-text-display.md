# <a name="international-fonts-and-text-display"></a>國際字型和文字顯示

國際字型和文字顯示可在 Windows 平臺上提供國際字型、國際文字和精細印刷樣式的平臺支援。

國際字型和文字是以 Windows 的基本印刷樣式、文字和先進閱讀技術為基礎，可為 Windows 平臺的全球使用者提供字型和文字。 此技術可增強端對端字型和文字顯示案例。

包含下列技術：

-   [國際字型管理](about-international-fonts-and-text.md)。 介紹國際字型的基本功能。
-   [使用 Ms Shell dlg 和 Ms Shell dlg 2](using-ms-shell-dlg-and-ms-shell-dlg-2.md)。 在不同的 Windows 版本上，您會瞭解 MS shell Dlg 和 MS Shell Dlg 2 字型，以及 MS Shell Dlg 和 MS Shell Dlg 2 的字型對應。
-   [國際字型列舉和選取](using-international-fonts-and-text.md)。 說明列舉國際字型的可能方式，讓使用者選取他們想要的字型。
-   [國際文字顯示](creating-your-own-format-selection-user-interface.md)。 說明顯示國際文字的可能方式。
-   [Uniscribe](uniscribe.md)。 允許高度的樣式控制和處理複雜的腳本。
