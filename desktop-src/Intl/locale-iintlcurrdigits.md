# <a name="locale_iintlcurrdigits"></a>地區設定 \_ IINTLCURRDIGITS

\[\_不支援地區設定 IINTLCURRDIGITS，未來可能會變更或無法使用。 請改為使用 [地區設定 \_ ICURRDIGITS](locale-icurrdigits.md)。\]

國際貨幣格式的小數位數。 此字串允許的最大字元數為三個字元，包括結束的 null 字元。 例如，美國貨幣的國際貨幣格式的小數位數為2，如 $345.25。
