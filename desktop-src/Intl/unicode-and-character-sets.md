# <a name="unicode-and-character-sets"></a>Unicode 和字元集

Microsoft Windows 透過[Unicode](unicode.md)和傳統字元集，為國際 marketplace 的許多不同書寫語言提供支援。

Unicode 是全球字元編碼標準，可提供唯一的數位來代表新式運算中使用的每個字元，包括用於發佈的技術符號和特殊字元。 Unicode 是新式標準（例如 XML 和 ECMAScript） (JavaScript) 的必要項，而且是執行 ISO/IEC 10646 的官方機制。 許多作業系統、所有新式瀏覽器及許多其他產品都支援它。 新的 Windows 應用程式應該使用 Unicode 來避免不同字碼頁的不一致，並協助簡化當地語系化。

傳統字元集是先前的字元編碼標準，例如 Windows 使用8位程式碼值的[字碼頁](code-pages.md)或8位值的組合，以代表特定語言或地理區域中所使用的字元。
