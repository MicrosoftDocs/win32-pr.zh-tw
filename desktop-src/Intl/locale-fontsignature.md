# <a name="locale_fontsignature"></a>地區設定 \_ FONTSIGNATURE

**Windows Me/98/95，Windows NT 4.0：** 特定的位模式，可決定支援地區設定和字型內容所需的字元涵蓋範圍之間的關聯性。 請注意，地區設定 \_ FONTSIGNATURE 資料會採用與其他所有地區設定資訊不同的表單。 所有其他地區設定資訊都可以字串形式或數位表示。 地區設定 \_ FONTSIGNATURE 資料會在 [LOCALESIGNATURE](/windows/win32/api/wingdi/ns-wingdi-localesignature) 結構中取出。
