# <a name="locale_ipos-constants"></a>地區設定 \_ IPOS \* 常數

本主題定義 NLS 所使用的地區設定 \_ IPOS \* 常數。



| 值                   | 意義                                                                                                                                                                   |
|-------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 地區設定 \_ IPOSSEPBYSPACE  | 以正數貨幣值分隔貨幣符號。 如果貨幣符號是以正數的空格隔開，則此值為1，否則為0。        |
| 地區設定 \_ IPOSSIGNPOSN    | 格式化正數值的索引。 索引會使用與 [地區設定 \_ INEGSIGNPOSN](locale-ineg-constants.md)相同的值，不同之處在于它不會使用零索引。    |
| 地區設定 \_ IPOSSYMPRECEDES | 貨幣符號的位置，以正數貨幣值表示。 如果貨幣符號在正數的正上方，則此值為1，如果符號符合該數量，則為0。 |
