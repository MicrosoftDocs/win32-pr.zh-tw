# <a name="supporting-system-language-settings"></a>支援的系統語言設定

您的 MUI 應用程式設定應用程式語言喜好設定的慣用方式，是支援與目標作業系統相同的一組使用者介面語言。 應用程式只需依賴作業系統，就能選取正確的使用者介面語言來顯示。 在 [ [MUI：系統設定應用程式範例](mui-system-settings-application-sample.md)] 中提供相關聯的範例。
