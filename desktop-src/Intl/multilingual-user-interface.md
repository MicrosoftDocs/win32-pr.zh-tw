# <a name="multilingual-user-interface"></a>多語系使用者介面

多語系消費者介面 (MUI) 可為全球化應用程式提供使用者介面的當地語系化。 MUI 也支援建立任意數量的使用者介面語言的資源。

本檔說明 MUI，並告訴您如何將 MUI 功能新增至您的完整全球化應用程式。 但是，它不會嘗試描述 Windows 上全球化應用程式開發的所有詳細資料。 如需此類型的詳細資訊，請參閱 Microsoft [Go 全球開發人員中心](https://msdn.microsoft.com/goglobal)。

MUI 檔包含下列各節：

-   [關於多語系消費者介面](about-multilingual-user-interface.md)
-   [使用多語系消費者介面](using-multilingual-user-interface.md)
-   [多語系消費者介面參考](multilingual-user-interface-reference.md)
