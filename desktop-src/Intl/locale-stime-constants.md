# <a name="locale_stime-constants"></a>地區設定 \_ STIME \* 常數

本主題定義 NLS 所使用的地區設定 \_ STIME \* 常數。



| 值               | 意義                                                                                                                                                                                                                                                                                                                                                                                   |
|---------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 地區設定 \_ STIME       | 時間分隔符號) 字元 (s。 此字串允許的最大字元數為四個字元，包括結束的 null 字元。 <br/> **Windows Vista 和更新版本：** 這個常數已被取代。 請改用 LOCALE \_ STIMEFORMAT。 自訂地區設定可能沒有單一的統一分隔字元。 例如，"03： 56 ' 23" 之類的格式是有效的。<br/> |
| 地區設定 \_ STIMEFORMAT | 地區設定的時間格式字串。 此字串允許的最大字元數為80，包括結束的 null 字元。 此字串可以包含 [小時、分鐘和第二種格式圖片](hour--minute--and-second-format-pictures.md)的組合。                                                                                                      |
