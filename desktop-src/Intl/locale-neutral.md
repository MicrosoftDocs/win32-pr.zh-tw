# <a name="locale_neutral"></a>地區設定 \_ 中性

中性地區設定。 呼叫 NLS Api 時，通常不會使用這個常數。 請改為使用 [地區設定 \_ 系統 \_ 預設值](locale-system-default.md) 或 [地區設定 \_ 使用者 \_ 預設值](locale-user-default.md)。 地區設定中性的值 \_ 為0x0000。
