# <a name="locale_idefault-constants"></a>地區設定 \_ IDEFAULT \* 常數

本主題定義 NLS 所使用的地區設定 \_ IDEFAULT \* 常數。



| 值                          | 意義                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
|--------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 地區設定 \_ IDEFAULTANSICODEPAGE   | 針對不支援 Unicode 的應用程式，其地區設定所使用的 ANSI 字碼頁。 此字串允許的最大字元數為六個字元，包括結束的 null 字元。 如果沒有可用的 ANSI 字碼頁，則只有 Unicode 可以用於地區設定。 在此情況下，值為 CP \_ ACP (0) 。 這類地區設定無法設定為系統地區設定。 不支援 Unicode 的應用程式無法正確搭配標示為「僅 Unicode」的地區設定。 如需 ANSI 和其他字碼頁的清單，請參閱 [字碼頁識別碼](code-page-identifiers.md)。 |
| 地區設定 \_ IDEFAULTCODEPAGE       | 原始設備製造商 (與國家/地區相關聯的 OEM) 字碼頁。 OEM 字碼頁是用來從以 ms-dos 為基礎的文字模式應用程式轉換。 如果地區設定未使用 OEM 字碼頁，則值為 CP \_ OEMCP (1) 。 此字串允許的最大字元數為六個字元，包括結束的 null 字元。 如需 OEM 和其他字碼頁的清單，請參閱 [字碼頁識別碼](code-page-identifiers.md)。                                                                                                               |
| 地區設定 \_ IDEFAULTCOUNTRY        | 已過時。 請勿使用。 提供這個值的目的是要使用預設值來完成部分指定的地區設定。 部分指定的地區設定現在已被取代。                                                                                                                                                                                                                                                                                                                                                                                               |
| 地區設定 \_ IDEFAULTEBCDICCODEPAGE | **Windows 2000：** 預設擴充二進位編碼的十進位交換程式碼 (EBCDIC) 與地區設定相關聯的字碼頁。 此字串允許的最大字元數為六個字元，包括結束的 null 字元。 如需 EBCDIC 和其他字碼頁的清單，請參閱 [字碼頁識別碼](code-page-identifiers.md)。                                                                                                                                                                                                                                     |
| 地區設定 \_ IDEFAULTLANGUAGE       | 已過時。 請勿使用。 提供這個值的目的是要使用預設值來完成部分指定的地區設定。 部分指定的地區設定現在已被取代。                                                                                                                                                                                                                                                                                                                                                                                               |
| 地區設定 \_ IDEFAULTMACCODEPAGE    | 與地區設定相關聯的預設 Macintosh 字碼頁。 此字串允許的最大字元數為六個字元，包括結束的 null 字元。 如果地區設定未使用 Macintosh 字碼頁，則值為 CP \_ MACCP (2) 。 如需 Macintosh (MAC) 和其他字碼頁的清單，請參閱 [字碼頁識別碼](code-page-identifiers.md)。                                                                                                                                                                                                              |
