# <a name="ime-composition-string-values"></a>IME 組合字元串值

這些值會搭配 [**ImmGetCompositionString**](/windows/desktop/api/Imm/nf-imm-immgetcompositionstringa) 和 [**WM \_ IME \_ 組合**](wm-ime-composition.md)使用。



| 值                 | 描述                                                                                |
|-----------------------|--------------------------------------------------------------------------------------------|
| GC \_ COMPATTR         | 取得或更新組合字元串的屬性。                                |
| GC \_ COMPCLAUSE       | 取得或更新組合字元串的子句資訊。                           |
| GC \_ COMPREADATTR     | 取得或更新目前組合之讀取字串的屬性。        |
| GC \_ COMPREADCLAUSE   | 取得或更新撰寫字串之讀取字串的子句資訊。 |
| GC \_ COMPREADSTR      | 取得或更新目前組合的讀取字串。                          |
| GC \_ COMPSTR          | 取出或更新目前的組合字元串。                                         |
| GC \_ CURSORPOS        | 取得或更新組合字元串中的游標位置。                              |
| GC \_ DELTASTART       | 取得或更新組合字元串中任何變更的開始位置。             |
| GC \_ RESULTCLAUSE     | 取得或更新結果字串的子句資訊。                                |
| GC \_ RESULTREADCLAUSE | 取得或更新讀取字串的子句資訊。                               |
| GC \_ RESULTREADSTR    | 取出或更新讀取字串。                                                     |
| GC \_ RESULTSTR        | 取得或更新組合結果的字串。                                   |
