# <a name="globalization-services"></a>全球化服務

全球化服務公開了語言和區域的核心全球化支援，可讓您建立適用于各種國際客戶的全球化應用程式。 這項技術提供全球化基礎結構，以支援一致且符合語言和文化特性的 Windows 使用者體驗。 全球化服務包括下列技術：

-   [擴充語言服務](extended-linguistic-services.md)。 針對 Windows 開發人員提供一系列以文字為基礎的服務， (包括語言偵測、音譯和其他核心語言功能) 以及應用程式用來輕鬆取用服務的平臺和介面，以及可讓開發人員加入其專屬服務的介面。
-   [Unicode (ICU) 的國際元件 ](international-components-for-unicode--icu-.md)。 提供一組成熟、廣泛使用的開放原始碼全球化 Api。 ICU 利用 Unicode 的廣泛通用地區設定資料存放庫 (CLDR) 作為資料庫，提供軟體應用程式的全球化支援。 ICU 具有廣泛的可攜性，讓應用程式在所有平臺上都有相同的結果。
-   [國家語言支援](national-language-support.md)。 提供國際資料和 Api，以支援日期、時間和行事曆格式、鍵盤配置、排序行為、文字啟用的層面，以及其他核心地區設定資訊。
-   [拼寫檢查 API](spell-checker-api.md)。 提供一個 API，讓開發人員使用拼寫檢查功能來檢查文字、取得建議，以及維護設定和使用者字典。 開發人員也可以建立將拼寫檢查架構整合成提供者的拼寫檢查。
