# <a name="locale_idaylzero"></a>地區設定 \_ IDAYLZERO

日期欄位中前置零的規範（僅限簡短日期）。 此字串允許的最大字元數為二，包括結束的 null 字元。 最好讓您的應用程式使用地區設定 [ \_ SSHORTDATE](locale-sshortdate.md) 常數，而不是地區設定 \_ IDAYLZERO。



| 值 | 意義                   |
|-------|---------------------------|
| 0     | 天沒有前置零 |
| 1     | 天數的前置零    |
