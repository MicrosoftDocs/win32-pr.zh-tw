# <a name="locale_windows"></a>地區設定 \_ 視窗

**Windows Vista 和更新版本：** 列舉作業系統隨附的所有地區設定，包括取代地區設定，但不包括替代的排序。 如需詳細資訊，請參閱 [自訂地區](custom-locales.md)設定。
