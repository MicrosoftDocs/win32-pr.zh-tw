# <a name="hour-minute-and-second-format-pictures"></a>Hour、Minute 和 Second 格式的圖片

本主題說明用來撰寫時間字串格式圖片的字串格式類型。 每個格式圖片都包含每個格式類型的一個字串組合。

> [!Note]  
> 在格式類型中，字母 "m"、"s" 和 "t" 必須是小寫，而且字母 "h" 必須是小寫，以表示12小時制或大寫 ( "H" ) 表示24小時制。

單引號可以用來標示應該完全依照指定顯示的字元。 如果應用程式必須顯示單引號，則應該在資料列中放入兩個單引號。 例如，' abc ' ' bar ' 會顯示為 "abc'bar"。

下表定義用來表示小時的格式類型。

| String | 意義                                                             |
|--------|---------------------------------------------------------------------|
| h      | 單一位數的小時內不含前置零的小時 (12 小時制) 。 |
| hh     | 以前置零表示單一位數小時的小時 (12 小時制) 。    |
| H      | 單一位數的小時內不含前置零的小時 (24 小時制) 。 |
| HH     | 以前置零表示單一位數小時的小時 (24 小時制) 。    |

下表定義用來代表分鐘的格式類型。

| String | 意義                                                 |
|--------|---------------------------------------------------------|
| m      | 單一位數分鐘沒有前置零的分鐘數。 |
| mm     | 在單一位數分鐘內使用前置零的分鐘數。    |

下表定義用來代表秒鐘的格式類型。

| String | 意義                                                 |
|--------|---------------------------------------------------------|
| s      | 單一位數秒沒有前置零的秒數。 |
| ss     | 以前置零表示的秒數（以秒為單位）。    |

下表定義用來表示時間標記的格式類型。

| String | 意義|
| ---    | ---    |
| t      | 一個字元的時間標記字串。<br />**注意：** 不建議將此格式用於某些語言，例如日文 (日本) 。 使用此格式，應用程式一律會使用時間標記字串中的第一個字元（由 [LOCALE_S1159](locale-s1159.md) (AM) 和 [LOCALE_S2359](locale-s2359.md) (PM) 。 因此，應用程式可以使用 AM 和 PM 所用的相同字串來建立不正確的格式。|
| tt     | 多字元的時間標記字串。 |
