# <a name="locale_idate"></a>地區設定 \_ IDATE

簡短日期格式排序規範。 規範必須是下列值之一。 不允許使用者指定的值。 您的應用程式最好使用地區設定 [ \_ SSHORTDATE](locale-sshortdate.md) ，而不是地區設定 \_ IDATE。



| 值 | 意義        |
|-------|----------------|
| 0     | 月-日-年 |
| 1     | 日-月-年 |
| 2     | 年-月-日 |
