# <a name="locale_imonlzero"></a>地區設定 \_ IMONLZERO

僅限簡短日期的月份欄位中前置零的規範。 最好讓您的應用程式使用地區設定 [ \_ SSHORTDATE](locale-sshortdate.md) 常數，而不是地區設定 \_ IMONLZERO。



| 值 | 意義                     |
|-------|-----------------------------|
| 0     | 月份沒有前置零 |
| 1     | 月的前置零    |
