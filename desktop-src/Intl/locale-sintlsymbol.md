# <a name="locale_sintlsymbol"></a>地區設定 \_ SINTLSYMBOL

ISO 4217 中指定的國際貨幣符號的三個字元，後面接著用來分隔此字串與數量的字元。 此字串允許的最大字元數為九個字元，包括結束的 null 字元。
