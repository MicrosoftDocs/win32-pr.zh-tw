# <a name="locale_spositivesign"></a>地區設定 \_ SPOSITIVESIGN

地區設定之正號的當地語系化字串值。 例如， **地區設定 \_ SPOSITIVESIGN** 視為 "+"，適用于英文 (美國) 地區設定。 請注意，此字串的資料存在並不表示在格式化數位時，一定會使用正負號。 如果此資料為空白/空的，則會假設使用 "+"。 此字串允許的最大字元數為五個字元，包括結束的 null 字元。
