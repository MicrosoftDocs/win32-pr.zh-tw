# <a name="locale_igeoid"></a>地區設定 \_ IGEOID

可唯一識別 [地理位置](table-of-geographical-locations.md)的32位帶正負號的數位。 應用程式會使用這個常數，為客戶提供地區設定特定的服務。 例如，它可以用來作為存取資料庫記錄的索引鍵，其中包含與國家/地區相關的特定資訊。
