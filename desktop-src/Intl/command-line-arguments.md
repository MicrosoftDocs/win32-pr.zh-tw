# <a name="command-line-arguments"></a>命令列的引數

應用程式可以使用 Win32 [GetCommandLine](/windows/win32/api/processenv/nf-processenv-getcommandlinea) 函數來取出 Unicode 命令列引數。 應用程式應該呼叫 Win32 函數作為 Unicode 函數。
