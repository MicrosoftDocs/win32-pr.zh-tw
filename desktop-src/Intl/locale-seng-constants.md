# <a name="locale_seng-constants"></a>地區設定 \_ SENG \* 常數

本主題定義 NLS 所使用的地區設定 \_ SENG \* 常數。



| 值                | 意義                                                                                                                                                                   |
|----------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 地區設定 \_ SENGCOUNTRY  | 適用于 Windows 7 和更新版本的已淘汰。 國家/地區的完整英文名稱。 請參閱 [地區設定 \_ SENGLISHCOUNTRYNAME](locale-senglish-constants.md)。                            |
| 地區設定 \_ SENGCURRNAME | **Windows Me/98、windows 2000：** 與地區設定相關聯之貨幣的完整英文名稱。 此字串允許的字元數沒有限制。 |
| 地區設定 \_ SENGLANGUAGE | 適用于 Windows 7 和更新版本的已淘汰。 ISO 標準639中語言的完整英文名稱。 請參閱 [地區設定 \_ SENGLISHLANGUAGENAME](locale-senglish-constants.md)。           |
