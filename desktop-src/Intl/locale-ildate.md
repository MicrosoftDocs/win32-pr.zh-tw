# <a name="locale_ildate"></a>地區設定 \_ ILDATE

完整日期格式排序規範。 最好讓您的應用程式使用地區設定 [ \_ SLONGDATE](locale-slongdate.md) 常數，而不是地區設定 \_ ILDATE。



| 值 | 意義        |
|-------|----------------|
| 0     | 月-日-年 |
| 1     | 日-月-年 |
| 2     | 年-月-日 |
