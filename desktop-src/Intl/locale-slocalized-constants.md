# <a name="locale_slocalized-constants"></a>地區設定 \_ SLOCALIZED \* 常數

本主題定義 NLS 所使用的地區設定 \_ SLOCALIZED \* 常數。



| 值                          | 意義                                                                                                                                                                                                                                                                                                                                     |
|--------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 地區設定 \_ SLOCALIZEDCOUNTRYNAME  | **Windows 7 和更新版本：** 國家/地區的完整當地語系化名稱，例如，德國的 Deutschland。 此字串允許的最大字元數為80，包括結束的 null 字元。 因為此名稱是以產品的當地語系化為基礎，所以會針對每個當地語系化版本進行變更。 <br/>           |
| 地區設定 \_ SLOCALIZEDDISPLAYNAME  | **Windows 7 和更新版本：** 使用者介面語言地區設定的完整當地語系化名稱，例如，Deutsch (Deutschland) 德文 (德國) 「此字串允許的字元數沒有限制。 因為此名稱是以產品的當地語系化為基礎，所以會針對每個當地語系化版本進行變更。 <br/> |
| 地區設定 \_ SLOCALIZEDLANGUAGENAME | **Windows Vista：** 當地語系化顯示名稱中包含的使用者介面語言完整當地語系化的主要名稱，例如代表德文的 Deutsch。 因為此名稱是以產品的當地語系化為基礎，所以會針對每個當地語系化版本進行變更。 <br/>                                                               |
