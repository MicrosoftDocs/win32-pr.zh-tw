# <a name="locale_smon-constants"></a>地區設定 \_ SMON \* 常數

本主題定義 \_ \* NLS 用來表示貨幣值的地區設定 SMON 常數。



| 值                   | 意義                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|-------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 地區設定 \_ SMONDECIMALSEP  | 字元 (s) 用來作為貨幣小數分隔符號。 此字串允許的最大字元數為四個字元，包括結束的 null 字元。 例如，如果貨幣金額顯示為 "$3.40"，就像「三元和40美分」會顯示在美國，而貨幣小數分隔符號則是 "."。                                                                                                                                             |
| 地區設定 \_ SMONGROUPING    | 小數點左邊每個貨幣位數群組的大小。 此字串允許的最大字元數為十個字元，包括結束的 null 字元。 每個群組都需要明確的大小，且大小會以分號分隔。 如果最後一個值為0，則會重複上述值。 例如，若要將數以千計分組，請指定 3; 0。 印度文語言會將前一千分組，然後分組為數百。 例如12、34、56789是以3表示，2; 0。 |
| 地區設定 \_ SMONTHOUSANDSEP | 字元 (s) 用來作為小數點左邊數位群組之間的貨幣分隔符號。 此字串允許的最大字元數為四個字元，包括結束的 null 字元。 一般而言，這些群組代表上千個。 不過，根據針對地區設定 SMONGROUPING 所指定的值 \_ ，它們可以代表其他內容。                                                                                                                                 |
