# <a name="locale_ssort_constants"></a>地區設定 \_ SSORT \_ 常數

本主題定義 NLS 所使用的地區設定 \_ SSORT \* 常數。



| 值               | 意義                                                                                                                                                                                                           |
|---------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 地區設定 \_ SSORTLOCALE | **Windows 7 和更新版本：** 要用於排序或大小寫行為的地區設定名稱。                                                                                                                                |
| 地區設定 \_ SSORTNAME   | **Windows Me/98、windows 2000：** 指定地區設定識別碼之排序的完整當地語系化名稱，取決於 shell 的語言。 這個常數用來判斷大小寫和排序行為。 |
