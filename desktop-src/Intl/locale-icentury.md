# <a name="locale_icentury"></a>地區設定 \_ ICENTURY

僅限簡短日期的2位數或4位數的世紀。 世紀可以具有下列其中一個值。 您的應用程式最好使用地區設定 [ \_ SSHORTDATE](locale-sshortdate.md) ，而不是地區設定 \_ ICENTURY。



| 值 | 意義                     |
|-------|-----------------------------|
| 0     | 縮寫的2位數世紀 |
| 1     | 完整4位數的世紀        |
