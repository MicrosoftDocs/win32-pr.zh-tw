# <a name="uniscribe"></a>Uniscribe

Uniscribe 是一組 Api，可讓您以高度的樣式進行微調，以及處理複雜的腳本。 複雜的腳本和簡單的腳本（具有精細印刷效果）都需要進行特殊處理以顯示和編輯，因為字元 ( 「字元」 ) 不會以簡單的方式來配置。 針對複雜的腳本，管理字元成形和位置的規則是以 [Unicode 標準](https://www.unicode.org/)來指定和編目。

本主題討論精細印刷樣式和處理複雜字集的各個層面，如下所示：

-   [使用 Uniscribe](using-uniscribe.md)
-   [Uniscribe 參考](uniscribe-reference.md)
