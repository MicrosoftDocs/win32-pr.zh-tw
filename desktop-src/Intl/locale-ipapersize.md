# <a name="locale_ipapersize"></a>地區設定 \_ IPAPERSIZE

**Windows 2000：** 與地區設定相關聯的預設紙張大小。 大小通常具有下列其中一個值，不過它可以設定為任何由多工緩衝處理器理解的已定義紙張大小。



| 值 | 意義   |
|-------|-----------|
| 1     | 美國信件 |
| 5     | 美國法律  |
| 8     | A3        |
| 9     | A4        |
