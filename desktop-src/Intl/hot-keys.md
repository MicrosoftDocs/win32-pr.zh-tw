# <a name="hot-keys"></a>快速鍵

快速鍵為使用者提供一種方式，可快速變更 IME 的輸入模式或切換至另一個 IME。 雖然應用程式無法將快速鍵新增至作業系統，但它們可以使用 [**ImmSimulateHotKey**](/windows/desktop/api/Imm/nf-imm-immsimulatehotkey) 函式來初始化與快速鍵相同的動作。

HexToUnicode IME 也允許在十六進位和 Unicode 字元之間進行轉換。 如需詳細資訊，請參閱 [HEXTOUNICODE IME](hextounicode-ime.md)。
