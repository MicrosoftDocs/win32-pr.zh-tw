# <a name="locale_sdate"></a>地區設定 \_ SDATE

日期分隔符號的字元 (s) 。 此字串允許的最大字元數為四個字元，包括結束的 null 字元。

**Windows Vista 和更新版本：** 這個常數已被取代。 請改用 [LOCALE \_ SSHORTDATE](locale-sshortdate.md) 。 自訂地區設定可能沒有單一的統一分隔字元。 例如，"12/31，2006" 之類的格式是有效的。
