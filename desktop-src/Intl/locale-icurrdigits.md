# <a name="locale_icurrdigits"></a>地區設定 \_ ICURRDIGITS

本地貨幣格式的小數位數。 此字串允許的最大字元數為二，包括值0-9 和終止的 null 字元。 例如，200.00 的值為2，因為有兩個小數位數，而200的值為0，因為沒有顯示任何小數位數。
