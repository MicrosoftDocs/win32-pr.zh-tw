# <a name="locale_name-constants"></a>地區設定 \_ 名稱 \* 常數

本主題定義 NLS 所使用的地區設定 \_ 名稱 \* 常數。



<table>
<colgroup>
<col style="width: 50%" />
<col style="width: 50%" />
</colgroup>
<thead>
<tr class="header">
<th>值</th>
<th>意義</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td>LOCALE_NAME_INVARIANT</td>
<td>提供穩定地區設定和行事歷數據的不變地區設定名稱。</td>
</tr>
<tr class="even">
<td>LOCALE_NAME_MAX_LENGTH</td>
<td>地區設定名稱的最大長度。 此字串允許的最大字元數為85，包括結束的 null 字元。
<blockquote>
[!Note]<br />
您的應用程式必須使用常數來取得最大的地區設定名稱長度，而不是對值85進行硬式編碼 &quot; &quot; 。
</blockquote>
<br/></td>
</tr>
<tr class="odd">
<td>LOCALE_NAME_SYSTEM_DEFAULT</td>
<td>目前作業系統地區設定的名稱。</td>
</tr>
<tr class="even">
<td>LOCALE_NAME_USER_DEFAULT</td>
<td>目前使用者地區設定的名稱，符合主控台的 [地區及語言選項] 部分中的喜好設定。 此地區設定可能與目前使用者介面語言的地區設定不同。</td>
</tr>
</tbody>
</table>
