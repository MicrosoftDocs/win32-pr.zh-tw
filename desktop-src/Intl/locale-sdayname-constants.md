# <a name="locale_sdayname-constants"></a>地區設定 \_ SDAYNAME \* 常數

本主題定義 NLS 所使用的地區設定 \_ SDAYNAME \* 常數。



| 值             | 意義                                                                                                                                 |
|-------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| 地區設定 \_ SDAYNAME1 | 星期一的原生完整名稱。 此字串允許的最大字元數為80，包括結束的 null 字元。    |
| 地區設定 \_ SDAYNAME2 | 星期二的原生完整名稱。 此字串允許的最大字元數為80，包括結束的 null 字元。   |
| 地區設定 \_ SDAYNAME3 | 星期三的原生完整名稱。 此字串允許的最大字元數為80，包括結束的 null 字元。 |
| 地區設定 \_ SDAYNAME4 | 星期四的原生完整名稱。 此字串允許的最大字元數為80，包括結束的 null 字元。  |
| 地區設定 \_ SDAYNAME5 | 星期五的原生完整名稱。 此字串允許的最大字元數為80，包括結束的 null 字元。    |
| 地區設定 \_ SDAYNAME6 | 星期六的原生完整名稱。 此字串允許的最大字元數為80，包括結束的 null 字元。  |
| 地區設定 \_ SDAYNAME7 | 星期日的原生完整名稱。 此字串允許的最大字元數為80，包括結束的 null 字元。    |
