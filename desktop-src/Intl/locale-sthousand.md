# <a name="locale_sthousand"></a>地區設定 \_ STHOUSAND

用來分隔小數點左邊數位群組的字元。 此字串允許的最大字元數為四個字元，包括結束的 null 字元。 一般而言，這些群組代表數以千計。 不過，根據針對 [地區設定 \_ SGROUPING](locale-sgrouping.md)所指定的值，它們可以代表其他內容。
