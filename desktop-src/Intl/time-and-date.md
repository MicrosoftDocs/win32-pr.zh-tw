# <a name="time-and-date"></a>時間與日期

NLS 支援使用適用于目前地區設定的格式，來抓取任何時間或日期的字串。 在 Windows Vista 和更新版本上，NLS 也支援抓取任何地區設定的持續時間字串，包括自訂地區設定。 如需開發指示，請參閱 [取出時間和日期資訊](retrieving-time-and-date-information.md)。
