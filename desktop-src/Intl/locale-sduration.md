# <a name="locale_sduration"></a>地區設定 \_ SDURATION

**Windows Vista 和更新版本：** 由下表所列格式圖片組成的時間持續時間格式。 格式類似 [地區設定 \_ STIMEFORMAT](locale-stime-constants.md)的格式。 針對地區 \_ 設定 STIMEFORMAT，此格式也可以包含以單引號括住的任何字元字串。 格式可包含，例如 "h:mm： ss" 或 "d ' h'h '. fff '"。 相較于地區 \_ 設定 STIMEFORMAT，有一秒的小數部分會有其他格式的圖片。 因為這種格式是用於持續時間，而不是時間，所以不會指定12或24小時制的系統，也不會包含 AM/PM 指標。 例如，這個常數可能會用於顯示檔案時間的多媒體應用程式，或顯示完成時間的體育活動應用程式。



| 值 | 意義                                                                                                                                                                                                                             |
|-------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| h     | 單一位數小時沒有前置零的小時數                                                                                                                                                                                  |
| hh    | 單一位數小時的前置零小時                                                                                                                                                                                     |
| m     | 單一位數分鐘沒有前置零的分鐘數                                                                                                                                                                              |
| mm    | 單一位數分鐘的前置零分鐘                                                                                                                                                                                 |
| s     | 單一位數秒沒有前置零的秒數                                                                                                                                                                              |
| ss    | 以前置零表示的秒數（以秒為單位）                                                                                                                                                                                 |
| f     | 十分之一秒                                                                                                                                                                                                                  |
| ff    | 百分之一秒                                                                                                                                                                                                              |
| fff   | 秒數秒;字元 "f" 最多可有九次 (.fffffffff) ，雖然頻率計時器的支援限制為100毫微秒;如果有九個字元，最後兩個數字一律為零 |
