# <a name="locale_itlzero"></a>地區設定 \_ ITLZERO

前置零 in time 欄位的規範。 最好讓您的應用程式使用地區設定 [ \_ STIMEFORMAT](locale-stime-constants.md) 常數，而不是地區設定 \_ ITLZERO。



| 值 | 意義                    |
|-------|----------------------------|
| 0     | 小時沒有前置零 |
| 1     | 小時的前置零    |
