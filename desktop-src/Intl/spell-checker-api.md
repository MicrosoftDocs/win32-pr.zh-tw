# <a name="spell-checking-api"></a>拼寫檢查 API

拼寫檢查 API 可讓開發人員使用拼寫檢查功能來檢查文字、取得建議，以及維護設定和使用者字典。 開發人員也可以建立已整合至拼寫檢查架構作為提供者的拼寫檢查。

-   [關於拼寫檢查 API](about-the-spell-checker-api.md)
-   [拼寫檢查 API 參考](spell-checker-api-reference.md)
