# <a name="locale_inegativepercent"></a>地區設定 \_ INEGATIVEPERCENT

**Windows 7 和更新版本：** 地區設定的負百分比格式模式。 只能指定一個模式。 如果有一種以上的格式用於地區設定，請選擇慣用的選項。 例如，如果 "-9%" 針對 "負九%" 顯示負百分比，則這個常數的適當選擇是0。



| 值 | 格式                                                    |
|-------|-----------------------------------------------------------|
| 0     | 負號、數位、空格、百分比;例如，-\# % |
| 1     | 負號、數位、百分比;例如，-\#%         |
| 2     | 負號、百分比、數位;例如，-%\#         |
| 3     | 百分比、負號、數位;例如，%-\#         |
| 4     | 百分比、數位、負號;例如，%\#-         |
| 5     | 數位、負號、百分比;例如， \#-%         |
| 6     | Number、percent、負號;例如， \#%-         |
| 7     | 負號、百分比、空格、數位;例如，-% \# |
| 8     | 數位、空格、百分比、負號;例如， \# %- |
| 9     | 百分比、空格、數位、負號;例如，% \#- |
| 10    | 百分比、空格、負號、數位;例如，%-\# |
| 11    | Number、負號、空格、percent;例如， \# -% |
