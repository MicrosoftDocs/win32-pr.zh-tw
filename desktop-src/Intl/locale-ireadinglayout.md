# <a name="locale_ireadinglayout"></a>地區設定 \_ IREADINGLAYOUT

**Windows 7 和更新版本：** 文字的閱讀版面配置。 可能的值定義于下表中。



| 值 | 意義                                                                                                                                                             |
|-------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 0     | 從左至右讀取，如同英文 (美國) 地區設定。                                                                                                 |
| 1     | 從右至左讀取，如同阿拉伯文地區設定一樣。                                                                                                                     |
| 2     | 從上到下，從上往下讀取資料行（由右至左），或從左至右讀取水準資料列（如同日文 (日本) 地區設定）。 |
| 3     | 從上到下以垂直方式從上到下讀取資料行，如蒙古文 (蒙古) 地區設定。                                                  |
