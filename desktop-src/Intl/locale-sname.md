# <a name="locale_sname"></a>地區設定 \_ SNAME

**Windows Vista 和更新版本：** 地區設定名稱，這是用來唯一識別地區設定的多部分標記。 此字串允許的最大字元數為85，包括結束的 null 字元。 此標記是以 RFC 4646 的語言標記慣例為基礎。 [地區設定名稱](locale-names.md)中描述要使用的模式。
