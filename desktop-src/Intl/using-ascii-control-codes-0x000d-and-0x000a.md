# <a name="using-ascii-control-codes-0x000d-and-0x000a"></a>使用 ASCII 控制代碼0x000D 和0x000A

Unicode 標準不規定控制項程式碼的特定意義 0x000D (換行傳回) 和 0x000A (換行字元) 。 這些代碼不一定要用於組合。 如果個別使用，則這兩個程式碼只能代表本身或兩個程式碼。 應用程式必須一律決定這些程式碼所代表的意義。
