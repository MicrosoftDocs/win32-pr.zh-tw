# <a name="reading-or-copying-pixels"></a>讀取或複製圖元

您可以將畫面格緩衝區中的圖元讀取到記憶體中，以各種方式進行編碼，並使用 [**glReadPixels**](glreadpixels.md)將編碼的結果儲存在記憶體中。 此外，您可以使用 [**glCopyPixels**](glcopypixels.md)，將圖元值的矩形從畫面格緩衝區的一個區域複製到另一個區域。 函式 [**glReadBuffer**](glreadbuffer.md) 可控制讀取或複製圖元的色彩緩衝區。
