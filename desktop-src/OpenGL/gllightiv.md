# <a name="gllightiv-function"></a>glLightiv 函式

**GlLightiv** 函數會傳回燈光來源參數值。

## <a name="syntax"></a>語法


```C++
void WINAPI glLightiv(
         GLenum light,
         GLenum pname,
   const GLint  *params
);
```

## <a name="remarks"></a>備註

**GlLightiv** 函數會設定個別光源參數的值或值。 *Light* 參數會將燈光命名，而且是一個外型燈的符號名稱 \_ ，其中 0 = *i* < GL 的 \_ 最大 \_ 燈光。

*Pname* 參數會以符號名稱再次指定其中一個燈光來源參數。 *Params* 參數為單一值或陣列的指標，其中包含新的值。

您可以使用 [**glEnable**](glenable.md) 和 [**glDisable**](gldisable.md) 搭配引數 GL 光源來啟用和停用光源計算 \_ 。 當光源啟用時，已啟用的燈光來源會對光源計算造成影響。 您可以使用 **glEnable** 和 **glDisable** 搭配引數 GL 燈來啟用和停用燈光來源 *i* \_ 。**

這一律是 GL \_ 燈 = gl ** \_ LIGHT0 + *i* 的情況。
