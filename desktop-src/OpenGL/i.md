# <a name="i-opengl"></a>我 (OpenGL)

<dl> <dt>

<span id="opengl_image"></span><span id="OPENGL_IMAGE"></span>**圖像**
</dt> <dd>

在用戶端記憶體或畫面格緩衝區中的矩形陣列（以圖元為單位）。

</dd> <dt>

<span id="opengl_image_primitive"></span><span id="OPENGL_IMAGE_PRIMITIVE"></span>**影像基本** 
</dt> <dd>

點陣圖或影像。

</dd> <dt>

<span id="opengl_immediate_mode"></span><span id="OPENGL_IMMEDIATE_MODE"></span>**立即模式**
</dt> <dd>

直接呼叫 OpenGL 命令的模式，而不是從顯示清單。 沒有立即模式位存在;立即模式中的模式指的是 OpenGL 的使用方式，而不是特定的 OpenGL 狀態位。

</dd> <dt>

<span id="opengl_index"></span><span id="OPENGL_INDEX"></span>**指數**
</dt> <dd>

轉譯為絕對值的單一值，而不是指定範圍內的正規化值 (為元件) 。 色彩索引是色彩的名稱，可使用色彩對應以顯示硬體進行取值。 索引通常會在超出範圍時進行遮罩，而不是壓制。 例如，將索引0xf7 寫入至4位緩衝區時，會將索引遮罩至0x7， (色彩或樣板) 。 色彩索引和樣板索引一律會被視為索引，而不是元件。

</dd> <dt>

<span id="opengl_iris_gl"></span><span id="OPENGL_IRIS_GL"></span>**鳶尾花 GL**
</dt> <dd>

晶片圖形專屬圖形程式庫，從1982到1992開發。 OpenGL 的設計是以鳶尾花 GL 作為起點。

</dd> </dl>
