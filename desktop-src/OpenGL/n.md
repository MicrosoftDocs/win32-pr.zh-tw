<dl> <dt>

<span id="opengl_network"></span><span id="OPENGL_NETWORK"></span>**網路**
</dt> <dd>

兩部或多部電腦之間的連接，可讓每個電腦互相傳輸資料。

</dd> <dt>

<span id="opengl_nonconvex"></span><span id="OPENGL_NONCONVEX"></span>**nonconvex**
</dt> <dd>

多邊形平面中的線條與多邊形相交超過兩倍的多邊形狀態。

</dd> <dt>

<span id="opengl_normal"></span><span id="OPENGL_NORMAL"></span>**正常**
</dt> <dd>

三個元件的平面方程式，定義平面或表面的角度方向，但不是位置。

</dd> <dt>

<span id="opengl_normalize"></span><span id="OPENGL_NORMALIZE"></span>**規範**
</dt> <dd>

將一般的每個元件除以其平方總和的平方根。 然後，如果將標準視為從原點到點的向量 (nx '，ny '，nx ' ) ，則這個向量具有單位長度：

<dl> <dd>nx ' = nx/因數</dd> <dd>ny ' = ny/因數</dd> <dd>nz ' = nz/因數</dd> </dl> </dd> <dt>

<span id="opengl_normal_vector"></span><span id="OPENGL_NORMAL_VECTOR"></span>**一般向量**
</dt> <dd>

請參閱 *正常*。

</dd> <dt>

<span id="opengl_nurbs"></span><span id="OPENGL_NURBS"></span>**NURBS**
</dt> <dd>

非統一的有理 B 曲線。 指定參數曲線和表面的常見方式。

</dd> </dl>
