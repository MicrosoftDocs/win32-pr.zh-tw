# <a name="pixels"></a>阻擋的

片段會轉換成畫面格緩衝區中的圖元。 畫面格緩衝區會組織成一組邏輯緩衝的色彩、深度、樣板和累積緩衝區。 色彩緩衝區本身是由 front left、right、back、back、back 和某些數目的輔助緩衝區所組成。 您可以發出函式來控制這些緩衝區，並直接讀取或複製它們的圖元。  (請注意，您所使用的特定 OpenGL 內容可能不會提供所有這些緩衝區。 )

-   [讀取或複製圖元](reading-or-copying-pixels.md)
-   [圖元參考](pixels-reference.md)
