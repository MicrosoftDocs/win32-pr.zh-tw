# <a name="guide-to-documentation"></a>檔指南

在 Windows 中為 OpenGL 設定的檔包含五個元素。

-   *Opengl 參考手冊* 包含了 opengl 的運作方式，以及一組詳細的參考頁面。 參考頁面涵蓋所有115相異的 OpenGL 函式，以及 OpenGL 公用程式中的43函式 (X.GLU 隊) 程式庫。

-   *Opengl 程式設計指南* 說明如何使用 OpenGL 建立圖形程式。 其中包含下列主要主題的討論：

    -   繪製幾何形狀
    -   圖元、點陣圖、字型和影像
    -   視圖和矩陣轉換
    -   材質對應
    -   顯示清單
    -   Advanced 複合技術
    -   Color
    -   評估工具和 NURBS
    -   光源
    -   選取專案和意見反應
    -   混合、消除鋸齒和霧化
    -   先進的技術

    此外，Opengl 程式 *設計指南* 包含的附錄可討論 Opengl 公用程式程式庫和 Opengl 程式設計手冊輔助程式庫。

-   第三個檔元素是此總覽。 它描述 OpenGL 的 Windows 執行，並提供其元件的總覽。 它會討論轉譯內容、像素格式和緩衝區的概念;將 OpenGL 連接至 Windows 視窗系統的 WGL 函式;以及支援每個視窗像素格式的 Windows 函式，以及適用于 OpenGL 圖形視窗的視窗的雙重緩衝。 WGL 函式和函式是適用于 OpenGL 的 Windows 實作為專屬的。

-   第四個檔元素是 WGL 函式的參考頁面集合、剛才提及的 Windows 函式，以及 [**PIXELFORMATDESCRIPTOR**](/windows/win32/api/wingdi/ns-wingdi-pixelformatdescriptor) 資料結構。

-   第五個檔元素是移植指南。 它會討論如何將現有的 OpenGL 程式碼從其他環境移至 Windows。

前兩個元素是官方的 OpenGL 書籍;它們不包含在 Microsoft Windows SDK 中。
