# <a name="gllightmodelfv-function"></a>glLightModelfv 函式

[**GlLightModelfv**](gllightfv.md)函式會設定光源模型參數。

## <a name="syntax"></a>語法


```C++
void WINAPI glLightModelfv(
         GLenum  pname,
   const GLfloat *params
);
```

## <a name="remarks"></a>備註

**GlLightModelfv** 函數會設定光源模型參數。 *Pname* 參數會為參數命名，而 *param* 會提供新值。個別光源參數的值或值。

在 RGBA 模式中，頂點的燈光色彩是材質排放強度的總和、材質環境反射率的乘積以及光線模型的全場景環境濃度，以及每個已啟用光源的比重。 每個燈光來源都有三個詞彙的總和：環境、擴散和反射。

-   環境光源的比重是材質環境反射率和光線環境濃度的乘積。
-   擴散光源的比重是材質的材質反射率、光線的擴散濃度，以及頂點的標準向量（從頂點到光源）的點乘積乘積。
-   反射光源的比重是材質反射反射率的乘積、光線的反射濃度，以及正規化頂點對眼睛和頂點對光線向量的點乘積，並提升為材質的反光能力。

所有三個光源的貢獻都會根據從頂點到光源的距離，以及光源方向、分佈指數和分佈截止角度，平均衰減。 如果所有的點乘積都評估為負值，則會以零取代。

產生之淺色色彩的 Alpha 元件會設定為材質漫射反射率的 Alpha 值。

在色彩索引模式中，頂點的淺色索引值範圍從環境到使用 GL 色彩索引傳遞給 [**glMaterial**](glmaterial-functions.md) 的反射值 \_ \_ 。 擴散和反射係數，以 ( .30、59、.11) 加權的色彩、材質的反光、以及與 RGBA 案例相同的反映和衰減方程式來計算，以判斷產生的索引在環境上的程度。
