# <a name="t-opengl"></a>T (OpenGL) 

[](a.md) [B](b.md) [C](c.md) [D](d.md) [E](e.md) [F](f.md) [G](g.md) [H](h.md) [I](i.md) [J K](jk.md) [L](l.md) [M](m.md) [N](n.md) [O](o.md) [P](p.md) [Q](q.md) [R](r.md) [S](s.md) [T](u-v.md) [W](w.md) [X Y Z](x-y-z.md)

<dl> <dt>

<span id="opengl_tessellation"></span><span id="OPENGL_TESSELLATION"></span>**鑲嵌**
</dt> <dd>

將分析表面的一部分縮減為多邊形的格線，或將分析曲線的一部分縮減為一連串的線條。

</dd> <dt>

<span id="opengl_texel"></span><span id="OPENGL_TEXEL"></span>**材質**
</dt> <dd>

材質元素。 材質是從紋理記憶體取得，代表要套用至對應片段的材質色彩。 另請參閱 [片段](f.md)。

</dd> <dt>

<span id="opengl_texture"></span><span id="OPENGL_TEXTURE"></span>**紋理**
</dt> <dd>

用來修改點陣化所產生片段色彩的一或兩個維度影像。 另請參閱「 [光柵](r.md)化」。

</dd> <dt>

<span id="opengl_texture_mapping"></span><span id="OPENGL_TEXTURE_MAPPING"></span>**材質對應**
</dt> <dd>

將影像 (材質) 套用至基本的流程。 材質對應通常用來為場景增加逼真。 例如，您可以將大樓外觀的圖片套用至代表牆的多邊形。 另請參閱材質。

</dd> <dt>

<span id="opengl_texture_matrix"></span><span id="OPENGL_TEXTURE_MATRIX"></span>**材質矩陣**
</dt> <dd>

4x4 矩陣，可將材質座標從其所指定的座標轉換為用於插補和材質查閱的座標。

</dd> <dt>

<span id="opengl_transformation"></span><span id="OPENGL_TRANSFORMATION"></span>**轉型**
</dt> <dd>

扭曲的空間。 在 OpenGL 中，轉換僅限於 projective 轉換，這些轉換包含可由4x4 矩陣表示的任何事物。 這類轉換包括旋轉、翻譯、 (非統一) scalings 沿著座標軸、觀點轉換以及這些的組合。

</dd> <dt>

<span id="opengl_triangle"></span><span id="OPENGL_TRIANGLE"></span>**三角形**
</dt> <dd>

具有三個邊緣的多邊形。 三角形一律為凸凸。

</dd> </dl>
