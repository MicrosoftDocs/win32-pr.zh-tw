# <a name="alpha-test"></a>Alpha 測試

Alpha 測試 (只在 RGBA 模式中執行) 會捨棄片段，這取決於片段的 Alpha 值與常數參考值之間的比較結果。 您可以使用 [**glAlphaFunc**](glalphafunc.md)來指定比較函數和參考值。
