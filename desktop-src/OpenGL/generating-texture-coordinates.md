# <a name="generating-texture-coordinates"></a>產生材質座標

您可以使用[glTexGen \* ](gltexgen-functions.md)，讓 OpenGL 將它們產生為其他頂點資料的函式，而不是明確提供材質座標。 在指定或產生材質座標之後，材質矩陣會轉換它們。 此矩陣是使用用於矩陣轉換的相同函式來控制 (請參閱 [矩陣轉換](matrix-transformations.md)) 。
