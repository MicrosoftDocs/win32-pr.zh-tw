# <a name="logical-operations"></a>邏輯作業

邏輯運算可以在片段和儲存在畫面格緩衝區中對應位置的值之間套用。結果會取代目前的畫面格緩衝區值。 您可以使用 [**glLogicOp**](gllogicop.md)選擇所需的邏輯作業。 邏輯作業只會在色彩索引上執行，而不會在 RGBA 值上執行。
