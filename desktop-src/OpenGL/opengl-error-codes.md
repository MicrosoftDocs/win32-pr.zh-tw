# <a name="opengl-error-codes"></a>OpenGL 錯誤碼

OpenGL 包含下列錯誤碼。



| 錯誤碼             | 描述                                 |
|------------------------|---------------------------------------------|
| GL \_ 無效 \_ 列舉      | **GLenum** 引數超出範圍。           |
| GL \_ 無效 \_ 值     | 數值引數超出範圍。              |
| GL \_ 不正確 \_ 操作 | 作業在目前狀態中不合法。         |
| GL \_ 堆疊 \_ 溢位    | 函數會導致堆疊溢位。      |
| GL \_ 堆疊 \_ 下溢   | 函數會導致堆疊下溢。     |
| GL \_ \_ 記憶體不足 \_    | 剩餘的記憶體不足，無法執行函數。 |
