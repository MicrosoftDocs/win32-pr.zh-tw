# <a name="primitives-opengl"></a> (OpenGL) 的基本專案

在下一個處理階段中，會在下列步驟中將基本專案轉換成圖元片段：

-   會適當地裁剪基本專案。
-   需要對色彩和材質資料進行對應的調整，而相關的座標則會轉換成視窗座標。
-   柵格化會將裁剪的基本專案轉換成圖元片段。

如需詳細資訊，請參閱下列主題：

-   [裁剪](clipping.md)
-   [轉換成視窗座標](transforming-to-window-coordinates.md)
-   [點陣化](rasterizing.md)
-   [基本參考](primitives-reference.md)
