# <a name="glvertex2dv-function"></a>glVertex2dv 函式

指定頂點。

## <a name="syntax"></a>語法


```C++
void WINAPI glVertex2dv(
   const GLdouble *v
);
```



## <a name="parameters"></a>參數

<dl> <dt>

*v* 
</dt> <dd>

兩個元素的陣列指標。 元素是頂點的 x 和 y 座標。

</dd> </dl>

## <a name="return-value"></a>傳回值

此函式不會傳回值。
