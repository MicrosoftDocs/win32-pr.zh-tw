<dl> <dt>

<span id="opengl_server"></span><span id="OPENGL_SERVER"></span>**伺服器**
</dt> <dd>

執行 OpenGL 命令的電腦。 這可能不同于發出命令的電腦。 另請參閱 client。

</dd> <dt>

<span id="opengl_shading"></span><span id="OPENGL_SHADING"></span>**陰影**
</dt> <dd>

在柵格化期間，于多邊形內部或線條頂點之間插即插色彩的進程。

</dd> <dt>

<span id="opengl_single_buffering"></span><span id="OPENGL_SINGLE_BUFFERING"></span>**單一緩衝**
</dt> <dd>

沒有背面色彩緩衝區的 OpenGL 內容。

</dd> <dt>

<span id="opengl_stipple"></span><span id="OPENGL_STIPPLE"></span>**stipple**
</dt> <dd>

一或兩個維度的二進位模式，會在其值為零的情況下產生片段的產生。 Line stipples 是一維的，並且會相對於一行的開頭套用。 多邊形 stipples 是二維的，並且會以固定方向套用至視窗。

</dd> </dl>
