# <a name="w-opengl"></a>W (OpenGL)

<dl> <dt>

<span id="opengl_window"></span><span id="OPENGL_WINDOW"></span>**視窗**
</dt> <dd>

畫面格緩衝區的子領域，通常是矩形，其圖元全都具有相同的緩衝區設定。 OpenGL 內容會一次轉譯成一個視窗。

</dd> <dt>

<span id="opengl_window_aligned"></span><span id="OPENGL_WINDOW_ALIGNED"></span>**對齊視窗**
</dt> <dd>

輔助線段或多邊形邊緣時，表示這些會與視窗界限平行。  (在 OpenGL 中，視窗是矩形，水準和垂直邊緣) 。 參考多邊形模式時，表示模式是相對於視窗原點固定的。

</dd> <dt>

<span id="opengl_window_coordinates"></span><span id="OPENGL_WINDOW_COORDINATES"></span>**視窗座標**
</dt> <dd>

視窗的座標系統。 請務必區分圖元的名稱，這些名稱是離散的，而視窗座標系統則是連續的。 例如，視窗左下角的圖元是圖元 (0、0) ;這個圖元中央的視窗座標是 (0.5、0.5、z) 。 請注意，視窗座標包含深度或 z、元件，而且此元件也是連續的。

</dd> <dt>

<span id="opengl_wireframe"></span><span id="OPENGL_WIREFRAME"></span>**著色**
</dt> <dd>

物件的表示，其中只包含行區段。 通常，線段表示多邊形邊緣。

</dd> </dl>
