# <a name="initializing"></a>正在初始化

使用 X.GLU 隊1.1 版或更新版本時， [**gluGetString**](glugetstring.md) 會傳回 x.glu 隊程式庫的版本號碼，或版本號碼和任何廠商專屬的 x.glu 隊延伸模組呼叫。

**GluGetString** 的原型是：

const **GLubyte** \* **gluGetString** (**GLenum** *name* ) ;
