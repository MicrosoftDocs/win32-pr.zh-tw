<dl> <dt>

<span id="opengl_lighting"></span><span id="OPENGL_LIGHTING"></span>**照明**
</dt> <dd>

根據目前的燈光、材質屬性和光源模型模式來計算頂點色彩的流程。

</dd> <dt>

<span id="opengl_line"></span><span id="OPENGL_LINE"></span>**線**
</dt> <dd>

兩個頂點之間的有限寬度的直接區域。  (與數學線不同的是，OpenGL 線條具有有限的寬度和長度。 ) 帶狀線的每個區段都是一行。

</dd> <dt>

<span id="opengl_luminance"></span><span id="OPENGL_LUMINANCE"></span>**亮度**
</dt> <dd>

表面的認知亮度。 通常是指紅色、綠色和藍色色彩值的加權平均，這些值會提供已察覺的組合亮度。

</dd> </dl>
