# <a name="setting-lighting-and-coloring"></a>設定光源和著色

除了指定色彩和一般向量之外，您還可以使用 [**\* GlLight**](gllight-functions.md)和 [**glLightModel \***](gllightmodel-functions.md)來定義所需的光源條件，以及使用 [**glMaterial \***](glmaterial-functions.md)來定義所需的材質屬性。 控制照明計算執行方式的相關函式包括 [**glShadeModel**](glshademodel.md)、 [**glFrontFace**](glfrontface.md)和 [**glColorMaterial**](glcolormaterial.md)。
