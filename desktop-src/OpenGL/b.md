<dl> <dt>

<span id="opengl_back_face"></span><span id="OPENGL_BACK_FACE"></span>**背景臉部**
</dt> <dd>

請參閱 [臉部](f.md)。

</dd> <dt>

<span id="opengl_bit"></span><span id="OPENGL_BIT"></span>**位**
</dt> <dd>

二進位位數。 只有兩個可能值的狀態變數：0或1。 二進位數是一或多個位的結構。

</dd> <dt>

<span id="opengl_bitmap"></span><span id="OPENGL_BITMAP"></span>**點陣圖**
</dt> <dd>

位的矩形陣列。 此外， [**glBitmap**](glbitmap.md) 命令所呈現的基本類型，會使用其 bitmap 參數做為遮罩。

</dd> <dt>

<span id="opengl_bitplane"></span><span id="OPENGL_BITPLANE"></span>**bitplane**
</dt> <dd>

以圖元為單位對應的位矩形陣列。

</dd> <dt>

<span id="opengl_blending"></span><span id="OPENGL_BLENDING"></span>**混合**
</dt> <dd>

將兩個色彩元件減少為一個元件，通常是兩個元件之間的線性插補。

</dd> <dt>

<span id="opengl_buffer"></span><span id="OPENGL_BUFFER"></span>**緩衝區**
</dt> <dd>

一組 bitplanes，用來儲存單一元件 (例如深度或綠色) 或單一索引 (例如色彩索引或樣板索引) 。 有時會將紅色、綠色、藍色和 Alpha 緩衝區統稱為色彩緩衝區 (而不是色彩緩衝區) 。

</dd> </dl>
