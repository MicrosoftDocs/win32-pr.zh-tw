# <a name="glvertex2sv-function"></a>glVertex2sv 函式

指定頂點。

## <a name="syntax"></a>語法


```C++
void WINAPI glVertex2sv(
   const GLshort *v
);
```



## <a name="parameters"></a>參數

<dl> <dt>

*v* 
</dt> <dd>

兩個元素的陣列指標。 元素是頂點的 x 和 y 座標。

</dd> </dl>

## <a name="return-value"></a>傳回值

此函式不會傳回值。
