# <a name="glutessproperty-function"></a>gluTessProperty 函式

**GluTessProperty** 函式會設定鑲嵌式物件的屬性。

## <a name="syntax"></a>語法


```C++
void WINAPI gluTessProperty(
   GLUtesselator *tess,
   GLenum        which,
   GLdouble      value
);
```

## <a name="return-value"></a>傳回值

此函式不會傳回值。

## <a name="remarks"></a>備註

**GluTessProperty** 函式會控制儲存在鑲嵌式物件中的屬性。 這些屬性會影響多邊形的解讀和轉譯方式。
