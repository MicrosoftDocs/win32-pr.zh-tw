# <a name="depth-buffer-test"></a>深度緩衝區測試

深度緩衝區測試會在深度比較失敗時捨棄片段; [**glDepthFunc**](gldepthfunc.md) 指定比較函數。 如果 stenciling 已啟用，則深度比較的結果也會影響樣板緩衝區更新值。
