# <a name="copying-an-opengl-image-to-the-clipboard"></a>將 OpenGL 影像複製到剪貼簿

雖然目前的 Windows 在中執行 OpenGL 的版本不直接支援剪貼簿，但是您可以將 Windows 的 opengl 影像複製到剪貼簿。

**將 OpenGL 影像複製到剪貼簿**

1.  將影像繪製至記憶體點陣圖或增強的中繼檔。
2.  將點陣圖複製到剪貼簿。
