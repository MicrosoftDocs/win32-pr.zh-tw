<dl> <dt>

<span id="opengl_half_space"></span><span id="OPENGL_HALF_SPACE"></span>**一半空格**
</dt> <dd>

平面除以空間的結果。 平面會將空間分割成兩個半空格。

</dd> <dt>

<span id="opengl_homogenous_coordinates"></span><span id="OPENGL_HOMOGENOUS_COORDINATES"></span>**同質座標**
</dt> <dd>

一組 n + 1 座標，用來表示 n 維度 projective 空間中的點。 Projective 空間中的點可視為 Euclidean 空間中的點，以及無限大的一些點。 座標是同質，因為依相同非零值常數調整每個座標，並不會改變座標所參考的點。 同質座標適用于 projective 幾何的計算，因此在電腦圖形中，場景必須投射到視窗上。

</dd> </dl>
