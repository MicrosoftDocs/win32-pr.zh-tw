# <a name="glvertex3iv-function"></a>glVertex3iv 函式

指定頂點。

## <a name="syntax"></a>語法


```C++
void WINAPI glVertex3iv(
   const GLint *v
);
```



## <a name="parameters"></a>參數

<dl> <dt>

*V* 
</dt> <dd>

三個元素的陣列指標。 元素是頂點的 x、y 和 z 座標。

</dd> </dl>

## <a name="return-value"></a>傳回值

此函式不會傳回值。
