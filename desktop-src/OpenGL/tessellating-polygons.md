# <a name="tessellating-polygons"></a><span data-ttu-id="2d326-111">鑲嵌多邊形</span><span class="sxs-lookup"><span data-stu-id="2d326-111">Tessellating Polygons</span></span>

<span data-ttu-id="2d326-112">OpenGL 只能直接顯示簡單的凸形多邊形。</span><span class="sxs-lookup"><span data-stu-id="2d326-112">OpenGL can directly display only simple convex polygons.</span></span> <span data-ttu-id="2d326-113">多邊形很簡單，如下所示：</span><span class="sxs-lookup"><span data-stu-id="2d326-113">A polygon is simple if:</span></span>

-   <span data-ttu-id="2d326-114">邊緣只會在頂點上相交。</span><span class="sxs-lookup"><span data-stu-id="2d326-114">The edges intersect only at vertices.</span></span>
-   <span data-ttu-id="2d326-115">沒有重複的頂點。</span><span class="sxs-lookup"><span data-stu-id="2d326-115">There are no duplicate vertices.</span></span>
-   <span data-ttu-id="2d326-116">正好有兩個邊緣符合任何頂點。</span><span class="sxs-lookup"><span data-stu-id="2d326-116">Exactly two edges meet at any vertex.</span></span>

<span data-ttu-id="2d326-117">若要顯示包含孔洞的簡單 nonconvex 多邊形或簡單多邊形，您必須先分成三角形多邊形 (將它們細分成凸多邊形) 。</span><span class="sxs-lookup"><span data-stu-id="2d326-117">To display simple nonconvex polygons or simple polygons containing holes, you must first triangulate the polygons (subdivide them into convex polygons).</span></span> <span data-ttu-id="2d326-118">這類細分稱為 *鑲嵌* 式。</span><span class="sxs-lookup"><span data-stu-id="2d326-118">Such subdivision is called *tessellation*.</span></span> <span data-ttu-id="2d326-119">X.GLU 隊提供執行鑲嵌的函式集合。</span><span class="sxs-lookup"><span data-stu-id="2d326-119">GLU provides a collection of functions that perform tessellation.</span></span> <span data-ttu-id="2d326-120">請注意，X.GLU 隊鑲嵌函數無法處理簡單多邊形;沒有標準的 OpenGL 方法可處理這類多邊形。</span><span class="sxs-lookup"><span data-stu-id="2d326-120">Note that the GLU tessellation functions can't handle nonsimple polygons; there is no standard OpenGL method to handle such polygons.</span></span>

<span data-ttu-id="2d326-121">因為鑲嵌通常是必要的，而且可能很難處理，所以本節會詳細說明 X.GLU 隊鑲嵌函數。</span><span class="sxs-lookup"><span data-stu-id="2d326-121">Because tessellation is often required and can be rather tricky, this section describes the GLU tessellation functions in detail.</span></span> <span data-ttu-id="2d326-122">這些函式會採用輸入任意簡單的多邊形（其中可能包含洞），並傳回三角形、三角形網格和三角形風扇的組合。</span><span class="sxs-lookup"><span data-stu-id="2d326-122">These functions take as input arbitrary simple polygons that might include holes, and they return some combination of triangles, triangle meshes, and triangle fans.</span></span> <span data-ttu-id="2d326-123">如果您不想要處理網格或風扇，可以指定鑲嵌式函數只傳回三角形。</span><span class="sxs-lookup"><span data-stu-id="2d326-123">If you don't want to deal with meshes or fans, you can specify that the tessellation functions return only triangles.</span></span> <span data-ttu-id="2d326-124">不過，網格和風扇資訊可提升效能。</span><span class="sxs-lookup"><span data-stu-id="2d326-124">However, mesh and fan information improves performance.</span></span> <span data-ttu-id="2d326-125">多邊形鑲嵌式函數會分成三角形具有一或多個輪廓的凹多邊形。</span><span class="sxs-lookup"><span data-stu-id="2d326-125">The polygon tessellation functions triangulate a concave polygon with one or more contours.</span></span>

<span data-ttu-id="2d326-126">**使用多邊形鑲嵌**</span><span class="sxs-lookup"><span data-stu-id="2d326-126">**To use polygon tessellation**</span></span>

    <span data-ttu-id="2d326-130">當多邊形描述完成時，鑲嵌式功能會視需要叫用您的回呼函式。</span><span class="sxs-lookup"><span data-stu-id="2d326-130">When the polygon description is complete, the tessellation facility invokes your callback functions as necessary.</span></span>

    <span data-ttu-id="2d326-131">您可以使用 [**gluDeleteTess**](gludeletetess.md)終結不必要的鑲嵌式物件。</span><span class="sxs-lookup"><span data-stu-id="2d326-131">You can destroy unneeded tessellation objects with [**gluDeleteTess**](gludeletetess.md).</span></span>

<span data-ttu-id="2d326-132">如需儲存鑲嵌式資料的詳細資訊，請參閱 [使用回呼函數](using-callback-functions.md)。</span><span class="sxs-lookup"><span data-stu-id="2d326-132">For more information on saving the tessellation data, see [Using Callback Functions](using-callback-functions.md).</span></span>
