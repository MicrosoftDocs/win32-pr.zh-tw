# <a name="porting-other-glx-code"></a>移植其他 GLX 程式碼

除了上述各節所述的 Xlib 和 GLX 函式之外，您的程式可能還包含 [翻譯 GLX 程式庫](translating-the-glx-library.md)中列出的其他 GLX 或 Xlib 函式。 將您的 X 視窗系統程式碼重寫為 Windows，並替代適當的函式。
