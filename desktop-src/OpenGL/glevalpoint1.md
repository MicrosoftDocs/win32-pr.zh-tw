# <a name="glevalpoint1-function"></a>glEvalPoint1 函式

[**GlEvalPoint1**](glevalpoint.md)和 **glEvalPoint2** 函數會產生和評估網格中的單一點。

## <a name="syntax"></a>語法


```C++
void glEvalPoint1(
   GLint i
);
```



## <a name="parameters"></a>參數

<dl> <dt>

*i* 
</dt> <dd>

方格網域變數 *i* 的整數值。

</dd> </dl>

## <a name="return-value"></a>傳回值

此函式不會傳回值。
