# <a name="opengl"></a>Opengl

## <a name="purpose"></a>目的

OpenGL 是圖形硬體的軟體介面，可將多維度物件轉譯成畫面格緩衝區。 Microsoft 針對 Windows 作業系統執行 OpenGL 是業界標準的圖形軟體，程式設計師可以用它來建立高品質的靜止和動畫三維色彩影像。 本節所述的 OpenGL 版本為1.1。

如需在 Windows 上執行之 OpenGL ES 的詳細資訊，請參閱[Windows 存放區的角度](https://github.com/microsoft/angle/wiki)。

## <a name="where-applicable"></a>適用時

OpenGL 是針對硬體和作業系統之間的相容性所建立的。 此架構可讓您輕鬆地將 OpenGL 程式從某個系統移植到另一個系統。 雖然每個作業系統都有獨特的需求，但許多程式中的 OpenGL 程式碼可依原樣使用。

## <a name="developer-audience"></a>開發人員對象

OpenGL 是針對 C/c + + 程式設計人員所設計，需要熟悉 Windows 的圖形化使用者介面以及訊息驅動架構。

## <a name="run-time-requirements"></a>執行階段需求求

如需特定功能需要哪些作業系統的詳細資訊，請參閱函式檔集的需求一節。
