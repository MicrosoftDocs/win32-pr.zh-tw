# <a name="appendix-opengl"></a> (OpenGL) 附錄

OpenGL，最初是由晶片圖形所開發，為其圖形工作站 (的 SGI) ，可讓應用程式建立與視窗系統、作業系統和硬體無關的高品質色彩影像。

OpenGL 架構審核面板 (ARB) （工業協會）目前負責定義 OpenGL。 ARB 的成員包含隨附的晶片圖形，Microsoft Corporation、Intel、IBM 和數位設備公司。

Opengl 第1版的官方參考檔是 opengl *參考手冊*，由 Opengl 架構審核委員會 (ISBN 0-201-63276-4) 。 學習 OpenGL 第1版的官方指南是 Opengl 程式 *設計指南*，Jackie Neider、Tom Davis 和 Mason WOO (ISBN 0-201-63274-8) 。 這兩本書都是由 Addison-Wesley 發佈。
