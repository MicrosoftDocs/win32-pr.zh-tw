# <a name="g-opengl"></a>G (OpenGL)

<dl> <dt>

<span id="opengl_gamma_correction"></span><span id="OPENGL_GAMMA_CORRECTION"></span>**gamma 修正**
</dt> <dd>

一種函式，適用于儲存在框架緩衝區中的色彩，以更正眼睛 (的非線性回應，有時監視器) 色彩濃度值的線性變更。

</dd> <dt>

<span id="opengl_geometric_model"></span><span id="OPENGL_GEOMETRIC_MODEL"></span>**幾何模型**
</dt> <dd>

描述物件的物件座標頂點和參數。 請注意，OpenGL 不會定義幾何模型的語法，而是用來呈現幾何模型的語法和語義。

</dd> <dt>

<span id="opengl_geometric_object"></span><span id="OPENGL_GEOMETRIC_OBJECT"></span>**幾何物件**
</dt> <dd>

幾何模型。

</dd> <dt>

<span id="opengl_geometric_primitive"></span><span id="OPENGL_GEOMETRIC_PRIMITIVE"></span>**幾何基本**
</dt> <dd>

點、線條或多邊形。

</dd> <dt>

<span id="opengl_gouraud_shading"></span><span id="OPENGL_GOURAUD_SHADING"></span>**Gouraud 網底**
</dt> <dd>

在多邊形或線段線段之間平滑色彩的插補。 色彩會指派于頂點，並以線性方式插入至基本型別，以產生相對較平滑的色彩變化。 也稱為平滑陰影。

</dd> <dt>

<span id="opengl_group"></span><span id="OPENGL_GROUP"></span>**組**
</dt> <dd>

一組、二、三或四個元素，代表用戶端記憶體中影像的每個圖元。 因此，在用戶端記憶體映射的內容中，群組和圖元就是相同的東西。

</dd> </dl>
