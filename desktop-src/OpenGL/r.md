# <a name="r-opengl"></a>R (OpenGL)

<dl> <dt>

<span id="opengl_rasterize"></span><span id="OPENGL_RASTERIZE"></span>**光柵**
</dt> <dd>

將投影點、線條或多邊形，或點陣圖或影像的圖元轉換成片段，每個片段都對應到畫面格緩衝區中的圖元。 請注意，所有基本專案都是柵格化的，而不只是點、線條和多邊形

</dd> <dt>

<span id="opengl_rectangle"></span><span id="OPENGL_RECTANGLE"></span>**矩形**
</dt> <dd>

四邊形，其替代邊緣會以物件座標平行處理。 以 glRect ( ) 指定的多邊形 \* 一律為矩形; 其他 quadrilaterals 可能是矩形。

</dd> <dt>

<span id="opengl_rendering"></span><span id="OPENGL_RENDERING"></span>**渲染**
</dt> <dd>

將物件座標中指定的基本類型轉換成畫面格緩衝區中的影像。 轉譯是 OpenGL 的主要作業。

</dd> <dt>

<span id="opengl_rgba"></span><span id="OPENGL_RGBA"></span>**RGBA**
</dt> <dd>

RGBA 模式的紅色、綠色、藍色和 Alpha 色彩元件。

</dd> <dt>

<span id="opengl_rgba_mode"></span><span id="OPENGL_RGBA_MODE"></span>**RGBA 模式**
</dt> <dd>

OpenGL 內容，其中的色彩緩衝區會儲存紅色、綠色、藍色和 Alpha 色彩元件，而不是色彩索引。

</dd> </dl>
