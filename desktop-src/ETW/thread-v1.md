# <a name="thread_v1-class"></a>Thread \_ V1 類別

這個類別是執行緒事件的父類別。

以下是從 MOF 程式碼簡化的語法。

## <a name="syntax"></a>語法

``` syntax
[Guid("{3d6fa8d1-fe05-11d0-9dda-00c04fd7ba7c}"), EventVersion(1)]
class Thread_V1 : MSNT_SystemTrace
{
};
```

## <a name="members"></a>成員

**執行緒 \_ V1** 類別未定義任何成員。
