# <a name="isr-class"></a>ISR 類別

這個類別是插斷服務常式的事件種類類別， (ISR) 事件。

以下是從 MOF 程式碼簡化的語法。

## <a name="syntax"></a>語法

``` syntax
[EventType{67}, EventTypeName{"ISR"}]
class ISR : PerfInfo
{
  object InitialTime;
  uint32 Routine;
  uint8  ReturnValue;
  uint8  Vector;
  uint16 Reserved;
};
```

## <a name="members"></a>成員

**ISR** 類別具有下列類型的成員：

-   [屬性](#properties)
