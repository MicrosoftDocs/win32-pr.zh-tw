# <a name="image_v1_load-class"></a>映射 \_ V1 \_ 載入類別

這個類別是影像載入事件的事件種類類別。

以下是從 MOF 程式碼簡化的語法。

## <a name="syntax"></a>語法

``` syntax
[EventType(10), EventTypeName("Load")]
class Image_V1_Load : Image_V1
{
  uint32 ImageBase;
  uint32 ImageSize;
  uint32 ProcessId;
  string FileName;
};
```

## <a name="members"></a>成員

**映射 \_ V1 \_ 載入** 類別具有下列類型的成員：

-   [屬性](#properties)

**映射 \_ V1 \_ 載入** 類別具有這些屬性。
