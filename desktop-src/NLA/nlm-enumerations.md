# <a name="nlm-enumerations"></a>NLM 列舉

以下是 NLM 列舉。

| 列舉型別                                                                 | 描述                                                                                |
|-----------------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [**NLM \_ 連接 \_ 成本**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_connection_cost)                        | 指定 Windows 8 成本 api 中支援的一組成本等級和成本旗標。            |
| [**NLM \_ 連接 \_ 屬性 \_ 變更**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_connection_property_change) | 定義對網路連接的屬性所做的變更。                            |
| [**NLM \_ 連接**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_connectivity)                               | 當連線相關的參數變更時提供通知。               |
| [**NLM \_ 網域 \_ 類型**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_domain_type)                                | 指定網路的網欄位型別。                                                    |
| [**NLM \_ 列舉 \_ 網路**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_enum_network)                              | 指定列舉的網路類型。                                           |
| [**NLM \_ 網際網路連線 \_ 能力**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_internet_connectivity)            | 定義一組旗標，以提供 IPv4 或 IPv6 網路連線的額外資料。 |
| [**NLM \_ NETWORK \_ 類別**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_network_class)                            | 定義一組旗標，指定是否已識別網路。                      |
| [**NLM \_ 網路 \_ 類別**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_network_category)                      | 指定網路的類別目錄類型。                                                  |
| [**NLM \_ 網路 \_ 屬性 \_ 變更**](/windows/desktop/api/Netlistmgr/ne-netlistmgr-nlm_network_property_change)       | 定義對網路屬性進行的變更。                                       |
