# <a name="nlm-structures"></a>NLM 結構

下列結構支援網路清單管理員 API：



| 結構                                                           | Description                                                          |
|---------------------------------------------------------------------|----------------------------------------------------------------------|
| [**NLM \_ 模擬 \_ 設定檔 \_ 資訊**](/windows/desktop/api/Netlistmgr/ns-netlistmgr-nlm_simulated_profile_info) | 代表要在成本模擬期間使用的設定檔資訊。    |
| [**NLM \_ SOCKADDR**](/windows/desktop/api/Netlistmgr/ns-netlistmgr-nlm_sockaddr)                               | 代表 IPv4/IPv6 目的地位址。                        |
| [**NLM \_ 使用量 \_ 資料**](/windows/desktop/api/Netlistmgr/ns-netlistmgr-nlm_usage_data)                          | 表示通話方案的資料使用方式的資訊。 |
| [**NLM \_ 資料 \_ 方案 \_ 狀態**](https://www.bing.com/search?q=**NLM\_DATA\_PLAN\_STATUS**)             | 表示通話方案的目前狀態。                        |
