# <a name="network-list-manager"></a>網路清單管理員

## <a name="purpose"></a>目的

網路清單管理員 API 可讓應用程式抓取可用網路連接的清單。 應用程式可以根據屬性和簽章來篩選網路，並選擇最適合其工作的網路。 網路清單管理員基礎結構會通知應用程式網路環境中的變更，進而讓應用程式動態更新網路連線。

## <a name="developer-audience"></a>開發人員對象

網路清單管理員是設計來供 COM 程式設計人員使用。

## <a name="run-time-requirements"></a>執行階段需求求

從 Windows Vista 開始支援網路清單管理員 API。
