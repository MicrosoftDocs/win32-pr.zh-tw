# <a name="filled-shapes"></a>填滿的圖案

*填滿的圖案* 是使用目前畫筆所概述並使用目前筆刷填滿的幾何形式。 有五個填滿的圖形：

-   橢圓形
-   Chord
-   Pie
-   Polygon
-   矩形

此總覽描述填滿的圖形。

-   [使用填滿的圖形](using-filled-shapes.md)
