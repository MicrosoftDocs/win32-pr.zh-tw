# <a name="about-lines-and-curves"></a>關於線條和曲線

許多類型的應用程式都使用線條和曲線來繪製點陣裝置上的圖形輸出。 電腦輔助設計 (CAD) 和繪圖應用程式會使用線條和曲線來勾勒出物件、指定物件的中心、物件的維度等等。 試算表應用程式會使用線條和曲線來繪製格線、圖表和圖形。 文書處理應用程式使用線條在文字頁面上建立規則和框線。
