# <a name="pen-width"></a>畫筆寬度

Width 屬性會以裝置單位來指定表面寬度。 不過，當與幾何畫筆搭配使用時，它會以邏輯單元來指定畫筆的寬度。 如需裝置單位的詳細資訊，請參閱 [座標空間和轉換](coordinate-spaces-and-transformations.md)。

目前，系統會將裝飾筆的寬度限制為單一圖元;不過，未來的版本可能會移除這項限制。
