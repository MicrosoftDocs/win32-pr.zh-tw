# <a name="font-families"></a>字型系列

*字型家族* 是一組具有一般筆觸寬度和 serif 特性的字型。 有五種字型家族。 第六個系列可讓應用程式使用預設字型。 下表說明字型系列。



| 字型家族 | Description                                                                                                                                   |
|-------------|-----------------------------------------------------------------------------------------------------------------------------------------------|
| 裝飾  | 指定新奇字型。 例如，舊的英文。                                                                                          |
| Dontcare    | 指定泛型系列名稱。 當字型的相關資訊不存在或不重要時，就會使用此名稱。 使用預設字型。 |
| 新式      | 指定具有或不具有襯線的等寬字型。 等寬字型通常是新式的;範例包括十二點活字、精英和新新的。         |
| 羅馬       | 指定具有襯線的比例字型。 這是新的羅馬時間範例。                                                                     |
| 指令碼      | 指定設計來看起來像手寫的字型;範例包括腳本和草書。                                              |
| 瑞士       | 指定沒有襯線的比例字型。 例如，Arial。                                                                            |



 

這些字型系列對應于 Wingdi .h 檔案中的常數： FF \_ 裝飾性、ff \_ DONTCARE、ff \_ 新式、FF \_ 羅馬、ff \_ 腳本和 ff \_ 瑞士。 當應用程式建立字型、選取字型或抓取字型的相關資訊時，會使用這些常數。

系列內的字型可依大小 (10 點、24點等，依此類推) 和樣式 (一般、斜體等) 。
