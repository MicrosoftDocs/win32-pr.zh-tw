# <a name="about-filled-shapes"></a>關於填滿的圖形

應用程式會針對各種不同的工作使用填滿的圖形。 例如，試算表應用程式會使用填滿的形狀來建立圖表和圖形，而且繪製和繪製應用程式會使用填滿的圖形來允許使用者繪製圖形和圖例。
