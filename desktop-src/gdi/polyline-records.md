# <a name="polyline-records"></a>聚合線條記錄

聚合[線條記錄是](/windows/desktop/api/Wingdi/nf-wingdi-polyline)一系列的點;在各點之間繪製的線條描述字元的外框。 聚合線條記錄是從上一筆記錄中的最後一個點開始 (或者，對於輪廓中的第一筆記錄而言，起始點) 。 記錄中的每個點都是在圖像大綱上，而且只要使用直線即可連接。
