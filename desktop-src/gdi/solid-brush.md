# <a name="solid-brush"></a>實心筆刷

*實心筆刷* 是包含相同色彩64圖元的邏輯筆刷。 應用程式可以藉由呼叫 [**CreateSolidBrush**](/windows/desktop/api/Wingdi/nf-wingdi-createsolidbrush) 函式來建立穩固的邏輯筆刷，以指定所需的筆刷色彩。 建立實心筆刷之後，應用程式就可以選取它的裝置內容，並用它來繪製填滿的圖形。
