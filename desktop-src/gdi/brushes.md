# <a name="brushes-windows-gdi"></a> (Windows GDI) 的筆刷

*筆刷* 是一種圖形工具，應用程式用來繪製多邊形、橢圓形和路徑的內部。 繪圖應用程式會使用筆刷繪製圖形;文字處理應用程式使用筆刷繪製規則;電腦輔助設計 (CAD) 應用程式使用筆刷來繪製跨區段視圖的內部;和試算表應用程式會使用筆刷來繪製圓形圖的區段和橫條圖中的橫條。
