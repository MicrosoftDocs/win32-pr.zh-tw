# <a name="framing-regions"></a>框架區域

應用程式會藉由呼叫 [**FrameRgn**](/windows/desktop/api/Wingdi/nf-wingdi-framergn) 函式並指定系統在繪製框架時使用的框線寬度和筆刷模式，在區域周圍繪製框線。
