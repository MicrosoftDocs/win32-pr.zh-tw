# <a name="content-of-descriptive-properties"></a>描述性屬性的內容

[**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible)介面提供描述性的屬性，描述物件的各種層面。 其中一些屬性是特定內容;其他屬性則包含由伺服器所提供的描述性文字所組成的內容。 每個屬性的資訊類型會根據物件而有所不同。

下列主題描述用戶端從這些屬性取得的資訊。 它們也提供伺服器選擇內容的建議。

-   [Name 屬性](name-property.md)
-   [Role 屬性](role-property.md)
-   [State 屬性](state-property.md)
-   [Value 屬性](value-property.md)
-   [Description 屬性](description-property.md)
-   [Help 屬性](help-property.md)
-   [HelpTopic 屬性](helptopic-property.md)
-   [KeyboardShortcut 屬性](keyboardshortcut-property.md)
-   [DefaultAction 屬性](defaultaction-property.md)

當您設計可存取的物件時，伺服器開發人員也應該參考下列主題：

-   [選擇要支援的屬性](choosing-which-properties-to-support.md)
-   [選擇描述性屬性的內容](choosing-the-content-for-descriptive-properties.md)

如需這些屬性的參數和傳回值的詳細資訊，請參閱 Microsoft Active Accessibility [C/c + + 參考](c-c---reference.md)的 [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible)一節。
