# <a name="appearstonotsupporttabbing"></a>AppearsToNotSupportTabbing

## <a name="text"></a>Text

應用程式似乎不支援 tab 鍵。

## <a name="type"></a>類型

錯誤

## <a name="description"></a>描述

應用程式不支援標準鍵盤導覽 (Tab 或 Shift + Tab) 。

由於應用程式中沒有任何專案可被探索，因此，此問題會造成依賴螢幕閱讀程式或鍵盤進行導覽的人員使用性問題。

## <a name="possible-causes"></a>可能的原因

-   應用程式的 UI 架構不支援 MSAA。
-   MSAA 不會在應用程式中的任何控制項上執行。
