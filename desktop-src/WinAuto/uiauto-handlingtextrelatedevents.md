# <a name="handling-text-related-events"></a>處理文字相關事件

本主題說明當內容變更、選取或取消選取時，以文字為基礎的控制項所引發的事件。


當文字內容變更時，以文字為基礎的控制項會引發 [**UIA \_ Text \_ TextChangedEventId**](uiauto-event-ids.md) 事件。 每次選取或取消選取文字內容時，它們都會引發 [**UIA \_ Text \_ TextSelectionChangedEventId**](uiauto-event-ids.md) 事件。 如需詳細資訊，請參閱 [消費者介面自動化事件總覽](uiauto-eventsoverview.md) 和 [訂閱消費者介面自動化事件](uiauto-eventsforclients.md)。
