# <a name="methodreturnedunexpectedhresult"></a>MethodReturnedUnexpectedHResult

## <a name="text"></a>Text

{0} 傳回 {1} ，但預期的傳回值應為 {2}

## <a name="type"></a>類型

警告

## <a name="description"></a>描述

嘗試取出專案的所有子系時，傳回未預期的 HRESULT。

## <a name="possible-causes"></a>可能的原因

不正確或不正確 MSAA 執行。
