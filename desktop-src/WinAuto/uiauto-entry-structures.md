# <a name="ui-automation-structures"></a>消費者介面自動化結構

本節說明消費者介面自動化用戶端應用程式和 Microsoft Win32 的消費者介面自動化提供者所使用的結構。

## <a name="in-this-section"></a>本節內容



| 結構                                                                            | Description                                                                                   |
|--------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [**ExtendedProperty**](/windows/desktop/api/UIAutomationClient/ns-uiautomationclient-extendedproperty)<br/>                       | 包含擴充屬性的相關資訊。 <br/>                                  |
| [**UIAutomationEventInfo**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiautomationeventinfo)<br/>       | 包含自訂事件的相關資訊。<br/>                                         |
| [**UIAutomationMethodInfo**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiautomationmethodinfo)<br/>     | 包含自訂控制項模式所支援之方法的相關資訊。<br/> |
| [**UIAutomationParameter**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiautomationparameter)<br/>       | 包含自訂控制項模式之參數的相關資訊。<br/>                |
| [**UIAutomationPatternInfo**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiautomationpatterninfo)<br/>   | 包含自訂控制項模式的相關資訊。<br/>                               |
| [**UIAutomationPropertyInfo**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiautomationpropertyinfo)<br/> | 包含自訂屬性的相關資訊。<br/>                                      |
| [**UiaChangeInfo**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiachangeinfo)<br/>                                    | 包含發生之消費者介面自動化變更的相關資料。<br/>                          |
| [**UiaPoint**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiapoint)<br/>                                 | 包含點的座標。 <br/>                                              |
| [**UiaRect**](/windows/desktop/api/UIAutomationCore/ns-uiautomationcore-uiarect)<br/>                                   | 包含矩形的位置和大小（以螢幕座標表示）。<br/>              |
