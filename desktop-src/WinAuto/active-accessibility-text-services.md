# <a name="active-accessibility-text-services"></a>Active Accessibility 文字服務

本節討論 Active Accessibility 的文字服務介面。

> [!Note]  
> Active Accessibility 的文字服務已淘汰。 如需有關 advanced Text 輸入和自然語言技術的詳細資訊，請參閱 [Microsoft Windows 文字服務架構](../tsf/text-services-framework.md) 。

## <a name="in-this-section"></a>本節內容

| 主題                                                     | 描述                                                                                                                    |
|-----------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| [**IAccServerDocMgr**](/windows/desktop/api/msaatext/nn-msaatext-iaccserverdocmgr)   | 公開可讓用戶端應用程式存取檔的方法。                              |
| [**IAccClientDocMgr**](/windows/desktop/api/msaatext/nn-msaatext-iaccclientdocmgr)   | 公開用戶端應用程式取出檔的方法。                                      |
| [**IAccDictionary**](/windows/desktop/api/msaatext/nn-msaatext-iaccdictionary)       | 公開字串操作的方法。                                                            |
| [**ICoCreateLocally**](/windows/desktop/api/msaatext/nn-msaatext-icocreatelocally)   | 公開方法，可讓用戶端應用程式在伺服器內容中建立 helper 物件。 |
| [**ICoCreatedLocally**](/windows/desktop/api/msaatext/nn-msaatext-icocreatedlocally) | 公開方法，以傳回本機物件的相關資訊。                                        |
| [**IVersionInfo**](/windows/desktop/api/msaatext/nn-msaatext-iversioninfo)           | 公開提供可存取元素之版本資訊的方法。                            |
