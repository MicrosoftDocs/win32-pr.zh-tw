# <a name="event-handling-interfaces-for-clients"></a>用戶端的事件處理介面

本節說明非受控消費者介面自動化用戶端應用程式的事件處理介面。

## <a name="in-this-section"></a>本節內容



| 介面                                                                                                              | 描述                                                                                                                      |
|------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------|
| [**IUIAutomationChangesEventHandler**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationchangeseventhandler)<br/>                         | 公開方法，以處理變更屬性時所發生的 Microsoft 消費者介面自動化事件。<br/>                      |
| [**IUIAutomationEventHandler**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationeventhandler)<br/>                                       | 公開處理消費者介面自動化事件的方法。<br/>                                                                      |
| [**IUIAutomationFocusChangedEventHandler**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationfocuschangedeventhandler)<br/>               | 公開方法，以處理鍵盤焦點移至另一個消費者介面自動化專案時所引發的事件。<br/>     |
| [**IUIAutomationNotificationEventHandler**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationnotificationeventhandler)<br/>               | 公開處理消費者介面自動化通知事件的方法。<br/>                                                         |
| [**IUIAutomationPropertyChangedEventHandler**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationpropertychangedeventhandler)<br/>         | 公開方法，以處理屬性變更時所發生的消費者介面自動化事件。<br/>                                |
| [**IUIAutomationStructureChangedEventHandler**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationstructurechangedeventhandler)<br/>       | 公開方法，以處理消費者介面自動化樹狀結構變更時所發生的事件。<br/>                        |
| [**IUIAutomationTextEditTextChangedEventHandler**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextedittextchangedeventhandler)<br/> | 公開方法，以處理當消費者介面自動化從文字編輯控制項報告文字變更的事件時所發生的事件。<br/> |
