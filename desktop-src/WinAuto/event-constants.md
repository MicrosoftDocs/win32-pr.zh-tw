# <a name="event-constants-winuserh"></a> (Winuser 的事件常數) 

本主題說明由作業系統和伺服器應用程式所產生的事件。 這些常數會依字母順序列出。

在使用這些事件之前，用戶端應用程式應該使用 [可存取的事件](accessible-event-watcher.md) 監看員來確認 UI 元素是否使用這些事件。

如需有關一般事件的詳細資訊，請參閱 [什麼是 WinEvents？](what-are-winevents.md) 和 [系統層級和物件層級事件](system-level-and-object-level-events.md)。 如需系統所傳送之事件的詳細資訊，請參閱 [附錄 A：支援的消費者介面元素參考](appendix-a--supported-user-interface-elements-reference.md)。
