# <a name="labeling-owner-drawn-controls"></a>Owner-Drawn 控制項加上標籤

應用程式開發人員應該針對所有主控描繪控制項定義標籤（也稱為標題文字），即使在螢幕上看不到標籤。 標籤必須緊接在其標記的控制項之前，而且必須在定位順序中的下一個專案。

例如，如果您在建立主控描繪按鈕時定義標籤，其中顯示圖形而非一般的標題文字，用戶端應用程式會取得標籤，並用它來識別控制項。
