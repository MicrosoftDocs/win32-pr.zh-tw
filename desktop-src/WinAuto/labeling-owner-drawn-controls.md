# <a name="labeling-owner-drawn-controls"></a><span data-ttu-id="2b53c-103">Owner-Drawn 控制項加上標籤</span><span class="sxs-lookup"><span data-stu-id="2b53c-103">Labeling Owner-Drawn Controls</span></span>

<span data-ttu-id="2b53c-104">應用程式開發人員應該針對所有主控描繪控制項定義標籤（也稱為標題文字），即使在螢幕上看不到標籤。</span><span class="sxs-lookup"><span data-stu-id="2b53c-104">Application developers should define a label, also known as caption text, for all owner-drawn controls, even if the label is not visible on the screen.</span></span> <span data-ttu-id="2b53c-105">標籤必須緊接在其標記的控制項之前，而且必須在定位順序中的下一個專案。</span><span class="sxs-lookup"><span data-stu-id="2b53c-105">The label must come immediately before the control that it is labeling, and must be in the next item in the tab order.</span></span>

<span data-ttu-id="2b53c-106">例如，如果您在建立主控描繪按鈕時定義標籤，其中顯示圖形而非一般的標題文字，用戶端應用程式會取得標籤，並用它來識別控制項。</span><span class="sxs-lookup"><span data-stu-id="2b53c-106">For example, if you define a label when creating an owner-drawn button in which a graphic is displayed instead of the normal caption text, client applications get the label and use it to identify the control.</span></span>
