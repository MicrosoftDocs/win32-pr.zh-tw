# <a name="unsupported-user-interface-elements"></a>不支援的消費者介面元素

Microsoft Active Accessibility 不支援下表所列的通用控制項。



| 控制項名稱                  | 視窗類別名稱   |
|-------------------------------|---------------------|
| 日期和時間選擇器控制項 | DATETIMEPICK \_ 類別 |
| 呼機控制項                | WC \_ PAGESCROLLER    |
| Rebar 控制項                | REBARCLASSNAME      |



 

Microsoft Active Accessibility 不支援下表所列的預先定義控制項和視窗元素。



| 視窗類別名稱 | 描述                                                 |
|-------------------|-------------------------------------------------------------|
| DDEMLEvent        | Windows NT： DDEML 事件的類別                      |
| 訊息           | Windows 2000 和更新版本：僅限訊息視窗的類別 |
| \#32772           | Windows NT：圖示標題的類別                       |
