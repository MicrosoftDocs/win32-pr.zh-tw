# <a name="elementshouldbeoffscreen"></a>ElementShouldBeOffScreen

## <a name="text"></a>Text

元素應該在螢幕上

## <a name="type"></a>類型

警告

## <a name="description"></a>描述

專案的周框會由其父元素的周框矩形裁剪。 應忽略元素，並將其視為超出桌面的範圍。
