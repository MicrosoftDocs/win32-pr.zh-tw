# <a name="taborderinquestion"></a>TabOrderInQuestion

## <a name="text"></a>Text

定位順序可能未遵循標準 Windows 指導方針

## <a name="type"></a>類型

資訊

## <a name="description"></a>Description

使用標準鍵盤流覽 (Tab 或 Shift + Tab) 會公開非標準的定位順序。 例如，根據本機系統設定，定位順序通常會模仿讀取順序。

這個問題會讓依賴螢幕讀取器和鍵盤進行流覽的人員遇到問題，因為進行中的專案可能不穩定且無法預測。
