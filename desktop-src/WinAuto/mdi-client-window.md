<span data-ttu-id="6977f-105">多重文件介面 (MDI) 是一種規格，可定義針對 Windows 撰寫之應用程式的標準使用者介面。</span><span class="sxs-lookup"><span data-stu-id="6977f-105">The multiple document interface (MDI) is a specification that defines the standard user interface for applications written for Windows.</span></span> <span data-ttu-id="6977f-106">MDI 應用程式可讓使用者一次使用一個以上的檔。</span><span class="sxs-lookup"><span data-stu-id="6977f-106">An MDI application enables a user to work with more than one document at a time.</span></span>
