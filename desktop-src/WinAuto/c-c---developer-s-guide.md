# <a name="cc-developers-guide"></a>C/c + + 開發人員指南

本節提供從 C 或 c + + 應用程式開發人員的觀點來看，Microsoft Active Accessibility 如何運作的相關資訊。 它涵蓋了用戶端和伺服器開發人員的特定問題。

請注意， *用戶端* 是協助工具輔助或其他應用程式，使用 Microsoft Active Accessibility 來取得使用者介面元素的相關資訊。 *伺服器* 是一種應用程式，它會使用 Microsoft Active Accessibility 來提供使用者介面元素的相關資訊。
