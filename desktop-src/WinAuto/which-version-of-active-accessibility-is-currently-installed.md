# <a name="determine-the-installed-version-of-active-accessibility"></a>判斷 Active Accessibility 的安裝版本

若要找出您的系統上安裝的 Microsoft Active Accessibility (MSAA) 版本，您應該在硬碟的 system32 資料夾中檢查 Oleacc.dll 檔案的版本。 下表將 Microsoft Active Accessibility 版本對應到 Oleacc.dll 版本。

| Active Accessibility 版本     | Oleacc.dll 版本                                                                       |
|----------------------------------|------------------------------------------------------------------------------------------|
| 1.0                              | 4.01.0970.0                                                                              |
| 1.2                              | 4.02.2900.0                                                                              |
| 1.3                              | 4.02.3100.0                                                                              |
| 2.0                              | 5.0.0.0                                                                                  |
| 3.0 (Windows 自動化 API 3.0)  | 7.0. x. y，其中 *x* 是組建編號，而 *y* 是 (快速修正工程) QFE 號碼 |
