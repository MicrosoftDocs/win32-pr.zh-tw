# <a name="common-infrastructure"></a>一般基礎結構

Windows 提供兩種適用于使用者介面協助工具和軟體測試自動化的 API 規格： Microsoft Active Accessibility 和 Microsoft 消費者介面自動化。 本節說明 Microsoft Active Accessibility 和消費者介面自動化兩者通用的功能和元件，並讓它們可相交互操作。
