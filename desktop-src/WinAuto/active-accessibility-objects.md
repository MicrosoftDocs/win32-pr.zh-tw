# <a name="active-accessibility-objects"></a>Active Accessibility 物件

在 Microsoft Active Accessibility 術語中，有 *可存取的物件* 和 *簡單的元素*。 雖然大部分的應用程式都包含這兩個應用程式，但可存取的物件比簡單元素更常見 本節定義和討論可存取的物件和簡單的元素，並提供適當的 UI 範例。
