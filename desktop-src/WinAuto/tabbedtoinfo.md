# <a name="tabbedtoinfo"></a>TabbedToInfo

## <a name="text"></a>Text

**訊息1**

擲回額外的焦點事件

**訊息2**

回到開頭處

## <a name="type"></a>類型

資訊

## <a name="description"></a>Description

**訊息1**

使用標準鍵盤流覽 (Tab 或 Shift + Tab) 時，單一元素會引發多個焦點事件。 會捨棄偽造的事件。

**訊息2**

焦點會改回定位順序中的起始元素。 因為定位順序顯示為圓形，所以在這個元素上已終止定位鍵。
