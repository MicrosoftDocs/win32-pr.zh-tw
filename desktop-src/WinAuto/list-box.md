# <a name="list-box-msaa-ui-element-reference"></a>清單方塊 (MSAA UI 元素參考) 

> [!Note]  
> 本主題描述用於 MSAA UI 專案參考之 **清單方塊** 物件的用途。 此處未說明如何在各種 UI 架構中建立 **清單方塊** 物件。 請參閱您所使用之 UI 架構的 API 參考檔。

 

清單方塊會顯示一個清單，使用者可以從中選取一或多個專案。 清單方塊會使用垂直捲動條、水準控制列或兩者來建立。

清單方塊的視窗類別名稱是「LISTBOX」。

## <a name="iaccessible-methods"></a>IAccessible 方法

清單方塊支援下列 [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) 方法：



| 方法                                                                    | 註解                                                                                                                                                                                                                                                                   |
|---------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**accDoDefaultAction**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accdodefaultaction) | 清單方塊物件本身沒有 **DefaultAction** 屬性;不過， [**accDoDefaultAction**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accdodefaultaction) 按兩下清單方塊專案。                                                                                    |
| [**accHitTest**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acchittest)                 | 如果測試的點在清單方塊中，而不是在清單方塊中的專案上，則 [**accHitTest**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acchittest) 方法會傳回零。 如果測試的點位於清單方塊中的專案上， **accHitTest** 會傳回清單方塊專案的索引。 |
| [**accLocation**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acclocation)               |                                                                                                                                                                                                                                                                            |
| [**accNavigate**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accnavigate)               |                                                                                                                                                                                                                                                                            |
| [**accSelect**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accselect)                   |                                                                                                                                                                                                                                                                            |
