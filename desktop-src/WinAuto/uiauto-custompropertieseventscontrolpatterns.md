# <a name="custom-properties-events-and-control-patterns"></a>自訂屬性、事件和控制項模式

Microsoft 消費者介面自動化 API 會指定一組預先定義的核心屬性、控制項模式和事件。 不過，應用程式並不限於使用這些預先定義的規格。 消費者介面自動化的擴充性功能可讓協力廠商引進自訂、相互同意的屬性、事件和控制項模式，以支援新的 UI 元素和應用程式案例。 消費者介面自動化提供者和用戶端可以立即開始使用自訂屬性、事件和控制模式，而不需要更新核心消費者介面自動化架構。
