# <a name="accessibleobjectfrompointreturnednullchildid"></a>AccessibleObjectFromPointReturnedNullChildId

## <a name="text"></a>Text

AccessibleObjectFromPoint ({0} ， {1}) 傳回 null ChildId

## <a name="type"></a>類型

錯誤

## <a name="description"></a>描述

針對指定座標取得的物件 [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) 介面 CHILDID 為 Null。

## <a name="possible-causes"></a>可能的原因

驗證期間的使用者互動，例如將焦點移至非目標 HWND，並干擾驗證程式。
