# <a name="direct-annotation"></a>直接注釋

本節提供使用直接注釋的相關資訊。

## <a name="summary"></a>總結

直接注釋是動態注釋最簡單的形式。 它是用來覆寫可存取專案的一或多個屬性值。 它最適合用於可存取的專案，其標注屬性不會相依于控制項的狀態，也不需要由 [值對應注釋](value-map-annotation.md) 和 [伺服器批註](server-annotation.md)提供的額外支援。

## <a name="when-to-use-this-technique"></a>使用這項技術的時機

當您想要覆寫一個或多個可存取專案的屬性值時，請使用這項技術。

如需詳細資訊，請參閱 [使用直接批註](using-direct-annotation.md)。
