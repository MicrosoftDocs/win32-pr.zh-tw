# <a name="ui-automation-fundamentals"></a>UI 自動化基礎觀念

Microsoft 消費者介面自動化可讓輔助技術應用程式和自動化測試控管與其他應用程式的 UI 控制項互動。 本節說明消費者介面自動化所依據的基本概念。

消費者介面自動化 API 分為兩個部分。 *消費者介面自動化提供者* 應用程式會使用一個部分，而另一個部分則由 *消費者介面自動化用戶端* 應用程式使用。 提供者 API 可讓 Microsoft Win32 自訂控制項和其他控制項架構的開發人員將這些控制項公開給消費者介面自動化，並讓用戶端應用程式看見這些控制項。 用戶端 API 可讓應用程式與其他應用程式中的控制項互動，並取得其相關資訊。
