# <a name="boundingrectnothittestable"></a>BoundingRectNotHitTestable

## <a name="text"></a>Text

報告的矩形似乎未包含可使用點擊測試來抓取此元素的點。

## <a name="type"></a>類型

警告

## <a name="description"></a>描述

點擊測試無法連接元素的周框。 例如，它是在實體畫面之外，可 (或可查看的區域) 。
