# <a name="gatheringelement"></a>GatheringElement

## <a name="text"></a>Text

已收集這個元素

## <a name="type"></a>類型

警告

## <a name="description"></a>描述

此訊息不是警告或錯誤，且不需要任何動作。 它會指出元素的資訊已收集並儲存在記錄檔中。
