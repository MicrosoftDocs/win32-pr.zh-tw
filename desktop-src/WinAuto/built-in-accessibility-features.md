# <a name="built-in-accessibility-features"></a>內建協助工具功能

內建的協助工具功能是作業系統的延伸模組。 無論正在執行哪一種應用程式，這些功能都會影響系統的行為。

Win32 支援下列內建的協助工具功能。



| 功能       | 描述                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
|---------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| AccessTimeout | 可讓使用者指定在經過一段時間後，系統會自動停用系統範圍的協助工具功能。 AccessTimeout 功能適用于多個使用者使用不同喜好設定所共用的電腦。 每個人都可以使用主控台中的快速鍵或輕鬆存取中心來啟用慣用的功能。 存取範圍超時期限是在作業系統自動關閉協助工具功能之前，必須經過鍵盤和滑鼠輸入才能傳遞的時間長度。 受時間影響的協助工具功能是「篩選器」、「滑鼠鍵」、「相黏鍵」、「切換器」和「 [高對比](high-contrast-parameter.md)」。 [**ACCESSTIMEOUT**](/windows/win32/api/winuser/ns-winuser-accesstimeout)結構會定義這項功能的參數。 |
| L    | 啟用鍵盤屬性的控制，例如輸入接受按鍵之前的時間量，以及按鍵開始重複之前的時間量。 篩選器功能也會在按下按鍵以及接受為輸入時，提供音效的意見反應。 [**篩選篩選**](/windows/win32/api/winuser/ns-winuser-filterkeys)的結構會定義這項功能的參數。                                                                                                                                                                                                                                                                                                                                                                                                                            |
| 方向鍵     | 可讓使用者使用數位鍵台控制滑鼠指標。 [**滑鼠鍵**](/windows/win32/api/winuser/ns-winuser-mousekeys)結構會定義這項功能的參數。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| R    | 可讓使用者使用附加至電腦序列埠的替代輸入裝置來提供輸入。 這項功能是專為無法使用電腦標準鍵盤和滑鼠的使用者所設計。 [**序列鍵**](/windows/win32/api/winuser/ns-winuser-serialkeysa)結構會定義此功能的參數。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| 衛士   | 當以 Microsoft Win32 為基礎的應用程式或在視窗中執行的 Microsoft MS-DOS 應用程式產生音效時，會顯示視覺信號。 「 [**聲音感測**](/windows/win32/api/winuser/ns-winuser-soundsentrya) 」結構會定義這項功能的參數，包括 [*SoundSentryProc*](/previous-versions/windows/desktop/legacy/dd373647(v=vs.85)) 回呼函式的位址。                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| 粘連    | 讓使用者可以依照順序（而不是同時）輸入按鍵組合（例如 CTRL + ALT + DEL）。 [**相黏鍵**](/windows/win32/api/winuser/ns-winuser-stickykeys)結構會定義這項功能的參數。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| $    | 當使用者開啟或關閉 CAPS LOCK、NUM LOCK 或 SCROLL LOCK 鍵時，提供音效的意見反應。 [**切換**](/windows/win32/api/winuser/ns-winuser-togglekeys)參數結構會定義這項功能的參數。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |



 

[**SystemParametersInfo**](/windows/desktop/api/winuser/nf-winuser-systemparametersinfoa)函式會提供內建協助工具功能的程式設計介面。 這項功能在主控台應用程式或其他允許使用者自訂環境的程式中很有用。 **SystemParametersInfo** 函式可以執行下列工作：

-   查詢系統以判斷指定內建協助工具功能的可用性和目前狀態。
-   啟用功能並指定控制其行為的參數。
-   停用功能。
-   啟用可讓使用者啟用或停用功能的快速鍵。

針對每個內建功能，Win32 會定義應用程式搭配 [**SystemParametersInfo**](/windows/desktop/api/winuser/nf-winuser-systemparametersinfoa) 使用的結構，以查詢或設定功能的參數。

應用程式可以使用 [**SystemParametersInfo**](/windows/desktop/api/winuser/nf-winuser-systemparametersinfoa) 來直接啟用或停用功能。 針對某些功能， **SystemParametersInfo** 可以啟用快速鍵或按鍵組合，讓使用者可以切換功能的狀態。 串列按鍵或聲音感測功能沒有快速鍵，或針對 [聲音標示] 參數。 下列協助工具功能有快速鍵。



| 功能                                           | 描述                                                                                                                                                                                                                                                                                                                                                                                                                                              |
|---------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| L                                        | 按住右邊的 SHIFT 鍵八秒，即可開啟或關閉篩選篩選功能。 將右 SHIFT 鍵按住12秒，可針對 [**篩選篩選**](/windows/win32/api/winuser/ns-winuser-filterkeys)結構的 **iBounceMSec** 成員使用最保守的值來進行篩選。 按住右 SHIFT 鍵16秒，即可使用 **篩選篩選篩選** 的 **iWaitMSec** 成員最保守的值來進行篩選。 |
| 方向鍵                                         | 按左 ALT + 左 SHIFT + NUM LOCK。                                                                                                                                                                                                                                                                                                                                                                                                                  |
| 粘連                                        | 按 SHIFT 鍵5次。                                                                                                                                                                                                                                                                                                                                                                                                                             |
| $                                        | 按住 NUM LOCK 鍵5秒。                                                                                                                                                                                                                                                                                                                                                                                                                |
| [高對比模式](high-contrast-parameter.md) | 按左 ALT + 左 SHIFT + PRINT SCREEN。                                                                                                                                                                                                                                                                                                                                                                                                              |
