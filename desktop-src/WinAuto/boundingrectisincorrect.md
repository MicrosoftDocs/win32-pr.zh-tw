# <a name="boundingrectisincorrect"></a>BoundingRectIsIncorrect

## <a name="text"></a>Text

{0}報表矩形的邊緣與矩形的點擊測試點不一致 {1} {2} 。

## <a name="type"></a>類型

錯誤

## <a name="description"></a>描述

專案的周框矩形與預期的點擊測試結果不一致。 例如，在專案的周框外面的點進行點擊測試時，不會傳回不同的元素。
