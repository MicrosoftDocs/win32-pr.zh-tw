# <a name="methodexceptionoccured"></a>MethodExceptionOccured

## <a name="text"></a>Text

發生例外狀況 \\ " {0} \\ "

## <a name="type"></a>類型

錯誤

## <a name="description"></a>描述

遇到未預期的例外狀況或 HRESULT。 某些例外狀況/HRESULT 是有效的，但在此情況下，不需要結果。 應修正此錯誤。
