# <a name="dynamic-annotation"></a>動態注釋

動態批註先前僅供 Microsoft Active Accessibility 伺服器和用戶端使用。 這項技術現在可供所有消費者介面自動化用戶端使用，以及一些 MSAA 伺服器和消費者介面自動化提供者的實現。 此外，使用動態注釋對自訂控制項所進行的批註 (所有三種技術) 都可以消費者介面自動化用戶端存取。

如需詳細資訊，請參閱 [動態批註 API](dynamic-annotation-api.md)。
