# <a name="header-and-lib-files"></a>標頭檔和 Lib 檔案

下列標頭檔和程式庫檔案支援 Microsoft Active Accessibility 應用程式的開發。



| 檔案         | 描述                                                                                                                                                                  |
|--------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Winuser。h    | 使用 Windows 控制項的 c 和 c + + 標頭檔                                                                                                                             |
| Oleacc。h     | C 和 c + + 標頭檔，包含 Microsoft Active Accessibility 用戶端和伺服器所需的定義                                                                    |
| Winver。h     | C 和 c + + 標頭檔中的版本資訊                                                                                                                                |
| Oleacc .lib   | Oleacc.dll 的匯入程式庫                                                                                                                                                |
| Oleacc.dll   | 動態連結程式庫，可提供 Microsoft Active Accessibility 執行時間、管理來自 Microsoft Active Accessibility 用戶端的要求，以及 proxy 通用 UI 元素 |
| Oleaccrc.dll | Oleacc.dll 使用的多語系資源動態連結程式庫                                                                                                                |
