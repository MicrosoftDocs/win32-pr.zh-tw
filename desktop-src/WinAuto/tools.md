# <a name="tools"></a>工具

您可以使用下列工具來測試 Microsoft Active Accessibility 的執行。 如需測試控管的詳細資訊，請參閱 [測試控管](testing-tools.md)。



| 工具                                                     | 描述                                                                                                                                                                           |
|----------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [協助工具事件監控程式](accessible-event-watcher.md) | 記錄在系統內發生的 [WinEvents](winevents-infrastructure.md) 。                                                                                                         |
| [檢查](inspect-objects.md)                           | 識別在 Microsoft Active Accessibility 的可存取物件，以及在 Microsoft 消費者介面自動化的自動化元素中，執行屬性和方法的問題。 |
