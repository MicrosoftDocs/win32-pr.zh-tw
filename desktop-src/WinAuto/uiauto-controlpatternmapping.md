# <a name="control-types-and-their-supported-control-patterns"></a><span data-ttu-id="4d0ea-111">控制項類型及其支援的控制項模式</span><span class="sxs-lookup"><span data-stu-id="4d0ea-111">Control Types and Their Supported Control Patterns</span></span>

<span data-ttu-id="4d0ea-112">用戶端應用程式會透過 [**IUIAutomationElement**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationelement) 介面取得一般屬性，以及藉由控制項所支援的控制項模式介面來取得特定類型的屬性，藉以取得控制項的相關資訊。</span><span class="sxs-lookup"><span data-stu-id="4d0ea-112">Client applications get information about a control by retrieving general properties through the [**IUIAutomationElement**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationelement) interface, and by retrieving type-specific properties through the control pattern interfaces that the control supports.</span></span> <span data-ttu-id="4d0ea-113">本主題列出 Microsoft 消費者介面自動化控制項類型，並識別每個類型支援的控制項模式。</span><span class="sxs-lookup"><span data-stu-id="4d0ea-113">This topic lists the Microsoft UI Automation control types and identifies the control patterns that each type supports.</span></span>

<span data-ttu-id="4d0ea-114">下表將控制項模式整理為下列類別：</span><span class="sxs-lookup"><span data-stu-id="4d0ea-114">The following table organizes the control patterns into the following categories:</span></span>

-   <span data-ttu-id="4d0ea-115">必須支援。</span><span class="sxs-lookup"><span data-stu-id="4d0ea-115">Must support.</span></span> <span data-ttu-id="4d0ea-116">控制項必定支援此控制項模式。</span><span class="sxs-lookup"><span data-stu-id="4d0ea-116">The control must support this control pattern.</span></span>
-   <span data-ttu-id="4d0ea-117">有條件支援。</span><span class="sxs-lookup"><span data-stu-id="4d0ea-117">Conditional support.</span></span> <span data-ttu-id="4d0ea-118">控制項可依據控制項的狀態決定是否支援此控制項模式。</span><span class="sxs-lookup"><span data-stu-id="4d0ea-118">The control may support this control pattern depending on the state of the control.</span></span>
-   <span data-ttu-id="4d0ea-119">不支援。</span><span class="sxs-lookup"><span data-stu-id="4d0ea-119">Does not support.</span></span> <span data-ttu-id="4d0ea-120">控制項不支援此控制項模式;不過，自訂版本的控制項可能會支援此控制項模式。</span><span class="sxs-lookup"><span data-stu-id="4d0ea-120">The control does not support this control pattern; however, a customized version of the control may support this control pattern.</span></span>

<span data-ttu-id="4d0ea-121">根據控制項的功能而定，某些控制項有數種控制項模式的條件支援。</span><span class="sxs-lookup"><span data-stu-id="4d0ea-121">Some controls have conditional support for several control patterns, depending on the functionality of the control.</span></span> <span data-ttu-id="4d0ea-122">例如，功能表項目控制項具有 [Invoke](uiauto-implementinginvoke.md)、 [ExpandCollapse](uiauto-implementingexpandcollapse.md)、 [切換](uiauto-implementingtoggle.md)或 [SelectionItem](uiauto-implementingselectionitem.md) 控制項模式的條件支援，視其在功能表控制項中的功能而定。</span><span class="sxs-lookup"><span data-stu-id="4d0ea-122">For example, the menu item control has conditional support for the [Invoke](uiauto-implementinginvoke.md), [ExpandCollapse](uiauto-implementingexpandcollapse.md), [Toggle](uiauto-implementingtoggle.md), or [SelectionItem](uiauto-implementingselectionitem.md) control pattern, depending on its function in the menu control.</span></span>
