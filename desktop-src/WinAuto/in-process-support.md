# <a name="in-process-support"></a>In-Process 支援

動態注釋的目前執行完全是同進程，因此只允許擁有這些 UI 元素的進程標注 UI 專案。 一個進程不可能對另一個進程的 UI 元素加上批註。

請注意，這只會影響設定注釋的動作;它不會干擾用戶端存取屬性 (批註或其他進程中的 UI 元素) 的能力。
