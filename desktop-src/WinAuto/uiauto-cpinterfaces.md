# <a name="control-pattern-interfaces-for-providers"></a>提供者的控制項模式介面

本節說明由消費者介面自動化提供者所執行的 *控制項模式* 介面，以公開非受控 Windows 應用程式中控制項的相關資訊。

## <a name="in-this-section"></a>本節內容



| 介面                                                                          | 描述                                                                                                                                                                                                                                                                                                         |
|------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IAnnotationProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-iannotationprovider)<br/>           | 公開檔中批註的屬性。<br/>                                                                                                                                                                                                                                                   |
| [**IDockProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-idockprovider)<br/>                           | 提供對停駐容器中專案的存取權。 <br/>                                                                                                                                                                                                                                                   |
| [**IDragProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-idragprovider)<br/>                           | 讓 Microsoft 消費者介面自動化專案將自身描述為可拖曳為拖放作業一部分的元素。<br/>                                                                                                                                                                     |
| [**IDropTargetProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-idroptargetprovider)<br/>               | 啟用消費者介面自動化專案，將其描述為可在消費者介面自動化拖放作業中接收拖曳專案卸載的元素。<br/>                                                                                                                                        |
| [**IExpandCollapseProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iexpandcollapseprovider)<br/>       | 提供控制項的存取權，以視覺化方式展開以顯示內容，並折迭以隱藏內容。 <br/>                                                                                                                                                                                                   |
| [**IGridItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-igriditemprovider)<br/>                   | 提供可讓您存取執行 [**IGridProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-igridprovider)之容器的個別子控制項。<br/>                                                                                                                                                                                 |
| [**IGridProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-igridprovider)<br/>                           | 提供控制項的存取權，這些控制項的作用如同以二維邏輯座標系統組織的子專案集合的容器， (也就是，消費者介面自動化用戶端可以使用鍵盤移至連續的控制項) 。<br/>                                              |
| [**IInvokeProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iinvokeprovider)<br/>                       | 提供控制項的存取權，這些控制項可起始或執行單一、明確的動作，而不會在啟用時維護狀態。<br/>                                                                                                                                                                              |
| [**IItemContainerProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iitemcontainerprovider)<br/>         | 提供可作為其他控制項容器（例如虛擬清單-view）之控制項的存取權。 <br/>                                                                                                                                                                                                      |
| [**ILegacyIAccessibleProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-ilegacyiaccessibleprovider)<br/> | 可讓消費者介面自動化用戶端存取 Microsoft Active Accessibility 專案的基礎 [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) 執行。<br/>                                                                                                                                                     |
| [**IMultipleViewProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-imultipleviewprovider)<br/>           | 提供對控制項的存取，這些控制項可以在同一組資訊或子控制項的多種標記法之間切換。<br/>                                                                                                                                                     |
| [**IObjectModelProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-iobjectmodelprovider)<br/>         | 提供由控制項或應用程式所執行之基礎物件模型的存取。 <br/>                                                                                                                                                                                                                 |
| [**IRangeValueProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irangevalueprovider)<br/>               | 提供可設定為範圍內某個值之控制項的存取權。 <br/>                                                                                                                                                                                                                                  |
| [**IScrollItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iscrollitemprovider)<br/>               | 提供可讓您存取執行 [**IScrollProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iscrollprovider)之容器的個別子控制項。 <br/>                                                                                                                                                                            |
| [**IScrollProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iscrollprovider)<br/>                       | 提供控制項的存取權，這些控制項可作為子物件集合的可滾動的容器。<br/>                                                                                                                                                                                                         |
| [**ISelectionItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iselectionitemprovider)<br/>         | 提供可供執行 [**ISelectionProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iselectionprovider)之容器的個別可選取子控制項的存取權。 <br/>                                                                                                                                                          |
| [**ISelectionProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iselectionprovider)<br/>                 | 提供控制項的存取權，這些控制項可作為個別、可選取之子專案集合的容器。<br/>                                                                                                                                                                                               |
| [**ISelectionProvider2**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iselectionprovider2)<br/>               | 擴充 [**ISelectionItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iselectionitemprovider) 介面，以提供所選取專案的相關資訊。<br/>                                                                                                                                                                       |
| [**ISpreadsheetItemProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-ispreadsheetitemprovider)<br/> | 提供試算表中 (資料格) 之專案相關資訊的存取權。 <br/>                                                                                                                                                                                                                                   |
| [**ISpreadsheetProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-ispreadsheetprovider)<br/>         | 可存取試算表中 (資料格) 的專案。<br/>                                                                                                                                                                                                                                                       |
| [**IStylesProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-istylesprovider)<br/>                   | 提供與檔內容相關聯之視覺化樣式的存取權。<br/>                                                                                                                                                                                                                          |
| [**ISynchronizedInputProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-isynchronizedinputprovider)<br/> | 可讓消費者介面自動化用戶端應用程式將滑鼠或鍵盤輸入導向至特定的 UI 元素。 <br/>                                                                                                                                                                                               |
| [**ITableItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itableitemprovider)<br/>                 | 提供可讓您存取執行 [**ITableProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itableprovider)之容器的子控制項。 <br/>                                                                                                                                                                                         |
| [**ITableProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itableprovider)<br/>                         | 提供控制項的存取權，這些控制項可作為子項目集合的容器。 這個專案的子系必須執行 [**ITableItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itableitemprovider) ，並在可以使用鍵盤進行往返的二維邏輯座標系統中進行組織。 <br/> |
| [**ITextChildProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-itextchildprovider)<br/>            | 提供以文字為基礎的控制項 (或內嵌于文字) 的物件（也就是另一個以文字為基礎之控制項的子系或子系）的存取。 <br/>                                                                                                                                                                    |
| [**ITextEditProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-itexteditprovider)<br/>               | 擴充 [**ITextProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextprovider) 介面，讓消費者介面自動化提供者可以公開程式設計的文字編輯動作。<br/>                                                                                                                                                          |
| [**ITextProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextprovider)<br/>                           | 提供對包含文字之控制項的存取。 <br/>                                                                                                                                                                                                                                                          |
| [**ITextProvider2**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-itextprovider2)<br/>                     | 擴充 [**ITextProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextprovider) 介面，讓消費者介面自動化提供者可以公開做為批註目標的文字內容，以及屬於該提供者之插入點的相關資訊。<br/>                                                                         |
| [**ITextRangeProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextrangeprovider)<br/>                 | 提供在執行 [**ITextProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextprovider) 或 [**ITextProvider2**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextprovider2)之文字容器中的連續文字範圍存取權。<br/>                                                                                                                       |
| [**ITextRangeProvider2**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextrangeprovider2)<br/>               | 擴充 [**ITextRangeProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextrangeprovider) 介面，以啟用消費者介面自動化提供者來叫用操作功能表。<br/>                                                                                                                                                                 |
| [**IToggleProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itoggleprovider)<br/>                       | 提供控制項的存取權，這些控制項可以迴圈處理一組狀態，並在設定之後維護狀態。<br/>                                                                                                                                                                                                 |
| [**ITransformProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itransformprovider)<br/>                 | 提供可在二維空間內移動、調整大小及/或旋轉之控制項的存取權。 <br/>                                                                                                                                                                                                  |
| [**ITransformProvider2**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-itransformprovider2)<br/>           | 擴充 [**ITransformProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itransformprovider) 介面，讓消費者介面自動化提供者公開屬性，以支援控制項的區縮放功能。 <br/>                                                                                                        |
| [**IValueProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-ivalueprovider)<br/>                         | 提供控制項的存取權，這些控制項的內部值不跨越某個範圍，而且可以表示為字串。 <br/>                                                                                                                                                                           |
| [**IVirtualizedItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-ivirtualizeditemprovider)<br/>     | 提供虛擬化專案的存取權，這些專案是由消費者介面自動化樹狀結構中的預留位置自動化元素所表示的專案。<br/>                                                                                                                                                                 |
| [**IWindowProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iwindowprovider)<br/>                       | 提供控制項之基礎視窗功能的存取權。<br/>                                                                                                                                                                                                                              |
