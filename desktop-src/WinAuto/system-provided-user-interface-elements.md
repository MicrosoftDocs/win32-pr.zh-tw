# <a name="system-provided-user-interface-elements"></a>System-Provided 消費者介面元素

Microsoft Active Accessibility 針對大部分預先定義和通用的控制項提供支援。 下圖顯示 Microsoft Active Accessibility 公開的一般視窗和某些系統提供的使用者介面元素，例如標題列、功能表、下拉式方塊、工具列控制項、樹狀檢視控制項、狀態列、大小掌握、清單視圖控制項和捲軸。

![顯示數個不同使用者介面元素的螢幕擷取畫面](images/explorer.gif)

Microsoft Active Accessibility 會將系統提供的使用者介面元素公開給伺服器應用程式，而不需要伺服器開發人員執行 [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) 介面。 包含這些元素的任何應用程式都會自動繼承其存取範圍。

如需 Microsoft Active Accessibility 支援的控制項和其他系統提供的使用者介面元素清單，請參閱 [附錄 a：支援的消費者介面專案參考](appendix-a--supported-user-interface-elements-reference.md)。
