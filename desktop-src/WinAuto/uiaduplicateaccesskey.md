# <a name="uiaduplicateaccesskey"></a>UiaDuplicateAccessKey

## <a name="text"></a>Text

有一個具有相同存取金鑰/快速鍵的同級自動化元素

## <a name="type"></a>類型

錯誤

## <a name="description"></a>描述

指派給元素的存取/快速鍵也會指派給驗證目標中的另一個元素。

此問題會造成依賴鍵盤進行導覽的人員發生問題，因為可能會叫用多個元素，或使用相同的快捷方式或快速鍵來迴圈焦點。
