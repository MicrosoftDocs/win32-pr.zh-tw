# <a name="interfaces-active-accessibility-user-interface-services"></a>介面 (Active Accessibility 消費者介面服務) 

本章節包含與 Microsoft Active Accessibility 搭配使用之介面的相關資訊。

## <a name="in-this-section"></a>本節內容



| 主題                                                                                | 描述                                                                                                                                                                                          |
|--------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible)<br/>                                        | 公開方法和屬性，讓用戶端應用程式可以存取使用者介面元素及其子系。<br/>                                                                     |
| [**IAccessibleWindowlessSite**](/windows/desktop/api/oleacc/nn-oleacc-iaccessiblewindowlesssite)<br/> | Microsoft ActiveX 控制項網站會執行這個介面，以啟用無視窗的 ActiveX 控制項，而該控制項有 Microsoft Active Accessibility 的執行來表達其存取範圍。 <br/> |
| [**IAccIdentity**](/windows/desktop/api/oleacc/nn-oleacc-iaccidentity)<br/>                                      | 公開方法，這個方法會提供可存取專案的唯一識別碼。<br/>                                                                                                             |
| [**IAccPropServer**](/windows/desktop/api/oleacc/nn-oleacc-iaccpropserver)<br/>                                  | 公開方法，該方法會抓取可存取專案的屬性值。<br/>                                                                                                               |
| [**IAccPropServices**](/windows/desktop/api/oleacc/nn-oleacc-iaccpropservices)<br/>                              | 公開批註可存取專案和操作識別字串的方法。 <br/>                                                                                                |
| [**IAccessibleHandler**](/windows/desktop/api/oleacc/nn-oleacc-iaccessiblehandler)<br/>                          | 公開方法，這個方法會從物件識別碼抓取可存取的專案。<br/>                                                                                                                  |
