|---------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MSAAMENUINFO**](/windows/win32/api/oleacc/ns-oleacc-msaamenuinfo)<br/>   | 由伺服器開發人員用來公開主控描繪功能表項目的名稱。<br/>                                                                                      |
| [**HWINEVENTHOOK**](hwineventhook.md)<br/> | 用來宣告視窗事件攔截函式。<br/>                                                                                                                 |
