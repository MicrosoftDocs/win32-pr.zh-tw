# <a name="constants-active-accessibility-user-interface-services"></a> (Active Accessibility 消費者介面服務的常數) 

本章節包含與 Microsoft Active Accessibility 搭配使用之常數的相關資訊。

## <a name="in-this-section"></a>本節內容



| 主題                                                               | 描述                                                                                                                                                                                                                                                                                                                                                                           |
|---------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**事件常數**](event-constants.md)<br/>               | 本主題說明由作業系統和伺服器應用程式所產生的事件。<br/>                                                                                                                                                                                                                                                                     |
| [**導覽常數**](navigation-constants.md)<br/>     | 本主題說明在 oleacc 中定義的常數值，指出 *空間* (向上、向下、向左和向右) 或 *邏輯* (第一個子系、最後一個、下一個，以及當用戶端使用 [**IAccessible：： accNavigate**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accnavigate) 從某個使用者介面專案流覽至相同容器內的另一個使用者介面專案時所觀察到的) 方向。<br/> |
| [**物件識別碼**](object-identifiers.md)<br/>         | 本主題說明 Microsoft Active Accessibility 物件識別碼、32位值，以識別視窗內可存取物件的 *類別* 。<br/>                                                                                                                                                                                                                |
| [**物件角色**](object-roles.md)<br/>                     | 本主題說明用來描述應用程式中各種 UI 物件角色的常數值。<br/>                                                                                                                                                                                                                                                               |
| [**物件狀態常數**](object-state-constants.md)<br/> | 本主題說明用來描述應用程式 UI 中物件狀態的常數值。<br/>                                                                                                                                                                                                                                                                       |
| [**SELFLAG 常數**](selflag.md)<br/>                     | 本主題說明用來指定如何選取可存取物件或取得焦點的常數值。<br/>                                                                                                                                                                                                                                                     |
