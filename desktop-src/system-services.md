# <a name="system-services"></a>系統服務

本節包含適用于桌面應用程式 Windows 所提供之系統 api 和服務的參考。 這些包括傳統可用的服務：

-   元件物件模型 (COM) 。
-   檔案壓縮。
-   動態連結程式庫。
-   記憶體管理。
-   電源管理。
-   執行多個執行緒的建立和協調。
-   服務應用程式的開發。
-   Windows 訊息。
-   取得 Windows 系統資訊。
-   說明 API。
