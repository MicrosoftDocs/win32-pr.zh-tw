# <a name="client-shell-api-enumerations"></a>用戶端 Shell API 列舉

下表概要說明 Windows 遠端管理 (WinRM) 用戶端 Shell 應用程式開發介面 (API) 中的列舉。



| 列舉                                                 | Description                                                                                      |
|--------------------------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**WSManAuthenticationFlags**](/windows/desktop/api/Wsman/ne-wsman-wsmanauthenticationflags) | 定義用於伺服器和 proxy 驗證的驗證配置。               |
| [**WSManCallbackFlags**](/windows/desktop/api/Wsman/ne-wsman-wsmancallbackflags)             | 定義所有回呼函式所使用的一組旗標。                                           |
| [**WSManDataType**](/windows/desktop/api/Wsman/ne-wsman-wsmandatatype)                       | 指定 [**WSMAN \_ 資料**](/windows/desktop/api/Wsman/ns-wsman-wsman_data) 結構中聯集的目前資料類型。 |
| [**WSManProxyAccessType**](/windows/desktop/api/Wsman/ne-wsman-wsmanproxyaccesstype)         | 定義 proxy 存取類型。                                                                   |
| [**New-wsmansessionoption**](/windows/desktop/api/Wsman/ne-wsman-wsmansessionoption)             | 定義會話的延伸選項組。                                               |
