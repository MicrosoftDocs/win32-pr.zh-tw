# <a name="resourcelocatorerror-property"></a>ResourceLocator。 Error 屬性

取得其他錯誤資訊的 XML 表示。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
ResourceLocator.Error As BSTR
```



## <a name="property-value"></a>屬性值

其他錯誤資訊的 XML 表示。
