# <a name="enumeratorerror-property"></a>列舉值。 Error 屬性

取得其他錯誤資訊的 XML 表示。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
Enumerator.Error As String
```



## <a name="property-value"></a>屬性值

其他錯誤資訊的 XML 表示。
