# <a name="wsmancommandline-property"></a>WSMan 屬性

取得載入 automation 元件之進程的命令列。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
WSMan.CommandLine As BSTR
```



## <a name="property-value"></a>屬性值

包含命令列的字串。
