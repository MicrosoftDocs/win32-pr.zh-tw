# <a name="programming-the-microsoft-agent-control"></a>Microsoft Agent 控制項程式設計

\[Microsoft Agent 已于 Windows 7 淘汰，在後續的 Windows 版本中可能無法使用。\]

雖然應用程式可以使用其 automation 伺服器介面直接寫入 Microsoft Agent 服務，但 Microsoft 代理程式也包含 ActiveX (OLE) 控制項。 此控制項支援使用指令碼語言（例如 Microsoft Visual Basic Scripting Edition (VBScript) 或其他支援 ActiveX 控制項介面的語言）進行簡單的程式設計。
