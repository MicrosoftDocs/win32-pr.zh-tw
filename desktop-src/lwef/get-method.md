## <a name="remarks"></a>備註

如果您使用 HTTP 通訊協定載入字元 (。ACF 檔) ，您必須先使用 **Get** 方法來抓取動畫資料，才能播放動畫。 如果您使用 UNC 通訊協定將字元載入 (，就不會使用這個方法。ACS 檔案) 。 如果您使用 UNC 通訊協定 ( 載入該字元，您也無法使用 **Get** 抓取字元的 HTTP 資料。) 的 ACS 字元檔。

如果您宣告物件參考，並將它設定為此方法，則會傳回 [**Request**](/windows/desktop/lwef/the-request-object) 物件。 如果關聯的動畫無法載入，伺服器會將 **要求** 物件的 [**Status**](status-property.md) 屬性設為「失敗」，並提供適當的錯誤號碼。 您可以使用 [**RequestComplete**](requestcomplete-event.md) 事件來檢查狀態，並判斷要採取的動作。

使用 Get 方法取出的動畫或音效資料會儲存在瀏覽器的 **快取** 中。 後續的呼叫將會檢查快取，如果動畫資料已經存在，控制項會直接從快取載入資料。 載入之後，您可以使用 [**Play**](play-method.md) 或 [**朗讀**](speak-method.md) 方法來播放動畫或音效資料。
