# <a name="advanced-character-options-window-microsoft-agent-user-interface"></a>[Advanced Character Options] 視窗 (Microsoft Agent 消費者介面) 

\[Microsoft Agent 已于 Windows 7 淘汰，在後續的 Windows 版本中可能無法使用。\]

Microsoft Agent 會維護特定的全域設定，讓使用者能夠控制所有字元的互動。 [先進的字元選項] 視窗會顯示這些選項及其目前的設定，並可透過使用 Microsoft Agent 程式設計介面的任何裝載應用程式來顯示。 Microsoft Agent 會記住上次查看的頁面，並在屬性工作表出現時顯示該頁面。

-   [輸出頁面](the-output-page.md)
-   [語音輸入頁面](the-speech-input-page.md)
-   [著作權頁面](the-copyright-page.md)
