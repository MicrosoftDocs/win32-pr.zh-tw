# <a name="creating-an-instance-of-the-control"></a>建立控制項的實例

\[Microsoft Agent 已于 Windows 7 淘汰，在後續版本的 Windows 中可能無法使用。\]

存取 Microsoft Agent 的控制權需要先建立控制項的實例。 最簡單的方法是將控制項的實例放在表單上。 您可能必須先將控制項加入至 [工具箱]，然後再將它新增至您的表單。 根據您的程式設計語言，您也可以在執行時間建立控制項。
