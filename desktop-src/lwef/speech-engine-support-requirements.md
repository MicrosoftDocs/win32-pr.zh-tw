# <a name="speech-engine-support-requirements"></a>語音引擎支援需求

\[Microsoft Agent 已于 Windows 7 淘汰，在後續版本的 Windows 中可能無法使用。\]

Microsoft Agent 會使用 Microsoft 語音應用程式開發介面 (的 SAPI) ，以支援語音輸入 (語音辨識，或使用 SR) 和語音輸出 (文字轉換語音或 TTS) 。 藉由支援這項標準，其他語音引擎可支援 Microsoft 代理程式的語音服務。 本檔說明 Microsoft 代理程式所使用的所需的 SAPI 介面。 如需有關 SAPI 的詳細資訊，請參閱 [Microsoft Speech](https://msdn.microsoft.com/library/ee705648.aspx) group 的網站。

-   [文字轉換語音引擎的需求](requirements-for-text-to-speech-engines.md)
-   [語音辨識引擎的需求](requirements-for-speech-recognition-engines.md)
-   [支援 Microsoft 語言資訊音效編輯工具的需求](requirements-for-supporting-the-microsoft-linguistic-information-sound-editing-tool.md)
