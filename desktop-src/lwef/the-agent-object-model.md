# <a name="the-agent-object-model"></a>代理程式物件模型

\[Microsoft Agent 已于 Windows 7 淘汰，在後續版本的 Windows 中可能無法使用。\]

Microsoft 代理程式物件模型包含下列物件：

-   要求
-   代理程式 (控制項) 
-    (集合) 的字元
-   字元
-   命令 (集合) 
-   命令
-   氣球
-   AnimationNames (集合) 
-   SpeechInput
-   AudioOutput
-   CommandsWindow
-   PropertySheet

這些物件會組織在下列階層中。  (物件後面的虛線表示可以存在多個物件。 )
