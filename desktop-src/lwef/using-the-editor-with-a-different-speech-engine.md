# <a name="using-the-editor-with-a-different-speech-engine"></a>使用編輯器搭配不同的語音引擎

\[Microsoft Agent 已于 Windows 7 淘汰，在後續的 Windows 版本中可能無法使用。\]

雖然音效編輯器會將 Microsoft 語音辨識引擎安裝 (4.0) ，但如果該引擎支援語音引擎需求檔中記載的必要介面，則可與另一個語音引擎搭配使用。 在嘗試使用編輯器搭配其他引擎之前，請向您的廠商確認他們符合這些需求。

若要將音效編輯器與另一個語音引擎搭配使用，請選擇 [ **編輯** ] 功能表上的 [語音引擎] 命令。 這會顯示一個對話方塊，顯示目前使用中的引擎。 若要選擇另一個引擎，請顯示引擎清單，然後按一下 **[確定]**。 如果未列出任何其他引擎，則表示您沒有包含任何其他相容的引擎。
