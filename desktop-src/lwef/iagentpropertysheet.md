# <a name="iagentpropertysheet"></a>IAgentPropertySheet

\[Microsoft Agent 已于 Windows 7 淘汰，在後續版本的 Windows 中可能無法使用。\]

**IAgentPropertySheet** 會定義一個介面，可讓應用程式 (視窗) 設定和查詢 Microsoft Agent 屬性工作表的屬性。

**依照 Vtable 順序的方法**



| IAgentPropertySheet 方法                             | Description                                                                                  |
|---------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [**GetVisible**](iagentpropertysheet--getvisible.md)   | 傳回 Microsoft Agent 屬性工作表是否可見。                               |
| [**SetVisible**](iagentpropertysheet--setvisible.md)   | 設定 Microsoft Agent 屬性工作表的 [**Visible**](visible-property.md) 屬性。 |
| [**GetPosition**](iagentpropertysheet--getposition.md) | 傳回 Microsoft Agent 屬性工作表的位置。                                  |
| [**GetSize**](iagentpropertysheet--getsize.md)         | 傳回 Microsoft Agent 屬性工作表的大小。                                      |
| [**GetPage**](iagentpropertysheet--getpage.md)         | 傳回 Microsoft Agent 屬性工作表的目前頁面。                             |
| [**SetPage**](iagentpropertysheet--setpage.md)         | 設定 Microsoft Agent 屬性工作表的目前頁面。                                |
