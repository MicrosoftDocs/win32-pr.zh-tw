# <a name="the-characters-object"></a>字元物件

\[Microsoft Agent 已于 Windows 7 淘汰，在後續版本的 Windows 中可能無法使用。\]

您的用戶端應用程式可以支援一或多個字元。 此外，您可以在數個應用程式之間共用一個字元。 Microsoft Agent 會將 [**字元**](/windows/desktop/lwef/the-characters-object) 物件定義為字元的集合。 若要存取字元，請將字元的資料載入至 [**字元**](/windows/desktop/lwef/the-characters-object) 集合，並指定集合中的專案使用該字元所支援的方法和屬性。

-   [字元物件方法](characters-object-methods.md)
-   [字元物件方法](character-object-methods.md)
-   [字元物件屬性](character-object-properties.md)
