# <a name="create-personality"></a>建立個人化

\[Microsoft Agent 已于 Windows 7 淘汰，在後續的 Windows 版本中可能無法使用。\]

我們會根據最簡單的社交提示（例如狀態、手勢、外觀、文字選擇和樣式），快速地分類我們所符合之人員的特質。 因此，一個字元的第一印象非常重要。 建立個人化並不需要人工智慧或實際呈現。 很棒的 animators 多年來都知道這一點，並使用最簡單的社交提示，為無生命物件建立豐富的特質。 例如，請考慮迪士尼的 *Aladdin* 和 Lassiter 的 *Luxo Jr* 的飛行地毯，這是一組幽默的動畫影片。 在迪士尼開始 animators 時，通常會面臨繪製 flour sacks 以表示表情的挑戰。

一個字元的名稱、它本身的使用方式、其說話方式、移動的方式，以及它如何回應使用者輸入，全都有助於建立其基本特性。 例如，您可以建立判斷提示的授權或主要樣式，並示範信賴度和發出命令，而 submissive 的個人化可能是藉由將事物視為問題或提出建議來表徵。 同樣地，您可以在互動順序中傳達個人化。 主要特質一律會優先執行。 無論您要建立何種類型的個人化類型，都必須提供獨特、定義完善的特質型別。 每個人通常會不喜歡弱式定義或不明確的特質。

您為字元選擇的特性種類取決於您的目標。 如果字元的目的是要將使用者導向特定的目標，請使用主要、assertive 的個人化。 如果字元的目的是要回應使用者的要求，請使用更 submissive 的特性。

另一種方法是將字元的特質調整為使用者的個人。 研究顯示，使用者偏好與特質互動。 您可以使用不同的特質來提供使用者選擇的字元，或觀察使用者與該字元的互動樣式，並修改字元的互動式樣式。 研究顯示當您嘗試比對使用者的個人特性時，不一定要有100% 的正確。 人們傾向于顯示其關聯性的彈性，而且由於社交關系的本質，也可能會修改自己的行為，而不是使用字元。
