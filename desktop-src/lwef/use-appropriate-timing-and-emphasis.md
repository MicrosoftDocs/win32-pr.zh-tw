# <a name="use-appropriate-timing-and-emphasis"></a>使用適當的計時和強調

\[Microsoft Agent 已于 Windows 7 淘汰，在後續的 Windows 版本中可能無法使用。\]

如同所有意見反應，語音輸出的有效性取決於計時和強調。 有一項很多的資訊可以用在說話時所用的步調、數量和音調來傳達。 如果您使用文字轉換語音引擎作為字元的聲音，大部分引擎都可讓您設定文字或片語的速度、暫停、音調和強調。 您可以使用這些屬性來指出您對字元的興趣和理解，以及指示使用者的注意或指出您的字元意圖。 如需有關如何設定語音屬性的詳細資訊，請參閱 [Microsoft Agent 語音輸出標記](microsoft-agent-speech-output-tags.md)。 如果您使用音效檔作為字元的輸出，請在錄製的音訊中也考慮這些因素。
