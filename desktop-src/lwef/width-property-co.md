## <a name="remarks"></a>備註

[**Width**](width-property.md)屬性一律以圖元表示。 這個屬性的設定會套用到字元的所有用戶端。

雖然字元會出現在不規則的形狀區域視窗中，但字元的位置是以使用 Microsoft Agent 字元編輯器來編譯字元時所使用之矩形動畫框架的外部維度為基礎。
