# <a name="starting-the-sound-editor"></a>啟動音效編輯器

\[Microsoft Agent 已于 Windows 7 淘汰，在後續版本的 Windows 中可能無法使用。\]

若要執行 Microsoft 語言資訊音效編輯工具，請從 [ **開始** ] 功能表中選擇它，或按兩下音效編輯器的圖示。 [音效編輯器] 的視窗隨即開啟，顯示其功能表、常用命令的工具列、輸入編輯器用來處理音效檔的文字的文字方塊，以及用來查看和編輯音訊和語言資料的顯示區域。

![顯示 Microsoft 語言資訊音效編輯工具起始頁面的螢幕擷取畫面。](images/f1listliq.gif)

當音效編輯器啟動之後，您就可以開始錄製新的音效檔或載入現有的聲音檔。
