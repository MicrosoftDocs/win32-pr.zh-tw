# <a name="animation-services"></a>動畫服務

\[Microsoft Agent 已于 Windows 7 淘汰，在後續版本的 Windows 中可能無法使用。\]

Microsoft 代理程式的動畫服務會在螢幕上的自己視窗中管理字元影像的動畫和移動。 動畫會定義為一系列的計時和可選擇分支的框架，其中包含一或多個影像。

-   [載入字元](loading-a-character.md)
-   [正在載入預設字元](loading-the-default-character.md)
-   [製作字元動畫](animating-a-character.md)
