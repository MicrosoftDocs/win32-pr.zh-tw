# <a name="animation-sound-effects"></a>動畫音效效果

\[Microsoft Agent 已于 Windows 7 淘汰，在後續的 Windows 版本中可能無法使用。\]

Microsoft Agent 也可讓您在字元的動畫中包含音效效果。 您可以使用「Microsoft 代理程式字元編輯器」來指定標準 Windows 音效 ( 的檔案名。要在指定的框架上觸發的 WAV) 檔案。 請注意，Microsoft 代理程式不會混合音效效果和語音輸出，因此在音效效果完成之前，不會開始語音輸出。 因此，請避免任何長或迴圈的音效效果成為字元動畫的一部分。
