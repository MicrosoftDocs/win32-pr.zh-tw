# <a name="endpoint-data-loss-prevention-structures"></a>端點資料遺失防止結構

端點資料遺失防止功能所使用的結構。



| 結構                                                       | Description                                                           |
|-------------------------------------------------------------------|-----------------------------------------------------------------------|
| [DLP_APP_OP_ENLIGHTENED_LEVEL](endpointdlp-dlp_app_op_enlightened_level.md)                       | 使端點資料遺失防止 (DLP) 動作與強制等級產生關聯。                                  |
| [DLP_DOCUMENT_INFO](endpointdlp-dlp_document_info.md)                       | 指定與端點 DLP 操作相關聯之檔的相關資訊。                                  |
| [DLP_POSTOP_STATUS](enpointdlp-dlp_postop_status.md)                         | 指定有關端點 DLP 操作的狀態資訊。  |
| [DLP_PRINT_INFO](endpointdlp-dlp_print_info.md)                       | 指定與端點 DLP 列印操作相關聯之檔的相關資訊。                                  |
