# <a name="version-property"></a>Version 屬性

\[Microsoft Agent 已于 Windows 7 淘汰，在後續版本的 Windows 中可能無法使用。\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**描述**
</dt> <dd>

傳回字元的版本。

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**語法**
</dt> <dd>

*agent. ***字元 ( "**_CharacterID_*_" ) 。版本_*

</dd> </dl>

## <a name="remarks"></a>備註

**Version** 屬性會傳回字串，該字串會對應至編譯字元之標準動畫集定義的版本。 當您使用 Microsoft 代理程式字元編輯器來建立時，會自動設定字元的版本號碼。
