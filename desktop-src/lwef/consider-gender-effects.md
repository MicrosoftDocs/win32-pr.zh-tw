# <a name="consider-gender-effects"></a>考慮性別效應

\[Microsoft Agent 已于 Windows 7 淘汰，在後續的 Windows 版本中可能無法使用。\]

社交回應可能會受性別影響。 例如，Nas 和 Reeves 的工作指出，在美國，男性特質 stereotypically 被視為瞭解更多有關技術主題的，而女性多則是在像是像愛和關聯性的公司中提供更好的專家。 這並不表示您應該永久保存性別造型，只是您知道它們可能存在，並瞭解它們可能會如何影響與您的字元的互動。

請記住，字元的認知性別並非完全以其名稱或外觀來建立。 即使是性別中性字元，也可以根據其語音特性、語音模式和移動來傳達性別。
