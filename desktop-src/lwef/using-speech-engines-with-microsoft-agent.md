# <a name="using-speech-engines-with-microsoft-agent"></a>搭配 Microsoft 代理程式使用語音引擎

\[Microsoft Agent 已于 Windows 7 淘汰，在後續版本的 Windows 中可能無法使用。\]

Microsoft 提供一組語音輸入 (辨識) 引擎和語音輸出 (tex 對語音或 TTS) 引擎，您可以搭配已啟用 Microsoft 代理程式的應用程式或網頁使用。 這些引擎的使用會受限於 Microsoft 代理程式授權合約以及當您安裝引擎時所顯示的補充補充增補。 此授權可提供您散發引擎的許可權，但只有在透過 Microsoft Agent API 和使用可見字元來使用時才會提供。

這裡的各節會說明如何使用 Microsoft Agent 的語音引擎。

-   [存取程式碼中的語音引擎](accessing-a-speech-engine-in-your-code.md)
-   [Microsoft 代理程式語音輸出標記](microsoft-agent-speech-output-tags.md)
-   [文字轉換語音引擎文字正規化](text-to-speech-engines-text-normalization.md)
-   [其他與 Microsoft Agent 相容的語音引擎](other-speech-engines-compatible-with-microsoft-agent.md)

在下列連結中尋找適用于 Windows Vista 的 Microsoft 代理程式語音和語音辨識功能的變更。

-   [Windows Vista 中的 Microsoft Agent 變更](microsoft-agent-changes-in-windows-vista.md)
