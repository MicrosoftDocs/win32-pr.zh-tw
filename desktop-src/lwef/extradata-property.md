# <a name="extradata-property"></a>ExtraData 屬性

\[Microsoft Agent 已于 Windows 7 淘汰，在後續版本的 Windows 中可能無法使用。\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**描述**
</dt> <dd>

傳回字串，這個字串會指定儲存為字元一部分的其他資料。

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**語法**
</dt> <dd>

*代理程式*。**( "**_CharacterID_*_" ) 的字元。ExtraData_*

</dd> </dl>

## <a name="remarks"></a>備註

使用 Microsoft Agent 字元編輯器來編譯字元時，會定義字元之 **ExtraData** 屬性的預設值。 它無法在執行時間變更或指定。

> [!Note]  
> **ExtraData** 屬性設定為選擇性，而且可能不會提供給所有字元。
