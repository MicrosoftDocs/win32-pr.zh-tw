# <a name="audio-output-support"></a>音訊輸出支援

\[Microsoft Agent 已于 Windows 7 淘汰，在後續版本的 Windows 中可能無法使用。\]

Microsoft 代理程式可讓您將音訊檔案用於字元的語音輸出。 您可以記錄音訊檔案，並使用「 [**說話**](speak-method.md) 」方法來播放該資料。 Microsoft Agent 動畫服務會使用音訊檔案的音訊特性，自動支援與字元中的邊緣同步處理。 Microsoft 代理程式也支援音訊檔案的特殊格式，包括額外的音素和斷詞資訊，以提供更多增強的 lip 同步支援。 您可以使用 Microsoft 語言資訊音效編輯工具來產生這個特殊格式。
