# <a name="social-interaction"></a>社交互動

\[Microsoft Agent 已于 Windows 7 淘汰，在後續的 Windows 版本中可能無法使用。\]

人們的溝通本質上是社交的。 從我們所出生的時間開始，我們會開始回應環境中的社交提示，並學習適用于有效互動的適當規則，其中包括像是聲調或文字排序等口頭行為，以及非口頭的行為，例如狀態、手勢和臉部運算式。 這些行為會傳達對我們的通訊進行色彩的看法、身分識別和表情。 我們通常會為通道建立替代慣例，而不會針對非口頭的提示（例如電子郵件或線上交談會話）自然提供頻寬。

可惜的是，大部分的軟體介面設計主要著重于通訊的認知層面、忽略大部分的社會層面。 不過，最近的研究已經示範過，人類對以互動內容呈現的社交刺激有自然的反應。 此外，這些反應通常會遵循人們彼此使用的相同慣例。 即使是最小的提示（例如顯示的色彩或訊息中的文字選擇），也可以觸發這項自動回應。 具有眼睛和嘴的動畫字元呈現，可對比度對字元之回應的社會期望和強度。 絕對不會假設使用者認為字元的行為不太適合社交，因為他們知道它是人工的。 知道這一點，在設計字元互動時，請務必考慮互動的社會層面。 *媒體方程式：人們如何將電腦、電視和新媒體作為真實的人員和地點* Byron Reeves 和 Clifford Nas (紐約：康橋大學按) 是此區域目前研究的絕佳參考。
