# <a name="the-speech-input-page"></a>語音輸入頁面

\[Microsoft Agent 已于 Windows 7 淘汰，在後續版本的 Windows 中可能無法使用。\]

使用者可以在此屬性頁面上調整語音輸入選項。 使用者可以停用語音輸入、設定接聽輸入機碼、選擇是否要顯示接聽提示視窗，以及選擇是否要在語音輸入可用時播放 MIDI 語氣。
