## <a name="remarks"></a>備註

標準 RGB 色彩的有效範圍為0到 16777215 (&HFFFFFF) 。 此範圍中數位的最大位元組等於 0;較低的3個位元組（從最高到最大的位元組）分別決定紅色、綠色和藍色的數量。 紅、綠和藍三種元素分別由 0 到 255 (&HFF) 之間的數字表示。
