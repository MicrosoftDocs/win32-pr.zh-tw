# <a name="helpcomplete-event"></a>HelpComplete 事件

\[Microsoft Agent 已于 Windows 7 淘汰，在後續版本的 Windows 中可能無法使用。\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**描述**
</dt> <dd>

指出已結束內容相關的說明模式。

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**語法**
</dt> <dd>

**子***代理程式。 * * * (byval* *  *CharacterID * * *，byval* *  *名稱 * * *，byval* *  *原因 * * * )**



| 部分          | 描述                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
|---------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *CharacterID* | 以字串形式傳回已按下字元的識別碼。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| *名稱*        | 傳回字串值，識別命令 (識別碼) 的名稱。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| *原因*       | 傳回值，這個值表示導致說明模式完成的原因。 1使用者選取了您的應用程式所提供的命令。<br/> 2使用者選取了另一個用戶端的 [**命令**](/windows/desktop/lwef/the-commands-collection-object) 物件。<br/> 3使用者已選取 [開啟語音命令] 命令。<br/> 4使用者選取了 [關閉語音命令] 命令。<br/> 5使用者選取 [顯示 *CharacterName* ] 命令。<br/> 6使用者選取了 [隱藏 *CharacterName* ] 命令。<br/> 7 (按一下) 字元來選取使用者。<br/> |



 

</dd> </dl>

### <a name="remarks"></a>備註

一般而言，當使用者按一下或拖曳字元，或從字元的快顯功能表中選取命令時，[說明] 模式就會完成。 按一下其他字元或螢幕上的其他位置，並不會取消說明模式。 設定字元之說明模式的用戶端可以將 [**HelpModeOn**](helpmodeon-property.md) 設定為 **False**，以取消說明模式。  (這不會觸發 **HelpComplete** 事件。 ) 

當使用者從 [說明] 模式中的字元快顯功能表選取命令時，伺服器會移除功能表、使用 [**命令指定的**](helpcontextid-property.md)協助工具來呼叫說明，並傳送此事件。 與內容相關的 (也稱為「這是什麼？ ) 說明視窗會顯示在指標位置。 如果使用者選取語音輸入的命令，則會在字元上顯示 [說明] 視窗。 如果字元是在螢幕上，視窗會在畫面上顯示最接近字元的目前位置。

如果伺服器將名稱以空字串的形式傳回 ( "" ) ，表示使用者已選取伺服器提供的命令。

此事件只會傳送至用戶端應用程式，該應用程式會將字元放在說明模式中。
