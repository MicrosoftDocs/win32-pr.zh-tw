# <a name="removeall-method"></a>RemoveAll 方法

\[Microsoft Agent 已于 Windows 7 淘汰，在後續的 Windows 版本中可能無法使用。\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**描述**
</dt> <dd>

從 [**命令**](/windows/desktop/lwef/the-commands-collection-object)集合中移除所有的 [**命令**](/windows/desktop/lwef/the-command-object)物件。

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**語法**
</dt> <dd>

*代理程式 ***。字元 ( "*** CharacterID * * *" ) . RemoveAll**

</dd> </dl>

## <a name="remarks"></a>備註

從集合中移除 [**命令**](/windows/desktop/lwef/the-command-object) 物件時，當您的用戶端應用程式為輸入使用中時，它就不會再顯示。
