# <a name="creating-a-new-sound-file"></a>建立新的音效檔

\[Microsoft Agent 已于 Windows 7 淘汰，在後續的 Windows 版本中可能無法使用。\]

當您第一次啟動編輯器時，您可以從 [**音訊**] 功能表選擇 [**錄製**]，或按一下 [音效編輯器] 工具列上的 [**錄製**] 按鈕，然後說到連接到您系統的麥克風，來建立新的音效檔。 按一下工具列上的 [ **停止** ] 按鈕以停止錄製。 您可以從 **音訊** 功能表或工具列中選取 [**播放**] 命令，以查看 Microsoft 代理程式如何處理音效檔，而不需要語言增強。 若要建立另一個新檔案，請從 [檔案] 功能表或工具列中 **選取 [新增** ]。
