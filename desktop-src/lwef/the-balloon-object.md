# <a name="the-balloon-object"></a>球形球形物件

\[Microsoft Agent 已于 Windows 7 淘汰，在後續的 Windows 版本中可能無法使用。\]

Microsoft Agent 支援使用卡通文字氣球的 [**語音**](speak-method.md) 方法文字字幕。 「 [**思考**](think-method.md) 」方法可讓您在「想像」的文字氣球中顯示沒有音訊輸出的文字。

字元的初始文字氣球視窗預設值會在 Microsoft Agent 字元編輯器中定義和編譯。 執行之後，使用者可以覆寫氣球的 [**啟用**](enabled-property.md) 和 [**字型**](https://www.bing.com/search?q=**Font**) 屬性。 如果使用者變更文字氣球的屬性，則會影響所有字元。 「 [**說話**](speak-method.md) 」和「 [**想法**](think-method.md) 」兩個字提示都使用相同的「大小」屬性設定。 您可以透過 [**球形球形**](/windows/desktop/lwef/the-balloon-object) 物件（即 [**字元**](/windows/desktop/lwef/the-characters-object) 物件的子系）來存取字元的字組批註的屬性。

[**球形球形**](/windows/desktop/lwef/the-balloon-object)物件支援下列屬性：

-   [**顏色**](backcolor-property.md)
-   [**BorderColor**](bordercolor-property.md)
-   [**CharsPerLine**](charsperline-property.md)
-   [**啟用**](enabled-property.md)
-   [**FontCharSet**](fontcharset-property.md)
-   [**FontName**](fontname-property-bal.md)
-   [**FontBold**](fontbold-property.md)
-   [**FontItalic**](fontitalic-property.md)
-   [**FontSize**](fontsize-property-bal.md)
-   [**FontStrikeThru**](fontstrikethru-property.md)
-   [**FontUnderline**](fontunderline-property.md)
-   [**ForeColor**](forecolor-property.md)
-   [**NumberOfLines**](numberoflines-property.md)
-   [**風格**](style-property.md)
-   [**可見**](visible-property-bal.md)
