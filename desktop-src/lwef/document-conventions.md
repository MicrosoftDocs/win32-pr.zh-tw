# <a name="document-conventions"></a>文件慣例

\[Microsoft Agent 已于 Windows 7 淘汰，在後續的 Windows 版本中可能無法使用。\]

本檔會使用下列印刷慣例：



| 慣例                       | Description                                                                    |
|----------------------------------|--------------------------------------------------------------------------------|
| **Sub、Visible、Caption**        | 粗體的單字（以字母大寫的字母大寫表示關鍵字）。               |
| *agent、String、Now*             | 斜體單字指出您所提供之資訊的預留位置。                 |
| ENTER、F1                        | 全大寫字母的單字表示檔案名、索引鍵名稱和索引鍵序列。 |
| `Agent1.Commands.Enabled = True` | 此字型中的文字表示程式碼範例。                                       |
| `' This is a comment`            | 表示程式碼批註的單引號 (`'`) 。                                  |
| `Agent1.Commands.Add "Test1", _` | 空格和底線 (\_) 繼續執行一行程式碼。                       |
| `[words or expression]`          | 方括弧內的專案是選擇性的。                                     |
| `This | That`                    | 分隔號表示兩個或多個專案之間的選擇。                   |
| *代理*                          | 斜體中的「代理程式」一詞代表您所使用的 agent 控制項名稱。  |



 

本檔中的程式設計介面描述遵循 Microsoft VBScript 的慣例。 不過，它們通常也適用于其他語言。
