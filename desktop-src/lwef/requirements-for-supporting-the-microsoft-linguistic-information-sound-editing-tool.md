# <a name="requirements-for-supporting-the-microsoft-linguistic-information-sound-editing-tool"></a>支援 Microsoft 語言資訊音效編輯工具的需求

\[Microsoft Agent 已于 Windows 7 淘汰，在後續的 Windows 版本中可能無法使用。\]

Microsoft 語言資訊音效編輯工具會使用語音辨識引擎來產生標準 Windows wave 音效 ( .wav) 檔案的斷詞和語音資訊。 2.0 版本現在支援搭配其他語音引擎使用。 希望支援音效編輯器的廠商必須確保其引擎完全支援無內容文法引擎的 SAPI 4.0 規格，以及下列需求。

-   [ISRResGraphEx 和 IAttributes](isrresgraphex-and-iattributes.md)
