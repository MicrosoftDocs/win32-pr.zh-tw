# <a name="installing-your-character"></a>安裝您的字元

\[Microsoft Agent 已于 Windows 7 淘汰，在後續版本的 Windows 中可能無法使用。\]

若要在助理資源庫中顯示您的字元，請將它安裝在 \\ 使用者的應用程式資料夾內的 Microsoft Office 動作專案 \\ 資料夾內。 您可以在 Windows 目錄或 Windows 目錄內的使用者設定檔目錄中找到應用程式資料檔案夾。 例如，在 Windows 2000 中，您可以在 [C： \\ Documents] 和 [設定 \\ YourLoginName \\ Application Data \\ Microsoft \\ Office \\ 演員] 中安裝字元。
