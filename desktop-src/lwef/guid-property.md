## <a name="remarks"></a>備註

這個屬性會傳回字串，表示伺服器用來參考唯一識別字元的內部識別碼。 使用 Microsoft 代理程式字元編輯器來編譯時，會設定字元識別碼。 此為唯讀屬性。
