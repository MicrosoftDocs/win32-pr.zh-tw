# <a name="the-propertysheet-object"></a>PropertySheet 物件

\[Microsoft Agent 已于 Windows 7 淘汰，在後續版本的 Windows 中可能無法使用。\]

如果您想要操作 Microsoft Agent 屬性工作表的相對字元 (也稱為 [先進字元選項] 視窗) ， [**PropertySheet**](https://www.bing.com/search?q=**PropertySheet**) 物件會提供數個可供您使用的屬性。

-   [**高度**](height-property-pso.md)
-   [**離開**](left-property-pso.md)
-   [**網頁**](page-property.md)
-   [**返回頁首**](top-property-pso.md)
-   [**可見**](visible-property.md)
-   [**寬度**](width-property-pso.md)

如果您在顯示內容表之前查詢 [**高度**](height-property-pso.md)、 [**左方**](left-property-pso.md)、 [**頂端**](top-property-pso.md)和 [**寬度**](width-property-pso.md) 屬性，其值會以零 (0) 傳回。 一旦顯示之後，這些屬性會傳回視窗的最後一個位置和大小 (相對於您目前的螢幕解析度) 。
