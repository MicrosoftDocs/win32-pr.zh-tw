# <a name="windows-defender-reference"></a>Windows Defender 參考

Windows Defender 可協助保護電腦不受間諜軟體和其他垃圾軟體所造成的快顯視窗、效能變慢和安全性威脅的影響。 應用程式可以查詢用戶端的資訊，例如電腦健全狀況、特定的惡意程式碼資訊，以及惡意程式碼記錄，並可叫用隨選掃描。
