# <a name="installing-the-sound-editor"></a>安裝音效編輯器

\[Microsoft Agent 已于 Windows 7 淘汰，在後續的 Windows 版本中可能無法使用。\]

使用音效編輯器的建議系統設定是具有 Pentium 166 的電腦，至少 48 MB 的 RAM，以及 Windows 相容的音效卡。 如果您想要使用工具錄製說話輸入，您也需要一個相容的麥克風。

若要安裝 Microsoft 語言音效編輯工具，請開啟其自我解壓縮安裝檔案。 這會自動在您的系統上安裝適當的檔案。 如果您從 Microsoft 代理程式網站下載音效編輯器，則可以選擇在下載之後安裝編輯器，或將它儲存到您的磁片，以便後續開啟和安裝。 安裝工具將會建議您在 Microsoft 代理程式的 [工具] 子目錄中自行安裝。 建議您使用這個位置。

Microsoft 命令和控制語音辨識引擎 (版本。 4.0) 也必須先安裝，才能使用音效編輯器。 這通常會與音效編輯器一起安裝，但如果之後卸載，您可以從 Microsoft Platform SDK 或 Microsoft Agent 網站重新安裝它 <https://www.microsoft.com/downloads/details.aspx?FamilyID=914fef6b-d764-4eca-88dc-d49d8912625c> 。 音效編輯器只能根據語音引擎所支援的語言來產生語言資訊。 若要產生其他語言的資訊，必須安裝適用于該語言的相容語音辨識引擎。 請洽詢您的語音引擎廠商，以判斷是否支援 Microsoft 語言音效編輯工具。
