# <a name="accessing-microsoft-agent-services-directly"></a>直接存取 Microsoft Agent 服務

\[Microsoft Agent 已于 Windows 7 淘汰，在後續版本的 Windows 中可能無法使用。\]

如果您使用 C、c + + 或 JAVA，則可以使用 ActiveX (OLE) 介面，直接存取 Microsoft 代理程式伺服器。 如需這些介面的詳細資訊，請參閱 [程式設計 Microsoft Agent 伺服器介面](programming-the-microsoft-agent-server-interface.md)。
