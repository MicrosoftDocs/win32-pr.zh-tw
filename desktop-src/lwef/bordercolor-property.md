# <a name="bordercolor-property"></a>邊框出屬性

\[Microsoft Agent 已于 Windows 7 淘汰，在後續版本的 Windows 中可能無法使用。\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**描述**
</dt> <dd>

傳回目前針對指定字元之文字氣球視窗顯示的框線色彩。

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**語法**
</dt> <dd>

*代理程式 ***。 ( "**_CharacterID_*_" ) 的字元。_* 球形邊框

</dd> </dl>

## <a name="remarks"></a>備註

標準 RGB 色彩的有效範圍為0到 16777215 (&HFFFFFF) 。 此範圍中數位的最大位元組等於 0;較低的3個位元組（從最高到最大的位元組）分別決定紅色、綠色和藍色的數量。 紅、綠和藍三種元素分別由 0 到 255 (&HFF) 之間的數字表示。
