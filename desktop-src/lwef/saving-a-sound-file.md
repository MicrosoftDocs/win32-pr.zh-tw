# <a name="saving-a-sound-file"></a>儲存音效檔

\[Microsoft Agent 已于 Windows 7 淘汰，在後續的 Windows 版本中可能無法使用。\]

當您準備好要儲存音效檔時，請 **選擇 [檔案] 功能表上** 的 [儲存] 命令，或在編輯器的工具列上選擇 [**儲存**] 命令。 編輯器會顯示 [ **另存** 新檔] 對話方塊，並根據您是否產生檔案的語言資訊來建議名稱和預設檔案類型。 如果您將檔案儲存為音效檔 ( .wav) ，編輯器只會儲存音訊資料。 如果您將檔案資訊儲存為語言增強的音效檔 ( lwv) ，則會自動將 word 和音素資訊納入修改的音效檔中。 在您確認或編輯名稱、位置、檔案類型和格式之後，請選擇 [ **儲存** ] 按鈕。

![顯示 [另存新檔] 對話方塊的螢幕擷取畫面，其中已選取 [檔案名]、[另存新檔類型]、[格式] 和 [屬性]。](images/f7listsave.gif)

如果您想要使用新的名稱、不同的位置或不同的格式來儲存音效檔，請 **選擇 [檔案] 功能表上** 的 [**另存** 新檔] 命令。 當 [ **另存** 新檔] 對話方塊出現時，請輸入新的檔案名，然後按一下 [ **儲存** ] 按鈕。

您也可以儲存部分的音效檔。 例如，您可能想要儲存檔案，而不會在其開頭或結尾有過度的無回應。 在 [音訊] 表示中，選取您要儲存的檔案部分，然後選擇 [檔案 **] 功能表中的 [** **儲存選取專案**]。 只有當您在 **音訊標記法** 中有選取專案時，才會啟用此命令。
