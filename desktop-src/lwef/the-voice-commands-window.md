# <a name="the-voice-commands-window"></a>語音命令視窗

\[Microsoft Agent 已于 Windows 7 淘汰，在後續的 Windows 版本中可能無法使用。\]

如果有安裝相容的語音引擎，Microsoft 代理程式會提供一個稱為 [ **語音命令] 視窗** 的特殊視窗，其中顯示已啟用語音辨識語音的命令。 [ **語音命令] 視窗** 可作為輸入 (命令的視覺提示，無法使用滑鼠) 選取。

![語音命令對話方塊](images/f2voice.gif)

當使用者選取 [ **開啟語音命令] 視窗** 命令（藉由說出命令或以滑鼠右鍵按一下字元，然後從字元的快顯功能表中選擇命令）時，就會出現此視窗。 不過，如果使用者停用語音輸入，則無法存取 [ **語音命令] 視窗** 。

[ **語音命令] 視窗** 會將啟用語音的命令顯示為樹狀結構。 如果目前的裝載應用程式提供語音命令，它們會出現在視窗的頂端。 使用字元的其他應用程式也會顯示專案。 此視窗也包含 Microsoft Agent 提供的全域語音命令。 如果目前的裝載應用程式沒有語音命令，全域語音命令會出現在視窗的頂端。

使用者可以調整 [ **語音命令] 視窗** 的大小和移動。 如果使用者關閉並重新開啟視窗，Microsoft Agent 會記住視窗的最後一個位置，並在該位置重新加以重新開啟。 如果視窗中的專案超過視窗目前的顯示大小，則會顯示捲軸。
