<span data-ttu-id="31968-108">當使用者選取 [ **開啟語音命令] 視窗** 命令（藉由說出命令或以滑鼠右鍵按一下字元，然後從字元的快顯功能表中選擇命令）時，就會出現此視窗。</span><span class="sxs-lookup"><span data-stu-id="31968-108">The window appears when a user selects the **Open Voice Commands Window** command, either by speaking the command or right-clicking the character and choosing the command from the character's pop-up menu.</span></span> <span data-ttu-id="31968-109">不過，如果使用者停用語音輸入，則無法存取 [ **語音命令] 視窗** 。</span><span class="sxs-lookup"><span data-stu-id="31968-109">However, if the user disables speech input, the **Voice Commands Window** is not accessible.</span></span>

<span data-ttu-id="31968-110">[ **語音命令] 視窗** 會將啟用語音的命令顯示為樹狀結構。</span><span class="sxs-lookup"><span data-stu-id="31968-110">The **Voice Commands Window** displays voice-enabled commands as a tree.</span></span> <span data-ttu-id="31968-111">如果目前的裝載應用程式提供語音命令，它們會出現在視窗的頂端。</span><span class="sxs-lookup"><span data-stu-id="31968-111">If the current hosting application supplies voice commands, they appear expanded and at the top of the window.</span></span> <span data-ttu-id="31968-112">使用字元的其他應用程式也會顯示專案。</span><span class="sxs-lookup"><span data-stu-id="31968-112">Entries also appear for other applications using the character.</span></span> <span data-ttu-id="31968-113">此視窗也包含 Microsoft Agent 提供的全域語音命令。</span><span class="sxs-lookup"><span data-stu-id="31968-113">The window also includes the global voice commands supplied by Microsoft Agent.</span></span> <span data-ttu-id="31968-114">如果目前的裝載應用程式沒有語音命令，全域語音命令會出現在視窗的頂端。</span><span class="sxs-lookup"><span data-stu-id="31968-114">If the current hosting application has no voice commands, the global voice commands appear expanded and at the top of the window.</span></span>
