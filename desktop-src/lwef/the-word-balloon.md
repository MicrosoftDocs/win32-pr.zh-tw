# <a name="the-word-balloon"></a>文字提示

\[Microsoft Agent 已于 Windows 7 淘汰，在後續版本的 Windows 中可能無法使用。\]

除了說出音訊輸出，Microsoft Agent 介面也支援以卡通樣式的文字批註形式，以文字輸出的形式來支援文字字幕。 單字會在說話時出現在氣球中。 當讀出的輸出完成時，會隱藏球。 [先進字元選項] 視窗提供的選項可用來停用氣球的顯示，以及控制項的外觀相關屬性。
