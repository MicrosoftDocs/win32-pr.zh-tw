# <a name="speech-output"></a>語音輸出

\[Microsoft Agent 已于 Windows 7 淘汰，在後續的 Windows 版本中可能無法使用。\]

就像語音輸入一樣，語音輸出是一種熟悉且自然的通訊形式，因此它也是以字元為基礎的介面中的適當補充。 不過，語音輸出也有其負債。 在某些環境中，語音輸出可能不是慣用或聲音。 此外，語音也不可見，而且沒有持續性輸出，而是大量依賴短期記憶體。 這些因素會限制處理大量資訊的容量和速度。 同樣地，語音輸出也可能會中斷使用者輸入，特別是當語音是輸入方法時。 語音引擎通常有少量的支援，可讓使用者在語音或其他音訊有輸出頻道時中斷。

因此，請避免使用語音作為輸出的專屬形式。 不過，由於 Microsoft Agent 會提供字元作為 Windows 介面的一部分，因此它會比僅限語音環境提供數個優點。 字元可以與其他形式的輸入和輸出結合，以便顯示選項和動作，讓介面比僅限視覺或語音型的介面更有效。

此外，若要讓語音輸出更容易看見，Microsoft 代理程式會提供以卡通樣式的文字氣球撰寫字元的選項。 其他設定可讓您決定文字在球標中的顯示方式，以及何時移除批註提示。 您也可以決定要使用的字型。 雖然您可以設定字元的文字氣球屬性，但請注意，使用者可以覆寫這些設定。
