# <a name="input-services"></a>輸入服務

\[Microsoft Agent 已于 Windows 7 淘汰，在後續的 Windows 版本中可能無法使用。\]

用戶端應用程式會提供主要使用者介面，以便與字元互動。 您可以程式設計一個字元來回應任何形式的輸入，從按鈕點按下輸入的文字。 此外，Microsoft Agent 也提供事件，讓您可以程式設計當使用者按一下、按兩下或拖曳字元時會發生什麼事。 伺服器會傳遞這些事件之指標和任何輔助按鍵狀態的座標。
