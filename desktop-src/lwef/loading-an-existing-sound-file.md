# <a name="loading-an-existing-sound-file"></a>載入現有的音效檔

\[Microsoft Agent 已于 Windows 7 淘汰，在後續版本的 Windows 中可能無法使用。\]

您也可以 **選擇 [檔案] 功能表或** 工具列中的 [**開啟**] 命令，將現有的 Windows 音效檔 ( .wav) 或語言增強的音效檔 ( lwv) 。 這會顯示 [ **開啟** ] 對話方塊。 選取檔案，然後按一下 [ **開啟** ]，將檔案載入編輯器中。

![載入現有檔案時顯示 [開啟] 對話方塊的螢幕擷取畫面。](images/f2listdial.gif)
