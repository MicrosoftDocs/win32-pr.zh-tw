# <a name="the-output-page"></a>輸出頁面

\[Microsoft Agent 已于 Windows 7 淘汰，在後續版本的 Windows 中可能無法使用。\]

此頁面包含控制字元輸出的屬性。 例如，使用者可以決定是否要在文字提示字元中顯示輸出、決定如何顯示氣球輸出、以音訊播放語音輸出、播放字元音效效果、顯示重新開機提示，以及調整說話速度。
