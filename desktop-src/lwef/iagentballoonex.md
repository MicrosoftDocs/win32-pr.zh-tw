# <a name="iagentballoonex"></a>IAgentBalloonEx

\[Microsoft Agent 已于 Windows 7 淘汰，在後續版本的 Windows 中可能無法使用。\]

[**IAgentBalloonEx**](/windows/desktop/lwef/iagentballoon) 衍生自 **IAgentBalloon** 介面。 它包含所有的 **IAgentBalloon** 方法，並提供其他函式的存取權。

**依照 Vtable 順序的方法**



| IAgentBalloonEx 方法                                  | Description                                                        |
|----------------------------------------------------------|--------------------------------------------------------------------|
| [**GetStyle**](https://www.bing.com/search?q=**GetStyle**)                     | 傳回文字球的輸出樣式。                           |
| [**>setstyle**](https://www.bing.com/search?q=**SetStyle**)                     | 設定文字球的輸出樣式。                              |
| [**SetNumLines**](https://www.bing.com/search?q=**SetNumLines**)               | 設定文字球中的數位行輸出。                  |
| [**SetNumCharsPerLine**](https://www.bing.com/search?q=**SetNumCharsPerLine**) | 設定文字氣球中每一行輸出的字元數。 |
