# <a name="use-the-active-voice"></a>使用主動語音

\[Microsoft Agent 已于 Windows 7 淘汰，在後續的 Windows 版本中可能無法使用。\]

使用「語音輸出」來提供指示詞資訊或要得出使用者回應時，請使用主動語音，並清楚地指定使用者的預期動作。 下列範例說明差異：



|                                      |                               |
|--------------------------------------|-------------------------------|
| 讓我重複您的號碼。           | 無使用者動作                |
| 將會重複此數位。         | 被動式語音，無使用者動作 |
| 在號碼重複時接聽。 | 被動式聲音                 |
| 接聽重複。            | 最佳選擇                   |



 

此外，請建立您的輸出，在片語的結尾處展開重要資訊，如下列範例所示：



|             |                                      |                                            |
|-------------|--------------------------------------|--------------------------------------------|
| 不要這樣撰寫： | 「有三個下一個數位？」           | 使用：「是下一個數位三？」            |
| 不要這樣撰寫： | 「按一下 [確定] 開始。」                 | 使用：「若要開始，請按一下 [確定]。」                 |
| 不要這樣撰寫： | 「說「完成」以完成您的訂單。」 | 使用： "若要完成您的訂單，請說「完成」。 |
