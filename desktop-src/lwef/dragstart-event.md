# <a name="dragstart-event"></a>DragStart 事件

\[Microsoft Agent 已于 Windows 7 淘汰，在後續的 Windows 版本中可能無法使用。\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**描述**
</dt> <dd>

發生于使用者開始拖曳字元時。

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**語法**
</dt> <dd>

**子***代理程式 * * * \_ DragStart* *  **(ByVal** *CharacterID*、 **(ByVal** *按鈕*、 **(byval** *Shift*、 **(byval** *X*、 **(byval** *Y * * * )**



| 部分          | 描述                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
|---------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *CharacterID* | 以字串形式傳回已按下字元的識別碼。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| *按鈕*      | 傳回整數，這個整數會識別已按下並釋放以引發事件的按鈕。 按鈕引數是位在左按鈕 (位 0) 、右按鈕 (位 1) ，以及中間按鈕 (位 2) 的位位。 這些位會分別對應至值1、2和4。 只會設定其中一個位，表示造成事件的按鈕。                                                                                                                                                                                                                                                                                |
| *Shift 鍵*       | 當按下或放開按鈕引數中所指定的按鈕時，傳回對應至 SHIFT、CTRL 和 ALT 鍵狀態的整數。 如果金鑰已關閉，則會設定位。 Shift 引數是具有最高有效位（對應至 SHIFT 鍵 (bit 0) 、CTRL 鍵 (bit 1) 和 ALT 鍵 (bit 2) 的位位。 這些位會分別對應至值1、2和4。 Shift 引數表示這些索引鍵的狀態。 您可以設定部分、全部或全部的位，表示未按下部分、全部或未按下任何鍵。 例如，如果按下 CTRL 和 ALT，shift 的值會是6。 |
| *X、Y*         | 傳回整數，這個整數會指定滑鼠指標的目前位置。 X 和 Y 值一律以圖元表示，相對於畫面的左上角。                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |



 

</dd> </dl>

### <a name="remarks"></a>備註

此事件只會傳送到字元的輸入-主動用戶端。 當使用者拖曳沒有輸入-主動用戶端的字元時，伺服器會將它的最後一個輸入-主動用戶端設定為目前輸入-主動用戶端，將 [**ActivateInput**](activateinput-event.md) 事件傳送到該用戶端，然後傳送 **DragStart** 事件。
