# <a name="mft_enum_video_renderer_extension_profile-attribute"></a>MFT \_ 列舉 \_ 影片轉譯器延伸模組 \_ \_ \_ 配置檔案屬性

包含 UWP 應用程式資訊清單檔案的 **VideoRendererExtensionProfiles** 標記中所有 **VideoRendererExtensionProfile** 專案的清單。 [**MFTEnumEx**](/windows/desktop/api/mfapi/nf-mfapi-mftenumex) 會將此資訊儲存在 [**IMFActivate**](/windows/desktop/api/mfobjects/nn-mfobjects-imfactivate) 物件的屬性存放區上 **MFTEnumEx** 為 MFTs 建立的，而該具有包含 **VideoRendererExtensionProfiles** 標記的相關聯 UWP 資訊清單。
