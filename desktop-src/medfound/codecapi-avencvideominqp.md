# <a name="codecapi_avencvideominqp-property"></a>CODECAPI \_ AVEncVideoMinQP 屬性

指定適用于影片編碼 (QP) 的最小量化參數。

## <a name="data-type"></a>資料類型

**ULONG** (VT \_ UI4) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVEncVideoMinQP**

## <a name="property-value"></a>屬性值

值是用於變數-QP 編碼的最小 QP。
