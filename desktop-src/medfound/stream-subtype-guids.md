# <a name="stream-subtype-guids"></a>資料流程子類型 Guid

下列子類型 Guid 會定義在標頭檔 mfapi 中。 若要指定子類型，請在媒體類型上設定 [ [**MF \_ MT \_ 子類型**](mf-mt-subtype-attribute.md) ] 屬性。

使用這些子類型時，請將 [ [MF \_ MT \_ 主要 \_ 類型](mf-mt-major-type-attribute.md) ] 屬性設定為 **MFMediaType \_ Stream**。



| GUID                               | 描述              |
|------------------------------------|--------------------------|
| **MFStreamFormat \_ MPEG2Program**   | MPEG-2 程式資料流程    |
| **MFStreamFormat \_ MPEG2Transport** | MPEG-2 傳輸串流。 |
