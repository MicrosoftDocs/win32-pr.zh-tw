# <a name="mfpkey_fullframerate-property"></a>MFPKEY \_ FULLFRAMERATE 屬性

指定是否允許 endoder 卸載框架。

## <a name="constant-for-ipropertybag"></a>IPropertyBag 的常數

**g \_ wszWMVCFullFrameRate**

## <a name="data-type"></a>資料類型

**VT \_ I4**

## <a name="remarks"></a>備註

如果您將此屬性設定為1，則編碼器不會捨棄畫面格。 如果您將此屬性設定為0，則編碼器可能會捨棄框架。
