# <a name="codecapi_avencadaptivemode-property"></a>CODECAPI \_ AVEncAdaptiveMode 屬性

指定影片編碼器的自我調整編碼模式。

## <a name="data-type"></a>資料類型

**ULONG** (VT \_ UI4) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVEncAdaptiveMode**

## <a name="property-value"></a>屬性值

這個屬性的值是 [**eAVEncAdaptiveMode**](/windows/win32/api/codecapi/ne-codecapi-eavencadaptivemode)列舉中值的位 **or** 。
