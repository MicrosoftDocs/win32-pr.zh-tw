# <a name="codecapi_avencvideoselectlayer-property"></a>CODECAPI \_ AVEncVideoSelectLayer 屬性

設定影片編碼器的影片選取圖層。

## <a name="data-type"></a>資料類型

**ULONG (VT \_ UI4)**

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVEncVideoSelectLayer**

## <a name="remarks"></a>備註

這個屬性也會與 h.264 [UVC 1.5 攝影機編碼器](camera-encoder-h264-uvc-1-5.md)一起使用。
