# <a name="mf_devicestream_frameserver_shared-attribute"></a>MF \_ DEVICESTREAM \_ FRAMESERVER \_ 共用屬性

在資料流程上設定此屬性時，會明確地將該資料流程標示為框架伺服器所共用的資料流程。

## <a name="data-type"></a>資料類型

**UINT32**

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|-------------------------------------|------------------------------------------------------------------------------------|
| 最低支援的用戶端<br/> | Windows 10， \[ 僅限1703版桌面應用程式\]<br/>                          |
| 最低支援的伺服器<br/> | 都不支援<br/>                                                          |
| 標頭<br/>                   | <dl> <dt>Mfidl。h</dt> </dl> |
