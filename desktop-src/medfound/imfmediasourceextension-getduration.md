# <a name="imfmediasourceextensiongetduration-method"></a>IMFMediaSourceExtension：： GetDuration 方法

取得媒體來源的持續時間（100-毫微秒單位）。

## <a name="syntax"></a>語法


```C++
double GetDuration();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

媒體來源的持續時間（100-毫微秒單位）。
