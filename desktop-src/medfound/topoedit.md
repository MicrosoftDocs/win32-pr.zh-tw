# <a name="topoedit"></a>TopoEdit

TopoEdit 是用來建立和測試拓撲的視覺化檢視。 透過 TopoEdit，您可以執行下列工作：

-   藉由新增各種拓撲節點（例如來源、轉換和輸出節點）來建立拓撲。
-   連線並解決拓撲結構。
-   透過播放來測試拓撲的功能。

從 Windows Server 2008 的 Windows SDK 開始，Windows SDK 可以使用 TopoEdit。
