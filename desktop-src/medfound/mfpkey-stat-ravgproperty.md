# <a name="mfpkey_stat_ravg-property"></a>MFPKEY \_ STAT \_ RAVG 屬性

指定編碼資料流程的平均位元速率（以每秒位數為單位）。 唯讀。

## <a name="constant-for-ipropertybag"></a>IPropertyBag 的常數

**g \_ wszWMVCAvgBitrate**

## <a name="data-type"></a>資料類型

**VT \_ I4**

## <a name="remarks"></a>備註

若要判斷編碼資料流程的平均位元速率，請在編碼結束時讀取這個屬性。
