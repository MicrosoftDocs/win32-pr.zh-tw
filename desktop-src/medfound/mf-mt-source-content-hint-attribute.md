# <a name="mf_mt_source_content_hint-attribute"></a>MF \_ MT \_ 來源 \_ 內容 \_ 提示屬性

描述影片媒體類型的預期外觀比例。

## <a name="data-type"></a>資料類型

**UINT32**

## <a name="remarks"></a>備註

這個屬性的值是 [**MFVideoSrcContentHintFlags**](/windows/desktop/api/mfapi/ne-mfapi-mfvideosrccontenthintflags) 列舉的成員。

這個屬性的 GUID 常數是從 mfuuid 匯出。
