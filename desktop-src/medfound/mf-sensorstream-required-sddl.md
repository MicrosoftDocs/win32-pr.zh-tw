# <a name="mf_sensorstream_required_sddl-attribute"></a>MF \_ SENSORSTREAM \_ 必要的 \_ SDDL 屬性

這個屬性是用來指定在資料流程上 (SDDL) 的安全描述項定義語言，以便為指定的感應器指定更細緻的存取權限。

## <a name="data-type"></a>資料類型

**WCHAR\***

| 需求 | 值 |
|-------------------------------------|------------------------------------------------------------------------------------|
| 最低支援的用戶端<br/> | Windows 10， \[ 僅限1703版桌面應用程式\]<br/>                          |
| 最低支援的伺服器<br/> | 都不支援<br/>                                                          |
