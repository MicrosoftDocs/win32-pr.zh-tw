# <a name="mf_stf_version_info-attribute"></a>MF \_ STF \_ 版本 \_ 資訊屬性

此屬性代表版本資訊感應器轉換 factory。 這是感應器轉換所需的唯一屬性。 此值是由「感應器轉換開發人員」定義，媒體管線會將其視為不透明。

## <a name="data-type"></a>資料類型

**WCHAR\***
