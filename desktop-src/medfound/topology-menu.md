# <a name="topology-menu"></a>拓撲功能表

[ **拓撲** ] 功能表提供下列命令：

<colgroup>
<col style="width: 50%" />
<col style="width: 50%" />
</colgroup>
<thead>
<tr class="header">
<th>命令</th>
<th>描述</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><strong>新增來源 .。。</strong></td>
<td>新增指定媒體檔案的來源節點。 請參閱 <a href="adding-source-nodes-with-topoedit.md">使用 TopoEdit 新增來源節點</a>。</td>
</tr>
<tr class="even">
<td><strong>新增 SAR</strong></td>
<td>將串流音訊轉譯器的輸出節點新增 (SAR) 。 請參閱 <a href="adding-output-nodes-with-topoedit.md">使用 TopoEdit 新增輸出節點</a>。</td>
</tr>
<tr class="odd">
<td><strong>新增 EVR</strong></td>
<td>新增 (EVR) 之增強型影片轉譯器的輸出節點。 請參閱 <a href="adding-output-nodes-with-topoedit.md">使用 TopoEdit 新增輸出節點</a>。</td>
</tr>
<tr class="even">
<td><strong>新增轉換 .。。</strong></td>
<td>加入指定的轉換節點。 請參閱 <a href="adding-transform-nodes-with-topoedit.md">使用 TopoEdit 新增轉換節點</a>。</td>
</tr>
<tr class="odd">
<td><strong>新增 t a</strong></td>
<td>加入用來將輸出傳送至多個輸出節點的 t 節點。</td>
</tr>
<tr class="even">
<td><strong>新增自訂 MFT .。。</strong></td>
<td>將自訂媒體基礎轉換 (MFT) 。 請參閱 <a href="adding-transform-nodes-with-topoedit.md">使用 TopoEdit 新增轉換節點</a>。</td>
</tr>
<tr class="odd">
<td><strong>新增自訂接收器 .。。</strong></td>
<td>加入自訂資料流程接收的輸出節點。 請參閱 <a href="adding-output-nodes-with-topoedit.md">使用 TopoEdit 新增輸出節點</a>。</td>
</tr>
<tr class="even">
<td><strong>新增影片捕獲來源 .。。</strong></td>
<td>新增影片捕獲裝置的來源節點。<br/>
<blockquote>
[!Note]<br />
這項功能需要 Windows 7。
</blockquote>
<br/></td>
</tr>
<tr class="odd">
<td><strong>新增音訊捕獲來源 .。。</strong></td>
<td>新增音訊捕獲裝置的來源節點。<br/>
<blockquote>
[!Note]<br />
這項功能需要 Windows 7。
</blockquote>
<br/></td>
</tr>
<tr class="even">
<td><strong>解析拓撲</strong></td>
<td>建立完整的播放拓撲。 請參閱 <a href="resolving-a-topology-with-topoedit.md">使用 TopoEdit 解析拓撲</a>。</td>
</tr>
<tr class="odd">
<td><strong>刪除</strong></td>
<td>從 [ <strong>拓撲] 窗格</strong>中移除專案、拓撲節點或節點連接。</td>
</tr>
