# <a name="topology-menu"></a>拓撲功能表

[ **拓撲** ] 功能表提供下列命令：




| 命令 | 描述 | 
|---------|-------------|
| <strong>新增來源 .。。</strong> | 新增指定媒體檔案的來源節點。 請參閱 <a href="adding-source-nodes-with-topoedit.md">使用 TopoEdit 新增來源節點</a>。 | 
| <strong>新增 SAR</strong> | 將串流音訊轉譯器的輸出節點新增 (SAR) 。 請參閱 <a href="adding-output-nodes-with-topoedit.md">使用 TopoEdit 新增輸出節點</a>。 | 
| <strong>新增 EVR</strong> | 新增 (EVR) 之增強型影片轉譯器的輸出節點。 請參閱 <a href="adding-output-nodes-with-topoedit.md">使用 TopoEdit 新增輸出節點</a>。 | 
| <strong>新增轉換 .。。</strong> | 加入指定的轉換節點。 請參閱 <a href="adding-transform-nodes-with-topoedit.md">使用 TopoEdit 新增轉換節點</a>。 | 
| <strong>新增 t a</strong> | 加入用來將輸出傳送至多個輸出節點的 t 節點。 | 
| <strong>新增自訂 MFT .。。</strong> | 將自訂媒體基礎轉換 (MFT) 。 請參閱 <a href="adding-transform-nodes-with-topoedit.md">使用 TopoEdit 新增轉換節點</a>。 | 
| <strong>新增自訂接收器 .。。</strong> | 加入自訂資料流程接收的輸出節點。 請參閱 <a href="adding-output-nodes-with-topoedit.md">使用 TopoEdit 新增輸出節點</a>。 | 
| <strong>新增影片捕獲來源 .。。</strong> | 新增影片捕獲裝置的來源節點。<br /><blockquote>[!Note]<br />這項功能需要 Windows 7。</blockquote><br /> | 
| <strong>新增音訊捕獲來源 .。。</strong> | 新增音訊捕獲裝置的來源節點。<br /><blockquote>[!Note]<br />這項功能需要 Windows 7。</blockquote><br /> | 
| <strong>解析拓撲</strong> | 建立完整的播放拓撲。 請參閱 <a href="resolving-a-topology-with-topoedit.md">使用 TopoEdit 解析拓撲</a>。 | 
| <strong>刪除</strong> | 從 [ <strong>拓撲] 窗格</strong>中移除專案、拓撲節點或節點連接。 |
