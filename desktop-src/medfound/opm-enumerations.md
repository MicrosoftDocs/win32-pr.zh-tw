# <a name="opm-enumerations"></a>OPM 列舉

[輸出保護管理員](output-protection-manager.md) (OPM) 會使用下列常數和列舉類型。



| 列舉型別                                                                    | 描述                                                                                                                                                                             |
|--------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**OPM 匯流排類型旗標**](opm-bus-type-flags.md)                               | 指定圖形介面卡所使用的 i/o 匯流排類型。                                                                                                                             |
| [**OPM 連接器類型旗標**](opm-connector-type-flags.md)                   | 指定影片輸出的實體連接器類型。                                                                                                                            |
| [**OPM 保護類型旗標**](opm-protection-type-flags.md)                 | 指定輸出保護管理員 (OPM) 的輸出保護機制。                                                                                                             |
| [**OPM 狀態旗標**](opm-status-flags.md)                                   | 指定 OPM 會話的狀態。                                                                                                                                                 |
| [**OPM \_ ACP \_ 保護 \_ 等級**](/windows/desktop/api/opmapi/ne-opmapi-opm_acp_protection_level)                | 指定類比禁止複製 (ACP) 的保護層級。                                                                                                                        |
| [**OPM \_ DPCP \_ 保護 \_ 等級**](/windows/desktop/api/opmapi/ne-opmapi-opm_dpcp_protection_level)              | 指定 DisplayPort 內容保護 (DPCP) 的保護層級。                                                                                                               |
| [**OPM \_ HDCP \_ 保護 \_ 等級**](/windows/desktop/api/opmapi/ne-opmapi-opm_hdcp_protection_level)              | 指定 High-Bandwidth 數位內容保護 (HDCP) 的保護層級。                                                                                                    |
| [**OPM \_ 影像 \_ 外觀 \_ 比例 \_ EN300294**](/windows/desktop/api/opmapi/ne-opmapi-opm_image_aspect_ratio_en300294) | 指定 ETSI EN 300 294 的長寬比。                                                                                                                                         |
| [**OPM \_ 影片 \_ 輸出 \_ 語義**](/windows/desktop/api/opmapi/ne-opmapi-opm_video_output_semantics)            | 指定 [**IOPMVideoOutput**](/windows/desktop/api/opmapi/nn-opmapi-iopmvideooutput) 介面是否會使用輸出保護管理員 (OPM) 的語義或認證的輸出保護通訊協定 (COPP) 語義。 |
