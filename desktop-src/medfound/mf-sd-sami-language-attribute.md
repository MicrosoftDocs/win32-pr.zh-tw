# <a name="mf_sd_sami_language-attribute"></a>MF \_ SD \_ 薩米文 \_ 語言屬性

包含針對資料流程所定義的已同步可存取媒體交換 (薩米) 語言名稱。

這個屬性會出現在從薩米文媒體來源傳回的資料流程描述項中。

## <a name="data-type"></a>資料類型

寬字元字串

## <a name="remarks"></a>備註

薩米文的語言名稱是在薩米文的檔案中指定。

這個屬性的 GUID 常數是從 mfuuid 匯出。
