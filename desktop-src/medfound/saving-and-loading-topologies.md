# <a name="saving-and-loading-topologies"></a>儲存和載入拓撲

透過 TopoEdit，您可以將拓撲儲存為 XML 檔案，並于稍後載入。 載入完整拓撲時，必須重新解析。 如需此程式的詳細資訊，請參閱 [使用 TopoEdit 解決拓撲](resolving-a-topology-with-topoedit.md)。

## <a name="to-save-a-topology"></a>若要儲存拓撲

1.  在 [檔案] 功能表上，按一下 [儲存]。

    [ **儲存** 盤案] 對話方塊隨即開啟。

2.  流覽至必須儲存拓撲檔案的資料夾。

3.  在 [ **檔案名：** ] 欄位中，輸入檔案的名稱。

4.  按一下 [儲存]。

拓撲檔案將會儲存為 XML 檔案。

## <a name="to-load-a-saved-topology"></a>載入已儲存的拓撲

1.  在 **[檔案]** 功能表上按一下 **[開啟舊檔]**。

    [ **開啟** 檔案] 對話方塊隨即開啟。

2.  流覽至拓撲 XML 檔案所在的資料夾。

3.  在 [ **檔案名：** ] 欄位中指定檔案名，或從清單中選取該檔案。

4.  按一下 [開啟]。

拓撲檔案會載入 **拓撲窗格** 中。
