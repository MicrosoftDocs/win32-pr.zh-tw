# <a name="decoding-to-spdif-microsoft-media-foundation"></a>解碼至 S/PDIF (Microsoft 媒體基礎) 

用於轉換 Windows Media 音訊 9.1 Professional 內容以透過索尼/Philips 數位互連格式傳遞的內容（ (S/PDIF) 是 CLSID CWMAudioSpdTxDMO）之解碼器物件的類別識別碼 \_ 。 您可以使用此物件，就像使用任何其他的解碼器一樣。 它支援與 Windows Media 音訊9解碼器相同的屬性。 此解碼器列舉的輸出類型都是 S/PDIF 格式。

若要完全支援 S/PDIF 輸出，用戶端電腦必須安裝特殊硬體。
