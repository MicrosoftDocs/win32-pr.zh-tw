# <a name="adding-source-nodes-with-topoedit"></a>使用 TopoEdit 新增來源節點

來源節點代表媒體檔案中的資料流程。 若要在 TopoEdit 中建立來源節點，請指定媒體檔案。 TopoEdit 會列舉檔案中的資料流程，並建立適當的來源節點。

如需使用媒體基礎 Api 以程式設計方式新增來源節點的相關資訊，請參閱 [建立來源節點](creating-source-nodes.md)。

## <a name="to-add-a-source-node-to-a-topology"></a>將來源節點新增至拓撲

1.  在 [ **拓撲** ] 功能表上，按一下 [ **新增來源**]。

    [ **選取媒體來源** ] 對話方塊隨即開啟。

2.  流覽至媒體檔案所在的資料夾。

3.  在 [ **檔案名：** ] 欄位中，輸入檔案的名稱。

4.  按一下 [開啟]。

TopoEdit 會建立資料流程的來源節點。 [ **拓撲] 窗格** 會顯示灰色方塊中所含的來源節點，以顯示媒體檔案的名稱。 來源節點會顯示節點的資料流程類型。
