# <a name="why-does-the-audio-encoder-report-no-supported-input-types"></a>音訊編碼器為什麼不會報告任何支援的輸入類型？

除非您設定輸出類型，否則音訊編碼器無法判斷支援的輸入類型。 列舉輸出類型，然後選取符合您需求的類型。 設定輸出類型之後，物件會傳回支援的輸入類型。
