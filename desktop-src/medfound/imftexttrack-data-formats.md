# <a name="imftimedtexttrack-data-format-guids"></a>IMFTimedTextTrack 資料格式 Guid

下列常數會識別 [**IMFTimedTextTrack**](/windows/desktop/api/mfmediaengine/nn-mfmediaengine-imftimedtexttrack)的基礎資料格式。 藉由呼叫 [**IMFTimedTextTrack：： GetDataFormat**](/windows/desktop/api/mfmediaengine/nf-mfmediaengine-imftimedtexttrack-getdataformat)來取得計時文字播放軌的格式。

<dl> <dt>

<span id="MFSubtitleFormat_TTML"></span><span id="mfsubtitleformat_ttml"></span><span id="MFSUBTITLEFORMAT_TTML"></span>**MFSubtitleFormat \_ TTML**
</dt> <dd> <dl> <dt>



此追蹤會使用計時文字標記語言 (TTML) 格式。


</dt> </dl> </dd> <dt>

<span id="MFSubtitleFormat_ATSC"></span><span id="mfsubtitleformat_atsc"></span><span id="MFSUBTITLEFORMAT_ATSC"></span>**MFSubtitleFormat \_ ATSC**
</dt> <dd> <dl> <dt>



此曲目使用 Advanced 電視 Systems 委員會 (ATSC) 格式。


</dt> </dl> </dd> <dt>

<span id="MFSubtitleFormat_ATSC"></span><span id="mfsubtitleformat_atsc"></span><span id="MFSUBTITLEFORMAT_ATSC"></span>**MFSubtitleFormat \_ ATSC**
</dt> <dd> <dl> <dt>



此播放軌使用 Web 影片文字來追蹤 (WebVTT) 格式。


</dt> </dl> </dd> </dl>
