# <a name="mf_mt_max_frame_average_luminance_level-attribute"></a>MF \_ MT \_ 最大 \_ 框架 \_ 平均 \_ 亮度 \_ 層級屬性

指定 Nits 內容之每個畫面格平均亮度層級的最大平均值。 這個屬性與 CEA-861.3 標準中所定義的 **MaxFALL** 具有相同的語義。

## <a name="data-type"></a>資料類型

**UINT32**
