# <a name="mf_mt_interlace_mode-attribute"></a>MF \_ MT \_ 交錯 \_ 模式屬性

描述影片媒體類型中的框架如何交錯。

## <a name="data-type"></a>資料類型

**UINT32**

## <a name="remarks"></a>備註

這個屬性的值是 [**MFVideoInterlaceMode**](/windows/desktop/api/mfobjects/ne-mfobjects-mfvideointerlacemode) 列舉的成員。

這個屬性的 GUID 常數是從 mfuuid 匯出。
