# <a name="overview-of-table-of-contents-parser"></a>目錄剖析器總覽

目錄剖析器 (TOC 剖析器) 是 Windows 元件，可讓您在媒體檔案中儲存 advanced index 資訊。 Advanced index 資訊的範例包括目錄、語義索引和案例面板。

您可以使用 TOC 剖析器撰寫軟體，讓使用者在影片檔案中尋找感興趣的場景。 例如，您可以為使用者提供目錄，讓他或她可以識別影片的主要區段，然後再查看特定區段。 您也可以為使用者提供索引，讓使用者可以找到與特定單字或片語相關的場景清單。

目錄產生器 (TOC 產生器) 是一種與 TOC 剖析器一起運作的 Windows 元件。 TOC 產生器可以偵測影片檔案中的場景和場景，然後產生一份目錄，讓您用來作為軟體的起點。

下列清單提供一些案例，讓您可以使用 TOC 剖析器或 TOC 產生器所提供的服務。

-   從頭開始建立目錄，並將其內嵌于影片檔案中。
-   讀取已內嵌于影片檔案中的目錄。
-   編輯已內嵌于影片檔案中的目錄。
-   您可以使用 TOC 產生器，根據在影片檔案中偵測到的場景和叢集來建立目錄。 編輯產生的目錄，並將其內嵌于影片檔案中。
