# <a name="mf_mt_forward_custom_sei-attribute"></a>MF \_ MT \_ 向前 \_ 自訂 \_ SEI 屬性

指定 (SEI) 單位類型的增補增強資訊應該在輸出範例上，由解碼器轉送。

## <a name="data-type"></a>資料類型

**UINT32**

## <a name="remarks"></a>備註

如果解碼器剖析 SEI，則不會轉送。
