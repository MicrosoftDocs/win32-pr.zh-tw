# <a name="mf_capture_metadata_digitalwindow-attribute"></a>MF \_ CAPTURE \_ METADATA \_ DIGITALWINDOW 屬性

值，表示影片捕獲數位視窗的界限設定。

## <a name="data-type"></a>資料類型

**BLOB**

## <a name="remarks"></a>備註

這個屬性的 **BLOB** 值是 [DigitalWindowSetting](/windows/win32/api/mfapi/ns-mfapi-digitalwindowsetting) 結構。
