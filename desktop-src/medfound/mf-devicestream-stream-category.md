# <a name="mf_devicestream_stream_category-attribute"></a>MF \_ DEVICESTREAM \_ 串流 \_ 類別目錄屬性

表示資料流程類別目錄。

## <a name="data-type"></a>資料類型

**GUID**

## <a name="remarks"></a>備註

此屬性代表識別 ksmedia 中定義之資料流程類別目錄的 GUID。
