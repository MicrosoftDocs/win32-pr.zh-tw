# <a name="mf_capture_engine_mediasource_config-attribute"></a>MF \_ CAPTURE \_ ENGINE \_ MEDIASOURCE \_ CONFIG 屬性

包含 capture 來源的設定屬性。

## <a name="data-type"></a>資料類型

**IPropertyStore \* *_ 儲存為 _* IUnknown\***
