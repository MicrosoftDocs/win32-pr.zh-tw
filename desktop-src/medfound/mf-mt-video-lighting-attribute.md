# <a name="mf_mt_video_lighting-attribute"></a>MF \_ MT \_ 視頻 \_ 光源屬性

指定影片媒體類型的最佳光源條件。

## <a name="data-type"></a>資料類型

**UINT32**

## <a name="remarks"></a>備註

這個屬性的值是 [**MFVideoLighting**](/windows/desktop/api/mfobjects/ne-mfobjects-mfvideolighting) 列舉的成員。

這個屬性的 GUID 常數是從 mfuuid 匯出。
