# <a name="mf_user_data_payload-attribute"></a>MF \_ 使用者 \_ 資料承載 \_ 屬性

設定是否要在輸出範例中包含使用者資料承載。

## <a name="data-type"></a>資料類型

**BLOB**

## <a name="remarks"></a>備註

使用者資料可能會包含資料，例如關閉標題資料或橫條圖資料。

傳遞使用者資料時不會進行任何解碼。
