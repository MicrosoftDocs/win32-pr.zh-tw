## <a name="remarks"></a>備註

DLNA 設定檔識別碼是可識別內容編碼設定檔的字串。

設定檔識別碼會在 [DLNA 網路裝置互通性指南](https://www.dlna.org/about/organization)中說明。  (此資源可能無法在某些語言及國家/地區使用。 )
