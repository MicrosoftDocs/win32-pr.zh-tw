# <a name="codecapi_avencvideocontenttype-property"></a>CODECAPI \_ AVEncVideoContentType 屬性

指出影片編碼器的影片內容類型。

## <a name="data-type"></a>資料類型

**ULONG**

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVEncVideoContentType**

## <a name="property-value"></a>屬性值

這個屬性的值是 [**eAVEncVideoContentType**](/windows/win32/api/codecapi/ne-codecapi-eavencvideocontenttype) 列舉的成員。

## <a name="remarks"></a>備註

此屬性是編碼器的提示，可將其編碼設定優化。
