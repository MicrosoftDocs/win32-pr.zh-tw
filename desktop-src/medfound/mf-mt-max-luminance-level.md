# <a name="mf_mt_max_luminance_level-attribute"></a>MF \_ MT \_ 最大 \_ 亮度 \_ 層級屬性

指定 Nits 中內容的最大亮度層級。 這個屬性與 CEA-861.3 標準中所定義的 **MaxCLL** 具有相同的語義。

## <a name="data-type"></a>資料類型

**UINT32**
