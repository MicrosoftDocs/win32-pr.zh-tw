# <a name="mf_mt_mpeg2_standard-attribute"></a>MF \_ MT \_ MPEG2 \_ 標準屬性

如需描述 MPEG-2 程式串流 (PS) 或傳輸串流 (TS) 的媒體類型，請指定用來多工資料流程的標準。

## <a name="data-type"></a>資料類型

**UINT32**



| 值                                                                                                | 意義                                                                    |
|------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------|
| <span id="0"></span><dl> <dt>**0**</dt> </dl> | 標準 MPEG-2 PS 或 TS。<br/>                                       |
| <span id="1"></span><dl> <dt>**1**</dt> </dl> | 先進的電視系統委員會 (ATSC) 標準。<br/>          |
| <span id="2"></span><dl> <dt>**2**</dt> </dl> | 數位視訊廣播 (DVB-T) 標準。<br/>                      |
| <span id="3"></span><dl> <dt>**3**</dt> </dl> | 無線電產業和企業的關聯 (ARIB) 標準。<br/> |
