# <a name="mf_mt_dv_aaux_src_pack_1-attribute"></a>MF \_ MT \_ DV \_ AAUX \_ SRC \_ PACK \_ 1 屬性

音訊輔助 (AAUX 數位視訊中第二個音訊區塊的) 來源套件 (DV) 媒體類型。

## <a name="data-type"></a>資料類型

**UINT32**

## <a name="remarks"></a>備註

這個屬性會對應至 DirectShow [**DVINFO**](/windows/win32/api/strmif/ns-strmif-dvinfo)結構的 **dwDVAAuxSrc1** 成員。

這個屬性的 GUID 常數是從 mfuuid 匯出。
