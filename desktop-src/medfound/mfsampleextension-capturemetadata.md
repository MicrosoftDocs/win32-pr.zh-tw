# <a name="mfsampleextension_capturemetadata-attribute"></a>MFSampleExtension \_ CaptureMetadata 屬性

與捕捉管線相關之所有中繼資料的 [**IMFAttributes**](/windows/desktop/api/mfobjects/nn-mfobjects-imfattributes) 存放區。

## <a name="data-type"></a>資料類型

**[**IMFAttributes**](/windows/desktop/api/mfobjects/nn-mfobjects-imfattributes)** 儲存為 **IUnknown** 

## <a name="remarks"></a>備註

這個屬性可以存在於任何 [**IMFSample**](/windows/desktop/api/mfobjects/nn-mfobjects-imfsample)上。

這個屬性是唯讀的。
