# <a name="direct3d-9-video-interfaces"></a>Direct3D 9 Video 介面

描述 Microsoft Direct3D 9 video 介面。



| 介面                                                                | 描述                                                                                                |
|--------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**IDirect3D9ExOverlayExtension**](/windows/desktop/api/d3d9/nn-d3d9-idirect3d9exoverlayextension)     | 查詢 Direct3D 裝置的重迭硬體功能。                                            |
| [**IDirect3DAuthenticatedChannel9**](/windows/desktop/api/d3d9/nn-d3d9-idirect3dauthenticatedchannel9) | 提供圖形驅動程式或 Direct3D 執行時間的通道。                        |
| [**IDirect3DCryptoSession9**](/windows/desktop/api/d3d9/nn-d3d9-idirect3dcryptosession9)               | 代表密碼編譯會話。                                                                        |
| [**IDirect3DDevice9Video**](/windows/desktop/api/d3d9/nn-d3d9-idirect3ddevice9video)                   | 讓應用程式使用由圖形驅動程式所執行的內容保護和加密服務。 |
| [**IDirect3DDXVADevice9**](idirect3ddxvadevice9.md)                     | 代表 DirectX Video 加速的硬體加速器 (DXVA) 1.0。                               |
| [**IDirect3DVideoDevice9**](idirect3dvideodevice9.md)                   | 使用 DXVA 版本1.0，從 Direct3D 裝置啟用硬體加速解碼。                      |
