# <a name="codecapi_avdecsoftwaredynamicformatchange-property"></a>CODECAPI \_ AVDecSoftwareDynamicFormatChange 屬性

設定是否要使用軟體動態格式變更進行內部調整大小。

## <a name="data-type"></a>資料類型

**Bool**

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVDecSoftwareDynamicFormatChange**

## <a name="property-value"></a>屬性值

如果設定為 **TRUE**，則會使用軟體動態格式變更進行內部調整大小。
