# <a name="introduction-to-topoedit"></a>TopoEdit 簡介

在媒體基礎中，媒體資料會透過管線層處理，而媒體會話則會控制資料流程並提供播放控制項。 每個管線元件中的資料（來源、轉換和接收器）都會表示為 *拓撲節點*。 *拓撲物件* 是拓撲節點的集合，代表整個媒體會話管線的整個資料流程路徑。

如需拓撲的詳細資訊，請參閱 [關於拓撲](about-topologies.md)。

TopoEdit 是用來建立和測試拓撲的視覺化檢視。 透過 TopoEdit，您可以執行下列工作：

-   藉由新增各種拓撲節點（例如來源、轉換和輸出節點）來建立拓撲。

-   連接並解決拓撲。

-   透過播放來測試拓撲的功能。
