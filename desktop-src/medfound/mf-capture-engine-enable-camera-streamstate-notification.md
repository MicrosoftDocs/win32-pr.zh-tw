# <a name="mf_capture_engine_enable_camera_streamstate_notification-attribute"></a>MF \_ 捕獲 \_ 引擎 \_ 啟用 \_ 攝影機 \_ STREAMSTATE \_ 通知屬性

指出是否應該啟用資料流程狀態通知。

## <a name="data-type"></a>資料類型

## <a name="remarks"></a>備註

非零值指定應啟用通知。 如果屬性不存在，或屬性的值為0，則不會啟用通知。
