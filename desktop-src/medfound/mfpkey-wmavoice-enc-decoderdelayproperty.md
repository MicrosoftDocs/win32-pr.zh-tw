# <a name="mfpkey_wmavoice_enc_decoderdelay-property"></a>MFPKEY \_ WMAVOICE \_ ENC \_ DecoderDelay 屬性

指定包單位中的編碼器延遲。

## <a name="constant-for-ipropertybag"></a>IPropertyBag 的常數

僅可使用 [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore)。

## <a name="data-type"></a>資料類型

VT \_ I4

## <a name="remarks"></a>備註

將這個屬性的值設為3、4或5。 當值從3增加到5時，品質和延遲也會增加。
