# <a name="mf_mt_timestamp_can_be_dts-attribute"></a>MF \_ MT \_ TIMESTAMP \_ 可以 \_ 是 \_ DTS 屬性

指定在設定時間戳記時，解碼器是否可以使用 (DTS) 的解碼時間戳記。

## <a name="data-type"></a>資料類型

**BOOL** 儲存為 **UINT32**
