# <a name="mf_mt_video_3d-attribute"></a>MF \_ MT \_ VIDEO \_ 3d 屬性

指定影片資料流程是否包含3D 內容。

## <a name="data-type"></a>資料類型

**BOOL** 儲存為 **UINT32**

## <a name="remarks"></a>備註

此屬性適用于影片媒體類型。 如果此屬性為 **TRUE**，則影片串流包含3d 內容。 預設值為 **FALSE**。
