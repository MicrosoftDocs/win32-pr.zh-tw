# <a name="mf_mt_max_mastering_luminance-attribute"></a>MF \_ MT \_ 最大 \_ 主控 \_ 亮度屬性

指定在 Nits 中撰寫內容的顯示最大亮度。 這個屬性與 CEA-861.3 標準中所定義的 **max \_ display 掌控 \_ \_ 亮度** 具有相同的語義。

## <a name="data-type"></a>資料類型

**UINT32**
