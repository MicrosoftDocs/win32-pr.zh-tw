# <a name="audio-renderer-attributes"></a>音訊轉譯器屬性

您可以使用下列屬性來設定音訊轉譯器。



| 屬性                                                                                                     | 描述                                                                                                 |
|---------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------|
| [**MF \_ 音訊轉譯器 \_ \_ 屬性 \_ 端點 \_ 識別碼**](mf-audio-renderer-attribute-endpoint-id-attribute.md)     | 指定音訊端點裝置的識別碼。                                                     |
| [**MF \_ 音訊轉譯器 \_ \_ 屬性 \_ 端點 \_ 角色**](mf-audio-renderer-attribute-endpoint-role-attribute.md) | 指定音訊轉譯器的音訊端點角色。                                                   |
| [**MF \_ 音訊轉譯器 \_ \_ 屬性 \_ 旗標**](mf-audio-renderer-attribute-flags-attribute.md)                  | 包含用來設定音訊轉譯器的旗標。                                                             |
| [**MF \_ 音訊轉譯器 \_ \_ 屬性 \_ 會話 \_ 識別碼**](mf-audio-renderer-attribute-session-id-attribute.md)       | 指定音訊轉譯器的音訊原則類別。                                                    |
| [MF \_ 音訊轉譯器 \_ \_ 屬性 \_ 資料流程 \_ 類別目錄](mf-audio-renderer-attribute-stream-category.md)           | 指定 [串流音訊](streaming-audio-renderer.md) 轉譯器的音訊串流類別 (SAR) 。 |
