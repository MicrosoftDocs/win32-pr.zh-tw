# <a name="mfprotectionattribute_fail_over-attribute"></a>MFPROTECTIONATTRIBUTE \_ 故障 \_ 轉移屬性

指出如果最大的努力失敗，保護是否會容錯移轉到此情況。 這個屬性可以與 [**IMFOutputSchema**](/windows/desktop/api/mfidl/nn-mfidl-imfoutputschema) 物件搭配使用。

## <a name="data-type"></a>資料類型

**BOOL** 儲存為 **UINT32**

## <a name="remarks"></a>備註

這個屬性可以與 [**IMFOutputSchema**](/windows/desktop/api/mfidl/nn-mfidl-imfoutputschema) 物件搭配使用。
