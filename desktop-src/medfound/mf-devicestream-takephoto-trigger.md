# <a name="mf_devicestream_takephoto_trigger-attribute"></a>MF \_ DEVICESTREAM \_ TAKEPHOTO \_ 觸發程式屬性

指定是否將「拍攝相片」觸發程式封裝至裝置來源。

## <a name="data-type"></a>資料類型

**UINT32**

## <a name="remarks"></a>備註

將此屬性設定為非零，以將「拍攝相片」觸發程式封裝至裝置來源。
