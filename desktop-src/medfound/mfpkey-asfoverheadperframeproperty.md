# <a name="mfpkey_asfoverheadperframe-property"></a>MFPKEY \_ ASFOVERHEADPERFRAME 屬性

以位元組為單位，指定用來儲存壓縮內容的容器所需的額外負荷（以位元組為單位）。

## <a name="constant-for-ipropertybag"></a>IPropertyBag 的常數

g \_ wszWMVPacketOverhead

## <a name="data-type"></a>資料類型

VT \_ I4

## <a name="default-value"></a>預設值

17

## <a name="remarks"></a>備註

如果您使用 Advanced Systems 格式 (ASF) 檔案結構，請勿變更此值的預設值。 如果您不是將資料儲存在 ASF 檔案中，則必須將此值設定為0。
