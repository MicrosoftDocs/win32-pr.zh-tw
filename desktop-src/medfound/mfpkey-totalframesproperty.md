# <a name="mfpkey_totalframes-property"></a>MFPKEY \_ TOTALFRAMES 屬性

指定在 endocing 過程中傳遞至編碼器的影片畫面總數。

## <a name="constant-for-ipropertybag"></a>IPropertyBag 的常數

g \_ wszWMVCTotalFrames

## <a name="data-type"></a>資料類型

VT \_ I4

## <a name="remarks"></a>備註

完成傳遞範例之後，您可以取得此值。
