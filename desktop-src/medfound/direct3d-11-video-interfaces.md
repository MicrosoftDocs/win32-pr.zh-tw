# <a name="direct3d-11-video-interfaces"></a>Direct3D 11 Video 介面

本章節包含 Microsoft Direct3D 11 video 介面的參考資訊。

| 主題                                                                                 | 描述                                                                                              |
|---------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| [**ID3D11AuthenticatedChannel**](/windows/desktop/api/d3d11/nn-d3d11-id3d11authenticatedchannel)<br/>           | 提供與圖形驅動程式或 Microsoft Direct3D runtime 的通道。 <br/> |
| [**ID3D11CryptoSession**](/windows/desktop/api/d3d11/nn-d3d11-id3d11cryptosession)<br/>                         | 代表密碼編譯會話。 <br/>                                                          |
| [**ID3D11VideoCoNtext**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videocontext)<br/>                           | 提供 Direct3D 11 裝置的影片功能。 <br/>                                    |
| [**ID3D11VideoCoNtext1**](/windows/desktop/api/d3d11_1/nn-d3d11_1-id3d11videocontext1)<br/>                         | 提供 Direct3D 11 裝置的影片功能。<br/>                                     |
| [**ID3D11VideoCoNtext2**](/windows/win32/api/d3d11_4/nn-d3d11_4-id3d11videocontext2)<br/>                         | 提供 Direct3D 11 裝置的影片功能。 <br/>                                    |
| [**ID3D11VideoDecoder**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videodecoder)<br/>                           | 代表 Direct3D 11 的硬體加速影片解碼器。<br/>                              |
| [**ID3D11VideoDecoderOutputView**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videodecoderoutputview)<br/>       | 識別影片解碼期間可以存取的輸出表面。<br/>                    |
| [**ID3D11VideoDevice**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videodevice)<br/>                             | 提供 Direct3D 11 裝置的影片解碼和影片處理功能。 <br/>       |
| [**ID3D11VideoDevice1**](/windows/desktop/api/d3d11_1/nn-d3d11_1-id3d11videodevice1)<br/>                           | 提供 Direct3D 11 裝置的影片解碼和影片處理功能。 <br/>       |
| [**ID3D11VideoProcessor**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videoprocessor)<br/>                       | 代表 Direct3D 11 的視頻處理器。<br/>                                                 |
| [**ID3D11VideoProcessorEnumerator**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videoprocessorenumerator)<br/>   | 列舉 Direct3D 11 裝置的視頻處理器功能。 <br/>                         |
| [**ID3D11VideoProcessorEnumerator1**](/windows/desktop/api/d3d11_1/nn-d3d11_1-id3d11videoprocessorenumerator1)<br/> | 列舉 Direct3D 11 裝置的視頻處理器功能。<br/>                          |
| [**ID3D11VideoProcessorInputView**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videoprocessorinputview)<br/>     | 識別影片處理期間可以存取的輸入介面。<br/>                   |
| [**ID3D11VideoProcessorOutputView**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videoprocessoroutputview)<br/>   | 識別影片處理期間可以存取的輸出表面。<br/>                  |
