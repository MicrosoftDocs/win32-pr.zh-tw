# <a name="mfpkey_conv_inputframerate-property"></a>MFPKEY \_ 約定 \_ INPUTFRAMERATE 屬性

將輸入畫面播放速率指定為比例。

## <a name="constant-for-ipropertybag"></a>IPropertyBag 的常數

僅可使用 [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore)。

## <a name="data-type"></a>資料類型

VT \_ UI8

## <a name="applies-to"></a>套用至

-   [畫面播放速率轉換器](framerateconverter.md)

## <a name="remarks"></a>備註

將分子以較高的4個位元組和分母儲存在較低4個位元組中。
