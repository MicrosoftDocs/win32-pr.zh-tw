# <a name="controls-menu"></a>控制項功能表

[ **控制項** ] 功能表提供在媒體會話上設定之拓撲的傳輸控制命令。 只有在成功解決拓撲之後，才會啟用此功能表。 若要快速存取這些命令，您可以使用 [ [TopoEdit] 工具列](topoedit-toolbar.md)。

[ **控制項** ] 功能表提供下列命令：



| 命令   | 描述                                                              |
|-----------|--------------------------------------------------------------------------|
| **播放**  | 開始播放最後解析的拓撲。                           |
| **停止**  | 停止播放目前現正播放的最後解析拓撲。  |
| **暫停** | 暫停目前現正播放的最後已解決拓撲的播放。 |
