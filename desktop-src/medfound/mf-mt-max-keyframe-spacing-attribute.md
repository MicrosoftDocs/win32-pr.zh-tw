# <a name="mf_mt_max_keyframe_spacing-attribute"></a>MF \_ MT \_ 最大主要畫面格 \_ \_ 間距屬性

影片媒體類型中，從一個主要畫面格到下一個的畫面格最大數目。

## <a name="data-type"></a>資料類型

**UINT32**

## <a name="remarks"></a>備註

這個屬性的 GUID 常數是從 mfuuid 匯出。
