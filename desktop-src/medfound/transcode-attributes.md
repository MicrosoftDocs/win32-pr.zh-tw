# <a name="transcode-attributes"></a>轉碼屬性

下列屬性會與轉碼 API 搭配使用。



| 屬性                                                                          | 描述                                                                                  |
|------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [MF \_ 轉碼 \_ 調整 \_ 設定檔](mf-transcode-adjust-profile.md)                  | 定義轉碼拓撲之資料流程設定的設定檔旗標。                    |
| [MF \_ 轉碼 \_ CONTAINERTYPE](mf-transcode-containertype.md)                     | 指定編碼檔案的容器類型。                                             |
| [MF \_ 轉碼 \_ 沒有 \_ 插入 \_ 編碼器](mf-transcode-donot-insert-encoder.md)     | 指定編碼器是否必須包含在轉碼拓撲中。                     |
| [MF \_ 轉碼 \_ ENCODINGPROFILE](mf-transcode-encodingprofile.md)                 | 指定用於編碼 Advanced 串流格式 (ASF) 檔案的裝置一致性設定檔。 |
| [MF \_ 轉碼 \_ QUALITYVSSPEED](mf-transcode-qualityvsspeed.md)                   | 指定編碼品質與編碼速度之間的取捨。                          |
| [MF \_ 轉碼 \_ 略過 \_ 中繼資料 \_ 傳輸](mf-transcode-skip-metadata-transfer.md) | 指定中繼資料是否要寫入至轉碼檔案。                                |
| [**MF \_ 轉碼 \_ TOPOLOGYMODE**](/windows/desktop/api/mfidl/ne-mfidl-mf_transcode_topologymode_flags)             | 指定拓撲載入器是否會載入以硬體為基礎的轉換。                   |
