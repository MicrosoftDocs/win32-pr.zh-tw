# <a name="table-of-contents-parser-interfaces"></a>目錄剖析器介面

下列介面可證明存取目錄剖析器的服務。



| 介面                                | 描述                                                                                                           |
|------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**IToc**](/windows/desktop/api/wmcodecdsp/nn-wmcodecdsp-itoc)                     | 表示個別的目錄。                                                                           |
| [**ITocCollection**](/windows/desktop/api/wmcodecdsp/nn-wmcodecdsp-itoccollection) | 代表目錄的集合。                                                                        |
| [**ITocEntry**](/windows/desktop/api/wmcodecdsp/nn-wmcodecdsp-itocentry)           | 代表目錄中的個別專案。                                                                |
| [**ITocEntryList**](/windows/desktop/api/wmcodecdsp/nn-wmcodecdsp-itocentrylist)   | 代表目錄中的專案清單。                                                                  |
| [**ITocParser**](/windows/desktop/api/wmcodecdsp/nn-wmcodecdsp-itocparser)         | 提供在影片檔案中內嵌目錄，以及從影片檔案讀取目錄的方法。 |
