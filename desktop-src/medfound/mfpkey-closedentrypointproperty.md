# <a name="mfpkey_closedentrypoint-property"></a>MFPKEY \_ CLOSEDENTRYPOINT 屬性

指定編碼器在一組圖片的開頭使用的編碼模式。

## <a name="constant-for-ipropertybag"></a>IPropertyBag 的常數

**g \_ wszWMVCClosedEntryPoint**

## <a name="data-type"></a>資料類型

**VT \_ BOOL**

## <a name="default-value"></a>預設值

**VARIANT \_ FALSE**

## <a name="remarks"></a>備註

如果您將此屬性設定為 **VARIANT \_ TRUE**，則會 BBIBBP 編碼模式。 如果您將此屬性設定為 **VARIANT \_ FALSE**，則會 IBBPBBP 編碼模式。
