# <a name="mf_sensorstream_required_capabilities-attribute"></a>MF \_ SENSORSTREAM \_ 需要的 \_ 功能屬性

此屬性包含以分號分隔的功能字串清單，可指定特定資料流程所需的功能。 如需可包含在這個屬性中的功能字串清單，請參閱 [DeviceCapability](/uwp/schemas/appxpackage/appxmanifestschema/element-devicecapability)。

## <a name="data-type"></a>資料類型

**WCHAR\***
