# <a name="topoedit-main-window"></a>TopoEdit 主視窗

啟動 TopoEdit 之後，下列螢幕擷取畫面中顯示的視窗隨即出現。

![顯示 topoedit 主視窗的螢幕擷取畫面](images/809bce6e-43b7-4399-a41b-deff894ba151.jpg)

TopoEdit 主視窗元素如下：

-   **拓撲窗格**。 顯示在視窗的左邊。 此區域可用來新增拓撲節點、查看拓撲，以及連接拓撲節點。 如需使用此窗格的詳細資訊，請參閱 [使用 TopoEdit 建立拓撲](building-topologies-by-using-topoedit.md)。
-   [**屬性] 窗格**。 出現在視窗右側。 此窗格會顯示目前所選取拓撲節點的屬性。 如需此窗格所顯示之屬性的相關資訊，請參閱 [查看拓撲資訊](viewing-topology-information.md)。
-   功能表。 請參閱 [TopoEdit 功能表](topoedit-menus.md)。
-   工具 欄。 請參閱 [TopoEdit 工具列](topoedit-toolbar.md)。
