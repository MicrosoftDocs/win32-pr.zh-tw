# <a name="media-session-attributes"></a>媒體會話屬性

下列屬性是用來設定媒體會話。



| 屬性                                                                                            | 描述                                                                                                 |
|------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------|
| [**MF \_ 會話 \_ 內容 \_ 保護 \_ 管理員**](mf-session-content-protection-manager-attribute.md) | 提供回呼介面，讓應用程式接收來自 PMP 會話的內容啟用程式物件。 |
| [**MF \_ 會話 \_ 全球 \_ 時間**](mf-session-global-time-attribute.md)                                | 指定拓撲是否會有全域開始和停止時間。                                        |
| [**MF \_ 會話 \_ 品質 \_ 管理員**](mf-session-quality-manager-attribute.md)                        | 包含媒體會話之品質管制員的 CLSID。                                              |
| [**MF \_ 會話 \_ 遠端 \_ 來源 \_ 模式**](mf-session-remote-source-mode-attribute.md)                 | 指定媒體來源正在遠端進程中執行。                                             |
| [**MF \_ 會話 \_ 伺服器 \_ 內容**](mf-session-server-context-attribute.md)                          | 讓兩個媒體會話實例共用相同的 PMP 進程。                                   |
| [**MF \_ 會話 \_ TOPOLOADER**](mf-session-topoloader-attribute.md)                                   | 包含媒體會話拓撲載入器的 CLSID。                                              |
