# <a name="how-do-i-decompress-streams-from-an-avi-file-or-other-file-about-which-i-have-no-format-information"></a>如何? 從 AVI 檔案或其他沒有格式資訊的檔案解壓縮串流？

如果沒有格式資訊，包括延伸格式資料，您就無法解壓縮以 Windows 媒體編解碼器編碼的資料。 編解碼器是設計用來建立資料流程，以搭配 Advanced 系統格式 (ASF) 檔案容器使用。 ASF 檔案的標頭區段會儲存檔案中所有資料流程的格式資訊。 當您使用在 ASF 檔案以外的其中一個 Windows Media 音訊和影片編解碼器編碼的內容時，您必須確定格式資訊可供使用。 這樣做的方式會因不同的容器而異。
