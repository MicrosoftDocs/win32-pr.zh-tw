# <a name="mfsampleextension_encryption_hardwareprotection-attribute"></a>MFSampleExtension \_ Encryption \_ HardwareProtection 屬性

指定媒體範例是否受硬體保護。

## <a name="data-type"></a>資料類型

**BOOL** 儲存為 **UINT32**

## <a name="remarks"></a>備註

值為0表示範例不受硬體保護。 非零值表示範例受硬體保護。
