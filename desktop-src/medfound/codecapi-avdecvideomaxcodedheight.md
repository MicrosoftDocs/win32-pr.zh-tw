# <a name="codecapi_avdecvideomaxcodedheight-property"></a>CODECAPI \_ AVDecVideoMaxCodedHeight 屬性

設定影片解碼器將接受做為輸入類型的最大圖片高度。

## <a name="data-type"></a>資料類型

**LONG** (**VT \_ I4**) 

## <a name="property-guid"></a>屬性 GUID

CODECAPI \_ AVDecVideoMaxCodedHeight

|-------------------------------------|---------------------------------------------------------------------------------------|
