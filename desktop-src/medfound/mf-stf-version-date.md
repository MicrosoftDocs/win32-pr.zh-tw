# <a name="mf_stf_version_date-attribute"></a>MF \_ STF \_ 版本 \_ 日期屬性

這個屬性會儲存一個 [FILETIME](/windows/desktop/api/minwinbase/ns-minwinbase-filetime) ，代表感應器轉換 factory 的日期/時間戳記。

## <a name="data-type"></a>資料類型

**UINT64**

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|-------------------------------------|------------------------------------------------------------------------------------|
| 最低支援的用戶端<br/> | Windows 10， \[ 僅限1703版桌面應用程式\]<br/>                          |
| 最低支援的伺服器<br/> | 都不支援<br/>                                                          |
| 標頭<br/>                   | <dl> <dt>Mfidl。h</dt> </dl> |
