# <a name="mf_device_thermal_state_changed-attribute"></a>MF \_ 裝置 \_ 熱 \_ 狀態 \_ 已變更屬性

代表裝置中的溫度狀態變更信號的事件。

## <a name="data-type"></a>資料類型

**UINT32**

## <a name="remarks"></a>備註

這個屬性的值會指定裝置的溫度值（以百分比表示）。
