# <a name="media-foundation-pipeline"></a>媒體基礎管線

Microsoft 媒體基礎中的 *管線* 層是直接產生或處理媒體資料的架構層。

大部分的應用程式不會直接在管線元件上呼叫方法，雖然可以這樣做。 如果您要撰寫自訂管線元件，請閱讀這些主題。

## <a name="in-this-section"></a>本節內容



| 主題                                                                     | 描述                                                                                                                           |
|---------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| [媒體來源](media-sources.md)<br/>                             | 媒體來源會產生媒體資料，通常是從檔案、捕獲裝置或網路串流。 <br/>                              |
| [媒體基礎轉換](media-foundation-transforms.md)<br/> | 媒體基礎轉換 (MFTs) 處理媒體資料。 例如，媒體基礎中的編解碼器會實作為 MFTs。 <br/>   |
| [媒體接收器](media-sinks.md)<br/>                                 | 媒體接收器會使用媒體資料。 例如，媒體接收器可能會將資料寫入檔案，或透過網路傳送資料。<br/> |
