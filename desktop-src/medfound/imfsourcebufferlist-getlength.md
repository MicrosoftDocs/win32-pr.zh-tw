# <a name="imfsourcebufferlistgetlength-method"></a>IMFSourceBufferList：： GetLength 方法

取得清單中的 [**IMFSourceBuffer**](/windows/desktop/api/mfmediaengine/nn-mfmediaengine-imfsourcebuffer) 物件數目。

## <a name="syntax"></a>語法


```C++
DWORD GetLength();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

清單中的來源緩衝區數目。
