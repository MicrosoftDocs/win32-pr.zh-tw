# <a name="mf_mt_video_3d_num_views-attribute"></a>MF \_ MT \_ VIDEO \_ 3d \_ NUM \_ VIEWS 屬性

3D 影片順序中的 views 數目。

## <a name="data-type"></a>資料類型

**UINT32**

## <a name="remarks"></a>備註

典型的3D 影片順序包含兩個視圖：左邊和右邊。 目前，這個屬性的值必須是1或2。
