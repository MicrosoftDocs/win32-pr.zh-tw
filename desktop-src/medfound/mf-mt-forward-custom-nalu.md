# <a name="mf_mt_forward_custom_nalu-attribute"></a>MF \_ MT \_ 向前 \_ 自訂 \_ NALU 屬性

指定 (NAL) 單位類型的網路抽象層應該在輸出範例上由解碼器轉送。

## <a name="data-type"></a>資料類型

**UINT32**

## <a name="remarks"></a>備註

如果解碼器剖析 NALU，則不會轉送。
