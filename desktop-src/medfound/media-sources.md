# <a name="media-sources"></a>媒體來源

*媒體來源* 是在媒體基礎管線中產生媒體資料的物件。 本節將詳細說明媒體來源 Api。 如果您要執行自訂媒體來源，或使用媒體基礎管線以外的媒體來源，請閱讀本節。

如果您的應用程式使用控制層，則只需要使用有限的媒體來源 Api 子集。 如需詳細資訊，請參閱 [使用媒體來源和媒體會話](using-media-sources-with-the-media-session.md)的主題。
