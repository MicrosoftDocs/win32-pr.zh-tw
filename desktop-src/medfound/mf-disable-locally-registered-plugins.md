# <a name="mf_disable_locally_registered_plugins-attribute"></a>MF \_ 停用 \_ 本機 \_ 註冊的 \_ 外掛程式屬性

指定是否停用本機註冊的外掛程式。

## <a name="data-type"></a>資料類型

**UINT32**

## <a name="remarks"></a>備註

將此屬性設定為 **TRUE** ，以防止 MF 拓撲載入器插入本機註冊的轉換。
