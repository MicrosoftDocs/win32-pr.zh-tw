# <a name="content-protection-commands"></a>內容保護命令

列出 [**IDirect3DAuthenticatedChannel9：： Configure**](/windows/desktop/api/d3d9/nf-d3d9-idirect3dauthenticatedchannel9-configure) 方法的命令。



| 命令                                                                                                           | 描述                                                                                                           |
|-------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**D3DAUTHENTICATEDCONFIGURE \_ INITIALIZE**](d3dauthenticatedconfigure-initialize.md)                             | 初始化已驗證的通道。                                                                                |
| [**D3DAUTHENTICATEDCONFIGURE \_ CRYPTOSESSION**](d3dauthenticatedconfigure-cryptosession.md)                       | 將密碼編譯會話與 DirectX Video 加速 2 (DXVA-2) 解碼器裝置和 Direct3D 裝置產生關聯。 |
| [**D3DAUTHENTICATEDCONFIGURE \_ ENCRYPTIONWHENACCESSIBLE**](d3dauthenticatedconfigure-encryptionwhenaccessible.md) | 設定在受保護的內容變成可供 CPU 或匯流排存取之前，所執行的加密層級。         |
| [**D3DAUTHENTICATEDCONFIGURE \_ 保護**](d3dauthenticatedconfigure-protection.md)                             | 啟用或停用裝置的保護。                                                                        |
| [**D3DAUTHENTICATEDCONFIGURE \_ SHAREDRESOURCE**](d3dauthenticatedconfigure-sharedresource.md)                     | 讓進程開啟共用資源，或停用開啟共用資源的進程。                     |
