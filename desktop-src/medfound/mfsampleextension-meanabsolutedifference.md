# <a name="mfsampleextension_meanabsolutedifference-attribute"></a>MFSampleExtension \_ MeanAbsoluteDifference 屬性

這個屬性會傳回 Y 平面中所有宏區塊 (MAD) 的平均絕對差異。

## <a name="data-type"></a>資料類型

**UINT32**

## <a name="remarks"></a>備註

應用程式可以使用 [CODECAPI \_ AVEncVideoMeanAbsoluteDifference](codecapi-avencvideomeanabsolutedifference.md)要求編碼器在輸出範例上設定此屬性。
