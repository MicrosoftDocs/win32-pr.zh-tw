# <a name="topoedit-toolbar"></a>TopoEdit 工具列

依預設，工具列會顯示在應用程式視窗的頂端，以及功能表列下方。 工具列可讓您快速存取 [ [控制] 功能表](controls-menu.md) 項。 只有在成功解決拓撲之後，才會啟用工具列按鈕。

下列螢幕擷取畫面顯示 [TopoEdit] 工具列。

![顯示 topoedit 工具列的螢幕擷取畫面](images/410257c5-ae5c-454c-a9a8-ba05fc5235b5.jpg)
