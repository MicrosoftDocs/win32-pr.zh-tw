# <a name="file-menu"></a>檔案功能表

[檔案 **] 功能表提供** 下列命令：



<table>
<colgroup>
<col style="width: 50%" />
<col style="width: 50%" />
</colgroup>
<thead>
<tr class="header">
<th>命令</th>
<th>描述</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><strong>新增功能。。。</strong></td>
<td>清除現有的拓撲，並從 [ <strong>拓撲] 窗格</strong>移除拓撲節點。<br/>
<ul>
<li>如果拓撲目前為開啟狀態，則會提示使用者儲存拓撲。 如需儲存拓撲的詳細資訊，請參閱 <a href="saving-and-loading-topologies.md">儲存和載入</a>拓撲。<br/></li>
<li>建立空的拓撲。 使用者可以將拓撲節點新增至新的拓撲。 如需詳細資訊，請參閱 <a href="building-topologies-by-using-topoedit.md">使用 TopoEdit 建立拓撲</a>。<br/></li>
</ul></td>
</tr>
<tr class="even">
<td><strong>轉譯檔案 .。。</strong></td>
<td>建立媒體檔案的播放拓撲。 TopoEdit 會自動建立並連接拓撲中的節點。 如需詳細資訊，請參閱 <a href="building-a-playback-topology-with-topoedit.md">使用 TopoEdit 建立播放拓撲</a>。</td>
</tr>
<tr class="odd">
<td><strong>轉譯轉碼 .。。</strong></td>
<td>建立轉碼拓撲。 如需詳細資訊，請參閱 <a href="building-a-transcode-topology-with-topoedit.md">使用 TopoEdit 建立轉碼拓撲</a>。<br/>
<blockquote>
[!Note]<br />
這項功能需要 Windows 7。
</blockquote>
<br/></td>
</tr>
<tr class="even">
<td><strong>打開。。。</strong></td>
<td>載入儲存為 XML 檔案的現有拓撲。 請參閱 <a href="saving-and-loading-topologies.md">儲存和載入拓撲</a>。</td>
</tr>
<tr class="odd">
<td><strong>救。。。</strong></td>
<td>將目前開啟的拓撲儲存為 XML 檔案。 請參閱 <a href="saving-and-loading-topologies.md">儲存和載入拓撲</a>。</td>
</tr>
<tr class="even">
<td><strong>結束</strong></td>
<td>關閉 TopoEdit 應用程式。</td>
</tr>
