# <a name="mfsampleextension_forwardeddecodeunittype-attribute"></a>MFSampleExtension \_ ForwardedDecodeUnitType 屬性

指定連接到 [MFSampleExtension \_ ForwardedDecodeUnits](mfsampleextension-forwardeddecodeunits.md)集合中之 [**IMFSample**](/windows/desktop/api/mfobjects/nn-mfobjects-imfsample)的單位類型（NALU 或 SEI）。

## <a name="data-type"></a>資料類型

**UINT32**

## <a name="remarks"></a>備註

這個屬性的值是來自 [**MF \_ 自訂解碼 \_ \_ 單位 \_ 類型**](mf-custom-decode-unit-type-enum.md) 列舉的值。
