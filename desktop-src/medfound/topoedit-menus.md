# <a name="topoedit-menus"></a>TopoEdit 功能表

下表說明 [TopoEdit] 功能表：



| 功能表                               | 描述                                                                    |
|------------------------------------|--------------------------------------------------------------------------------|
| [檔案功能表](file-menu.md)         | 開啟新的或現有的拓撲。                                              |
| [拓撲功能表](topology-menu.md) | 加入來源、轉換和輸出節點。 解析和刪除拓撲節點。 |
| [控制項功能表](controls-menu.md) | 播放、暫停和停止已解決的拓撲。                                  |
| [工具功能表](tools-menu.md)       | 記錄所選取拓撲節點的活動。                                   |
| [說明功能表](help-menu.md)         | 提供 TopoEdit 說明主題的連結。                                   |
