# <a name="codecapi_videoencoderdisplaycontenttype-property"></a>CODECAPI \_ VideoEncoderDisplayContentType 屬性

設定內容是否為全螢幕的影片，而不是可能有較小影片視窗或完全沒有影片的螢幕內容。

## <a name="data-type"></a>資料類型

**VT \_ UI4**

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ VideoEncoderDisplayContentType**

## <a name="property-value"></a>屬性值

如果值為非零值，則內容會是全螢幕的影片。 如果值為零，則內容不是全螢幕的影片。

## <a name="remarks"></a>備註

在 Windows 10 上，不會使用這個屬性的值。
