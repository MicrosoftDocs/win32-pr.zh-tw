# <a name="mfpkey_stat_bmax-property"></a>MFPKEY \_ STAT \_ BMAX 屬性

指定編碼資料流程的最大緩衝區視窗（以毫秒為單位）。 唯讀。

## <a name="constant-for-ipropertybag"></a>IPropertyBag 的常數

**g \_ wszWMVCMaxBitrate**

## <a name="data-type"></a>資料類型

**VT \_ I4**

## <a name="remarks"></a>備註

若要判斷已編碼資料流程的緩衝區視窗上限，請在編碼結束時讀取這個屬性。
