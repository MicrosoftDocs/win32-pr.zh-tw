# <a name="mf_mt_pad_control_flags-attribute"></a>MF \_ MT \_ 板 \_ 控制項 \_ 旗標屬性

指定影片媒體類型之輸出矩形的長寬比。

## <a name="data-type"></a>資料類型

**UINT32**

## <a name="remarks"></a>備註

這個屬性的值是 [**MFVideoPadFlags**](/windows/desktop/api/mfapi/ne-mfapi-mfvideopadflags) 列舉的成員。

這個屬性的 GUID 常數是從 mfuuid 匯出。
