# <a name="axwindowsmediaplayersettings-property"></a>AxWindowsMediaPlayer 設定屬性

settings 屬性會取得 IWMPSettings 介面，以提供修改各種 Windows Media Player 設定的方式。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```CSharp
public IWMPSettings settings {get;}
```


```VB

Public ReadOnly Property settings As IWMPSettings
```





## <a name="property-value"></a>屬性值

WMPLib. IWMPSettings 介面。
