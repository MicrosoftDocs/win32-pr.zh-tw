# <a name="axwindowsmediaplayerdvd-property"></a>AxWindowsMediaPlayer dvd 屬性

Dvd 屬性會取得 IWMPDVD 介面，以提供使用 Dvd 的屬性和方法。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```CSharp
public IWMPDVD dvd {get;}
```


```VB

Public ReadOnly Property dvd As IWMPDVD
```





## <a name="property-value"></a>屬性值

WMPLib. IWMPDVD 介面。
