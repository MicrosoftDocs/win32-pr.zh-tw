# <a name="viewtitle"></a>VIEW. 標題

**Title** 屬性會指定或抓取 **視圖** 的標題。

``` syntax
        elementID.title
```

## <a name="possible-values"></a>可能的值

這個屬性是在設計階段指定的 **字串** ，之後是唯讀的。 長度最多可達255個字元，而且沒有預設值。

## <a name="remarks"></a>備註

指定或取出的標題是主題的顯示名稱。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
