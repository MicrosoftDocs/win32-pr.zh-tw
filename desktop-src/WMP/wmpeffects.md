# <a name="wmpeffects"></a>WMPEFFECTS

這是具有下列預設值的預先定義 **效果** 。

``` syntax
horizontalAlignment="stretch"
verticalAlignment="stretch"
height="200"
width="250"
tabStop="false"
onclick="next();"
```

## <a name="remarks"></a>備註

這會建立 **效果** 元素，當使用者按一下控制項時，將會逐步執行視覺效果預設。 當播放程式調整大小時，也會延展視覺效果。

顯示的初始視覺效果預設是在 [**視覺效果**] 下的 [ **View** ] 功能表上選取的視覺效果。 當播放程式處於面板模式時，變更此功能表上的選取專案將會自動變更此元素所顯示的預設值。 [ **View** ] 功能表會以播放程式的完整模式顯示，或在面板中將 [ **標題** ] 屬性設定為 [true] 時顯示。

此 **效果** 專案的所有屬性都可以透過明確指定來覆寫。
