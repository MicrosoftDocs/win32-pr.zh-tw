# <a name="eventaltkey"></a>altKey

**AltKey** 屬性會抓取值，指出在事件發生時，ALT 鍵是否已關閉。

``` syntax
event.altKey
```

## <a name="possible-values"></a>可能的值

這個屬性是唯讀的 **布林值**。



| 值 | 描述                                     |
|-------|-------------------------------------------------|
| true  | 指出 ALT 鍵位於下位置。 |
| false | 指出 ALT 鍵在上的位置。   |



 

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
