# <a name="titlenum-attribute"></a>titleNum 屬性

**TitleNum** 屬性是 DVD 標題號碼。

## <a name="applies-to"></a>套用至

-   [DVD](dvd-attributes.md)

## <a name="remarks"></a>備註

若要判斷是否可以變更這個屬性的值，請使用 [isReadOnlyItem](media-isreadonlyitem.md) 方法。

| 需求 | 值 |
|--------------------|---------------------------------------------------|
