# <a name="iwmpclosedcaption-vb-and-c-interface"></a>IWMPClosedCaption (VB 和 c # ) 介面

提供一種方式，可包含包含數位媒體檔案的標題。 字幕文字位於已同步處理的可存取媒體交換 (薩米) 檔。

## <a name="members"></a>成員

**IWMPClosedCaption (VB 和 c # )** 介面不會定義任何成員。

**IWMPClosedCaption** 介面會公開下列屬性。



| 屬性                                                                             | 描述                                                                                |
|--------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [captioningId](wmplibiwmpclosedcaption-iwmpclosedcaption-captioningid--vb-and-c.md) | 取得或設定顯示字幕的 HTML 元素名稱。                       |
| [SAMIFileName](wmplibiwmpclosedcaption-iwmpclosedcaption-samifilename--vb-and-c.md) | 取得或設定包含隱藏式輔助字幕所需資訊的檔案名。 |
| [SAMILang](wmplibiwmpclosedcaption-iwmpclosedcaption-samilang--vb-and-c.md)         | 取得或設定針對隱藏式輔助字幕所顯示的語言。                                 |
| [SAMIStyle](wmplibiwmpclosedcaption-iwmpclosedcaption-samistyle--vb-and-c.md)       | 取得或設定隱藏式輔助字幕樣式。                                                  |



 

使用下列屬性來取得 **IWMPClosedCaption** 介面。



| Object                                                            | 屬性                                                                   |
|-------------------------------------------------------------------|----------------------------------------------------------------------------|
| [AxWindowsMediaPlayer](axwindowsmediaplayer-object--vb-and-c.md) | [closedCaption](axwmplib-axwindowsmediaplayer-closedcaption--vb-and-c.md) |
