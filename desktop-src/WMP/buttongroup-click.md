# <a name="buttongroupclick"></a>BUTTONGROUP。按一下

**Click** 方法會使用指定的索引，呼叫針對 **BUTTONELEMENT** 所定義的 **onclick** 事件處理常式。

``` syntax
        elementID.click(button)
```

## <a name="parameters"></a>參數

<dl> <dt>

<span id="button"></span><span id="BUTTON"></span>*按鈕*
</dt> <dd>

**Number** (**Long**) ，其中包含 **BUTTONELEMENT** 的索引，以及要呼叫的 **onclick** 事件處理常式。

</dd> </dl>

## <a name="return-value"></a>傳回值

這個方法不會傳回值。

## <a name="remarks"></a>備註

您可以使用這個方法，提供執行與 **BUTTONGROUP** 內所含按鈕相關聯之程式碼的替代方法。
