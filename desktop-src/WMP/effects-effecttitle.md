# <a name="effectseffecttitle"></a>效果。 effectTitle

**EffectTitle** 方法會使用指定的登錄索引來抓取視覺效果的顯示標題。

``` syntax
        elementID.effectTitle(index)
```

## <a name="parameters"></a>參數

<dl> <dt>

<span id="index"></span><span id="INDEX"></span>*指數*
</dt> <dd>

包含視覺效果登錄索引的 (**長**) **數目**。

</dd> </dl>

## <a name="return-value"></a>傳回值

這個方法會傳回 **字串**。

## <a name="remarks"></a>備註

這個方法是用來在使用者介面中顯示視覺效果標題。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|---------------------------------------------------|
| 版本<br/> | Windows Media Player 9 系列或更新版本<br/> |
