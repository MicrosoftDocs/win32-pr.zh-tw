# <a name="playlistname"></a>Playlist.name

`name`屬性會指定或抓取播放清單的名稱。

## <a name="syntax"></a>Syntax

*玩家*。*currentPlaylist*。**名稱**

## <a name="possible-values"></a>可能的值

這個屬性是讀取/寫入 **字串**。

## <a name="remarks"></a>備註

若要取得這個屬性的值，需要有程式庫的讀取權限。 若要指定值，則需要完整存取權。 如需詳細資訊，請參閱連結 [庫存取](library-access.md)。

**Windows Media Player 10** 行動裝置版：這個屬性是唯讀的。
