# <a name="iwmpcontrols3audiolanguagecount-property"></a>IWMPControls3：： audioLanguageCount 屬性

**AudioLanguageCount** 屬性會取得支援的音訊語言計數。

## <a name="syntax"></a>Syntax


```CSharp
public System.Int32 audioLanguageCount {get; set;}
```


```VB

Public ReadOnly Property audioLanguageCount As System.Int32
```





## <a name="property-value"></a>屬性值

支援的音訊語言計數的 **system.object** 。

## <a name="remarks"></a>備註

針對 Windows Media 內容，與語言選取相關的屬性和方法只支援單一輸出。
