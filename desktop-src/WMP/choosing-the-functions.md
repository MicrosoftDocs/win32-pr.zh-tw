# <a name="choosing-the-functions"></a>選擇函數

此外觀的目的是要提供播放音訊的基本功能。 將會使用下列函數：



| 函式                     | 描述                                                                                                                                                                                                                                                                                                                                                                                                                          |
|------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| PlayPause 或 PlayPauseStop\* | 啟動媒體專案的重要性很重要。 如果您要建立 Windows Media Player 10 行動裝置版或更新版本的面板，則應該使用 PlayPauseStop。 如果您使用的是舊版 Windows Media Player 行動裝置版，則必須使用 PlayPause。 這兩個函式可讓您新增暫停的功能，但只有在無法使用暫停功能時，才可使用 PlayPauseStop 來停止即時廣播。 |
| 停止                         | 您會想要新增停止以停止播放專案。 如果您要為 Windows Media Player 10 行動裝置版或更新版本建立面板，PlayPauseStop 函式已提供此功能。                                                                                                                                                                                                                                          |
| 下一個                         | 如果您有播放清單，您會想要能夠選擇下一個專案。 您需要這項程式才能進行數位媒體的基本導覽。                                                                                                                                                                                                                                                                                                       |
| 上一頁                         | 雖然您可以使用 [下一步] 迴圈播放播放清單，直到您開始進行為止，新增 [上一步] 可讓使用者在流覽播放清單時，可以更輕鬆地回溯和向前移動。                                                                                                                                                                                                                                   |
| VolumeUp 或 VolumeDown\*     | 您會想要提供使用者開啟或關閉磁片區的能力。                                                                                                                                                                                                                                                                                                                                                    |



 

\*適用于 Windows Media Player 10 行動裝置版或更新版本。
