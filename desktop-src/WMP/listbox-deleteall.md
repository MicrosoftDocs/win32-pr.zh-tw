# <a name="listboxdeleteall"></a>LISTBOX. deleteAll

**DeleteAll** 方法會刪除清單方塊控制項中的所有專案。

``` syntax
        elementID.deleteAll()
```

## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|---------------------------------------------------------|
| 版本<br/> | 適用于 Windows XP 或更新版本的 Windows Media Player<br/> |
