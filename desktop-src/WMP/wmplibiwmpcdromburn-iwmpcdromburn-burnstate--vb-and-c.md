# <a name="iwmpcdromburnburnstate-property"></a>IWMPCdromBurn：： burnState 屬性

**BurnState** 屬性會取得指出目前燒錄狀態的列舉值。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```CSharp
public WMPBurnState burnState {get;}
```


```VB

Public ReadOnly Property burnState As WMPBurnState
```





## <a name="property-value"></a>屬性值

**WMPLib** ，這是 **WMPBurnState** 列舉中表示目前狀態的值。
