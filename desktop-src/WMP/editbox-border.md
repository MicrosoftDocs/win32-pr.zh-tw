# <a name="editboxborder"></a>編輯方塊。框線

**框線** 屬性會指定或抓取值，指出編輯方塊控制項是否有框線。 只能在設計階段設定。

``` syntax
        elementID.border
```

## <a name="possible-values"></a>可能的值

這個屬性是在設計階段指定的 **布林值** ，之後是唯讀的。 預設值為 true。

## <a name="remarks"></a>備註

框線會以雙圖元寬的凹陷框線顯示在編輯方塊控制項的系統色彩周圍。
