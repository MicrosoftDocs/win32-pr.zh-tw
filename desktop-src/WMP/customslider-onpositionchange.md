# <a name="customslideronpositionchange"></a>CUSTOMSLIDER.onPositionChange

**OnPositionChange** 事件處理常式會處理當滑杆的位置由於使用者按一下或拖曳而變更時所發生的事件。

``` syntax
onPositionChange
```

## <a name="remarks"></a>備註

如果自訂滑杆的位置因為腳本中的 **值** 屬性修改而變更，則不會引發此事件。 為了滿足這種可能性，請改為執行 **值 \_ onchange** 事件處理常式。
