# <a name="downloadmanagercreatedownloadcollection-method"></a>DownloadManager. createDownloadCollection 方法

> [!Note]  
> 本章節描述專為線上商店使用而設計的功能。 不支援在線上商店的內容之外使用這項功能。

 

**CreateDownloadCollection** 方法會建立下載集合。

## <a name="syntax"></a>語法


```JScript
retVal = DownloadManager.createDownloadCollection()
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法會傳回 **DownloadCollection** 物件。
