# <a name="navigation-for-type-2-online-stores"></a>類型2線上商店的流覽

Windows Media Player 10 或更新版本所裝載的網頁（例如線上商店服務工作窗格中的網頁），可以使用[NavigateTaskPaneURL](external-navigatetaskpaneurl.md)方法來控制 Windows Media Player 內的導覽。 導覽可能會以下列方式發生：

-   在服務工作窗格之間。 請參閱 [在服務工作窗格之間流覽](navigating-between-service-task-panes.md)
-   從資訊中心 View、HTMLView 網頁和 [專輯資訊] 頁面等功能到 [服務工作窗格]。 請參閱[從 Windows Media Player 功能流覽](navigating-from-windows-media-player-features.md)。
-   在線上商店之間。 請參閱 [流覽至其他線上商店](navigating-to-other-online-stores.md)。
