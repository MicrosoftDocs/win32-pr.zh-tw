# <a name="ambient-attributes"></a>環境屬性

所有適當的面板元素都支援環境屬性和方法，除非有注明。

下列屬性為環境。



| 屬性                                                        | 描述                                                                                                                               |
|------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| [accName](ambientattributes-accname.md)                         | 指定或抓取任何元素的名稱。                                                                                            |
| [accDescription](ambientattributes-accdescription.md)           | 指定或抓取任何元素的描述。                                                                                     |
| [accKeyboardShortcut](ambientattributes-acckeyboardshortcut.md) | 指定或抓取任何元素的鍵盤快速鍵描述。                                                                   |
| [AlphaBlend](ambientattributes-alphablend.md)                   | 指定或抓取 Alpha 混合任何 **視圖**、子 **視圖** 或 UI widget 的值。                                                |
| [底部](ambientattributes-bottom.md)                           | 指定或抓取控制項的底部座標。                                                                              |
| [clippingColor](ambientattributes-clippingcolor.md)             | 指定或抓取從 **clippingImage** 點陣圖裁剪的色彩。                                                           |
| [clippingImage](ambientattributes-clippingimage.md)             | 指定或抓取要裁剪控制項的區域。                                                                                 |
| [elementType](ambientattributes-elementtype.md)                 | 抓取 (的元素類型，例如 **按鈕**) 。                                                                             |
| [啟用](ambientattributes-enabled.md)                         | 指定或抓取值，指出控制項是否已啟用或停用。                                                     |
| [height](ambientattributes-height.md) (高度)                           | 指定或抓取控制項的高度。                                                                                         |
| [>horizontalalignment](ambientattributes-horizontalalignment.md) | 指定或抓取值，指出調整 **視圖** 或父子 **視圖** 大小時，控制項的水準位置。 |
| [id](ambientattributes-id.md)                                   | 指定或抓取控制項的識別碼。 只能在設計階段設定。                                                       |
| [離開](ambientattributes-left.md)                               | 指定或抓取控制項的左座標。                                                                                |
| [nineGridMargins](ambientattributes-ninegridmargins.md)         | 指定面板元素之非統一縮放的邊界寬度。                                                                      |
| [通過](ambientattributes-passthrough.md)                 | 指定或抓取值，指出控制項是否會將所有滑鼠事件傳遞至其下的控制項。                 |
| [resizeImages](ambientattributes-resizeimages.md)               | 指定或抓取值，指出控制項大小變更時，控制項中包含的影像是否會自動調整大小。     |
| [對](ambientattributes-right.md)                             | 指定或抓取控制項的右側座標。                                                                               |
| [tabStop](ambientattributes-tabstop.md)                         | 指定或抓取值，指出控制項是否會在定位順序中。                                               |
| [top](ambientattributes-top.md)                                 | 指定或抓取控制項的上座標。                                                                                 |
| [verticalAlignment](ambientattributes-verticalalignment.md)     | 指定或抓取值，指出調整 **視圖** 或父子 **視圖** 大小時，控制項的垂直位置。   |
| [可見](ambientattributes-visible.md)                         | 指定或抓取控制項的可見度。                                                                                     |
| [width](ambientattributes-width.md) (寬度)                             | 指定或抓取控制項的寬度。                                                                                          |
| [zIndex](ambientattributes-zindex.md)                           | 指定或抓取控制項的呈現順序。                                                                        |



 

下列是環境的方法。



| 方法                                             | 描述                                                                                                                                                          |
|----------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [AlphaBlendTo](ambientattributes-alphablendto.md) | 調整 **AlphaBlend** 屬性一段時間。                                                                                                           |
| [moveSizeTo](ambientattributes-movesizeto.md)     | 移動控制項，並在新的位置指定控制項的新大小。 控制項會在指定的時間週期內以動畫方式移動和調整大小。 |
| [moveTo](ambientattributes-moveto.md)             | 以線性速度將控制項移至新的位置。                                                                                                               |
| [slideTo](ambientattributes-slideto.md)           | 將控制項移至新的位置。 控制項在指定的時間週期內以非線性方式移動。                                                       |
