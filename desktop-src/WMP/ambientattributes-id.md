# <a name="ambientattributesid"></a>AmbientAttributes.id

**Id** 屬性會指定或抓取控制項的識別碼。 這個屬性會支援 Microsoft Visual Basic for Applications 所允許的任何名稱。 只能在設計階段設定。

``` syntax
        elementID.id
```

## <a name="possible-values"></a>可能的值

這個屬性是在設計階段指定的 **字串** ，之後是唯讀的。 預設值為

未命名的 \_ *elementtype* \_ *num*

其中 *elementtype* 是控制項類型的名稱，而 *num* 是一個數位，其對應至主題中所有未命名控制項的順序中的控制項位置。 每個新主題的這個數位會重設為零。
