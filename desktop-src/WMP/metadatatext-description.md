# <a name="metadatatextdescription"></a>MetadataText。描述

**Description** 屬性會捕獲中繼資料文字的描述。

## <a name="syntax"></a>Syntax

*玩家*。*currentMedia*。**getItemInfoByType** ( *name*、 *language*、 *index*) 。**描述**

## <a name="possible-values"></a>可能的值

這個屬性是唯讀 **字串**。

## <a name="remarks"></a>備註

若要取得這個屬性的值，需要有程式庫的讀取權限。 如需詳細資訊，請參閱連結 [庫存取](library-access.md)。

**Windows Media Player 10** 行動裝置版：不支援這個屬性。
