# <a name="listboxfirstvisibleitem"></a>LISTBOX. firstVisibleItem

**FirstVisibleItem** 屬性會指定或抓取清單方塊控制項中第一個可見行的索引。

``` syntax
        elementID.firstVisibleItem
```

## <a name="possible-values"></a>可能的值

這個屬性是 (**長**) 的讀取/寫入 **號碼**。

## <a name="remarks"></a>備註

當滾動清單方塊時，這個屬性可用來判斷第一個可見的行。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|---------------------------------------------------------|
| 版本<br/> | 適用于 Windows XP 或更新版本的 Windows Media Player<br/> |
