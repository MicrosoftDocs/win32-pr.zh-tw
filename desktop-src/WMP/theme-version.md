# <a name="themeversion"></a>主題。版本

**version** 屬性會指定或抓取用來撰寫面板的 Windows Media Player 版本號碼。

``` syntax
theme.version
```

## <a name="possible-values"></a>可能的值

這個屬性是 (**浮點數**) 的 **數位**。 在設計階段或執行時間指定此屬性的值沒有任何作用。 此值一律為1.0。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
