# <a name="playlistcollectiongetall-method"></a>PlaylistCollection. getAll 方法

**GetAll** 方法會抓取 **PlaylistArray** 物件，其中包含文件庫中的所有播放清單。

## <a name="syntax"></a>語法


```JScript
retVal = PlaylistCollection.getAll()
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法會傳回 **PlaylistArray** 物件。

## <a name="remarks"></a>備註

若要使用此方法，需要有程式庫的讀取權限。 如需詳細資訊，請參閱連結 [庫存取](library-access.md)。
