# <a name="building-a-dsp-plug-in"></a>建立 DSP 外掛程式

您可以使用 Visual Studio 和 Windows Media Player 外掛程式嚮導來建立 DSP 外掛程式。 如需設定 wizard 和建立外掛程式的詳細資訊，請參閱 [建立外掛程式](building-a-plug-in.md)。

當您執行嚮導時，將會提供數種不同類型之外掛程式的選項。選擇 [ **DSP 外掛程式]**。

下列主題說明您可以使用 wizard 建立之範例音訊和影片 DSP 外掛程式的功能。

-   [關於範例音訊 DSP 外掛程式](about-the-sample-audio-dsp-plug-in.md)
-   [關於範例影片 DSP 外掛程式](about-the-sample-video-dsp-plug-in.md)
