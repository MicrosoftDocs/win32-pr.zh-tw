# <a name="skin-creation-guide"></a>外觀建立指南

本指南是如何建立不同類型之外觀的一系列詳細說明。 如需有關面板的一般資訊，請參閱 [關於外觀](about-skins.md)。 如需有關面板中所使用之每個屬性、方法和事件的特定詳細資料，請參閱面板程式 [設計參考](skin-programming-reference.md)。 當您在面板的程式設計方面有更多的相關資訊時，您可能會想要閱讀此 SDK 中涵蓋[Windows Media Player 物件模型](windows-media-player-object-model.md)的部分。

在本指南中，會為 Adobe Photoshop 5.5 （熱門的藝術操作程式）提供建立藝術的相關指示。 如果您有類似的尖端程式，例如 Jasc 小畫家店 Pro 或 Sonic Foundry Viscosity，則特定的指示可能會有所不同，但概念會相同。 因為有兩個原因，所以選擇了 Photoshop：它是適用于商業演出者的熱門程式，且適用于圖層。 您將在教學課程中看到，圖層在建立面板時非常有用。

本指南將涵蓋下列工作。



| 工作                                                     | 描述                                                                                     |
|----------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [建立您的第一個外觀](building-your-first-skin.md) | 啟動和停止 Windows Media Player 的簡單面板逐步解說。                     |
| [新增播放清單](adding-a-playlist.md)               | 如何使用簡單的播放清單。                                                                   |
| [選擇檔案](choosing-files.md)                     | 如何使用 [開啟檔案] 對話方塊來挑選檔案。                                                 |
| [新增影片](adding-video.md)                         | 如何將影片視窗放入您的外觀。                                                       |
| [新增視覺效果](adding-visualizations.md)       | 如何新增視覺效果。                                                                      |
| [新增滑杆](adding-a-slider.md)                   | 如何使用滑杆來追蹤媒體內容的進度 .。。然後讓使用者變更！ |
| [建立自訂滑杆](creating-custom-sliders.md)   | 如何使用自訂滑杆控制磁片區。                                                 |
