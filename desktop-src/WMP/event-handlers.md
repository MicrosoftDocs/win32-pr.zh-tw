# <a name="event-handlers"></a>事件處理常式

Microsoft JScript 用來處理面板定義檔中的事件。 如需事件處理常式的詳細資訊，請參閱 [處理事件](handling-events.md) 。

事件處理常式中可以有一行以上的程式碼，但必須小心不要超過 JScript 允許的行長度。 以分號分隔程式程式碼。


```C++
onclick = "JScript: player.URL = 'https://proseware.com/cool.wma' ; myText.value = 'Playing'; "

```
