# <a name="playlistdropdownbackgroundimage"></a>播放清單。 dropDownBackgroundImage

**DropDownBackgroundImage** 屬性會指定或抓取顯示在下拉式清單背景中之影像的名稱。

``` syntax
        elementID.dropDownBackgroundImage
```

## <a name="possible-values"></a>可能的值

這個屬性是包含影像檔案名稱的讀取/寫入 **字串** 。 它沒有預設值。

## <a name="remarks"></a>備註

這個屬性支援 PNG、JPG、BMP 和 GIF 檔案。 如果影像是8位的 BMP 檔案，則可以使用 **hueShift** 和 **飽和度** 屬性動態變更其色調和飽和度值。
