# <a name="iwmperroritemcustomurl-property"></a>IWMPErrorItem：： customUrl 屬性

**CustomUrl** 屬性會取得網站的 URL，以顯示有關編解碼器下載失敗的特定資訊。

## <a name="syntax"></a>Syntax


```CSharp
public System.String customUrl {get; set;}
```


```VB

Public ReadOnly Property customUrl As System.String
```





## <a name="property-value"></a>屬性值

**System.string** ，此為自訂 URL。
