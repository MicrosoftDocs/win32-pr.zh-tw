# <a name="type-2-online-stores"></a>輸入2個線上商店

> [!Note]  
> 本章節描述專為線上商店使用而設計的功能。 不支援在線上商店的內容之外使用這項功能。

 

Windows Media Player 11 支援類型1和類型2線上商店。 類型1存放區比類型2存放區更緊密整合到玩家的使用者介面。 Windows Media Player 10 支援類型2存放區，但不支援類型1存放區。 本檔僅適用于類型2存放區，分為下列各節。



| 區段                                                                                      | 描述                                                                                                                            |
|----------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------|
| [關於 Type 2 線上商店](about-type-2-online-stores.md)                                 | 提供類型2線上商店的總覽，以及外掛程式和範例的一般資訊。                                |
| [Type 2 線上商店的程式設計指南](programming-guide-for-type-2-online-stores.md) | 說明如何建立 type 2 線上商店的外掛程式，並提供相關網頁和下載管理員的相關資訊。 |
| [類型 2 線上商店的參考](reference-for-type-2-online-stores.md)                 | 提供類型2線上商店所使用之介面、物件、列舉和登錄專案的詳細參考。              |
