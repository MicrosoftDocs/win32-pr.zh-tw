# <a name="about-the-closedcaption-object"></a>關於 ClosedCaption 物件

**ClosedCaption** 物件會管理 Windows Media Player 的標題介面。 **ClosedCaption** 物件是透過 **Player** 物件的 **ClosedCaption** 屬性取得。 例如，若要取得 **SAMIFileName** 屬性值，請輸入下列內容：


```C++
myfile = player.closedcaption.SAMIFileName;

```
