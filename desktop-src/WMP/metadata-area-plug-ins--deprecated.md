# <a name="metadata-area-plug-ins-deprecated"></a>中繼資料區域外掛程式 (已淘汰) 

本頁記載了未來版本的 Windows Media Player 和 Windows Media Player SDK 可能無法使用的功能。

[中繼資料區域 UI 外掛程式] 會顯示在 [ **目前播放** ] 窗格的 [中繼資料] 區域中，位於主要視覺效果區域和 [工具] 窗格的右邊，並在 [播放清單] 視圖下 (（如果已啟用) ）。 一次只能啟用一個中繼資料區域外掛程式。

中繼資料區域外掛程式適合用來顯示有關目前播放媒體專案、簡單控制項或超連結的詳細資訊。
