# <a name="playlistdisableditemcolor"></a>播放清單。 disabledItemColor

**DisabledItemColor** 屬性會指定或抓取離線時已停用的 CD 播放軌或線上內容的色彩。

``` syntax
        elementID.disabledItemColor
```

## <a name="possible-values"></a>可能的值

這個屬性是包含任何 Microsoft Internet Explorer 色彩值的讀取/寫入 **字串** 。 其預設值為 "graytext"。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
