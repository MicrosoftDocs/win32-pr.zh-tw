# <a name="background-plug-ins"></a>背景外掛程式

背景 UI 外掛程式實際上並沒有使用者介面，除非它們會執行屬性頁。 當一或多個背景外掛程式正在執行時，在執行曲目時間旁的 Windows Media Player 顯示中會出現特殊圖示。 您可以按一下這個圖示，在 [選項] 對話方塊中顯示背景外掛程式選項。 使用者可以從該處，選擇開啟任何支援該外掛程式之外掛程式的 [屬性] 頁面。

背景外掛程式適用于提供不需要使用者輸入的自動服務。
