# <a name="iwmpdvdresume-method"></a>IWMPDVD：： resume 方法

**Resume** 方法會從功能表模式回到播放模式，並在叫用功能表時的相同位置繼續進行。

## <a name="syntax"></a>語法


```CSharp
public void resume();
```


```VB

Public Sub resume()
Implements IWMPDVD.resume
```





## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
