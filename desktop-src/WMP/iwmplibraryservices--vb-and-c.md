# <a name="iwmplibraryservices-vb-and-c-interface"></a>IWMPLibraryServices (VB 和 c # ) 介面

提供列舉程式庫的方法。

## <a name="members"></a>成員

**IWMPLibraryServices (VB 和 c # )** 介面都有下列類型的成員：

-   [方法](#methods)

### <a name="methods"></a>方法

**IWMPLibraryServices (VB 和 c # )** 介面都有這些方法。



| 方法                                                                                               | 描述                                                                                                        |
|:-----------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------|
| [**getCountByType**](wmplibiwmplibraryservices-iwmplibraryservices-getcountbytype--vb-and-c.md)     | 傳回指定類型的可用程式庫計數。<br/>                                           |
| [**getLibraryByType**](wmplibiwmplibraryservices-iwmplibraryservices-getlibrarybytype--vb-and-c.md) | 傳回 **IWMPLibrary** 介面，代表具有指定之類型和索引的程式庫。<br/> |



 

藉由轉換從 **AxWindowsMediaPlayer. GetOcx** 方法傳回的物件來取得 **IWMPLibraryServices** 介面。
