# <a name="commonly-used-windows-media-file-attributes"></a>常用的 Windows Media 檔案屬性

Windows Media 檔案可包含任意數目的屬性，包括使用者定義的屬性。

如果檔案已新增至程式庫，則也可以使用與媒體類型相關聯的其他屬性。 查看 [音訊專案](audio-item-attributes.md) 或其他屬性的 [影片專案](video-item-attributes.md) 。

如需 Windows Media 檔案屬性的詳細資訊，請參閱 [Windows Media 中繼資料使用指導方針](/previous-versions/ms867702(v=msdn.10))。

下列清單包含常用屬性。

-   [Author 屬性](author-attribute.md)
-   [Title 屬性](title-attribute.md)
-   [WM/AlbumArtist 屬性](wm-albumartist-attribute.md)
-   [WM/ContentDistributor 屬性](wm-contentdistributor-attribute.md)
-   [WM/內容類型屬性](wm-genre-attribute.md)
-   [WM/MCDI 屬性](wm-mcdi-attribute.md)
-   [WM/MediaClassPrimaryID 屬性](wm-mediaclassprimaryid-attribute.md)
-   [WM/MediaClassSecondaryID 屬性](wm-mediaclasssecondaryid-attribute.md)
-   [WM/提供者屬性](wm-provider-attribute.md)
-   [WM/TrackNumber 屬性](wm-tracknumber-attribute.md)
