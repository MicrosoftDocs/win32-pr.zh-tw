# <a name="videostretchtofit"></a>StretchToFit

**StretchToFit** 屬性指定一個值，指出影片是否將延展到針對影片控制項定義的寬度和高度。

``` syntax
        elementID.stretchToFit
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **布林值**。



| 值 | 描述                                             |
|-------|---------------------------------------------------------|
| true  | 影片將會延展以符合控制項。              |
| false | 預設值。 影片將不會伸展以容納控制項。 |



 

## <a name="remarks"></a>備註

如果未指定寬度或高度，則會忽略這個屬性。

| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
