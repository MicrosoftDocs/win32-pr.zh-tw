# <a name="bitrate-attribute"></a>位元速率屬性

**位元速率** 屬性是專案的位元速率，以每秒位數為單位。

## <a name="applies-to"></a>套用至

-   [音訊專案](audio-item-attributes.md)
-   [常用的 Windows Media 檔案](commonly-used-windows-media-file-attributes.md)
-   [其他專案](other-item-attributes.md)
-   [單選項目](radio-item-attributes.md)
-   [影片專案](video-item-attributes.md)

## <a name="remarks"></a>備註

這個屬性會儲存在文件庫和數位媒體檔案中。

這個屬性的 Windows Media Format SDK 常數是 g \_ wszWMBitrate。

若要判斷是否可以變更這個屬性的值，請使用 [isReadOnlyItem](media-isreadonlyitem.md) 方法。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|---------------------------------------------------|
| 版本<br/> | Windows Media Player 9 系列或更新版本<br/> |
