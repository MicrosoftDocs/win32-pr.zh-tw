# <a name="playlistcolumnorder"></a>播放清單. columnOrder

**ColumnOrder** 屬性會指定或抓取播放清單資料行的順序。

``` syntax
        elementID.columnOrder
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **字串** ，可指定以分號分隔的播放清單資料行索引清單。 預設值為 "0; 1; 2; 3"。 開頭和尾端的分號和空格不應該存在。

| 需求 | 值 |
|--------------------|------------------------------------------------------|
