# <a name="eventscreenwidth"></a>screenWidth

**ScreenWidth** 屬性會抓取可用螢幕大小的寬度（以圖元為單位）。

``` syntax
event.screenWidth
```

## <a name="possible-values"></a>可能的值

這個屬性是唯讀的 **數位** (**long**) 。

## <a name="remarks"></a>備註

這適用于判斷監視器中有多少空間。 如果有兩個監視器，它會計算這兩個監視器的空間。
