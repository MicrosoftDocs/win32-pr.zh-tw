# <a name="dvd-object"></a>DVD 物件

**Dvd** 物件提供使用 dvd 的屬性和方法。

**DVD** 物件支援下列屬性。



| 屬性                           | 描述                                                                                        |
|------------------------------------|----------------------------------------------------------------------------------------------------|
| [域](dvd-domain.md)           | 抓取 DVD 的目前網域。                                                           |
| [isAvailable](dvd-isavailable.md) | 抓取指定的資訊類型是否可用，或是否可以執行指定的動作。 |



 

**DVD** 物件支援下列方法。



| 方法                         | 描述                                                                                          |
|--------------------------------|------------------------------------------------------------------------------------------------------|
| [返回](dvd-back.md)           | 將顯示從子功能表變更為其父功能表。                                               |
| [恢復](dvd-resume.md)       | 從功能表模式變更播放模式，並在叫用功能表時的相同位置繼續進行。 |
| [titleMenu](dvd-titlemenu.md) | 停止播放並顯示標題功能表。                                                          |
| [topMenu](dvd-topmenu.md)     | 停止播放並顯示根功能表。                                                           |



 

**DVD** 物件可透過下列屬性來存取。



| Object                      | 屬性              |
|-----------------------------|-----------------------|
| [球員](player-object.md) | [Dvd](player-dvd.md) |
