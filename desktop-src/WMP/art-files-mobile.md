# <a name="art-files-mobile"></a> (Mobile) 的美工檔案

您將需要為您的面板建立美工檔案。 您可以建立數種類型的美工檔案，而且它們必須是特定的藝術檔案格式。

下表描述您可以為面板建立的七種類型的美工檔案。



| 美工檔案類型                            | 描述                                                                                                |
|------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [背景檔](background-file.md)   | 必要。 背景是所有按鈕影像的疊加影像                        |
| [停用的檔案](disabled-file.md)       | 必要。 用來表示推送按鈕不會有任何作用的影像。                                |
| [推送的檔案](pushed-file.md)           | 必要。 定義使用者推送按鈕時所顯示的影像。                              |
| [區域檔案](region-files.md)         | 定義點陣圖，以使用色彩區域表示點擊類型按鈕的點-回應位置      |
| [Super 檔案](super-files.md)           | 為 trackbars 定義停用的映射。                                                                 |
| [SeekThumb 檔案](seekthumb-file.md)     | 定義要搭配使用中的影像，以表示媒體專案中的目前位置。 |
| [VolumeThumb 檔案](volumethumb-file.md) | 定義要搭配使用中的影像，以表示音量控制位置。            |



 

下列各節提供有關美工檔案的詳細資訊。

-   [美工檔案格式](art-file-formats-mobile.md)
-   [色彩調色板](color-palettes.md)
-   [映射大小](image-sizes.md)
-   [按鈕位置](button-placement.md)

如需有關美工檔案類型的詳細資訊，以及如何在面板定義檔中識別這些檔案的詳細資訊，請參閱面板參考的 [點陣圖](bitmaps.md) 區段。
