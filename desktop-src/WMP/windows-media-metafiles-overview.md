# <a name="windows-media-metafiles-overview"></a>Windows Media 中繼檔總覽

成功使用 Windows Media 中繼檔的最重要部分是使用中繼檔元素的正確語法。 Windows Media 中繼檔中的語法錯誤可能會導致任何來自被忽略之單一屬性的任何事物，而不會將中繼檔辨識為有效且無法運作。

最重要的是元素出現在 Windows Media 中繼檔中的順序。 某些專案的屬性會暫時覆寫中繼檔不同區段中類似專案的屬性。 有一個已定義 [的優先順序](order-of-precedence.md)。

Windows Media 中繼檔播放清單是 Windows Media 中繼檔，可提供 Windows Media Player 用來接收單點傳送串流、多播串流以及其他內部網路或網際網路支援之媒體的資訊。 中繼檔播放清單基本上是媒體內容的快捷方式。 中繼檔播放清單可以當做電子郵件傳送，作為網頁上的連結參考、使用 Active Server Pages (ASP) 動態建立，或以獨立檔案的形式存在於本機磁片磁碟機上。 中繼檔播放清單可以參考另一個中繼檔播放清單、ASP 頁面或 (副檔名為 .nsc 的 Windows Media 站檔案) 。 .Nsc 檔案是用來定義要 Windows Media Player 的 Windows Media 站。 每個案例的基本處理常式都相同。
