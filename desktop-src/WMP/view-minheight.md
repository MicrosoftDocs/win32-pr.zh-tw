# <a name="viewminheight"></a>MinHeight

當調整大小時， **minHeight** 屬性會指定或抓取 **視圖** 的最小高度（以圖元為單位）。

``` syntax
        elementID.minHeight
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 (**長**) 值為零或更大的 **數位**。 預設值為零，表示對 **視圖** 的最小高度沒有任何限制。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
