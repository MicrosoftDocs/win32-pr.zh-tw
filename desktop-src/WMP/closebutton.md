# <a name="closebutton"></a>CLOSEBUTTON

這是預先定義的 **按鈕** ，具有下列預設值。


```C++
onclick="jscript:view.close();"
upToolTip="Close"

```



## <a name="remarks"></a>備註

這會建立關閉播放程式的 **按鈕** 控制項。 工具提示已當地語系化。 此 **按鈕** 的所有屬性都可以藉由明確指定來覆寫。
