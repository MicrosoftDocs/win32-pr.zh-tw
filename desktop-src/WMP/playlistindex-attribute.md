# <a name="playlistindex-attribute"></a>PlaylistIndex 屬性

**PlaylistIndex** 屬性是其父播放清單中媒體專案的索引。

## <a name="applies-to"></a>套用至

-   [音訊專案](audio-item-attributes.md)
-   [CD 曲目](cd-track-attributes.md)
-   [其他專案](other-item-attributes.md)
-   [影片專案](video-item-attributes.md)

## <a name="remarks"></a>備註

這個屬性是暫存屬性。 它不會儲存在文件庫或數位媒體檔案中。

媒體專案可以是播放清單元素。 這個屬性是以零為基底的索引，表示播放清單中媒體專案的位置。 如果媒體專案不是播放清單的一部分，則此屬性的值為空字串 ( "" ) 。
