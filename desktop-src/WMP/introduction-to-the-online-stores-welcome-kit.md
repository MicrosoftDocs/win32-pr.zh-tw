# <a name="introduction-to-the-online-stores-welcome-kit"></a>線上商店歡迎套件簡介

線上商店歡迎套件說明將線上數位媒體存放區帶入 Windows Media Player 的流程。 Windows Media Player 支援兩種類型的線上商店：類型1和類型2。 如需類型1和類型2存放區之間差異的相關資訊，請參閱[Windows Media Player 線上商店](windows-media-player-online-stores.md)。

下列各節所述的內建程式適用于類型1和類型2存放區，但有一些差異會注明。

## <a name="eligibility"></a>資格

您的公司必須符合線上商店合約中提供的要求和維護需求，才能擁有與 Windows Media Player 整合的線上音樂商店。 合約中的需求類型包含但不限於下列各項。

-   您的公司必須有權透過 Windows Media Player 發佈線上內容。
-   您的公司必須同意對 Windows Media Player 的使用者提供特定數目的追蹤。
-   您的公司必須提供特定層級的不中斷服務。

合約中的某些特定需求會在 [線上音樂商店的需求](requirements-for-online-music-stores.md)中反白顯示。

## <a name="contacting-microsoft"></a>聯絡 Microsoft

如果您有關于線上商店和上線流程的問題，請洽詢 Windows Media Player Services 虛擬團隊 mpsvctm@microsoft.com 。
