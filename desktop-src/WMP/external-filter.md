# <a name="externalfilter"></a>外部. 篩選

> [!Note]  
> 本主題說明針對線上商店使用而設計的功能。 不支援在線上商店的內容之外使用這項功能。

 

**Filter** 屬性會抓取 Windows Media Player 目前正在使用的搜尋篩選準則。

``` syntax
window.external.filter
      
```

## <a name="possible-values"></a>可能的值

這個屬性是唯讀 **字串**。

## <a name="remarks"></a>備註

搜尋篩選準則是使用者在播放程式的 word 滾輪控制項中輸入的文字。
