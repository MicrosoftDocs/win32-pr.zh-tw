# <a name="equalizersettingsreset"></a>EQUALIZERSETTINGS 重設

**Reset** 方法會將所有波段的增益層級重設為零分貝。

``` syntax
        elementID.reset()
```

## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
