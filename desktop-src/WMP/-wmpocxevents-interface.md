# <a name="_wmpocxevents-interface"></a>\_WMPOCXEvents 介面

**\_ WMPOCXEvents** 介面提供與 **IWMPEvents**、 **IWMPEvents2**、 **IWMPEvents3** 和 **IWMPEvents4** 介面相同的事件，但繼承自 **IDispatch** 而不是 **IUnknown**。 這可讓您透過 **IDispatch** 事件接收器接收 Windows Media Player 事件通知。
