# <a name="ambientattributesright"></a>AmbientAttributes。 right

**右邊** 的屬性會指定或抓取控制項的右座標。

``` syntax
        elementID.right
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **號碼** (**long**) 代表從控制項到父 **視圖****或子視圖右** 邊緣的距離（以圖元為單位）。 負數值的行為，或未指定 [AmbientAttributes](ambientattributes-width.md) 時，未定義。

|--------------------|------------------------------------|
