# <a name="automenushow"></a>AUTOMENU。顯示

**Show** 方法會顯示 **快速存取面板**。

``` syntax
        elementID.show("Play")
        
```

## <a name="parameters"></a>參數

<dl> <dt>

<span id="_Play_"></span><span id="_play_"></span><span id="_PLAY_"></span>*發出*
</dt> <dd>

一律指定此值。

</dd> </dl>

## <a name="return-value"></a>傳回值

這個方法不會傳回值。

## <a name="remarks"></a>備註

**快速存取面板** 會出現在 **AUTOMENU** 元素的 **left** 和 **top** 屬性所指定的位置。

|--------------------|---------------------------------------------------|
| 版本<br/> | Windows Media Player 9 系列或更新版本<br/> |
