# <a name="effectscurrentpresettitle"></a>效果。 currentPresetTitle

**CurrentPresetTitle** 屬性會抓取目前視覺效果目前預設值的標題。

``` syntax
        elementID.currentPresetTitle
```

## <a name="possible-values"></a>可能的值

這個屬性是唯讀 **字串**。

| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
