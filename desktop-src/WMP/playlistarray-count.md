# <a name="playlistarraycount"></a>PlaylistArray 計數

**Count** 屬性會抓取播放清單陣列中的播放清單數目。

## <a name="syntax"></a>Syntax

playlistCollection. getAll () . count

## <a name="possible-values"></a>可能的值

這個屬性是唯讀的 **數位** (**long**) 。

## <a name="remarks"></a>備註

若要取得這個屬性的值，需要有程式庫的讀取權限。 如需詳細資訊，請參閱連結 [庫存取](library-access.md)。
