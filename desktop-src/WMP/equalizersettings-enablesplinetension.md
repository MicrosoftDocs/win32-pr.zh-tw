# <a name="equalizersettingsenablesplinetension"></a>EQUALIZERSETTINGS.enableSplineTension

**EnableSplineTension** 屬性會指定或抓取值，指出是否已啟用曲線張力。

``` syntax
        elementID.enableSplineTension
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **布林** 值，預設值為 true。

## <a name="remarks"></a>備註

啟用曲線張力可讓使用者更順暢地調整等化器頻寬，讓頻率層級之間沒有大型跳躍。
