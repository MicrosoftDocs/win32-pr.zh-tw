# <a name="ambientattributeselementtype"></a>AmbientAttributes 的 elementType

**ElementType** 屬性會抓取 (的元素類型，例如按鈕) 。

``` syntax
        elementID.elementType
```

## <a name="possible-values"></a>可能的值

這個屬性是唯讀 **字串** ，表示元素的名稱。

## <a name="remarks"></a>備註

這個屬性有助於判斷引發事件的專案類型，並為這個類別的專案撰寫泛型事件處理常式。

|--------------------|------------------------------------------------------|
