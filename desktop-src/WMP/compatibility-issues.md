# <a name="compatibility-issues"></a>相容性問題

## <a name="netscape-support"></a>Netscape 支援

HTMLView 功能所開啟的網頁會使用 Microsoft Internet Explorer 的內嵌實例來顯示。 HTMLView 內容不需要 Netscape 相容性。

## <a name="no-htmlview-support-in-earlier-versions-of-windows-media-player"></a>舊版 Windows Media Player 不支援 HTMLView

目前，只有 Windows Media Player 9 系列或更新版本才支援 HTMLView 功能。 舊版播放程式和為其他平臺建立的版本，會在剖析 .asx 檔案時忽略 HTMLView 參數。 當發生這種情況時，數位媒體內容會如預期般播放，但以網路為基礎的內容不會顯示在 [ **目前播放** ] 功能中。
