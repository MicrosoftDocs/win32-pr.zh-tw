# <a name="button-functions"></a>按鈕函式

您會想要為使用者提供按鈕，以控制媒體專案的選取和播放。 您可以使用適當的按鈕來播放、暫停或停止媒體專案、移至播放清單中的不同專案、使音效靜音，或提供所播放媒體專案的相關資訊。

您不需要在面板中提供所有按鈕功能，但最佳的使用者體驗可能至少包含 PlayPause、停止或 PlayPauseStop、上一個按鈕和下一個按鈕功能。

可用的按鈕函數如下：

-   播放
-   暫停
-   停止
-   上一頁
-   下一個
-   Shuffle
-   Info
-   RepeatPlaylist
-   Mute
-   PlayPause
-   FullScreen
-   WindowsMedia\*
-   PlayPauseStop\*
-   VolumeUp\*
-   VolumeDown\*

\*適用于 Windows Media Player 10 行動裝置版或更新版本。

如需按鈕函式的詳細資訊，請參閱面板參考的 [按鈕](buttons.md) 區段。
