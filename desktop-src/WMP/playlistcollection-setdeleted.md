# <a name="playlistcollectionsetdeleted-method"></a>PlaylistCollection. setDeleted 方法

**SetDeleted** 方法會將播放清單移至 [刪除的專案] 資料夾。

## <a name="syntax"></a>語法


```JScript
PlaylistCollection.setDeleted(
  playlist,
  true
)
```



## <a name="parameters"></a>參數

<dl> <dt>

*播放清單* \[在\]
</dt> <dd>

要移動的 **播放清單** 物件。

</dd> <dt>

*true* \[在\]
</dt> <dd>

一律指定此值。

</dd> </dl>

## <a name="return-value"></a>傳回值

這個方法不會傳回值。

## <a name="remarks"></a>備註

**Windows Media Player 10** 行動裝置版：不支援此方法。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 版本<br/> | 適用于 Windows XP 的 Windows Media Player 7.0 版、Windows Media Player 7.1 版或 Windows Media Player。 Windows Media Player 9 系列或更新版本不支援這個方法。<br/> |
| DLL<br/>     | <dl> <dt>Wmp.dll</dt> </dl>                                                                                                              |
