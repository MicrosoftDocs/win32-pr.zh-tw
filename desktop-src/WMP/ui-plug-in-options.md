# <a name="ui-plug-in-options"></a>UI 外掛程式選項

五個 UI 外掛程式類型的任一個都可以有屬性頁，讓使用者可以在其中修改外掛程式設定。 這個屬性頁可以設定為第一次載入外掛程式時自動顯示。 也可以從 [選項] 對話方塊的 [外掛程式] 索引標籤存取。

當 Windows Media Player 啟動時，可以將 UI 外掛程式設定為自動載入。 如果沒有自動啟動，則使用者可以從 [**工具**] 功能表上的 [**外掛程式]** 清單中選取它來載入它。

顯示區域外掛程式可以有多個預設值，也就是外掛程式可讓使用者使用的不同視圖。 使用者可以變更預設值，方法是從 [ **View** ] 功能表上的 [**視覺效果和流覽** 器] 清單中選取不同的專案，或使用狀態訊息和傳輸控制項正上方的 [**目前播放**] 窗格底部提供的箭號按鈕。

背景和個別的視窗 UI 外掛程式可進行程式設計，以接受使用者傳送給它的媒體專案或播放清單。 如果外掛程式支援這項功能，其名稱會出現在播放清單控制項或程式庫的快捷方式功能表中的 [ **傳送到** ] 清單上。

UI 外掛程式可進行程式設計，以在具有鍵盤焦點時攔截鍵盤快速鍵。 載入多個攔截相同鍵盤快速鍵的 UI 外掛程式時，需要鍵盤焦點以防止發生衝突。 雖然這樣會造成衝突，但也會對使用者造成混淆。 基於這個理由，不建議使用具有 UI 外掛程式的鍵盤快速鍵。 不過，使用這些方法時，不應攔截 Windows Media Player 所使用的任何鍵盤快速鍵。 如需播放程式所使用之鍵盤快速鍵的完整清單，請參閱 Windows Media Player 說明。
