# <a name="sliderthumbdisabledimage"></a>滑杆. thumbDisabledImage

**ThumbDisabledImage** 屬性會指定或抓取控制項停用時使用之滑杆控制項的捲動方塊影像。

``` syntax
        elementID.thumbDisabledImage
```

## <a name="possible-values"></a>可能的值

這個屬性是包含影像檔案名稱的讀取/寫入 **字串** 。

## <a name="remarks"></a>備註

**ThumbDisabledImage** 是選擇性的。 如果未指定，則會改用 **thumbImage** 。

支援的格式為 BMP、JPG、PNG 和 GIF (不包括動畫 Gif) 。
