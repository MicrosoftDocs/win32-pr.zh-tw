# <a name="effectsnext"></a>效果。下一步

**下** 一個方法會顯示下一個視覺效果預設值，視需要移至下一個視覺效果。

``` syntax
        elementID.next()
```

## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。

## <a name="remarks"></a>備註

如果目前的預設值是所有可用視覺效果系列中的最後一個預設值，則第一個視覺效果的第一個預設會成為最新的。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
