# <a name="playerenabled"></a>Player。已啟用

**enabled** 屬性會指定或抓取值，指出是否已啟用 Windows Media Player 控制項。

## <a name="syntax"></a>Syntax

*玩家* 。**已啟用**

## <a name="possible-values"></a>可能的值

這個屬性是讀取/寫入 **布林值**。



| 值 | 描述                                           |
|-------|-------------------------------------------------------|
| true  | 預設值。 已啟用 Windows Media Player 控制項。 |
| false | Windows Media Player 控制項已停用。         |



 

## <a name="remarks"></a>備註

如果 **enabled** 等於 false，則在全螢幕播放期間 Windows Media Player 隱藏使用者控制項。
