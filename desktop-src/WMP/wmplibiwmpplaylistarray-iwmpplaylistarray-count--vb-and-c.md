# <a name="iwmpplaylistarraycount-property"></a>IWMPPlaylistArray：： count 屬性

**Count** 屬性會取得播放清單陣列中的播放清單數目。

## <a name="syntax"></a>Syntax


```CSharp
public System.Int32 count {get; set;}
```


```VB

Public ReadOnly Property count As System.Int32
```





## <a name="property-value"></a>屬性值

System.string **，這是** 陣列中的播放清單數目。

## <a name="remarks"></a>備註

使用這個屬性之前，您必須擁有程式庫的讀取權限。 如需詳細資訊，請參閱連結 [庫存取](library-access.md)。
