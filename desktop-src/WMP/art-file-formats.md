# <a name="art-file-formats"></a>美工檔案格式

適用于面板的 Windows Media Player 可辨識下列藝術檔案格式：



| 格式                                  | 分機   | 描述                                                                                        |
|-----------------------------------------|-------------|----------------------------------------------------------------------------------------------------|
| 點陣圖                                  | .bmp        | 建議使用點陣圖影像，因為它們提供對確切影像和色彩的最大控制權。 |
| 圖形交換格式 (GIF)       | .gif        | 用於網頁的壓縮影像格式。 支援動畫 Gif。                            |
| JPEG 格式 (JPEG) | .jpeg、.jpg | 用於網頁的壓縮影像格式。                                                         |
| 可攜式網路圖形 (PNG)         | .png        | 用於網頁的壓縮影像格式。                                                         |



 

如果您使用其中一種壓縮檔案格式來定義網頁瀏覽器的透明色彩，請勿在影像檔中將色彩定義為透明。 使用可見色彩表示影像中的透明區域，然後在外觀定義檔中將該色彩定義為透明。 例如，如果您建立的 GIF 檔案中有些區域是透明的，則在您的最終影像中將不會是透明的，而且您將無法使用在您的 GIF 檔案中設定為透明的色彩來呈現您的外觀。
