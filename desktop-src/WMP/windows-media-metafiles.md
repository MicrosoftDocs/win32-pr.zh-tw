# <a name="windows-media-metafiles"></a>Windows Media 中繼檔案  (可能為英文網頁)

此參考檔記載 Windows Media 中繼檔，其使用地板蠟、. 300k.wvx、. wmx 和 .asx 副檔名。 其中包含總覽和程式設計指南章節，以及中繼檔元素標記、其屬性和值，以及與每個專案相關的特殊條件的完整參考區段。

中繼檔是包含其他檔案相關資訊的檔案。 中繼檔可以用來列出要依序播放的媒體內容檔案群組。 Windows Media 中繼檔播放清單，在本參考檔中只是指播放清單，是 Microsoft Windows Media 技術最強大的功能之一。 播放清單可讓您控制和自訂媒體內容。 例如，有了播放清單，您可以將內容排程為連續播放，或是將廣告或特殊興趣剪輯插入簡報中。 播放清單的另一項好處是，不是播放資料流程、停止、啟動下一個資料流程，然後等候它完成緩衝處理，而 Windows Media Services 和 Windows Media Player 會一起運作，以最短的緩衝時間或其間的中斷來播放剪輯。

Windows Media 技術和其他網際網路產品提供您工具，以瞭解使用者人口統計，並針對個別使用者動態自訂廣播或訊息。
