# <a name="installing-your-skin"></a>安裝您的面板

使用下列程式來安裝您的面板：

1.  將您所有的面板檔案複製到新資料夾。
2.  將包含面板檔案的資料夾複製到可攜式裝置上的 Windows Media Player 資料夾。

    開始 Windows Media Player Mobile。 這會啟動面板 **選擇器**，其中有兩個瀏覽按鈕可讓您流覽可用的面板，並顯示每個面板的預覽。 針對您的面板顯示的名稱將會是您的外觀定義檔名稱。

3.  按一下 **[確定]** ，選取您的外觀。
