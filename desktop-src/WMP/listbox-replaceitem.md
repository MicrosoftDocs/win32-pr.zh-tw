# <a name="listboxreplaceitem"></a>LISTBOX. replaceItem

**ReplaceItem** 方法會以指定的文字取代指定索引處的文字。

``` syntax
        elementID.replaceItem(index, text)
```

## <a name="parameters"></a>參數

<dl> <dt>

<span id="index"></span><span id="INDEX"></span>*指數*
</dt> <dd>

包含要取出的行之索引的 (**長**) **數目**。

</dd> <dt>

<span id="text"></span><span id="TEXT"></span>*文本*
</dt> <dd>

包含新文字的 **字串**。

</dd> </dl>

## <a name="return-value"></a>傳回值

這個方法不會傳回值。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|---------------------------------------------------------|
| 版本<br/> | 適用于 Windows XP 或更新版本的 Windows Media Player<br/> |
