# <a name="listboxsetselectedstate"></a>LISTBOX. setSelectedState

**SetSelectedState** 方法會選取或取消選取具有指定索引的專案。

``` syntax
        elementID.setSelectedState(index, selected)
```

## <a name="parameters"></a>參數

<dl> <dt>

<span id="index"></span><span id="INDEX"></span>*指數*
</dt> <dd>

**Number** (**Long**) ，其中包含要選取或取消選取之專案的索引。

</dd> <dt>

<span id="selected"></span><span id="SELECTED"></span>*選擇*
</dt> <dd>

指出是否要選取專案 (true) 或未選取 (false) 的 **布林** 值。

</dd> </dl>

## <a name="return-value"></a>傳回值

這個方法不會傳回值。

## <a name="remarks"></a>備註

這是用來選取或取消選取多行。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|---------------------------------------------------------|
| 版本<br/> | Windows XP 或更新版本的 Windows Media Player<br/> |
