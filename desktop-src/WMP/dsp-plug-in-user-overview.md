# <a name="dsp-plug-in-user-overview"></a>DSP 外掛程式使用者總覽

對使用者而言，DSP 外掛程式是附加元件功能，會以某種方式變更使用者體驗 Windows Media Player 所播放音訊或影片的方式。 DSP 外掛程式的結果可能會很明顯，例如影片影像中的色彩會反轉，使其看起來像是一種相片，或是非常微妙的東西，例如音訊中的燈光效果，以將感覺 spaciousness 至音效。

使用者可以在 Windows Media Player 中使用 [選項] 對話方塊的 [ **外掛程式** ] 索引標籤，來安裝和卸載 DSP 外掛程式。 使用者接著可以使用 [ **外掛程式** ] 索引標籤上提供的核取方塊來啟用和停用已安裝的外掛程式。 DSP 外掛程式可能會執行屬性頁，而使用者也可以從 [ **外掛程式** ] 索引標籤存取該頁面。屬性頁提供使用者變更設定的方式，這些設定會影響 DSP 外掛程式的運作方式。 例如，屬性頁可能包含滑動軸控制項，可讓使用者變更「回音」效果的衰減時間。
