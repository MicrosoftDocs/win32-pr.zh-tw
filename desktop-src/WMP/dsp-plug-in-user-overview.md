# <a name="dsp-plug-in-user-overview"></a><span data-ttu-id="3e40b-109">DSP 外掛程式使用者總覽</span><span class="sxs-lookup"><span data-stu-id="3e40b-109">DSP Plug-in User Overview</span></span>

<span data-ttu-id="3e40b-110">對使用者而言，DSP 外掛程式是附加元件功能，會以某種方式變更使用者體驗 Windows Media Player 所播放音訊或影片的方式。</span><span class="sxs-lookup"><span data-stu-id="3e40b-110">To the user, DSP plug-ins are add-on features that somehow change the way the user experiences the audio or video that Windows Media Player is playing.</span></span> <span data-ttu-id="3e40b-111">DSP 外掛程式的結果可能會很明顯，例如影片影像中的色彩會反轉，使其看起來像是一種相片，或是非常微妙的東西，例如音訊中的燈光效果，以將感覺 spaciousness 至音效。</span><span class="sxs-lookup"><span data-stu-id="3e40b-111">The result of a DSP plug-in could be something obvious, like the colors in the video image being reversed so it looks like a photographic negative, or something very subtle, like a light reverb effect in the audio to add a feeling of spaciousness to the sound.</span></span>

<span data-ttu-id="3e40b-112">使用者可以在 Windows Media Player 中使用 [選項] 對話方塊的 [ **外掛程式** ] 索引標籤，來安裝和卸載 DSP 外掛程式。</span><span class="sxs-lookup"><span data-stu-id="3e40b-112">Users can install and uninstall DSP plug-ins by using the **Plug-ins** tab of the Options dialog box in Windows Media Player.</span></span> <span data-ttu-id="3e40b-113">使用者接著可以使用 [ **外掛程式** ] 索引標籤上提供的核取方塊來啟用和停用已安裝的外掛程式。 DSP 外掛程式可能會執行屬性頁，而使用者也可以從 [ **外掛程式** ] 索引標籤存取該頁面。屬性頁提供使用者變更設定的方式，這些設定會影響 DSP 外掛程式的運作方式。</span><span class="sxs-lookup"><span data-stu-id="3e40b-113">Users can then enable and disable installed plug-ins using the checkboxes provided on the **Plug-ins** tab. DSP plug-ins may implement a property page, which users can also access from the **Plug-ins** tab. A property page provides the user with a way to change settings that affect the way a DSP plug-in works.</span></span> <span data-ttu-id="3e40b-114">例如，屬性頁可能包含滑動軸控制項，可讓使用者變更「回音」效果的衰減時間。</span><span class="sxs-lookup"><span data-stu-id="3e40b-114">For example, a property page might include a slider control that allows the user to change the decay time of a reverb effect.</span></span>
