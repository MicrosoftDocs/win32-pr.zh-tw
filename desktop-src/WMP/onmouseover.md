# <a name="onmouseover"></a>onmouseover

**Onmouseover** 事件處理常式會處理使用者第一次將指標放在元素上時所發生的事件。

``` syntax
onmouseover
```

## <a name="remarks"></a>備註

此事件處理常式不適用於 **播放清單** 元素。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|-----------------------------------------------------|
| 版本<br/> | Windows Media Player 70 版或更新版本<br/> |
