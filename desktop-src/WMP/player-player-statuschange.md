# <a name="playerstatuschange-event"></a>StatusChange 事件

當 **status** 屬性變更值時，就會發生 **StatusChange** 事件。

## <a name="syntax"></a>語法


```JScript
Player.StatusChange()
```



## <a name="parameters"></a>參數

此事件沒有任何參數。

## <a name="return-value"></a>傳回值

此事件不會傳回值。
