# <a name="writing-code"></a>撰寫程式碼

外觀的真正威力在於能夠回應事件。 當使用者按一下按鈕時，您想要讓您的面板能夠執行某些動作，而您也會想要回應 Windows Media Player 所發生的變更，例如播放的數位媒體檔案或串流的進度。

下列各節討論如何撰寫適用于面板的程式碼：



| 區段                                | 描述                                 |
|----------------------------------------|---------------------------------------------|
| [處理事件](handling-events.md) | 說明如何在外觀程式碼中處理事件。 |
| [使用 JScript](using-jscript.md)     | 說明如何在外觀中撰寫腳本。 |
