# <a name="dsp-plug-in-developer-overview"></a>DSP 外掛程式開發人員總覽

從開發人員的觀點來看，DSP 外掛程式是一種軟體程式，可接收 Windows Media Player 在轉譯階段之前所提供的音訊或影片資料，然後處理該資料，並將資料傳回 Windows Media Player 以進行轉譯。

下列各節提供有關 Windows Media Player DSP 外掛程式的一般資訊：

-   [資料輸入和輸出](data-input-and-output.md)
-   [格式協調](format-negotiation.md)
-   [提供消費者介面](providing-a-user-interface.md)
-   [DSP 外掛程式嚮導](dsp-plug-in-wizard.md)
-   [必要的介面](required-interfaces.md)
-   [註冊 DSP 外掛程式](registering-dsp-plug-ins.md)
-   [更新現有的 DSP 外掛程式](updating-existing-dsp-plug-ins.md)
