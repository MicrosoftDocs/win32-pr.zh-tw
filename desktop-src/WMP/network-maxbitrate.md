# <a name="networkmaxbitrate"></a>MaxBitRate

**MaxBitRate** 屬性會抓取最大可能的影片位元速率。

## <a name="syntax"></a>Syntax

*玩家*。*網路*。**maxBitRate**

## <a name="possible-values"></a>可能的值

這個屬性是唯讀的 **數位** (**long**) 。
