# <a name="ffwdelement"></a>FFWDELEMENT

這是預先定義的 **BUTTONELEMENT** ，具有下列預設值。

``` syntax
onclick="jscript:player.controls.fastForward()"
upToolTip="Fast Forward"
cursor="system"
enabled="wmpenabled:player.controls.fastForward"
```

## <a name="remarks"></a>備註

此 **BUTTONELEMENT** 的所有屬性都可以藉由明確指定來覆寫。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
