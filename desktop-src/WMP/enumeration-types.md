# <a name="windows-media-player-sdk-enumeration-types"></a>Windows Media Player SDK 列舉類型

Windows Media Player SDK 支援下列列舉類型。



| 列舉類型                                                             | Description                                                                                                            |
|------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [WMPBurnFormat](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpburnformat)                                           | 定義燒錄的 Cd 可能類型。                                                                         |
| [WMPBurnState](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpburnstate)                                             | 定義 Windows Media Player 在燒錄 CD 時可能的操作狀態。                                      |
| [WMPDeviceStatus](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpdevicestatus)                                       | 定義裝置目前狀態的可能值。                                                        |
| [WMPFolderScanState](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpfolderscanstate)                                 | 定義 Windows Media Player 的可能操作狀態，因為它會監視數位媒體內容的檔案資料夾。 |
| [WMPLibraryType](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmplibrarytype)                                         | 定義 Windows Media Player 可以連接的可能程式庫類型。                                          |
| [WMPOpenState](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpopenstate)                                             | 定義在開啟數位媒體檔案時，Windows Media Player 可能的操作狀態。                      |
| [WMPPlaylistChangeEventType](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpplaylistchangeeventtype)                 | 定義可以對播放清單進行的變更類型。                                                           |
| [WMPPlayState](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpplaystate)                                             | 定義播放數位媒體檔案時，Windows Media Player 可能的操作狀態。                      |
| [WMPRipState](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpripstate)                                               | 定義 Windows Media Player 的可能操作狀態，因為它會將 CD 翻錄。                                       |
| [WMPStringCollectionChangeEventType](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpstringcollectionchangeeventtype) | 定義字串集合中可能發生之變更的類型。                                                    |
| [WMPSyncState](/previous-versions/windows/desktop/api/wmp/ne-wmp-wmpsyncstate)                                             | 定義 Windows Media Player 的可能操作狀態，因為它會將數位媒體同步至裝置。          |
