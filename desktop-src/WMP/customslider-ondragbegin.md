# <a name="customsliderondragbegin"></a>CUSTOMSLIDER.onDragBegin

**OnDragBegin** 事件處理常式會處理使用者按一下並按住滑鼠左鍵並開始拖曳滑鼠時所發生的事件。

``` syntax
onDragBegin
```
