# <a name="using-the-windows-media-player-control-in-a-net-framework-solution"></a>在 .NET Framework 方案中使用 Windows Media Player 控制項

Windows Media Player 9 系列或更新版本的 ActiveX 控制項支援在 .NET Framework 應用程式中內嵌。 下列各節提供使用 Windows Media Player 控制項搭配 .NET Framework 的特定資訊。



| 區段                                                                                                                                                      | 描述                                                                                                                  |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------|
| [使用 Windows Media Player 控制項搭配 Microsoft Visual Studio](using-the-windows-media-player-control-with-microsoft-visual-studio.md)               | 提供有關將 Windows Media Player 9 系列或更新版本的控制項新增至 Visual Studio 方案的詳細資料。                |
| [在 c # 方案中內嵌 Windows Media Player 控制項](embedding-the-windows-media-player-control-in-a-c--solution.md)                              | 提供建立使用 Windows Media Player 控制項之簡單 c # 應用程式的指示。                      |
| [在 Visual Basic .NET 方案中內嵌 Windows Media Player 控制項](embedding-the-windows-media-player-control-in-a-visual-basic--net-solution.md) | 提供如何建立使用 Windows Media Player 控制項之簡單 Visual Basic .NET 應用程式的指示。        |
| [以程式設計方式建立 Windows Media Player 控制項](creating-the-windows-media-player-control-programmatically.md)                                 | 描述如何在不使用 Visual Studio 工具箱的情況下，在應用程式中建立 Windows Media Player 控制項。            |
| [關於 .NET Framework 資料類型](about--net-framework-data-types.md)                                                                                       | 提供將腳本導向的物件模型參考轉譯成 .NET Framework 基底資料類型所需的資訊。 |
