# <a name="about-user-interface-plug-ins"></a>關於消費者介面外掛程式

Windows Media Player 提供各種控制台，可讓使用者修改播放程式的各個層面，例如影片和圖形等化器設定。 這些控制台可以增強 Windows Media Player 所提供的多媒體經驗，但不一定會提供使用者可能要尋找的所有功能。

面板是提供額外功能的一種方式，但需要開發人員重新建立整個使用者介面， (UI) 。 或者，Windows Media Player 可讓您建立在播放程式的完整模式中顯示的自訂 UI 外掛程式。 這項功能是透過遵循標準 Microsoft 元件物件模型 (COM) 指導方針的程式設計介面來提供。 您可以使用 Windows Media Player 外掛程式嚮導來協助您開始使用，在 Microsoft Visual C++ 中執行這個介面。

下列主題會更詳細地說明 UI 外掛程式。



| 主題                                              | 描述                                                   |
|----------------------------------------------------|---------------------------------------------------------------|
| [UI 外掛程式總覽](ui-plug-in-overview.md)     | 提供 UI 外掛程式的總覽。                          |
| [建立 UI 外掛程式](building-a-ui-plug-in.md) | 提供建立 UI 外掛程式所需的技術詳細資料。 |
