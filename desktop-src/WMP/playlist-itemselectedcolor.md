# <a name="playlistitemselectedcolor"></a>播放清單。 itemSelectedColor

**ItemSelectedColor** 屬性會指定或抓取值，指出播放清單中所選取專案的文字色彩。

``` syntax
        elementID.itemSelectedColor
```

## <a name="possible-values"></a>可能的值

這個屬性是代表色彩的讀取/寫入 **字串** 。 預設值為 Windows 醒目提示文字色彩。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------|
| 版本<br/> | Windows Media Player 10<br/> |
