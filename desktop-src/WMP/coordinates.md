# <a name="coordinates"></a>座標

這會定義相對於背景檔案之影像的 x 和 y 座標。 需要兩個正整數（以逗號分隔）。 您必須使用的座標類型有兩種，視您所定義的映射類型而定。 下表顯示點陣圖類型所允許的座標值。



| 點陣圖類型               | 座標值                                                             |
|---------------------------|-------------------------------------------------------------------------------|
| 背景或超級       | 0、0                                                                           |
| 停用、推送或區域 | 相對於背景影像的影像左上角（以圖元為單位）。 |
