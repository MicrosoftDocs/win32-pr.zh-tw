# <a name="themeauthorversion"></a>AuthorVersion

**AuthorVersion** 屬性會指定或抓取作者指派的面板版本號碼。

``` syntax
theme.authorVersion
```

## <a name="possible-values"></a>可能的值

這個屬性是唯讀 **字串** ，沒有預設值。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
