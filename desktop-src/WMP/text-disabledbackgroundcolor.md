# <a name="textdisabledbackgroundcolor"></a>DisabledBackgroundColor

**DisabledBackgroundColor** 屬性會指定或抓取文字控制項停用時所用的背景色彩。

``` syntax
        elementID.disabledBackgroundColor
```

## <a name="possible-values"></a>可能的值

這個屬性是包含任何 Microsoft Internet Explorer 色彩值的讀取/寫入 **字串** 。

## <a name="remarks"></a>備註

如果未指定 **disabledBackgroundColor** ，則會使用 **backgroundColor** 。

如需說明如何使用 **TEXT** 元素屬性的範例，請參閱 [value](text-value.md)屬性。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
