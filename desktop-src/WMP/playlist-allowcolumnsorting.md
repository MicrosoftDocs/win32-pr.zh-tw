# <a name="playlistallowcolumnsorting"></a>播放清單。 allowColumnSorting

**AllowColumnSorting** 屬性會指定或抓取值，指出是否允許排序資料行內容。

``` syntax
        elementID.allowColumnSorting
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **布林值**。



| 值 | 描述                                  |
|-------|----------------------------------------------|
| true  | 預設值。 允許排序資料行內容。 |
| false | 不允許排序資料行內容。      |



 

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
