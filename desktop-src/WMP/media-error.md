# <a name="mediaerror"></a>媒體。錯誤

如果媒體專案有錯誤條件， **錯誤** 屬性會抓取 **ErrorItem** 物件。

## <a name="syntax"></a>Syntax

*玩家*。*currentMedia*。**錯誤**

## <a name="possible-values"></a>可能的值

這個屬性是唯讀的 **ErrorItem** 物件。

## <a name="remarks"></a>備註

如果無法播放媒體專案，這個屬性會抓取 **ErrorItem** 物件，其中包含發生之問題的相關資訊。

## <a name="requirements"></a>規格需求

|--------------------|------------------------------------------------------------------------------------|
| 版本<br/> | 適用于 Windows XP 或更新版本的 Windows Media Player。<br/>                           |
