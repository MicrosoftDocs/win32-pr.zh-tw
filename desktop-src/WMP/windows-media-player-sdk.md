# <a name="windows-media-player-sdk"></a>Windows Media PlayerSDK

本檔說明 Microsoft Windows Media Player 軟體發展工具組 (SDK) 。 Windows Media Player SDK 是 Microsoft Windows SDK 的其中一個元件。 其他媒體元件包含 Microsoft 媒體基礎 SDK、microsoft Windows 媒體格式 SDK 和 microsoft Windows Media Services SDK。

Windows Media Player SDK 檔的程式設計技術，可用來擴充 Windows Media Player 和 Windows Media Player 行動裝置的功能。 這些技術記載于下列各節：



| 區段                                                                      | 描述                                                                                                                                                                                                                                                                                                                       |
|------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [關於 Windows Media Player SDK](about-the-windows-media-player-sdk.md) | 本節提供有關如何在 SDK 中尋找特定資訊的詳細資料。 其中包含關於新功能的章節，以及有關如何使用 SDK 隨附範例的資訊。                                                                                                                                     |
| [Windows Media Player物件模型](windows-media-player-object-model.md)   | microsoft Windows Media Player 控制項是一種 microsoft ActiveX 控制項，可用來將數位媒體播放功能新增至網頁。 它提供了用來轉譯數位媒體檔案和資料流程的程式設計介面。                                                                                                     |
| [Windows Media Player皮膚](windows-media-player-skins.md)                 | 外觀是一種以 XML 為基礎的技術，用來自訂 Windows Media Player 的使用者介面。 您也可以使用 Windows Media Player 行動面板來自訂 Windows Media Player mobile 的使用者介面。                                                                                                                |
| [Windows Media Player外掛程式](windows-media-player-plug-ins.md)           | 外掛程式是以各種方式擴充 Windows Media Player 功能的物件。 外掛程式類型包括自訂視覺效果、使用者介面外掛程式、DSP 外掛程式和轉譯外掛程式。Windows Media Player本節也會說明行動外掛程式支援。                                              |
| [Windows媒體中繼檔](windows-media-metafiles.md)                       | Windows媒體中繼檔是提供媒體專案及其簡報相關資訊的 XML 檔。 您可以使用中繼檔將媒體專案組織成可包含順暢串流切換、廣告插入和其他功能功能的播放清單。                                                          |
| [Windows媒體播放清單](windows-media-playlists.md)                       | 播放清單是使用 XML 專案來定義媒體專案的動態「智慧型」播放清單或一組靜態媒體專案的檔案。                                                                                                                                                                                              |
| [Windows Media Player線上商店](windows-media-player-online-stores.md) | Windows Media Player 提供可讓數位媒體內容提供者將其服務與 Windows Media Player 整合的功能。 播放程式和線上數位媒體存放區之間的整合，可讓使用者尋找內容、下載和管理檔案、播放內容，以及將內容複寫到 Cd 或裝置。 |
| [Windows Media Player](windows-media-player.md)                             | SDK 的某些功能適用于 Windows Media Player、Windows Media Player ActiveX 控制項和 Windows Media Player 行動裝置。 本節提供這些功能的相關資訊。                                                                                                                                        |
| [詞彙](glossary.md)                                                     | 本章節包含整個 SDK 中使用的詞彙定義。                                                                                                                                                                                                                                                               |



 

> [!Note]  
> 安裝 Windows SDK 不會安裝 Windows Media Player 或 Windows Media Player 行動裝置。 您必須安裝 Windows Media Player，才能使用此 SDK 中的材質。 Windows Media Player行動裝置是 Windows mobile 的一部分，只有在裝置製造商或行動操作員提供的 Pocket PC 或 Smartphone 上才能使用。
