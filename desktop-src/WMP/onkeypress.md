# <a name="onkeypress"></a>onkeypress

當按下英數位鍵 **時，會處理此事件** 處理常式處理的事件。

``` syntax
onkeypress
```

## <a name="remarks"></a>備註

此事件處理常式不適用於播放 *影片* 時的 **播放清單** 元素、**效果** 元素或 **影片** 元素。**無視窗** 為 false。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|-----------------------------------------------------|
| 版本<br/> | Windows Media Player 70 版或更新版本<br/> |
