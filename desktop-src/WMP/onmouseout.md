# <a name="onmouseout"></a>onmouseout

**Onmouseout** 事件處理常式會處理使用者將指標移出元素時所發生的事件。

``` syntax
onmouseout
```

## <a name="remarks"></a>備註

此事件處理常式不適用於 **播放清單** 元素。

| 需求 | 值 |
|--------------------|-----------------------------------------------------|
| 版本<br/> | Windows Media Player 70 版或更新版本<br/> |
