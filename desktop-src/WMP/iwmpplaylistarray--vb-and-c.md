# <a name="iwmpplaylistarray-vb-and-c-interface"></a>IWMPPlaylistArray (VB 和 c # ) 介面

**IWMPPlaylistArray** 介面會公開下列屬性。

## <a name="members"></a>成員

**IWMPPlaylistArray (VB 和 c # )** 介面都有下列類型的成員：

-   [方法](#methods)
-   [屬性](#properties)

### <a name="methods"></a>方法

**IWMPPlaylistArray (VB 和 c # )** 介面都有這些方法。



| 方法                                                                   | 描述                                                                                        |
|:-------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------|
| [**Item**](wmplibiwmpplaylistarray-iwmpplaylistarray-item--vb-and-c.md) | 傳回 **IWMPPlaylist** 介面，代表指定索引處的播放清單。<br/> |



 

### <a name="properties"></a>屬性

**IWMPPlaylistArray (VB 和 c # )** 介面都有這些屬性。



| 屬性                                                                              | 存取類型          | Description                                                   |
|:--------------------------------------------------------------------------------------|:---------------------|:--------------------------------------------------------------|
| [**count**](wmplibiwmpplaylistarray-iwmpplaylistarray-count--vb-and-c.md)<br/> | 唯讀<br/> | 取得播放清單陣列中的播放清單數目<br/> |
