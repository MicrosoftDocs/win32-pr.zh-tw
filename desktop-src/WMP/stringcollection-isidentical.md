# <a name="stringcollectionisidentical-method"></a>StringCollection. isIdentical 方法

**IsIdentical** 方法會抓取值，指出提供的物件是否與目前的物件相同。

## <a name="syntax"></a>語法


```JScript
bRetVal = StringCollection.isIdentical(
  stringCollection
)
```



## <a name="parameters"></a>參數

<dl> <dt>

*stringCollection* \[在\]
</dt> <dd>

要與目前物件比較的 **StringCollection** 物件。

</dd> </dl>

## <a name="return-value"></a>傳回值

這個方法會傳回 **布林值**。

## <a name="remarks"></a>備註

若要使用此方法，需要有程式庫的讀取權限。 如需詳細資訊，請參閱連結 [庫存取](library-access.md)。
