# <a name="windows-media-metafile-elements-reference"></a>Windows媒體中繼檔元素參考

本章節包含用戶端應用程式之所有 Windows 媒體中繼檔元素的參考檔集。 參考專案包含元素的定義、其屬性和值，以及與每個專案相關的特殊條件。

用戶端應用程式支援下列中繼檔元素。



| 元素                                           | 描述                                                                                                            |
|---------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [抽象](abstract-element.md)                  | 包含描述相關聯的 **ASX**、 **橫幅** 或 **ENTRY** 元素的文字。                                 |
| [.ASX](asx-element.md)                            | 將檔案定義為 Windows 媒體中繼檔。                                                                            |
| [作者](author-element.md)                      | 包含媒體剪輯或 Windows 媒體中繼檔的作者名稱。                                           |
| [旗幟](banner-element.md)                      | 指定出現在 Windows Media Player 顯示面板中之圖形的 URL。                               |
| [基地](base-element.md)                          | 指定附加至傳送給 Windows Media Player 之 Url 前面的字串。                                 |
| [註解](comments.md)                          | 指定批註的 XML 語法 ( <!--...--> ).                                                            |
| [版權](copyright-element.md)                | 包含 **ASX** 或 **ENTRY** 元素的著作權資訊。                                                |
| [DURATION](duration-element.md)                  | 指定 Windows Media Player 呈現資料流程的時間長度。                                                    |
| [ENDMARKER](endmarker-element.md)                | 指定 Windows Media Player 停止呈現資料流程的標記。                                           |
| [進入](entry-element.md)                        | 指定媒體剪輯的路徑。                                                                                   |
| [ENTRYREF](entryref-element.md)                  | 連結至外部 Windows 媒體中繼檔中具有 .asx、. 地板蠟、. 300k.wvx 或 wmx 副檔名的 **專案專案**。 |
| [事件](event-element.md)                        | 定義在收到標記為事件的指令碼命令時，Windows Media Player 所採取的行為或動作。      |
| [LOGURL](logurl-element.md)                      | 指示 Windows Media Player 將任何記錄資料提交至指定的 URL。                                            |
| [MOREINFO](moreinfo-element.md)                  | 指定與顯示、剪輯或橫幅相關聯之網站、電子郵件地址或指令碼命令的 URL。               |
| [參數](param-element.md)                        | 指定與播放清單或播放清單元素相關聯之自訂參數的值。                      |
| [PREVIEWDURATION](previewduration-element.md)    | 指定在預覽模式下播放剪輯的時間長度。                                                         |
| [裁判](ref-element.md)                            | 指定一段數位媒體內容的 URL。                                                                  |
| [重複](repeat-element.md)                      | 指定 Windows Media Player 重複一或多個 **ENTRY** 或 **ENTRYREF** 元素的次數。             |
| [面板 (已淘汰) ](skin-element--deprecated.md) | 指定內嵌面板的 URL。                                                                                   |
| [STARTMARKER](startmarker-element.md)            | 指定 Windows Media Player 開始呈現資料流程的標記。                                          |
| [STARTTIME](starttime-element.md)                | 指定 Windows Media Player 將開始呈現資料流程的時間。                                        |
| [標題](title-element--metafile.md)              | 包含 **ASX** 或 **ENTRY** 元素的標題。                                                                 |
