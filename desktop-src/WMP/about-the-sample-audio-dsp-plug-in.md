# <a name="about-the-sample-audio-dsp-plug-in"></a>關於範例音訊 DSP 外掛程式

範例音訊 DSP 外掛程式提供簡單的處理常式，可根據使用者在屬性頁中提供的因素來調整音訊的波幅。 外掛程式接受介於0和1之間的值。 預設值為 1。 值為1時，不會影響音效;其他縮放因素是音訊樣本的乘數。 例如，值 .5 會導致磁片區中的6分貝減少。 值為零會導致無聲。

範例音訊 DSP 外掛程式適用于身歷聲或 mono PCM 音訊。
