# <a name="balanceslider"></a>BALANCESLIDER

這是具有下列預設值的預先定義 **滑杆** 。

``` syntax
toolTip="Balance"
max="100"
min="-100"
value="wmpprop:player.settings.balance"
value_onchange="jscript:player.settings.balance=value;"
```

## <a name="remarks"></a>備註

這會建立 **滑杆** 控制項來設定音訊餘額。 工具提示已當地語系化。 此 **滑杆** 的所有屬性都可以藉由明確指定來覆寫。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|----------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 或更新版本<br/> |
