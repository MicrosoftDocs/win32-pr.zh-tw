# <a name="column-element"></a>COLUMN 元素

**Column** 元素表示播放清單控制項內的資料行。 資料 **行** 元素一律是 **播放清單** 元素的子系。

**COLUMN** 元素支援下列屬性。



| 屬性                                       | 描述                                                        |
|-------------------------------------------------|--------------------------------------------------------------------|
| [columnName](column-columnname.md)             | 指定或抓取 **播放清單** 控制項中的資料行名稱。  |
| [columnID](column-columnid.md)                 | 指定或抓取 **播放清單** 控制項中的資料行識別碼。    |
| [columnResizeMode](column-columnresizemode.md) | 指定或抓取此資料行的調整大小模式。            |
| [columnWidth](column-columnwidth.md)           | 指定或抓取 **播放清單** 控制項中的資料行寬度。 |
