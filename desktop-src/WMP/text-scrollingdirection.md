# <a name="textscrollingdirection"></a>ScrollingDirection

**ScrollingDirection** 屬性會指定或抓取滾動文字的方向。

``` syntax
        elementID.scrollingDirection
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **字串**。



| 值 | 描述                         |
|-------|-------------------------------------|
| Left  | 預設值。 從右至左滾動。 |
| Right | 從左至右滾動。          |



 

## <a name="remarks"></a>備註

如果 [ **滾動** ] 設定為 [false]，則會忽略 **scrollingDirection** 。

如需說明如何使用 **TEXT** 元素屬性的範例，請參閱 [value](text-value.md)屬性。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
