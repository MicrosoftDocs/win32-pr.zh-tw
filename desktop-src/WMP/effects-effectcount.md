# <a name="effectseffectcount"></a>效果。 effectCount

**EffectCount** 屬性會抓取可用視覺效果的數目。

``` syntax
        elementID.effectCount
```

## <a name="possible-values"></a>可能的值

這個屬性是唯讀的 **數位** (**long**) 。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|---------------------------------------------------------|
| 版本<br/> | 適用于 Windows XP 或更新版本的 Windows Media Player<br/> |
