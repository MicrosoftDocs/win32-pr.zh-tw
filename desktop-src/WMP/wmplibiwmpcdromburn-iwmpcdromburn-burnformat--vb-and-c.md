# <a name="iwmpcdromburnburnformat-property"></a>IWMPCdromBurn：： burnFormat 屬性

**BurnFormat** 屬性會取得值，指出要燒錄的 CD 類型。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```CSharp
public WMPBurnFormat burnFormat {get;}
```


```VB

Public ReadOnly Property burnFormat As WMPBurnFormat
```





## <a name="property-value"></a>屬性值

**WMPLib** ，這是 **WMPBurnFormat** 列舉中的值，表示要燒錄的 CD 類型。
