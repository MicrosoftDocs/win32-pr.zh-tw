# <a name="media-item-attributes"></a>媒體專案屬性

當您查看 Windows Media Player 中的程式庫時，您通常會看到有關媒體專案的大量資訊。 例如，除了音訊播放軌的名稱之外，您也可能會看到播放軌的專輯名稱、演出者和編輯器的名稱、音樂的類型等等。

在 Windows Media Player 物件模型中，這些中繼資料專案稱為屬性。 物件模型包含屬性和方法，您可以使用這些屬性和方法來查詢媒體專案和屬性的播放清單。 然後，您可以在使用者介面中顯示內容值，或是您的程式碼可以根據屬性的值採取動作。

下列主題將說明如何使用媒體專案屬性。



| 主題                                                                                      | 描述                                                                                     |
|--------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [瞭解屬性來源](understanding-attribute-sources.md)                     | 描述媒體專案的來源如何影響可能與其相關聯的屬性。 |
| [讀取屬性值](reading-attribute-values.md)                                   | 顯示用於取出屬性值的方法。                                     |
| [具有多個值的屬性](attributes-with-multiple-values.md)                     | 顯示用來取得多重值屬性值的方法。                         |
| [從 CD 或 DVD 讀取屬性值](reading-attribute-values-from-a-cd-or-dvd.md) | 提供範例應用程式，以讀取媒體專案的所有屬性。                 |
| [變更屬性值](changing-attribute-values.md)                                 | 顯示變更屬性值的方法。                                        |
| [電視內容的屬性值](attribute-values-for-tv-content.md)                     | 顯示如何指定包含電視節目之數位視訊內容的屬性。               |
