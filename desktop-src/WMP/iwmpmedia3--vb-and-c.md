-   [方法](#methods)

### <a name="methods"></a>方法

**IWMPMedia3 (VB 和 c # )** 介面都有這些方法。



| 方法                                                                                           | 描述                                                                                            |
|:-------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------|
| [**getAttributeCountByType**](wmplibiwmpmedia3-iwmpmedia3-getattributecountbytype--vb-and-c.md) | 傳回與指定的屬性類型相關聯的屬性數目。<br/>              |
