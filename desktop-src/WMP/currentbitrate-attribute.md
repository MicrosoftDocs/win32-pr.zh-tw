# <a name="currentbitrate-attribute"></a>CurrentBitrate 屬性

**CurrentBitrate** 屬性是專案目前的位元速率，以每秒位數為單位。

## <a name="applies-to"></a>套用至

-   [音訊專案](audio-item-attributes.md)
-   [影片專案](video-item-attributes.md)

## <a name="remarks"></a>備註

若要判斷是否可以變更這個屬性的值，請使用 [isReadOnlyItem](media-isreadonlyitem.md) 方法。

這個屬性的 Windows Media Format SDK 常數是 g \_ wszWMCurrentBitrate。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|-----------------------------------------------|
| 版本<br/> | 僅 Windows Media Player 9 系列<br/> |
