# <a name="about-the-metadatapicture-object"></a>關於 MetadataPicture 物件

**WM/圖片** 中繼資料屬性會提供有關中繼資料影像的詳細資訊，例如專輯封面。 **MetadataPicture** 物件提供一種方式，可取得與影像相關的個別資料。 您可能會將 **WM/圖片** 中繼資料屬性視為複合屬性。 當您使用 *媒體* 時。**getItemInfoByType** 若要取得中繼資料影像的相關資訊，您可以取出具有包含個別資料之屬性的 **MetadataPicture** 物件。
