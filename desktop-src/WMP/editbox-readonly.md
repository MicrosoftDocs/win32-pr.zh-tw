# <a name="editboxreadonly"></a>編輯方塊 readOnly

**ReadOnly** 屬性會指定或抓取值，指出編輯方塊控制項中的文字是唯讀的或可編輯的。

``` syntax
        elementID.readOnly
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **布林** 值，預設值為 false。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|---------------------------------------------------------|
| 版本<br/> | 適用于 Windows XP 或更新版本的 Windows Media Player<br/> |
