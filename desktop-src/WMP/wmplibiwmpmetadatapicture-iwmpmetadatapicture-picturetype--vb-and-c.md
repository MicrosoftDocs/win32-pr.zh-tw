# <a name="iwmpmetadatapicturepicturetype-property"></a>IWMPMetadataPicture：:p ictureType 屬性

`pictureType`屬性會取得中繼資料屬性所代表之影像的圖片類型。

## <a name="syntax"></a>Syntax


```CSharp
public System.String pictureType {get; set;}
```


```VB

Public ReadOnly Property pictureType As System.String
```





## <a name="property-value"></a>屬性值

**System.string** ，此為影像的圖片類型。

## <a name="remarks"></a>備註

使用這個屬性之前，您必須擁有程式庫的讀取權限。 如需詳細資訊，請參閱連結 [庫存取](library-access.md)。
