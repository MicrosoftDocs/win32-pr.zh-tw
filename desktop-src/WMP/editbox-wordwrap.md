# <a name="editboxwordwrap"></a>編輯方塊. wordWrap

**WordWrap** 屬性會指定或抓取值，指出是否已啟用自動換行。

``` syntax
        elementID.wordWrap
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **布林** 值，預設值為 true。

## <a name="remarks"></a>備註

只有當 **editStyle** 設定為 "多行" 時，這個屬性才有用。

如果停用自動換行，且文字無法容納在控制項中，則文字會被截斷。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|---------------------------------------------------------|
| 版本<br/> | 適用于 Windows XP 或更新版本的 Windows Media Player<br/> |
