# <a name="music-file-attributes"></a>音樂檔案屬性

下列屬性適用于不在文件庫中的音樂檔案。 這些屬性會對應至 [Windows Media Player **Advanced Tag Editor** ] 對話方塊中找到的輸入。

-   [Description 屬性](description-attribute.md)
-   [WM/OriginalAlbumTitle 屬性](wm-originalalbumtitle-attribute.md)
-   [WM/OriginalArtist 屬性](wm-originalartist-attribute.md)
-   [WM/OriginalLyricist 屬性](wm-originallyricist-attribute.md)
-   [WM/PartOfSet 屬性](wm-partofset-attribute.md)
