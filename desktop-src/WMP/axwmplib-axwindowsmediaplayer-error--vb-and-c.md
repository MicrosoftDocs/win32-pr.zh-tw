# <a name="axwindowsmediaplayererror-property"></a>AxWindowsMediaPlayer。 Error 屬性

Error 屬性會取得 IWMPError 介面，以提供 IWMPErrorItem 介面集合的存取權。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```CSharp
public IWMPError Error {get;}
```


```VB

Public ReadOnly Property Error As IWMPError
```





## <a name="property-value"></a>屬性值

WMPLib. IWMPError 介面。
