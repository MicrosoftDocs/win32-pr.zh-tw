# <a name="ambientattributesleft"></a>AmbientAttributes left

**左邊** 的屬性會指定或抓取控制項的左座標。

``` syntax
        elementID.left
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **號碼** (**long**) 代表從控制項到父 **視圖****或子視圖左邊** 緣的距離（以圖元為單位）。 預設值為零。 允許負數，在這種情況下， **VIEW** **或子視圖的** 左邊框線會裁剪控制項。
