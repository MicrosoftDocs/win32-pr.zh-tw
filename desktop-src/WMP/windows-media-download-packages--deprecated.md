# <a name="windows-media-download-packages-deprecated"></a>Windows媒體下載套件 (已淘汰) 

本頁記載了未來版本的 Windows Media Player 和 Windows Media Player SDK 可能無法使用的功能。

Windows媒體下載套件會將 Windows Media Player 框線、播放清單資訊和多媒體內容，結合 wmd 副檔名的單一可下載檔案中。 Windows 媒體下載套件可能包含整個專輯的音樂影片，這些影片也會以圖形商標的形式顯示廣告，並連結至線上音樂零售商網站。

使用者只要按一下連結，就可以從網站下載 Windows 媒體下載套件。 將套件下載到使用者的電腦之後，Windows Media Player 會自動解壓縮套件內的檔案，然後將封裝的播放清單新增至 [播放清單] 下拉式方塊、將內容新增至文件庫、在 [完整模式] Windows Media Player 的 [**立即播放**] 窗格中顯示框線外觀，然後播放播放清單中的初始專案。

Windows媒體下載套件提供下列優點：

-   單鍵下載、解壓縮及目錄配合多個檔案到使用者的電腦。
-   內容會在下載後立即播放。
-   自訂框線可以顯示廣告和商標資訊。
-   封裝的播放清單會在媒體櫃中進行編目。
-   網站重新導向可能會從 Windows Media Player 內的相關網站進行。
-   有各種互動式框線控制項可供使用。
-   有多個音訊和影片檔案類型支援封裝。

所有這些優點都可以一起使用。 例如，單一 Windows 媒體下載套件可讓使用者有機會在播放歌曲時看到歌詞、顯示歌曲隨附的影片、查看有關記錄散發者的廣告資訊、觀看專輯封面、造訪風扇網站，以及在文件庫中將內容編目。
