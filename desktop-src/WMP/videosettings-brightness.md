# <a name="videosettingsbrightness"></a>VIDEOSETTINGS。亮度

**亮度** 屬性會指定或抓取影片的亮度設定。

``` syntax
        elementID.brightness
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **號碼** (**long**) ，其值範圍從-127 到 + 127，預設值為0。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|-----------------------------------------------------------------|
| 版本<br/> | Windows XP 或更新版本的 Windows Media Player 版本<br/> |
