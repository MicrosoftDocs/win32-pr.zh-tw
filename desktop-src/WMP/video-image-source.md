# <a name="video-image-source"></a>影片影像來源

影片顯示區域不需要個別的映射。 您可能會想要在背景影像（預設會顯示）中建立一個區域，作為預留位置。 使用您想要在面板定義檔中指定的相同寬度和高度，讓區域矩形在圖形中。 播放影片時，影片將會疊加在背景檔案中的該區域。
