# <a name="listboxitemcount"></a>LISTBOX. itemCount

**ItemCount** 屬性會捕獲清單方塊控制項中的專案數。

``` syntax
        elementID.itemCount
```

## <a name="possible-values"></a>可能的值

這個屬性是唯讀的 **數位** (**long**) 。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|---------------------------------------------------------|
| 版本<br/> | 適用于 Windows XP 或更新版本的 Windows Media Player<br/> |
