# <a name="customslidermax"></a>CUSTOMSLIDER。最大值

**Max** 屬性會指定或抓取自訂滑杆所定義範圍的最大值。

``` syntax
        elementID.max
```

## <a name="possible-values"></a>可能的值

這個屬性是 (**float**) 的讀/寫 **編號**，預設值為100。

## <a name="remarks"></a>備註

**Max** 的值必須大於 **最小** 值。

## <a name="examples"></a>範例

如需說明如何使用 **CUSTOMSLIDER** 元素之屬性的範例，請參閱 [positionImage](customslider-positionimage.md)屬性。
