# <a name="settingsplaycount"></a>設定. playCount

**PlayCount** 屬性會指定或抓取媒體專案的播放次數。

## <a name="syntax"></a>Syntax

playCount

## <a name="possible-values"></a>可能的值

這個屬性是一個讀/寫 **數位** (**long**) ，其最小值為1，預設值為1。

**Windows Media Player 10 Mobile**：不支援此屬性。
