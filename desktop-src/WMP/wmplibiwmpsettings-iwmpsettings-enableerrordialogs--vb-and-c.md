# <a name="iwmpsettingsenableerrordialogs-property"></a>IWMPSettings：： enableErrorDialogs 屬性

**EnableErrorDialogs** 屬性會取得或設定值，指出是否會自動顯示錯誤對話方塊。

## <a name="syntax"></a>Syntax


```CSharp
public System.Boolean enableErrorDialogs {get; set;}
```


```VB

Public Property enableErrorDialogs As System.Boolean
```





## <a name="property-value"></a>屬性值

System.string **值，** 指出是否會自動顯示錯誤對話方塊。 預設值為 **True**。
