# <a name="playlistitemselectedbackgroundfocuslostcolor"></a>播放清單。 itemSelectedBackgroundFocusLostColor

**ItemSelectedBackgroundFocusLostColor** 屬性會指定或抓取值，指出播放清單中所選取專案的文字色彩。

``` syntax
        elementID.itemSelectedBackgroundFocusLostColor
```

## <a name="possible-values"></a>可能的值

這個屬性是代表色彩的讀取/寫入 **字串** 。 預設值是 Windows 按鈕臉部色彩。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------|
| 版本<br/> | Windows Media Player 10<br/> |
