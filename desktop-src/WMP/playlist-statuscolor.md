# <a name="playliststatuscolor"></a>播放清單。 statusColor

**StatusColor** 屬性會指定或抓取 **播放清單** 元素中的狀態行色彩。

``` syntax
        elementID.statusColor
```

## <a name="possible-values"></a>可能的值

這個屬性是包含任何 Microsoft Internet Explorer 色彩值的讀取/寫入 **字串** 。 它的預設值等於 **backgroundColor** 屬性的值。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|---------------------------------------------------|
| 版本<br/> | Windows Media Player 9 系列或更新版本<br/> |
