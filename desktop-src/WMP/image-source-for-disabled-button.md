# <a name="image-source-for-disabled-button"></a>已停用按鈕的影像來源

當按鈕已停用或有對應于 off 的狀態時，您必須定義您想要顯示的影像來源。 影像來自您在 [點陣圖] 區段中定義為已停用的檔案。 您必須輸入映射類型，後面接著一個空格和 @ 符號以及另一個空格。 然後，您必須輸入兩個正整數，以定義您想要在點陣圖檔案內使用之影像的左上座標 (（以圖元為單位）) 。 影像的顯示位置來自針對相對於背景影像的按鈕所定義的座標，但它的位置是由 "Disabled @" 之後的兩個數字所定義，相對於您正在讀取影像的已停用點陣圖。

例如，若要使用已停用之點陣圖中的影像，而該點陣圖位於位於50、60圖元的左上角，則使用下列程式碼：


```C++
Disabled @ 50,60

```



您必須定義停用的點陣圖。 如果您不想要顯示不同的影像，您可以將背景點陣圖定義為已停用的點陣圖，位移為0，0：


```C++
Disabled @ 50,60

```



在上述範例中，50，60是您在已停用的檔案中使用之按鈕的位置 (在此案例中，與背景點陣圖上的按鈕相同的位置) 。 不過，強烈建議您針對所有按鈕顯示已停用的影像，以提供使用者視覺指示，因為在某些情況下，有許多按鈕將無法使用，例如空白播放清單。 如果使用者知道按鈕已停用，就不會繼續嘗試按一下它，或認為您的面板未依照設計的方式運作。
