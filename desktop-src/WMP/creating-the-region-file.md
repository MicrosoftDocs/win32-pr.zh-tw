# <a name="creating-the-region-file"></a>建立區域檔案

您會想要建立一個區域檔案，以顯示哪些區域會回應點擊。 您可以直接將每個元素的基底圖層複製到新圖層，並將其色彩對應到您將在面板定義檔中使用的色彩數位。 確定您在此圖層中建立的影像是穩固的，且不會套用任何效果。 您可能會想要寫下每個影像的色彩數位。 您可以從 Photoshop 色彩選擇器取得色彩數位。 您想要 RGB 數位，也就是三個小數位數。 每個數位的範圍是從0到255。 例如，純紅色會是255、0、0。

> [!Note]  
> 因為 Windows Media Player 10 行動裝置版或更新版本不支援按鈕類型，所以不會在 Windows Media Player 10 行動面板中使用區域檔案。

 

下圖是區域檔案。

![區域檔案](images/ceswmreg.png)

此檔案中只有四個影像，因為只有 [PlayPause]、[停止]、[下一步] 和 [上一個] 按鈕是按類型按鈕。
