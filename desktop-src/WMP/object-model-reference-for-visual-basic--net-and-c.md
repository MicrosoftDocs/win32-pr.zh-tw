# <a name="object-model-reference-for-visual-basic-net-and-c"></a>Visual Basic .net 和 C 的物件模型參考#

> [!Note]  
> 只有 Windows Media Player 9 系列 SDK 和更新版本才支援使用 Windows Media Player ActiveX control 搭配 Visual Basic .net 和 c #。

 

這個參考章節提供 **AxWindowsMediaPlayer** 物件和介面給 Windows Media Player ActiveX 控制項的相關檔。 當您在 Visual Studio .net 2002 或更新版本的解決方案中使用 Windows Media Player ActiveX control 至 COM 互通性時，可以使用這項功能。

如需使用 Windows Media Player 控制項搭配 .NET Framework 的示範，請參閱此 SDK 隨附的 Visual Basic 和 c # 範例。

下列各節記載 Visual Studio 所建立 COM interop 元件所提供的 Visual Basic .net 和 c # 物件和介面。

> [!Note]  
> 需要遠端處理 Windows Media Player 控制項的功能不支援 .net 程式設計。

 



| 區段                                                                                | 描述                                                                                                                    |
|----------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| [AxWindowsMediaPlayer 物件 (VB 和 c # ) ](axwindowsmediaplayer-object--vb-and-c.md)  | 列出適用于 .net 程式設計人員之 Windows Media Player 控制項根物件的屬性、方法和事件。 |
| [適用于 Visual Basic .net 和 C 的介面#](interfaces-for-visual-basic--net-and-c.md) | 列出使用 Windows Media Player ActiveX 控制項之 .net 程式設計人員可用的介面。                           |
