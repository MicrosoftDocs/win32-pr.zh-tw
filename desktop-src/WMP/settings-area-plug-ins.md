# <a name="settings-area-plug-ins"></a>設定區域外掛程式

若為 Microsoft Windows Media Player 11 及更早版本，設定區域 UI 外掛程式會顯示在 [ **目前播放** ] 窗格的 [工具] 區域中，顯示在主顯示區域下方和傳輸控制項上方。 針對 Microsoft Windows Media Player 12 和更新版本，設定區域 UI 外掛程式會顯示為個別的視窗。 一次只能啟用一個設定區域外掛程式。

設定區域外掛程式適用于顯示自訂控制項面板，類似于圖形等化器或影片設定控制項。
