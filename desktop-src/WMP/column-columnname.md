# <a name="columncolumnname"></a>資料行. columnName

**ColumnName** 屬性會指定或抓取 **播放清單** 控制項中的資料行名稱。

``` syntax
        elementID.columnName
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **字串**。

## <a name="remarks"></a>備註

資料行名稱是顯示在資料行標頭中的易記名稱。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|---------------------------------------------------|
| 版本<br/> | Windows Media Player 9 系列或更新版本<br/> |
