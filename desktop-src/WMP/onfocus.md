# <a name="onfocus"></a>onfocus

**Onfocus** 事件處理常式會處理當元素收到鍵盤焦點時所發生的事件。

``` syntax
onfocus
```

## <a name="remarks"></a>備註

**Onfocus** 事件處理常式不適用於 **效果** 元素或 **POPUP** 元素。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|-----------------------------------------------------|
| 版本<br/> | Windows Media Player 70 版或更新版本<br/> |
