# <a name="axwindowsmediaplayernetwork-property"></a>AxWindowsMediaPlayer： network 屬性

Network 屬性會取得 IWMPNetwork 介面，以提供網路資訊和設定的存取權。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```CSharp
public IWMPNetwork network {get;}
```


```VB

Public ReadOnly Property network As IWMPNetwork
```





## <a name="property-value"></a>屬性值

WMPLib. IWMPNetwork 介面。
