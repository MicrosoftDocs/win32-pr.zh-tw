# <a name="publishing-codecs-for-windows-media-player"></a>發佈 Windows Media Player 的編解碼器

WMPlugins 網站已不再支援發佈您的編解碼器，並可讓 Windows Media Player 使用者探索。 您可以改為在您選擇的網站上發佈編解碼器。
