# <a name="wmwmshadowfilesourcedrmtype-windows-media-player-sdk"></a>WM/WMShadowFileSourceDRMType (Windows Media Player SDK) 

**WM/WMShadowFileSourceDRMType** 是數位版權管理系統，用來保護陰影檔案中包含的檔案。

## <a name="remarks"></a>備註

陰影檔案可以是原始程式檔的包裝函式。 這個屬性是一個字串，用來識別來源檔案的 DRM 型別。

您可以使用 [ShadowFilePath](shadowfilepath-attribute.md) 屬性來指定陰影檔。
