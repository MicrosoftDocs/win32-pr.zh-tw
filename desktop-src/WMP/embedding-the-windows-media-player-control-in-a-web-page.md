# <a name="embedding-the-windows-media-player-control-in-a-web-page"></a>將 Windows Media Player 控制項內嵌在網頁中

當您在網頁中內嵌 Windows Media Player 控制項時，您可以採用基本方法或更詳盡的方法。

在基本方法中，您只需使用控制項來播放背景音樂，或將影片顯示新增至您的頁面。

在更詳盡的方法中，您可以藉由新增腳本來透過 Player 物件模型操作控制項，讓您的頁面成為互動式。 這可讓您為自己的設計提供使用者介面，或以動態方式回應指令碼命令或事件，例如媒體轉換。

如需詳細資訊，請參閱[使用網頁中的 Windows Media Player 控制項](using-the-windows-media-player-control-in-a-web-page.md)。
