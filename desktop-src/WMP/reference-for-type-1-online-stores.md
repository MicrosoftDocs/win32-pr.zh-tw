# <a name="reference-for-type-1-online-stores"></a>類型 1 線上商店的參考

> [!Note]  
> 本章節描述專為線上商店使用而設計的功能。 不支援在線上商店的內容之外使用這項功能。

 

Windows Media Player 11 提供目錄編譯器、物件，以及數個支援類型1線上商店的介面。 下列各節將詳細說明這些詳細資料。



| 區段                                                                                    | 描述                                                                                                                                                                                                                        |
|--------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [類型1線上商店的目錄編譯器](catalog-compiler-for-type-1-online-stores.md) | 提供編譯器的參考資料，讓線上商店用來編譯其音樂目錄。                                                                                                                               |
| [IWMPContentContainer 介面](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentcontainer)                                 | 提供線上商店的外掛程式呼叫以檢查內容容器的方法參考頁面。 內容容器代表要下載或購買的一組媒體專案。 由 Windows Media Player 所執行。 |
| [IWMPContentContainerList 介面](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentcontainerlist)                         | 提供線上商店的外掛程式呼叫以檢查內容容器清單的方法參考頁面。 由 Windows Media Player 所執行。                                                                           |
| [IWMPContentPartner 介面](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentpartner)                                     | 提供方法的參考頁面，這些方法會 Windows Media Player 呼叫以從線上商店取得資訊，以及通知線上商店有關使用者的動作。 由線上商店的外掛程式所執行。                  |
| [IWMPContentPartnerCallback 介面](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentpartnercallback)                     | 提供線上商店的外掛程式呼叫的方法參考頁面，以通知 Windows Media Player 非同步作業是否完成。 由 Windows Media Player 所執行。                                    |
| [類型1線上商店的外部物件](external-object-for-type-1-online-stores.md)   | 提供在線上商店提供的網頁中，腳本所使用之屬性、方法和事件的參考頁面。                                                                                                              |
| [類型1線上商店的列舉](enumerations-for-type-1-online-stores.md)         | 提供用於 Windows Media Player 與線上商店外掛程式之間通訊的列舉參考頁面。                                                                                               |
| [類型1線上商店的結構](structures-for-type-1-online-stores.md)             | 針對 Windows Media Player 與線上商店的外掛程式之間的通訊所使用的結構提供參考頁面。                                                                                                 |
