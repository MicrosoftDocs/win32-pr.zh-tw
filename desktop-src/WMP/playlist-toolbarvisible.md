# <a name="playlisttoolbarvisible"></a>播放清單。 toolbarVisible

**ToolbarVisible** 屬性會指定或抓取值，指出播放清單工具列是否會顯示。

``` syntax
        elementID.toolbarVisible
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **布林值**。 預設值為 false。

## <a name="remarks"></a>備註

播放清單工具列提供可讓您輕鬆使用播放清單的使用者介面元素。
