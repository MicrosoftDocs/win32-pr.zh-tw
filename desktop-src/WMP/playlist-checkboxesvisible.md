# <a name="playlistcheckboxesvisible"></a>播放清單。 checkboxesVisible

**CheckboxesVisible** 屬性會指定或抓取表示核取方塊是否可見的值。

``` syntax
        elementID.checkboxesVisible
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **布林值**。



| 值 | 描述                           |
|-------|---------------------------------------|
| true  | 核取方塊是可見的。              |
| false | 預設值。 不會顯示覆選框。 |



 

## <a name="remarks"></a>備註

核取方塊會出現在最左邊的資料行中。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
