# <a name="effectsprevious"></a>效果。上一個

**先前** 的方法會顯示先前的視覺效果預設，視需要移至先前視覺效果的最後一個預設值。

``` syntax
        elementID.previous()
```

## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。

## <a name="remarks"></a>備註

如果目前的預設值是所有可用視覺效果系列中的第一個預設值，最後一個視覺效果的預設值會設為最新的。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
