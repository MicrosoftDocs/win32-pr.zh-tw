# <a name="bits"></a>BITS

[背景智慧型傳送服務](/windows/desktop/Bits/background-intelligent-transfer-service-portal)是 Windows 作業系統的一項功能。 BITS 會在用戶端與伺服器之間)  (下載或上傳檔案，並提供與傳輸相關的進度資訊。 如果您想要使用 c + + 程式碼來傳輸檔案，BITS 是要使用的正確技術。 如果您想要使用線上商店網頁中 JScript 程式碼來傳送檔案，請改用下載管理員。

因為 Windows Media Player 下載管理員會使用 bits，所以您可以採取步驟來設定 bits 複製工作，以 Windows Media Player 管理您的工作。 若要這樣做，您必須遵循[Windows Media Player bits 作業慣例](windows-media-player-bits-job-convention.md)] 區段中所述的慣例，將工作新增至 BITS 傳送佇列。 本節也會說明從線上商店網頁抓取您的 BITS 作業相關資訊的機制。
