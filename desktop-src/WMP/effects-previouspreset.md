# <a name="effectspreviouspreset"></a>效果。 previousPreset

**PreviousPreset** 方法會顯示目前視覺效果的先前預設值。

``` syntax
        elementID.previousPreset()
```

## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。

## <a name="remarks"></a>備註

如果目前的預設值是目前視覺效果清單中的第一個預設值，則會將最後一個預設值設為最新的。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
