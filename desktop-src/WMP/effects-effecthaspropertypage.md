# <a name="effectseffecthaspropertypage"></a>效果。 effectHasPropertyPage

**EffectHasPropertyPage** 屬性會抓取值，指出目前的視覺效果是否有屬性頁。

``` syntax
        elementID.effectHasPropertyPage
```

## <a name="possible-values"></a>可能的值

這個屬性是唯讀的 **布林值**。



| 值 | 描述                                  |
|-------|----------------------------------------------|
| true  | 視覺效果具有屬性頁。           |
| false | 視覺效果沒有屬性頁。 |



 

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
