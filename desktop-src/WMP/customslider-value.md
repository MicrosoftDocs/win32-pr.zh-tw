# <a name="customslidervalue"></a>CUSTOMSLIDER。值

**Value** 屬性會指定或抓取滑杆的目前位置。

``` syntax
        elementID.value
```

## <a name="possible-values"></a>可能的值

這個屬性是 (**float**) 的讀/寫 **數位**，預設值等於 **min** 屬性。

## <a name="remarks"></a>備註

值必須大於或等於 **最小** 值，且小於或等於 **最大****值**。如果值超出範圍，則會發出警告，而且值不會變更。

## <a name="examples"></a>範例

如需說明如何使用 **CUSTOMSLIDER** 元素之屬性的範例，請參閱 [positionImage](customslider-positionimage.md)屬性。
