# <a name="tracknametext"></a>TRACKNAMETEXT

這是預先定義的 **TEXT** 元素，具有下列預設值。

``` syntax
value="wmpprop:player.currentMedia.name"
tabstop="true"
```

## <a name="remarks"></a>備註

這會建立 **文字** 專案，以顯示目前媒體的名稱。 此 **文字** 專案的所有屬性都可以藉由明確指定來覆寫。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|----------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 或更新版本<br/> |
