# <a name="downloaditemresume-method"></a>DownloadItem. resume 方法

> [!Note]  
> 本章節描述專為線上商店使用而設計的功能。 不支援在線上商店的內容之外使用這項功能。

 

**Resume** 方法會繼續進行下載。

## <a name="syntax"></a>語法


```JScript
DownloadItem.resume()
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。

## <a name="remarks"></a>備註

**Resume** 方法對即時下載沒有任何影響。
