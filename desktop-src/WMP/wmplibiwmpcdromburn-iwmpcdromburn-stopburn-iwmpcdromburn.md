# <a name="iwmpcdromburnstopburn-method"></a>IWMPCdromBurn：： stopBurn 方法

**StopBurn** 方法會停止 CD 燒錄進程。

## <a name="syntax"></a>語法


```CSharp
public void stopBurn();
```


```VB

Public Sub stopBurn()
Implements IWMPCdromBurn.stopBurn
```





## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
