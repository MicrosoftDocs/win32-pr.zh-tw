# <a name="dvdresume-method"></a>DVD. resume 方法

**Resume** 方法會從功能表模式回到播放模式，其標題位置與叫用功能表時相同。

## <a name="syntax"></a>語法


```JScript
DVD.resume()
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。

## <a name="remarks"></a>備註

**Windows Media Player 10** 行動裝置版：這個方法一律會成功，但不會執行預定的作業。
