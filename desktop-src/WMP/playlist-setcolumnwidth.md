# <a name="playlistsetcolumnwidth"></a>播放清單。 setColumnWidth

**SetColumnWidth** 方法會指定資料行寬度，並將資料行的調整大小模式變更為「固定」。

``` syntax
        elementID.setColumnWidth(column, width)
```

## <a name="parameters"></a>參數

<dl> <dt>

<span id="column"></span><span id="COLUMN"></span>*列*
</dt> <dd>

**Number** (**long**) 指出正在變更之資料行的索引。

</dd> <dt>

<span id="width"></span><span id="WIDTH"></span>*寬度*
</dt> <dd>

表示新寬度（以圖元為單位） (**長**) **數目**。

</dd> </dl>

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
