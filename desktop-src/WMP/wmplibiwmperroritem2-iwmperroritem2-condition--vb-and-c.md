# <a name="iwmperroritem2condition-property"></a>IWMPErrorItem2：： condition 屬性

**Condition** 屬性會取得值，指出錯誤的條件。

## <a name="syntax"></a>Syntax


```CSharp
public System.Int32 condition {get; set;}
```


```VB

Public ReadOnly Property condition As System.Int32
```





## <a name="property-value"></a>屬性值

作為條件碼的 **system.object** 。

## <a name="remarks"></a>備註

條件碼是 Microsoft 用來為技術支援人員提供其他資訊的值。
