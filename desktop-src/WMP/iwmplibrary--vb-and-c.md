# <a name="iwmplibrary-vb-and-c-interface"></a>IWMPLibrary (VB 和 c # ) 介面

代表程式庫。

**IWMPLibrary** 介面會公開下列屬性。

## <a name="members"></a>成員

**IWMPLibrary (VB 和 c # )** 介面都有下列類型的成員：

-   [方法](#methods)
-   [屬性](#properties)

### <a name="methods"></a>方法

**IWMPLibrary (VB 和 c # )** 介面都有這些方法。



| 方法                                                                     | 描述                                                                                           |
|:---------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------|
| [**isIdentical**](wmplibiwmplibrary-iwmplibrary-isidentical--vb-and-c.md) | 傳回值，這個值會指出提供的物件是否與目前的物件相同。<br/> |



 

### <a name="properties"></a>屬性

**IWMPLibrary (VB 和 c # )** 介面都有這些屬性。



| 屬性                                                                                      | 存取類型          | Description                                                                   |
|:----------------------------------------------------------------------------------------------|:---------------------|:------------------------------------------------------------------------------|
| [**mediaCollection**](wmplibiwmplibrary-iwmplibrary-mediacollection--vb-and-c.md)<br/> | 唯讀<br/> | 取得目前程式庫的 **IWMPMediaCollection** 介面。<br/> |
| [**名字**](wmplibiwmplibrary-iwmplibrary-name--vb-and-c.md)<br/>                       | 唯讀<br/> | 取得目前程式庫的顯示名稱。<br/>                      |
| [**類型**](wmplibiwmplibrary-iwmplibrary-type--vb-and-c.md)<br/>                       | 唯讀<br/> | 取得值，這個值表示程式庫類型。<br/>                      |



 

使用下列方法取得 **IWMPLibrary** 介面。



| Object                                                   | 屬性                                                         |
|----------------------------------------------------------|------------------------------------------------------------------|
| [IWMPLibraryServices](iwmplibraryservices--vb-and-c.md) | [**getLibraryByType**](/previous-versions/windows/desktop/api/wmp/nf-wmp-iwmplibraryservices-getlibrarybytype) |
