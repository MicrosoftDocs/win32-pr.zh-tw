# <a name="submitting-your-visualization"></a>提交您的視覺效果

不再支援透過 WMPlugins 網站提交您的視覺效果。

## <a name="test-your-visualization"></a>測試您的視覺效果

您可能會想要嘗試在多部電腦上安裝視覺效果，以確定一切都能以您想要的方式運作。 請嘗試每個預設值，以確定您的程式碼正常運作。

## <a name="sign-your-visualization"></a>簽署您的視覺效果

您必須提交程式碼簽署的視覺效果，讓使用者可以在安裝視覺效果之前，決定是否信任您的視覺效果。 如需程式碼簽署的詳細資訊，請參閱程式 [代碼簽署簡介](/previous-versions/windows/internet-explorer/ie-developer/platform-apis/ms537361(v=vs.85))。

## <a name="distributing-your-visualization"></a>散佈您的視覺效果

雖然 WMPlugins 網站不再支援發佈您的視覺效果，但您仍然可以從您選擇的網站散發視覺效果。
