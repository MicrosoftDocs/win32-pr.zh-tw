# <a name="about-compatibility"></a>關於相容性

並非每個版本的 Windows Media Player Mobile 都能在每個裝置上運作。 此外，某些版本會預先安裝在裝置上，而其他版本則可以從 Microsoft 網站下載及安裝。 較新版本可能會提供較早版本不支援的外觀功能。

下表詳細說明 Windows Media Player Mobile 與各種裝置的相容性。



| 軟體                          | 硬體                                                                                                                                                                          | 可用性                                                     | 外觀功能                                     |
|-----------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------|---------------------------------------------------|
| Windows Media Player 1。0          | **POCKET PC 3.0 裝置：** Casio Cassiopeia E-115/125，EM-500<br/> Compaq Aero 15xx、Aero 21xx、iPAQ 3150、iPAQ 36xx<br/> HP Jornada 54x<br/>                   | 隨附于裝置。                                             | 僅限音訊。                                       |
| Windows Media Player 1。1          | **掌上大小的電腦：** Casio Cassiopeia E-10 *x* 類別裝置<br/> Compaq Aero 21 *xx* 和 Aero 15 *xx*<br/> HP Jornada 430<br/>                                      | Web 版本。                                                     | 僅限音訊。                                       |
| Windows Media Player 1。2          | **掌上型電腦：** HP Jornada 820、HP Jornada 680/690/720<br/> Vadem Clio C-1050<br/> 利 Telios HC-VJ1c/HC-AJ2<br/> NEC Mobile Pro 780/880 裝置<br/> |  (HPC 3.0) 的 Web 版本。 隨附于裝置 (HPC 3.1) 。<br/> | 僅限音訊。                                       |
| Windows Media Player 7            | **POCKET PC 3.0 裝置：** Casio Cassiopeia E-115/125，例如-800，EM-500<br/> Compaq iPAQ 3150、iPAQ 36 *xx*<br/> HP Jornada 54 *x*<br/>                             | Web 版本。                                                     | 本機音訊和影片。                            |
| Windows Media Player 7。1          | **POCKET PC 3.0 裝置：** Casio Cassiopeia E-115/125，例如-800，EM-500<br/> Compaq iPAQ 3150、iPAQ 36 *xx*<br/> HP Jornada 54 *x*<br/>                             | Web 版本。                                                     | 串流音訊和影片。                        |
| Windows Media Player 9 系列     | **執行 Windows Mobile 2003 的裝置**                                                                                                                                           | 隨附于裝置。                                             | 多個顯示方向。                    |
| Windows Media Player 10 行動裝置版    | **執行 Windows Mobile 2003 第二版的裝置**                                                                                                                            | 隨附于裝置。                                             | 已新增新的按鈕函式和專輯封面元素。 |
| Windows Media Player 10.1 行動裝置。 | **執行 Windows Mobile 5.0 的裝置。**                                                                                                                                           | 隨附于裝置。                                             | 已新增評等元素。                            |
