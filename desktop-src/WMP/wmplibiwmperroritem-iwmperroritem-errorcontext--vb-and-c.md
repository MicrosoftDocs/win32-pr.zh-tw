# <a name="iwmperroritemerrorcontext-property"></a>IWMPErrorItem：： errorCoNtext 屬性

**ErrorCoNtext** 屬性會取得值，指出錯誤的內容。

## <a name="syntax"></a>Syntax


```CSharp
public System.Object errorContext {get; set;}
```


```VB

Public ReadOnly Property errorContext As System.Object
```





## <a name="property-value"></a>屬性值

屬於錯誤內容的 **system.object** 。

## <a name="remarks"></a>備註

錯誤內容是 Microsoft 用來為技術支援人員提供其他資訊的資訊。
