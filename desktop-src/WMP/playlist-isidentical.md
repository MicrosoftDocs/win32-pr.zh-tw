# <a name="playlistisidentical-method"></a>IsIdentical 方法

**IsIdentical** 方法會抓取值，指出提供的播放清單物件是否與目前的 **播放清單** 物件相同。

## <a name="syntax"></a>語法


```JScript
bRetVal = Playlist.isIdentical(
  playlist
)
```



## <a name="parameters"></a>參數

<dl> <dt>

*播放清單* \[在\]
</dt> <dd>

要與目前的 **播放清單** 物件比較。

</dd> </dl>

## <a name="return-value"></a>傳回值

這個方法會傳回 **布林值**。
