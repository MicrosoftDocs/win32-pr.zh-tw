# <a name="sliderdirection"></a>滑杆。方向

**Direction** 屬性會指定或抓取滑杆影像的配置方向。

``` syntax
        elementID.direction
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **字串**。



| 值      | 描述                                                                             |
|------------|-----------------------------------------------------------------------------------------|
| 水平 | 預設值。 最小位置是在左邊，而最大位置則是右邊。 |
| 垂直   | 最小位置是在底部，最大位置是在最上方。          |



 

## <a name="remarks"></a>備註

**ForegroundColor** 或 **foregroundImage** 的填滿或移動是根據指定的值，沿著垂直或水準軸。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
