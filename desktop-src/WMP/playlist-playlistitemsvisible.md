# <a name="playlistplaylistitemsvisible"></a>播放清單。 playlistItemsVisible

**PlaylistItemsVisible** 屬性會指定或抓取值，指出播放清單專案區域是否可見。

``` syntax
        elementID.playlistItemsVisible
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **布林值**。



| 值 | 描述                              |
|-------|------------------------------------------|
| true  | 預設值。 播放清單專案區域是可見的。 |
| false | 看不到播放清單專案區域。      |



 

## <a name="remarks"></a>備註

播放清單專案區域包含資料行標題、資料行的內容，以及捲軸 (（如果有) 的話）。

| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
