# <a name="onmouseup"></a>onmouseup

當滑鼠指標在專案上方時，當使用者放開滑鼠按鍵時， **onmouseup** 事件處理常式會處理發生的事件。

``` syntax
onmouseup
```

## <a name="remarks"></a>備註

此事件處理常式不適用於 **播放清單** 元素。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|-----------------------------------------------------|
| 版本<br/> | Windows Media Player 70 版或更新版本<br/> |
