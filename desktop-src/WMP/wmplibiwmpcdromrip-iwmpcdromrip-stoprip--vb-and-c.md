# <a name="iwmpcdromripstoprip-method"></a>IWMPCdromRip：： stopRip 方法

**StopRip** 方法會停止 CD 翻錄進程。

## <a name="syntax"></a>語法


```CSharp
public void stopRip();
```


```VB

Public Sub stopRip()
Implements IWMPCdromRip.stopRip
```





## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
