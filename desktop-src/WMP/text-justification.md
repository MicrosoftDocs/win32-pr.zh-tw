# <a name="textjustification"></a>文字。對齊方式

[ **對齊** ] 屬性會指定或抓取文字控制項內的文字對齊方式。

``` syntax
        elementID.justification
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **字串**。



| 值  | 描述                                                   |
|--------|---------------------------------------------------------------|
| Left   | 預設值。 將文字對齊文字控制項的左邊。     |
| Right  | 將文字對齊文字控制項的右邊。             |
| Center | 將文字對齊文字控制項的水準置中。 |
