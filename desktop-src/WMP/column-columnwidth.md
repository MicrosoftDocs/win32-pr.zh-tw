# <a name="columncolumnwidth"></a>ColumnWidth

**ColumnWidth** 屬性會指定或抓取 **播放清單** 控制項中的資料行寬度。

``` syntax
        elementID.columnWidth
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **號碼** (**long**) 表示資料行的寬度（以圖元為單位）。

## <a name="remarks"></a>備註

**ColumnResizeMode** 屬性必須設定為「固定」，此屬性才能運作。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|---------------------------------------------------|
| 版本<br/> | Windows Media Player 9 系列或更新版本<br/> |
