# <a name="customslidertooltip"></a>CUSTOMSLIDER 工具提示

**Tooltip** 屬性會指定或抓取滑杆的工具提示文字。

``` syntax
        elementID.toolTip
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **字串** ，最大長度為1024個字元。 它沒有預設值。

## <a name="remarks"></a>備註

當這個屬性設定為 "" (空白字串) 時，不會顯示任何工具提示。

## <a name="examples"></a>範例

如需說明如何使用 **CUSTOMSLIDER** 元素之屬性的範例，請參閱 [positionImage](customslider-positionimage.md)屬性。
