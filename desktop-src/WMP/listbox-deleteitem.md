# <a name="listboxdeleteitem"></a>LISTBOX. deleteItem

**DeleteItem** 方法會刪除指定索引處的清單方塊控制項專案。

``` syntax
        elementID.deleteItem(index)
```

## <a name="parameters"></a>參數

<dl> <dt>

<span id="index"></span><span id="INDEX"></span>*指數*
</dt> <dd>

**Number** (**Long**) ，其中包含要刪除之專案的索引。

</dd> </dl>

## <a name="return-value"></a>傳回值

這個方法不會傳回值。

## <a name="remarks"></a>備註

已刪除行底下的行將會向上移動一個索引值。
