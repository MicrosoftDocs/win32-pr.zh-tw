# <a name="attribute-reference"></a>屬性參考

Windows Media Player 屬性參考記載 Windows Media Player SDK 開發人員感興趣的中繼資料屬性。 下表列出各種類別中的這些屬性。



| 類別                                                                                       | 描述                                                                                                              |
|------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [字母順序參考](alphabetical-attribute-reference.md)                                 | 依字母順序提供預先定義的屬性清單。                                                                  |
| [音訊專案](audio-item-attributes.md)                                                       | 列出媒體櫃中音訊專案可用的屬性。                                                    |
| [CD 播放清單](cd-playlist-attributes.md)                                                     | 列出可供 CD 使用的屬性。                                                                            |
| [CD 曲目](cd-track-attributes.md)                                                           | 列出可供 CD 播放軌使用的屬性。                                                                      |
| [常用 Windows 媒體檔案屬性](commonly-used-windows-media-file-attributes.md) | 列出可用於數位媒體檔案的屬性。                                                            |
| [DVD](dvd-attributes.md)                                                                     | 列出 DVD 可用的屬性。                                                                           |
| [音樂檔案](music-file-attributes.md)                                                       | 列出不在文件庫中的音樂檔案可用的屬性。                                         |
| [其他專案](other-item-attributes.md)                                                       | 列出程式庫中不是音訊、影片、播放清單、相片或選項按鈕專案的可用屬性。 |
| [相片專案](photo-item-attributes.md)                                                       | 列出媒體櫃中的相片專案可用的屬性。                                                     |
| [播放清單](playlist-attributes-ref.md)                                                       | 列出可供從程式庫以媒體專案形式取出的播放清單的屬性。                             |
| [單選項目](radio-item-attributes.md)                                                       | 列出程式庫中的選項按鈕可用的屬性。                                                     |
| [影片專案](video-item-attributes.md)                                                       | 列出媒體櫃中可供影片專案使用的屬性。                                                     |



 

如需中繼資料屬性的詳細資訊，請參閱[Windows 媒體中繼資料使用指導方針](/previous-versions/ms867702(v=msdn.10))。
