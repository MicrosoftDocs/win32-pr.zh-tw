# <a name="pushed-image-source"></a>推送的影像來源

當使用者推送按鈕時，您必須定義您想要顯示的影像來源。 影像來自您在點陣圖區段中所定義的檔案。 您必須輸入映射類型，後面接著一個空格和 @ 符號以及另一個空格。 然後，您必須輸入兩個正整數，以定義您想要在推送影像檔案中使用之影像的上和左座標 (以圖元為單位的) 。 影像的顯示位置來自于相對於背景影像的按鈕所定義的座標;但其來源位置是由 "推 @" 之後的兩個數字所定義，而且相對於您正在讀取影像的已推送影像。

例如，若要使用推送的影像中的影像，其左上角位置是50，60，您會使用下列程式碼：


```C++
Pushed @ 50,60

```



如果您不想要顯示不同的影像，您可以將背景點陣圖定義為已推送的點陣圖。 若要這樣做，請將背景檔案指定為您推送的檔案，並指定按鈕左上角的位移：


```C++
Pushed @ 50,60

```



如果您這樣做，則所有按鈕都不會顯示已推送的影像。 建議您針對所有按鈕顯示已推入的影像，以提供使用者視覺指示，因為它不一定會清楚是否會產生結果，尤其是在播放音樂或關閉點擊音效時。
