# <a name="about-skins"></a>關於外觀

有了 Windows Media Player，使用者可以從各種標準的面板中選擇，每個都提供額外的視覺體驗來增強聆聽和觀賞的樂趣。 Windows Media Player 隨附了數個可供選擇的外觀，但建立和散發自訂的面板相當容易。

外觀只是一或多個電腦美工檔案的集合，以文字檔來組織，告訴 Windows Media Player 如何使用這些檔案來顯示面板。 您可以使用各種程式來修改現有的藝術，或建立自己的簡易藝術。 例如，使用 Microsoft PowerPoint 所提供的繪圖工具和美工圖案，只要拖放就可以建立圖稿。 同樣地，您不需要是程式設計師，就能建立用來定義個別面板的文字檔。

外觀相當容易建立的其中一個原因，就是您可以查看個別的美工圖案和任何面板的原始程式碼，並查看如何將每個面板放在一起。 此 SDK 提供數個簡單的範例面板，以及面板元素和屬性的參考。
