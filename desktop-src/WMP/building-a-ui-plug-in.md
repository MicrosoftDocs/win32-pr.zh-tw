# <a name="building-a-ui-plug-in"></a>建立 UI 外掛程式

您可以使用 Visual Studio 和 Windows Media Player 外掛程式嚮導來建立 UI 外掛程式。 如需設定 wizard 和建立外掛程式的詳細資訊，請參閱 [建立外掛程式](building-a-plug-in.md)。

當您執行嚮導時，將會提供數種不同類型之外掛程式的選項。選擇 [UI 外掛程式]。

Wizard 會建立 UI 外掛程式，您可以將其作為起點。 下列主題將說明如何自訂外掛程式的用途：

-   [自訂 UI 外掛程式](customizing-the-ui-plug-in.md)
