# <a name="equalizersettingswowlevel"></a>EQUALIZERSETTINGS.wowLevel

**WowLevel** 屬性會指定或抓取 SRS WOW 效果等級。

``` syntax
        elementID.wowLevel
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **號碼** (**長**) 範圍從0到100，預設值是50。

## <a name="remarks"></a>備註

SRS WOW 效果是音訊增強效果。 如果 **enhancedAudio** 設定為 false，則會忽略這個屬性。
