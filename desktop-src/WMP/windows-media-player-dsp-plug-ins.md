# <a name="windows-media-player-dsp-plug-ins"></a>Windows Media PlayerDSP 外掛程式

Microsoft Windows Media Player 提供的架構可讓使用者安裝及啟動外掛程式程式，以新增數位信號處理 (DSP) 功能。 DSP 外掛程式是一種 Microsoft DirectX 媒體物件 (DMO) 或媒體基礎轉換 (使用 COM 介面連接到播放程式的) 。 典型的 DSP 外掛程式可能是音訊等化器或影片色調控制項。 SDK 的這個部分會提供您建立自己的 DSP 外掛程式所需的程式設計資訊。

DSP 外掛程式檔分成下列三個區段。



| 區段                                                                      | 描述                                                                                                                                     |
|------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------|
| [關於 DSP 外掛程式](about-dsp-plug-ins.md)                                 | 提供用於 DSP 外掛程式的架構總覽。閱讀本節，以瞭解與此技術相關的一般概念。  |
| [DSP 外掛程式程式設計指南](dsp-plug-ins-programming-guide.md)         | 說明建立 DSP 外掛程式的必要動作。 本節包含範例程式碼和逐步執行程式。                           |
| [DSP 外掛程式程式設計參考](dsp-plug-ins-programming-reference.md) | 提供適用于 DSP 外掛程式的 Windows Media Player SDK 所支援的 COM 介面、方法和列舉類型的詳細參考。 |
