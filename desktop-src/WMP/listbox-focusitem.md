# <a name="listboxfocusitem"></a>LISTBOX. focusItem

**FocusItem** 屬性會指定或抓取包含焦點的行。

``` syntax
        elementID.focusItem
```

## <a name="possible-values"></a>可能的值

這個屬性是 (**長**) 的讀取/寫入 **號碼**。

## <a name="remarks"></a>備註

焦點專案和選取的專案可能會不同。 您可以透過腳本選取專案，但是可以透過腳本將焦點提供給其他專案。 具有焦點的專案在其周圍會有點狀矩形，並且會在按下 ENTER 鍵時選取。
