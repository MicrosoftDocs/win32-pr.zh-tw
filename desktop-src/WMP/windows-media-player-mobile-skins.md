# <a name="windows-media-player-mobile-skins"></a>Windows Media Player 行動外觀

Windows Media Player 的行動套裝軟體含預設的面板，其中包含各種可供您用來播放、倒帶或停止數位媒體的按鈕和滑杆。 面板是可自訂的使用者介面，可為 Windows Media Player Mobile 提供獨特的外觀和功能。

下列各節說明如何建立 Windows Media Player Mobile 的外觀。



| 區段                               | 描述                                                                                                   |
|---------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [關於外觀](about-skins-mobile.md) | 討論瞭解外觀架構所需的概念，以及建立面板時必須遵循的規則。 |
| [外觀指南](skin-guide.md)          | 顯示如何建立簡單的外觀。                                                                            |
| [外觀參考](skin-reference.md)  | 包含面板所有部分的詳細參考。                                                     |
