# <a name="type-1-online-stores"></a>輸入1個線上商店

> [!Note]  
> 本章節描述專為線上商店使用而設計的功能。 不支援在線上商店的內容之外使用這項功能。

 

Windows Media Player 11 支援類型1和類型2線上商店。 類型1存放區比類型2存放區更緊密整合到玩家的使用者介面。 本檔僅適用于類型1存放區，分為下列各節。



| 區段                                                                                      | 描述                                                                                                                                      |
|----------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| [關於類型1線上商店](about-type-1-online-stores.md)                                 | 討論類型1線上商店的主要功能，包括可下載的目錄、程式庫整合和探索頁面。               |
| [類型1線上商店的程式設計指南](programming-guide-for-type-1-online-stores.md) | 提供建立類型1線上商店外掛程式的指引。                                                                                     |
| [類型 1 線上商店的參考](reference-for-type-1-online-stores.md)                 | 提供類型1線上商店所使用之介面、物件、列舉、結構、常數和登錄專案的詳細參考。 |
