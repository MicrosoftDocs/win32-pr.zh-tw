# <a name="type-1-online-store-sample"></a>輸入1線上商店範例

Windows Media Player SDK 包含 Windows Media Player 11 支援的範例類型1線上商店。

若要查看範例類型1線上商店，請依照 [類型1線上商店的登錄機碼和專案](registry-keys-and-entries-for-a-type-1-online-store.md)中所述，設定您的登錄。 範例類型1存放區的索引鍵為2573。
