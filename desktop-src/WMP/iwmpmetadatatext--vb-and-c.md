# <a name="iwmpmetadatatext-vb-and-c-interface"></a>IWMPMetadataText (VB 和 c # ) 介面

提供屬性，以取得複雜文字中繼資料屬性的相關資訊。

**IWMPMetadataText** 介面會公開下列屬性。



| 屬性                                                                         | 描述                              |
|----------------------------------------------------------------------------------|------------------------------------------|
| [說明](wmplibiwmpmetadatatext-iwmpmetadatatext-description--vb-and-c.md) | 取得中繼資料文字的描述。 |
| [text](wmplibiwmpmetadatatext-iwmpmetadatatext-text--vb-and-c.md)               | 取得中繼資料文字。                  |



 

將屬性名稱（例如 **WM/UserWebURL** ）傳遞給下列方法，並轉換傳回的物件，以取得 **IWMPMetadataText** 介面。



|                                        |                                                                                  |
|----------------------------------------|----------------------------------------------------------------------------------|
| 介面                              | 屬性                                                                         |
| [IWMPMedia3](iwmpmedia3--vb-and-c.md) | [getItemInfoByType](wmplibiwmpmedia3-iwmpmedia3-getiteminfobytype--vb-and-c.md) |



 

## <a name="members"></a>成員

**IWMPMetadataText (VB 和 c # )** 介面不會定義任何成員。
