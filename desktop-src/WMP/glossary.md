# <a name="glossary-windows-media-player-sdk"></a>詞彙 (Windows Media Player SDK) 

<dl> <dt>

<span id="projectid515.ambient"></span><span id="PROJECTID515.AMBIENT"></span>**環境**
</dt> <dd>

由一個以上的外觀元素支援。

</dd> <dt>

<span id="projectid515.analog"></span><span id="PROJECTID515.ANALOG"></span>**類比**
</dt> <dd>

使用聲波或類比訊號傳輸音訊及視訊的傳統格式。 類比訊號可能無法與數位喇叭搭配使用；電腦通常是使用數位訊號。

</dd> <dt>

<span id="projectid515.anchor_window"></span><span id="PROJECTID515.ANCHOR_WINDOW"></span>**錨定視窗**
</dt> <dd>

當 Windows Media Player 處於面板模式時，可以出現在螢幕右下角的小視窗。 您可以使用該視窗回復到完整模式並存取其他命令。

</dd> <dt>

<span id="projectid515.announcement"></span><span id="PROJECTID515.ANNOUNCEMENT"></span>**公告**
</dt> <dd>

Windows Media 中繼檔，可讓播放程式取得接收內容所需的資訊。 宣告檔案包含可延伸標記語言 (XML)  (XML) 腳本。

</dd> <dt>

<span id="projectid515.aspect_ratio"></span><span id="PROJECTID515.ASPECT_RATIO"></span>**外觀比例**
</dt> <dd>

影像寬度與其高度的比例。

</dd> <dt>

<span id="projectid515.attribute"></span><span id="PROJECTID515.ATTRIBUTE"></span>**屬性**
</dt> <dd>

名稱/值資料組。

</dd> <dt>

<span id="projectid515.bandwidth"></span><span id="PROJECTID515.BANDWIDTH"></span>**頻寬**
</dt> <dd>

傳輸媒體的資料傳輸容量。

</dd> <dt>

<span id="projectid515.bit_rate"></span><span id="PROJECTID515.BIT_RATE"></span>**位元速率**
</dt> <dd>

每單位時間傳輸的位元數，通常會以每秒的位元數來表示。

</dd> <dt>

<span id="projectid515.broadcast"></span><span id="PROJECTID515.BROADCAST"></span>**廣播**
</dt> <dd>

用戶端用來接收資料流程的方法。 在廣播連接期間，用戶端無法控制串流。 這與隨選簡報相反。

</dd> <dt>

<span id="projectid515.broadcast_multicast"></span><span id="PROJECTID515.BROADCAST_MULTICAST"></span>**廣播多播**
</dt> <dd>

Windows Media 伺服器將一個串流傳遞給許多用戶端。 從用戶端的觀點來看，因為用戶端永遠不會連接到 Windows Media 伺服器，所以廣播是無連線的體驗。

</dd> <dt>

<span id="projectid515.buffer"></span><span id="PROJECTID515.BUFFER"></span>**緩衝區**
</dt> <dd>

在接收裝置或應用程式使用資料前，用於暫時保留資料所預留的記憶體區域。 緩衝可防止資料傳輸的中斷。

</dd> <dt>

<span id="projectid515.caption"></span><span id="PROJECTID515.CAPTION"></span>**標題**
</dt> <dd>

隨附影像或影片的文字，做為補充描述或說出單字的文字記錄。

</dd> <dt>

<span id="projectid515.cd"></span><span id="PROJECTID515.CD"></span>**CD**
</dt> <dd>

請參閱： compact 光碟 (CD) 的定義

</dd> <dt>

<span id="projectid515.client"></span><span id="PROJECTID515.CLIENT"></span>**用戶端**
</dt> <dd>

任何連接到其他電腦或程式或是要求其他電腦或程式提供服務的電腦或程式。 用戶端也可以參考啟用電腦或程式來建立連線的軟體。

</dd> <dt>

<span id="projectid515.clipping_image"></span><span id="PROJECTID515.CLIPPING_IMAGE"></span>**裁剪影像**
</dt> <dd>

未顯示的影像，而是定義另一個影像的可見區域。

</dd> <dt>

<span id="projectid515.closed_disc"></span><span id="PROJECTID515.CLOSED_DISC"></span>**封閉光碟**
</dt> <dd>

CD 或 DVD 在燒錄期間會經過的程序，可向要播放的程式及裝置顯示光碟內容。 光碟完成後，就無法再將額外的資料燒錄進去。 某些播放裝置 (如某些可攜式、家用或車用 CD 及 DVD 播放機) 無法播放未完成的光碟。

</dd> <dt>

<span id="projectid515.codec"></span><span id="PROJECTID515.CODEC"></span>**編解碼器**
</dt> <dd>

壓縮程式/解壓縮程式的縮寫。 用來壓縮及解壓縮數位媒體的軟硬體。

</dd> <dt>

<span id="projectid515.compact_disc__cd_"></span><span id="PROJECTID515.COMPACT_DISC__CD_"></span>光碟 **(光碟)**
</dt> <dd>

數位資料的光學儲存媒介。

</dd> <dt>

<span id="projectid515.compression"></span><span id="PROJECTID515.COMPRESSION"></span>**壓縮**
</dt> <dd>

從數位媒體檔案或串流中移除重複資料，以減少大小或所用頻寬的程序。

</dd> <dt>

<span id="projectid515.content"></span><span id="PROJECTID515.CONTENT"></span>**內容**
</dt> <dd>

數位媒體檔或串流包含的音訊、視訊、影像、文字或任何其他資訊。

</dd> <dt>

<span id="projectid515.down_image"></span><span id="PROJECTID515.DOWN_IMAGE"></span>**下圖**
</dt> <dd>

當滑鼠按鍵位於控制項的關閉位置時所顯示的影像。

</dd> <dt>

<span id="projectid515.download_def"></span><span id="PROJECTID515.DOWNLOAD_DEF"></span>**下載**
</dt> <dd>

利用數據機或網路將檔案從一台電腦複製到另一台電腦中。

</dd> <dt>

<span id="projectid515.element"></span><span id="PROJECTID515.ELEMENT"></span>**元素**
</dt> <dd>

標記語言的基本語法單位，例如 HTML 或 XML。 元素會以開始標記和結束標記分隔。 空白元素是使用空白元素標記定義。

</dd> <dt>

<span id="projectid515.encode"></span><span id="PROJECTID515.ENCODE"></span>**編碼**
</dt> <dd>

將音訊和影片內容轉換成指定的數位格式，通常涉及壓縮技術或加密技術。

</dd> <dt>

<span id="projectid515.error_correction"></span><span id="PROJECTID515.ERROR_CORRECTION"></span>**錯誤修正**
</dt> <dd>

在 Windows Media Player 中，可確保在播放或複製期間，正確地從 CD-ROM 光碟機讀取數位音訊資料。 使用錯誤修正可防止不屬於原始資料的雜音。

</dd> <dt>

<span id="projectid515.extensible_markup_language__xml_"></span><span id="PROJECTID515.EXTENSIBLE_MARKUP_LANGUAGE__XML_"></span>**可延伸標記語言 (XML)  (XML)**
</dt> <dd>

提供描述結構化資料格式的標記語言。 XML 是全球資訊網協會 (W3C) 規格，而且是標準通用標記語言的子集 (SGML) 。

</dd> <dt>

<span id="projectid515.file_format"></span><span id="PROJECTID515.FILE_FORMAT"></span>**檔案格式**
</dt> <dd>

檔案中資料的結構或組織。 檔案格式通常可由檔案副檔名表示。

</dd> <dt>

<span id="projectid515.file_name_extension"></span><span id="PROJECTID515.FILE_NAME_EXTENSION"></span>**副檔名**
</dt> <dd>

加在檔案名稱尾端的字元組，可用來識別檔案類型或格式。

</dd> <dt>

<span id="projectid515.file_type"></span><span id="PROJECTID515.FILE_TYPE"></span>**檔案類型**
</dt> <dd>

一種檔案格式，通常由其副檔名指示。 檔案類型可指示檔案是由何種程式所建立，以及該用何種程式開啟。

</dd> <dt>

<span id="projectid515.frame_rate"></span><span id="PROJECTID515.FRAME_RATE"></span>**畫面播放速率**
</dt> <dd>

每秒顯示的視訊畫面數目。 畫面播放速率越高，圖片中的動作通常就越流暢。

</dd> <dt>

<span id="projectid515.full_mode"></span><span id="PROJECTID515.FULL_MODE"></span>**完整模式**
</dt> <dd>

Windows Media Player 的預設操作狀態，其中所有功能都可供使用。 Player 也可以是面板模式、迷你播放機模式或全螢幕模式。

</dd> <dt>

<span id="projectid515.global_attribute"></span><span id="PROJECTID515.GLOBAL_ATTRIBUTE"></span>**全域屬性**
</dt> <dd>

與整個面板定義檔相關聯的屬性，可以從任何專案存取。

</dd> <dt>

<span id="projectid515.hover_image"></span><span id="PROJECTID515.HOVER_IMAGE"></span>**停留影像**
</dt> <dd>

當滑鼠指標停留在控制項上方時顯示的影像。

</dd> <dt>

<span id="projectid515.http"></span><span id="PROJECTID515.HTTP"></span>**HTTP**
</dt> <dd>

請參閱：超文字傳輸通訊協定 (HTTP) 的定義

</dd> <dt>

<span id="projectid515.hypertext_transfer_protocol__http_"></span><span id="PROJECTID515.HYPERTEXT_TRANSFER_PROTOCOL__HTTP_"></span>**超文字傳輸通訊協定 (HTTP)**
</dt> <dd>

一組標準規則，可讓電腦透過網際網路轉送資訊。

</dd> <dt>

<span id="projectid515.intelligent_streaming"></span><span id="PROJECTID515.INTELLIGENT_STREAMING"></span>**智慧型串流**
</dt> <dd>

一種串流，可偵測網路狀況，並調整影片或音訊串流的內容，以將品質最大化。

</dd> <dt>

<span id="projectid515.library"></span><span id="PROJECTID515.LIBRARY"></span>連結 **庫**
</dt> <dd>

在 Windows Media Player 中，包含電腦、網路磁碟機機或網際網路上數位媒體檔案相關資訊的資料庫。

</dd> <dt>

<span id="projectid515.licensed_file"></span><span id="PROJECTID515.LICENSED_FILE"></span>**授權** 的檔案
</dt> <dd>

Windows Media 檔案，其具有相關聯的授權，可定義檔案的播放方式。 根據授權建立者的不同，授權中註明的限制也會不同。 使用 Windows Media Player 複製 CD 曲目時，可以將授權指派給新建立的檔案。 在該授權下，只能在建立檔案的電腦上播放該檔案。

</dd> <dt>

<span id="projectid515.lossy_compression"></span><span id="PROJECTID515.LOSSY_COMPRESSION"></span>**損量壓縮**
</dt> <dd>

將被視為不必要資訊的資料壓縮的程式會被移除，而且無法在解壓縮時復原。 通常會搭配音訊和視覺資料使用，在此情況下，品質會稍微降低。

</dd> <dt>

<span id="projectid515.mapping_image"></span><span id="PROJECTID515.MAPPING_IMAGE"></span>**對應影像**
</dt> <dd>

未顯示的影像，而是透過使用色彩來定義另一個影像上的重要區域。

</dd> <dt>

<span id="projectid515.media_usage_rights"></span><span id="PROJECTID515.MEDIA_USAGE_RIGHTS"></span>**媒體許可權**
</dt> <dd>

以特定方式使用受保護之 Windows Media 檔案的許可權。 例如，像是線上音樂商店這類的內容提供者可授與您在電腦上播放歌曲的權限 (播放權限)、將歌曲燒錄至音訊 CD 的權限 (燒錄權限)，或是將歌曲同步處理到可攜式裝置的權限 (同步處理權限)。 內容提供者會指定許可權的條款 (例如，它授與您的許可權，以及) 和 Windows Media 數位 Rights Management (DRM) 元件強制執行這些許可權的時間長度。 也稱為授權。

</dd> <dt>

<span id="projectid515.metadata"></span><span id="PROJECTID515.METADATA"></span>**中繼資料**
</dt> <dd>

關於資料的資料。 檔案中繼資料的範例包括標題、主旨、作者及大小等。

</dd> <dt>

<span id="projectid515.metafile_playlist"></span><span id="PROJECTID515.METAFILE_PLAYLIST"></span>**中繼檔播放清單**
</dt> <dd>

Windows Media 中繼檔，其中包含一系列數位媒體專案的相關資訊。

</dd> <dt>

<span id="projectid515.method"></span><span id="PROJECTID515.METHOD"></span>**方法**
</dt> <dd>

與特定物件相關的程式設計函數，可以呼叫來修改物件的屬性，或達成物件負責的特定效果。 方法可以傳遞稱為參數的資訊片段，以修改產生的行為。 方法呼叫的結果通常是傳回不同的資訊片段，然後由程式的另一個部分使用。

</dd> <dt>

<span id="projectid515.microsoft_media_server__mms__protocol"></span><span id="PROJECTID515.MICROSOFT_MEDIA_SERVER__MMS__PROTOCOL"></span>**Microsoft Media Server (MMS) 通訊協定**
</dt> <dd>

使用 UDP 或 TCP 將內容傳遞為單播資料流程的專屬通訊協定。

</dd> <dt>

<span id="projectid515.midi"></span><span id="PROJECTID515.MIDI"></span>**MIDI**
</dt> <dd>

請參閱的定義：音樂檢測數位介面 (MIDI) 

</dd> <dt>

<span id="projectid515.mime"></span><span id="PROJECTID515.MIME"></span>**MIME**
</dt> <dd>

請參閱下列定義：多用途網際網路郵件延伸 (MIME) 

</dd> <dt>

<span id="projectid515.mms_protocol"></span><span id="PROJECTID515.MMS_PROTOCOL"></span>**MMS 通訊協定**
</dt> <dd>

請參閱： Microsoft Media Server (MMS) 通訊協定的定義

</dd> <dt>

<span id="projectid515.moving_picture_experts_group__mpeg_"></span><span id="PROJECTID515.MOVING_PICTURE_EXPERTS_GROUP__MPEG_"></span>**MPEG 格式 (MPEG)**
</dt> <dd>

建立國際標準以將音訊視訊資訊編碼為數位壓縮格式的委員會。 由該委員會開發的個別規則之前會加上 MPEG 這個縮寫字。 例如，MPEG-2 指的是標準 ISO/IEC-11172。

</dd> <dt>

<span id="projectid515.mpeg"></span><span id="PROJECTID515.MPEG"></span>**MPEG**
</dt> <dd>

請參閱： MPEG 格式 (MPEG) 的定義

</dd> <dt>

<span id="projectid515.multicast"></span><span id="PROJECTID515.MULTICAST"></span>**多播**
</dt> <dd>

一種內容傳遞方法，可從媒體伺服器將單一串流傳輸到多個用戶端。 用戶端與伺服器之間並沒有連線。 相反地，伺服器會透過網路將一份串流傳送到使用多點傳送的路由器，而路由器會複寫資料。 然後用戶端就可監視特定多點傳送 IP 位址及連接埠來接收串流。

</dd> <dt>

<span id="projectid515.multipurpose_internet_mail_extension__mime_"></span><span id="PROJECTID515.MULTIPURPOSE_INTERNET_MAIL_EXTENSION__MIME_"></span>多 **用途網際網路郵件延伸 (MIME)**
</dt> <dd>

延伸 Simple Mail Transport Protocol (SMTP) 的標準，用於編碼非 ASCII 資料檔案，例如影片、音效和二進位檔案，以供附件到網際網路電子郵件。

</dd> <dt>

<span id="projectid515.musical_instrument_digital_interface__midi_"></span><span id="PROJECTID515.MUSICAL_INSTRUMENT_DIGITAL_INTERFACE__MIDI_"></span>**(MIDI) 的音樂檢測數位介面**
</dt> <dd>

MIDI 廠商協會 (MMA) 的規格。 此規格定義描述音樂資料 (如按鍵及離鍵訊息)、儲存音樂資料的檔案格式 (稱為標準 MIDI) 及標準硬體介面的通訊協定。

</dd> <dt>

<span id="projectid515.object"></span><span id="PROJECTID515.OBJECT"></span>**物件**
</dt> <dd>

提供特定用途並視為一個單位之屬性和方法的集合。

</dd> <dt>

<span id="projectid515.online_store"></span><span id="PROJECTID515.ONLINE_STORE"></span>**線上商店**
</dt> <dd>

在 Windows Media Player 中，提供依訂用帳戶或購買之數位媒體內容的網站。

</dd> <dt>

<span id="projectid515.packet"></span><span id="PROJECTID515.PACKET"></span>封 **包**
</dt> <dd>

在網路上從某一部電腦或裝置傳輸到另一部的資訊單位。

</dd> <dt>

<span id="projectid515.parameter"></span><span id="PROJECTID515.PARAMETER"></span>**參數**
</dt> <dd>

傳遞至方法以達成特定結果的資訊片段。

</dd> <dt>

<span id="projectid515.partnership"></span><span id="PROJECTID515.PARTNERSHIP"></span>**合作關係**
</dt> <dd>

請參閱：同步合作關係的定義

</dd> <dt>

<span id="projectid515.playlist"></span><span id="PROJECTID515.PLAYLIST"></span>**播放清單**
</dt> <dd>

數位媒體內容的清單。

</dd> <dt>

<span id="projectid515.port"></span><span id="PROJECTID515.PORT"></span>**埠**
</dt> <dd>

以硬體或軟體為主的介面，可用於在電腦與其他裝置之間傳輸資訊。 硬體連接埠為電腦外部可見的實際連線。 軟體連接埠則是程式中的編號閘道，軟體程式可用來交換資訊。

</dd> <dt>

<span id="projectid515.portable_device"></span><span id="PROJECTID515.PORTABLE_DEVICE"></span>**可攜式裝置**
</dt> <dd>

可與電腦或裝置交換檔案或其他資料的行動電子裝置。 可攜式裝置的範例包含掌上型 PC、可攜式數位音樂播放機及Smartphone。

</dd> <dt>

<span id="projectid515.preset"></span><span id="PROJECTID515.PRESET"></span>預設 **值**
</dt> <dd>

預先定義的設定。 針對 Windows Media Player，會針對 [等化器設定] 和大部分的視覺效果定義預設。

</dd> <dt>

<span id="projectid515.progress_bar"></span><span id="PROJECTID515.PROGRESS_BAR"></span>**進度** 列
</dt> <dd>

圖形化使用者介面中的類似計量指標，會顯示進程的進度，例如檔案下載或安裝。

</dd> <dt>

<span id="projectid515.property"></span><span id="PROJECTID515.PROPERTY"></span>**屬性**
</dt> <dd>

指定物件特定詳細資料的一段資訊。

</dd> <dt>

<span id="projectid515.protocol"></span><span id="PROJECTID515.PROTOCOL"></span>**通訊協定**
</dt> <dd>

一組讓電腦能夠交換資訊的標準格式和程序。

</dd> <dt>

<span id="projectid515.proxy_server"></span><span id="PROJECTID515.PROXY_SERVER"></span>**proxy 伺服器**
</dt> <dd>

位於用戶端軟體間網路上的伺服器，例如網頁瀏覽器及其他伺服器。 它可攔截對伺服器提出的所有要求，以判斷是否可自行滿足這些要求。 如果不行，會將要求轉送給其他伺服器。

</dd> <dt>
