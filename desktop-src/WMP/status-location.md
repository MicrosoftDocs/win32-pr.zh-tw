# <a name="status-location"></a>狀態位置

您必須定義您想要使用的狀態顯示位置。 您必須提供四個以逗號分隔的正整數，以圖元為單位來定義狀態顯示的左邊、頂端、寬度和高度（以圖元為單位），相對於背景影像。
