# <a name="imagebutton"></a>IMAGEBUTTON

這是預先定義的 **按鈕** ，具有下列預設值。

``` syntax
cursor="Hand"
```

## <a name="remarks"></a>備註

所有按鈕都可以包含影像。 這個元素適用于程式碼說明，以指出主要作為影像容器使用的按鈕。

此 **按鈕** 的所有屬性都可以藉由明確指定來覆寫。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
