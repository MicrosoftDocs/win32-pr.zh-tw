# <a name="iwmperroritem2-vb-and-c-interface"></a>IWMPErrorItem2 (VB 和 c # ) 介面

提供用來取得錯誤條件程式碼的屬性。

除了繼承自 **IWMPErrorItem** 的屬性之外， **IWMPErrorItem2** 介面也會公開下列屬性。



| 屬性                                                                 | 描述                                          |
|--------------------------------------------------------------------------|------------------------------------------------------|
| [條件](wmplibiwmperroritem2-iwmperroritem2-condition--vb-and-c.md) | 取得值，指出錯誤的條件。 |



 

藉由轉換 **專案** 屬性所傳回的值來取得 **IWMPErrorItem2** 介面， (在 c # 中， **IWMPErrorItem** 介面的 **get \_ Item 方法**) 。

## <a name="members"></a>成員

**IWMPErrorItem2 (VB 和 c # )** 介面不會定義任何成員。
