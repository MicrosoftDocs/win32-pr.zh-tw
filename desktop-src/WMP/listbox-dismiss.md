# <a name="listboxdismiss"></a>LISTBOX。關閉

**關閉** 方法會隱藏控制項。

``` syntax
        elementID.dismiss()
```

## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。

## <a name="remarks"></a>備註

這個方法主要是與 **POPUP** 元素搭配使用，這表示只有在需要時才會顯示的清單方塊控制項。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|---------------------------------------------------------|
| 版本<br/> | Windows XP 或更新版本的 Windows Media Player<br/> |
