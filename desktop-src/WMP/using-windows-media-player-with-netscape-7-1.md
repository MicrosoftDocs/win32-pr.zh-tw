# <a name="using-windows-media-player-with-netscape-71"></a>使用 Windows Media Player 搭配 Netscape 7.1

Netscape 7.1 網頁瀏覽器提供使用 Gecko ActiveX 元件 Windows Media Player ActiveX 控制項的原生支援。 這表示您可以使用 Microsoft Internet Explorer 所用的相同語法，將 Windows Media Player 控制項內嵌在您的 Netscape 7.1 的網頁中。 Netscape 7.1 也支援以類似的方式編寫控制項和事件處理的腳本。

如果您想要在 Netscape 中使用 Windows Media Player 控制項，建議您使用 Netscape 7.1。

如需有關使用 Windows Media Player 控制項搭配 Netscape 7.1 的詳細資訊，請參閱 [netscape 網站](https://developer.mozilla.org/docs/Archive/Windows_Media_in_Netscape)。

如需有關 Gecko ActiveX 元件的詳細資訊，請參閱 [StreamingMedia.com 網站](https://www.streamingmedia.com/Articles/Editorial/Featured-Articles/The-Windows-Media-ActiveX-Control-8211%3b-Not-Just-for-Internet-Explorer-Anymore-64608.aspx)。

此 SDK 的完整下載包含示範網頁瀏覽器偵測的範例。 如需 SDK 隨附範例的詳細資訊，請參閱 [範例](samples.md)。
