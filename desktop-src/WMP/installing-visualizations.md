# <a name="installing-visualizations"></a>安裝視覺效果

您必須為視覺效果的使用者提供安裝程式。 您也必須為使用者提供卸載進程。 目前版本的 Windows Media Player 不會從使用者介面安裝視覺效果。

## <a name="installing-to-the-visualization-folder"></a>安裝至視覺效果資料夾

建議您在安裝 Windows Media Player 的資料夾的 [視覺效果] 子資料夾中安裝所有視覺效果。

## <a name="registering-your-visualization"></a>註冊您的視覺效果

視覺效果是 COM Dll，並遵循所有安裝和移除的一般規則。 您可以使用 regsvr32.exe 或其他安裝工具來註冊您的視覺效果。
