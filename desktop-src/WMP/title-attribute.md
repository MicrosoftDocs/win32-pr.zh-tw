# <a name="title-attribute"></a>Title 屬性

**標題** 屬性是內容的標題。

## <a name="remarks"></a>備註

這個屬性會儲存在程式庫 (或快取) 和數位媒體檔案中。

**Name** 是這個屬性的別名。

這個屬性的 Windows Media Format SDK 常數是 g \_ wszWMTitle。

若要判斷是否可以變更這個屬性的值，請使用 [isReadOnlyItem](media-isreadonlyitem.md) 方法。
