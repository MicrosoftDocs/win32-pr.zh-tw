# <a name="ambientattributesbottom"></a>AmbientAttributes 底部

**底部** 屬性會指定或抓取控制項的底部座標。

``` syntax
        elementID.bottom
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **號碼** (**long**) 代表從控制項到父 **視圖****或子視圖下** 邊緣的距離（以圖元為單位）。 預設值為零。 負數值的行為，或未指定 [AmbientAttributes](ambientattributes-height.md) 時，是未定義的。
