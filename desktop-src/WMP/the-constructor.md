# <a name="the-constructor"></a>函數

Wizard 提供的函式會在成員變數中儲存主要外掛程式類別的指標。 Wizard 會使用這個參考來取得 CSearch 類別中宣告之成員變數的存取權。 不需要執行任何變更。
