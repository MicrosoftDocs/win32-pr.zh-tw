# <a name="dropdownplaylist"></a>DROPDOWNPLAYLIST

這是具有下列預設值的預先定義 **播放清單** 元素。

``` syntax
playlistItemsVisible="false"
```

## <a name="remarks"></a>備註

這將會建立簡單的下拉式 **清單** ，且不會顯示任何專案。 此 **播放清單** 的所有屬性都可以透過明確指定來覆寫。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|----------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 或更新版本<br/> |
