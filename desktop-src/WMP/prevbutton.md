# <a name="prevbutton"></a>PREVBUTTON

這是預先定義的 **按鈕** ，具有下列預設值。

``` syntax
onclick="jscript:player.controls.previous()"
upToolTip="Previous"
cursor="system"
enabled="wmpenabled:player.controls.previous"
```

## <a name="remarks"></a>備註

此按鈕的所有屬性都可以藉由明確指定來覆寫。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
