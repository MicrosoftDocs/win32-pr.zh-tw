# <a name="viewcategory"></a>VIEW. category

**Category** 屬性會指定或抓取適用于 **視圖** 的類別。

``` syntax
        elementID.category
```

## <a name="possible-values"></a>可能的值

這個屬性是包含下列其中一個值的讀取/寫入 **字串** 。



| 值 | 描述                           |
|-------|---------------------------------------|
| 全部   | 預設值。 所有媒體類型的主題 |
| 選項 | 用於收音機播放的 UI             |
| CD    | 用於 CD 播放的 UI                |
| DVD   | 播放 DVD 的 UI               |
| 音樂 | MP3、WAV、MIDI、WMA 的 UI        |
| 影片 | 影片播放的 UI             |



 

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
