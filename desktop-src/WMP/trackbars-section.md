# <a name="trackbars-section"></a>Trackbars 區段

Trackbars 區段包含任何您想要包含之 Trackbars 的相關資訊：


```C++
[ Trackbars ]

//  <Function>  <Location>     <Dis Image Src>  <Thumb Image Src>  <Thumb Size>
//  ----------  ----------     ---------------  -----------------  ------------
    Volume      40,200,170,40  Disabled @ 15,10    VolumeThumb.bmp    23,22

```



這裡只會定義磁片區的對等。 如需有關面板定義檔之 [區段] 區段的詳細資訊，請參閱面板參考中的 [Trackbars](trackbars.md) 。
