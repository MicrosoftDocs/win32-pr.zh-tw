# <a name="displaying-web-pages-in-windows-media-player"></a>顯示 Windows Media Player 中的網頁

Windows Media Player HTMLView 功能可讓網際網路內容提供者 (icp) ，以在播放數位媒體內容時，使用完整模式播放程式的「**立即播放**」功能來顯示網路內容。 Windows Media Player 控制項不支援 (HTMLView。 ) 過去，icp 必須在個別的網頁瀏覽器 (視窗中顯示廣告，稱為「快顯廣告」 ) 或必須將自訂數位媒體播放機新增至其網頁。 透過 HTMLView，icp 可以提供專為 Windows Media Player 9 系列和 Windows Media Player 10 所建立的個人化與整合式體驗，同時確保數位媒體內容仍會在舊版 Windows Media Player 中播放。

本節提供使用 HTMLView、建立 HTMLView 內容，以及解決在下列主題中使用 HTMLView 功能時可能遇到的潛在問題的相關資訊：

-   [使用 HTMLView 的優點](advantages-of-using-htmlview.md)
-   [建立 HTMLView 簡報](creating-an-htmlview-presentation.md)
-   [加入內嵌 Windows Media Player 控制項](adding-an-embedded-windows-media-player-control.md)
-   [確保 Windows Media Player 開啟 HTMLView 內容](ensuring-that-windows-media-player-opens-the-htmlview-content.md)
-   [網頁問題](web-page-issues.md)
-   [控制播放體驗](controlling-the-playback-experience.md)
-   [相容性問題](compatibility-issues.md)
