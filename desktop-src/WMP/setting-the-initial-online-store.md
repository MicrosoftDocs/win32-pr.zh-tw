# <a name="setting-the-initial-online-store"></a>設定初始線上商店

如需使用您的應用程式轉散發 Windows Media Player 軟體的詳細資訊，請參閱轉散發 [Windows Media Player 軟體](redistributing-windows-media-player-software.md)。

當使用者第一次安裝 Windows Media Player 時，安裝應用程式會將初始使用中的線上商店設定為 Microsoft 所選擇的預設值。 個人電腦的原始設備製造商 (Oem) 可以選擇變更此設定。

若要指定及設定使用者安裝 Windows Media Player 時所看到的第一個線上商店，您必須：

-   建立您在使用者的硬碟上安裝的 ServiceInfo 檔。 本檔包含如何設定初始主動線上商店的相關資訊。
-   執行安裝應用程式時，請附加一組命令列參數。

安裝 Windows Media Player 和設定初始線上商店的案例有三種。 下列各節提供每個案例的詳細資料：

-   [離線時安裝](installing-while-offline.md)
-   [在線上從 CD 安裝](installing-from-a-cd-while-online.md)
-   [在線上時從 Web 安裝](installing-from-the-web-while-online.md)
