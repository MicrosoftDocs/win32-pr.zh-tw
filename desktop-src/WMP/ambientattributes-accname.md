# <a name="ambientattributesaccname"></a>AmbientAttributes.accName

**AccName** 屬性會指定或抓取任何元素的名稱。

``` syntax
        elementID.accName
```

## <a name="possible-values"></a>可能的值

這個屬性是具有等於 **識別碼** 屬性之預設值的讀取/寫入 **字串**。

## <a name="remarks"></a>備註

這個屬性會用於協助工具用途。 它允許讀取器程式大聲讀出任何元素的名稱。

這個屬性也適用于按鈕群組控制項內的按鈕元素。
