# <a name="sliderthumbdownimage"></a>滑杆. thumbDownImage

**ThumbDownImage** 屬性會指定或抓取表示 thumb 關閉狀態的影像。

``` syntax
        elementID.thumbDownImage
```

## <a name="possible-values"></a>可能的值

這個屬性是包含影像檔案名稱的讀取/寫入 **字串** 。

## <a name="remarks"></a>備註

**ThumbDownImage** 是選擇性的，而是用來表示 **thumbImage** 之關閉狀態的影像。 如果未提供 **thumbDownImage** ，滑杆將會使用 **thumbImage**。

支援的格式為 BMP、JPG、PNG 和 GIF (不包括動畫 Gif) 。
