# <a name="iwmpcdrom-vb-and-c-interface"></a>IWMPCdrom (VB 和 c # ) 介面

提供一種方式來存取磁片磁碟機中的 CD 或 DVD。

**IWMPCdrom** 介面會公開下列屬性。

## <a name="members"></a>成員

**IWMPCdrom (VB 和 c # )** 介面都有下列類型的成員：

-   [方法](#methods)
-   [屬性](#properties)

### <a name="methods"></a>方法

**IWMPCdrom (VB 和 c # )** 介面都有這些方法。



| 方法                                                     | 描述                                     |
|:-----------------------------------------------------------|:------------------------------------------------|
| [**彈出**](wmplibiwmpcdrom-iwmpcdrom-eject--vb-and-c.md) | 從磁片磁碟機中取出 CD 或 DVD。<br/> |



 

### <a name="properties"></a>屬性

**IWMPCdrom (VB 和 c # )** 介面都有這些屬性。



| 屬性                                                                                | 存取類型          | Description                                                                                                                                  |
|:----------------------------------------------------------------------------------------|:---------------------|:---------------------------------------------------------------------------------------------------------------------------------------------|
| [**driveSpecifier**](wmplibiwmpcdrom-iwmpcdrom-drivespecifier--vb-and-c.md)<br/> | 唯讀<br/> | 取得 CD 或 DVD 光碟機號。<br/>                                                                                                  |
| [**播放 清單**](wmplibiwmpcdrom-iwmpcdrom-playlist--vb-and-c.md)<br/>             | 唯讀<br/> | 取得 [**IWMPPlaylist**](/previous-versions/windows/desktop/api/wmp/nn-wmp-iwmpplaylist) 介面，代表 CD 上的曲目或 DVD 的根層級標題專案。<br/> |
