# <a name="windows-media-player-object-model-reference"></a>Windows Media Player物件模型參考

Windows Media Player 物件模型參考章節包含有關 Windows Media Player ActiveX 控制項物件模型的詳細資訊。 物件模型提供的介面可讓開發人員將 Windows Media Player 功能新增至網頁、c + + 程式和 .net 應用程式。

腳本的物件模型參考會以設計成與 Microsoft JScript 之類的指令碼語言搭配使用，而且會以物件的相關屬性、方法和事件來記錄物件模型。

c + + 的物件模型參考提供 COM 介面的檔，c + + 程式設計人員可以使用這些介面與 Windows Media Player 的 ActiveX 控制項或 Windows Media Player 的 Mobile ActiveX 控制項進行通訊。 這些介面大多是由 Windows Media Player 控制項所執行，並由 c + + 應用程式呼叫。 不過，某些介面 (例如， **IWMPEvents** 和 **IWMPRemoteMediaServices**) 是由 c + + 應用程式所執行，並由 Windows Media Player 控制項呼叫。

Visual Basic .net 和 c # 的物件模型參考會提供 **AxWindowsMediaPlayer** 物件的屬性、方法和事件的相關檔，以及適用于 Windows Media Player ActiveX 控制項的介面，該介面可透過 Visual Studio .net 2002 或更新版本中編碼的解決方案中的 COM 互通性取得。

Windows Media Player 物件模型參考包含下列各節。

|----------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [腳本的物件模型參考](object-model-reference-for-scripting.md)                               | 提供物件模型可供指令碼語言使用之屬性、方法和事件的參考檔集。 這項資訊也適用于 Visual Basic 6.0 程式設計。 |
| [C + + 的物件模型參考](object-model-reference-for-c.md)                                             | 提供 c + + 介面的參考檔。                                                                                                                                                     |
| [Visual Basic .net 和 C 的物件模型參考#](object-model-reference-for-visual-basic--net-and-c.md) | 提供可透過 COM 互通性 Visual Basic .net 和 c # 程式設計人員使用之物件和介面的參考檔。                                                             |
