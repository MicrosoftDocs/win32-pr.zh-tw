# <a name="windows-media-player-object-model-reference"></a>Windows Media Player 物件模型參考

Windows Media Player 物件模型參考章節包含 Windows Media Player ActiveX 控制項物件模型的詳細資訊。 物件模型提供的介面可讓開發人員將 Windows Media Player 功能新增至網頁、c + + 程式和 .NET 應用程式。

腳本的物件模型參考會以設計用來搭配 Microsoft JScript 等指令碼語言使用的樣式呈現，而它會以物件的相關屬性、方法和事件來記錄物件模型。

C + + 的物件模型參考提供 COM 介面的檔，c + + 程式設計人員可以使用這些介面與 Windows Media Player 的 ActiveX 控制項或 Windows Media Player 的行動 ActiveX 控制項進行通訊。 這些介面大多是由 Windows Media Player 控制項所執行，並由 c + + 應用程式呼叫。 不過，某些介面 (例如， **IWMPEvents** 和 **IWMPRemoteMediaServices**) 是由 c + + 應用程式所執行，並由 Windows Media Player 控制項呼叫。

Visual Basic .NET 和 c # 的物件模型參考會提供 **AxWindowsMediaPlayer** 物件的屬性、方法和事件的相關檔，以及適用于 Windows Media Player ActiveX 控制項的介面，該控制項可透過 Visual Studio .NET 2002 或更新版本中編碼的解決方案中的 COM 互通性取得。

Windows Media Player 物件模型參考包含下列各節。

| [C + + 的物件模型參考](object-model-reference-for-c.md)                                             | 提供 c + + 介面的參考檔。                                                                                                                                                     |
