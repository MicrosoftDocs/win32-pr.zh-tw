# <a name="make-a-sketch"></a>製作草圖

您不需要這麼做，但讓您的設計看起來像您想要的方式，可能會非常有説明。 藉由事先規劃，您就可以節省自己必須在多個點陣圖檔案上四處移動的資訊。

啟動 Photoshop，並建立所需大小的新映射。 建立新的圖層以保存粗略的草圖。 如果您有草圖和掃描器，您可能會想要將草圖掃描到 Photoshop。 下圖顯示粗略草圖的範例。

![粗略草圖](images/ceswmskt.png)

[PlayPause] 按鈕位於左上方、[停止]、[下一步] 和 [上一步] 按鈕位於左側，而 [音量] 側條是在底部，而且字幕是在 [PlayPause] 按鈕和 [音量] 的 [標籤] 之間。

建立 Windows Media Player 10 行動裝置版或更新版本的面板時，應使用 [PlayPauseStop] 按鈕，而不是 [PlayPause] 按鈕。 不需要額外的停止按鈕。
