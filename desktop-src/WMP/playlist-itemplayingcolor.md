# <a name="playlistitemplayingcolor"></a>播放清單。 itemPlayingColor

**ItemPlayingColor** 屬性會指定或抓取表示播放清單中目前播放專案的反白顯示色彩。

``` syntax
        elementID.itemPlayingColor
```

## <a name="possible-values"></a>可能的值

這個屬性是包含任何 Microsoft Internet Explorer 色彩值的讀取/寫入 **字串** 。 其預設值為 " \# 00FF00"。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
