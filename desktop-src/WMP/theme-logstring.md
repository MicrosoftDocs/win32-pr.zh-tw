# <a name="themelogstring"></a>LogString

如果啟用記錄功能， **logString** 方法會將使用者定義的字串記錄至錯誤檔。

``` syntax
        theme.logString(message)
```

## <a name="parameters"></a>參數

<dl> <dt>

<span id="message"></span><span id="MESSAGE"></span>*消息*
</dt> <dd>

**字串**，指定要記錄的訊息。

</dd> </dl>

## <a name="return-value"></a>傳回值

這個方法不會傳回值。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
