# <a name="onclick"></a>onclick

**Onclick** 事件處理常式會處理使用者按一下專案時所發生的事件。

``` syntax
onclick
```

## <a name="remarks"></a>備註

當 *影片* 時，這個事件處理常式不適用於 **播放清單** 元素、 **POPUP** 元素、 **VIDEO** 元素、 **LISTBOX** 元素。[**無視窗**] 為 false，**或效果元素** *。***視窗** 化是 true。
