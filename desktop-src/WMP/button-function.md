# <a name="button-function"></a>Button 函數

下表顯示對按鈕函式有效的值。 您不需要定義您將不會在面板中使用之函式的值。



| 值           | 描述                                                                                                                                                  |
|-----------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 播放            | 播放播放清單中目前的選取範圍。                                                                                                                 |
| 暫停           | 暫停目前媒體專案的播放。                                                                                                                   |
| 停止            | 停止播放目前的媒體專案。                                                                                                                    |
| 上一頁            | 移至播放清單中的上一個專案。                                                                                                                  |
| 下一個            | 移至播放清單中的下一個專案。                                                                                                                      |
| Shuffle         | 隨機洗牌播放清單。                                                                                                                              |
| RepeatPlaylist  | 無限期地重複播放清單。                                                                                                                           |
| Mute            | 讓閉嘴音訊輸出。                                                                                                                                   |
| PlayPause       | 切換播放狀態和暫停狀態。                                                                                                          |
| Info            | 提供現正播放之媒體專案的相關資訊。 若為 Windows Media Player 10 個行動裝置版或更新版本的外觀，則不會使用此函數。                     |
| FullScreen      | 將影片大小調整為裝置顯示器的大小上限。                                                                                                 |
| WindowsMedia\*  | 開啟 Pocket Internet Explorer 並載入 Windowsmedia.com 網站。                                                                                       |
| PlayPauseStop\* | 除了在播放狀態和暫停狀態之間切換之外，還可讓使用者在無法使用暫停功能時，停止即時廣播。 |
| VolumeUp\*      | 增加音訊輸出。                                                                                                                                  |
| VolumeDown\*    | 減少音訊輸出。                                                                                                                                  |



 

\*適用于 Windows Media Player 10 行動裝置版或更新版本。
