# <a name="buttonelementdowntooltip"></a>BUTTONELEMENT.downToolTip

**DownToolTip** 屬性會指定或抓取滑鼠移到 **BUTTONELEMENT** 上且 **BUTTONELEMENT** 處於關閉狀態時所顯示的工具提示文字。

``` syntax
        elementID.downToolTip
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入字串，其預設值為 "" (空白字串) ，最大長度為1024個字元。

## <a name="remarks"></a>備註

當這個屬性設定為 "" (空白字串) 時，不會顯示任何工具提示。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
