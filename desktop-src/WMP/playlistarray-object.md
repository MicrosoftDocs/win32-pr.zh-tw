# <a name="playlistarray-object"></a>PlaylistArray 物件

**PlaylistArray** 物件提供依索引編號存取播放清單的方法。

**PlaylistArray** 物件支援下列屬性。



| 屬性                         | 描述                                              |
|----------------------------------|----------------------------------------------------------|
| [計數](playlistarray-count.md) | 捕獲播放清單陣列中的播放清單數目。 |



 

**PlaylistArray** 物件支援下列方法。



| 方法                         | 描述                                                              |
|--------------------------------|--------------------------------------------------------------------------|
| [item](playlistarray-item.md) | 取得指定索引處的 [播放清單](playlist-object.md) 物件。 |



 

**PlaylistArray** 物件是透過下列方法來存取。



| Object                                              | 方法                                                                                 |
|-----------------------------------------------------|----------------------------------------------------------------------------------------|
| [PlaylistCollection](playlistcollection-object.md) | [getAll](playlistcollection-getall.md)、 [getByName](playlistcollection-getbyname.md) |



 

例如， *播放* 程式的用途。*playlistCollection*。**getAll** 用於參考語法區段中。
