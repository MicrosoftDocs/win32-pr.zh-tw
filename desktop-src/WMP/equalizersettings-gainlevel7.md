# <a name="equalizersettingsgainlevel7"></a>EQUALIZERSETTINGS.gainLevel7

**GainLevel7** 屬性會指定或抓取帶狀7的增益等級。

``` syntax
        elementID.gainLevel7
```

## <a name="possible-values"></a>可能的值

這個屬性是一個讀/寫 **號碼** (**Float**) ，其值通常範圍從20到 + 20。 預設值為零。

## <a name="remarks"></a>備註

這個屬性會調整以2kHz 為中心的音訊頻率範圍部分。

如果未指定此屬性，則會保留先前的值。
