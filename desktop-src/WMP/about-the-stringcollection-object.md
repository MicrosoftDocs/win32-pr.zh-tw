# <a name="about-the-stringcollection-object"></a>關於 StringCollection 物件

**StringCollection** 物件會管理字串集合，用來為 **MediaCollection** 物件提供字串處理功能。 **StringCollection** 物件可以透過 **MediaCollection** 物件的 **getAttributeStringCollection** 方法來抓取。 您可以使用這個來處理代表各種媒體專案屬性值的字串集合。
