# <a name="playerapplicationswitchtocontrol-method"></a>PlayerApplication. switchToControl 方法

**switchToControl** 方法會將遠端 Windows Media Player 控制項切換至停駐的狀態。

## <a name="syntax"></a>語法


```JScript
PlayerApplication.switchToControl()
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。

## <a name="remarks"></a>備註

只有在遠端 Windows Media Player 控制項時，才會使用這個方法。

**Windows Media Player 10** 行動裝置版：不支援這個方法。
