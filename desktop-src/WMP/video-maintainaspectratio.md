# <a name="videomaintainaspectratio"></a>MaintainAspectRatio

**MaintainAspectRatio** 屬性會指定或抓取值，指出影片是否會在嘗試符合控制項定義的寬度和高度時維持其外觀比例。

``` syntax
        elementID.maintainAspectRatio
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **布林值**。



| 值 | 描述                                              |
|-------|----------------------------------------------------------|
| true  | 預設值。 影片在調整大小時維持其外觀比例。 |
| false | 當調整大小時，影片不會維持外觀比例。  |



 

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
