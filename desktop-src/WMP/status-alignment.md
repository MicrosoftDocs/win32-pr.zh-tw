# <a name="status-alignment"></a>狀態對齊

您必須定義您想要使用之狀態顯示的對齊方式。 下表顯示有效的值。



| 值  | 描述                                                                       |
|--------|-----------------------------------------------------------------------------------|
| Left   | 文字會靠左對齊， (對齊狀態顯示) 的左邊界。   |
| Center | 文字是在狀態顯示中的中央。                                       |
| Right  | 文字是靠右對齊的 (對齊狀態顯示) 的右邊界。 |
