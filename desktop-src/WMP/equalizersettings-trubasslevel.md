# <a name="equalizersettingstrubasslevel"></a>EQUALIZERSETTINGS.truBassLevel

**TruBassLevel** 屬性會指定或抓取 SRS TruBass 層級。

``` syntax
        elementID.truBassLevel
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **號碼** (**長**) 範圍從0到100，預設值是50。

## <a name="remarks"></a>備註

TruBass 是增強音訊播放軌低音等級音效的效果。如果 **enhancedAudio** 設定為 false，則會忽略這個屬性。
