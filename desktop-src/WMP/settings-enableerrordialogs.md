# <a name="settingsenableerrordialogs"></a>設定. enableErrorDialogs

**EnableErrorDialogs** 屬性會指定或抓取值，指出是否會自動顯示錯誤對話方塊。

## <a name="syntax"></a>Syntax

enableErrorDialogs

## <a name="possible-values"></a>可能的值

這個屬性是讀取/寫入 **布林值**。



| 值 | 描述                                     |
|-------|-------------------------------------------------|
| true  | 預設值。 錯誤對話方塊會自動顯示。 |
| false | 錯誤對話方塊不會自動顯示。      |



 

## <a name="remarks"></a>備註

這個屬性會展示播放程式控制項之遠端實例的特定行為。 如需詳細資訊，請參閱 [遠端處理 Windows Media Player 控制項](remoting-the-windows-media-player-control.md)。
