# <a name="axwindowsmediaplayerctlcontrols-property"></a>AxWindowsMediaPlayer. Ctlcontrols 屬性

Ctlcontrols 屬性會取得 IWMPControls 介面，這個介面會提供一種方式來操作媒體專案的播放。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```CSharp
public IWMPControls Ctlcontrols {get;}
```


```VB

Public ReadOnly Property Ctlcontrols As IWMPControls
```





## <a name="property-value"></a>屬性值

WMPLIB。IWMPControls 介面。
