# <a name="settingsdefaultaudiolanguage"></a>設定. defaultAudioLanguage

**DefaultAudioLanguage** 屬性會抓取 Windows Media Player 中指定之預設音訊語言的 LCID。

## <a name="syntax"></a>Syntax

defaultAudioLanguage

## <a name="possible-values"></a>可能的值

這個屬性是唯讀的 **數位** (**long**) 。

## <a name="remarks"></a>備註

LCID 可唯一識別特定語言方言，稱為地區設定。
