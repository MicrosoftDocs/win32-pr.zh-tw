# <a name="axwindowsmediaplayerclosedcaption-property"></a>AxWindowsMediaPlayer. closedCaption 屬性

ClosedCaption 屬性會取得目前媒體專案的 **IWMPClosedCaption** 介面。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```CSharp
public IWMPClosedCaption closedCaption {get;}
```


```VB

Public ReadOnly Property closedCaption As IWMPClosedCaption
```





## <a name="property-value"></a>屬性值

WMPLIB。IWMPClosedCaption 介面。
