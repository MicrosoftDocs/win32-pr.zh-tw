# <a name="cameramanufacturer-attribute"></a>CameraManufacturer 屬性

**CameraManufacturer** 屬性是代表相機製造商的字串。

## <a name="applies-to"></a>套用至

-   相片專案

## <a name="remarks"></a>備註

若要判斷是否可以變更這個屬性的值，請使用 [isReadOnlyItem](media-isreadonlyitem.md) 方法。
