# <a name="marquee-location"></a>天棚位置

您必須定義要使用的 [字幕] 顯示方塊的位置。 您必須輸入四個正整數（以逗號分隔），以定義相對於背景影像的文字顯示方塊的左邊、頂端、寬度和高度（以圖元為單位）。

一般值為：


```C++
180,46,50,30 

```
