# <a name="automenu-element"></a>AUTOMENU 元素

**AUTOMENU** 元素提供在面板中顯示 **快速存取面板** 的方法。 當您按一下 [ **立即播放** ] 按鈕右邊的箭號時，就會出現在播放程式完整模式中的功能表。 它可讓您以各種方式（例如播放清單中或依專輯、演出者或內容類型）組織的數位媒體檔案的立即存取。 使用 **left** 和 **top** 屬性來指定呼叫 **show** 方法時，應該出現功能表的位置。

**AUTOMENU** 元素支援下列方法。



| 方法                    | 描述                          |
|---------------------------|--------------------------------------|
| [show](automenu-show.md) | 顯示 **快速存取面板**。 |



 

**AUTOMENU** 元素支援下列環境屬性： left 和 top。

> [!Note]  
> 此元素需要 Windows Media Player 9 系列或更新版本。
