# <a name="creating-the-skin-definition-file"></a>建立外觀定義檔

外觀定義檔是簡單的文字檔。 您可以在任何文字編輯器中建立面板定義檔，以節省 ASCII 純文字檔，例如 Microsoft 記事本、Microsoft Visual InterDev®或 Microsoft Visual Studio。 由於您將撰寫 XML 而非 HTML，因此您可能不會想要使用 HTML 編輯器，因為它可能會新增您不想要的額外程式碼，或可能會告訴您您沒有的錯誤。 您可能會想要避免像 Microsoft Word 這樣的字組處理器;雖然 Word 可以儲存純文字檔案，但您可能不想記得在每次儲存時儲存為文字。

-   [新增 BUTTONGROUP](adding-buttongroup.md)
-   [新增播放 BUTTONELEMENT](adding-the-play-buttonelement.md)
-   [新增 Close BUTTONELEMENT](adding-the-close-buttonelement.md)
