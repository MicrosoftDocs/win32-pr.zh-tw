# <a name="purpose-of-a-metafile-playlist"></a>中繼檔播放清單的用途

中繼檔播放清單的基本目的是要將串流處理媒體內容從瀏覽器重新導向至 Windows Media Player，因為大部分的瀏覽器會嘗試下載內容，而不是串流處理。 中繼檔播放清單提供 Windows Media Player 用來接收資料流程的資訊，例如統一資源定位器 (URL) 。 中繼檔播放清單可透過網際網路、內部網路或直接儲存在網路或本機磁片磁碟機上的媒體，來提供媒體檔案的路徑。 將處理重新導向至 Windows Media Player 之後，您就可以更充分掌控串流處理媒體。

中繼檔播放清單是用來控制串流體驗。 藉由將不同的媒體檔案結合成單一內容串流，您可以控制和自訂其簡報。
