# <a name="ondblclick"></a>ondblclick

**Ondblclick** 事件處理常式會處理使用者按兩下元素時所發生的事件。

``` syntax
ondblclick
```

## <a name="remarks"></a>備註

此事件處理常式不適用於 **播放清單** 元素、 **POPUP** **元素、video 元素（** 當 *影片* 時）。[**無視窗**] 為 false，**或效果元素** *。***視窗** 化是 true。

|--------------------|-----------------------------------------------------|
| 版本<br/> | Windows Media Player 70 版或更新版本<br/> |
