# <a name="playlistdropdownlist"></a>播放清單。 dropDownList

**DropDownList** 屬性會指定或抓取值，指出哪些元素會顯示在 **播放** 清單元素之指定實例的下拉式清單方塊中。

``` syntax
        elementID.dropDownList
```

## <a name="possible-values"></a>可能的值

這個屬性是包含下列其中一個值的讀取/寫入 **字串** 。



| 值       | 描述                                                                                      |
|-------------|--------------------------------------------------------------------------------------------------|
| showAlbums  | 顯示專輯。                                                                                    |
| showAll     | 預設值。 顯示所有可用的元素，包括 CD 播放清單、使用者播放清單和選項按鈕。 |
| showCD      | 顯示 CD 播放清單。                                                                           |
| showClips   | 顯示所有剪輯。                                                                                 |
| showCurrent | 顯示目前、未儲存的播放清單。                                                             |
| showLibrary | 只顯示文件庫播放清單。                                                                    |
| showRadio   | 顯示選項按鈕。                                                                             |
| showQueries | 顯示查詢。                                                                                   |
