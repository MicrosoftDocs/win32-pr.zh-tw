# <a name="the-glow-sample"></a>發光範例

本節說明如何建立稱為「發光」的簡單視覺效果。 使用的預設值有三種：紅色、綠色和藍色。 自訂視覺效果嚮導已執行，並已選擇名稱發光。

「發光」只是將時間的色彩變更為音樂。 發光閃爍的原因是，波形的音訊功能有時會在快照集的目前跨越零。 選擇此範例是因為這是最簡單的範例之一。
