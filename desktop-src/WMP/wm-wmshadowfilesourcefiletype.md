# <a name="wmwmshadowfilesourcefiletype-windows-media-player-sdk"></a>WM/WMShadowFileSourceFileType (Windows Media Player SDK) 

**WM/WMShadowFileSourceFileType** 是陰影檔案中包含之檔案的檔案類型。

## <a name="remarks"></a>備註

陰影檔案可以是原始程式檔的包裝函式。 這個屬性是一個字串，其中包含原始程式檔的副檔名 (沒有點分隔符號) 。 例如，如果來源檔案是 AAC 檔案，這個屬性就會包含 "AAC" 字串。

您可以使用 [ShadowFilePath](shadowfilepath-attribute.md) 屬性來指定陰影檔。
