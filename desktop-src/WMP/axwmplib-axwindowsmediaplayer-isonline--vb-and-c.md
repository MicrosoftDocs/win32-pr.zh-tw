# <a name="axwindowsmediaplayerisonline-property"></a>AxWindowsMediaPlayer. isOnline 屬性

IsOnline 屬性會取得值，指出使用者是否已連線到網路。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```CSharp
public System.Boolean isOnline {get;}
```


```VB

Public ReadOnly Property isOnline As System.Boolean
```





## <a name="property-value"></a>屬性值

指出使用者是否已連接到網路的布林值。
