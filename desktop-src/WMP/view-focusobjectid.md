# <a name="viewfocusobjectid"></a>FocusObjectID

**FocusObjectID** 屬性會指定或抓取值，指出哪個元素具有鍵盤焦點。 只有在執行時間才可設定。

``` syntax
        elementID.focusObjectID
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **字串** ，可指定具有焦點之專案的 **識別碼** 。 它沒有預設值。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
