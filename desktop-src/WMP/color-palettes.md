# <a name="color-palettes-windows-media-player-sdk"></a>色彩調色板 (Windows Media Player SDK) 

不同的裝置具有不同的色彩調色板。 您可能會想要為每個色板建立不同的外觀。 建議使用下列調色板：

-   16 位元
-   8 位元
-   2位灰階
