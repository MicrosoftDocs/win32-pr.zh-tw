# <a name="wmdirector-attribute"></a>WM/Director 屬性

**WM/director** 屬性是 Director 的名稱。

## <a name="applies-to"></a>套用至

-   [常用的 Windows Media 檔案屬性](commonly-used-windows-media-file-attributes.md)
-   [DVD](dvd-attributes.md)
-   [影片專案](video-item-attributes.md)

## <a name="remarks"></a>備註

這個屬性會儲存在程式庫 (或快取) 和數位媒體檔案中。

**Director** 是這個屬性的別名。

這個屬性的 Windows Media Format SDK 常數是 g \_ wszWMDirector。

若要判斷是否可以變更這個屬性的值，請使用 [isReadOnlyItem](media-isreadonlyitem.md) 方法。

|--------------------|---------------------------------------------------|
| 版本<br/> | Windows Media Player 9 系列或更新版本<br/> |
