# <a name="playermediacollection"></a>MediaCollection

**MediaCollection** 屬性會捕獲 **mediaCollection** 物件。

## <a name="syntax"></a>Syntax

*玩家* 。**mediaCollection**

## <a name="possible-values"></a>可能的值

這個屬性是唯讀的 **MediaCollection** 物件。

## <a name="remarks"></a>備註

若要取得這個屬性的值，需要有程式庫的讀取權限。 如需詳細資訊，請參閱連結 [庫存取](library-access.md)。
