# <a name="playlistabortcopy"></a>播放清單。 abortCopy

**AbortCopy** 方法會取消複製作業。

``` syntax
        elementID.abortCopy()
```

## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
