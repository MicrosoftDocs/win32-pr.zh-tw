# <a name="modifying-the-echo-sample-property-page"></a>修改 Echo 範例屬性頁

Windows Media Player 外掛程式 Wizard 的 DSP 外掛程式範例所提供的預設屬性頁執行，包含單一編輯方塊控制項，可接收使用者的縮放比例。 您必須修改屬性頁，以接收 Echo 範例所使用的兩個屬性值。

如需有關 DSP 外掛程式屬性頁的總覽，請參閱 [執行音訊 DSP 外掛程式](implementing-an-audio-dsp-plug-in.md)。

下列各節將逐步引導您完成修改範例屬性頁的程式：

-   [修改 Echo 對話資源](modifying-the-echo-dialog-resource.md)
-   [新增和修改事件](adding-and-modifying-the-events.md)
-   [Echo 範例如何保存資料](how-the-echo-sample-persists-data.md)
-   [執行 CEchoPropPage：： OnInitDialog](implementing-cechoproppage--oninitdialog.md)
-   [執行 CEchoPropPage：： Apply](implementing-cechoproppage--apply.md)
-   [執行 CEcho：： FinalConstruct](implementing-cecho--finalconstruct.md)
