# <a name="working-with-the-library"></a>使用程式庫

程式庫是儲存在使用者電腦上或已在該處播放之媒體內容的相關資訊資料庫。 部分資訊會顯示在 Windows Media Player 的 [程式庫] 功能中。 您可以用程式設計的方式存取更大的一組資訊。

媒體專案可以組織在播放清單中。 播放清單可以由使用者或程式碼建立。 Windows Media Player 物件模型提供物件和方法，可讓您用來操作個別的媒體專案和播放清單。

下列各節說明使用程式庫的技術：

-   [程式庫存取](library-access.md)
-   [管理媒體專案](managing-media-items.md)
-   [媒體專案屬性](media-item-attributes.md)
-   [管理播放清單](managing-playlists.md)
