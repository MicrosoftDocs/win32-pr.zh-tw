# <a name="about-windows-media-metafiles"></a>關於 Windows 媒體中繼檔

Microsoft Windows 媒體中繼檔是提供檔案資料流程及其簡報相關資訊的文字檔。 Windows媒體中繼檔是以可延伸標記語言 (XML)  (XML) 語法為基礎，而且可以用 ANSI 或 UNICODE (UTF-8) 格式進行編碼。 它們是由各種元素所組成，以及其相關聯的標記和屬性。 Windows 媒體中繼檔中的每個元素都會定義 Windows Media Player 中的特定設定或動作。 某些元素必須位於檔案中相對於其他元素的特定位置。 有些元素有必要的標記和屬性必須在 Windows 的媒體中繼檔中定義。

Windows媒體下載套件提供一種方式，可將 Windows Media Player 框線、播放清單資訊和多媒體內容合併成單一下載的檔案。
