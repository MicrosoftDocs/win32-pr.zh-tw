# <a name="editboxlinecount"></a>編輯方塊. lineCount

**LineCount** 屬性會抓取編輯方塊控制項中的行數。

``` syntax
        elementID.lineCount
```

## <a name="possible-values"></a>可能的值

這個屬性是唯讀的 **數位** (**long**) 。

## <a name="remarks"></a>備註

只有當 **editStyle** 設定為 "多行" 時，這個屬性才有用。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|---------------------------------------------------------|
| 版本<br/> | Windows XP 或更新版本的 Windows Media Player<br/> |
