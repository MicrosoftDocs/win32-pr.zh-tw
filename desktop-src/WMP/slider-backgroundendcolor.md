# <a name="sliderbackgroundendcolor"></a>滑杆. backgroundEndColor

**BackgroundEndColor** 屬性會指定或抓取滑杆控制項的背景結束色彩。

``` syntax
        elementID.backgroundEndColor
```

## <a name="possible-values"></a>可能的值

這個屬性是包含任何 Microsoft Internet Explorer 色彩值的讀取/寫入 **字串** 。 它沒有預設值。

## <a name="remarks"></a>備註

**BackgroundEndColor** 會與 **backgroundColor** 搭配使用。 所建立的效果是從 **backgroundColor** 到 **backgroundEndColor** 的漸層漸層。
