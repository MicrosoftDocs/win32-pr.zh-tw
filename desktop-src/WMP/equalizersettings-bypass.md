# <a name="equalizersettingsbypass"></a>EQUALIZERSETTINGS，略過

**略過** 屬性會指定或抓取值，指出篩選圖形中是否略過等化器篩選。

``` syntax
        elementID.bypass
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **布林值**。



| 值 | 描述                                |
|-------|--------------------------------------------|
| true  | 預設值。 已略過等化器篩選。 |
| false | 使用的是等化器篩選器。              |



 

## <a name="remarks"></a>備註

如果未指定此屬性，則會保留先前的值。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
