# <a name="creating-the-art"></a>建立藝術

這個面板的藝術是使用 Adobe Photoshop 所建立。 您可以使用其他圖形編輯程式，例如 Jasc 油漆店 Pro 或 Microsoft 小畫家。 Photoshop 被選為支援圖層之圖形編輯應用程式的範例，這有助於建立多個點陣圖檔案，以便彼此對齊。

使用下列步驟建立藝術。 您可以使用許多其他的程式和工具;這只是其中一種做法。 要牢記在心的重要事項是，您想要以面板所需的特定點陣圖檔案為結尾。

-   [製作草圖](make-a-sketch.md)
-   [定義矩形](define-the-rectangles.md)
-   [建立背景](creating-the-background.md)
-   [建立區域檔案](creating-the-region-file.md)
-   [建立推送的檔案](creating-the-pushed-file.md)
-   [建立停用的檔案](creating-the-disabled-file.md)
-   [建立超級檔案](creating-the-super-file.md)
-   [建立 VolumeThumb 檔案](creating-the-volumethumb-file.md)
