# <a name="art-files"></a>美工檔案

您必須為您的面板建立一或多個美工檔案。 如果沒有藝術，使用者將不會看到任何內容。 您可以建立不可見的面板，但看不到它。 甚至，您仍然必須建立美工檔案來保存隱藏的影像，因為面板定義檔需要特定專案的美工檔案。

在面板中有三種用途：主要影像、地圖影像和替代影像。

## <a name="primary-images"></a>主要映射

您必須為您的面板建立主要映射。 這是使用者安裝面板時將看到的內容。 主要映射是由特定面板控制項所建立的一或多個影像所組成。

如果您有一個以上的控制項，您必須指定迭置順序，以定義哪些控制項會顯示在其他控制項的前方。 每個 **View** 元素都會有一個背景影像，您可以將其他元素影像新增至其中，讓您建立主要複合影像。

您也可能有次要影像（例如滑動紙匣）第一次出現時未顯示，但會在使用者採取某些動作時顯示。 這些會遵循與主要映射相同的規則，因為它們是以一組控制項來建立。

## <a name="mapping-images"></a>對應影像

Windows Media Player 面板最強大的功能之一，就是您可以使用影像對應來觸發面板的事件。 影像對應是包含特殊映射的檔案。 不過，影像地圖檔中的影像並不是要供使用者查看，而是由 Windows Media Player 在使用者按一下您的面板時，用來執行動作。

不同的控制項需要不同類型的影像地圖。 例如，如果您將某個影像地圖的一部分標示為特定的紅色值，而且使用者按一下主要影像的對應區域，按鈕就會引發事件。 色彩是用來定義在面板的特定區域中按一下所觸發的事件。

## <a name="alternate-images"></a>替代映射

您也可以設定當使用者執行某些工作時，要顯示的替代影像。 例如，您可以建立按鈕的替代影像，只有在滑鼠停留在按鈕上時才會顯示。 這是讓使用者知道其運作方式的好方法，也可讓使用者介面擁有高度可探索性。 藉由謹慎使用工具提示和暫留影像，您可以建立不尋常的使用者介面，讓使用者有關于哪些可用選項的意見反應。
