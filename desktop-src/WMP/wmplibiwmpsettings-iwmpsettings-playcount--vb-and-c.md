# <a name="iwmpsettingsplaycount-property"></a>IWMPSettings：:p layCount 屬性

**PlayCount** 屬性會取得或設定媒體專案的播放次數。

## <a name="syntax"></a>Syntax


```CSharp
public System.Int32 playCount {get; set;}
```


```VB

Public Property playCount As System.Int32
```





## <a name="property-value"></a>屬性值

是播放次數的 **system.object** ，最小值為1，預設值為1。
