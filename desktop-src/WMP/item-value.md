# <a name="itemvalue"></a>ITEM。值

**Value** 屬性會指定或抓取針對清單方塊或快顯項目目顯示的文字。

``` syntax
        elementID.value
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **字串**。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|---------------------------------------------------------|
| 版本<br/> | Windows XP 或更新版本的 Windows Media Player<br/> |
