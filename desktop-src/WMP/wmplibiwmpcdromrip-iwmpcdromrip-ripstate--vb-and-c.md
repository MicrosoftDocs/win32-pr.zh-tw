# <a name="iwmpcdromripripstate-property"></a>IWMPCdromRip：： ripState 屬性

**RipState** 屬性會取得列舉值，指出目前的翻錄進程狀態。

## <a name="syntax"></a>Syntax


```CSharp
public WMPRipState ripState {get; set;}
```


```VB

Public ReadOnly Property ripState As WMPRipState
```





## <a name="property-value"></a>屬性值

**WMPLib** ，這是 **WMPRipState** 列舉中表示目前狀態的值。
