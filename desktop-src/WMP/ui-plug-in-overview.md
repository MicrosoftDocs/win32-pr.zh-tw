# <a name="ui-plug-in-overview"></a>UI 外掛程式總覽

您可以使用 Windows Media Player 中 [選項] 對話方塊的 [外掛程式] 索引標籤，來安裝、卸載和設定 UI 外掛程式。

Windows Media Player 支援五種類型的 UI 外掛程式。 其中三種類型會對應至播放程式完整模式的 [ **立即播放** ] 窗格的不同區域。 其他兩種類型適用于在個別視窗中顯示的外掛程式，以及沒有顯示 (的外掛程式，但屬性頁可能) 。

當 UI 外掛程式關閉或停用時，就會將其卸載。 當使用者選擇相同類型的另一個外掛程式，或從 [**現正播放**] 窗格中切換時，也會卸載出現在 [**現正播放**] 窗格中的 UI 外掛程式。 當使用者切換至面板模式時，所有外掛程式都會保持載入狀態，但不會顯示。 當 Windows Media Player 關閉時，如果個別的視窗或背景外掛程式正在執行，則會在下次播放程式啟動時自動重載。
