# <a name="jscript-files"></a>JScript檔

JScript 的檔案會以 **VIEW** 元素的 **scriptFile** 屬性載入。 它們必須是文字檔，且應使用副檔名 .js。 如果您的 JScript 檔案與面板定義檔的名稱相同，則會在面板定義檔的相同時間載入 JScript 檔。 例如，如果您有一個名為 laure 的面板定義檔，而且您有一個稱為 laure.js 的 JScript 檔案，就會自動載入 laure.js 檔。

您可以使用 JScript，在您的面板背後建立詳盡的功能。 藉由在 JScript 中建立函式，您幾乎可以使用想像得到來進行任何動作。 例如，您可以在一周的每一天使用不同的播放清單，但在星期五一律有相同的播放清單。

如需搭配使用 JScript 與外觀的詳細資訊，請參閱[使用 JScript](using-jscript.md) 。 請注意，Visual Basic 腳本版本 (VBScript) （在網頁中內嵌 Windows Media Player 控制項時可以使用），不支援搭配使用外觀。
