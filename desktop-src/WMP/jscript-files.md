# <a name="jscript-files"></a>JScript 檔案

JScript 檔案是使用 **VIEW** 專案的 **scriptFile** 屬性載入的。 它們必須是文字檔，而且應該使用副檔名 .js。 如果您的 JScript 檔案與面板定義檔的名稱相同，則會同時載入該 JScript 檔案與面板定義檔。 例如，如果您有一個名為 laure 的面板定義檔，而且您有一個稱為 laure.js 的 JScript 檔案，就會自動載入 laure.js 檔。

您可以使用 JScript，在您的面板背後建立詳盡的功能。 藉由在 JScript 中建立函式，您幾乎可以使用面板來想像得到任何功能。 例如，您可以在一周的每一天使用不同的播放清單，但在星期五一律有相同的播放清單。

如需搭配使用 JScript 與外觀的詳細資訊，請參閱 [使用 jscript](using-jscript.md) 。 請注意，當您在網頁中內嵌 Windows Media Player 控制項時，不支援使用面板來 Visual Basic Scripting Edition (VBScript) 。
