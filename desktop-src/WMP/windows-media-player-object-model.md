# <a name="windows-media-player-object-model"></a>Windows Media Player 物件模型

Microsoft Windows Media Player ActiveX 控制項物件模型所提供的介面，可讓開發人員將 Windows Media Player 和 Windows Media Player 行動功能新增至網頁和其他程式。 下列各節會提供 Windows Media Player 的 ActiveX 控制項物件模型檔。

> [!Note]  
> Windows Media Player 的行動 ActiveX 控制項是以 Windows Media Player ActiveX 控制項為基礎，因此會利用相同的物件模型。 因此，除非另有說明，否則在網頁或應用程式中內嵌 Windows Media Player ActiveX 控制項的任何檔也適用于 Windows Media Player 的行動 ActiveX 控制項。

|------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [關於 Player 物件模型](about-the-player-object-model.md)                             | 以抽象術語討論物件模型的架構。                                                                                                                  |
| [播放機控制指南](player-control-guide.md)                                               | 提供在各種程式設計案例中使用控制項的相關資訊，包括在 .NET Framework 應用程式中使用控制項。                                    |
| [Windows Media Player 物件模型參考](windows-media-player-object-model-reference.md) | 提供腳本、c + + 和 .NET 介面的參考頁面給 Windows Media Player ActiveX 控制項。 也提供列舉類型和媒體屬性的檔。 |
