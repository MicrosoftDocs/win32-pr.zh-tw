# <a name="video-location"></a>影片位置

您必須在面板中定義影片顯示區域的位置。 您必須輸入四個正整數（以逗號分隔），以圖元為單位來定義影片顯示的左邊、頂端、寬度和高度（以圖元為單位）（相對於背景影像）。
