# <a name="downloadcollectionclear-method"></a>DownloadCollection Clear 方法

> [!Note]  
> 本章節描述專為線上商店使用而設計的功能。 不支援在線上商店的內容之外使用這項功能。

 

**Clear** 方法會從下載集合中移除所有專案。

## <a name="syntax"></a>語法


```JScript
DownloadCollection.Clear()
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
