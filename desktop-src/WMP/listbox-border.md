# <a name="listboxborder"></a>LISTBOX. 框線

**框線** 屬性會指定或抓取值，指出清單方塊控制項是否有框線。

``` syntax
        elementID.border
```

## <a name="possible-values"></a>可能的值

這個屬性是 **布林** 值。



| 值 | 描述                                  |
|-------|----------------------------------------------|
| true  | 預設值。 清單方塊控制項有框線。  |
| false | 清單方塊控制項沒有框線。 |



 

## <a name="remarks"></a>備註

框線會顯示為清單方塊控制項周圍的雙圖元寬凹陷框線與系統色彩。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|---------------------------------------------------------|
| 版本<br/> | 適用于 Windows XP 或更新版本的 Windows Media Player<br/> |
