# <a name="playliststatustextcolor"></a>播放清單。 statusTextColor

**StatusTextColor** 屬性會指定或抓取值，指出狀態文字的色彩。

``` syntax
        elementID.statusTextColor
```

## <a name="possible-values"></a>可能的值

這個屬性是代表色彩的讀取/寫入 **字串** 。 預設值為 "none"。

| 需求 | 值 |
|--------------------|------------------------------------|
