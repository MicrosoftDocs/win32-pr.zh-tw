# <a name="iwmpstringcollection2-vb-and-c-interface"></a>IWMPStringCollection2 (VB 和 c # ) 介面

提供補充 **IWMPStringCollection** 介面的方法。

## <a name="members"></a>成員

**IWMPStringCollection2 (VB 和 c # )** 介面都有下列類型的成員：

-   [方法](#methods)

### <a name="methods"></a>方法

**IWMPStringCollection2 (VB 和 c # )** 介面都有這些方法。



| 方法                                                                                                                 | 描述                                                                                                                           |
|:-----------------------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------|
| [**getAttributeCountByType**](wmplibiwmpstringcollection2-iwmpstringcollection2-getattributecountbytype--vb-and-c.md) | 傳回與指定的字串集合專案索引、屬性名稱和語言相關聯的屬性數目。<br/> |
| [**getItemInfo**](wmplibiwmpstringcollection2-iwmpstringcollection2-getiteminfo--vb-and-c.md)                         | 傳回對應至指定之字串集合專案索引和名稱的字串。<br/>                                   |
| [**getItemInfoByType**](wmplibiwmpstringcollection2-iwmpstringcollection2-getiteminfobytype--vb-and-c.md)             | 傳回對應至指定之字串集合專案索引、名稱、語言和屬性索引的值。<br/>        |
| [**isIdentical**](wmplibiwmpstringcollection2-iwmpstringcollection2-isidentical--vb-and-c.md)                         | 傳回值，指出提供的物件是否與目前的物件相同。<br/>                                     |



 

藉由轉換 [**IWMPMediaCollection. getAttributeStringCollection**](/previous-versions/windows/desktop/api/wmp/nf-wmp-iwmpmediacollection-getattributestringcollection)方法所傳回的值來取得 **IWMPStringCollection2** 介面。
