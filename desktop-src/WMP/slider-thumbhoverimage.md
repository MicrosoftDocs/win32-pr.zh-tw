# <a name="sliderthumbhoverimage"></a>滑杆. thumbHoverImage

**ThumbHoverImage** 屬性會指定或抓取將滑鼠停留在其上方時所顯示捲動方塊的影像。

``` syntax
        elementID.thumbHoverImage
```

## <a name="possible-values"></a>可能的值

這個屬性是包含影像檔案名稱的讀取/寫入 **字串** 。

## <a name="remarks"></a>備註

**ThumbHoverImage** 是選擇性的。 如果未提供，滑杆控制項將不會在滑鼠停留在其上方時變更外觀。

支援的格式為 BMP、JPG、PNG 和 GIF (不包括動畫 Gif) 。
