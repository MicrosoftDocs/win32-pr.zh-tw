# <a name="pixelaspectratioy"></a>PixelAspectRatioY

**PixelAspectRatioY** 屬性是影片外觀比例的垂直部分。

## <a name="applies-to"></a>套用至

-   [影片專案](video-item-attributes.md)

## <a name="remarks"></a>備註

針對外觀比例為16:9 的影片， **PixelAspectRatioY** 會等於 "9"。

| 需求 | 值 |
|--------------------|------------------------------------|
| 版本<br/> | Windows Media Player 11<br/> |
