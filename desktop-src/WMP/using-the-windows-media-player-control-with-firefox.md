# <a name="using-the-windows-media-player-control-with-firefox"></a>搭配 Firefox 使用 Windows Media Player 控制項

Microsoft 提供一個外掛程式，可讓您將 Windows Media Player 的 ActiveX 控制項內嵌在網頁中，讓 Firefox 瀏覽器能夠正確地顯示該控制項。 您可以從 [Microsoft Port25 網站](https://www.mozilla.org/)下載外掛程式（必須安裝在用戶端電腦上）。

如需 Firefox 瀏覽器的詳細資訊，請參閱 [firefox 網站](https://www.mozilla.org/)。

如需有關搭配 Firefox 使用 Windows Media Player 的詳細資訊，請參閱 [mozillaZine 網站](http://kb.mozillazine.org/Windows_Media_Player)的 Windows Media Player 一節。

下列主題描述如何將 Windows Media Player 控制項內嵌在可由 Internet Explorer 和 Firefox 正確顯示的網頁中。



| 主題                                                                                                                                    | 描述                                                                                                           |
|------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [將播放機控制項內嵌在 Firefox 所顯示的網頁中](embedding-the-player-control-in-a-web-page-displayed-by-firefox.md)   | 描述如何撰寫腳本，以建立與 Internet Explorer 和 Firefox 相容的物件元素。 |
| [在 Firefox 所顯示的網頁中使用 PARAM 元素](using-param-elements-in-a-web-page-displayed-by-firefox.md)                   | 描述 Firefox 外掛程式所支援的 PARAM 元素。                                               |
| [在 Firefox 所顯示的網頁中使用腳本](using-script-in-a-web-page-displayed-by-firefox.md)                                   | 描述 Firefox 外掛程式所支援的 Windows Media Player 物件。                                 |
| [在 Firefox 所顯示的網頁中處理事件](handling-events-in-a-web-page-displayed-by-firefox.md)                             | 描述如何在由 Firefox 顯示的網頁中撰寫腳本，以處理 Windows Media Player 事件。           |
| [在 Firefox 所顯示的網頁中使用 invokeURLs 屬性](using-the-invokeurls-property-in-a-web-page-displayed-by-firefox.md) | 描述如何在 Firefox 所顯示的網頁中回應 URL 類型的指令碼命令。                               |
