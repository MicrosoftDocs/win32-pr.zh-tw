# <a name="viewontimer"></a>VIEW. ontimer

**Ontimer** 事件會處理計時器事件。

``` syntax
ontimer
```

## <a name="remarks"></a>備註

**TimerInterval** 屬性會指定將引發計時器事件的間隔。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
