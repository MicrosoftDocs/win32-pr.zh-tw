# <a name="customslider-element"></a>CUSTOMSLIDER 元素

**CUSTOMSLIDER** 元素提供一種方式來建立和操作任何形狀的滑杆控制項，例如圓形旋鈕。 下表列出此元素支援的屬性和事件處理常式。

**CUSTOMSLIDER** 元素支援下列屬性。



| 屬性                                               | 描述                                                                                                                 |
|---------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------|
| [cursor](customslider-cursor.md)                       | 指定或抓取當滑鼠停留在滑杆上時，所顯示之滑杆控制項游標的值。               |
| [disabledImage](customslider-disabledimage.md)         | 指定或抓取滑杆停用時使用的滑杆影像。                                            |
| [downImage](customslider-downimage.md)                 | 指定或抓取自訂滑杆的向下狀態影像。                                                           |
| [hoverImage](customslider-hoverimage.md)               | 指定或抓取當滑鼠停留在自訂滑杆上時所顯示的影像。                                    |
| [image](customslider-image.md)                         | 指定或抓取檔案的名稱，該檔案包含對應至自訂滑杆之各種狀態的影像。 |
| [max](customslider-max.md)                             | 指定或抓取自訂滑杆所定義範圍的最大值。                                         |
| [min](customslider-min.md)                             | 指定或抓取自訂滑杆所定義範圍的最小值。                                         |
| [positionImage](customslider-positionimage.md)         | 指定或抓取用來決定要顯示之 **圖像** 檔案位置影像的影像地圖。             |
| [提示](customslider-tooltip.md)                     | 指定或抓取滑杆的工具提示文字。                                                                     |
| [transparencyColor](customslider-transparencycolor.md) | 指定或抓取自訂滑杆影像的透明色彩。                                                  |
| [value](customslider-value.md)                         | 指定或抓取滑杆的目前位置。                                                                  |



 

**CUSTOMSLIDER** 元素可以執行下列事件處理常式。



| 事件處理常式                                         | Description                                                                                                          |
|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [onDragBegin](customslider-ondragbegin.md)           | 處理當使用者按一下並按住滑鼠左鍵並開始拖曳滑鼠時所發生的事件。 |
| [onDragEnd](customslider-ondragend.md)               | 處理在拖曳作業之後放開滑鼠左鍵時所發生的事件。                      |
| [onPositionChange](customslider-onpositionchange.md) | 處理當滑杆的位置由於使用者按一下或拖曳而變更時所發生的事件。   |



 

**CUSTOMSLIDER** 元素支援環境屬性，並可執行環境事件處理常式。 如需詳細資訊，請參閱 [環境屬性](ambient-attributes.md) 和 [環境事件處理常式](ambient-event-handlers.md)。
