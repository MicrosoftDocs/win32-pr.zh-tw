# <a name="playlistdropdowntooltip"></a>播放清單。 dropDownToolTip

**DropDownToolTip** 屬性會指定或抓取當使用者將滑鼠停留在 [**播放清單** 元素] 下拉式清單上時顯示的工具提示。

``` syntax
        elementID.dropDownToolTip
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **字串** ，其預設值為「顯示播放清單、音訊、影片或無線電站」。

## <a name="remarks"></a>備註

當這個屬性設定為 "" (空白字串) 時，不會顯示任何工具提示。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
