# <a name="iwmpclosedcaption2-vb-and-c-interface"></a>IWMPClosedCaption2 (VB 和 c # ) 介面

提供補充 **IWMPClosedCaption** 介面之隱藏式輔助字幕的屬性和方法。

除了繼承自 **IWMPClosedCaption** 的屬性之外， **IWMPClosedCaption2** 介面也會公開下列屬性。

## <a name="members"></a>成員

**IWMPClosedCaption2 (VB 和 c # )** 介面都有下列類型的成員：

-   [方法](#methods)
-   [屬性](#properties)

### <a name="methods"></a>方法

**IWMPClosedCaption2 (VB 和 c # )** 介面都有這些方法。



| 方法                                                                                             | 描述                                                                                       |
|:---------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------|
| [**getSAMILangID**](wmplibiwmpclosedcaption2-iwmpclosedcaption2-getsamilangid--vb-and-c.md)       | 傳回目前薩米文檔案所支援語言 (LCID) 的地區設定識別碼。<br/> |
| [**getSAMILangName**](wmplibiwmpclosedcaption2-iwmpclosedcaption2-getsamilangname--vb-and-c.md)   | 傳回目前的薩米文檔案所支援的語言名稱。<br/>                     |
| [**getSAMIStyleName**](wmplibiwmpclosedcaption2-iwmpclosedcaption2-getsamistylename--vb-and-c.md) | 傳回目前的薩米文檔案所支援的樣式名稱。<br/>                        |



 

### <a name="properties"></a>屬性

**IWMPClosedCaption2 (VB 和 c # )** 介面都有這些屬性。



| 屬性                                                                                                | 存取類型           | 描述                                                                         |
|:--------------------------------------------------------------------------------------------------------|:----------------------|:------------------------------------------------------------------------------------|
| [**SAMILangCount**](wmplibiwmpclosedcaption2-iwmpclosedcaption2-samilangcount--vb-and-c.md)<br/> | 讀取/寫入<br/> | 取得或設定目前的薩米文檔案所支援的語言數目。<br/> |
| [**SAMIStyleCount**](wmplibiwmpclosedcaption2-samistylecount.md)<br/>                            | 讀取/寫入<br/> | 取得或設定目前的薩米文檔案所支援的樣式數目。<br/>    |



 

藉由轉換 [**AxWindowsMediaPlayer closedCaption**](axwmplib-axwindowsmediaplayer-closedcaption--vb-and-c.md)屬性所傳回的值來取得 **IWMPClosedCaption2** 介面。
