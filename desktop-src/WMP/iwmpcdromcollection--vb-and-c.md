# <a name="iwmpcdromcollection-vb-and-c-interface"></a>IWMPCdromCollection (VB 和 c # ) 介面

提供一種方式來組織和存取一組 CD 或 DVD 光碟機。

**IWMPCdromCollection** 介面會公開下列屬性。

## <a name="members"></a>成員

**IWMPCdromCollection (VB 和 c # )** 介面都有下列類型的成員：

-   [方法](#methods)
-   [屬性](#properties)

### <a name="methods"></a>方法

**IWMPCdromCollection (VB 和 c # )** 介面都有這些方法。



| 方法                                                                                                     | 描述                                                                              |
|:-----------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------|
| [**getByDriveSpecifier**](wmplibiwmpcdromcollection-iwmpcdromcollection-getbydrivespecifier--vb-and-c.md) | 傳回與特定磁碟機號相關聯的 **IWMPCdrom** 介面。<br/> |
| [**項目**](wmplibiwmpcdromcollection-iwmpcdromcollection-item--vb-and-c.md)                               | 傳回指定索引處的 **IWMPCdrom** 介面。<br/>                        |



 

### <a name="properties"></a>屬性

**IWMPCdromCollection (VB 和 c # )** 介面都有這些屬性。



| 屬性                                                                                  | 存取類型          | Description                                                              |
|:------------------------------------------------------------------------------------------|:---------------------|:-------------------------------------------------------------------------|
| [**count**](wmplibiwmpcdromcollection-iwmpcdromcollection-count--vb-and-c.md)<br/> | 唯讀<br/> | 取得系統上的可用 CD 和 DVD 光碟機數目。<br/> |



 

使用下列屬性來取得 **IWMPCdromCollection** 介面。



| Object                                                                   | 屬性                                                                           |
|--------------------------------------------------------------------------|------------------------------------------------------------------------------------|
| [AxWindowsMediaPlayer 物件](axwindowsmediaplayer-object--vb-and-c.md) | [**cdromCollection**](axwmplib-axwindowsmediaplayer-cdromcollection--vb-and-c.md) |
