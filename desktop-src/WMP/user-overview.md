# <a name="user-overview"></a>使用者總覽

對使用者而言，視覺效果是 dance 時間到音樂的影像。 他們提供額外的娛樂體驗以增強音樂體驗。 視覺效果越有趣，更好！

一般視覺效果隨即出現，如下圖所示。

![一般視覺效果](images/vizanim.gif)

## <a name="standard-visualizations"></a>標準視覺效果

Windows Media Player 包含數個視覺效果，如下所示：

-   氛圍
-   Plenoptic
-   穗

## <a name="standard-presets"></a>標準預設

此外，每個視覺效果都有特定的預設值，可改變視覺效果的顯示方式。 例如，尖峰視覺效果共有兩個：

-   穗
-   阿米巴

這兩種效果都屬於相同的視覺效果，但有不同的預設值。 預設值是特定視覺效果的屬性值集合。

## <a name="skin-attributes"></a>外觀屬性

如果您建立 Windows Media Player 的外觀，您可以使用 **效果** 元素在您的面板中顯示標準視覺效果。 如需在 [外觀] 中使用視覺效果的詳細資訊，請參閱 [Windows Media Player 的外觀](windows-media-player-skins.md)。
