# <a name="windows-media-player"></a>Windows Media Player

Windows Media Player SDK 提供的功能會影響 Windows Mobile Windows Media Player 和 Windows Media Player 的行為。

下列各節詳細說明一般適用于 Windows Media Player 的 SDK 功能。



| 區段                                                                                                        | 描述                                                                                                                                                           |
|----------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [登錄設定](registry-settings.md)                                                                     | 詳細說明您可以在使用者的登錄中變更的值，讓 Windows Media Player 辨識自訂副檔名。                                    |
| [命令列參數](command-line-parameters.md)                                                         | 詳細說明命令列參數的集合，這些參數會指定在啟動時 Windows Media Player 的行為。                                                              |
| [音訊輸出](audio-outputs.md)                                                                             | 描述 Windows Media Player 和 Windows Media Player ActiveX 控制項如何選擇預設音訊輸出裝置。                                                  |
| [轉散發 Windows Media Player 軟體](redistributing-windows-media-player-software.md)               | 提供轉散發 Windows Media Player 的相關資訊。                                                                                                       |
| [發佈 Windows Media Player 的編解碼器](publishing-codecs-for-windows-media-player.md)                   | 不再支援在 WMPlugins 網站上發佈編解碼器。                                                                                                |
| [調整授權取得對話方塊的大小](resizing-the-license-acquisition-dialog-box.md)                 | 說明如何修改 Windows Media 檔案屬性，以指定 Windows Media Player 10 或更新版本的授權取得對話方塊的大小。                     |
| [加速中繼資料傳輸的裝置擴充功能](device-extensions-for-accelerated-metadata-transfer.md) | 描述 Windows Media Player 如何從可攜式裝置抓取有關同步處理會話之間特定內容專案變更的資訊。 |
| [用於報告取得內容的裝置擴充功能](device-extensions-for-reporting-acquired-content.md)       | 描述 Windows Media Player 可如何在同步處理會話之間取得可攜式裝置所取得的新內容清單。                                 |
| [播放清單物件喜好設定的裝置擴充功能](device-extensions-for-playlist-object-preferences.md)     | 描述可攜式裝置如何指定 Windows Media Player 必須在自動同步處理期間複製到裝置的播放清單物件類型。                |
| [裝置 i/o 控制碼](device-i-o-control-codes.md)                                                       | 列出 Windows Media Player 10 或更新版本用來透過 Windows Media 裝置管理員 SDK 與可攜式裝置通訊的裝置 i/o 控制碼。            |
| [裝置的自訂映射支援](custom-image-support-for-devices.md)                                       | 描述可攜裝置製造商建立的兩個影像檔案，以自訂 Windows Media Player 10 或更新版本中的品牌。                                    |
