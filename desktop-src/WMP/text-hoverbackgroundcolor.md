# <a name="texthoverbackgroundcolor"></a>HoverBackgroundColor

當滑鼠游標停留在文字控制項上方時， **hoverBackgroundColor** 屬性會指定或抓取用於文字控制項的背景色彩。

``` syntax
        elementID.hoverBackgroundColor
```

## <a name="possible-values"></a>可能的值

這個屬性是包含任何 Microsoft Internet Explorer 色彩值的讀取/寫入 **字串** 。

## <a name="remarks"></a>備註

如果未指定 **hoverBackgroundColor** ，則會使用 **backgroundColor** 。

如需說明如何使用 **TEXT** 元素屬性的範例，請參閱 [value](text-value.md)屬性。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
