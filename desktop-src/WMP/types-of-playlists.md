# <a name="types-of-playlists"></a>播放清單的類型

Windows媒體中繼檔播放清單，只要在本檔中稱為播放清單，就會有地板蠟、. 300k.wvx 或 .asx 的副檔名。 使用這些副檔名的播放清單會分別用來存取副檔名為 .wma、.wmv 和 .asf 的 Windows 媒體檔案。
