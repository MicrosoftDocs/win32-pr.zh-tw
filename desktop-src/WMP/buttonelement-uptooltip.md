# <a name="buttonelementuptooltip"></a>BUTTONELEMENT.upToolTip

**UpToolTip** 屬性會指定或抓取滑鼠移到 **BUTTONELEMENT** 上且 **BUTTONELEMENT** 處於 Up 狀態時所顯示的工具提示文字。

``` syntax
        elementID.upToolTip
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **字串** ，其預設值為 "" (空白字串) ，最大長度為1024個字元。

## <a name="remarks"></a>備註

當這個屬性設定為 "" (空白字串) 時，不會顯示任何工具提示。

## <a name="examples"></a>範例

如需說明如何使用此屬性的範例，請參閱 [mappingColor](buttonelement-mappingcolor.md) 屬性。
