# <a name="sliderdisabledcolor"></a>滑杆. disabledColor

**DisabledColor** 屬性會指定或抓取滑杆控制項停用的色彩。

``` syntax
        elementID.disabledColor
```

## <a name="possible-values"></a>可能的值

這個屬性是包含任何 Microsoft Internet Explorer 色彩值的讀取/寫入 **字串** 。 它沒有預設值。

## <a name="remarks"></a>備註

使用前景和背景色彩指定滑杆控制項時，停用的色彩會在 [ **已啟用** ] 設定為 [false] 時指定控制項的色彩。
