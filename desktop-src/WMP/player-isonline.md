# <a name="playerisonline"></a>IsOnline

**IsOnline** 屬性會抓取值，指出使用者是否已連線到網路。

## <a name="syntax"></a>Syntax

*玩家* 。**isOnline**

## <a name="possible-values"></a>可能的值

這個屬性是唯讀的 **布林值**。



| 值 | 描述                             |
|-------|-----------------------------------------|
| true  | 使用者已連線到網路。     |
| false | 使用者未連線到網路。 |



 

## <a name="remarks"></a>備註

**Windows Media Player 10** 行動裝置版：這個屬性一律會傳回 true。
