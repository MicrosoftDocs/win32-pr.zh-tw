# <a name="windows-media-player-plug-ins"></a>Windows Media Player 外掛程式

Microsoft Windows Media Player 公開介面，可讓您以數種方式擴充功能。 下列各節說明支援的外掛程式架構，以及建立外掛程式的程式：



| 區段                                                                                                         | 描述                                                                                                                                                                                                                                                                                             |
|-----------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [建立外掛程式](building-a-plug-in.md)                                                                    | 您可以使用 Visual Studio 和 Windows Media Player 外掛程式嚮導來建立數種類型的外掛程式。                                                                                                                                                                                             |
| [Windows Media Player 自訂視覺效果](windows-media-player-custom-visualizations.md)                    | Windows Media Player 的視覺效果是元件物件模型 (COM) 物件，可用來顯示與播放程式媒體播放播放音訊部分同步的視覺效果影像。 您可以使用 Microsoft Visual C++ 來建立自訂視覺效果。                                             |
| [Windows Media Player 消費者介面外掛程式](windows-media-player-user-interface-plug-ins.md)                | Windows Media Player 的使用者介面外掛程式會將新功能新增至完整模式播放程式的 [ **立即播放** ] 窗格。 您可以建立使用視覺效果區域、個別視窗、設定區域、中繼資料區域或背景外掛程式的外掛程式，以公開沒有可見的使用者介面。 |
| [Windows Media Player 的 DSP 外掛程式](windows-media-player-dsp-plug-ins.md)                                      | Windows Media Player 的 DSP 外掛程式會在播放程式呈現音訊和影片之前，修改或處理音訊和影片資料。 DSP 外掛程式是透過 COM 介面連接到播放程式的 DirectX 媒體物件 (SQL-DMO) 。                                                                                           |
| [Windows Media Player 轉譯外掛程式 (已淘汰) ](/previous-versions/windows/desktop/legacy/dd564683(v=vs.85)) | 如有必要，Windows Media Player 轉譯外掛程式解碼 () 和轉譯 Windows Media 格式資料流程中包含的自訂資料。 轉譯外掛程式是透過 COM 介面連接到播放程式的 DirectX 媒體物件 (SQL-DMO) 。                                                                  |
| [Windows Media Player 轉換外掛程式](windows-media-player-conversion-plug-ins.md)                        | Windows Media Player 轉換外掛程式會將使用 Microsoft 未提供的技術所建立的數位媒體檔案轉換成 Advanced Systems 格式 (ASF) 。                                                                                                                                 |
