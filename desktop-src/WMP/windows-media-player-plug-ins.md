# <a name="windows-media-player-plug-ins"></a>Windows Media Player外掛程式

Microsoft Windows Media Player 公開介面，可讓您以數種方式擴充功能。 下列各節說明支援的外掛程式架構，以及建立外掛程式的程式：



| 區段                                                                                                         | 描述                                                                                                                                                                                                                                                                                             |
|-----------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [建立外掛程式](building-a-plug-in.md)                                                                    | 您可以使用 Visual Studio 和 Windows Media Player 外掛程式嚮導來建立數種類型的外掛程式。                                                                                                                                                                                             |
| [Windows Media Player自訂視覺效果](windows-media-player-custom-visualizations.md)                    | Windows Media Player 的視覺效果是元件物件模型 (COM) 物件，可用來顯示與播放程式媒體播放播放音訊部分同步的視覺效果影像。 您可以使用 Microsoft Visual C++ 來建立自訂視覺效果。                                             |
| [Windows Media Player消費者介面外掛程式](windows-media-player-user-interface-plug-ins.md)                | Windows Media Player 的使用者介面外掛程式會將新功能新增至完整模式播放程式的 [**立即播放**] 窗格。 您可以建立使用視覺效果區域、個別視窗、設定區域、中繼資料區域或背景外掛程式的外掛程式，以公開沒有可見的使用者介面。 |
| [Windows Media PlayerDSP 外掛程式](windows-media-player-dsp-plug-ins.md)                                      | Windows Media Player在播放程式轉譯音訊和影片之前，DSP 外掛程式會修改或處理音訊和影片資料。 DSP 外掛程式是 (透過 COM 介面連接到播放程式 DMO) 的 DirectX 媒體物件。                                                                                           |
| [Windows Media Player (已淘汰的轉譯外掛程式) ](/previous-versions/windows/desktop/legacy/dd564683(v=vs.85)) | 如有必要，Windows Media Player 轉譯外掛程式解碼 () 和轉譯 Windows 媒體格式資料流程中包含的自訂資料。 轉譯外掛程式是 (透過 COM 介面連接到播放程式 DMO) 的 DirectX 媒體物件。                                                                  |
| [Windows Media Player轉換外掛程式](windows-media-player-conversion-plug-ins.md)                        | Windows Media Player 轉換外掛程式會將使用 Microsoft 未提供的技術所建立的數位媒體檔案轉換成 Advanced Systems 格式 (ASF) 。                                                                                                                                 |
