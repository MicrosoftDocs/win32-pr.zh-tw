# <a name="using-the-echo-sample-dsp-plug-in"></a>使用 Echo 範例 DSP 外掛程式

當您建立 Windows Media Player echo DSP 外掛程式的無錯誤組建時，您可以在 Windows Media Player 中使用它來聽取效果。 當您啟動 Windows Media Player 時，預設應該啟用外掛程式。 如果您使用音訊播放音訊或影片，您應該會聽到音效中的一秒回應，在與原始音效相同的音量上播放。

您可以藉由變更 [echo 外掛程式] 屬性頁中的值，來試驗不同的延遲時間和效果層級。 嘗試對不同類型的來來源資料（例如音樂或語音）產生效果。 效果層級為30的延遲時間為200毫秒，將會產生緊密間距的 echo，這聽起來像是在早期搖滾和推出記錄上所使用的效果。 效果層級大於50時，會導致磁片區中的原始信號較小，而不是回應，這稱為回應前效果。
