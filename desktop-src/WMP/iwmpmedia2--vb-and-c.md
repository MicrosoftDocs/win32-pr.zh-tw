# <a name="iwmpmedia2-vb-and-c-interface"></a>IWMPMedia2 (VB 和 c # ) 介面

提供其他媒體專案屬性的存取權。

除了繼承自 **IWMPMedia** 的屬性和方法之外， **IWMPMedia2** 介面也會公開下列屬性。



| 屬性                                                 | 描述                                                                   |
|----------------------------------------------------------|-------------------------------------------------------------------------------|
| [錯誤](wmplibiwmpmedia2-iwmpmedia2-error--vb-and-c.md) | 取得 **IWMPErrorItem** 介面（如果媒體專案有錯誤狀況）。 |



 

藉由轉換下列其中一個屬性所傳回的值，或透過下列物件或介面所存取的方法，來取得 **IWMPMedia2** 介面。



| 物件或介面                                               | 屬性或方法                                                                                                                |
|-------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| [IWMPControls](iwmpcontrols--vb-and-c.md)                        | [currentItem](wmplibiwmpcontrols-iwmpcontrols-currentitem--vb-and-c.md)                                                          |
| [AxWindowsMediaPlayer](axwindowsmediaplayer-object--vb-and-c.md) | [currentMedia](axwmplib-axwindowsmediaplayer-currentmedia--vb-and-c.md) 、 [newMedia](axwmplib-axwindowsmediaplayer-newmedia.md) |
| [IWMPPlaylist](iwmpplaylist--vb-and-c.md)                        | 在 c # 中 **取得 \_ 專案** ([專案](iwmpplaylist-item--vb-and-c.md))                                                                    |



 

## <a name="members"></a>成員

**IWMPMedia2 (VB 和 c # )** 介面不會定義任何成員。
