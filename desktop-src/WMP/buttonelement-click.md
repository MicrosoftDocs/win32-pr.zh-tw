# <a name="buttonelementclick"></a>BUTTONELEMENT。按一下

**Click** 方法會呼叫針對 **BUTTONELEMENT** 所定義的 **onclick** 事件處理常式。

``` syntax
        elementID.click()
```

## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。

## <a name="remarks"></a>備註

您可以使用這個方法，提供執行與按鈕相關聯之程式碼的替代方法。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|---------------------------------------------------|
| 版本<br/> | Windows Media Player 9 系列或更新版本<br/> |
