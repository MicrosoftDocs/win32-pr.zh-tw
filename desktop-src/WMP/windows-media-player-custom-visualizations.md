# <a name="windows-media-player-custom-visualizations"></a>Windows Media Player 自訂視覺效果

Microsoft Windows Media Player 提供視覺效果，可讓使用者查看在播放時，會同步處理至媒體內容音效的視覺效果影像。 Windows Media Player 包含數個標準的視覺效果，包括尖峰、橫條和 DotPlane。 SDK 的這一節提供程式設計資訊，可讓您建立自己的視覺效果。

自訂視覺效果檔分為三個部分：



| 區段                                                                                      | 描述                                                                                                                                                          |
|----------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [關於自訂視覺效果](about-custom-visualizations.md)                               | 以抽象術語討論自訂視覺效果的架構。 您應該閱讀本節，以瞭解視覺效果與 Windows Media Player 的關聯性。 |
| [自訂視覺效果程式設計指南](custom-visualization-programming-guide.md)         | 說明建立自訂視覺效果所需的動作。 本章節包含實用的提示和秘訣，可協助您建立有趣的視覺效果。     |
| [自訂視覺效果程式設計參考](custom-visualization-programming-reference.md) | 提供您必須執行之介面的參考，以及其他技術參考資訊。                                                          |
