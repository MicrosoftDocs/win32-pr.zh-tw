# <a name="customslidermin"></a>CUSTOMSLIDER，分鐘

**Min** 屬性會指定或抓取自訂滑杆所定義範圍的最小值。

``` syntax
        elementID.min
```

## <a name="possible-values"></a>可能的值

這個屬性是 (**float**) 的讀/寫 **數位**，預設值為零。

## <a name="remarks"></a>備註

[ **最小** 值] 的值必須小於 [ **最大** 值]。

## <a name="examples"></a>範例

如需說明如何使用 **CUSTOMSLIDER** 元素之屬性的範例，請參閱 [positionImage](customslider-positionimage.md)屬性。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
