# <a name="listboxreadonly"></a>LISTBOX. readOnly

**ReadOnly** 屬性會指定或抓取值，指出文字為唯讀或可由使用者選取。 只能在設計階段設定。

``` syntax
        elementID.readOnly
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **布林值**。



| 值 | 描述                                |
|-------|--------------------------------------------|
| true  | 文字是唯讀的。                         |
| false | 預設值。 使用者可以選取文字。 |



 

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|---------------------------------------------------------|
| 版本<br/> | Windows XP 或更新版本的 Windows Media Player<br/> |
