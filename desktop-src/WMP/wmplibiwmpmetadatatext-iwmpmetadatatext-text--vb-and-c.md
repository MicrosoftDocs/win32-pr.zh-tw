# <a name="iwmpmetadatatexttext-property"></a>IWMPMetadataText：： text 屬性

`text`屬性會取得中繼資料文字。

## <a name="syntax"></a>Syntax


```CSharp
public System.String text {get; set;}
```


```VB

Public ReadOnly Property text As System.String
```





## <a name="property-value"></a>屬性值

表示中繼資料文字的 **system.string** 。

## <a name="remarks"></a>備註

使用這個屬性之前，您必須擁有程式庫的讀取權限。 如需詳細資訊，請參閱連結 [庫存取](library-access.md)。
