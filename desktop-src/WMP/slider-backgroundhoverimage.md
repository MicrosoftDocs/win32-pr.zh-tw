# <a name="sliderbackgroundhoverimage"></a>滑杆. backgroundHoverImage

**BackgroundHoverImage** 屬性會指定或抓取游標停留在滑鼠上方時所顯示之滑杆的背景影像。

``` syntax
        elementID.backgroundHoverImage
```

## <a name="possible-values"></a>可能的值

這個屬性是包含影像檔案名稱的讀取/寫入 **字串** 。

## <a name="remarks"></a>備註

此屬性是選擇性的。 如果未指定，則會使用 **backgroundImage** 。

支援的格式為 BMP、JPG、PNG 和 GIF (不包括動畫 Gif) 。
