# <a name="background-file"></a>背景檔

每個面板都需要一個背景檔案，其中包含第一次顯示面板時，使用者會看到的影像。 這個檔案會定義為面板定義檔中的背景檔案。

下圖是一般的背景檔。

![背景檔](images/cesdkbak.png)

這是使用者一開始會看到的背景影像。 不過，它們也會在 trackbars 上的適當位置看到兩個 thumb 影像，以搜尋 (上述) 和磁片區 () 。 此外，已停用的任何按鈕都會改為使用其停用的影像來顯示。
