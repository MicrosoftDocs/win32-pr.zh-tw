# <a name="button-location"></a>按鈕位置

您必須定義每個按鈕影像的位置。 若要這樣做，您可以提供四個以逗號分隔的正整數。 這些數位定義了左上角的座標、按鈕的寬度和高度（以圖元為單位）。

如果您使用區域按鈕 (PushHit、ToggleHit 和 2PushHit) ，您仍然必須輸入這四個數字。 這些值是用來判斷次要影像的位置，例如推送和停用。

> [!Note]  
> 在 Windows Media Player 10 行動裝置版或更新版本的面板中，按鈕類型已被取代。
