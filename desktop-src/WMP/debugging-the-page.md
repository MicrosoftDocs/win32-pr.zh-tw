# <a name="debugging-the-page"></a>頁面的調試

當 Windows Media Player 裝載您的線上商店網頁時，它會停用從網頁開啟對話方塊的功能。 若要對您的網頁程式碼進行偵錯工具，您應該改為加入可暫時將輸出列印到網頁上 HTML 專案的程式碼，例如 DIV 或 TEXT 元素。
