# <a name="managing-playlists"></a>管理播放清單

**播放清單** 物件代表一組媒體專案。 您可以使用物件的屬性和方法來操作其包含的媒體專案，以及取得播放清單的相關中繼資料。 若要管理播放清單本身，請使用 **PlaylistCollection** 物件和 **MediaCollection** 物件。

下列主題描述如何使用 Windows Media Player 控制項物件模型來管理播放清單：

-   [播放清單屬性](playlist-attributes.md)
-   [播放清單和媒體專案](playlists-and-media-items.md)
-   [播放清單和 PlaylistCollection 物件](playlists-and-the-playlistcollection-object.md)
-   [靜態和自動播放清單](static-and-auto-playlists.md)
-   [播放清單和 MediaCollection 物件](playlists-and-the-mediacollection-object.md)
