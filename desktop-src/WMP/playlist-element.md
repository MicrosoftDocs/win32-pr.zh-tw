# <a name="playlist-element"></a>播放清單元素

**播放** 清單元素提供一種方式，可使用下列屬性和方法，在清單中組織媒體專案以方便操作。 為了方便起見，也提供預先定義的 **播放清單** 元素。 您可以將資料 **行** 元素包含為 **播放清單** 元素的子系，以指定播放清單的自訂資料行。

**播放清單** 元素支援下列屬性。



| 屬性                                                                                 | 描述                                                                                                                                |
|-------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------|
| [allowColumnSorting](playlist-allowcolumnsorting.md)                                     | 指定或抓取值，指出是否允許在資料行標頭上排序。                                                    |
| [allowItemEditing](playlist-allowitemediting.md)                                         | 指定或抓取值，指出播放清單中的專案是否支援就地編輯。                                            |
| [backgroundColor](playlist-backgroundcolor.md)                                           | 指定或抓取背景色彩。                                                                                               |
| [backgroundImage](playlist-backgroundimage.md)                                           | 指定或抓取背景影像。                                                                                               |
| [checkboxesVisible](playlist-checkboxesvisible.md)                                       | 指定或抓取值，指出是否會顯示覆選框。                                                                  |
| [columnCount](playlist-columncount.md)                                                   | 抓取顯示的資料行數目。                                                                                                     |
| [columnOrder](playlist-columnorder.md)                                                   | 指定或抓取播放清單資料行的順序。                                                                                  |
| [列](playlist-columns.md)                                                           | 定義出現在 **播放清單** 元素中的資料行。                                                                               |
| [columnsVisible](playlist-columnsvisible.md)                                             | 指定或抓取值，指出是否顯示資料行。                                                                       |
| [複製](playlist-copying.md)                                                           | 抓取值，指出 **播放清單** 元素是否在複製的動作中。                                                    |
| [disabledItemColor](playlist-disableditemcolor.md)                                       | 指定或抓取離線時已停用的 CD 播放軌或線上內容的色彩。                                                 |
| [dropDownBackgroundImage](playlist-dropdownbackgroundimage.md)                           | 指定或抓取顯示在下拉式清單背景中之影像的名稱。                                        |
| [dropDownImage](playlist-dropdownimage.md)                                               | 指定或抓取下拉式清單按鈕（顯示在下拉式清單的右邊緣）所使用之影像的名稱。 |
| [dropDownList](playlist-dropdownlist.md)                                                 | 指定或抓取值，指出哪些元素會顯示在 **播放** 清單元素之指定實例的下拉式清單中。   |
| [dropDownToolTip](playlist-dropdowntooltip.md)                                           | 指定或抓取當使用者將滑鼠停留在 [ **播放清單** 專案] 下拉式功能表上時，所顯示的工具提示。                                |
| [dropDownVisible](playlist-dropdownvisible.md)                                           | 指定或抓取值，指出 **播放清單** 元素下拉式清單選取器是否可見。                                  |
| [editButtonVisible](playlist-editbuttonvisible.md)                                       | 指定或抓取值，指出 **播放清單** 元素編輯按鈕是否可見。                                         |
| [foregroundColor](playlist-foregroundcolor.md)                                           | 指定或抓取前景色彩。                                                                                               |
| [hueShift](playlist-hueshift.md)                                                         | 指定或抓取下拉式影像之色調的位移量。                                                     |
| [itemCount](playlist-itemcount.md)                                                       | 抓取 **播放清單** 元素目前所顯示的專案數。                                                             |
| [itemErrorColor](playlist-itemerrorcolor.md)                                             | 指定或抓取表示有錯誤條件之播放清單專案的反白顯示色彩。                                     |
| [itemMedia](playlist-itemmedia.md)                                                       | 抓取與 **播放清單** 元素中的指定索引對應的 **媒體** 物件。                                               |
| [itemPlayingBackgroundColor](playlist-itemplayingbackgroundcolor.md)                     | 指定或抓取目前播放播放清單專案的背景色彩。                                                        |
| [itemPlayingColor](playlist-itemplayingcolor.md)                                         | 指定或抓取反白顯示色彩，指出播放清單中目前現正播放的專案。                                      |
| [itemPlaylist](playlist-itemplaylist.md)                                                 | 抓取 **播放清單** 元素中指定索引處顯示之媒體專案的播放清單。                                |
| [itemSelectedBackgroundColor](playlist-itemselectedbackgroundcolor.md)                   | 指定或抓取值，指出播放清單中所選取專案的背景色彩。                                         |
| [itemSelectedBackgroundFocusLostColor](playlist-itemselectedbackgroundfocuslostcolor.md) | 指定或抓取值，指出播放清單中所選取專案的文字色彩。                                               |
| [itemSelectedColor](playlist-itemselectedcolor.md)                                       | 指定或抓取值，指出播放清單中所選取專案的文字色彩。                                               |
| [itemSelectedFocusLostColor](playlist-itemselectedfocuslostcolor.md)                     | 指定或抓取值，指出播放清單失去焦點時，播放清單中所選取專案的文字色彩。                 |
| [leftStatus](playlist-leftstatus.md)                                                     | 指定或抓取 **播放清單** 元素左邊和底部顯示的狀態文字。                          |
| [播放清單](playlist-playlist.md)                                                         | 指定或抓取播放 **清單** 元素提供介面的 **播放清單** 物件。                                    |
| [playlistItemsVisible](playlist-playlistitemsvisible.md)                                 | 指定或抓取值，指出播放清單中的專案是否可見。                                                       |
| [rightStatus](playlist-rightstatus.md)                                                   | 指定或抓取 **播放清單** 元素右側和底部顯示的狀態文字。                         |
| [飽和](playlist-saturation.md)                                                     | 指定或抓取下拉式影像的飽和度值。                                                                       |
| [statusColor](playlist-statuscolor.md)                                                   | 指定或抓取 **播放清單** 元素中的狀態行色彩。                                                           |
| [statusTextColor](playlist-statustextcolor.md)                                           | 指定或抓取值，指出狀態文字的色彩。                                                                    |
| [toolbarVisible](playlist-toolbarvisible.md)                                             | 指定或抓取值，指出是否顯示播放清單工具列。                                                           |



 

**播放清單** 元素支援下列方法。



| 方法                                                              | 描述                                                                                                               |
|---------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------|
| [abortCopy](playlist-abortcopy.md)                                 | 取消複製操作。                                                                                                 |
| [addSelectedToPlaylist](playlist-addselectedtoplaylist.md)         | 將選取的專案加入至播放清單。                                                                                   |
| [copy](playlist-copy.md)                                           | 從 CD 開始複製操作。                                                                                      |
| [deleteSelected](playlist-deleteselected.md)                       | 從播放清單中刪除選取的專案。                                                                              |
| [deleteSelectedFromLibrary](playlist-deleteselectedfromlibrary.md) | 從播放清單和媒體櫃中刪除選取的專案。                                                         |
| [getNextCheckedItem](playlist-getnextcheckeditem.md)               | 以指定的索引之後，抓取播放清單中下一個核取專案的索引。                               |
| [getNextCheckedItem2](playlist-getnextcheckeditem2.md)             | 以指定的索引之後，抓取播放清單中下一個核取專案的索引。 適用于嵌套的播放清單。  |
| [getNextSelectedItem](playlist-getnextselecteditem.md)             | 以指定的索引之後，抓取播放清單中下一個選取專案的索引。                              |
| [getNextSelectedItem2](playlist-getnextselecteditem2.md)           | 以指定的索引之後，抓取播放清單中下一個選取專案的索引。 適用于嵌套的播放清單。 |
| [moveSelectedDown](playlist-moveselecteddown.md)                   | 將選取的專案向下移動清單中的一個位置。                                                                    |
| [moveSelectedUp](playlist-moveselectedup.md)                       | 將選取的專案在清單中向上移動一個位置。                                                                      |
| [setCheckedState](playlist-setcheckedstate.md)                     | 指定檢查播放清單中的索引項目目。                                                               |
| [setCheckedState2](playlist-setcheckedstate2.md)                   | 使用 **播放清單** 元素中的指定索引，設定專案的已核取狀態。 適用于嵌套的播放清單。     |
| [setColumnResizeMode](playlist-setcolumnresizemode.md)             | 指定索引資料行本身的大小。                                                                            |
| [setColumnWidth](playlist-setcolumnwidth.md)                       | 指定資料行寬度，並將資料行的調整大小模式變更為 "wmpcrmFixed"。                                    |
| [setSelectedState](playlist-setselectedstate.md)                   | 指定已選取播放清單中的索引項目目。                                                              |
| [setSelectedState2](playlist-setselectedstate2.md)                 | 使用 **播放清單** 元素中的指定索引，設定專案的選取狀態。 適用于嵌套的播放清單。    |
| [sortColumn](playlist-sortcolumn.md)                               | 排序指定之資料行中的資料。                                                                                   |



 

**播放清單** 元素支援環境屬性，而且可以執行環境事件處理常式（除非有注明）。 如需詳細資訊，請參閱 [環境屬性](ambient-attributes.md) 和 [環境事件處理常式](ambient-event-handlers.md)。

預先定義的播放清單是一般的 **播放清單** 元素，預設會指定各種不同的通用屬性設定。 您可以使用下列預先定義的播放清單。



| 預先定義的播放清單                      | 描述                                                       |
|------------------------------------------|-------------------------------------------------------------------|
| [DROPDOWNPLAYLIST](dropdownplaylist.md) | 沒有專案可見的下拉式 **清單** 。                   |
| [ITEMSPLAYLIST](itemsplaylist.md)       | 未顯示任何專案或資料行標題的下拉式 **清單** 。 |
