<span data-ttu-id="3bf67-119">上述範例會使用代表第一個磁片磁碟機 DVD 媒體的第一個專案來抓取 **播放清單** 物件。</span><span class="sxs-lookup"><span data-stu-id="3bf67-119">The preceding example retrieves a **Playlist** object with a first entry representing the DVD media in the first drive.</span></span> <span data-ttu-id="3bf67-120">其他專案代表個別的 DVD 標題。</span><span class="sxs-lookup"><span data-stu-id="3bf67-120">Additional entries represent individual DVD titles.</span></span> <span data-ttu-id="3bf67-121">因此，可用的標題數目可以計算如下：</span><span class="sxs-lookup"><span data-stu-id="3bf67-121">Therefore, the number of titles available can be calculated as follows:</span></span>
