# <a name="mutebutton"></a>MUTEBUTTON

這是預先定義的按鈕，具有下列預設值。

``` syntax
onclick="jscript:player.settings.mute=down;"
upToolTip="Mute"
downToolTip="Sound"
down="wmpprop:player.settings.mute"
sticky="true"
```

## <a name="remarks"></a>備註

這會建立 **按鈕** 控制項，將音訊靜音並取消靜音。 工具提示已當地語系化。 此 **按鈕** 的所有屬性都可以藉由明確指定來覆寫。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|----------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 或更新版本<br/> |
