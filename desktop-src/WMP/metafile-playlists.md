# <a name="metafile-playlists"></a>中繼檔播放清單

Windows Media 中繼檔播放清單是用來組織串流內容和簡報。 播放清單可以直接將 Windows Media Player 重新導向至要串流處理的內容，但更常見的方式是提供要依序播放的檔案清單。 更複雜的播放清單可能包含重複、商業插入，以及清單中專案的相關資訊。

播放清單的另一個優點是，不是播放資料流程、停止、啟動下一個資料流程，然後等候它完成緩衝處理，Windows Media Services 和 Windows Media Player 會一起運作，以最短的緩衝處理時間或中斷來播放剪輯。
