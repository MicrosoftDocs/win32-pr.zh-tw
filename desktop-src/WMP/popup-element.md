# <a name="popup-element"></a>POPUP 元素

**POPUP** 元素提供一個方法，讓使用者從清單中選取專案。 您可以使用 **ITEM 專案** 作為 **POPUP** 元素的子系來指定這些專案。 **Popup** 元素與 **LISTBOX** 元素相同，但 **popup** 屬性的預設值為 true。 當 **popUp** 設定為 true 時，只有在呼叫 **show** 方法時才會顯示控制項，而且其寬度設定為內容的寬度 (環境 **寬度** 屬性會忽略) 。 當使用者從清單中選取一個專案，或在呼叫 **解除** 方法時，就會自動隱藏控制項。

> [!Note]  
> 此元素需要 Windows XP 或更新版本的 Windows Media Player。
