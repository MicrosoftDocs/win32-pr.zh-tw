# <a name="returnbutton"></a>RETURNBUTTON

這是預先定義的 **按鈕** ，具有下列預設值。

``` syntax
onclick="jscript:view.returnToMediaCenter();"
upToolTip="Return to Full Mode"
```

## <a name="remarks"></a>備註

這會建立可將播放程式傳回 media center 的 **按鈕** 控制項。 工具提示已當地語系化。 此 **按鈕** 的所有屬性都可以藉由明確指定來覆寫。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|----------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 或更新版本<br/> |
