# <a name="iwmplibrarymediacollection-property"></a>IWMPLibrary：： mediaCollection 屬性

**MediaCollection** 屬性會取得目前程式庫的 **IWMPMediaCollection** 介面。

## <a name="syntax"></a>Syntax


```CSharp
public IWMPMediaCollection mediaCollection {get; set;}
```


```VB

Public ReadOnly Property mediaCollection As IWMPMediaCollection
```





## <a name="property-value"></a>屬性值

目前程式庫的 WMPLib IWMPMediaCollection 介面。
