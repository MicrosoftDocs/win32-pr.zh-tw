# <a name="formattag-attribute"></a>FormatTag 屬性

**FormatTag** 屬性是音訊編解碼器的格式標記。

## <a name="applies-to"></a>套用至

-   [音訊專案](audio-item-attributes.md)
-   [影片專案](video-item-attributes.md)

## <a name="remarks"></a>備註

值是十進位格式標記值的字串表示。 例如，針對 Windows Media 音訊9編解碼器， **FormatTag** 屬性會包含字串 "354"。

| 需求 | 值 |
|--------------------|------------------------------------|
| 版本<br/> | Windows Media Player 11<br/> |
