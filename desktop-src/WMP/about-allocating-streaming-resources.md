<span data-ttu-id="06dd3-109">Windows Media Player 外掛程式 Wizard 產生的範例 DSP 外掛程式不需要任何額外的串流緩衝區。</span><span class="sxs-lookup"><span data-stu-id="06dd3-109">The sample DSP plug-in generated by the Windows Media Player Plug-in Wizard does not require any additional streaming buffers.</span></span> <span data-ttu-id="06dd3-110">不過，您可能會想要為您的 DSP 外掛程式配置記憶體資源。</span><span class="sxs-lookup"><span data-stu-id="06dd3-110">However, you might want to allocate memory resources for your DSP plug-in.</span></span> <span data-ttu-id="06dd3-111">例如，產生 echo 效果的外掛程式需要次要緩衝區來建立必要的時間延遲。</span><span class="sxs-lookup"><span data-stu-id="06dd3-111">For example, a plug-in that produces an echo effect would require a secondary buffer to create the necessary time delay.</span></span>
