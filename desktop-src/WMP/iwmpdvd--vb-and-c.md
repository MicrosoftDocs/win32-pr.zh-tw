# <a name="iwmpdvd-vb-and-c-interface"></a>IWMPDVD (VB 和 c # ) 介面

**IWMPDVD** 介面提供使用 dvd 的屬性和方法。

**IWMPDVD** 介面會公開下列屬性。

## <a name="members"></a>成員

**IWMPDVD (VB 和 c # )** 介面都有下列類型的成員：

-   [方法](#methods)
-   [屬性](#properties)

### <a name="methods"></a>方法

**IWMPDVD (VB 和 c # )** 介面都有這些方法。



| 方法                                                         | 描述                                                                                                     |
|:---------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------|
| [**返回**](wmplibiwmpdvd-iwmpdvd-back--vb-and-c.md)           | 將顯示從子功能表變更為其父功能表。<br/>                                               |
| [**恢復**](wmplibiwmpdvd-iwmpdvd-resume--vb-and-c.md)       | 從功能表模式變更播放模式，並在叫用功能表時的相同位置繼續進行。<br/> |
| [**titleMenu**](wmplibiwmpdvd-iwmpdvd-titlemenu--vb-and-c.md) | 停止播放並顯示標題功能表。<br/>                                                          |
| [**topMenu**](wmplibiwmpdvd-iwmpdvd-topmenu--vb-and-c.md)     | 停止播放並顯示根功能表。<br/>                                                           |



 

### <a name="properties"></a>屬性

**IWMPDVD (VB 和 c # )** 介面都有這些屬性。



| 屬性                                                            | 存取類型          | 描述                                                                                                                                                                          |
|:--------------------------------------------------------------------|:---------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**域**](wmplibiwmpdvd-iwmpdvd-domain--vb-and-c.md)<br/> | 唯讀<br/> | 取得 DVD 的目前網域。<br/>                                                                                                                                       |
| [**isAvailable**](iwmpdvd-isavailable--vb-and-c.md)<br/>     | 唯讀<br/> | 取得值，這個值表示指定的資訊類型是否可用，或是否可以執行指定的動作。 在 c # 中，這是 **get \_ isAvailable** 方法。<br/> |



 

使用下列屬性來取得 **IWMPDVD** 介面。



| Object                                                                   | 屬性                                                   |
|--------------------------------------------------------------------------|------------------------------------------------------------|
| [AxWindowsMediaPlayer 物件](axwindowsmediaplayer-object--vb-and-c.md) | [**Dvd**](axwmplib-axwindowsmediaplayer-dvd--vb-and-c.md) |
