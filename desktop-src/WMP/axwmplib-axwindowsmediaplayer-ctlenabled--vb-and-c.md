# <a name="axwindowsmediaplayerctlenabled-property"></a>AxWindowsMediaPlayer. Ctlenabled 屬性

Ctlenabled 屬性會取得或設定值，指出是否已啟用 Windows Media Player 控制項。

## <a name="syntax"></a>Syntax


```CSharp
public System.Boolean Ctlenabled {get; set;}
```


```VB

Public Property Ctlenabled As System.Boolean
```





## <a name="property-value"></a>屬性值

指出是否已啟用 Windows Media Player 控制項的 System.object 值。 預設值為 true。

## <a name="remarks"></a>備註

如果 [ **Ctlenabled** ] 屬性設定為 [false]，則 Windows Media Player 在全螢幕播放期間隱藏使用者控制項。
