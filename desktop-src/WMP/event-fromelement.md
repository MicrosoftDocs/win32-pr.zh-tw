# <a name="eventfromelement"></a>fromElement

**FromElement** 屬性會抓取事件來自的元素。 如果沒有任何專案通過事件，此值會是 **Null**。

``` syntax
event.fromElement
```

## <a name="possible-values"></a>可能的值

這個屬性是唯讀物件。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
