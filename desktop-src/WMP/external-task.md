# <a name="externaltask"></a>External. task

> [!Note]  
> 本主題說明針對線上商店使用而設計的功能。 不支援在線上商店的內容之外使用這項功能。

 

**Task** 屬性會抓取目前工作窗格的名稱。

## <a name="syntax"></a>Syntax

``` syntax
window.external.task
```

## <a name="possible-values"></a>可能的值

這個屬性是唯讀 **字串**。 可能的值為 [流覽]、[燒錄] 和 [同步]。
