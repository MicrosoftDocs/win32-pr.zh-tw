# <a name="image-sizes"></a>映射大小

建立面板的圖稿時，您必須考慮顯示區域大小。 下表顯示不同螢幕設定的顯示區域大小。



| DPI (模式)                     | 寬度（以圖元為單位） | 高度（以圖元為單位） |
|-------------------------------|-----------------|------------------|
| 96 (縱向)                  | 240             | 320              |
| 96 (橫向)                 | 320             | 240              |
| 192 (縱向)                 | 480             | 640              |
| 192 (橫向)                | 640             | 220              |
| 96 (平方)                    | 240             | 240              |
| 192 (平方)                   | 480             | 480              |
| 適用于 Smartphone) 的 96 (縱向  | 176             | 220              |
| 適用于 Smartphone) 的 131 (縱向 | 240             | 320              |



 

> [!Note]  
> Windows Mobile 2003 Windows Media Player 9 系列之前的版本僅支援 96 DPI 直向模式的外觀。 Windows Media Player 10 個行動裝置版或更新版本在 Smartphone 裝置上支援 131 DPI 直向模式的外觀，而這項支援預期會在未來的 Windows Media Player 行動版中繼續進行。
