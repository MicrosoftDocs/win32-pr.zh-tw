# <a name="iwmperroritem-vb-and-c-interface"></a>IWMPErrorItem (VB 和 c # ) 介面

提供錯誤資訊的存取。

**IWMPErrorItem** 介面會公開下列屬性。



| 屬性                                                                             | 描述                                                                                |
|--------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [customUrl](wmplibiwmperroritem-iwmperroritem-customurl--vb-and-c.md)               | 取得網站的 URL，這個網站會顯示有關編解碼器下載失敗的特定資訊。 |
| [errorCode](wmplibiwmperroritem-iwmperroritem-errorcode--vb-and-c.md)               | 取得目前的錯誤碼。                                                               |
| [errorCoNtext](wmplibiwmperroritem-iwmperroritem-errorcontext--vb-and-c.md)         | 取得值，指出錯誤的內容。                                          |
| [errorDescription](wmplibiwmperroritem-iwmperroritem-errordescription--vb-and-c.md) | 取得錯誤的描述。                                                           |
| [補救](wmplibiwmperroritem-iwmperroritem-remedy--vb-and-c.md)                     | 保留供未來使用。                                                                   |



 

使用下列屬性取得 **IWMPErrorItem** 介面。



| 介面                            | 屬性                                                                   |
|--------------------------------------|----------------------------------------------------------------------------|
| [IWMPError](iwmperror--vb-and-c.md) | C # 中的 [專案](iwmperror-item--vb-and-c.md) (使用 **取得 \_ 專案** 方法)  |



 

## <a name="members"></a>成員

**IWMPErrorItem (VB 和 c # )** 介面不會定義任何成員。
