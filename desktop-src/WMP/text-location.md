# <a name="text-location"></a>文字位置

您必須定義您想要使用之文字顯示方塊的位置。 若要這樣做，您必須提供四個正整數（以逗號分隔），以定義相對於背景影像的文字顯示方塊的左邊、頂端、寬度和高度（以圖元為單位）。


```C++
180,46,50,30 

```
