# <a name="custom-graphics-in-windows-media-player"></a>Windows Media Player 中的自訂圖形

透過 Windows Media 中繼檔腳本，您可以在資料流程播放時，將圖形元素加入 Windows Media Player。 您可以新增的圖形有一種基本類型。



| 類型   | Description                                                                                                                                                                                                                        |
|--------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 旗幟 | 使用 **橫幅** 元素，您可以在影片顯示區域底部的 Windows Media Player 上，將橫幅 (32 圖元高、194圖元寬 ) 影像。 然後，您可以使用子項目將超連結新增至橫幅。 |
