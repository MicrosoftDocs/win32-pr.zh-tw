# <a name="playlistitemplayingbackgroundcolor"></a>播放清單。 itemPlayingBackgroundColor

**ItemPlayingBackgroundColor** 屬性會指定或抓取目前播放播放清單專案的背景色彩。

``` syntax
        elementID.itemPlayingBackgroundColor
```

## <a name="possible-values"></a>可能的值

這個屬性是包含任何 Microsoft Internet Explorer 色彩值的讀取/寫入 **字串** 。 預設值為 " \# 222222"。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
