# <a name="buttongroupgetbutton"></a>BUTTONGROUP. getButton

**GetButton** 方法會使用指定的索引來捕獲 **BUTTONELEMENT** 。

``` syntax
        elementID.getButton(button)
```

## <a name="parameters"></a>參數

<dl> <dt>

<span id="button"></span><span id="BUTTON"></span>*按鈕*
</dt> <dd>

包含要取得之 **BUTTONELEMENT** 索引的 (**長**) **數目**。

</dd> </dl>

## <a name="return-value"></a>傳回值

這個方法會傳回對應至 **BUTTONELEMENT** 元素的物件。

| 需求 | 值 |
|--------------------|---------------------------------------------------|
| 版本<br/> | Windows Media Player 9 系列或更新版本<br/> |
