# <a name="externalbaskettitle"></a>BasketTitle

> [!Note]  
> 本主題說明針對線上商店使用而設計的功能。 不支援在線上商店的內容之外使用這項功能。

 

**BasketTitle** 屬性會在 [清單] 窗格中抓取按鈕的標題 (也稱為 Windows Media Player 中的購物籃) 。

``` syntax
window.external.basketTitle
      
```

## <a name="possible-values"></a>可能的值

這個屬性是唯讀 **字串**。
