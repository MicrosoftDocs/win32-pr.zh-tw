# <a name="editbox-element"></a>編輯方塊元素

**編輯方塊** 元素提供一個方法，讓使用者在面板內輸入文字。

**編輯方塊** 元素支援下列屬性。



| 屬性                                      | 描述                                                                                                   |
|------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [backgroundColor](editbox-backgroundcolor.md) | 指定或抓取編輯方塊控制項的背景色彩。                                         |
| [邊境](editbox-border.md)                   | 指定或抓取值，指出編輯方塊控制項是否有框線。                          |
| [editStyle](editbox-editstyle.md)             | 指定或抓取編輯方塊控制項的樣式。                                                     |
| [fontFace](editbox-fontface.md)               | 指定或抓取編輯方塊控制項中用於文字的字型。                                        |
| [fontSize](editbox-fontsize.md)               | 指定或抓取編輯方塊控制項的字型大小。                                                |
| [fontStyle](editbox-fontstyle.md)             | 指定或抓取編輯方塊控制項的字型樣式。                                               |
| [foregroundColor](editbox-foregroundcolor.md) | 指定或抓取編輯方塊控制項中的文字色彩。                                                |
| [理由](editbox-justification.md)     | 指定或抓取編輯方塊控制項內的文字對齊方式。                                 |
| [lineCount](editbox-linecount.md)             | 抓取編輯方塊控制項中的行數。                                                        |
| [readOnly](editbox-readonly.md)               | 指定或抓取值，指出編輯方塊控制項中的文字是否為唯讀或可編輯。 |
| [textLimit](editbox-textlimit.md)             | 指定或抓取使用者可以在編輯方塊控制項中輸入的最大字元數。       |
| [value](editbox-value.md)                     | 指定或抓取編輯方塊控制項中顯示的文字。                                    |
| [wordWrap](editbox-wordwrap.md)               | 指定或抓取值，指出編輯方塊控制項中是否已啟用自動換行。               |



 

**編輯方塊** 元素支援下列方法。



| 方法                                             | 描述                                                                                         |
|----------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [getLine](editbox-getline.md)                     | 使用指定的索引，抓取行的文字。                                           |
| [getLineFromChar](editbox-getlinefromchar.md)     | 抓取指定字元索引的行索引。                                         |
| [getLineIndex](editbox-getlineindex.md)           | 抓取指定行索引的字元索引。                                         |
| [getSelectionEnd](editbox-getselectionend.md)     | 抓取編輯方塊控制項中所選取文字的結束位置。                         |
| [getSelectionStart](editbox-getselectionstart.md) | 抓取編輯方塊控制項中所選取文字的開始位置。                       |
| [replaceSelection](editbox-replaceselection.md)   | 以指定的文字取代目前的選取範圍。                                             |
| [setSelection](editbox-setselection.md)           | 從指定的開始索引到指定的結束索引，選取編輯方塊控制項中的文字。 |



 

**編輯方塊** 元素支援環境屬性，並可執行所有環境事件處理常式，但 **onclick** 除外。 如需詳細資訊，請參閱 [環境屬性](ambient-attributes.md) 和 [環境事件處理常式](ambient-event-handlers.md)。

> [!Note]  
> 此元素需要 Windows XP 或更新版本的 Windows Media Player。
