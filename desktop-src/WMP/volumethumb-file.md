# <a name="volumethumb-file"></a>VolumeThumb 檔案

VolumeThumb 檔案會定義映射，以指出磁片區播放程式的播放音量層級。 您可以使用一個檔案並定義它，以供 SeekThumb 和 VolumeThumb 使用。 與其他美工檔案不同的是，thumb 檔案是在外觀定義檔的 Trackbars 區段中定義。

下圖是一般的 VolumeThumb 檔案。

![volumethumb 檔案](images/cesdkvol.png)

這兩個按鈕檔看起來相同，但大小稍有不同。 左邊的影像是使用者看到的一般觀點，而右邊的影像則是使用者在按下按鈕時所看到的推播影像。

您可能會想要讓 thumb 影像的特定區域變成透明的。 這可讓您在矩形以外的圖形中建立 thumb 影像。 您以 RGB 值255、0、255所指定的色彩填滿的任何 thumb 影像區域，在您的面板中將會顯示為透明。
