# <a name="wmpvideo"></a>WMPVIDEO

這是預先定義的 **影片** 元素，具有下列預設值。

``` syntax
backgroundColor="black"
horizontalAlignment="stretch"
verticalAlignment="stretch"
```

## <a name="remarks"></a>備註

這將會建立一個 **影片** 專案，以在調整面板大小時延展影片視窗。 顯示在此視窗中的數位影片將會縮放以符合可用空間，或根據 [**影片大小**] 下的 **[播放清單**] 功能表上的設定相應增加或減少。 [ **View** ] 功能表會以播放程式的完整模式顯示，或在面板中將 [ **標題** ] 屬性設定為 [true] 時顯示。

您可以明確地指定此 **影片** 專案的所有屬性，以覆寫這些屬性。
