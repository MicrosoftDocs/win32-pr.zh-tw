# <a name="listboxbackgroundcolor"></a>LISTBOX. backgroundColor

**BackgroundColor** 屬性會指定或抓取清單方塊控制項中的背景色彩。

``` syntax
        elementID.backgroundColor
```

## <a name="possible-values"></a>可能的值

這個屬性是包含任何 Microsoft Internet Explorer 色彩值或值為 "none" 的讀取/寫入 **字串** 。 它的預設值等於 Windows 背景色彩。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|---------------------------------------------------------|
| 版本<br/> | 適用于 Windows XP 或更新版本的 Windows Media Player<br/> |
