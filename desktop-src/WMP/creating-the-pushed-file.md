# <a name="creating-the-pushed-file"></a>建立推送的檔案

您可以採用您所建立的元素，並重複使用它們來建立您所需的其他映射。 藉由結合圖層並將結果著色，您可以建立包含您想要在使用者按下按鈕時顯示之影像的推播點陣圖。 此檔案也包含 PlayPause 按鈕暫停狀態的次要影像。

下圖是已推送的檔案。

![推送的檔案](images/ceswmpus.png)

右側的兩個按鈕適用于正常暫停狀態，以及 PlayPause 按鈕的已暫停狀態。 按鈕的色彩是黃色，但 PlayPause 按鈕的一般暫停狀態，與用於所有按鈕之正常狀態的綠色色彩有很大的差異。
