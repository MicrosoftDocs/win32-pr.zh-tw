# <a name="creating-the-mapping-file"></a>建立對應檔

當您建立了主要藝術檔案的片段之後，建立對應檔案會相當簡單。 您將會結合您已建立的兩個按鈕圖層中的美工圖案，以建立新的對應檔案。

1.  您必須採用您為主要圖片建立的兩個按鈕，並將其複製到新的圖層。 使用下列步驟：複製 [關閉] 按鈕圖層、移除任何圖層效果，然後將它重新命名為「關閉地圖」。 美工圖案看起來應該是平面的，沒有斜角。
2.  使用色彩選擇器來建立純紅色的前景色彩。 請確定 color number 值為 " \# FF0000"。 然後使用油漆桶工具，在接近地圖圖層的圓形內填滿。
3.  複製 [播放] 按鈕圖層，移除任何圖層效果，並將它重新命名為「Play 地圖」。 同樣地，美工圖案看起來應該是平面的。 您不想要在對應圖層中產生任何效果，因為您只是定義點陣圖的區域，Windows Media Player 將用來判斷滑鼠執行動作的位置，以及您想要執行的動作。
4.  使用色彩選擇器來建立純綠的前景色彩。 請確定 color number 值為 " \# 00FF00"。 然後使用油漆桶工具來填滿播放地圖圖層的圓形內。

您現在已準備好建立地圖藝術檔案。 隱藏所有圖層，然後依序顯示下列層級 (從上到下) ：

播放地圖

關閉對應

使用 [檔案] 功能表中的 [儲存複製] 命令，儲存至新的檔案。 在 [儲存複本] 對話方塊的 [另存新檔] 對話方塊中，選取 [BMP] 選項，然後輸入您稍後會在面板定義檔中參考的檔案名。 在理想的情況下，它應該與您的外觀定義檔位於相同的目錄中。 例如，您可以呼叫此檔案 map.bmp。 選擇預設設定，然後儲存檔案。

您的對應檔案看起來應該如下圖所示。

![對應檔案](images/g01map.png)

您可能會猜到，綠色區域將用來決定何時 Windows Media Player 開始，而紅色區域則是告訴它停止。 您可以使用任何兩種色彩，只要您在設定面板定義檔時使用顏色編號即可。 請確定地圖中的色彩是您想要使用且具有不同邊緣之區域的純顏色。 單純的色彩會是區域中的每個單一圖元都有相同色彩值的一種。 使用效果可能會模糊或扭曲邊緣，進而稍微修改部分圖元的色彩。 對應檔案只會由滑鼠看到，而不是人類看見，因此不會將它裝飾，而是從其他圖層中移除任何可能已執行的圖層效果。

當您儲存檔案時，您選擇的檔案名稍後將用來做為面板定義檔中 **BUTTONGROUP** 元素的 **mappingImage** 屬性值。
