# <a name="videosettingsreset"></a>VIDEOSETTINGS 重設

**Reset** 方法會將所有屬性重設為其預設值零。

``` syntax
        elementID.reset()
```

## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|-----------------------------------------------------------------|
| 版本<br/> | 適用于 Windows XP 或更新版本的 Windows Media Player 版本<br/> |
