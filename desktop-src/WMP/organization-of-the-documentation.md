# <a name="organization-of-the-documentation"></a>檔的組織

Windows Media Player SDK 記載了與 Windows Media Player 相關的各種技術。 為了讓您更輕鬆地找到所需的資訊，檔分為幾個章節，可協助您快速找出有關特定功能或技術的主題。

在適當的情況下，每個主要區段都包含三個子區段：關於、程式設計指南和參考。 [About （關於）] 區段說明可協助您瞭解如何使用的特定功能或技術。 《程式設計指南》一節新增了教學課程，讓您更深入瞭解技術，並提供逐步指示。 參考區段會詳細說明可供您使用的介面、物件、方法、屬性、事件、元素和屬性。
