# <a name="building-a-visualization"></a>建立視覺效果

您可以使用 Visual Studio 以及 Windows Media Player 外掛程式嚮導來建立自訂視覺效果。 如需設定 wizard 和建立外掛程式的詳細資訊，請參閱 [建立外掛程式](building-a-plug-in.md)。

當您執行嚮導時，將會提供數種不同類型之外掛程式的選項。選擇 [視覺效果]。

當您完成嚮導並建立外掛程式之後，請在 Windows Media Player 中啟動一首歌曲，然後按一下視覺效果。 自動為您建立的預設 *值稱為「專案名稱* 列」 *和「專案名稱* 」 Wave *，其中專案名稱是* 專案的名稱。
