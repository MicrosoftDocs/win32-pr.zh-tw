# <a name="eventtoelement"></a>toElement

**ToElement** 屬性會抓取鍵盤焦點移至的元素。 這個屬性只適用于 **onblur** 事件;對於所有其他事件，其值為 **Null**。

``` syntax
event.toElement
```

## <a name="possible-values"></a>可能的值

這個屬性是唯讀物件。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
