# <a name="textscrolling"></a>文字。滾動

**滾動** 屬性會指定或抓取值，指出文字是否會滾動。

``` syntax
        elementID.scrolling
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **布林值**。



| 值 | 描述                     |
|-------|---------------------------------|
| true  | 已啟用滾動。           |
| false | 預設值。 滾動已停用。 |



 

## <a name="remarks"></a>備註

滾動功能會在文字結尾和重複行的開頭之間提供兩個空間的緩衝區。

[ **對齊** ] 屬性會指定文字首次出現的位置，然後才開始滾動。

如需說明如何使用 **TEXT** 元素屬性的範例，請參閱 [value](text-value.md)屬性。
