# <a name="pixelaspectratiox"></a>PixelAspectRatioX

**PixelAspectRatioX** 屬性是影片外觀比例的水準部分。

## <a name="applies-to"></a>套用至

-   [影片專案](video-item-attributes.md)

## <a name="remarks"></a>備註

針對外觀比例為16:9 的影片， **PixelAspectRatioX** 會等於 "16"。

| 需求 | 值 |
|--------------------|------------------------------------|
| 版本<br/> | Windows Media Player 11<br/> |
