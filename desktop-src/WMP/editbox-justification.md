# <a name="editboxjustification"></a>編輯方塊。理由

[ **對齊** ] 屬性會指定或抓取編輯方塊控制項內的文字對齊方式。

``` syntax
        elementID.justification
```

## <a name="possible-values"></a>可能的值

這個屬性是包含下列其中一個值的讀取/寫入 **字串** 。



| 值  | 描述                                                        |
|--------|--------------------------------------------------------------------|
| Left   | 預設值。 將文字對齊編輯方塊控制項的左邊緣。 |
| Right  | 將文字對齊編輯方塊控制項的右邊緣。         |
| Center | 將文字對齊編輯方塊控制項的水準置中。  |



 

## <a name="remarks"></a>備註

有兩個圖元的邊界。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|---------------------------------------------------------|
| 版本<br/> | Windows XP 或更新版本的 Windows Media Player<br/> |
