# <a name="trackbar-location"></a>進行中的位置

您必須定義您想要使用的 [執行程式] 的位置。 您必須提供四個以逗號分隔的正整數，以圖元為單位來定義並排顯示的左邊、頂端、寬度和高度（以圖元為單位）。


```C++
180,46,50,30 

```
