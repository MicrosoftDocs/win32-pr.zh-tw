# <a name="progressbar-element"></a>PROGRESSBAR 元素

**PROGRESSBAR** 元素提供一種方式，可在水準或垂直控制項中顯示進度資訊。 這個元素與 **滑杆** 元素相同。 進度列通常是不是互動式的滑杆，但是沒有任何專案可防止它以互動方式使用。 這個元素只是為了方便標記而提供，可讓您在程式碼中 noninteractivity 清楚，如果您選擇使用該方法。 它沒有其他值。
