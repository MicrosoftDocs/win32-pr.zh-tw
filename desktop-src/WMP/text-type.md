# <a name="text-type"></a>文字類型

您必須定義要使用的文字顯示方塊類型。 下表顯示下列有效類型。



| 類型        | 描述                                                                                                                                                                                                                                                                         |
|-------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Time        | 媒體專案中目前的播放位置（以分鐘和秒為單位）。                                                                                                                                                                                                                 |
| 播放清單    | 目前播放清單的名稱。                                                                                                                                                                                                                                                       |
| Track\#     | 播放清單中目前專案的數目。                                                                                                                                                                                                                                         |
| \#軌道    | 目前播放清單中的專案數。                                                                                                                                                                                                                                            |
| 標題       | 目前播放清單中目前專案的標題。 在某些情況下，這種類型可能無法正確顯示。 或者，使用 Title 作為字幕的一部分。                                                                                                                 |
| 作者      | 目前的播放清單中目前專案的作者。 在某些情況下，這種類型可能無法正確顯示。 或者，您也可使用 Author 作為字幕的一部分。                                                                                                               |
| 著作權   | 目前播放清單中目前專案的著作權資料。 在某些情況下，這種類型可能無法正確顯示。 或者，使用著作權作為字幕的一部分。                                                                                                    |
| 檔案名稱    | 目前的播放清單中目前專案的檔案名。                                                                                                                                                                                                                              |
| FilenameExt | 目前的播放清單中目前專案的副檔名。 唯一支援的值為 "wma" 和 "wav"。                                                                                                                                                                 |
| Bitrate     | 目前媒體專案的編碼位元速率 (Kbps) 。                                                                                                                                                                                                                           |
| 頻率   | 取樣頻率 (目前媒體專案的 KHz) 。                                                                                                                                                                                                                           |
| 狀態\*    | 播放期間目前音訊專案的演出者、專輯、內容類型和位元速率，以及目前影片專案的適用)  (的位速率和字幕。 當媒體專案未播放時，[狀態] 文字類型會顯示目前的播放狀態 (例如，[已停止] 或 [已暫停]) 。 |



 

\*適用于 Windows Media Player 10 行動裝置版或更新版本。
