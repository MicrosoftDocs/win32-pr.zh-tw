# <a name="object-model-reference-for-c"></a>C + + 的物件模型參考

> [!Note]  
> Windows Media Player 9 系列或更新版本支援在 c + + 中使用 Windows Media Player ActiveX 控制項。

 

c + + 的物件模型參考提供 Windows Media Player 的檔，以及 Windows Media Player 的行動介面和列舉類型。

如果您想要搭配 Windows Media Player 控制項使用遠端或外觀，您也需要瞭解 **IWMPPlayerServices** 和 **IWMPRemoteMediaServices** 介面，這些介面是設計來搭配 c + + 使用。

如需在 c + + 程式中內嵌 Windows Media Player 控制項的示範，請參閱此 SDK 隨附的 c + + 範例。

下列各節記載 Windows Media Player 物件模型所提供的 c + + 介面和列舉。



| 區段                                    | 描述                                                                                                     |
|--------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| [介面](interfaces.md)               | 列出可供使用 Windows Media Player ActiveX 控制項的 c + + 程式設計人員使用的介面和方法。 |
| [列舉類型](enumeration-types.md) | 列出可在 c + + 程式碼中使用的列舉類型。                                                              |
