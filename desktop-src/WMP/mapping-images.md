# <a name="mapping-images"></a>對應影像

對應影像可用於特定的控制項，以指定哪些區域會回應滑鼠點擊，以及判斷哪些控制項會收到哪些事件。 地圖影像的高度和寬度必須與主要影像相同，而且應該建立，如此一來，這兩個檔案中的影像都能完全對齊。 基於這個理由，建議使用支援圖層的圖形編輯程式，因為您可以確定影像完全對齊。

不同的控制項需要不同類型的對應。 **ButtonElement** 元素需要針對每個按鈕具有不同色彩的地圖。 對應檔中的彩色區域必須對應到您要對應之按鈕的區域。
