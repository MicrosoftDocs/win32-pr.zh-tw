# <a name="playlistplaylist"></a>播放清單。播放清單

**播放** 清單屬性會指定或抓取 **播放清單** 元素提供介面的 **播放清單** 物件。

``` syntax
        elementID.playlist
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **播放清單** 物件，沒有預設值。

## <a name="remarks"></a>備註

如果指定的播放清單無效或未指定任何值， **播放清單** 元素會顯示目前播放的媒體專案。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
