# <a name="creating-the-super-file"></a>建立超級檔案

Super 檔案包含已停用的 trackbars 映射。

> [!Note]  
> Windows Media Player 10 行動版或更新版本的外觀中不會使用超級檔案，因為搜尋 trackbars 的已停用映射位於 seekthumb 檔案中。

 

下圖是超級檔。

![super 檔案](images/ceswmsup.png)

它包含已停用之磁片區的對等的影像。
