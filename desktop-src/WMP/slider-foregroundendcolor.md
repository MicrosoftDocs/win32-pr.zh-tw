# <a name="sliderforegroundendcolor"></a>滑杆. foregroundEndColor

**ForegroundEndColor** 屬性會指定或抓取滑杆控制項的前景結束色彩。

``` syntax
        elementID.foregroundEndColor
```

## <a name="possible-values"></a>可能的值

這個屬性是包含任何 Microsoft Internet Explorer 色彩值的讀取/寫入 **字串** 。 它沒有預設值。

## <a name="remarks"></a>備註

**ForegroundEndColor** 會與 **foregroundColor** 搭配使用。 所建立的效果是從 **foregroundColor** 到 **foregroundEndColor** 的漸層漸層。
