# <a name="primary-images"></a>主要映射

您會想要為您的面板建立主要映射。 主要映射是由 **VIEW** 元素所定義的一個背景影像，以及任何其他使用特定元素的影像所組成。 例如，如果您建立 **button** 專案，則必須包含該按鈕的美工圖案，並確定它的迭置順序高於背景影像。
