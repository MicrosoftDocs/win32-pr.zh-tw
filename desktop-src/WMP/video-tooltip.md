# <a name="videotooltip"></a>VIDEO。工具提示

**Tooltip** 屬性會指定或抓取影片視窗的工具提示文字。

``` syntax
        elementID.toolTip
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **字串**，長度不得超過1024個字元。 它沒有預設值。

## <a name="remarks"></a>備註

當這個屬性設定為 "" (空白字串) 時，不會顯示任何工具提示。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
