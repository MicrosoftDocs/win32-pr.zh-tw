# <a name="playlistcollectionremove-method"></a>PlaylistCollection 方法

**Remove** 方法會從文件庫中移除播放清單。

## <a name="syntax"></a>語法


```JScript
PlaylistCollection.remove(
  playlist
)
```



## <a name="parameters"></a>參數

<dl> <dt>

*播放清單* \[在\]
</dt> <dd>

要移除的 **播放清單** 物件。

</dd> </dl>

## <a name="return-value"></a>傳回值

這個方法不會傳回值。

## <a name="remarks"></a>備註

若要使用此方法，需要有程式庫的完整存取權。 如需詳細資訊，請參閱連結 [庫存取](library-access.md)。
