# <a name="eventbutton"></a>事件。按鈕

**按鈕** 屬性會抓取一個值，指出事件發生時哪一個滑鼠按鍵已關閉。

``` syntax
event.button
```

## <a name="possible-values"></a>可能的值

這個屬性是唯讀的 **數位** (**long**) 。



| 值 | 描述                      |
|-------|----------------------------------|
| 0     | 未關閉任何滑鼠按鍵。      |
| 1     | 滑鼠左鍵已關閉。  |
| 2     | 滑鼠右鍵已關閉。 |
| 3     | 這兩個滑鼠按鍵都已關閉。    |



 

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
