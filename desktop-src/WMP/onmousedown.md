# <a name="onmousedown"></a>onmousedown

**Onmousedown** 事件處理常式會處理使用者按下滑鼠按鍵時所發生的事件。

``` syntax
onmousedown
```

## <a name="remarks"></a>備註

此事件處理常式不適用於 **播放清單** 元素（影片中的 **video** 元素 *）。*[**無視窗**] 為 false，**或效果元素** *。***視窗** 化是 true。
