# <a name="stringcollectioncount"></a>StringCollection 計數

**Count** 屬性會捕獲字串集合中的專案數。

## <a name="syntax"></a>Syntax

*玩家*。*mediaCollection*。**getAttributeStringCollection** ( *屬性*， *媒體*) 。**計數**

## <a name="possible-values"></a>可能的值

這個屬性是唯讀的 **數位** (**long**) 。

## <a name="remarks"></a>備註

若要取得這個屬性的值，需要有程式庫的讀取權限。 如需詳細資訊，請參閱連結 [庫存取](library-access.md)。
