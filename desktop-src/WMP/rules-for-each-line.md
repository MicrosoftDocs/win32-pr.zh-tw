# <a name="rules-for-each-line"></a>每一行的規則

面板定義檔中的每個定義都必須在不同的行上。 您必須使用下列規則來輸入文字行：

-   每一行的結尾都必須是段落。 某些行的長度可能會超過文字編輯器的可見行長度，但不會中斷這些線條。 您可能會想要關閉換行，以確定您的線條不會中斷。
-   只使用面板參考中定義的可見字元。
-   行是由一或多個專案所組成，以空格分隔。
-   您可以在專案之間輸入任意數量的空格，以提供可讀性。
-   請勿在檔案中的任何位置使用任何定位字元。
-   會忽略開頭為兩個連續正斜線 (//) 的行。 將這些用於批註。 也會忽略空白行。
-   在括弧和區段名稱之間加入空格。
