# <a name="externalpluginrunning"></a>PluginRunning

> [!Note]  
> 本主題說明針對線上商店使用而設計的功能。 不支援在線上商店的內容之外使用這項功能。

 

**PluginRunning** 屬性會抓取值，指出線上商店的外掛程式是否正在執行。

## <a name="syntax"></a>Syntax

``` syntax
window.external.pluginRunning
```

## <a name="possible-values"></a>可能的值

這個屬性是唯讀的 **布林值**。 **TRUE** 表示外掛程式正在執行。
