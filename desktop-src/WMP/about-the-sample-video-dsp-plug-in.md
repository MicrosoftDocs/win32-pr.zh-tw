# <a name="about-the-sample-video-dsp-plug-in"></a>關於範例影片 DSP 外掛程式

範例影片 DSP 外掛程式提供簡單的處理常式，可根據使用者在屬性頁中提供的因素來調整影片的色彩飽和度。 外掛程式接受介於0和1之間的值。 預設值為 1。 值為1時，不會影響影片影像;其他比例因素會反滲透影像色彩。 例如，值 .5 會導致50% 的色彩飽和度。 零的值會產生灰階影片。

範例影片 DSP 外掛程式適用于下列影片格式：

-   NV12
-   YV12
-   YUY2
-   UYVY
-   RGB32
-   RGB24
-   RGB555
-   RGB565
