# <a name="about-custom-visualizations"></a>關於自訂視覺效果

Windows Media Player 提供可建立自訂視覺效果的介面。 本節概述視覺效果的架構、它們與 Windows Media Player 的相關資訊，以及建立自訂視覺效果所需的技術詳細資料。

請注意，程式碼中的數個位置，您會看到文字效果。 視覺效果是效果。

下列主題會更詳細地說明自訂視覺效果。



| 主題                                                              | 描述                                                                                |
|--------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [使用者總覽](user-overview.md)                                 | 描述使用者觀點的視覺效果。                                          |
| [開發人員總覽](developer-overview.md)                       | 描述從開發人員觀點來看的視覺效果。                                     |
| [建立視覺效果](building-a-visualization.md)           | 描述如何建立視覺效果專案。                                           |
| [執行您的程式碼](implementing-your-code.md)               | 描述如何修改視覺效果外掛程式 wizard 產生的範例程式碼。     |
| [安裝視覺效果](installing-visualizations.md)         | 描述如何在使用者的電腦上安裝視覺效果。                             |
| [提交您的視覺效果](submitting-your-visualization.md) | 不再支援將您已完成的視覺效果發佈至 WMPlugins 網站。 |
