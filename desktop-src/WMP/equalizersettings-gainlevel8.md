# <a name="equalizersettingsgainlevel8"></a>EQUALIZERSETTINGS.gainLevel8

**GainLevel8** 屬性會指定或抓取寬線8的增益層級。

``` syntax
        elementID.gainLevel8
```

## <a name="possible-values"></a>可能的值

這個屬性是一個讀/寫 **號碼** (**Float**) ，其值通常範圍從20到 + 20。 預設值為零。

## <a name="remarks"></a>備註

這個屬性會調整以4kHz 為中心的音訊頻率範圍部分。

如果未指定此屬性，則會保留先前的值。
