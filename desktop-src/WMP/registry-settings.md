# <a name="registry-settings"></a>登錄設定

Windows Media Player 9 系列或更新版本會使用登錄來儲存特定設定。 您可以對這些設定進行變更，以變更 Windows Media Player 和 Windows Media Player 控制項的行為。

下列各節詳細說明支援的登錄設定。



| 區段                                                                                                        | 描述                                                                                                                                                           |
|----------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [副檔名登錄設定](file-name-extension-registry-settings.md)                             | 說明如何註冊自訂的副檔名。                                                                                                               |
| [自訂配置登錄設定](custom-scheme-registry-settings.md)                                         | 說明如何註冊自訂配置。                                                                                                                            |
| [資料夾監控登錄設定](folder-monitoring-registry-settings.md)                                 | 說明如何註冊 Windows Media Player 要監視的檔案資料夾。                                                                                       |
| [防火牆埠登錄設定](firewall-port-registry-settings.md)                                         | 描述 Windows Media Player 設定的登錄專案，以指定防火牆是否應讓埠保持開啟，以供程式庫共用。                                   |
| [無訊息取得登錄設定](silent-acquisition-registry-setting.md)                                 | 描述 Windows Media Player 用來決定是否要在使用者播放或同步處理受保護的內容時，自動下載許可權的登錄專案。 |
| [註冊應用程式相依性](registering-application-dependency.md)                                   | 提供如何註冊應用程式，以確保使用者電腦上有正確的執行時間檔案的相關資訊。                                         |
| [用於追蹤安裝進度的登錄專案](registry-entries-for-tracking-installation-progress.md) | 描述安裝程式可用來追蹤 Windows Media Player 設定進度的登錄專案。                                                    |
