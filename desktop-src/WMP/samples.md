# <a name="wmp-sdk-samples"></a>WMP SDK 範例

Windows Media Player SDK 包含各種不同的範例，這些範例會示範本檔中所述的許多程式設計技術。

-   腳本範例示範如何將 Player ActiveX 控制項內嵌在網頁中。 完整的網頁範例會示範如何偵測目前版本的播放程式 ActiveX 控制項。
-   外觀範例示範如何變更視覺外觀和 Windows Media Player 的行為。
-   Microsoft Visual C++ 的 wizard 可以建立數種外掛程式類型的唯一工作範例：視覺效果外掛程式、使用者介面外掛程式、轉譯外掛程式，以及數位信號處理外掛程式（）。
-   線上商店的範例包含 Microsoft Visual C++ 的 wizard，可建立線上商店 COM 物件的唯一工作範例。
-   Microsoft Visual C++ 的範例示範內嵌 Windows Media Player ActiveX 控制項、遠端播放播放程式 ActiveX 控制項，以及使用程式庫。
-   C # 範例示範如何搭配使用 Windows Media Player ActiveX 控制項與 .NET Framework。
-   Visual Basic .NET 範例會示範如何使用 Windows Media Player ActiveX 控制項搭配 .NET Framework。

若要使用範例，您必須安裝 Windows SDK，其中包含 Windows Media Player SDK。 若要下載 Windows SDK，請移至 Windows 開發人員中心的 [ [Windows SDK](https://msdn.microsoft.com/windows/bb980924.aspx) ] 頁面。
