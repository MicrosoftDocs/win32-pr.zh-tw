# <a name="text-editors"></a>文字編輯器

您必須使用不包含任何格式化字元的文字編輯器。 建議使用簡單的文字編輯器（例如 [記事本]），雖然開發環境（例如 Microsoft Visual Studio）提供先進的功能。 您不應該使用像是 Microsoft Word 的編輯器，因為預設 Word 檔案格式會儲存其他無法使用的格式化程式碼。 如果您使用 word 處理器（例如 Word），則每次儲存檔案時，都必須記得將您的工作儲存為未格式化的文字。

如果您的文字編輯器允許，您可能會關閉自動換行，而您可能想要開啟查看所有字元的功能，以確保不會不慎加入任何索引標籤或其他控制字元。 請記住，在您的外觀定義檔中包含定位字元會導致面板失敗。
