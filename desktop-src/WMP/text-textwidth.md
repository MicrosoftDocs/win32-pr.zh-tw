# <a name="texttextwidth"></a>TextWidth

**TextWidth** 屬性會以圖元為單位來抓取 **值** 屬性中所含文字的寬度（以圖元為單位）。

``` syntax
        elementID.textWidth
```

## <a name="possible-values"></a>可能的值

這個屬性是唯讀的 (**int**) 的 **號碼**。

## <a name="remarks"></a>備註

傳回的值是根據 **fontFace**、 **fontSize** 和 **fontStyle** 屬性，以及 **值** 屬性字串中的實際字元。

如需說明如何使用 **TEXT** 元素屬性的範例，請參閱 [value](text-value.md)屬性。
