# <a name="playlistcolumnsvisible"></a>播放清單。 columnsVisible

**ColumnsVisible** 屬性會指定或抓取值，指出是否顯示資料行標頭。

``` syntax
        elementID.columnsVisible
```

## <a name="possible-values"></a>可能的值

此屬性是讀取/寫入 **布林值**。



| 值 | 描述                        |
|-------|------------------------------------|
| true  | 預設值。 顯示資料行標題。 |
| false | 不會顯示資料行標題。      |



 

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|------------------------------------------------------|
| 版本<br/> | Windows Media Player 7.0 版或更新版本<br/> |
