# <a name="multicast"></a>多點傳送

泛型 Winsock multipoint 函數支援 IP 多播。 不過，支援多播的 TCP/IP 傳輸提供者也必須提供 Berkeley 軟體散發 (BSD) 樣式–支援對應的多播選項來支援多播。 這可簡化將現有多播應用程式移植至 Windows 通訊端2的程式。
