# <a name="quality-of-service-in-the-windows-sockets-2-spi"></a>Windows 通訊端 2 SPI 中的服務品質

服務品質 (qos) 會透過 Windows Server 2003、Windows XP 和 Windows 2000 所支援的各種 QoS 元件在 Windows 中執行。 QoS 及其元件的詳細說明，位於 Microsoft Windows 軟體開發套件 (SDK) 的服務品質專屬區段中。 本 QoS 章節將說明每個 QoS 元件，以及它在 QoS 程式中扮演的角色，並提供額外的指導方針，讓您能夠執行具備 QoS 功能的應用程式和服務。 如需詳細資訊，請參閱 Windows SDK 的[服務品質 (QoS) ](/previous-versions/windows/desktop/qos/qos-start-page)一節。

本節說明 Winsock 開發人員可用的服務功能品質。 下列清單說明本節中的主題：

-   [使用量模型](usage-model-2.md)
-   [QoS 更新](qos-updates-2.md)
-   [SPI 中的預設值](default-values-in-the-spi-2.md)
-   [SPI 中的 QoS 範本](qos-templates-in-the-spi-2.md)
