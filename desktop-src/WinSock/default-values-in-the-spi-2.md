# <a name="default-values-in-the-spi"></a>SPI 中的預設值

服務品質 (QoS) 的執行詳細資料，可在 Microsoft Windows 軟體開發套件 (SDK) 的個別專屬 QoS 區段中提供。 如需有關預設值的 QoS 使用指導方針的詳細資訊，請參閱 Windows SDK 的「 [服務品質](/previous-versions/windows/desktop/qos/qos-start-page) 」一節。
