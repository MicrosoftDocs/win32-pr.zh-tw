# <a name="multipoint-and-multicast-semantics"></a>Multipoint 和多播語義

在考慮如何在 Windows 通訊端 2 (Winsock) 中支援 multipoint 和多播語義時，有許多現有和提議的配置 (包括 IP 多播、ATM 點對點連線、聖 II、T. 120、.h、MCU) 。 若要啟用各種不同配置的一致討論，請先建立分類，以區分每個架構的基本屬性。 在本檔中， *multipoint* 一詞代表 multipoint 和多播。
