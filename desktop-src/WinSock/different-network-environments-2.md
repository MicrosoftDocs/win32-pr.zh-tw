# <a name="different-network-environments"></a>不同的網路環境

數個網路環境會影響應用程式的網路行為。 區分網路環境的屬性包括低和高頻寬，以及低和高 RTT。 網路環境會以不同的方式影響交易式和串流處理應用程式。 交易式應用程式對 RTT 而言更為敏感;串流應用程式對頻寬延遲產品而言更加敏感。

![此圖顯示不同的網路環境如何影響應用程式的網路行為。](images/hperf-1.png)

撥號網路和某些無線網路具有變數 RTT。 衛星網路的上游和下游之間通常會有非對稱的頻寬。 無線區域網路和 xDSL 是一個很好的網路範例，其頻寬延遲產品類似于快速乙太網路。
