# <a name="event-objects-in-the-windows-sockets-2-spi"></a>Windows 通訊端 2 SPI 中的事件物件

事件物件會在 Windows 通訊端2中引進，作為 Winsock 服務提供者和應用程式之間的一般同步處理機制。 這些用途有許多用途，包括指出重迭作業的完成，以及一或多個網路事件的發生次數。
