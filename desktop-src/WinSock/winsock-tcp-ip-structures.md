# <a name="winsock-tcpip-structures"></a>Winsock TCP/IP 結構

下列清單提供每個 Winsock TCP/IP 結構的簡短描述。 如需任何結構的詳細資訊，請按一下結構名稱。



| TCP/IP 結構                                 | Description                                                                                                                              |
|--------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| [**介面 \_ 資訊**](/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-interface_info)      | 與 ioctl 命令搭配使用，以取得介面 IP 位址的相關資訊。                                                         |
| [**介面 \_ 資訊 \_ EX**](/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-interface_info_ex) | 搭配 **SIO \_ GET \_ interface \_ LIST IOCTL** 命令使用，以取得介面 IP 位址的相關資訊，包括 IPv6 位址。 |
| [**sockaddr \_ gen**](/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-sockaddr_gen)            | 提供一般通訊端位址資訊。 搭配 [**介面 \_ 資訊**](/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-interface_info) 結構使用。                        |
