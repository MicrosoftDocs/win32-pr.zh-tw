# <a name="winsock-annexes"></a>Winsock 附件

Winsock 附件提供通用 Winsock 通訊協定套件的實資訊，並討論如何搭配 Winsock 使用每個通訊協定。

本附錄中的每個通訊協定都有慣例、行為或特殊功能，這些都不會對一般實行產生任何特性。 本節記載開發人員在執行或使用所述的通訊協定時，必須考慮的詳細資料。

Winsock 附件包含下列各節：

-   [Winsock ATM 附錄](winsock-atm-annex.md)
-   [Winsock IPX/SPX 附錄](winsock-ipx-spx-annex.md)
-   [Winsock TCP/IP 附錄](winsock-tcp-ip-annex.md)
