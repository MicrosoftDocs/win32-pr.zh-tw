# <a name="performance-dimensions"></a>效能維度

應用程式效能會依觀點而有所不同。 對於使用者而言，理想的應用程式可提供高效能和回應性。 對於系統管理員而言，理想的應用程式會耗用最少量的網路資源。 效能也會根據應用程式類型而有所不同。 大量檔案傳輸應用程式的效能可能會與交易式應用程式的效能有不同的含意。

下列各節會解決不同的效能維度和應用程式類型，以及網路環境如何影響效能。

-   [效能需求：使用者和系統管理員](performance-needs-users-and-administrators-2.md)
-   [交易式與串流應用程式](transactional-versus-streaming-applications-2.md)
-   [不同的網路環境](different-network-environments-2.md)
