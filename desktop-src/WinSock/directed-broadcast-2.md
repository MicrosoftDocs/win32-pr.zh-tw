# <a name="directed-broadcast"></a>導向廣播

通常視為比所有路由廣播更容易使用的網路，導向的廣播會將自己限制在您指定的區域網路。 以二進位的網路編號和 **sa \_ nodenum** 填入 **sa \_ netnum** 。 路由器會將此要求轉送到目的地網路，以作為本機廣播。 中繼網路不會看到此封包為廣播。
