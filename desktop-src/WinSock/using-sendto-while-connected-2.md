# <a name="using-sendto-while-connected"></a>連接時使用 Sendto

[**WSPSendTo**](/previous-versions/windows/desktop/legacy/ms742291(v=vs.85)) 一律會將資料傳遞至指定的位址，即使已在 [**WSPConnect**](/previous-versions/windows/hardware/network/ff566275(v=vs.85))中建立傳送通訊端的指定對等互連也是一樣。
