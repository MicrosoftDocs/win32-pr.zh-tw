# <a name="sio_multipoint_loopback-command-code-for-wsaioctl"></a>\_ \_ 適用于 WSAIOCTL 的 SIO MULTIPOINT 回送命令程式碼

當 \_ 您在 nonrooted 資料平面中使用 d 分葉通訊端時，會想要將傳送的流量送回相同的通訊端上。 適用于 \_ WSAIoctl 的 SIO MULTIPOINT \_ 回[](/windows/desktop/api/Winsock2/nf-winsock2-wsaioctl)送命令程式碼可用來啟用或停用 MULTIPOINT 流量的回送。
