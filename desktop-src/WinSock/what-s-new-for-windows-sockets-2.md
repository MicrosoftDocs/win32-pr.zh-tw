## <a name="updated-for-windows-8-and-windows-server-2012"></a>已更新 Windows 8 和 Windows Server 2012

Microsoft Windows 8 和 Windows Server 2012 引進了新的 Windows 通訊端程式設計項目。

以較低的延遲和抖動功能提高網路效能，可提供一組高速網路擴充功能。 這些以伺服器應用程式為目標的延伸模組是使用預先註冊的資料緩衝區和完成佇列來提高效能。
