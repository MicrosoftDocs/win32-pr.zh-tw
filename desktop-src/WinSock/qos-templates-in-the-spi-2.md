# <a name="qos-templates-in-the-spi"></a>SPI 中的 QoS 範本

服務品質 (QoS) 的執行詳細資料，可在 Microsoft Windows 軟體開發套件 (SDK) 的個別專屬 QoS 區段中提供。 如需 QoS 使用指導方針的詳細資訊，包括可用的 QoS 範本及其相關聯的 [**FLOWSPEC**](/windows/win32/api/qos/ns-qos-flowspec)值，請參閱 Windows SDK 的「[服務品質](/previous-versions/windows/desktop/qos/qos-start-page)」一節。
