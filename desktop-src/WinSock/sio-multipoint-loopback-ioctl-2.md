# <a name="sio_multipoint_loopback-ioctl"></a>SIO \_ MULTIPOINT \_ 回送 Ioctl

在 \_ nonrooted 資料平面中使用 d 分葉通訊端時，通常會想要能夠控制是否也會在相同的通訊端上接收送出的流量。 適用于 \_ WSPIoctl 的 SIO MULTIPOINT \_ 回[](/previous-versions/windows/hardware/network/ff566296(v=vs.85))送命令程式碼可用來啟用或停用 MULTIPOINT 流量的回送。
