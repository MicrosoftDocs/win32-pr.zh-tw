# <a name="winsock-spi-structures"></a>Winsock SPI 結構

下列清單提供每個 Winsock SPI 結構的精確描述。 如需任何 Winsock SPI 結構的詳細資訊，請按一下結構名稱。



| Winsock SPI 結構                                         | Description                                                                                                   |
|---------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**NSP \_ 常式**](/windows/desktop/api/Ws2spi/ns-ws2spi-nsp_routine)                           | 包含命名空間服務提供者第1版 (NSPv1) 提供者所執行函數的資訊。 |
| [**NSPV2 \_ 常式**](/windows/desktop/api/Ws2spi/ns-ws2spi-nspv2_routine)                       | 包含命名空間服務提供程式第2版 (NSPv2) 提供者所執行函數的資訊。 |
| [**WSATHREADID**](/windows/desktop/api/Ws2spi/ns-ws2spi-wsathreadid)                          | 可讓提供者識別非同步程式呼叫 (Apc) 可以排入佇列的執行緒。           |
| [**WSC \_ 提供者 \_ 審核 \_ 資訊**](/windows/desktop/api/Ws2spi/ns-ws2spi-wsc_provider_audit_info) | 包含多層式服務提供者的審核資訊 (LSP) 專案。                                        |
| [**WSPDATA**](/windows/desktop/api/Ws2spi/ns-ws2spi-wspdata)                                  | 包含服務提供者資訊。                                                                        |
| [**WSPPROC \_ 資料表**](/windows/desktop/api/Ws2spi/ns-ws2spi-wspproc_table)                       | 包含服務提供者函數之指標的表格。                                                   |
| [**WSPUPCALLTABLE**](/windows/desktop/api/Ws2spi/ns-ws2spi-wspupcalltable)                      | 包含服務提供者 upcall 函式之指標的表格。                                            |
