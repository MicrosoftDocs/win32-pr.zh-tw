# <a name="qos-updates"></a>QoS 更新

服務品質 (QoS) 的執行詳細資料，可在 Microsoft Windows 軟體開發套件 (SDK) 的個別專屬 QoS 區段中提供。 如需 QoS 更新的詳細資訊，請參閱 Windows SDK 的「[服務品質](/previous-versions/windows/desktop/qos/qos-start-page)」一節。
