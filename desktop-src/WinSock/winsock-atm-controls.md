# <a name="winsock-atm-controls"></a>Winsock ATM 控制項

「Windows 通訊端2」規格原本就支援「ATM 點對點」和「點對點」連接設定和卸載。 事實上，Windows 通訊端 2 QoS 規格和與通訊協定無關的 multipoint/多播機制的設計，與其他通訊協定搭配使用。 請參閱分別適用于 Windows 通訊端2、服務品質和 Multipoint 支援 Windows 通訊端 2 API 規格的第2.7 節和附錄 D。 因此，本檔中不需要引進任何 ATM 特定 Ioctls。
