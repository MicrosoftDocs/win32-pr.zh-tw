# <a name="winsock-ipxspx-structures"></a>Winsock IPX/SPX 結構

下列清單提供每個 Winsock IPX/SPX 結構的精確描述。 如需任何結構的詳細資訊，請按一下結構名稱。



| IPX/SPX 結構                                            | Description                                                          |
|--------------------------------------------------------------|----------------------------------------------------------------------|
| [**IPX \_ 位址 \_ 資料**](/windows/desktop/api/Wsnwlink/ns-wsnwlink-ipx_address_data)             | 提供與 IPX 系結的特定介面卡相關的資訊。 |
| [**IPX \_ NETNUM \_ 資料**](/windows/desktop/api/Wsnwlink/ns-wsnwlink-ipx_netnum_data)               | 提供指定之 IPX 網路編號的相關資訊。           |
| [**IPX \_ SPXCONNSTATUS \_ 資料**](/windows/desktop/api/Wsnwlink/ns-wsnwlink-ipx_spxconnstatus_data) | 提供有關已連線之 SPX 通訊端的資訊。                   |
