# <a name="internet-protocol-version-6-ipv6"></a>網際網路通訊協定第 6 版 (IPv6)

請務必確定新的 Winsock 應用程式以及現有的應用程式與 IPv6 完全相容。 適用于新 IPv4 位址配置的 IPv4 位址空間可用性，已在亞洲和太平洋2011中耗盡。 世界上的其他部分預期會在幾年內用盡。

使用 IPv6 位址可取得新網站和服務的成長百分比。 新興市場的許多網際網路使用者都依存于 IPv6 以進行網際網路存取。

Microsoft 長期致力於支援 IPv6。 完整的 IPv6 支援是在 Windows XP Service Pack 1 (SP1) 和更新版本中提供。

本檔提供 IPv6 Winsock 支援的相關資訊：

-   [使用 IPv6](using-ipv6-2.md)
-   [使用 Internet Explorer 來存取 IPv6 網站](using-internet-explorer-to-access-ipv6-web-sites-2.md)
-   [IPv6 的建議設定](recommended-configurations-2.md)
-   [其他 IPv6 主題](additional-topics-2.md)

如需將 IPv6 功能新增至 Windows 通訊端應用程式的詳細資訊，請參閱 [Windows 通訊端應用程式的 Ipv6 指南](ipv6-guide-for-windows-sockets-applications-2.md)。

Technet 的 [Microsoft 網際網路通訊協定第6版 (ipv6) ](/previous-versions/windows/it-pro/windows-server-2008-R2-and-2008/dd379473(v=ws.10))可在 Technet 上取得 IPv6 通訊協定的簡介，以及部署和 IPv6 轉換技術的概述。
