# <a name="winsock-enumerations"></a>Winsock 列舉

下列清單提供每個 Winsock 列舉的精確描述。 如需任何列舉的詳細資訊，請按一下列舉名稱。

| 列舉型別 | 描述 |
|-|-|
| [**控制 \_ 通道 \_ 觸發程式 \_ 狀態**](/windows/win32/api/mstcpip/ne-mstcpip-control_channel_trigger_status) | 指定與 [**ControlChannelTrigger**](/uwp/api/Windows.Networking.Sockets.ControlChannelTrigger)搭配使用的 TCP 通訊端之 **即時 \_ \_ 通知 \_ 功能** 傳輸設定的查詢狀態，以接收 Windows 執行階段應用程式中的背景網路通知。 |
| [**eWINDOW \_ 前進 \_ 方法**](/windows/win32/api/wsrm/ne-wsrm-ewindow_advance_method) | 指定用於可靠多播的視窗預先模式。 |
| [**保證**](./guarantee-2.md) | 不再使用。 |
| [**多播 \_ 模式 \_ 類型**](/windows/win32/api/ws2ipdef/ne-ws2ipdef-multicast_mode_type) | 指定多播群組位址的篩選模式。 |
| [**NAPI \_ 提供者 \_ 層級**](/windows/win32/api/nsemail/ne-nsemail-napi_provider_level) | 為指定的網域指定 NS \_ 電子郵件命名空間提供者的提供者授權層級。 |
| [**NAPI \_ 提供者 \_ 類型**](/windows/win32/api/nsemail/ne-nsemail-napi_provider_type) | 指定命名空間提供者預期的裝載類型。 |
| [**RIO \_ 通知 \_ 完成 \_ 類型**](/windows/win32/api/mswsock/ne-mswsock-rio_notification_completion_type) | 指定當使用 Winsock 註冊的 i/o 延伸模組傳送或接收資料時，要與 [**RIONotify**](/windows/win32/api/mswsock/nc-mswsock-lpfn_rionotify) 函數搭配使用的完成佇列通知類型。 |
| [**通訊端 \_ 安全性 \_ 通訊協定**](/windows/win32/api/mstcpip/ne-mstcpip-socket_security_protocol) | 指出要在通訊端上使用的安全性通訊協定類型。 |
| [**通訊端 \_ 使用 \_ 類型**](/windows/win32/api/Mstcpip/ne-mstcpip-socket_usage_type) | 用來指定通訊端的使用類型。 |
| [**TCPSTATE**](/windows/win32/api/mstcpip/ne-mstcpip-tcpstate) | 指出傳輸控制通訊協定 (TCP) 連接的可能狀態。 |
| [**WSAECOMPARATOR**](/windows/win32/api/Winsock2/ne-winsock2-wsaecomparator) | 用於版本比較的語法。 |
| [**WSC \_ 提供者 \_ 資訊 \_ 類型**](/windows/win32/api/ws2spi/ne-ws2spi-wsc_provider_info_type) | 指定多層式服務通訊協定的資訊類別 (LSP) 。 |
