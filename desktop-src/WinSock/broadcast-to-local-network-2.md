# <a name="broadcast-to-local-network"></a>廣播到區域網路

您可以藉由將 **sa \_ netnum** 設定為二進位零 (0) ，並將 **sa \_ nodenum** 設定為二進位 (1) ，對本機連接的網路進行廣播。 此廣播可能會傳送到裝置的主要網路，或是服務提供者選項的所有本機連接網路。 路由器不會傳播至區域網路的廣播。
