# <a name="flow-specification-quality-of-service"></a>Flow規格服務品質

服務品質是透過各種系統 QoS 元件在 Windows 中執行。 如需完整的詳細資料和實行指導方針，請參閱 [服務品質](/previous-versions/windows/desktop/qos/qos-start-page)。

如需 QoS 範本的詳細資訊，請參閱 [Qos 範本](/previous-versions/aa374099(v=vs.80))。

如需服務品質的詳細資料和實行指導方針，請參閱 [**FLOWSPEC**](/windows/win32/api/qos/ns-qos-flowspec) 結構。
