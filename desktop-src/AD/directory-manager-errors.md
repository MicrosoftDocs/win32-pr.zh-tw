# <a name="directory-manager-errors"></a>目錄管理員錯誤

本主題包含 Active Directory Domain Services 中的函式之目錄管理員錯誤的傳回值。



| 值                    | 描述                                              |
|--------------------------|----------------------------------------------------------|
| **hrPMRecDeleted**       | 記錄已刪除。                                  |
| **hrRemainingVersions**  | 剩餘閒置工作。                            |
| **hrFLDKeyTooBig**       | 因為索引鍵超過255個位元組，所以已截斷。 |
| **hrFLDTooManySegments** | 索引鍵區段太多。                         |
| **hrFLDNullKey**         | 索引鍵為 **Null**。                                     |
