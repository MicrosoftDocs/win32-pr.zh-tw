# <a name="access-rights-for-active-directory-domain-services-objects"></a>Active Directory Domain Services 物件的存取權限

所有目錄物件都會使用與一般目錄作業對應的一組預先定義的存取權限。 這些許可權會在 [**ADS \_ 許可權 \_ 列舉**](/windows/win32/api/iads/ne-iads-ads_rights_enum) 列舉型別中描述。

此外，您還可以定義私用存取權限， (控制存取權) 超過預先定義之許可權所涵蓋的作業。 請參閱 [控制存取權限](control-access-rights.md)。
