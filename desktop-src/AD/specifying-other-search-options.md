# <a name="specifying-other-search-options"></a>指定其他搜尋選項

Active Directory Domain Services 支援搜尋技術所提供的大部分搜尋選項。 如需其他搜尋選項的詳細資訊，請參閱 [使用 >idirectorysearch 指定其他搜尋選項](/windows/desktop/ADSI/specifying-other-search-options-with-idirectorysearch)。
