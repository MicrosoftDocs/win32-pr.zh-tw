# <a name="nesting-a-group-in-another-group"></a>在另一個群組中嵌套群組

將群組新增為另一個群組的成員，稱為「 *嵌套*」。 針對通訊群組，混合模式和原生模式都支援嵌套。 針對安全性群組，只有在原生模式中執行的網域才支援「嵌套」。

若要在另一個群組中嵌套群組，請使用在 [定義域中將成員新增至群組](adding-members-to-groups-in-a-domain.md)所述的相同技術。 請注意，根據群組的範圍而定，群組只能包含特定類型和其他群組的範圍。

這些嵌套選項也取決於網域處於混合模式或原生模式。 如需不同網域模式中群組的詳細資訊，請參閱 [在原生模式中進行嵌套](nesting-in-native-mode.md) 和 [在混合模式](nesting-in-mixed-mode.md)中進行嵌套。

如需群組嵌套的詳細資訊，請參閱 [常見的錯誤](common-errors.md)。
