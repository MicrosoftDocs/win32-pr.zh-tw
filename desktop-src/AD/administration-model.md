# <a name="administration-model"></a>管理模型

授權使用者在 Active Directory Domain Services 中執行管理。 使用者會獲得較高的許可權，以便在目錄的某些已識別子樹中的指定物件和物件類別集合上執行一組指定的動作。 這稱為 *委派系統管理*。 委派的系統管理可讓您更精細地控制誰可以做什麼，並在不授與較高許可權的情況下啟用授權委派。
