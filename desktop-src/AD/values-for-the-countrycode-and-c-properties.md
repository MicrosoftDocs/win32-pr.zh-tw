# <a name="values-for-the-countrycode-and-c-properties"></a>CountryCode 和 c 屬性的值

**CountryCode** 和 **c** 屬性的值是從 ISO 標準3166取得的。 這項標準可在 [國際標準組織](https://www.iso.org/) 網站取得。 **CountryCode** 屬性值是 iso 3166 的數位國家/地區代碼， **c** 屬性值是 iso 3166 的兩個字母國家/地區指定。
