# <a name="managing-users-on-member-servers-and-windows-2000-professional"></a>管理成員伺服器與 Windows 2000 Professional 上的使用者

在成員伺服器與 Windows 2000 Professional 上，有一個本機安全性資料庫。 該本機安全性資料庫可以包含自己的本機使用者，其範圍只是其建立所在的特定電腦。 在成員伺服器和工作站上管理這些類型的使用者時，您會使用 WinNT 提供者。

使用 ADSI 管理 Windows 2000 網域上的使用者時，您會使用 LDAP 提供者。 在成員伺服器和工作站上管理使用者時，您會使用 WinNT 提供者。
