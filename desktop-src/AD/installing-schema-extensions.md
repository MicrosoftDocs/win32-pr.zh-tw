# <a name="installing-schema-extensions"></a>安裝架構延伸模組

架構更新會嚴格地在大部分的客戶網站上控制-這是建議的作法。 如果服務需要架構延伸，則架構延伸模組必須個別安裝。 這可讓客戶在安裝的其餘部分之前，分別執行架構更新。
