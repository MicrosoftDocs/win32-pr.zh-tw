# <a name="cfstr_ds_parenthwnd"></a>CFSTR \_ DS \_ PARENTHWND

<dl> <dt>

<span id="CFSTR_DS_PARENTHWND"></span><span id="cfstr_ds_parenthwnd"></span>**CFSTR \_ DS \_ PARENTHWND**
</dt> <dd> <dl> <dt>

"DsAdminParentHwndClipFormat"
</dt> <dt>



**CFSTR \_ DS \_ PARENTHWND** 剪貼簿格式提供全域記憶體控制碼 (**HGLOBAL**) ，其中包含主 MMC 視窗 (**HWND**) 的視窗控制碼。

> [!Note]  
> 此剪貼簿格式值未定義于已發行的標頭檔中。 若要使用此剪貼簿格式值，請使用所示的確切格式來定義它。
