# <a name="inheritance-ad-ds"></a>繼承 (AD DS) 

[*繼承*](/previous-versions/windows/desktop/legacy/ms681910(v=vs.85)) 可讓指定的 ACE 從套用至容器之所有子系的容器傳播。 您可以結合繼承與委派，在單一作業中將系統管理許可權授與目錄的整個子樹。
