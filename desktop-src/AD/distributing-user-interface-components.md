# <a name="distributing-user-interface-components"></a>散發消費者介面元件

使用者介面擴充功能所使用的 COM 元件、應用程式及（或）檔案，可透過建立 Windows Installer 套件，並使用群組原則來散發套件來散發。 如需建立安裝程式套件的詳細資訊，請參閱 [Windows Installer](/windows/desktop/Msi/windows-installer-portal)。 如需群組原則的詳細資訊，請參閱 [群組原則](/previous-versions/windows/desktop/Policy/group-policy-start-page)。
