# <a name="wmi-provider-classes-in-active-directory-domain-services"></a>Active Directory Domain Services 中的 WMI 提供者類別

Active Directory Domain Services 具有下列可與 WMI 腳本搭配使用的類別：

-   [**MSAD 的 \_ 控制器**](msad-domaincontroller.md)
-   [**MSAD \_ NamingCoNtext**](msad-namingcontext.md)
-   [**MSAD \_ ReplCursor**](msad-replcursor.md)
-   [**MSAD \_ ReplNeighbor**](msad-replneighbor.md)
-   [**MSAD \_ ReplPendingOp**](msad-replpendingop.md)

提供者本身會以名稱為 "ReplProv1" 的 [**ReplicationProvider1**](replicationprovider1.md) 類別表示。
