# <a name="creating-dynamic-objects"></a>建立動態物件

在具現化物件時，將輔助類別 **dynamicObject** 指定為 **objectClass** 屬性的值。 在此情況下，特殊輔助類別會顯示在類別清單中，此類別是物件的 **objectClass** 屬性值。 如果在加入特殊輔助類別時未提供 **entryTTL** 屬性的值，Active Directory Domain Services 將會提供預設值（如稍後所指定）。

如需將輔助類別加入至物件的詳細資訊，請參閱 [動態輔助類別](dynamic-auxiliary-classes.md)。
