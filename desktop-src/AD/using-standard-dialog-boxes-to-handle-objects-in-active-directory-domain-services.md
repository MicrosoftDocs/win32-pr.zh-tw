# <a name="using-standard-dialog-boxes-to-handle-objects-in-active-directory-domain-services"></a>使用標準對話方塊來處理 Active Directory Domain Services 中的物件

Windows 提供一般作業的通用對話方塊程式庫，例如 [開啟檔案]、[檔案流覽] 等等。 從 Windows 2000 開始，系統會提供可用於 Active Directory Domain Services 中一般使用者介面作業的對話方塊。

下列對話方塊可用於 Active Directory Domain Services：

-   [目錄物件選擇器](directory-object-picker.md)
-   [網域瀏覽器](domain-browser.md)
-   [容器瀏覽器](container-browser.md)
-   [Active Directory 消費者和電腦屬性工作表](active-directory-users-and-computers-property-sheets.md)

此外，也可以從應用程式叫用 Active Directory Domain Services 建立的嚮導。 如需詳細資訊，請參閱 [從您的應用程式叫用建立嚮導](invoking-creation-wizards-from-your-application.md)。
