# <a name="documenting-schema-extensions"></a>記載架構延伸模組

除了為架構延伸提供個別的安裝程式之外，架構延伸的本質也必須清楚記載。 檔應包含：

-   任何新類別和屬性的清單，其中包含每個類別和屬性的說明
-   以新屬性擴充的任何標準類別清單，以及屬性清單
-   所有新屬性頁面的清單，其中包含如何使用每個屬性頁的說明
-   說明服務如何使用架構延伸的說明
