# <a name="mdm_windowsadvancedthreatprotection-class"></a>MDM \_ WindowsAdvancedThreatProtection 類別

\[某些資訊與預先發行的產品有關，在正式發行之前可能會經過大幅修改。 Microsoft 對此處提供的資訊，不做任何明確或隱含的瑕疵擔保。\]

**MDM \_ WindowsAdvancedThreatProtection** 類別是用來上架及下架 Windows Defender Advanced 威脅防護 (WDATP) 的端點。

下列語法是簡化自 MOF 程式碼，且包含所有繼承的屬性。

## <a name="syntax"></a>語法

``` syntax
[InPartition("local-system"), dynamic, provider("DMWmiBridgeProv1")]
class MDM_WindowsAdvancedThreatProtection
{
  string InstanceID;
  string ParentID;
  string Onboarding;
  string Offboarding;
};
```
