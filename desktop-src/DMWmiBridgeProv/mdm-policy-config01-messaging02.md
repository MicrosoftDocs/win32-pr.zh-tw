MDM \_ Policy \_ Config01 \_ Messaging02 類別可讓文字訊息在任何地方進行備份和還原和訊息傳送。 此原則可讓組織停用這些功能，以避免將資訊儲存在其控制之外的伺服器上。

下列語法是簡化自 MOF 程式碼，且包含所有繼承的屬性。

## <a name="syntax"></a>語法

``` syntax
[InPartition("local-system"), dynamic, provider("DMWmiBridgeProv")]
class MDM_Policy_Config01_Messaging02
{
  string InstanceID;
  string ParentID;
  sint32 AllowMessageSync;
};
```
