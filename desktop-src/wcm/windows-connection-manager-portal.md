# <a name="windows-connection-manager"></a>Windows 連線管理員

## <a name="purpose"></a>目的

Windows 連線管理員 (WCM) 可讓您建立及設定連線管理員軟體。

## <a name="developer-audience"></a>開發人員對象

Windows 連線管理員是專為使用 C/c + + 開發軟體的開發人員所設計。

## <a name="run-time-requirements"></a>執行階段需求求

Windows 8 和 Windows Server 2012 都支援 Windows 連線管理員 API。
