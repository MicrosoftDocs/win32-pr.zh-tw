# <a name="windows-portable-devices"></a>Windows 可攜式裝置

Windows可攜式裝置 (WPD) 可讓電腦與連接的媒體和存放裝置進行通訊。 WPD 提供彈性且健全的方式，讓電腦與音樂播放機、存放裝置、行動電話、攝影機，以及許多其他類型的連線裝置進行通訊。 此系統會取代 Windows 媒體裝置管理員和 Windows 映像取得。

以 WPD 為基礎的應用程式可以探索裝置、傳送和接收內容，甚至控制裝置，例如拍攝圖片或傳送文字訊息。 系統設計為具有彈性，因此可以探索許多類型的裝置，並可進行擴充，讓驅動程式開發人員可以定義自訂裝置的自訂屬性和命令。

您可以使用 WPD 撰寫 Windows 的應用程式和 Web 應用程式。 您可以使用 WPD 應用程式開發介面來建立 Windows 應用程式。 這些應用程式可以用 c + +、c \# .net 或 Visual Basic .net 來撰寫。 您可以使用 WPD Automation 物件模型來撰寫 web 應用程式。 這些應用程式是以 JScript 和 HTML 來撰寫。

Windows 7、Windows Vista 和 Windows XP 作業系統都支援 WPD 應用程式設計介面。 只有 Windows 7 才支援 WPD Automation 物件模型。

如需有關建立 WPD Windows 應用程式的詳細資訊，請參閱[WPD 應用程式開發介面](./wpd_sdk/wpd-application-programming-interface.md)檔和範例。

如需有關建立 WPD Automation 應用程式的詳細資訊，請參閱 [WPD Automation 物件模型](/previous-versions/windows/desktop/legacy/dd389295(v=vs.85)) 檔和範例。
