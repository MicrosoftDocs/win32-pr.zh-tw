# <a name="animation-direct3d-9-graphics"></a> (Direct3D 9 圖形) 的動畫

包含參考先前框架的動畫。 它應該包含一個框架的參考，以及至少一組 [**AnimationKey**](animationkey.md) 範本。 它也可以包含 [**AnimationOptions**](animationoptions.md) 資料物件。

``` syntax
template Animation
{
    < 3D82AB4F-62DA-11cf-AB39-0020AF71E433 >
    [...]
} 
```

其中：

-   \[ ... \] -您可以在這裡使用任何的. x 檔案範本。 這可讓架構成為可擴充的。
