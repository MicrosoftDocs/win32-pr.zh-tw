# <a name="d3dx_channel"></a>D3DX \_ 通道

下列旗標用來指定要在材質中操作的通道。



| \#定義                 | Description                                                                   |
|--------------------------|-------------------------------------------------------------------------------|
| D3DX \_ 頻道 \_ 紅色       | 表示應該使用紅色通道。                                     |
| D3DX \_ CHANNEL \_ BLUE      | 表示應該使用藍色通道。                                    |
| D3DX \_ CHANNEL \_ 綠     | 表示應該使用綠色通道。                                   |
| D3DX \_ 頻道 \_ ALPHA     | 表示應該使用 Alpha 色板。                                   |
| D3DX \_ 通道 \_ 亮度 | 指出應該使用紅色、綠色和藍色通道的 luminances。 |
