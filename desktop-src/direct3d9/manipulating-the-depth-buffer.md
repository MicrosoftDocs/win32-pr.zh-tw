# <a name="manipulating-the-depth-buffer-direct3d-9"></a>操作 (Direct3D 9) 的深度緩衝區

深度緩衝區會與裝置相關聯。 應用程式必須在設定呈現目標時移動深度緩衝區。 [**IDirect3DDevice9：： GetDepthStencilSurface**](/windows/desktop/api)和 [**IDirect3DDevice9：： SetDepthStencilSurface**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-setdepthstencilsurface)方法是用來操作深度緩衝區。
