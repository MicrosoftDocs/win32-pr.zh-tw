# <a name="id3dxprtcompbuffergetnumsamples-method"></a>ID3DXPRTCompBuffer：： GetNumSamples 方法

抓取取樣 (或材質) 的頂點數目。

## <a name="syntax"></a>語法


```C++
UINT GetNumSamples();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **[ **UINT**](../winprog/windows-data-types.md)**

傳回取樣 (或材質) 的頂點數目。
