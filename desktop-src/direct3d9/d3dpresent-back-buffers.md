# <a name="d3dpresent_back_buffers"></a>D3DPRESENT \_ 後臺 \_ 緩衝區

指定裝置可使用的後端緩衝區數目。



| \#定義                           | 值 | 描述                                               |
|------------------------------------|-------|-----------------------------------------------------------|
| D3DPRESENT \_ 後臺 \_ 緩衝區 \_ 最大值     | 3     | Direct3D 9 中支援的背景緩衝區數目上限。   |
| D3DPRESENT \_ 回 \_ 緩衝區 \_ 最大值 \_ | 30    | Direct3D 9Ex 中支援的背景緩衝區數目上限。 |
