# <a name="id3dxprtenginegetnumverts-method"></a>ID3DXPRTEngine：： GetNumVerts 方法

抓取網格中的頂點數目，包括自我調整空間取樣之後加入的任何新頂點。

## <a name="syntax"></a>語法


```C++
UINT GetNumVerts();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **[ **UINT**](../winprog/windows-data-types.md)**

網格中的頂點數目，包括因適應性取樣而加入的任何新頂點。
