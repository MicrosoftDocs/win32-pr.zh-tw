# <a name="matrix4x4"></a>Matrix4x4

定義 4 x 4 矩陣。 這會用來做為框架轉換矩陣。

``` syntax
template Matrix4x4
{
    < F6F23F45-7686-11cf-8F52-0040333594A3 >
    array float matrix[16];
} 
```

其中：

-   陣列 float 矩陣 \[ 16 \] -16 浮點數的陣列。
