# <a name="id3dxprtbufferistexture-method"></a>ID3DXPRTBuffer：： IsTexture 方法

指出緩衝區是否包含材質。

## <a name="syntax"></a>語法


```C++
BOOL IsTexture();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **[ **BOOL**](../winprog/windows-data-types.md)**

如果緩衝區包含材質，則傳回 **TRUE** ;否則傳回 **FALSE** 。
