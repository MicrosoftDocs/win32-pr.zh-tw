# <a name="d3dconvolutionmono"></a>D3DCONVOLUTIONMONO

指定紋理數量詞篩選的最大維度。



|                               |       |                                                            |
|-------------------------------|-------|------------------------------------------------------------|
| \#定義                      | 值 | 描述                                                |
| D3DCONVOLUTIONMONO \_ MAXHEIGHT | 7     | 單色紋理卷積濾波器的最大高度。 |
| D3DCONVOLUTIONMONO \_ MAXWIDTH  | 7     | 單色紋理卷積濾波器的最大寬度。  |
