# <a name="id3dxfontgetdc-method"></a>ID3DXFont：： GetDC 方法

傳回已設定字型 (DC) 的顯示裝置內容控制碼。

## <a name="syntax"></a>語法


```C++
HDC GetDC();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **[ **HDC**](../winprog/windows-data-types.md)**

顯示 DC 的控制碼。
