# <a name="id3dxanimationcontrollergetmaxnumanimationsets-method"></a>ID3DXAnimationController：： GetMaxNumAnimationSets 方法

取得動畫控制器可支援的動畫集合數目上限。

## <a name="syntax"></a>語法


```C++
UINT GetMaxNumAnimationSets();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **[ **UINT**](../winprog/windows-data-types.md)**

控制器可管理的動畫組數目上限。
