# <a name="states-direct3d-9"></a>狀態 (Direct3D 9) 

Direct3D 裝置會使用狀態來設定所有管線處理。 取樣器狀態會藉由定義材質定址和篩選模式來控制紋理取樣器的設定。 材質狀態：設定多材質 blender，以控制如何結合和混合材質紋理。 轉譯狀態會控制大部分的管線狀態。

狀態欄塊是儲存和還原狀態的便利機制。 請參閱 [狀態欄塊儲存和還原狀態 (Direct3D 9) ](state-blocks-save-and-restore-state.md)
