# <a name="d3dissue_begin"></a>D3DISSUE \_ 開始

此宏會建立 [**問題**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3dquery9-issue) 用來發出查詢開始的值。

``` syntax
#define D3DISSUE_BEGIN (1 << 1)
```

## <a name="remarks"></a>備註

D3DISSUE \_ BEGIN 對下列查詢類型有效。

-   D3DQUERYTYPE \_ 遮蔽
