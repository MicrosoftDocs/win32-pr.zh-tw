# <a name="id3dxconstanttablegetbuffersize-method"></a>ID3DXConstantTable：： GetBufferSize 方法

取得常數資料表的緩衝區大小。

## <a name="syntax"></a>語法


```C++
DWORD GetBufferSize();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **[ **DWORD**](../winprog/windows-data-types.md)**

傳回緩衝區的大小（以位元組為單位）。
