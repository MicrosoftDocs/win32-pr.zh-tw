# <a name="id3dxbasemeshgetnumvertices-method"></a>ID3DXBaseMesh：： GetNumVertices 方法

抓取網格中的頂點數目。

## <a name="syntax"></a>語法


```C++
DWORD GetNumVertices();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **[ **DWORD**](../winprog/windows-data-types.md)**

傳回網格中的頂點數目。
