# <a name="id3dxprtcompbuffergetwidth-method"></a>ID3DXPRTCompBuffer：： GetWidth 方法

抓取紋理的寬度（以圖元為單位）。

## <a name="syntax"></a>語法


```C++
UINT GetWidth();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **[ **UINT**](../winprog/windows-data-types.md)**

傳回材質的寬度（以圖元為單位）。
