# <a name="dxfile-constants"></a>DXFILE 常數

下列旗標用來指定要在材質中操作的通道。 已取代。

## <a name="dxfileformat"></a>DXFILEFORMAT



|                          |       |                  |
|--------------------------|-------|------------------|
| \#定義                 | 值 | 描述      |
| DXFILEFORMAT \_ 二進位檔     | 0     | 二進位檔案。     |
| DXFILEFORMAT \_ 文字       | 1     | 文字檔。       |
| DXFILEFORMAT \_ 壓縮 | 2     | 壓縮檔案。 |



 

這些 \# 定義是在 Dxfile 中宣告。

## <a name="dxfileloadoptions"></a>DXFILELOADOPTIONS



|                          |       |                              |
|--------------------------|-------|------------------------------|
| \#定義                 | 值 | 描述                  |
| DXFILELOAD \_ FROMFILE     | 0x00L | 從檔案載入檔案。     |
| DXFILELOAD \_ FROMRESOURCE | 0x01L | 從資源載入檔案。 |
| DXFILELOAD \_ FROMMEMORY   | 0x02L | 從記憶體載入檔案。     |
| DXFILELOAD \_ FROMSTREAM   | 0x04L | 從資料流程載入檔案。   |
| DXFILELOAD \_ FROMURL      | 0x08L | 從 URL 載入檔案。      |



 

這些 \# 定義是在 Dxfile 中宣告。
