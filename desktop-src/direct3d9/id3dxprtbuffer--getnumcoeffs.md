# <a name="id3dxprtbuffergetnumcoeffs-method"></a>ID3DXPRTBuffer：： GetNumCoeffs 方法

抓取記憶體中用來儲存範例的每個顏色通道的純量數目。

## <a name="syntax"></a>語法


```C++
UINT GetNumCoeffs();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **[ **UINT**](../winprog/windows-data-types.md)**

傳回記憶體中用來儲存範例的每個顏色通道的純量數目。
