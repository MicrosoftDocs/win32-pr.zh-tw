# <a name="direct3d-textures-direct3d-9"></a>Direct3d 材質 (Direct3D 9) 

紋理是在電腦產生 3D 影像中創造真實感的利器。 Direct3D 支援廣泛的紋理功能設定，提供開發人員輕鬆存取進階紋理技術。

本節將協助您開始使用紋理。

-   [ (Direct3D 9) 的基本紋理概念 ](basic-texturing-concepts.md)
-   [ (Direct3D 9) 的材質座標 ](texture-coordinates.md)
-   [ (Direct3D 9) 的材質篩選 ](texture-filtering.md)
-   [ (Direct3D 9) 的材質資源 ](texture-resources.md)
-   [ (Direct3D 9) 的材質換行 ](texture-wrapping.md)
-   [ (Direct3D 9) 的材質混合 ](texture-blending.md)

這些主題將詳細說明其他紋理功能。

-   [自動產生 Mipmap (Direct3D 9) ](automatic-generation-of-mipmaps.md)
-   [自動材質管理 (Direct3D 9) ](automatic-texture-management.md)
-   [ (Direct3D 9) 的壓縮材質資源 ](compressed-texture-resources.md)
-   [ (Direct3D 9) 進行紋理的硬體考慮 ](hardware-considerations-for-texturing.md)
-   [ (Direct3D 9) 的音量材質資源 ](volume-texture-resources.md)

若要改善效能，請考慮使用動態紋理。 動態紋理可以在每個影格中鎖定、寫入，或是解鎖。 請參閱 [使用動態紋理](performance-optimizations.md)。
