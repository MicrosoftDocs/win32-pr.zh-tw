# <a name="id3dxbuffergetbufferpointer-method"></a>ID3DXBuffer：： GetBufferPointer 方法

## <a name="syntax"></a>語法


```C++
LPVOID GetBufferPointer();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **[ **LPVOID**](../winprog/windows-data-types.md)**

傳回緩衝區中資料的指標。
