# <a name="fog-types-direct3d-9"></a> (Direct3D 9) 的霧化類型

有兩種方式可以在場景中加入霧化：使用圖元霧化和/或頂點霧化。 下列主題說明用於霧化方的公式，以及如何執行頂點和圖元霧化。 應用程式可以使用頂點著色器和圖元霧化（如有需要）來執行霧化。

-   [霧化公式](fog-formulas.md)
-   [霧化參數](fog-parameters.md)
-   [霧化混色](fog-blending.md)
-   [霧化色彩](fog-color.md)
-   [頂點霧化](vertex-fog.md)
-   [圖元霧化](pixel-fog.md)

霧化混色由轉譯狀態控制;它不是可程式化的圖元管線的一部分。
