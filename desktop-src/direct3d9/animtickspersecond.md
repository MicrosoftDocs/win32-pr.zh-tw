# <a name="animtickspersecond"></a>AnimTicksPerSecond

描述動畫比例因數。

``` syntax
template AnimTicksPerSecond
{
    < 9E415A43-7BA6-4a73-8743-B73D47E88476 >
    DWORD AnimTicksPerSecond;
} 
```

其中：

-   AnimTicksPerSecond：將動畫時間戳記轉換為通用時間的縮放比例。 它的值會將動畫時間除以30、60、24或25。
