# <a name="fire-flares-and-explosions-direct3d-9"></a>)  (Direct3D 9 的火災、光暈和爆炸

您可以使用 Microsoft Direct3D 來模擬涉及能源的自然現象。 例如，應用程式可以將類似火焰的材質套用至一組公告欄，以產生火災的外觀。 如果應用程式使用一連串的火材質，以在火災中的每個佈告欄上建立點火的動畫效果，這就特別有效。 將動畫播放的速度從佈告欄切換至佈告欄會增加實際點火的外觀。 混合3D 點火的類似可透過將公告欄上的公告欄和材質分層來達成。

您可以藉由將連續亮亮的燈光對應套用至場景中的所有基本型別，來模擬光暈和閃爍。 雖然這是一種計算高度負荷的技術，但它可讓您的應用程式模擬當地語系化的光暈或 flash。 也就是說，flare 或 flash 源自的場景部分，會先使其變亮。

另一種方法是在場景前方放置佈告欄，以涵蓋整個轉譯目的地區域。 應用程式會將連續的 whiter 紋理套用至佈告欄，並隨著時間減少透明度。 整個場景會隨著時間的推移淡化為白色。 這是建立光暈的低負擔方法。 不過，使用這項技術時，可能很難從單一點光源來源產生亮閃 flash 的外觀。

爆炸可以顯示在類似于引發、閃爍和光暈的3D 場景程式中。 比方說，您的應用程式可能會使用佈告欄來顯示震動波，並在發生爆炸時增加 plume。 同時，您的應用程式可以使用一組公告欄來模擬點火。 此外，它也可以將單一佈告欄放置在場景前方，以將燈光的光暈新增至整個場景。

您可以使用公告欄來模擬能源字形狀。 您的應用程式也可以使用定義為行清單或換行的基本專案來顯示它們。 如需詳細資訊，請參閱 [行清單](line-lists.md) 和 [行條紋](line-strips.md)。

您的應用程式可以使用定義為三角形清單的公告欄或基本專案來建立強制欄位。 若要從三角形清單建立強制欄位，請在三角形清單中定義一組連續的三角形，而這些三角形會在 [強制] 欄位所涵蓋的區域上平均分佈。 三角形之間的間隙可讓使用者查看三角形背後的場景，如您在查看 [force] 欄位時所預期的情況。 將材質套用至三角形清單，讓三角形成為具有能源發光的外觀。 如需詳細資訊，請參閱 [三角形清單](triangle-lists.md)。
