## <a name="other-constants"></a>其他常數

| \#定義                              | 值                                             | 描述                                                        |
|---------------------------------------|---------------------------------------------------|--------------------------------------------------------------------|
| D3D \_ 最大 \_ 同時 \_ RENDERTARGETS | 4                                                 | Rendertargets 數目上限。                               |
| D3DDMAPSAMPLER                        | 256                                               | 置換地圖樣本的最大數目。                    |
| D3DDP \_ MAXTEXCOORD                    | 8                                                 | 材質座標的最大數目。                         |
| MAXD3DDECLLENGTH                      | 64 (不包含 "end" 標記頂點元素)  | 頂點宣告中的元素數目上限。                |
| MAXD3DDECLUSAGEINDEX                  | 15                                                | 可以在頂點宣告中使用的最大索引 (0-15) 。 |



 

這些 \# 定義是在 d3d9types 中宣告。
