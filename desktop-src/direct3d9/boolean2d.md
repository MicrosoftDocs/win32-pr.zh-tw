# <a name="boolean2d"></a>Boolean2d

定義一組兩個布林值，用於 [**MeshFaceWraps**](meshfacewraps.md) 範本以定義個別臉部的材質拓撲。

``` syntax
template Boolean2d
{
    < 4885AE63-78E8-11cf-8F52-0040333594A3 >
    Boolean u;
    Boolean v;
} 
```

其中：

-   u-布林值。 請參閱 [**布林值**](boolean.md)。
-   v 布林值。 請參閱 [**布林值**](boolean.md)。
