# <a name="id3dxprtenginefreessdata-method"></a>ID3DXPRTEngine：： FreeSSData 方法

釋出用於暫存地下輕量模擬資料的記憶體。

## <a name="syntax"></a>語法


```C++
HRESULT FreeSSData();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **[ **HRESULT**](https://msdn.microsoft.com/library/Bb401631(v=MSDN.10).aspx)**

傳回值為「D3D \_ 正常」。
