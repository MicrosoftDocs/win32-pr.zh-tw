# <a name="id3dxskininfogetminboneinfluence-method"></a>ID3DXSkinInfo：： GetMinBoneInfluence 方法

取得最小的骨骼影響。 影響小於此值的值會被忽略。

## <a name="syntax"></a>語法


```C++
FLOAT GetMinBoneInfluence();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **[ **FLOAT**](../winprog/windows-data-types.md)**

傳回最小的骨骼影響值。
