# <a name="id3dxlinegetgllines-method"></a>ID3DXLine：： GetGLLines 方法

取得 OpenGL 樣式的線條繪圖模式。

## <a name="syntax"></a>語法


```C++
BOOL GetGLLines();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **[ **BOOL**](../winprog/windows-data-types.md)**

如果已啟用 OpenGL 樣式的行，則傳回 **TRUE** ; 如果啟用 Direct3D 樣式的行，則傳回 **FALSE** 。
