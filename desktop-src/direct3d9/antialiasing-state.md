# <a name="antialiasing-state-direct3d-9"></a> (Direct3D 9) 的消除鋸齒狀態

[ (Direct3D 9) 的消除鋸齒 ](antialiasing.md) 是讓線條和邊緣在螢幕上顯得更平滑的方法。 根據預設，Direct3D 不會執行消除鋸齒。 若要啟用全場景消除鋸齒，請將 D3DRS \_ MULTISAMPLEANTIALIAS 轉譯狀態設為 **TRUE**。 若要停用，請將 D3DRS \_ MULTISAMPLEANTIALIAS 設定為 **FALSE**。
