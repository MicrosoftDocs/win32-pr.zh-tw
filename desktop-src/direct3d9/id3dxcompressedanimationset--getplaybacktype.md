# <a name="id3dxcompressedanimationsetgetplaybacktype-method"></a>ID3DXCompressedAnimationSet：： GetPlaybackType 方法

取得動畫集合播放迴圈的型別。

## <a name="syntax"></a>語法


```C++
D3DXPLAYBACK_TYPE GetPlaybackType();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **[ **D3DXPLAYBACK \_ 類型**](./d3dxplayback-type.md)**

動畫集合播放迴圈的型別。 請參閱 [**D3DXPLAYBACK \_ 類型**](./d3dxplayback-type.md)。
