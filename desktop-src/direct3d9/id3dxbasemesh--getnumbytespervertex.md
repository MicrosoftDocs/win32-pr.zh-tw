# <a name="id3dxbasemeshgetnumbytespervertex-method"></a>ID3DXBaseMesh：： GetNumBytesPerVertex 方法

取得每個頂點的位元組數目。

## <a name="syntax"></a>語法


```C++
DWORD GetNumBytesPerVertex();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **[ **DWORD**](../winprog/windows-data-types.md)**

傳回每個頂點的位元組數目。
