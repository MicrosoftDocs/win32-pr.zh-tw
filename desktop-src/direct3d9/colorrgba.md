# <a name="colorrgba"></a>ColorRGBA

使用 Alpha 元件定義 color 物件。 這會用於材質範本定義中的臉部色彩。

``` syntax
template ColorRGBA
{
    < 35FF44E0-6C7C-11cf-8F52-0040333594A3 >
    float red;
    float green;
    float blue;
    float alpha;
} 
```
