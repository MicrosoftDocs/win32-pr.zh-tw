# <a name="d3dclear"></a>D3DCLEAR

這些旗標會識別呼叫 [**Clear**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-clear)時要重設的介面。



|                   |                                                                                                                                             |
|-------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| \#定義          | Description                                                                                                                                 |
| D3DCLEAR \_ 樣板 | 清除樣板緩衝區。                                                                                                                   |
| D3DCLEAR \_ 目標  | 清除轉譯目標或多重轉譯目標中的所有目標。 請參閱 [ (Direct3D 9) 的多重轉譯目標 ](multiple-render-targets.md)。 |
| D3DCLEAR \_ ZBUFFER | 清除深度緩衝區。                                                                                                                     |
