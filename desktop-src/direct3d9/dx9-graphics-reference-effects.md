# <a name="effect-reference-direct3d-9-graphics"></a> (Direct3D 9 圖形) 的效果參考

效果是較高層級的抽象概念，可利用頂點和圖元著色器來封裝轉譯效果。 您可以使用高層級語言或著色器元件語言來撰寫效果。

API 主題如下。

-   [效果介面](dx9-graphics-reference-effects-interfaces.md)
-   [效果函數](dx9-graphics-reference-effects-functions.md)
-   [效果結構](dx9-graphics-reference-effects-structures.md)
-   [效果常數](dx9-graphics-reference-effects-constants.md)
-   [ (Direct3D 9) 的效果格式 ](dx9-graphics-reference-effects-file-format.md)
-   [DirectX 標準注釋和語義參考](dx9-graphics-reference-effects-dxsas.md)

您也可以使用命令列編譯工具，做為使用 API 建立效果的替代方案。 請參閱 [效果編譯器工具](../direct3dtools/fxc.md)。
