# <a name="id3dxtexturegutterhelpergetheight-method"></a>ID3DXTextureGutterHelper：： GetHeight 方法

抓取紋理的高度（以圖元為單位）。

## <a name="syntax"></a>語法


```C++
UINT GetHeight();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **[ **UINT**](../winprog/windows-data-types.md)**

傳回紋理的高度（以圖元為單位）。
