# <a name="id3dxbuffergetbuffersize-method"></a>ID3DXBuffer：： GetBufferSize 方法

抓取緩衝區中資料的總大小。

## <a name="syntax"></a>語法


```C++
DWORD GetBufferSize();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **[ **DWORD**](../winprog/windows-data-types.md)**

傳回緩衝區中資料的大小總計（以位元組為單位）。
