# <a name="id3dxlinegetantialias-method"></a>ID3DXLine：： GetAntialias 方法

取得線條消除鋸齒狀態。

## <a name="syntax"></a>語法


```C++
BOOL GetAntialias();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **[ **BOOL**](../winprog/windows-data-types.md)**

傳回消除鋸齒參數值。 **TRUE** 表示消除鋸齒是開啟的， **FALSE** 表示消除鋸齒功能。
