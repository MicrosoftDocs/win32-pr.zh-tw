# <a name="x-files-legacy-direct3d-9"></a>X 檔案 (舊版)  (Direct3D 9) 

X 檔案格式參考副檔名為 X 的檔案。 X 檔案是透過 DirectX 2.0 引進。 此格式的二進位版本後續是與 DirectX 3.0 一起發行，本檔也有相關說明。 DirectX 6.0 引進了介面和方法，可讓您讀取和寫入. x 檔案。

X 檔案提供範本驅動的格式，可讓您儲存網格、紋理、動畫和使用者可定義的物件。 動畫集合的支援可讓您儲存預先定義的路徑以供即時播放。 也支援實例和階層。 實例可啟用多個物件的參考，例如網格，而且每個檔案只會儲存一次其資料。 階層可用來表示資料記錄之間的關聯性。

. X 檔案格式提供低層級的資料基本資料，讓應用程式透過範本來定義較高層級的基本專案。

以謹慎的 3ds max 或 Alias \| Wavefront 的 Maya 應用程式建立的立體模型，可以轉換成具有適用于別名 Maya 之 DirectX 延伸模組的. x 檔案。
