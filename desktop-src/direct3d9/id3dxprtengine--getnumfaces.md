# <a name="id3dxprtenginegetnumfaces-method"></a>ID3DXPRTEngine：： GetNumFaces 方法

抓取網格中的臉部數目，包括自調適型空間取樣的結果所加入的任何新臉部。

## <a name="syntax"></a>語法


```C++
UINT GetNumFaces();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **[ **UINT**](../winprog/windows-data-types.md)**

網格中的臉部數目，包括因適應性取樣而加入的任何新臉部。
