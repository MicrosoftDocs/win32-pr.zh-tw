# <a name="d3dxcolorscale-function"></a>D3DXColorScale 函式

調整色彩值。

## <a name="syntax"></a>語法


```C++
D3DXCOLOR* D3DXColorScale(
  _Inout_       D3DXCOLOR *pOut,
  _In_    const D3DXCOLOR *pC,
  _In_          FLOAT     s
);
```



## <a name="parameters"></a>參數

<dl> <dt>

*不悅* \[in、out\]
</dt> <dd>

類型： **[ **D3DXCOLOR**](d3dxcolor.md)\***

[**D3DXCOLOR**](d3dxcolor.md)結構的指標，該結構為運算的結果。

</dd> <dt>

*電腦* \[在\]
</dt> <dd>

Type： **Const [**D3DXCOLOR**](d3dxcolor.md) \***

來源 [**D3DXCOLOR**](d3dxcolor.md) 結構的指標。

</dd> <dt>

 \[ 中的 s\]
</dt> <dd>

類型： **[ **FLOAT**](../winprog/windows-data-types.md)**

比例因數。 它會調整色彩，例如，將它視為4D 向量。 的值沒有限制。 如果 s 是1，則產生的色彩會是原始色彩。

</dd> </dl>

## <a name="return-value"></a>傳回值

類型： **[ **D3DXCOLOR**](d3dxcolor.md)\***

此函式會傳回 [**D3DXCOLOR**](d3dxcolor.md) 結構的指標，此結構是縮放的色彩值。

## <a name="remarks"></a>備註

此函式的傳回值與不悅參數中所傳回的值相同。 如此一來， **D3DXColorScale** 函式就可以用來做為另一個函式的參數。

此函式會將 [**D3DXCOLOR**](d3dxcolor.md) 結構的色彩元件乘以指定的縮放比例，以計算縮放的色彩值，如下列範例所示。


```
pOut->r = pC->r * s;
```
