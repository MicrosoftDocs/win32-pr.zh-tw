定義兩個維度向量，用來定義網格的 (u，v) 材質座標。

``` syntax
template Coords2d
{
    < F6F23F44-7686-11cf-8F52-0040333594A3 >
    float u;
    float v;
} 
```

-   u-u 座標值。
-   v 座標值。
