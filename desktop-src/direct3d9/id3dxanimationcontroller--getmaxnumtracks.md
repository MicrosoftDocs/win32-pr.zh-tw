# <a name="id3dxanimationcontrollergetmaxnumtracks-method"></a>ID3DXAnimationController：： GetMaxNumTracks 方法

取得動畫控制器中的曲目數目上限。

## <a name="syntax"></a>語法


```C++
UINT GetMaxNumTracks();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **[ **UINT**](../winprog/windows-data-types.md)**

曲目數目。

## <a name="remarks"></a>備註

控制器可管理的最大追蹤數目。
