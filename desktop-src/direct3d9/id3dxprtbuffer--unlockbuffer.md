# <a name="id3dxprtbufferunlockbuffer-method"></a>ID3DXPRTBuffer：： UnlockBuffer 方法

結束 [**ID3DXPRTBuffer：： LockBuffer**](id3dxprtbuffer--lockbuffer.md)所傳回之 ppData 指標的存留期。

## <a name="syntax"></a>語法


```C++
HRESULT UnlockBuffer();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **[ **HRESULT**](https://msdn.microsoft.com/library/Bb401631(v=MSDN.10).aspx)**

如果方法成功，則傳回值為「D3D \_ 正常」。
