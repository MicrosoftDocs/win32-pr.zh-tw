# <a name="id3dxanimationcontrollergetmaxnumanimationoutputs-method"></a>ID3DXAnimationController：： GetMaxNumAnimationOutputs 方法

取得動畫控制器可支援的動畫輸出數目上限。

## <a name="syntax"></a>語法


```C++
UINT GetMaxNumAnimationOutputs();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **[ **UINT**](../winprog/windows-data-types.md)**

控制器可管理的動畫輸出數目上限。
