# <a name="d3dcursorcaps"></a>D3DCURSORCAPS

驅動程式資料指標功能旗標。



|                       |             |                                                                                                 |
|-----------------------|-------------|-------------------------------------------------------------------------------------------------|
| \#定義              | 值       | 描述                                                                                     |
| D3DCURSORCAPS \_ 色彩  | 0x00000001L | 驅動程式支援至少高解析度模式的硬體顏色游標 (高度 >= 400) 。 |
| D3DCURSORCAPS \_ LOWRES | 0x00000002L | 此驅動程式支援低解析度模式的硬體顏色游標 (高度 < 400) 。            |
