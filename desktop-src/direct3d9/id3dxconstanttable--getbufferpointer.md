# <a name="id3dxconstanttablegetbufferpointer-method"></a>ID3DXConstantTable：： GetBufferPointer 方法

取得包含常數資料表之緩衝區的指標。

## <a name="syntax"></a>語法


```C++
LPVOID GetBufferPointer();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **[ **LPVOID**](../winprog/windows-data-types.md)**

傳回緩衝區的指標。
