# <a name="working-with-device-windows-direct3d-9"></a>使用裝置 Windows (Direct3D 9) 

本節列出在 Direct3D 應用程式中使用裝置視窗時，可能會遇到的問題。

-   Direct3D 只會使用 Direct3D 訊息處理函式來連結焦點視窗，而不是裝置視窗，而且只會處理焦點視窗訊息。 因此，焦點視窗應該是任何裝置視窗的父系。
