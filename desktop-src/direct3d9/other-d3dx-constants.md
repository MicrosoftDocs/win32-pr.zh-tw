# <a name="other-d3dx-constants"></a>其他 D3DX 常數

其他 D3DX 常數包含下列各項：

## <a name="16-bit-floating-point-numbers"></a>16位浮點數



| \#定義                | Description                             |
|-------------------------|-----------------------------------------|
| D3DX \_ 16F \_ 深入探討          | 十進位位數精確度   |
| D3DX \_ 16F \_ EPSILON      | 最小，因此 1.0 + epsilon！ = 1。0 |
| D3DX \_ 16F \_ MANT \_ 發掘    | 尾數中的位元數              |
| D3DX \_ 16F \_ MAX          | 最大值                           |
| D3DX \_ 16F \_ MAX \_ 10 \_ EXP | 最大十進位指數                |
| D3DX \_ 16F \_ MAX \_ EXP     | 最大二進位指數                 |
| D3DX \_ 16F \_ MIN          | 最小正值                  |
| D3DX \_ 16F \_ MIN \_ 10 \_ EXP | 最小十進位指數                |
| D3DX \_ 16F \_ 最小 \_ EXP     | 最小二進位指數                 |
| D3DX \_ 16F \_ 基數        | 指數基數                          |
| D3DX \_ 16F \_ 四捨五入       | 加法四捨五入︰接近                 |
| D3DX \_ 1BYPI             | 1/pi                                    |
| D3DX \_ PI                | pi                                      |
| D3DX \_ 預設 \_ 浮點數    | 最大浮點數值                     |



 

這些 \# 定義是在 d3dx9 和 d3dx9math 中宣告的。

## <a name="prt-constants"></a>PRT 常數



| \#定義         | Description                                |
|------------------|--------------------------------------------|
| D3DXSH \_ MINORDER | 模擬的最低允許順序。  |
| D3DXSH \_ MAXORDER | 模擬的最高允許順序。 |



 

這些 \# 定義是在 d3dx9math 中宣告。 如需 PRT 的詳細資訊，請參閱 [預先計算 Radiance 傳輸 (Direct3D 9) ](precomputed-radiance-transfer.md)。

## <a name="texture-constants"></a>材質常數



| \#定義               | Description                                                        |
|------------------------|--------------------------------------------------------------------|
| 從檔案 D3DFMT \_ \_     | 從檔案完全採用格式。                               |
| D3DX \_ 預設值          | 預設值。                                                   |
| D3DX \_ 預設 \_ NONPOW2 | 請勿將寬度或高度等數位四捨五入為二的乘冪。 |
| 從檔案 D3DX \_ \_       | 從檔案完全採用材質維度。                   |



 

這些 \# 定義是在 d3dx9 中宣告。

## <a name="other-d3dx-constants"></a>其他 D3DX 常數

在 D3dx9mesh 中定義的 UNUSED16 和 D3DX \_ 版本常數會在內部使用。 請勿使用這些常數。
