# <a name="id3dxpatchmeshgetoptions-method"></a>ID3DXPatchMesh：： GetOptions 方法

取得修補程式的類型。

## <a name="syntax"></a>語法


```C++
DWORD GetOptions();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **[ **DWORD**](../winprog/windows-data-types.md)**

修補類型。

## <a name="remarks"></a>備註

如需修補程式類型的詳細資訊，請參閱 [**D3DXPATCHMESHTYPE**](./d3dxpatchmeshtype.md)。
