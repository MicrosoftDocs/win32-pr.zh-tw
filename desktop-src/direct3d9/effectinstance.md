# <a name="effectinstance"></a>EffectInstance

定義效果實例。

``` syntax
template EffectInstance
{
    < E331F7E4-0559-4cc2-8E99-1CEC1657928F >
    STRING EffectFilename;
    [ ... ]
} 
```

其中：

-   EffectFilename 效果檔案名。
-   \[ ... \] -您可以在這裡使用任何的. x 檔案範本。 這可讓架構成為可擴充的。
