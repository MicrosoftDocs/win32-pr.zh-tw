# <a name="d3dcomposerects_maxnumrects"></a>D3DCOMPOSERECTS \_ MAXNUMRECTS

指定用來在 [**ComposeRects**](/windows/desktop/api/d3d9/nf-d3d9-idirect3ddevice9ex-composerects)中撰寫字元的矩形數目上限。



| \#定義                     | Description                                                                                                                          |
|------------------------------|--------------------------------------------------------------------------------------------------------------------------------------|
| D3DCOMPOSERECTS \_ MAXNUMRECTS | 要在文字字串中一起組合的矩形字元數上限。 請參閱 [**ComposeRects**](/windows/desktop/api/d3d9/nf-d3d9-idirect3ddevice9ex-composerects)。 |
