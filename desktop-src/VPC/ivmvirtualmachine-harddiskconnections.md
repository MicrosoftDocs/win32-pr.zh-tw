# <a name="ivmvirtualmachineharddiskconnections-property"></a>IVMVirtualMachine：： HardDiskConnections 屬性

\[WindowsVirtual PC 不再適用于 Windows 8。 請改為使用 [HYPER-V WMI 提供者 (V2) ](/windows/desktop/HyperV_v2/windows-virtualization-portal)。\]

抓取硬碟連接的可列舉集合。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```C++
HRESULT get_HardDiskConnections(
  [out, retval] IVMHardDiskConnectionCollection **hardDiskConnectionCollection
);
```



## <a name="property-value"></a>屬性值

[**IVMHardDiskConnectionCollection**](ivmharddiskconnectioncollection.md)物件。
