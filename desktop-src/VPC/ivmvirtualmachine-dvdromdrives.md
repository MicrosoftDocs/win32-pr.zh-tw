# <a name="ivmvirtualmachinedvdromdrives-property"></a>IVMVirtualMachine：:D VDROMDrives 屬性

\[WindowsVirtual PC 不再適用于 Windows 8。 請改為使用 [HYPER-V WMI 提供者 (V2) ](/windows/desktop/HyperV_v2/windows-virtualization-portal)。\]

抓取連接到虛擬機器之 CD 和 DVD 光碟機的可列舉集合。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```C++
HRESULT get_DVDROMDrives(
  [out, retval] IVMDVDDriveCollection **dvdROMCollection
);
```



## <a name="property-value"></a>屬性值

[**IVMDVDDriveCollection**](ivmdvddrivecollection.md)物件。
