# <a name="windows-virtual-pc"></a>Windows Virtual PC

\[WindowsVirtual PC 不再適用于 Windows 8。 請改為使用 [HYPER-V WMI 提供者 (V2) ](../hyperv_v2/windows-virtualization-portal.md)。\]

## <a name="purpose"></a>目的

本檔提供 Windows Virtual PC COM 介面的相關資訊。

## <a name="where-applicable"></a>適用時

WindowsVirtual PC 是最新的 Microsoft 虛擬化技術;它可讓您直接從 Windows 7，在虛擬 Windows 環境上執行許多生產力應用程式。 您可以在 Windows 7 桌上型電腦上建立個別的虛擬機器。 每部虛擬機器都是在獨立的獨立軟體環境中，模擬一個完整的硬體系統（從處理器到網路卡），以啟用其他不相容系統的平行作業。

## <a name="developer-audience"></a>開發人員對象

Windows virtual PC COM 介面適用于建立用戶端應用程式的開發人員，以自動化虛擬機器的部署和操作。

## <a name="run-time-requirements"></a>執行階段需求求

WindowsVirtual PC 需要下列其中一項 Windows 7 版本： home Basic、home 進階版、Professional、旗艦版或 Enterprise 版。

若要取得相關的標頭檔和程式庫檔案，請從[Microsoft 下載中心](https://www.microsoft.com/download/details.aspx?id=8279)安裝 Windows 7 的 Windows SDK。
