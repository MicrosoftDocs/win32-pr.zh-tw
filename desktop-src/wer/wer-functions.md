# <a name="wer-functions"></a>WER 函數

Windows 錯誤報告中使用下列函數。 如需使用這些函式的範例，請參閱 WinBase \\ WindowsErrorReporting 和 WinBase \\ Debug FailFast 的 Windows SDK 中的範例 \\ 。

## <a name="in-this-section"></a>本節內容



| 主題                                                                                                               | 描述                                                                                                                                                                                                                                                                                                                        |
|---------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AddERExcludedApplication**](/windows/desktop/api/ErrorRep/nf-errorrep-adderexcludedapplicationa)<br/>                                             | 從錯誤報表中排除指定的應用程式。<br/>                                                                                                                                                                                                                                                                |
| [*PFN \_ WER \_ 執行時間 \_ 例外狀況 \_ 事件*](/windows/desktop/api/Werapi/nc-werapi-pfn_wer_runtime_exception_event)<br/>                          | WER 會呼叫這個函式，以判斷例外狀況處理常式是否宣稱損毀。<br/>                                                                                                                                                                                                                               |
| [*PFN \_ WER \_ 執行時間 \_ 例外狀況 \_ 偵錯工具 \_ 啟動*](/windows/desktop/api/Werapi/nc-werapi-pfn_wer_runtime_exception_debugger_launch)<br/> | WER 會呼叫這個函式，讓您自訂偵錯工具啟動選項和啟動字串。<br/>                                                                                                                                                                                                                             |
| [*PFN \_ WER \_ 執行時間 \_ 例外狀況 \_ 事件 \_ 簽章*](/windows/desktop/api/Werapi/nc-werapi-pfn_wer_runtime_exception_event_signature)<br/>      | WER 可以多次呼叫此函數，以取得可唯一描述問題的報表參數。<br/>                                                                                                                                                                                                              |
| [**ReportFault**](/windows/desktop/api/ErrorRep/nf-errorrep-reportfault)<br/>                                                                       | 啟用可執行自己的例外狀況處理的應用程式，以向 Microsoft 報告錯誤。<br/>                                                                                                                                                                                                                          |
| [**WerAddExcludedApplication**](/windows/desktop/api/Werapi/nf-werapi-weraddexcludedapplication)<br/>                                           | 將指定的應用程式新增至要從錯誤報表中排除的應用程式清單。<br/>                                                                                                                                                                                                                |
| [**WerFreeString**](/windows/desktop/api/Werapi/nf-werapi-werfreestring)<br/>                                                                   | 釋出用來儲存報表索引鍵字串的記憶體。 一旦使用了特定的報表索引鍵字串且不再需要時，應該在每次後續呼叫 [**WerStoreGetFirstReportKey**](/windows/desktop/api/Werapi/nf-werapi-werstoregetfirstreportkey) 或 [**WerStoreGetNextReportKey**](/windows/desktop/api/Werapi/nf-werapi-werstoregetnextreportkey)之後呼叫此參數。<br/> |
| [**WerGetFlags**](/windows/desktop/api/Werapi/nf-werapi-wergetflags)<br/>                                                                       | 抓取指定進程的錯誤報表設定。<br/>                                                                                                                                                                                                                                                       |
| [**WerRegisterAdditionalProcess**](/windows/desktop/api/Werapi/nf-werapi-werregisteradditionalprocess)<br/>                                     | 註冊要包含在錯誤報表中的進程，以及主要的應用程式處理常式。 （選擇性）在該註冊的進程內指定要從中取得其他資料的執行緒。<br/>                                                                                                                               |
| [**WerRegisterAppLocalDump**](/windows/desktop/api/Werapi/nf-werapi-werregisterapplocaldump)<br/>                                               | 針對呼叫應用程式的本機應用程式存放區註冊相對的路徑，其中 Windows 錯誤報告 (WER) 應該在應用程式的其中一個進程停止回應時，儲存 WER 所收集的診斷記憶體傾印複本。<br/>                                                                     |
| [**WerRegisterCustomMetadata**](/windows/desktop/api/Werapi/nf-werapi-werregistercustommetadata)<br/>                                           | 以索引鍵/值字串的形式，註冊要收集的應用程式專屬中繼資料 () 在 WER 建立錯誤報表時。<br/>                                                                                                                                                                                                    |
| [**WerRegisterExcludedMemoryBlock**](/windows/desktop/api/Werapi/nf-werapi-werregisterexcludedmemoryblock)<br/>                                 | 標示預設包含在錯誤報表中的記憶體區塊 (，) 要從錯誤報表中排除。<br/>                                                                                                                                                                                                      |
| [**WerRegisterFile**](/windows/desktop/api/Werapi/nf-werapi-werregisterfile)<br/>                                                               | 註冊當 WER 建立錯誤報表時要收集的檔案。<br/>                                                                                                                                                                                                                                                      |
| [**WerRegisterMemoryBlock**](/windows/desktop/api/Werapi/nf-werapi-werregistermemoryblock)<br/>                                                 | 註冊當 WER 建立錯誤報表時要收集的記憶體區塊。<br/>                                                                                                                                                                                                                                              |
| [**WerRegisterRuntimeExceptionModule**](/windows/desktop/api/Werapi/nf-werapi-werregisterruntimeexceptionmodule)<br/>                           | 註冊自訂執行時間例外狀況處理常式，用來提供自訂錯誤報表給當機。<br/>                                                                                                                                                                                                                |
| [**WerRemoveExcludedApplication**](/windows/desktop/api/Werapi/nf-werapi-werremoveexcludedapplication)<br/>                                     | 從要從錯誤報表排除的應用程式清單中移除指定的應用程式。<br/>                                                                                                                                                                                                           |
| [**WerReportAddDump**](/windows/desktop/api/Werapi/nf-werapi-werreportadddump)<br/>                                                             | 將指定類型的傾印加入至指定的報表。<br/>                                                                                                                                                                                                                                                              |
| [**WerReportAddFile**](/windows/desktop/api/Werapi/nf-werapi-werreportaddfile)<br/>                                                             | 將檔案加入至指定的報表。<br/>                                                                                                                                                                                                                                                                                    |
| [**WerReportCloseHandle**](/windows/desktop/api/Werapi/nf-werapi-werreportclosehandle)<br/>                                                     | 關閉指定的報表。<br/>                                                                                                                                                                                                                                                                                            |
| [**WerReportCreate**](/windows/desktop/api/Werapi/nf-werapi-werreportcreate)<br/>                                                               | 建立描述應用程式事件的問題報表。<br/>                                                                                                                                                                                                                                                           |
| [**WerReportHang**](/windows/desktop/api/ErrorRep/nf-errorrep-werreporthang)<br/>                                                                   | 在指定的視窗上起始「無回應」報告。<br/>                                                                                                                                                                                                                                                              |
| [**WerReportSetParameter**](/windows/desktop/api/Werapi/nf-werapi-werreportsetparameter)<br/>                                                   | 設定可唯一識別指定報表之事件的參數。<br/>                                                                                                                                                                                                                                           |
| [**WerReportSetUIOption**](/windows/desktop/api/Werapi/nf-werapi-werreportsetuioption)<br/>                                                     | 設定指定之報表的使用者介面選項。<br/>                                                                                                                                                                                                                                                               |
| [**WerReportSubmit**](/windows/desktop/api/Werapi/nf-werapi-werreportsubmit)<br/>                                                               | 提交指定的報表。<br/>                                                                                                                                                                                                                                                                                           |
| [**WerSetFlags**](/windows/desktop/api/Werapi/nf-werapi-wersetflags)<br/>                                                                       | 設定目前進程的錯誤報表設定。<br/>                                                                                                                                                                                                                                                              |
| [**WerStoreClose**](/windows/desktop/api/Werapi/nf-werapi-werstoreclose)<br/>                                                                   | 關閉已儲存報表的集合。<br/>                                                                                                                                                                                                                                                                                |
| [**WerStoreGetFirstReportKey**](/windows/desktop/api/Werapi/nf-werapi-werstoregetfirstreportkey)<br/>                                           | 取得報表存放區中第一個報表的參考。<br/>                                                                                                                                                                                                                                                               |
| [**WerStoreGetNextReportKey**](/windows/desktop/api/Werapi/nf-werapi-werstoregetnextreportkey)<br/>                                             | 取得錯誤報表存放區中下一個報表的參考。<br/>                                                                                                                                                                                                                                                          |
| [**WerStoreOpen**](/windows/desktop/api/Werapi/nf-werapi-werstoreopen)<br/>                                                                     | 開啟預存錯誤報表的集合。<br/>                                                                                                                                                                                                                                                                           |
| [**WerStoreQueryReportMetadataV2**](/windows/desktop/api/Werapi/nf-werapi-werstorequeryreportmetadatav2)<br/>                                   | 抓取存放區中報表的相關中繼資料。<br/>                                                                                                                                                                                                                                                                         |
| [**WerUnregisterAdditionalProcess**](/windows/desktop/api/Werapi/nf-werapi-werunregisteradditionalprocess)<br/>                                 | 從要包含在錯誤報表中的其他進程清單中移除處理常式。<br/>                                                                                                                                                                                                                             |
| [**WerUnregisterAppLocalDump**](/windows/desktop/api/Werapi/nf-werapi-werunregisterapplocaldump)<br/>                                           | 取消呼叫 [**WerRegisterAppLocalDump**](/windows/desktop/api/Werapi/nf-werapi-werregisterapplocaldump) 函式所進行的註冊，以指定 WINDOWS 錯誤報告 (wer) 應儲存當應用程式的其中一個進程停止回應時，wer 所收集的診斷記憶體傾印複本。<br/>              |
| [**WerUnregisterCustomMetadata**](/windows/desktop/api/Werapi/nf-werapi-werunregistercustommetadata)<br/>                                       | 移除在應用程式的錯誤報表期間收集的應用程式專屬中繼資料專案。<br/>                                                                                                                                                                                                                    |
| [**WerUnregisterExcludedMemoryBlock**](/windows/desktop/api/Werapi/nf-werapi-werunregisterexcludedmemoryblock)<br/>                             | 移除先前標示為排除的記憶體區塊 (它會再次包含在錯誤報表中) 。<br/>                                                                                                                                                                                                             |
| [**WerUnregisterFile**](/windows/desktop/api/Werapi/nf-werapi-werunregisterfile)<br/>                                                           | 從要加入至為目前進程產生之報表的檔案清單中移除檔案。<br/>                                                                                                                                                                                                                         |
| [**WerUnregisterMemoryBlock**](/windows/desktop/api/Werapi/nf-werapi-werunregistermemoryblock)<br/>                                             | 從要在應用程式的錯誤報表期間收集的資料清單中移除記憶體區塊。<br/>                                                                                                                                                                                                                |
| [**WerUnregisterRuntimeExceptionModule**](/windows/desktop/api/Werapi/nf-werapi-werunregisterruntimeexceptionmodule)<br/>                       | 移除 WER 例外狀況處理常式的註冊。<br/>                                                                                                                                                                                                                                                                 |
