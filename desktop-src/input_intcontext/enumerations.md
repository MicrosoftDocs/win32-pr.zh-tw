# <a name="interaction-context-enumerations"></a>互動內容列舉

本節中的主題提供 [互動內容](interaction-context-portal.md) 列舉的參考規格。

## <a name="in-this-section"></a>本節內容

| 主題 | 描述 |
|---|---|
| [**跨 \_ 幻燈片 \_ 旗標**](/windows/win32/api/interactioncontext/ne-interactioncontext-cross_slide_flags)<br/> | 指定跨幻燈片互動的狀態。<br/> |
| [**交叉投影 \_ 片 \_ 閾值**](/windows/win32/api/interactioncontext/ne-interactioncontext-cross_slide_threshold)<br/> | 指定交叉投影片的行為臨界值。<br/> |
| [**慣性 \_ 參數**](/windows/win32/api/interactioncontext/ne-interactioncontext-inertia_parameter)<br/> | 為操作指定慣性值 (轉譯、旋轉、調整) 。<br/> |
| [**互動 \_ 設定 \_ 旗標**](/windows/win32/api/interactioncontext/ne-interactioncontext-interaction_configuration_flags)<br/> | 指定在設定 [互動內容](interaction-context-portal.md) 物件時要啟用的互動。<br/> |
| [**互動 \_ 上下文 \_ 屬性**](/windows/win32/api/interactioncontext/ne-interactioncontext-interaction_context_property)<br/> | 指定 [互動內容](interaction-context-portal.md) 物件的屬性。 <br/> |
| [**互動 \_ 旗標**](/windows/win32/api/interactioncontext/ne-interactioncontext-interaction_flags)<br/> | 指定互動的狀態。<br/> |
| [**互動 \_ 識別碼**](/windows/win32/api/interactioncontext/ne-interactioncontext-interaction_id)<br/> | 指定用於設定 [互動內容](interaction-context-portal.md) 物件的互動狀態。 互動可以是靜態 (單一連絡人，而不需要操作，例如點一下、點一下、點一下、按住) 或動態 (一或多個連絡人進行操作，例如轉譯、旋轉或調整) 。<br/> |
| [**互動 \_ 狀態**](/windows/win32/api/interactioncontext/ne-interactioncontext-interaction_state)<br/> | 指定 [互動內容](interaction-context-portal.md) 物件的狀態。<br/> |
| [**操作 \_ 滑軌 \_ 狀態**](/windows/win32/api/interactioncontext/ne-interactioncontext-manipulation_rails_state)<br/> | 指定互動的滑軌狀態。<br/> |
| [**滑鼠 \_ 滾輪 \_ 參數**](/windows/win32/api/interactioncontext/ne-interactioncontext-mouse_wheel_parameter)<br/> | 指定可對應至滑鼠滾輪旋轉的操作。<br/> |
