# <a name="wic-samples"></a>WIC 範例

本節包含 Windows 影像處理元件 (WIC) 範例。

WIC 提供兩種類型的範例：

-   範例：使用數個 WIC 功能的程式碼區段。 範例的原始程式碼會在 Windows SDK 中傳遞。
-   程式碼範例：示範特定功能的較短範例。 SDK 不提供範例的原始程式碼。
