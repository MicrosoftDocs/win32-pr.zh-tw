# <a name="windows-imaging-component"></a>Windows Imaging Component

## <a name="purpose"></a>目的

Windows 影像處理元件 (WIC) 是可延伸的平臺，可為數字影像提供低層級的 API。  WIC 支援標準 web 映射格式、高動態範圍影像和原始相機資料。  WIC 也支援其他功能，例如：

-   標準元資料格式的內建支援
-   映射編解碼器、像素格式和元資料格式的可擴充架構。
-   範圍廣泛的像素格式支援。
-   高彩支援;包含30位的延伸範圍、30位高精確度，以及48位高精確度和寬範圍像素格式。
-   漸進式影像解碼。

## <a name="developer-audience"></a>開發人員讀者

WIC 的設計是為了符合下列開發人員類別的需求：

-   在應用程式中讀取和寫入影像資料的開發人員。
-   在應用程式中讀取和寫入影像中繼資料的開發人員。
-   需要針對新設計或實作為映射編解碼器進行執行時間編解碼器探索的開發人員。
-   開發人員設計或執行新的映射編解碼器和元資料格式。
