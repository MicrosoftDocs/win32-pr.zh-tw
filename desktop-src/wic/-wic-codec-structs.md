# <a name="wic-structures"></a>WIC 結構

本節包含 Windows 影像處理元件 (WIC) 結構的相關資訊。

## <a name="in-this-section"></a>本節內容



| 主題                                                                          | 描述                                                                                                                   |
|--------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------|
| [**WICBitmapPattern**](/windows/desktop/api/Wincodec/ns-wincodec-wicbitmappattern)<br/>             | 包含成員，這些成員可在影像檔案中識別可用於識別特定格式的模式。<br/>   |
| [**WICBitmapPlane**](/windows/desktop/api/Wincodec/ns-wincodec-wicbitmapplane)<br/>                            | 指定平面像素格式之元件平面的像素格式、緩衝區、跨距和大小。<br/>                |
| [**WICBitmapPlaneDescription**](/windows/desktop/api/Wincodec/ns-wincodec-wicbitmapplanedescription)<br/>      | 指定元件平面的像素格式和大小。<br/>                                                          |
| [**WICDdsFormatInfo**](/windows/desktop/api/Wincodec/ns-wincodec-wicddsformatinfo)<br/>                        | 指定 DDS 格式的 [**DXGI \_ 格式**](/windows/desktop/api/dxgiformat/ne-dxgiformat-dxgi_format) 和區塊資訊。<br/>                  |
| [**WICDdsParameters**](/windows/desktop/api/Wincodec/ns-wincodec-wicddsparameters)<br/>                        | 指定包含資料的 DDS 影像維度、 [**DXGI \_ 格式**](/windows/desktop/api/dxgiformat/ne-dxgiformat-dxgi_format) 和 Alpha 模式。<br/>  |
| [**WICImageParameters**](/windows/desktop/api/Wincodec/ns-wincodec-wicimageparameters)<br/>                    | 這會定義參數，您可以用來覆寫在編碼影像時通常使用的預設參數。 <br/> |
| [**WICJpegFrameHeader**](/windows/desktop/api/wincodec/ns-wincodec-wicjpegframeheader)<br/>                    | 表示 JPEG 框架標題。<br/>                                                                                    |
| [**WICJpegScanHeader**](/windows/desktop/api/wincodec/ns-wincodec-wicjpegscanheader)<br/>                      | 表示 JPEG 框架標題。<br/>                                                                                    |
| [**WICMetadataHeader**](/windows/desktop/api/Wincodecsdk/ns-wincodecsdk-wicmetadataheader)<br/>           | 表示中繼資料標頭。<br/>                                                                                        |
| [**WICMetadataPattern**](/windows/desktop/api/Wincodecsdk/ns-wincodecsdk-wicmetadatapattern)<br/>         | 表示中繼資料模式。<br/>                                                                                     |
| [**WICRawCapabilitiesInfo**](/windows/desktop/api/Wincodec/ns-wincodec-wicrawcapabilitiesinfo)<br/> | 定義原始編解碼器的能力。<br/>                                                                                     |
| [**WICRawToneCurve**](/windows/desktop/api/Wincodec/ns-wincodec-wicrawtonecurve)<br/>               | 表示原始影像音調曲線。<br/>                                                                                 |
| [**WICRawToneCurvePoint**](/windows/desktop/api/Wincodec/ns-wincodec-wicrawtonecurvepoint)<br/>     | 表示原始影像音調曲線點。<br/>                                                                           |
| [**WICRect**](/windows/desktop/api/Wincodec/ns-wincodec-wicrect)<br/>                               | 表示 WIC API 的矩形。<br/>                                                                                |
