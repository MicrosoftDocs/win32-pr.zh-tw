# <a name="about-security-wmi-providers"></a>關於安全性 WMI 提供者

安全性 WMI 提供者可讓應用程式與信賴平臺模組進行互動 (TPM) 和 BitLocker 磁碟機加密 (的) Windows Management Instrumentation () WMI 的整合管理架構。



| 區段                                                                  | 描述                                                                                                                    |
|--------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| [安全性 WMI 提供者參考](security-wmi-providers-reference.md) | [**Win32 \_ EncryptableVolume**](win32-encryptablevolume.md)和 [**WIN32 \_ Tpm**](win32-tpm.md)類別的 API 檔。 |



 

## <a name="remarks"></a>備註

受控物件格式 (MOF) 檔包含 Windows Management Instrumentation (WMI) 類別的定義。 MOF 檔案不會安裝為 Windows SDK 的一部分。 當您使用伺服器管理員新增相關聯的角色時，它們會安裝在伺服器上。 如需 MOF 檔案的詳細資訊，請參閱 [受控物件格式 (mof) ](../wmisdk/managed-object-format--mof-.md)。
