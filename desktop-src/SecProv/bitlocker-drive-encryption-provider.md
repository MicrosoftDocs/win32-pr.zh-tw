# <a name="bitlocker-drive-encryption-provider"></a>BitLocker 磁碟機加密提供者

WMI 提供者介面是用來管理和設定 Windows Server 2008 R2、Windows Server 2008 上的 BitLocker 磁碟機加密 (的) ，以及僅限特定版本的 Windows 7、Windows Vista Enterprise 和 Windows Vista 旗艦版。 這是為了讓撰寫腳本、使用者介面專案或其他的系統管理工具，以及想要深入瞭解 MANAGE-BDE 功能的人員閱讀。
