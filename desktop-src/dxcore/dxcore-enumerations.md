# <a name="dxcore-enumerations"></a>DXCore 列舉

下列列舉是在 dxcore_interface 中宣告的。

## <a name="in-this-section"></a>本節內容

| 主題和描述 |
|-|
| [**DXCoreAdapterPreference**](./dxcore_interface/ne-dxcore_interface-dxcoreadapterpreference.md)。 定義常數，指定要當做清單排序準則使用的 DXCore 介面卡喜好設定。 |
| [**DXCoreAdapterProperty**](./dxcore_interface/ne-dxcore_interface-dxcoreadapterproperty.md)。 定義指定 DXCore 介面卡屬性的常數。 |
| [**DXCoreAdapterState**](./dxcore_interface/ne-dxcore_interface-dxcoreadapterstate.md)。 定義常數，指定 DXCore 介面卡狀態的種類。 |
| [**DXCoreNotificationType**](./dxcore_interface/ne-dxcore_interface-dxcorenotificationtype.md)。 定義常數，指定 [IDXCoreAdapter](./dxcore_interface/nn-dxcore_interface-idxcoreadapter.md) 或 [IDXCoreAdapterList](./dxcore_interface/nn-dxcore_interface-idxcoreadapterlist.md) 物件所引發的通知類型。 |
| [**DXCoreSegmentGroup**](./dxcore_interface/ne-dxcore_interface-dxcoresegmentgroup.md)。 定義常數，指定介面卡的記憶體區段群組。 |
