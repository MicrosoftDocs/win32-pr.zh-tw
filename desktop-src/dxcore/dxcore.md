# <a name="dxcore"></a>DXCore

DXCore 是圖形和計算裝置的介面卡列舉 API，因此其某些設備會與 [DXGI](../direct3ddxgi/dx-graphics-dxgi.md)的部分重迭。 DXCore 是由 Direct3D 12 使用。

此檔集包含使用 DXCore 進行程式設計的相關資訊，以及 DXCore 參考。

## <a name="requirements"></a>規格需求

| &nbsp; | &nbsp; |
| ---- |:---- |
| **支援的執行時間環境** | Windows/c + + |
| **建議的程式設計語言** | C++ |
| **最低支援的用戶端** | Windows 10，版本 2004 (10.0;組建 19041)  |
| **標頭** | dxcore .h 和 dxcore_interface。h |
| **程式庫** | dxcore .lib |
| **DLL** | dxcore.dll |
