# <a name="idxcoreadapterfactory-interface"></a>IDXCoreAdapterFactory 介面

 **IDXCoreAdapterFactory**   介面會實作為產生 DXCore 介面卡列舉物件的方法，以及用來取得其詳細資料的方法。 **IDXCoreAdapterFactory** 繼承自 [IUnknown](/windows/win32/api/unknwn/nn-unknwn-iunknown) 介面。 如需程式設計指引和程式碼範例，請參閱 [使用 DXCore 來列舉介面卡](../dxcore-enum-adapters.md)。
