# <a name="idxcoreadapterlist-interface"></a>IDXCoreAdapterList 介面

 **IDXCoreAdapterList** 介面會實作為   從產生的清單中取得介面卡專案的方法，以及清單的詳細資料。 **IDXCoreAdapterList** 繼承自 [IUnknown](/windows/win32/api/unknwn/nn-unknwn-iunknown) 介面。 如需程式設計指引和程式碼範例，請參閱 [使用 DXCore 來列舉介面卡](../dxcore-enum-adapters.md)。
