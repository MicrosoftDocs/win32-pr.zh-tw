# <a name="eventarg-element"></a>eventArg 元素

指定產生的函數中是否包含相關的事件引數。

## <a name="usage"></a>使用量

``` syntax
<eventArg/>
```

## <a name="attributes"></a>屬性

沒有任何屬性。

## <a name="child-elements"></a>子元素

沒有任何子項目。

## <a name="parent-elements"></a>父元素



| 元素                                                               | 描述                                                                                     |
|-----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [**idlFunctionDeclarations**](idlfunctiondeclarations.md)<br/> | 針對埠類型作業產生 proxy 函式的 IDL 宣告。<br/> <br/> |
| [**stubDefinitions**](stubdefinitions.md)<br/>                 | 針對埠類型作業產生存根函式的實作為。<br/> <br/>   |



## <a name="remarks"></a>備註

可能的值為 1 (包含) 的事件引數，而 0 (預設值為) 中排除的事件引數。

|-------------------------------------|---------------|
