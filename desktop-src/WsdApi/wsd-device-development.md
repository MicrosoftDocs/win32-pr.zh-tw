# <a name="wsd-device-development"></a>WSD 裝置開發

裝置製造商和其他裝置實作者可以建立與以 Windows 為基礎的 WSDAPI 用戶端和主機交互操作的裝置，而不需要使用 WSDAPI。

除了開發裝置功能，裝置實作者必須 [針對 Web 服務 (DPWS) 來執行裝置設定檔](https://specs.xmlsoap.org/ws/2006/02/devprof/) 。

盡可能使用 WSDAPI 來建立以 Windows 為基礎的 WSD 用戶端或裝置主機應用程式。 使用 WSDAPI 可降低開發的複雜度並節省時間。 如需詳細資訊，請參閱 [Windows 上的 WSD 應用程式開發](wsd-application-development-on-windows.md)。
