# <a name="stubfunction-element"></a>stubFunction 元素

指定存根函式參考是否應該包含在單向和雙向作業的埠類型定義中的作業結構中。

## <a name="usage"></a>使用方式

``` syntax
<stubFunction/>
```

## <a name="attributes"></a>屬性

沒有任何屬性。

## <a name="child-elements"></a>子元素

沒有任何子項目。

## <a name="parent-elements"></a>父元素



| 元素                                                       | 描述                                                  |
|---------------------------------------------------------------|--------------------------------------------------------------|
| [**portTypeDefinitions**](porttypedefinitions.md)<br/> | 產生埠類型的 C 常數。<br/> <br/> |



## <a name="remarks"></a>備註

存根函式參考用於單向和雙向作業的裝載案例中。

此元素的有效值為 1 (TRUE/stub 函數參考包含) 和 0 (FALSE/沒有包含) 的存根函數參考。
