# <a name="wsdapi-xml-type-bytecodes"></a>WSDAPI XML 類型位元組

WSDAPI 會使用這些位元組來剖析和產生 XML，方法是使用可描述複雜類型的位元組程式碼資料表。

型別的位元組程式碼資料表表示二進位 (記憶體內部) ，以及類型的 XML (線上) 表示。 位元組程式碼資料表是由特殊運算終止的作業清單所組成，亦即 `OpEndOfTable` 。 作業包含一個位元組的作業程式碼，後面接著零或多個位元組的引數資料。
