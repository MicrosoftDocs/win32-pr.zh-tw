# <a name="web-services-on-devices-code-generator"></a>裝置上的 Web 服務程式碼產生器

裝置上的 Web 服務程式碼產生器 WsdCodeGen 是一種命令列公用程式，可從服務描述產生程式碼。 服務描述會儲存在 WSDL 和/或 XSD 檔案中。 WsdCodeGen 會建立 c + + 和 IDL (介面定義語言) 檔。 開發人員可以使用 WsdCodeGen 來建立 WSDAPI 應用程式，而不需要擔心資料如何在網路上進行封送處理和表示。

WsdCodeGen 可以用來產生服務、用戶端或兩者的程式碼。 公用程式會產生服務的 stub 程式碼、用戶端的 proxy 程式碼，以及兩者的介面和類型程式碼。

這項工具適用于 Microsoft Windows 軟體開發套件 (SDK) 以及 Windows 驅動程式套件 (WDK) 。 SDK 工具位於下列目錄： <Windows SDK Install Folder> \\ Bin。

Windows SDK 包含 WsdCodeGen 所產生的範例程式碼。 如需詳細資訊，請參閱 [WSDAPI 範例](wsdapi-samples.md)。
