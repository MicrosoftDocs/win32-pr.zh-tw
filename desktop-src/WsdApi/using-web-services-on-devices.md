# <a name="using-web-services-on-devices"></a>在裝置上使用 Web 服務

Microsoft Web Services on Devices API (WSDAPI) 支援執行用戶端控制的裝置和服務，以及符合 Web 服務 (DPWS) 之 [裝置設定檔](https://specs.xmlsoap.org/ws/2006/02/devprof/) 的裝置主機。 您可以使用 WSDAPI 來開發用戶端和伺服器 (裝置) 的實施。

盡可能使用 WSDAPI 來建立以 Windows 為基礎的 WSD 用戶端或裝置主機應用程式。 使用 WSDAPI 可降低開發的複雜度並節省時間。 如需詳細資訊，請參閱 Windows 和[WSDAPI 開發工具](wsdapi-development-tools.md)[上的 WSD 應用程式開發](wsd-application-development-on-windows.md)。

您也可以建立未執行 Windows 的 WSD 裝置，並可與 Windows 型 WSD 用戶端和裝置主機互通。 這些裝置必須符合特定需求，才能維持互通性。 如需詳細資訊，請參閱 [WSD 裝置開發](wsd-device-development.md)。
