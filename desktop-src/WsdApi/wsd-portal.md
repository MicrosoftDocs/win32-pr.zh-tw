# <a name="web-services-on-devices"></a>裝置上的 Web 服務

## <a name="purpose"></a>目的

Microsoft Web Services on Devices API (WSDAPI) 支援執行用戶端控制的裝置和服務，以及符合 Web 服務 (DPWS) 之 [裝置設定檔](https://specs.xmlsoap.org/ws/2006/02/devprof/) 的裝置主機。 WSDAPI 使用 [WS 探索](https://specs.xmlsoap.org/ws/2005/04/discovery/ws-discovery.pdf) 進行裝置探索。

WSDAPI 可用於用戶端和服務的開發。

## <a name="where-applicable"></a>適用時

裝置上的 Web 服務可讓用戶端在網路上探索及存取遠端裝置及其相關聯的服務。 它支援裝置探索、描述、控制項和事件。 開發人員可以為裝置主機建立 WSDAPI 用戶端 proxy 和對應的存根。

## <a name="developer-audience"></a>開發人員對象

[裝置上的 Web 服務] 檔適用于 C/c + + 程式設計人員，以及建立符合 DPWS 規範之產品的裝置廠商。

## <a name="run-time-requirements"></a>執行階段需求求

從 Windows Vista 和 Windows Server 2008 開始，支援使用 WSDAPI 的用戶端應用程式。
