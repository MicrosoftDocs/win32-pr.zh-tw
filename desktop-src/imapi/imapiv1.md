# <a name="imapiv1-support"></a>IMAPIv1 支援

映射主控 API 第1版 (IMAPIv1) 可讓應用程式階段，以及將簡單的音訊或資料影像燒錄到 CD-R 和 cd-rw 裝置。 API 支援 Redbook 音訊和資料光碟的 Joliet 和 ISO 9660 格式。

> [!Note]  
> IMAPI.EXE 已針對2.0 版進行了完整重新設計。 版本2.0 提供大部分的1.0 版功能。 我們鼓勵這些撰寫映射主控應用程式或執行新的裝置和格式開發，以使用 [版本 2.0](using-imapi.md) ，而不是1.0 版。 如果您在 Windows Vista 上執行1.0 版的應用程式，1.0 版介面會將這些呼叫轉換成適當的2.0 版呼叫，以達到相同的功能。

 

如需在 Windows XP 上使用1.0 版的詳細資訊，請參閱下列主題：

-   [關於映射主控 API](about-the-image-mastering-api.md)
-   [使用映射主控 API](using-the-image-mastering-api.md)
-   [IMAPIv1 參考](imapiv1-reference.md)
