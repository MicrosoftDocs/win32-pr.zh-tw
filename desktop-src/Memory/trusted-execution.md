# <a name="trusted-execution"></a>受信任的執行

記憶體保護區可用來建立受信任的執行環境。 記憶體保護區是程式碼的隔離區域，以及應用程式位址空間內的資料。 只有在記憶體保護區中執行的程式碼可以存取相同記憶體保護區內的資料。
