# <a name="structural-object-class-attribute"></a>結構物件類別屬性

這個建立的屬性會儲存包含在類別階層中的類別清單，包括抽象類別。 這份清單包含動態連結的輔助類別。



| 進入 | 值 |
|-------------------|-----------------------------------------------------------------|
| CN                | 結構物件類別                                         |
| Ldap-顯示名稱 | structuralObjectClass                                           |
| 大小              | \-                                                              |
| 更新許可權  | 此值是由系統所設定。                                |
| 更新頻率  | 在類別建立時。                                              |
| Attribute-Id      | 2.5.21.9                                                        |
| 系統識別碼-Guid    | 3860949f-f6a8-4b38-9950-81ecb6bc2982                            |
| Syntax            | [**String(Object-Identifier)**](s-string-object-identifier.md) |
