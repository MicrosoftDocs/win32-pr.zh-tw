| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | Netboot-SIF-File                            |
| Ldap-顯示名稱 | netbootSIFFile                              |
| 大小              | \-                                          |
| 更新許可權  | 此值是由系統所設定。            |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1240                     |
| 系統識別碼-Guid    | 2df90d84-009f-11d2-aa4c-00c04fd7d83a        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
