# <a name="acs-rsvp-account-files-location-attribute"></a>ACS-RSVP-帳戶-檔案-位置屬性

RSVP 帳戶檔案的目錄位置。 預設為位於預設系統目錄中的 "system32" 子目錄。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | ACS-RSVP-帳戶-檔案-位置             |
| Ldap-顯示名稱 | aCSRSVPAccountFilesLocation                 |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.900                      |
| 系統識別碼-Guid    | f072230f-aef5-11d1-bdcf-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
