|-------------------|--------------------------------------|
| CN                | MSMQ-Services                        |
| Ldap-顯示名稱 | mSMQServices                         |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.950               |
| 系統識別碼-Guid    | 9a0dc33d-c100-11d1-bbc5-0080c76670c0 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
