# <a name="print-color-attribute"></a>Print-Color 屬性

如果印表機可以彩色列印，**則為 TRUE** 。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | Print-Color                          |
| Ldap-顯示名稱 | printColor                           |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.243               |
| 系統識別碼-Guid    | 281416d3-1968-11d0-a28f-00aa003049e2 |
| Syntax            | [**Boolean**](s-boolean.md)         |
