# <a name="non-security-member-attribute"></a>非安全性成員屬性

非安全性群組的成員。 用於 Exchange 通訊群組清單。



| 進入 | 值 |
|-------------------|--------------------------------------------------|
| CN                | 非安全性成員                              |
| Ldap-顯示名稱 | nonSecurityMember                                |
| 大小              | \-                                               |
| 更新許可權  | 網域系統管理員                             |
| 更新頻率  | 每次在 DL 中新增或移除使用者時。 |
| Attribute-Id      | 1.2.840.113556.1.4.530                           |
| 系統識別碼-Guid    | 52458018-ca6a-11d0-afff-0000f80367c1             |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md)          |
