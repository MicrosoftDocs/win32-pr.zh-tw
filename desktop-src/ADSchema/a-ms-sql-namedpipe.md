# <a name="ms-sql-namedpipe-attribute"></a>NamedPipe 屬性

具名管道連接。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | NamedPipe                            |
| Ldap-顯示名稱 | NamedPipe                            |
| 大小              | \-                                          |
| 更新許可權  | 此值是由系統所設定。            |
| 更新頻率  | 在系統啟動時。                          |
| Attribute-Id      | 1.2.840.113556.1.4.1374                     |
| 系統識別碼-Guid    | 7b91c840-ccee-11d2-9993-0000f87a57d4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
