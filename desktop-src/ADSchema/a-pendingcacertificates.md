# <a name="pending-ca-certificates-attribute"></a>擱置-CA-憑證屬性

即將對此憑證授權單位單位有效的憑證。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | 擱置-CA-憑證                               |
| Ldap-顯示名稱 | pendingCACertificates                                 |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.693                                |
| 系統識別碼-Guid    | 963d273c-48be-11d1-a9c3-0000f80367c1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
