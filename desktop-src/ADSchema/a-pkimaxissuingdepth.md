| Ldap-顯示名稱 | pKIMaxIssuingDepth                   |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |

| 系統識別碼-Guid    | f0bfdefa-3b9d-11d2-90cc-00c04fd91ab1 |
