# <a name="msmq-foreign-attribute"></a>MSMQ-Foreign 屬性

指出這部電腦是否為外部 MSMQ 電腦。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | MSMQ-Foreign                         |
| Ldap-顯示名稱 | mSMQForeign                          |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.934               |
| 系統識別碼-Guid    | 9a0dc32f-c100-11d1-bbc5-0080c76670c0 |
| Syntax            | [**Boolean**](s-boolean.md)         |
