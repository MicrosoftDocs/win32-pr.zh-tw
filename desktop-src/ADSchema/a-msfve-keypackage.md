# <a name="ms-fve-keypackage-attribute"></a>FVE-KeyPackage 屬性

| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | ms-FVE-KeyPackage                                     |
| Ldap-顯示名稱 | msFVE-KeyPackage                                      |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1999                               |
| 系統識別碼-Guid    | 1fd55ea8-88a7-47dc-8129-0daa97186a54                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
