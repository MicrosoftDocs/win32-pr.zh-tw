# <a name="inter-site-topology-generator-attribute"></a>網站間拓撲-產生器屬性

這個屬性將用來支援指定網站中執行知識一致性檢查程式產生的電腦上的容錯移轉。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | 網站間拓撲-產生器           |
| Ldap-顯示名稱 | interSiteTopologyGenerator              |
| 大小              | \-                                      |
| 更新許可權  | \-                                      |
| 更新頻率  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1246                 |
| 系統識別碼-Guid    | b7c69e5e-2cc7-11d2-854e-00a0c983f608    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |



## <a name="implementations"></a>實作
