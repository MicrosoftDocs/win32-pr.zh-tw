# <a name="recalculate-security-inheritance-extended-right"></a>重新計算-安全性-繼承擴充許可權

需要有擴充的許可權，才能強制 DS 在命名內容上重新計算 ACL 繼承。



| 進入 | 值 |
|--------------|--------------------------------------|
| CN           | 重新計算-安全性-繼承     |
| 顯示名稱 | 重新計算安全性繼承     |
| 許可權-GUID  | 62dd28a8-7f46-11d2-b9ad-00c04f79f805 |
