# <a name="uniquemember-attribute"></a>uniqueMember 屬性

群組成員的分辨名稱。 由 groupOfUniqueNames 使用。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | uniqueMember                            |
| Ldap-顯示名稱 | uniqueMember                            |
| 大小              | \-                                      |
| 更新許可權  | \-                                      |
| 更新頻率  | \-                                      |
| Attribute-Id      | 2.5.4.50                                |
| 系統識別碼-Guid    | 8f888726-f80a-44d7-b1ee-cb9df21392c8    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
