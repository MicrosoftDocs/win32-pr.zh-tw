|-------------------|---------------------------------------------|
| CN                | Localized-Description                       |
| Ldap-顯示名稱 | localizedDescription                        |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |

| 系統識別碼-Guid    | d9e18316-8939-11d1-aebc-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
