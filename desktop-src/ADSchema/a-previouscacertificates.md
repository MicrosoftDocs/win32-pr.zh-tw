# <a name="previous-ca-certificates-attribute"></a>上一個 CA-憑證屬性

此憑證授權單位單位的上次過期憑證。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | 上一個 CA-憑證                              |
| Ldap-顯示名稱 | previousCACertificates                                |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.692                                |
| 系統識別碼-Guid    | 963d2739-48be-11d1-a9c3-0000f80367c1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
