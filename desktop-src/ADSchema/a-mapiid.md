# <a name="mapi-id-attribute"></a>MAPI 識別碼屬性

MAPI 用戶端用來識別這個屬性的整數。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | MAPI 識別碼                              |
| Ldap-顯示名稱 | mAPIID                               |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.49                |
| 系統識別碼-Guid    | bf9679b7-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
