# <a name="operating-system-attribute"></a>Operating-System 屬性

作業系統名稱，例如 Windows Vista Enterprise。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | Operating-System                            |
| Ldap-顯示名稱 | operatingSystem                             |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.363                      |
| 系統識別碼-Guid    | 3e978925-8c01-11d0-afda-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
