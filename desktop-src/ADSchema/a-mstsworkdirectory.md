| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | ms-TS-Work-Directory                        |
| Ldap-顯示名稱 | msTSWorkDirectory                           |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1989                     |
| 系統識別碼-Guid    | a744f666-3d3c-4cc8-834b-9d4f6f687b8b        |
