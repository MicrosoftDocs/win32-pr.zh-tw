# <a name="com-clsid-attribute"></a>COM-CLSID 屬性

這個屬性包含與這個物件類別相關聯的 GUID。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | COM-CLSID                                   |
| Ldap-顯示名稱 | cOMCLSID                                    |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.249                      |
| 系統識別碼-Guid    | 281416d9-1968-11d0-a28f-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
