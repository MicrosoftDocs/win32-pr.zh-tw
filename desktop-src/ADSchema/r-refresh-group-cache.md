# <a name="refresh-group-cache-extended-right"></a>重新整理-群組-快取擴充許可權

這不適用於 GC 登入。 沒有 GC 登入需要快取群組成員資格，而此控制存取許可權是用來授與系統管理員和操作員許可權，以立即重新整理快取，並聯系可用的 GC。



| 進入 | 值 |
|--------------|--------------------------------------|
| CN           | 重新整理-群組-快取                  |
| 顯示名稱 | 重新整理登入的群組快取       |
| 許可權-GUID  | 9432c620-033c-4db7-8b58-14ef6d0bf477 |
