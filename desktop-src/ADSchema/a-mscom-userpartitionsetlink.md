# <a name="ms-com-userpartitionsetlink-attribute"></a>UserPartitionSetLink 屬性

用來將使用者與 COM + PartitionSet 建立關聯的連結。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | UserPartitionSetLink             |
| Ldap-顯示名稱 | msCOM-UserPartitionSetLink              |
| 大小              | \-                                      |
| 更新許可權  | \-                                      |
| 更新頻率  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1426                 |
| 系統識別碼-Guid    | 8e940c8a-e477-4367-b08d-ff2ff942dcd7    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
