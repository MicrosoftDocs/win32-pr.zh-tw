# <a name="ms-pki-template-minor-revision-attribute"></a>ms-PKI-範本-次要修訂屬性

持續追蹤類別中的屬性變更。 不過，這不會觸發自動註冊。



| 進入 | 值 |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-範本-次要修訂                                                                    |
| Ldap-顯示名稱 | Mspki-certificate-name-flag-範本-次要修訂                                                                     |
| 大小              | 4 個位元組                                                                                           |
| 更新許可權  | 網域系統管理員                                                                              |
| 更新頻率  | 當您編輯、建立或複製憑證範本時 (ms-PKI-憑證範本) 物件。 |
| Attribute-Id      | 1.2.840.113556.1.4.1435                                                                           |
| 系統識別碼-Guid    | 13f5236c-1884-46b1-b5d0-484e38990d58                                                              |
| Syntax            | [**列舉型別**](s-enumeration.md)                                                              |
