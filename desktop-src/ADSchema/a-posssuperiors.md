# <a name="poss-superiors-attribute"></a>Poss-Superiors 屬性

可包含這個類別的物件清單。



| 進入 | 值 |
|-------------------|-----------------------------------------------------------------|
| CN                | Poss-Superiors                                                  |
| Ldap-顯示名稱 | possSuperiors                                                   |
| 大小              | \-                                                              |
| 更新許可權  | 網域系統管理員                                            |
| 更新頻率  | 定義類別時。                                      |
| Attribute-Id      | 1.2.840.113556.1.2.8                                            |
| 系統識別碼-Guid    | bf9679fa-0de6-11d0-a285-00aa003049e2                            |
| Syntax            | [**String(Object-Identifier)**](s-string-object-identifier.md) |
