# <a name="object-class-category-attribute"></a>物件類別類別目錄屬性

這個屬性包含類別類型，例如 abstract、輔助或結構化。



| 進入 | 值 |
|-------------------|---------------------------------------------------------------------------------|
| CN                | 物件類別-類別目錄                                                           |
| Ldap-顯示名稱 | objectClassCategory                                                             |
| 大小              | 4個位元組。 結構1、抽象2、輔助3。 不應使用類別88、0。 |
| 更新許可權  | 物件的設計工具會設定此值。                                |
| 更新頻率  | 此值永遠不會變更。                                                 |
| Attribute-Id      | 1.2.840.113556.1.2.370                                                          |
| 系統識別碼-Guid    | bf9679e6-0de6-11d0-a285-00aa003049e2                                            |
| Syntax            | [**列舉型別**](s-enumeration.md)                                            |
