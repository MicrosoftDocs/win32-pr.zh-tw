# <a name="ms-sql-characterset-attribute"></a>CharacterSet 屬性

SQL Server 目前實例的字元集。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | CharacterSet                  |
| Ldap-顯示名稱 | CharacterSet                  |
| 大小              | \-                                   |
| 更新許可權  | 此值是由系統所設定。     |
| 更新頻率  | 在系統啟動時。                   |
| Attribute-Id      | 1.2.840.113556.1.4.1370              |
| 系統識別碼-Guid    | 696177a6-ccee-11d2-9993-0000f87a57d4 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
