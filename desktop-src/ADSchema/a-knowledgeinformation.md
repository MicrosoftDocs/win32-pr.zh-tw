# <a name="knowledge-information-attribute"></a>Knowledge-Information 屬性

指定由特定目錄服務代理程式所主控之知識的累積描述。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | Knowledge-Information                       |
| Ldap-顯示名稱 | knowledgeInformation                        |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 2.5.4.2                                     |
| 系統識別碼-Guid    | 1677581f-47f3-11d1-a9c3-0000f80367c1        |
| Syntax            | [**String(Teletex)**](s-string-teletex.md) |
