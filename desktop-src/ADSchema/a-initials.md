# <a name="initials-attribute"></a>縮寫屬性

包含使用者全名部分的縮寫。 這可用來做為 Windows 通訊錄中的中間首。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | 縮寫                                    |
| Ldap-顯示名稱 | Initials                                    |
| 大小              | \-                                          |
| 更新許可權  | 網域系統管理員或帳戶擁有者。      |
| 更新頻率  | \-                                          |
| Attribute-Id      | 2.5.4.43                                    |
| 系統識別碼-Guid    | f0f8ff90-1191-11d0-a060-00aa006c33ed        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
