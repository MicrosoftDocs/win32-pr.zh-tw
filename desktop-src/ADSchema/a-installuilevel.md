# <a name="install-ui-level-attribute"></a>安裝-Ui 層級屬性

此屬性包含用於使用者介面之安裝 (層級) 類型的資訊。 可能的安裝層級如下所示： 2 INSTALLUILEVEL \_ 無無訊息安裝 (無訊息安裝) 3 INSTALLUILEVEL \_ 基本 (簡單安裝，錯誤處理) 4 INSTALLUILEVEL \_ 減少 (撰寫的 ui \_



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 安裝-Ui 層級                     |
| Ldap-顯示名稱 | installUiLevel                       |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.847               |
| 系統識別碼-Guid    | 96a7dd64-9118-11d1-aebc-0000f80367c1 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
