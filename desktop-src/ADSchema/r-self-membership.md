# <a name="self-membership-validated-writes"></a>Self-Membership 驗證的寫入

已驗證的寫入權限，可讓您在新增/移除一個帳戶時，更新群組的成員資格。



| 進入 | 值 |
|--------------|--------------------------------------|
| CN           | Self-Membership                      |
| 顯示名稱 | 新增/移除本身為成員            |
| 許可權-GUID  | bf9679c0-0de6-11d0-a285-00aa003049e2 |
