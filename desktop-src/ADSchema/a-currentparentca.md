| CN                | Current-父系-CA                       |
| Ldap-顯示名稱 | currentParentCA                         |
| 大小              | \-                                      |
| 更新許可權  | \-                                      |

| 系統識別碼-Guid    | 963d273f-48be-11d1-a9c3-0000f80367c1    |
