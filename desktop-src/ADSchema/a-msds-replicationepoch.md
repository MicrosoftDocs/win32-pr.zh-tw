# <a name="ms-ds-replicationepoch-attribute"></a>ReplicationEpoch 屬性

這是用來保存所有 Dc 正在複寫的 epoch。 Epoch 是網域具有特定名稱的時間週期。 新的 epoch 會在定義功能變數名稱稱變更時啟動。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ReplicationEpoch               |
| Ldap-顯示名稱 | ReplicationEpoch                |
| 大小              | \-                                   |
| 更新許可權  | 此值是由系統所設定。     |
| 更新頻率  | 只有在網域重建時。      |
| Attribute-Id      | 1.2.840.113556.1.4.1720              |
| 系統識別碼-Guid    | 08e3aa79-eb1c-45b5-af7b-8f94246c8e41 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
