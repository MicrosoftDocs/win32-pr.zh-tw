# <a name="validated-dns-host-name-validated-writes"></a>已驗證-DNS-主機名稱驗證的寫入

| 進入 | 值 |
|--------------|--------------------------------------|
| CN           | 已驗證-DNS-主機名稱              |
| 顯示名稱 | 已驗證寫入 DNS 主機名稱     |
| 許可權-GUID  | 72e39547-7b18-11d1-adef-00c04fd8d5cd |
