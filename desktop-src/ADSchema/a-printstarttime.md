# <a name="print-start-time-attribute"></a>列印-開始時間屬性

列印佇列開始服務工作的時間。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 列印-開始時間                     |
| Ldap-顯示名稱 | printStartTime                       |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.233               |
| 系統識別碼-Guid    | 281416c9-1968-11d0-a28f-00aa003049e2 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
