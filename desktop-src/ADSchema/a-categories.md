# <a name="categories-attribute"></a>Category 屬性

適用于此應用程式的類別目錄識別碼 (Guid) 清單。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | 類別                                  |
| Ldap-顯示名稱 | categories                                  |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.672                      |
| 系統識別碼-Guid    | 7bfdcb7e-4807-11d1-a9c3-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
