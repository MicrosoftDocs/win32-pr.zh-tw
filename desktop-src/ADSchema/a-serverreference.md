# <a name="server-reference-attribute"></a>Server-Reference 屬性

在網站電腦物件中找到。 包含網域命名內容中網域控制站的分辨名稱。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | Server-Reference                        |
| Ldap-顯示名稱 | serverReference                         |
| 大小              | \-                                      |
| 更新許可權  | 此值是由系統所設定。        |
| 更新頻率  | 每次建立網站時。             |
| Attribute-Id      | 1.2.840.113556.1.4.515                  |
| 系統識別碼-Guid    | 26d9736d-6070-11d1-a9c6-0000f80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
