# <a name="subschemasubentry-attribute"></a>SubSchemaSubEntry 屬性

定義類別或屬性之 ubschema 物件位置的分辨名稱。



| 進入 | 值 |
|-------------------|----------------------------------------------------------------------------------|
| CN                | SubSchemaSubEntry                                                                |
| Ldap-顯示名稱 | subSchemaSubEntry                                                                |
| 大小              | \-                                                                               |
| 更新許可權  | 架構系統管理員                                                             |
| 更新頻率  | 當 Active Directory 已載入或定義新的類別或屬性時。 |
| Attribute-Id      | 2.5.18.10                                                                        |
| 系統識別碼-Guid    | 9a7ad94d-ca53-11d1-bbd0-0080c76670c0                                             |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md)                                          |
