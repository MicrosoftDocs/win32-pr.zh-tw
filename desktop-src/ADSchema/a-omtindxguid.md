# <a name="omt-indx-guid-attribute"></a>OMT-Indx-Guid 屬性

連結-追蹤-物件-移動資料表專案的索引識別碼。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | OMT-Indx-Guid                                         |
| Ldap-顯示名稱 | oMTIndxGuid                                           |
| 大小              | 16 個位元組                                              |
| 更新許可權  | 此值是由系統所設定。                      |
| 更新頻率  | 每當檔案的連結變更時。                 |
| Attribute-Id      | 1.2.840.113556.1.4.333                                |
| 系統識別碼-Guid    | 1f0075fa-7e40-11d0-afd6-00c04fd930c9                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |



## <a name="implementations"></a>實作

-   [**Windows Server 2003**](#windows-server-2003)
