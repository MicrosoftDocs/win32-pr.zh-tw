# <a name="ms-ds-az-generate-audits-attribute"></a>ms DS-Az-產生-審核屬性

此為布林值欄位，指出是否需要開啟執行時間審核 (包括存取檢查的檢查等) 。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | ms DS-Az-產生-審核                |
| Ldap-顯示名稱 | AzGenerateAudits                   |
| 大小              | \-                                      |
| 更新許可權  | AzRoles 系統管理員                           |
| 更新頻率  | 在初始化或原則變更期間。 |
| Attribute-Id      | 1.2.840.113556.1.4.1805                 |
| 系統識別碼-Guid    | f90abab0-186c-4418-bb85-88447c87222a    |
| Syntax            | [**Boolean**](s-boolean.md)            |
