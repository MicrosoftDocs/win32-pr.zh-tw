|-------------------|---------------------------------------------------------------|
| CN                | FRS-時間-最後一個命令                                         |
| Ldap-顯示名稱 | fRSTimeLastCommand                                            |
| 大小              | \-                                                            |
| 更新許可權  | \-                                                            |
| 更新頻率  | \-                                                            |
| Attribute-Id      | 1.2.840.113556.1.4.880                                        |
| 系統識別碼-Guid    | 2a132583-9373-11d1-aebc-0000f80367c1                          |
| Syntax            | [**String(Generalized-Time)**](s-string-generalized-time.md) |
