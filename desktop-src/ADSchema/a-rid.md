# <a name="rid-attribute"></a>Rid 屬性

物件的相對識別碼。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | 擺脫                                         |
| Ldap-顯示名稱 | rid                                         |
| 大小              | 4 個位元組                                     |
| 更新許可權  | 此值是由系統所設定。            |
| 更新頻率  | 當需要 RID 的物件建立時。 |
| Attribute-Id      | 1.2.840.113556.1.4.153                      |
| 系統識別碼-Guid    | bf967a22-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**列舉型別**](s-enumeration.md)        |
