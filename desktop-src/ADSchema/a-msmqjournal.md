# <a name="msmq-journal-attribute"></a>MSMQ-Journal 屬性

指出從佇列中取出的訊息是否應該保留在日誌中。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | MSMQ-Journal                         |
| Ldap-顯示名稱 | mSMQJournal                          |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.918               |
| 系統識別碼-Guid    | 9a0dc321-c100-11d1-bbc5-0080c76670c0 |
| Syntax            | [**Boolean**](s-boolean.md)         |
