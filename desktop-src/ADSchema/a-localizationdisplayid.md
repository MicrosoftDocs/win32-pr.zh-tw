# <a name="localization-display-id-attribute"></a>當地語系化-顯示識別碼屬性

用來編制檔案 Extrts.mc 的索引，以取得物件的當地語系化 displayName 以進行 UI 用途。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 當地語系化-顯示識別碼              |
| Ldap-顯示名稱 | localizationDisplayId                |
| 大小              | 4 個位元組                              |
| 更新許可權  | 架構系統管理員                 |
| 更新頻率  | 永遠不應變更識別碼。       |
| Attribute-Id      | 1.2.840.113556.1.4.1353              |
| 系統識別碼-Guid    | a746f0d1-78d0-11d2-9916-0000f87a57d4 |
| Syntax            | [**列舉型別**](s-enumeration.md) |



## <a name="implementations"></a>實作
