# <a name="print-duplex-supported-attribute"></a>列印-雙工支援的屬性

指出印表機所擁有的雙工支援類型。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | 列印-雙工-支援                                |
| Ldap-顯示名稱 | printDuplexSupported                                  |
| 大小              | 4個位元組。 雙工支援 = 2。 單工 = 1 或0。 |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1311                               |
| 系統識別碼-Guid    | 281416cc-1968-11d0-a28f-00aa003049e2                  |
| Syntax            | [**Boolean**](s-boolean.md)                          |
