# <a name="comment-attribute-ad-schema"></a> (AD 架構) 的批註屬性

使用者的批註。 這個字串可以是 null 字串。



| 進入 | 值 |
|-------------------|--------------------------------------------------------------------------|
| CN                | 註解                                                                  |
| Ldap-顯示名稱 | info                                                                     |
| 大小              | \-                                                                       |
| 更新許可權  | 網域系統管理員或帳戶擁有者。                                   |
| 更新頻率  | 每次使用者或系統管理員需要在帳戶上新增留言。 |
| Attribute-Id      | 1.2.840.113556.1.2.81                                                    |
| 系統識別碼-Guid    | bf96793e-0de6-11d0-a285-00aa003049e2                                     |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                              |
