# <a name="transport-address-attribute-attribute"></a>傳輸-位址屬性

傳輸的網址類別型名稱。



| 進入 | 值 |
|-------------------|-----------------------------------------------------------------|
| CN                | 傳輸-位址-屬性                                     |
| Ldap-顯示名稱 | transportAddressAttribute                                       |
| 大小              | \-                                                              |
| 更新許可權  | 此值是由系統所設定。                                |
| 更新頻率  | 連接網站時。                                          |
| Attribute-Id      | 1.2.840.113556.1.4.895                                          |
| 系統識別碼-Guid    | c1dc867c-a261-11d1-b606-0000f80367c1                            |
| Syntax            | [**String(Object-Identifier)**](s-string-object-identifier.md) |
