# <a name="acs-aggregate-token-rate-per-user-attribute"></a>ACS-匯總-每位使用者的權杖速率屬性

任何使用者對所有流程可能具有的最大權杖速率。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ACS-匯總-權杖-每位使用者的費率    |
| Ldap-顯示名稱 | aCSAggregateTokenRatePerUser         |
| 大小              | 8 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.760               |
| 系統識別碼-Guid    | 7f56127d-5301-11d1-a9c5-0000f80367c1 |
| Syntax            | [**間隔**](s-interval.md)       |
