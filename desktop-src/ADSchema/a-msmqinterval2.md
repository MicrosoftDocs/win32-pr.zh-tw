| CN                | MSMQ-Interval2                       |
| Ldap-顯示名稱 | mSMQInterval2                        |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1309              |
| 系統識別碼-Guid    | 99b88f52-3b7b-11d2-90cc-00c04fd91ab1 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
