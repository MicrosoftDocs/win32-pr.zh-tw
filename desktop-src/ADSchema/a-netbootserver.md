# <a name="netboot-server-attribute"></a>netboot-Server 屬性

NetBoot 伺服器的分辨名稱。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | netboot-伺服器                          |
| Ldap-顯示名稱 | netbootServer                           |
| 大小              | \-                                      |
| 更新許可權  | 此值是由系統所設定。        |
| 更新頻率  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.860                  |
| 系統識別碼-Guid    | 07383081-91df-11d1-aebc-0000f80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
