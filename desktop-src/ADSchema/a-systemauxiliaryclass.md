# <a name="system-auxiliary-class-attribute"></a>系統輔助類別屬性

使用者無法修改的輔助類別清單。



| 進入 | 值 |
|-------------------|--------------------------------------------------------------------|
| CN                | 系統輔助類別                                             |
| Ldap-顯示名稱 | systemAuxiliaryClass                                               |
| 大小              | \-                                                                 |
| 更新許可權  | 架構系統管理員                                               |
| 更新頻率  | 建立類別或加入新的輔助類別。 |
| Attribute-Id      | 1.2.840.113556.1.4.198                                             |
| 系統識別碼-Guid    | bf967a43-0de6-11d0-a285-00aa003049e2                               |
| Syntax            | [**String(Object-Identifier)**](s-string-object-identifier.md)    |
