# <a name="usn-last-obj-rem-attribute"></a>USN-最後-Obj-Rem 屬性

包含從伺服器移除之最後一個非系統物件的更新序號 (USN) 。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | USN-最後-Obj-Rem                     |
| Ldap-顯示名稱 | uSNLastObjRem                        |
| 大小              | 8 個位元組                              |
| 更新許可權  | 此值是由系統所設定。     |
| 更新頻率  | 每當目錄物件變更時。 |
| Attribute-Id      | 1.2.840.113556.1.2.121               |
| 系統識別碼-Guid    | bf967a73-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**間隔**](s-interval.md)       |
