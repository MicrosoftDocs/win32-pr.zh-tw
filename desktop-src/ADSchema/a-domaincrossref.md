# <a name="domain-cross-ref-attribute"></a>網域交互參考屬性

這是從信任的網域物件到受信任網域的交互參照物件的參考。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | 網域交互參考                        |
| Ldap-顯示名稱 | domainCrossRef                          |
| 大小              | \-                                      |
| 更新許可權  | \-                                      |
| 更新頻率  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.472                  |
| 系統識別碼-Guid    | b000ea7b-a086-11d0-afdd-00c04fd930c9    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
