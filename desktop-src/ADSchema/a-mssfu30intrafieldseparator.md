| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | msSFU-30-欄位內-分隔符號              |
| Ldap-顯示名稱 | msSFU30IntraFieldSeparator                  |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.6.18.1.303                 |
| 系統識別碼-Guid    | 95b2aef0-27e4-4cb9-880a-a2d9a9ea23b8        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
