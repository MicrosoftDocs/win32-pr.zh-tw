# <a name="msmq-sites-attribute"></a>MSMQ-Sites 屬性

電腦所屬的網站清單 (網站 objectGUIDs 的陣列，通常不能有一個以上的 GUID) 。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | MSMQ-Sites                                            |
| Ldap-顯示名稱 | mSMQSites                                             |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.927                                |
| 系統識別碼-Guid    | 9a0dc32a-c100-11d1-bbc5-0080c76670c0                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
