# <a name="print-bin-names-attribute"></a>Print-Bin-Names 屬性

印表機 bin 名稱的清單。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | 列印-Bin 名稱                             |
| Ldap-顯示名稱 | printBinNames                               |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.237                      |
| 系統識別碼-Guid    | 281416cd-1968-11d0-a28f-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
