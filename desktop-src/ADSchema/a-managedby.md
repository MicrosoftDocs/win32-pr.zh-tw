# <a name="managed-by-attribute"></a>Managed-By 屬性

指派來管理此物件之使用者的分辨名稱。



| 進入 | 值 |
|-------------------|-------------------------------------------------------------------------------------|
| CN                | Managed-By                                                                          |
| Ldap-顯示名稱 | managedBy                                                                           |
| 大小              | \-                                                                                  |
| 更新許可權  | 網域系統管理員或帳戶擁有者。                                              |
| 更新頻率  | 當使用者的記錄建立時，以及每當受管理的物件需要變更時。 |
| Attribute-Id      | 1.2.840.113556.1.4.653                                                              |
| 系統識別碼-Guid    | 0296c120-40da-11d1-a9c0-0000f80367c1                                                |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md)                                             |
