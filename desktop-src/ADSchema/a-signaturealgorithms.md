# <a name="signature-algorithms-attribute"></a>Signature-Algorithms 屬性

這個屬性工作表示在驗證過程中，必須用來解碼數位簽章的演算法類型。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | Signature-Algorithms                        |
| Ldap-顯示名稱 | signatureAlgorithms                         |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.824                      |
| 系統識別碼-Guid    | 2a39c5b2-8960-11d1-aebc-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
