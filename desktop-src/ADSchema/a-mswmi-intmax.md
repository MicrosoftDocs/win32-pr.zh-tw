# <a name="ms-wmi-intmax-attribute"></a>intMax 屬性

WMI 32 位整數參數物件的最大值。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | intMax                        |
| Ldap-顯示名稱 | msWMI-IntMax                         |
| 大小              | 4 個位元組                              |
| 更新許可權  | 群組原則系統管理員           |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1629              |
| 系統識別碼-Guid    | fb920c2c-f294-4426-8ac1-d24b42aa2bce |
| Syntax            | [**列舉型別**](s-enumeration.md) |
