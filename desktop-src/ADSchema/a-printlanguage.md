# <a name="print-language-attribute"></a>Print-Language 屬性

支援的分頁描述語言 (例如 PostScript、PCL) 。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | Print-Language                              |
| Ldap-顯示名稱 | printLanguage                               |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.246                      |
| 系統識別碼-Guid    | 281416d6-1968-11d0-a28f-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
