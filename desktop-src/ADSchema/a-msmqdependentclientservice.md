# <a name="msmq-dependent-client-service-attribute"></a>MSMQ 相依用戶端-服務屬性

指出此伺服器是否提供相依的用戶端服務。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | MSMQ 相依用戶端-服務        |
| Ldap-顯示名稱 | mSMQDependentClientService           |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1239              |
| 系統識別碼-Guid    | 2df90d83-009f-11d2-aa4c-00c04fd7d83a |
| Syntax            | [**Boolean**](s-boolean.md)         |
