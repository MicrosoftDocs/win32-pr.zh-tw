# <a name="ms-ds-per-user-trust-tombstones-quota-attribute"></a>每位使用者-信任----配額屬性

用來在授權根據比對使用者 SID 時，強制刪除 Trusted-Domain 物件的每一使用者配額。



| 進入 | 值 |
|-------------------|-------------------------------------------|
| CN                | 每一使用者的 MS DS-信任     |
| Ldap-顯示名稱 | PerUserTrustTombstonesQuota          |
| 大小              | \-                                        |
| 更新許可權  | 網域系統管理員                      |
| 更新頻率  | 在建立樹系時，很少發生。 |
| Attribute-Id      | 1.2.840.113556.1.4.1790                   |
| 系統識別碼-Guid    | 8b70a6c6-50f9-4fa3-a71e-1ce03040449b      |
| Syntax            | [**列舉型別**](s-enumeration.md)      |
