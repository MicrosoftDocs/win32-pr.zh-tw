# <a name="www-page-other-attribute"></a>WWW-頁面-其他屬性

替代網頁的清單。



| 進入 | 值 |
|-------------------|-----------------------------------------------------------------------------|
| CN                | WWW-頁面-其他                                                              |
| Ldap-顯示名稱 | url                                                                         |
| 大小              | \-                                                                          |
| 更新許可權  | 網域系統管理員或帳戶擁有者。                                      |
| 更新頻率  | 當使用者的記錄建立時，以及每當網頁需要變更時。 |
| Attribute-Id      | 1.2.840.113556.1.4.749                                                      |
| 系統識別碼-Guid    | 9a9a0221-4a5b-11d1-a9c3-0000f80367c1                                        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                 |
