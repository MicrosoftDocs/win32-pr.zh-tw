# <a name="boolean-syntax"></a>布林語法

表示布林值。



| 進入 | 值 |
|--------------|--------------------------------------------------------------------------|
| 名稱         | Boolean                                                                  |
| 語法識別碼    | 2.5.5.8                                                                  |
| OM 識別碼        | 1                                                                        |
| MAPI 類型    | BOOLEAN                                                                  |
| ADS 類型     | ADSTYPE \_ 布林值                                                         |
| Variant 類型 | VT \_ BOOL                                                                 |
| SDS 類型     | [System.Boolean](/dotnet/api/system.boolean) |



## <a name="remarks"></a>備註

針對在篩選中包含布林語法屬性的查詢，請指定 "TRUE" 或 "FALSE" (例如 "myboolattr = TRUE" ) 。
