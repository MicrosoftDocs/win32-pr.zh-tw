# <a name="ms-ts-initial-program-attribute"></a>ms-TS-初始程式屬性

終端機服務會話初始程式會指定使用者登入終端機伺服器時，使用者想要自動啟動之應用程式的路徑和檔案名。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | ms-TS-Initial-Program                       |
| Ldap-顯示名稱 | msTSInitialProgram                          |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1990                     |
| 系統識別碼-Guid    | 9201ac6f-1d69-4dfb-802e-d95510109599        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
