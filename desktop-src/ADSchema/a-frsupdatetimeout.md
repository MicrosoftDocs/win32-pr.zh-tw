# <a name="frs-update-timeout-attribute"></a>FRS-更新-Timeout 屬性

在放棄之前，要等候完成更新的最長時間（以分鐘為單位）。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | FRS-更新-超時                   |
| Ldap-顯示名稱 | fRSUpdateTimeout                     |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.485               |
| 系統識別碼-Guid    | 1be8f172-a9ff-11d0-afe2-00c04fd930c9 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
