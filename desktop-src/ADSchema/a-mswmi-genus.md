# <a name="ms-wmi-genus-attribute"></a>拼出動植物屬性

識別編碼的物件類型。



| 進入 | 值 |
|-------------------|-----------------------------------------------------------------|
| CN                | 拼出動植物                                                    |
| Ldap-顯示名稱 | msWMI-拼出動植物                                                     |
| 大小              | 4個位元組。 1表示類別。 2表示實例。          |
| 更新許可權  | 網域系統管理員                                            |
| 更新頻率  | 只要加入或修改包含屬性的類別。 |
| Attribute-Id      | 1.2.840.113556.1.4.1677                                         |
| 系統識別碼-Guid    | 50c8673a-8f56-4614-9308-9e1340fb9af3                            |
| Syntax            | [**列舉型別**](s-enumeration.md)                            |
