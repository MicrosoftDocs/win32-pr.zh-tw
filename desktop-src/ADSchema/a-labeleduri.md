# <a name="labeleduri-attribute"></a>labeledURI 屬性

後面加上標籤的統一資源識別項。 標籤是用來描述 URI 所指向的資源，而這會用來做為人們可讀取的顯示名稱。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | labeledURI                                  |
| Ldap-顯示名稱 | labeledURI                                  |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.3.6.1.4.1.250.1.57                        |
| 系統識別碼-Guid    | c569bb46-c680-44bc-a273-e6c227d71b45        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
