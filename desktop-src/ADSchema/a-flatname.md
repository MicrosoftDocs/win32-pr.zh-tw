# <a name="flat-name-attribute"></a>Flat-Name 屬性

針對 Windows NT 網域，一般名稱是 NetBIOS 名稱。 若為非 Windows NT 網域的連結，則一般名稱是該網域的識別名稱，或為 **Null**。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | Flat-Name                                   |
| Ldap-顯示名稱 | flatName                                    |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.511                      |
| 系統識別碼-Guid    | b7b13117-b82e-11d0-afee-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
