# <a name="rpc-ns-annotation-attribute"></a>rpc-Ns-Annotation 屬性

Describs 指定 RPC 設定檔元素的字串。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | rpc-Ns-批註                           |
| Ldap-顯示名稱 | rpcNsAnnotation                             |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.366                      |
| 系統識別碼-Guid    | 88611bde-8cf4-11d0-afda-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
