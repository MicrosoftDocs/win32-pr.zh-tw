# <a name="ms-ds-default-quota-attribute"></a>ms DS-預設配額屬性

如果沒有涵蓋安全性主體的配額規格，則會套用至在 NC 中建立物件的安全性主體所適用的預設配額。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ms DS-預設配額                  |
| Ldap-顯示名稱 | DefaultQuota                    |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1846              |
| 系統識別碼-Guid    | 6818f726-674b-441b-8a3a-f40596374cea |
| Syntax            | [**列舉型別**](s-enumeration.md) |
