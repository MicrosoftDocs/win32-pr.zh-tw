# <a name="ipserviceprotocol-attribute"></a>ipServiceProtocol 屬性

包含服務對應的區段，此區段會儲存 UNIX 服務的通訊協定號碼。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ipServiceProtocol                    |
| Ldap-顯示名稱 | ipServiceProtocol                    |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.16                     |
| 系統識別碼-Guid    | cd96ec0b-1ed6-43b4-b26b-f170b645883f |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
