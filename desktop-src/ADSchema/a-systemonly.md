# <a name="system-only-attribute"></a>System-Only 屬性

布林值，指定是否只有 Active Directory 可以修改類別。 只能由目錄服務代理程式來建立或刪除僅限系統的類別。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | System-Only                          |
| Ldap-顯示名稱 | systemOnly                           |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.170               |
| 系統識別碼-Guid    | bf967a46-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Boolean**](s-boolean.md)         |
