# <a name="ms-wmi-propertyname-attribute"></a>ms-WMI-PropertyName 屬性

參數物件的目標原則物件名稱。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | ms-WMI-PropertyName                         |
| Ldap-顯示名稱 | msWMI-PropertyName                          |
| 大小              | 少於二十個字元。                |
| 更新許可權  | 群組原則系統管理員                  |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1641                     |
| 系統識別碼-Guid    | ab920883-e7f8-4d72-b4a0-c0449897509d        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
