# <a name="class-display-name-attribute"></a>類別-顯示名稱屬性

要顯示在對話方塊中的物件名稱。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | 類別-顯示名稱                          |
| Ldap-顯示名稱 | classDisplayName                            |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.610                      |
| 系統識別碼-Guid    | 548e1c22-dea6-11d0-b010-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
