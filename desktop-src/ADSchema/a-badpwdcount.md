# <a name="bad-pwd-count-attribute"></a>錯誤-Pwd-Count 屬性

使用者嘗試使用不正確的密碼登入帳戶的次數。 值為0表示值不明。



| 進入 | 值 |
|-------------------|-------------------------------------------|
| CN                | 錯誤-Pwd-計數                             |
| Ldap-顯示名稱 | badPwdCount                               |
| 大小              | 4 個位元組                                   |
| 更新許可權  | 此值是由系統所設定。          |
| 更新頻率  | 每次使用者輸入錯誤密碼時。 |
| Attribute-Id      | 1.2.840.113556.1.4.12                     |
| 系統識別碼-Guid    | bf96792e-0de6-11d0-a285-00aa003049e2      |
| Syntax            | [**列舉型別**](s-enumeration.md)      |
