# <a name="ms-pki-enrollment-flag-attribute"></a>ms-PKI-註冊-旗標屬性

包含註冊相關的旗標。



| 進入 | 值 |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-註冊-旗標                                                                            |
| Ldap-顯示名稱 | Mspki-certificate-name-flag-註冊-旗標                                                                             |
| 大小              | 4 個位元組                                                                                           |
| 更新許可權  | 網域系統管理員                                                                              |
| 更新頻率  | 當您編輯、建立或複製憑證範本時 (ms-PKI-憑證範本) 物件。 |
| Attribute-Id      | 1.2.840.113556.1.4.1430                                                                           |
| 系統識別碼-Guid    | d15ef7d8-f226-46db-ae79-b34e560bd12c                                                              |
| Syntax            | [**列舉型別**](s-enumeration.md)                                                              |
