# <a name="country-name-attribute"></a>Country-Name 屬性

使用者位址中的國家/地區。 國家/地區會以 ISO-3166 為基礎，以2個字元的代碼表示。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | Country-Name                                |
| Ldap-顯示名稱 | c                                           |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 2.5.4.6                                     |
| 系統識別碼-Guid    | bf967945-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
