# <a name="ms-ds-optional-feature-flags-attribute"></a>ms-DS-選用-功能旗標屬性

| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ms-DS-選用-功能-旗標         |
| Ldap-顯示名稱 | OptionalFeatureFlags            |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.2063              |
| 系統識別碼-Guid    | 8a0560c1-97b9-4811-9db7-dc061598965b |
| Syntax            | [**列舉型別**](s-enumeration.md) |
