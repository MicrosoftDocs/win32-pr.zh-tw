# <a name="ms-ds-cached-membership-time-stamp-attribute"></a>ms DS 快取-成員資格時間戳記屬性

在權杖評估期間，安全性帳戶管理員會使用以 ms DS 快取的 **成員資格時間戳記** 屬性進行群組擴充。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ms DS 快取-成員資格時間戳記   |
| Ldap-顯示名稱 | 預先快取-成員資格時間戳記    |
| 大小              | 8 個位元組                              |
| 更新許可權  | 此值是由系統所設定。     |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1442              |
| 系統識別碼-Guid    | 3566bf1f-beee-4dcb-8abe-ef89fcfec6c1 |
| Syntax            | [**間隔**](s-interval.md)       |
