# <a name="managed-objects-attribute"></a>Managed-Objects 屬性

包含由使用者管理的物件清單。 列出的物件是屬性 managedBy 屬性設定為此使用者的物件。 清單中的每個專案都是 managed 物件的連結參考。



| 進入 | 值 |
|-------------------|------------------------------------------------------------------------------------|
| CN                | Managed-Objects                                                                    |
| Ldap-顯示名稱 | Managedobjects 之下                                                                     |
| 大小              | \-                                                                                 |
| 更新許可權  | 此值是由系統所設定。                                                   |
| 更新頻率  | 當使用者記錄建立時，以及每當受管理物件需要變更時。 |
| Attribute-Id      | 1.2.840.113556.1.4.654                                                             |
| 系統識別碼-Guid    | 0296c124-40da-11d1-a9c0-0000f80367c1                                               |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md)                                            |
