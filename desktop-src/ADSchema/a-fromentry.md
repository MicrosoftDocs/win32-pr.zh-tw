# <a name="from-entry-attribute"></a>From-Entry 屬性

如果物件是可寫入的，則此為已建立的屬性，如果是唯讀的 **，則為** **FALSE** ，例如，GC 複本實例。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | From-Entry                           |
| Ldap-顯示名稱 | fromEntry                            |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.910               |
| 系統識別碼-Guid    | 9a7ad949-ca53-11d1-bbd0-0080c76670c0 |
| Syntax            | [**Boolean**](s-boolean.md)         |
