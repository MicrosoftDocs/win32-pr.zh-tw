# <a name="domain-certificate-authorities-attribute"></a>網域-憑證授權單位單位屬性

指定網域的憑證授權單位單位清單。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | 網域-憑證-授權單位          |
| Ldap-顯示名稱 | domainCAs                               |
| 大小              | \-                                      |
| 更新許可權  | \-                                      |
| 更新頻率  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.668                  |
| 系統識別碼-Guid    | 7bfdcb7a-4807-11d1-a9c3-0000f80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
