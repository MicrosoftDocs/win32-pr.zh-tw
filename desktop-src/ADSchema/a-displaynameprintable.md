# <a name="display-name-printable-attribute"></a>顯示名稱-可列印屬性

物件的可列印顯示名稱。 可列印的顯示名稱通常是使用者名字、中間名縮寫和姓氏的組合。



| 進入 | 值 |
|-------------------|----------------------------------------|
| CN                | 顯示-名稱-可列印                 |
| Ldap-顯示名稱 | displayNamePrintable                   |
| 大小              | \-                                     |
| 更新許可權  | 網域系統管理員或帳戶擁有者。 |
| 更新頻率  | \-                                     |
| Attribute-Id      | 1.2.840.113556.1.2.353                 |
| 系統識別碼-Guid    | bf967954-0de6-11d0-a285-00aa003049e2   |
| Syntax            | [**String(IA5)**](s-string-ia5.md)    |
