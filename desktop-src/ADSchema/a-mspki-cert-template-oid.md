# <a name="ms-pki-cert-template-oid-attribute"></a>ms-PKI-Cert-Template-OID 屬性

指定憑證範本的物件識別碼。



| 進入 | 值 |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-Cert-Template-OID                                                                          |
| Ldap-顯示名稱 | Mspki-certificate-name-flag-Cert-Template-OID                                                                           |
| 大小              | 64位元組                                                                                          |
| 更新許可權  | 網域系統管理員                                                                              |
| 更新頻率  | 當您編輯、建立或複製憑證範本時 (ms-PKI-憑證範本) 物件。 |
| Attribute-Id      | 1.2.840.113556.1.4.1436                                                                           |
| 系統識別碼-Guid    | 3164c36a-ba26-468c-8bda-c1e5cc256728                                                              |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                                       |
