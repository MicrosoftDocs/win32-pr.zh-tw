# <a name="ms-ts-default-to-main-printer-attribute"></a>ms-TS-預設為主要印表機屬性

「終端服務預設列印到主要印表機」指定是否自動列印到用戶端的預設印表機。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ms-TS-Default-To-Main-Printer        |
| Ldap-顯示名稱 | msTSDefaultToMainPrinter             |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1988              |
| 系統識別碼-Guid    | c0ffe2bd-cacf-4dc7-88d5-61e9e95766f6 |
| Syntax            | [**Boolean**](s-boolean.md)         |
