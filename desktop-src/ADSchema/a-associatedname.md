# <a name="associatedname-attribute"></a>associatedName 屬性

AssociatedName 屬性類型會指定與 DNS 網域相關聯之組織 DIT 中的專案。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | associatedName                          |
| Ldap-顯示名稱 | associatedName                          |
| 大小              | \-                                      |
| 更新許可權  | \-                                      |
| 更新頻率  | \-                                      |
| Attribute-Id      | 0.9.2342.19200300.100.1.38              |
| 系統識別碼-Guid    | f7fbfc45-85ab-42a4-a435-780e62f7858b    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
