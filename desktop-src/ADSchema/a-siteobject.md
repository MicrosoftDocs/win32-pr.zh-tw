# <a name="site-object-attribute"></a>Site-Object 屬性

此子網所屬網站的分辨名稱。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | Site-Object                             |
| Ldap-顯示名稱 | siteObject                              |
| 大小              | \-                                      |
| 更新許可權  | 網域系統管理員                    |
| 更新頻率  | 每次建立網站時。             |
| Attribute-Id      | 1.2.840.113556.1.4.512                  |
| 系統識別碼-Guid    | 3e10944c-c354-11d0-aff8-0000f80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
