# <a name="priority-attribute"></a>Priority 屬性

進程的目前優先權 (、列印工作等) 。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 優先順序                             |
| Ldap-顯示名稱 | priority                             |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.231               |
| 系統識別碼-Guid    | 281416c7-1968-11d0-a28f-00aa003049e2 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
