# <a name="bridgehead-server-list-bl-attribute"></a>橋頭-伺服器-清單-BL 屬性

作為複寫之橋頭的伺服器清單。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | 橋頭-伺服器-清單-BL               |
| Ldap-顯示名稱 | bridgeheadServerListBL                  |
| 大小              | \-                                      |
| 更新許可權  | 此值是由系統所設定。        |
| 更新頻率  | 每次設定網站時。               |
| Attribute-Id      | 1.2.840.113556.1.4.820                  |
| 系統識別碼-Guid    | d50c2cdb-8951-11d1-aebc-0000f80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
