# <a name="uniqueidentifier-attribute"></a>uniqueIdentifier 屬性

UniqueIdentifier 屬性型別會指定目錄中所代表物件的「唯一識別碼」。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | 唯一                            |
| Ldap-顯示名稱 | 唯一                            |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.44                  |
| 系統識別碼-Guid    | ba0184c7-38c5-4bed-a526-75421470580c        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
