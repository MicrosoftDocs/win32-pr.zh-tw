# <a name="msi-script-name-attribute"></a>Msi-腳本-名稱屬性

此應用程式的 Microsoft installer 公告腳本檔名稱。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | Msi-腳本-名稱                             |
| Ldap-顯示名稱 | msiScriptName                               |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.845                      |
| 系統識別碼-Guid    | 96a7dd62-9118-11d1-aebc-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
