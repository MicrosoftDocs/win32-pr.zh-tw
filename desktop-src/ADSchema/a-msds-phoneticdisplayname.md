# <a name="ms-ds-phonetic-display-name-attribute"></a>ms-DS-注音-顯示名稱屬性

物件的注音標示顯示名稱。 如果沒有語音顯示名稱，則會使用現有的顯示名稱。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | ms-DS-Phonetic-Display-Name                 |
| Ldap-顯示名稱 | msDS-PhoneticDisplayName                    |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1946                     |
| 系統識別碼-Guid    | e21a94e4-2d66-4ce5-b30d-0ef87a776ff0        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
