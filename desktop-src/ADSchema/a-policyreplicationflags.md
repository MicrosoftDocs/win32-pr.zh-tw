# <a name="policy-replication-flags-attribute"></a>原則-複寫-旗標屬性

決定要將哪些 LSA 屬性複寫到用戶端。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 原則-複寫-旗標             |
| Ldap-顯示名稱 | policyReplicationFlags               |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.633               |
| 系統識別碼-Guid    | 19405b96-3cfa-11d1-a9c0-0000f80367c1 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
