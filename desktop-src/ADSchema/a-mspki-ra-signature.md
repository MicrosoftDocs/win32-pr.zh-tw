# <a name="ms-pki-ra-signature-attribute"></a>ms-PKI-RA-Signature 屬性

指定註冊要求中所需的註冊註冊授權單位簽章數目。



| 進入 | 值 |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-RA-簽章                                                                               |
| Ldap-顯示名稱 | Mspki-certificate-name-flag-RA-簽章                                                                                |
| 大小              | 4 個位元組                                                                                           |
| 更新許可權  | 網域系統管理員                                                                              |
| 更新頻率  | 當您編輯、建立或複製憑證範本時 (ms-PKI-憑證範本) 物件。 |
| Attribute-Id      | 1.2.840.113556.1.4.1429                                                                           |
| 系統識別碼-Guid    | fe17e04b-937d-4f7e-8e0e-9292c8d5683e                                                              |
| Syntax            | [**列舉型別**](s-enumeration.md)                                                              |
