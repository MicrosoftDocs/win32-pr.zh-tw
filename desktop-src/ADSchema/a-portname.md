# <a name="port-name-attribute"></a>Port-Name 屬性

埠名稱的清單。 例如，適用于印表機埠或通訊埠。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | Port-Name                                   |
| Ldap-顯示名稱 | portName                                    |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.228                      |
| 系統識別碼-Guid    | 281416c4-1968-11d0-a28f-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
