# <a name="show-in-advanced-view-only-attribute"></a>顯示為僅限視圖的屬性

如果這個屬性是在 UI 的 Advanced 模式中顯示，則為 **TRUE** 。



| 進入 | 值 |
|-------------------|-----------------------------------------------------------------------------------------|
| CN                | 顯示-內建-僅限視圖                                                              |
| Ldap-顯示名稱 | showInAdvancedViewOnly                                                                  |
| 大小              | 4 個位元組                                                                                 |
| 更新許可權  | 網域系統管理員                                                                    |
| 更新頻率  | 當建立物件時，或屬性的視圖層面需要變更時。 |
| Attribute-Id      | 1.2.840.113556.1.2.169                                                                  |
| 系統識別碼-Guid    | bf967984-0de6-11d0-a285-00aa003049e2                                                    |
| Syntax            | [**Boolean**](s-boolean.md)                                                            |
