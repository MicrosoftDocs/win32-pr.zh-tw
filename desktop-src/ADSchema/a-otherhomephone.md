# <a name="phone-home-other-attribute"></a>電話-首頁-其他屬性

備用家用電話號碼的清單。



| 進入 | 值 |
|-------------------|----------------------------------------------------------------------------------|
| CN                | 電話-首頁-其他                                                                 |
| Ldap-顯示名稱 | otherHomePhone                                                                   |
| 大小              | \-                                                                               |
| 更新許可權  | 網域系統管理員或帳戶擁有者。                                           |
| 更新頻率  | 當使用者的記錄建立時，以及每次電話號碼需要變更時。 |
| Attribute-Id      | 1.2.840.113556.1.2.277                                                           |
| 系統識別碼-Guid    | f0f8ffa2-1191-11d0-a060-00aa006c33ed                                             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                      |
