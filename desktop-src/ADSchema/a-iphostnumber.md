# <a name="iphostnumber-attribute"></a>ipHostNumber 屬性

以點分隔的十進位標記法包含主機的 IP 位址，並省略前置零。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ipHostNumber                         |
| Ldap-顯示名稱 | ipHostNumber                         |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.19                     |
| 系統識別碼-Guid    | de8bb721-85dc-4fde-b687-9657688e667e |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
