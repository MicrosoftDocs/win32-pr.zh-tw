# <a name="shell-property-pages-attribute"></a>Shell-屬性頁屬性

用來管理 Active Directory 物件的屬性頁的順序編號和 GUID。 您可以從 Windows shell 存取這些屬性頁。 如需詳細資訊，請參閱檔，擴充目錄物件的消費者介面。



| 進入 | 值 |
|-------------------|------------------------------------------------|
| CN                | Shell-屬性頁                           |
| Ldap-顯示名稱 | shellPropertyPages                             |
| 大小              | \-                                             |
| 更新許可權  | 網域系統管理員或應用程式開發人員。 |
| 更新頻率  | 每當加入或移除屬性頁時。  |
| Attribute-Id      | 1.2.840.113556.1.4.563                         |
| 系統識別碼-Guid    | 52458039-ca6a-11d0-afff-0000f80367c1           |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)    |
