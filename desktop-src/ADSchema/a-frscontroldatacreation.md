| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | FRS-控制-資料建立                   |
| Ldap-顯示名稱 | fRSControlDataCreation                      |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.871                      |
| 系統識別碼-Guid    | 2a13257a-9373-11d1-aebc-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
