# <a name="package-name-attribute"></a>Package-Name 屬性

此屬性包含應用程式的部署名稱。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | Package-Name                                |
| Ldap-顯示名稱 | packageName                                 |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.326                      |
| 系統識別碼-Guid    | 7d6c0e98-7e20-11d0-afd6-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
