# <a name="locality-name-attribute"></a>Locality-Name 屬性

表示位置的名稱，例如城鎮或縣/市。



| 進入 | 值 |
|-------------------|------------------------------------------------------------------------------|
| CN                | Locality-Name                                                                |
| Ldap-顯示名稱 | l                                                                            |
| 大小              | \-                                                                           |
| 更新許可權  | 網域系統管理員或帳戶擁有者。                                       |
| 更新頻率  | 當使用者的記錄建立時，以及每次需要變更位置時。 |
| Attribute-Id      | 2.5.4.7                                                                      |
| 系統識別碼-Guid    | bf9679a2-0de6-11d0-a285-00aa003049e2                                         |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                  |
