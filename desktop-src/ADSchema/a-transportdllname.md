# <a name="transport-dll-name-attribute"></a>傳輸-DLL-名稱屬性

將管理傳輸的 DLL 名稱。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | 傳輸-DLL-名稱                          |
| Ldap-顯示名稱 | transportDLLName                            |
| 大小              | \-                                          |
| 更新許可權  | 此值是由系統所設定。            |
| 更新頻率  | 連接網站時。                      |
| Attribute-Id      | 1.2.840.113556.1.4.789                      |
| 系統識別碼-Guid    | 26d97372-6070-11d1-a9c6-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
