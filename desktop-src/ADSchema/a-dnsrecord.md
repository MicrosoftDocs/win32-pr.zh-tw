# <a name="dns-record-attribute"></a>Dns-Record 屬性

用來將二進位 DNS 資源記錄儲存在 DNS 物件上。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | Dns-Record                                            |
| Ldap-顯示名稱 | dnsRecord                                             |
| 大小              | \-                                                    |
| 更新許可權  | 此值是由系統所設定。                      |
| 更新頻率  | DNS 資源記錄變更時。                 |
| Attribute-Id      | 1.2.840.113556.1.4.382                                |
| 系統識別碼-Guid    | e0fa1e69-9b45-11d0-afdd-00c04fd930c9                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |



## <a name="implementations"></a>實作
