# <a name="gecos-attribute"></a>gecos 屬性

包含儲存在 GECOS 欄位中的資訊。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | gecos                                |
| Ldap-顯示名稱 | gecos                                |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.2                      |
| 系統識別碼-Guid    | a3e03f1f-1d55-4253-a0af-30c2a784e46e |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
