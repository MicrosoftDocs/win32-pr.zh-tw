# <a name="acs-maximum-sdu-size-attribute"></a>ACS-SDU 大小上限屬性

**ACS-最大 SDU 大小** 屬性僅供內部使用。 以 RFC2210 為基礎。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ACS-最大-SDU-大小                 |
| Ldap-顯示名稱 | aCSMaximumSDUSize                    |
| 大小              | 8 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1314              |
| 系統識別碼-Guid    | 87a2d8f9-3b90-11d2-90cc-00c04fd91ab1 |
| Syntax            | [**間隔**](s-interval.md)       |
