# <a name="revision-attribute"></a>修訂屬性

安全描述項或其他變更的修訂層級。 僅用於 sam 伺服器和 ds ui 設定物件。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 修訂版                             |
| Ldap-顯示名稱 | revision                             |
| 大小              | 4 個位元組                              |
| 更新許可權  | 此值是由系統所設定。     |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.145               |
| 系統識別碼-Guid    | bf967a21-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
