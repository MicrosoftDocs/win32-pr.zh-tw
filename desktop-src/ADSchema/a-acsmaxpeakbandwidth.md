# <a name="acs-max-peak-bandwidth-attribute"></a>ACS-最大尖峰頻寬屬性

可以保留的尖峰頻寬。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ACS-最大尖峰頻寬               |
| Ldap-顯示名稱 | aCSMaxPeakBandwidth                  |
| 大小              | 8 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.767               |
| 系統識別碼-Guid    | 7f561284-5301-11d1-a9c5-0000f80367c1 |
| Syntax            | [**間隔**](s-interval.md)       |
