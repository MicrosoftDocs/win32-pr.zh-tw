# <a name="ipsec-id-attribute"></a>Ipsec 識別碼屬性

**IPSEC ID** 屬性僅供內部使用。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | Ipsec 識別碼                                    |
| Ldap-顯示名稱 | ipsecID                                     |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.621                      |
| 系統識別碼-Guid    | b40ff81d-427a-11d1-a9c2-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
