# <a name="extended-chars-allowed-attribute"></a>擴充字元-允許的屬性

指出這個屬性的值是否允許擴充字元。 只適用于 IA5、Numeric、可列印和 Teletex 字串屬性。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 擴充字元-允許               |
| Ldap-顯示名稱 | extendedCharsAllowed                 |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.380               |
| 系統識別碼-Guid    | bf967966-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Boolean**](s-boolean.md)         |
