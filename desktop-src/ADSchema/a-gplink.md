# <a name="gp-link-attribute"></a>GP-Link 屬性

群組原則選項的排序清單。 每個選項都是 **DWORD**。 使用 UNICODE 字串是很方便的。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | GP-Link                                     |
| Ldap-顯示名稱 | gPLink                                      |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.891                      |
| 系統識別碼-Guid    | f30e3bbe-9ff0-11d1-b603-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
