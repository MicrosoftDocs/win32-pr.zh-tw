# <a name="msmq-recipient-formatname-attribute"></a>MSMQ-收件者-Msmq.formatname 屬性

當做 MSMQ 自訂收件者類別中的收件者格式名稱使用。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | MSMQ-收件者-Msmq.formatname                   |
| Ldap-顯示名稱 | msMQ-收件者-Msmq.formatname                   |
| 大小              | 1到255個字元。                        |
| 更新許可權  | 佇列擁有者。                            |
| 更新頻率  | 只有在移動外部佇列之後。    |
| Attribute-Id      | 1.2.840.113556.1.4.1695                     |
| 系統識別碼-Guid    | 3bfe6748-b544-485a-b067-1b310c4334bf        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
