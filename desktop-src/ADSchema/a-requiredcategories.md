# <a name="required-categories-attribute"></a>Required-Categories 屬性

此屬性包含元件類別識別碼的清單，物件 (例如) 需要執行的應用程式。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | Required-Categories                                   |
| Ldap-顯示名稱 | requiredCategories                                    |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.321                                |
| 系統識別碼-Guid    | 7d6c0e93-7e20-11d0-afd6-00c04fd930c9                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
