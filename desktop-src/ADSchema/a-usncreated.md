# <a name="usn-created-attribute"></a>USN-Created 屬性

更新序號 (USN) 在建立物件時指派。 另請參閱 [**USN 變更**](a-usnchanged.md)。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | USN-Created                          |
| Ldap-顯示名稱 | uSNCreated                           |
| 大小              | 8 個位元組                              |
| 更新許可權  | 此值是由系統所設定。     |
| 更新頻率  | 建立物件時。          |
| Attribute-Id      | 1.2.840.113556.1.2.19                |
| 系統識別碼-Guid    | bf967a70-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**間隔**](s-interval.md)       |
