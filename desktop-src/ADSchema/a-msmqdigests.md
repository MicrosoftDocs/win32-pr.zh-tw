# <a name="msmq-digests-attribute"></a>MSMQ-Digests 屬性

屬性 mSMQ 簽署憑證中對應憑證的摘要陣列。 它們是用來將摘要對應至憑證。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | MSMQ-Digests                                          |
| Ldap-顯示名稱 | mSMQDigests                                           |
| 大小              | 每個摘要都是16個位元組。                              |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.948                                |
| 系統識別碼-Guid    | 9a0dc33c-c100-11d1-bbc5-0080c76670c0                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
