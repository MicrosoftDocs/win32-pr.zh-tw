# <a name="acs-max-size-of-rsvp-log-file-attribute"></a>ACS-最大-最大值-記錄檔屬性

RSVP 記錄檔的大小上限（以位元組為單位）。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ACS-大小上限-記錄檔        |
| Ldap-顯示名稱 | aCSMaxSizeOfRSVPLogFile              |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.775               |
| 系統識別碼-Guid    | 1cb3559d-56d0-11d1-a9c6-0000f80367c1 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
