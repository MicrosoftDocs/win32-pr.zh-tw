# <a name="ms-ts-max-connection-time-attribute"></a>ms-TS-最大連線時間屬性

| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ms-TS-Max-Connection-Time            |
| Ldap-顯示名稱 | msTSMaxConnectionTime                |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1982              |
| 系統識別碼-Guid    | 1d960ee2-6464-4e95-a781-e3b5cd5f9588 |
