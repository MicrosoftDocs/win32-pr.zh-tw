# <a name="ms-fve-volumeguid-attribute"></a>FVE-VolumeGuid 屬性

| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | ms-FVE-VolumeGuid                                     |
| Ldap-顯示名稱 | msFVE-VolumeGuid                                      |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1998                               |
| 系統識別碼-Guid    | 85e5a5cf-dcee-4075-9cfd-ac9db6a2f245                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
