# <a name="admin-count-attribute"></a>Admin-Count 屬性

指出指定的物件已將其 Acl 變更為系統中較安全的值，因為它是直接或可轉移)  (其中一個系統管理群組的成員。



| 進入 | 值 |
|-------------------|-----------------------------------------------------|
| CN                | Admin-Count                                         |
| Ldap-顯示名稱 | adminCount                                          |
| 大小              | 4 個位元組                                             |
| 更新許可權  | 此值是由系統所設定。                    |
| 更新頻率  | 當物件加入至系統管理群組時。 |
| Attribute-Id      | 1.2.840.113556.1.4.150                              |
| 系統識別碼-Guid    | bf967918-0de6-11d0-a285-00aa003049e2                |
| Syntax            | [**列舉型別**](s-enumeration.md)                |
