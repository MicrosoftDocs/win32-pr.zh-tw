# <a name="ms-dfs-link-identity-guid-v2-attribute"></a>ms-DFS-Link-Identity-GUID-v2 屬性

僅建立連結時才會設定。 只要沒有以相同名稱的另一個連結取代連結，就可以在重新命名或移動時穩定。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | ms-DFS-Link-Identity-GUID-v2                          |
| Ldap-顯示名稱 | msDFS-LinkIdentityGUIDv2                              |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.2041                               |
| 系統識別碼-Guid    | edb027f3-5726-4dee-8d4e-dbf07e1ad1f1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
