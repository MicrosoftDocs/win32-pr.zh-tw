# <a name="friendly-names-attribute"></a>Friendly-Names 屬性

目錄所支援的預設顯示名稱定義清單。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | Friendly-Names                              |
| Ldap-顯示名稱 | friendlyNames                               |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.682                      |
| 系統識別碼-Guid    | 7bfdcb88-4807-11d1-a9c3-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
