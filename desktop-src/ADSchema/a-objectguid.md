# <a name="object-guid-attribute"></a>Object-Guid 屬性

物件的唯一識別碼。



| 進入 | 值 |
|-------------------|---------------------------------------------------------------------|
| CN                | Object-Guid                                                         |
| Ldap-顯示名稱 | objectGUID                                                          |
| 大小              | 16 個位元組                                                            |
| 更新許可權  | 此值是由系統所設定。                                    |
| 更新頻率  | 此值是在建立物件時設定的，而且無法變更。 |
| Attribute-Id      | 1.2.840.113556.1.4.2                                                |
| 系統識別碼-Guid    | bf9679e7-0de6-11d0-a285-00aa003049e2                                |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md)               |
