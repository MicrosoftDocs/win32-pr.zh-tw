# <a name="frs-root-path-attribute"></a>FRS-根目錄路徑屬性

複寫檔案系統樹狀目錄的根路徑。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | FRS-根目錄路徑                               |
| Ldap-顯示名稱 | fRSRootPath                                 |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.487                      |
| 系統識別碼-Guid    | 1be8f174-a9ff-11d0-afe2-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
