# <a name="ms-pki-certificate-name-flag-attribute"></a>ms-PKI-憑證名稱-旗標屬性

包含在已發行的憑證中用來建立主體名稱的相關旗標。



| 進入 | 值 |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-憑證名稱-旗標                                                                      |
| Ldap-顯示名稱 | Mspki-certificate-name-flag-憑證-名稱-旗標                                                                       |
| 大小              | 4 個位元組                                                                                           |
| 更新許可權  | 網域系統管理員                                                                              |
| 更新頻率  | 當您編輯、建立或複製憑證範本時 (ms-PKI-憑證範本) 物件。 |
| Attribute-Id      | 1.2.840.113556.1.4.1432                                                                           |
| 系統識別碼-Guid    | ea1dddc4-60ff-416e-8cc0-17cee534bce7                                                              |
| Syntax            | [**列舉型別**](s-enumeration.md)                                                              |



## <a name="implementations"></a>實作

-   [**Windows Server 2003 R2**](#windows-server-2003-r2)
-   [**Windows Server 2008**](#windows-server-2008)
