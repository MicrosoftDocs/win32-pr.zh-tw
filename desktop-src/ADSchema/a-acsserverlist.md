# <a name="acs-server-list-attribute"></a>ACS-伺服器清單屬性

包含允許執行 ACS 之 Windows NT 伺服器的 DNS 名稱。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | ACS-伺服器清單                             |
| Ldap-顯示名稱 | aCSServerList                               |
| 大小              | \-                                          |
| 更新許可權  | 網域系統管理員                        |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1312                     |
| 系統識別碼-Guid    | 7cbd59a5-3b90-11d2-90cc-00c04fd91ab1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |



## <a name="implementations"></a>實作
