# <a name="ms-ds-machine-account-quota-attribute"></a>MS DS-電腦-帳戶-配額屬性

允許使用者在網域中建立的電腦帳戶數目。



| 進入 | 值 |
|-------------------|--------------------------------------------------|
| CN                | MS-DS-電腦帳戶-配額                      |
| Ldap-顯示名稱 | MachineAccountQuota                        |
| 大小              | 4 個位元組                                          |
| 更新許可權  | 網域系統管理員                             |
| 更新頻率  | 每當網域的配額需要變更時。 |
| Attribute-Id      | 1.2.840.113556.1.4.1411                          |
| 系統識別碼-Guid    | d064fb68-1480-11d3-91c1-0000f87a57d4             |
| Syntax            | [**列舉型別**](s-enumeration.md)             |
