# <a name="ms-wmi-intdefault-attribute"></a>intDefault 屬性

WMI 32 位整數參數物件的預設值。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | intDefault                    |
| Ldap-顯示名稱 | msWMI-IntDefault                     |
| 大小              | 4 個位元組                              |
| 更新許可權  | 群組原則系統管理員           |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1628              |
| 系統識別碼-Guid    | 1b0c07f8-76dd-4060-a1e1-70084619dc90 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
