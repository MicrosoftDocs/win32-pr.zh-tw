# <a name="partial-attribute-deletion-list-attribute"></a>部分屬性-刪除清單屬性

追蹤部分複本的內部複寫狀態 (也就是在 Gc) 上。 部分複本 NC 物件的屬性。 當 GC 正在從其部分複本 Nc 中的物件中移除屬性時使用。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | 部分屬性-刪除-清單                       |
| Ldap-顯示名稱 | partialAttributeDeletionList                          |
| 大小              | \-                                                    |
| 更新許可權  | 此值是由系統所設定。                      |
| 更新頻率  | 複寫期間                                    |
| Attribute-Id      | 1.2.840.113556.1.4.663                                |
| 系統識別碼-Guid    | 28630ec0-41d5-11d1-a9c1-0000f80367c1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
