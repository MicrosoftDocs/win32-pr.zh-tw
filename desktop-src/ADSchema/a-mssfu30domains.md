| Ldap-顯示名稱 | msSFU30Domains                       |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |

| 系統識別碼-Guid    | 93095ed3-6f30-4bdd-b734-65d569f5f7c9 |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
