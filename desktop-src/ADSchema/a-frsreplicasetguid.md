# <a name="frs-replica-set-guid-attribute"></a>FRS-複本集-GUID 屬性

識別 FRS 複本集的 GUID。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | FRS-複本集-GUID                                  |
| Ldap-顯示名稱 | fRSReplicaSetGUID                                     |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.533                                |
| 系統識別碼-Guid    | 5245801a-ca6a-11d0-afff-0000f80367c1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
