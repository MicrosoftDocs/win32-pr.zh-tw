# <a name="service-principal-name-attribute"></a>服務主體名稱屬性

用來在這部電腦上與服務實例進行相互驗證的主體名稱清單。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | 服務主體名稱                      |
| Ldap-顯示名稱 | servicePrincipalName                        |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.771                      |
| 系統識別碼-Guid    | f3a64788-5306-11d1-a9c5-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
