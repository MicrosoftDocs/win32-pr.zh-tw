# <a name="admin-display-name-attribute"></a>Admin-顯示名稱屬性

要在系統管理員畫面上顯示的名稱。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | 系統管理員-顯示名稱                          |
| Ldap-顯示名稱 | adminDisplayName                            |
| 大小              | \-                                          |
| 更新許可權  | 網域系統管理員                        |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.194                      |
| 系統識別碼-Guid    | bf96791a-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
