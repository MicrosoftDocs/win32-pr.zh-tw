# <a name="ms-ts-max-idle-time-attribute"></a>ms-TS-最大閒置時間屬性

終端機服務會話最大閒置時間是終端機服務會話可保持閒置的最長時間（以分鐘為單位）。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ms-TS-Max-Idle-Time                  |
| Ldap-顯示名稱 | msTSMaxIdleTime                      |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1983              |
| 系統識別碼-Guid    | ff739e9c-6bb7-460e-b221-e250f3de0f95 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
