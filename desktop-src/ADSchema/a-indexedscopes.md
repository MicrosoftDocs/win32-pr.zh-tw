# <a name="indexedscopes-attribute"></a>IndexedScopes 屬性

目錄範圍清單 (例如，C： \\ 或 D： \) 已編制索引。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | IndexedScopes                               |
| Ldap-顯示名稱 | indexedScopes                               |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.681                      |
| 系統識別碼-Guid    | 7bfdcb87-4807-11d1-a9c3-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
