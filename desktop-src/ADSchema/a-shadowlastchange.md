| Ldap-顯示名稱 | shadowLastChange                     |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |

| 系統識別碼-Guid    | f8f2689c-29e8-4843-8177-e8b98e15eeac |
