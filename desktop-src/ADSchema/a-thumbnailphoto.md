# <a name="picture-attribute"></a>圖片屬性

使用者的影像。 建議使用以空格有效率的格式，例如 JPEG 或 GIF。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | Picture                                               |
| Ldap-顯示名稱 | thumbnailPhoto                                        |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 2.16.840.1.113730.3.1.35                              |
| 系統識別碼-Guid    | 8d3bca50-1d7e-11d0-a081-00aa006c33ed                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
