# <a name="ipnetworknumber-attribute"></a>ipNetworkNumber 屬性

以點分隔的十進位標記法包含 IP 網路編號，並省略前置零。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ipNetworkNumber                      |
| Ldap-顯示名稱 | ipNetworkNumber                      |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.20                     |
| 系統識別碼-Guid    | 4e3854f4-3087-42a4-a813-bb0c528958d3 |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
