# <a name="ms-wmi-querylanguage-attribute"></a>QueryLanguage 屬性

識別 WQL 查詢語言。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | QueryLanguage                        |
| Ldap-顯示名稱 | msWMI-QueryLanguage                         |
| 大小              | 少於二十個字元。                |
| 更新許可權  | 群組原則系統管理員                  |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1643                     |
| 系統識別碼-Guid    | 7d3cfa98-c17b-4254-8bd7-4de9b932a345        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
