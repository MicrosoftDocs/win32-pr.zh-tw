# <a name="rdn-attribute"></a>RDN 屬性

物件的相對辨別名稱 (RDN) 。 RDN 是辨別名稱的相對部分 (DN) ，可唯一識別 LDAP 物件。



| 進入 | 值 |
|-------------------|------------------------------------------------|
| CN                | RDN                                            |
| Ldap-顯示名稱 | NAME                                           |
| 大小              | \-                                             |
| 更新許可權  | 此值是由架構管理員所設定。 |
| 更新頻率  | \-                                             |
| Attribute-Id      | 1.2.840.113556.1.4.1                           |
| 系統識別碼-Guid    | bf967a0e-0de6-11d0-a285-00aa003049e2           |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)    |
