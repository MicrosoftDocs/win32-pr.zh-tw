# <a name="script-path-attribute"></a>Script-Path 屬性

這個屬性會指定使用者登入腳本的路徑。 字串可以是 null。



| 進入 | 值 |
|-------------------|------------------------------------------------------------------------|
| CN                | Script-Path                                                            |
| Ldap-顯示名稱 | scriptPath                                                             |
| 大小              | \-                                                                     |
| 更新許可權  | 網域系統管理員或帳戶擁有者。                                 |
| 更新頻率  | 當使用者記錄建立時，以及每次需要變更路徑時。 |
| Attribute-Id      | 1.2.840.113556.1.4.62                                                  |
| 系統識別碼-Guid    | bf9679a8-0de6-11d0-a285-00aa003049e2                                   |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                            |
