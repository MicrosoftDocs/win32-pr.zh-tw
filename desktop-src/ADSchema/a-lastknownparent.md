# <a name="last-known-parent-attribute"></a>最後一個已知的父屬性

孤立物件的最後一個已知父系 (DN) 的分辨名稱。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | 最後一個已知-父系                       |
| Ldap-顯示名稱 | lastKnownParent                         |
| 大小              | \-                                      |
| 更新許可權  | 此值是由系統所設定。        |
| 更新頻率  | 每次物件孤立時。         |
| Attribute-Id      | 1.2.840.113556.1.4.781                  |
| 系統識別碼-Guid    | 52ab8670-5709-11d1-a9c6-0000f80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
