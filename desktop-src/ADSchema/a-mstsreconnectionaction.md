# <a name="ms-ts-reconnection-action-attribute"></a>ms-TS-重新連接-動作屬性

[終端機服務會話重新連線] 動作會指定是否允許從任何用戶端電腦重新連線到中斷連線的終端機服務會話。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ms-TS-Reconnection-Action            |
| Ldap-顯示名稱 | msTSReconnectionAction               |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1984              |
| 系統識別碼-Guid    | 366ed7ca-3e18-4c7f-abae-351a01e4b4f7 |
| Syntax            | [**Boolean**](s-boolean.md)         |
