# <a name="gpc-functionality-version-attribute"></a>GPC 功能-Version 屬性

建立此物件群組原則編輯器的版本。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | GPC-功能-版本            |
| Ldap-顯示名稱 | gPCFunctionalityVersion              |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.893               |
| 系統識別碼-Guid    | f30e3bc0-9ff0-11d1-b603-0000f80367c1 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
