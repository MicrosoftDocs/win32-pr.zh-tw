# <a name="min-pwd-age-attribute"></a>最小-Pwd-Age 屬性

密碼有效的最小時間量（以 100-毫微秒間隔）。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 最小-Pwd-使用期限                          |
| Ldap-顯示名稱 | minPwdAge                            |
| 大小              | \-                                   |
| 更新許可權  | 網域系統管理員                 |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.78                |
| 系統識別碼-Guid    | bf9679c2-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**間隔**](s-interval.md)       |
