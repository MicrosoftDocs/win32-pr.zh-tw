# <a name="options-attribute"></a>Options 屬性

位位，其中位的意義會因 objectClass 而異。 可在網站間傳輸、NTDS 連線、NTDS DSA、NTDS 設定和 Site-Link 物件上進行。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 選項                              |
| Ldap-顯示名稱 | 選項                              |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.307               |
| 系統識別碼-Guid    | 19195a53-6da0-11d0-afd3-00c04fd930c9 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
