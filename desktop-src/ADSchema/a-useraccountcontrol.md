# <a name="user-account-control-attribute"></a>使用者帳戶控制屬性

控制使用者帳戶行為的旗標。



| 進入 | 值 |
|-------------------|---------------------------------------|
| CN                | 使用者帳戶控制                  |
| Ldap-顯示名稱 | userAccountControl                    |
| 大小              | 4個位元組。                              |
| 更新許可權  | 此值是由系統所設定。      |
| 更新頻率  | 每次帳戶原則變更時。 |
| Attribute-Id      | 1.2.840.113556.1.4.8                  |
| 系統識別碼-Guid    | bf967a68-0de6-11d0-a285-00aa003049e2  |
| Syntax            | [**列舉型別**](s-enumeration.md)  |
