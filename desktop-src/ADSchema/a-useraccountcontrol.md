# <a name="user-account-control-attribute"></a>使用者帳戶控制屬性

控制使用者帳戶行為的旗標。



| 進入 | 值 |
|-------------------|---------------------------------------|
| CN                | 使用者帳戶控制                  |
| Ldap-顯示名稱 | userAccountControl                    |
| 大小              | 4個位元組。                              |
| 更新許可權  | 此值是由系統所設定。      |
| 更新頻率  | 每次帳戶原則變更時。 |
| Attribute-Id      | 1.2.840.113556.1.4.8                  |
| 系統識別碼-Guid    | bf967a68-0de6-11d0-a285-00aa003049e2  |
| Syntax            | [**列舉型別**](s-enumeration.md)  |

## <a name="remarks"></a>備註

這個屬性值可以是零或一或多個下列值的組合。




| 十六進位值 | Iads 中定義的識別碼 ()  | Description | 
|-------------------|--------------------------------|-------------|
| 0x00000001 | <a href="/windows/desktop/api/iads/ne-iads-ads_user_flag_enum"><strong>ADS_UF_SCRIPT</strong></a> | 執行登入腳本。 | 
| 0x00000002 | <a href="/windows/desktop/api/iads/ne-iads-ads_user_flag_enum"><strong>ADS_UF_ACCOUNTDISABLE</strong></a> | 使用者帳戶已停用。 | 
| 0x00000008 | <a href="/windows/desktop/api/iads/ne-iads-ads_user_flag_enum"><strong>ADS_UF_HOMEDIR_REQUIRED</strong></a> | 需要主目錄。 | 
| 0x00000010 | <a href="/windows/desktop/api/iads/ne-iads-ads_user_flag_enum"><strong>ADS_UF_LOCKOUT</strong></a> | 帳戶目前已鎖定。 | 
| 0x00000020 | <a href="/windows/desktop/api/iads/ne-iads-ads_user_flag_enum"><strong>ADS_UF_PASSWD_NOTREQD</strong></a> | 不需要密碼。 | 
| 0x00000040 | <a href="/windows/desktop/api/iads/ne-iads-ads_user_flag_enum"><strong>ADS_UF_PASSWD_CANT_CHANGE</strong></a> | 使用者無法變更密碼。<blockquote>[!Note]<br />您無法直接修改 UserAccountControl 屬性來指派 PASSWD_CANT_CHANGE 的許可權設定。 如需詳細資訊和顯示如何防止使用者變更密碼的程式碼範例，請參閱 <a href="/windows/desktop/ADSI/user-cannot-change-password">使用者無法變更密碼</a>。</blockquote><br /> : | 
| 0x00000080 | <a href="/windows/desktop/api/iads/ne-iads-ads_user_flag_enum"><strong>ADS_UF_ENCRYPTED_TEXT_PASSWORD_ALLOWED</strong></a> | 使用者可以傳送加密密碼。 | 
| 0x00000100 | <a href="/windows/desktop/api/iads/ne-iads-ads_user_flag_enum"><strong>ADS_UF_TEMP_DUPLICATE_ACCOUNT</strong></a> | 這是主要帳戶位於另一個網域的使用者帳戶。 此帳戶可讓使用者存取此網域，但不提供任何信任此網域的網域。 也稱為本機使用者帳戶。 | 
| 0x00000200 | <a href="/windows/desktop/api/iads/ne-iads-ads_user_flag_enum"><strong>ADS_UF_NORMAL_ACCOUNT</strong></a> | 這是代表一般使用者的預設帳戶類型。 | 
| 0x00000800 | <a href="/windows/desktop/api/iads/ne-iads-ads_user_flag_enum"><strong>ADS_UF_INTERDOMAIN_TRUST_ACCOUNT</strong></a> | 這是允許信任其他網域之系統網域的帳戶。 | 
| 0x00001000 | <a href="/windows/desktop/api/iads/ne-iads-ads_user_flag_enum"><strong>ADS_UF_WORKSTATION_TRUST_ACCOUNT</strong></a> | 這是屬於此網域成員之電腦的電腦帳戶。 | 
| 0x00002000 | <a href="/windows/desktop/api/iads/ne-iads-ads_user_flag_enum"><strong>ADS_UF_SERVER_TRUST_ACCOUNT</strong></a> | 這是屬於此網域成員之系統備份網域控制站的電腦帳戶。 | 
| 0x00004000 | N/A | 未使用。 | 
| 0x00008000 | N/A | 未使用。 | 
| 0x00010000 | <a href="/windows/desktop/api/iads/ne-iads-ads_user_flag_enum"><strong>ADS_UF_DONT_EXPIRE_PASSWD</strong></a> | 此帳戶的密碼永遠不會過期。 | 
| 0x00020000 | <a href="/windows/desktop/api/iads/ne-iads-ads_user_flag_enum"><strong>ADS_UF_MNS_LOGON_ACCOUNT</strong></a> | 這是 MNS 登入帳戶。 | 
| 0x00040000 | <a href="/windows/desktop/api/iads/ne-iads-ads_user_flag_enum"><strong>ADS_UF_SMARTCARD_REQUIRED</strong></a> | 使用者必須使用智慧卡登入。 | 
| 0x00080000 | <a href="/windows/desktop/api/iads/ne-iads-ads_user_flag_enum"><strong>ADS_UF_TRUSTED_FOR_DELEGATION</strong></a> | 服務帳戶 (用來執行服務的使用者或電腦帳戶) ，受 Kerberos 委派的信任。 任何這類服務都可以模擬要求服務的用戶端。 | 
| 0x00100000 | <a href="/windows/desktop/api/iads/ne-iads-ads_user_flag_enum"><strong>ADS_UF_NOT_DELEGATED</strong></a> | 即使服務帳戶設定為受信任的 Kerberos 委派，也不會將使用者的安全性內容委派給服務。 | 
| 0x00200000 | <a href="/windows/desktop/api/iads/ne-iads-ads_user_flag_enum"><strong>ADS_UF_USE_DES_KEY_ONLY</strong></a> | 限制此主體僅使用資料加密標準 (DES) 金鑰加密類型。 | 
| 0x00400000 | <a href="/windows/desktop/api/iads/ne-iads-ads_user_flag_enum"><strong>ADS_UF_DONT_REQUIRE_PREAUTH</strong></a> | 此帳戶不需要 Kerberos 預先驗證即可登入。 | 
| 0x00800000 | <a href="/windows/desktop/api/iads/ne-iads-ads_user_flag_enum"><strong>ADS_UF_PASSWORD_EXPIRED</strong></a> | 使用者密碼已過期。 此旗標是由系統使用來自 <a href="a-pwdlastset.md"><strong>Pwd 的 Last 設定</strong></a> 屬性和網域原則的資料所建立。 | 
| 0x01000000 | <a href="/windows/desktop/api/iads/ne-iads-ads_user_flag_enum"><strong>ADS_UF_TRUSTED_TO_AUTHENTICATE_FOR_DELEGATION</strong></a> | 帳戶已啟用委派。 這是安全性相關設定;啟用此選項的帳戶應受到嚴格的控制。 此設定可讓在此帳戶下執行的服務採用用戶端身分識別，並以該使用者身分向網路上的其他遠端伺服器進行驗證。 |
