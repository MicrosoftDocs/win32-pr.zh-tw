# <a name="ms-pki-minimal-key-size-attribute"></a>ms-PKI-基本金鑰大小屬性

指出最小私用金鑰大小。



| 進入 | 值 |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-基本金鑰大小                                                                           |
| Ldap-顯示名稱 | Mspki-certificate-name-flag-基本-索引鍵大小                                                                            |
| 大小              | 4 個位元組                                                                                           |
| 更新許可權  | 網域系統管理員                                                                              |
| 更新頻率  | 當您編輯、建立或複製憑證範本時 (ms-PKI-憑證範本) 物件。 |
| Attribute-Id      | 1.2.840.113556.1.4.1433                                                                           |
| 系統識別碼-Guid    | e96a63f5-417f-46d3-be52-db7703c503df                                                              |
| Syntax            | [**列舉型別**](s-enumeration.md)                                                              |
