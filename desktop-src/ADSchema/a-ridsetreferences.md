|-------------------|-----------------------------------------|
| CN                | RID-Set 參考                      |
| Ldap-顯示名稱 | rIDSetReferences                        |
| 大小              | \-                                      |
| 更新許可權  | \-                                      |
| 更新頻率  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.669                  |
| 系統識別碼-Guid    | 7bfdcb7b-4807-11d1-a9c3-0000f80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
