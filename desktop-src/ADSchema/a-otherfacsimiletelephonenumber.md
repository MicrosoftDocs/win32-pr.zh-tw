# <a name="phone-fax-other-attribute"></a>電話-Fax-其他屬性

替代傳真號碼的清單。



| 進入 | 值 |
|-------------------|----------------------------------------------------------------------------------|
| CN                | 電話-Fax-其他                                                                  |
| Ldap-顯示名稱 | otherFacsimileTelephoneNumber                                                    |
| 大小              | \-                                                                               |
| 更新許可權  | 網域系統管理員或帳戶擁有者。                                           |
| 更新頻率  | 當使用者的記錄建立時，以及每次電話號碼需要變更時。 |
| Attribute-Id      | 1.2.840.113556.1.4.646                                                           |
| 系統識別碼-Guid    | 0296c11d-40da-11d1-a9c0-0000f80367c1                                             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                      |
