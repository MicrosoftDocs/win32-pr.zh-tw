| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | UPN-Suffixes                                |
| Ldap-顯示名稱 | uPNSuffixes                                 |
| 大小              | \-                                          |
| 更新許可權  | 網域系統管理員                        |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.890                      |
| 系統識別碼-Guid    | 032160bf-9824-11d1-aec0-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
