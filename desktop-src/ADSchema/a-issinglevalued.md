# <a name="is-single-valued-attribute"></a>是-單一值屬性

若 **為 TRUE**，則這個屬性只能儲存一個值。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 是-單一值                     |
| Ldap-顯示名稱 | isSingleValued                       |
| 大小              | \-                                   |
| 更新許可權  | 架構系統管理員                 |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.33                |
| 系統識別碼-Guid    | bf967992-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Boolean**](s-boolean.md)         |
