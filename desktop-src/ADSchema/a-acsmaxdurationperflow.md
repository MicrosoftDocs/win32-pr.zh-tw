# <a name="acs-max-duration-per-flow-attribute"></a>ACS-每個流程的最大持續時間屬性

任何單一流程的最大持續時間（以秒為單位）。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ACS-每個流程的最大持續時間            |
| Ldap-顯示名稱 | aCSMaxDurationPerFlow                |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.761               |
| 系統識別碼-Guid    | 7f56127e-5301-11d1-a9c5-0000f80367c1 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
