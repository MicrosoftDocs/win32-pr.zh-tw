# <a name="proxy-addresses-attribute"></a>Proxy-Addresses 屬性

proxy 位址是在外部郵件系統中識別 Microsoft Exchange Server 收件者物件的位址。 所有收件者物件都需要 Proxy 位址，例如自訂收件者和通訊群組清單。



| 進入 | 值 |
|-------------------|------------------------------------------------------------------------------------------------------|
| CN                | Proxy-Addresses                                                                                      |
| Ldap-顯示名稱 | proxyAddresses                                                                                       |
| 大小              | \-                                                                                                   |
| 更新許可權  | 此值是由系統所設定。                                                                     |
| 更新頻率  | 在安裝網址類別型時，由 Address-Type directory 物件提供的 DLL 所建立。 |
| Attribute-Id      | 1.2.840.113556.1.2.210                                                                               |
| 系統識別碼-Guid    | bf967a06-0de6-11d0-a285-00aa003049e2                                                                 |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                                          |
