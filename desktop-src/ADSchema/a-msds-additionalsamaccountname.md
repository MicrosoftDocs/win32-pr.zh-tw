# <a name="ms-ds-additional-sam-account-name-attribute"></a>ms-DS-其他 Sam-帳戶名稱屬性

這個屬性是用來儲存 SAM 帳戶名稱，這些名稱會對應至以 ms DS 為依據的 DNS 主機名稱。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | ms DS-其他-Sam-帳戶-名稱           |
| Ldap-顯示名稱 | AdditionalSamAccountName               |
| 大小              | 小於20位元組。                         |
| 更新許可權  | 此值是由系統所設定。            |
| 更新頻率  | 當 DC 重新命名時。                     |
| Attribute-Id      | 1.2.840.113556.1.4.1718                     |
| 系統識別碼-Guid    | 975571df-a4d5-429a-9f59-cdc6581d91e6        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |



## <a name="implementations"></a>實作

-   [**Windows Server 2003 R2**](#windows-server-2003-r2)
-   [**Windows Server 2008**](#windows-server-2008)
