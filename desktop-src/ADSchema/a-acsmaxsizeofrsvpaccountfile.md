# <a name="acs-max-size-of-rsvp-account-file-attribute"></a>ACS-最大---------Account-File 屬性

RSVP 帳戶檔案的大小上限（以位元組為單位）。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ACS-最大----------------File    |
| Ldap-顯示名稱 | aCSMaxSizeOfRSVPAccountFile          |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.902               |
| 系統識別碼-Guid    | f0722311-aef5-11d1-bdcf-0000f80367c1 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
