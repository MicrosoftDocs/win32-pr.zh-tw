|-------------------|-------------------------------------------------------|

| Ldap-顯示名稱 | fRSExtensions                                         |
| 大小              | \-                                                    |
| 更新許可權  | 此值是由系統所設定。                      |
| 更新頻率  | \-                                                    |

| 系統識別碼-Guid    | 52458020-ca6a-11d0-afff-0000f80367c1                  |
