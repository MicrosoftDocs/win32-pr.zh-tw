# <a name="transport-type-attribute"></a>Transport-Type 屬性

用來將網站連接在一起的傳輸類型的分辨名稱。 此值可指向 IP 或 SMTP 傳輸。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | Transport-Type                          |
| Ldap-顯示名稱 | transportType                           |
| 大小              | \-                                      |
| 更新許可權  | 此值是由系統所設定。        |
| 更新頻率  | 連接網站時。                  |
| Attribute-Id      | 1.2.840.113556.1.4.791                  |
| 系統識別碼-Guid    | 26d97374-6070-11d1-a9c6-0000f80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
