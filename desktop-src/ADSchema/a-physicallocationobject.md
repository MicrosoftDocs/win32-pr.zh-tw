| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | 實體位置-物件                |
| Ldap-顯示名稱 | physicalLocationObject                  |
| 大小              | \-                                      |
| 更新許可權  | \-                                      |
| 更新頻率  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.514                  |
| 系統識別碼-Guid    | b7b13119-b82e-11d0-afee-0000f80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
