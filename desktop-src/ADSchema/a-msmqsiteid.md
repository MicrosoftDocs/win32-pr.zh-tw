**Msmq 網站識別碼** 屬性包含 msmq 混合模式資訊。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | MSMQ-網站識別碼                                          |
| Ldap-顯示名稱 | mSMQSiteID                                            |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.953                                |
| 系統識別碼-Guid    | 9a0dc340-c100-11d1-bbc5-0080c76670c0                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
