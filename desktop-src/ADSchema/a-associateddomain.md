# <a name="associateddomain-attribute"></a>associatedDomain 屬性

AssociatedDomain 屬性類型會指定與物件相關聯的 DNS 網域。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | associatedDomain                     |
| Ldap-顯示名稱 | associatedDomain                     |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 0.9.2342.19200300.100.1.37           |
| 系統識別碼-Guid    | 3320fc38-c379-4c17-a510-1bdf6133c5da |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
