|-------------------|---------------------------------------------|
| CN                | ms-WMI-作者                               |
| Ldap-顯示名稱 | msWMI-作者                                |
| 大小              | 少於25個字元。                    |
| 更新許可權  | 群組原則系統管理員                  |

| 系統識別碼-Guid    | 6366c0c1-6972-4e66-b3a5-1d52ad0c0547        |
