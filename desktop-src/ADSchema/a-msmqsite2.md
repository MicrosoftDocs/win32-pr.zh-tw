# <a name="msmq-site-2-attribute"></a>MSMQ-網站2屬性

所連接之配對的第二個網站的 DN。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | MSMQ-網站2                             |
| Ldap-顯示名稱 | mSMQSite2                               |
| 大小              | \-                                      |
| 更新許可權  | \-                                      |
| 更新頻率  | 每次建立 MSMQ 網站行時。 |
| Attribute-Id      | 1.2.840.113556.1.4.944                  |
| 系統識別碼-Guid    | 9a0dc338-c100-11d1-bbc5-0080c76670c0    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
