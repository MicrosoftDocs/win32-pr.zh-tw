# <a name="print-max-copies-attribute"></a>列印-Max-複製屬性

裝置可列印的最大複本數目。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 列印-最大複本                     |
| Ldap-顯示名稱 | printMaxCopies                       |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.241               |
| 系統識別碼-Guid    | 281416d1-1968-11d0-a28f-00aa003049e2 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
