# <a name="apply-group-policy-extended-right"></a>套用-群組-原則擴充許可權

群組原則引擎用來判斷 GPO 是否適用于使用者或電腦的擴充許可權。



| 進入 | 值 |
|--------------|--------------------------------------|
| CN           | 套用-群組-原則                   |
| 顯示名稱 | 申請群組原則                   |
| 許可權-GUID  | edacfd8f-ffb3-11d1-b41d-00a0c968f939 |
