# <a name="ms-sql-size-attribute"></a>ms-chap-SQL 大小屬性

資料庫的大小（以 kb 為單位）。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 毫秒-SQL 大小                          |
| Ldap-顯示名稱 | 毫秒-SQL 大小                          |
| 大小              | 8 個位元組                              |
| 更新許可權  | 此值是由系統所設定。     |
| 更新頻率  | 當資料庫的大小變更時。      |
| Attribute-Id      | 1.2.840.113556.1.4.1396              |
| 系統識別碼-Guid    | e9098084-ccee-11d2-9993-0000f87a57d4 |
| Syntax            | [**間隔**](s-interval.md)       |
