# <a name="msmq-authenticate-attribute"></a>MSMQ-Authenticate 屬性

指出佇列是否只接受已驗證的訊息。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | MSMQ-Authenticate                    |
| Ldap-顯示名稱 | mSMQAuthenticate                     |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.923               |
| 系統識別碼-Guid    | 9a0dc326-c100-11d1-bbc5-0080c76670c0 |
| Syntax            | [**Boolean**](s-boolean.md)         |
