| CN                | MS-TS-LicenseVersion2                       |
| Ldap-顯示名稱 | msTSLicenseVersion2                         |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |

| 系統識別碼-Guid    | 4b0df103-8d97-45d9-ad69-85c3080ba4e7        |
