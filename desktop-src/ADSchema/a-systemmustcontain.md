# <a name="system-must-contain-attribute"></a>System-必須包含屬性

類別的強制屬性清單。 建立類別的實例時，必須指定這些屬性。 屬性清單只能由系統修改。



| 進入 | 值 |
|-------------------|-------------------------------------------------------------------------------|
| CN                | 系統-必須包含                                                           |
| Ldap-顯示名稱 | systemMustContain                                                             |
| 大小              | \-                                                                            |
| 更新許可權  | 架構系統管理員                                                          |
| 更新頻率  | 當建立類別或將新的強制屬性加入至類別時。 |
| Attribute-Id      | 1.2.840.113556.1.4.197                                                        |
| 系統識別碼-Guid    | bf967a45-0de6-11d0-a285-00aa003049e2                                          |
| Syntax            | [**String(Object-Identifier)**](s-string-object-identifier.md)               |



## <a name="implementations"></a>實作

-   [**Windows 2000 Server**](#windows-2000-server)
-   [**Windows Server 2003**](#windows-server-2003)
