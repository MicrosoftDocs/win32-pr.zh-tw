# <a name="msmq-migrated-attribute"></a>MSMQ-Migrated 屬性

**Msmq 遷移** 的屬性包含 msmq 混合模式資訊。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | MSMQ-Migrated                        |
| Ldap-顯示名稱 | mSMQMigrated                         |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.952               |
| 系統識別碼-Guid    | 9a0dc33f-c100-11d1-bbc5-0080c76670c0 |
| Syntax            | [**Boolean**](s-boolean.md)         |
