| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | Lockout-Threshold                    |
| Ldap-顯示名稱 | lockoutThreshold                     |
| 大小              | \-                                   |
| 更新許可權  | 網域系統管理員                 |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.73                |
| 系統識別碼-Guid    | bf9679a6-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
