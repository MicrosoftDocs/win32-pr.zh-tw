# <a name="ms-pki-supersede-templates-attribute"></a>ms-PKI-取代範本屬性

指定目前範本所取代的憑證範本名稱。



| 進入 | 值 |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-取代-範本                                                                        |
| Ldap-顯示名稱 | Mspki-certificate-name-flag-取代-範本                                                                         |
| 大小              | 64位元組                                                                                          |
| 更新許可權  | 網域系統管理員                                                                              |
| 更新頻率  | 當您編輯、建立或複製憑證範本時 (ms-PKI-憑證範本) 物件。 |
| Attribute-Id      | 1.2.840.113556.1.4.1437                                                                           |
| 系統識別碼-Guid    | 9de8ae7d-7a5b-421d-b5e4-061f79dfd5d7                                                              |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                                       |



## <a name="implementations"></a>實作

-   [**Windows Server 2003**](#windows-server-2003)
-   [**Windows Server 2003 R2**](#windows-server-2003-r2)
-   [**Windows Server 2008**](#windows-server-2008)
