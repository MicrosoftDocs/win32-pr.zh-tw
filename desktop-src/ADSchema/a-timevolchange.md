# <a name="time-vol-change-attribute"></a>時間-音量-變更屬性

這個屬性工作表示上次變更遠端存放裝置磁片區中檔案的時間。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 時間-音量-變更                      |
| Ldap-顯示名稱 | timeVolChange                        |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.502               |
| 系統識別碼-Guid    | ddac0cf0-af8f-11d0-afeb-00c04fd930c9 |
| Syntax            | [**間隔**](s-interval.md)       |
