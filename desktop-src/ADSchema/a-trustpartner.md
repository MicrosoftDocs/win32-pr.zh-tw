# <a name="trust-partner-attribute"></a>Trust-Partner 屬性

存在信任之網域的名稱。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | Trust-Partner                               |
| Ldap-顯示名稱 | trustPartner                                |
| 大小              | \-                                          |
| 更新許可權  | 此值是由系統所設定。            |
| 更新頻率  | 當新的信任建立時。                |
| Attribute-Id      | 1.2.840.113556.1.4.133                      |
| 系統識別碼-Guid    | bf967a5d-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
