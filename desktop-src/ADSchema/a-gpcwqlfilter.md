# <a name="gpc-wql-filter-attribute"></a>GPC-WQL-Filter 屬性

用來儲存包含篩選的 GUID 和 WMI 命名空間路徑的字串。



| 進入 | 值 |
|-------------------|---------------------------------------------------------------------------------|
| CN                | GPC-WQL-篩選                                                                  |
| Ldap-顯示名稱 | gPCWQLFilter                                                                    |
| 大小              | \-                                                                              |
| 更新許可權  | 網域系統管理員                                                            |
| 更新頻率  | 只有當系統管理員變更群組原則 UI 中的篩選屬性時。 |
| Attribute-Id      | 1.2.840.113556.1.4.1694                                                         |
| 系統識別碼-Guid    | 7bd4c7a6-1add-4436-8c04-3999a880154c                                            |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                     |
