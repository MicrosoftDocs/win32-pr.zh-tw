# <a name="treat-as-leaf-attribute"></a>視為分葉屬性

此旗標設為 **TRUE** 的顯示規範會強制將相關類別顯示為分葉類別，即使它有子系也一樣。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 視為分葉                        |
| Ldap-顯示名稱 | treatAsLeaf                          |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.806               |
| 系統識別碼-Guid    | 8fd044e3-771f-11d1-aeae-0000f80367c1 |
| Syntax            | [**Boolean**](s-boolean.md)         |
