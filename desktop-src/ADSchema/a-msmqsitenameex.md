# <a name="msmq-site-name-ex-attribute"></a>MSMQ-網站名稱-Ex 屬性

**Msmq-網站名稱-Ex** 屬性包含 msmq 混合模式資訊。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | MSMQ-網站名稱-例如                           |
| Ldap-顯示名稱 | mSMQSiteNameEx                              |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1416                     |
| 系統識別碼-Guid    | 422144fa-c17f-4649-94d6-9731ed2784ed        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
