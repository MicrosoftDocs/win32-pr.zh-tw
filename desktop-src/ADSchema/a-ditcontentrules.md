# <a name="dit-content-rules-attribute"></a>DIT-Content 規則屬性

這會藉由使用一組選擇性的輔助物件類別，以及必要的、選擇性和排除屬性，來指定特定結構物件類別的專案所允許的內容。 集體屬性包含在 DIT 內容規則中，如 RFC 2251 中所定義。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | DIT-內容-規則                           |
| Ldap-顯示名稱 | dITContentRules                             |
| 大小              | \-                                          |
| 更新許可權  | 此值是由系統所設定。            |
| 更新頻率  | 每次建立新的類別。            |
| Attribute-Id      | 2.5.21.2                                    |
| 系統識別碼-Guid    | 9a7ad946-ca53-11d1-bbd0-0080c76670c0        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
