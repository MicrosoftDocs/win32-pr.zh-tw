# <a name="ms-ds-all-users-trust-quota-attribute"></a>MS DS-所有使用者-信任-配額屬性

用來強制合併的使用者使用新的 control 存取權限建立的 Trusted-Domain 物件總數，建立-------------------樹系信任。



| 進入 | 值 |
|-------------------|-------------------------------------------|
| CN                | MS-DS-所有使用者-信任-配額               |
| Ldap-顯示名稱 | AllUsersTrustQuota                   |
| 大小              | \-                                        |
| 更新許可權  | 網域系統管理員                      |
| 更新頻率  | 在建立樹系時，很少發生。 |
| Attribute-Id      | 1.2.840.113556.1.4.1789                   |
| 系統識別碼-Guid    | d3aa4a5c-4e03-4810-97aa-2b339e7a434b      |
| Syntax            | [**列舉型別**](s-enumeration.md)      |
