# <a name="name-service-flags-attribute"></a>名稱-服務-旗標屬性

RPC 名稱服務的設定旗標。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 名稱-服務-旗標                   |
| Ldap-顯示名稱 | nameServiceFlags                     |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.753               |
| 系統識別碼-Guid    | 80212840-4bdc-11d1-a9c4-0000f80367c1 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
