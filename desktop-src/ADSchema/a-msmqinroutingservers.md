# <a name="msmq-in-routing-servers-attribute"></a>MSMQ 路由-伺服器屬性

MSMQ 路由伺服器的 DN 連結，此伺服器會透過這些伺服器來路由傳送至此電腦的所有連入流量。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | MSMQ 路由傳送伺服器                 |
| Ldap-顯示名稱 | mSMQInRoutingServers                    |
| 大小              | \-                                      |
| 更新許可權  | \-                                      |
| 更新頻率  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.929                  |
| 系統識別碼-Guid    | 9a0dc32c-c100-11d1-bbc5-0080c76670c0    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
