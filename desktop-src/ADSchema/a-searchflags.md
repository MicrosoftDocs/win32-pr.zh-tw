## <a name="remarks"></a>備註

這個屬性可以是零或一或多個下列值的組合。



| 值            | 描述                                                                                                                                                                                                                                                                                                                                                               |
|------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 1 (0x00000001)   | 建立屬性的索引。                                                                                                                                                                                                                                                                                                                                        |
| 2 (0x00000002)    | 在每個容器中建立屬性的索引。                                                                                                                                                                                                                                                                                                                      |
| 4 (0x00000004)    | 將此屬性新增至 (ANR) 設定的明確名稱解析。 這是用來協助在只提供部分資訊時尋找物件。 例如，如果 LDAP 篩選準則是 (ANR = JEFF) ，搜尋會尋找每個物件，其中的名字、姓氏、電子郵件地址或其他 ANR 屬性等於 JEFF。 必須為此索引設定 Bit 0 才會生效。 |
| 8 (0x00000008)    | 針對已刪除的物件，將這個屬性保留在標記物件中。                                                                                                                                                                                                                                                                                                      |
| 16 (0x00000010)   | 複製物件時，請複製這個屬性的值。                                                                                                                                                                                                                                                                                                              |
| 32 (0x00000020)   | 建立屬性的元組索引。 這會改善搜尋字串開頭出現萬用字元的搜尋。 例如， (sn = \* mith) 。                                                                                                                                                                                                                |
| 64 (0x00000040)    | 從 ADAM 開始支援。 建立索引，以大幅協助 VLV 任意屬性的效能。                                                                                                                                                                                                                                                                  |
| 128 (0x00000080)  | 將屬性標示為機密。 已忽略基底架構屬性 (systemFlags = 0x10) 。                                                                                                                                                                                                                                                                                    |
| 64 (0x00000040)   | 從 Windows Server 2008 開始支援。 建立索引，以改善這個屬性的 VLV 搜尋效能。                                                                                                                                                                                                                                                        |
