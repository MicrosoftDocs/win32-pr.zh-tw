# <a name="time-refresh-attribute"></a>Time-Refresh 屬性

此屬性具有時間間隔，在這段期間內應針對 DNS 伺服器重新整理 Active Directory 整合區域中包含的資源記錄。 預設間隔為7天。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | Time-Refresh                         |
| Ldap-顯示名稱 | timeRefresh                          |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.503               |
| 系統識別碼-Guid    | ddac0cf1-af8f-11d0-afeb-00c04fd930c9 |
| Syntax            | [**間隔**](s-interval.md)       |
