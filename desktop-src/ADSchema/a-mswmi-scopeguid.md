# <a name="ms-wmi-scopeguid-attribute"></a>ScopeGuid 屬性

相關編碼所在範圍的 GUID。



| 進入 | 值 |
|-------------------|-----------------------------------------------------------------|
| CN                | ScopeGuid                                                |
| Ldap-顯示名稱 | msWMI-ScopeGuid                                                 |
| 大小              | 16 個位元組                                                        |
| 更新許可權  | 網域系統管理員                                            |
| 更新頻率  | 只要加入或修改包含屬性的類別。 |
| Attribute-Id      | 1.2.840.113556.1.4.1686                                         |
| 系統識別碼-Guid    | 87b78d51-405f-4b7f-80ed-2bd28786f48d                            |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                     |
