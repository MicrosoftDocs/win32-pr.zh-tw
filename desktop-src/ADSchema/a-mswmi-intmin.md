# <a name="ms-wmi-intmin-attribute"></a>intMin 屬性

WMI 32 位整數參數物件的最小值。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | intMin                        |
| Ldap-顯示名稱 | msWMI-IntMin                         |
| 大小              | 4 個位元組                              |
| 更新許可權  | 群組原則系統管理員           |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1630              |
| 系統識別碼-Guid    | 68c2e3ba-9837-4c70-98e0-f0c33695d023 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
