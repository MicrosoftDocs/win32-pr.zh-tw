# <a name="version-number-attribute"></a>Version-Number 屬性

一般用途版本號碼。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | Version-Number                       |
| Ldap-顯示名稱 | versionNumber                        |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.141               |
| 系統識別碼-Guid    | bf967a76-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
