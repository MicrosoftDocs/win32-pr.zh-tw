|-------------------|---------------------------------------------|
| CN                | Other-Name                                  |
| Ldap-顯示名稱 | middleName                                  |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 2.16.840.1.113730.3.1.34                    |
| 系統識別碼-Guid    | bf9679f2-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
