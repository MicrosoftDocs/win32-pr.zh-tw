# <a name="msmq-ds-services-attribute"></a>MSMQ-Ds-服務屬性

指出安裝在這部電腦上的 MSMQ 是否提供 MSMQ DS 服務。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | MSMQ-Ds-服務                     |
| Ldap-顯示名稱 | mSMQDsServices                       |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1228              |
| 系統識別碼-Guid    | 2df90d78-009f-11d2-aa4c-00c04fd7d83a |
| Syntax            | [**Boolean**](s-boolean.md)         |
