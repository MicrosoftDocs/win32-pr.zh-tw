# <a name="current-location-attribute"></a>Current-Location 屬性

已移動之物件的電腦位置。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | Current-Location                                      |
| Ldap-顯示名稱 | currentLocation                                       |
| 大小              | \-                                                    |
| 更新許可權  | 此值是由系統所設定。                      |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.335                                |
| 系統識別碼-Guid    | 1f0075fc-7e40-11d0-afd6-00c04fd930c9                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
