# <a name="trust-direction-attribute"></a>Trust-Direction 屬性

信任的方向。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | Trust-Direction                      |
| Ldap-顯示名稱 | trustDirection                       |
| 大小              | 4 個位元組                              |
| 更新許可權  | 此值是由系統所設定。     |
| 更新頻率  | 當新的信任建立時。         |
| Attribute-Id      | 1.2.840.113556.1.4.132               |
| 系統識別碼-Guid    | bf967a5c-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
