| 更新許可權  | 此值是由架構管理員所設定。 |

| 系統識別碼-Guid    | e0fa1e62-9b45-11d0-afdd-00c04fd930c9           |

## <a name="remarks"></a>備註

這個屬性可以是零或一或多個下列值的組合。



| 值                   | 描述                                                                                                                                                                                                                                                                                     |
|-------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 1 (0x00000001)          | 套用至屬性時，將不會複寫屬性。套用至 [**交互參考**](c-crossref.md) 物件時，命名內容會在 NTDS 中。<br/>                                                                                                                          |
| 2 (0x00000002)           | 套用至屬性時，屬性會複寫至通用類別目錄。套用至 [**交互參考**](c-crossref.md) 物件時，命名內容為定義域。<br/>                                                                                                       |
| 4 (0x00000004)           | 當套用至屬性時，會建立屬性。                                                                                                                                                                                                                                     |
| 16 (0x00000010)          | 設定時，表示物件為類別1物件。 類別目錄1物件是包含在系統所包含的基底架構中的類別或屬性。                                                                                                                                |
| 33554432 (0x02000000)    | 物件被刪除時，不會移至已刪除的物件容器。 它會立即刪除。                                                                                                                                                                                    |
| 67108864 (0x04000000)    | 無法移動物件。                                                                                                                                                                                                                                                                     |
| 134217728 (0x08000000)   | 無法重新命名物件。                                                                                                                                                                                                                                                                   |
| 268435456 (0x10000000)   | 針對設定分割區中的物件，如果設定此旗標，則可以使用限制來移動物件;否則，就無法移動物件。 根據預設，不會在設定磁碟分割下建立的新物件上設定這個旗標。 此旗標只能在物件建立期間設定。 |
| 536870912 (0x20000000)   | 針對設定分割區中的物件，如果設定此旗標，則可以移動物件;否則，就無法移動物件。 根據預設，不會在設定磁碟分割下建立的新物件上設定這個旗標。 此旗標只能在物件建立期間設定。                   |
| 1073741824 (0x40000000)  | 針對設定分割區中的物件，如果設定此旗標，則可以重新命名物件;否則，就無法重新命名物件。 根據預設，不會在設定磁碟分割下建立的新物件上設定這個旗標。 此旗標只能在物件建立期間設定。               |
| 2147483648 (0x80000000)  | 無法刪除物件。                                                                                                                                                                                                                                                                   |
