# <a name="msi-file-list-attribute"></a>Msi 檔案清單屬性

此屬性包含 Microsoft installer 檔案的清單，例如基礎 MSI 檔案 ( .msi) 和 MST 轉換檔案 ( .mst) 。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | Msi-檔案清單                               |
| Ldap-顯示名稱 | msiFileList                                 |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.671                      |
| 系統識別碼-Guid    | 7bfdcb7d-4807-11d1-a9c3-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
