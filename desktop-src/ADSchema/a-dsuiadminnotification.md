# <a name="ds-ui-admin-notification-attribute"></a>DS-UI-管理通知屬性

這是 COM 物件的 Guid 清單，支援在物件上透過 UI 執行動作時 DSAdmin 呼叫的回呼介面。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | DS-UI-系統管理-通知                    |
| Ldap-顯示名稱 | dSUIAdminNotification                       |
| 大小              | \-                                          |
| 更新許可權  | 網域系統管理員                        |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1343                     |
| 系統識別碼-Guid    | f6ea0a94-6f91-11d2-9905-0000f87a57d4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
