| 大小              | \-                                                                                         |
| 更新許可權  | 企業系統管理員                                                                   |
| 更新頻率  | 每次建立新範本，或編輯現有範本的屬性時。 |

| 系統識別碼-Guid    | 3c91fbbf-4773-4ccd-a87b-85d53e7bcf6a                                                       |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                                |
