| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | MS-SQL-組建                         |
| Ldap-顯示名稱 | mS-SQL-組建                         |
| 大小              | \-                                   |
| 更新許可權  | 此值是由系統所設定。     |
| 更新頻率  | 在系統啟動時。                   |
| Attribute-Id      | 1.2.840.113556.1.4.1368              |
| 系統識別碼-Guid    | 603e94c4-ccee-11d2-9993-0000f87a57d4 |
