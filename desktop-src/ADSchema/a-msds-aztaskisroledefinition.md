# <a name="ms-ds-az-task-is-role-definition-attribute"></a>ms-chap-Az-Task-Role-Definition 屬性

指出 AzTask 為傳統工作或角色定義的布林值欄位。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | ms-chap-Az-Task-角色定義        |
| Ldap-顯示名稱 | AzTaskIsRoleDefinition             |
| 大小              | \-                                      |
| 更新許可權  | AzRoles 系統管理員                           |
| 更新頻率  | 在初始化或原則變更期間。 |
| Attribute-Id      | 1.2.840.113556.1.4.1818                 |
| 系統識別碼-Guid    | 7b078544-6c82-4fe9-872f-ff48ad2b2e26    |
| Syntax            | [**Boolean**](s-boolean.md)            |
