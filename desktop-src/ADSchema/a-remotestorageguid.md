# <a name="remote-storage-guid-attribute"></a>遠端儲存體-GUID 屬性

此屬性包含遠端儲存物件的 GUID。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | 遠端儲存體-GUID                         |
| Ldap-顯示名稱 | remoteStorageGUID                           |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.809                      |
| 系統識別碼-Guid    | 2a39c5b0-8960-11d1-aebc-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
