# <a name="msmq-version-attribute"></a>MSMQ-Version 屬性

MSMQ DS 資訊的版本號碼。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | MSMQ-Version                         |
| Ldap-顯示名稱 | mSMQVersion                          |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.942               |
| 系統識別碼-Guid    | 9a0dc336-c100-11d1-bbc5-0080c76670c0 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
