# <a name="ms-wmi-int8max-attribute"></a>int8Max 屬性

WMI 64 位整數參數物件的最大值。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | int8Max                       |
| Ldap-顯示名稱 | msWMI-Int8Max                        |
| 大小              | 8 個位元組                              |
| 更新許可權  | 群組原則系統管理員           |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1633              |
| 系統識別碼-Guid    | e3d8b547-003d-4946-a32b-dc7cedc96b74 |
| Syntax            | [**間隔**](s-interval.md)       |
