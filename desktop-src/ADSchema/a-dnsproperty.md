# <a name="dns-property-attribute"></a>DNS-Property 屬性

用來將二進位設定 (屬性) 儲存在 DNS 區域物件上。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | DNS-Property                                          |
| Ldap-顯示名稱 | dNSProperty                                           |
| 大小              | \-                                                    |
| 更新許可權  | 此值是由系統所設定。                      |
| 更新頻率  | 每次 DNS 區域記錄變更時。                     |
| Attribute-Id      | 1.2.840.113556.1.4.1306                               |
| 系統識別碼-Guid    | 675a15fe-3b70-11d2-90cc-00c04fd91ab1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
