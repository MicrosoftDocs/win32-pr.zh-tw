# <a name="default-object-category-attribute"></a>預設-物件-類別目錄屬性

如果未指定物件，則為物件使用的物件類別。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | 預設-物件-類別                 |
| Ldap-顯示名稱 | defaultObjectCategory                   |
| 大小              | \-                                      |
| 更新許可權  | 架構系統管理員                    |
| 更新頻率  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.783                  |
| 系統識別碼-Guid    | 26d97367-6070-11d1-a9c6-0000f80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
