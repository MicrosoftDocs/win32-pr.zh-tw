# <a name="ms-pki-template-schema-version-attribute"></a>ms-PKI-Template-Schema-Version 屬性

持續追蹤 PKI 憑證範本類別的架構更新。



| 進入 | 值 |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-Template-Schema-Version                                                                    |
| Ldap-顯示名稱 | Mspki-certificate-name-flag-Template-Schema-Version                                                                     |
| 大小              | 4 個位元組                                                                                           |
| 更新許可權  | 網域系統管理員                                                                              |
| 更新頻率  | 當您編輯、建立或複製憑證範本時 (ms-PKI-憑證範本) 物件。 |
| Attribute-Id      | 1.2.840.113556.1.4.1434                                                                           |
| 系統識別碼-Guid    | 0c15e9f5-491d-4594-918f-32813a091da9                                                              |
| Syntax            | [**列舉型別**](s-enumeration.md)                                                              |
