# <a name="ms-wmi-targetnamespace-attribute"></a>ms-WMI-TargetNameSpace 屬性

要在其中建立物件的命名空間。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | ms-WMI-TargetNameSpace                      |
| Ldap-顯示名稱 | msWMI-TargetNameSpace                       |
| 大小              | 少於二十個字元。                |
| 更新許可權  | 群組原則系統管理員                  |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1646                     |
| 系統識別碼-Guid    | 1c4ab61f-3420-44e5-849d-8b5dbf60feb7        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
