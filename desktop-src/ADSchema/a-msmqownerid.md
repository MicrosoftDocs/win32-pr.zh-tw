| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | MSMQ-擁有者識別碼                                         |
| Ldap-顯示名稱 | mSMQOwnerID                                           |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.925                                |
| 系統識別碼-Guid    | 9a0dc328-c100-11d1-bbc5-0080c76670c0                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
