# <a name="anr-attribute"></a>ANR 屬性

在物件之間進行選擇時，要使用的明確名稱解析屬性。



| 進入 | 值 |
|-------------------|-------------------------------------------------------------------|
| CN                | Anr                                                               |
| Ldap-顯示名稱 | Anr                                                               |
| 大小              | \-                                                                |
| 更新許可權  | 架構系統管理員                                              |
| 更新頻率  | 當需要在 ANR 清單中新增或移除屬性時。 |
| Attribute-Id      | 1.2.840.113556.1.4.1208                                           |
| 系統識別碼-Guid    | 45b01500-c419-11d1-bbc9-0080c76670c0                              |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                       |
