# <a name="ms-wmi-changedate-attribute"></a>ChangeDate 屬性

此物件上次變更的日期。



| 進入 | 值 |
|-------------------|---------------------------------------------------|
| CN                | ChangeDate                                 |
| Ldap-顯示名稱 | msWMI-ChangeDate                                  |
| 大小              | WMI datetime 資料類型表示-26 個字元。 |
| 更新許可權  | 此值是由系統所設定。                  |
| 更新頻率  | \-                                                |
| Attribute-Id      | 1.2.840.113556.1.4.1624                           |
| 系統識別碼-Guid    | f9cdf7a0-ec44-4937-a79b-cd91522b3aa8              |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)       |
