# <a name="msmq-ds-service-attribute"></a>MSMQ-Ds-服務屬性

指出此伺服器是否提供 DS 服務。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | MSMQ-Ds-服務                      |
| Ldap-顯示名稱 | mSMQDsService                        |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1238              |
| 系統識別碼-Guid    | 2df90d82-009f-11d2-aa4c-00c04fd7d83a |
| Syntax            | [**Boolean**](s-boolean.md)         |
