# <a name="lock-out-observation-window-attribute"></a>鎖定-觀察-視窗屬性

| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 鎖定-觀察-視窗          |
| Ldap-顯示名稱 | Msds-lockoutobservationwindow             |
| 大小              | \-                                   |
| 更新許可權  | 網域系統管理員                 |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.61                |
| 系統識別碼-Guid    | bf9679a4-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**間隔**](s-interval.md)       |
