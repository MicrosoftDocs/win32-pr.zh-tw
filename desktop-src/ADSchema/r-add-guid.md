# <a name="add-guid-extended-right"></a>新增-GUID 擴充許可權

| 進入 | 值 |
|--------------|--------------------------------------|
| CN           | Add-GUID                             |
| 顯示名稱 | 加入 GUID                             |
| 許可權-GUID  | 440820ad-65b4-11d1-a3da-0000f875ae0d |
