# <a name="ms-wmi-int8default-attribute"></a>int8Default 屬性

WMI 64 位整數參數物件的預設值。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | int8Default                   |
| Ldap-顯示名稱 | msWMI-Int8Default                    |
| 大小              | 8 個位元組                              |
| 更新許可權  | 群組原則系統管理員           |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1632              |
| 系統識別碼-Guid    | f4d8085a-8c5b-4785-959b-dc585566e445 |
| Syntax            | [**間隔**](s-interval.md)       |
