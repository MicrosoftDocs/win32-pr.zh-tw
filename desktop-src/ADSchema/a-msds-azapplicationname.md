# <a name="ms-ds-az-application-name-attribute"></a>ms-chap-Az-應用程式名稱屬性

可唯一識別應用程式物件的字串。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | ms-DS-Az-應用程式名稱                   |
| Ldap-顯示名稱 | AzApplicationName                      |
| 大小              | 128 個字元                              |
| 更新許可權  | AzRoles 系統管理員                               |
| 更新頻率  | 在初始化或原則變更期間。     |
| Attribute-Id      | 1.2.840.113556.1.4.1798                     |
| 系統識別碼-Guid    | db5b0728-6208-4876-83b7-95d3e5695275        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
