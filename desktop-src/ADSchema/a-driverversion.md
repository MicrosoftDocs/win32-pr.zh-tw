# <a name="driver-version-attribute"></a>Driver-Version 屬性

設備磁碟機的版本號碼。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | Driver-Version                       |
| Ldap-顯示名稱 | driverVersion                        |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.276               |
| 系統識別碼-Guid    | ba305f6e-47e3-11d0-a1a6-00c04fd930c9 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
