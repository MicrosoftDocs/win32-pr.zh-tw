# <a name="ds-ui-shell-maximum-attribute"></a>DS-UI-Shell-最大屬性

這是 shell UI 在容器中顯示的預設最大物件數目。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | DS-UI-Shell-最大值                  |
| Ldap-顯示名稱 | dSUIShellMaximum                     |
| 大小              | \-                                   |
| 更新許可權  | 網域系統管理員                 |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1345              |
| 系統識別碼-Guid    | fcca766a-6f91-11d2-9905-0000f87a57d4 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
