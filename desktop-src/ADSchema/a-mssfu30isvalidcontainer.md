# <a name="mssfu-30-is-valid-container-attribute"></a>msSFU-30-有效-容器屬性

| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | msSFU-30-有效-容器          |
| Ldap-顯示名稱 | msSFU30IsValidContainer              |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.18.1.350          |
| 系統識別碼-Guid    | 0dea42f5-278d-4157-b4a7-49b59664915b |
| Syntax            | [**列舉型別**](s-enumeration.md) |
