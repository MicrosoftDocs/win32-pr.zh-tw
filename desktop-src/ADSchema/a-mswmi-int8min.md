# <a name="ms-wmi-int8min-attribute"></a>int8Min 屬性

WMI 64 位整數參數物件的最小值。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | int8Min                       |
| Ldap-顯示名稱 | msWMI-Int8Min                        |
| 大小              | 8 個位元組                              |
| 更新許可權  | 群組原則系統管理員           |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1634              |
| 系統識別碼-Guid    | ed1489d1-54cc-4066-b368-a00daa2664f1 |
| Syntax            | [**間隔**](s-interval.md)       |
