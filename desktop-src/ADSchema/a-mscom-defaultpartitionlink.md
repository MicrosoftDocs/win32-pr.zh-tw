# <a name="ms-com-defaultpartitionlink-attribute"></a>DefaultPartitionLink 屬性

用來識別 COM + PartitionSet 中「預設」 COM + 分割的連結。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | DefaultPartitionLink             |
| Ldap-顯示名稱 | msCOM-DefaultPartitionLink              |
| 大小              | \-                                      |
| 更新許可權  | \-                                      |
| 更新頻率  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1427                 |
| 系統識別碼-Guid    | 998b10f7-aa1a-4364-b867-753d197fe670    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
