# <a name="msnpallowdialin-attribute"></a>msNPAllowDialin 屬性

指出帳戶是否有權撥入 RAS 伺服器。 請勿直接修改此值。 使用適當的 [RAS 管理功能](/windows/desktop/RRAS/ras-administration-functions) 來修改此值。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | msNPAllowDialin                      |
| Ldap-顯示名稱 | msNPAllowDialin                      |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1119              |
| 系統識別碼-Guid    | db0c9085-c1f2-11d1-bbc5-0080c76670c0 |
| Syntax            | [**Boolean**](s-boolean.md)         |
