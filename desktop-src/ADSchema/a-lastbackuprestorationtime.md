# <a name="last-backup-restoration-time-attribute"></a>上次備份-還原時間屬性

上次發生系統還原的時間。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 上次備份-還原時間         |
| Ldap-顯示名稱 | lastBackupRestorationTime            |
| 大小              | 8 個位元組                              |
| 更新許可權  | 此值是由系統所設定。     |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.519               |
| 系統識別碼-Guid    | 1fbb0be8-ba63-11d0-afef-0000f80367c1 |
| Syntax            | [**間隔**](s-interval.md)       |
