# <a name="houseidentifier-attribute"></a>houseIdentifier 屬性

指定用來識別特定大樓的語言結構，例如，與街道、市、城鎮、城市等相關的房屋或房子名稱。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | houseIdentifier                             |
| Ldap-顯示名稱 | houseIdentifier                             |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 2.5.4.51                                    |
| 系統識別碼-Guid    | a45398b7-c44a-4eb6-82d3-13c10946dbfe        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
