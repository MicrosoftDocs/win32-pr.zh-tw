# <a name="ms-ieee-80211-data-type-attribute"></a>ms-chap-ieee-80211-資料類型屬性

Msieee80211 資料 BLOB 的內部使用資料類型。



| 進入 | 值 |
|-------------------|----------------------------------------------------------------------------------|
| CN                | ms-chap-ieee-80211-資料類型                                                          |
| Ldap-顯示名稱 | msieee80211-DataType                                                             |
| 大小              | \-                                                                               |
| 更新許可權  | 網域系統管理員                                                             |
| 更新頻率  | 每次網域管理員變更網域或 OU 的無線網路原則時。 |
| Attribute-Id      | 1.2.840.113556.1.4.1822                                                          |
| 系統識別碼-Guid    | 6558b180-35da-4efe-beed-521f8f48cafb                                             |
| Syntax            | [**列舉型別**](s-enumeration.md)                                             |
