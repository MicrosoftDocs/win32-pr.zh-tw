# <a name="server-state-attribute"></a>Server-State 屬性

指出伺服器是否已啟用或停用。 值為1表示已啟用伺服器。 值為2表示伺服器已停用。 所有其他值都無效。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | Server-State                         |
| Ldap-顯示名稱 | serverState                          |
| 大小              | \-                                   |
| 更新許可權  | 此值是由系統所設定。     |
| 更新頻率  | 當使用者的原則變更時。  |
| Attribute-Id      | 1.2.840.113556.1.4.154               |
| 系統識別碼-Guid    | bf967a34-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
