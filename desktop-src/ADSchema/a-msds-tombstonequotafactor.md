# <a name="ms-ds-tombstone-quota-factor-attribute"></a>ms DS-標記-配額因素屬性

|-------------------|--------------------------------------|
| CN                | ms-DS-標記-配額因素         |
| Ldap-顯示名稱 | TombstoneQuotaFactor            |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1847              |
| 系統識別碼-Guid    | 461744d7-f3b6-45ba-8753-fb9552a5df32 |
