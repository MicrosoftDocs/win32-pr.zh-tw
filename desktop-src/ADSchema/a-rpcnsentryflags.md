# <a name="rpc-ns-entry-flags-attribute"></a>rpc-Ns-Entry 旗標屬性

表示已明確建立 RPC NS 專案的旗標。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | rpc-Ns-Entry-旗標                   |
| Ldap-顯示名稱 | rpcNsEntryFlags                      |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.754               |
| 系統識別碼-Guid    | 80212841-4bdc-11d1-a9c4-0000f80367c1 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
