# <a name="acs-permission-bits-attribute"></a>ACS-許可權-Bits 屬性

允許指定使用者的多播流程來源。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ACS-許可權位                  |
| Ldap-顯示名稱 | aCSPermissionBits                    |
| 大小              | 8 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.765               |
| 系統識別碼-Guid    | 7f561282-5301-11d1-a9c5-0000f80367c1 |
| Syntax            | [**間隔**](s-interval.md)       |
