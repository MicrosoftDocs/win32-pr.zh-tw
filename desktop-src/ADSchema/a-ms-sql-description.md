# <a name="ms-sql-description-attribute"></a>TRANSACT-SQL-Description 屬性

SQL Server 資料庫的描述。



| 進入 | 值 |
|-------------------|--------------------------------------------------|
| CN                | MS-SQL-Description                               |
| Ldap-顯示名稱 | mS-SQL-Description                               |
| 大小              | \-                                               |
| 更新許可權  | 網域系統管理員                             |
| 更新頻率  | 當使用者決定變更描述時。 |
| Attribute-Id      | 1.2.840.113556.1.4.1390                          |
| 系統識別碼-Guid    | 8386603c-ccef-11d2-9993-0000f87a57d4             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)      |
