布林值，指定這個類別之新實例的 showInAdvancedViewOnly 屬性預設設定。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 預設值-隱藏-值                 |
| Ldap-顯示名稱 | defaultHidingValue                   |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.518               |
| 系統識別碼-Guid    | b7b13116-b82e-11d0-afee-0000f80367c1 |
| Syntax            | [**Boolean**](s-boolean.md)         |
