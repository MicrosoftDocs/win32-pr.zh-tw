# <a name="bridgehead-transport-list-attribute"></a>橋頭-傳輸-清單屬性

此伺服器為其橋頭的傳輸。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | 橋頭-傳輸-清單               |
| Ldap-顯示名稱 | bridgeheadTransportList                 |
| 大小              | \-                                      |
| 更新許可權  | 此值是由系統所設定。        |
| 更新頻率  | 每次設定網站時。               |
| Attribute-Id      | 1.2.840.113556.1.4.819                  |
| 系統識別碼-Guid    | d50c2cda-8951-11d1-aebc-0000f80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
