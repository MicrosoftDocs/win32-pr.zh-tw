# <a name="ms-wmi-name-attribute"></a>ms-WMI-Name 屬性

最上層原則物件的顯示名稱。 用於通用類別目錄中。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | ms-WMI-名稱                                 |
| Ldap-顯示名稱 | msWMI-名稱                                  |
| 大小              | \-                                          |
| 更新許可權  | 群組原則系統管理員                  |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1639                     |
| 系統識別碼-Guid    | c6c8ace5-7e81-42af-ad72-77412c5941c4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
