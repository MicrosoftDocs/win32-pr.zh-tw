# <a name="global-address-list-attribute"></a>全域通訊清單屬性

這個屬性在 Microsoft Exchange 容器中是用來儲存新建立的全域通訊清單 (GAL) 的分辨名稱。 在您讓訊息應用程式發展介面 (MAPI) 用戶端使用 GAL 之前，這個屬性必須擁有一個項目。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | 全域通訊清單                     |
| Ldap-顯示名稱 | globalAddressList                       |
| 大小              | \-                                      |
| 更新許可權  | \-                                      |
| 更新頻率  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1245                 |
| 系統識別碼-Guid    | f754c748-06f4-11d2-aa53-00c04fd7d83a    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |



## <a name="implementations"></a>實作
