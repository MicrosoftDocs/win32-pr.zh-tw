# <a name="tree-name-attribute"></a>Tree-Name 屬性

位於樹狀結構根目錄之網域的 DNS 名稱。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | Tree-Name                                   |
| Ldap-顯示名稱 | treeName                                    |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.660                      |
| 系統識別碼-Guid    | 28630ebd-41d5-11d1-a9c1-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
