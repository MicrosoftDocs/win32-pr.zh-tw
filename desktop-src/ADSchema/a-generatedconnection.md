# <a name="generated-connection-attribute"></a>Generated-Connection 屬性

如果此連接是由自動拓撲產生所建立，**則為 True** 。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | Generated-Connection                 |
| Ldap-顯示名稱 | generatedConnection                  |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.41                |
| 系統識別碼-Guid    | bf96797a-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Boolean**](s-boolean.md)         |
