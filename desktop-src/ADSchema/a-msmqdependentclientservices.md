# <a name="msmq-dependent-client-services-attribute"></a>MSMQ 相依-用戶端服務屬性

指出安裝在這部電腦上的 MSMQ 是否提供 MSMQ 相依用戶端服務。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | MSMQ 相依-用戶端服務       |
| Ldap-顯示名稱 | mSMQDependentClientServices          |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1226              |
| 系統識別碼-Guid    | 2df90d76-009f-11d2-aa4c-00c04fd7d83a |
| Syntax            | [**Boolean**](s-boolean.md)         |
