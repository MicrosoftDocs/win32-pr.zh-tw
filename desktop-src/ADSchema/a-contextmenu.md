# <a name="context-menu-attribute"></a>Context-Menu 屬性

要用於物件的內容功能表的順序編號和 GUID。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | Context-Menu                                |
| Ldap-顯示名稱 | coNtextMenu                                 |
| 大小              | \-                                          |
| 更新許可權  | 架構系統管理員                        |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.499                      |
| 系統識別碼-Guid    | 4d8601ee-ac85-11d0-afe3-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
