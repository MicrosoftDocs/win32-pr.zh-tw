# <a name="print-keep-printed-jobs-attribute"></a>列印-保持列印-工作屬性

如果保留列印工作，**則為 TRUE** 。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 列印-保留列印-工作              |
| Ldap-顯示名稱 | printKeepPrintedJobs                 |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.275               |
| 系統識別碼-Guid    | ba305f6d-47e3-11d0-a1a6-00c04fd930c9 |
| Syntax            | [**Boolean**](s-boolean.md)         |
