# <a name="email-information-property-set"></a>Email-Information 屬性集

包含使用者屬性的屬性集，這些屬性會描述使用者電子郵件的相關資訊。



| 進入 | 值 |
|--------------|--------------------------------------|
| CN           | Email-Information                    |
| 顯示名稱 | 電話和 Mail 選項               |
| 許可權-GUID  | e45795b2-9455-11d1-aebd-0000f80367c1 |
