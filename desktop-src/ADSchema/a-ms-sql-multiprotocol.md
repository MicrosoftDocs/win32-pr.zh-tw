# <a name="ms-sql-multiprotocol-attribute"></a>MS-CHAP-多重通訊協定屬性

RPC 連接點。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | MS-CHAP-多重通訊協定                        |
| Ldap-顯示名稱 | Ms-chap-多重通訊協定                        |
| 大小              | \-                                          |
| 更新許可權  | 此值是由系統所設定。            |
| 更新頻率  | 在系統啟動時。                          |
| Attribute-Id      | 1.2.840.113556.1.4.1375                     |
| 系統識別碼-Guid    | 8157fa38-ccee-11d2-9993-0000f87a57d4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
