# <a name="rpc-ns-priority-attribute"></a>rpc-Ns-Priority 屬性

給定 RPC 設定檔專案的優先權。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | rpc-Ns-優先順序                      |
| Ldap-顯示名稱 | rpcNsPriority                        |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.117               |
| 系統識別碼-Guid    | bf967a27-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
