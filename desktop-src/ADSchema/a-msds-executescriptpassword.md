# <a name="ms-ds-executescriptpassword-attribute"></a>ExecuteScriptPassword 屬性

在網域重新命名期間使用。 此值無法使用 LDAP 寫入或讀取。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | ExecuteScriptPassword                           |
| Ldap-顯示名稱 | ExecuteScriptPassword                            |
| 大小              | \-                                                    |
| 更新許可權  | 此值是由系統所設定。                      |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1783                               |
| 系統識別碼-Guid    | 9d054a5a-d187-46c1-9d85-42dfc44a56dd                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
