# <a name="ms-ds-allowed-dns-suffixes-attribute"></a>允許的 ms-DS-DNS 尾碼屬性

電腦物件中 dNSHostName 屬性允許的尾碼清單。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | ms-DS-允許的 DNS 尾碼                  |
| Ldap-顯示名稱 | >msds-alloweddnssuffixes                     |
| 大小              | \-                                          |
| 更新許可權  | 網域系統管理員                        |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1710                     |
| 系統識別碼-Guid    | 8469441b-9ac4-4e45-8205-bd219dbf672d        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
