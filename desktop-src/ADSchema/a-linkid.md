# <a name="link-id-attribute"></a>連結識別碼屬性

表示屬性為連結屬性的整數。 偶數的整數是轉寄連結，而奇數整數是反向連結。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 連結識別碼                              |
| Ldap-顯示名稱 | linkID                               |
| 大小              | \-                                   |
| 更新許可權  | 架構系統管理員                 |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.50                |
| 系統識別碼-Guid    | bf96799b-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
