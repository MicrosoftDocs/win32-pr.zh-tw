# <a name="cost-attribute"></a>Cost 屬性

包含透過特定網站連接器路由傳送訊息的相對成本。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 成本                                 |
| Ldap-顯示名稱 | cost                                 |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.135               |
| 系統識別碼-Guid    | bf967944-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
