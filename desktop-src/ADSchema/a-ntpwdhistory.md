# <a name="nt-pwd-history-attribute"></a>Nt-Pwd-歷程記錄屬性

Windows NT 單向格式的使用者密碼歷程記錄 (OWF) 。 Windows 2000 使用 Windows NT OWF。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | Nt-Pwd-歷程記錄                                        |
| Ldap-顯示名稱 | ntPwdHistory                                          |
| 大小              | \-                                                    |
| 更新許可權  | 此值是由系統所設定。                      |
| 更新頻率  | 每次使用者變更密碼時。                 |
| Attribute-Id      | 1.2.840.113556.1.4.94                                 |
| 系統識別碼-Guid    | bf9679e2-0de6-11d0-a285-00aa003049e2                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
