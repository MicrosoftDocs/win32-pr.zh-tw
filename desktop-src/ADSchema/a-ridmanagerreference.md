# <a name="rid-manager-reference-attribute"></a>RID 管理員-Reference 屬性

物件之 RID 管理員的分辨名稱。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | RID 管理員-參考                   |
| Ldap-顯示名稱 | rIDManagerReference                     |
| 大小              | \-                                      |
| 更新許可權  | 此值是由系統所設定。        |
| 更新頻率  | 每次 RID 管理員變更時。       |
| Attribute-Id      | 1.2.840.113556.1.4.368                  |
| 系統識別碼-Guid    | 66171886-8f3c-11d0-afda-00c04fd930c9    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
