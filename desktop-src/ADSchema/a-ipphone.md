# <a name="phone-ip-primary-attribute"></a>電話-Ip-主要屬性

電話的 TCP/IP 位址。 由電話語音使用。



| 進入 | 值 |
|-------------------|----------------------------------------------------------------------------------|
| CN                | 電話-Ip-主要                                                                 |
| Ldap-顯示名稱 | ipPhone                                                                          |
| 大小              | \-                                                                               |
| 更新許可權  | 網域系統管理員或帳戶擁有者。                                           |
| 更新頻率  | 當使用者的記錄建立時，以及每次電話號碼需要變更時。 |
| Attribute-Id      | 1.2.840.113556.1.4.721                                                           |
| 系統識別碼-Guid    | 4d146e4a-48d4-11d1-a9c3-0000f80367c1                                             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                      |
