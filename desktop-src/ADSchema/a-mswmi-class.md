# <a name="ms-wmi-class-attribute"></a>ms WMI 類別屬性

相關編碼 (中 WMI 類別物件的名稱，例如 Win32 \_) 。 編碼可以是實例或類別物件。



| 進入 | 值 |
|-------------------|-----------------------------------------------------------------|
| CN                | ms WMI 類別                                                    |
| Ldap-顯示名稱 | msWMI 類別                                                     |
| 大小              | \-                                                              |
| 更新許可權  | 網域系統管理員                                            |
| 更新頻率  | 只要加入或修改包含屬性的類別。 |
| Attribute-Id      | 1.2.840.113556.1.4.1676                                         |
| 系統識別碼-Guid    | 90c1925f-4a24-4b07-b202-be32eb3c8b74                            |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                     |
