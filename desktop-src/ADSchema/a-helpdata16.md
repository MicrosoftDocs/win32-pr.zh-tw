# <a name="help-data16-attribute"></a>Help-Data16 屬性

這個屬性是用於 Exchange 4.0 的 Win16 說明檔案格式。 它不會用於任何其他版本的 Exchange。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | Help-Data16                                           |
| Ldap-顯示名稱 | helpData16                                            |
| 大小              | \-                                                    |
| 更新許可權  | 系統會使用此方法。                           |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.2.402                                |
| 系統識別碼-Guid    | 5fd424a7-1262-11d0-a060-00aa006c33ed                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
