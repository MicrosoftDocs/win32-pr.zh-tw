# <a name="template-roots-attribute"></a>Template-Roots 屬性

這個屬性在 Exchange config 容器中用來指出範本容器的儲存位置。 此資訊供 Active Directory MAPI 提供者使用。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | Template-Roots                          |
| Ldap-顯示名稱 | templateRoots                           |
| 大小              | \-                                      |
| 更新許可權  | 系統會使用此方法。             |
| 更新頻率  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1346                 |
| 系統識別碼-Guid    | ed9de9a0-7041-11d2-9905-0000f87a57d4    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |



## <a name="implementations"></a>實作
