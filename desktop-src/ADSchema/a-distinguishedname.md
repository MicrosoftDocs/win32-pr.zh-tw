# <a name="obj-dist-name-attribute"></a>Obj-Dist-Name 屬性

與物件的辨別名稱相同。 供 Exchange 使用。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | Obj-Dist 名稱                           |
| Ldap-顯示名稱 | distinguishedName                       |
| 大小              | \-                                      |
| 更新許可權  | 此值是由系統所設定。        |
| 更新頻率  | 每次建立或移動物件時。 |
| Attribute-Id      | 2.5.4.49                                |
| 系統識別碼-Guid    | bf9679e4-0de6-11d0-a285-00aa003049e2    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
