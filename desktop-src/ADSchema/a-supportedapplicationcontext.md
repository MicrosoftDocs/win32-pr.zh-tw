指定 OSI 應用程式支援之應用程式內容的物件識別碼。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | 支援-應用程式內容                         |
| Ldap-顯示名稱 | supportedApplicationCoNtext                           |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 2.5.4.30                                              |
| 系統識別碼-Guid    | 1677588f-47f3-11d1-a9c3-0000f80367c1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
