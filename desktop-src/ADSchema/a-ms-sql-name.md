# <a name="ms-sql-name-attribute"></a>MS-CHAP-Name 屬性

SQL Server 實例的名稱。 預設值為 MSSQLSERVER。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-名稱                                 |
| Ldap-顯示名稱 | mS-SQL-名稱                                 |
| 大小              | \-                                          |
| 更新許可權  | 此值是由系統所設定。            |
| 更新頻率  | 在系統設定時。                            |
| Attribute-Id      | 1.2.840.113556.1.4.1363                     |
| 系統識別碼-Guid    | 3532dfd8-ccee-11d2-9993-0000f87a57d4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
