# <a name="ds-check-stale-phantoms-extended-right"></a>DS-檢查過時-幻像擴充許可權

| 進入 | 值 |
|--------------|--------------------------------------|
| CN           | DS-檢查過時-幻像              |
| 顯示名稱 | 檢查過時的幻像                 |
| 許可權-GUID  | 69ae6200-7f46-11d2-b9ad-00c04f79f805 |
