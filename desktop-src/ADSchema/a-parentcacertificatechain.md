# <a name="parent-ca-certificate-chain-attribute"></a>父系 CA-憑證鏈屬性

父憑證授權單位單位的 DER 編碼 x.509v3 憑證。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | 上層 CA-憑證鏈                           |
| Ldap-顯示名稱 | parentCACertificateChain                              |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.685                                |
| 系統識別碼-Guid    | 963d2733-48be-11d1-a9c3-0000f80367c1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
