# <a name="ms-ds-az-domain-timeout-attribute"></a>ms-chap-Az-Domain-Timeout 屬性

偵測到無法連線到網域之後，以及重新嘗試 DC 之前的時間（以毫秒為單位）。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | ms-chap-Az-Domain-Timeout                 |
| Ldap-顯示名稱 | AzDomainTimeout                    |
| 大小              | \-                                      |
| 更新許可權  | AzRoles 系統管理員                           |
| 更新頻率  | 在初始化或原則變更期間。 |
| Attribute-Id      | 1.2.840.113556.1.4.1795                 |
| 系統識別碼-Guid    | 6448f56a-ca70-4e2e-b0af-d20e4ce653d0    |
| Syntax            | [**列舉型別**](s-enumeration.md)    |
