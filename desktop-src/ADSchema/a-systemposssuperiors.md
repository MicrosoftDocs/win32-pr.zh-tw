# <a name="system-poss-superiors-attribute"></a>系統 Poss-Superiors 屬性

可包含這個類別的類別清單。 這份清單只能由系統修改。



| 進入 | 值 |
|-------------------|-----------------------------------------------------------------------------|
| CN                | 系統-Poss-Superiors                                                       |
| Ldap-顯示名稱 | systemPossSuperiors                                                         |
| 大小              | \-                                                                          |
| 更新許可權  | 架構系統管理員                                                        |
| 更新頻率  | 當建立類別，或將新的可能高階加入至類別時。 |
| Attribute-Id      | 1.2.840.113556.1.4.195                                                      |
| 系統識別碼-Guid    | bf967a47-0de6-11d0-a285-00aa003049e2                                        |
| Syntax            | [**String(Object-Identifier)**](s-string-object-identifier.md)             |



## <a name="implementations"></a>實作

-   [**Windows 2000 Server**](#windows-2000-server)
-   [**Windows Server 2003**](#windows-server-2003)
-   [**亞當**](#adam)
-   [**Windows Server 2003 R2**](#windows-server-2003-r2)
