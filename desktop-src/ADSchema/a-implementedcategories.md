# <a name="implemented-categories-attribute"></a>Implemented-Categories 屬性

此屬性包含此物件所執行之元件類別識別碼的清單。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | Implemented-Categories                                |
| Ldap-顯示名稱 | implementedCategories                                 |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.320                                |
| 系統識別碼-Guid    | 7d6c0e92-7e20-11d0-afd6-00c04fd930c9                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |



## <a name="implementations"></a>實作

-   [**Windows 2000 Server**](#windows-2000-server)

-   [**Windows Server 2008 R2**](#windows-server-2008-r2)
