# <a name="x500uniqueidentifier-attribute"></a>x500uniqueIdentifier 屬性

在重複使用辨別名稱時用來區別物件。 這是來自 uid 和 uniqueIdentifier 類型的不同屬性類型。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | x500uniqueIdentifier                                  |
| Ldap-顯示名稱 | x500uniqueIdentifier                                  |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 2.5.4.45                                              |
| 系統識別碼-Guid    | d07da11f-8a3d-42b6-b0aa-76c962be719a                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
