# <a name="documentidentifier-attribute"></a>documentIdentifier 屬性

DocumentIdentifier 屬性類型會指定檔的唯一識別碼。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | documentIdentifier                          |
| Ldap-顯示名稱 | documentIdentifier                          |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.11                  |
| 系統識別碼-Guid    | 0b21ce82-ff63-46d9-90fb-c8b9f24e97b9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
