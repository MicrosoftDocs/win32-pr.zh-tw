# <a name="group-type-attribute"></a>Group-Type 屬性

包含一組旗標，這些旗標會定義群組物件的類型和範圍。 如需這個屬性的可能值，請參閱備註。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | Group-Type                           |
| Ldap-顯示名稱 | groupType                            |
| 大小              | \-                                   |
| 更新許可權  | 網域系統管理員                 |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.750               |
| 系統識別碼-Guid    | 9a9a021e-4a5b-11d1-a9c3-0000f80367c1 |
| Syntax            | [**列舉型別**](s-enumeration.md) |

## <a name="remarks"></a>備註

這個屬性可以是零或一或多個下列值的組合。



| 值                   | 描述                                                                                  |
|-------------------------|----------------------------------------------------------------------------------------------|
| 1 (0x00000001)          | 指定系統所建立的群組。                                             |
| 2 (0x00000002)           | 指定具有全域領域的群組。                                                         |
| 4 (0x00000004)           | 指定具有網域區域範圍的群組。                                                   |
| 8 (0x00000008)           | 指定具有通用範圍的群組。                                                      |
| 16 (0x00000010)          | 指定 Windows Server 授權管理員的應用程式 \_ 基本群組。                      |
| 32 (0x00000020)          | 指定 Windows Server 授權管理員的應用程式 \_ 查詢群組。                      |
| 2147483648 (0x80000000)  | 指定安全性群組。 如果未設定此旗標，則群組為通訊群組。 |



 

如需群組類型與範圍的詳細資訊，請參閱 Microsoft TechNet 上的 [群組類型](/previous-versions/windows/it-pro/windows-server-2003/cc781446(v=ws.10)) 和 [群組領域](/previous-versions/windows/it-pro/windows-server-2003/cc755692(v=ws.10)) 主題。
