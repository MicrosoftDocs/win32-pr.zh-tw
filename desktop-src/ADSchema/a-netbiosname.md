# <a name="netbios-name-attribute"></a>NETBIOS-Name 屬性

要在 NetBIOS 上使用的物件名稱。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | NETBIOS-Name                                |
| Ldap-顯示名稱 | nETBIOSName                                 |
| 大小              | \-                                          |
| 更新許可權  | 網域系統管理員                        |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.87                       |
| 系統識別碼-Guid    | bf9679d8-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
