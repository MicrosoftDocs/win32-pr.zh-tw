# <a name="ms-ds-other-settings-attribute"></a>ms DS-其他-設定屬性

此多重值屬性是用來儲存以 NAME = VALUE 格式儲存的 DS 任何可設定的設定。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | ms DS-其他-設定                        |
| Ldap-顯示名稱 | 其他設定                         |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1621                     |
| 系統識別碼-Guid    | 79d2f34c-9d7d-42bb-838f-866b3e4400e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
