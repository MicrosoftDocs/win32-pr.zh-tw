# <a name="ms-ds-consistency-child-count-attribute"></a>MS DS-一致性-子計數屬性

這個屬性是用來藉由比較子物件的計數，來檢查目錄與另一個物件、資料庫或應用程式之間的一致性。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | MS DS-一致性-子計數        |
| Ldap-顯示名稱 | ConsistencyChildCount          |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1361              |
| 系統識別碼-Guid    | 178b7bc2-b63a-11d2-90e1-00c04fd91ab1 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
