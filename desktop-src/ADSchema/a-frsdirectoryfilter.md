# <a name="frs-directory-filter-attribute"></a>FRS-目錄-篩選屬性

| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | FRS-目錄-篩選                        |
| Ldap-顯示名稱 | fRSDirectoryFilter                          |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.484                      |
| 系統識別碼-Guid    | 1be8f171-a9ff-11d0-afe2-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
