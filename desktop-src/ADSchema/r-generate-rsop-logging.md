# <a name="generate-rsop-logging-extended-right"></a>產生-RSoP-記錄延伸許可權

擁有 OU 或網域之許可權的使用者，將能夠針對 OU 內的使用者或電腦產生記錄模式的 RSoP 資料。



| 進入 | 值 |
|--------------|--------------------------------------------|
| CN           | 產生-RSoP-記錄                      |
| 顯示名稱 | 產生原則結果組 (記錄) |
| 許可權-GUID  | b7b1b3de-ab09-4242-9e30-9980e5d322f7       |
