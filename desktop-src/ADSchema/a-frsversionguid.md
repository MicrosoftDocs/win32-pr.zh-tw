# <a name="frs-version-guid-attribute"></a>FRS-Version-GUID 屬性

如果有的話，變更此值表示已在此複本集上進行設定變更。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | FRS-版本-GUID                                      |
| Ldap-顯示名稱 | fRSVersionGUID                                        |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.43                                 |
| 系統識別碼-Guid    | 26d9736c-6070-11d1-a9c6-0000f80367c1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
