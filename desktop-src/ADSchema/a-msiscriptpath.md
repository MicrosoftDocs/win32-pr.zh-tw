# <a name="msi-script-path-attribute"></a>Msi-腳本-Path 屬性

此應用程式之 Microsoft installer 公告腳本檔案的檔案路徑。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | Msi-腳本-路徑                             |
| Ldap-顯示名稱 | msiScriptPath                               |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.15                       |
| 系統識別碼-Guid    | bf967937-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
