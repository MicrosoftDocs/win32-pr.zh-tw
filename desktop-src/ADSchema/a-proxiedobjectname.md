# <a name="proxied-object-name-attribute"></a>Proxy-物件名稱屬性

這個屬性是由 Active Directory 在內部使用，以協助追蹤有域移動的移動。



| 進入 | 值 |
|-------------------|-------------------------------------------------|
| CN                | Proxy-物件名稱                             |
| Ldap-顯示名稱 | proxiedObjectName                               |
| 大小              | \-                                              |
| 更新許可權  | 系統會使用此方法。                     |
| 更新頻率  | \-                                              |
| Attribute-Id      | 1.2.840.113556.1.4.1249                         |
| 系統識別碼-Guid    | e1aea402-cd5b-11d0-afff-0000f80367c1            |
| Syntax            | [**Object(DN-Binary)**](s-object-dn-binary.md) |
