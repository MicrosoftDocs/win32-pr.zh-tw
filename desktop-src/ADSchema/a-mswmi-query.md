# <a name="ms-wmi-query-attribute"></a>ms-WMI-查詢屬性

單一 WQL 查詢。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | ms-WMI-查詢                                |
| Ldap-顯示名稱 | msWMI-查詢                                 |
| 大小              | 少於250個字元。                   |
| 更新許可權  | 群組原則系統管理員                  |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1642                     |
| 系統識別碼-Guid    | 65fff93e-35e3-45a3-85ae-876c6718297f        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
