# <a name="userclass-attribute"></a>userClass 屬性

UserClass 屬性類型會指定電腦使用者的類別。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | userClass                                   |
| Ldap-顯示名稱 | userClass                                   |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.8                   |
| 系統識別碼-Guid    | 11732a8a-e14d-4cc5-b92f-d93f51c6d8e4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
