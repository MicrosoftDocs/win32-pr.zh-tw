| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | 其他受信任的服務名稱            |
| Ldap-顯示名稱 | additionalTrustedServiceNames               |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |

| 系統識別碼-Guid    | 032160be-9824-11d1-aec0-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
