# <a name="link-track-secret-attribute"></a>連結-追蹤-秘密屬性

這個屬性會儲存秘密金鑰的連結，讓加密的檔案能夠轉譯為純文字。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | 連結-追蹤-秘密                                     |
| Ldap-顯示名稱 | linkTrackSecret                                       |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.269                                |
| 系統識別碼-Guid    | 2ae80fe2-47b4-11d0-a1a4-00c04fd930c9                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
