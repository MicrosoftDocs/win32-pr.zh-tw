# <a name="bootparameter-attribute"></a>bootParameter 屬性

提供啟動無磁片用戶端所需的資料。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | bootParameter                        |
| Ldap-顯示名稱 | bootParameter                        |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.23                     |
| 系統識別碼-Guid    | d72a0750-8c7c-416e-8714-e65f11e908be |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
