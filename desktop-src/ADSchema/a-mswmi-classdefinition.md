# <a name="ms-wmi-classdefinition-attribute"></a>ClassDefinition 屬性

保存要在某些 WMI 命名空間中 enstated 的類別定義。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | ClassDefinition                      |
| Ldap-顯示名稱 | msWMI-ClassDefinition                       |
| 大小              | \-                                          |
| 更新許可權  | 此值是由系統所設定。            |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1625                     |
| 系統識別碼-Guid    | 2b9c0ebc-c272-45cb-99d2-4d0e691632e0        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
