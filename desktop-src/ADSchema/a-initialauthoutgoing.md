# <a name="initial-auth-outgoing-attribute"></a>初始-驗證-傳出屬性

包含驗證服務器對要求驗證的用戶端所傳送之初始傳出驗證的相關資訊。 使用這個屬性的伺服器會接收來自驗證服務器的授權，並將其傳送至用戶端。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | 初始-驗證-外寄                       |
| Ldap-顯示名稱 | initialAuthOutgoing                         |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.540                      |
| 系統識別碼-Guid    | 52458024-ca6a-11d0-afff-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
