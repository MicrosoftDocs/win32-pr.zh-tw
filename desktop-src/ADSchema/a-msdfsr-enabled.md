# <a name="ms-dfsr-enabled-attribute"></a>啟用 ms-chap 的屬性

指定是否啟用物件。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 啟用 ms-DFSR                      |
| Ldap-顯示名稱 | msDFSR-已啟用                       |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.9            |
| 系統識別碼-Guid    | 03726ae7-8e7d-4446-8aae-a91657c00993 |
| Syntax            | [**Boolean**](s-boolean.md)         |
