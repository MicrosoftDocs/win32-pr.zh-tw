| Ldap-顯示名稱 | extendedAttributeInfo                       |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |

| 系統識別碼-Guid    | 9a7ad947-ca53-11d1-bbd0-0080c76670c0        |
