# <a name="account-expires-attribute"></a>Account-Expires 屬性

帳戶到期的日期。 此值代表自1601年1月1日 (UTC) 起的 100-毫微秒間隔數。 值為0或 0x7FFFFFFFFFFFFFFF (9223372036854775807) 表示帳戶永遠不會過期。



| 進入 | 值 |
|-------------------|------------------------------------------------------------------------|
| CN                | Account-Expires                                                        |
| Ldap-顯示名稱 | accountExpires                                                         |
| 大小              | 8 個位元組                                                                |
| 更新許可權  | 網域系統管理員會設定此屬性。                          |
| 更新頻率  | 當先前的到期日過期且需要更新時。 |
| Attribute-Id      | 1.2.840.113556.1.4.159                                                 |
| 系統識別碼-Guid    | bf967915-0de6-11d0-a285-00aa003049e2                                   |
| Syntax            | [**間隔**](s-interval.md)                                         |
