|-------------------|-------------------------------------------------------|
| CN                | 憑證-撤銷清單                           |
| Ldap-顯示名稱 | t                             |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 2.5.4.39                                              |
| 系統識別碼-Guid    | 1677579f-47f3-11d1-a9c3-0000f80367c1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
