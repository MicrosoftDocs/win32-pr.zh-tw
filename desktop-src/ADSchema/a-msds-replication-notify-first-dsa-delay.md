# <a name="ms-ds-replication-notify-first-dsa-delay-attribute"></a>ms DS-複寫-通知-第一個-DSA-延遲屬性

這個屬性會控制對 DS 的變更之間的延遲時間，以及 NC 第一個複本夥伴的通知。



| 進入 | 值 |
|-------------------|------------------------------------------|
| CN                | ms DS-複寫-通知-第一個-DSA-延遲 |
| Ldap-顯示名稱 | 複寫-通知-第一個-DSA-延遲  |
| 大小              | \-                                       |
| 更新許可權  | 此值是由系統所設定。         |
| 更新頻率  | \-                                       |
| Attribute-Id      | 1.2.840.113556.1.4.1663                  |
| 系統識別碼-Guid    | 85abd4f4-0a89-4e49-bdec-6f35bb2562ba     |
| Syntax            | [**列舉型別**](s-enumeration.md)     |
