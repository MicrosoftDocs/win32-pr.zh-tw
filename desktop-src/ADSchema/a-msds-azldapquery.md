# <a name="ms-ds-az-ldap-query-attribute"></a>ms DS-Az-LDAP-查詢屬性

定義 LDAP 查詢的字串， (最大長度 4096) ，以決定使用者物件對群組的成員資格。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | ms DS-Az-LDAP-查詢                         |
| Ldap-顯示名稱 | AzLDAPQuery                            |
| 大小              | 4096個字元                             |
| 更新許可權  | AzRoles 系統管理員                               |
| 更新頻率  | 在初始化或原則變更期間。     |
| Attribute-Id      | 1.2.840.113556.1.4.1792                     |
| 系統識別碼-Guid    | 5e53368b-fc94-45c8-9d7d-daf31ee7112d        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
