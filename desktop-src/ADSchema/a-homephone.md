# <a name="phone-home-primary-attribute"></a>電話-首頁-主要屬性

使用者的主要家庭電話號碼。



| 進入 | 值 |
|-------------------|----------------------------------------------------------------------------------|
| CN                | 電話-首頁-主要                                                               |
| Ldap-顯示名稱 | homePhone                                                                        |
| 大小              | \-                                                                               |
| 更新許可權  | 網域系統管理員或帳戶擁有者。                                           |
| 更新頻率  | 當使用者的記錄建立時，以及每次電話號碼需要變更時。 |
| Attribute-Id      | 0.9.2342.19200300.100.1.20                                                       |
| 系統識別碼-Guid    | f0f8ffa1-1191-11d0-a060-00aa006c33ed                                             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                      |
