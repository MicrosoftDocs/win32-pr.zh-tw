| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | User-Cert                                             |
| Ldap-顯示名稱 | userCert                                              |
| 大小              | \-                                                    |
| 更新許可權  | 此值是由系統所設定。                      |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.645                                |
| 系統識別碼-Guid    | bf967a69-0de6-11d0-a285-00aa003049e2                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
