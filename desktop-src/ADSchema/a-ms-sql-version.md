# <a name="ms-sql-version-attribute"></a>TRANSACT-SQL-Version 屬性

SQL Server 目前實例的版本。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-版本                              |
| Ldap-顯示名稱 | mS-SQL-版本                              |
| 大小              | \-                                          |
| 更新許可權  | 此值是由系統所設定。            |
| 更新頻率  | 在系統設定時。                            |
| Attribute-Id      | 1.2.840.113556.1.4.1388                     |
| 系統識別碼-Guid    | c07cc1d0-ccee-11d2-9993-0000f87a57d4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
