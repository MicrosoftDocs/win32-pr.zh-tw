# <a name="parent-guid-attribute"></a>Parent-GUID 屬性

這是已建立來支援 DirSync 控制項的已建立屬性。 這個屬性會在複寫物件的建立、重新命名或移動時，保存物件父系的 objectGuid。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | Parent-GUID                                           |
| Ldap-顯示名稱 | parentGUID                                            |
| 大小              | 16 個位元組                                              |
| 更新許可權  | 此值是由系統所設定。                      |
| 更新頻率  | 複寫期間                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1224                               |
| 系統識別碼-Guid    | 2df90d74-009f-11d2-aa4c-00c04fd7d83a                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
