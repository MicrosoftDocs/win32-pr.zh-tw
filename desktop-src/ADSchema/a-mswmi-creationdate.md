# <a name="ms-wmi-creationdate-attribute"></a>CreationDate 屬性

建立物件實例的日期和時間戳記。



| 進入 | 值 |
|-------------------|---------------------------------------------------|
| CN                | CreationDate                               |
| Ldap-顯示名稱 | msWMI-CreationDate                                |
| 大小              | WMI datetime 資料類型表示-26 個字元。 |
| 更新許可權  | 此值是由系統所設定。                  |
| 更新頻率  | \-                                                |
| Attribute-Id      | 1.2.840.113556.1.4.1626                           |
| 系統識別碼-Guid    | 748b0a2e-3351-4b3f-b171-2f17414ea779              |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)       |
