# <a name="stringutc-time-syntax"></a>字串 (UTC 時間) 語法

由 ASN. 1 標準定義的時間字串格式。 如需詳細資訊，請參閱 ISO 8601 和 X680。 使用此語法，以 UTC-Time 格式儲存時間值。



| 進入 | 值 |
|--------------|----------------------------------------------------------------------------|
| 名稱         | String(UTC-Time)                                                           |
| 語法識別碼    | 2.5.5.11                                                                   |
| OM 識別碼        | 23                                                                         |
| MAPI 類型    | SYSTIME                                                                    |
| ADS 類型     | ADSTYPE \_ UTC \_ 時間                                                         |
| Variant 類型 | VT \_ 日期                                                                   |
| SDS 類型     | [System.DateTime](/dotnet/api/system.datetime) |
