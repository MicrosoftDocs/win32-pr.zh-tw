# <a name="msmq-site-name-attribute"></a>MSMQ-網站名稱屬性

Msmq **網站名稱** 屬性已由 [**Msmq-網站名稱（Ex**](a-msmqsitenameex.md) ）屬性取代。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | MSMQ-網站名稱                              |
| Ldap-顯示名稱 | mSMQSiteName                                |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.965                      |
| 系統識別碼-Guid    | ffadb4b2-de39-11d1-90a5-00c04fd91ab1        |
| Syntax            | [**String(Teletex)**](s-string-teletex.md) |
