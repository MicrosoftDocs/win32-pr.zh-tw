# <a name="driver-name-attribute"></a>Driver-Name 屬性

設備磁碟機名稱。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | Driver-Name                                 |
| Ldap-顯示名稱 | driverName                                  |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.229                      |
| 系統識別碼-Guid    | 281416c5-1968-11d0-a28f-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
