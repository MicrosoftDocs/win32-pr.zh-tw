| Ldap-顯示名稱 | printAttributes                      |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |

| 系統識別碼-Guid    | 281416d7-1968-11d0-a28f-00aa003049e2 |
