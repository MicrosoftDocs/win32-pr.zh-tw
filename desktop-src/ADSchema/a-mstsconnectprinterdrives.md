# <a name="ms-ts-connect-printer-drives-attribute"></a>ms-chap-連接-印表機-磁片磁碟機屬性

終端機服務會話連線印表機磁片磁碟機會指定是否要在登入時重新連線到對應的用戶端印表機。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ms-TS-Connect-Printer-Drives         |
| Ldap-顯示名稱 | msTSConnectPrinterDrives             |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1987              |
| 系統識別碼-Guid    | 8ce6a937-871b-4c92-b285-d99d4036681c |
| Syntax            | [**Boolean**](s-boolean.md)         |
