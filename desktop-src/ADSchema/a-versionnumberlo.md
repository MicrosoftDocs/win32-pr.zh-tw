# <a name="version-number-lo-attribute"></a>版本號碼-Lo 屬性

一般用途次要版本號碼。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 版本號碼-Lo                    |
| Ldap-顯示名稱 | versionNumberLo                      |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.329               |
| 系統識別碼-Guid    | 7d6c0e9b-7e20-11d0-afd6-00c04fd930c9 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
