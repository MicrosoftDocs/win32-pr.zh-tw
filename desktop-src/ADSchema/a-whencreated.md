# <a name="when-created-attribute"></a>When-Created 屬性

此物件的建立日期。 此值會複寫並位於通用類別目錄中。



| 進入 | 值 |
|-------------------|---------------------------------------------------------------|
| CN                | When-Created                                                  |
| Ldap-顯示名稱 | whenCreated                                                   |
| 大小              | 8 個位元組                                                       |
| 更新許可權  | 此值是由系統所設定。                              |
| 更新頻率  | 建立物件時。                                   |
| Attribute-Id      | 1.2.840.113556.1.2.2                                          |
| 系統識別碼-Guid    | bf967a78-0de6-11d0-a285-00aa003049e2                          |
| Syntax            | [**String(Generalized-Time)**](s-string-generalized-time.md) |
