# <a name="frs-service-command-attribute"></a>FRS-服務-命令屬性

系統管理員可以設定來將命令傳遞給每個複本集成員的 Unicode 字串。 未使用。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | FRS-服務-命令                         |
| Ldap-顯示名稱 | fRSServiceCommand                           |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.500                      |
| 系統識別碼-Guid    | ddac0cee-af8f-11d0-afeb-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
