# <a name="domain-policy-reference-attribute"></a>網域原則參考屬性

原則物件從中複製之網域原則物件的分辨名稱。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | 網域原則-參考                 |
| Ldap-顯示名稱 | domainPolicyReference                   |
| 大小              | \-                                      |
| 更新許可權  | \-                                      |
| 更新頻率  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.422                  |
| 系統識別碼-Guid    | 80a67e2a-9f22-11d0-afdd-00c04fd930c9    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
