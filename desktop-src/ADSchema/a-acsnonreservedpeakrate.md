# <a name="acs-non-reserved-peak-rate-attribute"></a>ACS-非保留的尖峰率屬性

**ACS-非保留的尖峰率** 屬性僅供內部使用。 以 RFC2814 為基礎。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ACS-非保留的尖峰率           |
| Ldap-顯示名稱 | aCSNonReservedPeakRate               |
| 大小              | 8 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1318              |
| 系統識別碼-Guid    | a331a73f-3b90-11d2-90cc-00c04fd91ab1 |
| Syntax            | [**間隔**](s-interval.md)       |
