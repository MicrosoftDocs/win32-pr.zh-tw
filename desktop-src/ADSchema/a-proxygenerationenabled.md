# <a name="proxy-generation-enabled-attribute"></a>Proxy 產生啟用的屬性

如果已啟用 proxy 產生，**則為 TRUE** 。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | Proxy 產生-已啟用             |
| Ldap-顯示名稱 | proxyGenerationEnabled               |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.523               |
| 系統識別碼-Guid    | 5fd424d6-1262-11d0-a060-00aa006c33ed |
| Syntax            | [**Boolean**](s-boolean.md)         |
