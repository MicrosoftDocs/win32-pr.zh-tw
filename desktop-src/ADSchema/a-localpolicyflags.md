| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 本機原則-旗標                   |
| Ldap-顯示名稱 | localPolicyFlags                     |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.56                |
| 系統識別碼-Guid    | bf96799e-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
