# <a name="acs-dsbm-refresh-attribute"></a>ACS-DSBM-Refresh 屬性

此屬性包含的間隔計時器值，可決定指定的子網路頻寬管理員 (DSBM) 送出重新整理訊息 (\_ 我 \_) 到網域中的所有子網路頻寬管理員。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ACS-DSBM-重新整理                     |
| Ldap-顯示名稱 | aCSDSBMRefresh                       |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.777               |
| 系統識別碼-Guid    | 1cb3559f-56d0-11d1-a9c6-0000f80367c1 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
