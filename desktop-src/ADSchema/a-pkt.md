| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | PKT                                                   |
| Ldap-顯示名稱 | pKT                                                   |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.206                                |
| 系統識別碼-Guid    | 8447f9f1-1027-11d0-a05f-00aa006c33ed                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
