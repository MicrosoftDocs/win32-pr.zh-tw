# <a name="trust-type-attribute"></a>Trust-Type 屬性

信任類型，例如 Windows NT 或 MIT。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | Trust-Type                           |
| Ldap-顯示名稱 | trustType                            |
| 大小              | 4 個位元組                              |
| 更新許可權  | 此值是由系統所設定。     |
| 更新頻率  | 當新的信任建立時。         |
| Attribute-Id      | 1.2.840.113556.1.4.136               |
| 系統識別碼-Guid    | bf967a60-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
