# <a name="home-drive-attribute"></a>Home-Drive 屬性

指定要對應 [**homeDirectory**](a-homedirectory.md)所指定之 UNC 路徑的磁碟機號。 磁碟機號必須以 *r * * *：** 格式指定，其中 *r* 是要對應之磁片磁碟機的字母。 *R* 必須是單一大寫字母和冒號 (： ) 是必要的。



| 進入 | 值 |
|-------------------|-----------------------------------------------------------------------------------|
| CN                | Home-Drive                                                                        |
| Ldap-顯示名稱 | homeDrive                                                                         |
| 大小              | 2 個位元組                                                                           |
| 更新許可權  | 網域系統管理員會設定此值。                                         |
| 更新頻率  | 當使用者的記錄建立時，以及每次主要磁碟磁碟機需要變更時。 |
| Attribute-Id      | 1.2.840.113556.1.4.45                                                             |
| 系統識別碼-Guid    | bf967986-0de6-11d0-a285-00aa003049e2                                              |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                       |
