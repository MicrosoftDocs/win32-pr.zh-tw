# <a name="acs-non-reserved-token-size-attribute"></a>ACS-非保留-Token 大小屬性

**ACS 非保留的 Token 大小** 屬性僅供內部使用。 以 RFC2814 為基礎。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ACS-非保留-權杖-大小          |
| Ldap-顯示名稱 | aCSNonReservedTokenSize              |
| 大小              | 8 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1319              |
| 系統識別碼-Guid    | a916d7c9-3b90-11d2-90cc-00c04fd91ab1 |
| Syntax            | [**間隔**](s-interval.md)       |
