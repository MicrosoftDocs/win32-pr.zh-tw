# <a name="ms-ts-home-directory-attribute"></a>ms-TS-主目錄屬性

「終端服務主目錄」指定使用者的主目錄。

|-------------------|---------------------------------------------|
| CN                | ms-TS-Home-Directory                        |
| Ldap-顯示名稱 | msTSHomeDirectory                           |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |

| 系統識別碼-Guid    | 5d3510f0-c4e7-4122-b91f-a20add90e246        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
