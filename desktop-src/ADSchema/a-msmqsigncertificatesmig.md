# <a name="msmq-sign-certificates-mig-attribute"></a>MSMQ-簽署-憑證-Svrmig.mig 屬性

在 MSMQ 混合模式中，屬性包含先前的 mSMQSignCertificates 值。 MSMQ 支援從 MSMQ 1.0 DS 遷移至 Windows 2000 DS，而混合模式則指定某些 DS 的部分未升級為 Windows 2000 的狀態。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | MSMQ-簽署-憑證-Svrmig.mig                            |
| Ldap-顯示名稱 | mSMQSignCertificatesMig                               |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.967                                |
| 系統識別碼-Guid    | 3881b8ea-da3b-11d1-90a5-00c04fd91ab1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
