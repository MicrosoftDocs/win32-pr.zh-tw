# <a name="ms-sql-lastbackupdate-attribute"></a>LastBackupDate 屬性

上次備份資料庫的日期。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | LastBackupDate                       |
| Ldap-顯示名稱 | LastBackupDate                       |
| 大小              | \-                                          |
| 更新許可權  | 此值是由系統所設定。            |
| 更新頻率  | 備份資料庫時。             |
| Attribute-Id      | 1.2.840.113556.1.4.1398                     |
| 系統識別碼-Guid    | f2b6abca-ccee-11d2-9993-0000f87a57d4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
