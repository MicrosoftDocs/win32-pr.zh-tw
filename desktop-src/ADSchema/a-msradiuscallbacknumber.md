# <a name="msradiuscallbacknumber-attribute"></a>msRADIUSCallbackNumber 屬性

**MsRADIUSCallbackNumber** 屬性會在內部使用。 請勿直接修改此值。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | msRADIUSCallbackNumber               |
| Ldap-顯示名稱 | msRADIUSCallbackNumber               |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1145              |
| 系統識別碼-Guid    | db0c909c-c1f2-11d1-bbc5-0080c76670c0 |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
