# <a name="show-in-address-book-attribute"></a>顯示通訊錄屬性

這個屬性是用來指出物件將出現在哪些 MAPI 通訊錄中。 它通常是由 Exchange 收件者更新服務維護。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | 顯示位址-書籍                    |
| Ldap-顯示名稱 | showInAddressBook                       |
| 大小              | \-                                      |
| 更新許可權  | 系統會使用此方法。             |
| 更新頻率  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.644                  |
| 系統識別碼-Guid    | 3e74f60e-3e73-11d1-a9c0-0000f80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
