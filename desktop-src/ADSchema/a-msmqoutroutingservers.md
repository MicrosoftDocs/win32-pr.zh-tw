# <a name="msmq-out-routing-servers-attribute"></a>MSMQ 輸出路由-伺服器屬性

MSMQ 路由伺服器的 DN 連結，此電腦的所有連出流量都應路由傳送。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | MSMQ 輸出路由-伺服器                |
| Ldap-顯示名稱 | mSMQOutRoutingServers                   |
| 大小              | \-                                      |
| 更新許可權  | \-                                      |
| 更新頻率  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.928                  |
| 系統識別碼-Guid    | 9a0dc32b-c100-11d1-bbc5-0080c76670c0    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
