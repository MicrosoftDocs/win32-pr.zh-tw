# <a name="ms-ts-managingls3-attribute"></a>ManagingLS3 屬性

| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | MS-TS-ManagingLS3                           |
| Ldap-顯示名稱 | msTSManagingLS3                             |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.2005                     |
| 系統識別碼-Guid    | fad5dcc1-2130-4c87-a118-75322cd67050        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
