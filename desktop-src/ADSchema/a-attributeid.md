| 進入 | 值 |
|-------------------|-----------------------------------------------------------------|
| CN                | 屬性識別碼                                                    |
| Ldap-顯示名稱 | attributeID                                                     |
| 大小              | \-                                                              |
| 更新許可權  | 架構系統管理員                                            |
| 更新頻率  | \-                                                              |
| Attribute-Id      | 1.2.840.113556.1.2.30                                           |
| 系統識別碼-Guid    | bf967922-0de6-11d0-a285-00aa003049e2                            |
| Syntax            | [**String(Object-Identifier)**](s-string-object-identifier.md) |
