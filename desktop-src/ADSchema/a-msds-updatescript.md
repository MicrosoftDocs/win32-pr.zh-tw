# <a name="ms-ds-updatescript-attribute"></a>UpdateScript 屬性

這是用來以網域重建指令來保存腳本。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | UpdateScript                          |
| Ldap-顯示名稱 | UpdateScript                           |
| 大小              | 大小上限為10K 個位元組。                     |
| 更新許可權  | 此值是由系統所設定。            |
| 更新頻率  | 只有在網域重建時。             |
| Attribute-Id      | 1.2.840.113556.1.4.1721                     |
| 系統識別碼-Guid    | 146eb639-bb9f-4fc1-a825-e29e00c77920        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |



## <a name="implementations"></a>實作

-   [**Windows Server 2008**](#windows-server-2008)
