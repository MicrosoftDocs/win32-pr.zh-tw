# <a name="mastered-by-attribute"></a>Mastered-By 屬性

Nc 屬性的後置連結。 其 NTDS 設定物件的分辨名稱。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | Mastered-By                             |
| Ldap-顯示名稱 | masteredBy                              |
| 大小              | \-                                      |
| 更新許可權  | 此值是由系統所設定。        |
| 更新頻率  | 當命名內容建立時。     |
| Attribute-Id      | 1.2.840.113556.1.4.1409                 |
| 系統識別碼-Guid    | e48e64e0-12c9-11d3-9102-00c04fd91ab1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
