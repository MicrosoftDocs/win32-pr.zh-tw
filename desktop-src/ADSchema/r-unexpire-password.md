# <a name="unexpire-password-extended-right"></a>Unexpire-Password 擴充許可權

可讓使用者還原使用者物件過期密碼的 [擴充控制] 存取權限。



| 進入 | 值 |
|--------------|--------------------------------------|
| CN           | 未過期密碼                    |
| 顯示名稱 | Unexpire 密碼                    |
| 許可權-GUID  | ccc2dc7d-a6ad-4a7a-8846-c04e3cc53501 |
