# <a name="rights-guid-attribute"></a>Rights-Guid 屬性

用來代表存取控制專案內擴充許可權的 GUID。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | Rights-Guid                                 |
| Ldap-顯示名稱 | rightsGuid                                  |
| 大小              | 16 個位元組                                    |
| 更新許可權  | 此值是由系統所設定。            |
| 更新頻率  | 建立新的擴充許可權時。       |
| Attribute-Id      | 1.2.840.113556.1.4.340                      |
| 系統識別碼-Guid    | 8297931c-86d3-11d0-afda-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
