# <a name="update-password-not-required-bit-extended-right"></a>更新-密碼-不需要位延伸許可權

[擴充控制] 存取權限，可讓使用者啟用或停用使用者物件的 [不需要密碼] 設定。



| 進入 | 值 |
|--------------|--------------------------------------|
| CN           | 更新-密碼-不需要位     |
| 顯示名稱 | 更新密碼不需要位     |
| 許可權-GUID  | 280f369c-67c7-438e-ae98-1d46f3c6f541 |
