# <a name="site-object-bl-attribute"></a>網站物件 BL 屬性

屬於此網站之子網的辨別名稱清單。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | 網站-物件-BL                          |
| Ldap-顯示名稱 | siteObjectBL                            |
| 大小              | \-                                      |
| 更新許可權  | 此值是由系統所設定。        |
| 更新頻率  | 每次建立或刪除網站時。  |
| Attribute-Id      | 1.2.840.113556.1.4.513                  |
| 系統識別碼-Guid    | 3e10944d-c354-11d0-aff8-0000f80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
