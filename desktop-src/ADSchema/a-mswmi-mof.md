# <a name="ms-wmi-mof-attribute"></a>ms-WMI-Mof 屬性

保存某些 WMI 物件的 MOF 定義。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | ms-WMI-Mof                                  |
| Ldap-顯示名稱 | msWMI-Mof                                   |
| 大小              | \-                                          |
| 更新許可權  | 群組原則系統管理員                  |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1638                     |
| 系統識別碼-Guid    | 6736809f-2064-443e-a145-81262b1f1366        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
