# <a name="ms-rras-attribute-attribute"></a>ms RRAS-Attribute 屬性

包含安裝在路由器上之 vendorID：號碼的字串。 VendorID 是廠商 SMI-S，而數位是在路由器身分識別字典物件中定義的 ms RRAS 屬性編號。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | ms RRAS-屬性                           |
| Ldap-顯示名稱 | msRRASAttribute                             |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.884                      |
| 系統識別碼-Guid    | f39b98ad-938d-11d1-aebd-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
