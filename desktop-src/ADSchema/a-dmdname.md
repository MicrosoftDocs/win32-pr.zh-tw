# <a name="dmd-name-attribute"></a>DMD-Name 屬性

| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | DMD-Name                                    |
| Ldap-顯示名稱 | dmdName                                     |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.598                      |
| 系統識別碼-Guid    | 167757b9-47f3-11d1-a9c3-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
