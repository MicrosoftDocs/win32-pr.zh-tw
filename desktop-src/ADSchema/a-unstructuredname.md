# <a name="unstructuredname-attribute"></a>unstructuredName 屬性

路由器的 DNS 名稱 (例如： router1.fabrikam.com) 。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | unstructuredName                     |
| Ldap-顯示名稱 | unstructuredName                     |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113549.1.9.2                 |
| 系統識別碼-Guid    | 9c8ef177-41cf-45c9-9673-7716c0c8901b |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
