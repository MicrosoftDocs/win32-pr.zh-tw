# <a name="print-pages-per-minute-attribute"></a>列印-每分鐘頁面屬性

驅動程式提供的列印速率（以每分鐘頁面數為單位）。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 列印-每分鐘頁面               |
| Ldap-顯示名稱 | printPagesPerMinute                  |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.631               |
| 系統識別碼-Guid    | 19405b97-3cfa-11d1-a9c0-0000f80367c1 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
