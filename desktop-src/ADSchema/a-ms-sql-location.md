# <a name="ms-sql-location-attribute"></a>MS SQL-Location 屬性

使用者定義的字串。 預設值是設定為 [位置]。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | 毫秒-SQL-位置                             |
| Ldap-顯示名稱 | 毫秒-SQL-位置                             |
| 大小              | \-                                          |
| 更新許可權  | 網域系統管理員                        |
| 更新頻率  | 在系統設定時。                            |
| Attribute-Id      | 1.2.840.113556.1.4.1366                     |
| 系統識別碼-Guid    | 561c9644-ccee-11d2-9993-0000f87a57d4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
