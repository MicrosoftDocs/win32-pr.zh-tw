# <a name="template-roots2-attribute"></a>Template-Roots2 屬性

這個屬性在 Exchange config 容器中用來指出範本容器的儲存位置。 Active Directory MAPI 提供者會使用這項資訊



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | Template-Roots2                         |
| Ldap-顯示名稱 | templateRoots2                          |
| 大小              | \-                                      |
| 更新許可權  | \-                                      |
| 更新頻率  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.2048                 |
| 系統識別碼-Guid    | b1cba91a-0682-4362-a659-153e201ef069    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
