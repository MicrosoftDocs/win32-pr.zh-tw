# <a name="ms-ieee-80211-id-attribute"></a>ms-ieee-80211-ID 屬性

適用于 AD 上無線原則物件的識別碼。



| 進入 | 值 |
|-------------------|----------------------------------------------------------------------------------|
| CN                | ms-chap-ieee-80211-識別碼                                                                 |
| Ldap-顯示名稱 | msieee80211-ID                                                                   |
| 大小              | \-                                                                               |
| 更新許可權  | 網域系統管理員                                                             |
| 更新頻率  | 每次網域管理員變更網域或 OU 的無線網路原則時。 |
| Attribute-Id      | 1.2.840.113556.1.4.1823                                                          |
| 系統識別碼-Guid    | 7f73ef75-14c9-4c23-81de-dd07a06f9e8b                                             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                      |
