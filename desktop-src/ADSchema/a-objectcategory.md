# <a name="object-category-attribute"></a>Object-Category 屬性

物件類別名稱，用來將這個或衍生類別的物件分組。



| 進入 | 值 |
|-------------------|--------------------------------------------------|
| CN                | Object-Category                                  |
| Ldap-顯示名稱 | objectCategory                                   |
| 大小              | 平均大約20個位元組。                       |
| 更新許可權  | 物件的設計工具會設定此值。 |
| 更新頻率  | 此值永遠不會變更。                  |
| Attribute-Id      | 1.2.840.113556.1.4.782                           |
| 系統識別碼-Guid    | 26d97369-6070-11d1-a9c6-0000f80367c1             |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md)          |
