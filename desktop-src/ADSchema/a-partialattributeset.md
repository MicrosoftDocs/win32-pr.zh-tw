# <a name="partial-attribute-set-attribute"></a>部分屬性集屬性

追蹤部分複本的內部複寫狀態 (也就是在 Gc) 上。 部分複本 NC 物件的屬性。 定義特定部分複本 NC 上存在的屬性集。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | 部分屬性集                                 |
| Ldap-顯示名稱 | partialAttributeSet                                   |
| 大小              | \-                                                    |
| 更新許可權  | 此值是由系統所設定。                      |
| 更新頻率  | 複寫期間                                    |
| Attribute-Id      | 1.2.840.113556.1.4.640                                |
| 系統識別碼-Guid    | 19405b9e-3cfa-11d1-a9c0-0000f80367c1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
