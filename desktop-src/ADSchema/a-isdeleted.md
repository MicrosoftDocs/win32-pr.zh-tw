# <a name="is-deleted-attribute"></a>Is-Deleted 屬性

若 **為 TRUE**，表示此物件已標示為要刪除，無法具現化。 在標記期間過期之後，系統就會將它從系統中移除。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | Is-Deleted                           |
| Ldap-顯示名稱 | isDeleted                            |
| 大小              | \-                                   |
| 更新許可權  | 網域系統管理員                 |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.48                |
| 系統識別碼-Guid    | bf96798f-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Boolean**](s-boolean.md)         |
