# <a name="ms-ds-az-class-id-attribute"></a>ms-chap-Az-Class ID 屬性

AzApplication 物件上 AzRoles UI 所需的類別識別碼。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | ms-chap-Az-Class-ID                           |
| Ldap-顯示名稱 | AzClassId                              |
| 大小              | 36個字元                               |
| 更新許可權  | AzRoles 系統管理員                               |
| 更新頻率  | 在初始化或原則變更期間。     |
| Attribute-Id      | 1.2.840.113556.1.4.1816                     |
| 系統識別碼-Guid    | 013a7277-5c2d-49ef-a7de-b765b36a3f6f        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
