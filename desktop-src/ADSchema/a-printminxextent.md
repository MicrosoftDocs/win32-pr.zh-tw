| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 列印-最小-X-範圍                   |
| Ldap-顯示名稱 | printMinXExtent                      |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.279               |
| 系統識別碼-Guid    | ba305f71-47e3-11d0-a1a6-00c04fd930c9 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
