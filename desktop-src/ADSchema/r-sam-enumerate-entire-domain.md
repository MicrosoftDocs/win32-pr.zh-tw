# <a name="sam-enumerate-entire-domain-extended-right"></a>SAM-列舉-整個定義域擴充許可權

您可以使用這個 control 存取權限來限制誰可以使用舊版 API，例如 NetQueryDisplayInformation 和 NetUser/GroupEnum，以及列舉整個網域。



| 進入 | 值 |
|--------------|--------------------------------------|
| CN           | SAM-列舉-整個網域          |
| 顯示名稱 | 列舉整個 SAM 網域          |
| 許可權-GUID  | 91d67418-0135-4acc-8d79-c08e857cfbec |
