# <a name="print-rate-attribute"></a>Print-Rate 屬性

驅動程式提供的列印速率。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | Print-Rate                           |
| Ldap-顯示名稱 | printRate                            |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.285               |
| 系統識別碼-Guid    | ba305f77-47e3-11d0-a1a6-00c04fd930c9 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
