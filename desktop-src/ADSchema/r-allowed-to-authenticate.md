# <a name="allowed-to-authenticate-extended-right"></a>允許驗證擴充許可權

Control 存取權限可控制誰可以驗證特定的電腦或服務。 基本上是在電腦、使用者和 InetOrgPerson 物件上。 如果整個網域允許存取，它也適用于網域物件。 它可以套用至 Ou，讓使用者能夠在包含一組使用者或電腦物件的 Ou 上設定可繼承 Ace。



| 進入 | 值 |
|--------------|--------------------------------------|
| CN           | 允許驗證              |
| 顯示名稱 | 允許驗證              |
| 許可權-GUID  | 68b1d179-0d15-4d4f-ab71-46152e79a7bc |
