# <a name="ms-ds-members-for-az-role-attribute"></a>ms DS-成員--Az-Role 屬性

連結至 Az 角色的成員應用程式群組或使用者清單。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | ms DS-成員--Az-Role               |
| Ldap-顯示名稱 | MembersForAzRole                   |
| 大小              | \-                                      |
| 更新許可權  | AzRoles 系統管理員                           |
| 更新頻率  | 在初始化或原則變更期間。 |
| Attribute-Id      | 1.2.840.113556.1.4.1806                 |
| 系統識別碼-Guid    | cbf7e6cd-85a4-4314-8939-8bfe80597835    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
