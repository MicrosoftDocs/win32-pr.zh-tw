# <a name="frs-root-security-attribute"></a>FRS-根目錄安全性屬性

檔案複寫的複本集根目錄的安全描述項。



| 進入 | 值 |
|-------------------|-----------------------------------------------------|
| CN                | FRS-根目錄-安全性                                   |
| Ldap-顯示名稱 | fRSRootSecurity                                     |
| 大小              | \-                                                  |
| 更新許可權  | \-                                                  |
| 更新頻率  | \-                                                  |
| Attribute-Id      | 1.2.840.113556.1.4.535                              |
| 系統識別碼-Guid    | 5245801f-ca6a-11d0-afff-0000f80367c1                |
| Syntax            | [**String(NT-Sec-Desc)**](s-string-nt-sec-desc.md) |
