# <a name="logon-hours-attribute"></a>Logon-Hours 屬性

允許使用者登入網域的時數。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | Logon-Hours                                           |
| Ldap-顯示名稱 | logonHours                                            |
| 大小              | \-                                                    |
| 更新許可權  | 網域系統管理員                                  |
| 更新頻率  | 只要帳戶的登入時數需要變更。   |
| Attribute-Id      | 1.2.840.113556.1.4.64                                 |
| 系統識別碼-Guid    | bf9679ab-0de6-11d0-a285-00aa003049e2                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
