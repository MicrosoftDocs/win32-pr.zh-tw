這個屬性是用來支援文字格式的400個位址。

|-------------------|---------------------------------------------|
| CN                | 文字編碼或位址                     |
| Ldap-顯示名稱 | textEncodedORAddress                        |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.2                   |
| 系統識別碼-Guid    | a8df7489-c5ea-11d1-bbcb-0080c76670c0        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
