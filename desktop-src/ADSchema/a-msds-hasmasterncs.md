# <a name="ms-ds-has-master-ncs-attribute"></a>Nc 屬性（ms DS）

包含網域控制站所持有的命名內容。 這個屬性淘汰具有-Nc。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | ms-DS-Nc                    |
| Ldap-顯示名稱 | hasMasterNCs                       |
| 大小              | \-                                      |
| 更新許可權  | 此值是由系統所設定         |
| 更新頻率  | 每次建立新 NC 時。          |
| Attribute-Id      | 1.2.840.113556.1.4.1836                 |
| 系統識別碼-Guid    | ae2de0e2-59d7-4d47-8d47-ed4dfe4357ad    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
