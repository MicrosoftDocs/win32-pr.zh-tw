|-------------------|-----------------------------------------|
| CN                | 預設類別-存放區                     |
| Ldap-顯示名稱 | defaultClassStore                       |
| 大小              | \-                                      |
| 更新許可權  | \-                                      |
| 更新頻率  | \-                                      |

| 系統識別碼-Guid    | bf967948-0de6-11d0-a285-00aa003049e2    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
