# <a name="repl-topology-stay-of-execution-attribute"></a>複寫-拓撲-持續執行屬性

刪除伺服器物件與從複寫拓撲中永久移除之間的延遲。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 複寫-拓撲-持續執行      |
| Ldap-顯示名稱 | replTopologyStayOfExecution          |
| 大小              | 4 個位元組                              |
| 更新許可權  | 此值是由系統所設定。     |
| 更新頻率  | 每次刪除伺服器物件時。 |
| Attribute-Id      | 1.2.840.113556.1.4.677               |
| 系統識別碼-Guid    | 7bfdcb83-4807-11d1-a9c3-0000f80367c1 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
