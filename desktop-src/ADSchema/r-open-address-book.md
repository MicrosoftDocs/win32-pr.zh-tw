# <a name="open-address-book-extended-right"></a>開啟-Address Book 擴充許可權

開啟通訊錄視圖的通訊錄物件時，會檢查延伸的許可權。



| 進入 | 值 |
|--------------|--------------------------------------|
| CN           | 開啟-通訊錄                    |
| 顯示名稱 | 開啟通訊清單                    |
| 許可權-GUID  | a1990816-4298-11d1-ade2-00c04fd8d5cd |
