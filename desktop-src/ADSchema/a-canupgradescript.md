# <a name="can-upgrade-script-attribute"></a>可升級-腳本屬性

這個屬性會儲存可以升級的應用程式封裝清單，或可升級此應用程式封裝的應用程式封裝。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | 可升級-腳本                          |
| Ldap-顯示名稱 | canUpgradeScript                            |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.815                      |
| 系統識別碼-Guid    | d9e18314-8939-11d1-aebc-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
