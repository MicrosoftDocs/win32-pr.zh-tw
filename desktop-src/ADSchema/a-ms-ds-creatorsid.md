# <a name="ms-ds-creator-sid-attribute"></a>MS DS-Creator-SID 屬性

包含此屬性之物件的建立者的安全識別碼。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | MS-DS-Creator-SID                    |
| Ldap-顯示名稱 | CreatorSID                     |
| 大小              | \-                                   |
| 更新許可權  | 此值是由系統所設定。     |
| 更新頻率  | 每次建立新的物件時。    |
| Attribute-Id      | 1.2.840.113556.1.4.1410              |
| 系統識別碼-Guid    | c5e60132-1480-11d3-91c1-0000f87a57d4 |
| Syntax            | [**(Sid) 的字串**](s-string-sid.md)  |
