# <a name="photo-attribute"></a>相片屬性

在 G3 fax 中編碼的物件（如建議的建議）中所述，使用 asn.1 包裝函式，使其與在 < 420 中定義的400個正文相符。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | 相片                                                 |
| Ldap-顯示名稱 | 相片                                                 |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 0.9.2342.19200300.100.1.7                             |
| 系統識別碼-Guid    | 9c979768-ba1a-4c08-9632-c6a5c1ed649a                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
