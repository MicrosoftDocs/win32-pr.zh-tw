# <a name="last-update-sequence-attribute"></a>上次更新順序屬性

這個屬性包含已變更之類別存放區中最後一個專案的更新序號。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | 上次更新順序                        |
| Ldap-顯示名稱 | lastUpdateSequence                          |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.330                      |
| 系統識別碼-Guid    | 7d6c0e9c-7e20-11d0-afd6-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
