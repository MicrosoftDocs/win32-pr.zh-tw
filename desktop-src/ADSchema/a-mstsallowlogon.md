# <a name="ms-ts-allow-logon-attribute"></a>ms-TS-允許-登入屬性

指定是否允許使用者登入終端機伺服器。 若允許登入，則此值為 1；若不允許，則此值為 0。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ms-TS-Allow-Logon                    |
| Ldap-顯示名稱 | msTSAllowLogon                       |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1979              |
| 系統識別碼-Guid    | 3a0cd464-bc54-40e7-93ae-a646a6ecc4b4 |
| Syntax            | [**Boolean**](s-boolean.md)         |
