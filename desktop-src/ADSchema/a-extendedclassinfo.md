# <a name="extended-class-info-attribute"></a>擴充類別資訊屬性

多值屬性，其中包含代表每個類別之其他資訊的字串。 每個值都包含類別的 governsID、lDAPDisplayName 和 schemaIDGUID。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | 擴充類別資訊                         |
| Ldap-顯示名稱 | extendedClassInfo                           |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.908                      |
| 系統識別碼-Guid    | 9a7ad948-ca53-11d1-bbd0-0080c76670c0        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
