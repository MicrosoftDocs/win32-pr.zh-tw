此屬性代表 TSV 部署中電腦的 VM 名稱。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | ms-TS-端點-資料                         |
| Ldap-顯示名稱 | msTSEndpointData                            |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.2070                     |
| 系統識別碼-Guid    | 40e1c407-4344-40f3-ab43-3625a34a63a2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
