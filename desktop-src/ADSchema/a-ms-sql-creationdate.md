# <a name="ms-sql-creationdate-attribute"></a>CreationDate 屬性

資料庫的建立日期。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | CreationDate                         |
| Ldap-顯示名稱 | CreationDate                         |
| 大小              | \-                                          |
| 更新許可權  | 此值是由系統所設定。            |
| 更新頻率  | 建立新的資料庫時。             |
| Attribute-Id      | 1.2.840.113556.1.4.1397                     |
| 系統識別碼-Guid    | ede14754-ccee-11d2-9993-0000f87a57d4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
