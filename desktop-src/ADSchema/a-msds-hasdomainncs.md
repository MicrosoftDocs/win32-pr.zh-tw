# <a name="ms-ds-has-domain-ncs-attribute"></a>Nc 屬性（ms DS）

詳細說明特定伺服器上存在之網域命名內容的目錄服務複寫資訊。



| 進入 | 值 |
|-------------------|-----------------------------------------------------|
| CN                | ms-DS-Nc                                |
| Ldap-顯示名稱 | HasDomainNCs                                   |
| 大小              | \-                                                  |
| 更新許可權  | 此值是由系統所設定                     |
| 更新頻率  | 它是由 DCPromo 設定，之後也未曾修改過。 |
| Attribute-Id      | 1.2.840.113556.1.4.1820                             |
| 系統識別碼-Guid    | 6f17e347-a842-4498-b8b3-15e007da4fed                |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md)             |



## <a name="implementations"></a>實作

-   [**Windows Server 2003**](#windows-server-2003)
-   [**亞當**](#adam)
-   [**Windows Server 2003 R2**](#windows-server-2003-r2)
-   [**Windows Server 2008**](#windows-server-2008)
