| CN                | MS-TS-LicenseVersion3                       |
| Ldap-顯示名稱 | msTSLicenseVersion3                         |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |

| 系統識別碼-Guid    | f8ba8f81-4cab-4973-a3c8-3a6da62a5e31        |
