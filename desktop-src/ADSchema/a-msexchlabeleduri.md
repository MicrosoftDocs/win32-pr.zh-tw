# <a name="ms-exch-labeleduri-attribute"></a>Ms-exch-assistant-name-LabeledURI 屬性

後接標籤的交換 URI。 標籤可用來描述 URI 所指向的資源，並將其視為人們可讀取的名稱。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | Ms-exch-assistant-name-LabeledURI                          |
| Ldap-顯示名稱 | msExchLabeledURI                            |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.593                      |
| 系統識別碼-Guid    | 16775820-47f3-11d1-a9c3-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
