# <a name="creation-wizard-attribute"></a>Creation-Wizard 屬性

建立這個類別的物件時，要啟動的 Wizard。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | Creation-Wizard                             |
| Ldap-顯示名稱 | creationWizard                              |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.498                      |
| 系統識別碼-Guid    | 4d8601ed-ac85-11d0-afe3-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
