# <a name="per-msg-dialog-display-table-attribute"></a>每個訊息-對話方塊-顯示資料表屬性

| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | 每個訊息-對話方塊-顯示資料表                          |
| Ldap-顯示名稱 | perMsgDialogDisplayTable                              |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.2.325                                |
| 系統識別碼-Guid    | 5fd424d3-1262-11d0-a060-00aa006c33ed                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
