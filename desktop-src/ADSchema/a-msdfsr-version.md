# <a name="ms-dfsr-version-attribute"></a>ms-DFSR-Version 屬性

包含分散式檔案系統 (DFS) Replication service 的版本號碼。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | ms-DFSR-版本                             |
| Ldap-顯示名稱 | msDFSR-版本                              |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.1                   |
| 系統識別碼-Guid    | 1a861408-38c3-49ea-ba75-85481a77c655        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
