# <a name="ipnetmasknumber-attribute"></a>ipNetmaskNumber 屬性

以點分隔的十進位標記法包含 IP 網路遮罩，省略前置零。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ipNetmaskNumber                      |
| Ldap-顯示名稱 | ipNetmaskNumber                      |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.21                     |
| 系統識別碼-Guid    | 6ff64fcd-462e-4f62-b44a-9a5347659eb9 |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
