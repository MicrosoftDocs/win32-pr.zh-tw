# <a name="ipserviceport-attribute"></a>ipServicePort 屬性

包含服務對應的區段，可指定 UNIX 服務的可用埠。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ipServicePort                        |
| Ldap-顯示名稱 | ipServicePort                        |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.15                     |
| 系統識別碼-Guid    | ff2daebf-f463-495a-8405-3e483641eaa2 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
