# <a name="netboot-guid-attribute"></a>Netboot-GUID 屬性

無磁片開機：電腦的機載 GUID。 對應于電腦的網路介面卡 MAC 位址。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | Netboot-GUID                                          |
| Ldap-顯示名稱 | netbootGUID                                           |
| 大小              | \-                                                    |
| 更新許可權  | 此值是由系統所設定。                      |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.359                                |
| 系統識別碼-Guid    | 3e978921-8c01-11d0-afda-00c04fd930c9                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
