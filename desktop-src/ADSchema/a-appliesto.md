# <a name="applies-to-attribute"></a>Applies-To 屬性

包含擴充許可權適用的物件類別清單。 在清單中，物件類別是由其 schemaClass 物件的 schemaIDGUID 屬性所代表。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | Applies-To                                  |
| Ldap-顯示名稱 | appliesTo                                   |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.341                      |
| 系統識別碼-Guid    | 8297931d-86d3-11d0-afda-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
