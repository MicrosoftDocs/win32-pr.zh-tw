# <a name="domain-identifier-attribute"></a>Domain-Identifier 屬性

|-------------------|--------------------------------------|
| CN                | Domain-Identifier                    |
| Ldap-顯示名稱 | domainIdentifier                     |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.755               |
| 系統識別碼-Guid    | 7f561278-5301-11d1-a9c5-0000f80367c1 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
