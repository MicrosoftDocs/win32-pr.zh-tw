# <a name="ms-ts-remote-control-attribute"></a>ms-TS-遠端控制屬性

終端機服務遠端控制可指定是否允許遠端觀察或遠端控制使用者的終端機服務會話。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ms-TS-Remote-Control                 |
| Ldap-顯示名稱 | msTSRemoteControl                    |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1980              |
| 系統識別碼-Guid    | 15177226-8642-468b-8c48-03ddfd004982 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
