# <a name="ms-ds-keyversionnumber-attribute"></a>KeyVersionNumber 屬性

此帳戶目前金鑰的 Kerberos 版本號碼。 這是一個結構化屬性。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | KeyVersionNumber               |
| Ldap-顯示名稱 | KeyVersionNumber                |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1782              |
| 系統識別碼-Guid    | c523e9c0-33b5-4ac8-8923-b57b927f42f6 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
