# <a name="last-content-indexed-attribute"></a>上次內容索引屬性

此磁片區上次編制內容索引的時間。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 上次內容-內容索引                 |
| Ldap-顯示名稱 | lastContentIndexed                   |
| 大小              | 8 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.50                |
| 系統識別碼-Guid    | bf967995-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**間隔**](s-interval.md)       |
