# <a name="print-number-up-attribute"></a>列印編號屬性

每張工作表的頁面影像數目。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 列印-編號                      |
| Ldap-顯示名稱 | printNumberUp                        |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.290               |
| 系統識別碼-Guid    | 3bcbfcf4-4d3d-11d0-a1a6-00c04fd930c9 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
