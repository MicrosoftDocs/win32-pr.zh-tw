# <a name="ms-sql-language-attribute"></a>MS SQL 語言屬性

SQL Server 目前實例的語言。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | MS SQL 語言                             |
| Ldap-顯示名稱 | mS SQL 語言                             |
| 大小              | \-                                          |
| 更新許可權  | 網域系統管理員                        |
| 更新頻率  | 在系統設定時。                            |
| Attribute-Id      | 1.2.840.113556.1.4.1389                     |
| 系統識別碼-Guid    | c57f72f4-ccee-11d2-9993-0000f87a57d4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
