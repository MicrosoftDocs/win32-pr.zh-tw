# <a name="ipprotocolnumber-attribute"></a>ipProtocolNumber 屬性

包含通訊協定對應的區段，此區段會儲存識別通訊協定的唯一編號。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ipProtocolNumber                     |
| Ldap-顯示名稱 | ipProtocolNumber                     |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.17                     |
| 系統識別碼-Guid    | ebf5c6eb-0e2d-4415-9670-1081993b4211 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
