# <a name="acs-policy-name-attribute"></a>ACS-原則名稱屬性

| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | ACS-原則-名稱                             |
| Ldap-顯示名稱 | aCSPolicyName                               |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.772                      |
| 系統識別碼-Guid    | 1cb3559a-56d0-11d1-a9c6-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
