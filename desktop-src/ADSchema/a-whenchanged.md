# <a name="when-changed-attribute"></a>When-Changed 屬性

上次變更此物件的日期。 此值不會複寫，而且存在於通用類別目錄中。



| 進入 | 值 |
|-------------------|---------------------------------------------------------------|
| CN                | When-Changed                                                  |
| Ldap-顯示名稱 | whenChanged                                                   |
| 大小              | 8 個位元組                                                       |
| 更新許可權  | 此值是由系統所設定。                              |
| 更新頻率  | 每次物件變更時。                              |
| Attribute-Id      | 1.2.840.113556.1.2.3                                          |
| 系統識別碼-Guid    | bf967a77-0de6-11d0-a285-00aa003049e2                          |
| Syntax            | [**String(Generalized-Time)**](s-string-generalized-time.md) |
