# <a name="rid-next-rid-attribute"></a>RID-下一個 RID 屬性

目前集區中的下一個可用相對識別碼。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | RID-下一個 RID                         |
| Ldap-顯示名稱 | rIDNextRID                           |
| 大小              | 4 個位元組                              |
| 更新許可權  | 此值是由系統所設定。     |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.374               |
| 系統識別碼-Guid    | 6617188c-8f3c-11d0-afda-00c04fd930c9 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
