# <a name="site-server-attribute"></a>Site-Server 屬性

授權主伺服器以取得指定的網站。



| 進入 | 值 |
|-------------------|----------------------------------------------|
| CN                | Site-Server                                  |
| Ldap-顯示名稱 | siteServer                                   |
| 大小              | \-                                           |
| 更新許可權  | 網域系統管理員                         |
| 更新頻率  | 每次需要變更授權網站。 |
| Attribute-Id      | 1.2.840.113556.1.4.494                       |
| 系統識別碼-Guid    | 1be8f17c-a9ff-11d0-afe2-00c04fd930c9         |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md)      |
