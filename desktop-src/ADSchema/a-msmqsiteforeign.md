| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | MSMQ-網站-外部                    |
| Ldap-顯示名稱 | mSMQSiteForeign                      |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.961               |
| 系統識別碼-Guid    | fd129d8a-d57e-11d1-90a2-00c04fd91ab1 |
| Syntax            | [**Boolean**](s-boolean.md)         |
