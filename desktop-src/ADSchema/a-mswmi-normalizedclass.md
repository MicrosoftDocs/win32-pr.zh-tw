# <a name="ms-wmi-normalizedclass-attribute"></a>NormalizedClass 屬性

核心 WMI 原則類別的名稱。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | NormalizedClass                      |
| Ldap-顯示名稱 | msWMI-NormalizedClass                       |
| 大小              | 少於二十個字元。                |
| 更新許可權  | 群組原則系統管理員                  |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1640                     |
| 系統識別碼-Guid    | eaba628f-eb8e-4fe9-83fc-693be695559b        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
