# <a name="given-name-attribute"></a>Given-Name 屬性

包含使用者名字 (名字) 。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | 指定的名稱                                  |
| Ldap-顯示名稱 | givenName                                   |
| 大小              | \-                                          |
| 更新許可權  | 網域系統管理員或帳戶擁有者。      |
| 更新頻率  | 當使用者的記錄建立時。          |
| Attribute-Id      | 2.5.4.42                                    |
| 系統識別碼-Guid    | f0f8ff8e-1191-11d0-a060-00aa006c33ed        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
