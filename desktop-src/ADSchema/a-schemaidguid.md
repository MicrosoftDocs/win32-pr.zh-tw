# <a name="schema-id-guid-attribute"></a>架構識別碼-GUID 屬性

屬性的唯一識別碼。



| 進入 | 值 |
|-------------------|---------------------------------------------------------------------|
| CN                | 架構識別碼-GUID                                                      |
| Ldap-顯示名稱 | schemaIDGUID                                                        |
| 大小              | 16 個位元組                                                            |
| 更新許可權  | 此值是由系統所設定。                                    |
| 更新頻率  | 此值是在建立物件時設定的，而且無法變更。 |
| Attribute-Id      | 1.2.840.113556.1.4.148                                              |
| 系統識別碼-Guid    | bf967923-0de6-11d0-a285-00aa003049e2                                |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md)               |
