# <a name="print-stapling-supported-attribute"></a>列印裝訂-支援的屬性

如果印表機支援裝訂，**則為 TRUE** 。 由驅動程式提供。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 列印-裝訂-支援             |
| Ldap-顯示名稱 | printStaplingSupported               |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.281               |
| 系統識別碼-Guid    | ba305f73-47e3-11d0-a1a6-00c04fd930c9 |
| Syntax            | [**Boolean**](s-boolean.md)         |
