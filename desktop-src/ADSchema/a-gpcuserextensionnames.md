# <a name="gpc-user-extension-names-attribute"></a>GPC-使用者-Extension-Names 屬性

用於使用者原則的群組原則物件。



| 進入 | 值 |
|-------------------|---------------------------------------------------------------------------------|
| CN                | GPC-使用者延伸-名稱                                                        |
| Ldap-顯示名稱 | gPCUserExtensionNames                                                           |
| 大小              | 取決於在此 GPO 中具有原則的用戶端延伸模組數目。 |
| 更新許可權  | 網域或原則管理員。                                                 |
| 更新頻率  | 每次透過 GPE 更新 GPO 時。                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1349                                                         |
| 系統識別碼-Guid    | 42a75fc6-783f-11d2-9916-0000f87a57d4                                            |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                     |
