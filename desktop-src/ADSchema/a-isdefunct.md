# <a name="is-defunct-attribute"></a>Is-Defunct 屬性

若 **為 TRUE**，則表示類別或屬性無法再使用。 此物件的舊版可能存在，但無法建立新的版本。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | Is-Defunct                           |
| Ldap-顯示名稱 | isDefunct                            |
| 大小              | \-                                   |
| 更新許可權  | 架構系統管理員                 |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.661               |
| 系統識別碼-Guid    | 28630ebe-41d5-11d1-a9c1-0000f80367c1 |
| Syntax            | [**Boolean**](s-boolean.md)         |
