# <a name="usn-changed-attribute"></a>USN-Changed 屬性

更新序號 (USN) 由本機目錄指派以進行最新變更，包括建立。 另請參閱 [**USN 建立**](a-usncreated.md)。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | USN-Changed                          |
| Ldap-顯示名稱 | uSNChanged                           |
| 大小              | 8 個位元組                              |
| 更新許可權  | 此值是由系統所設定。     |
| 更新頻率  | 當物件變更時。          |
| Attribute-Id      | 1.2.840.113556.1.2.120               |
| 系統識別碼-Guid    | bf967a6f-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**間隔**](s-interval.md)       |
