# <a name="acs-non-reserved-tx-limit-attribute"></a>ACS-非保留的 Tx-Limit 屬性

使用者應用程式可以在保留備妥之前傳輸的最大頻寬。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ACS-非保留的 Tx-限制            |
| Ldap-顯示名稱 | aCSNonReservedTxLimit                |
| 大小              | 8 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.780               |
| 系統識別碼-Guid    | 1cb355a2-56d0-11d1-a9c6-0000f80367c1 |
| Syntax            | [**間隔**](s-interval.md)       |
