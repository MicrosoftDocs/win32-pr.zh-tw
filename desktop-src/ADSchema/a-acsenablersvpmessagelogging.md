# <a name="acs-enable-rsvp-message-logging-attribute"></a>ACS-啟用-RSVP-訊息記錄屬性

如果已啟用 RSVP 記錄，則為 True。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ACS-啟用-RSVP-訊息記錄      |
| Ldap-顯示名稱 | aCSEnableRSVPMessageLogging          |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.768               |
| 系統識別碼-Guid    | 7f561285-5301-11d1-a9c5-0000f80367c1 |
| Syntax            | [**Boolean**](s-boolean.md)         |
