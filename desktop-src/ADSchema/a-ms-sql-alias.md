# <a name="ms-sql-alias-attribute"></a>MS-CHAP-Alias 屬性

用來連接到資料庫的別名。 預設值設定為 [別名]。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-別名                                |
| Ldap-顯示名稱 | mS-SQL-別名                                |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1395                     |
| 系統識別碼-Guid    | e0c6baae-ccee-11d2-9993-0000f87a57d4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
