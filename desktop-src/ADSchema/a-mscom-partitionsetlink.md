# <a name="ms-com-partitionsetlink-attribute"></a>PartitionSetLink 屬性

用來將 COM + 分割與 COM + PartitionSet 物件產生關聯的連結。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | PartitionSetLink                 |
| Ldap-顯示名稱 | msCOM-PartitionSetLink                  |
| 大小              | \-                                      |
| 更新許可權  | 此值是由系統所設定。        |
| 更新頻率  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1424                 |
| 系統識別碼-Guid    | 67f121dc-7d02-4c7d-82f5-9ad4c950ac34    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |



## <a name="implementations"></a>實作

-   [**Windows Server 2012**](#windows-server-2012)
