# <a name="supplemental-credentials-attribute"></a>Supplemental-Credentials 屬性

用於驗證的預存認證。 使用者密碼的加密版本。 這個屬性不是可讀取或可寫入的。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | Supplemental-Credentials                              |
| Ldap-顯示名稱 | supplementalCredentials                               |
| 大小              | \-                                                    |
| 更新許可權  | 此值是由系統所設定。                      |
| 更新頻率  | 當使用者的密碼變更時。                     |
| Attribute-Id      | 1.2.840.113556.1.4.125                                |
| 系統識別碼-Guid    | bf967a3f-0de6-11d0-a285-00aa003049e2                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
