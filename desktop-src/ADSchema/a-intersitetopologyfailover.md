# <a name="inter-site-topology-failover-attribute"></a>網站間拓撲-容錯移轉屬性

表示在網站間拓撲產生器的上一次保持運作之後，必須經過多少時間才會被視為失效。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 網站間拓撲-容錯移轉         |
| Ldap-顯示名稱 | interSiteTopologyFailover            |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1248              |
| 系統識別碼-Guid    | b7c69e60-2cc7-11d2-854e-00a0c983f608 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
