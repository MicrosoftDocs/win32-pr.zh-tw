# <a name="min-ticket-age-attribute"></a>最小票證-年齡屬性

這個屬性會決定使用者的票證授與票證 (TGT) 的最小時間週期（以小時為單位），在提出要求以更新票證之前，可以使用 Kerberos 驗證。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 最小票證-年齡                       |
| Ldap-顯示名稱 | minTicketAge                         |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.80                |
| 系統識別碼-Guid    | bf9679c4-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**間隔**](s-interval.md)       |
