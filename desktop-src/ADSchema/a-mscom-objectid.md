| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | ms-COM-ObjectId                                       |
| Ldap-顯示名稱 | msCOM-ObjectId                                        |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1428                               |
| 系統識別碼-Guid    | 430f678b-889f-41f2-9843-203b5a65572f                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
