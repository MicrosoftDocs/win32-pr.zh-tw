# <a name="com-unique-libid-attribute"></a>COM-Unique-LIBID 屬性

| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | COM-唯一-LIBID                            |
| Ldap-顯示名稱 | cOMUniqueLIBID                              |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.250                      |
| 系統識別碼-Guid    | 281416da-1968-11d0-a28f-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
