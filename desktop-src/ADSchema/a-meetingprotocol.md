|-------------------|---------------------------------------------|
| CN                | meetingProtocol                             |
| Ldap-顯示名稱 | meetingProtocol                             |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.570                      |
| 系統識別碼-Guid    | 11b6cc81-48c4-11d1-a9c3-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
