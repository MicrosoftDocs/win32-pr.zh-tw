| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | Address-Syntax                                        |
| Ldap-顯示名稱 | addressSyntax                                         |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.2.255                                |
| 系統識別碼-Guid    | 5fd42463-1262-11d0-a060-00aa006c33ed                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
