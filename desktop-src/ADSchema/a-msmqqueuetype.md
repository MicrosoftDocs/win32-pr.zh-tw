# <a name="msmq-queue-type-attribute"></a>MSMQ-Queue-Type 屬性

描述佇列所提供之服務類型的 GUID。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | MSMQ-佇列類型                                       |
| Ldap-顯示名稱 | mSMQQueueType                                         |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.917                                |
| 系統識別碼-Guid    | 9a0dc320-c100-11d1-bbc5-0080c76670c0                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
