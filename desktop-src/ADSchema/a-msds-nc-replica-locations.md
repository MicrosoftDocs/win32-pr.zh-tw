| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | ms DS-----複本-位置              |
| Ldap-顯示名稱 | 高階 NC-複本-位置               |
| 大小              | \-                                      |
| 更新許可權  | \-                                      |
| 更新頻率  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1661                 |
| 系統識別碼-Guid    | 97de9615-b537-46bc-ac0f-10720f3909f3    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
