# <a name="ms-ds-repl-value-meta-data-attribute"></a>ms-DS-複寫-值-中繼資料屬性

屬性之每個值的中繼資料清單。 中繼資料會指出上次變更值的人員。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | ms-DS-複製-值中繼資料                  |
| Ldap-顯示名稱 | ReplValueMetaData                      |
| 大小              | \-                                          |
| 更新許可權  | 此值是由系統所設定。            |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1708                     |
| 系統識別碼-Guid    | 2f5c8145-e1bd-410b-8957-8bfa81d5acfd        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
