# <a name="logon-count-attribute"></a>Logon-Count 屬性

帳戶成功登入的次數。 值為0表示值不明。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | Logon-Count                          |
| Ldap-顯示名稱 | logonCount                           |
| 大小              | 4 個位元組                              |
| 更新許可權  | 網域系統管理員                 |
| 更新頻率  | 每次使用者登入時。          |
| Attribute-Id      | 1.2.840.113556.1.4.169               |
| 系統識別碼-Guid    | bf9679aa-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**列舉型別**](s-enumeration.md) |

## <a name="remarks"></a>備註

這個屬性不會複寫，並且會保留在網域中的每個網域控制站上。 若要取得使用者在網域中成功登入嘗試總數的精確值，必須查詢網域中的每個網域控制站，並使用值的總和。 請記住，此屬性不會複寫，因此已淘汰的網域控制站也可能會有使用者的計數登入，而這將會從計數中遺失。

> [!IMPORTANT]
> 由於與16位版 LAN Manager 的相容性，因此屬性的上限為65535。 達到此限制之後，您就無法將它用作為此網域控制站上的使用者活動指標。
