# <a name="ms-ts-connect-client-drives-attribute"></a>ms-TS-Connect-用戶端-磁片磁碟機屬性

| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ms-TS-Connect-Client-Drives          |
| Ldap-顯示名稱 | msTSConnectClientDrives              |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1986              |
| 系統識別碼-Guid    | 23572aaf-29dd-44ea-b0fa-7e8438b9a4a3 |
| Syntax            | [**Boolean**](s-boolean.md)         |
