# <a name="printer-name-attribute"></a>Printer-Name 屬性

附加印表機的顯示名稱。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | Printer-Name                                |
| Ldap-顯示名稱 | printerName                                 |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.300                      |
| 系統識別碼-Guid    | 244b296e-5abd-11d0-afd2-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
