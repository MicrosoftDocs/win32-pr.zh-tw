# <a name="shadowwarning-attribute"></a>shadowWarning 屬性

包含密碼到期前的天數，以警告使用者。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | shadowWarning                        |
| Ldap-顯示名稱 | shadowWarning                        |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.8                      |
| 系統識別碼-Guid    | 7ae89c9c-2976-4a46-bb8a-340f88560117 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
