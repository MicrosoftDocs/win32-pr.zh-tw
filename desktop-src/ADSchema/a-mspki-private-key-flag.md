# <a name="ms-pki-private-key-flag-attribute"></a>ms-PKI-私用金鑰旗標屬性

包含私密金鑰相關的旗標。



| 進入 | 值 |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-私用金鑰旗標                                                                           |
| Ldap-顯示名稱 | Mspki-certificate-name-flag-私用金鑰旗標                                                                            |
| 大小              | 4 個位元組                                                                                           |
| 更新許可權  | 網域系統管理員                                                                              |
| 更新頻率  | 當您編輯、建立或複製憑證範本時 (ms-PKI-憑證範本) 物件。 |
| Attribute-Id      | 1.2.840.113556.1.4.1431                                                                           |
| 系統識別碼-Guid    | bab04ac2-0435-4709-9307-28380e7c7001                                                              |
| Syntax            | [**列舉型別**](s-enumeration.md)                                                              |
