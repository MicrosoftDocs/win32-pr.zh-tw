# <a name="acs-enable-acs-service-attribute"></a>ACS-啟用-ACS-服務屬性

如果要啟用 ACS 服務，則為 True。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ACS-啟用-ACS-服務               |
| Ldap-顯示名稱 | aCSEnableACSService                  |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.770               |
| 系統識別碼-Guid    | 7f561287-5301-11d1-a9c5-0000f80367c1 |
| Syntax            | [**Boolean**](s-boolean.md)         |
