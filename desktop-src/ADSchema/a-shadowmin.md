# <a name="shadowmin-attribute"></a>shadowMin 屬性

包含陰影變更之間的最小天數。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | shadowMin                            |
| Ldap-顯示名稱 | shadowMin                            |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.6                      |
| 系統識別碼-Guid    | a76b8737-e5a1-4568-b057-dc12e04be4b2 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
