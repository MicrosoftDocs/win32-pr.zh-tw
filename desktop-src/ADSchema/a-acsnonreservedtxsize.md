# <a name="acs-non-reserved-tx-size-attribute"></a>ACS-非保留的 Tx 大小屬性

應用程式可以在保留備妥之前使用的權杖 bucket 大小。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ACS-非保留-德克薩斯維大小             |
| Ldap-顯示名稱 | aCSNonReservedTxSize                 |
| 大小              | 8 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.898               |
| 系統識別碼-Guid    | f072230d-aef5-11d1-bdcf-0000f80367c1 |
| Syntax            | [**間隔**](s-interval.md)       |
