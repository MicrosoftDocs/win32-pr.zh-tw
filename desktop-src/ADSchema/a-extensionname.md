# <a name="extension-name-attribute"></a>Extension-Name 屬性

用來擴充目錄物件之 UI 的屬性頁名稱。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | Extension-Name                              |
| Ldap-顯示名稱 | extensionName                               |
| 大小              | \-                                          |
| 更新許可權  | 擴充物件的建立者。        |
| 更新頻率  | 每次建立新的 UI 擴充功能時。     |
| Attribute-Id      | 1.2.840.113556.1.2.227                      |
| 系統識別碼-Guid    | bf967972-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
