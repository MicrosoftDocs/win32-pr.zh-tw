| Ldap-顯示名稱 | serviceDNSNameType                          |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |

| 系統識別碼-Guid    | 28630eba-41d5-11d1-a9c1-0000f80367c1        |
