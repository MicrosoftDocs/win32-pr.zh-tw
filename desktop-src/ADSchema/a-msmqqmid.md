# <a name="msmq-qm-id-attribute"></a>MSMQ-QM-ID 屬性

| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | MSMQ-QM-ID                                            |
| Ldap-顯示名稱 | mSMQQMID                                              |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.951                                |
| 系統識別碼-Guid    | 9a0dc33e-c100-11d1-bbc5-0080c76670c0                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
