| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | rpc-Ns-物件識別碼                            |
| Ldap-顯示名稱 | rpcNsObjectID                               |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.312                      |
| 系統識別碼-Guid    | 29401c48-7a27-11d0-afd6-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
