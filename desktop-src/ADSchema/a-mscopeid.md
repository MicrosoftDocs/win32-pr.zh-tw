# <a name="mscope-id-attribute"></a>Mscope-Id 屬性

指出指定的 DHCP 伺服器上有多播領域。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | Mscope-Id                            |
| Ldap-顯示名稱 | mscopeId                             |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.716               |
| 系統識別碼-Guid    | 963d2751-48be-11d1-a9c3-0000f80367c1 |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
