# <a name="inter-site-topology-renew-attribute"></a>網站間-拓撲-更新屬性

這個類別指出網站間拓撲產生器更新傳送至相同網站中的網域控制站之 keep-alive 訊息的頻率。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 網站間-拓撲-更新            |
| Ldap-顯示名稱 | interSiteTopologyRenew               |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1247              |
| 系統識別碼-Guid    | b7c69e5f-2cc7-11d2-854e-00a0c983f608 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
