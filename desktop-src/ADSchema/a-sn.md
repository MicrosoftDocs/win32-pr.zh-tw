# <a name="surname-attribute"></a>姓氏屬性

此屬性包含使用者的系列或姓氏。



| 進入 | 值 |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Surname                                                                          |
| Ldap-顯示名稱 | sn                                                                               |
| 大小              | \-                                                                               |
| 更新許可權  | 任何人都可以根據所建立物件的安全性來更新這個物件。 |
| 更新頻率  | \-                                                                               |
| Attribute-Id      | 2.5.4.4                                                                          |
| 系統識別碼-Guid    | bf967a41-0de6-11d0-a285-00aa003049e2                                             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                      |
