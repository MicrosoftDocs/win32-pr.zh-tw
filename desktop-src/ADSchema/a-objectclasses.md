# <a name="object-classes-attribute"></a>Object-Classes 屬性

多值屬性，其中包含表示架構中每個類別的字串。 每個值都包含 governsID、lDAPDisplayName、mustContain、mayContain 等等。



| 進入 | 值 |
|-------------------|--------------------------------------------------|
| CN                | Object-Classes                                   |
| Ldap-顯示名稱 | objectClasses                                    |
| 大小              | 每個類別名稱平均大約20個位元組。        |
| 更新許可權  | 物件的設計工具會設定此值。 |
| 更新頻率  | 每當類別的設計變更時。        |
| Attribute-Id      | 2.5.21.6                                         |
| 系統識別碼-Guid    | 9a7ad94b-ca53-11d1-bbd0-0080c76670c0             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)      |
