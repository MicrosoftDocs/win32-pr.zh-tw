# <a name="print-end-time-attribute"></a>列印-結束時間屬性

列印佇列停止服務工作的時間。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 列印-結束時間                       |
| Ldap-顯示名稱 | printEndTime                         |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.234               |
| 系統識別碼-Guid    | 281416ca-1968-11d0-a28f-00aa003049e2 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
