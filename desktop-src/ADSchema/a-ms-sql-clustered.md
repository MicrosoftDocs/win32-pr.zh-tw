# <a name="ms-sql-clustered-attribute"></a>MS-SQL-叢集屬性

如果伺服器已叢集化，則為 True。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | MS-CHAP-叢集                     |
| Ldap-顯示名稱 | Ms-chap-叢集                     |
| 大小              | 4 個位元組                              |
| 更新許可權  | 此值是由系統所設定。     |
| 更新頻率  | 在系統啟動時。                   |
| Attribute-Id      | 1.2.840.113556.1.4.1373              |
| 系統識別碼-Guid    | 7778bd90-ccee-11d2-9993-0000f87a57d4 |
| Syntax            | [**Boolean**](s-boolean.md)         |
