# <a name="acs-minimum-delay-variation-attribute"></a>ACS-最小延遲-變化屬性

**ACS-最小延遲變化** 屬性僅供內部使用。 以 RFC2210 為基礎。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ACS-最小延遲-變化          |
| Ldap-顯示名稱 | aCSMinimumDelayVariation             |
| 大小              | 8 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1317              |
| 系統識別碼-Guid    | 9c65329b-3b90-11d2-90cc-00c04fd91ab1 |
| Syntax            | [**間隔**](s-interval.md)       |
