# <a name="ms-sql-memory-attribute"></a>MS SQL 記憶體屬性

電腦上的記憶體數量。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | MS SQL 記憶體                        |
| Ldap-顯示名稱 | mS SQL 記憶體                        |
| 大小              | 8 個位元組                              |
| 更新許可權  | 此值是由系統所設定。     |
| 更新頻率  | 在系統啟動時。                   |
| Attribute-Id      | 1.2.840.113556.1.4.1367              |
| 系統識別碼-Guid    | 5b5d448c-ccee-11d2-9993-0000f87a57d4 |
| Syntax            | [**間隔**](s-interval.md)       |
