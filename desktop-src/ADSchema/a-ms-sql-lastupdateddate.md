# <a name="ms-sql-lastupdateddate-attribute"></a>LastUpdatedDate 屬性

此值代表 SQL 上次更新 Active Directory 的時間。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | LastUpdatedDate                      |
| Ldap-顯示名稱 | LastUpdatedDate                      |
| 大小              | \-                                          |
| 更新許可權  | 此值是由系統所設定。            |
| 更新頻率  | 當系統啟動或更新 AD 時。    |
| Attribute-Id      | 1.2.840.113556.1.4.1381                     |
| 系統識別碼-Guid    | 9fcc43d4-ccee-11d2-9993-0000f87a57d4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
