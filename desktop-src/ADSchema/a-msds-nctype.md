| Ldap-顯示名稱 | msDS-NcType                          |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.2024              |
| 系統識別碼-Guid    | 5a2eacd7-cc2b-48cf-9d9a-b6f1a0024de9 |
