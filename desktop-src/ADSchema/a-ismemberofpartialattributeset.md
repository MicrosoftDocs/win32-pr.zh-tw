# <a name="is-member-of-partial-attribute-set-attribute"></a>是-局部屬性集屬性的成員

若 **為 TRUE**，則會將此屬性複寫至通用類別目錄。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 是-Partial 屬性集的成員   |
| Ldap-顯示名稱 | isMemberOfPartialAttributeSet        |
| 大小              | \-                                   |
| 更新許可權  | 架構系統管理員                 |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.639               |
| 系統識別碼-Guid    | 19405b9d-3cfa-11d1-a9c0-0000f80367c1 |
| Syntax            | [**Boolean**](s-boolean.md)         |
