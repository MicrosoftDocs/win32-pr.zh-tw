# <a name="department-attribute"></a>部門屬性

包含使用者工作所在部門的名稱。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | department                                  |
| Ldap-顯示名稱 | department                                  |
| 大小              | \-                                          |
| 更新許可權  | 網域系統管理員或帳戶擁有者。      |
| 更新頻率  | 當部門需要變更時。    |
| Attribute-Id      | 1.2.840.113556.1.2.141                      |
| 系統識別碼-Guid    | bf96794f-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
