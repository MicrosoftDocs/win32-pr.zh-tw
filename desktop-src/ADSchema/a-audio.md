| CN                | 音訊                                                 |
| Ldap-顯示名稱 | 音訊                                                 |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |

| 系統識別碼-Guid    | d0e1d224-e1a0-42ce-a2da-793ba5244f35                  |
