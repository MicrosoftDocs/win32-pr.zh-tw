# <a name="fsmo-role-owner-attribute"></a>FSMO 角色-擁有者屬性

彈性的 Single-Master 作業：可以修改架構之 DC 的分辨名稱。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | FSMO 角色-擁有者                         |
| Ldap-顯示名稱 | fSMORoleOwner                           |
| 大小              | \-                                      |
| 更新許可權  | 架構系統管理員                    |
| 更新頻率  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.369                  |
| 系統識別碼-Guid    | 66171887-8f3c-11d0-afda-00c04fd930c9    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
