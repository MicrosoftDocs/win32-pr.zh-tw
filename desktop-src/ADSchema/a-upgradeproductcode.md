# <a name="upgrade-product-code-attribute"></a>升級-產品代碼屬性

此屬性包含其他封裝的產品代碼（例如應用程式），可由此封裝升級，或可升級此封裝。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | 升級-產品代碼                                  |
| Ldap-顯示名稱 | upgradeProductCode                                    |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.813                                |
| 系統識別碼-Guid    | d9e18312-8939-11d1-aebc-0000f80367c1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
