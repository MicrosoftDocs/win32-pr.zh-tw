# <a name="print-spooling-attribute"></a>Print-Spooling 屬性

代表印表機幕後處理類型的字串。



| 進入 | 值 |
|-------------------|----------------------------------------------------------------------|
| CN                | Print-Spooling                                                       |
| Ldap-顯示名稱 | printSpooling                                                        |
| 大小              | 可能的值： PrintDirect、PrintWhileSpooling、PrintAfterSpooled。 |
| 更新許可權  | \-                                                                   |
| 更新頻率  | \-                                                                   |
| Attribute-Id      | 1.2.840.113556.1.4.274                                               |
| 系統識別碼-Guid    | ba305f6c-47e3-11d0-a1a6-00c04fd930c9                                 |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                          |
