# <a name="common-name-attribute"></a>Common-Name 屬性

代表物件的名稱。 用來執行搜尋。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | Common-Name                                 |
| Ldap-顯示名稱 | cn                                          |
| 大小              | 平均大小為64個位元組。                   |
| 更新許可權  | 網域系統管理員                        |
| 更新頻率  | 只有在建立新的物件時才會發生。          |
| Attribute-Id      | 2.5.4.3                                     |
| 系統識別碼-Guid    | bf96793f-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
