| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | 查詢-原則-BL                         |
| Ldap-顯示名稱 | queryPolicyBL                           |
| 大小              | \-                                      |
| 更新許可權  | 此值是由系統所設定。        |
| 更新頻率  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.608                  |
| 系統識別碼-Guid    | e1aea404-cd5b-11d0-afff-0000f80367c1    |
