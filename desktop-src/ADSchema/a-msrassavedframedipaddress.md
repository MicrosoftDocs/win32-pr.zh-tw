| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | msRASSavedFramedIPAddress            |
| Ldap-顯示名稱 | msRASSavedFramedIPAddress            |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1190              |
| 系統識別碼-Guid    | db0c90c6-c1f2-11d1-bbc5-0080c76670c0 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
