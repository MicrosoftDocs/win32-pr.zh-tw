# <a name="rid-allocation-pool-attribute"></a>RID 組態集區屬性

當 RID 先前的組態集區已用完時，被 RID 管理員所使用的集區。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | RID-組態集區                  |
| Ldap-顯示名稱 | rIDAllocationPool                    |
| 大小              | 8 個位元組                              |
| 更新許可權  | 此值是由系統所設定。     |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.371               |
| 系統識別碼-Guid    | 66171889-8f3c-11d0-afda-00c04fd930c9 |
| Syntax            | [**間隔**](s-interval.md)       |
