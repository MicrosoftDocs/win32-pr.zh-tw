# <a name="dns-tombstoned-attribute"></a>DNS-Tombstoned 屬性

如果此物件已被加上標記，則為 True。 這個屬性可讓您更輕鬆且快速地搜尋已標記的記錄。 已刪除的物件是已刪除但尚未從目錄移除的物件。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | DNS-Tombstoned                       |
| Ldap-顯示名稱 | dNSTombstoned                        |
| 大小              | 4 個位元組                              |
| 更新許可權  | 此值是由系統所設定。     |
| 更新頻率  | 每次刪除物件時。       |
| Attribute-Id      | 1.2.840.113556.1.4.1414              |
| 系統識別碼-Guid    | d5eb2eb7-be4e-463b-a214-634a44d7392e |
| Syntax            | [**Boolean**](s-boolean.md)         |
