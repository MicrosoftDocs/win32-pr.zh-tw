# <a name="governs-id-attribute"></a>控管識別碼屬性

這個 Class-Schema 物件所定義之類別的唯一物件識別碼。



| 進入 | 值 |
|-------------------|-----------------------------------------------------------------|
| CN                | 控管-識別碼                                                      |
| Ldap-顯示名稱 | governsID                                                       |
| 大小              | \-                                                              |
| 更新許可權  | \-                                                              |
| 更新頻率  | \-                                                              |
| Attribute-Id      | 1.2.840.113556.1.2.22                                           |
| 系統識別碼-Guid    | bf96797d-0de6-11d0-a285-00aa003049e2                            |
| Syntax            | [**String(Object-Identifier)**](s-string-object-identifier.md) |
