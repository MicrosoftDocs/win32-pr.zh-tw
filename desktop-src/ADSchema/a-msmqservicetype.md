# <a name="msmq-service-type-attribute"></a>MSMQ-服務類型屬性

| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | MSMQ-服務類型                    |
| Ldap-顯示名稱 | mSMQServiceType                      |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.930               |
| 系統識別碼-Guid    | 9a0dc32d-c100-11d1-bbc5-0080c76670c0 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
