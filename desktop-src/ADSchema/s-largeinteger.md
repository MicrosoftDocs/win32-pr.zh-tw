# <a name="largeinteger-syntax"></a>LargeInteger 語法

代表64位帶正負號的整數值。



| 進入 | 值 |
|--------------|------------------------------------------------------------------------------------|
| 名稱         | LargeInteger                                                                       |
| 語法識別碼    | 2.5.5.16                                                                           |
| OM 識別碼        | 65                                                                                 |
| MAPI 類型    | \-                                                                                 |
| ADS 類型     | ADSTYPE \_ 大型 \_ 整數                                                            |
| Variant 類型 | VT \_ 分派                                                                       |
| SDS 類型     | 可以轉換成 [**IADsLargeInteger**](/windows/desktop/api/iads/nn-iads-iadslargeinteger)的 COM 物件。 |
