# <a name="ms-ds-consistency-guid-attribute"></a>MS DS-一致性-Guid 屬性

這個屬性是用來藉由比較 Guid 來檢查目錄與另一個物件、資料庫或應用程式之間的一致性。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | MS-DS-Consistency-Guid                                |
| Ldap-顯示名稱 | ConsistencyGuid                                 |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1360                               |
| 系統識別碼-Guid    | 23773dc2-b63a-11d2-90e1-00c04fd91ab1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
