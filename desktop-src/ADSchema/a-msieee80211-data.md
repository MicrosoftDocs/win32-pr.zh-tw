# <a name="ms-ieee-80211-data-attribute"></a>ms-ieee-80211-Data 屬性

在無線網路的群組原則中儲存慣用網路設定的清單。



| 進入 | 值 |
|-------------------|----------------------------------------------------------------------------------|
| CN                | ms-ieee-80211-資料                                                               |
| Ldap-顯示名稱 | msieee80211-資料                                                                 |
| 大小              | \-                                                                               |
| 更新許可權  | 網域系統管理員                                                             |
| 更新頻率  | 每次網域管理員變更網域或 OU 的無線網路原則時。 |
| Attribute-Id      | 1.2.840.113556.1.4.1821                                                          |
| 系統識別碼-Guid    | 0e0d0938-2658-4580-a9f6-7a0ac7b566cb                                             |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md)                            |
