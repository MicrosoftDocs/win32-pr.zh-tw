# <a name="site-list-attribute"></a>Site-List 屬性

連接至此連結化物件的網站清單。



| 進入 | 值 |
|-------------------|----------------------------------------------|
| CN                | Site-List                                    |
| Ldap-顯示名稱 | 清單                                     |
| 大小              | \-                                           |
| 更新許可權  | 網域系統管理員                         |
| 更新頻率  | 每次新增或移除網站。 |
| Attribute-Id      | 1.2.840.113556.1.4.821                       |
| 系統識別碼-Guid    | d50c2cdc-8951-11d1-aebc-0000f80367c1         |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md)      |
