# <a name="ms-ds-az-scope-name-attribute"></a>ms-chap-Az-Scope-Name 屬性

可唯一識別範圍物件的字串。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | ms-chap-Az-範圍名稱                         |
| Ldap-顯示名稱 | AzScopeName                            |
| 大小              | \-                                          |
| 更新許可權  | AzRoles 系統管理員                               |
| 更新頻率  | 在初始化或原則變更期間。     |
| Attribute-Id      | 1.2.840.113556.1.4.1799                     |
| 系統識別碼-Guid    | 515a6b06-2617-4173-8099-d5605df043c6        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
