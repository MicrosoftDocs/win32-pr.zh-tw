# <a name="unixhomedirectory-attribute"></a>unixHomeDirectory 屬性

| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | unixHomeDirectory                    |
| Ldap-顯示名稱 | unixHomeDirectory                    |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.3                      |
| 系統識別碼-Guid    | bc2dba12-000f-464d-bf1d-0808465d8843 |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
