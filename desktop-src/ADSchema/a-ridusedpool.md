# <a name="rid-used-pool-attribute"></a>RID 使用的集區屬性

DC 所使用的 RID 集區。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | RID-已使用集區                        |
| Ldap-顯示名稱 | rIDUsedPool                          |
| 大小              | 8 個位元組                              |
| 更新許可權  | 此值是由系統所設定。     |
| 更新頻率  | 每次清空 RID 集區時。      |
| Attribute-Id      | 1.2.840.113556.1.4.373               |
| 系統識別碼-Guid    | 6617188b-8f3c-11d0-afda-00c04fd930c9 |
| Syntax            | [**間隔**](s-interval.md)       |
