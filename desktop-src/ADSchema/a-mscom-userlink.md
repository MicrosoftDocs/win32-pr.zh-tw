# <a name="ms-com-userlink-attribute"></a>UserLink 屬性

用來將 COM + PartitionSet 與使用者物件產生關聯的連結。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | UserLink                         |
| Ldap-顯示名稱 | msCOM-UserLink                          |
| 大小              | \-                                      |
| 更新許可權  | 此值是由系統所設定。        |
| 更新頻率  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1425                 |
| 系統識別碼-Guid    | 9e6f3a4d-242c-4f37-b068-36b57f9fc852    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
