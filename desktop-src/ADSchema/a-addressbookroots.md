# <a name="address-book-roots-attribute"></a>通訊錄-根屬性

供 Exchange 使用。 Exchange 設定通訊錄容器的樹狀目錄以顯示於 MAPI 通訊錄中。 Exchange 設定物件上的這個屬性會列出通訊錄容器樹狀結構的根。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | 位址-書籍-根目錄                      |
| Ldap-顯示名稱 | addressBookRoots                        |
| 大小              | \-                                      |
| 更新許可權  | \-                                      |
| 更新頻率  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1244                 |
| 系統識別碼-Guid    | f70b6e48-06f4-11d2-aa53-00c04fd7d83a    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
