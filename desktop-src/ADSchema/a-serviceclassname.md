# <a name="service-class-name-attribute"></a>服務類別名稱屬性

管理點所代表之服務的字串名稱。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | 服務類別名稱                          |
| Ldap-顯示名稱 | serviceClassName                            |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.509                      |
| 系統識別碼-Guid    | b7b1311d-b82e-11d0-afee-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
