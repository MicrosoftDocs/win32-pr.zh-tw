# <a name="other-login-workstations-attribute"></a>其他-登入工作站屬性

使用者可以登入的非 Windows NT 或 LAN Manager 工作站。

> [!Note]  
> Active Directory 不會使用或填入此欄位。

 



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | 其他-登入工作站                    |
| Ldap-顯示名稱 | otherLoginWorkstations                      |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.91                       |
| 系統識別碼-Guid    | bf9679f1-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
