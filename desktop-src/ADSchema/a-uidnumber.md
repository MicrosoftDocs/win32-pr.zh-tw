# <a name="uidnumber-attribute"></a>uidNumber 屬性

包含可唯一識別系統管理網域中使用者的整數。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | uidNumber                            |
| Ldap-顯示名稱 | uidNumber                            |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.0                      |
| 系統識別碼-Guid    | 850fcc8f-9c6b-47e1-b671-7c654be4d5b3 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
