| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 預先設定的時間                       |
| Ldap-顯示名稱 | priorSetTime                         |
| 大小              | 8 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.99                |
| 系統識別碼-Guid    | bf967a01-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**間隔**](s-interval.md)       |
