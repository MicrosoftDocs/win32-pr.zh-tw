# <a name="ms-ds-source-object-dn-attribute"></a>ms DS-來源-物件-DN 屬性

包含另一個樹系中對應至這個物件之物件辨別名稱的字串表示。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | ms-DS-來源-物件-DN                      |
| Ldap-顯示名稱 | SourceObjectDN                         |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1879                     |
| 系統識別碼-Guid    | 773e93af-d3b4-48d4-b3f9-06457602d3d0        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
