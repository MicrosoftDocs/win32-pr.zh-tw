# <a name="modify-time-stamp-attribute"></a>修改時間戳記屬性

計算的屬性，表示上次變更此物件的日期。 此值不會複寫。



| 進入 | 值 |
|-------------------|---------------------------------------------------------------|
| CN                | 修改時間戳記                                             |
| Ldap-顯示名稱 | modifyTimeStamp                                               |
| 大小              | 8 個位元組                                                       |
| 更新許可權  | 此值是由系統所設定。                              |
| 更新頻率  | \-                                                            |
| Attribute-Id      | 2.5.18.2                                                      |
| 系統識別碼-Guid    | 9a7ad94a-ca53-11d1-bbd0-0080c76670c0                          |
| Syntax            | [**String(Generalized-Time)**](s-string-generalized-time.md) |
