# <a name="ms-ds-security-group-extra-classes-attribute"></a>ms-DS-安全性群組-額外類別屬性

可透過 Active Directory 消費者和電腦嵌入式管理單元新增至安全性群組之非標準類別的一般名稱。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | ms-DS-安全性群組-額外類別          |
| Ldap-顯示名稱 | 安全性群組-額外類別           |
| 大小              | \-                                          |
| 更新許可權  | 網域系統管理員                        |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1688                     |
| 系統識別碼-Guid    | 4f146ae8-a4fe-4801-a731-f51848a4f4e4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |



## <a name="implementations"></a>實作

-   [**Windows Server 2003 R2**](#windows-server-2003-r2)
