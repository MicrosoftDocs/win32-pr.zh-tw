# <a name="ms-tapi-ip-address-attribute"></a>ms-TAPI-Ip 位址屬性

使用者登入之 TAPI 用戶端電腦的 IP 位址。 這個屬性可用來作為用來保存電腦 IP 位址的一般屬性。



| 進入 | 值 |
|-------------------|------------------------------------------------------------|
| CN                | ms-TAPI-Ip 位址                                         |
| Ldap-顯示名稱 | msTAPI-IpAddress                                           |
| 大小              | 每個 IP 位址會以字串的形式儲存在 b.. D 標記法中。 |
| 更新許可權  | 不需要任何特殊許可權。                            |
| 更新頻率  | 通常很低。                                        |
| Attribute-Id      | 1.2.840.113556.1.4.1701                                    |
| 系統識別碼-Guid    | efd7d7f7-178e-4767-87fa-f8a16b840544                       |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                |
