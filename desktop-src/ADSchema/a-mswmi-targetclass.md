# <a name="ms-wmi-targetclass-attribute"></a>>targetclass 屬性

要建立之原則物件的類別名稱。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | >targetclass                          |
| Ldap-顯示名稱 | msWMI->targetclass                           |
| 大小              | 少於二十個字元。                |
| 更新許可權  | 群組原則系統管理員                  |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1645                     |
| 系統識別碼-Guid    | 95b6d8d6-c9e8-4661-a2bc-6a5cabc04c62        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
