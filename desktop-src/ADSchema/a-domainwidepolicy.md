# <a name="domain-wide-policy-attribute"></a>全網域原則屬性

這是為了將使用者可延伸的原則複寫到用戶端。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | 全網域原則                                    |
| Ldap-顯示名稱 | domainWidePolicy                                      |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.421                                |
| 系統識別碼-Guid    | 80a67e29-9f22-11d0-afdd-00c04fd930c9                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
