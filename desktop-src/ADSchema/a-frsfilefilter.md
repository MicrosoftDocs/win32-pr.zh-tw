# <a name="frs-file-filter-attribute"></a>FRS-檔案篩選器屬性

從檔案複寫中排除的副檔名清單。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | FRS-檔案-篩選                             |
| Ldap-顯示名稱 | fRSFileFilter                               |
| 大小              | \-                                          |
| 更新許可權  | 網域系統管理員                        |
| 更新頻率  | 當複寫設定時。                  |
| Attribute-Id      | 1.2.840.113556.1.4.483                      |
| 系統識別碼-Guid    | 1be8f170-a9ff-11d0-afe2-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
