# <a name="international-isdn-number-attribute"></a>國際 ISDN 號碼屬性

指定與物件相關聯的國際 ISDN 號碼。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | 國際 ISDN 號碼                   |
| Ldap-顯示名稱 | internationalISDNNumber                     |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 2.5.4.25                                    |
| 系統識別碼-Guid    | bf96798d-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Numeric)**](s-string-numeric.md) |
