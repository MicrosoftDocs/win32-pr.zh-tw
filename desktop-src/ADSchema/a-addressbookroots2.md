# <a name="address-book-roots2-attribute"></a>Roots2 屬性

供 Exchange 使用。 Exchange 設定通訊錄容器的樹狀目錄以顯示於 MAPI 通訊錄中。 Exchange 設定物件上的這個屬性會列出通訊錄容器樹狀結構的根。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | Address-Book-Roots2                     |
| Ldap-顯示名稱 | addressBookRoots2                       |
| 大小              | \-                                      |
| 更新許可權  | \-                                      |
| 更新頻率  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.2046                 |
| 系統識別碼-Guid    | 508ca374-a511-4e4e-9f4f-856f61a6b7e4    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
