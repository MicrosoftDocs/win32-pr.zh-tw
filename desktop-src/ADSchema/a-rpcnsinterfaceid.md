# <a name="rpc-ns-interface-id-attribute"></a>rpc-Ns-Interface-ID 屬性

指定伺服器所支援的介面識別碼。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | rpc-Ns-Interface-ID                         |
| Ldap-顯示名稱 | rpcNsInterfaceID                            |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.115                      |
| 系統識別碼-Guid    | bf967a25-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
