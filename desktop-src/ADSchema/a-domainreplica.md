Unicode 字串屬性，提供 Windows NT 4.0 複寫網域控制站的清單。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | Domain-Replica                              |
| Ldap-顯示名稱 | domainReplica                               |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.158                      |
| 系統識別碼-Guid    | bf96795e-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
