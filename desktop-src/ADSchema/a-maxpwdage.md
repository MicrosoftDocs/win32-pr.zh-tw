# <a name="max-pwd-age-attribute"></a>Max-Pwd-Age 屬性

密碼的最大時間量，以 100-毫微秒間隔為有效。 此值會儲存為一個大型整數，表示密碼在密碼到期前的設定時間，100-納秒間隔的數目。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 最大-Pwd-使用期限                          |
| Ldap-顯示名稱 | maxPwdAge                            |
| 大小              | \-                                   |
| 更新許可權  | 網域系統管理員                 |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.74                |
| 系統識別碼-Guid    | bf9679bb-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**間隔**](s-interval.md)       |
