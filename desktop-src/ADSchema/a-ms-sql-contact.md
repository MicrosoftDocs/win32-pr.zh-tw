# <a name="ms-sql-contact-attribute"></a>MS-SQL-Contact 屬性

使用者定義的字串。 預設值是設定為 Contact。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-連絡人                              |
| Ldap-顯示名稱 | mS-SQL-連絡人                              |
| 大小              | \-                                          |
| 更新許可權  | 網域系統管理員                        |
| 更新頻率  | 在系統設定時。                            |
| Attribute-Id      | 1.2.840.113556.1.4.1365                     |
| 系統識別碼-Guid    | 4f6cbdd8-ccee-11d2-9993-0000f87a57d4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
