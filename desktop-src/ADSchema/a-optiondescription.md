# <a name="option-description-attribute"></a>Option-Description 屬性

此屬性包含在 DHCP 伺服器上設定之選項的描述。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | Option-Description                          |
| Ldap-顯示名稱 | optionDescription                           |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.712                      |
| 系統識別碼-Guid    | 963d274d-48be-11d1-a9c3-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
