# <a name="ms-wmi-sourceorganization-attribute"></a>SourceOrganization 屬性

一開始建立原則物件的商務組織。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | SourceOrganization                   |
| Ldap-顯示名稱 | msWMI-SourceOrganization                    |
| 大小              | 少於50個字元。                 |
| 更新許可權  | 群組原則系統管理員                  |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1644                     |
| 系統識別碼-Guid    | 34f7ed6c-615d-418d-aa00-549a7d7be03e        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
