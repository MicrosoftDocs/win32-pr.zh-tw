# <a name="last-set-time-attribute"></a>Last 設定時間屬性

上次變更密碼的時間。 此值會儲存為大整數，表示自1601年1月1日起 (UTC) 的 100-2-2-2-2-秒間隔。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 上次設定時間                        |
| Ldap-顯示名稱 | lastSetTime                          |
| 大小              | 8 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.53                |
| 系統識別碼-Guid    | bf967998-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**間隔**](s-interval.md)       |
