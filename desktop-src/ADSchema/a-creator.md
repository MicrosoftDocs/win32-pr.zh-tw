| Ldap-顯示名稱 | 創造者                                     |
| 大小              | \-                                          |
| 更新許可權  | 此值是由系統所設定。            |

| 系統識別碼-Guid    | 7bfdcb85-4807-11d1-a9c3-0000f80367c1        |
