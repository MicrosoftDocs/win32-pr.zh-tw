# <a name="bootfile-attribute"></a>bootFile 屬性

包含 UNIX 身分識別管理支援的開機映射名稱。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | bootFile                             |
| Ldap-顯示名稱 | bootFile                             |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.24                     |
| 系統識別碼-Guid    | e3f3cb4e-0f20-42eb-9703-d2ff26e52667 |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
