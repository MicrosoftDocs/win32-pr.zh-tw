# <a name="macaddress-attribute"></a>macAddress 屬性

包含最大、冒號分隔的十六進位標記法中的 MAC 位址。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | macAddress                           |
| Ldap-顯示名稱 | macAddress                           |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.22                     |
| 系統識別碼-Guid    | e6a522dd-9770-43e1-89de-1de5044328f7 |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
