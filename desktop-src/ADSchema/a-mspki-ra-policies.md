# <a name="ms-pki-ra-policies-attribute"></a>ms-PKI-RA-原則屬性

包含簽署註冊要求之註冊授權單位中的必要原則 Oid 清單。



| 進入 | 值 |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-RA-原則                                                                                |
| Ldap-顯示名稱 | Mspki-certificate-name-flag-RA-原則                                                                                 |
| 大小              | 專案數目的64位元組倍。                                                             |
| 更新許可權  | 網域系統管理員                                                                              |
| 更新頻率  | 當您編輯、建立或複製憑證範本時 (ms-PKI-憑證範本) 物件。 |
| Attribute-Id      | 1.2.840.113556.1.4.1438                                                                           |
| 系統識別碼-Guid    | d546ae22-0951-4d47-817e-1c9f96faad46                                                              |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                                       |
