# <a name="product-code-attribute"></a>Product-Code 屬性

此屬性包含特定產品版本之應用程式的唯一識別碼（以字串 GUID 表示），例如 " {12345678-1234-1234-1234-123456789012} "。 此 GUID 中使用的字母必須為大寫。 此識別碼必須因不同版本和語言而有所不同。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | Product-Code                                          |
| Ldap-顯示名稱 | productCode                                           |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.818                                |
| 系統識別碼-Guid    | d9e18317-8939-11d1-aebc-0000f80367c1                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
