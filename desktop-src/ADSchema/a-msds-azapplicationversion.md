# <a name="ms-ds-az-application-version-attribute"></a>ms DS-Az-應用程式版本屬性

版本號碼，表示 AzApplication 已更新。



| 進入 | 值 |
|-------------------|------------------------------------------------------|
| CN                | ms DS-Az-應用程式版本                         |
| Ldap-顯示名稱 | AzApplicationVersion                            |
| 大小              | \-                                                   |
| 更新許可權  | AzRoles 系統管理員                                        |
| 更新頻率  | 當應用程式的版本號碼變更時。 |
| Attribute-Id      | 1.2.840.113556.1.4.1817                              |
| 系統識別碼-Guid    | 7184a120-3ac4-47ae-848f-fe0ab20784d4                 |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)          |
