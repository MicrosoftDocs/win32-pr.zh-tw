| CN                | From-Server                             |
| Ldap-顯示名稱 | fromServer                              |
| 大小              | \-                                      |
| 更新許可權  | \-                                      |

| 系統識別碼-Guid    | bf967979-0de6-11d0-a285-00aa003049e2    |
