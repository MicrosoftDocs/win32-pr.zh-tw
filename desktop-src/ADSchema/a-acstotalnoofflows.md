指定使用者可能擁有的流程總數。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ACS-總計-無流程                |
| Ldap-顯示名稱 | aCSTotalNoOfFlows                    |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.763               |
| 系統識別碼-Guid    | 7f561280-5301-11d1-a9c5-0000f80367c1 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
