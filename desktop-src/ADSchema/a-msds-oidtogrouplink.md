# <a name="ms-ds-oidtogroup-link-attribute"></a>OIDToGroup-Link 屬性

若為 OID，會識別對應至這個 OID 所代表之發行原則的群組物件。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | OIDToGroup-連結                   |
| Ldap-顯示名稱 | OIDToGroupLink                     |
| 大小              | \-                                      |
| 更新許可權  | \-                                      |
| 更新頻率  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.2051                 |
| 系統識別碼-Guid    | f9c9a57c-3941-438d-bebf-0edaf2aca187    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
