# <a name="default-security-descriptor-attribute"></a>預設值-安全描述項屬性

第一次建立物件時，要指派給該物件的安全描述項。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | 預設-安全性-描述元                 |
| Ldap-顯示名稱 | defaultSecurityDescriptor                   |
| 大小              | \-                                          |
| 更新許可權  | 架構系統管理員                        |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.224                      |
| 系統識別碼-Guid    | 807a6d30-1669-11d0-a064-00aa006c33ed        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
