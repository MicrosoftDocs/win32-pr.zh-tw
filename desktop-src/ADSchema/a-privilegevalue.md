# <a name="privilege-value-attribute"></a>Privilege-Value 屬性

代表 Windows NT 許可權的值。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | Privilege-Value                      |
| Ldap-顯示名稱 | privilegeValue                       |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.635               |
| 系統識別碼-Guid    | 19405b99-3cfa-11d1-a9c0-0000f80367c1 |
| Syntax            | [**間隔**](s-interval.md)       |
