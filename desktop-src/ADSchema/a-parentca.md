# <a name="parent-ca-attribute"></a>父系-CA 屬性

父 CA 的憑證授權單位單位 (CA) 物件的分辨名稱。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | 父系-CA                               |
| Ldap-顯示名稱 | parentCA                                |
| 大小              | \-                                      |
| 更新許可權  | \-                                      |
| 更新頻率  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.557                  |
| 系統識別碼-Guid    | 5245801b-ca6a-11d0-afff-0000f80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
