# <a name="stringnt-sec-desc-syntax"></a>字串 (NT-Sec-Desc) 語法

包含 Windows NT 或 Windows 2000 安全描述項的八位字串。



| 進入 | 值 |
|--------------|------------------------------------------------------------------------------------------------|
| 名稱         | String(NT-Sec-Desc)                                                                            |
| 語法識別碼    | 2.5.5.15                                                                                       |
| OM 識別碼        | 66                                                                                             |
| MAPI 類型    | BINARY                                                                                         |
| ADS 類型     | ADSTYPE \_ NT \_ 安全 \_ 描述項                                                              |
| Variant 類型 | VT \_ 分派                                                                                   |
| SDS 類型     | 可以轉換成 [**IADsSecurityDescriptor**](/windows/desktop/api/iads/nn-iads-iadssecuritydescriptor)的 COM 物件。 |
