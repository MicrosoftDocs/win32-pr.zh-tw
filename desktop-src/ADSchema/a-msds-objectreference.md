# <a name="ms-ds-object-reference-attribute"></a>ms DS-物件參考屬性

物件的連結，該物件會使用儲存在包含這個屬性之物件中的資料。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | ms-DS-物件參考                  |
| Ldap-顯示名稱 | ObjectReference                    |
| 大小              | \-                                      |
| 更新許可權  | \-                                      |
| 更新頻率  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1840                 |
| 系統識別碼-Guid    | 638ec2e8-22e7-409c-85d2-11b21bee72de    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
