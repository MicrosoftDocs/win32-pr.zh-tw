# <a name="msmq-csp-name-attribute"></a>MSMQ-CSP-Name 屬性

MSMQ 使用的密碼編譯提供者類型。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | MSMQ-CSP-名稱                               |
| Ldap-顯示名稱 | mSMQCSPName                                 |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.940                      |
| 系統識別碼-Guid    | 9a0dc334-c100-11d1-bbc5-0080c76670c0        |
| Syntax            | [**String(Teletex)**](s-string-teletex.md) |
