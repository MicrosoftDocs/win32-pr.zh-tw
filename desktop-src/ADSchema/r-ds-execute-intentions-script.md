# <a name="ds-execute-intentions-script-extended-right"></a>DS-執行-意圖-編寫擴充許可權

必須授與資料分割容器的 Control 存取權限，以允許在網域重新命名中使用 Rendom.exe 或準備作業。 此控制項存取權也會在執行 Rendom.exe 或執行步驟作業時，顯示為僅限審核許可權。



| 進入 | 值 |
|--------------|--------------------------------------|
| CN           | DS-執行-意圖-腳本         |
| 顯示名稱 | 執行樹系更新腳本         |
| 許可權-GUID  | 2f16c4a5-b98e-432c-952a-cb388ba33f2e |
