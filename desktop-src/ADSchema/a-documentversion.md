# <a name="documentversion-attribute"></a>documentVersion 屬性

DocumentVersion 屬性類型會指定檔的版本號碼。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | documentVersion                             |
| Ldap-顯示名稱 | documentVersion                             |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.13                  |
| 系統識別碼-Guid    | 94b3a8a9-d613-4cec-9aad-5fbcc1046b43        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
