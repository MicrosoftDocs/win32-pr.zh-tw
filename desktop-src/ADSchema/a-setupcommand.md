# <a name="setup-command-attribute"></a>Setup-Command 屬性

這個屬性會指出設定此應用程式是否需要安裝命令。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | Setup-Command                               |
| Ldap-顯示名稱 | setupCommand                                |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.325                      |
| 系統識別碼-Guid    | 7d6c0e97-7e20-11d0-afd6-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
