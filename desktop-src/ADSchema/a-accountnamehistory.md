|-------------------|---------------------------------------------|
| CN                | 帳戶-名稱-歷程記錄                        |
| Ldap-顯示名稱 | accountNameHistory                          |
| 大小              | \-                                          |
| 更新許可權  | 網域系統管理員                        |

| 系統識別碼-Guid    | 031952ec-3b72-11d2-90cc-00c04fd91ab1        |
