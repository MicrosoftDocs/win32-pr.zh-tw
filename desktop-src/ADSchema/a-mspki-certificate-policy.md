# <a name="ms-pki-certificate-policy-attribute"></a>ms-PKI-憑證-原則屬性

包含在已發行的憑證中的原則 Oid 和其選擇性 Csp 的清單。



| 進入 | 值 |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-憑證-原則                                                                         |
| Ldap-顯示名稱 | Mspki-certificate-name-flag-憑證-原則                                                                          |
| 大小              | 專案數目的64位元組倍。                                                             |
| 更新許可權  | 網域系統管理員                                                                              |
| 更新頻率  | 當您編輯、建立或複製憑證範本時 (ms-PKI-憑證範本) 物件。 |
| Attribute-Id      | 1.2.840.113556.1.4.1439                                                                           |
| 系統識別碼-Guid    | 38942346-cc5b-424b-a7d8-6ffd12029c5f                                                              |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                                       |
