# <a name="netboot-allow-new-clients-attribute"></a>netboot-允許-新增-用戶端屬性

**Netboot-Allow-New-Clients** 屬性已保留供內部使用。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | netboot-允許-新增-用戶端            |
| Ldap-顯示名稱 | netbootAllowNewClients               |
| 大小              | \-                                   |
| 更新許可權  | 此值是由系統所設定。     |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.849               |
| 系統識別碼-Guid    | 07383076-91df-11d1-aebc-0000f80367c1 |
| Syntax            | [**Boolean**](s-boolean.md)         |
