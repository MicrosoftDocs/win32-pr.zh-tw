# <a name="ms-ds-behavior-version-attribute"></a>ms DS-行為-版本屬性

這個屬性是用來追蹤網域或樹系行為版本。 它是用來啟用某些 Active Directory 功能的單純遞增數位。



| 進入 | 值 |
|-------------------|------------------------------------------------|
| CN                | ms DS-行為-版本                         |
| Ldap-顯示名稱 | 行為-版本                          |
| 大小              | \-                                             |
| 更新許可權  | 此值是由網域系統管理員所設定。 |
| 更新頻率  | \-                                             |
| Attribute-Id      | 1.2.840.113556.1.4.1459                        |
| 系統識別碼-Guid    | d31a8757-2447-4545-8081-3bb610cacbf2           |
| Syntax            | [**列舉型別**](s-enumeration.md)           |
