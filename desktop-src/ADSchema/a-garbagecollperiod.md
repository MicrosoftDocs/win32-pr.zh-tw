# <a name="garbage-coll-period-attribute"></a>Coll 期間屬性

這個屬性位於 CN = Directory 服務，CN = Windows NT，CN = Services，CN = Configuration,.。。物件。 它代表 DS 垃圾收集執行之間的時間（以小時為單位）。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 垃圾 Coll 期間                  |
| Ldap-顯示名稱 | garbageCollPeriod                    |
| 大小              | 4 個位元組                              |
| 更新許可權  | 網域系統管理員                 |
| 更新頻率  | 非常罕見。                           |
| Attribute-Id      | 1.2.840.113556.1.2.301               |
| 系統識別碼-Guid    | 5fd424a1-1262-11d0-a060-00aa006c33ed |
| Syntax            | [**列舉型別**](s-enumeration.md) |
