# <a name="acs-max-token-rate-per-flow-attribute"></a>ACS-最大權杖-每 Flow 的屬性

特定使用者的所有單一流程可能具有的最大權杖速率。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | ACS-最大權杖-每 Flow          |
| Ldap-顯示名稱 | aCSMaxTokenRatePerFlow               |
| 大小              | 8 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.758               |
| 系統識別碼-Guid    | 7f56127b-5301-11d1-a9c5-0000f80367c1 |
| Syntax            | [**間隔**](s-interval.md)       |
