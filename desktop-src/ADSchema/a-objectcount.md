# <a name="object-count-attribute"></a>Object-Count 屬性

指定磁片區的追蹤檔案配額。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | Object-Count                         |
| Ldap-顯示名稱 | objectCount                          |
| 大小              | 4 個位元組                              |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.506               |
| 系統識別碼-Guid    | 34aaa216-b699-11d0-afee-0000f80367c1 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
