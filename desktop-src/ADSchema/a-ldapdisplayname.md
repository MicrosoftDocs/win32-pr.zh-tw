# <a name="ldap-display-name-attribute"></a>LDAP-顯示名稱屬性

LDAP 用戶端所使用的名稱，例如 ADSI LDAP 提供者，使用 LDAP 通訊協定來讀取和寫入屬性。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | LDAP-顯示名稱                           |
| Ldap-顯示名稱 | lDAPDisplayName                             |
| 大小              | \-                                          |
| 更新許可權  | 架構系統管理員                        |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.460                      |
| 系統識別碼-Guid    | bf96799a-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
