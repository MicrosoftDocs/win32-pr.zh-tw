# <a name="shadowflag-attribute"></a>shadowFlag 屬性

| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | shadowFlag                           |
| Ldap-顯示名稱 | shadowFlag                           |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.11                     |
| 系統識別碼-Guid    | 8dfeb70d-c5db-46b6-b15e-a4389e6cee9b |
| Syntax            | [**列舉型別**](s-enumeration.md) |
