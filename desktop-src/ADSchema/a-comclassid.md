# <a name="com-classid-attribute"></a>COM-ClassID 屬性

這個屬性會儲存在此應用程式封裝中所執行的 Classid 清單。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | COM-ClassID                                 |
| Ldap-顯示名稱 | cOMClassID                                  |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.19                       |
| 系統識別碼-Guid    | bf96793b-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
