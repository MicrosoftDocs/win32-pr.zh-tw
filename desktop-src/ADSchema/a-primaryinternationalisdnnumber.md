# <a name="phone-isdn-primary-attribute"></a>電話-ISDN-主要屬性

主要 ISDN。



| 進入 | 值 |
|-------------------|----------------------------------------------------------------------------------|
| CN                | 電話-ISDN-主要                                                               |
| Ldap-顯示名稱 | primaryInternationalISDNNumber                                                   |
| 大小              | \-                                                                               |
| 更新許可權  | 網域系統管理員或帳戶擁有者。                                           |
| 更新頻率  | 當使用者的記錄建立時，以及每次電話號碼需要變更時。 |
| Attribute-Id      | 1.2.840.113556.1.4.649                                                           |
| 系統識別碼-Guid    | 0296c11f-40da-11d1-a9c0-0000f80367c1                                             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                      |
