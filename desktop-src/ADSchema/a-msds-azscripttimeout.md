# <a name="ms-ds-az-script-timeout-attribute"></a>ms DS-Az-Script-Timeout 屬性

等候腳本完成審核特定原則的最長時間（以毫秒為單位）。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | ms DS-Az-Script-Timeout                 |
| Ldap-顯示名稱 | AzScriptTimeout                    |
| 大小              | \-                                      |
| 更新許可權  | AzRoles 系統管理員                           |
| 更新頻率  | 在初始化或原則變更期間。 |
| Attribute-Id      | 1.2.840.113556.1.4.1797                 |
| 系統識別碼-Guid    | 87d0fb41-2c8b-41f6-b972-11fdfd50d6b0    |
| Syntax            | [**列舉型別**](s-enumeration.md)    |
