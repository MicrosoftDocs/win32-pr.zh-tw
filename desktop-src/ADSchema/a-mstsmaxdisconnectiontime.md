# <a name="ms-ts-max-disconnection-time-attribute"></a>ms-TS-最大中斷連線時間屬性

終端機服務會話最大中斷連線時間是中斷連線的終端機服務會話在終端機伺服器上維持作用中狀態的最大時間（以分鐘為單位）。

|-------------------|--------------------------------------|
| CN                | ms-TS-Max-Disconnection-Time         |
| Ldap-顯示名稱 | msTSMaxDisconnectionTime             |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1981              |
| 系統識別碼-Guid    | 326f7089-53d8-4784-b814-46d8535110d2 |
