# <a name="short-server-name-attribute"></a>簡短伺服器名稱屬性

適用于列印伺服器的 Windows 2000 相容伺服器名稱。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | 簡短伺服器名稱                           |
| Ldap-顯示名稱 | shortServerName                             |
| 大小              | \-                                          |
| 更新許可權  | 網域系統管理員                        |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1209                     |
| 系統識別碼-Guid    | 45b01501-c419-11d1-bbc9-0080c76670c0        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |



## <a name="implementations"></a>實作
