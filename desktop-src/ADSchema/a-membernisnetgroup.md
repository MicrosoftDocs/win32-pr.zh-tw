|-------------------|--------------------------------------|
| CN                | memberNisNetgroup                    |
| Ldap-顯示名稱 | memberNisNetgroup                    |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.13                     |
| 系統識別碼-Guid    | 0f6a17dc-53e5-4be8-9442-8f3ce2f9012a |
