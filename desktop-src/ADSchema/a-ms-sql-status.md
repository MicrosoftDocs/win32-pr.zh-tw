# <a name="ms-sql-status-attribute"></a>TRANSACT-SQL-Status 屬性

SQL server 的狀態。 如果伺服器正在執行，則狀態為 **TRUE** 。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | MS-SQL-狀態                        |
| Ldap-顯示名稱 | mS-SQL-狀態                        |
| 大小              | 8 個位元組                              |
| 更新許可權  | 此值是由系統所設定。     |
| 更新頻率  | 在系統啟動時。                   |
| Attribute-Id      | 1.2.840.113556.1.4.1380              |
| 系統識別碼-Guid    | 9a7d4770-ccee-11d2-9993-0000f87a57d4 |
| Syntax            | [**間隔**](s-interval.md)       |
