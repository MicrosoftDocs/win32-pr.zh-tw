| CN                | Reps-From                                             |
| Ldap-顯示名稱 | repsFrom                                              |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.2.91                                 |
| 系統識別碼-Guid    | bf967a1d-0de6-11d0-a285-00aa003049e2                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
