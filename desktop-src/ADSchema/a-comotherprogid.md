| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | COM-其他-進程識別碼                           |
| Ldap-顯示名稱 | cOMOtherProgId                              |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.253                      |
| 系統識別碼-Guid    | 281416dd-1968-11d0-a28f-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
