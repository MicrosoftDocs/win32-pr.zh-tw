# <a name="schema-version-attribute"></a>Schema-Version 屬性

架構的版本號碼。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | Schema-Version                       |
| Ldap-顯示名稱 | schemaVersion                        |
| 大小              | 4 個位元組                              |
| 更新許可權  | 此值是由系統所設定。     |
| 更新頻率  | 每次發行架構時。    |
| Attribute-Id      | 1.2.840.113556.1.2.471               |
| 系統識別碼-Guid    | bf967a2c-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
