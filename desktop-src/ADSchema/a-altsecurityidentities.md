# <a name="alt-security-identities-attribute"></a>Alt-Security-識別屬性

針對驗證目的，包含 x.509 憑證或外部 Kerberos 使用者帳戶到此使用者的對應。



| 進入 | 值 |
|-------------------|-----------------------------------------------------|
| CN                | Alt-Security-身分識別                             |
| Ldap-顯示名稱 | altSecurityIdentities                               |
| 大小              | \-                                                  |
| 更新許可權  | 網域系統管理員                                |
| 更新頻率  | 每次需要新的驗證機制時。 |
| Attribute-Id      | 1.2.840.113556.1.4.867                              |
| 系統識別碼-Guid    | 00fbf30c-91fe-11d1-aebc-0000f80367c1                |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)         |
