# <a name="ms-ds-non-members-attribute"></a>ms DS-非成員屬性

這個屬性的用途與非安全性成員屬性相同，但套用了範圍規則。



| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | ms DS-非成員                       |
| Ldap-顯示名稱 | 非成員                         |
| 大小              | \-                                      |
| 更新許可權  | AzRoles 系統管理員                           |
| 更新頻率  | 在初始化和原則變更時。    |
| Attribute-Id      | 1.2.840.113556.1.4.1793                 |
| 系統識別碼-Guid    | cafcb1de-f23c-46b5-adf7-1e64957bd5db    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
