# <a name="package-type-attribute"></a>Package-Type 屬性

此屬性描述應用程式封裝所需的安裝類型，例如 MSI、EXE、CAB。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | Package-Type                         |
| Ldap-顯示名稱 | packageType                          |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.324               |
| 系統識別碼-Guid    | 7d6c0e96-7e20-11d0-afd6-00c04fd930c9 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
