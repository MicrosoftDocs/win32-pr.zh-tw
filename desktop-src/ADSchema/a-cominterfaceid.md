# <a name="com-interfaceid-attribute"></a>COM-InterfaceID 屬性

這個屬性會儲存在此應用程式封裝中執行的介面清單。



| 進入 | 值 |
|-------------------|----------------------------------------------------------------------------------|
| CN                | COM-InterfaceID                                                                  |
| Ldap-顯示名稱 | cOMInterfaceID                                                                   |
| 大小              | \-                                                                               |
| 更新許可權  | 任何人都可以根據所建立物件的安全性來更新這個物件。 |
| 更新頻率  | \-                                                                               |
| Attribute-Id      | 1.2.840.113556.1.4.20                                                            |
| 系統識別碼-Guid    | bf96793c-0de6-11d0-a285-00aa003049e2                                             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                      |
