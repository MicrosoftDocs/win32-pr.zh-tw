| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | MS-TS-ManagingLS2                           |
| Ldap-顯示名稱 | msTSManagingLS2                             |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.2002                     |
| 系統識別碼-Guid    | 349f0757-51bd-4fc8-9d66-3eceea8a25be        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
