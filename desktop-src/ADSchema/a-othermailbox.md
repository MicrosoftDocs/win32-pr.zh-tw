# <a name="other-mailbox-attribute"></a>Other-Mailbox 屬性

在表單中包含其他其他郵寄地址，例如 CCMAIL： BruceKeever。



| 進入 | 值 |
|-------------------|----------------------------------------------------------|
| CN                | Other-Mailbox                                            |
| Ldap-顯示名稱 | otherMailbox                                             |
| 大小              | \-                                                       |
| 更新許可權  | 網域系統管理員                                     |
| 更新頻率  | 每次使用者發出額外的電子郵件地址。 |
| Attribute-Id      | 1.2.840.113556.1.4.651                                   |
| 系統識別碼-Guid    | 0296c123-40da-11d1-a9c0-0000f80367c1                     |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)              |
