# <a name="netboot-answer-only-valid-clients-attribute"></a>netboot-僅回應-有效-用戶端屬性

判斷伺服器是否會回應所有（或僅限預先設置的）用戶端電腦。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | netboot-僅限回應-有效-用戶端    |
| Ldap-顯示名稱 | netbootAnswerOnlyValidClients        |
| 大小              | \-                                   |
| 更新許可權  | 此值是由系統所設定。     |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.854               |
| 系統識別碼-Guid    | 0738307b-91df-11d1-aebc-0000f80367c1 |
| Syntax            | [**Boolean**](s-boolean.md)         |
