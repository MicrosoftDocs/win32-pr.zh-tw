# <a name="msmq-nt4-flags-attribute"></a>MSMQ-Nt4 旗標屬性

**Msmq-Nt4 旗標** 屬性包含 msmq 混合模式資訊。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | MSMQ-Nt4-旗標                       |
| Ldap-顯示名稱 | mSMQNt4Flags                         |
| 大小              | \-                                   |
| 更新許可權  | \-                                   |
| 更新頻率  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.964               |
| 系統識別碼-Guid    | eb38a158-d57f-11d1-90a2-00c04fd91ab1 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
