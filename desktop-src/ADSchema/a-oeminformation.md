|-------------------|---------------------------------------------|
| CN                | OEM-Information                             |
| Ldap-顯示名稱 | oEMInformation                              |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |

| 系統識別碼-Guid    | bf9679ea-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
