# <a name="www-home-page-attribute"></a>WWW-首頁屬性

網站主要登陸頁面的網頁。



| 進入 | 值 |
|-------------------|-----------------------------------------------------------------------------|
| CN                | WWW-首頁                                                               |
| Ldap-顯示名稱 | wWWHomePage                                                                 |
| 大小              | \-                                                                          |
| 更新許可權  | 網域系統管理員或帳戶擁有者。                                      |
| 更新頻率  | 當使用者的記錄建立時，以及每當網頁需要變更時。 |
| Attribute-Id      | 1.2.840.113556.1.2.464                                                      |
| 系統識別碼-Guid    | bf967a7a-0de6-11d0-a285-00aa003049e2                                        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                 |
