# <a name="file-ext-priority-attribute"></a>File-Ext-Priority 屬性

此屬性包含應用程式封裝中的副檔名清單及其相關聯的優先順序。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | File-Ext-Priority                           |
| Ldap-顯示名稱 | fileExtPriority                             |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.816                      |
| 系統識別碼-Guid    | d9e18315-8939-11d1-aebc-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
