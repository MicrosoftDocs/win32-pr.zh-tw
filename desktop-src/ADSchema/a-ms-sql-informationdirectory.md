# <a name="ms-sql-informationdirectory-attribute"></a>InformationDirectory 屬性

SQL Server 目前實例的資訊目錄。



| 進入 | 值 |
|-------------------|------------------------------------------|
| CN                | InformationDirectory              |
| Ldap-顯示名稱 | InformationDirectory              |
| 大小              | \-                                       |
| 更新許可權  | 網域系統管理員                     |
| 更新頻率  | 當系統啟動或更新 AD 時。 |
| Attribute-Id      | 1.2.840.113556.1.4.1392                  |
| 系統識別碼-Guid    | d0aedb2e-ccee-11d2-9993-0000f87a57d4     |
| Syntax            | [**Boolean**](s-boolean.md)             |
