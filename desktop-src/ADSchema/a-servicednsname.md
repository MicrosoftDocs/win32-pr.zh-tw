# <a name="service-dns-name-attribute"></a>服務-DNS-名稱屬性

要查詢以尋找執行此服務之伺服器的 DNS 名稱。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | 服務-DNS-名稱                            |
| Ldap-顯示名稱 | serviceDNSName                              |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.657                      |
| 系統識別碼-Guid    | 28630eb8-41d5-11d1-a9c1-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
