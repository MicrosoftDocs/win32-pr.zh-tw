# <a name="ms-ds-sd-reference-domain-attribute"></a>ms DS-SD-Reference-Domain attribute

| 進入 | 值 |
|-------------------|-----------------------------------------|
| CN                | ms DS-SD-Reference-網域               |
| Ldap-顯示名稱 | SDReferenceDomain                  |
| 大小              | \-                                      |
| 更新許可權  | \-                                      |
| 更新頻率  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1711                 |
| 系統識別碼-Guid    | 4c51e316-f628-43a5-b06b-ffb695fcb4f3    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
