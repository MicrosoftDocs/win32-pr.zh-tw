# <a name="trust-posix-offset-attribute"></a>Trust-Posix-Offset 屬性

適用于受信任網域的可移植作業系統介面 (POSIX) 位移。



| 進入 | 值 |
|-------------------|--------------------------------------|
| CN                | 信任-Posix-Offset                   |
| Ldap-顯示名稱 | trustPosixOffset                     |
| 大小              | 4 個位元組                              |
| 更新許可權  | 此值是由系統所設定。     |
| 更新頻率  | 當新的信任建立時。         |
| Attribute-Id      | 1.2.840.113556.1.4.134               |
| 系統識別碼-Guid    | bf967a5e-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**列舉型別**](s-enumeration.md) |
