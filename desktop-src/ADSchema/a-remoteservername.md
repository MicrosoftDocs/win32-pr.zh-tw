# <a name="remote-server-name-attribute"></a>遠端伺服器名稱屬性

在必須儲存一或多個電腦名稱稱的地方使用。



| 進入 | 值 |
|-------------------|---------------------------------------------|
| CN                | 遠端伺服器名稱                          |
| Ldap-顯示名稱 | remoteServerName                            |
| 大小              | \-                                          |
| 更新許可權  | \-                                          |
| 更新頻率  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.105                      |
| 系統識別碼-Guid    | bf967a12-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
