# <a name="control-access-rights-attribute"></a>控制項存取權屬性

由 DS 安全性用來判斷哪些使用者可以在主機物件上執行特定作業。



| 進入 | 值 |
|-------------------|-------------------------------------------------------|
| CN                | 控制存取權                                 |
| Ldap-顯示名稱 | controlAccessRights                                   |
| 大小              | \-                                                    |
| 更新許可權  | \-                                                    |
| 更新頻率  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.200                                |
| 系統識別碼-Guid    | 6da8a4fc-0e52-11d0-a286-00aa003049e2                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
