# <a name="vml-h-element"></a>VML H 元素

本主題說明 VML，這是 Windows Internet Explorer 9 淘汰的功能。 依賴 VML 的網頁和應用程式應該遷移至 SVG 或其他廣泛支援的標準。

> [!Note]  
> 從2011年12月起，本主題已封存。 因此，它不會再主動維護。 如需詳細資訊，請參閱封存的 [內容](/previous-versions/windows/internet-explorer/ie-developer/)。 如需目前 Windows Internet Explorer 版本的相關資訊、建議和指引，請參閱 [Internet Explorer 開發人員中心](https://msdn.microsoft.com/ie/)。

 

定義圖形的控制碼。

下列屬性會修改控制碼。



| 屬性                                                | 描述                                                    |
|----------------------------------------------------------|----------------------------------------------------------------|
| [InvX](msdn-online-vml-invx-attribute.md)               | 判斷控制碼的 *x* 位置是否反轉。 |
| [InvY](msdn-online-vml-invy-attribute.md)               | 判斷控制碼的 *y* 位置是否反轉。 |
| [地圖](msdn-online-vml-map-attribute.md)                 | 定義控制碼的對應範圍。                         |
| [極座標圖](msdn-online-vml-polar-attribute.md)             | 定義極座標圖的中心位置。                 |
| [位置](position-attribute--h--vml.md)               | 指定控制碼的 *x* 和 *y* 值。                |
| [RadiusRange](msdn-online-vml-radiusrange-attribute.md) | 定義極座標控點的範圍。                           |
| [開關](msdn-online-vml-switch-attribute.md)           | 決定是否要交換控制碼方向。          |
| [XRange](msdn-online-vml-xrange-attribute.md)           | 定義控制碼的 *x* 範圍。                             |
| [YRange](msdn-online-vml-yrange-attribute.md)           | 定義控制碼的 *y* 範圍。                             |



 

**備註**

這個元素必須定義在 [Shape](shape-element--vml.md) 元素內。
