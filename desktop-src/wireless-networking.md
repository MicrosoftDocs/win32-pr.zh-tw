| 主題                                                                          | 描述                                                                                                                                                                                                                                                                                                     |
|--------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Bluetooth](./bluetooth/bluetooth-start-page.md)<br/>                         | 藍牙是業界標準的通訊協定，可讓電腦、掌上型裝置、行動電話和其他裝置使用無線連線能力。<br/>                                                                                                                                                     |
| [紅外線資料關聯 (IrDA) ](/previous-versions/windows/desktop/irda/irda-start-page)<br/>            | 紅外線資料關聯 (IrDA) 是一種通訊協定套件，其設計目的是要在裝置之間提供無線、連線的連線能力。 <br/>                                                                                                                                                                      |
| [行動寬頻](./mbn/mobile-broadband-networks-portal.md)<br/>            | 行動寬頻 API 是用來執行對行動電話通訊網路的連線能力。 應用程式不應該直接與這類行動寬頻裝置通訊。 相反地，他們必須使用 Mobile 寬頻 API。 <br/>                                                                                  |
| [原生 Wifi](./nativewifi/portal.md)<br/>                                         | 原生 Wifi 自動設定元件會設定、連線和中斷連線無線網路。 原生 Wifi 可將設定檔儲存在與其互動的網路上，並以 XML 檔的形式儲存。 <br/>                                                                                   |
| [Windows Connect Now](./wcn/portal.md)<br/>                                   | Windows Connect Now (的 WCN) 允許行動裝置和 embedded 裝置、802.11 存取點 (Ap) 和電腦，以安全地連線，以及彼此交換設定。 WCN 是針對家用或小型企業使用者所設計，可在容易使用和健全的安全性之間提供合理的折衷。 <br/> |
| [Windows 連線管理員](./wcm/windows-connection-manager-portal.md)<br/> | Windows 連線管理員 (WCM) 可讓您建立及設定連線管理員軟體。<br/>                                                                                                                                                                                              |
