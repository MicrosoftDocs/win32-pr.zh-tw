# <a name="sensor_category_orientation"></a>感應器 \_ 類別 \_ 方向

[感應器 \_ 類別] \_ 方向類別包含的感應器可提供實體方向的相關資訊。 羅盤會提供導覽方向，例如根據磁性北部的方向。 Inclinometers 測量斜率或提高許可權。 距離感應器會測量某個物件與感應器的相近度。
