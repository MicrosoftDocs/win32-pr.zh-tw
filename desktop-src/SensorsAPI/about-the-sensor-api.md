# <a name="about-the-sensor-api"></a>關於感應器 API

感應器 API 會透過一組 COM 介面提供其功能。 感應器 API 會公開其中一些介面，這表示您可以在其上呼叫方法。 其他介面會由 API 定義，讓您可以執行感應器 API 為程式提供事件通知所需的方法。
