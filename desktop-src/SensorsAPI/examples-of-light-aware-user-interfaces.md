# <a name="examples-of-light-aware-user-interfaces"></a>Light-Aware 使用者介面的範例

現在您已瞭解針對不同光源條件優化 UI 的一些基本原則，您可以查看一些範例，說明這項優化在戶外觀賞內容時的差異。 下列影像是直接日光的膝上型電腦並存比較。

下列影像會將燈光感知的 UI 與膝上型電腦上一般 UI 的直接照明比較。 第一張相片顯示當膝上型電腦使用電池使用量顯示設定時，哪些內容通常看起來像戶外。 第二張相片會顯示彈性亮度和淺感知 UI 的組合，以及這些功能如何一起運作，以增加螢幕可讀性。

![處於40% 亮度的一般 ui](images/ui-unaware.png)![以100% 亮度的輕感知 ui](images/ui-lightaware.png)

下列相片顯示的是已關閉並開啟明亮感知的戶外流覽程式。 室內內容不是戶外可用，而室外內容很容易辨認。 另請注意，使用「黑白」色彩配置時的反射會如何最小化，如第二張相片所示。
