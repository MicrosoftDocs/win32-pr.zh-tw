# <a name="com-interfaces-sensor-api"></a>COM 介面 (感應器 API) 

感應器 API 會公開下列 COM 介面。



| 介面                                              | 描述                                                                                                      |
|--------------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [**ILocationPermissions**](/windows/desktop/api/sensorsapi/nn-sensorsapi-ilocationpermissions)   | 提供可讓使用者變更位置設定之系統設定的狀態。                         |
| [**ILogicalSensorManager**](/previous-versions/windows/desktop/legacy/dd318934(v=vs.85)) | 提供邏輯感應器作者用來管理邏輯感應器連接的方法。                            |
| [**ISensor**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensor)                             | 代表感應器。                                                                                             |
| [**ISensorCollection**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensorcollection)         | 代表感應器的集合。                                                                              |
| [**ISensorDataReport**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensordatareport)         | 代表感應器資料包表。                                                                                 |
| [**ISensorEvents**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensorevents)                 | 接收感應器事件的回呼介面。                                                              |
| [**ISensorManager**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensormanager)               | 提供用來探索和取出可用感應器的方法，以及要求感應器管理員事件的方法。 |
| [**ISensorManagerEvents**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensormanagerevents)   | 用來接收感應器管理員事件的回呼介面。                                                      |
