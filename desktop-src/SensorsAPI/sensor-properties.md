# <a name="sensor-properties"></a>感應器屬性

感應器和位置平臺會定義用來識別感應器屬性的常數。 感應器製造商也可以定義自己的屬性。

平臺會定義下列感應器屬性的 **PROPERTYKEY** 值。 除非另有說明，否則這些屬性都是唯讀的。

每個平臺定義的感應器屬性 **PROPERTYKEY** 都是以名為「感應器 \_ 屬性 \_ 一般 guid」的一般 guid 為基礎 \_ ：

{7F8383EC-D3EC-495C-A8CF-B8BBE85C2920}.

> [!IMPORTANT]
> 請勿使用這個基底值來定義您自己的屬性索引鍵。

 

指定為讀取/寫入的屬性值可以由用戶端應用程式指定。 指定為 static 的屬性值不能隨著時間而變更。 感應器必須支援指定為 [必要] 的屬性。
