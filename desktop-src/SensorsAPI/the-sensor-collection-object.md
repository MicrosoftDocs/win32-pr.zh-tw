# <a name="the-sensor-collection-object"></a>感應器集合物件

感應器集合物件包含一組感應器物件。

有時候您會想要以群組的形式來使用感應器。 例如，您可能會想要使用感應器管理員來探索所有可用的移動感應器。 感應器集合物件提供使用感應器物件群組的方法。 您可以使用 [**ISensorCollection**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensorcollection) 介面來呼叫方法，以計算、新增、移除和取出集合中的感應器。
