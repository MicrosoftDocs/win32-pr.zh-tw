# <a name="privacy-and-security-in-the-windows-sensor-and-location-platform"></a>Windows 感應器和位置平台的隱私權與安全性

Windows 的感應器和位置平臺包含隱私權設定，以協助保護使用者的個人資訊。

此平臺可協助確保感應器資料在需要隱私權時維持私密，方法如下：

-   感應器預設為關閉。 由於平臺設計會假設任何感應器都可以提供個人資料，因此每個感應器都會停用，直到使用者明確同意存取感應器資料為止。
-   Windows 提供洩漏訊息和使用者的說明內容。 此內容可協助使用者瞭解感應器會如何影響其個人資料的隱私權，並協助使用者做出明智的決策。
-   提供感應器的許可權需要系統管理員許可權。
-   啟用時，感應器裝置適用于在特定使用者帳戶下執行的所有程式 (或所有使用者帳戶) 。 這包括非互動式使用者和服務，例如 ASPNET 或 SYSTEM。 例如，如果您為使用者帳戶啟用 GPS 感應器，則只有在您的使用者帳戶下執行的程式才能存取 GPS。 如果您為所有使用者啟用 GPS，則在任何使用者帳戶下執行的任何程式都可以存取 GPS。
-   使用感應器的程式可以呼叫方法來開啟系統對話方塊，該對話方塊會提示使用者啟用所需的感應器裝置。 這項功能可讓開發人員和使用者在程式需要時輕鬆地確保感應器能正常運作，同時保有使用者對感應器資料洩漏的控制權。
-   感應器驅動程式會使用特殊物件來處理所有 i/o 要求。 此物件可確保只有具有使用者權限的程式可以存取感應器資料。
