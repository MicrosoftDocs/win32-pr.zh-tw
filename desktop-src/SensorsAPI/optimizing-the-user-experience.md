# <a name="optimizing-the-user-experience"></a>優化使用者體驗

當您在執行燈光感知的使用者介面時，您應該仔細考慮使用者如何回應程式的行為。 避免 jarring 轉換或頻繁變更程式的內容和使用者介面。 只有在需要時才會進行順暢且漸進的轉換。 在理想的情況下，您的程式應該以真實世界的照明條件和使用者案例中的使用者進行測試。 最後，請考慮建立一個機制，讓使用者手動變更程式的輕量優化或停用功能。
