# <a name="sensor-api-glossary"></a>感應器 API 詞彙

<dl> <dt>

<span id="winsensors.sensors_glossary_ambient_light_sensor"></span><span id="WINSENSORS.SENSORS_GLOSSARY_AMBIENT_LIGHT_SENSOR"></span>**環境光線感應器**
</dt> <dd>

偵測目前光源層級的感應器。

</dd> <dt>

<span id="winsensors.sensors_glossary_geodetic_reference_system"></span><span id="WINSENSORS.SENSORS_GLOSSARY_GEODETIC_REFERENCE_SYSTEM"></span>**geodetic 參考系統**
</dt> <dd>

橢圓體的組合，指定地球的大小和形狀，以及參考所有其他點之緯度和經度的基底點。

</dd> <dt>

<span id="winsensors.sensors_glossary_light-aware_user_interface__ui_"></span><span id="WINSENSORS.SENSORS_GLOSSARY_LIGHT-AWARE_USER_INTERFACE__UI_"></span>**輕感知的使用者介面 (UI)**
</dt> <dd>

此程式會使用環境光線感應器資料，將其使用者介面和內容優化，以進行特定的光源條件。

</dd> <dt>

<span id="winsensors.sensors_glossary_logical_id"></span><span id="WINSENSORS.SENSORS_GLOSSARY_LOGICAL_ID"></span>**邏輯識別碼**
</dt> <dd>

識別特定邏輯感應器 (GUID) 的全域唯一識別碼。 每次連接到邏輯感應器時，都需要邏輯識別碼。

</dd> <dt>

<span id="winsensors.sensors_glossary_logical_sensor"></span><span id="WINSENSORS.SENSORS_GLOSSARY_LOGICAL_SENSOR"></span>**邏輯感應器**
</dt> <dd>

在不使用硬體裝置的情況下提供資料的感應器。 例如，邏輯感應器可以使用在資料表中查閱 IP 位址的服務，來提供使用者目前位置的相關資料。

</dd> <dt>

<span id="winsensors.sensors_glossary_lux_value"></span><span id="WINSENSORS.SENSORS_GLOSSARY_LUX_VALUE"></span>**lux 值**
</dt> <dd>

照亮表面的光線量（以每個平方英尺的流明為單位）。 環境光線感應器提供 lux 值的資料。

</dd> <dt>

<span id="winsensors.sensors_glossary_reference_ellipsoid"></span><span id="WINSENSORS.SENSORS_GLOSSARY_REFERENCE_ELLIPSOID"></span>**參考橢圓體**
</dt> <dd>

與 geodetic 參考系統相關聯的橢圓體，其介面近似于地球的大小和形狀。

</dd> <dt>

<span id="winsensors.sensors_glossary_sensor"></span><span id="WINSENSORS.SENSORS_GLOSSARY_SENSOR"></span>**感應器**
</dt> <dd>

可以偵測到外來事件或環境狀況的硬體或軟體。 例如，全球定位系統 (GPS) 接收器是可判斷使用者目前實體位置的感應器。

</dd> <dt>

<span id="winsensors.sensors_glossary_sensor_category"></span><span id="WINSENSORS.SENSORS_GLOSSARY_SENSOR_CATEGORY"></span>**感應器類別**
</dt> <dd>

廣泛的感應器裝置類別。 例如，影片遊戲控制器中的感應器，可偵測玩家手的位置和移動屬於感應器的方向類別。

</dd> <dt>

<span id="winsensors.sensors_glossary_sensor_collection"></span><span id="WINSENSORS.SENSORS_GLOSSARY_SENSOR_COLLECTION"></span>**感應器集合**
</dt> <dd>

可以用來做為群組的多個感應器。 例如，感應器集合可以包含所有可用的動作感應器，而且可以呼叫方法來抓取此集合。

</dd> <dt>

<span id="winsensors.sensors_glossary_sensor_data_report"></span><span id="WINSENSORS.SENSORS_GLOSSARY_SENSOR_DATA_REPORT"></span>**感應器資料包表**
</dt> <dd>

包含由感應器產生之資料欄值的報表。

</dd> <dt>

<span id="winsensors.sensors_glossary_sensor_device"></span><span id="WINSENSORS.SENSORS_GLOSSARY_SENSOR_DEVICE"></span>**感應器裝置**
</dt> <dd>

可以偵測到外來事件或環境狀況的硬體裝置。 單一硬體裝置可以包含多個感應器。

</dd> <dt>

<span id="winsensors.sensors_glossary_sensor_manager"></span><span id="WINSENSORS.SENSORS_GLOSSARY_SENSOR_MANAGER"></span>**感應器管理員**
</dt> <dd>

程式設計物件，可維護可用感應器的清單並提供其存取權。

</dd> <dt>

<span id="winsensors.sensors_glossary_sensor_type"></span><span id="WINSENSORS.SENSORS_GLOSSARY_SENSOR_TYPE"></span>**感應器類型**
</dt> <dd>

特定種類的感應器，例如三維加速計。

</dd> </dl>
