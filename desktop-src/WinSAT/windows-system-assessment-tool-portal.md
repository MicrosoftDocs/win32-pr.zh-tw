# <a name="windows-system-assessment-tool-provider"></a>Windows系統評定工具提供者

\[在 Windows 8.1 之後，可能會變更或無法使用 WinSAT。\]

## <a name="purpose"></a>目的

Windows 系統評定工具 (WinSAT) 會公開一些可評估電腦效能特性和功能的類別。 開發人員可以使用此 API 來開發可存取電腦效能和功能資訊的軟體，以根據該電腦的效能功能來判斷最佳的應用程式設定。

使用者可以使用正式評量的結果，判斷哪些案例和應用程式會在電腦上順利執行。 例如，如果軟體套件包含其封裝的評等，則使用者可以使用評等來判斷軟體是否可在電腦上順利執行。

## <a name="developer-audience"></a>開發人員對象

WinSAT 是針對 c、c + + 及 Visual Basic 開發人員和撰寫腳本的開發人員所設計。

## <a name="run-time-requirements"></a>執行階段需求求

從 Windows Vista 作業系統開始，用戶端電腦上可以使用 WinSAT。
