# <a name="winsat-mof-classes"></a>WinSAT MOF 類別

\[WinSAT MOF 類別可能會在 Windows 8.1 之後變更或無法使用。\]

Windows 系統評定工具 (WinSAT) 會定義下列 Windows Management Instrumentation () MOF 受控物件格式類別（存在於 \\ 根 cimv2 命名空間） () MOF 類別 \\ ：

-   [**Win32 \_ WinSAT**](win32-winsat.md)

您可以使用這個類別來取得最新正式評量的摘要資訊。
