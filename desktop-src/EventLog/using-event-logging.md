# <a name="using-event-logging"></a>使用事件記錄

應用程式可以使用事件記錄 API 來報告和查看事件。 如需報告事件的詳細資訊，請參閱 [報告事件](reporting-an-event.md)。

若要查看已寫入記錄檔的事件，請參閱 [查詢事件來源訊息](querying-for-event-source-messages.md) 和 [接收事件通知](receiving-event-notification.md)。 您也可以使用事件檢視器來查看事件。
