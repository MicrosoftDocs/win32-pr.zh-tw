## <a name="comments"></a><span data-ttu-id="57f21-113">註解</span><span class="sxs-lookup"><span data-stu-id="57f21-113">Comments</span></span>

```

<span data-ttu-id="57f21-117">針對區塊批註，以分號開始每一行，然後將 C/c + + open block 批註指標 (/) 放在 \* 第一行的分號後面，並在 \* 最後一行的分號之後 (/) 。</span><span class="sxs-lookup"><span data-stu-id="57f21-117">For a block comment, begin each line with a semicolon, then place a C/C++ open block comment indicator (/\*) after the semicolon on the first line and the close block comment indicator (\*/) after the semicolon on the last line.</span></span>

;*/
```
