# <a name="win32_componentcategory-class"></a>Win32 \_ ComponentCategory 類別

**Win32 \_ ComponentCategory** [WMI 類別](/windows/desktop/WmiSdk/retrieving-a-class)代表元件類別。 元件類別是元件物件模型的群組， (COM) 類別，並在兩者之間共用已定義的功能集。 使用這些介面的用戶端會查詢登錄中的分類標題和唯一識別碼 **，稱為 [類別目錄**]，這是從全域唯一識別碼 (GUID) 所建立。

下列語法已經過受管理物件格式 (MOF) 程式碼簡化，並包含所有已繼承的屬性。 屬性會依字母順序列出，而不是依 MOF 順序列出。

## <a name="syntax"></a>語法

``` syntax
[Dynamic, Provider("CIMWin32"), UUID("{0F73ED5A-8ED9-11d2-B340-00105A1F8569}"), AMENDMENT]
class Win32_ComponentCategory : CIM_LogicalElement
{
  string   Caption;
  string   Description;
  datetime InstallDate;
  string   Status;
  string   CategoryId;
  string   Name;
};
```
