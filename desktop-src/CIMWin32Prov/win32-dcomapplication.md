# <a name="win32_dcomapplication-class"></a>Win32 \_ DCOMApplication 類別

**Win32 \_ DCOMApplication** [WMI 類別](/windows/desktop/WmiSdk/retrieving-a-class)代表 DCOM 應用程式的屬性。

下列語法已經過受管理物件格式 (MOF) 程式碼簡化，並包含所有已繼承的屬性。 屬性會依字母順序列出，而不是依 MOF 順序列出。

## <a name="syntax"></a>語法

``` syntax
[Dynamic, Provider("CIMWin32"), UUID("{0F73ED52-8ED9-11d2-B340-00105A1F8569}"), AMENDMENT]
class Win32_DCOMApplication : Win32_COMApplication
{
  string   Caption;
  string   Description;
  datetime InstallDate;
  string   Name;
  string   Status;
  string   AppID;
};
```
