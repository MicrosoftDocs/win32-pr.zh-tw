# <a name="power-management-event-provider"></a>電源管理事件提供者

電源管理提供者可讓 WMI 建立 Windows 電源管理通訊協定的模型。 作為事件提供者，電源管理提供者會將資訊提供給 [**Win32 \_ PowerManagementEvent**](win32-powermanagementevent.md) 類別，以描述電源狀態變更所造成的電源管理事件。 這些狀態變更與 advanced 電源管理 (APM) 或 advanced configuration and power interface (ACPI) 系統管理通訊協定相關聯。 [**\_ \_ Win32Provider**](/windows/desktop/WmiSdk/--win32provider)實例名稱是「MS \_ 電源 \_ 管理 \_ 事件 \_ 提供者」。
