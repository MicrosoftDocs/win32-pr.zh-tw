|---|---|
| [筆跡展示者](ink-presenter.md)<br/> | 筆墨演講者 Api 可讓 Microsoft Win32 應用程式透過插入應用程式 [DirectComposition](../directcomp/directcomposition-portal.md)視覺化樹狀結構的 [**InkPresenter**](/uwp/api/Windows.UI.Input.Inking.InkPresenter)物件，來管理筆墨輸入的輸入、處理和轉譯 (標準和修改過的) 。<br/> |
