# <a name="file-management-local-file-systems"></a>檔案管理 (本機檔案系統) 

*File 物件* 提供資源的表示 (實體裝置或實體裝置上的資源，可由 i/o 系統管理) 。 就像其他物件一樣，它們會啟用資源的共用、其名稱、以物件為基礎的安全性加以保護，而且支援同步處理。 I/o 系統也可讓您讀取或寫入資源。
