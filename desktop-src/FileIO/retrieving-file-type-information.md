# <a name="retrieving-file-type-information"></a>正在取出檔案類型資訊

[**GetFileType**](/windows/desktop/api/FileAPI/nf-fileapi-getfiletype)函式會抓取檔案的類型：磁片、字元 (例如主控台) 、管道或不明。 [**GetBinaryType**](/windows/desktop/api/WinBase/nf-winbase-getbinarytypea)函式會判斷檔案是否為可執行檔，如果是，則是可執行檔的類型。 如需支援的可執行檔案類型清單，請參閱 **GetBinaryType** 。
