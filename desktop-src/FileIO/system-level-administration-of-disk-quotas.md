# <a name="system-level-administration-of-disk-quotas"></a>磁片配額的系統層級管理

系統管理員可以設定磁片區上特定使用者的配額。 系統管理員也可以設定磁片區的預設配額。 除非系統管理員特別針對該使用者建立配額，否則磁片區上的新使用者會收到預設配額。

系統管理員可以查詢配額追蹤和強制執行 (或配額狀態的層級) 、預設配額限制，以及每個使用者的配額資訊。 每個使用者的配額資訊包含使用者的固定配額限制、警告閾值和配額使用量。 系統管理員也可以啟用或停用配額強制。

有三個配額狀態，如下表所示。



| 州          | 描述                                                                                                                                                                              |
|----------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 停用配額 | 不會追蹤配額使用量變更，但不會移除配額限制。 在此狀態下，效能不會受到磁片配額的影響。 這是預設狀態。                         |
| 已追蹤配額  | 系統會追蹤配額使用量變更，但不會強制執行配額限制。 處於此狀態時，不會產生任何配額違規事件，而且因為磁片配額違規，所以沒有任何檔案作業失敗。 |
| 已強制執行配額 | 系統會追蹤配額使用量變更，並強制執行配額限制。                                                                                                                           |
