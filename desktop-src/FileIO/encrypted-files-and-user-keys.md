# <a name="encrypted-files-and-user-keys"></a>加密檔案和使用者金鑰

若要為使用者建立新的金鑰，請使用 [**SetUserFileEncryptionKey**](/windows/desktop/api/Winefs/nf-winefs-setuserfileencryptionkey) 函數。 若要將使用者金鑰新增至加密的檔案，請使用 [**AddUsersToEncryptedFile**](/windows/desktop/api/Winefs/nf-winefs-adduserstoencryptedfile) 函數。 若要查詢加密檔案的使用者金鑰，請使用 [**QueryUsersOnEncryptedFile**](/windows/desktop/api/Winefs/nf-winefs-queryusersonencryptedfile) 函數。 若要從加密的檔案中移除使用者金鑰，請使用 [**RemoveUsersFromEncryptedFile**](/windows/desktop/api/Winefs/nf-winefs-removeusersfromencryptedfile) 函數。
