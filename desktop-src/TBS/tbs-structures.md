| 主題                                                           | 描述                                                                                                                                          |
|-----------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**TB \_ 內容 \_ 參數**](/windows/win32/api/tbs/ns-tbs-tbs_context_params)<br/>   | 指定 TB 內容執行的版本。<br/>                                                                                  |
| [**TB \_ 內容 \_ PARAMS2**](/windows/win32/api/tbs/ns-tbs-tbs_context_params2)<br/> | 指定 TB 內容執行的版本。 如果您的應用程式適用于這兩個版本的 TPM，您必須使用此結構。<br/> |
| [**TPM \_ 裝置 \_ 資訊**](/windows/desktop/api/Tbs/ns-tbs-tpm_device_info)<br/>         | 提供 TPM 版本的相關資訊。<br/>                                                                                        |
