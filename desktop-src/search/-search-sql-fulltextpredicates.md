# <a name="full-text-predicates"></a>Full-Text 述詞

Microsoft Windows Search 查詢語言支援兩個全文檢索搜尋述詞。 CONTAINS 述詞會對包含文字的資料行執行比較。 CONTAINS 子句可以根據搜尋詞彙的相近程度，執行單一單字或片語的比對。 相較之下，FREETEXT 述詞經過調整，以符合搜尋片語對文字資料行的意義。

本節包含下列主題：

-   [CONTAINS 述詞](-search-sql-contains.md)
-   [FREETEXT 述詞](-search-sql-freetext.md)
