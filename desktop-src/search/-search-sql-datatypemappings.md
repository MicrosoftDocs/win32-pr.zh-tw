| VT \_ CY            | DBTYPE \_ CY       |
| VT \_ 日期          | DBTYPE \_ 日期     |

| VT \_ FILETIME      | DBTYPE \_ FILETIME |
| VT \_ GUID          | DBTYPE \_ GUID     |
| VT \_ I1            | DBTYPE \_ I1       |
| VT \_ I2            | DBTYPE \_ I2       |
| VT \_ I4            | DBTYPE \_ I4       |
| VT \_ I8            | DBTYPE \_ I8       |
| VT \_ Null          | DBTYPE \_ Null     |
| VT \_ R4            | DBTYPE \_ R4       |
| VT \_ R8            | DBTYPE \_ UI8      |
| VT \_ 向量        | DBTYPE \_ 向量   |
| VT \_ WSTR          | DBTYPE \_ WSTR     |
