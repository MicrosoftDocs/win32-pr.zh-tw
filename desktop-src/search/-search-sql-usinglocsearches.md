# <a name="using-localized-searches"></a>使用當地語系化的搜尋

Microsoft Windows Search 支援完整的 Unicode 搜尋。 您可以為使用 Unicode 的檔編制索引，並建立使用 Unicode 識別碼和常值的搜尋查詢。 但是，應用程式或作業系統的語言設定可能會對 Windows Search 的運作方式產生最多的影響。

本節包含下列主題：

-   [檔和系統地區設定設定](-search-sql-docandlocalesystemsettings.md)
-   [指定語言](-search-sql-specifyinglanguages.md)
