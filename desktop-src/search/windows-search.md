# <a name="windows-search"></a>Windows 搜尋

Windows Search 是一種桌面搜尋平臺，可針對最常見的檔案和資料類型（例如電子郵件、連絡人、行事曆約會、檔、相片、多媒體和其他可由協力廠商開發人員擴充的格式）提供即時搜尋功能。 這些功能可讓使用者尋找、管理和組織家用和企業環境中常見的資料量。
