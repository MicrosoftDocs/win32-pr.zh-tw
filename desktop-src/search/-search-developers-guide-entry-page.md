# <a name="windows-search-developers-guide"></a>Windows搜尋開發人員指南

協力廠商可以建立應用程式，以程式設計方式查詢資料的索引，並可延伸 Windows Search 從自訂檔案格式和資料存放區索引資料。 若要建立 Windows Search 的應用程式，協力廠商開發人員必須先將 Shell 資料存放區實作為，才能達到合理的使用者體驗。 如需詳細資訊，請參閱 [執行基本資料夾物件介面](/previous-versions/windows/desktop/legacy/cc144093(v=vs.85))。

您也必須下載 Windows Search 程式庫的[Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx) 。 [Windows Search SDK 範例](https://www.microsoft.com/downloads/details.aspx?FamilyID=645300AE-5E7A-4CE7-95F0-49793F8F76E8)包含實用的程式碼範例，以及使用 managed 程式碼進行開發的 interopability 元件。 如需使用程式碼範例的詳細資訊，請參閱[Windows Search 程式碼範例](-search-3x-wds-sampleentry.md)。

本節的組織方式如下：

- [管理索引](-search-3x-wds-mngidx-overview.md)
- [以程式設計方式查詢索引](-search-3x-wds-qryidx-overview.md)
- [擴充索引](-search-3x-wds-extidx-overview.md)
- [擴充語言資源](extending-language-resources-in-windows-search.md)
