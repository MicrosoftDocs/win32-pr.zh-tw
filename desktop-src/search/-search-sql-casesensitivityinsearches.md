# <a name="case-sensitivity-in-searches"></a>搜尋中的區分大小寫

Microsoft Windows Search 查詢不區分大小寫。 這包括查詢關鍵字 (SELECT 等同于 Select、select 和 sELect) 和搜尋字詞。 如果識別碼或常值使用具有大小寫對應語義的 Unicode 字元，則搜尋引擎會將所有案例視為相等。
