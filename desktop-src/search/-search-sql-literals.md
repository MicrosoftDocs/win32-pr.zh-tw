# <a name="literals"></a>常值

常值是代表查詢語句中值的字元字串。 您可以使用常值來比較資料行的值，或指定搜尋詞彙。 Windows Search 支援下列類型的常值。


-   **字串常** 值可以是任何長度，而且可以包含 ANSI 或 Unicode 字元。 您必須以單引號括住字串常值 ( ' ) 。 若要在字串常值內加入單引號，請使用兩個單引號 ( ' ' ) 。 將空字串表示為兩個連續的單引號 ( ' ' ) 。
-   **數值常** 值可以包含數位0-9、句號，以及字母 E (或 E) 。 數值常值代表數位，包括正數和負整數、十進位數和貨幣值。 您可以使用科學記號標記法來定義數值常值 (例如 2.3 E-05) 。 請勿將數值常值括在單引號中，否則會被視為字串常值，並使用字串比較技巧進行比較。 貨幣值不能包含貨幣符號。
-   **十六進位常** 值可以包含數位0-9 和字母 a-f 和 a-f。 十六進位常值表示以十六進位標記法指定的不帶正負號的整數。 十六進位常值的開頭必須是0x。
    > [!Note]  
    > SQL-92 標準要求以單引號括住十六進位常值;不過，Windows Search 不支援該標記法。

     

-   **布林常** 值代表邏輯值，而且可以是 **TRUE** 或 **FALSE**。 請勿將布林值常值括在單引號中，或將它解釋為字串常值。
-   **日期常** 值代表特定日期、時間戳記或相對時間，並以單引號括住。 您必須將日期以年/月/日時數：分：秒或年-月-日時數：分鐘：秒，其中 month、day 和 year 是數位。 以四位數值指定年份，例如2004。 時間值的格式必須為小時：分鐘：秒。 相對時間語法是以 [DATEADD 函數](-search-sql-dateadd.md)為基礎。
