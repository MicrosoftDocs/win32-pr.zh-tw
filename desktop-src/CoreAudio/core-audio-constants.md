# <a name="core-audio-constants"></a>核心音訊常數

本節說明 Windows Vista 和更新版本中核心音訊 api 所使用的常數。

| 常數                                                                            | 描述                                                                         |
|-------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------|
| [**AUDCLNT \_ SESSIONFLAGS \_ XXX**](audclnt-sessionflags-xxx-constants.md)<br/> | 在串流初始化期間開啟之音訊會話的旗標。<br/> |
| [**AUDCLNT \_ STREAMFLAGS \_ XXX**](audclnt-streamflags-xxx-constants.md)              | 音訊串流的初始化旗標。                                           |
| [**裝置 \_ 狀態 \_ XXX**](device-state-xxx-constants.md)                            | 音訊端點裝置的狀態旗標。                                           |
| [**端點 \_ 硬體 \_ 支援 \_ XXX**](endpoint-hardware-support-xxx-constants.md) | 音訊端點裝置的硬體支援旗標。                                |
