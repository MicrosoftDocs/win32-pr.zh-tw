# <a name="pkey_mfx_processingmodes_supported_for_streaming-property"></a>PKEY \_ MFX \_ ProcessingModes \_ 支援 \_ \_ 串流屬性

列出模式效果 APO 所支援的信號處理模式。

PROPVARIANT 類型 (vt) ： **vt \_ 向量** \| **vt \_ LPWSTR**

## <a name="remarks"></a>備註

此清單只包含信號處理模式，APO 會在串流期間實際處理音訊信號。

這份清單不包含 APO 針對探索用途所支援的任何信號處理模式。
