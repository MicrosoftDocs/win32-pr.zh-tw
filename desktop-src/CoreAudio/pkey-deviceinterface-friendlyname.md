# <a name="pkey_deviceinterface_friendlyname"></a>PKEY \_ DeviceInterface \_ FriendlyName

**PKEY \_ DeviceInterface \_ FriendlyName** 屬性包含端點裝置所連接之音訊介面卡的易記名稱 (例如「XYZ 音訊介面卡」 ) 。

**PROPVARIANT** 結構的 **vt** 成員會設定為 vt \_ LPWSTR。

**PROPVARIANT** 結構的 **pwszVal** 成員指向以 null 結尾的寬字元字串，其中包含易記名稱。
