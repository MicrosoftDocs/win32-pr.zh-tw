# <a name="core-audio-enumerations"></a>核心音訊列舉

本節說明 Windows Vista 和更新版本中核心音訊 Api 使用的列舉。



| 列舉型別                                                                   | 描述                                                                                        |
|-------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------|
| [**\_AUDCLNT \_ BUFFERFLAGS**](/windows/win32/api/audioclient/ne-audioclient-_audclnt_bufferflags)                        | 音訊端點緩衝區的狀態旗標。                                                         |
| [**AUDCLNT \_ SHAREMODE**](/windows/desktop/api/Audiosessiontypes/ne-audiosessiontypes-audclnt_sharemode)                               | 資料流程的共用模式。                                                                     |
| [**AUDCLNT \_ STREAMOPTIONS**](/windows/desktop/api/audioclient/ne-audioclient-audclnt_streamoptions)                       | 定義描述音訊資料流程特性的值。                               |
| [**音訊 \_ 串流 \_ 類別**](/windows/desktop/api/audiosessiontypes/ne-audiosessiontypes-audio_stream_category)                      | 指定音訊資料流程的類別。                                                         |
| [**AudioSessionState**](/windows/desktop/api/Audiosessiontypes/ne-audiosessiontypes-audiosessionstate)                                | 音訊會話的狀態。                                                                     |
| [**ConnectorType**](/windows/win32/api/devicetopology/ne-devicetopology-connectortype)                                        | 裝置拓撲中的連接器類型。                                                        |
| [**資料流程**](/windows/win32/api/devicetopology/ne-devicetopology-dataflow)                                                  | 音訊串流的資料流程方向。                                                        |
| [**EDataFlow**](/windows/win32/api/mmdeviceapi/ne-mmdeviceapi-edataflow)                                                | 音訊資料在音訊端點裝置與用戶端應用程式之間流動的方向。 |
| [**EndpointFormFactor**](/windows/win32/api/mmdeviceapi/ne-mmdeviceapi-endpointformfactor)                              | 音訊端點裝置的一般實體屬性。                                       |
| [**ERole**](/windows/win32/api/mmdeviceapi/ne-mmdeviceapi-erole)                                                        | 音訊端點裝置的系統角色。                                                       |
| [**KSJACK \_ 接收 \_ CONNECTIONTYPE**](/windows/win32/api/devicetopology/ne-devicetopology-ksjack_sink_connectiontype)<br/> | 連接的類型。<br/>                                                                 |
| [**PartType**](/windows/win32/api/devicetopology/ne-devicetopology-parttype)                                                  | 元件的元件類型 (連接器，或是裝置拓撲中的子單位) 。                               |
