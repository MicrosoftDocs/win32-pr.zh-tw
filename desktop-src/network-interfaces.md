# <a name="network-interfaces"></a><span data-ttu-id="18f27-103">網路介面</span><span class="sxs-lookup"><span data-stu-id="18f27-103">Network interfaces</span></span>

<span data-ttu-id="18f27-104">本主題說明 Windows 上的高階網路介面概念，包括可在程式碼中識別它們的方式，以及它們的屬性。</span><span class="sxs-lookup"><span data-stu-id="18f27-104">This topic describes high-level network interface concepts on Windows, including the ways they can be identified in code and their properties.</span></span> 

> [!IMPORTANT]

## <a name="overview"></a><span data-ttu-id="18f27-107">概觀</span><span class="sxs-lookup"><span data-stu-id="18f27-107">Overview</span></span>

<span data-ttu-id="18f27-108">*網路介面* 是兩個網路設備或通訊協定層連接的點。</span><span class="sxs-lookup"><span data-stu-id="18f27-108">A *network interface* is the point where two pieces of network equipment or protocol layers connect.</span></span> <span data-ttu-id="18f27-109">一般來說，這是以實體網路介面卡來代表電腦與私人或公用網路之間的連接 (NIC) 。</span><span class="sxs-lookup"><span data-stu-id="18f27-109">Typically, this is represented by a physical Network Interface Card (NIC) for connection between a computer and a private or public network.</span></span> <span data-ttu-id="18f27-110">不過，它也可以採用僅軟體元件的形式，例如 `127.0.0.1` IPv4 或 IPv6) 的回送介面 (`::1` 。</span><span class="sxs-lookup"><span data-stu-id="18f27-110">However, it can also take the form of a software-only component such as the loopback interface (`127.0.0.1` for IPv4 or `::1` for IPv6).</span></span>

<span data-ttu-id="18f27-111">網路介面是由網際網路工程任務推動小組 (IETF) 在 [RFC 2863](https://tools.ietf.org/html/rfc2863) 中定義，而不是由 Windows 定義。</span><span class="sxs-lookup"><span data-stu-id="18f27-111">Network interfaces are defined by the Internet Engineering Task Force (IETF) in [RFC 2863](https://tools.ietf.org/html/rfc2863) and are not meant to be defined by Windows.</span></span> <span data-ttu-id="18f27-112">如需有關網路介面識別碼（例如 **ifIndex**）意義的詳細問題，請參閱 IETF 的定義。</span><span class="sxs-lookup"><span data-stu-id="18f27-112">For detailed questions about the meaning of network interface identifiers such as **ifIndex**, see the IETF's definitions of them.</span></span> <span data-ttu-id="18f27-113">本主題的其餘部分將討論 Windows 特定的執行詳細資料。</span><span class="sxs-lookup"><span data-stu-id="18f27-113">The rest of this topic discusses Windows-specific implementation details.</span></span>
