# <a name="software-device-api"></a>軟體裝置 API

## <a name="purpose"></a>目的

您可以使用軟體裝置 API 從應用程式建立 PnP 裝置。 API 可讓您將裝置列舉為任何現有父裝置的子系。 此 API 也可讓您向列舉的裝置註冊裝置介面，以及設定裝置和介面的屬性。

## <a name="developer-audience"></a>開發人員對象

軟體裝置 API 的設計目的是要讓想要在 PnP 「目錄」中發佈裝置功能或載入設備磁碟機的開發人員使用。
