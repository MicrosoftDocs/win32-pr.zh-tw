# <a name="richer-application-experiences"></a>更豐富的應用程式體驗

Windows 7 可讓開發人員建立獨特且直覺的應用程式，以大幅提升可搜尋性、可用性和極大的享受。 桌面整合的新方法將應用程式功能直接放在使用者的手中，Windows 檔案總管和連結 *庫* 可讓您輕鬆存取高價值的資訊。 Windows 功能區控制項和 Windows 動畫架構可讓您更輕鬆地建立互動式且吸引人的使用者介面。 新的觸控 Api 透過多點觸控和手指移動來啟用自然互動，而操作和慣性 Api 則可實現令人印象深刻的視覺效果。

硬體和軟體技術的快速進展也促使了更高精確度的使用者體驗。 Windows 7 以可充分利用此技術的全新且彈性的 Api，在開發人員控制項下帶來這些進步，同時讓開發引人注目的應用程式變得更容易。
