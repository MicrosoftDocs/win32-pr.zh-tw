# <a name="introduction-windows-7-developer-guide"></a>簡介 (Windows 7 開發人員指南) 

開發軟體一直都不太令人興奮。 世界從來沒有更好的連線、互動式和行動裝置。 使用者體驗和內容從未有更高的精確度。 連線性、行動性、自然介面、圖形和媒體的進展，為開發人員提供了新的案例和商機。 在此同時，效能、彈性和互通性的期望絕對不會更高。 世界級的開發人員平臺必須為開發人員及其解決方案提供有彈性且完整的基礎，讓他們能夠建立應用程式，以在競賽中設定這些應用程式，同時將其投資最大化。

Windows 7 作業系統是開發人員的基本平臺。 傾聽我們的客戶、展望未來的案例，以及建立 Windows Vista 的功能，Microsoft 為開發人員提供了 Windows 開發人員平臺內各種不同的選擇和功能，同時讓開發人員能夠提供行動感知、連線、高精確度的創意解決方案，並提供高度直覺的使用者體驗。 最重要的是，開發人員平臺的基本概念，例如安全性、效能和相容性，是 Windows 7 的首要考慮。

本指南摘要說明下列三個領域的主要開發人員進展：

[穩固的基礎](solid-foundation.md)

絕佳的使用者體驗從穩固的基礎開始。 Windows 7 提供穩固的開發平臺和創新的工具，可提供您更多的選項、更高的應用程式相容性、更佳的效能，以及精密的檔支援。 這會為您的應用程式產生簡單、更可靠的環境，並為您的使用者提供熟悉且直覺的環境。

我們讓使用者介面保持一致、可預測且便於使用，同時新增多點觸控支援、滾動和其他直覺的互動功能，以利用最新的電腦和行動裝置功能。 有了 Windows 7，開發人員可以為他們的應用程式打造適當的使用者體驗。  (參閱 [Windows 使用者經驗互動指導方針](/windows/apps/desktop/)。 ) 

[更豐富的應用程式體驗](richer-application-experiences.md)

Windows 7 可讓開發人員立即建立將在未來電腦上執行的應用程式。 啟用服務的軟體和裝置，例如行動電話、便攜媒體播放機和數位相機、需求固定的連線能力和先進的應用程式。 Windows 7 為兩者提供一個平臺，讓開發人員可以輕鬆利用下一代硬體的功能與功能，同時確保使用者永遠保持開機且一律保持連線。  (參閱 [Windows 硬體開發人員中心](https://www.microsoft.com/whdc/default.mspx)。 ) 

[Windows 和 Web 的最佳](the-best-of-windows-and-the-web.md)

在 Windows 7 中，您可以輕鬆地連線並保持連線。 Windows 網路功能提供開發人員選項，可讓您更妥善地快取和共用資料，以改善網路效能。 Windows 7 中的網路診斷會提供開發人員相關資訊來監視網路問題。 Windows Internet Explorer 8 是建立快速、豐富 web 體驗的必要平臺。 無論目標是要連接裝置、連線到網路，還是連接到 *網際網路*，Windows 7 都能提供增強的平臺。  (查看 [Web 開發](https://msdn.microsoft.com/library/aa155073.aspx)。 )
