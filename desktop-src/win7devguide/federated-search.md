# <a name="federated-search"></a>同盟搜尋

Windows 7 支援搜尋使用者電腦以外的檔。 開發人員和 IT 專業人員可以在不需要撰寫和部署用戶端程式代碼的情況下，從 Windows 7 搜尋搜尋引擎、檔存放庫、web 應用程式及專屬資料存放區。 如此一來，使用者就可以輕鬆搜尋公司內部網路或網路，因為他們可以從相同熟悉的 Windows 介面中搜尋本機檔案。

開發人員和 IT 專業人員可以將 Windows 7 相容的 *OpenSearch* 支援新增至任何現有的可搜尋 web 應用程式，方法是新增 *RSS* 或 *ATOM* 輸出（如公用的 [OpenSearch](http://www.opensearch.org/) 標準所定義）。 然後，開發人員可以撰寫 [ *OpenSearch* 描述檔]，並使用 advanced users 來定義連線參數，讓桌面用戶端建立服務的搜尋連接器。

SharePoint Search Server 也可以查詢這些相容的 *OpenSearch* 服務。 這也可讓您從 *SharePoint* 搜尋中心 web 介面存取您的搜尋服務。
