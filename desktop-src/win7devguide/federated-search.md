# <a name="federated-search"></a>同盟搜尋

Windows 7 支援搜尋使用者電腦以外的檔。 開發人員和 IT 專業人員可以從 Windows 7 中搜尋搜尋引擎、檔存放庫、web 應用程式及專屬資料存放區，而不需要撰寫和部署用戶端程式代碼。 如此一來，使用者就能搜尋其公司內部網路或網路，就像搜尋本機檔案一樣容易，因為它們都是從相同的熟悉的 Windows 介面內完成。

開發人員和 IT 專業人員可以將 Windows 7 相容的 *OpenSearch* 支援新增至任何現有的可搜尋 web 應用程式，方法是新增 *RSS* 或 *ATOM* 輸出（如公用 [OpenSearch](http://www.opensearch.org/)標準所定義）。 然後，開發人員可以撰寫 *OpenSearch* 的說明檔，並使用 advanced 使用者定義連線參數，讓桌面用戶端建立服務的搜尋連接器。

SharePoint搜尋伺服器也可以查詢這些相容的 *OpenSearch* 服務。 這也可讓您從 *SharePoint* 搜尋中心 web 介面存取您的搜尋服務。
