<span data-ttu-id="a82cd-115">最後，Internet Explorer 8 可讓您透過新的瀏覽器功能，輕鬆地將使用者連接到創新的 web 服務，以達成超越網頁的體驗。</span><span class="sxs-lookup"><span data-stu-id="a82cd-115">Finally, Internet Explorer 8 enables to fall back to Windows Internet Explorer 7- compatible (or earlier) behavior if necessary.</span></span> <span data-ttu-id="a82cd-120">為了協助開發人員充分利用 HTML 4 所提供的元素，Internet Explorer 8 提供數個表示性元素的升級支援。</span><span class="sxs-lookup"><span data-stu-id="a82cd-120">To assist developers in taking full advantage of the elements offered by HTML 4, Internet Explorer 8 provides upgraded support for several presentational elements.</span></span> <span data-ttu-id="a82cd-121">例如，Q 元素代表內嵌引號的字串，而物件專案現在可代表任何「物件」（包括影像）。</span><span class="sxs-lookup"><span data-stu-id="a82cd-121">For
