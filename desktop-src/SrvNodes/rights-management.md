# <a name="microsoft-rights-management-sdks"></a>Microsoft Rights Management Sdk

現已推出三代的 Rights Management SDK： [RIGHTS MANAGEMENT SDK 4.2](/previous-versions/windows/desktop/msipcthin2/active-directory-rights-management-services-multi-platform-thin-client-sdk-portal)、 [Microsoft Rights Management SDK 2.1](/previous-versions/windows/desktop/msipc/microsoft-information-protection-and-control-client-portal) 和 [ACTIVE DIRECTORY RIGHTS MANAGEMENT SERVICES SDK](/previous-versions/windows/desktop/adrms_sdk/active-directory-rights-management-services-sdk-portal)，以及腳本 api （ [Active Directory Rights Management Services 腳本 api](/previous-versions/windows/desktop/adrms_script/adrms-script-portal)），可讓您自訂 RMS 伺服器的管理。

## <a name="in-this-section"></a>本節內容

<dl> <dt>

[程式碼範例和工具](rms-code-samples-and-tools.md)
</dt> <dd>

Microsoft 針對所有支援的作業系統提供的 RMS 程式碼範例和開發人員支援工具集合。Android、iOS/OS X、Windows Phone 和 Windows 桌面。

</dd> <dd>

RMS SDK 4.2 是簡化的新一代工具組，可透過 Microsoft Rights Management 服務，提供輕量的開發體驗，讓您的 Android、iOS 和 Mac OS X 裝置應用程式提供資訊保護。

</dd> <dd>

RMS SDK 2.1 是適用于 Windows 桌面應用程式開發人員和以伺服器為基礎的解決方案提供者的強大 SDK 供應專案，可讓其產品具有 rights management。

</dd> <dt>

[Active Directory Rights Management Services SDK](/previous-versions/windows/desktop/adrms_sdk/active-directory-rights-management-services-sdk-portal)
</dt> <dd>

> [!Note]  
> 利用 Msdrm.dll 中用戶端所公開功能的 AD RMS SDK，可在 Windows Server 2012、Windows 8、Windows Server 2008 R2、Windows 7、Windows Server 2008 和 Windows Vista 中使用。 它在後續版本中可能會變更或無法使用。 相反地，請使用 [Microsoft Rights Management Services SDK 2.1](/previous-versions/windows/desktop/msipc/microsoft-information-protection-and-control-client-portal)，這會利用用戶端在 Msipc.dll 中公開的功能。

 

</dd> <dt>

[Active Directory Rights Management Services Scripting API](/previous-versions/windows/desktop/adrms_script/adrms-script-portal)
</dt> <dd>

您可以使用 Windows Active Directory Rights Management Services (AD RMS) 腳本 API 來建立腳本，以管理 AD RMS 安裝。

</dd> </dl>
