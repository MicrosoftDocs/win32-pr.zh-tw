# <a name="active-directory-federation-services"></a>Active Directory Federation Services

Active Directory 同盟服務 (AD FS) 是在 Windows 伺服器上執行的驗證與授權服務。 AD FS 為使用者提供企業環境與 web 應用程式之間的單一登入存取。
