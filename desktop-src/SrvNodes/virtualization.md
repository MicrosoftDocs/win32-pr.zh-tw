# <a name="virtualization"></a>虛擬化

## <a name="in-this-section"></a>本節內容

<dl> <dt>

[Microsoft Virtual Server](/previous-versions/windows/desktop/msvs/microsoft-virtual-server-portal)
</dt> <dd>

Microsoft Virtual Server 2005 可讓您在 Microsoft Windows 桌上型電腦上建立個別的虛擬機器，讓您幾乎可以安裝任何 Intel 架構的作業系統。 每部虛擬機器都是在獨立的獨立軟體環境中，模擬一個完整的硬體系統（從處理器到網路卡），以啟用其他不相容系統的平行作業。

</dd> <dt>

[Windows Virtual PC](/windows/desktop/VPC/virtual-pc-portal)
</dt> <dd>

Windows Virtual PC 是最新的 Microsoft 虛擬化技術;它可讓您直接從 Windows 7 在虛擬 Windows 環境中執行許多生產力應用程式，只需按一下即可。 您可以在 Windows 7 桌上型電腦上建立個別的虛擬機器。 每部虛擬機器都是在獨立的獨立軟體環境中，模擬一個完整的硬體系統（從處理器到網路卡），以啟用其他不相容系統的平行作業。

</dd> </dl>
