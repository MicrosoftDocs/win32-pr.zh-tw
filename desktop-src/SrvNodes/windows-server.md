# <a name="windows-server"></a>Windows Server

Windows伺服器是一組由 Microsoft 所設計的作業系統，可支援企業層級的管理、資料儲存、應用程式和通訊。 舊版的 Windows Server 著重于穩定性、安全性、網路功能，以及檔案系統的各種改進。 其他改良功能也包含部署技術的增強功能，以及增加的硬體支援。 Microsoft 也建立了 Windows Server 的特製化 sku，著重于家用和小型企業市場。 Windows Server 2012R2 是 Windows Server 的最新版本，並著重于雲端運算。

## <a name="windows-server-versions"></a>Windows伺服器版本

下表說明 Windows Server 的目前記錄版本。



| 伺服器版本                                                                                                      | 描述                                                                                                                                                                                                                                                                                                                                                         |
|---------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Windows Server 2012 R2 與 Windows Server 2012](what-s-new-for-windows-server-2012-r2.md)<br/>              | 利用改良的 IP 定址功能、更新的 Hyper-v 以及新的檔案系統 (ReFS) 來強調雲端支援。 Windows Server 2012R2 包含虛擬化、管理、儲存體、網路、虛擬桌面基礎結構、存取保護、資訊保護、web 服務和應用程式平臺基礎結構的增強功能。<br/> |
| [Windows Server 2008 R2 和 Windows Server 2008](https://msdn.microsoft.com/library/jj873947.aspx)<br/> | 包含許多與 Windows Vista 共用的額外安全性和管理功能：重新撰寫的網路堆疊、改良的防火牆、額外的 .NET Framework 技術，以及許多核心、檔案和記憶體系統的改進。<br/>                                                                                               |
| [Windowsserver 2003 R2 和 Windows server 2003](https://msdn.microsoft.com/library/dn609950.aspx)<br/> | 整合 Windows XP 的許多功能。 包含改良的網路安裝和整合、改良的 web 服務，以及 DTFS 的增強功能。<br/>                                                                                                                                                                                  |
