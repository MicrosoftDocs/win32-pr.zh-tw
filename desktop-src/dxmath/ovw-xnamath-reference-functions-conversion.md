# <a name="directxmath-library-conversion-functions"></a>DirectXMath 程式庫轉換函式

列出 DirectXMath 所提供的轉換函數。

## <a name="in-this-section"></a>本節內容



| 主題                                                                       | 描述                                                                                                                                                          |
|-----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMConvertFloatToHalf**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmconvertfloattohalf)<br/>             | 將單精確度浮點數轉換為半精確度浮點值。<br/>                                                                |
| [**XMConvertFloatToHalfStream**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmconvertfloattohalfstream)<br/> | 將單精確度浮點數的資料流程轉換成半精確度浮點值的資料流程。<br/>                                          |
| [**XMConvertHalfToFloat**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmconverthalftofloat)<br/>             | 將半精確度浮點數轉換成單精確度浮點值。<br/>                                                                |
| [**XMConvertHalfToFloatStream**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmconverthalftofloatstream)<br/> | 將半精確度浮點數的資料流程轉換成單精確度浮點值的資料流程。<br/>                                          |
| [**XMConvertToDegrees**](/windows/desktop/api/DirectXMath/nf-directxmath-xmconverttodegrees)<br/>                 | 將以弧度為單位的角度，轉換成以度數測量的角度。<br/>                                                                                       |
| [**XMConvertToRadians**](/windows/desktop/api/DirectXMath/nf-directxmath-xmconverttoradians)<br/>                 | 將以度為單位的角度，轉換成以弧度為單位的角度。<br/>                                                                                       |
| [**XMConvertVectorFloatToInt**](/windows/win32/api/directxmath/nf-directxmath-xmconvertvectorfloattoint)<br/>   | 將具有 **float** 元件的 [**XMVECTOR**](xmvector-data-type.md)轉換成具有 **int32 \_ t** 元件的 **XMVECTOR** ，並套用統一偏差。<br/>  |
| [**XMConvertVectorFloatToUInt**](/windows/win32/api/directxmath/nf-directxmath-xmconvertvectorfloattouint)<br/> | 將具有 **float** 元件的 [**XMVECTOR**](xmvector-data-type.md)轉換成具有 **uint32 \_ t** 元件的 **XMVECTOR** ，並套用統一偏差。<br/> |
| [**XMConvertVectorIntToFloat**](/windows/win32/api/directxmath/nf-directxmath-xmconvertvectorinttofloat)<br/>   | 將具有 **int32 \_ t** 元件的 [**XMVECTOR**](xmvector-data-type.md)轉換成具有 **float** 元件的 **XMVECTOR** ，並套用統一偏差。<br/>  |
| [**XMConvertVectorUIntToFloat**](/windows/win32/api/directxmath/nf-directxmath-xmconvertvectoruinttofloat)<br/> | 將具有 **uint32 \_ t** 元件的 [**XMVECTOR**](xmvector-data-type.md)轉換為具有 **float** 元件的 **XMVECTOR** ，並套用統一偏差。<br/> |
