# <a name="xmuint2-extensions"></a>XMUINT2 延伸模組

[**XMUINT2**](/windows/win32/api/directxmath/ns-directxmath-xmuint2) 描述由兩個不帶正負號的整數值所組成的2d 向量。

> [!Note]  
> 此處所列的功能只有在使用 c + + 進行開發時才可使用。
