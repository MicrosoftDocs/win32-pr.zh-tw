# <a name="xmxdecn4-extensions"></a>XMXDECN4 延伸模組

[**XMXDECN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmxdecn4) 定義了4d 向量，用來將已簽署的正規化值以10位帶正負號的 x、y 和 z 元件和不帶正負號的正規化值儲存為2位不帶正負號的 w 元件。

> [!Note]  
> 此處所列的功能只有在使用 c + + 進行開發時才可使用。
