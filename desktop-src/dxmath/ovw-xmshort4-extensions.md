# <a name="xmshort4-extensions"></a>XMSHORT4 延伸模組

[**XMSHORT4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshort4) 描述由16位帶正負號的整陣列件所組成的4d 向量。

> [!Note]  
> 此處所列的功能只有在使用 c + + 進行開發時才可使用。
