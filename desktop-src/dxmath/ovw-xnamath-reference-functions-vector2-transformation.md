# <a name="directxmath-library-2d-vector-transformation-functions"></a>DirectXMath 程式庫2D 向量轉換函式

列出2D 向量轉換函式。

## <a name="in-this-section"></a>本節內容



| 主題                                                                               | 描述                                                                                                                                      |
|-------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector2Transform**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transform)<br/>                         | 依矩陣轉換2D 向量。<br/>                                                                                                   |
| [**XMVector2TransformCoord**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transformcoord)<br/>               | 依指定的矩陣轉換2D 向量，將結果投射回 w = 1。<br/>                                                      |
| [**XMVector2TransformCoordStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transformcoordstream)<br/>   | 依指定的矩陣轉換2D 向量的資料流程，投射產生的向量，使其 w 座標等於1.0。<br/> |
| [**XMVector2TransformNormal**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transformnormal)<br/>             | 依指定的矩陣轉換2D 向量標準。<br/>                                                                                  |
| [**XMVector2TransformNormalStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transformnormalstream)<br/> | 依指定的矩陣轉換2D 一般向量的資料流程。<br/>                                                                           |
| [**XMVector2TransformStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transformstream)<br/>             | 依指定的矩陣轉換2D 向量的資料流程。<br/>                                                                                  |
