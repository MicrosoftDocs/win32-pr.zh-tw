# <a name="xmint4-extensions"></a>XMINT4 延伸模組

[**XMINT4**](/windows/win32/api/directxmath/ns-directxmath-xmint4) 描述包含由四個整數值組成之4d 向量的結構。

> [!Note]  
> 此處所列的功能只有在使用 c + + 進行開發時才可使用。
