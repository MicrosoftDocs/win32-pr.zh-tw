# <a name="directxmath-library-template-functions"></a>DirectXMath 程式庫範本函數

列出 DirectXMath 提供的範本函數。

## <a name="in-this-section"></a>本節內容



| 主題                                                                  | 描述                                                                                                                                                                                                                               |
|------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMMax**](xmmax-template.md)<br/>                             | 比較兩個數值資料類型實例或物件的兩個實例，以支援 < 的多載，並傳回兩個實例中較大的一個。 引數的資料類型和傳回值相同。<br/>  |
| [**XMMin**](xmmin-template.md)<br/>                             | 比較兩個數值資料類型實例或物件的兩個實例，以支援 < 的多載，並傳回兩個實例中較小的一個。 引數的資料類型和傳回值相同。<br/> |
| [**XMVectorPermute**](xmvectorpermute-template.md)<br/>         | Permutes 兩個向量的元件來建立新的向量。<br/>                                                                                                                                                                 |
| [**XMVectorRotateLeft**](xmvectorrotateleft-template.md)<br/>   | 將向量向左旋轉指定的32位元素數目。<br/>                                                                                                                                                                  |
| [**XMVectorRotateRight**](xmvectorrotateright-template.md)<br/> | 將向量向右旋轉指定的32位元素數目。<br/>                                                                                                                                                                 |
| [**XMVectorShiftLeft**](xmvectorshiftleft-template.md)<br/>     | 將向量向左移位指定數目的32位元素，並使用第二個向量的元素來填滿空的元素。<br/>                                                                                                    |
| [**XMVectorSwizzle**](xmvectorswizzle-template.md)<br/>         | Swizzles 向量。<br/>                                                                                                                                                                                                             |
| [**XMVectorInsert**](xmvectorinsert-template.md)<br/>           | 將向量旋轉指定的32位元件數目，並將該結果的選取元素插入另一個向量。<br/>                                                                                                  |
