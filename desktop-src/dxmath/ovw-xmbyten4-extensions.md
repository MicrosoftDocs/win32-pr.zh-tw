# <a name="xmbyten4-extensions"></a>XMBYTEN4 延伸模組

[**XMBYTEN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmbyten4) 描述用來將已簽署的正規化值儲存為帶正負號的8位 (1 位元組) 整數的3d 向量。

> [!Note]  
> 此處所列的功能只有在使用 c + + 進行開發時才可使用。
