# <a name="xmhalf2-extensions"></a>XMHALF2 延伸模組

[**XMHALF2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmhalf2) 描述2d 向量，其中包含兩個半精確度 (16bit) 浮點數。

> [!Note]  
> 此處所列的功能只有在使用 c + + 進行開發時才可使用。
