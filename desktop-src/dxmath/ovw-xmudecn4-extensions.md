# <a name="xmudecn4-extensions"></a>XMUDECN4 延伸模組

[**XMUDECN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmudecn4)用來將未簽署、正規化的值儲存為10位不帶正負號 x、y 和 z 元件，以及2位不帶正負號 w 元件的4D 向量。

> [!Note]  
> 此處所列的功能只有在使用 c + + 進行開發時才可使用。
