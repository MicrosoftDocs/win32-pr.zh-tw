# <a name="xmuint4-extensions"></a>XMUINT4 延伸模組

[**XMUINT4**](/windows/win32/api/directxmath/ns-directxmath-xmuint4) 描述包含由四個不帶正負號的整數值組成之4d 向量的結構。

> [!Note]  
> 此處所列的功能只有在使用 c + + 進行開發時才可使用。
