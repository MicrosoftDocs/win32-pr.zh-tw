# <a name="bit-wise-vector-functions"></a>Bit-Wise 向量函數

列出位向量函數。

## <a name="in-this-section"></a>本節內容



| 主題                                                         | 描述                                                                                                                             |
|---------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVectorAndCInt**](/windows/win32/api/directxmath/nf-directxmath-xmvectorandcint)<br/>         | 以負號的第二個向量計算一個向量的邏輯 AND，將每個元件視為不帶正負號的整數。<br/> |
| [**XMVectorAndInt**](/windows/win32/api/directxmath/nf-directxmath-xmvectorandint)<br/>           | 計算兩個向量的邏輯 AND，將每個元件視為不帶正負號的整數。<br/>                                     |
| [**XMVectorNorInt**](/windows/win32/api/directxmath/nf-directxmath-xmvectornorint)<br/>           | 計算兩個向量的邏輯或，將每個元件視為不帶正負號的整數。<br/>                                     |
| [**XMVectorNotEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvectornotequal)<br/>       | 針對兩個向量是否不相等執行每個元件的測試。<br/>                                                             |
| [**XMVectorNotEqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvectornotequalint)<br/> | 針對兩個向量的不相等執行每個元件測試，並將每個元件視為不帶正負號的整數。<br/>             |
| [**XMVectorOrInt**](/windows/win32/api/directxmath/nf-directxmath-xmvectororint)<br/>             | 計算兩個向量的邏輯 OR，將每個元件視為不帶正負號的整數。<br/>                                      |
| [**XMVectorXorInt**](/windows/win32/api/directxmath/nf-directxmath-xmvectorxorint)<br/>           | 計算兩個向量的邏輯 XOR，將每個元件視為不帶正負號的整數。<br/>                                     |
