# <a name="xmushort2-extensions"></a>XMUSHORT2 延伸模組

[**XMUSHORT2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushort2) 描述用來將未簽署的正規化值儲存為不帶正負號16位整數的 a2D 向量， (類型 `uint16_t`) 。

> [!Note]  
> 此處所列的功能只有在使用 c + + 進行開發時才可使用。
