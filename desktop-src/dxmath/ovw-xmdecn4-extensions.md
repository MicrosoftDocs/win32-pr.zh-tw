# <a name="xmdecn4-extensions"></a>XMDECN4 延伸模組

[**XMDECN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmdecn4) 描述的4d 向量可將已簽署的正規化值儲存為10位帶正負號的 x、y 和 z 元件，以及2位帶正負號的 w 元件。

> [!Note]  
> 此處所列的功能只有在使用 c + + 進行開發時才可使用。
