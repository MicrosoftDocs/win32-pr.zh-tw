# <a name="xmvector-operators"></a>XMVECTOR 運算子

下列運算子是由 [**XMVECTOR**](xmvector-data-type.md) 結構所公開。

> [!Note]  
> 此處所列的運算子僅適用于 c + +。

 

## <a name="in-this-section"></a>本節內容



| 方法                                                    | 描述                                                                                                                                 |
|------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| [**運算子 + =**](/previous-versions/windows/desktop/legacy/ee421394(v=vs.85))<br/>  | 將浮點值加入至 `XMVECTOR` 實例，並傳回已更新之實例的參考。 <br/>                         |
| [**operator-=**](/previous-versions/windows/desktop/legacy/ee421383(v=vs.85))<br/>    | 從目前的實例減去浮點值，並傳回 `XMVECTOR` 更新的目前實例中的結果。 <br/> |
| [**運算元 \***](xmvector-operator-mul.md)<br/>    | 乘法運算子<br/>                                                                                                         |
| [**運算元 \*=**](xmvector-operator-muleq.md)<br/> | 乘法指派運算子<br/>                                                                                              |
| [**運算子**](xmvector-operator-div.md)<br/>     | 除法運算子。<br/>                                                                                                               |
| [**operator/=**](xmvector-operator-diveq.md)<br/>  | 除法指派運算子。<br/>                                                                                                    |
| [**運算子**](xmvector-operator--.md)<br/>       | 減法和負運算子<br/>                                                                                               |
| [**運算子 +**](xmvector-operator-pls.md)<br/>     | 加法運算子。<br/>                                                                                                              |
