# <a name="component-wise-vector-functions"></a>元件的向量函數

列出元件的向量函數。

## <a name="in-this-section"></a>本節內容



| 主題                                                             | 描述                                                                                                                              |
|-------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVectorInsert**](/windows/win32/api/directxmath/nf-directxmath-xmvectorinsert)<br/>               | 將向量旋轉指定的32位元件數目，並將該結果的選取元素插入另一個向量。<br/> |
| [**XMVectorMergeXY**](/windows/win32/api/directxmath/nf-directxmath-xmvectormergexy)<br/>             | 藉由結合兩個向量的 x 和 y 元件來建立新的向量。<br/>                                                      |
| [**XMVectorMergeZW**](/windows/win32/api/directxmath/nf-directxmath-xmvectormergezw)<br/>             | 藉由結合兩個向量的 z 和 w 元件來建立新的向量。<br/>                                                      |
| [**XMVectorPermute**](/windows/win32/api/directxmath/nf-directxmath-xmvectorpermute)<br/>             | Permutes 兩個向量的元件來建立新的向量。<br/>                                                                |
| [**XMVectorRotateLeft**](/windows/win32/api/directxmath/nf-directxmath-xmvectorrotateleft)<br/>       | 將向量向左旋轉指定的32位元素數目。<br/>                                                                 |
| [**XMVectorRotateRight**](/windows/win32/api/directxmath/nf-directxmath-xmvectorrotateright)<br/>     | 將向量向右旋轉指定的32位元素數目。<br/>                                                                |
| [**XMVectorSelect**](/windows/win32/api/directxmath/nf-directxmath-xmvectorselect)<br/>               | 在兩個輸入向量之間執行每個元件的選取專案，並傳回產生的向量。<br/>                                |
| [**XMVectorSelectControl**](/windows/win32/api/directxmath/nf-directxmath-xmvectorselectcontrol)<br/> | 定義用於 [**XMVectorSelect**](/windows/win32/api/directxmath/nf-directxmath-xmvectorselect)的控制向量。<br/>                                                 |
| [**XMVectorShiftLeft**](/windows/win32/api/directxmath/nf-directxmath-xmvectorshiftleft)<br/>         | 將向量向左移位指定數目的32位元素，並使用第二個向量的元素來填滿空的元素。<br/>   |
| [**XMVectorSplatW**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatw)<br/>               | 將向量的 w 元件複製到所有元件。<br/>                                                              |
| [**XMVectorSplatX**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatx)<br/>               | 將向量的 x 元件複製到所有元件。<br/>                                                              |
| [**XMVectorSplatY**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplaty)<br/>               | 將向量的 y 元件複製到所有元件。<br/>                                                              |
| [**XMVectorSplatZ**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatz)<br/>               | 將向量的 z 元件複製到所有元件。<br/>                                                              |
| [**XMVectorSwizzle**](/windows/win32/api/directxmath/nf-directxmath-xmvectorswizzle)<br/>             | Swizzles 向量。<br/>                                                                                                            |
