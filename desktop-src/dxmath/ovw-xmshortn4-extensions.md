# <a name="xmshortn4-extensions"></a>XMSHORTN4 延伸模組

[**XMSHORTN4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshortn4) 會描述用於將已簽署的正規化值儲存為帶正負號16位整數的4d 向量， (類型 `int16_t`) 。

> [!Note]  
> 此處所列的功能只有在使用 c + + 進行開發時才可使用。
