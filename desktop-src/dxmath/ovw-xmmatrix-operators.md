# <a name="xmmatrix-operators"></a>XMMATRIX 運算子

[**XMMATRIX**](/windows/win32/api/directxmath/ns-directxmath-xmmatrix)結構會公開下列運算子。

> [!Note]  
> 此處所列的運算子僅適用于 c + +。

 

## <a name="in-this-section"></a>本節內容



| 方法                                                    | 描述                                                                                                                                                                                   |
|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**運算子 \** _](/windows/win32/api/directxmath/nf-directxmath-xmmatrix-operator-mult)<br/>    | 執行另一個實例之目前實例的矩陣乘法 `XMMATRIX` `XMMATRIX` 。<br/>                                                                          |
| [_ *運算子 \* =**](/windows/win32/api/directxmath/nf-directxmath-xmmatrix-operator-mult-assign)<br/> | 執行另一個實例的目前實例的矩陣乘法， `XMMATRIX` 並傳回 `XMMATRIX` 目前實例的參考，而該實例已更新。 <br/> |
| [**運算子 =**](/windows/win32/api/directxmath/nf-directxmath-xmmatrix-operator-assign)<br/>      | 將一個實例的矩陣資料指派 `XMMATRIX` 給目前的實例 `XMMATRIX` ，並將傳回 `reference` 至目前的實例。 <br/>                                    |
| [**運算子 ()**](xmmatrix-operator-parens.md)<br/> | 從目前的實例存取資料列和資料行所參考的特定矩陣元素 `XMMATRIX` 。 <br/>                                                                           |
