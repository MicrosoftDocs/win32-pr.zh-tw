# <a name="directxmath-library-utility-functions"></a>DirectXMath 程式庫公用程式函式

列出 DirectXMath 所提供的公用程式函數。

## <a name="in-this-section"></a>本節內容



| 主題                                                                       | 描述                                                                                                                |
|-----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------|
| [**XMComparisonAllFalse**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonallfalse)<br/>             | 測試比較值，以判斷是否所有比較的元件都是 false。<br/>                            |
| [**XMComparisonAllInBounds**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonallinbounds)<br/>       | 測試比較值，以判斷是否所有比較的元件都在設定界限內。<br/>                |
| [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue)<br/>               | 測試比較值，以判斷所有比較的元件是否為 true。<br/>                             |
| [**XMComparisonAnyFalse**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonanyfalse)<br/>             | 測試比較值，以判斷是否有任何比較的元件為 false。<br/>                            |
| [**XMComparisonAnyOutOfBounds**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonanyoutofbounds)<br/> | 測試比較值，以判斷是否有任何比較的元件在集合界限之外。<br/>           |
| [**XMComparisonAnyTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonanytrue)<br/>               | 測試比較值，以判斷是否有任何比較的元件為 true。<br/>                             |
| [**XMComparisonMixed**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonmixed)<br/>                   | 測試比較值，以判斷比較的元件是否具有混合的結果--某些 true 和一些 false。<br/> |
| [**XMFresnelTerm**](/windows/win32/api/directxmath/nf-directxmath-xmfresnelterm)<br/>                           | 計算 unpolarized light 的 Fresnel 字詞。<br/>                                                              |
| [**XMVerifyCPUSupport**](/windows/win32/api/directxmath/nf-directxmath-xmverifycpusupport)<br/>                 | 指出 DirectXMath 程式庫是否支援目前的平臺。<br/>                                             |
