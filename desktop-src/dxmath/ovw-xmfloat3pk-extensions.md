# <a name="xmfloat3pk-extensions"></a>XMFLOAT3PK 延伸模組

3D 向量，其 X 和 Y 元件儲存為11位浮點數，而 Z 元件儲存為10位浮點值。

> [!Note]  
> 此處所列的功能只有在使用 c + + 進行開發時才可使用。
