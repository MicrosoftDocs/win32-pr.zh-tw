# <a name="xmmatrix-extensions"></a>XMMATRIX 延伸模組

[**XMMATRIX**](/windows/win32/api/directxmath/ns-directxmath-xmmatrix) 描述的 4 \* 4 矩陣，會對應至4個位元組的界限，並對應到四個硬體向量暫存器。

> [!Note]  
> 此處所列的功能只適用于 c + +。
