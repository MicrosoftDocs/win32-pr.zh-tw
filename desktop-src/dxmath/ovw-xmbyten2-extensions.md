# <a name="xmbyten2-extensions"></a>XMBYTEN2 延伸模組

[**XMBYTEN2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmbyten2) 說明將已簽署的正規化值儲存為帶正負號的8位 (1 位元組) 整數的2d 向量。

> [!Note]  
> 此處所列的功能只有在使用 c + + 進行開發時才可使用。
