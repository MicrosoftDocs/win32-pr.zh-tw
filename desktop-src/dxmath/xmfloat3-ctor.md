# <a name="xmfloat3-constructors"></a>XMFLOAT3 函式

公開的函式 `XMFLOAT3` 。

結構 [**XMFLOAT3**](/windows/win32/api/directxmath/ns-directxmath-xmfloat3) 結構的實例。

> [!Note]  
> 此處所列的函式僅適用于 c + +。

 

### <a name="overload-list"></a>多載清單




| 建構函式 | 描述 | 
|-------------|-------------|
| [<strong>XMFLOAT3 () </strong>](/windows/win32/api/directxmath/nf-directxmath-xmfloat3-xmfloat3(constfloat)) | 的預設函數 <code>XMFLOAT3</code> 。<br /> [<strong>XMFLOAT3</strong>](/windows/win32/api/directxmath/ns-directxmath-xmfloat3)的預設的函式。<br /><blockquote>[!Note]<br />這個函式僅適用于 c + +。</blockquote><br /> | 
| [<strong>XMFLOAT3 (const float * ) </strong>](/windows/win32/api/directxmath/nf-directxmath-xmfloat3a-xmfloat3a(constfloat)) | <code>XMFLOAT3</code>從三個元素陣列引數，初始化的新實例 <code>float</code> 。<br /> 這個函式會從三個元素陣列引數，初始化 [<strong>XMFLOAT3</strong>](/windows/win32/api/directxmath/ns-directxmath-xmfloat3) 的新實例 <code>float</code> 。<br /><blockquote>[!Note]<br />這個函式僅適用于 c + +。</blockquote><br /> | 
| [<strong>XMFLOAT3 (float，float，float) </strong>](/windows/win32/api/directxmath/nf-directxmath-xmfloat3-xmfloat3(float_float_float)) | <code>XMFLOAT3</code>從三個引數初始化的新實例 <code>float</code> 。<br /> 這個函式會從三個引數初始化 [<strong>XMFLOAT3</strong>](/windows/win32/api/directxmath/ns-directxmath-xmfloat3) 的新實例 <code>float</code> 。<br /><blockquote>[!Note]<br />這個函式僅適用于 c + +。</blockquote><br /> |
