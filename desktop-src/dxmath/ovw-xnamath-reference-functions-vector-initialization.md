# <a name="vector-initialization-functions"></a>向量初始化函數

列出向量初始化函數。

## <a name="in-this-section"></a>本節內容



| 主題                                                                     | 描述                                                                                                                                      |
|---------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVectorFalseInt**](/windows/win32/api/directxmath/nf-directxmath-xmvectorfalseint)<br/>                   | 傳回零 (false) 向量。<br/>                                                                                                      |
| [**XMVectorReplicate**](/windows/win32/api/directxmath/nf-directxmath-xmvectorreplicate)<br/>                 | 將浮點值複寫至向量的所有四個元件。<br/>                                                               |
| [**XMVectorReplicateInt**](/windows/win32/api/directxmath/nf-directxmath-xmvectorreplicateint)<br/>           | 將整數值複寫至向量的所有四個元件。<br/>                                                                     |
| [**XMVectorReplicateIntPtr**](/windows/win32/api/directxmath/nf-directxmath-xmvectorreplicateintptr)<br/>     | 將指標所參考的整數值複寫至向量的所有四個元件。<br/>                                            |
| [**XMVectorReplicatePtr**](/windows/win32/api/directxmath/nf-directxmath-xmvectorreplicateptr)<br/>           | 將指標所參考的浮點值複寫至向量的所有四個元件。<br/>                                         |
| [**XMVectorSet**](/windows/win32/api/directxmath/nf-directxmath-xmvectorset)<br/>                             | 使用四個浮點值建立向量。<br/>                                                                                    |
| [**XMVectorSetBinaryConstant**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsetbinaryconstant)<br/> | 建立向量，其中每個元件都是 0.0 f 或 1.0 f。<br/>                                                                    |
| [**XMVectorSetInt**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsetint)<br/>                       | 建立具有不帶正負號整陣列件的向量。<br/>                                                                                    |
| [**XMVectorSplatConstant**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatconstant)<br/>         | 使用相同的浮點元件建立向量。 每個元件都是將整數除以兩個整數指數的常數。<br/> |
| [**XMVectorSplatConstantInt**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatconstantint)<br/>   | 建立具有相同整陣列件的向量。<br/>                                                                                   |
| [**XMVectorSplatEpsilon**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatepsilon)<br/>           | 傳回向量，其中每個元件都是 epsilon (1.192092896 e-7) 。<br/>                                                              |
| [**XMVectorSplatInfinity**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatinfinity)<br/>         | 傳回向量，其中每個元件都是無限大 (0x7F800000) 。<br/>                                                                 |
| [**XMVectorSplatOne**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatone)<br/>                   | 傳回向量，其中每個元件都是一個。<br/>                                                                                   |
| [**XMVectorSplatQNaN**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatqnan)<br/>                 | 傳回向量，其中每個元件都 QNaN (0x7CF00000) 。<br/>                                                                     |
| [**XMVectorSplatSignMask**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatsignmask)<br/>         | 傳回向量，其中每個元件都是正負號遮罩 (0x80000000) 。<br/>                                                            |
| [**XMVectorTrueInt**](/windows/win32/api/directxmath/nf-directxmath-xmvectortrueint)<br/>                     | 傳回向量，其中每個元件都代表 true (0xFFFFFFFF) 。<br/>                                                              |
| [**XMVectorZero**](/windows/win32/api/directxmath/nf-directxmath-xmvectorzero)<br/>                           | 建立零向量。<br/>                                                                                                              |
