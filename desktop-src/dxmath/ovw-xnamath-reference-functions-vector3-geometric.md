| 主題                                                                                 | 描述                                                                                                                       |
|---------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|

| [**XMVector3AngleBetweenNormalsEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector3anglebetweennormalsest)<br/> | 估計兩個正規化3D 向量之間的弧度角度。<br/>                                                          |

| [**XMVector3ClampLength**](/windows/win32/api/directxmath/nf-directxmath-xmvector3clamplength)<br/>                       | 將3D 向量的長度個至指定的範圍。<br/>                                                                     |
| [**XMVector3ClampLengthV**](/windows/win32/api/directxmath/nf-directxmath-xmvector3clamplengthv)<br/>                     | 將3D 向量的長度個至指定的範圍。<br/>                                                                     |
