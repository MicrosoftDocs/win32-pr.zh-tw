# <a name="xmushortn2-extensions"></a>XMUSHORTN2 延伸模組

[**XMUSHORTN2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushortn2) 描述用來將未簽署的正規化值儲存為不帶正負號16位整數的2d 向量， (類型 `int16_t`) 。

> [!Note]  
> 此處所列的功能只有在使用 c + + 進行開發時才可使用。
