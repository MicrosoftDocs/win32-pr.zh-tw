# <a name="xmdec4-operators"></a>XMDEC4 運算子

> [!Note]  
> 此處所列的運算子僅適用于 c + +。

 

## <a name="in-this-section"></a>本節內容



| 方法                                                           | 描述                                                                                                          |
|-------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**運算子 uint32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmdec4-operatoruint32_t)<br/> | 傳回的實例， `uint32_t` 其中包含 `XMDEC4` 封裝格式之實例的元件。 <br/> |
