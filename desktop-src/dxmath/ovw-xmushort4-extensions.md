# <a name="xmushort4-extensions"></a>XMUSHORT4 延伸模組

[**XMUSHORT4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushort4) 描述由16位不帶正負號整陣列件所組成的4d 向量。

> [!Note]  
> 此處所列的功能只有在使用 c + + 進行開發時才可使用。
