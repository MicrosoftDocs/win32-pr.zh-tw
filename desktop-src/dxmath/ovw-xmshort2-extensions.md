# <a name="xmshort2-extensions"></a>XMSHORT2 延伸模組

[**XMSHORT2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshort2) 描述由16位帶正負號和正規化的整陣列件所組成的2d 向量。

> [!Note]  
> 此處所列的功能只有在使用 c + + 進行開發時才可使用。
