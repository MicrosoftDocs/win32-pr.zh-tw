# <a name="xmcolor-extensions"></a>XMCOLOR 延伸模組

[**XMCOLOR**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmcolor) 描述32位的紅色綠色藍色 (ARGB) 色向量，其中每個顏色通道都是由8個位指定。

> [!Note]  
> 此處所列的功能只有在使用 c + + 進行開發時才可使用。
