# <a name="xmvector-extensions"></a>XMVECTOR 延伸模組

[**XMVECTOR**](xmvector-data-type.md) 是可用來代表 4 32 位浮點或整陣列件向量的可移植型別，每個都能以最佳方式對齊，並對應至硬體向量暫存器。

> [!Note]  
> 此處所列的功能只適用于 c + +。
