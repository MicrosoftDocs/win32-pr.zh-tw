# <a name="xmubyten4-extensions"></a>XMUBYTEN4 延伸模組

[**XMUBYTEN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmubyten4) 描述用來將未簽署的正規化值儲存為不帶正負號的8位 (1 位元組) 整數的3d 向量。

> [!Note]  
> 此處所列的功能只有在使用 c + + 進行開發時才可使用。
