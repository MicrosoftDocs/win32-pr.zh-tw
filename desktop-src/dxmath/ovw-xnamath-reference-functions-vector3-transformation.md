# <a name="directxmath-3d-vector-transformation-functions"></a>DirectXMath 3D 向量轉換函式

列出3D 向量轉換函式。

## <a name="in-this-section"></a>本節內容



| 主題                                                                               | 描述                                                                                                                                      |
|-------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector3InverseRotate**](/windows/win32/api/directxmath/nf-directxmath-xmvector3inverserotate)<br/>                 | 使用四元數的反向旋轉3D 向量。<br/>                                                                                |
| [**XMVector3Project**](/windows/win32/api/directxmath/nf-directxmath-xmvector3project)<br/>                             | 將物件空間中的3D 向量投影至螢幕空間。<br/>                                                                              |
| [**XMVector3ProjectStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3projectstream)<br/>                 | 將3D 向量的資料流程從物件空間投射到螢幕空間。<br/>                                                                  |
| [**XMVector3Rotate**](/windows/win32/api/directxmath/nf-directxmath-xmvector3rotate)<br/>                               | 使用四元數旋轉3D 向量。<br/>                                                                                               |
| [**XMVector3Transform**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transform)<br/>                         | 依矩陣轉換3D 向量。<br/>                                                                                                   |
| [**XMVector3TransformCoord**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformcoord)<br/>               | 依指定的矩陣轉換3D 向量，將結果投射回 w = 1。<br/>                                                      |
| [**XMVector3TransformCoordStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformcoordstream)<br/>   | 依指定的矩陣轉換3D 向量的資料流程，投射產生的向量，使其 w 座標等於1.0。<br/> |
| [**XMVector3TransformNormal**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformnormal)<br/>             | 依指定的矩陣轉換一般的3D 向量。<br/>                                                                                  |
| [**XMVector3TransformNormalStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformnormalstream)<br/> | 依指定的矩陣轉換3D 一般向量的資料流程。<br/>                                                                           |
| [**XMVector3TransformStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformstream)<br/>             | 依指定的矩陣轉換3D 向量的資料流程。<br/>                                                                                  |
| [**XMVector3Unproject**](/windows/win32/api/directxmath/nf-directxmath-xmvector3unproject)<br/>                         | 從螢幕空間將3D 向量投射到物件空間。<br/>                                                                             |
| [**XMVector3UnprojectStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3unprojectstream)<br/>             | 將3D 向量的資料流程從螢幕空間轉換成物件空間。<br/>                                                                  |
