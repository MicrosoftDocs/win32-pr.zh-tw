# <a name="xmint2-extensions"></a>XMINT2 延伸模組

[**XMINT2**](/windows/win32/api/directxmath/ns-directxmath-xmint2) 描述由兩個整數值所組成的2d 向量。

> [!Note]  
> 此處所列的功能只有在使用 c + + 進行開發時才可使用。
