# <a name="geometric-vector-functions"></a>幾何向量函數

列出幾何向量函數。

## <a name="in-this-section"></a>本節內容



| 主題                                                           | 描述                                                                                                                                                                                                      |
|-----------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVectorBaryCentric**](/windows/win32/api/directxmath/nf-directxmath-xmvectorbarycentric)<br/>   | 使用指定的位置向量，傳回 Barycentric 座標中的點。<br/>                                                                                                                     |
| [**XMVectorBaryCentricV**](/windows/win32/api/directxmath/nf-directxmath-xmvectorbarycentricv)<br/> | 使用指定的位置向量，傳回 Barycentric 座標中的點。<br/>                                                                                                                     |
| [**XMVectorCatmullRom**](/windows/win32/api/directxmath/nf-directxmath-xmvectorcatmullrom)<br/>     | 使用指定的位置向量，執行 Catmull-Rom 插補。<br/>                                                                                                                           |
| [**XMVectorCatmullRomV**](/windows/win32/api/directxmath/nf-directxmath-xmvectorcatmullromv)<br/>   | 使用指定的位置向量，執行 Catmull-Rom 插補。<br/>                                                                                                                           |
| [**XMVectorHermite**](/windows/win32/api/directxmath/nf-directxmath-xmvectorhermite)<br/>           | 使用指定的向量執行 Hermite 曲線插補。<br/>                                                                                                                                 |
| [**XMVectorHermiteV**](/windows/win32/api/directxmath/nf-directxmath-xmvectorhermitev)<br/>         | 使用指定的向量執行 Hermite 曲線插補。<br/>                                                                                                                                 |
| [**XMVectorInBounds**](/windows/win32/api/directxmath/nf-directxmath-xmvectorinbounds)<br/>         | 測試給定向量的元件是否在設定界限內。<br/>                                                                                                                                 |
| [**XMVectorInBoundsR**](/windows/win32/api/directxmath/nf-directxmath-xmvectorinboundsr)<br/>       | 測試給定向量的元件是否在特定範圍內，並設定可使用 [**XMComparisonAllTrue**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue)等函數檢查的比較值。<br/> |
| [**XMVectorLerp**](/windows/win32/api/directxmath/nf-directxmath-xmvectorlerp)<br/>                 | 在兩個向量之間執行線性插補。<br/>                                                                                                                                                  |
| [**XMVectorLerpV**](/windows/win32/api/directxmath/nf-directxmath-xmvectorlerpv)<br/>               | 在兩個向量之間執行線性插補。<br/>                                                                                                                                                  |
