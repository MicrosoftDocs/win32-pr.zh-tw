# <a name="xmxdec4-operators"></a>XMXDEC4 運算子

> [!Note]  
> 此處所列的運算子僅適用于 c + +。

 

## <a name="in-this-section"></a>本節內容



| 方法                                                            | 描述                                                                                                           |
|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**運算子 uint32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmxdec4-operatoruint32_t)<br/> | 傳回的實例， `uint32_t` 其中包含 `XMXDEC4` 封裝格式之實例的元件。 <br/> |
| [**運算子 =**](xmxdec4-operator-eq.md)<br/>               | 將向量元件資料指派給目前的實例 `XMXDEC4` 。 <br/>                                       |
