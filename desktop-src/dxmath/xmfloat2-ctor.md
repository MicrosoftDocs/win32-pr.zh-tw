# <a name="xmfloat2-constructors"></a>XMFLOAT2 函式

公開的函式 `XMFLOAT2` 。

結構 [**XMFLOAT2**](/windows/win32/api/directxmath/ns-directxmath-xmfloat2) 結構的實例。

> [!Note]  
> 此處所列的函式僅適用于 c + +。

 

### <a name="overload-list"></a>多載清單




| 建構函式 | 描述 | 
|-------------|-------------|
| [<strong>XMFLOAT2 () </strong>](/windows/win32/api/directxmath/nf-directxmath-xmfloat2-xmfloat2(constxmfloat2_)) | 的預設函數 <code>XMFLOAT2</code> 。 <br /> [<strong>XMFLOAT2</strong>](/windows/win32/api/directxmath/ns-directxmath-xmfloat2)的預設的函式。 <br /><blockquote>[!Note]<br />這個函式僅適用于 c + +。</blockquote><br /> | 
| [<strong>XMFLOAT2 (float，float) </strong>](/windows/win32/api/directxmath/nf-directxmath-xmfloat2-xmfloat2(float_float)) | <code>XMFLOAT2</code>從兩個引數初始化的新實例 <code>float</code> 。 <br /> 這個函式會從兩個引數初始化 [<strong>XMFLOAT2</strong>](/windows/win32/api/directxmath/ns-directxmath-xmfloat2) 的新實例 <code>float</code> 。 <br /><blockquote>[!Note]<br />這個函式僅適用于 c + +。</blockquote><br /> | 
| [<strong>XMFLOAT2 (const float * ) </strong>](/windows/win32/api/directxmath/nf-directxmath-xmfloat2-xmfloat2(constfloat)) | <code>XMFLOAT2</code>從兩個元素陣列引數初始化的新實例 <code>float</code> 。 <br /> 這個函式會從兩個元素陣列引數，初始化 [<strong>XMFLOAT2</strong>](/windows/win32/api/directxmath/ns-directxmath-xmfloat2) 的新實例 <code>float</code> 。 <br /><blockquote>[!Note]<br />這個函式僅適用于 c + +。</blockquote><br /> |
