# <a name="directxmath-library-scalar-functions"></a>DirectXMath 程式庫純量函數

列出 DirectXMath 提供的純量函數。

## <a name="in-this-section"></a>本節內容



| 主題                                                     | 描述                                                          |
|-----------------------------------------------------------|----------------------------------------------------------------------|
| [**XMScalarACos**](/windows/win32/api/directxmath/nf-directxmath-xmscalaracos)<br/>           | 計算浮點數的反余弦值。<br/>        |
| [**XMScalarACosEst**](/windows/win32/api/directxmath/nf-directxmath-xmscalaracosest)<br/>     | 估計浮點數的反余弦值。<br/>       |
| [**XMScalarASin**](/windows/win32/api/directxmath/nf-directxmath-xmscalarasin)<br/>           | 計算浮點數的反正弦值。<br/>          |
| [**XMScalarASinEst**](/windows/win32/api/directxmath/nf-directxmath-xmscalarasinest)<br/>     | 估計浮點數的反正弦值。<br/>         |
| [**XMScalarCos**](/windows/win32/api/directxmath/nf-directxmath-xmscalarcos)<br/>             | 計算弧度角度的余弦值。<br/>                    |
| [**XMScalarCosEst**](/windows/win32/api/directxmath/nf-directxmath-xmscalarcosest)<br/>       | 估計弧度角度的余弦值。<br/>                   |
| [**XMScalarModAngle**](/windows/win32/api/directxmath/nf-directxmath-xmscalarmodangle)<br/>   | 計算-XM \_ pi 和 XM pi 之間的角度 \_ 。<br/>             |
| [**XMScalarNearEqual**](/windows/win32/api/directxmath/nf-directxmath-xmscalarnearequal)<br/> | 判斷兩個浮點值是否幾乎相等。<br/> |
| [**XMScalarSin**](/windows/win32/api/directxmath/nf-directxmath-xmscalarsin)<br/>             | 計算弧度角度的正弦。<br/>                      |
| [**XMScalarSinCos**](/windows/win32/api/directxmath/nf-directxmath-xmscalarsincos)<br/>       | 計算弧度角度的正弦和余弦。<br/>      |
| [**XMScalarSinCosEst**](/windows/win32/api/directxmath/nf-directxmath-xmscalarsincosest)<br/> | 估計弧度角度的正弦和余弦。<br/>     |
| [**XMScalarSinEst**](/windows/win32/api/directxmath/nf-directxmath-xmscalarsinest)<br/>       | 估計弧度角度的正弦。<br/>                     |
