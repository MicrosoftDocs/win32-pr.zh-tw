# <a name="xmint3-extensions"></a>XMINT3 延伸模組

[**XMINT3**](/windows/win32/api/directxmath/ns-directxmath-xmint3) 描述的結構包含三個整數值所組成的3d 向量。

> [!Note]  
> 此處所列的功能只有在使用 c + + 進行開發時才可使用。
