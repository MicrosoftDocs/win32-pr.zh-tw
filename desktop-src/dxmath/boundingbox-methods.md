# <a name="boundingbox-methods"></a>BoundingBox 方法



| 方法                                                              | 描述                                                                                            |
|---------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**包含**](boundingbox-contains.md)<br/>                 | 測試 BoundingBox 是否包含指定的物件。<br/>                                  |
| [**CreateFromPoints**](boundingbox-createfrompoints.md)<br/> | 從點建立 BoundingBox。<br/>                                                          |
| [**相交**](boundingbox-intersects.md)<br/>             | 測試 BoundingBox 與另一個物件的交集。<br/>                                 |
| [**Transform**](boundingbox-transform.md)<br/>               | 轉換 BoundingBox。<br/>                                                                 |
| [**ContainedBy**](/windows/desktop/api/DirectXCollision/nf-directxcollision-boundingbox-containedby)<br/>           | 測試 [**BoundingBox**](/windows/desktop/api/DirectXCollision/ns-directxcollision-boundingbox) 是否包含在指定的錐中。<br/> |
| [**CreateFromSphere**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-createfromsphere)<br/> | 建立夠大的 BoundingBox，以包含指定的 BoundingSphere。<br/>               |
| [**CreateMerged**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-createmerged)<br/>         | 建立夠大的 BoundingBox，以包含兩個指定的 BoundBox 實例。<br/>             |
