# <a name="operator--operators"></a>operator + 運算子

加法運算子。

### <a name="overload-list"></a>多載清單




| 運算子 | 描述 | 
|----------|-------------|
| <a href="/previous-versions/windows/desktop/legacy/ee421395(v=vs.85)"><strong>XMVECTOR：： operator + (XMVECTOR) </strong></a> | 在實例上執行身分識別作業的效能 <code>XMVECTOR</code> 。<br /> <code>operator +</code>會採用<a href="xmvector-data-type.md"><strong>XMVECTOR 資料類型</strong></a>的實例，並傳回的新實例，並將身分 <code>XMVECTOR</code> 識別運算子套用至每個元件。<br /><blockquote>[!Note]<br />此運算子僅適用于 c + +。</blockquote><br /> | 
| <a href="/previous-versions/windows/desktop/legacy/ee421396(v=vs.85)"><strong>XMVECTOR：： operator + (XMVECTOR、XMVECTOR) </strong></a> | 加入兩個實例 <code>XMVECTOR</code> ，並在新的實例中傳回結果。<br /> 會 <code>operator +</code> 加入兩個 <a href="xmvector-data-type.md"><strong>XMVECTOR 資料類型</strong></a>之實例的每個元件，然後在新的實例中傳回結果 <code>XMVECTOR</code> 。 <br /><blockquote>[!Note]<br />此運算子僅適用于 c + +。</blockquote><br /> |
