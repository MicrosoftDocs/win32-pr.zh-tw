# <a name="xmu565-extensions"></a>XMU565 延伸模組

[**XMU565**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmu565) 描述具有5位不帶正負號 x 和 z 元件以及6位不帶正負號之 y 元件的4d 向量。

> [!Note]  
> 此處所列的功能只有在使用 c + + 進行開發時才可使用。
