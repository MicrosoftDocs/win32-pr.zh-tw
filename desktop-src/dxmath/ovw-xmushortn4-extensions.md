# <a name="xmushortn4-extensions"></a>XMUSHORTN4 延伸模組

[**XMUSHORTN4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushortn4) 描述用來將未簽署的正規化值儲存為帶正負號16位整數的4d 向量， (類型 `uint16_t`) 。

> [!Note]  
> 此處所列的功能只有在使用 c + + 進行開發時才可使用。
