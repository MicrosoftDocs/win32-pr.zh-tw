# <a name="xmdec4-extensions"></a>XMDEC4 延伸模組

[**XMDEC4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmdec4)會描述具有 x、y 和 z 元件的4d 向量（以10位帶正負號的整數值表示），並以2位帶正負號的整數值來描述 w 元件。

> [!Note]  
> 此處所列的功能只有在使用 c + + 進行開發時才可使用。
