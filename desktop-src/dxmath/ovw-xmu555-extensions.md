# <a name="xmu555-extensions"></a>XMU555 延伸模組

[**XMU555**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmu555) 會描述具有 x、y 和 z 元件的4d 向量（以5位不帶正負號的整數值表示），以及以1位整數值表示 w 元件。

> [!Note]  
> 此處所列的功能只有在使用 c + + 進行開發時才可使用。
