# <a name="xmubyten2-extensions"></a>XMUBYTEN2 延伸模組

[**XMUBYTEN2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmubyten2) 說明將未簽署的正規化值儲存為不帶正負號的8位 (1 位元組) 整數的2d 向量。

> [!Note]  
> 此處所列的功能只有在使用 c + + 進行開發時才可使用。
