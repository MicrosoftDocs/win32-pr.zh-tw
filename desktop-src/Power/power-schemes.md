# <a name="power-schemes"></a>電源配置

電源配置是設定的集合，可控制電腦的電源使用量。 您可以使用電源配置來減少個別裝置或整個系統的耗電量。 您可以設定選用功能，例如支援休眠、系統必須閒置多少時間才能關閉顯示器，或系統閒置計時器到期時所使用的睡眠狀態。

有幾種方式可以建立電源配置：主控台中的電源選項應用程式、PowerCfg.exe 命令列工具，以及電源配置 API。 如需電源選項或 PowerCfg.exe 的詳細資訊，請參閱作業系統隨附的說明。

下列主題說明電源配置 API 所提供的電源配置支援：

-   [電源原則設定](power-policy-settings.md)
-   [電源配置管理](managing-power-schemes.md)
-   [系統管理員覆寫](administrator-overrides.md)
