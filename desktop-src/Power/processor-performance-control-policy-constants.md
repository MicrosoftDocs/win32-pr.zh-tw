# <a name="processor-performance-control-policy-constants"></a>處理器效能控制原則常數

處理器效能控制原則常數指出適用于電源配置的處理器效能控制演算法。



| 常數/值                                                                                                                                                                                                                             | Description                                                                                                                                                                                                                                                                                                                                                                                                                                       |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="PO_THROTTLE_ADAPTIVE"></span><span id="po_throttle_adaptive"></span><dl> <dt>**PO \_節流 \_**</dt>調整 <dt>3</dt> </dl> | 嘗試使處理器效能符合目前的需求。 這項原則會使用高和低的電壓和頻率狀態。 這項原則會將處理器的效能降至最低，只要沒有足夠的需求，就可以證明較高的電壓。 如果未使用 C3 狀態，並回應熱事件，此原則將會觸及處理器時鐘節流。<br/> |
| <span id="PO_THROTTLE_CONSTANT"></span><span id="po_throttle_constant"></span><dl> <dt>**PO \_節流 \_ 常數**</dt> <dt>1</dt> </dl> | 不允許處理器使用任何高電壓的效能狀態。 除了回應熱事件之外，此原則不會與處理器時鐘節流互動。<br/>                                                                                                                                                                                                                                                                 |
| <span id="PO_THROTTLE_DEGRADE"></span><span id="po_throttle_degrade"></span><dl> <dt>**PO \_節流 \_ 降級**</dt> <dt>2</dt> </dl>    | 不允許處理器使用任何高電壓的效能狀態。 當電池低於特定閾值、未使用 C3 狀態，或回應熱事件時，此原則將會觸及處理器時鐘節流。<br/>                                                                                                                                                                                    |
| <span id="PO_THROTTLE_NONE"></span><span id="po_throttle_none"></span><dl> <dt>**PO \_節流 \_ 無**</dt> <dt>0</dt> </dl>             | 未套用任何處理器效能控制。 此原則一律會以最高的可能效能層級執行處理器。 除了回應熱事件之外，此原則不會與處理器時鐘節流互動。<br/>                                                                                                                                                                                                            |
