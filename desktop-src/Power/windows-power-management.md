# <a name="windows-power-management"></a>Windows電源管理

Windows 電源管理可讓使用者隨時接觸按鈕或按鍵來存取電腦。 它也可確保系統的所有元素（應用程式、裝置和使用者介面）都能充分利用電源管理技術與功能的強大改良功能。

Windows 作業系統使用電源管理硬體使電腦進入低電源 *睡眠狀態*，而不是完全關閉，讓系統可以快速繼續運作。 當電腦處於閒置狀態時，或當使用者按下按鈕來指出目前的工作會話已結束時，作業系統會自動進入睡眠狀態。 對使用者而言，系統似乎已關閉。 處於睡眠狀態時，電腦的處理器不會執行程式碼，也不會為使用者完成任何工作。 不過，您可以啟用系統中來自兩個硬體裝置和系統時鐘的事件，讓系統結束睡眠狀態 (也就是「喚醒」 ) 並快速返回工作狀態。

當電腦處於睡眠狀態時，電腦硬體、系統和電腦上執行的應用程式必須能夠立即回應電源交換器、通訊事件和其他動作。 如果所有應用程式都能正常地處理電源狀態轉換，則使用者會察覺到更精緻且整合的系統。 未處理這些轉換的應用程式可能會在電源關閉時失敗，然後因為資料遺失或可能已移除之裝置的相依性而失敗。

以下是 Windows 電源管理的優點：

-   消除啟動和關機延遲。 當使用者啟動睡眠狀態時，電腦不需要執行完整系統開機，或在使用者起始睡眠狀態時，完全關機。
-   讓自動化工作在電腦處於睡眠狀態時執行。 工作排程器可讓使用者排定要執行的應用程式;即使系統處於睡眠狀態，也可以執行已排程的事件。 工作排程器會使用 [可等候計時器](/windows/desktop/Sync/waitable-timer-objects) ，以確保當應用程式已排程執行時，系統已準備就緒。 如需詳細資訊，請參閱工作排程器隨附的說明檔。
-   啟用每個裝置的電源管理。 未使用的裝置可進入睡眠狀態以節省電源。
-   改善電源效率。 在可攜式電腦上的電源效率特別重要。 減少系統電源耗用量可直接轉譯為較低的能源成本和較長的電池壽命。
-   可讓使用者透過主控台中的電源選項應用程式，建立 [電源配置](power-schemes.md)、設定警示，以及指定電池選項。 作業系統會根據電源原則設定來協調所有電源管理活動。 如需詳細資訊，請參閱電源選項應用程式隨附的說明檔。
