# <a name="power-management-control-codes"></a>電源管理控制碼

下列控制碼適用于 [電池](battery-information.md)。



| 控制程式代碼                                                                  | Description                                        |
|-------------------------------------------------------------------------------|----------------------------------------------------|
| [**IOCTL \_ 電池 \_ 查詢 \_ 資訊**](ioctl-battery-query-information.md) | 抓取各種不同的電池資訊 |
| [**IOCTL \_ 電池 \_ 查詢 \_ 狀態**](ioctl-battery-query-status.md)           | 捕獲電池的目前狀態。      |
| [**IOCTL \_ 電池 \_ 查詢 \_ 標記**](ioctl-battery-query-tag.md)                 | 抓取電池的目前標記。               |
| [**IOCTL \_ 電池 \_ 設定 \_ 資訊**](ioctl-battery-set-information.md)     | 設定各種電池資訊。                  |



 

下列控制項碼是 [背光控制項介面](backlight-control-interface.md)的一部分。



| 控制程式代碼                                                                                 | Description                                                                   |
|----------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------|
| [**IOCTL \_ 影片 \_ 查詢 \_ 顯示 \_ 亮度**](ioctl-video-query-display-brightness.md)     | 抓取目前的 AC 和 DC 背光等級和目前的電源狀態。 |
| [**IOCTL \_ 影片 \_ 查詢 \_ 支援的 \_ 亮度**](ioctl-video-query-supported-brightness.md) | 抓取支援的背光等級。                                     |
| [**IOCTL \_ 影片 \_ 集 \_ 顯示 \_ 亮度**](ioctl-video-set-display-brightness.md)         | 設定目前的 AC 和 DC 背光等級                                   |
