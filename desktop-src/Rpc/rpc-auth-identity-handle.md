# <a name="rpc_auth_identity_handle"></a>RPC \_ 驗證身分 \_ 識別 \_ 控制碼

「 **RPC \_ 驗證身分 \_ 識別 \_ 控制碼** 」資料類型會宣告指向結構的控制碼，該結構包含為遠端程序呼叫指定的用戶端驗證和授權認證。


```C++
typedef void* RPC_AUTH_IDENTITY_HANDLE;
```
