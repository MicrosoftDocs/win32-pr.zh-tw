# <a name="data-and-language-features"></a>資料和語言功能

Microsoft 介面定義語言 (MIDL) 提供一組擴充 C 程式設計語言的功能，以支援遠端程序呼叫。 MIDL 不是 C 的變化;它是強型別正式語言，可讓您控制透過網路傳輸的資料。 MIDL 的設計是為了讓熟悉 C 的開發人員能夠快速地學習。

本主題討論三種語言功能：

-   [強型別](strong-typing.md)
-   [方向 (參數) 屬性](directional-parameter-attributes.md)
-   [資料表示](data-representation.md)
