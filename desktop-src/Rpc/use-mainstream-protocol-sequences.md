# <a name="use-mainstream-protocol-sequences"></a>使用主流通訊協定序列

使用 [**ncacn 的 \_ ip \_ tcp**](/windows/desktop/Midl/ncacn-ip-tcp)、 [**ncacn \_ HTTP**](/windows/desktop/Midl/ncacn-http) 或 [**ncacn \_ np**](/windows/desktop/Midl/ncacn-np) 進行遠端 RPC 呼叫。 使用 [**ncalrpc**](/windows/desktop/Midl/ncalrpc) 進行本機 RPC 呼叫。 這些通訊協定最廣泛使用，並可獲得最多審查和測試。
