# <a name="structs-rpc"></a>RPC) 的結構 (

一般 C 語義適用于基底類型的欄位。 您可以使用類型或 [**欄位 \_ 屬性**](field-attributes.md)來修改更複雜類型的欄位，例如指標、陣列和其他結構化類型。 如需詳細資訊，請參閱 [**結構**](/windows/desktop/Midl/struct)。
