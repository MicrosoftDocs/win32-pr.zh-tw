# <a name="i-rpc"></a>我 (RPC) 

[](a-glos.md) [B](b-glos.md) [C](c-glos.md) [D](d-glos.md) [E](e-glos.md) [F](f-glos.md) G [](v-glos.md) H I J K [L](l-glos.md) [M](m-glos.md) [N](n-glos.md) [O](o-glos.md) [P](p-glos.md) [Q](q.md) [R](r-glos.md) [S](s-glos.md) [T](t-glos.md) [](u-glos.md) [W](w-glos.md) X Y Z

<dl> <dt>

<span id="_rpc_idempotent_glos"></span><span id="_RPC_IDEMPOTENT_GLOS"></span>**冪**
</dt> <dd>

在每次使用相同的輸入參數呼叫時，不會變更狀態並傳回相同資訊的遠端程序呼叫。

</dd> <dt>

<span id="_rpc_input_parameter_glos"></span><span id="_RPC_INPUT_PARAMETER_GLOS"></span>**輸入參數**
</dt> <dd>

包含用戶端應用程式正在傳送至伺服器應用程式之資料的參數。 **In** 屬性指定輸入參數。

</dd> <dt>

<span id="_rpc_idl_glos"></span><span id="_RPC_IDL_GLOS"></span>**介面定義語言 (IDL)**
</dt> <dd>

憑證-DCE 標準語言，用於指定遠端程序呼叫的介面。 另請參閱 [*MIDL*](m-glos.md)。

</dd> <dt>

<span id="_rpc_ip_glos"></span><span id="_RPC_IP_GLOS"></span>**網際網路通訊協定 (IP)**
</dt> <dd>

不需連線的網路層通訊協定。 另請參閱 [*資料包*](d-glos.md)。

</dd> <dt>

<span id="_rpc_ipx_glos"></span><span id="_RPC_IPX_GLOS"></span>**(IPX) 的網路封包交換**
</dt> <dd>

Novell NetWare 通訊協定，它會使用 [*資料包*](d-glos.md) 通訊端，透過區域網路和廣域網路路由傳送資訊封包。

</dd> <dt>

<span id="_rpc_intranet_glos"></span><span id="_RPC_INTRANET_GLOS"></span>**網**
</dt> <dd>

使用網際網路產品和技術的私人網路 (例如，網頁伺服器) ，但無法供外部網際網路使用者使用。

</dd> </dl>
