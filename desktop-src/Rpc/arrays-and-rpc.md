# <a name="arrays-and-rpc"></a>陣列和 RPC

C 和 c + + 程式設計語言基本上提供兩種類型的陣列：單一維度和多維度。 RPC 可讓開發人員使用 MIDL 屬性指定額外的陣列類型，以描述分散式應用程式中陣列的特性。

本節描述 RPC 下可用的陣列類型。 此外，還討論了可供開發人員用來描述單一維度和多維陣列的 MIDL 屬性。 本主題提供下列主題：

-   [陣列的種類](kinds-of-arrays.md)
-   [陣列屬性](array-attributes.md)
-   [多維陣列](multidimensional-arrays.md)
