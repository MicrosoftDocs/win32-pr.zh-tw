# <a name="configuring-the-security-server"></a>設定安全性伺服器

下列程式詳細說明如何設定 Microsoft RPC 的安全性伺服器。

**設定 RPC 的安全性伺服器**

1.  啟動主控台。
2.  在主控台中，按一下 [ **網路** ] 圖示。 [ **網路設定** ] 對話方塊隨即出現。
3.  在 [ **已安裝的網路軟體** ] 清單中，選取 [ **RPC** 設定]。 [ **RPC** 設定] 對話方塊隨即出現。
4.  在 [ **安全性服務提供者** ] 清單中，選取一或多個安全性提供者。
5.  按一下 [確定]  。
