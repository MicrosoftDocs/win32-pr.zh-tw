# <a name="u-rpc"></a>U (RPC) 

[](a-glos.md) [B](b-glos.md) [C](c-glos.md) [D](d-glos.md) [E](e-glos.md) [F](f-glos.md) G H [I](i-glos.md) J K [L](l-glos.md) [M](m-glos.md) [N](n-glos.md) [O](o-glos.md) [P](p-glos.md) [Q](q.md) [R](r-glos.md) [S](s-glos.md) [T](t-glos.md) [](v-glos.md) [W](w-glos.md) X Y Z

<dl> <dt>

<span id="_rpc_udp_glos"></span><span id="_RPC_UDP_GLOS"></span>**使用者資料包協定 (UDP)**
</dt> <dd>

使用無連接 [*資料包*](d-glos.md) 通訊端且在網際網路通訊協定上階層式網路傳輸 ([*IP)*](i-glos.md)。

</dd> <dt>

<span id="_rpc_unbind_glos"></span><span id="_RPC_UNBIND_GLOS"></span>**先**
</dt> <dd>

終止用戶端與伺服器之間邏輯連接的進程。

</dd> <dt>

<span id="_rpc_unique_pointer_glos"></span><span id="_RPC_UNIQUE_POINTER_GLOS"></span>**唯一指標**
</dt> <dd>

可以是 null 或指向現有資料的指標，其值可以在遠端程序呼叫期間變更。 唯一的指標不能有別名。 \[ [Unique](/windows/desktop/Midl/unique) \] 屬性指定唯一指標。 另請參閱 [*完整指標*](f-glos.md)、 [*參考指標*](r-glos.md)。

</dd> <dt>

<span id="_rpc_uuid_glos"></span><span id="_RPC_UUID_GLOS"></span>**通用唯一識別碼 (UUID)**
</dt> <dd>

 (或 [**GUID**](/windows/win32/api/guiddef/ns-guiddef-guid)) 128 位值，用於跨進程通訊以識別實體，例如用戶端和伺服器介面、管理員進入點向量和 RPC 物件。 另請參閱 [uuidgen.exe](/windows)。

</dd> <dt>

<span id="_rpc_unmarshaling_glos"></span><span id="_RPC_UNMARSHALING_GLOS"></span>**送**
</dt> <dd>

跨進程界限傳送的解除封裝參數的處理常式。

</dd> <dt>

<span id="_rpc_uuidgen_glos"></span><span id="_RPC_UUIDGEN_GLOS"></span>**uuidgen.exe**
</dt> <dd>

隨附于平臺軟體發展工具組 (SDK) 的公用程式，它會使用時間值和電腦的網路介面卡識別碼來產生保證是唯一的 [uuid](/windows) 。

</dd> </dl>
