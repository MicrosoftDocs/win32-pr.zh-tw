# <a name="enumerated-types-rpc"></a> (RPC) 的列舉類型

[**列舉**](/windows/desktop/Midl/enum)宣告不會轉譯成 **\# 定義** 語句，因為它是由某些 DCE 編譯器所提供，但會在產生的標頭檔中重現為 C 語言 **列舉** 宣告。
