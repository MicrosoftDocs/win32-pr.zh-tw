# <a name="authentication-services"></a>驗證服務

安全性支援提供者介面 (SSPI) 提供 RPC 的基礎驗證服務。 因此，當您的應用程式指定驗證服務時，它會選取 SSP。 如需 SPPI 目前支援的 Ssp 清單，請參閱 [驗證-服務常數](authentication-service-constants.md)。 如需詳細資訊，請參閱 [ (SSPI) 的安全性支援提供者介面 ](security-support-provider-interface-sspi-.md)。

> [!Note]  
> 條款安全性提供者和驗證服務可互換。
