# <a name="building-rpc-applications"></a>建立 RPC 應用程式

 (RPC) 應用程式中建立分散式遠端程序呼叫的程式會因下列情況而稍有不同：

-   您正在開發的作業系統平臺。
-   目標平臺。
-   您使用的 MIDL 和 C 或 c + + 編譯器版本。
-   您使用的 API 程式庫。

本節討論使用 Microsoft 遠端程序呼叫建立用戶端/伺服器應用程式的流程。 本主題提供下列主題中的資訊：

-   [一般組建程式](general-build-procedure.md)
-   [例外狀況處理](exception-handling.md)
