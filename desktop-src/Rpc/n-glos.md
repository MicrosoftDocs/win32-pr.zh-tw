# <a name="n-rpc"></a>N (RPC) 

[](a-glos.md) [B](b-glos.md) [C](c-glos.md) [D](d-glos.md) [E](e-glos.md) [F](f-glos.md) G [](u-glos.md) [](v-glos.md) H [I](i-glos.md) J K [L](l-glos.md) [M](m-glos.md) N [O](o-glos.md) [P](p-glos.md) [Q](q.md) [R](r-glos.md) [S](s-glos.md) [T](t-glos.md) [W](w-glos.md) X Y Z

<dl> <dt>

<span id="_rpc_name_service_glos"></span><span id="_RPC_NAME_SERVICE_GLOS"></span>**名稱服務**
</dt> <dd>

此服務會將名稱對應至物件，並將名稱/物件組儲存在資料庫中。 例如，RPC 名稱服務會將邏輯名稱對應至系結 [*控制碼*](b-glos.md) ，因此用戶端應用程式可以參考該邏輯名稱，而不是通訊協定順序和網路位址。 另請參閱名稱服務-介面 daemon ([nsid)](/windows)、用戶端目錄服務 ([*cd)*](c-glos.md)、 [*定位器*](l-glos.md)。

</dd> <dt>

<span id="_rpc_nca_glos"></span><span id="_RPC_NCA_GLOS"></span>**(NCA) 的網路運算架構**
</dt> <dd>

分散式運算的指導方針。 RPC 通訊協定會遵循這些指導方針。

</dd> <dt>

<span id="_rpc_nsi_glos"></span><span id="_RPC_NSI_GLOS"></span>**名稱服務獨立 (NSI)**
</dt> <dd>

API 函式的標準，可讓分散式應用程式透過各種名稱服務提供者（例如憑證-DCE Cell Directory 服務或 Microsoft 定位器）來存取 RPC 名稱服務資料庫元素。 另請參閱 [-介面 daemon](/windows) (nsid) 。

</dd> <dt>

<span id="_rpc_nsid_glos"></span><span id="_RPC_NSID_GLOS"></span>**名稱服務-介面 daemon (nsid)**
</dt> <dd>

提供 Microsoft [*定位器*](l-glos.md) 與憑證-DCE Cell Directory 服務名稱服務資料庫之間的介面的服務，以進行 RPC 名稱服務功能。

</dd> <dt>

<span id="_rpc_named_pipe_glos"></span><span id="_RPC_NAMED_PIPE_GLOS"></span>**具名管道**
</dt> <dd>

以連接導向的通訊協定（以伺服器訊息區為基礎） (Smb) 和 [NetBIOS](/windows)，用於伺服器進程與一或多個用戶端進程之間的通訊。

</dd> <dt>

<span id="_rpc_netbeui_glos"></span><span id="_RPC_NETBEUI_GLOS"></span>**NetBIOS 延伸消費者介面 (NetBEUI)**
</dt> <dd>

LAN Manager 原生傳輸通訊協定和網路設備磁碟機。 另請參閱 [NetBIOS](/windows)。

</dd> <dt>

<span id="_rpc_netbios_glos"></span><span id="_RPC_NETBIOS_GLOS"></span>**網路基本輸入/輸出系統 (NetBIOS)**
</dt> <dd>

Microsoft MS-DOS 作業系統、i/o 匯流排和局域網路之間的軟體介面。

</dd> <dt>

<span id="_rpc_ndr_glos"></span><span id="_RPC_NDR_GLOS"></span>**(NDR) 的網路資料表示**
</dt> <dd>

在網路傳輸期間使用的標準格式，與任何特定電腦架構上的資料類型格式無關。 傳送的資料包含指定其 NDR 格式的資訊。

</dd> <dt>

<span id="_rpc_network_address_glos"></span><span id="_RPC_NETWORK_ADDRESS_GLOS"></span>**網路位址**
</dt> <dd>

識別網路上伺服器的位址。

</dd> <dt>

<span id="_rpc_nonencapsulated_union_glos"></span><span id="_RPC_NONENCAPSULATED_UNION_GLOS"></span>**nonencapsulated 聯集**
</dt> <dd>

在判別和等位未緊密系結的情況之下，與封裝聯集較不嚴格的差異聯 [*集*](d-glos.md)。 如果 union 是參數，則判別是另一個參數;如果聯集是結構欄位，判別就是另一個結構欄位。 IDL 關鍵字 \[ [**參數 \_ 是**](/windows/desktop/Midl/switch-is) \] ， \[ [**參數 \_ 類型**](/windows/desktop/Midl/switch-type)會 \] 識別判別及其類型。 另請參閱 [*封裝聯集*](e-glos.md)。

</dd> <dt>

<span id="_rpc_nonidempotent_glos"></span><span id="_RPC_NONIDEMPOTENT_GLOS"></span>**nonidempotent**
</dt> <dd>

指出遠端程序呼叫無法執行一次以上的指標，因為它會傳回不同的值或變更狀態。

</dd> </dl>
