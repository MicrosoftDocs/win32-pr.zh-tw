<dl> <dt>

<span id="_rpc_odl_glos"></span><span id="_RPC_ODL_GLOS"></span>**物件描述語言 (ODL)**
</dt> <dd>

[*MIDL*](m-glos.md)屬性、關鍵字、語句和指示詞的子集，用來定義 OLE Automation 應用程式的類型程式庫。

</dd> <dt>

<span id="_rpc_open_array_glos"></span><span id="_RPC_OPEN_ARRAY_GLOS"></span>**開啟陣列**
</dt> <dd>

符合和變化的陣列;也就是說，其大小和傳輸專案的範圍，都是由其他參數、結構或運算式在執行時間決定。

</dd> <dt>

<span id="_rpc_osf_glos"></span><span id="_RPC_OSF_GLOS"></span>**Open Software Foundation (憑證)**
</dt> <dd>

形成的公司聯盟，形成用來定義分散式運算環境 (DCE) 。

</dd> <dt>

<span id="_rpc_orphaned_memory_glos"></span><span id="_RPC_ORPHANED_MEMORY_GLOS"></span>**孤立的記憶體**
</dt> <dd>

用戶端上先前由伺服器重設為 null 的指標參數所參考的記憶體。

</dd> <dt>

<span id="_rpc_output_parameter_glos"></span><span id="_RPC_OUTPUT_PARAMETER_GLOS"></span>**output 參數**
</dt> <dd>

包含伺服器應用程式傳送至用戶端應用程式之資料的參數。 \[ [Out](/windows/desktop/Midl/out-idl) \] 屬性指定了 output 參數。

</dd> </dl>
