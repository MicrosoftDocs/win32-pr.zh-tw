# <a name="server-side-pipe-implementation"></a>Server-Side 管道執行

使用管道的分散式應用程式的伺服器程式不需要執行任何發送、提取或配置函式。 他們必須包含用戶端可以從遠端叫用的程式，以起始資料傳輸。 在下列主題中，本節會說明如何執行伺服器的遠端程式。

-   [在伺服器上執行輸入管道](implementing-input-pipes-on-the-server.md)
-   [在伺服器上執行輸出管道](implementing-output-pipes-on-the-server.md)
