# <a name="using-more-than-one-context-handle-per-call"></a>每次呼叫使用一個以上的內容控制碼

Windows Vista 和更新版本的作業系統中，可以使用內容控制碼的陣列做為參數。 不過，在應用程式中應該謹慎使用這項功能。 例如，在多個呼叫中使用內容控制碼的情況下，會越來越難以追蹤其用途。
