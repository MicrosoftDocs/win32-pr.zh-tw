<span data-ttu-id="36e61-122">在此範例中，內容資訊為檔案控制代碼。</span><span class="sxs-lookup"><span data-stu-id="36e61-122">In this example, the context information is a file handle.</span></span> <span data-ttu-id="36e61-123">它會持續追蹤檔案中的目前位置。</span><span class="sxs-lookup"><span data-stu-id="36e61-123">It keeps track of the current location in the file.</span></span> <span data-ttu-id="36e61-124">介面會將檔案控制代碼封裝為內容處理常式，並將它當作參數傳遞給遠端程序呼叫。</span><span class="sxs-lookup"><span data-stu-id="36e61-124">The interface packages the file handle as a context handle and passes it as a parameter to remote procedure calls.</span></span> <span data-ttu-id="36e61-125">結構包含檔案名和檔案控制代碼。</span><span class="sxs-lookup"><span data-stu-id="36e61-125">A structure contains the file name and the file handle.</span></span>

``` syntax

typedef [ref] PCONTEXT_HANDLE_TYPE * PPCONTEXT_HANDLE_TYPE;

    [in, string] unsigned char * pszFile);

    [in] PCONTEXT_HANDLE_TYPE phContext,
```
