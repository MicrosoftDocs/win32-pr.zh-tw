# <a name="client-context-reset"></a>用戶端內容重設

當伺服器變成無法使用時，用戶端應用程式可以藉由呼叫 RPC 函數 [**RpcSsDestroyClientCoNtext**](/windows/desktop/api/Rpcndr/nf-rpcndr-rpcssdestroyclientcontext)來釋放其內容資料。
