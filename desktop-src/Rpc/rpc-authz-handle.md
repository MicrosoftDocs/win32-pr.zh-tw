# <a name="rpc_authz_handle"></a>RPC \_ 授權 \_ 控制碼

**RPC \_ AUTHZ \_ 控制碼** 資料類型宣告了授權控制碼。 這個控制碼會指向進行遠端程序呼叫之用戶端應用程式的許可權資訊。


```C++
typedef void* RPC_AUTHZ_HANDLE;
```
