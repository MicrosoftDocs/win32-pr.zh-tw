# <a name="running-the-application"></a>執行應用程式

若要在單一電腦上執行應用程式，請開啟兩個主控台視窗。 在第一個視窗中，輸入

**hellos**

在第二個視窗中，輸入

**helloc**

若要試驗不同的通訊協定順序、端點和其他選項，請從 \\ 平臺軟體發展工具組 (SDK) 的 RPC hello 目錄中的來源檔案，建立範例 hello 應用程式。

> [!Note]  
> **RPC 介面限制** 可能會導致此應用程式失敗，並在 Windows XP Service Pack 2 (SP2) 和更新版本的 Windows 發生 **拒絕存取** 錯誤。 如需詳細資訊，請參閱 [RPC 介面限制](/previous-versions/windows/it-pro/windows-xp/bb457156(v=technet.10)#EGAA)。
