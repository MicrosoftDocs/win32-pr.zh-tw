# <a name="security-rpc"></a>安全性 (RPC) 

隨著分散式應用程式的使用，在用戶端和伺服器部分的應用程式之間進行安全通訊的需求非常重要。 遠端程序呼叫 (RPC) 執行時間程式庫會為用戶端和伺服器提供驗證服務的標準化介面。 伺服器主機系統上的驗證服務會提供 RPC 驗證。 應用程式會使用已驗證的遠端程序呼叫，以確保所有呼叫都來自授權的用戶端。 它們也可以協助確保所有伺服器回復都來自已驗證的伺服器。

本節提供下列各節中已驗證 RPC 的總覽：

-   [RPC 安全性基本知識](rpc-security-essentials.md)
-   [安全性方法](security-methods.md)
