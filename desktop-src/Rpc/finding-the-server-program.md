# <a name="finding-the-server-program"></a>尋找伺服器程式

在用戶端 RPC 執行時間連接到系結控制碼所要求的伺服器主機系統之後，用戶端 RPC 執行時間程式庫會尋找伺服器進程。 若要這樣做，它會查詢伺服器主機系統上的端點對應。 端點對應包含伺服器正在接聽之端點的相關資訊。
