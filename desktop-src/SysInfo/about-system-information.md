# <a name="about-system-information"></a>關於系統資訊

本總覽中所述的函式會取得各種資料。 系統組態資訊描述電腦硬體、電腦名稱稱和使用者名稱。 作業系統配置資訊描述作業系統版本、金鑰目錄和環境變數。 系統參數會描述系統屬性。 系統計量會在顯示區域中提供元素的維度。

下列各節將說明如何取得這些類型的資訊：

-   [硬體設定](hardware-configuration.md)
-   [電腦名稱稱](computer-names.md)
-   [作業系統版本](operating-system-version.md)
-   [作業系統配置](operating-system-configuration.md)
