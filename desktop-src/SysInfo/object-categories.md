# <a name="object-categories"></a>物件類別

系統提供三種類別的物件：使用者、圖形裝置介面 (GDI) 和核心。 系統會使用使用者物件來支援視窗管理、支援圖形的 GDI 物件，以及支援記憶體管理、進程執行和處理序間通訊 (IPC) 的核心物件。 如需建立和使用特定物件的詳細資訊，請參閱相關的總覽。
