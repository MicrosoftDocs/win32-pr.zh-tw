# <a name="handle-and-object-functions"></a>控制碼和物件函式

下列函式可用於控制碼和物件。



| 函式                                             | 描述                                                                                  |
|------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [**CloseHandle**](/windows/win32/api/handleapi/nf-handleapi-closehandle)                   | 關閉開啟的物件控制碼。                                                                |
| [**CompareObjectHandles**](/windows/desktop/api/handleapi/nf-handleapi-compareobjecthandles) | 比較兩個物件控制碼，以判斷它們是否參考相同的基礎核心物件。 |
| [**DuplicateHandle**](/windows/win32/api/handleapi/nf-handleapi-duplicatehandle)           | 複製物件控制碼。                                                                 |
| [**GetHandleInformation**](/windows/win32/api/handleapi/nf-handleapi-gethandleinformation) | 抓取物件控制碼的特定屬性。                                            |
| [**SetHandleInformation**](/windows/win32/api/handleapi/nf-handleapi-sethandleinformation) | 設定物件控制碼的特定屬性。                                                 |
