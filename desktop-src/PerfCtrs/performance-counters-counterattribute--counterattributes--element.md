# <a name="counterattribute-counterattributes-element"></a>counterAttribute (counterAttributes) 元素

識別計數器的屬性，此屬性會描述應該如何顯示計數器。

``` syntax
<xs:element name="counterAttribute"
    type="man:counterAttribute"
 />
```

**CounterAttribute** 元素是由 [**counterAttributes**](performance-counters-counterattributes-complex-type.md)複雜型別定義。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|-------------------------------------|------------------------------------------------------|
| 最低支援的用戶端<br/> | \[僅限 Windows Vista 桌面應用程式\]<br/>       |
| 最低支援的伺服器<br/> | 僅限 Windows Server 2008 \[ desktop 應用程式\]<br/> |
