# <a name="limited-user-access-support"></a>有限的使用者存取支援

只有電腦的系統管理員或效能記錄使用者群組中的使用者可以記錄計數器資料。 只有在使用 [以 **系統管理員身分執行**] 開啟的 [命令提示字元] 視窗中啟動，系統管理員群組中的使用者才能記錄計數器資料。互動式登入會話中的任何使用者都可以查看計數器資料。 不過，非互動式登入會話中的使用者必須在 Performance Monitoring Users 群組中，才能查看計數器資料。

**WINDOWS XP：** 系統管理員群組中的系統管理員或使用者可以記錄和查看計數器資料，而不受限制。
