# <a name="struct-structs-element"></a>結構 (結構) 元素

識別包含父計數器集合中計數器值的結構。

``` syntax
<xs:element name="struct"
    type="man:struct"
 />
```

**Struct** 元素是由 [**結構**](performance-counters-structs-complex-type.md)複雜型別定義。
