# <a name="counterattributes-counter-element"></a>counterAttributes (counter) 元素

識別描述如何顯示計數器的屬性清單。

``` syntax

    type="man:counterAttributes"
 />
```

**CounterAttributes** 元素是由 [**counter**](performance-counters-counter-complex-type.md)複雜型別定義。
