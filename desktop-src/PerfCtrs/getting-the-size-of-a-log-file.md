# <a name="getting-the-size-of-a-log-file"></a>取得記錄檔的大小

[**PdhGetLogFileSize**](/windows/desktop/api/Pdh/nf-pdh-pdhgetlogfilesize) 會傳回指定記錄檔的大小（以位元組為單位）。 當您需要為將接收記錄檔內容的緩衝區配置記憶體時，請使用此函數。
