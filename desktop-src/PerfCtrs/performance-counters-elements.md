# <a name="performance-counters-elements"></a>效能計數器元素

效能計數器架構會定義下列元素：

-   [**計數器 (counterSet)**](performance-counters-counter--counterset--element.md)
-   [**counterAttribute (counterAttributes)**](performance-counters-counterattribute--counterattributes--element.md)
-   [**counterAttributes (計數器)**](performance-counters-counterattributes--counter--element.md)
-   [**計數器**](performance-counters-counters-element.md)
-   [**counterSet (提供者)**](performance-counters-counterset--provider--element.md)
-   [**提供者 (計數器)**](performance-counters-provider--counters--element.md)
-   [**結構 (結構)**](performance-counters-struct--structs--element.md)
-   [**結構 (counterSet)**](performance-counters-structs--counterset--element.md)

此區段包含您在資訊清單中使用的元素名稱;不過，若要取得每個專案的詳細資料，請參閱包含元素的複雜類型。 資訊清單效能計數器區段的根項目是 [**計數器**](performance-counters-counters-element.md) 元素。
