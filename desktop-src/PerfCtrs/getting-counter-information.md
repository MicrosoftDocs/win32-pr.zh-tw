# <a name="getting-counter-information"></a>取得計數器資訊

若要抓取計數器的相關資訊（例如計數器類型、解說文字和路徑），請呼叫 [**PdhGetCounterInfo**](/windows/desktop/api/Pdh/nf-pdh-pdhgetcounterinfoa) 函數。 函數會以 [**PDH \_ 計數器 \_ 資訊**](/windows/desktop/api/Pdh/ns-pdh-pdh_counter_info_a) 結構的形式傳回計數器資訊。
