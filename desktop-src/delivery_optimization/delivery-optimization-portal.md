# <a name="delivery-optimization-do"></a>傳遞最佳化 () 

傳遞最佳化是雲端管理的點對點用戶端，以及可將更新提供給組織網路裝置的下載程式服務。 傳遞最佳化可讓裝置從替代 (來源（例如，網路) 上的其他對等）下載更新（除了 Microsoft 伺服器）。 傳遞最佳化結合本機裝置的部分位，以及 Microsoft 伺服器的部分位，以更新網路環境中的裝置。 預期的結果是頻寬使用量降低，以及更快速的更新程式。

如需詳細資訊，請參閱 [設定 Windows 10 更新傳遞最佳化](/windows/deployment/update/waas-delivery-optimization) 。
