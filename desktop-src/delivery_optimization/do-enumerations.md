# <a name="do-enumerations"></a>進行列舉

[傳遞最佳化 () 介面](do-interfaces.md)會使用下列列舉。

## <a name="in-this-section"></a>本節內容

| 主題 | 描述 |
|-|-|
| [**BG_ERROR_CONTEXT**](bg-error-context.md) | **BG_ERROR_CONTEXT** 列舉會定義常數值，以指定發生錯誤的內容。 |
| [**BG_JOB_PRIORITY**](bg-job-priority-.md) | **BG_JOB_PRIORITY** 列舉會定義常數值，以指定工作的優先權層級。  |
| [**BG_JOB_STATE**](bg-job-state-.md) | **BG_JOB_STATE** 列舉會針對作業的不同狀態定義常數值。 |
| [**BG_JOB_TYPE**](bg-job-type.md) | **BG_JOB_TYPE** 列舉會定義常數值，以指定傳送作業的類型，例如 [下載]。 |
| [**BITS_FILE_PROPERTY_ID**](bits-file-property-id-.md) | **BITS_FILE_PROPERTY_ID** 列舉會指定定義對應至 **BACKGROUNDCOPYFILE** 屬性之識別碼值的值。 |
| [**BITS_JOB_PROPERTY_ID**](bits-job-property-id.md) | **BITS_JOB_PROPERTY_ID** 列舉會指定 DO 作業的屬性識別碼。 此列舉會在 [**BITS_JOB_PROPERTY_VALUE**](bits-job-property-value-.md) 聯集內使用，以決定包含在聯集內的數值型別。 |
| [**BITS_JOB_TRANSFER_POLICY**](bits-job-transfer-policy-.md) | **BITS_JOB_TRANSFER_POLICY** 列舉會定義對應至 DO 屬性的識別碼值。 |
| [**DeliveryOptimizationFileProperty**](deliveryoptimizationfileproperty.md) | 指定 DO 檔案的選擇性屬性識別碼。 |
| [**DODownloadCostPolicy**](./deliveryoptimizationdownloadtypes/ne-deliveryoptimizationdownloadtypes-dodownloadcostpolicy.md) | 指定與 **DODownloadProperty_CostPolicy** 屬性相關聯之成本原則選項的識別碼。 |
| [**DODownloadProperty**](./deliveryoptimizationdownloadtypes/ne-deliveryoptimizationdownloadtypes-dodownloadproperty.md) | 指定「進行下載」作業的屬性識別碼。 |
| [**DODownloadPropertyEx**](./dodownloadinternal/ne-dodownloadinternal-dodownloadpropertyex.md) | 指定「進行下載」作業的擴充屬性識別碼。 |
| [**DODownloadState**](./deliveryoptimizationdownloadtypes/ne-deliveryoptimizationdownloadtypes-dodownloadstate.md) | 指定目前下載狀態的識別碼，這是 **DO_DOWNLOAD_STATUS** 結構的一部分。 |
| [**DownloadMode**](downloadmode.md) | 定義傳遞最佳化使用的不同下載模式。  |
| [**SwarmStatus**](swarmstatus.md) | 定義傳遞優化用戶端內檔案的狀態。 |
