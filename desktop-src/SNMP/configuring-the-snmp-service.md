# <a name="configuring-the-snmp-service"></a>設定 SNMP 服務

若要設定 SNMP 服務，您必須知道網路使用的群體名稱、每個社區的設陷目的地，以及每個 SNMP 管理主機的電腦名稱稱或 IP 位址。

下表列出的主題包含設定 SNMP 服務的相關資訊。

| 主題                                                                        | 目錄                                                                                                                              |
|------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| [Community名字](community-names.md)                                       | 描述在 SNMP 中使用群體名稱的方式。                                                                                         |
| [主機名稱和 IP 位址](host-names-and-ip-addresses.md)               | 描述在 SNMP 中使用主機名稱和 IP 位址的方式。                                                                             |
| [設定 SNMP 安全性](configuring-snmp-security.md)                   | 描述 SNMP 安全性功能。                                                                                                     |
| [設定 SNMP 群組原則](configuring-snmp-group-policy.md)           | 描述與使用 Microsoft Management Console 為 SNMP 設定以登錄為基礎的原則設定相關聯的子機碼。 |
| [設定 SNMP 代理程式資訊](configuring-snmp-agent-information.md) | 描述在 SNMP 中使用代理程式資訊的方式。                                                                                       |
