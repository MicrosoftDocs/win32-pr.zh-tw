# <a name="community-names"></a>Community名字

群體名稱可識別 SNMP 管理員和代理程式的集合。 使用群體名稱可針對接收要求及起始陷阱作業的代理程式和管理員，提供基本的安全性和內容檢查。 代理程式不會接受來自社區外部管理員的要求。

請注意，除了受信任的網路以外，標準的安全性建議建議不要使用 SNMP，因為依設計的通訊協定提供最基本的安全性。 如需詳細資訊，請參閱設定 [SNMP 安全性](configuring-snmp-security.md)。
