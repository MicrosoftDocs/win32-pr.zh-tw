# <a name="rfcs-relevant-to-winsnmp"></a>與 WinSNMP 相關的 Rfc

TCP/IP 標準是在 (Rfc) 的評論中定義，這些要求是由網際網路工程任務推動小組 (IETF) 發佈。 下表列出與 WinSNMP 功能相關的 Rfc。



| RFC 編號 | 標題                                                                                                  |
|------------|--------------------------------------------------------------------------------------------------------|
| 1155       | 「以 TCP/IP 為基礎的網際網路的管理資訊的結構和識別」                    |
| 1157       | 「簡單的網路管理通訊協定 (SNMP) 」                                                          |
| 1213       | 「TCP/IP 型網際網路的網路管理管理資訊基礎： MIB-II」                 |
| 1901       | 「以 Community 為基礎的 SNMPv2 簡介」                                                               |
| 1902       | 「簡易網路管理通訊協定第2版的管理資訊結構 (SNMPv2) 」 |
| 1903       | 「簡易網路管理通訊協定第2版的文字慣例 (SNMPv2) 」                 |
| 1904       | 「簡易網路管理通訊協定第2版的一致性聲明 (SNMPv2) 」              |
| 1905       | 「簡易網路管理通訊協定第2版的通訊協定作業 (SNMPv2) 」                 |
| 1906       | 「簡易網路管理通訊協定第2版的傳輸對應 (SNMPv2) 」                  |
| 1907       | 「簡易網路管理通訊協定第2版的管理資訊基礎 (SNMPv2) 」         |
| 1908       | 「在第1版與第2版的網際網路之間共存-標準網路管理架構」    |
| 2089       | 「V2ToV1 將 SNMPv2 對應至 bi 語言的 SNMP 代理程式內的 SNMPv1」                                     |
