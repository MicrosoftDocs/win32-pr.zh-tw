# <a name="winsnmp-sessions"></a>WinSNMP 會話

*會話* 是呼叫的 winsnmp 應用程式與 Microsoft WinSNMP 實行之間的資源和通訊管理的基本單位。 建議應用程式使用會話來組織其作業，並降低對執行中資源的需求。

如需詳細資訊，請參閱 [開啟和關閉 WinSNMP 會話](opening-and-closing-a-winsnmp-session.md)。
