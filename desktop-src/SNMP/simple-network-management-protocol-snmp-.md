# <a name="simple-network-management-protocol-snmp"></a>簡易網路管理通訊協定 (SNMP)

\[SNMP 可用於 [需求] 區段中指定的作業系統。 它在後續版本中可能會變更或無法使用。 相反地，請使用[Windows 遠端管理](/windows/desktop/WinRM/portal)，也就是 Microsoft 對 ws-atomictransaction 的實。\]

簡易網路管理通訊協定 (SNMP) 是網際網路標準通訊協定，可在管理主控台應用程式（例如 HP Openview、Novell NMS、IBM NetView 或 Sun Net Manager）和受控實體之間交換管理資訊。 受管理的實體可以包括主機、路由器、橋接器和集線器。
