# <a name="snmp-data-types"></a>SNMP 資料類型

\[SNMP 可用於 [需求] 區段中指定的作業系統。 它在後續版本中可能會變更或無法使用。 相反地，請使用 [Windows 遠端管理](/windows/desktop/WinRM/portal)，也就是 MICROSOFT 對 ws-atomictransaction 的實。\]

SNMP API 會定義下列資料類型。



| 資料類型                      | 描述                                                            |
|--------------------------------|------------------------------------------------------------------------|
| **AsnObjectName**<br/>   | 請參閱 [**AsnObjectIdentifier**](/windows/desktop/api/Snmp/ns-snmp-asnobjectidentifier)。<br/> |
| **AsnObjectSyntax**<br/> | 請參閱 [**AsnAny**](/windows/desktop/api/Snmp/ns-snmp-asnany)。<br/>                           |
