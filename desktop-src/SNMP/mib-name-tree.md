# <a name="mib-name-tree"></a>MIB 名稱樹狀結構

MIB 物件識別碼的命名空間為階層式。 它是結構化的，因此可以將全域唯一名稱指派給每個可管理的物件。

命名空間部分的授權單位會指派給個別組織。 這可讓組織指派名稱，而不需為每個指派的網際網路授權單位進行參考。 例如，指派給 Microsoft 的命名空間是 1.3.6.1.4.1.311.，在 MSFT 中定義。Mib。 Microsoft 有權將名稱指派給該命名空間下任何位置的物件。

階層中的物件識別碼會以從根目錄開始的 subidentifiers 序列撰寫，並在物件結尾。 Subidentifiers 會以句號分隔。
