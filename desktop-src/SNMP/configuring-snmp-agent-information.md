# <a name="configuring-snmp-agent-information"></a>設定 SNMP 代理程式資訊

SNMP 代理程式資訊可讓您指定使用者的相關批註以及電腦的實體位置，並指出要報告的服務類型。 可以報告的服務類型是以電腦的設定為基礎。
