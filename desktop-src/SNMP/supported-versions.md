# <a name="supported-versions"></a>支援的版本

\[SNMP 可用於 [需求] 區段中指定的作業系統。 它在後續版本中可能會變更或無法使用。 相反地，請使用 [Windows 遠端管理](/windows/desktop/WinRM/portal)，也就是 MICROSOFT 對 ws-atomictransaction 的實。\]

Windows Vista 和 Windows Server 2008 支援 SNMPv1 和 SNMPv2c。 本主題提供每個 SNMP 版本的基本資訊。

簡易網路管理通訊協定第1版 (SNMPv1) 是最早的通訊協定版本。 SNMPv1 是由 RFC 1157、RFC 1155 和 RFC 1212 所定義。 針對 SNMPv1，安全性是以社區字串為基礎。

簡易網路管理通訊協定2c 版 (SNMPv2c) 是由 RFC 1901、RFC 1905 和 RFC 1906 所定義。 SNMPv2c 會更新在 SNMPv2p 中定義的通訊協定作業和資料類型。 SNMPv2c （稱為「以字串為基礎的 SNMPv2）」使用 SNMPv1 的以社區為基礎的安全性。

Windows 2000 和更新版本的用戶端和伺服器作業系統平臺（包括 Windows XP 和 Windows Server 2003）都支援 SNMPv2x。
