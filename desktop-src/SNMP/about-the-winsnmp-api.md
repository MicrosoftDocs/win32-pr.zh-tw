# <a name="about-the-winsnmp-api"></a>關於 WinSNMP API

\[SNMP 可用於 [需求] 區段中指定的作業系統。 它在後續版本中可能會變更或無法使用。 相反地，請使用[Windows 遠端管理](/windows/desktop/WinRM/portal)，也就是 Microsoft 對 ws-atomictransaction 的實。\]

WinSNMP API 是一種介面，可用於開發啟用 SNMP 的網路管理應用程式。

WinSNMP API 提供下列功能：

-   網路管理應用程式的 SNMP 啟用技術
-   SNMP 第1版 (SNMPv1) 和 SNMP 2C 版 (SNMPv2C) 支援

除了 SNMP 管理員作業之外，WinSNMP API 版本2.0 也支援 SNMP 代理程式作業。

WinSNMP API 支援32位的應用程式，而且會在單一線程和多執行緒環境中執行。 在 Windows 2000 作業環境或更新版本中執行的應用程式，可使用支援的 WinSNMP API。
