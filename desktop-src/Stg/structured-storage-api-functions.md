# <a name="structured-storage-api-functions"></a>結構化儲存體 API 函數

結構化儲存體的某些 API 函式是協助程式函式，可對其他 API 函式和介面方法執行一連串的呼叫。 您可以使用這些 helper 函式作為簡短的剪下。

其他 API 函式可讓您存取 COM 的結構化儲存體、複合檔案的執行。

另外還有另一組 API 函數可讓您將 OLE 1 物件轉換成結構化儲存區。 您可以使用這些函式來判斷物件類別是否來自 OLE 1，以及轉換 OLE 1 和目前 COM 儲存格式之間的物件。

最後，有 API 函式可用於轉換和模擬其他物件。 這些函式會為一個伺服器應用程式提供服務，以處理來自另一個應用程式的資料。 資料可以轉換成原生格式的伺服器應用程式，方法是從其原始格式讀取資料，但以物件應用程式的原生格式寫入資料。 或者，資料可以維持原始格式，而伺服器應用程式會以其原始格式讀取和寫入資料。

如需詳細資訊，請參閱下一節的 [結構化儲存體存取模式](structured-storage-access-modes.md)。
