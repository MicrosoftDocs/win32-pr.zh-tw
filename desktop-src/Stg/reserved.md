# <a name="reserved"></a>保留

**DWORD** 資料類型的值已保留供日後使用。 屬性集的寫入器應該將此值設為 1;屬性集的讀取器應確定此值至少為1。 此指導方針的例外狀況是，針對 [DocumentSummaryInformation 和使用者配置的屬性集](the-documentsummaryinformation-and-userdefined-property-sets.md)，這個值可能是2。
