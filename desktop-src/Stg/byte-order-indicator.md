# <a name="byte-order-indicator"></a>Byte-Order 指標

位元組順序指標是 **單字** 資料類型，而且應該一律保留值0xFFFE。 這與 Unicode 位元組順序指標相同。 此值一律會以 Intel 位元組順序寫入，因此會以0xFE、0xFF 的形式出現在檔案或資料流程中。
