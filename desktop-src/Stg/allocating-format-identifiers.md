# <a name="allocating-format-identifiers"></a>配置格式識別碼

系統會建立 FMTIDs，並以與 COM Clsid 和介面識別碼相同的方式來表示。 若要建立唯一的 FMTID，請使用 Win32® API 中包含的 UUIDGEN.EXE 程式。
