# <a name="class-identifier"></a>類別識別碼

 (CLSID) 的類別識別碼是可顯示及/或提供以程式設計方式存取屬性值的類別。 如果沒有這類類別，建議您將此值設定為與 FMTID 相同 (如下所示) 。 或者，您可以將此值設定為零。不過，使用 FMTID 可讓未來有更大的彈性。
