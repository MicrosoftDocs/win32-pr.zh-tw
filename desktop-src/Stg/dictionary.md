# <a name="dictionary"></a>Dictionary

[**IPropertyStorage：： WritePropertyNames**](/windows/desktop/api/Propidl/nf-propidl-ipropertystorage-writepropertynames) 是使用 [屬性識別碼0字典](/windows/desktop/Stg/reserved-property-identifiers)所執行。 無法透過 [**IPropertyStorage：： ReadMultiple**](/windows/desktop/api/Propidl/nf-propidl-ipropertystorage-readmultiple) 或 [**IPropertyStorage：： WRITEMULTIPLE**](/windows/desktop/api/Propidl/nf-propidl-ipropertystorage-writemultiple)存取屬性識別碼0。
