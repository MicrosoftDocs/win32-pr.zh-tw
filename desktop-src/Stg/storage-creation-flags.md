# <a name="storage-creation-flags"></a>儲存體建立旗標

儲存體建立旗標指定當現有的儲存體、資料流程或 lockbytes 物件的名稱與您正在建立的新儲存體或資料流程物件相同時，COM 應採取的動作。 預設值是傳回錯誤訊息，而不是建立新的物件。 您只能在指定的建立呼叫中使用上述其中一個旗標。
