# <a name="access-permission-flags"></a>存取權限旗標

存取權限旗標指定用戶端應用程式對開啟物件的存取類型：讀取、寫入或讀取/寫入。 [讀取] 許可權可讓應用程式讀取資料流程的內容，但不會寫入資料流程的變更。 寫入權限可讓應用程式呼叫函式，以認可物件儲存的變更，而不是讀取資料流程的內容。 讀取/寫入權限可讓應用程式讀取物件的資料流程或寫入物件的儲存體。
