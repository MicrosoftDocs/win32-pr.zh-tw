# <a name="shared-access-flags"></a>共用存取旗標

共用存取旗標會指定其他應用程式對您的應用程式開啟之物件的存取程度。 您可以拒絕讀取權限、拒絕寫入存取權，或拒絕所有存取權。 您也可以明確指定不會拒絕存取的類型。
