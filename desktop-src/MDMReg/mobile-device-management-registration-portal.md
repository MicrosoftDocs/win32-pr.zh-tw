# <a name="mobile-device-management-mdm-registration"></a>行動裝置管理 (MDM) 註冊

## <a name="purpose"></a>用途

產業趨勢的開發是讓員工將其個人行動運算裝置連接到公司網路和資源 (內部部署或透過雲端) 來執行工作場所工作。 此趨勢需要支援輕鬆設定網路和資源，讓員工可以在公司註冊個人裝置，以進行工作相關的用途。 支援輕鬆設定的應用程式和技術也必須讓 IT 專業人員管理與將未受管理的裝置連線到公司網路相關聯的風險。 行動裝置管理 (MDM) 註冊會將裝置註冊到管理服務中。
