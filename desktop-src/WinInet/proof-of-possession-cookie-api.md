# <a name="proof-of-possession-cookie-api"></a>擁有權 cookie API 的證明

可讓您建立擁有權的 cookie 證明，以支援 OAuth 網際網路驗證通訊協定所驗證之 web 應用程式的單一登入。 您可以使用此 API 所傳回的資訊來執行 Microsoft 帳戶的單一登入。

|-----------------------------------------------------------------------------------------------|------------------------------------------------------------------|
| [**IProofOfPossessionCookieInfoManager**](/windows/desktop/api/ProofOfPossessionCookieInfo/nn-proofofpossessioncookieinfo-iproofofpossessioncookieinfomanager)<br/> | 支援建立擁有權的 cookie 證明。<br/> |
