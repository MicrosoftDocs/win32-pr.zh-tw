# <a name="dictionaries"></a>字典

字典允許 token 化通用節點資料，例如二進位 XML 格式內的元素/屬性名稱、命名空間和一般文字。


您可以將 [**字典>**](/windows/desktop/api/WebServices/ns-webservices-ws_xml_dictionary) 指定為 [**WS \_ XML \_ 字串**](/windows/desktop/api/WebServices/ns-webservices-ws_xml_string) 類型的一部分。 單一 **字典>** 中的所有字串都必須是唯一的。

所有編碼都支援字典識別碼， (不是) 的二進位 XML 格式。

下列函式會搭配字典使用：

-   [**WsGetDictionary**](/windows/desktop/api/WebServices/nf-webservices-wsgetdictionary)
