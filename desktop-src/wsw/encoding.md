# <a name="encoding-windows-web-services"></a> (Windows Web 服務) 編碼

編碼是 XML 和線性電傳格式之間的轉換。 解碼是反向轉換。


編碼是使用 [**WS \_ 編碼**](/windows/desktop/api/WebServices/ne-webservices-ws_encoding) 常數來識別。
