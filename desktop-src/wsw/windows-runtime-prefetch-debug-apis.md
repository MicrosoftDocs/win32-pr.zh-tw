# <a name="prefetch-debug-apis-for-windows-store-apps"></a>Windows Store 應用程式的預先提取 debug api

下列 api 用來在執行 [**ContentPrefetcher**](/uwp/api/Windows.Networking.BackgroundTransfer.ContentPrefetcher)類別的 Windows 存放區應用程式中，用來偵測和測試內容預先提取行為。



| 介面                                                              | 描述                                                                                                                                                                      |
|------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IContentPrefetcherTaskTrigger**](/windows/desktop/api/IContentPrefetcherTaskTrigger/nn-icontentprefetchertasktrigger-icontentprefetchertasktrigger) | 使用此介面來確認已針對此背景工作註冊安裝的 Windows 存放區應用程式套件，並手動觸發其內容預先提取作業。<br/> |
