# <a name="windows-web-services-api"></a>WindowsWeb 服務 API

## <a name="purpose"></a>目的

WWSAPI 是 SOAP 的原生程式碼執行，可支援一組 WS 和來提供核心網路通訊功能 \* 。網路 \* 協定系列。 WWSAPI 是設計來供屬於下列其中一個類別的元件/應用程式使用：

-   原生程式碼的要求
-   需要基本的相依性
-   需要較短的啟動時間
-   記憶體限制的環境

## <a name="developer-audience"></a>開發人員對象

WindowsWeb 服務 API (WWSAPI) 提供 C/c + + 應用程式開發介面 (API) ，以建立以 SOAP 為基礎的 web 服務和用戶端。 使用此 API 時，對 web 服務和與其相關聯的通訊協定有基本的瞭解。

## <a name="run-time-requirements"></a>執行階段需求求

WindowsWeb 服務 API (WWSAPI) 是 Windows 7 和 Windows Server 2008 R2 或更新版本的 Microsoft Windows 的作業系統元件。
