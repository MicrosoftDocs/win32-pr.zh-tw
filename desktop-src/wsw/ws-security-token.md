# <a name="ws_security_token"></a>WS \_ 安全性 \_ 權杖

代表 [安全性權杖](security-processing-results.md)的不透明控制碼。


```C++
typedef struct _WS_SECURITY_TOKEN WS_SECURITY_TOKEN;
```



## <a name="remarks"></a>備註

此物件不是安全線程。 如需詳細資訊，請參閱 [執行緒安全性](thread-safety.md)。
