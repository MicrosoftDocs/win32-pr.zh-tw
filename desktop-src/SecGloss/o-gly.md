# <a name="o-security-glossary"></a>O (安全性詞彙) 

[](a-gly.md) [B](b-gly.md) [C](c-gly.md) [D](d-gly.md) [E](e-gly.md) [](u-gly.md) [](v-gly.md) F [G](g-gly.md) [H](h-gly.md) [I](i-gly.md) J [K](k-gly.md) [L](l-gly.md) [M](m-gly.md) [N](n-gly.md) O [P](p-gly.md) Q [R](r-gly.md) [S](s-gly.md) [T](t-gly.md) [W](w-gly.md) [X](x-gly.md) Y Z

<dl> <dt>

<span id="_security_object_identifier_gly"></span><span id="_SECURITY_OBJECT_IDENTIFIER_GLY"></span>**物件識別碼**
</dt> <dd>

 (OID) 可唯一識別物件類別或屬性的數位。 物件識別碼會表示為以點分隔的十進位字串，例如1.2.3.4。 物件識別碼會組織成全域階層。 國家註冊授權單位會向個人或組織發出根物件識別項，這些個人或組織會管理其根物件識別項之下的階層。

另請參閱 [*抽象語法標記法 1*](a-gly.md)、 [*屬性*](a-gly.md)。

</dd> <dt>

<span id="_security_ocsp_gly"></span><span id="_SECURITY_OCSP_GLY"></span>**OCSP**
</dt> <dd>

請參閱 *線上憑證狀態通訊協定*。

</dd> <dt>

<span id="_security_ofb_gly"></span><span id="_SECURITY_OFB_GLY"></span>**OFB**
</dt> <dd>

查看 *輸出意見* 反應。

</dd> <dt>

<span id="_security_oid_gly"></span><span id="_SECURITY_OID_GLY"></span>**老**
</dt> <dd>

請參閱 *物件識別碼*。

</dd> <dt>

<span id="_security_online_certificate_status_protocol_gly"></span><span id="_SECURITY_ONLINE_CERTIFICATE_STATUS_PROTOCOL_GLY"></span>**線上憑證狀態通訊協定**
</dt> <dd>

 (OCSP) 用來取得 x.509 數位憑證之撤銷狀態的網際網路通訊協定。

</dd> <dt>

<span id="_security_opaque_blob_gly"></span><span id="_SECURITY_OPAQUE_BLOB_GLY"></span>**不透明的 BLOB**
</dt> <dd>

用來儲存工作階段金鑰的位元組序列。 不透明的 Blob 包含基本金鑰材質和所有目前的狀態資訊。 這包括 salt 值、初始化向量和索引鍵資料表等資訊。 不透明 Blob 的格式已解除發佈。 每個 CSP 廠商都會決定自己的 BLOB 格式，此格式應包含使用某種對稱金鑰來加密不透明的 Blob。

</dd> <dt>

<span id="_security_outer_content_gly"></span><span id="_SECURITY_OUTER_CONTENT_GLY"></span>**外部內容**
</dt> <dd>

某些封裝資料的增強功能。 本文主要用於討論 PKCS 7 訊息中 (內部內容) 的增強型資料 \# 。

</dd> <dt>

<span id="_security_output_feedback_gly"></span><span id="_SECURITY_OUTPUT_FEEDBACK_GLY"></span>**輸出意見反應**
</dt> <dd>

 (OFB) 一個區塊加密模式，它會使用與加密意見反應 (CFB) 模式類似的意見反應。 這兩種模式之間的唯一差異在於如何填滿 shift 註冊。

另請參閱 [*加密意見*](c-gly.md)反應。

</dd> </dl>
