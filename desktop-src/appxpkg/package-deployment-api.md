# <a name="package-deployment-api"></a>套件部署 API

瞭解套件部署 API，您可以用來在系統上安裝、更新及卸載應用程式套件。 每個應用程式套件都包含構成 Windows 應用程式的檔案，以及描述要 Windows 之軟體的資訊清單檔案。
