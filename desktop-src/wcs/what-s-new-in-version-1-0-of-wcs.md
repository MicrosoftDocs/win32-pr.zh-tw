# <a name="whats-new-in-version-10-of-wcs"></a>WCS 1.0 版的新功能

 (ICM) 在 Microsoft Windows 95 中提供的映射色彩管理版本1.0，並在 Windows 裝置內容中提供基本的色彩管理功能。

ICM 2.0 版包含在 Windows 98、Windows Millennium Edition、Windows 2000 和 Windows XP 中，並包含在裝置內容以外執行色彩管理的各種新功能。 這些新函式適用于需求較高的色彩管理需求，並讓應用程式更能掌控色彩管理程式。

Microsoft Windows 的 Windows Vista 版本包含其他功能。

-   選擇靜態/整合式和動態/模組化的 CMM/設定檔工作流程
    -   HeidelbergerDruckmaschinen 在 Win98 ICM2 中提供的靜態，Canon Inc. 所提供的動態功能。使用 Canon 的 Kyuanos 技術
    -   基準裝置、色彩外觀和 gamut 對應模型設定檔
    -   黑色世代支援、
    -   黑色保留支援，
    -   高動態範圍支援，
    -   每個通道精確度的浮點數/大於16位，
    -   預設寬範圍工作空間設定檔支援，
    -   彈性控制裝置內的目標裝置特性，不同于裝置間的特性，
    -   能夠在兩個裝置之間控制單一 gamut 對應演算法
    -   支援在色彩管理的工作流程中，將來源裝置主要複本對應至目的地主要複本。
-   二進位和文字色彩配置檔案格式的選擇
    -   二進位檔符合以 ICC TIFF 標記為基礎的配置檔案格式和使用 XML 架構的文字
-   ICC 第4版支援，
-   中央色彩管理控制台，
-   維護應用程式和裝置的舊版相容性
-   透過協力廠商外掛程式裝置模型和 gamut 對應模型的擴充性
-   支援 DM 與 GMM 外掛程式之間的直接通訊，藉此支援 advanced device/gamut 對應，
-   靜態 (ICM2) CMM 現在預設為相對色度意圖，而不是絕對色度意圖。
