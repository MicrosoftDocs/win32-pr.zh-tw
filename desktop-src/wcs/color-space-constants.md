# <a name="color-space-constants"></a>色彩空間常數

表示定義設定檔資料之色彩空間的簽章值。 可以是下列任何一個值。



| Profile 類別     | 簽名         |
|-------------------|-------------------|
| XYZ               | 空間 \_ XYZ        |
| 實驗室               | 空間 \_ 實驗室        |
| 親愛的               | 空間 \_ Luv        |
| YCbCr             | 空間 \_ YCbCr      |
| Yxy               | 空間 \_ Yxy        |
| RGB               | 空格 \_ RGB        |
| 灰度        | 空格 \_ 灰色       |
| HSV               | 空間 \_ HSV        |
| HLS               | 空間 \_ HLS        |
| CMYK              | 空格 \_ CMYK       |
| CMY               | 空間 \_ CMY        |
| 一般2通道 | 空間 \_ 2 \_ 頻道 |
| 一般3通道 | 空間 \_ 3 \_ 通道 |
| 一般4通道 | 空間 \_ 4 \_ 通道 |
| 一般5通道 | 空間 \_ 5 \_ 通道 |
| 一般6通道 | 空間 \_ 6 \_ 通道 |
| 一般7通道 | 空間 \_ 7 \_ 通道 |
| 一般8通道 | 空間 \_ 8 \_ 通道 |
