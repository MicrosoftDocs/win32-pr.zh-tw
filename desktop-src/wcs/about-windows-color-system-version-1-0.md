# <a name="about-windows-color-system-version-10"></a>關於 Windows Color System 版本1。0

Microsoft Windows Color System (WCS) 技術可確保色彩影像、圖形或文字物件在任何裝置上都能盡可能接近其原始意圖，儘管裝置之間的映射技術和色彩功能有所差異。 無論您是掃描彩色掃描器上的影像或其他圖形、透過網際網路進行下載、在螢幕上進行觀看或編輯，或是將它輸出到紙張、電影或其他媒體，WCS 1.0 都能協助您保持其色彩一致且正確。

WCS 1.0 是 Microsoft Windows 不可或缺的一部分，因為 WCS 1.0 以一組 WIN32 API 函式的形式內建在 Windows 系列的作業系統中，因此可供任何應用程式、設備磁碟機、裝置校正工具或色彩管理模組 (的 CMM) 。

映射色彩管理的版本 1.0 (ICM) 已在 Microsoft Windows 95 中提供，並在 Windows 裝置內容中提供基本的色彩管理功能。

ICM 2.0 版包含在 Windows 98、Windows Millennium Edition、Windows 2000 和 Windows XP 中，以及在裝置內容以外執行色彩管理的包含函式。 這些函式適用于需求較高的色彩管理需求，並讓應用程式更能掌控色彩管理程式。

WCS 1.0 版包含在 Windows Vista 中，並包含各種新功能，可針對廠商的靈活、透明度、可預測性和 extensiblity 提供顯著的改進。

使用 WCS 1.0 可受益于哪些類型的應用程式？ 現今在 Windows Vista 環境中執行的應用程式幾乎都是如此。 基本色彩管理功能已成為各種應用程式的需求。

過去幾年來，色彩顯示和列印已經過改良，現在可讓您不只是在桌上型電腦發行，也可以在各式各樣的資料和展示應用程式中使用相片實際色彩影像和複雜色彩圖形。 COM 和 ActiveX 這類物件技術在幾乎每一種資料空間中都有內嵌的色彩物件。

服裝目錄、線上藝術、家族相片專輯、商務標誌、圖表、圖形、簡報、預覽列印和色彩模擬，都只是一些色彩重要的日常應用程式範例。

因此，有越來越多的使用者要求其應用程式能夠正確地顯示色彩。 色彩轉譯不良會毀損對使用者日益重要的影像和圖形。

在基本層級中，幾乎所有應用程式都應該能夠自動調整色彩，使其輸出在不同的監視器和印表機上看起來相同。 WCS 1.0 提供一組函式來傳遞這種對使用者而言是透明的色彩管理，並且在應用程式中需要很少的額外負荷。

在較高的層級上，WCS 1.0 會提供額外的功能，以提供更複雜且可控制的色彩管理。 圖形和桌上型電腦發佈應用程式需要這些額外的功能，讓使用者在整個生產過程中都能以一致的色彩在多部裝置之間精確地運作。
