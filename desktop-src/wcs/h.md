# <a name="h"></a>H

## <a name="hard-proofing"></a>硬校樣

藉由列印影像的固定複本來預覽或校樣影像。

## <a name="hls"></a>HLS

演出者常用的色彩空間。 其元件為色調、亮度和飽和度 (色度) 。

## <a name="hsv"></a>HSV

演出者常用的色彩空間。 它的元件包括色調、飽和度 (色度) 和值 (亮度) 。

## <a name="hue"></a>色調

純粹的色彩。 色調是在電磁輻射可見部分的特定位置。
