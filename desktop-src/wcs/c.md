# <a name="c"></a>C

## <a name="camp"></a>營地

色彩外觀模型配置檔案格式是以 XML 為基礎的檔案格式，其中包含描述 DMP CIEXYZ 值與 CIEJab 值之間關聯性所需的查看條件。

## <a name="cmyk"></a>Cmyk

青色、洋紅、黃色和黑色色空間。 它通常會在印表機上執行。

## <a name="colorant"></a>著色 劑

某些東西（尤其是 pigment 或筆墨）會以色彩或修改其他事物的色調。

## <a name="color-channel"></a>色彩頻道

色彩空間的元件。 例如，RGB 色彩空間具有紅色、綠色和藍色色頻。

## <a name="color-conversion"></a>色彩轉換

將色彩從某個色彩空間轉換成另一個色彩的進程。

## <a name="color-management-module-cmm"></a>色彩管理模組 (CMM) 

使用裝置設定檔執行色彩轉換和色彩對應的程式碼模組。

## <a name="color-mapping"></a>色彩對應

請參閱色彩比對。

## <a name="color-matching"></a>配色

將已轉換的色彩符合目的色彩空間中，其視覺效果最接近的色彩。

## <a name="color-model"></a>色彩模型

請參閱色彩空間。

## <a name="color-space"></a>色彩空間

色彩元件在三個維度的幾何座標系統上的對應。
