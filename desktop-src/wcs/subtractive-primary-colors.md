# <a name="subtractive-primary-colors"></a>Subtractive 主要色彩

色彩混合的 subtractive 方法是根據以白色開頭的假設。 所有色彩都有相同的數量。 如果您從白色減去青色和黃色，則產生的影像會是洋紅。 在綠色的影像中，減去青色會導致影像變更為黃色。

Subtractive [主要色彩](p.md) 為青色、黃色和洋紅。 下圖示范 subtractive 主要色彩混合。
