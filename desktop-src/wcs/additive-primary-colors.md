# <a name="additive-primary-colors"></a>加法主要色彩

色彩混合的加法方法是根據您以黑色開頭的假設。 也就是說，如果沒有其他色彩 (影像為黑色) 而您新增紅色，則影像會顯示紅色。 如果您接著新增藍色，影像會顯示為洋紅。

三種加總的 [主要色彩](p.md) 為紅色、綠色和藍色。 下圖說明附加的主要色彩混合。
