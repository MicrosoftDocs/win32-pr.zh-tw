# <a name="using-color-management-on-the-internet"></a>使用網際網路上的色彩管理

色彩影像和圖形是與網際網路使用者的資訊通訊日益重要的方式。 色彩轉換資訊可以內嵌在某些檔案格式中。 目前使用的內嵌技術並不精簡。 網際網路頻寬的考慮通常會讓它們變得不切實際。 網際網路考慮包括下列主題：

-   [sRGB：標準色彩空間](srgb--a-standard-color-space.md)
-   [輸入色彩空間和輸出設定檔的 WCS 1.0 預設值](srgb--a-standard-color-space.md)
-   [sRGB 和 Embedded 設定檔](srgb--a-standard-color-space.md)
