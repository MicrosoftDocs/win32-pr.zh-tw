# <a name="hsv-color-spaces"></a>HSV 色彩空間

演出者通常會使用色調、飽和度和值 (HSV) [色彩空間](c.md) 。 「色調」是我們通常會將其視為色彩的結果。 它是一種色彩的屬性，我們為它命名，例如 "red" 或 "blue"。 「值」是「亮度」的另一個字，這種色彩的屬性會相當於黑色和白色之間的灰色陰影。 飽和度是一種量值，會顯示色彩如何從相同 [亮度](b.md)的灰色顯示。 零飽和度表示沒有任何色調，而只是灰色尺規。 HSV 色彩空間已正規化。

![hsv 色彩空間](images/hsvline.png)

上圖顯示以 hexcone 形式 HSV 空間的線條繪圖。 其每個交叉區段都是六邊形。 每個交叉區段的頂點都有紅色、黃色、綠色、青色、藍色和洋紅等色彩。 HSV 空間中的色彩是藉由指示色調角度、色度層級和亮度層級所指定。 色調為零的角度為紅色。 色調角度會以逆時針方向增加。 互補色彩是分開的180。

HSV 色彩空間可與裝置相依或與裝置無關。
