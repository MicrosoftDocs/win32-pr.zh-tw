# <a name="obsolete-wcs-functions"></a>過時的 WCS 函數

|                                            |                                              |
|--------------------------------------------|----------------------------------------------|
| [**UpdateICMRegKey**](/windows/desktop/api/Wingdi/nf-wingdi-updateicmregkeya) | 已 *淘汰*：管理色彩設定檔和 cmm。 |
