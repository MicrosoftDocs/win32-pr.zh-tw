# <a name="r"></a>R

## <a name="rendering-intent"></a>轉譯意圖

將色彩從某個色彩空間轉譯至另一個色彩的方法。 ICC 設定檔中使用的轉譯意圖是由 (ICC) 的國際色彩協會所定義。

## <a name="rgb"></a>RGB

紅色、綠色和藍色的色彩空間。

## <a name="rgb-triple"></a>RGB 三重

包含 RGB 色彩空間值的數量。
