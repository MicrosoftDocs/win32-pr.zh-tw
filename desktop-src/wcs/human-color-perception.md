# <a name="human-color-perception"></a>人類色彩認知

人們可以藉由將視力為 *棒敲* 和 *圓錐* 的視覺資料格存在，來取得人為願景。 研究顯示，棒敲似乎不牽涉到人類的色彩認知。 Retina 中有三種不同的圓錐類型。 每種類型的錐形都會偵測到紅色、綠色或藍色。 人們看到的所有其他色彩都是這三種色彩的混合。 比方說，當出現大約相等的紅色、綠色和藍色時，就會察覺到白色。 當眼睛未偵測到任何紅色、綠色或藍色 (很少或沒有任何) 光線時，就會出現黑色。 人們看到的色彩數量也取決於光源的強度、集中程度和位置。 光源可能對色彩的認知有深遠的影響。

在影像處理系統中，色彩可以用不同的方式混合使用，以產生所需的眼睛結果。 最常使用的混合方法是以加總 *主要色彩* 和 *subtractive 主要色彩* 為基礎。 所有色彩都可以使用任一方法來重現。
