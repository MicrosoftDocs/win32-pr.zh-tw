# <a name="d"></a>D

## <a name="dac"></a>DAC

數位到類比轉換器。

## <a name="device-link-profile"></a>裝置連結設定檔

包含常使用之色彩轉換串連的檔案。

## <a name="device-profile"></a>裝置設定檔

檔案，其中包含如何描繪裝置色彩特性的資訊。 ICC 裝置設定檔會將裝置範圍中的色彩轉換成電腦。 它也會保留將電腦的色彩轉換成裝置範圍所需的資訊。 WCS XML 裝置設定檔提供測量資料和演算法參考，以在原生裝置空間與 CIEXYZ 之間轉換。

## <a name="dmp"></a>DMP

色彩裝置模型配置檔案格式是以 XML 為基礎的檔案格式，其中包含描述原生裝置色彩空間值與 CIEXYZ 值之間關聯性所需的度量。
