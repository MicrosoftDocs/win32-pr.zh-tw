# <a name="color-in-imaging"></a>影像中的色彩

色彩管理不是新的主題。 幾乎只要有商業列印和發佈公司，幾乎就能使用它。 在過去幾年來，我們已在最近幾年套用完善開發的理論和經驗的主體，以電腦化映射和發佈系統。 這些理論和實務中所述的概念是以人類色彩的認知為基礎。

應用程式開發人員並不需要具備生理學眼睛和緊張系統的高層級專長。 此外，也不一定要成為光線物理的專家。 不過，瞭解視覺和影像處理的基本概念，可協助開發人員使用 Windows 色彩系統1.0。 這些基本概念包括：

-   [人類色彩認知](human-color-perception.md)
-   [加法主要色彩](additive-primary-colors.md)
-   [Subtractive 主要色彩](subtractive-primary-colors.md)
-   [描述色彩](describing-color.md)
