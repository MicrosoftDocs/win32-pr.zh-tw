# <a name="rgb-color-spaces"></a>RGB 色彩空間

RGB [色彩空間](c.md) 的建立方式是將紅色、綠色和藍色色彩對應到立體笛卡兒座標系統。 此結果是立體 cube，如下圖所示。 此圖顯示兩個不同角度的相同 RGB cube。 請注意，座標系統的原點是黑色。 這是紅色、綠色和藍色 (RGB) 色彩元件全部0.0 的位置。 Cube 對角的邊角是白色，其中 RGB 色彩元件的最大值為。

![rgb 色彩空間 cube （最大值）](images/rgbclrs1.png)

![rgb 色彩空間 cube （最小值）](images/rgbclrs2.png)

如同大部分的 [色彩](c.md)空間，RGB 色彩空間已正規化。 也就是說，所有色彩值都受限於零到一個（含）的範圍。 黑色是 (0.0、0.0、0.0) ，而白色 (1.0、1.0、1.0) 。

在 RGB 色彩空間中， [主要色彩](p.md) 為紅色、綠色和藍色。 次要色彩為青色、黃色和洋紅。

RGB 色彩空間可與裝置相依或與裝置無關。
