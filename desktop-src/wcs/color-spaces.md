# <a name="color-spaces"></a>色彩空間

人類眼通常可以偵測到比數位裝置可重現更多的色彩。 比方說，如果您看一頁空白的白色頁面，您的眼睛可能偵測到至少100個不同的白色陰影。 白色牆很容易有1500的白色陰影。

高品質的數位相機、掃描器和其他影像取得裝置也可以偵測數十萬或甚至數百萬個色彩。 由於有這麼多可偵測的色彩，因此影像專業人員已創造出模型來指定色彩。 這些模型稱為 *色彩空間*。

這些模型稱為色彩空間的原因是它們大部分都可以對應到2D、立體或四維座標系統，類似于笛卡兒座標系統。 因此，您可以將色彩視為二維、立體或立體空間中的座標。 色彩空間中的色彩元件也稱為 *色彩通道*。

某些色彩空間的用途，是要與用來產生色彩影像的任何裝置無關。 有些是裝置相依的。 下列各節將討論裝置相關和與裝置無關的色彩空間：

-   [RGB 色彩空間](rgb-color-spaces.md)
-   [HSV 色彩空間](hsv-color-spaces.md)
-   [HLS 色彩空間](hls-color-spaces.md)
-   [CMY 和 CMYK 色彩空間](cmy-and-cmyk-color-spaces.md)
-   [裝置相關的色彩空間](device-dependent-color-spaces.md)
-   [與裝置無關的色彩空間](device-independent-color-spaces.md)
