# <a name="hls-color-spaces"></a>HLS 色彩空間

HLS [色彩空間](c.md) 也會由演出者廣泛使用。 其色彩元件為色調、亮度和飽和度 (色度) 。

色調與 HSV 模型具有相同的意義，但在此模型中，色調角度0會對應到藍色。 洋紅色在60，紅色是120。 就像 HSV 模型一樣，互補色彩也是180。

亮度是色彩中的黑色或白色的量。 提高亮度可將白色增加到色調。 減少亮度會將黑色增加到色調。

HLS 模型中的[飽和度](s.md)是色調的「純度」量值。 當飽和度減少時，色調會變得更灰色。 飽和度值為零會產生灰色的值。

下圖是 HLS 空間的線條繪圖，也就是雙重 hexcone。 HLS 色彩空間的任何水準交叉區段都是六邊形。 HLS 是正規化的色彩空間。 也就是說，亮度和飽和度值的範圍是從0.0 到1.0 （含）。 色調會因0和360（含）而異。

HLS 色彩空間可與裝置相依或與裝置無關。
