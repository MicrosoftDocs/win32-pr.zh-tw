# <a name="describing-color"></a>描述色彩

色彩可以在元件方面描述。 影像專業人員最常使用的元件為 *色調*、 *色度* 和 *亮度*。 色調通常會視為色彩。 色彩的色調會修正其在可見光線範圍中的位置。 色彩的色度是色彩的「純粹」或「強式」。 中性灰色表示有零飽和度。 色彩的亮度是指影像所反映或傳輸的光線濃度。

色彩影像處理文獻中也可以使用詞彙的 *色調*、 *語氣* 和 *陰影* 。 色彩的色調是藉由將色調與白色混合來取得。 色彩的色調是藉由將色調與灰色混合來建立。 色彩的陰影是藉由在色調中加上黑色來進行。
