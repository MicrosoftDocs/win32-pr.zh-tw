# <a name="g"></a>G

## <a name="gamma-correction"></a>Gamma 修正

裝置的原生實體語氣特性通常會因不同類型的裝置而有所不同。 Gamma 更正因數會補償這些差異。

## <a name="gamma-curve"></a>gamma 曲線

針對整個色彩範圍的 gamma 校正集合。 繪製在圖形上時，會形成曲線。

## <a name="gamma-ramp"></a>gamma 斜坡

請參閱 gamma 曲線。

## <a name="gamut"></a>域

裝置可以產生的色彩空間中的一組色彩。

## <a name="gmmp"></a>GMMP

色階地圖模型配置檔案格式是以 XML 為基礎的檔案格式，可控制來源與目的地裝置 gamuts 之間的色彩對應。
