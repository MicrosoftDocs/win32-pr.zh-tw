# <a name="wcs-10-availability"></a>WCS 1.0 可用性

ICM1 適用于 Microsoft Windows 95。 但不能用於 MS DOS、Windows 3。*x* 或 Windows NT 4.0 或更早版本。

ICM2 適用于 Microsoft Windows 98、Windows Millennium Edition (Me) 、Windows 2000 及 Windows XP。 但不能用於 MS DOS、Windows 3。*x*、Windows 95 或 Windows NT 4.0 或更早版本。

WCS 1.0 適用于 Microsoft Windows Vista。 Windows XP 或更早版本無法使用。
