# <a name="_alldiv-routine"></a>\_alldiv 常式

將兩個 **LONGLONG** 整數相除。 例如，若要將兩個 int64 值相除，編譯器可能會產生 \_ Alldiv 常式的呼叫。

## <a name="remarks"></a>備註

\_alldiv 常式是 C 編譯器的 helper 常式。 編譯器是否呼叫 \_ Alldiv 常式完全取決於優化集。
