# <a name="about-resource-enumeration"></a>關於資源列舉

資源列舉可提供應用程式執行時所發生之檢測活動的關閉和精確查看。 您可以在偵錯工具中，將檢測資訊抽象化並分類為一組處理常式特定的專案。

工具（例如偵錯工具）通常需要存取檢測資訊，但可能會發現它無法使用。 為了解決這個問題， [**VerifierEnumerateResource**](/windows/desktop/api/Avrfsdk/nf-avrfsdk-verifierenumerateresource) 函式會從應用程式中解壓縮這類資訊，為所要進行的問題提供更佳的內容資訊。
