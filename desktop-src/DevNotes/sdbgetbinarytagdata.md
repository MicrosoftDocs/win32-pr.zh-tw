# <a name="sdbgetbinarytagdata-function"></a>SdbGetBinaryTagData 函式

抓取指定之 **TAGID** 的二進位資料。

## <a name="syntax"></a>語法


```C++
PVOID WINAPI SdbGetBinaryTagData(
  _In_ PDB   pdb,
  _In_ TAGID tiWhich
);
```



## <a name="parameters"></a>參數

<dl> <dt>

*pdb* \[在\]
</dt> <dd>

填充碼資料庫的控制碼。

</dd> <dt>

*tiWhich* \[在\]
</dt> <dd>

對應至要抓取之資料的 **TAGID** 。

</dd> </dl>

## <a name="return-value"></a>傳回值

函數會傳回二進位資料的指標，如果 **TAGID** 無效，則 **為 Null** 。
