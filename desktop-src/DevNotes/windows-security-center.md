# <a name="windows-security-center"></a>Windows 資訊安全中心

Windows 安全性 Center (WSC) 是全方位的報告工具，可協助使用者建立和維護其電腦系統的保護安全層。 一旦建立安全層之後，Windows 安全性中心會 inconspicuous 監視電腦的健全狀況狀態。 但是，如果有弱點存在，WSC 會提供警示和規範性指導方針，以協助使用者完成透過「動作中心」向終端使用者呈現的安全狀態。

為了讓協力廠商安全性解決方案 (防毒軟體、反惡意程式碼或反間諜功能) 符合 Windows 的規範，並且成功地將狀態回報給「動作中心」，他們必須向「安全性中心」註冊自己，並使用私人 Api 來報告任何後續狀態變更，以與 WSC 通訊。 接著，WSC 會將這些更新傳達給「動作中心」，最後向使用者顯示這些更新。
