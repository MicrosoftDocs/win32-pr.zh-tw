# <a name="resource-enumeration"></a>資源列舉

開發人員可以使用資源列舉作為檢測的一部分，以提供有關其應用程式的特定類型資訊。

Microsoft 會在 Windows 作業系統中建立檢測，但這些功能緊密地系結到作業系統，因此應用程式開發人員可能會限制使用這些功能。 Microsoft 正在建立資源列舉應用程式開發介面 (API) 可搭配下列檢測資源使用：堆積配置和處理追蹤資訊。

下列主題描述如何在您的應用程式中使用此 API：

-   [關於資源列舉](about-resource-enumeration.md)
-   [資源列舉參考](resource-enumeration-reference.md)
