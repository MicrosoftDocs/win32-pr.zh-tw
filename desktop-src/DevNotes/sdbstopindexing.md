# <a name="sdbstopindexing-function"></a>SdbStopIndexing 函式

停用指定資料庫的索引建立和修改。

## <a name="syntax"></a>語法


```C++
BOOL WINAPI SdbStopIndexing(
  _In_ PDB     pdb,
  _In_ INDEXID iiWhich
);
```



## <a name="parameters"></a>參數

<dl> <dt>

*pdb* \[在\]
</dt> <dd>

填充碼資料庫的控制碼。

</dd> <dt>

*iiWhich* \[在\]
</dt> <dd>

索引識別碼。

</dd> </dl>

## <a name="return-value"></a>傳回值

當失敗時，函數會傳回 **TRUE** 或 **FALSE** 。
