# <a name="sndvol32exe"></a>SndVol32.exe

SndVol32.exe 程式可控制您系統的磁片區設定。 若要顯示此音量控制項，請移至主控台並啟動聲音和音訊裝置。 移至 [ **裝置磁片** 區]，然後按一下 [ **Advanced**]。

若要使用命令列引數來執行此程式，請移至 [ **開始** ] 功能表，按一下 [ **執行**]，然後輸入下列命令。

**SndVol32.exe \[ -R \| -D** *n * * * \]**

<dl> <dt>

<span id="-R"></span><span id="-r"></span>-R
</dt> <dd>

在記錄模式中啟動應用程式。

</dd> <dt>

<span id="-D_n"></span><span id="-d_n"></span><span id="-D_N"></span>-D *n*
</dt> <dd>

指定裝置識別碼。

</dd> </dl>

## <a name="remarks"></a>備註

此程式是 Windows 中包含的原生音訊音量和混音器控制項;無法從 Microsoft 下載。 程式通常會安裝在下列目錄中： C： \\ Windows \\ System32。

若要在您的系統上找出此檔案，請開啟 Windows 檔案總管、按一下 [ **搜尋** ] 按鈕，然後選取 [ **所有檔案和資料夾**]。 輸入檔案名 (SndVol32.exe) 至第一個可用的行，然後按一下 [ **搜尋** ] 按鈕。

如需使用此控制項的詳細資訊 **，請啟動程式並選取 [** 說明]。 請注意，無法以程式設計方式存取此程式的功能。
