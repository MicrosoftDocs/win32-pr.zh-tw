# <a name="sapcreateinstance-method"></a>SapCreateInstance 方法

使用安全音訊路徑驅動程式建立通道。

## <a name="syntax"></a>語法


```C++
MSSAPDLLObject* SapCreateInstance();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

**MSSAPDLLObject** 的指標。
