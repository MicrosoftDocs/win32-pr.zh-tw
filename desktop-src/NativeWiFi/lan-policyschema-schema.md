# <a name="lan_policy-schema"></a>區域網路 \_ 原則架構

區域網路 \_ 原則架構定義有線 (LAN) 原則設定檔。

區域網路原則設定檔的根項目是 [**LANPolicy**](lan-policyschema-lanpolicy-element.md) 元素。 每個設定檔都只會有一個根項目。 所有 LAN \_ 原則元素都在命名空間中 `https://www.microsoft.com/networking/LAN/policy/v1` 。

-   [LAN \_ PolicySchema 元素](lan-policyschema-elements.md)
-   [LAN \_ PolicySchema 簡單類型](lan-policyschema-simple-types.md)
