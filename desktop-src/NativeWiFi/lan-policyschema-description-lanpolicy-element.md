# <a name="description-lanpolicy-element"></a>描述 (LANPolicy) 元素

描述 (LANPolicy) 元素包含有線區域網路原則的描述。

``` syntax
<xs:element name="description"
    type="nameType"
 />
```

**Description** 元素是由 [**LANPolicy**](lan-policyschema-lanpolicy-element.md)元素定義。
