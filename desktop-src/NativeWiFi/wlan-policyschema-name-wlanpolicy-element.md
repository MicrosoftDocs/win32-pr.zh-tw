# <a name="name-wlanpolicy-element"></a>WLANPolicy) 專案的名稱 (

名稱 (WLANPolicy) 元素包含無線區域網路原則的名稱。

``` syntax
<xs:element name="name"
    type="nameType"
 />
```

**Name** 元素是由 [**WLANPolicy**](wlan-policyschema-wlanpolicy-element.md)元素所定義。

## <a name="remarks"></a>備註

名稱不區分大小寫。
