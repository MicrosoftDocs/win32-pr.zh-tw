# <a name="native-wifi"></a>原生 Wifi

## <a name="purpose"></a>目的

原生 Wifi 自動設定元件會設定、連線和中斷連線無線網路。 原生 Wifi 可將設定檔儲存在與其互動的網路上，並以 XML 檔的形式儲存。

## <a name="developer-audience"></a>開發人員對象

原生 Wifi API 是針對 C/c + + 開發人員所設計。 程式設計人員應該熟悉無線網路概念和術語。

## <a name="run-time-requirements"></a>執行階段需求求

原生 Wifi 元件需要執行 Windows Vista、Windows XP Service Pack 3 (SP3) 或適用于 Windows XP Service Pack 2 (SP2) 的無線區域網路 API 的用戶端。
