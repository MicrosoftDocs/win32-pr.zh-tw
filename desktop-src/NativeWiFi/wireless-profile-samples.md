# <a name="wireless-profile-samples"></a>無線設定檔範例

下列主題顯示各種無線設定的範例 XML 設定檔。 這些 XML 設定檔會使用 [WLAN \_ 設定檔架構](wlan-profileschema-elements.md) 專案和 [OneX 架構元素](onexschema-elements.md)。

您可以藉由呼叫 [**WlanSetProfile**](/windows/desktop/api/wlanapi/nf-wlanapi-wlansetprofile)，以程式設計方式將無線設定檔新增至設定檔存放區。 您也可以使用 **netsh** 命令列公用程式，將設定檔新增至設定檔存放區。 如需詳細資訊，請參閱 [無線區域網路 (wlan) 的 Netsh 命令 ](/previous-versions/windows/it-pro/windows-server-2008-R2-and-2008/cc755301(v=ws.10))。
