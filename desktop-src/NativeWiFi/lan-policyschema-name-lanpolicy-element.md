# <a name="name-lanpolicy-element"></a>LANPolicy) 專案的名稱 (

名稱 (LANPolicy) 元素包含有線區域網路原則的名稱。

``` syntax
<xs:element name="name"
    type="nameType"
 />
```

**Name** 元素是由 [**LANPolicy**](lan-policyschema-lanpolicy-element.md)元素所定義。

## <a name="remarks"></a>備註

名稱不區分大小寫。
