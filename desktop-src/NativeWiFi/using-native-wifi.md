# <a name="using-native-wifi"></a>使用原生 Wifi

本節說明原生 Wifi 採用的程式和程式設計技術。 它包含基本的原生 Wifi 程式設計技術，例如 [原生 WIFI API 範例](native-wifi-api-sample.md)，以及適用于有經驗的 NativeWifi 開發人員的先進技術。

本節將說明下列主題：

-   [無線消費者介面 Api](wireless-user-interface-apis.md)
-   [使用 Wi-Fi Direct 函數](using-the-wi-fi-direct-api.md)
-   [使用託管網路和網際網路連線共用](using-hosted-network-and-internet-connection-sharing.md)
-   [原生 Wifi API 範例](native-wifi-api-sample.md)
-   [無線託管網路範例](wireless-hosted-network-sample.md)
-   [有線設定檔範例](wired-profile-samples.md)
-   [無線設定檔範例](wireless-profile-samples.md)

[無線消費者介面 api](wireless-user-interface-apis.md)主題顯示一些用於啟動無線網路管理使用者介面的程式碼。

[使用 Wi-Fi direct](using-the-wi-fi-direct-api.md)函式主題列出 Wi-Fi 直接函式的清單，並討論如何使用這些函數。

[使用託管網路和網際網路連線共用](using-hosted-network-and-internet-connection-sharing.md)主題討論如何使用 (ICS) 的無線裝載網路和網際網路連線共用。

Microsoft Windows 軟體開發套件 (SDK) 包含兩個示範原生 Wifi API 使用的程式碼範例。 此外，也提供範例設定檔，以示範各種原生 Wifi 設定檔架構的使用。

下列主題包含有關範例的詳細資訊：

-   [原生 Wifi API 範例](native-wifi-api-sample.md)
-   [無線託管網路範例](wireless-hosted-network-sample.md)
-   [有線設定檔範例](wired-profile-samples.md)
-   [無線設定檔範例](wireless-profile-samples.md)
