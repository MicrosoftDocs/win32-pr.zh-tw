# <a name="lan_profile-schema"></a>LAN \_ 設定檔架構

LAN \_ 設定檔架構會定義有線 (LAN) 網路設定檔。 LAN 設定檔包含乙太網路介面的網路設定。

LAN 設定檔的根項目是 [**LANProfile**](lan-profileschema-lanprofile-element.md) 元素。 每個設定檔都只會有一個根項目。 所有 LAN \_ 設定檔元素都在命名空間中 `https://www.microsoft.com/networking/LAN/profile/v1` 。

若要查看範例 LAN 設定檔，請參閱 [有線設定檔範例](wired-profile-samples.md)。

-   [LAN \_ 設定檔架構元素](lan-profileschema-elements.md)
