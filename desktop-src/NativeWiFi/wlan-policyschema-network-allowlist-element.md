# <a name="network-allowlist-element"></a>network (允許清單) 元素

Network (允許清單) 元素會定義允許的網路。 必須允許電腦連線到允許的網路。

``` syntax
<xs:element name="network"
    type="networkItemType"
 />
```

**Network** 元素是由 [**允許清單**](wlan-policyschema-allowlist-networkfilter-element.md)元素定義。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|-------------------------------------|------------------------------------------------------|
| 最低支援的用戶端<br/> | \[僅限 Windows Vista 桌面應用程式\]<br/>       |
| 最低支援的伺服器<br/> | 僅限 Windows Server 2008 \[ desktop 應用程式\]<br/> |
