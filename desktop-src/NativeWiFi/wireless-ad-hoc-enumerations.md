# <a name="wireless-ad-hoc-enumerations"></a>無線特定列舉

無線臨機操作介面會定義並使用下列列舉。

-   [**DOT11 臨機操作 \_ \_ 驗證 \_ 演算法**](/windows/win32/api/adhoc/ne-adhoc-dot11_adhoc_auth_algorithm)
-   [**DOT11 臨機操作 \_ \_ 密碼 \_ 演算法**](/windows/win32/api/adhoc/ne-adhoc-dot11_adhoc_cipher_algorithm)
-   [**DOT11 臨機操作 \_ \_ 連接 \_ 失敗 \_ 原因**](/windows/win32/api/adhoc/ne-adhoc-dot11_adhoc_connect_fail_reason)
-   [**DOT11 臨機操作 \_ \_ 網路 \_ \_ 線上狀態**](/windows/win32/api/adhoc/ne-adhoc-dot11_adhoc_network_connection_status)

> [!Note]  
> 未來的 Windows 版本可能無法使用臨機操作模式。 從 Windows 8.1 和 Windows Server 2012 R2 開始，請改用 [Wi-Fi Direct](about-the-wi-fi-direct-api.md) 。
