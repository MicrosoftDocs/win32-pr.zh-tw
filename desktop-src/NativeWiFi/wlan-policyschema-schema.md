# <a name="wlan_policy-schema"></a>WLAN \_ 原則架構

WLAN \_ 原則架構會定義原生 Wifi 自動設定服務所使用的 wlan 原則設定檔。

WLAN 原則設定檔的根項目是 [**WLANPolicy**](wlan-policyschema-wlanpolicy-element.md) 元素。 每個設定檔都只會有一個根項目。 所有 \_ 的 WLAN 原則元素都在命名空間中 `https://www.microsoft.com/networking/WLAN/policy/v1` 。

-   [WLAN \_ PolicySchema 元素](wlan-policyschema-elements.md)
-   [WLAN \_ PolicySchema 簡單類型](wlan-policyschema-simple-types.md)
-   [WLAN \_ PolicySchema 複雜類型](wlan-policyschema-complex-types.md)
