# <a name="firewall-dynamic-keywords-structures"></a>防火牆動態關鍵字結構

防火牆動態關鍵字包含下列結構。

## <a name="in-this-section"></a>本節內容

| 主題 | 描述 |
|-|-|
| [**FW_DYNAMIC_KEYWORD_ADDRESS_DATA0**](/windows/win32/api/netfw/ns-netfw-fw_dynamic_keyword_address_data0) | 當呼叫 **列舉** api 時，保存傳回給用戶端的資料。 |
| [**FW_DYNAMIC_KEYWORD_ADDRESS0**](/windows/win32/api/netfw/ns-netfw-fw_dynamic_keyword_address0) | 允許用戶端建立動態關鍵字位址，其中包含 IP 位址的清單。 |
