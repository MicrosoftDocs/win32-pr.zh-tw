# <a name="primary-group-of-a-new-object"></a>新物件的主要群組

新物件的主要群組是來自物件建立者所指定之 [*安全描述項*](/windows/desktop/SecGloss/s-gly) 的主要群組。 如果物件的建立者未指定主要群組，則物件的主要群組是來自建立者主要或模擬 [*權杖*](/windows/desktop/SecGloss/i-gly)的預設主要群組。
