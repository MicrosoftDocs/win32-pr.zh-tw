# <a name="permissions-property-page"></a>許可權屬性頁面

[**許可權**] 屬性頁包含可在物件的 [*DACL*](/windows/desktop/SecGloss/d-gly)中預先編輯 [*ace*](/windows/desktop/SecGloss/a-gly)的控制項。 此頁面可讓使用者查看和編輯 DACL 中的所有資訊，包括 [ [基本安全性] 屬性頁](basic-security-property-page.md)上未提供的資訊。 Advanced 資訊包含 [物件特定的 ace](object-specific-aces.md) 和 [ACE 繼承](ace-inheritance.md)的相關資訊。

當使用者按一下 [基本安全性] 屬性頁上的 [ **advanced** ] 按鈕時，[**許可權**] 屬性頁一律會包含在 [ [advanced security] 屬性工作表](advanced-security-property-sheet.md)中。
