# <a name="using-authorization-in-script"></a>在腳本中使用授權

您可以使用授權管理員 API 來控制對應用程式資源的存取。

本節中的範例是使用 Visual Basic Scripting Edition (VBScript) 和 Active Server Pages (ASP) 來撰寫。

如需其他授權工作的詳細資訊，請參閱 [腳本中的支援](supporting-tasks-for-authorization-in-script.md)工作。



| 主題                                                                                                                      | 描述                                                                                              |
|----------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| [在腳本中定義許可權](defining-permissions-in-script.md)                                                       | 藉由建立授權原則存放區來定義哪些使用者可以存取哪些應用程式資源。 |
| [在腳本中確認用戶端對要求的資源的存取](verifying-client-access-to-a-requested-resource-in-script.md) | 檢查用戶端是否可以存取一或多個作業。                                           |
| [委派腳本中的許可權定義](delegating-the-defining-of-permissions-in-script.md)                   | 將儲存在 Active Directory 中的授權原則存放區委派給管理。          |
