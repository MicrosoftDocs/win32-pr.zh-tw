# <a name="advantages-of-role-based-authorization"></a>以角色為基礎的授權優點

授權管理員 API 提供的授權模型有幾個優點，優於低層級的 (ACL 型) 授權：

-   簡化存取控制管理。
-   使授權可供使用 Microsoft Visual Basic 開發系統開發的腳本和應用程式使用。
-   提供在檢查存取時套用執行時間商務邏輯的機制。
