# <a name="role-based-access-control"></a>角色型存取控制

授權管理員 API 和 MMC 嵌入式管理單元可為應用程式提供以角色為基礎的存取控制架構。 授權管理員 API （也稱為 AzMan）提供簡化的開發模型，可讓您管理彈性群組和商務規則，以及儲存授權原則。 如需詳細資訊，請參閱下列主題：

-   [以角色為基礎的授權優點](advantages-of-role-based-authorization.md)
-   [授權管理員模型](authorization-manager-model.md)

**WINDOWS XP：** 授權管理員 API 和 MMC 嵌入式管理單元可從下載 (Windows Server 2003 Administration Tools Pack) [https://www.microsoft.com/downloads](https://www.microsoft.com/Downloads/) 。 此下載支援授權管理員原則的系統管理與開發;它不支援部署使用授權管理員來控制存取權的應用程式。
