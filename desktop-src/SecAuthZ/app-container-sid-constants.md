# <a name="app-container-sid-constants"></a>應用程式容器 SID 常數

應用程式容器的特定 SID 常數會指示應用程式封裝授權單位。 雖然開發人員可以直接使用這些常數，但大部分的開發人員不需要定義這些應用程式容器 Sid。
