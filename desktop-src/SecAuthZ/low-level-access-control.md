# <a name="low-level-access-control"></a>低層級存取控制

低層級的安全性功能可協助您使用 [*安全描述項*](/windows/desktop/SecGloss/s-gly)、 [*存取控制清單*](/windows/desktop/SecGloss/a-gly) (acl) ，以及 (ace) 的 [*存取控制專案*](/windows/desktop/SecGloss/a-gly) 。

如需模型的描述，請參閱 [存取控制模型](access-control-model.md)。



| 主題                                                                                                  | 描述                                                                                                                                                                                                                                                                                                  |
|--------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [低層級安全描述項函式](low-level-security-descriptor-functions.md)                 | 用來設定和取得物件 [*安全描述項*](/windows/desktop/SecGloss/s-gly)的函式。                                                                                                                                                |
| [建立低層級安全描述項](low-level-security-descriptor-creation.md)                   | 用來建立 [*安全描述項*](/windows/desktop/SecGloss/s-gly) ，以及取得和設定安全描述項元件的函數。                                                                                                        |
| [絕對和 Self-Relative 的安全描述項](absolute-and-self-relative-security-descriptors.md) | 用來檢查或轉換 [*絕對*](/windows/desktop/SecGloss/a-gly) 或 [*自我相關*](/windows/desktop/SecGloss/s-gly) 格式的函式。 |
| [低層級 ACL 和 ACE 函式](low-level-acl-and-ace-functions.md)                                 | 用於管理 Acl 和 Ace 的函式。                                                                                                                                                                                                                                                                        |
