# <a name="access-control-for-application-resources"></a>應用程式資源的存取控制

Windows Server 2003 中提供的授權管理員 API 和 MMC 嵌入式管理單元，可為應用程式提供彈性的角色型存取控制架構。 此架構包含一組適用于 web 和企業營運應用程式的 strippable COM 物件。

下列各節提供控制應用程式資源存取權的相關資訊：

-   [以角色為基礎的存取控制](role-based-access-control.md)
-   [以 ACL 為基礎的存取控制](acl-based-access-control.md)
