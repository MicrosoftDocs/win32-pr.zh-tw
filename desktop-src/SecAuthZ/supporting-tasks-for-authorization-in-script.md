下列工作支援在 [腳本中使用 [授權](using-authorization-in-script.md)] 中所列的主要工作。



| 主題                                                                                                    | 描述                                                                                                                                                                                                                                  |
|----------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [在腳本中建立授權原則存放區](creating-an-authorization-policy-store-in-script.md) | 在安裝應用程式之前或期間建立授權原則。                                                                                                                                                          |
| [在腳本中建立用戶端內容](establishing-a-client-context-in-script.md)                   | 使用權杖的控制碼、網域和使用者名稱，或用戶端的 [*安全識別碼*](/windows/desktop/SecGloss/s-gly) (SID) 的字串表示，建立用戶端內容。 |
| [使用腳本中的商務邏輯來限定存取權](qualifying-access-with-business-logic-in-script.md)   | 提供用來檢查存取的執行時間邏輯。                                                                                                                                                                                                  |
