# <a name="about-authorization"></a>關於授權

下列各節提供授權的相關資訊。



| 區段                                                                                             | 描述                                                                                                                                                                                                                                     |
|-----------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [存取控制](access-control.md)<br/>                                                     | 控制哪些人可以存取作業系統中資源的安全性功能。 應用程式會呼叫存取控制功能，以設定可以存取特定資源的人員，或控制應用程式所提供資源的存取權。<br/>     |
| [存取控制編輯器](access-control-editor.md)<br/>                                       | 建立和使用屬性工作表和屬性頁，讓使用者能夠查看和修改物件 [*安全描述項*](/windows/desktop/SecGloss/s-gly)的元件。<br/> |
| [用戶端/伺服器存取控制](client-server-access-control.md)<br/>                         | 提供服務給用戶端的伺服器應用程式。<br/>                                                                                                                                                                                |
| [應用程式資源的存取控制](access-control-for-application-resources.md)<br/> | 應用程式資源的角色型和 ACL 型存取控制。<br/>                                                                                                                                                                   |
| [認證相符控制](mandatory-integrity-control.md)<br/>                           | 安全物件的系統層級存取控制。<br/>                                                                                                                                                                                   |
| [使用者帳戶控制](user-account-control.md)<br/>                                         | 這項安全性功能可讓使用者以非系統管理員的身分（稱為標準使用者）或系統管理員身分執行一般工作，而不需要切換使用者、登出或使用 [ **執行** 身分]。<br/>                                           |
