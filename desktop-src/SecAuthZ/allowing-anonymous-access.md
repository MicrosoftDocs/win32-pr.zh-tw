# <a name="allowing-anonymous-access"></a>允許匿名存取

預設安全性原則會將匿名本機存取限制為沒有許可權。 系統管理員可以在適當的情況下新增或減少許可權。

具有與所有人相同存取權限之應用程式的本機存取群組存在。 系統管理員可以接著適當地增加或減少該群組中的使用者數目，名為 **Windows 2000 相容的存取群組**。

如需詳細資訊，請參閱在 [本機群組函數](/windows/desktop/NetMgmt/local-group-functions) 主題的表格中所列函式的參考頁面。
