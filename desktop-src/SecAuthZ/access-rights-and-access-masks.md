<span data-ttu-id="8d132-104">*訪問* 許可權是位旗標，對應至執行緒可以在安全物件上執行的一組特定作業。</span><span class="sxs-lookup"><span data-stu-id="8d132-104">An *access right* is a bit flag that corresponds to a particular set of operations that a thread can perform on a securable object.</span></span> <span data-ttu-id="8d132-105">例如，登錄機碼具有金鑰 \_ 集 \_ 值存取權限，這會對應到執行緒在機碼下設定值的能力。</span><span class="sxs-lookup"><span data-stu-id="8d132-105">For example, a registry key has the KEY\_SET\_VALUE access right, which corresponds to the ability of a thread to set a value under the key.</span></span> <span data-ttu-id="8d132-106">如果執行緒嘗試對物件執行作業，但沒有物件的必要存取權限，系統就不會執行此作業。</span><span class="sxs-lookup"><span data-stu-id="8d132-106">If a thread tries to perform an operation on an object, but does not have the necessary access right to the object, the system does not carry out the operation.</span></span>

<span data-ttu-id="8d132-107">[*存取遮罩*](/windows/desktop/SecGloss/a-gly)是一種32位的值，其位會對應至物件所支援的存取權。</span><span class="sxs-lookup"><span data-stu-id="8d132-107">An [*access mask*](/windows/desktop/SecGloss/a-gly) is a 32-bit value whose bits correspond to the access rights supported by an object.</span></span> <span data-ttu-id="8d132-108">所有 Windows 安全物件都使用 [存取遮罩格式](access-mask-format.md) ，其中包含下列存取權類型的位：</span><span class="sxs-lookup"><span data-stu-id="8d132-108">All Windows securable objects use an [access mask format](access-mask-format.md) that includes bits for the following types of access rights:</span></span>

-   [<span data-ttu-id="8d132-110">標準存取權限</span><span class="sxs-lookup"><span data-stu-id="8d132-110">Standard access rights</span></span>](standard-access-rights.md)
-   [<span data-ttu-id="8d132-111">SACL 存取權限</span><span class="sxs-lookup"><span data-stu-id="8d132-111">SACL access right</span></span>](sacl-access-right.md)
-   [<span data-ttu-id="8d132-112">目錄服務存取權限</span><span class="sxs-lookup"><span data-stu-id="8d132-112">Directory services access rights</span></span>](directory-services-access-rights.md)

<span data-ttu-id="8d132-113">當執行緒嘗試開啟物件的控制碼時，執行緒通常會指定存取遮罩來 [要求一組存取權限](requesting-access-rights-to-an-object.md)。</span><span class="sxs-lookup"><span data-stu-id="8d132-113">When a thread tries to open a handle to an object, the thread typically specifies an access mask to [request a set of access rights](requesting-access-rights-to-an-object.md).</span></span> <span data-ttu-id="8d132-114">例如，需要設定和查詢登錄機碼值的應用程式，可以使用存取遮罩來要求金鑰 \_ 集 \_ 值和金鑰 \_ 查詢 \_ 值存取權限，來開啟金鑰。</span><span class="sxs-lookup"><span data-stu-id="8d132-114">For example, an application that needs to set and query the values of a registry key can open the key by using an access mask to request the KEY\_SET\_VALUE and KEY\_QUERY\_VALUE access rights.</span></span>
