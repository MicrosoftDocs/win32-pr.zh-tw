# <a name="collections-authorization"></a> (授權) 的集合

授權管理員 API 提供表示其他物件集合的介面。 例如， [**IAzRoles**](/windows/desktop/api/Azroles/nn-azroles-iazroles) 介面代表 [**IAzRole**](/windows/desktop/api/Azroles/nn-azroles-iazrole) 物件的集合。

代表集合的其他介面為 [**IAzApplications**](/windows/desktop/api/Azroles/nn-azroles-iazapplications)、 [**IAzApplicationGroups**](/windows/desktop/api/Azroles/nn-azroles-iazapplicationgroups)、 [**IAzOperations**](/windows/desktop/api/Azroles/nn-azroles-iazoperations)、 [**IAzScopes**](/windows/desktop/api/Azroles/nn-azroles-iazscopes)和 [**IAzTasks**](/windows/desktop/api/Azroles/nn-azroles-iaztasks)。 這些介面每一個都提供可協助列舉集合中物件的屬性。
