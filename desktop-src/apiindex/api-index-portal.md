# <a name="api-index-for-desktop-windows-applications"></a>桌面 Windows 應用程式的 API 索引

本文提供可在 desktop Windows 應用程式中使用之 api 的參考檔連結。

## <a name="win32-windows-api"></a>Win32 (Windows API)

Win32 api (也稱為 Windows API) 是 Windows 應用程式的原生平臺。 此 API 最適合需要直接存取系統功能和硬體的桌面應用程式。 Windows API 可在所有桌面應用程式中使用，且32位和64位 Windows 上通常會支援相同的函式。

* [依功能的 WIN32 API 參考](windows-api-list.md)
* [依標頭的 WIN32 API 參考](/windows/win32/api/)
* [適用於 UWP 應用程式的 Win32 與 COM API](/uwp/win32-and-com/win32-and-com-for-uwp-apps)
* [Windows 傘程式庫](windows-umbrella-libraries.md)
* [Windows API 集合](windows-apisets.md)

## <a name="windows-runtime-winrt"></a>Windows執行時間 (WinRT) 

WinRT 是 Windows 10 應用程式和遊戲（包括桌面應用程式）的領先 edge 平臺。 WinRT API 適用于需要精密 UI、樣式自訂和圖形密集案例的原生 c + + 和 managed 桌面應用程式。

* [WinRT API 參考](/uwp/api/)
* [從桌面應用程式呼叫的 WinRT Api](uwp-apis-callable-from-a-classic-desktop-app.md)

.net 類別庫可讓您存取受管理的桌面應用程式 Windows 系統和 UI 功能，包括 WPF 和 Windows Forms 應用程式。
