# <a name="about-the-control-point-api"></a>關於控制點 API

UPnP 技術可提供智慧型設備、電腦和其他控制點的動態網路功能。 動態網路功能適用于各種實體媒體。 Microsoft 的控制點 API 採用 UPnP 技術，包含一組 COM 物件，可用來撰寫可尋找及控制以 UPnP 為基礎之裝置的應用程式與網頁。 例如，您可以從特別設計來控制 UPnP 的網頁來控制可支援 UPnP 的 VCR。 為了延伸這個概念，開發人員可以從中央、遠端或對等位置控制許多設備。

使用具有 UPnP 技術的控制點 API 所建立的應用程式，範圍從 HTML 網頁中內嵌的腳本到完備的 c + + 程式。 具有 UPnP 技術的控制點 API 支援 microsoft Visual Basic 開發系統、microsoft Visual Basic 腳本版本 (VBScript) 和 c + +。

下列各節將說明在一般情況下，控制點 API 與 UPnP 技術的運作方式：

-   [控制點基本概念](control-point-basics.md)
-   [尋找裝置](finding-devices.md)
-   [描述裝置](describing-devices.md)
-   [控制裝置](controlling-devices.md)

整個控制點 API 中的所有範例都有 UPnP 技術檔，稱為虛構裝置、裝置類型和字串。
