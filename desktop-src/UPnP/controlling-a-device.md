# <a name="controlling-a-device"></a>控制裝置

一旦探索到裝置之後，您就可以加以控制。

**若要查看裝置屬性**

1.  從 [ **找到的裝置** ] 清單中選取裝置。
2.  按一下 [ **裝置屬性**]。 [ **裝置屬性** ] 視窗隨即出現，並顯示所要求的資訊。

> [!Note]  
> 您無法在 c + + 範例程式碼中使用 **View Presentation** 功能。

 

**若要查看裝置的表示頁面**

1.  從 [ **找到的裝置** ] 清單中選取裝置。
2.  按一下 [ **View Presentation**]。 Internet Explorer 視窗隨即出現，並顯示要求的表示頁面。

> [!Note]  
> **View Service Desc。** c + + 範例程式碼無法使用功能。

 

**若要查看服務描述**

1.  您可以在服務 Desc 中輸入服務描述的 URL **。URL** 欄位。

    ![服務描述 url](images/ucp-url.png)

2.  按一下 [ **View Service Desc]。** [ **服務描述檢視器** ] 視窗隨即顯示。 然後，您可以使用樹狀檢視來流覽服務描述。 C + + 範例程式碼無法使用這項功能。

    ![服務描述檢視器視窗](images/ucp-serv.png)

    您可以在 [服務描述檢視器] 視窗中使用五個命令。



| Button                 | 執行的動作                                                                                                                                                                      |
|------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Go                     | 載入服務 Desc 中所顯示的檔案 **。URL** 欄位。                                                                                                                              |
| 設定動作             | 在服務描述樹狀目錄中選取動作名稱，然後按一下 [ **設定動作**]。 動作名稱會輸入到主要 **一般 UCP** 視窗的 [叫用 **動作**] 欄位中。       |
| Set Variable           | 在 [服務描述] 樹狀結構中選取變數名稱，然後按一下 [ **設定變數**]。 變數名稱會輸入到主要 **一般 UCP** 視窗的 **查詢變數** 欄位中。 |
| 填入變數清單 | 將所有服務的變數名稱載入至主要 **泛型 UCP** 視窗的 **查詢變數** 清單。                                                                           |
| 填入動作清單   | 將所有服務的動作名稱載入至主要 **一般 UCP** 視窗的叫用 **動作** 清單。                                                                              |



 

**控制裝置**

1.  從 [ **找到的裝置** ] 清單中，選取您要控制的裝置。
2.  從 [ **選擇服務** ] 清單中，選取您要叫用的服務或您想要查詢的狀態變數。

    ![控制裝置視窗](images/ucp-contr.png)

    > [!Note]  
    > **服務清單** 的內容是以 [**找到裝置**] 清單中選取的裝置為基礎。

     

3.  如果您想要找出所選服務的狀態變數值，請在 [服務] 的 [ **查詢變數** ] 欄位中輸入變數名稱。 按一下 **[查詢]**。 結果會顯示在 [ **查詢狀態值/動作輸出引數** ] 欄位中。
4.  如果您想要叫用服務的動作，請在 [叫用 **動作** ] 欄位中輸入動作名稱，然後在 [ **動作引數** ] 欄位中輸入任何引數。 按一下 **[** 叫用]。 結果會顯示在 [ **查詢狀態值/動作輸出引數** ] 欄位中（如果有的話）。

    傳回值（如果有的話）會包含在第一個 out 引數中。

    > [!Note]  
    > 如果某個動作有多個引數，請以空格分隔它們。

     

5.  在所選服務的 [事件] 欄位中，查看 **事件** 資訊。
