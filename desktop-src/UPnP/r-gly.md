# <a name="r-upnp-apis"></a>R (UPnP Api) 

包含以字母 R 開頭的 UPnP 相關詞彙。

[B](a-gly.md) [C](c-gly.md) [D](d-gly.md) [E](e-gly.md) F G [H](h-gly.md) I J K L M [N](n-gly.md) O [P](p-gly.md) Q R [S](s-gly.md) T W [](u-gly.md) X Y Z

<dl> <dt>

<span id="upnp.r_1_gly"></span><span id="UPNP.R_1_GLY"></span>**資原始目錄**
</dt> <dd>

資原始目錄是裝置的本機目錄，也就是圖示檔案等檔案的所在位置。

</dd> <dt>

<span id="upnp.r_2_gly"></span><span id="UPNP.R_2_GLY"></span>**根裝置**
</dt> <dd>

根裝置是裝置樹狀結構中最上層的裝置。 例如，如果您要存取稱為「觀賞站」的 VCR/電視組合裝置，則有三個裝置：觀賞站、VCR 和電視。 觀賞站被視為根裝置。

</dd> <dt>

<span id="upnp.r_3_gly"></span><span id="UPNP.R_3_GLY"></span>**正在執行裝置**
</dt> <dd>

執行中的裝置是在註冊之前已載入其程式碼的裝置。

這些是在註冊之前不知道服務描述的裝置。

這類裝置的範例是橋接器，例如 UPnP SCP 橋接器。 此橋接器會向裝置主機註冊執行中的裝置。

</dd> </dl>
