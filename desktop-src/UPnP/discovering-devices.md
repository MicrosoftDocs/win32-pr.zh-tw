# <a name="discovering-devices"></a>探索裝置

您可以透過下列三種方式來搜尋裝置：依類型、UDN 和非同步搜尋 (這是依裝置類型) 搜尋。

![探索裝置視窗](images/ucp-disc.png)

**探索裝置**

1.  選取您要使用的搜尋類型： **依類型尋找**、 **依 UDN 尋找** 或 **非同步尋找**。
2.  從清單中選取您想要從清單 (尋找的裝置類型或 UDN，然後依類型或 UDN) 來搜尋。 清單的內容會根據您先前編輯的 udn.txt 和 devtype.txt 檔案的內容而定。
3.  按一下 [開始探索]。 搜尋已啟動。 結果會顯示在 [ **找到裝置** ] 清單中。 如果找不到任何裝置，則會顯示一則訊息，指出這個狀況。 搜尋進度的狀態會顯示在 [ **狀態** ] 欄位中。

當您使用非同步搜尋時，找到的新裝置會顯示在 [ **找到的裝置** ] 清單和 **狀態列** 中。 非同步搜尋完成後， **狀態列** 會顯示一則訊息，指出這一點。 不過，因為非同步搜尋會在手動停止之前執行，所以在裝置出現在網路上時，新的裝置會出現在 [ **找到的裝置** ] 清單和 **狀態列** 中。

一旦探索到裝置之後，您就可以 [加以控制](controlling-a-device.md)。
