# <a name="querying-state-variables"></a>查詢狀態變數

[**IUPnPService**](/windows/desktop/api/Upnp/nn-upnp-iupnpservice)介面會提供 [**QueryStateVariable**](/windows/desktop/api/Upnp/nf-upnp-iupnpservice-querystatevariable)方法，以傳回指定狀態變數的值。

UPnP 論壇不鼓勵使用 [**QueryStateVariable**](/windows/desktop/api/Upnp/nf-upnp-iupnpservice-querystatevariable)。 建議裝置寫入器撰寫適當的動作以提供此資訊。 裝置寫入器不需要執行 **QueryStateVariable**。

請參閱叫用 [動作](invoking-actions.md)一節。
