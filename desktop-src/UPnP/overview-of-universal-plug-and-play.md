# <a name="overview-of-upnp-architecture"></a><span data-ttu-id="fb8ce-103">UPnP 架構總覽</span><span class="sxs-lookup"><span data-stu-id="fb8ce-103">Overview of UPnP Architecture</span></span>

<span data-ttu-id="fb8ce-104">UPnP 架構定義智慧型設備、裝置和 [*控制點*](c-gly.md)的對等網路連線能力。</span><span class="sxs-lookup"><span data-stu-id="fb8ce-104">The UPnP architecture defines peer-to-peer network connectivity of intelligent appliances, devices, and [*control points*](c-gly.md).</span></span> <span data-ttu-id="fb8ce-105">它的設計目的是要將容易使用、彈性、標準的連線帶到臨機操作、受控或非受控網路，不論這些網路是在家庭、小型企業或直接連接到網際網路。</span><span class="sxs-lookup"><span data-stu-id="fb8ce-105">It is designed to bring easy-to-use, flexible, standards-based connectivity to ad-hoc, managed, or unmanaged networks, whether these networks are in the home, small businesses, or attached directly to the Internet.</span></span> <span data-ttu-id="fb8ce-106">UPnP 架構是一種分散式的開放網路架構，它使用現有的 TCP/IP 和 Web 技術來啟用無縫近接網路，以及在網路裝置之間進行控制和資料傳輸。</span><span class="sxs-lookup"><span data-stu-id="fb8ce-106">The UPnP architecture is a distributed, open networking architecture that uses existing TCP/IP and Web technologies to enable seamless proximity networking, in addition to control and data transfer among networked devices.</span></span>

<span data-ttu-id="fb8ce-107">UPnP 是以 Web 服務通訊協定的初步版本（例如 XML 和簡易物件存取通訊協定 (SOAP) ）為基礎的通訊協定套件。</span><span class="sxs-lookup"><span data-stu-id="fb8ce-107">UPnP is an IP-based protocol suite based on preliminary versions of Web Services protocols such as XML and Simple Object Access Protocol (SOAP).</span></span> <span data-ttu-id="fb8ce-108">透過 UPnP，裝置可動態地加入網路、取得 IP 位址、傳達其功能，以及探索網路上其他裝置的存在和功能。</span><span class="sxs-lookup"><span data-stu-id="fb8ce-108">With UPnP, a device can dynamically join a network, obtain an IP address, convey its capability, and discover the presence and capabilities of other devices on the network.</span></span>

<span data-ttu-id="fb8ce-109">UPnP 裝置是服務和嵌套裝置的容器。</span><span class="sxs-lookup"><span data-stu-id="fb8ce-109">A UPnP device is a container of services and nested devices.</span></span> <span data-ttu-id="fb8ce-110">例如，VCR 可能包含磁帶傳輸服務、調諧器服務和時鐘服務。</span><span class="sxs-lookup"><span data-stu-id="fb8ce-110">For example, a VCR might consist of a tape transport service, a tuner service, and a clock service.</span></span> <span data-ttu-id="fb8ce-111">不同的 UPnP 裝置類別與不同的服務集和內嵌裝置相關聯。</span><span class="sxs-lookup"><span data-stu-id="fb8ce-111">Different categories of UPnP devices are associated with different sets of services and embedded devices.</span></span> <span data-ttu-id="fb8ce-112">例如，VCR 內的服務與印表機內的服務不同。</span><span class="sxs-lookup"><span data-stu-id="fb8ce-112">For example, services within a VCR are different from those within a printer.</span></span> <span data-ttu-id="fb8ce-113">特定裝置類型所能提供的一組服務的相關資訊，會在裝置所裝載的 XML 裝置描述檔中捕捉。</span><span class="sxs-lookup"><span data-stu-id="fb8ce-113">Information about the set of services that a particular device type can provide is captured in an XML device description document that the device hosts.</span></span> <span data-ttu-id="fb8ce-114">裝置描述也會列出與裝置相關聯的屬性，例如裝置名稱和圖示。</span><span class="sxs-lookup"><span data-stu-id="fb8ce-114">The device description also lists properties such as device name and icons associated with the device.</span></span> <span data-ttu-id="fb8ce-115">Microsoft 已增強 UPnP 支援，以包含與 [PnP X](/previous-versions/windows/desktop/fundisc/pnp-x) 和函式 [探索](/previous-versions/windows/desktop/fundisc/fd-portal)的整合。</span><span class="sxs-lookup"><span data-stu-id="fb8ce-115">Microsoft has enhanced UPnP support to include integration with [PnP-X](/previous-versions/windows/desktop/fundisc/pnp-x) and [Function Discovery](/previous-versions/windows/desktop/fundisc/fd-portal).</span></span>

<span data-ttu-id="fb8ce-116">UPnP 架構不僅僅是隨插即用周邊模型的簡單延伸。</span><span class="sxs-lookup"><span data-stu-id="fb8ce-116">The UPnP architecture is more than just a simple extension of the plug-and-play peripheral model.</span></span> <span data-ttu-id="fb8ce-117">它支援零設定、不可見的網路功能，以及從各式各樣的廠商探索各種裝置類別的自動探索。</span><span class="sxs-lookup"><span data-stu-id="fb8ce-117">It supports zero-configuration, invisible networking and automatic discovery for a range of device categories from a wide range of vendors.</span></span> <span data-ttu-id="fb8ce-118">這可讓裝置動態地加入網路、取得 IP 位址，並在要求時傳達其功能。</span><span class="sxs-lookup"><span data-stu-id="fb8ce-118">This enables a device to dynamically join a network, obtain an IP address, and convey its capabilities upon request.</span></span> <span data-ttu-id="fb8ce-119">然後，其他控制點可搭配 UPnP 技術使用控制點 API，以瞭解其他裝置的目前狀態和功能。</span><span class="sxs-lookup"><span data-stu-id="fb8ce-119">Then, other control points can use the Control Point API with UPnP technology to learn about the presence and capabilities of other devices.</span></span> <span data-ttu-id="fb8ce-120">裝置在不再使用網路時，可以順暢且自動地離開網路。</span><span class="sxs-lookup"><span data-stu-id="fb8ce-120">A device can leave a network smoothly and automatically when it is no longer in use.</span></span>

<span data-ttu-id="fb8ce-121">UPnP 技術的通用功能為何？</span><span class="sxs-lookup"><span data-stu-id="fb8ce-121">What is universal about UPnP technology?</span></span>

-   <span data-ttu-id="fb8ce-122">媒體和裝置獨立性。</span><span class="sxs-lookup"><span data-stu-id="fb8ce-122">Media and device independence.</span></span> <span data-ttu-id="fb8ce-123">UPnP 技術可以在任何媒體上執行，包括電話線路、電源線、乙太網路、RF 和1394。</span><span class="sxs-lookup"><span data-stu-id="fb8ce-123">UPnP technology can run on any medium including phone line, power line, Ethernet, RF, and 1394.</span></span>

<span data-ttu-id="fb8ce-136">UPnP 技術的範圍廣泛，是以家用網路、鄰近網路以及小型企業和商業大樓的網路為目標。</span><span class="sxs-lookup"><span data-stu-id="fb8ce-136">UPnP technology is broad in scope in that it targets home networks, proximity networks, and networks in small businesses and commercial buildings.</span></span> <span data-ttu-id="fb8ce-137">它會在網路上任何控制裝置的命令下，啟用任何兩個裝置之間的資料通訊。</span><span class="sxs-lookup"><span data-stu-id="fb8ce-137">It enables data communication between any two devices under the command of any control device on the network.</span></span> <span data-ttu-id="fb8ce-138">UPnP 技術與任何特定的作業系統、程式設計語言或實體媒體無關。</span><span class="sxs-lookup"><span data-stu-id="fb8ce-138">UPnP technology is independent of any particular operating system, programming language, or physical medium.</span></span>
