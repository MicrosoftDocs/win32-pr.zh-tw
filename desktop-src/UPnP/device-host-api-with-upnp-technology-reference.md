# <a name="device-host-api-reference"></a>裝置主機 API 參考

下列介面是裝置主機 API 與 UPnP 技術的一部分。 具有 UPnP 技術的裝置主機支援 Microsoft Visual Basic 和 c + +。

此 API 不提供 Microsoft Visual Basic Scripting Edition (VBScript) 的支援。



| 介面                                                  | 描述                                                                                         |
|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [**IUPnPDeviceControl**](/windows/desktop/api/Upnphost/nn-upnphost-iupnpdevicecontrol)           | 由裝置主機用來管理裝置和相關服務。                                     |
| [**IUPnPDeviceProvider**](/windows/desktop/api/Upnphost/nn-upnphost-iupnpdeviceprovider)         | 由裝置主機用來啟動和停止裝置提供者。                                         |
| [**IUPnPEventSink**](/windows/desktop/api/Upnphost/nn-upnphost-iupnpeventsink)                   | 供裝置用來將事件通知傳送至裝置主機。                                     |
| [**IUPnPEventSource**](/windows/desktop/api/Upnphost/nn-upnphost-iupnpeventsource)               | 由裝置主機用來訂閱或取消訂閱託管服務所提供的事件。       |
| [**IUPnPRegistrar**](/windows/desktop/api/Upnphost/nn-upnphost-iupnpregistrar)                   | 供裝置用來向裝置主機註冊。                                                   |
| [**IUPnPRemoteEndpointInfo**](/windows/desktop/api/Upnphost/nn-upnphost-iupnpremoteendpointinfo) | 裝置用來取得要求者的相關資訊 (也就是控制點) 和要求。 |
| [**IUPnPReregistrar**](/windows/desktop/api/Upnphost/nn-upnphost-iupnpreregistrar)               | 供裝置用來向裝置主機重新註冊。                                                |
