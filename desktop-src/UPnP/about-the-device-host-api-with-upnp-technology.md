# <a name="about-the-device-host-api"></a>關於裝置主機 API

具有 UPnP 技術的裝置主機 API 是在 Windows 平臺上執行以 UPnP 為基礎之裝置功能的架構。 使用裝置主機 API 搭配 UPnP 技術來建立裝置 (稱為「 [*託管裝置*](h-gly.md) 」的開發人員) 只需要執行裝置的核心功能。 開發人員可以依賴裝置主機來處理探索、描述、控制、事件和簡報的 UPnP 特定詳細資料。 裝置主機會驗證來自 UPnP 型用戶端的傳入資料，並根據 UPnP 裝置架構來格式化託管裝置上的所有傳出資料。

下列各節將說明一般情況下，UPnP 裝置主機的運作方式：

-   [向裝置主機註冊託管裝置](registering-a-hosted-device-with-the-device-host.md)
-   [裝置提供者](device-providers.md)
-   [事件](eventing.md)
-   [展示](presentation.md)
