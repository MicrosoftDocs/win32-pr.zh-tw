# <a name="what-is-the-control-point-object-model"></a>什麼是控制點物件模型？

下圖顯示基本控制點物件模型。

![控制點物件模型](images/upnp-object-model.png)

使用裝置搜尋工具來搜尋裝置會建立裝置集合。 裝置集合包含零或多個裝置物件。 應用程式可以使用各種裝置集合方法來存取個別的裝置物件。

裝置物件一律會包含包含一或多個服務物件的服務集合。 應用程式會使用這些服務物件來與裝置通訊及控制裝置。
