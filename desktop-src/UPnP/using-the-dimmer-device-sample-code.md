# <a name="using-the-dimmer-device-sample-code"></a>使用暗暗裝置範例程式碼

平臺軟體發展工具組 (SDK) 包含更暗的裝置範例程式碼。 此範例程式碼示範如何使用裝置主機 API 搭配 UPnP 技術來實行簡單的 UPnP 型裝置。

DCO \_ DimmerService 目錄包含裝置 .dll 的執行。 >registerdevice.js 目錄包含用來向電腦上的 UPnP 註冊機構服務註冊裝置 .dll 的執行。 逐步目錄包含 VC 6.0 專案，這些專案會對應至 Readme.txt 檔中所包含的「逐步指南」。 「逐步指南」適用于沒有先前 ATL/COM 經驗的開發人員。
