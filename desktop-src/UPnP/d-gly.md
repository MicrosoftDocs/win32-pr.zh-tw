# <a name="d-upnp-apis"></a>D (UPnP Api) 

包含以字母 D 開頭的 UPnP 相關詞彙。

[B](a-gly.md) [C](c-gly.md) D [E](e-gly.md) F G [H](h-gly.md) I J K L M [N](n-gly.md) O [P](p-gly.md) Q [R](r-gly.md) [S](s-gly.md) T W [](u-gly.md) X Y Z

<dl> <dt>

<span id="upnp.d_1_gly"></span><span id="UPNP.D_1_GLY"></span>**裝置**
</dt> <dd>

裝置是其他服務或裝置的容器。 裝置可以包含其他邏輯裝置。

</dd> <dt>

<span id="upnp.d_2_gly"></span><span id="UPNP.D_2_GLY"></span>**裝置控制物件**
</dt> <dd>

裝置控制物件是裝置服務物件的集中管理點。 註冊裝置時，其裝置控制物件會傳遞至裝置主機。

</dd> <dt>

<span id="upnp.d_3_gly"></span><span id="UPNP.D_3_GLY"></span>**裝置描述**
</dt> <dd>

裝置描述是實體裝置中所有裝置的一般屬性清單，包括服務、裝置的階層，以及其屬性。

</dd> <dt>

<span id="upnp.d_4_gly"></span><span id="UPNP.D_4_GLY"></span>**裝置主機**
</dt> <dd>

裝置主機會處理裝置功能的探索、描述、控制和事件部分。 裝置包含用來處理裝置特定功能的程式碼，例如倒帶 VCR。

</dd> <dt>

<span id="upnp.d_5_gly"></span><span id="UPNP.D_5_GLY"></span>**裝置提供者**
</dt> <dd>

裝置提供者是向裝置主機註冊的實體。 此裝置提供者不會在網路上發佈。 不過，裝置提供者會註冊其他裝置，並將它們發佈到網路上。

</dd> <dt>

<span id="upnp.d_6_gly"></span><span id="UPNP.D_6_GLY"></span>**裝置類型**
</dt> <dd>

標準裝置類型是以 urn：架構-upnp-org： device：後面接著 UPnP 論壇所指派的唯一名稱表示。

UPnP 型裝置範本與裝置類型之間有一對一的關聯性。 UPnP 廠商可能會指定其他裝置類型;這些是以 urn：功能變數名稱： device：後面接著廠商指派的唯一名稱來表示，其中網功能變數名稱稱是註冊給廠商的功能變數名稱。

</dd> </dl>
