# <a name="msvm_guestcommunicationservice-class"></a>Msvm \_ GuestCommunicationService 類別

代表來賓通訊服務。 它是用來與來自 Hyper-v 主機的來賓進行通訊。

下列語法已經過受管理物件格式 (MOF) 程式碼簡化，並包含所有已繼承的屬性。

## <a name="syntax"></a>語法

``` syntax
[Dynamic, Provider("VmmsWmiInstanceAndMethodProvider"), AMENDMENT]
class Msvm_GuestCommunicationService : Msvm_GuestService
{
};
```

## <a name="members"></a>成員

**Msvm \_ GuestCommunicationService** 類別未定義任何成員。
