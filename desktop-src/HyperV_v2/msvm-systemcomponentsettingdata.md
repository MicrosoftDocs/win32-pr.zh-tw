# <a name="msvm_systemcomponentsettingdata-class"></a>Msvm \_ SystemComponentSettingData 類別

用來設定資料類別的泛型基類，代表虛擬系統的元件。

下列語法已經過受管理物件格式 (MOF) 程式碼簡化，並包含所有已繼承的屬性。

## <a name="syntax"></a>語法

``` syntax
[Abstract, AMENDMENT]
class Msvm_SystemComponentSettingData : CIM_SettingData
{
};
```

## <a name="members"></a>成員

**Msvm \_ SystemComponentSettingData** 類別未定義任何成員。
