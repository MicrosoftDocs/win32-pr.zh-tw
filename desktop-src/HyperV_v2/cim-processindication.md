# <a name="cim_processindication-class"></a>CIM \_ ProcessIndication 類別

**CIM \_ProcessIndication** 是特殊指示類別的抽象超類別，可處理提供者和檢測所發佈的特定變更和警示。

## <a name="syntax"></a>語法

``` syntax
[Indication, Version("2.6.0"), UMLPackagePath("CIM::Event"), AMENDMENT]
class CIM_ProcessIndication : CIM_Indication
{
};
```

## <a name="members"></a>成員

**CIM \_ ProcessIndication** 類別不會定義任何成員。
