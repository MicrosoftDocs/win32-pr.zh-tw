# <a name="deletevhdsnapshot-method-of-the-msvm_imagemanagementservice-class"></a>Msvm ImageManagementService 類別的 DeleteVHDSnapshot 方法 \_

刪除 VHD 設定檔內的 VHD 快照集專案。

## <a name="syntax"></a>語法


```mof
uint32 DeleteVHDSnapshot(
  [in]  string              VHDSetPath,
  [in]  string              SnapshotId,
  [in]  boolean             PersistReferenceSnapshot,
  [out] CIM_ConcreteJob REF Job
);
```



## <a name="parameters"></a>參數

<dl> <dt>

*VHDSetPath* \[在\]
</dt> <dd>

字串，包含包含有問題快照集的 VHD 集檔案的路徑。

</dd> <dt>

*SnapshotId* \[在\]
</dt> <dd>

GUID，代表要刪除的 VHD 快照集專案的快照集識別碼。

</dd> <dt>

*PersistReferenceSnapshot* \[在\]
</dt> <dd>

是否應在刪除快照集之後保存僅參考的快照記錄。

</dd> <dt>

*作業* \[擴展\]
</dt> <dd>

如果工作已完成) ，則作業 (的參考可以是 null。

</dd> </dl>
