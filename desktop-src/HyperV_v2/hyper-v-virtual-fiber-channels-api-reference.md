# <a name="hyper-v-virtual-fibre-channel-api-reference"></a>Hyper-v virtual 光纖通道 API 參考

Hyper-v virtual 光纖通道 API 會定義下列程式設計項目。

## <a name="in-this-section"></a>本節內容



| 主題                                                                                    | 描述                                                                                                    |
|------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------|
| [**Msvm \_ ExternalFcPort**](msvm-externalfcport.md)<br/>                           | 表示外部光纖通道埠。<br/>                                                          |
| [**Msvm \_ FcActiveConnection**](msvm-fcactiveconnection.md)<br/>                   | 將交換器埠連接到埠所連接的光纖通道端點。<br/>                |
| [**Msvm \_ FcDeviceSAPImplementation**](msvm-fcdevicesapimplementation.md)<br/>     | 代表服務存取點與執行它的邏輯裝置之間的關聯。<br/> |
| [**Msvm \_ FcEndpoint**](msvm-fcendpoint.md)<br/>                                   | 代表光纖通道埠的邏輯連接點。<br/>                                   |
| [**Msvm \_ FcPortAllocationSettingData**](msvm-fcportallocationsettingdata.md)<br/> | 代表綜合光纖通道埠或光纖通道交換器埠的設定狀態。<br/>   |
| [**Msvm \_ FcSwitchPort**](msvm-fcswitchport.md)<br/>                               | 代表虛擬光纖通道參數上的埠。<br/>                                              |
| [**Msvm \_ SyntheticFcPort**](msvm-syntheticfcport.md)<br/>                         | 代表綜合光纖通道埠。<br/>                                                          |
| [**Msvm \_ SyntheticFcPortSettingData**](msvm-syntheticfcportsettingdata.md)<br/>   | 代表綜合光纖通道埠的已設定狀態。<br/>                                  |
| [**Msvm \_ VirtualFcSwitch**](msvm-virtualfcswitch.md)<br/>                         | 表示虛擬光纖通道參數。<br/>                                                          |
| [**Msvm \_ VirtualFcSwitchSettingData**](msvm-virtualfcswitchsettingdata.md)<br/>   | 代表虛擬光纖通道交換器的設定。<br/>                                     |
