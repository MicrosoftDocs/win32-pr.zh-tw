# <a name="resource-management-classes"></a>資源管理類別

以下是與資源管理相關的虛擬化 WMI 類別。

## <a name="in-this-section"></a>本節內容



| 主題                                                                                                        | 描述                                                                                                                                                      |
|--------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Msvm \_ AbstractResourcePoolSettingData**](msvm-abstractresourcepoolsettingdata.md)<br/>             | 表示非配置相關之 [**Msvm \_ ResourcePool**](msvm-resourcepool.md) 實例的設定。<br/>                                |
| [**Msvm \_ AllocationCapabilities**](msvm-allocationcapabilities.md)<br/>                               | 定義用戶端可以探索虛擬資源預設設定之有效範圍的方法。<br/>                                          |
| [**Msvm \_ ElementAllocatedFromPool**](msvm-elementallocatedfrompool.md)<br/>                           | 將已配置資源的實例與從中配置資源的資源集區產生關聯。<br/>                                                   |
| [**Msvm \_ ElementCapabilities**](msvm-elementcapabilities.md)<br/>                                     | 代表 managed 元素與其功能之間的關聯。<br/>                                                                           |
| [**Msvm \_ FeatureSettingData**](msvm-featuresettingdata.md)<br/>                                       | 抽象類別，表示系統或元件之特定功能的設定。<br/>                                                           |
| [**Msvm \_ FeatureSettingsDefineCapabilities**](msvm-featuresettingsdefinecapabilities.md)<br/>         | 提供乙太網路交換器功能實例與資源的最小值、最大值、增量和預設設定之間的連結。<br/> |
| [**Msvm \_ ResourceAllocationFromPool**](msvm-resourceallocationfrompool.md)<br/>                       | 將資源配置的實例與其配置來源的資源集區產生關聯。<br/>                                                    |
| [**Msvm \_ ResourceAllocationSettingData**](msvm-resourceallocationsettingdata.md)<br/>                 | 代表虛擬資源目前和記錄的配置狀態。<br/>                                                                          |
| [**Msvm \_ ResourcePool**](msvm-resourcepool.md)<br/>                                                   | 描述可在虛擬機器中使用的虛擬資源類型。<br/>                                                                           |
| [**Msvm \_ ResourcePoolConfigurationCapabilities**](msvm-resourcepoolconfigurationcapabilities.md)<br/> | 描述相關聯 [**Msvm \_ ResourcePoolConfigurationService**](msvm-resourcepoolconfigurationservice.md) 類別的功能。<br/>           |
| [**Msvm \_ ResourcePoolConfigurationService**](msvm-resourcepoolconfigurationservice.md)<br/>           | 提供資源集區的主動管理。<br/>                                                                                                     |
| [**Msvm \_ ResourcePoolSettingData**](msvm-resourcepoolsettingdata.md)<br/>                             | 表示非配置相關之 [**Msvm \_ ResourcePool**](msvm-resourcepool.md) 實例的設定。<br/>                                |
| [**Msvm \_ SettingsDefineCapabilities**](msvm-settingsdefinecapabilities.md)<br/>                       | 提供功能實例與資源的最小值、最大值、增量和預設設定之間的連結。<br/>                         |
