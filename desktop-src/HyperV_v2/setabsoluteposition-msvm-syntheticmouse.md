# <a name="setabsoluteposition-method-of-the-msvm_syntheticmouse-class"></a>Msvm SyntheticMouse 類別的 SetAbsolutePosition 方法 \_

設定滑鼠游標的水準和垂直位置。

## <a name="syntax"></a>語法


```mof
uint32 SetAbsolutePosition(
  [in] sint32 horizontalPosition,
  [in] sint32 verticalPosition
);
```



## <a name="parameters"></a>參數

<dl> <dt>

*horizontalPosition* \[在\]
</dt> <dd>

類型： **sint32**

滑鼠游標的新水準位置（以圖元為單位）。

</dd> <dt>

*verticalPosition* \[在\]
</dt> <dd>

類型： **sint32**

滑鼠游標的新垂直位置（以圖元為單位）。

</dd> </dl>

## <a name="return-value"></a>傳回值

類型： **uint32**

傳回值為零表示成功。 非零值表示無法改變捲軸位置。
