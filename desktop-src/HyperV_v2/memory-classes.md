# <a name="memory-classes"></a>記憶體類別

記憶體類別會建立虛擬機器的記憶體模型。

以下是與記憶體相關的虛擬化 WMI 類別。

## <a name="in-this-section"></a>本節內容



| 主題                                                                                      | 描述                                                                                                         |
|--------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [**Msvm \_ ElementAllocatedFromNumaNode**](msvm-elementallocatedfromnumanode.md)<br/> | 將已配置資源的實例與從中配置的實體 NUMA 節點產生關聯。<br/> |
| [**Msvm \_ 記憶體**](msvm-memory.md)<br/>                                             | 代表目前配置給虛擬機器的記憶體。<br/>                                          |
| [**Msvm \_ MemorySettingData**](msvm-memorysettingdata.md)<br/>                       | 代表虛擬機器的記憶體設定狀態。<br/>                                     |
| [**Msvm \_ NumaNode**](msvm-numanode.md)<br/>                                         | 表示虛擬系統 (NUMA) 節點的非統一記憶體存取。<br/>                                  |
