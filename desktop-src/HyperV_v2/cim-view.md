# <a name="cim_view-class"></a>CIM \_ View 類別

**CIM \_ View** 類別是類別的超類，可提供已取消標準化、匯總的 managed 資源標記法。

## <a name="syntax"></a>語法

``` syntax
[Abstract, Version("2.26.0"), UMLPackagePath("CIM::Core::CoreElements"), AMENDMENT]
class CIM_View : CIM_ManagedElement
{
};
```

## <a name="members"></a>成員

**CIM \_ View** 類別未定義任何成員。
