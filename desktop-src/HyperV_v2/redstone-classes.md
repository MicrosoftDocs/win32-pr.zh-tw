# <a name="redstone-classes"></a>Redstone 類別

包含 Windows 10 1703 版的新類別。

| 主題                                                                                                                | 描述                                                                                                                                                                                                                                                                                               |
|----------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Msvm \_ AssignableDeviceDismountSettingData**](msvm-assignabledevicedismountsettingdata.md)<br/>             | 代表要匯入的虛擬系統設定。 由 [**Msvm \_ AssignableDeviceService**](msvm-assignabledeviceservice.md)類別的 [**卸載**](msvm-assignabledeviceservice-dismountassignabledevice.md)方法所使用。<br/>                                                        |

| [**Msvm \_ CollectionReferencePointExportJob**](msvm-collectionreferencepointexportjob.md)<br/>                 | 此類別代表集合參考點匯出作業工作。<br/>                                                                                                                                                                                                                       |
| [**Msvm \_ EthernetSwitchHardwareOffloadSettingData**](msvm-ethernetswitchhardwareoffloadsettingdata.md)<br/>   | 代表切換卸載設定。<br/>                                                                                                                                                                                                                                                        |
| [**Msvm \_ EthernetSwitchPortMigrationQosSettingData**](msvm-ethernetswitchportmigrationqossettingdata.md)<br/> | 代表 VFP QOS 設定。<br/>                                                                                                                                                                                                                                                               |
| [**Msvm \_ EthernetSwitchPortRdmaSettingData**](msvm-ethernetswitchportrdmasettingdata.md)<br/>                 | 表示埠 RDMA 功能設定資料。<br/>                                                                                                                                                                                                                                                 |

| [**Msvm \_ GpuPartition**](msvm-gpupartition.md)<br/>                                                           | 代表 GPU 分割區的狀態。<br/>                                                                                                                                                                                                                                                     |
| [**Msvm \_ GpuPartitionSettingData**](msvm-gpupartitionsettingdata.md)<br/>                                     | 代表 GPU 磁碟分割裝置的設定狀態。<br/>                                                                                                                                                                                                                                     |
| [**Msvm \_ NetworkConnectionDiagnosticInformation**](msvm-networkconnectiondiagnosticinformation.md)<br/>       | 提供有關虛擬機器之網路連線能力的資訊。<br/>                                                                                                                                                                                                                     |
| [**Msvm \_ NetworkConnectionDiagnosticSettingData**](msvm-networkconnectiondiagnosticsettingdata.md)<br/>       | 表示用來測試虛擬機器的網路連線能力的設定。 <br/>                                                                                                                                                                                                           |
| [**Msvm \_ PartitionableGpu**](msvm-partitionablegpu.md)<br/>                                                   | 代表可分割 GPU。 每個 GPU 都可以分割成多個 GPU 磁碟分割，而這些分割區可指派給虛擬機器作為 vGPU。<br/>                                                                                                                                                  |
| [**Msvm \_ PciExpress**](msvm-pciexpress.md)<br/>                                                               | 表示 PCI Express 埠的狀態。<br/>                                                                                                                                                                                                                                                  |
| [**Msvm \_ PciExpressSettingData**](msvm-pciexpresssettingdata.md)<br/>                                         | 表示 PCI Express 埠已設定的狀態。<br/>                                                                                                                                                                                                                                         |
| [**Msvm \_ SecurityElement**](msvm-securityelement.md)<br/>                                                     | 代表 [**CIM \_**](cim-computersystem.md)的執行時間安全性設定。<br/>                                                                                                                                                                                               |
| [**Msvm \_ SecurityService**](msvm-securityservice.md)<br/>                                                     | 表示安全性服務。 它是用來設定虛擬系統安全性設定。<br/>                                                                                                                                                                                                  |
| [**Msvm \_ SecuritySettingData**](msvm-securitysettingdata.md)<br/>                                             | 代表所設定之安全性設定的狀態 <br/>                                                                                                                                                                                                                                  |

| [**Msvm \_ SummaryInformationBase**](msvm-summaryinformationbase.md)<br/>                                       | 用於 [**Msvm \_ VirtualSystemManagementService**](msvm-virtualsystemmanagementservice.md)類別中的 [**GetSummaryInformation**](getsummaryinformation-msvm-virtualsystemmanagementservice.md)方法，以快速取得與虛擬系統或快照集相關的一般資訊。<br/> |
| [**Msvm \_ SystemComponentSettingData**](msvm-systemcomponentsettingdata.md)<br/>                               | 用來設定資料類別的泛型基類，代表虛擬系統的元件。<br/>                                                                                                                                                                                                     |
| [**Msvm \_ VirtualSystemReferencePointExportJob**](msvm-virtualsystemreferencepointexportjob.md)<br/>           | 此類別代表虛擬系統參考點匯出作業工作。<br/>                                                                                                                                                                                                                   |
