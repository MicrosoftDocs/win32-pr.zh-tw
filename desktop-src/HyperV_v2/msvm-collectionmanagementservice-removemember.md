# <a name="removemember-method-of-the-msvm_collectionmanagementservice-class"></a>Msvm CollectionManagementService 類別的 RemoveMember 方法 \_

將指定的 managed 元素移除為指定之 [**CIM \_ CollectionOfMSEs**](cim-collectionofmses.md) 物件的成員。

## <a name="syntax"></a>語法


```mof
uint32 RemoveMember(
  [in]  CIM_ManagedElement   REF Member,
  [in]  CIM_CollectionOfMSEs REF Collection,
  [out] CIM_ConcreteJob      REF Job
);
```



## <a name="parameters"></a>參數

<dl> <dt>

*成員* \[在\]
</dt> <dd>

要移除的成員。

</dd> <dt>

*集合* \[在\]
</dt> <dd>

要從中移除成員的集合。

</dd> <dt>

*作業* \[擴展\]
</dt> <dd>

如果工作已完成) ，則作業 (的參考可以是 null。

</dd> </dl>
