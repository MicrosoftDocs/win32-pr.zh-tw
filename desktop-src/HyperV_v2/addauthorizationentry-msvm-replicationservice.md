# <a name="addauthorizationentry-method-of-the-msvm_replicationservice-class"></a>Msvm ReplicationService 類別的 AddAuthorizationEntry 方法 \_

將授權專案新增至復原伺服器。 這些專案是用來授權恢復伺服器的連線。

## <a name="syntax"></a>語法


```mof
uint32 AddAuthorizationEntry(
  [in]  string              AuthorizationEntry,
  [out] CIM_ConcreteJob REF Job
);
```



## <a name="parameters"></a>參數

<dl> <dt>

*AuthorizationEntry* \[在\]
</dt> <dd>

[**Msvm \_ ReplicationAuthorizationSettingData**](msvm-replicationauthorizationsettingdata.md)類別實例的字串表示，定義要加入的授權專案。

</dd> <dt>

*作業* \[擴展\]
</dt> <dd>

如果作業是以非同步方式執行，這個方法會傳回4096，而此參數會包含衍生自 [**CIM \_ ConcreteJob**](/previous-versions//cc136808(v=vs.85))之物件的參考。

</dd> </dl>
