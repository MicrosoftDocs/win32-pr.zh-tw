# <a name="msvm_persistentmemorycontroller-class"></a>Msvm \_ PersistentMemoryController 類別

代表虛擬持續性記憶體控制器。

下列語法已經過受管理物件格式 (MOF) 程式碼簡化，並包含所有已繼承的屬性。

## <a name="syntax"></a>語法

``` syntax
[Dynamic, Provider("VmmsWmiInstanceAndMethodProvider"), AMENDMENT]
class Msvm_PersistentMemoryController : CIM_Controller
{
};
```

## <a name="members"></a>成員

**Msvm \_ PersistentMemoryController** 類別未定義任何成員。
