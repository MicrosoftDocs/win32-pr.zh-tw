# <a name="hyper-v-wmi-classes"></a>Hyper-v WMI 類別

以下是 Hyper-v WMI 類別。

## <a name="in-this-section"></a>本節內容



| 主題                                                                                 | 描述                                                                                                                                                                                                                                                                                 |
|---------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [CIM 類別](cim-classes.md)<br/>                                             | CIM 類別是所有 WMI 物件衍生自的基類。<br/>                                                                                                                                                                                                               |
| [BIOS 類別](bios-classes.md)<br/>                                           | 虛擬 BIOS 是載入 RAM 的軟體映射，可設定部分的基本層面，並將電腦系統開機。 每台電腦系統都有一個 BIOS 元素，而且無法取代或移除該元素。<br/>                                                 |
| [輸入類別](input-classes.md)<br/>                                         | 使用者輸入裝置是由這些類別表示。 虛擬機器一律會有一個與其相關聯之類別的實例。<br/>                                                                                                                                             |
| [Integration services 類別](integration-services-classes.md)<br/>           | Integration services 是一項軟體服務，可在子虛擬機器內的客體作業系統上執行，並作為管理作業系統中虛擬化堆疊的一部分，以提供與管理作業系統的某種程度的整合。<br/> |
| [記憶體類別](memory-classes.md)<br/>                                       | 記憶體類別會建立虛擬機器的記憶體模型。<br/>                                                                                                                                                                                                                       |
| [處理器類別](processor-classes.md)<br/>                                 | 處理器類別會建立虛擬機器的處理器模型。<br/>                                                                                                                                                                                                                   |
| [設定檔註冊類別](profile-registration-classes.md)<br/>           | 與設定檔註冊相關的虛擬化 WMI 類別。<br/>                                                                                                                                                                                                                      |
| [遠端桌面類別](remotefx-classes.md)<br/>                             | 遠端桌面虛擬化 WMI 類別可用來管理遠端桌面和 RemoteFX 資源和虛擬機器。<br/>                                                                                                                                                     |
| [資源管理類別](resource-management-classes.md)<br/>             | 與資源管理相關的虛擬化 WMI 類別。<br/>                                                                                                                                                                                                                       |
| [序列裝置類別](serial-devices-classes.md)<br/>                       | 虛擬機器中的序列裝置包含序列控制器和序列埠。 每部虛擬機器只有一個串列控制器，且每個序列控制器都有兩個序列埠。<br/>                                                                         |
| [儲存類別](storage-classes.md)<br/>                                     | 存放裝置物件分成三種類型：控制器、磁片磁碟機和媒體。<br/>                                                                                                                                                                                              |
| [影片類別](video-classes.md)<br/>                                         | 所有虛擬機器都反映出模擬的 S3 影片控制器以及加速的綜合視訊控制器。<br/>                                                                                                                                                     |
| [虛擬系統類別](virtual-system-classes.md)<br/>                       | 虛擬機器相關的虛擬化 WMI 類別。<br/>                                                                                                                                                                                                                       |
| [虛擬系統管理類別](virtual-system-management-classes.md)<br/> | 虛擬機器管理的相關虛擬 WMI 類別。<br/>                                                                                                                                                                                                                |
