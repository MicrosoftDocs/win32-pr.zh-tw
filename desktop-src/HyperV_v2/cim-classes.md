# <a name="cim-classes-hyper-v-management"></a> (Hyper-v 管理) 的 CIM 類別

CIM 類別是所有 WMI 物件衍生自的基類。 每個提供者可能會使用略有不同的 CIM 類別版本;因此，大部分 v2 WMI 提供者都包含自己的 CIM 類別檔。

您可以在根 \\ 虛擬化 v2 命名空間中找到下列 CIM 類別 \\ 。

## <a name="in-this-section"></a>本節內容
