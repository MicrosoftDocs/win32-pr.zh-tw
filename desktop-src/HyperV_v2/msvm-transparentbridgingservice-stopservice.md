# <a name="stopservice-method-of-the-msvm_transparentbridgingservice-class"></a>Msvm TransparentBridgingService 類別的 StopService 方法 \_

## <a name="syntax"></a>語法


```mof
uint32 StopService();
```



## <a name="parameters"></a>參數

<dl> <dt>
