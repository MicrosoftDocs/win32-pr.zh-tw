# <a name="hyper-v-migration-api-reference"></a>Hyper-v 遷移 API 參考

Hyper-v 遷移 API 會定義下列程式設計項目。

## <a name="in-this-section"></a>本節內容



| 主題                                                                                                                                | 描述                                                                                                                                                                                         |
|--------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Msvm \_ MigrationJob**](msvm-migrationjob.md)<br/>                                                                           | 此類別代表由虛擬系統移轉服務為儲存或虛擬系統移轉所建立的遷移作業工作。<br/>                                                 |
| [**Msvm \_ VirtualSystemMigrationCapabilities**](msvm-virtualsystemmigrationcapabilities.md)<br/>                               | 定義用戶端可以探索遷移服務所提供之方法的方式，以及虛擬系統移轉設定資料的有效範圍。<br/>                                |
| [**Msvm \_ VirtualSystemMigrationNetworkSettingData**](msvm-virtualsystemmigrationnetworksettingdata.md)<br/>                   | 代表虛擬系統移轉服務接聽傳入虛擬系統移轉的網路。<br/>                                                                 |
| [**Msvm \_ VirtualSystemMigrationService**](msvm-virtualsystemmigrationservice.md)<br/>                                         | 表示虛擬系統移轉服務。 它是用來遷移虛擬系統，或將虛擬系統的儲存體從一個虛擬化平臺遷移到另一個虛擬化平臺。<br/> |
| [**Msvm \_ VirtualSystemMigrationServiceSettingData**](msvm-virtualsystemmigrationservicesettingdata.md)<br/>                   | 代表主機上虛擬系統移轉服務的設定。<br/>                                                                                                              |
| [**Msvm \_ VirtualSystemMigrationServiceSettingDataComponent**](msvm-virtualsystemmigrationservicesettingdatacomponent.md)<br/> | 用來表示虛擬系統移轉服務之虛擬系統移轉網路設定的關聯。<br/>                                                                      |
| [**Msvm \_ VirtualSystemMigrationSettingData**](msvm-virtualsystemmigrationsettingdata.md)<br/>                                 | 代表遷移虛擬系統和附加至虛擬系統之存放裝置的遷移設定。<br/>                                                                           |
