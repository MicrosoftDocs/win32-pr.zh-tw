處理器類別會建立虛擬機器的處理器模型。

以下是與處理器相關的虛擬化 WMI 類別。

## <a name="in-this-section"></a>本節內容



| 主題                                                                      | 描述                                                                               |
|----------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**Msvm \_ 處理器**](msvm-processor.md)<br/>                       | 代表虛擬機器中的虛擬處理器。<br/>                         |
| [**Msvm \_ ProcessorPool**](msvm-processorpool.md)<br/>               | 匯總可能配置給虛擬機器的處理器資源。<br/> |
| [**Msvm \_ ProcessorSettingData**](msvm-processorsettingdata.md)<br/> | 代表虛擬機器的虛擬處理器設定。<br/>               |
