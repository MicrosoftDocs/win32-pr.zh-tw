# <a name="msvm_tpm-class"></a>Msvm \_ TPM 類別

## <a name="syntax"></a>語法

``` syntax
[Dynamic, Provider("VmmsWmiInstanceAndMethodProvider"), AMENDMENT]
class Msvm_TPM : CIM_TPM
{
};
```
