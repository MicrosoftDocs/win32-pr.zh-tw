# <a name="stopservice-method-of-the-msvm_imagemanagementservice-class"></a>Msvm ImageManagementService 類別的 StopService 方法 \_

停止服務。

## <a name="syntax"></a>語法


```mof
uint32 StopService();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法會傳回下列其中一個值：

<dl> <dt>

**已完成，沒有錯誤** (0) 
</dt> <dt>

**不支援** (1) 
</dt> </dl>
