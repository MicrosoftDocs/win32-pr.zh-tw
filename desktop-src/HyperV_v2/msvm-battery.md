# <a name="msvm_battery-class"></a>Msvm \_ 電池類別

代表虛擬電池裝置。

下列語法已經過受管理物件格式 (MOF) 程式碼簡化，並包含所有已繼承的屬性。

## <a name="syntax"></a>語法

``` syntax
[Dynamic, Provider("VmmsWmiInstanceAndMethodProvider"), AMENDMENT]
class Msvm_Battery : CIM_LogicalDevice
{
};
```
