# <a name="typescancodes-method-of-the-msvm_keyboard-class"></a>Msvm 鍵盤類別的 TypeScancodes 方法 \_

使用掃描碼來模擬按鍵序列。

## <a name="syntax"></a>語法


```mof
uint32 TypeScancodes(
  [in] uint8 Scancodes[]
);
```



## <a name="parameters"></a>參數

<dl> <dt>

*Scancodes* \[在\]
</dt> <dd>

類型： **uint8 \[ \]**

陣列，包含要輸入的掃描碼。

</dd> </dl>
