# <a name="profile-registration-classes"></a>設定檔註冊類別

以下是與設定檔註冊相關的虛擬化 WMI 類別。

## <a name="in-this-section"></a>本節內容



| 主題                                                                                                    | 描述                                                                                                                                                                      |
|----------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Msvm \_ ElementConformsToProfile**](msvm-elementconformstoprofile.md)<br/>                       | 定義所參考系統符合的已註冊設定檔。<br/>                                                                                              |
| [**Msvm \_ HostedResourcePool**](msvm-hostedresourcepool.md)<br/>                                   | 表示系統元件關聯的特製化，該關聯會建立資源集區是在系統的內容中定義的。<br/>                      |
| [**Msvm \_ ReferencedProfile**](msvm-referencedprofile.md)<br/>                                     | 描述另一個已註冊設定檔所參考的設定檔。<br/>                                                                                                 |
| [**Msvm \_ RegisteredProfile**](msvm-registeredprofile.md)<br/>                                     | 描述一組具有必要屬性和方法的類別，這些類別具有必要的屬性和方法，需要用來管理真實世界的實體，或以互通的方式支援使用案例。<br/> |
| [**Msvm \_ ResourcePoolComponent**](msvm-resourcepoolcomponent.md)<br/>                             | 表示 Microsoft Windows Hyper-v 平臺的資源集區元素。<br/>                                                                                         |
| [**Msvm \_ ResourcePoolRegistration**](msvm-resourcepoolregistration.md)<br/>                       | 註冊提供全域資源集區相關物件的服務。<br/>                                                                                               |
| [**Msvm \_ ResourceTypeDefinition**](msvm-resourcetypedefinition.md)<br/>                           | 定義資源類型與其實類別的對應。<br/>                                                                                                   |
| [**Msvm \_ VirtualizationComponent**](msvm-virtualizationcomponent.md)<br/>                         | 代表 Microsoft Windows Hyper-v 平臺的服務。<br/>                                                                                                       |
| [**Msvm \_ VirtualizationComponentRegistration**](msvm-virtualizationcomponentregistration.md)<br/> | 表示 Microsoft Hyper-V 平臺中的服務註冊。<br/>                                                                                           |
| [**Msvm \_ VirtualSystemResourceComponent**](msvm-virtualsystemresourcecomponent.md)<br/>           | 代表 Microsoft Windows Hyper-v 平臺的虛擬裝置服務。<br/>                                                                                        |
| [**Msvm \_ VirtualSystemResourceRegistration**](msvm-virtualsystemresourceregistration.md)<br/>     | 註冊可提供虛擬機器特定資源相關物件的服務。<br/>                                                                                  |
