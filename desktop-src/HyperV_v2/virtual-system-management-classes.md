# <a name="virtual-system-management-classes"></a>虛擬系統管理類別

以下是與虛擬機器管理相關的虛擬化 WMI 類別。

## <a name="in-this-section"></a>本節內容



| 主題                                                                                                                | 描述                                                                                                                                                                                                                                                          |
|----------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Msvm \_ AffectedJobElement**](msvm-affectedjobelement.md)<br/>                                               | 表示作業與受管理元素之間的關聯，可能會受到其執行影響。<br/>                                                                                                                                                    |
| [**Msvm \_ ConcreteJob**](msvm-concretejob.md)<br/>                                                             | 代表工作單位，用來追蹤非同步作業的進度。<br/>                                                                                                                                                                   |
| [**Msvm \_ DiskMergeSettingData**](msvm-diskmergesettingdata.md)<br/>                                           | 代表虛擬機器的磁片合併設定的設定狀態。<br/>                                                                                                                                                                      |
| [**Msvm \_ ElementSettingData**](msvm-elementsettingdata.md)<br/>                                               | 將 managed 元素與其設定資料產生關聯。<br/>                                                                                                                                                                                                 |
| [**Msvm \_ 錯誤**](msvm-error.md)<br/>                                                                         | 包含嚴重性、原因、建議動作，以及與 CIM 作業失敗相關之其他資料的相關資訊。<br/>                                                                                                                            |
| [**Msvm \_ HostedDependency**](msvm-hosteddependency.md)<br/>                                                   | 將虛擬機器實例與代表實體、裝載系統的電腦系統物件產生關聯。<br/>                                                                                                                                       |
| [**Msvm \_ HostedService**](msvm-hostedservice.md)<br/>                                                         | 將服務與其主控電腦系統產生關聯。<br/>                                                                                                                                                                                                    |
| [**Msvm \_ LastAppliedSnapshot**](msvm-lastappliedsnapshot.md)<br/>                                             | 代表虛擬系統與最近套用至虛擬系統之快照集的設定資料之間的關聯。<br/>                                                                                                             |
| [**Msvm \_ OwningJobElement**](msvm-owningjobelement.md)<br/>                                                   | 表示作業與負責建立作業的 managed 元素之間的關聯。<br/>                                                                                                                                                  |
| [**Msvm \_ ServiceAffectsElement**](msvm-serviceaffectselement.md)<br/>                                         | 將虛擬機器實例與控制其狀態的管理服務產生關聯。<br/>                                                                                                                                                                |
| [**Msvm \_ ServicingSettings**](msvm-servicingsettings.md)<br/>                                                 | 包含在服務作業期間使用的設定。<br/>                                                                                                                                                                                                       |
| [**Msvm \_ SettingsDefineState**](msvm-settingsdefinestate.md)<br/>                                             | 將虛擬機器及其裝置與 [**CIM \_ SettingData**](/previous-versions//cc136911(v=vs.85)) 的實例產生關聯，以代表適用于這些物件的目前設定。<br/>                                                                               |
| [**Msvm \_ SystemExportSettingData**](msvm-systemexportsettingdata.md)<br/>                                     | 建立虛擬機器與其匯出設定資料的關聯。<br/>                                                                                                                                                                                                 |
| [**Msvm \_ VirtualSystemExportSettingData**](msvm-virtualsystemexportsettingdata.md)<br/>                       | 提供要與 [**Msvm \_ VirtualSystemManagementService**](msvm-virtualsystemmanagementservice.md)類別的 [**ExportSystemDefinition**](exportsystemdefinition-msvm-virtualsystemmanagementservice.md)方法搭配使用的其他資訊。<br/> |
| [**Msvm \_ VirtualSystemManagementCapabilities**](msvm-virtualsystemmanagementcapabilities.md)<br/>             | 描述相關聯 [**Msvm \_ VirtualSystemManagementService**](msvm-virtualsystemmanagementservice.md)的功能。<br/>                                                                                                                         |
| [**Msvm \_ VirtualSystemManagementService**](msvm-virtualsystemmanagementservice.md)<br/>                       | 代表存在於單一主機系統上的虛擬化服務。<br/>                                                                                                                                                                                    |
| [**Msvm \_ VirtualSystemManagementServiceSettingData**](msvm-virtualsystemmanagementservicesettingdata.md)<br/> | 代表存在於單一主機系統上的虛擬化服務的設定。<br/>                                                                                                                                                                   |
| [**Msvm \_ VirtualSystemSnapshotService**](msvm-virtualsystemsnapshotservice.md)<br/>                           | 用來建立、套用和終結虛擬機器快照集的服務。<br/>                                                                                                                                                                                      |
