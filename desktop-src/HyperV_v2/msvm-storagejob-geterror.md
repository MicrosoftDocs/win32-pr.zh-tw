# <a name="geterror-method-of-the-msvm_storagejob-class"></a>Msvm Get-storagejob 類別的 GetError 方法 \_

## <a name="syntax"></a>語法


```mof
uint32 GetError(
  [out] string Error
);
```

*錯誤* \[擴展\]

</dd> </dl>
