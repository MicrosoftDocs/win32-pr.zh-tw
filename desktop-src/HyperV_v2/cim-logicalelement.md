# <a name="cim_logicalelement-class-hyper-v-management"></a>CIM_LogicalElement 類別 (Hyper-v 管理) 

**CIM \_LogicalElement** 是系統所有元件的基類，代表抽象系統元件，例如檔案、進程和邏輯裝置。

## <a name="syntax"></a>語法

``` syntax
[Abstract, Version("2.6.0"), UMLPackagePath("CIM::Core::CoreElements"), AMENDMENT]
class CIM_LogicalElement : CIM_ManagedSystemElement
{
};
```

## <a name="members"></a>成員

**CIM \_ LogicalElement** 類別不會定義任何成員。
