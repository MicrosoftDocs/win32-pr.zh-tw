# <a name="dxgi"></a>DXGI

Microsoft DirectX Graphic Infrastructure (DXGI) 處理列舉圖形介面卡、列舉顯示模式、選取緩衝區格式、在進程之間共用資源 (例如，應用程式和桌面視窗管理員 (DWM) ) ，以及將轉譯的畫面格呈現至視窗或監視器以供顯示。

Direct3D 10、Direct3D 11 和 Direct3D 12 都會使用 DXGI。

雖然大部分圖形程式設計是使用 Direct3D 完成的，但您可以使用 DXGI 來呈現視窗、監視器或其他圖形元件的框架，以供最終撰寫和顯示。 您也可以使用 DXGI 來讀取監視器上的內容。

此檔集包含使用 DXGI 進行程式設計的相關資訊。
