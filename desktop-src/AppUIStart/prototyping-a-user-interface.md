## <a name="who-is-involved"></a><span data-ttu-id="9a937-145">牽涉到誰？</span><span class="sxs-lookup"><span data-stu-id="9a937-145">Who Is Involved?</span></span>

## <a name="what-is-the-scope"></a><span data-ttu-id="9a937-188">範圍為何？</span><span class="sxs-lookup"><span data-stu-id="9a937-188">What is the Scope?</span></span>
