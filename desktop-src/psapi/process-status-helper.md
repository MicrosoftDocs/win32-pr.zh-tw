# <a name="process-status-api"></a>進程狀態 API

進程狀態應用程式開發介面 (PSAPI.DLL) 是協助程式程式庫，可讓您更輕鬆地取得處理常式和設備磁碟機的相關資訊。 如需詳細資訊，請參閱下列主題：

-   [關於 PSAPI.DLL](about-psapi.md)
-   [使用 PSAPI.DLL](using-psapi.md)
-   [PSAPI.DLL 參考](psapi-reference.md)

這些函數可在 Psapi.dll 中取得。

相同的資訊通常可透過登錄中的效能資料取得。 如需相關資訊，請參閱 [Performance Counters](/windows/desktop/PerfCtrs/performance-counters-portal)。
