# <a name="application-gestures"></a>應用程式手勢

應用程式手勢是您選擇讓應用程式支援的手勢。 它們最常用於特別設計來搭配 tablet 畫筆的應用程式。 繼承應用程式可能會使用筆勢，但應用程式中現有的選取模型可能會讓手勢的支援變得複雜。 只有 [Tablet PC 輸入面板] 中支援的手勢可供繼承應用程式使用。

下列各節將說明應用程式手勢及其用法。

-   [應用程式手勢和語義行為](application-gestures-and-semantic-behavior.md)
-   [僅使用 Microsoft 手勢辨識器](using-the-microsoft-gesture-recognizer-only.md)
-   [使用自訂手勢辨識器](using-a-custom-gesture-recognizer.md)
-   [可能與應用程式手勢發生衝突](potential-conflicts-with-application-gestures.md)
-   [記錄應用程式所使用的手勢](documenting-the-gestures-used-by-your-applications.md)
-   [未實現的字元](unimplemented-glyphs.md)
