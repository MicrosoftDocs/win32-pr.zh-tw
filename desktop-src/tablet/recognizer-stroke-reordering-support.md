# <a name="recognizer-stroke-reordering-support"></a>辨識器筆觸重新排序支援

Microsoft 的拉丁腳本辨識器支援筆觸重新排列。

拉丁腳本的 Microsoft 手寫辨識器不會依賴以特定順序撰寫的線條中的筆劃。 相反地，辨識器會根據筆劃之間的空間關聯性，在內部排列筆劃的順序。 例如，拉丁腳本的辨識器支援常見的案例，也就是有人回到行以書寫左括弧 " (" 或加入引號。
