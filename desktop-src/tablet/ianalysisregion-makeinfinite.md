# <a name="ianalysisregionmakeinfinite-method"></a>IAnalysisRegion：： MakeInfinite 方法

展開 [**IAnalysisRegion**](ianalysisregion.md) 以代表無限區域。

## <a name="syntax"></a>語法


```C++
HRESULT MakeInfinite();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

如需傳回值的描述，請參閱 [類別和介面-筆跡分析](classes-and-interfaces---ink-analysis.md)。
