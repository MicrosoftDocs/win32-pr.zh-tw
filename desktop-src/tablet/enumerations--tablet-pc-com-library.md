# <a name="core-tablet-pc-enumerations"></a>核心 Tablet PC 列舉

本章節包含 Tablet PC 的核心列舉。

## <a name="in-this-section"></a>本節內容



| 列舉型別                                                                                  | 描述                                                                                                                                                                                                                                                                               |
|----------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**InkApplicationGesture**](/windows/desktop/api/msinkaut/ne-msinkaut-inkapplicationgesture)                                       | 指定一組應用程式特定手勢的興趣。<br/>                                                                                                                                                                                                               |
| [**InkBoundingBoxMode 列舉**](/windows/desktop/api/msinkaut/ne-msinkaut-inkboundingboxmode)                                 | 指定筆劃的特性，例如繪製屬性，用來計算筆墨的周框方塊。<br/>                                                                                                                                                    |
| [**InkClipboardFormats**](/windows/desktop/api/msinkaut/ne-msinkaut-inkclipboardformats)                                           | 指定儲存在剪貼簿上的筆墨格式。<br/>                                                                                                                                                                                                                   |
| [**InkClipboardModes**](/windows/desktop/api/msinkaut/ne-msinkaut-inkclipboardmodes)                                               | 指定剪貼簿的複製選項。<br/>                                                                                                                                                                                                                                   |
| [**InkCollectionMode**](/windows/desktop/api/msinkaut/ne-msinkaut-inkcollectionmode)                                               | 決定是否要將筆墨、手勢或筆跡和手勢辨識為使用者寫入。<br/>                                                                                                                                                                                       |
| [**InkCollectorEventInterest**](/windows/desktop/api/msinkaut/ne-msinkaut-inkcollectoreventinterest)                               | 指定是否在筆墨收集器上發生事件，如果有的話，會引發哪個事件。<br/>                                                                                                                                                                                         |
| [**InkCursorButtonState**](/windows/desktop/api/msinkaut/ne-msinkaut-inkcursorbuttonstate)                                         | 指定資料指標按鈕的狀態。<br/>                                                                                                                                                                                                                                        |
| [**InkDivisionType**](/windows/win32/api/msinkaut15/ne-msinkaut15-inkdivisiontype)                                                   | 定義 [**IInkDivisionResult**](/windows/desktop/api/msinkaut15/nn-msinkaut15-iinkdivisionresult) 物件中結構化類型的值。<br/>                                                                                                                                                                    |
| [**InkExtractFlags**](/windows/desktop/api/msinkaut/ne-msinkaut-inkextractflags)                                                   | 指定要從 [**InkDisp**](inkdisp-class.md) 物件中移除之筆劃的部分。<br/>                                                                                                                                                                                     |
| [**InkMouseButton**](/windows/desktop/api/msinkaut/ne-msinkaut-inkmousebutton)                                                     | 指定按下的滑鼠按鍵。<br/>                                                                                                                                                                                                                                      |
| [**InkMousePointer**](/windows/desktop/api/msinkaut/ne-msinkaut-inkmousepointer)                                                   | 指定要顯示的滑鼠指標類型。<br/>                                                                                                                                                                                                                                 |
| [**InkOverlayAttachMode**](/windows/desktop/api/msinkaut/ne-msinkaut-inkoverlayattachmode)                                         | 指定要在使用中層的後方或前方附加新的 [**InkOverlay**](inkoverlay-class.md) 物件的位置。<br/>                                                                                                                                                       |
| [**InkOverlayEditingMode**](/windows/desktop/api/msinkaut/ne-msinkaut-inkoverlayeditingmode)                                       | 指定 [**InkOverlay**](inkoverlay-class.md) 物件和 [InkPicture](inkpicture-control-reference.md) 控制項的行為模式。<br/>                                                                                                                               |
| [**InkOverlayEraserMode**](/windows/desktop/api/msinkaut/ne-msinkaut-inkoverlayerasermode)                                         | 指定從 [**InkOverlay**](inkoverlay-class.md) 物件和 [InkPicture](inkpicture-control-reference.md) 控制項中清除筆墨的方式。<br/> 當 [**InkOverlayEditingMode**](/windows/desktop/api/msinkaut/ne-msinkaut-inkoverlayeditingmode) 設定為 [刪除] 時，就會使用此模式。<br/> |
| [**InkPenTip**](/windows/desktop/api/msinkaut/ne-msinkaut-inkpentip)                                                               | 指定畫筆提示是圓形或矩形。<br/>                                                                                                                                                                                                                         |
| [**InkPersistenceCompressionMode**](/windows/desktop/api/msinkaut/ne-msinkaut-inkpersistencecompressionmode)                       | 指定用來將 [**InkDisp**](inkdisp-class.md) 物件儲存至序列化格式的壓縮模式。<br/>                                                                                                                                                      |
| [**InkPersistenceFormat**](/windows/desktop/api/msinkaut/ne-msinkaut-inkpersistenceformat)                                         | 指定如何保存筆墨。<br/>                                                                                                                                                                                                                                                |
| [**InkRasterOperation**](/windows/desktop/api/msinkaut/ne-msinkaut-inkrasteroperation)                                             | 定義用來對繪製的筆墨執行點陣作業的值。<br/>                                                                                                                                                                                                                  |
| [**InkRecognitionAlternatesSelection 列舉**](/windows/desktop/api/msinkaut/ne-msinkaut-inkrecognitionalternatesselection)   | 未實作。<br/>                                                                                                                                                                                                                                                               |
| [**InkRecognitionConfidence**](/windows/desktop/api/msinkaut/ne-msinkaut-inkrecognitionconfidence)                                 | 指出辨識器在辨識結果中的信心層級。<br/>                                                                                                                                                                                           |
| [**InkRecognitionModes**](/windows/desktop/api/msinkaut/ne-msinkaut-inkrecognitionmodes)                                           | 指定 recognzier 如何解讀筆墨並決定結果字串。<br/>                                                                                                                                                                                              |
| [**InkRecognitionStatus**](/windows/desktop/api/msinkaut/ne-msinkaut-inkrecognitionstatus)                                         | 指定是否在辨識期間發生錯誤，以及是否發生錯誤。<br/>                                                                                                                                                                                       |
| [**InkRecognizerCapabilities**](/windows/desktop/api/msinkaut/ne-msinkaut-inkrecognizercapabilities)                               | 指定辨識器的屬性。 當您搜尋已安裝的辨識器時，也可以使用這個列舉來決定要使用哪些屬性。<br/>                                                                                                                      |
| [**InkRecognizerCharacterAutoCompletionMode**](/windows/desktop/api/msinkaut/ne-msinkaut-inkrecognizercharacterautocompletionmode) | 指定字元輸入模式的類型。<br/>                                                                                                                                                                                                                                      |
| [**InkShiftKeyModifierFlags**](/windows/desktop/api/msinkaut/ne-msinkaut-inkshiftkeymodifierflags)                                 | 指定按下的輔助按鍵。<br/>                                                                                                                                                                                                                                      |
| [**InkSystemGesture**](/windows/desktop/api/msinkaut/ne-msinkaut-inksystemgesture)                                                 | 指定一組作業系統特定手勢的興趣。<br/>                                                                                                                                                                                                         |
| [**ItemSelectionConstants**](/windows/win32/api/msinkaut/ne-msinkaut-inkselectionconstants)                                     | 指定是否使用第一個元素或某個點或封包值群組內的所有元素。<br/>                                                                                                                                                                        |
| [**PanelType**](/windows/win32/api/peninputpanel/ne-peninputpanel-paneltype)                                                               | 定義 [**PenInputPanel**](peninputpanel-class.md) 物件中目前可用的輸入類型。<br/>                                                                                                                                                                      |
| [**SelectionHitResult**](/windows/desktop/api/msinkaut/ne-msinkaut-selectionhitresult)                                             | 指定點擊測試期間遇到的選取部分（如果有的話）。<br/>                                                                                                                                                                                                        |
| [**TabletHardwareCapabilities**](/windows/desktop/api/msinkaut/ne-msinkaut-tablethardwarecapabilities)                             | 指定 Tablet PC 的硬體功能。<br/>                                                                                                                                                                                                                          |
| [**TabletPropertyMetricUnit**](/windows/desktop/api/msinkaut/ne-msinkaut-tabletpropertymetricunit)                                 | 指出屬性的度量單位。<br/>                                                                                                                                                                                                                               |
