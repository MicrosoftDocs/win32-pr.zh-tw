<span data-ttu-id="6b60a-104">InkEdit 控制項可讓您收集筆跡、辨識筆墨，以及將筆墨顯示為文字。</span><span class="sxs-lookup"><span data-stu-id="6b60a-104">The InkEdit control enables you to collect ink, recognize ink, and display ink as text.</span></span> <span data-ttu-id="6b60a-105">此控制項可讓您啟用智慧型表單，以改善文字輸入的精確度。</span><span class="sxs-lookup"><span data-stu-id="6b60a-105">This control allows you to enable smart forms, which improves the accuracy of text input.</span></span>

<span data-ttu-id="6b60a-109">在透明控制項背後建立 InkEdit 控制項 (例如，加上 WS \_ EX \_ 透明屬性集) 的分組會導致 InkEdit 無法收集筆跡。</span><span class="sxs-lookup"><span data-stu-id="6b60a-109">Creating the InkEdit control behind a transparent control (such as a GroupBox with the WS\_EX\_TRANSPARENT property set) will prevent InkEdit from collecting ink.</span></span>
