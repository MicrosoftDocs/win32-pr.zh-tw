<span data-ttu-id="1dbea-113">使用者字典是使用者已新增至其 Tablet PC 的單字清單。</span><span class="sxs-lookup"><span data-stu-id="1dbea-113">The user dictionary is the list of words that the user has added to his or her Tablet PC.</span></span> <span data-ttu-id="1dbea-114">如果未設定 [**RecognizerCoNtext**](inkrecognizercontext-class.md)的 [**單詞表**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkrecognizercontext-get_wordlist)屬性， **RecognizerCoNtext** 會將使用者字典以單字清單傳遞給辨識器。</span><span class="sxs-lookup"><span data-stu-id="1dbea-114">If the [**WordList**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkrecognizercontext-get_wordlist) property of the [**RecognizerContext**](inkrecognizercontext-class.md) is not set, the **RecognizerContext** passes the user dictionary as a word list to the recognizer.</span></span> <span data-ttu-id="1dbea-115">但是，如果已設定 **RecognizerCoNtext** 物件的 [**單詞表**] 屬性，則會將此字組傳遞給辨識器，而不是使用者字典。</span><span class="sxs-lookup"><span data-stu-id="1dbea-115">However, if the **WordList** property of the **RecognizerContext** object is set, the word list is passed to the recognizer instead of the user dictionary.</span></span>

> [!Note]  
> <span data-ttu-id="1dbea-116">在您設定 [[**單詞表**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkrecognizercontext-get_wordlist)] 屬性之前， [**RecognizerCoNtext**](inkrecognizercontext-class.md)物件的 [[**筆劃**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkrecognizercontext-get_strokes)] 屬性必須是空的。</span><span class="sxs-lookup"><span data-stu-id="1dbea-116">The [**Strokes**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkrecognizercontext-get_strokes) property of the [**RecognizerContext**](inkrecognizercontext-class.md) object must be empty before you set the [**WordList**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkrecognizercontext-get_wordlist) property.</span></span> <span data-ttu-id="1dbea-117">如果 **筆觸** 屬性不是空的，則會擲回例外狀況。</span><span class="sxs-lookup"><span data-stu-id="1dbea-117">If the **Strokes** property is not empty, an exception is thrown.</span></span> <span data-ttu-id="1dbea-118">在將單字指派給 **RecognizerCoNtext** 物件之後，不應該將它們新增至字組清單。</span><span class="sxs-lookup"><span data-stu-id="1dbea-118">Words should never be added to a word list after it has been assigned to a **RecognizerContext** object.</span></span>

 

<span data-ttu-id="1dbea-119">在 [**RecognizerCoNtext**](inkrecognizercontext-class.md) 物件上設定模擬程式也會影響辨識器使用應用程式字典的方式。</span><span class="sxs-lookup"><span data-stu-id="1dbea-119">Setting a factoid on the [**RecognizerContext**](inkrecognizercontext-class.md) object also affects how application dictionaries are used by the recognizer.</span></span> <span data-ttu-id="1dbea-120">影響字典行為的 factoids 如下：</span><span class="sxs-lookup"><span data-stu-id="1dbea-120">The factoids that affect the behavior of dictionaries are:</span></span>

-   <span data-ttu-id="1dbea-122">**SystemDictionary**</span><span class="sxs-lookup"><span data-stu-id="1dbea-122">**SystemDictionary**</span></span>
-   <span data-ttu-id="1dbea-123">**None**</span><span class="sxs-lookup"><span data-stu-id="1dbea-123">**None**</span></span>
