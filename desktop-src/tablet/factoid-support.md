# <a name="factoid-support"></a>模擬支援

*Factoids* 藉由提供辨識器的定義和預期的單字、數位和標點符號順序，來改善辨識精確度。

如需 factoids 的詳細資訊，請參閱 [使用內容改善精確度](using-context-to-improve-accuracy.md)。
