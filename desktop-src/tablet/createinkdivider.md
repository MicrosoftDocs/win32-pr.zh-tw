# <a name="createinkdivider-function"></a>CreateInkDivider 函式

具現化 [**InkDivider**](inkdivider-class.md) 介面的執行，並傳回其控制碼。

此函式不適合應用程式程式碼使用。

## <a name="syntax"></a>語法


```C++
INT_PTR CreateInkDivider(void);
```



## <a name="parameters"></a>參數

此函式沒有參數。

## <a name="return-value"></a>傳回值

傳回已建立之物件的控制碼。

## <a name="remarks"></a>備註

這個類別繼承自 [**InkDivider**](inkdivider-class.md) 基類，並會實作為虛擬函式。
