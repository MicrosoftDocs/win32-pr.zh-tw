# <a name="text-ink-reference"></a>文字筆墨參考

除了支援標準 OLE 介面， (tInk) 的 text 筆墨物件也支援 [**IInkLineInfo**](/windows/desktop/api/msinkaut/nn-msinkaut-iinklineinfo)。

如需指定文字筆墨物件之顯示內容的詳細資訊，請參閱 [**INKMETRIC**](/windows/win32/api/msinkaut/ns-msinkaut-inkmetric) 結構。
