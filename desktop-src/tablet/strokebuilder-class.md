# <a name="strokebuilder-class"></a>StrokeBuilder 類別

實行 [**IStrokeBuilder 介面**](/windows/desktop/api/RTSCom/nn-rtscom-istrokebuilder)。

## <a name="com-implementation"></a>COM 執行

這個類別會實 [**IStrokeBuilder 介面**](/windows/desktop/api/RTSCom/nn-rtscom-istrokebuilder)。

您可以在 c + + 中呼叫 [**CoCreateInstance**](/windows/win32/api/combaseapi/nf-combaseapi-cocreateinstance) 方法來具現化這個類別。
