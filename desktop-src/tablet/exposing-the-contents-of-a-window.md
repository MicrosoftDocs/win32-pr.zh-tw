# <a name="exposing-the-contents-of-a-window"></a>公開視窗的內容

應用程式必須公開使用 Microsoft Active Accessibility 之自訂視窗類別的內容。 整個視窗應該視為自訂控制項，也就是其他標準、自訂或自訂控制項的容器。

如需使用 Active Accessibility 公開自訂視窗內容的詳細資訊，請參閱 [協助工具](/windows/desktop/accessibility) 一節。
