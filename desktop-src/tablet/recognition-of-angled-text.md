# <a name="recognition-of-angled-text"></a>角度文字的辨識

Microsoft 辨識器會處理以 +/-20 度為單位的寫入，但精確度會隨著寫入偏離理想水準而下降。 針對以水準軸超過20度書寫的筆墨，建議開發人員在傳遞至辨識器之前，將筆墨旋轉回水準。
