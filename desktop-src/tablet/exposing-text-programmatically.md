# <a name="exposing-text-programmatically"></a>以程式設計方式公開文字

以程式設計方式公開文字，可提供更可靠的協助工具協助工具，並在某些情況下更豐富的資訊解讀。

有幾個基本的方法可以公開物件，以程式設計方式公開文字。 不過，所有的協助工具輔助程式都沒有任何方法可供使用，因此也必須在螢幕上繪製文字，以便辨識。 如需公開大量文字的詳細資訊，請參閱 [使用 Active Accessibility 公開文字](using-active-accessibility-to-expose-text.md)。
