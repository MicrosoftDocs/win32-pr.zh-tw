# <a name="erasing-ink-with-the-pen"></a>使用畫筆來清除筆墨

除了啟用手寫和繪圖功能之外，您還可以建立應用程式，以支援使用 tablet 畫筆來清除筆墨。 使用 [**InkOverlay**](inkoverlay-class.md) 物件來執行此動作。

下列各節說明如何使用 tablet 畫筆來清除筆墨。

-   [使用 InkOverlay 物件進行清除](erasing-by-using-the-inkoverlay-object.md)
-   [使用畫筆進行清除](erasing-by-using-the-pen.md)
