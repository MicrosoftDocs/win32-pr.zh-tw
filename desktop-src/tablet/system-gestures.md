# <a name="system-gestures"></a>系統手勢

系統手勢對應到傳統的滑鼠訊息。 Tablet Pc 支援滑鼠左鍵的系統輸入手勢、按兩下、按一下滑鼠右鍵、向左鍵拖曳、以滑鼠右鍵拖曳，以及將滑鼠停留在滑鼠上。 針對 Tablet PC 開發的應用程式必須以支援滑鼠動作的相同方式支援這些手勢。 若是傳統的應用程式，手勢所觸發的許多事件都會被視為傳統的滑鼠訊息。
