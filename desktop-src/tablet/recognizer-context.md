# <a name="recognizer-context"></a>辨識器內容

[**RecognizerCoNtext**](inkrecognizercontext-class.md)物件會定義單一辨識器內容的設定。 它會收到要辨識的筆墨、定義筆跡輸入和辨識輸出的條件約束，並且可以查詢辨識結果。

如需筆墨資料的詳細資訊，請參閱 [筆墨資料](ink-data.md)。 如需筆跡集合的詳細資訊，請參閱 [筆墨收集](ink-collection.md)。

您可以在 [**RecognizerCoNtext**](inkrecognizercontext-class.md) 物件的筆墨輸入和辨識輸出上設定的條件約束包括：

-   辨識或傳回的語言
-   使用的字典
-   Factoids

如需您可以在 [**RecognizerCoNtext**](inkrecognizercontext-class.md) 物件上設定之條件約束的詳細資訊，請參閱 [字典和 Factoids](dictionaries-and-factoids.md)。
