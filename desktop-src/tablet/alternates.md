# <a name="alternates"></a>替代項目

替代項是辨識區段的可能字組相符項。 辨識器會產生替代專案，並以字典或模擬的筆墨或音訊輸入可接受的相符專案作為基礎。

> [!Note]  
> 信賴評估目前僅適用于 Microsoft 英文 (美國) 和手勢辨識器。

 

有時筆墨在區段之間可能會有不明確的區別。 辨識器會比較這些區段與辨識器字典，以判斷可能的相符專案。 當辨識器比較區段時，會維持一份可能的替代相符專案清單，並將信賴等級指派給每一個專案，以挑選最重要的選擇。

> [!Note]  
> 辨識器無法針對小於辨識區段的筆墨提供替代部分。
