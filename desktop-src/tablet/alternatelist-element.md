# <a name="alternatelist-element"></a>AlternateList 元素

包含 [**替代**](alternate-element.md) 專案的集合，這些專案會保留辨識器針對父 [**InkWord**](inkword-element.md) 元素所傳回的替代專案。 替代項會依辨識器在替代中的可信度進行排序，最高優先。

## <a name="definition"></a>定義

``` syntax
<xs:element name="AlternateList" type="AlternatesListType" />
```

## <a name="parent-elements"></a>父項目

[**InkWord**](inkword-element.md)

## <a name="child-elements"></a>子元素

[**替代**](alternate-element.md)

## <a name="attributes"></a>屬性

None

|-----------------|---------------------------------------------------------------------------|
| **元素類型**| [**AlternatesListType**](alternateslisttype-complex-type.md) complexType |
| **Namespace**   | urn：架構-microsoft-com：平板電腦： richink                                |
| **結構描述名稱** | 日誌讀者                                                            |
