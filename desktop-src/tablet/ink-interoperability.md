# <a name="ink-interoperability"></a>筆墨互通性

持續性和互通性是兩個極為重要的功能，特別是針對使用筆墨的應用程式。 若要啟用筆墨的持續性和互通性，應用程式必須允許使用者：

-   以完整的筆墨資料精確度儲存並開啟其檔。
-   使用剪貼簿，將資料移入和移出其他以 Microsoft Windows 為基礎的應用程式。

下列各節詳細說明筆墨互通性。

-   [筆墨資料格式](ink-data-formats.md)
-   [重要互通性案例](important-interoperability-scenarios.md)
-   [持續性格式](persistence-formats.md)
-   [選擇要支援的格式](choosing-which-formats-to-support.md)
-   [積木](building-blocks.md)
-   [以 HTML 儲存筆墨](storing-ink-in-html.md)

本檔假設您對剪貼簿有詳細的瞭解。 如需剪貼簿的詳細資訊，請參閱 [剪貼簿檔](/windows/desktop/dataxchg/clipboard)。
