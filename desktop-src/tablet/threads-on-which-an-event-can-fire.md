-   [RecognizerCoNtext 物件事件](recognizercontext-object-events.md)
-   [筆觸集合事件](strokes-collection-events.md)
-   [InkEdit 控制項事件](inkedit-control-events.md)
-   [InkPicture 控制項事件](inkpicture-control-events.md)
-   [PenInputPanel 物件事件](peninputpanel-object-events.md)
