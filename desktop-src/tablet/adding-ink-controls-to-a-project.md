# <a name="adding-ink-controls-to-a-project"></a>將筆墨控制項新增至專案

將 [InkEdit](inkedit-control-reference.md) 或 [InkPicture](inkpicture-control-reference.md) 控制項新增至 Microsoft Visual Basic .Net 或 microsoft Visual C \# 專案：

1.  在 [工具箱] 中按一下滑鼠右鍵，然後按一下 [ **自訂工具箱**]。
2.  在 [ **自訂工具箱** ] 對話方塊中，按一下 [ **.NET Framework 元件** ] 索引標籤。
3.  在清單方塊中，選取 [InkEdit] 和 [InkPicture] 控制項，然後按一下 **[確定]**。 這些控制項會新增至 [ **元件** ] 索引標籤下的 [工具箱]。
4.  從 [工具箱] 中選取您想要的控制項，然後將控制項拖曳至表單上您想要顯示的位置。

> [!Note]  
> Tablet PC 受控程式庫可以在部分信任的環境中支援 Web 控制項，也就是從網頁執行受管理的元件。 但是，不支援網頁上啟用筆墨功能的 ActiveX 控制項。
