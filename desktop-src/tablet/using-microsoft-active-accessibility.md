# <a name="using-microsoft-active-accessibility"></a>使用 Microsoft Active Accessibility

Tablet PC 上的語音辨識引擎會使用 Active Accessibility 直接從物件取出資訊，並在物件變更時收到通知。 標準 Microsoft Windows 控制項支援這個介面，也可供自訂控制項和自訂視窗類別使用。
