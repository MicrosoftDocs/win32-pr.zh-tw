# <a name="dictionaries-and-factoids"></a>字典和 Factoids

字典是一種內部詞典，辨識器會用來判斷文字元合和替代專案。 模擬程式會藉由選取特定欄位的特定內容來增加辨識精確度。

如需字典的詳細資訊，請參閱 [使用應用程式字典](using-application-dictionaries.md)。

如需 factoids 的詳細資訊，請參閱 [使用內容改善精確度](using-context-to-improve-accuracy.md)。
