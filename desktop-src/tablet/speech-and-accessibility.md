# <a name="speech-and-accessibility"></a>語音和協助工具

針對可存取的應用程式，必須使用語音辨識公用程式來公開畫面內容的相關資訊。 這些公用程式需要識別並命名特定的控制項，讓它們可以辨識所說的控制項名稱，並操控該控制項來回應語音命令。

例如，使用語音輸入公用程式時，使用者可能會說「檔案」，然後說「版面設定」以選擇對應的命令功能表項目，然後說「橫向」來選取對應的核取方塊。 您必須公開這些元素的相關資訊，才能完成所需的目標。

-   [公開控制項和功能表](exposing-controls-and-menus.md)
-   [公開視窗](exposing-a-window.md)
-   [以程式設計方式公開文字](exposing-text-programmatically.md)
