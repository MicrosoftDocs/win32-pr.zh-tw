# <a name="flicks-enumerations"></a>筆觸列舉

本章節包含在 [**WM \_ TABLET \_ 筆鋒訊息**](wm-tablet-flick-message.md)中使用的列舉。



| 列舉型別                                                 | 描述                                                                                                                                                   |
|-------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**FLICKDIRECTION**](/windows/desktop/api/tabflicks/ne-tabflicks-flickdirection)                    | 代表筆勢的方向。<br/>                                                                                                             |
| [**FLICKMODE**](/windows/desktop/api/tabflicks/ne-tabflicks-flickmode)                              | 指定可以看到筆觸手勢的位置。<br/>                                                                                                        |
| [**FLICKACTION \_ COMMANDCODE**](/windows/desktop/api/tabflicks/ne-tabflicks-flickaction_commandcode) | 指定在筆觸 [**\_ 資料結構**](/windows/desktop/api/tabflicks/ns-tabflicks-flick_data) 中記載的命令代碼，並代表要執行的筆鋒動作類型。<br/> |
| [**SCROLLDIRECTION**](/windows/desktop/api/tabflicks/ne-tabflicks-scrolldirection)                  | 指定捲動方塊的移動方向。<br/>                                                                                               |
| [**KEYMODIFIER**](/windows/desktop/api/tabflicks/ne-tabflicks-keymodifier)                          | 指定當筆觸手勢發生時，所按下的輔助按鍵（如果有的話）。<br/>                                                               |
