# <a name="the-divider-object"></a>分隔物件

Tablet PC 技術包含筆跡輸入的版面配置分析，結合手寫和繪圖。 [**分隔**](inkdivider-class.md)物件提供版面配置分析功能，可將筆劃分類並群組為不同的結構化元素。

[筆墨分割範例](ink-divider-sample.md)主題示範如何使用具有 [**筆墨**](inkdisp-class.md)物件的 [**分界線**](inkdivider-class.md)物件來執行筆墨版面配置分析。
