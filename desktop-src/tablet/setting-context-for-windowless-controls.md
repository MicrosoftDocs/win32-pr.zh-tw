# <a name="setting-context-for-windowless-controls"></a>設定無視窗控制項的內容

每次在無視窗控制項的元素之間移動焦點時，無視窗控制項都可以呼叫 [**SetInputScope**](/windows/win32/api/inputscope/nf-inputscope-setinputscope) 函式;不過，基於效能考慮，不建議這麼做。
