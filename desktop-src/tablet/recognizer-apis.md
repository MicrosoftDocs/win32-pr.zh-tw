# <a name="recognizer-reference"></a>辨識器參考

辨識器會處理：

-   判斷電腦上可用的辨識器。
-   識別要使用的辨識器。
-   建立辨識器內容。
-   取出辨識器結果和替代項。

## <a name="in-this-section"></a>本節內容



| 主題                                                      | 描述                                                                                                                                                           |
|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [HRECOALT 控制碼](hrecoalt-handle.md)                     | 抓取替代字串和屬性值，例如信賴等級和度量。<br/>                                                                  |
| [HRECOCONTEXT 控制碼](hrecocontext-handle.md)             | 將筆墨新增至內容、執行筆墨辨識，以及抓取辨識結果和替代項。<br/>                                                    |
| [HRECOGNIZER 控制碼](hrecognizer-handle.md)               | 建立辨識器內容、抓取其屬性和屬性，並決定辨識器需要哪些封包屬性來執行辨識。<br/> |
| [HRECOWORDLIST 控制碼](hrecowordlist-handle.md)           | 管理您附加至辨識器內容的字組清單。 這份清單可改善辨識結果。<br/>                                                   |
| [辨識器列舉](recognizer-api-enumerations.md) | 描述辨識器列舉類型。<br/>                                                                                                                |
| [辨識器結構](recognizer-api-structures.md)     | 描述辨識器結構。<br/>                                                                                                                       |
| [辨識器 Guid](recognizer-guids.md)                   | 指定封包屬性和辨識器屬性的屬性類型。<br/>                                                                            |
