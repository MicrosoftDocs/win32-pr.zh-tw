# <a name="core-tablet-pc-constants"></a>核心 Tablet PC 常數

本章節包含 Tablet PC 的核心常數。

## <a name="in-this-section"></a>本節內容



| 常數                                                 | 描述                                                                                                                                                                                                                                               |
|----------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [標記](factoid-constants.md)                         | 定義常數位符串值，藉由提供內容相關資訊給辨識器，來增加辨識的精確度。<br/>                                                                                                           |
| [PacketPropertyGuids](packetpropertyguids-constants.md) | 定義指定封包屬性的值。  (API) 的 Tablet PC 應用程式開發介面，會使用全域唯一識別碼 (Guid) 來識別封包屬性（在自動化中是常數位符串）。<br/>                     |
| [RecognitionProperty](recognitionproperty-constants.md) | 定義值，這些值會指定辨識替代項的屬性。  (API) 的 Tablet PC 應用程式開發介面，會使用全域唯一識別碼 (Guid) 來識別封包屬性（在自動化中是常數位符串）。<br/> |
