# <a name="listening-to-system-events"></a>接聽系統事件

應用程式可以使用 [**InkCollector**](inkcollector-class.md) 物件來接聽系統事件，並在其上接聽 [**SystemGesture**](inkcollector-systemgesture.md) 事件。 您可以設定應用程式要接聽的事件。 當 tablet 畫筆動作發生時，對應的 **SystemGesture** 事件會傳送至應用程式的 **InkCollector** 物件。 當應用程式收到事件時，可以取消對應至指定 **SystemGesture** 事件的滑鼠訊息。 如需取消滑鼠郵件的詳細資訊，請參閱 **SystemGesture** 事件。
