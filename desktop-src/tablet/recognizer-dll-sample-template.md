# <a name="recognizer-dll-sample-template"></a>辨識器 DLL 範例範本

辨識器 DLL 範例包含用來建立辨識器 DLL 的範本。 範本包含用來註冊和取消註冊伺服器的功能，以及主要辨識器函式的骨架。

此範本說明如何在登錄中註冊一個 CLSID。 當您使用安裝程式安裝辨識器時，也可以這麼做。 請務必為您的語言建立自己的唯一 CLSID，而且不要重複使用範例中顯示的 CLSID。

根據預設，範例會安裝在您安裝 SDK 的目錄下的範例目錄中。

如需如何從 Tablet PC 平臺 API 使用辨識器的範例，請參閱 [Advanced 辨識範例](advanced-recognition-sample.md)。
