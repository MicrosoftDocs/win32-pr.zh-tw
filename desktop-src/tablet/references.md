# <a name="references-tablet-pc"></a> (Tablet PC) 的參考

如需在應用程式中啟用語音辨識的詳細資訊，請參閱 Microsoft speech sdk （可在 [Microsoft 語音 sdk](https://www.microsoft.com/speech/download/sdk51/)中找到）。 本網站也包含 Microsoft 語音 API 和應用程式設計指導方針的總覽。

如需 Microsoft 協助工具的相關資訊，請參閱 [Microsoft 協助工具](https://www.microsoft.com/enable/)。

如需 Windows 應用程式 Microsoft Active Accessibility 指導方針的詳細資訊，請參閱[協助工具](../accessibility/accessibility.md)一節。
