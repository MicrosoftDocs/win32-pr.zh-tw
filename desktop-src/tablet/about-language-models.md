# <a name="about-language-models"></a>關於語言模型

手寫辨識器會使用指定語言的一組已定義規則，做為辨識單字的主要內容。 這些規則稱為語言模型。 語言模型是由辨識器所執行，開發人員無法使用 Microsoft Windows XP Tablet PC Edition 軟體發展工具組 (SDK) 直接變更。 不過，有一些方法可以針對應用程式中的特定欄位來精簡內容。 這些技術包括：

-   輸入範圍
-   Factoids
-   字典
-   片語清單
