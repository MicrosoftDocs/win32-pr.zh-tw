# <a name="windows-presentation-foundation-tablet-pc"></a>Windows Presentation Foundation (Tablet PC) 

Windows Presentation Foundation (WPF) 的 Tablet PC 功能不需要任何特殊設定。

遵循設定 WPF 的指示，以使用 InkCanvas 和其他 WPF 元件。
