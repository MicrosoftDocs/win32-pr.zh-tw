## <a name="com-implementation"></a>COM 執行

本主題包含文字輸入面板 COM 介面和列舉的參考。

您可以在 c + + 中呼叫 [**CoCreateInstance**](/windows/win32/api/combaseapi/nf-combaseapi-cocreateinstance) 方法來具現化這些介面。

## <a name="remarks"></a>備註

文字輸入面板介面和列舉是用來控制輸入面板的外觀和行為。

這些元素是在 Peninputpanel 中宣告。
