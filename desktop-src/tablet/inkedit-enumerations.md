# <a name="inkedit-enumerations"></a>InkEdit 列舉

此區段包含屬於 InkEdit 控制項的列舉。



| 列舉型別                                            | 描述                                                                                                                                              |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AppearanceConstants**](/windows/desktop/api/inked/ne-inked-appearanceconstants)     | 指定控制項是否顯示為平面或立體。<br/>                                                                                            |
| [**BorderStyleConstants**](/windows/desktop/api/inked/ne-inked-borderstyleconstants)   | 指定控制項是否有框線。<br/>                                                                                                   |
| [**InkApplicationGesture**](/windows/desktop/api/msinkaut/ne-msinkaut-inkapplicationgesture) | 指定一組應用程式特定手勢的興趣。<br/>                                                                             |
| [**InkDisplayMode**](/windows/desktop/api/inked/ne-inked-inkdisplaymode)               | 指定選取範圍是否顯示為筆墨或文字。<br/>                                                                                         |
| [**InkEditStatus**](/windows/desktop/api/inked/ne-inked-inkeditstatus)                 | 指定 [InkEdit](inkedit-control-reference.md) 控制項為閒置、收集筆墨或辨識筆跡。<br/>                           |
| [**InkInsertMode**](/windows/desktop/api/inked/ne-inked-inkinsertmode)                 | 指定如何將筆墨插入至 [InkEdit](inkedit-control-reference.md) 控制項。<br/>                                                      |
| [**InkMode**](/windows/desktop/api/inked/ne-inked-inkmode)                             | 指定繪製筆墨的收集模式設定-是否已停用筆墨收集、是否已收集筆跡，或收集筆跡和手勢。<br/> |
| [**InkMouseButton**](/windows/desktop/api/msinkaut/ne-msinkaut-inkmousebutton)               | 指定按下的滑鼠按鍵。<br/>                                                                                                     |
| [**InkMousePointer**](/windows/desktop/api/msinkaut/ne-msinkaut-inkmousepointer)             | 指定出現的滑鼠指標類型。<br/>                                                                                             |
| [**MouseButton**](/windows/desktop/api/inked/ne-inked-mousebutton)                     | 指定按下的滑鼠按鍵。<br/>                                                                                                     |
| [**ScrollBarsConstants**](/windows/desktop/api/inked/ne-inked-scrollbarsconstants)     | 指定 [InkEdit](inkedit-control-reference.md) 控制項捲軸出現在螢幕上的方式。<br/>                                    |
| [**SelAlignmentConstants**](/windows/desktop/api/inked/ne-inked-selalignmentconstants) | 指定段落相對於 [InkEdit](inkedit-control-reference.md) 控制項邊界的對齊方式。<br/>                     |
