# <a name="instantiating-the-peninputpanel-class"></a>具現化 PenInputPanel 類別

\[[**PenInputPanel**](peninputpanel-class.md) 已被 [>textinput](/previous-versions/dotnet/netframework-3.5/ms581554(v=vs.90))取代。 請參閱 [文字輸入面板](programming-the-text-input-panel.md)的程式設計。\]

您可以具現化多個 [**PenInputPanel**](peninputpanel-class.md) 物件，並將其附加至表單上的每個控制項，或您可以具現化一個 **PenInputPanel** 物件，並在焦點事件處理常式的控制項之間移動。 因為一次只能有一個編輯控制項，所以第二個方法可以節省記憶體。 稱為 [PenInputPanel 範例](peninputpanel-sample.md) 的自動宣告範例版本已實作為兩種技術的範例。

下列各節將說明如何建立及顯示 [**PenInputPanel**](peninputpanel-class.md) 物件。

-   [建立 PenInputPanel 物件](creating-a-peninputpanel-object.md)
-   [使用 PenInputPanel 搭配下拉式方塊](using-the-peninputpanel-with-combo-boxes.md)
-   [顯示輸入面板](displaying-the-input-panel.md)
