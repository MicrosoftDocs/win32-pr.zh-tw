# <a name="text-input-panel-enumerations"></a>文字輸入面板列舉

此區段包含 [**ITextInputPanel 介面**](/windows/desktop/api/peninputpanel/nn-peninputpanel-itextinputpanel)使用的列舉。

## <a name="in-this-section"></a>本節內容



| 列舉型別                                                  | 描述                                                                                                                               |
|--------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| [**CorrectionMode 列舉**](/windows/win32/api/peninputpanel/ne-peninputpanel-correctionmode)         | 定義代表 Tablet PC 輸入面板之修正模式的值。<br/>                                               |
| [**CorrectionPosition 列舉**](/windows/win32/api/peninputpanel/ne-peninputpanel-correctionposition) | 定義代表輸入面板之 post 插入更正位置的值。<br/>                                      |
| [**EventMask**](/windows/win32/api/peninputpanel/ne-peninputpanel-eventmask)                               | 指定您可以設定注意的事件。                                                                                      |
| [**InPlaceDirection 列舉**](/windows/win32/api/peninputpanel/ne-peninputpanel-inplacedirection)     | 定義值，這些值代表可放置輸入面板的位置，相對於目前的文字插入點。<br/> |
| [**InPlaceState 列舉**](/windows/win32/api/peninputpanel/ne-peninputpanel-inplacestate)             | 定義表示輸入面板在螢幕上顯示方式的值。<br/>                                                       |
| [**InteractionMode 列舉**](/windows/win32/api/peninputpanel/ne-peninputpanel-interactionmode)       | 定義指示輸入面板位置的值。<br/>                                                                |
| [**PanelInputArea 列舉**](/windows/win32/api/peninputpanel/ne-peninputpanel-panelinputarea)         | 定義代表輸入面板輸入區域的值。<br/>                                                              |
