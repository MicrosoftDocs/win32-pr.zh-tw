# <a name="working-with-the-text-ink-object"></a><span data-ttu-id="722ee-103">使用文字筆墨物件</span><span class="sxs-lookup"><span data-stu-id="722ee-103">Working with the Text Ink Object</span></span>

<span data-ttu-id="722ee-104">為了協助支援應用程式中的筆跡，有兩個物件可以內嵌，而且任何 OLE 容器都支援這些物件，文字筆墨物件 (tInk) 和草圖筆墨物件 (接收) 。</span><span class="sxs-lookup"><span data-stu-id="722ee-104">To aid the support of ink in applications, there are two objects, both of which can be embedded and are supported by any OLE container, the text ink object (tInk) and the sketch ink object (sInk).</span></span>

<span data-ttu-id="722ee-105">文字筆墨物件是代表預期會形成單字之筆墨的 OLE 物件。</span><span class="sxs-lookup"><span data-stu-id="722ee-105">The text ink object is an OLE object representing ink that is expected to form words.</span></span> <span data-ttu-id="722ee-106">文字筆墨物件可從替代清單中選擇，以將手寫筆墨轉換成文字。</span><span class="sxs-lookup"><span data-stu-id="722ee-106">A text ink object enables the handwritten ink to be converted to text by choosing from a list of alternates.</span></span> <span data-ttu-id="722ee-107">您可以透過程式設計方式設定文字筆墨物件的色彩和大小，而且可以根據物件周圍文字的屬性來設定。</span><span class="sxs-lookup"><span data-stu-id="722ee-107">The color and size of the text ink object can be set programmatically and can be based on the attributes of the text around the object.</span></span> <span data-ttu-id="722ee-108">文字筆墨物件的目的是要包含單一單字。</span><span class="sxs-lookup"><span data-stu-id="722ee-108">The text ink object is intended to contain a single word.</span></span>

<span data-ttu-id="722ee-109">文字筆墨物件支援內嵌和剪貼簿支援所需的一組標準 OLE 介面。</span><span class="sxs-lookup"><span data-stu-id="722ee-109">The text ink object supports the standard set of OLE interfaces required for embedding and clipboard support.</span></span> <span data-ttu-id="722ee-110">[**IPersistStream**](/windows/win32/api/objidl/nn-objidl-ipersiststream)介面會以筆跡序列化格式讀取和寫入資料流程， (ISF) 。</span><span class="sxs-lookup"><span data-stu-id="722ee-110">The [**IPersistStream**](/windows/win32/api/objidl/nn-objidl-ipersiststream) interface reads from and writes to a stream in ink serialized format (ISF).</span></span> <span data-ttu-id="722ee-111">文字筆墨物件提供 [**IInkLineInfo**](/windows/desktop/api/msinkaut/nn-msinkaut-iinklineinfo) 介面，可存取其顯示內容和辨識結果清單。</span><span class="sxs-lookup"><span data-stu-id="722ee-111">The text ink object provides the [**IInkLineInfo**](/windows/desktop/api/msinkaut/nn-msinkaut-iinklineinfo) interface to access its display properties, and recognition result list.</span></span>

<span data-ttu-id="722ee-112">文字筆墨物件可用於應用程式之間的互通性，方法是將它放在剪貼簿上的 OLE 物件位置，或將它內嵌在 .RTF 資料流程中，或將它保存在 ISF 資料流程中。</span><span class="sxs-lookup"><span data-stu-id="722ee-112">The text ink object can be used for interoperability between applications either by placing it in the OLE object slot on the Clipboard, by embedding it in RTF, or by persisting it in an ISF stream.</span></span>

<span data-ttu-id="722ee-113">您可以透過下列方式產生文字筆墨物件。</span><span class="sxs-lookup"><span data-stu-id="722ee-113">A text ink object can be generated in the following ways.</span></span>

-   <span data-ttu-id="722ee-114">[InkEdit](inkedit-control-reference.md)控制項會使用文字筆墨物件。</span><span class="sxs-lookup"><span data-stu-id="722ee-114">The [InkEdit](inkedit-control-reference.md) control makes use of the text ink object.</span></span> <span data-ttu-id="722ee-115">InkEdit 控制項的功能是標準 RichEdit 控制項功能的超級集合。</span><span class="sxs-lookup"><span data-stu-id="722ee-115">The InkEdit control's functionality is a super-set of the standard RichEdit control functionality.</span></span> <span data-ttu-id="722ee-116">筆墨會以文字筆墨物件的形式插入 InkEdit 控制項的 RTF 資料流程中。</span><span class="sxs-lookup"><span data-stu-id="722ee-116">Ink is inserted into the InkEdit control's RTF stream as a text ink object.</span></span>
-   <span data-ttu-id="722ee-117">當應用程式將 [InkStrokes](/previous-versions/windows/desktop/legacy/ms703293(v=vs.85)) 或 [InkEdit](inkedit-control-reference.md) 物件複製到剪貼簿，並設定 [**INKCLIPBOARDFORMATS 列舉**](/windows/desktop/api/msinkaut/ne-msinkaut-inkclipboardformats) 格式時，ole 物件剪貼簿位置會包含文字筆墨 OLE 物件。</span><span class="sxs-lookup"><span data-stu-id="722ee-117">When an application copies an [InkStrokes](/previous-versions/windows/desktop/legacy/ms703293(v=vs.85)) or an [InkEdit](inkedit-control-reference.md) object onto the Clipboard and the [**InkClipboardFormats Enumeration**](/windows/desktop/api/msinkaut/ne-msinkaut-inkclipboardformats) format is set, the OLE object Clipboard slot contains a text ink OLE object.</span></span>
-   <span data-ttu-id="722ee-118">Tablet PC 輸入面板可以產生文字筆墨物件。</span><span class="sxs-lookup"><span data-stu-id="722ee-118">Tablet PC Input Panel can generate text ink objects.</span></span>

<span data-ttu-id="722ee-119">例如，您的應用程式可以辨識手寫，並將辨識結果新增至筆劃。</span><span class="sxs-lookup"><span data-stu-id="722ee-119">For example, your application can recognize handwriting and add the recognition result to the strokes.</span></span> <span data-ttu-id="722ee-120">然後，如果您將筆劃複製並貼入 Microsoft Word 作為文字筆墨物件，Word 2003 和更新版本中就會提供該單字的替代項。</span><span class="sxs-lookup"><span data-stu-id="722ee-120">Then, if you copy and paste the strokes to Microsoft Word as a text ink object, alternates for that word is available in Word 2003 and later versions.</span></span>

<span data-ttu-id="722ee-121">為了成功包含文字筆墨物件，應用程式必須針對内嵌物件執行 OLE 容器支援。</span><span class="sxs-lookup"><span data-stu-id="722ee-121">In order to successfully contain text ink objects, an application must implement OLE container support for embedded objects.</span></span> <span data-ttu-id="722ee-122">然後，若要讓容器完全支援文字筆墨，您必須建立：</span><span class="sxs-lookup"><span data-stu-id="722ee-122">Then, to make the container fully support text ink, you must institute:</span></span>

-   <span data-ttu-id="722ee-123">修改應用程式以進行尋找和取代。</span><span class="sxs-lookup"><span data-stu-id="722ee-123">Modifications to the application for Find and Replace.</span></span> <span data-ttu-id="722ee-124">這些物件的類型必須是查核，而不是略過搜尋中的內嵌物件。</span><span class="sxs-lookup"><span data-stu-id="722ee-124">Instead of skipping embedded objects in the search, these objects must be interrogated for type.</span></span> <span data-ttu-id="722ee-125">如果它們是文字筆墨物件，則必須將它們具現化並針對其對應的文字進行查詢。</span><span class="sxs-lookup"><span data-stu-id="722ee-125">If they are a text ink object, they must be instantiated and queried for their corresponding text.</span></span>
-   <span data-ttu-id="722ee-126">修改選取行為。</span><span class="sxs-lookup"><span data-stu-id="722ee-126">Modifications to selection behavior.</span></span> <span data-ttu-id="722ee-127">文字筆墨物件的選取範圍絕對不能與調整大小控點一起出現。</span><span class="sxs-lookup"><span data-stu-id="722ee-127">Selection of text ink objects should never appear with sizing handles.</span></span> <span data-ttu-id="722ee-128">它們的選取方式應該與檔中選取文字的方式相同。</span><span class="sxs-lookup"><span data-stu-id="722ee-128">They should be selected in the same way that text is selected in the document.</span></span> <span data-ttu-id="722ee-129">物件的選取程式碼必須偵測型別是否為文字筆墨，並適當地顯示選取範圍。</span><span class="sxs-lookup"><span data-stu-id="722ee-129">Selection code for objects must detect if the type is text ink and display the selection appropriately.</span></span>
-   <span data-ttu-id="722ee-130">環境屬性的使用。</span><span class="sxs-lookup"><span data-stu-id="722ee-130">Use of ambient properties.</span></span> <span data-ttu-id="722ee-131">字型大小、色彩和粗體格式等環境屬性必須傳送至文字筆墨物件。</span><span class="sxs-lookup"><span data-stu-id="722ee-131">Ambient properties such as font size, color, and bold formatting need to be transmitted to the text ink object.</span></span> <span data-ttu-id="722ee-132">這些屬性的應用會變更手寫筆跡的寬度，因此需要透過呼叫 [**IInkLineInfo：： GetInkExtent**](/windows/desktop/api/msinkaut/nf-msinkaut-iinklineinfo-getinkextent) 或 [**IOleObject：： GetExtent**](/windows/win32/api/oleidl/nf-oleidl-ioleobject-getextent) 方法來更新大小。</span><span class="sxs-lookup"><span data-stu-id="722ee-132">Application of these properties changes the width of the handwritten ink, so a size update is required by calling the [**IInkLineInfo::GetInkExtent**](/windows/desktop/api/msinkaut/nf-msinkaut-iinklineinfo-getinkextent) or [**IOleObject::GetExtent**](/windows/win32/api/oleidl/nf-oleidl-ioleobject-getextent) method.</span></span>
