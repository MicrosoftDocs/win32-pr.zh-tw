# <a name="cng-algorithm-pseudo-handles"></a>CNG 演算法虛擬控制碼

從 Windows 10 開始，CNG 為許多演算法提供預先定義的演算法控制碼。 您可以在任何需要演算法控制碼的情況下使用這些控制碼。

## <a name="remarks"></a>備註

從 Windows 10 開始，CNG 為許多演算法提供預先定義的演算法控制碼。 您可以在任何需要演算法控制碼的情況下使用這些控制碼。 不過， [**BCryptSetProperty**](/windows/desktop/api/Bcrypt/nf-bcrypt-bcryptsetproperty) 的任何呼叫都會失敗，因為演算法控制碼是共用的，且無法修改。 此外，在核心模式中，這些控制碼不能以 IRQL = 分派的方式使用。
