# <a name="about-cng"></a>關於 CNG

密碼編譯 API：新一代 (CNG) 是 [*CryptoAPI*](/windows/desktop/SecGloss/c-gly)的長期取代。 CNG 的設計可在許多層級進行擴充，並在行為上進行中立性驗證。

## <a name="features"></a>功能

如需 CNG 功能的相關資訊，請參閱 [Cng 功能](cng-features.md)。

## <a name="cryptographic-primitives"></a>密碼編譯基本類型

如需 CNG 中密碼編譯基本類型的詳細資訊，請參閱 [密碼編譯基本](cryptographic-primitives.md)類型。

## <a name="key-storage-and-retrieval"></a>金鑰儲存和抓取

如需有關 CNG 中金鑰儲存和抓取的詳細資訊，請參閱 [金鑰儲存和](key-storage-and-retrieval.md)抓取。

## <a name="key-import-and-export"></a>金鑰匯入和匯出

如需匯入和匯出金鑰的詳細資訊，請參閱 [金鑰匯入和匯出](key-import-and-export.md)。

## <a name="data-protection-api-next-generation"></a>資料保護 API：新一代

CNG 提供一組 Api，可用來輕鬆地加密和解密多部電腦上的授權主體內容。 如需詳細資訊，請參閱 [CNG DPAPI](cng-dpapi.md)。
