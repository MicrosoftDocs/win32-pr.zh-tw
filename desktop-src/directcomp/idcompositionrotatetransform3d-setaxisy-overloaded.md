# <a name="idcompositionrotatetransform3dsetaxisy-methods"></a>IDCompositionRotateTransform3D：： SetAxisY 方法

變更或繪製旋轉轉換之 AxisY 屬性值的動畫。 AxisY 屬性會指定旋轉軸向量的 y 座標。 預設值為零。

### <a name="overload-list"></a>多載清單



| 方法                                                                                                       | 描述                                          |
|:-------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------|
| [**SetAxisY (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisy(float))                                     | 變更 AxisY 屬性的值。<br/>  |
| [**SetAxisY (IDCompositionAnimation \*)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisy(idcompositionanimation)) | 繪製 AxisY 屬性值的動畫。<br/> |
