# <a name="idcompositionskewtransformsetcentery-methods"></a>IDCompositionSkewTransform：： SetCenterY 方法

變更或繪製2D 扭曲轉換之 CenterY 屬性值的動畫。 CenterY 屬性會指定執行扭曲的點的 y 座標。

### <a name="overload-list"></a>多載清單



| 方法                                                                                                       | 描述                                            |
|:-------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetCenterY (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setcentery(float))                                     | 變更 CenterY 屬性的值。<br/>  |
| [**SetCenterY (IDCompositionAnimation \*)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setcentery(idcompositionanimation)) | 繪製 CenterY 屬性值的動畫。<br/> |
