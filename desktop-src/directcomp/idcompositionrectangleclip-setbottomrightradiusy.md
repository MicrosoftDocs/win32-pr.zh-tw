# <a name="idcompositionrectangleclipsetbottomrightradiusy-methods"></a>IDCompositionRectangleClip：： SetBottomRightRadiusY 方法

變更或繪製此剪輯之 BottomRightRadiusY 屬性值的動畫。 BottomRightRadiusY 屬性會指定將剪輯右下角四捨五入之橢圓形的 y 半徑。
