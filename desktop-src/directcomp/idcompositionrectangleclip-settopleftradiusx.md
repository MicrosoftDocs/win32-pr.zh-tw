# <a name="idcompositionrectangleclipsettopleftradiusx-methods"></a>IDCompositionRectangleClip：： SetTopLeftRadiusX 方法

變更或繪製此剪輯之 TopLeftRadiusX 屬性值的動畫。 TopLeftRadiusX 屬性會指定四捨五入剪輯左上角之橢圓形的 x 半徑。

| 方法                                                                                                                     | 描述                                                                |
|:---------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------|
| [**SetTopLeftRadiusX (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settopleftradiusx(idcompositionanimation))                                     | 變更此剪輯之 TopLeftRadiusX 屬性的值。<br/>  |
