# <a name="idcompositiontranslatetransformsetoffsetx-methods"></a>IDCompositionTranslateTransform：： SetOffsetX 方法

變更或繪製2D 轉譯轉換之 OffsetX 屬性值的動畫。 OffsetX 屬性會指定沿著 X 軸的平移。

### <a name="overload-list"></a>多載清單



| 方法                                                                                                            | 描述                                            |
|:------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetOffsetX (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform-setoffsetx(float))                                     | 變更 OffsetX 屬性的值。<br/>  |
| [**SetOffsetX (IDCompositionAnimation \*)**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform-setoffsetx(idcompositionanimation)) | 繪製 OffsetX 屬性值的動畫。<br/> |
