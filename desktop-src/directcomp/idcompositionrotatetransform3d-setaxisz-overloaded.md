# <a name="idcompositionrotatetransform3dsetaxisz-methods"></a>IDCompositionRotateTransform3D：： SetAxisZ 方法

變更或繪製3D 旋轉轉換之 AxisZ 屬性值的動畫。 AxisZ 屬性會指定旋轉軸向量的 z 座標。 預設值為 1.0。

### <a name="overload-list"></a>多載清單



| 方法                                                                                                       | 描述                                          |
|:-------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------|
| [**SetAxisZ (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisz(float))                                     | 變更 AxisZ 屬性的值。<br/>  |
| [**SetAxisZ (IDCompositionAnimation \*)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisz(idcompositionanimation)) | 繪製 AxisZ 屬性值的動畫。<br/> |
