# <a name="idcompositionrectangleclipsetbottomleftradiusy-methods"></a>IDCompositionRectangleClip：： SetBottomLeftRadiusY 方法

變更或繪製此剪輯之 BottomLeftRadiusY 屬性值的動畫。 BottomLeftRadiusY 屬性會指定將剪輯左下角四捨五入之橢圓形的 y 半徑。

| 方法                                                                                                                           | 描述                                                                   |
|:---------------------------------------------------------------------------------------------------------------------------------|:------------------------------------------------------------------------------|
| [**SetBottomLeftRadiusY (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setbottomleftradiusy(idcompositionanimation))                                     | 變更此剪輯之 BottomLeftRadiusY 屬性的值。<br/>  |
| [**SetBottomLeftRadiusY (IDCompositionAnimation \*)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setbottomleftradiusy(idcompositionanimation)) | 繪製此剪輯之 BottomLeftRadiusY 屬性值的動畫。<br/> |
