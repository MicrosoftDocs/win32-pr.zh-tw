# <a name="idcompositionrectangleclipsettop-methods"></a>IDCompositionRectangleClip：： SetTop 方法

動畫或變更裁剪矩形頂端屬性的值。 Top 屬性指定裁剪矩形左上角的 y 座標。

### <a name="overload-list"></a>多載清單



| 方法                                                                                               | 描述                                        |
|:-----------------------------------------------------------------------------------------------------|:---------------------------------------------------|
| [**SetTop (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settop(float))                                     | 變更 Top 屬性的值。<br/>  |
| [**SetTop (IDCompositionAnimation \*)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settop(idcompositionanimation)) | 繪製 Top 屬性值的動畫。<br/> |
