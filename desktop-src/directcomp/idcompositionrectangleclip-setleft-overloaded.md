# <a name="idcompositionrectangleclipsetleft-methods"></a>IDCompositionRectangleClip：： SetLeft 方法

動畫或變更剪輯矩形左邊屬性的值。 左邊的屬性會指定裁剪矩形左上角的 x 座標。

### <a name="overload-list"></a>多載清單



| 方法                                                                                                 | 描述                                         |
|:-------------------------------------------------------------------------------------------------------|:----------------------------------------------------|
| [**SetLeft (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setleft(float))                                     | 變更 Left 屬性的值。<br/>  |
| [**SetLeft (IDCompositionAnimation \*)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setleft(idcompositionanimation)) | 繪製左方屬性值的動畫。<br/> |
