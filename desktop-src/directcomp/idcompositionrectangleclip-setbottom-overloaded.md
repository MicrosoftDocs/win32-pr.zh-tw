# <a name="idcompositionrectangleclipsetbottom-methods"></a>IDCompositionRectangleClip：： SetBottom 方法

動畫或變更剪輯矩形底部屬性的值。 底部屬性指定裁剪矩形右下角的 y 座標。

### <a name="overload-list"></a>多載清單



| 方法                                                                                                     | 描述                                           |
|:-----------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**SetBottom (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setbottom(float))                                     | 變更下一個屬性的值。<br/>  |
| [**SetBottom (IDCompositionAnimation \*)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setbottom(idcompositionanimation)) | 將底端屬性的值繪製成動畫。<br/> |
