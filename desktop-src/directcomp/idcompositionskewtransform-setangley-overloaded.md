# <a name="idcompositionskewtransformsetangley-methods"></a>IDCompositionSkewTransform：： SetAngleY 方法

變更或繪製2D 扭曲轉換之 System.windows.media.skewtransform.angley 屬性值的動畫。 System.windows.media.skewtransform.angley 屬性會指定旋轉角度（以度為單位）。 預設值為零。

### <a name="overload-list"></a>多載清單



| 方法                                                                                                     | 描述                                           |
|:-----------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**SetAngleY (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setangley(float))                                     | 變更 System.windows.media.skewtransform.angley 屬性的值。<br/>  |
| [**SetAngleY (IDCompositionAnimation \*)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setangley(idcompositionanimation)) | 繪製 System.windows.media.skewtransform.angley 屬性值的動畫。<br/> |
