# <a name="idcompositionscaletransformsetscalex-methods"></a>IDCompositionScaleTransform：： SetScaleX 方法

變更或繪製2D 縮放轉換之 ScaleX 屬性值的動畫。 ScaleX 屬性會指定沿著 X 軸的縮放比例。

### <a name="overload-list"></a>多載清單



| 方法                                                                                                      | 描述                                           |
|:------------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**SetScaleX (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform-setscalex(float))                                     | 變更 ScaleX 屬性的值。<br/>  |
| [**SetScaleX (IDCompositionAnimation \*)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform-setscalex(idcompositionanimation)) | 將 ScaleX 屬性的值繪製成動畫。<br/> |
