# <a name="idcompositiontranslatetransform3dsetoffsety-methods"></a>IDCompositionTranslateTransform3D：： SetOffsetY 方法

變更或繪製3D 轉譯轉換效果之 OffsetY 屬性值的動畫。 OffsetY 屬性會指定沿著 y 軸的平移。

### <a name="overload-list"></a>多載清單



| 方法                                                                                                              | 描述                                            |
|:--------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetOffsetY (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform3d-setoffsety(float))                                     | 變更 OffsetY 屬性的值。<br/>  |
| [**SetOffsetY (IDCompositionAnimation \*)**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform3d-setoffsety(idcompositionanimation)) | 繪製 OffsetY 屬性值的動畫。<br/> |
