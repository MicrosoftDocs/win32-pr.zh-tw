# <a name="directcomposition-functions"></a>DirectComposition 函式

本節說明 Microsoft DirectComposition API 所提供的功能。

## <a name="in-this-section"></a>本節內容



| 主題                                                                                       | 描述                                                                                                                                          |
|---------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DCompositionAttachMouseDragToHwnd**](/windows/desktop/api/dcomp/nf-dcomp-dcompositionattachmousedragtohwnd)<br/>   | 建立互動/InputSink，以將滑鼠按鍵向下移動，並將任何後續的 move 和 up 事件路由至指定的 HWND。<br/>                      |
| [**DCompositionAttachMouseWheelToHwnd**](/windows/desktop/api/dcomp/nf-dcomp-dcompositionattachmousewheeltohwnd)<br/> | 建立互動/InputSink，以將滑鼠滾輪訊息路由至指定的 HWND。 <br/>                                                        |
| [**DCompositionCreateDevice**](/windows/desktop/api/Dcomp/nf-dcomp-dcompositioncreatedevice)<br/>                     | 建立新的裝置物件，可用來建立其他 DirectComposition 物件。<br/>                                                   |
| [**DCompositionCreateDevice2**](/windows/desktop/api/Dcomp/nf-dcomp-dcompositioncreatedevice2)<br/>                   | 建立新的裝置物件，可用來建立其他 DirectComposition 物件。<br/>                                                   |
| [**DCompositionCreateDevice3**](/windows/desktop/api/dcomp/nf-dcomp-dcompositioncreatedevice3)<br/>                   | 建立新的 DirectComposition 裝置物件，可用於建立其他 DirectComposition 物件。<br/>                               |
| [**DCompositionCreateSurfaceHandle**](/windows/desktop/api/Dcomp/nf-dcomp-dcompositioncreatesurfacehandle)<br/>       | 建立新的組合介面物件，該物件可以系結至 Microsoft DirectX 交換鏈或交換緩衝區，並與視覺效果建立關聯。<br/> |
| [**DCompositionGetFrameStatistics**](/previous-versions/windows/desktop/legacy/mt589902(v=vs.85))<br/>         | 抓取撰寫統計資訊。<br/>                                                                                             |
