# <a name="idcompositionrotatetransform3dsetangle-methods"></a>IDCompositionRotateTransform3D：： SetAngle 方法

變更或繪製3D 旋轉轉換之 Angle 屬性值的動畫。 Angle 屬性指定旋轉角度（以度為單位）。 預設值為零。

### <a name="overload-list"></a>多載清單



| 方法                                                                                                       | 描述                                          |
|:-------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------|
| [**SetAngle (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setangle(float))                                     | 變更 Angle 屬性的值。<br/>  |
| [**SetAngle (IDCompositionAnimation \*)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setangle(idcompositionanimation)) | 繪製 Angle 屬性值的動畫。<br/> |
