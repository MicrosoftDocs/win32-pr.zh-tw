# <a name="idcompositionrotatetransform3dsetcentery-methods"></a>IDCompositionRotateTransform3D：： SetCenterY 方法

變更或繪製3D 旋轉轉換之 CenterY 屬性值的動畫。 CenterY 屬性會指定執行旋轉的點的 y 座標。 預設值為零。

### <a name="overload-list"></a>多載清單



| 方法                                                                                                           | 描述                                            |
|:-----------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetCenterY (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setcentery(float))                                     | 變更 CenterY 屬性的值。<br/>  |
| [**SetCenterX (IDCompositionAnimation \*)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setcentery(idcompositionanimation)) | 繪製 CenterY 屬性值的動畫。<br/> |
