# <a name="directcomposition-interfaces"></a>DirectComposition 介面

本節說明 Microsoft DirectComposition API 所提供的介面。

## <a name="in-this-section"></a>本節內容



| 主題                                                                                               | 描述                                                                                                                                                                                                                                                                                                                                                                                               |
|-----------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IDCompositionAffineTransform2DEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionaffinetransform2deffect)<br/>     | 算術複合效果是用來結合2個影像，並使用輸入影像的加權加總圖元。 <br/>                                                                                                                                                                                                                                                                             |
| [**IDCompositionAnimation**](/windows/desktop/api/DcompAnimation/nn-dcompanimation-idcompositionanimation)<br/>                                 | 表示用來建立一或多個 DirectComposition 物件的一或多個屬性動畫的函式。<br/>                                                                                                                                                                                                                                                                                           |
| [**IDCompositionArithmeticCompositeEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionarithmeticcompositeeffect)<br/> | 算術複合效果是用來結合2個影像，並使用輸入影像的加權加總圖元。<br/>                                                                                                                                                                                                                                                                              |
| [**IDCompositionBlendEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionblendeffect)<br/>                             | Blend 效果用來合併2個影像。 <br/>                                                                                                                                                                                                                                                                                                                                                 |
| [**IDCompositionBrightnessEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionbrightnesseffect)<br/>                   | 亮度效果控制影像的亮度。<br/>                                                                                                                                                                                                                                                                                                                                    |
| [**IDCompositionColorMatrixEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositioncolormatrixeffect)<br/>                 | 色彩矩陣效果會改變點陣圖的 RGBA 值。<br/>                                                                                                                                                                                                                                                                                                                                    |
| [**IDCompositionCompositeEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositioncompositeeffect)<br/>                     | 複合效果用來合併2個或多個影像。 此效果有13個不同的複合模式。 複合效果接受2個以上的輸入。 當您指定2個影像時，目的地是第一個輸入 (索引 0) ，而來源是第二個輸入 (索引 1) 。 如果您指定2個以上的輸入，則影像會從第一個輸入開始，而第二個則是從第二個輸入開始，依此類推。 <br/> |
| [**IDCompositionClip**](/windows/win32/api/dcomp/nn-dcomp-idcompositionclip)<br/>                                           | 表示用來限制視覺子樹轉譯為矩形區域的剪輯物件。 <br/>                                                                                                                                                                                                                                                                                    |
| [**IDCompositionDesktopDevice**](/windows/win32/api/dcomp/nn-dcomp-idcompositiondesktopdevice)<br/>                         | 應用程式必須使用 IDCompositionDesktopDevice 介面，才能在 Win32 桌面應用程式中使用 DirectComposition。 此介面可讓應用程式將視覺化樹狀結構連接至視窗，以及裝載多層式子視窗以進行組合<br/>                                                                                                                                |
| [**IDCompositionDevice**](/windows/win32/api/dcomp/nn-dcomp-idcompositiondevice)<br/>                                       | 作為所有其他 DirectComposition 物件的 factory，並提供控制交易式組合的方法。<br/>                                                                                                                                                                                                                                                                         |
| [**IDCompositionDevice2**](/windows/win32/api/dcomp/nn-dcomp-idcompositiondevice2)<br/>                                     | 作為所有其他 DirectComposition 物件的 factory，並提供控制交易式組合的方法。<br/>                                                                                                                                                                                                                                                                         |
| [**IDCompositionDevice3**](/windows/win32/api/dcomp/nn-dcomp-idcompositiondevice3)<br/>                                     | 作為所有其他 DirectComposition 物件的 factory，並提供控制交易式組合的方法。 <br/>                                                                                                                                                                                                                                                                        |
| [**IDCompositionDeviceDebug**](/windows/win32/api/dcomp/nn-dcomp-idcompositiondevicedebug)<br/>                             | 提供對轉譯功能的存取，以協助進行應用程式的偵錯工具和效能微調。 您可以從 DirectComposition 裝置介面查詢這個介面。<br/>                                                                                                                                                                                                                  |
| [**IDCompositionEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositioneffect)<br/>                                       | 表示修改視覺效果子樹之點陣化的點陣圖效果。 <br/>                                                                                                                                                                                                                                                                                                             |
| [**IDCompositionEffectGroup**](/windows/win32/api/dcomp/nn-dcomp-idcompositioneffectgroup)<br/>                             | 表示一組點陣圖效果，會一起套用以修改視覺效果子樹的點陣化。 <br/>                                                                                                                                                                                                                                                                             |
| [**IDCompositionFilterEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionfiltereffect)<br/>                           | 表示篩選效果。<br/>                                                                                                                                                                                                                                                                                                                                                                    |
| [**IDCompositionFloodEffect**](/previous-versions/windows/desktop/legacy/dn919732(v=vs.85))<br/>                             | 大量效果是用來根據指定的色彩和 Alpha 值產生點陣圖。 當您想要特定色彩作為效果的輸入（例如背景色彩）時，可以使用此效果。 <br/>                                                                                                                                                                                        |
| [**IDCompositionGaussianBlurEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositiongaussianblureffect)<br/>               |                                                                                                                                                                                                                                                                                                                                                                                                           |
| [**IDCompositionHueRotationEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionhuerotationeffect)<br/>                 | 色調旋轉效果會根據旋轉角度套用色彩矩陣，來改變影像的色調。 <br/>                                                                                                                                                                                                                                                                                      |
| [**IDCompositionLinearTransferEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionlineartransfereffect)<br/>           | 線性傳輸效果是用來使用從您為每個通道提供的值清單建立的線性函式，來對應影像的色彩濃度。 <br/>                                                                                                                                                                                                                               |
| [**IDCompositionMatrixTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositionmatrixtransform)<br/>                     | 表示由3到2個矩陣定義的任意仿射2D 轉換。<br/>                                                                                                                                                                                                                                                                                                                   |
| [**IDCompositionMatrixTransform3D**](/windows/win32/api/dcomp/nn-dcomp-idcompositionmatrixtransform3d)<br/>                 | 表示由四個矩陣定義的任意3D 轉換。<br/>                                                                                                                                                                                                                                                                                                                          |
| [**IDCompositionRectangleClip**](/windows/win32/api/dcomp/nn-dcomp-idcompositionrectangleclip)<br/>                         | 表示將視覺子樹轉譯為指定之矩形區域的剪輯物件。 （選擇性）剪切物件可能會指定圓角。<br/>                                                                                                                                                                                                                 |
| [**IDCompositionRotateTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositionrotatetransform)<br/>                     | 表示會影響 Z 軸周圍視覺效果旋轉的2D 轉換。 座標系統會圍繞指定的中心點旋轉。 <br/>                                                                                                                                                                                                                                   |
| [**IDCompositionSaturationEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionsaturationeffect)<br/>                   | 此效果可用來改變影像的飽和度。 飽和度效果是色彩矩陣效果的特製化。 <br/>                                                                                                                                                                                                                                                                |
| [**IDCompositionRotateTransform3D**](/windows/win32/api/dcomp/nn-dcomp-idcompositionrotatetransform3d)<br/>                 | 表示3D 轉換，它會影響視覺效果沿著3D 空間中任意軸的旋轉。 座標系統會圍繞指定的中心點旋轉。 <br/>                                                                                                                                                                                                                 |
| [**IDCompositionScaleTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositionscaletransform)<br/>                       | 表示沿著 X 軸和 y 軸影響視覺效果尺規的2D 轉換。 座標系統會從指定的中心點縮放。 <br/>                                                                                                                                                                                                                               |
| [**IDCompositionScaleTransform3D**](/windows/win32/api/dcomp/nn-dcomp-idcompositionscaletransform3d)<br/>                   | 表示3D 轉換效果，其沿著 X 軸、y 軸和 Z 軸影響視覺效果的尺規。 座標系統會從指定的中心點縮放。 <br/>                                                                                                                                                                                                               |
| [**IDCompositionSkewTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositionskewtransform)<br/>                         | 表示會影響沿著 X 軸和 y 軸之視覺效果扭曲的2D 轉換。 座標系統會在指定的中心點周圍扭曲。 <br/>                                                                                                                                                                                                                              |
| [**IDCompositionShadowEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionshadoweffect)<br/>                           | 陰影效果是用來從影像的 Alpha 通道產生陰影。 針對較高的 Alpha 值，陰影較不透明，較低的 Alpha 值則更透明。 您可以設定模糊的數量和陰影的色彩。 <br/>                                                                                                                                                 |
| [**IDCompositionSurface**](/windows/win32/api/dcomp/nn-dcomp-idcompositionsurface)<br/>                                     | 表示可以與視覺效果建立關聯的實體點陣圖，以在視覺化樹狀結構中進行組合。 這個介面也可以用來更新點陣圖內容。 <br/>                                                                                                                                                                                                                            |
| [**IDCompositionTableTransferEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontabletransfereffect)<br/>             | 表格傳輸效果是用來對應影像的色彩濃度，其使用的傳輸函式是由您所提供的值清單來建立。<br/>                                                                                                                                                                                                                                  |
| [**IDCompositionSurfaceFactory**](/windows/win32/api/dcomp/nn-dcomp-idcompositionsurfacefactory)<br/>                       | 建立與應用程式提供之轉譯裝置相關聯的介面和虛擬介面物件。<br/>                                                                                                                                                                                                                                                                                          |
| [**IDCompositionTarget**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontarget)<br/>                                       | 代表 DirectComposition 視覺化樹狀結構與應該在其上撰寫視覺化樹狀結構之目的地之間的系結。 <br/>                                                                                                                                                                                                                                                            |
| [**IDCompositionTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontransform)<br/>                                 | 表示可以用來修改視覺化子樹之座標空間的2D 轉換。 <br/>                                                                                                                                                                                                                                                                                           |
| [**IDCompositionTransform3D**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontransform3d)<br/>                             | 表示3D 轉換效果，可用於修改視覺化子樹的點陣化。 <br/>                                                                                                                                                                                                                                                                                       |
| [**IDCompositionTranslateTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontranslatetransform)<br/>               | 表示2D 轉換，它只會影響沿著 X 軸和 y 軸的視覺效果位移。<br/>                                                                                                                                                                                                                                                                                           |
| [**IDCompositionTurbulenceEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionturbulenceeffect)<br/>                   | Turbulence 效果是用來根據 Perlin 雜訊函式來產生點陣圖。 Turbulence 效果沒有輸入影像。 <br/>                                                                                                                                                                                                                                                              |
| [**IDCompositionTranslateTransform3D**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontranslatetransform3d)<br/>           | 表示3D 轉換，其會影響沿著 X 軸、y 軸和 Z 軸的視覺效果位移。 <br/>                                                                                                                                                                                                                                                                                      |
| [**IDCompositionVirtualSurface**](/windows/win32/api/dcomp/nn-dcomp-idcompositionvirtualsurface)<br/>                       | 表示可與視覺效果樹狀結構中的視覺效果相關聯的稀疏配置點陣圖。<br/>                                                                                                                                                                                                                                                                                  |
| [**IDCompositionVisual**](/windows/win32/api/dcomp/nn-dcomp-idcompositionvisual)<br/>                                       | 表示 DirectComposition 視覺效果。 <br/>                                                                                                                                                                                                                                                                                                                                                        |
| [**IDCompositionVisual2**](/windows/win32/api/dcomp/nn-dcomp-idcompositionvisual2)<br/>                                     | 代表視覺化樹狀結構中的一個 DirectComposition 視覺效果。<br/>                                                                                                                                                                                                                                                                                                                                      |
| [**IDCompositionVisual3**](/windows/win32/api/dcomp/nn-dcomp-idcompositionvisual3)<br/>                                     | 代表視覺化樹狀結構中的一個 DirectComposition 視覺效果。<br/>                                                                                                                                                                                                                                                                                                                                      |
| [**IDCompositionVisualDebug**](/windows/win32/api/dcomp/nn-dcomp-idcompositionvisualdebug)<br/>                             | 表示 debug 視覺效果。<br/>                                                                                                                                                                                                                                                                                                                                                                     |
