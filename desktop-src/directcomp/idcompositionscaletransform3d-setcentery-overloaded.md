# <a name="idcompositionscaletransform3dsetcentery-methods"></a>IDCompositionScaleTransform3D：： SetCenterY 方法

變更或繪製3D 縮放轉換之 CenterY 屬性值的動畫。 CenterY 屬性會指定執行調整的起點的 y 座標。

### <a name="overload-list"></a>多載清單



| 方法                                                                                                          | 描述                                            |
|:----------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetCenterY (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setcentery(float))                                     | 變更 CenterY 屬性的值。<br/>  |
| [**SetCenterY (IDCompositionAnimation \*)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setcentery(idcompositionanimation)) | 繪製 CenterY 屬性值的動畫。<br/> |
