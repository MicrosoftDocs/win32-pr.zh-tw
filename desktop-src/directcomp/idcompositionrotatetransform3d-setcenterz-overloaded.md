# <a name="idcompositionrotatetransform3dsetcenterz-methods"></a>IDCompositionRotateTransform3D：： SetCenterZ 方法

變更或繪製3D 旋轉轉換之 CenterZ 屬性值的動畫。 CenterZ 屬性會指定執行旋轉的點的 z 座標。 預設值為零。

### <a name="overload-list"></a>多載清單



| 方法                                                                                                           | 描述                                            |
|:-----------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetCenterZ (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setcenterz(float))                                     | 變更 CenterZ 屬性的值。<br/>  |
| [**SetCenterZ (IDCompositionAnimation \*)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setcenterz(idcompositionanimation)) | 繪製 CenterZ 屬性值的動畫。<br/> |
