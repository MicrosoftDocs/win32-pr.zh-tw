# <a name="idcompositiontranslatetransform3dsetoffsetz-methods"></a>IDCompositionTranslateTransform3D：： SetOffsetZ 方法

變更或繪製3D 轉譯轉換效果之 OffsetZ 屬性值的動畫。 OffsetZ 屬性會指定沿著 Z 軸的平移。

### <a name="overload-list"></a>多載清單



| 方法                                                                                                              | 描述                                            |
|:--------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetOffsetZ (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform3d-setoffsetz(float))                                     | 變更 OffsetZ 屬性的值。<br/>  |
| [**SetOffsetZ (IDCompositionAnimation \*)**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform3d-setoffsetz(idcompositionanimation)) | 繪製 OffsetZ 屬性值的動畫。<br/> |
