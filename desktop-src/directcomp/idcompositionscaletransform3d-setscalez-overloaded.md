# <a name="idcompositionscaletransform3dsetscalez-methods"></a>IDCompositionScaleTransform3D：： SetScaleZ 方法

變更或繪製調整規模轉換之 ScaleZ 屬性值的動畫。 ScaleZ 屬性會指定沿著 Z 軸的縮放係數。

### <a name="overload-list"></a>多載清單



| 方法                                                                                                        | 描述                                           |
|:--------------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**SetScaleZ (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setscalez(float))                                     | 變更 ScaleZ 屬性的值。<br/>  |
| [**SetScaleZ (IDCompositionAnimation \*)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setscalez(idcompositionanimation)) | 繪製 ScaleZ 屬性值的動畫。<br/> |
