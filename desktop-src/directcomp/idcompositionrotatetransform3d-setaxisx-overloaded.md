# <a name="idcompositionrotatetransform3dsetaxisx-methods"></a>IDCompositionRotateTransform3D：： SetAxisX 方法

變更或繪製3D 旋轉轉換之 AxisX 屬性值的動畫。 AxisX 屬性會指定旋轉軸向量的 x 座標。 預設值為零。

### <a name="overload-list"></a>多載清單



| 方法                                                                                                       | 描述                                          |
|:-------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------|
| [**SetAxisX (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisx(float))                                     | 變更 AxisX 屬性的值。<br/>  |
| [**SetAxisX (IDCompositionAnimation \*)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisx(idcompositionanimation)) | 繪製 AxisX 屬性值的動畫。<br/> |
