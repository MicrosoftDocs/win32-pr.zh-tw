# <a name="idcompositionrectangleclipsetbottomleftradiusx-methods"></a>IDCompositionRectangleClip：： SetBottomLeftRadiusX 方法

變更或繪製此剪輯之 BottomLeftRadiusX 屬性值的動畫。 BottomLeftRadiusX 屬性會指定將剪輯左下角四捨五入的橢圓形 x 半徑。

| 方法                                                                                                                           | 描述                                                                   |
|:---------------------------------------------------------------------------------------------------------------------------------|:------------------------------------------------------------------------------|
| [**SetBottomLeftRadiusX (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setbottomleftradiusx(idcompositionanimation))                                     | 變更此剪輯之 BottomLeftRadiusX 屬性的值。<br/>  |
| [**SetBottomLeftRadiusX (IDCompositionAnimation \*)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setbottomleftradiusx(idcompositionanimation)) | 繪製此剪輯之 BottomLeftRadiusX 屬性值的動畫。<br/> |
