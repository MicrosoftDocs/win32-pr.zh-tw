# <a name="idcompositiontranslatetransform3dsetoffsetx-methods"></a>IDCompositionTranslateTransform3D：： SetOffsetX 方法

變更或繪製3D 轉譯轉換效果之 OffsetX 屬性值的動畫。 OffsetX 屬性會指定沿著 X 軸的平移。

### <a name="overload-list"></a>多載清單



| 方法                                                                                                              | 描述                                            |
|:--------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetOffsetX (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform3d-setoffsetx(float))                                     | 變更 OffsetX 屬性的值。<br/>  |
| [**SetOffsetX (IDCompositionAnimation \*)**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform3d-setoffsetx(idcompositionanimation)) | 繪製 OffsetX 屬性值的動畫。<br/> |
