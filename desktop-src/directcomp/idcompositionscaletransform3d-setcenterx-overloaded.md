# <a name="idcompositionscaletransform3dsetcenterx-methods"></a>IDCompositionScaleTransform3D：： SetCenterX 方法

變更或繪製3D 縮放轉換之 System.windows.media.rotatetransform.centerx 屬性值的動畫。 System.windows.media.rotatetransform.centerx 屬性會指定執行調整的起點的 x 座標。

### <a name="overload-list"></a>多載清單



| 方法                                                                                                          | 描述                                            |
|:----------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetCenterX (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setcenterx(float))                                     | 變更 System.windows.media.rotatetransform.centerx 屬性的值。<br/>  |
| [**SetCenterX (IDCompositionAnimation \*)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setcenterx(idcompositionanimation)) | 繪製 System.windows.media.rotatetransform.centerx 屬性值的動畫。<br/> |
