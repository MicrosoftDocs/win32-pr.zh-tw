# <a name="idcompositionskewtransformsetanglex-methods"></a>IDCompositionSkewTransform：： SetAngleX 方法

變更或繪製2D 扭曲轉換之 System.windows.media.skewtransform.anglex 屬性值的動畫。 System.windows.media.skewtransform.anglex 屬性會指定旋轉角度（以度為單位）。 預設值為零。

### <a name="overload-list"></a>多載清單



| 方法                                                                                                     | 描述                                           |
|:-----------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**SetAngleX (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setanglex(float))                                     | 變更 System.windows.media.skewtransform.anglex 屬性的值。<br/>  |
| [**SetAngleX (IDCompositionAnimation \*)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setanglex(idcompositionanimation)) | 繪製 System.windows.media.skewtransform.anglex 屬性值的動畫。<br/> |
