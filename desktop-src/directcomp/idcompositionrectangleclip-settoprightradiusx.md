# <a name="idcompositionrectangleclipsettoprightradiusx-methods"></a>IDCompositionRectangleClip：： SetTopRightRadiusX 方法

變更或繪製此剪輯之 TopRightRadiusX 屬性值的動畫。 TopRightRadiusX 屬性會指定將剪輯右上角四捨五入的橢圓形 x 半徑。

| 方法                                                                                                                       | 描述                                                                 |
|:-----------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------|
| [**SetTopRightRadiusX (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settoprightradiusx(idcompositionanimation))                                     | 變更此剪輯之 TopRightRadiusX 屬性的值。<br/>  |
| [**SetTopRightRadiusX (IDCompositionAnimation \*)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settoprightradiusx(idcompositionanimation)) | 繪製此剪輯之 TopRightRadiusX 屬性值的動畫。<br/> |
