# <a name="idcompositionskewtransformsetcenterx-methods"></a>IDCompositionSkewTransform：： SetCenterX 方法

變更或繪製2D 扭曲轉換之 System.windows.media.rotatetransform.centerx 屬性值的動畫。 System.windows.media.rotatetransform.centerx 屬性會指定執行扭曲之點的 x 座標。

### <a name="overload-list"></a>多載清單



| 方法                                                                                                       | 描述                                            |
|:-------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetCenterX (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setcenterx(float))                                     | 變更 System.windows.media.rotatetransform.centerx 屬性的值。<br/>  |
| [**SetCenterX (IDCompositionAnimation \*)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setcenterx(idcompositionanimation)) | 繪製 System.windows.media.rotatetransform.centerx 屬性值的動畫。<br/> |
