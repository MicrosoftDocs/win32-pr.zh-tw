# <a name="idcompositionrectangleclipsetright-methods"></a>IDCompositionRectangleClip：： SetRight 方法

建立動畫或變更剪輯矩形右邊屬性的值。 Right 屬性指定裁剪矩形右下角的 x 座標。

### <a name="overload-list"></a>多載清單



| 方法                                                                                                   | 描述                                          |
|:---------------------------------------------------------------------------------------------------------|:-----------------------------------------------------|
| [**SetRight (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setright(float))                                     | 變更 Right 屬性的值。<br/>  |
| [**SetRight (IDCompositionAnimation \*)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setright(idcompositionanimation)) | 將 Right 屬性的值繪製成動畫。<br/> |
