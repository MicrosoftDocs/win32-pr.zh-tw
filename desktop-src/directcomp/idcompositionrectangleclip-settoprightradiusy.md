# <a name="idcompositionrectangleclipsettoprightradiusy-methods"></a>IDCompositionRectangleClip：： SetTopRightRadiusY 方法

變更或繪製此剪輯之 TopRightRadiusY 屬性值的動畫。 TopRightRadiusY 屬性會指定將剪輯右上角四捨五入之橢圓形的 y 半徑。

| 方法                                                                                                                       | 描述                                                                 |
|:-----------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------|
| [**SetTopRightRadiusY (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settoprightradiusy(idcompositionanimation))                                     | 變更此剪輯之 TopRightRadiusY 屬性的值。<br/>  |
| [**SetTopRightRadiusY (IDCompositionAnimation \*)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settoprightradiusy(idcompositionanimation)) | 繪製此剪輯之 TopRightRadiusY 屬性值的動畫。<br/> |
