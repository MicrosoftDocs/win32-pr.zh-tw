# <a name="pipes-interprocess-communications"></a>管道 (處理序間通訊) 

*管道* 是處理用於通訊的共用記憶體區段。 建立管道的進程是 *管道伺服器*。 連接到管道的進程是 *管道用戶端*。 一個進程會將資訊寫入管道，而另一個進程會從管道讀取資訊。 本總覽說明如何建立、管理和使用管道。

-   [使用管道](using-pipes.md)
