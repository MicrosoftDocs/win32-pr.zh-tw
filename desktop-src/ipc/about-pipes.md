# <a name="about-pipes"></a>關於管道

管道有兩種類型： [匿名管道](anonymous-pipes.md) 和 [具名管道](named-pipes.md)。 匿名管道需要的額外負荷比具名管道少，但提供有限的服務。

在這裡使用的「管道」一詞表示使用管道作為資訊管道。 就概念而言，管道有兩個端點。 單向管道可讓一端的進程寫入管道，並允許另一端的進程從管道讀取程式。 雙向 (或雙工) 管道，可讓處理常式從管道的結尾讀取和寫入。

-   [匿名管道](anonymous-pipes.md)
-   [具名管道](named-pipes.md)
