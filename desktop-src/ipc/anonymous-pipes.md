# <a name="anonymous-pipes"></a>匿名管道

*匿名管道* 是未命名的單向管道，通常會在父進程和子進程之間傳輸資料。 匿名管道一律為本機;它們無法用來透過網路進行通訊。

如需詳細資訊，請參閱下列主題：

-   [匿名管道作業](anonymous-pipe-operations.md)
-   [管道控制碼繼承](pipe-handle-inheritance.md)
-   [匿名管道安全性和存取權限](anonymous-pipe-security-and-access-rights.md)
