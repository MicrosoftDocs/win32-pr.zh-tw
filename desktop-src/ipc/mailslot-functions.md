# <a name="mailslot-functions"></a>郵筒函數

下列函數可以搭配 mailslots 使用。



| 函式                                   | 描述                                                                  |
|--------------------------------------------|------------------------------------------------------------------------------|
| [**CreateMailslot**](/windows/desktop/api/Winbase/nf-winbase-createmailslota)   | 使用指定的名稱建立郵筒。                                  |
| [**GetMailslotInfo**](/windows/desktop/api/Winbase/nf-winbase-getmailslotinfo) | 抓取指定之郵筒的相關資訊。                          |
| [**SetMailslotInfo**](/windows/desktop/api/Winbase/nf-winbase-setmailslotinfo) | 設定指定的郵筒用於讀取作業的超時值。 |
