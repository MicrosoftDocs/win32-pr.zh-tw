# <a name="determining-shim-status"></a>判斷填充碼狀態

## <a name="platforms"></a>平台

<dl> 用戶端-Windows 8  
伺服器-Windows Server 2012  
</dl>

## <a name="description"></a>Description

Windows 8 會在基於各種原因而填充應用程式時記錄事件。 若要判斷您的應用程式是否正在填充，最簡單的方式就是檢查事件檢視器。 移至 [應用程式及服務] 記錄 \\ Microsoft Windows Application-Experience 程式 \\ 遙測。

## <a name="mitigation"></a>降低

取得填充碼銜接的最佳方式是將可執行檔重新命名，或增加主要版本 1 (重新編譯可執行檔) 。
