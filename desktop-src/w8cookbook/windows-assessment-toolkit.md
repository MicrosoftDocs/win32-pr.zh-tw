# <a name="windows-assessment-toolkit"></a>Windows 評定工具組

## <a name="platforms"></a>平台

**客戶** 端-Windows 7 \| Windows 8  

## <a name="description"></a>Description

Windows 評量工具組和 Windows 效能工具組會組成 Windows 的評定及部署套件 (ADK) 。 它們一起提供完整的解決方案，讓您評估整體電腦效能，以及自動化將 Windows 作業系統部署到新電腦的工作。

本主題著重于 **評定工具** 組。 評量結果是用來診斷潛在的問題，因此您開發的硬體和軟體都有回應，而且會對電池壽命、啟動效能和關機時間造成最大的影響。 OEM 合作夥伴、ISV/IHV 夥伴、愛好者和其他社區成員都可以使用相同的評量，以建立可測量、比較和審視品質層面的通用架構。

藉由使用 Windows 評量工具，您可以跨各種案例來測量效能的不同層面，從開機時間到電池壽命效能到高定義影片串流。 評量可找出潛在的問題、不一致的行為，並醒目提示要調查的區域。

在先前的 Windows 版本中，有多項工具可用來測量品質，包括速度測試套件 (VTS/VOS) 、基礎品質工具套件 (FQTS) ，以及系統電源和效能工具套件 (SPPTS) 。 評量工具組將這些效能診斷工具的功能結合成一組整合式工具，讓您更容易使用，並提供更廣泛、更有意義的結果。

Windows 評量工具組透過下列方式將消費者滿意度最大化：

-   協助您評估 Windows、附加價值的軟體和驅動程式，以及硬體設定的整體體驗
-   提供可協助您識別品質問題根本原因的詳細系統資料
-   藉由在開發期間洩漏問題來降低成本
-   從多個結果集建立比較圖形，一段時間後，從不同的電腦或同一組電腦產生結果的比較
-   以標準化格式產生意見反應，讓您可以用來改善產品品質

使用評量工具組可以達成重要的商務目標：

-   **量值 & 比較** -您可以使用資料來比較元件 (軟體、驅動程式或這兩種) 與其他類似的元件，以加速您的決策制定、建議和競爭對手標示
-   **提升品質** -您可以獨立運作或與相關的合作夥伴合作，根據預先定義的品質準則建立元件 (軟體、驅動程式或) 
-   **追蹤品質**   您可以建立一個流程，以有效率地追蹤元件版本的品質，並在每次反復專案之後偵測回歸

## <a name="usage-and-best-practices"></a>使用方式和最佳作法

評量是 XML 和二進位檔的組合，可在電腦上引發一組特定狀態、測量及記錄活動，以及保留記錄的結果。 作業是一或多個評量及其設定的集合，會在電腦上一次執行。 評量平臺提供一致執行和顯示作業、評量和結果的基礎結構。 結果通常包含診斷和修復資訊，可協助您判斷需要額外調查和修正動作的區域。 您可以：

-   使用 **Windows 評量主控台** (Windows AC) ，對單一電腦或小型電腦集合執行評定 <dl> 此案例適用于想要在一或多個不同的電腦設定上查看效能特性的使用者。 WindowsAC 是圖形化使用者介面， (GUI) 用來分組評量、建立作業、封裝工作、執行工作，以及管理工作結果。 結果通常包含建議的動作。  
    </dl>
-   Run assessments against multiple computers in a lab environment with **Windows Assessment Services** (Windows AS) <dl> 此案例主要是針對想要在開發環境中對桌上型電腦、膝上型電腦或平板電腦執行一系列定性評量的使用者。  
    </dl>

評量工具組提供下列功能：

-   簡單的圖形化使用者介面 (GUI) 和評量，可以用來評估電腦，而不需要對系統進行任何深入的技術知識
-   在主控台或 UI 中查看的評定結果，其中通常包含可協助您改善系統的建議
-   按一下即可執行預先設定工作的能力
-   每個預先設定的作業評定中預先定義的評定設定，讓您可以在多部電腦上執行作業，並確保結果可比較
-   可以自訂的作業，包括您想要使用的評量和您想要使用的設定
-   用於編寫腳本和自動化作業的評定平臺命令列語法
-   執行作業、查看結果、採取補救步驟來改善系統的能力，然後再次執行作業，並將新的結果與舊的結果並存比較，以查看系統的改進程度。

評量工具組通常用於下列案例：




| 狀況 | 描述 | 
|----------|-------------|
| 「黑箱」 | 執行預先定義的作業，並檢查結果是否有任何不尋常的值，或驅動程式、記憶體使用量或評量位址的其他區域出現問題的跡象。 | 
| 比較結果 | <ol><li>在執行支援之作業系統的任何電腦上，使用建議的設定來執行單一評量。</li><li>使用 Windows AC 來封裝要在另一部電腦上執行的作業。</li><li>將結果儲存至共用，讓您可以比較結果。</li><li>比較任何 Windows 電腦與任何其他受支援作業系統的結果，以找出差異。</li></ol><br /> | 
| 清理電腦 | 在只包含作業系統的全新電腦上執行評量，以建立基準系統結果。 | 
| 已新增硬體或軟體元件的電腦 | 將新的硬體或軟體新增至乾淨的電腦系統，然後重新執行評量，以將結果與乾淨的電腦結果進行比較。 | 
| 建立評量 | 使用公用 Api 來開發或擴充評量，或將評量與您的工具和基礎結構整合。 | 




 

您可以使用這些評定：



| 評量                                   | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
|----------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 驅動程式認證預先驗證          | **驅動程式** 憑證驗證前評估會確認正在執行之 Windows 作業系統上的驅動程式是否符合 Windows 認證計畫的資格。 結果包含的建議可協助您解決評量發現的問題，例如未簽署的驅動程式或過期的簽章。                                                                                                                                                                                                                                                            |
| 驅動程式驗證                          | **驅動程式驗證** 評量會確認離線 Windows 映像或執行中 Windows 作業系統是否包含一組正確的驅動程式。 結果包含的建議可協助您解決評量發現的任何問題。 這些問題可能包括遺失、重複、舊或不必要的驅動程式。                                                                                                                                                                                                                                         |
| 檔案處理                                | 檔案 **處理** 評估提供自動化的方式來執行一般檔案作業和捕獲計量。 計量會測量持續時間和輸送量，以協助您瞭解電腦在使用者檔案處理案例中執行的程度。 檔案處理評量會使用一組工作負載來模擬在用戶端系統上複製、移動、壓縮、解壓縮及刪除檔案和資料夾的使用者。                                                                                                                                       |
| 相片處理                               | **相片處理** 評量藉由模擬正在觀賞和操作相片的終端使用者，來測量電腦效能和電池壽命。                                                                                                                                                                                                                                                                                                                                                                                                                           |
| Internet Explorer 啟動]/[建立] 索引標籤          | **Internet Explorer 啟動效能** 評估有助於找出可能會影響啟動 Internet Explorer 所需時間的元件。 評量會測量完整轉譯空白頁面的時間，包括 IExplore.exe 程式的載入時間和框架建立和索引標籤建立間隔。 它也會測量系統上安裝的所有延伸模組、增益集和工具列的效果。 它不會測量任何網路或流覽效能。                                                                                         |
| 開啟/關閉                                       | 「**開啟/關閉」轉換** 評量會測量 Windows 8 開機、待命和休眠效能案例的效能。                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| Internet Explorer 流覽效能       | **Internet Explorer 流覽效能** 評估會測量 Internet Explorer 的流覽體驗品質，並評估 CPU 和圖形硬體功能。 提供三種不同的流覽工作負載，以各種方式來壓力電腦。                                                                                                                                                                                                                                                                                                  |
| 媒體轉碼效能                | **轉碼影片** 評量會測量將影片檔案變更為不同格式或位元速率的程式。 這項評量會以常見的輸入和輸出檔案格式和解決方式執行一系列的轉碼作業                                                                                                                                                                                                                                                                                                                                           |
| WindowsUI 效能                       | **Windows UI 效能** 評定會評估 Windows 使用者介面內某些基本體驗的效能。 評量會測量回應能力和轉譯品質，而評量會進行模擬使用者活動的工作負載，例如使用搜尋和從 Windows 存放區應用程式環境轉換到桌面。 回應結果是以毫秒為單位來測量。 數位很低表示電腦速度更快且更具回應性。 針對轉譯，結果會顯示畫面播放速率和發生的問題數目。 |
| 記憶體耗用量                             | 您可以使用記憶體使用量評 **量** 來從數量比較基準作業系統映射與其他作業系統映射。 然後，您可以識別會影響實體系統記憶體使用量的特定元件。 這些元件可以包括驅動程式、增益集應用程式、預先載入的軟體套件，以及防毒程式。                                                                                                                                                                                                           |
| 第一個開機效能                       | **第一次開機效能** 評定會找出影響 Windows 在電腦第一次啟動時需要多久的時間才會啟動，並顯示開始畫面的問題。 結果可協助 Oem 診斷造成延遲的原因，並提供改善體驗的建議。                                                                                                                                                                                                                                                                                      |
| 媒體串流                              | 當您使用 Internet Explorer 串流媒體時， **串流處理媒體效能** 評定可協助您評估電腦設定的效能。 您可以使用評定結果來瞭解、比較和改善串流媒體體驗。                                                                                                                                                                                                                                                                                                                 |
| WinSAT 全方位                         | **Windows 系統評量 (WinSAT)** 用來對數個系統元件（包括 CPU、記憶體、磁片和圖形）的電腦效能進行評分和改善。 WinSAT 全方位評估結果能以數位表示電腦的硬體設定能力。 較高的分數通常表示評估的電腦執行效能比較低分數的電腦更好且更快速。                                                                                                                                                        |
| 節能                            | **能源效率** 作業可讓您以自動化的方式評估電腦的電池壽命。 使用工作負載時，能源效率作業也會執行診斷，以評估系統元件在閒置時是否使用電源。                                                                                                                                                                                                                                                                                                               |
| 微篩選診斷設定               | **微篩選器診斷** 選項會在 Internet Explorer 啟動效能評估、檔案處理評估和開機效能 (Windows 8) 評量內執行。 在提供「篩選器」診斷選項的評量中選取此選項，會產生可協助您評估微篩選器作業對各種評定案例的影響的度量。                                                                                                                                                                                  |
| Windows Media Player 效能和品質 | **Windows Media Player 的效能和品質** 評定會啟動 WMP，然後播放多個媒體剪輯，以捕捉與媒體播放相關的效能和品質計量。                                                                                                                                                                                                                                                                                                                                                                          |
