## <a name="purpose"></a>目的

在操作 *手冊* 中，我們會提供 Windows 10、Windows 8、Windows 8.1、windows server 2012 和 windows Server 2012 R2 作業系統的變更和新功能的相關資訊。 我們也會提供指導方針，讓您驗證現有和計畫的程式與新作業系統的相容性。 我們假設您已經熟悉舊版的 Windows。

內容適用于：

-   Windows 10
-   Windows 8.1
-   Windows Server 2012 R2
-   Windows 8
-   Windows Server 2012
-   Windows 7
-   Windows Server 2008 R2
