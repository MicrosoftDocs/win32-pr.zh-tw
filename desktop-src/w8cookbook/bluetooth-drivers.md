# <a name="bluetooth-drivers"></a>藍牙驅動程式

在 Windows 7、Windows 8 或 Windows 8.1 的升級過程中，將不會將藍牙驅動程式和協力廠商設定檔套件遷移至 Windows 10。 請參閱 [這篇文章](/windows-hardware/design/component-guidelines/bluetooth) ，以取得 Windows 10 藍牙變更的詳細資訊。
