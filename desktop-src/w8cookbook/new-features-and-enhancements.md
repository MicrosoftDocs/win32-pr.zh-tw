# <a name="new-features-and-enhancements"></a>新功能和增強功能

本檔的這一節說明可改善使用者和開發人員體驗的增強功能和新功能。 不過，其中有幾個會影響開發人員必須留意現有和新產品的相關資訊。 以下是依功能區分組的新功能和增強功能清單：

## <a name="security"></a>安全性

-   早期啟動反惡意程式碼
-   核心模式驅動程式的安全開機功能簽署需求
-   測量開機

## <a name="performance"></a>效能

-   啟動應用程式
-   復原檔案系統
-   檔案伺服器 API 支援
-   自動維護

## <a name="usability"></a>可用性

-   協力廠商輸入法編輯器

## <a name="storage-and-file-system"></a>儲存體和檔案系統

-   在 Windows 8.1 中追蹤檔案的修改範圍
-   新的 API 可讓應用程式將「修剪和取消對應」提示傳送至存放裝置媒體
-   多重路徑 i/o 現在支援延伸儲存體要求區塊
-   復原檔案系統
-   檔案伺服器 API 支援
-   新的檔案歷程記錄功能
-   作業系統現在可控制磁片磁碟機的電源

## <a name="usb"></a>USB

-   USB 3.0 的支援
