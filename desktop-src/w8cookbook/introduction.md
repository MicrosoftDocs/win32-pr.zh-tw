# <a name="introduction-compatibility-cookbook-for-windows"></a>Windows) 的簡介 (相容性操作手冊

Windows 10、Windows 8、Windows 8.1 和 Windows Server 2012 R2 引進最新的作業系統技術和軟體發展平臺，供全球的應用程式開發人員和企業使用。 為了進一步增強 Windows 的安全性、可靠性、效能和使用者體驗，Microsoft 引進了許多新功能、改良的現有功能，並移除其他功能。

雖然 Windows 10、Windows 8、Windows 8.1 和 Windows Server 2012 R2 的目標是要與針對先前發行的作業系統所撰寫的大部分應用程式保持高度相容，但由於創新、強化的安全性和可靠性，某些相容性中斷是不可避免的。 整體來說，新作業系統與現有應用程式的相容性很高。

本逐步 *指南* 分為三個主要區段： Windows 10、Windows 8.1 和 Windows 8：

-   Windows 10
-   Windows 8.1 與 Windows Server 2012 R2
-   Windows 8 和 Windows Server 2012

以下是這一版 *手冊* 的新功能或已更新的主題：

-   Windows 映像檔案開機 (的 WimBoot) 
-   Windows 應用程式認證套件

我們邀請您查看這些主題，以瞭解如何優化您的應用程式，並利用此最新版本的 Windows 所提供的功能。

可以在[Windows Vista 和 Windows server 2008](/previous-versions/bb757005(v=msdn.10))和[Windows 7 和 Windows Server 2008 R2 應用程式品質逐步指南](/windows/desktop/Win7AppQual/windows-7-application-quality-cookbook)中，找到可解決舊版 windows 和 windows Server 作業系統的 Windows 相容性操作手冊版本
