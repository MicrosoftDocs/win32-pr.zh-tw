# <a name="undocumented-apis"></a>未記載的 API

您的 app 不應該呼叫未記載的 Windows API，或相依於特定的 Windows 檔案匯出或登錄機碼。 這可能會導致功能中斷、資料遺失和潛在的安全性問題。 如果您 app 需要的功能無法使用，您可透過平常使用的意見反應管道及 Windows 測試人員計畫提供意見反應。
