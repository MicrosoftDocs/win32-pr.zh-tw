# <a name="moving-a-user-account-or-group-to-a-different-role"></a>將使用者帳戶或群組移至不同的角色

當使用者或群組的成員不應再獲得指派給角色的應用程式資源存取權時，您可以從角色中移除使用者帳戶或群組。

**若要從角色移除使用者帳戶或群組**

1.  在 [元件服務] 系統管理工具的主控台樹中，找出包含您感興趣之角色的 COM + 應用程式。 在主控台樹中展開視圖，直到角色的使用者可以看見為止。

2.  以滑鼠右鍵按一下您要移除的使用者帳戶或群組，然後按一下 [ **刪除**]。

3.  當 [ **確認專案刪除** ] 對話方塊出現提示時，請按一下 [ **是]** 以刪除使用者帳戶或群組。

您從角色移除的使用者帳戶或群組不會再出現在 [ **使用者** ] 資料夾中。 新的角色成員資格會在應用程式下次啟動時生效。 如果您已對 **系統應用程式** 進行變更，您必須重新開機電腦，變更才會生效。
