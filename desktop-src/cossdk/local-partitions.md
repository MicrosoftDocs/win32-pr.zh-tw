# <a name="local-partitions"></a>本機分割區

分割區可以使用的其中一種方式是在本機應用程式伺服器上。 在此案例中，系統管理員可以使用 [元件服務] 系統管理工具來建立磁碟分割。 本機使用者只能存取在本機執行的資料分割，而不是在 Active Directory 內的資料分割，而不是網域中其他電腦上的使用者。 建立分割區並在其中安裝 COM + 應用程式之後，系統管理員可以將該磁碟分割指派為該伺服器上一或多個使用者的預設資料分割。 若要將預設分割區指派給使用者，系統管理員可以使用 [元件服務] 系統管理工具。
