# <a name="partition-properties"></a>分割區屬性

磁碟分割有五個可用的屬性。 這些屬性和其描述如下：

<dl> <dt>

<span id="Partition_Name"></span><span id="partition_name"></span><span id="PARTITION_NAME"></span>分割區名稱
</dt> <dd>

與資料分割相關聯的文字字串。

</dd> <dt>

<span id="Partition_ID"></span><span id="partition_id"></span><span id="PARTITION_ID"></span>分割區識別碼
</dt> <dd>

系統用來讓使用者與分割區相互關聯的 GUID。

</dd> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>描述
</dt> <dd>

分割區的文字描述。

</dd> <dt>

<span id="Disable_Deletion"></span><span id="disable_deletion"></span><span id="DISABLE_DELETION"></span>停用刪除
</dt> <dd>

禁止使用者刪除分割區的機制。

</dd> <dt>

<span id="Disable_Changes"></span><span id="disable_changes"></span><span id="DISABLE_CHANGES"></span>停用變更
</dt> <dd>

禁止使用者變更分割區的機制。

</dd> </dl>

以滑鼠右鍵按一下 [元件服務] 系統管理工具內的分割區名稱，然後選擇 [ **屬性**]，即可看到這些屬性。

如需有關 [資料分割識別碼] 屬性的詳細資訊，請參閱在分割區 [中註冊和啟用元件](registering-and-activating-components-in-partitions.md)。
