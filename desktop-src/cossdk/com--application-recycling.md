# <a name="com-application-recycling"></a>COM + 應用程式回收

應用程式回收可大幅提升應用程式的整體穩定性。 因為大部分應用程式的效能可能會因為像是記憶體流失、依賴協力廠商程式碼，以及 nonscalable 資源使用情形等因素而降低，所以 COM + 應用程式回收提供簡單的解決方案，可正常關閉與應用程式相關聯的進程並重新啟動。

下列主題提供有關在 COM + 中使用應用程式回收的背景和工作資訊。

| [COM + 應用程式回收工作](com--application-recycling-tasks.md)       | 提供設定 COM + 應用程式回收的指示。 |
