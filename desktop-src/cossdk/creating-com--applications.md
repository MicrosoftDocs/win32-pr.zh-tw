# <a name="creating-com-applications"></a>建立 COM + 應用程式

本節說明如何建立新的 (空白) COM + 應用程式，然後將元件新增至該應用程式。 它也會說明如何將 COM 元件加入至現有的 COM + 應用程式，並將其從現有的 COM + 應用程式中移除，以及如何在 COM + 應用程式之間移動和複製元件至另一個。



| 主題                                                                                                       | 描述                                                                        |
|-------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------|
| [建立新的 COM + 應用程式](creating-a-new-com--application.md)<br/>                           | 說明如何建立新的 COM + 應用程式。<br/>                         |
| [安裝新元件](installing-new-components.md)<br/>                                       | 描述如何安裝新元件。<br/>                                |
| [匯入元件](importing-components.md)<br/>                                                 | 說明如何匯入元件。<br/>                                     |
| [從 COM + 應用程式移除元件](removing-a-component-from-a-com--application.md)<br/> | 描述如何從 COM + 應用程式刪除元件。<br/>            |
| [移動元件](moving-components.md)<br/>                                                       | 說明如何將元件從一個 COM + 應用程式移到另一個 COM + 應用程式。<br/> |
| [複製元件](copying-components.md)<br/>                                                     | 說明如何將元件從一個 COM + 應用程式複製到另一個 COM + 應用程式。<br/> |
| [刪除 COM + 應用程式](deleting-a-com--application.md)<br/>                                   | 描述如何刪除 COM + 應用程式。<br/>                             |



 

如需本節所述程式的概念資訊，請參閱 [Com + 應用程式總覽](com--application-overview.md) 和 [部署 Com + 應用程式](deploying-com--applications.md)。

如需有關匯出和安裝 COM + 應用程式之管理工作的程式資訊，請參閱《元件服務系統管理員指南》中的「安裝 COM + 應用程式」。
