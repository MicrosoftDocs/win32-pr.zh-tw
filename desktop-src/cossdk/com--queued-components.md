# <a name="com-queued-components"></a>COM + 佇列元件

[COM + 佇列元件] 功能可讓用戶端從 COM + 伺服器應用程式要求服務，即使伺服器應用程式暫時無法使用也是一樣。 用戶端要求會儲存在佇列中，並在伺服器變成可用時進行處理。

> [!Note]  
> 若要使用 COM + 佇列的元件服務，您必須在電腦上安裝 [訊息佇列](/previous-versions/windows/desktop/legacy/ms711472(v=vs.85)) 。
