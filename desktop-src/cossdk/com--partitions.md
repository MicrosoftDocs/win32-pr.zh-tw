# <a name="com-partitions"></a>COM + 磁碟分割

您可以使用 COM + 分割服務，在電腦上安裝不同版本的 COM + 應用程式，並同時執行它們。

**WINDOWS XP：** 無法使用建立、設定或委派 COM + 分割的能力。 全域分割區是唯一可用的 COM + 分割。

* * Windows 2000： * * COM + 分割服務無法在 Windows 2000 中使用。

下列主題提供 COM + 磁碟分割服務的相關資訊。



| 主題                                                               | 描述                                                                 |
|---------------------------------------------------------------------|-----------------------------------------------------------------------------|
| [COM + 磁碟分割概念](com--partitions-concepts.md)<br/> | 提供使用 COM + 磁碟分割的總覽。<br/>                   |
| [COM + 分割工作](com--partitions-tasks.md)<br/>       | 提供建立和管理 COM + 磁碟分割的指示。<br/> |



 

> [!Note]  
> 預設不會啟用 COM + 分割服務。 若要使用 COM + 分割服務，您必須透過 [元件服務] 系統管理工具啟用它，或將 [**LocalComputer**](localcomputer.md) 集合上的 PartitionsEnabled 屬性變更為 True。
