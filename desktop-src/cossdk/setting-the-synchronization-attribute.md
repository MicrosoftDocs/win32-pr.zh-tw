# <a name="setting-the-synchronization-attribute"></a>設定同步處理屬性

使用 [元件服務] 系統管理工具或 COM + 系統管理程式庫來設定同步處理屬性。  (參閱 [自動化 COM + 管理](automating-com--administration.md)。 ) 

**使用元件服務系統管理工具設定屬性**

1.  安裝元件之後，請開啟元件的 [ **屬性** ] 對話方塊。

2.  按一下 [ **並行** 存取] 索引標籤。

3.  在 **[同步處理支援**] 下，按一下您要為元件選取的值旁邊的選項按鈕。

4.  按一下 [確定]。
