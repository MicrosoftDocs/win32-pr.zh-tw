# <a name="optimizing-interactions-with-the-com-business-logic-tier"></a>優化與 COM + 商務邏輯層的互動

分散式應用程式的架構通常會將系統細分為三個 (或更) 層。 這些層級之間的互動可能會大幅影響您的應用程式是否成功。 本節討論如何優化商務邏輯層和展示層與資料層之間的互動。 如需詳細資訊，請參閱下列主題：

-   [將 COM + 商務邏輯層和展示層之間的互動優化](optimizing-interactions-between-the-com--business-logic-tier-and-the-presentation-tier.md)
-   [優化 COM + 商務邏輯層與資料層之間的互動](optimizing-interactions-between-the-com--business-logic-tier-and-the-data-tier.md)
