# <a name="com-partitions-concepts"></a>COM + 磁碟分割概念

在需要在同一部電腦上使用不同版本或應用程式設定的運算環境中，當某個版本的應用程式需要不同的元件，或當某個版本需要電腦上與其他應用程式版本不相容的元件時，就會發生衝突。 在過去，解決此問題的唯一方法是維護多部電腦，並在每部電腦上執行不同版本的應用程式。 這種方法既昂貴又沒有效率，因為它會增加硬體成本和系統管理負擔。

COM + 藉由提供 COM + 分割功能來解決這個問題。 您可以使用 COM + 分割，在電腦上安裝不同版本的 COM + 應用程式，並同時執行它們。
