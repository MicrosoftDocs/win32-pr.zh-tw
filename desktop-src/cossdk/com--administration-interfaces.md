# <a name="com-administration-interfaces"></a>COM + 系統管理介面

以下是 COM + 系統管理介面。



| 介面                                        | 描述                                                                                                                                                                                      |
|--------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ICatalogCollection**](/windows/desktop/api/ComAdmin/nn-comadmin-icatalogcollection) | 可讓您列舉、新增、移除及取出 COM + 目錄中任何集合中的專案，以及存取相關集合。                                                               |
| [**ICatalogObject**](/windows/desktop/api/ComAdmin/nn-comadmin-icatalogobject)         | 可讓您取得並放置 COM + 目錄中的物件所公開的屬性。                                                                                                                    |
| [**ICOMAdminCatalog**](/windows/desktop/api/ComAdmin/nn-comadmin-icomadmincatalog)     | 起始會話以進行程式設計的 COM + 管理、存取目錄中的集合、安裝 COM + 應用程式和元件、啟動和停止服務，以及連接到遠端伺服器。 |
| [**ICOMAdminCatalog2**](/windows/desktop/api/ComAdmin/nn-comadmin-icomadmincatalog2)   | 控制應用程式、元件和磁碟分割的互動。                                                                                                                           |
