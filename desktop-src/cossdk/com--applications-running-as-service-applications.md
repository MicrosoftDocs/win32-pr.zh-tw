# <a name="com-applications-running-as-service-applications"></a>以服務應用程式形式執行的 COM + 應用程式

COM + 開發人員可以使用 [元件服務] 系統管理工具，將 COM + 伺服器應用程式設定和實作為服務應用程式。 這表示，如果您的應用程式一律需要執行，則可以自動啟動或重新開機伺服器;如果您的 COM + 應用程式需要執行特殊許可權的作業，您的 COM + 應用程式可以執行為本機系統帳戶;而且您的應用程式相依服務現在可以自動啟動。

下列主題提供有關將應用程式設定為以服務應用程式的方式執行的背景和工作資訊。



| 主題                                                                      | 描述                                                                                         |
|----------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [COM + 服務應用程式概念](com--service-application-concepts.md) | 提供執行 COM + 伺服器應用程式做為服務應用程式之優點的總覽。 |
| [COM + 服務應用程式工作](com--service-application-tasks.md)       | 提供將 COM + 伺服器應用程式設定為服務的指示。                       |
