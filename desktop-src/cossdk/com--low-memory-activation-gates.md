# <a name="com-low-memory-activation-gates"></a>COM + Low-Memory 啟用閘道

COM + 低記憶體啟用閘道服務會在 COM + 伺服器或物件建立之前，自動檢查記憶體。 如果應用程式可用的虛擬記憶體百分比低於固定閾值，則在建立物件之前啟用會失敗。 如此一來，低記憶體啟用閘道服務可大幅提高系統可靠性。

下列主題提供 COM + 低記憶體啟用閘道服務的背景和工作資訊。

| [COM + Low-Memory 啟用閘道工作](com--low-memory-activation-gates-tasks.md)<br/>       | 這項功能無法設定，因此沒有相關聯的工作。<br/> |
