# <a name="com-transactions-concepts"></a>COM + 交易概念

雖然 COM + 會處理許多與交易處理相關的繁瑣程式設計細節，但是當您需要在 COM + 中撰寫交易時，對交易理論具有概念瞭解相當實用。

下表所述的主題將涵蓋適用于交易處理的概念。



| 主題                                                                                                                     | 描述                                                                                                                                                                                 |
|---------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [ACID 屬性](acid-properties.md)<br/>                                                                         | 描述交易的不可部分完成、一致、隔離及持久屬性。<br/>                                                                                              |
| [設定交易](configuring-transactions.md)<br/>                                                       | 描述您可以指派給元件的交易屬性值，以判斷要強制執行的交易保護程度。<br/>                              |
| [設定交易隔離等級](configuring-transaction-isolation-levels.md)<br/>                       | 描述您可以指派給交易的隔離等級，可讓您根據效能和擴充性需求來增加或減少並行。<br/> |
| [管理 COM + 中的自動交易](managing-automatic-transactions-in-com-.md)<br/>                         | COM + 支援的自動化交易程式總覽。 <br/>                                                                                                                |
| [在交易中使用非交易式元件](using-non-transactional-components-in-a-transaction.md)<br/> | 描述如何在交易中使用非交易式元件，以及使用它們的時機。<br/>                                                                                  |
| [整合您自己的交易 (BYOT) ](bring-your-own-transaction--byot-.md)<br/>                                     | 描述如何允許元件繼承外部交易。<br/>                                                                                                              |
