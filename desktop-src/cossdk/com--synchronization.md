# <a name="com-synchronization"></a>COM + 同步處理

COM + 同步處理服務會提供從元件流向元件的同步處理，並禁止多個呼叫端在任何指定時間輸入元件。 同步處理會決定執行緒可以將呼叫分派給物件的時間。
