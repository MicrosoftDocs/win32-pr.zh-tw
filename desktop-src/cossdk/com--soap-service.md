# <a name="com-soap-service"></a>COM + SOAP 服務

只要選取 [元件服務] 系統管理工具中的 [ **使用 SOAP** ] 核取方塊，您就可以將 com + 應用程式公開為 XML web service。 您也可以使用 COM 元件，以透明的方式使用 XML web service （不論是否使用 COM + 部署）。 這表示您可以輕鬆地從現有的 COM + 應用程式建立新的 XML web service，並輕鬆地將 XML web service 併入新的 COM + 應用程式。

下表所述的主題提供有關使用 COM + 將已設定的 COM 元件公開為 XML web service 的背景和工作資訊。



| 主題                                                                   | 描述                                                                            |
|-------------------------------------------------------------------------|----------------------------------------------------------------------------------------|
| [COM + SOAP 服務概念](com--soap-service-concepts.md)<br/> | 提供 XML web service 和 SOAP 的總覽。<br/>                          |
| [COM + SOAP 服務工作](com--soap-service-tasks.md)<br/>       | 提供使用 COM + 來建立和部署 XML web service 的指示。<br/> |
