# <a name="com-functions"></a>COM + 函式

以下是 COM + 函式。



| 函式                                                                 | 描述                                                                                                                             |
|--------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**CoCreateActivity**](/windows/desktop/api/ComSvcs/nf-comsvcs-cocreateactivity)                             | 建立活動來執行可以使用 COM+ 服務的同步或非同步 (Asynchronous) 批次工作，而不需建立 COM+ 元件。 |
| [**CoEnterServiceDomain**](/windows/desktop/api/ComSvcs/nf-comsvcs-coenterservicedomain)                     | 用來輸入可接著使用 COM + 服務的程式碼。                                                                                     |
| [**CoGetDefaultCoNtext**](/windows/desktop/api/combaseapi/nf-combaseapi-cogetdefaultcontext)                       | 抓取指定之單元的預設內容參考。                                                                |
| [**CoLeaveServiceDomain**](/windows/desktop/api/ComSvcs/nf-comsvcs-coleaveservicedomain)                     | 用來離開使用 COM + 服務的程式碼。                                                                                             |
| **ComPlusCompleteCbbSetup**               | 完成目的地電腦上的目錄遷移。                                                                              |
| **GetComPlusPackageInstallStatus** | 指出是否已安裝64位的 Common Language Runtime (CLR) 。                                                                |
| [**GetDispenserManager**](/windows/desktop/api/MtxDM/nf-mtxdm-getdispensermanager)                       | 抓取分配器管理員的 [**IDispenserManager**](/windows/desktop/api/ComSvcs/nn-comsvcs-idispensermanager) 介面。                                             |
| [**GetManagedExtensions**](/windows/desktop/api/ComSvcs/nf-comsvcs-getmanagedextensions)                     | 判斷已安裝的 COM + 版本是否支援提供特殊功能來管理 (受控物件) 的服務元件。    |
| [**GetObjectCoNtext**](/windows/desktop/api/ComSvcs/nf-comsvcs-getobjectcontext)                             | 捕獲與目前 COM + 物件相關聯之內容的參考。                                                   |
| [**MTSCreateActivity**](/windows/desktop/api/ComSvcs/nf-comsvcs-mtscreateactivity)                           | 在單一執行緒的單元中建立活動，以執行同步或非同步批次工作。                                        |
| [**RecycleSurrogate**](/windows/desktop/api/ComSvcs/nf-comsvcs-recyclesurrogate)                             | 回收呼叫進程。                                                                                                           |
| [**SafeRef**](/windows/desktop/api/ComSvcs/nf-comsvcs-saferef)                                               | 目前請勿使用。                                                                                                                   |
