# <a name="com-administration-classes"></a>COM + 系統管理類別

以下是 COM + 系統管理類別。



| 類別                                                          | 描述                                                                                                                                            |
|----------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**COMAdminCatalog**](comadmincatalog.md)                     | 存取儲存在 COM + 目錄中的資料。                                                                                                  |
| [**COMAdminCatalogCollection**](comadmincatalogcollection.md) | 代表 COM + 目錄中的任何集合。 您可以使用它來列舉、新增、移除和取出集合中的專案，以及存取相關集合。 |
| [**COMAdminCatalogObject**](comadmincatalogobject.md)         | 代表 COM + 目錄中集合內的專案。 您可以使用它來取得和修改集合中的專案所公開的屬性。                      |
