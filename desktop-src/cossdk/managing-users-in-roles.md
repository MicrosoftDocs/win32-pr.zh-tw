# <a name="managing-users-in-roles"></a>管理角色中的使用者

設定 COM + 應用程式安全性的大部分管理工作，都牽涉到在安裝應用程式時，以及持續進行時，向使用者擴展預先定義的角色。 如需有關這些安全性功能如何運作的簡介資訊，請參閱設定 [Role-Based 安全性](configuring-role-based-security.md)。

本節將提供下列工作的逐步指示：

-   [啟用 Role-Based 授權檢查](enabling-role-based-authorization-checking.md)
-   [將使用者帳戶或群組指派給角色](assigning-a-user-account-or-group-to-a-role.md)
-   [從角色中移除使用者帳戶或群組](removing-a-user-account-or-group-from-a-role.md)
-   [將使用者帳戶或群組移至不同的角色](moving-a-user-account-or-group-to-a-different-role.md)
