# <a name="com-events"></a>COM + 事件

COM + 事件服務是一個自動、鬆散結合的事件系統，可將來自不同發行者的事件資訊儲存在 COM + 目錄中。 訂閱者可以查詢此存放區，並選取他們想要聽取的事件。
