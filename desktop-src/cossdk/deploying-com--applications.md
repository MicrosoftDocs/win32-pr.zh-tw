# <a name="deploying-com-applications"></a>部署 COM + 應用程式

本節介紹 COM + 中可加速部署 COM + 應用程式的功能。 本節也說明 COM + 應用程式如何在 COM + 目錄中註冊。 如需詳細資訊，請參閱下表所述的主題。

> [!Note]  
> 如需部署 COM + 應用程式的詳細程式，請參閱元件服務管理說明中的「安裝 COM + 應用程式」。

| [部署應用程式 proxy](deploying-application-proxies.md)<br/>                                               | 描述如何透過 [元件服務] 系統管理工具，將 COM + 伺服器應用程式匯出為應用程式 proxy。<br/>                                                       |

| [COMREPL 複製公用程式](the-comrepl-replication-utility.md)<br/>                                           | 描述 COMREPL 公用程式，此公用程式可用來將指定來源電腦的 COM + 類別目錄複寫到一或多部目的電腦。<br/>                                            |
