# <a name="moving-components"></a>移動元件

您可以將元件從一個 COM + 應用程式移到另一個 COM + 應用程式。 實際上，移動元件會從原始的 COM + 應用程式中移除元件，並將它安裝在目標 COM + 應用程式中。

**將元件從一個 COM + 應用程式移到另一個 COM + 應用程式**

1.  在 [元件服務] 系統管理工具的詳細資料窗格中，以滑鼠右鍵按一下您要移動的元件，然後按一下 [ **移動**]。

2.  在 [ **移動元件 (s)** ] 對話方塊中，選取您要移動元件的目的地 com + 應用程式。

3.  按一下 [確定]  。
