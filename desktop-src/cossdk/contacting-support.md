# <a name="contacting-support"></a>聯絡支援

如果您遇到無法解決的問題，您可以使用下列資訊來聯絡 Microsoft 技術支援：

-   發生錯誤的應用程式拓撲，例如應用程式、元件和介面的描述。
-   所有電腦上的應用程式事件記錄檔。
-   如何重現錯誤的描述（如果可能）。
