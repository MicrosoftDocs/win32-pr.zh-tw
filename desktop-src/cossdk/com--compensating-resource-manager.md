# <a name="com-compensating-resource-manager"></a>COM + 補償 Resource Manager

COM + 補償 Resource Manager (CRM) 提供快速且輕鬆的方式來整合應用程式資源與 Microsoft Distributed Transaction Coordinator (DTC) 交易。 使用 CRM 是開發完整 Microsoft 交易服務 (MTS) 資源管理員的替代方案。

下列主題提供有關使用 CRM 的背景和工作資訊。
