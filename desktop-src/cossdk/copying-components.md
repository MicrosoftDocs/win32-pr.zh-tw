# <a name="copying-components"></a>複製元件

您可以將元件從一個 COM + 應用程式複製到另一個 COM + 應用程式。 將元件複製到另一個 COM + 應用程式之後，您可以設定此元件的方式與原始元件不同。

**將元件從一個 COM + 應用程式複製到另一個 COM + 應用程式**

1.  在 [元件服務] 系統管理工具的詳細資料窗格中，以滑鼠右鍵按一下您要複製的元件，然後按一下 [ **複製**]。

2.  在 [ **複製元件** ] 對話方塊的 [ **請選取目的地** ] 窗格中，選取要將元件複製到其中的 com + 應用程式。

3.  輸入所複製元件的新 ProgID。

    > [!Note]  
    > 原始元件的 ProgID 會顯示為參考，而且無法變更。

     

4.  輸入所複製元件的新 CLSID。 COM + 會顯示自動產生的 CLSID。 在大部分的情況下，這就足以唯一識別複製的元件。 只有在您想要變更 COM + 所提供的 CLSID 時，才輸入新的 CLSID。

5.  按一下 [確定]。
