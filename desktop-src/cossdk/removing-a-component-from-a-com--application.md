# <a name="removing-a-component-from-a-com-application"></a>從 COM + 應用程式移除元件

您可以使用 [元件服務] 系統管理工具，從 COM + 應用程式中移除元件。

> [!Note]  
> 從 COM + 應用程式刪除及重新安裝個別元件時，請小心謹慎。 此程式需要有關元件類別的詳細資訊，包括其屬性、介面和方法的安全性設定，以及其他設定。

 

**從 COM + 應用程式移除元件**

1.  在 [元件服務] 系統管理工具的主控台樹中，選取包含您要移除之元件的應用程式。

2.  開啟應用程式的 [ **元件** ] 資料夾。

3.  選取您要移除的元件。

4.  在 [ **動作** ] 功能表上，按一下 [ **刪除**]。 您也可以選取元件、按一下滑鼠右鍵，然後按一下 [ **刪除**]。

5.  當系統提示您確認您的動作時，請按一下 **[是]** 。
