> <span data-ttu-id="0fab4-105">從 COM + 應用程式刪除及重新安裝個別元件時，請小心謹慎。</span><span class="sxs-lookup"><span data-stu-id="0fab4-105">Exercise care when deleting and reinstalling individual components from a COM+ application.</span></span> <span data-ttu-id="0fab4-106">此程式需要有關元件類別的詳細資訊，包括其屬性、介面和方法的安全性設定，以及其他設定。</span><span class="sxs-lookup"><span data-stu-id="0fab4-106">This process requires detailed knowledge about the component's classes, including their attributes, security configurations on interfaces and methods, and other settings.</span></span>

 

<span data-ttu-id="0fab4-107">**從 COM + 應用程式移除元件**</span><span class="sxs-lookup"><span data-stu-id="0fab4-107">**To remove a component from a COM+ application**</span></span>

1.  <span data-ttu-id="0fab4-108">在 [元件服務] 系統管理工具的主控台樹中，選取包含您要移除之元件的應用程式。</span><span class="sxs-lookup"><span data-stu-id="0fab4-108">In the console tree of the Component Services administrative tool, select the application that contains the component you want to remove.</span></span>

3.  <span data-ttu-id="0fab4-110">選取您要移除的元件。</span><span class="sxs-lookup"><span data-stu-id="0fab4-110">Select the component you want to remove.</span></span>

4.  <span data-ttu-id="0fab4-111">在 [ **動作** ] 功能表上，按一下 [ **刪除**]。</span><span class="sxs-lookup"><span data-stu-id="0fab4-111">On the **Action** menu, click **Delete**.</span></span> <span data-ttu-id="0fab4-112">您也可以選取元件、按一下滑鼠右鍵，然後按一下 [ **刪除**]。</span><span class="sxs-lookup"><span data-stu-id="0fab4-112">You can also select the component, right-click, and click **Delete**.</span></span>
