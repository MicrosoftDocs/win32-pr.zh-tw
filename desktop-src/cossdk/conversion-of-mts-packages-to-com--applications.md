# <a name="conversion-of-mts-packages-to-com-applications"></a>將 MTS 套件轉換為 COM + 應用程式

您可以將 COM + 視為 Microsoft Transaction Server (MTS) 和 COM 的合併，以及新功能的引進。

在大部分的情況下，您的 MTS 套件會在 Windows 安裝程式期間轉換為 com + 應用程式。 除了一般設定以外，您還可以利用所有新的 COM + 功能。 如需最重要 COM + 改良功能的清單和簡短說明，請參閱 [Com + 程式設計總覽](com--programming-overview.md)。

如果您想要深入瞭解如何在 Windows 安裝程式期間，自動將 MTS 套件轉換為 com + 應用程式，以及如何針對此程式中可能發生的任何錯誤進行疑難排解，請參閱[從 MTS 自動轉換](automatic-conversion-from-mts.md)。

如果您想要瞭解如何手動將 MTS 套件轉換為 COM + 應用程式，請參閱 [從 Mts 手動轉換](manual-conversion-from-mts.md)。

您也應該注意，安裝 COM + 引進了幾項新功能，可供您的 MTS 套件作為 COM + 應用程式使用。 這些將在 [Com + 轉換結果和問題](com--conversion-results-and-issues.md)中說明。
