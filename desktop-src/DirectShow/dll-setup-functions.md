# <a name="dll-setup-functions"></a>DLL 安裝函式

這些函數會註冊篩選準則。



| 函式                                                         | 描述                                        |
|------------------------------------------------------------------|----------------------------------------------------|
| [**AMovieDllRegisterServer**](amoviedllregisterserver.md)       | 已過時。                                          |
| [**AMovieDllRegisterServer2**](amoviedllregisterserver2.md)     | 註冊和取消註冊篩選。                 |
| [**AMovieDllUnregisterServer**](amoviedllunregisterserver.md)   | 已過時。                                          |
| [**AMovieSetupRegisterFilter**](amoviesetupregisterfilter.md)   | 已過時。                                          |
| [**AMovieSetupRegisterFilter2**](amoviesetupregisterfilter2.md) | 註冊篩選器的業績、釘選和媒體類型。 |
