# <a name="analog-television-audio"></a>類比電視音訊

透過類比電視，音訊來自電視調諧器。 不同的調諧器卡必須以不同的方式設定。 有些卡片會使用外部連接器，將音訊修補至音效卡的線路。 其他人使用內部連接器，將音訊修補至音效卡的 CD 輸入。

下列主題描述如何在 DirectShow 中預覽和捕獲電視音訊：

-   [預覽電視音訊](previewing-tv-audio.md)
-   [捕獲電視音訊](capturing-tv-audio.md)
