# <a name="mpeg-2-kernel-media-types"></a>MPEG-2 核心媒體類型

針對數個 MPEG-2 媒體類型 Guid，Windows DDK 會定義與 DirectShow 中使用的名稱不同的名稱。 下表顯示 (在 Ksuuids 中定義的 DirectShow 名稱) ，以及在 Ksmedia 中定義 (的對應核心模式名稱) 。



| Ksuuids 中的名稱              | Ksmedia 中的名稱                          |
|--------------------------------|--------------------------------------------|
| 格式 \_ WaveFormatEx           | KSDATAFORMAT \_ 規範 \_ WAVEFORMATEX      |
| 格式 \_ MPEG2Video             | KSDATAFORMAT \_ 規範 \_ MPEG2 \_ 影片      |
| MEDIASUBTYPE \_ ATSC \_ SI         | KSDATAFORMAT \_ 子類型 \_ ATSC \_ SI            |
| MEDIASUBTYPE \_ 杜 \_ AC3       | KSDATAFORMAT \_ 子類型 \_ AC3 \_ 音訊          |
| MEDIASUBTYPE \_ DVB-T \_ SI          | KSDATAFORMAT \_ 子類型 \_ DVB-T \_ SI             |
| MEDIASUBTYPE \_ DVD \_ LPCM \_ 音訊 | KSDATAFORMAT \_ 子類型 \_ LPCM \_ 音訊         |
| MEDIASUBTYPE \_ MPEG2 \_ 音訊     | KSDATAFORMAT \_ 子類型 \_ MPEG2 \_ 音訊        |
| MEDIASUBTYPE \_ MPEG2 \_ 計畫   | 靜態 \_ KSDATAFORMAT \_ 類型 \_ MPEG2 \_ 程式 |
| MEDIASUBTYPE \_ MPEG2 \_ 傳輸 | KSDATAFORMAT \_ 類型 \_ MPEG2 \_ 傳輸       |
| MEDIASUBTYPE \_ MPEG2 \_ 影片     | KSDATAFORMAT \_ 子類型的 \_ MPEG2 \_ 影片        |
| 媒體的 \_ MPEG2 \_ 區段     | KSDATAFORMAT \_ 類型 \_ MPEG2 \_ 區段        |
| 媒體 \_ 媒體               | KSDATAFORMAT \_ 類型 \_ 音訊                  |
| 媒體媒體的 \_ MPEG2 \_ pe          | KSDATAFORMAT \_ 類型 \_ MPEG2 \_ pe             |
| 媒體 \_ 媒體               | KSDATAFORMAT \_ 型別 \_ 影片                  |
