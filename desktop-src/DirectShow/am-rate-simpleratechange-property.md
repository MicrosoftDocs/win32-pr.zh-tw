# <a name="am_rate_simpleratechange-property"></a>AM \_ RATE \_ SimpleRateChange 屬性

這個屬性是用來將速率變更傳送至解碼器。 這個屬性的資料型別是 **AM \_ SimpleRateChange** 結構，它會提供新的播放速率，以及新速率生效的呈現時間。



| 標籤 | 值 |
|-------------------|-----------------------------------------------------|
| 屬性集 GUID | AM \_ KSPROPSETID \_ TSRateChange                       |
| 屬性識別碼       | AM \_ RATE \_ SimpleRateChange                          |
| 資料類型         | [**AM \_ SimpleRateChange**](/previous-versions/windows/desktop/api/Dvdmedia/ns-dvdmedia-am_simpleratechange) |
