# <a name="windowless-sample"></a>無視窗範例

## <a name="description"></a>描述

此範例示範影片混合轉譯器 9 (VMR-9) 篩選器中的無視窗模式。

您可以在影像上按一下滑鼠右鍵，或從 [**影像**] 功能表中選取 [**抓取目前的畫面** 格]，從影片視窗抓取影像。

## <a name="downloading-the-sample"></a>下載範例

若要下載 DirectShow SDK 範例，請安裝最新版本的[Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx)。
