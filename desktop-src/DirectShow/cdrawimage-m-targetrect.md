# <a name="cdrawimagem_targetrect-member"></a>CDrawImage：： m \_ TargetRect 成員

`m_TargetRect`成員變數會指定繪圖的目標矩形。 初始值是空的矩形，表示應該將影像繪製至整個目的地視窗。

## <a name="syntax"></a>Syntax


```C++
RECT m_TargetRect;
```
