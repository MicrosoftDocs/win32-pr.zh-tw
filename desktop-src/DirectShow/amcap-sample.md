# <a name="amcap-sample"></a>AmCap 範例

## <a name="description"></a>Description

影片捕獲應用程式。

這個範例應用程式示範與音訊和影片捕捉相關的下列工作：

-   捕捉至檔案
-   即時預覽
-   設定檔案
-   顯示裝置屬性頁
-   裝置列舉
-   Stream 控制項

AMCap 支援 MPEG-2 程式資料流程輸入，例如來自串流 MPEG-2 內容的類比 TV 調諧器。 需要 DirectShow 相容的 MPEG-2 解碼器才能解碼資料流程。

## <a name="downloading-the-sample"></a>下載範例

若要下載 DirectShow SDK 範例，請安裝最新版本的 [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx)。

此範例安裝在下列路徑下： \[ *SDK 根* \] \\ 範例 \\ 多媒體 \\ DirectShow \\ 捕獲 \\ AmCap。
