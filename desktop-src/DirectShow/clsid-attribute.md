# <a name="clsid-attribute"></a>clsid 屬性

> [!Note]  
> \[廢棄。 此 API 可能會從 Windows 的未來版本中移除。\]

 

`clsid`屬性會為物件指定 (CLSID) 的類別識別碼。

## <a name="possible-values"></a>可能的值

必須是格式為的字串 {00000000-0000-0000-0000-000000000000} 。

## <a name="applies-to"></a>套用至

[**at**](at-element.md)、 [**effect**](effect-element.md)、 [**轉換**](transition-element.md)
