# <a name="cbasicaudio-class"></a>CBasicAudio 類別

![cbasicaudio 類別階層](images/cutil07.png)

類別會實 `CBasicAudio` [**IBasicAudio**](/windows/desktop/api/Control/nn-control-ibasicaudio)介面的 **IDispatch** 方法。 其餘的 **IBasicAudio** 方法會保留為純虛擬方法，而且必須由衍生類別來執行。

此類別中的 **IDispatch** 方法是標準的實作為，在這裡不會詳細描述。
