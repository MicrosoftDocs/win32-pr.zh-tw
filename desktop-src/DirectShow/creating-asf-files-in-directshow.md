# <a name="creating-asf-files-in-directshow-directshow"></a>在 DirectShow (DirectShow) 中建立 ASF 檔案

本節說明如何使用 DirectShow [WM ASF 寫入](wm-asf-writer-filter.md) 器篩選器，以 Advanced Systems FORMAT (ASF) 來建立檔案。 ASF 是一種容器格式，可包含任何類型的壓縮或未壓縮資料。 Windows Media 格式檔案是一種 ASF 檔案，使用 Windows Media Format 軟體發展工具組 (SDK) 中指定的特定編解碼器。 這些檔案會針對 Windows Media 音訊檔案使用副檔名 ".wma"，Windows Media 視訊檔案使用 ".wmv"。

本主題包含下列幾節：

-   [DirectShow SDK 和 Windows Media Format SDK](the-directshow-sdk-and-the-windows-media-format-sdk.md)
-   [設定 ASF 寫入器](configuring-the-asf-writer.md)
-   [建立篩選圖形來寫入 ASF 檔案](building-filter-graphs-to-write-asf-files.md)
-   [設定設定檔和其他 ASF 檔案屬性](configuring-profiles-and-other-asf-file-properties.md)
-   [解除鎖定 Windows Media Format SDK](unlocking-the-windows-media-format-sdk.md)
