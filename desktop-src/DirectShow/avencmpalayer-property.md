# <a name="avencmpalayer-property"></a>AVEncMPALayer 屬性

指定 MPEG 音訊層。 這個屬性會套用至 MPEG 音訊編碼器。

這是可讀寫的屬性。

## <a name="data-type"></a>資料類型

**UINT32** (**VT \_ UI4**) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVEncMPALayer**

## <a name="property-value"></a>屬性值

這個屬性的值是 [**eAVEncMPALayer**](/windows/win32/api/codecapi/ne-codecapi-eavencmpalayer) 列舉的成員。
