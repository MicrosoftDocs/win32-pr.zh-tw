# <a name="coareftimeoperator-reference_time-method"></a>COARefTime。 operator 參考 \_ 時間方法

這個運算子會將參考時間抓取為 **參考 \_ 時間** 值。

## <a name="syntax"></a>語法


```C++
REFERENCE_TIME operator REFERENCE_TIME();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回 100-毫微秒單位的參考時間。
