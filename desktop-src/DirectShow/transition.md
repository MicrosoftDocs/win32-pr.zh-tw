# <a name="transition"></a>轉換

> [!Note]  
> \[廢棄。 此 API 可能會從 Windows 的未來版本中移除。\]

 

轉換物件會建立曲目或組合之間的影片轉換。 若要建立這個物件，請呼叫 [**IAMTimeline：： CreateEmptyNode**](iamtimeline-createemptynode.md) 方法。

轉換物件會公開下列介面：

-   [**IAMTimelineObj**](iamtimelineobj.md)
-   [**IAMTimelineSplittable**](iamtimelinesplittable.md)
-   [**IAMTimelineTrans**](iamtimelinetrans.md)
