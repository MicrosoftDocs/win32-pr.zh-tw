# <a name="cbasewindowunsetpalette-method"></a>CBaseWindow. UnsetPalette 方法

`UnsetPalette`方法會刪除視窗的目前調色板，並還原預設的系統調色板。

## <a name="syntax"></a>語法


```C++
void UnsetPalette();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
