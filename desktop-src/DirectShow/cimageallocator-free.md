# <a name="cimageallocatorfree-method"></a>CImageAllocator 方法

`Free`方法會釋放所有緩衝區記憶體。 這個方法會實作為純虛擬 [**CBaseAllocator：： Free**](cbaseallocator-free.md) 方法。

## <a name="syntax"></a>語法


```C++
void Free();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
