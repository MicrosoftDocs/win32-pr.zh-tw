# <a name="media-locator"></a>媒體定位器

> [!Note]  
> \[廢棄。 此 API 可能會從 Windows 的未來版本中移除。\]

 

媒體定位器物件會尋找與來源物件相關聯的檔案。 使用這個物件來驗證時間軸中的來源。 若要建立這個物件，請呼叫 **CoCreateInstance**。 類別識別碼是 CLSID \_ MediaLocator。

媒體定位器物件會公開下列介面：

-   [**IMediaLocator**](imedialocator.md)
