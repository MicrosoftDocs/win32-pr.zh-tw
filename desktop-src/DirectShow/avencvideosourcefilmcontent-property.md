# <a name="avencvideosourcefilmcontent-property"></a>AVEncVideoSourceFilmContent 屬性

指定輸入影片的原始來源是否為電影或影片。

這是可讀寫的屬性。

## <a name="data-type"></a>資料類型

**UINT32** (**VT \_ UI4**) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVEncVideoSourceFilmContent**

## <a name="property-value"></a>屬性值

這個屬性的值是 [**eAVEncVideoFilmContent**](/windows/win32/api/codecapi/ne-codecapi-eavencvideofilmcontent) 列舉的成員。
