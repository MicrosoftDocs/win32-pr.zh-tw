# <a name="avencvideooutputcolorprimaries-property"></a>AVEncVideoOutputColorPrimaries 屬性

指定編碼影片的色彩主要複本。

這是可讀寫的屬性。

## <a name="data-type"></a>資料類型

**UINT32** (**VT \_ UI4**) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVEncVideoOutputColorPrimaries**

## <a name="property-value"></a>屬性值

這個屬性的值是 [**eAVEncVideoColorPrimaries**](/windows/desktop/api/codecapi/ne-codecapi-eavencvideocolorprimaries) 列舉的成員。
