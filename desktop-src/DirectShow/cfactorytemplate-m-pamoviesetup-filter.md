# <a name="cfactorytemplatem_pamoviesetup_filter-member"></a>CFactoryTemplate：： m \_ pAMovieSetup \_ 篩選成員

[**AMOVIESETUP \_ 篩選**](amoviesetup-filter.md)結構的指標。

## <a name="syntax"></a>語法


```C++
const AMOVIESETUP_FILTER *m_pAMovieSetup_Filter;
```



## <a name="remarks"></a>備註

您可以使用此結構來讓篩選自我註冊。 如果您的元件不是篩選準則，此成員變數可以是 **Null**。 如需詳細資訊，請參閱如何註冊 DirectShow 篩選。
