# <a name="property-setter"></a>屬性 Setter

> [!Note]  
> \[廢棄。 此 API 可能會從 Windows 的未來版本中移除。\]

 

屬性 Setter 物件會在時間軸物件上設定 **IDispatch** 屬性。 若要建立這個物件，請呼叫 **CoCreateInstance**。 類別識別碼是 CLSID \_ PropertySetter。

屬性 Setter 物件會公開下列介面：

-   [**IAMSetErrorLog**](iamseterrorlog.md)
-   [**IPropertySetter**](ipropertysetter.md)
