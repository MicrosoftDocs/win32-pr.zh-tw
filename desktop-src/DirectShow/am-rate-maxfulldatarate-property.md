# <a name="am_rate_maxfulldatarate-property"></a>AM \_ RATE \_ MaxFullDataRate 屬性

傳回解碼器可以處理資料的最大速率。 這個屬性的值是解碼器的最大播放速度 x 10000。 例如，如果最大的向前速度是2.0，則這個屬性的值是20000。

這個屬性的資料類型是 **\_ MaxFullDataRate**，其為 `typedef` **LONG**。

這是唯讀的屬性。



|                   |                                    |
|-------------------|------------------------------------|
| 屬性集 GUID | AM \_ KSPROPSETID \_ TSRateChange      |
| 屬性識別碼       | AM \_ RATE \_ MaxFullDataRate          |
| 資料類型         | **AM \_MaxFullDataRate** (**LONG**)  |
