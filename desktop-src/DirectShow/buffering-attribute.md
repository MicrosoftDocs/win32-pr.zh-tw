# <a name="buffering-attribute"></a>緩衝屬性

> [!Note]  
> \[廢棄。 此 API 可能會從 Windows 的未來版本中移除。\]

 

`buffering`屬性指定預覽期間要預先轉譯的畫面格數目。

## <a name="possible-values"></a>可能的值

值必須是大於1的數位。 預設值是 30。
