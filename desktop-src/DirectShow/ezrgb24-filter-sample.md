# <a name="ezrgb24-filter-sample"></a>EZRGB24 篩選範例

## <a name="description"></a>Description

EZRGB24 篩選器是一種影像處理轉換篩選。 此篩選會執行一些個別的影片效果，包括紅色、綠色、藍色、暗灰色、XOR、模糊、灰色和浮凸效果。

這個範例會示範如何使用數個 DirectShow 基類，包括 [**CTransformFilter**](ctransformfilter.md)、 [**CPersistStream**](cpersiststream.md)和 [**CBasePropertyPage**](cbasepropertypage.md)。

## <a name="downloading-the-sample"></a>下載範例

若要下載 DirectShow SDK 範例，請安裝最新版本的 [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx)。

此範例安裝在下列路徑下： *\[ SDK \] 根* \\ 範例 \\ 多媒體 \\ DirectShow \\ 篩選 \\ EZRGB24。
