# <a name="key-transition"></a>金鑰轉換

> [!Note]  
> \[廢棄。 此 API 可能會從 Windows 的未來版本中移除。\]

 

金鑰轉換會根據 RGB 值、Alpha 值、色調或亮度來執行加密。

下圖顯示金鑰轉換：

![金鑰轉換](images/trans-key.png)

類別識別碼 (CLSID) ： {C5B19592-145E-11D3-9F04-006008039E37}

CLSID 變數名稱： CLSID \_ DxtKey

易記名稱： "DxtKey"

屬性



| 屬性   | 類型  | 有效範圍           | Description                                                                                                                                                                                                                                                | 套用至                     |
|------------|-------|-----------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------|
| 色調        | int   | 0-360                 | 要做為索引鍵的色調值。                                                                                                                                                                                                                             | 色調                            |
| Invert     | BOOL  | **FALSE** 或 **TRUE** | 布林值，指出是否要反轉索引鍵的預設運算。 如果 **為 FALSE**，則會以預設方式將上層影像中的圖元變成透明。 若 **為 TRUE**，則作業反轉。                                                   | 色度、色調、亮度、Nonred |
| KeyType    | int   | 請參閱備註           | 指定索引鍵的類型。 如需詳細資訊，請參閱＜備註＞。                                                                                                                                                                                              | 全部                            |
| Luminance  | int   | 0–100                 | 要做為索引鍵的亮度值。                                                                                                                                                                                                                       | Luminance                      |
| RGB        | DWORD | 0x0 –0xFFFFFF        | 按鍵的色彩。 此值為十六進位數位，格式為 0x *RRGGBB*，其中 *RR* 為紅色值， *GG* 為綠色值，而 *BB* 為藍色值。  (純紅色、綠色和藍色分別為0xFF0000、0x00FF00 和0x0000FF。 )  | 色度                         |
| 相似度 | int   | 0–100                 | 變成透明的色彩資料範圍。 在較高的值中，較大範圍的相似色彩是透明的。                                                                                                                                        | 色度、Nonred                 |



 

## <a name="remarks"></a>備註

執行的索引鍵類型取決於 **KeyType** 屬性的值，必須是下列其中一項：



| 值 | 列舉型別       | 描述                                           |
|-------|-------------------|-------------------------------------------------------|
| 0     | DXTKEY \_ RGB       | 以 RGB 值 (按鍵) 的色度按鍵。                        |
| 1     | DXTKEY \_ NONRED    | Nonred 鍵。  (使藍和綠區域透明。 )  |
| 2     | DXTKEY \_ 亮度 | 亮度索引鍵。                                        |
| 3     | DXTKEY \_ ALPHA     | 依字母值的索引鍵。                                   |
| 4     | DXTKEY \_ 色調       | 按鍵（依色調）。                                           |



 

索引鍵類型預設為 DXTKEY \_ ALPHA。
