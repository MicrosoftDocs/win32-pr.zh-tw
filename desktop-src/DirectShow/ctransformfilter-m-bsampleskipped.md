# <a name="ctransformfilterm_bsampleskipped-member"></a>CTransformFilter：： m \_ bSampleSkipped 成員

指出最近樣本是否已卸載的旗標。 如果 [**接收**](ctransformfilter-receive.md) 方法卸載範例，則會將值設定為 **TRUE**。

## <a name="syntax"></a>Syntax


```C++
BOOL m_bSampleSkipped;
```
