# <a name="tv-audio-filter"></a>電視音訊篩選器

電視音訊篩選器可控制電視音訊解碼、身歷聲或 monoaural 選取專案，以及 (SAP) 選取的次要音訊程式。



|                                          |                                                                                                                                |
|------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| 篩選介面                        | [**IAMTVAudio**](/windows/desktop/api/Strmif/nn-strmif-iamtvaudio)、 **IPersistPropertyBag**、 **ISpecifyPropertyPages**、 [**IKsPropertySet**](ikspropertyset.md) |
| 輸入 Pin 媒體類型                    | 媒體 \_ AnalogAudio                                                                                                         |
| 輸入 Pin 介面                     | **IKsPropertySet**                                                                                                             |
| 輸出 Pin 媒體類型                   | 媒體 \_ AnalogAudio                                                                                                         |
| 輸出 Pin 介面                    | **IKsPropertySet**                                                                                                             |
| 篩選 CLSID                             | CLSID \_ TVAudioFilter                                                                                                           |
| 屬性頁 CLSID                      | AM \_ KSCATEGORY \_ TVAUDIO                                                                                                        |
| [優點](merit.md)                       | \_ \_ 未 \_ 使用的業績                                                                                                            |
| [篩選準則分類](filter-categories.md) | CLSID \_ AudioInputDeviceCategory                                                                                                |
