# <a name="avddsurroundmode-property"></a>AVDDSurroundMode 屬性

指定音訊是否以杜住範圍編碼。

這是唯讀的屬性。

## <a name="data-type"></a>資料類型

**UINT32** (**VT \_ UI4**) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVDDSurroundMode**

## <a name="property-value"></a>屬性值

這個屬性的值是 [**eAVDDSurroundMode**](/windows/desktop/api/codecapi/ne-codecapi-eavddsurroundmode) 列舉的成員。
