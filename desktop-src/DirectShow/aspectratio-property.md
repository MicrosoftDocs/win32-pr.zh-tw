# <a name="aspectratio-property"></a>AspectRatio 屬性

> [!Note]  
> 此元件可用於 Microsoft Windows 2000、Windows XP 及 Windows Server 2003 作業系統。 它在後續版本中可能會變更或無法使用。

 

屬性會抓取 `AspectRatio` 目前影片串流的外觀比例，如光碟上所撰寫。

``` syntax
[  nAspectRatio = ] MSWebDVD.AspectRatio 
```

## <a name="return-value"></a>傳回值

傳回數位，代表在光碟上撰寫的目前影片的外觀比例 (1.33 的4:3 外觀比例，或 1.78 16:9 外觀比例) 。

## <a name="remarks"></a>備註

這個屬性是唯讀的，沒有預設值。
