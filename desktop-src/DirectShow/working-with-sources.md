# <a name="working-with-sources"></a>使用來源

\[此 API 不受支援，而且可能會在未來變更或無法使用。\]

來源物件代表來源檔案中的單一媒體資料流程。 來源提供專案的原始材料。 時間軸中的其他專案都有提供結構 (的追蹤、組合和群組) 或修改來源 (效果和轉換) 。

本文說明與 Microsoft® DirectShow®編輯服務 (DES) 中的來源相關的各種詳細資料。 如需將來源新增至時間軸所需之基本步驟的詳細資訊，請參閱 [建立時間軸](constructing-a-timeline.md)。

本文包含下列各節。

-   [設定來源的屬性](setting-properties-on-sources.md)
-   [使用媒體偵測器](using-the-media-detector.md)
-   [使用媒體定位器](using-the-media-locator.md)
-   [其他來源物件](other-source-objects.md)
