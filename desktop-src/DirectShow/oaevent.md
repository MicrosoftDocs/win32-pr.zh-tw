# <a name="oaevent"></a>OAEVENT

**OAEVENT** 資料類型會定義 OLE Automation 用戶端可以使用的事件控制碼，例如 Microsoft Visual Basic 6.0。


```C++
typedef LONG_PTR OAEVENT;
```
