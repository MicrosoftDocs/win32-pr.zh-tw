# <a name="directshow-editing-services"></a>DirectShow編輯服務

\[此 API 不受支援，而且可能會在未來變更或無法使用。\]

DirectShow編輯服務 (DES) 是 (API) 的應用程式開發介面，可大幅簡化影片編輯作業所需的工作。 DES 以核心 DirectShow 架構為基礎。 它會抽象化 DirectShow 的許多複雜性，並提供一組專門針對操作影片編輯專案所設計的介面。 做為應用程式開發人員，您可以獲得在架構中 DirectShow 的優點，更適合用來建立影片編輯應用程式。
