# <a name="dmo-interfaces"></a>DMO介面

本節包含所有 Microsoft DirectX 媒體物件 (DMO) COM 介面及其方法的參考專案。



| 介面                                                            | 描述                                                              |
|----------------------------------------------------------------------|--------------------------------------------------------------------------|
| [**IDMOQualityControl**](/previous-versions/windows/desktop/api/Mediaobj/nn-mediaobj-idmoqualitycontrol)                     | 支援 DMO 的品質控制。                                       |
| [**IDMOVideoOutputOptimizations**](/previous-versions/windows/desktop/api/Mediaobj/nn-mediaobj-idmovideooutputoptimizations) | 支援 DMO 的影片優化。                                   |
| [**IEnumDMO**](/previous-versions/windows/desktop/api/Mediaobj/nn-mediaobj-ienumdmo)                                         | 提供列舉 DMOs 的方法。                                   |
| [**IMediaBuffer**](/previous-versions/windows/desktop/api/Mediaobj/nn-mediaobj-imediabuffer)                                 | 提供處理資料緩衝區的方法。                         |
| [**IMediaObject**](/previous-versions/windows/desktop/api/Mediaobj/nn-mediaobj-imediaobject)                                 | 提供操作 DMO 的方法。                                 |
| [**IMediaObjectInPlace**](/previous-versions/windows/desktop/api/mediaobj/nn-mediaobj-imediaobjectinplace)                   | 提供就地處理資料的方法。                           |
| [**IMediaParamInfo**](/previous-versions/windows/desktop/api/Medparam/nn-medparam-imediaparaminfo)                           | 抓取物件所支援媒體參數的相關資訊 |
| [**IMediaParams**](/previous-versions/windows/desktop/api/Medparam/nn-medparam-imediaparams)                                 | 設定和抓取物件上的媒體參數。                        |
