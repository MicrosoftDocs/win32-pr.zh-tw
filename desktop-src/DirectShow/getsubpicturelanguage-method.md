# <a name="getsubpicturelanguage-method"></a>GetSubpictureLanguage 方法

> [!Note]  
> 此元件可用於 Microsoft Windows 2000、Windows XP 及 Windows Server 2003 作業系統。 它在後續版本中可能會變更或無法使用。

 

方法會抓取 `GetSubpictureLanguage` 指定之子圖片資料流程的語言。

``` syntax
[ sLang = ] MSWebDVD.GetSubpictureLanguage(iStream)
```

## <a name="parameters"></a>參數

<dl> <dt>

<span id="iStream"></span><span id="istream"></span><span id="ISTREAM"></span>*iStream*
</dt> <dd>

指定您想要取出其文字語言作為整數的子圖片資料流程號碼。

</dd> </dl>

## <a name="return-value"></a>傳回值

傳回人們可讀取的字串，識別目前標題中音訊資料流程的語言。
