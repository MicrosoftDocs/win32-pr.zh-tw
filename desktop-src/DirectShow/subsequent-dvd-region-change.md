# <a name="subsequent-dvd-region-change"></a>後續的 DVD 區域變更

在 Windows 2000 和更新版本中，DVD 導覽器會在裝置管理員的 DVD-ROM 光碟機裝置上叫用屬性頁。 當需要變更區域時，DVDPlay.exe 的應用程式也會啟動這個屬性頁面。 這個屬性頁的 UI 與 DVDRgn.exe 的 UI 非常類似，而且地區設定之在作業系統中。

針對 RPC1 磁片磁碟機，Windows 作業系統元件會管理區域變更。 RPC2 磁片磁碟機會在自己的硬體中維護區域設定。 在 RPC2 磁片磁碟機上耗盡允許的變更數目時，磁片磁碟機將無法執行變更區域的呼叫，而作業系統中的區域變更元件將會指出對使用者的變更。
