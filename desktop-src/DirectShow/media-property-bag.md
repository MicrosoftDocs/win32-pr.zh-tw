# <a name="media-property-bag"></a>媒體屬性包

媒體屬性包物件會設定或抓取 AVI 檔中的資訊和中的區塊。 藉由呼叫 **CoCreateInstance** 來建立此物件。



| 標籤 | 值 |
|------------------|------------------------------------------------|
| 類別識別碼 | CLSID \_ MediaPropertyBag                        |
| 介面       | [**IMediaPropertyBag**](/windows/desktop/api/Strmif/nn-strmif-imediapropertybag) |
