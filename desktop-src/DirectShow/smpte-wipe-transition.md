# <a name="smpte-wipe-transition"></a>SMPTE 清除轉換

> [!Note]  
> \[廢棄。 此 API 可能會從 Windows 的未來版本中移除。\]

 

「SMPTE 抹除」轉換會產生任何由運動圖片和電視工程師所定義的標準抹除， (smpte 檔 258M-1993 中的 SMPTE) ，但有四個分割的例外。

![清除轉換進行中](images/trans-wipe.png)

類別識別碼 (CLSID) ： {DE75D012-7A65-11D2-8CEA-00A0C9441E20}

CLSID 變數名稱： CLSID \_ DxtJpeg

易記名稱： "DxtJpeg"

屬性



| 屬性       | 類型   | 預設  | 描述                                                                                                                                                                                                                                                                                                      |
|----------------|--------|----------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| BorderColor    | long   | 0x000000 | 抹除模式邊緣周圍的框線色彩。 這個屬性的值是十六進位數位，格式為 0x *RRGGBB*，其中 *RR* 是紅色值， *GG* 是綠色值，而 *BB* 為藍色值。  (因此，純紅色、綠色和藍色分別為0xFF000、0x00FF00 和0x0000FF。 )  |
| BorderSoftness | long   | 0        | 清除模式邊緣周圍模糊區域的寬度。 針對沒有模糊區域的指定零。                                                                                                                                                                                                              |
| BorderWidth    | long   | 0        | 沿著抹除模式邊緣的實心框線寬度。 針對沒有框線指定零。                                                                                                                                                                                                                       |
| MaskName       | BSTR   | **NULL** | 如果不是 **Null**，則指定要作為抹除遮罩的 JPEG 檔案名，而不是標準的內建抹除。 檔案必須包含單色、每圖元8位的漸層。 漸層會用來做為遮罩來定義抹除的進度。                                                                 |
| MaskNum        | long   | 1        | 標準的 SMPTE 抹除程式碼，指定要使用的抹除樣式。 如需抹除碼及其相關聯的圖解清單，請參閱 258M-1993。                                                                                                                                                            |
| OffsetX        | long   | 0        | 從影像中央抹除來源的水準位移。 僅適用于從101到131的 **MaskNum** 值。                                                                                                                                                                                         |
| OffsetY        | long   | 0        | 從影像中央 Verical 抹除來源的位移。 僅適用于從101到131的 **MaskNum** 值。                                                                                                                                                                                            |
| ReplicateX     | long   | 0        | 水準複寫抹除模式的次數。 僅適用于從101到131的 **MaskNum** 值。                                                                                                                                                                                                |
| ReplicateY     | long   | 0        | 垂直複製抹除模式的次數。 僅適用于從101到131的 **MaskNum** 值。                                                                                                                                                                                                  |
| ScaleX         | double | 1.0      | 以水準方式延展抹除的量（以原始抹除定義的百分比表示）。 僅適用于從101到131的 **MaskNum** 值。                                                                                                                                                         |
| ScaleY         | double | 1.0      | 用來垂直延展抹除的數量，以原始抹除定義的百分比表示。 僅適用于從101到131的 **MaskNum** 值。                                                                                                                                                           |
