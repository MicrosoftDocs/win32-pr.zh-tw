# <a name="avencaudiomapdestchanneln-property"></a>AVEncAudioMapDestChannelN 屬性

指定將哪個音訊通道對應到編碼音訊串流中的 channel *N* 。

這是可讀寫的屬性。

## <a name="data-type"></a>資料類型

**UINT32** (**VT \_ UI4**) 

## <a name="property-guid"></a>屬性 GUID

**請參閱＜備註＞。**

## <a name="remarks"></a>備註

本參考主題描述一組相關的屬性。 每個屬性都會將輸入通道對應至指定的輸出通道。 會定義下列屬性。



| 屬性 GUID                            | Description             |
|------------------------------------------|-------------------------|
| **CODECAPI \_ AVEncAudioMapDestChannel0**  | 目的地通道0。  |
| **CODECAPI \_ AVEncAudioMapDestChannel1**  | 目的地頻道1。  |
| **CODECAPI \_ AVEncAudioMapDestChannel2**  | 目的地頻道2。  |
| **CODECAPI \_ AVEncAudioMapDestChannel3**  | 目的地頻道3。  |
| **CODECAPI \_ AVEncAudioMapDestChannel4**  | 目的地通道4。  |
| **CODECAPI \_ AVEncAudioMapDestChannel5**  | 目的地頻道5。  |
| **CODECAPI \_ AVEncAudioMapDestChannel6**  | 目的地通道6。  |
| **CODECAPI \_ AVEncAudioMapDestChannel7**  | 目的地 channel 7。  |
| **CODECAPI \_ AVEncAudioMapDestChannel8**  | 目的地頻道8。  |
| **CODECAPI \_ AVEncAudioMapDestChannel9**  | 目的地 channel 9。  |
| **CODECAPI \_ AVEncAudioMapDestChannel10** | 目的地通道10。 |
| **CODECAPI \_ AVEncAudioMapDestChannel11** | 目的地 channel 11。 |
| **CODECAPI \_ AVEncAudioMapDestChannel12** | 目的地 channel 12。 |
| **CODECAPI \_ AVEncAudioMapDestChannel13** | 目的地頻道13。 |
| **CODECAPI \_ AVEncAudioMapDestChannel14** | 目的地 channel 14。 |
| **CODECAPI \_ AVEncAudioMapDestChannel15** | 目的地頻道15。 |
