# <a name="directshow"></a>DirectShow

Microsoft DirectShow 應用程式開發介面 (API) 是適用于 Microsoft Windows 的媒體串流架構。 您的應用程式可以使用 DirectShow 來執行高品質的影片和音訊播放或捕捉。

您可以在 [Windows SDK](https://msdn.microsoft.com/windows/aa904949.aspx)中取得 DirectShow 標頭、程式庫、SDK 工具和範例。

> [!Note]  
> DirectX sdk 隨附舊版的 DirectShow SDK。 包含 DirectShow 的 DirectX SDK 的最新版本是 DirectX 9.0 SDK Update- (二月 2005) 額外專案。 在此版本之後，會將 DirectShow 移至 Windows SDK。 若要取得最新版本的 DirectShow 標頭、程式庫和範例，請下載 Windows SDK。
