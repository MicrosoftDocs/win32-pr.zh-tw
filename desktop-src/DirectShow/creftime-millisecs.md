# <a name="creftimemillisecs-method"></a>CRefTime. Millisecs 方法

`Millisecs`方法會將參考時間轉換成毫秒。

## <a name="syntax"></a>語法


```C++
LONG Millisecs();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回參考時間（以毫秒為單位）。
