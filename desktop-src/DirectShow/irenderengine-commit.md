# <a name="irenderenginecommit-method"></a>IRenderEngine：： Commit 方法

> [!Note]  
> \[廢棄。 此 API 可能會從 Windows 的未來版本中移除。\]

## <a name="syntax"></a>語法


```C++
HRESULT Commit();
```



## <a name="parameters"></a>參數
