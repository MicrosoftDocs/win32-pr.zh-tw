# <a name="avencstataudioaveragepcmvalue-property"></a>AVEncStatAudioAveragePCMValue 屬性

傳回音頻內容的平均音量層級。

這是可讀寫的屬性。

## <a name="data-type"></a>資料類型

**UINT32** (**VT \_ UI4**) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVEncStatAudioAveragePCMValue**

## <a name="remarks"></a>備註

編碼完成之後，就可以使用這個屬性。
