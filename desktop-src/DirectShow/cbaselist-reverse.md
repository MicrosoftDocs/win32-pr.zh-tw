# <a name="cbaselistreverse-method"></a>CBaseList. Reverse 方法

`Reverse`方法會反轉清單的順序。

## <a name="syntax"></a>語法


```C++
void Reverse();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
