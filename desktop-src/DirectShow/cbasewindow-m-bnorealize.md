# <a name="cbasewindowm_bnorealize-member"></a>CBaseWindow：： m \_ bNoRealize 成員

旗標，指定視窗是否應該實現其調色板。 若 **為 TRUE**，則表示視窗無法實現其調色板。 預設值為 **FALSE**。

## <a name="syntax"></a>Syntax


```C++
BYTE m_bNoRealize;
```
