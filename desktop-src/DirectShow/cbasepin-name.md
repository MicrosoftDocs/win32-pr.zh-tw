# <a name="cbasepinname-method"></a>CBasePin.Name 方法

方法會抓取 `Name` pin 識別碼。

## <a name="syntax"></a>語法


```C++
LPWSTR Name();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回 [**CBasePin：： m \_ pName**](cbasepin-m-pname.md) 成員變數的值。
