# <a name="cdrawimagem_bstretch-member"></a>CDrawImage：： m \_ bStretch 成員

`m_bStretch`成員變數會指出是否必須延展影片影像以符合目的地視窗。 如果來源矩形與目的地矩形的大小相同，則此值為 **FALSE** ，否則為 **TRUE** 。

## <a name="syntax"></a>Syntax


```C++
BOOL m_bStretch;
```
