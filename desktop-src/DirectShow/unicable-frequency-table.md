# <a name="unicable-frequency-table"></a>Unicable Frequency 資料表

由於其長度的緣故，此處不包含 Unicable 資料表。 Unicable 資料表背後的目標是要包含一組完整的頻率 (基本上結合了所有其他資料表，再加上一些真實世界的觀察) 在一個資料表中，以協助進行自動化的通道掃描，而不需要對整個頻率頻譜進行線性搜尋。 針對不包含在此表格中的頻率，取用者必須透過自訂使用者介面進入頻率，並導致登錄中的頻率覆寫專案。
