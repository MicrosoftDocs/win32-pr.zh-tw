# <a name="cbaserenderergetpincount-method"></a>CBaseRenderer. GetPinCount 方法

方法會抓取 `GetPinCount` 釘選數目。

## <a name="syntax"></a>語法


```C++
virtual int GetPinCount();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回 1。

## <a name="remarks"></a>備註

這個方法會實 [**CBaseFilter：： GetPinCount**](cbasefilter-getpincount.md) 方法，這是 **CBaseFilter** 類別中的純虛擬。
