# <a name="avdecddoperationalmode-property"></a>AVDecDDOperationalMode 屬性

指定杜比 AC-3 音訊串流的壓縮控制模式。

這是可讀寫的屬性。

## <a name="data-type"></a>資料類型

**UINT32** (**VT \_ UI4**) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVDecDDOperationalMode**

## <a name="property-value"></a>屬性值

這個屬性的值是 [**eAVDecDDOperationalMode**](/windows/desktop/api/codecapi/ne-codecapi-eavdecddoperationalmode) 列舉的成員。
