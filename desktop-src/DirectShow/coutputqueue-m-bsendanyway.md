# <a name="coutputqueuem_bsendanyway-member"></a>COutputQueue：： m \_ bSendAnyway 成員

覆寫批次處理的旗標。 將此旗標設定為 **TRUE** 會覆寫 [**COutputQueue：： m \_ bBatchExact**](coutputqueue-m-bbatchexact.md) 旗標，並傳遞所有暫止的範例。

## <a name="syntax"></a>Syntax


```C++
BOOL m_bSendAnyway;
```
