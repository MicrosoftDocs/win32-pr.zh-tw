# <a name="height-property-directshow"></a>Height 屬性 (DirectShow) 

> [!Note]  
> 此元件可在 Microsoft Windows 2000、Windows XP 和 Windows Server 2003 作業系統中使用。 它在後續版本中可能會變更或無法使用。

 

`Height`屬性會設定或抓取矩形的高度。

``` syntax
[ iHeight = ] DVDRect.Height
```

## <a name="return-value"></a>傳回值

傳回矩形的高度（以圖元為單位），表示為整數。

## <a name="remarks"></a>備註

此屬性是讀取/寫入，沒有預設值。
