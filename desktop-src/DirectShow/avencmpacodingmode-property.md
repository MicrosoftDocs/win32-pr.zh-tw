# <a name="avencmpacodingmode-property"></a>AVEncMPACodingMode 屬性

指定 MPEG 1/2 音訊編碼模式。 這個屬性會套用至 MPEG 音訊編碼器。

這是可讀寫的屬性。

## <a name="data-type"></a>資料類型

**UINT32** (**VT \_ UI4**) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVEncMPACodingMode**

## <a name="property-value"></a>屬性值

這個屬性的值是 [**eAVEncMPACodingMode**](/windows/desktop/api/codecapi/ne-codecapi-eavencmpacodingmode) 列舉的成員。
