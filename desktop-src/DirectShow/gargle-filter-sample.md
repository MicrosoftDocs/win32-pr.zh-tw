# <a name="gargle-filter-sample"></a>Gargle 篩選範例

## <a name="description"></a>Description

Gargle 篩選器是音訊效果轉換篩選。

Gargle 篩選器會藉由將波形乘以在篩選內以數學方式產生的另一個波形，來 modulates 波形。 根據預設，調整波形為三角形 wave。 屬性工作表也提供平方波的替代專案。 您可以透過篩選的屬性工作表來設定調整 wave 的頻率。 在低度的頻率下，效果聽起來像是 tremolo。 在更高的速度下，它聽起來像是扭曲，在原始 unmodulated 音效的上方和下方新增額外的頻率。

## <a name="downloading-the-sample"></a>下載範例

若要下載 DirectShow SDK 範例，請安裝最新版本的 [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx)。

此範例安裝在下列路徑下： *\[ SDK \] 根* \\ 範例 \\ 多媒體 \\ DirectShow \\ 篩選 \\ Gargle。
