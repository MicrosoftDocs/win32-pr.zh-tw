# <a name="cbasepinm_bcanreconnectwhenactive-member"></a>CBasePin：： m \_ bCanReconnectWhenActive 成員

指出 pin 是否支援動態重新連接的旗標。

## <a name="syntax"></a>語法


```C++
bool m_bCanReconnectWhenActive;
```



## <a name="remarks"></a>備註

此旗標的預設值為 **FALSE**。
