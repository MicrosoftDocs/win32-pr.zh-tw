# <a name="cbaserendererisstreaming-method"></a>CBaseRenderer. IsStreaming 方法

`IsStreaming`方法會查詢篩選是否為數據流處理資料。

## <a name="syntax"></a>語法


```C++
BOOL IsStreaming();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回 [**CBaseRenderer：： m \_ bStreaming**](cbaserenderer-m-bstreaming.md) 旗標。
