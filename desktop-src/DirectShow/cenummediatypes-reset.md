# <a name="cenummediatypesreset-method"></a>CEnumMediaTypes 重設方法

方法會將 `Reset` 列舉順序重設為開頭。 這個方法會實 [**IEnumMediaTypes：： Reset**](/windows/desktop/api/Strmif/nf-strmif-ienummediatypes-reset) 方法。

## <a name="syntax"></a>語法


```C++
HRESULT Reset();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回 S \_ OK。
