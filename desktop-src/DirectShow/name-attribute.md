# <a name="name-attribute"></a>name 屬性

> [!Note]  
> \[廢棄。 此 API 可能會從 Windows 的未來版本中移除。\]

 

`name`屬性會指定物件的名稱。

## <a name="possible-values"></a>可能的值

可以是任何值。

## <a name="applies-to"></a>套用至

[**群組**](group-element.md)、 [ **param**](param-element.md)

## <a name="remarks"></a>備註

針對 **群組** 專案，這個屬性會指定群組的應用程式定義名稱。

針對 **param** 元素，這個屬性會指定屬性的名稱。
