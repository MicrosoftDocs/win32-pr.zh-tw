# <a name="cbasepropertypagem_bdirty-member"></a>CBasePropertyPage：： m \_ bDirty 成員

**M \_ bDirty** 旗標會指出是否有任何屬性變更。 如果使用者透過對話方塊控制項變更任何屬性，請將此旗標設定為 **TRUE** 。

## <a name="syntax"></a>Syntax


```C++
BOOL m_bDirty;
```
