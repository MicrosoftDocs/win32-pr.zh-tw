# <a name="username-attribute"></a>username 屬性

> [!Note]  
> \[廢棄。 此 API 可能會從 Windows 的未來版本中移除。\]

 

`userid`屬性指定物件的應用程式定義名稱。

## <a name="possible-values"></a>可能的值

可以是任何值。
