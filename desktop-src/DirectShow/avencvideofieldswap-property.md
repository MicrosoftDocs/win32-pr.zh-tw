# <a name="avencvideofieldswap-property"></a>AVEncVideoFieldSwap 屬性

反轉來源影片中交錯欄位的順序。

這是可讀寫的屬性。

## <a name="data-type"></a>資料類型

**變異 \_BOOL** (**VT \_ bool**) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVEncVideoFieldSwap**

## <a name="remarks"></a>備註

如果值為 **VARIANT \_ TRUE**，編碼器會交換欄位順序。
