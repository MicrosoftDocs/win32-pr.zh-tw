# <a name="resume-method"></a>Resume 方法

> [!Note]  
> 此元件可用於 Microsoft Windows 2000、Windows XP 及 Windows Server 2003 作業系統。 它在後續版本中可能會變更或無法使用。

 

在 `Resume` 顯示功能表之後，方法會繼續播放。

``` syntax
MSWebDVD.Resume()
```

## <a name="return-value"></a>傳回值

沒有傳回值。
