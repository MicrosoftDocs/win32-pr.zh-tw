# <a name="avencinputvideosystem-property"></a>AVEncInputVideoSystem 屬性

指定來源內容的視頻系統。

這是可讀寫的屬性。

## <a name="data-type"></a>資料類型

**UINT32** (**VT \_ UI4**) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVEncInputVideoSystem**

## <a name="property-value"></a>屬性值

這個屬性的值是 [**eAVEncInputVideoSystem**](/windows/desktop/api/codecapi/ne-codecapi-eavencinputvideosystem) 列舉的成員。
