# <a name="avencaudiointervaltoskip-property"></a>AVEncAudioIntervalToSkip 屬性

指定要略過之編碼器的音訊樣本數。

這是可讀寫的屬性。

## <a name="data-type"></a>資料類型

**UINT64** (**VT \_ UI8**) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVEncAudioIntervalToSkip**
