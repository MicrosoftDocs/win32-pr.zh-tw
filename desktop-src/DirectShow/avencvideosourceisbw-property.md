# <a name="avencvideosourceisbw-property"></a>AVEncVideoSourceIsBW 屬性

指定影片是否為單色 (黑色和白色) 或包含色彩。

這是可讀寫的屬性。

## <a name="data-type"></a>資料類型

**變異 \_BOOL** (**VT \_ bool**) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVEncVideoSourceIsBW**

## <a name="property-value"></a>屬性值

如果值為 **VARIANT \_ TRUE**，則表示影片為單色。 否則，影片就會包含色彩。
