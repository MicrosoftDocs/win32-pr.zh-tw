![cbasecontrolvideo 類別階層](images/wctrl02.png)

<span data-ttu-id="2b935-106">**CBaseControlVideo** 類別會執行 [**IBasicVideo**](/windows/desktop/api/Control/nn-control-ibasicvideo)介面，並控制一般影片視窗的影片屬性。</span><span class="sxs-lookup"><span data-stu-id="2b935-106">The **CBaseControlVideo** class implements the [**IBasicVideo**](/windows/desktop/api/Control/nn-control-ibasicvideo) interface and controls the video properties of a generic video window.</span></span> <span data-ttu-id="2b935-107">一般而言， **CBaseControlVideo** 物件是影片轉譯器，可將影片繪製到顯示器上的視窗中。</span><span class="sxs-lookup"><span data-stu-id="2b935-107">Generally, a **CBaseControlVideo** object is a video renderer that draws video into a window on the display.</span></span>

<span data-ttu-id="2b935-108">許多 **CBaseControlVideo** 成員函式只需要影片轉譯器連接至篩選圖形。</span><span class="sxs-lookup"><span data-stu-id="2b935-108">Many **CBaseControlVideo** member functions require only that the video renderer be connected to a filter graph.</span></span> <span data-ttu-id="2b935-109">如果未連接，成員函式會傳回 **VFW \_ E \_ 未 \_ 連接**。</span><span class="sxs-lookup"><span data-stu-id="2b935-109">If it is not connected, member functions will return **VFW\_E\_NOT\_CONNECTED**.</span></span> <span data-ttu-id="2b935-110">在影片轉譯器上設定的屬性會在連續連接和中斷連接之間保存。</span><span class="sxs-lookup"><span data-stu-id="2b935-110">Properties set on a video renderer persist between successive connections and disconnections.</span></span> <span data-ttu-id="2b935-111">所有應用程式都應該確保它們在開始簡報之前重設轉譯器屬性。</span><span class="sxs-lookup"><span data-stu-id="2b935-111">All applications should ensure that they reset the renderer properties before starting a presentation.</span></span>

<span data-ttu-id="2b935-112">使用影片時，應用程式可以選取要使用的影片部分。</span><span class="sxs-lookup"><span data-stu-id="2b935-112">When working with video, the application can select a portion of the video to use.</span></span> <span data-ttu-id="2b935-113">這個部分是 **CBaseControlVideo** 物件所控制的來源矩形。</span><span class="sxs-lookup"><span data-stu-id="2b935-113">This portion is the source rectangle that the **CBaseControlVideo** object controls.</span></span> <span data-ttu-id="2b935-114">**CBaseControlVideo** 可讓您的應用程式設定和取出來源矩形。</span><span class="sxs-lookup"><span data-stu-id="2b935-114">**CBaseControlVideo** enables your application to set and retrieve the source rectangle.</span></span> <span data-ttu-id="2b935-115">**CBaseControlVideo** 使用的所有矩形都採用寬度和高度值，而不是右值和底部值。</span><span class="sxs-lookup"><span data-stu-id="2b935-115">All the rectangles that **CBaseControlVideo** uses employ width and height values rather than right and bottom values.</span></span> <span data-ttu-id="2b935-116">若未設定任何來源矩形，來源矩形的屬性會傳回完整的原生影片大小。</span><span class="sxs-lookup"><span data-stu-id="2b935-116">When no source rectangle has been set, the properties of the source rectangle return the full, native video size.</span></span>
