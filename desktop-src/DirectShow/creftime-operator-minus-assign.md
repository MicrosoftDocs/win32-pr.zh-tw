# <a name="creftimeoperator--method"></a>CRefTime. operator-= 方法

-= 運算子會減去另一個參考時間。

## <a name="syntax"></a>語法


```C++
CRefTime& operator-=(
  [ref] const CRefTime &rt
);
```



## <a name="parameters"></a>參數

<dl> <dt>

*rt* \[裁判\]
</dt> <dd>

**CRefTime** 物件的參考。

</dd> </dl>

## <a name="return-value"></a>傳回值
