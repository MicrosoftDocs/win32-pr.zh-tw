# <a name="cameventreset-method"></a>CAMEvent 重設方法

`Reset`方法會將事件的狀態設定為未收到信號。

## <a name="syntax"></a>語法


```C++
void Reset();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
