# <a name="cbaselistremoveall-method"></a>CBaseList. RemoveAll 方法

`RemoveAll`方法會從清單中移除所有節點。

## <a name="syntax"></a>語法


```C++
void RemoveAll();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。

## <a name="remarks"></a>備註

呼叫端必須負責刪除所有節點中包含的專案。
