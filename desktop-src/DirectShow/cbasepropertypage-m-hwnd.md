# <a name="cbasepropertypagem_hwnd-member"></a>CBasePropertyPage：： m \_ hwnd 成員

`m_hwnd`成員變數包含交談視窗的控制碼。 當 **CreateDialogParam** 函式傳回時，這個成員變數就會在物件建立對話方塊視窗之後初始化。

## <a name="syntax"></a>Syntax


```C++
HWND m_hwnd;
```
