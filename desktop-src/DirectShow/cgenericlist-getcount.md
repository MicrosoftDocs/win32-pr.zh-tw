# <a name="cgenericlistgetcount-method"></a>CGenericList. GetCount 方法

`GetCount`方法會捕獲清單中的專案數。

## <a name="syntax"></a>語法


```C++
int GetCount();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回清單中的專案數。
