# <a name="tooltipmaxwidth-property"></a>ToolTipMaxWidth 屬性

> [!Note]  
> 此元件可用於 Microsoft Windows 2000、Windows XP 及 Windows Server 2003 作業系統。 它在後續版本中可能會變更或無法使用。

 

`ToolTipMaxWidth`屬性會設定或抓取與 **MSWebDVD** 物件相關聯之工具提示的最大寬度。

``` syntax
[ iMaxWidth = ] MSWebDVD.ToolTipMaxWidth
```

## <a name="return-value"></a>傳回值

將應用程式工具提示的最大寬度傳回為整數。

## <a name="remarks"></a>備註

此屬性是讀取/寫入，預設值為200圖元。
