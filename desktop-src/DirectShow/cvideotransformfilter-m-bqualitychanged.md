# <a name="cvideotransformfilterm_bqualitychanged-member"></a>CVideoTransformFilter：： m \_ bQualityChanged 成員

指出篩選是否已卸載框架的布林值。 此成員變數會控制篩選傳送 [**EC \_ 品質 \_ 變更**](ec-quality-change.md) 事件的時間。

## <a name="syntax"></a>Syntax


