# <a name="crenderedinputpinactive-method"></a>CRenderedInputPin 方法

`Active`方法會通知釘選篩選現在已在使用中。 這個方法會覆寫 [**CBasePin：： Active**](cbasepin-active.md) 方法。

## <a name="syntax"></a>語法


```C++
HRESULT Active();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

\_如果成功，則傳回，否則傳回錯誤碼。
