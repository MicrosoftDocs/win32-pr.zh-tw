# <a name="bitdepth-attribute"></a>bitdepth 屬性

> [!Note]  
> \[廢棄。 此 API 可能會從 Windows 的未來版本中移除。\]

 

`bitdepth`屬性指定輸出影片的位深度（以位/圖元為單位）。

## <a name="possible-values"></a>可能的值

必須是16、24或32。 預設值為 16。

## <a name="applies-to"></a>套用至

[**組**](group-element.md)

## <a name="remarks"></a>備註

只有當 type 屬性為時，才設定這個屬性 `video` 。
