# <a name="multimedia-streaming-sample-code"></a>多媒體串流範例程式碼

> [!Note]  
> 這些 Api 已被取代。 應用程式應該使用 [**範例捕獲**](sample-grabber-filter.md)篩選器或執行自訂篩選，以從 DirectShow 的篩選圖形取得資料。

 

本文提供可執行多媒體串流介面的範例程式碼。 影片串流範例程式碼示範如何讀取檔案，並將它轉譯成主要的 Microsoft® DirectDraw®表面。 為了簡潔起見，此程式碼不包含錯誤檢查。

第二個程式碼範例會示範如何使用音訊串流介面來串流處理音訊資料。

本文包含下列各節。

-   [影片串流範例程式碼](video-streaming-sample-code.md)
-   [音訊串流範例程式碼](audio-streaming-sample-code.md)
