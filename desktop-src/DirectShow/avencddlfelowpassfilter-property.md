# <a name="avencddlfelowpassfilter-property"></a>AVEncDDLFELowPassFilter 屬性

指定是否要將低傳遞篩選套用至低頻率效果， (LFE) 通道在杜比數位音訊串流中。 此屬性適用于杜比數位音訊編碼器。

這是可讀寫的屬性。

## <a name="data-type"></a>資料類型

**變異 \_BOOL** (**VT \_ bool**) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVEncDDLFELowPassFilter**
