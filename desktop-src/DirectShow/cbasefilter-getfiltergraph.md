# <a name="cbasefiltergetfiltergraph-method"></a>CBaseFilter. GetFilterGraph 方法

## <a name="syntax"></a>語法


```C++
IFilterGraph* GetFilterGraph();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回 [**CBaseFilter：： m \_ pGraph**](cbasefilter-m-pgraph.md)的值。
