# <a name="ec_file_closed"></a>EC \_ 檔案 \_ 已關閉

因為發生未預期的事件，所以已關閉原始程式檔。 例如，網路伺服器已關機。

</dt> <dd>

</dd> <dt>

<span id="lParam2"></span><span id="lparam2"></span><span id="LPARAM2"></span>*lParam2*
</dt> <dd>

零個。

</dd> </dl>

## <a name="default-action"></a>預設動作

無。

## <a name="remarks"></a>備註

舊版 Windows Media 來源篩選傳送此事件。 新的篩選器不應傳送此事件。
