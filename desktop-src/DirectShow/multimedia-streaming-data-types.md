# <a name="multimedia-streaming-data-types"></a>多媒體串流資料類型

> [!Note]  
> 此 API 即將淘汰。 新的應用程式不應使用它。

 

Microsoft DirectShow 多媒體串流介面會使用下列資料類型。



| 資料類型                           | 描述                                            |
|-------------------------------------|--------------------------------------------------------|
| [**MSPID**](mspid.md)              | 識別媒體資料流程的用途。              |
| [**資料流程 \_ 時間**](stream-time.md) | 定義用來表示參考時間的資料類型。 |
