# <a name="crendererinputpinendflush-method"></a>CRendererInputPin. EndFlush 方法

`EndFlush`方法會結束清除作業。 這個方法會覆寫 [**CBaseInputPin：： EndFlush**](cbaseinputpin-endflush.md) 方法。

## <a name="syntax"></a>語法


```C++
HRESULT EndFlush();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回 **HRESULT** 值。
