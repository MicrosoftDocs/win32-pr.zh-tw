# <a name="samplingrate-attribute"></a>samplingrate 屬性

> [!Note]  
> \[廢棄。 此 API 可能會從 Windows 的未來版本中移除。\]

 

`samplingrate`屬性指定輸出音訊的取樣率，以 Hz 為限。

## <a name="possible-values"></a>可能的值

此值必須是8000、11025、22050、32000、44100或48000。 預設值為44100。

## <a name="applies-to"></a>套用至

[**組**](group-element.md)

## <a name="remarks"></a>備註

只有當 **type** 屬性為時，才設定這個屬性 `audio` 。
