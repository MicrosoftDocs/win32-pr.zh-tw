# <a name="media-types-directshow"></a>媒體類型 (DirectShow)

-   [**主要類型**](major-types.md)
-   [**音訊子類型**](audio-subtypes.md)
-   [**BDA 媒體類型**](bda-media-types.md)
-   [**DVD 媒體類型**](dvd-media-types.md)
-   [**第21行媒體類型**](line-21-media-types.md)
-   [**MPEG-2 媒體類型**](mpeg-1-media-types.md)
-   [MPEG-2 媒體類型](mpeg-2-media-types.md)
-   [**資料流程子類型**](stream-subtypes.md)
-   [VBI 媒體類型](vbi-media-types.md)
-   [影片子類型](video-subtypes.md)
