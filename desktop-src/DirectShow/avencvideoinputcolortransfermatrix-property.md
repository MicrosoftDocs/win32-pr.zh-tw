# <a name="avencvideoinputcolortransfermatrix-property"></a>AVEncVideoInputColorTransferMatrix 屬性

針對輸入影片，指定從 Y'Cb'Cr 的色彩空間到 R'G'B 的色彩空間的轉換矩陣。

這是可讀寫的屬性。

## <a name="data-type"></a>資料類型

**UINT32** (**VT \_ UI4**) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVEncVideoInputColorTransferMatrix**

## <a name="property-value"></a>屬性值

這個屬性的值是 [**eAVEncVideoColorTransferMatrix**](/windows/desktop/api/codecapi/ne-codecapi-eavencvideocolortransfermatrix) 列舉的成員。
