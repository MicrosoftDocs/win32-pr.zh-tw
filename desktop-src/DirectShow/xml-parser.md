# <a name="xml-parser"></a>XML 剖析器

> [!Note]  
> \[廢棄。 此 API 可能會從 Windows 的未來版本中移除。\]

 

XML 剖析器物件會在 XML 專案檔與時間軸之間轉換。 若要建立這個物件，請呼叫 **CoCreateInstance**。

XML 剖析器物件會公開下列介面：

-   [**IXml2Dex**](ixml2dex.md)

**類別識別碼：** CLSID \_ Xml2Dex
