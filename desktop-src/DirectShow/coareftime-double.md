# <a name="coareftimeoperator-double-method"></a>COARefTime 運算子 double 方法

這個運算子會將參考時間轉換成 **雙精度** 值。

## <a name="syntax"></a>語法


```C++
double operator double();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回參考時間（以秒為單位）。
