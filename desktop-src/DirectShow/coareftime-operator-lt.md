# <a name="coareftimeoperator-method"></a>COARefTime 運算子< 方法

這個運算子會測試某個參考時間是否小於另一個。

## <a name="syntax"></a>語法


```C++
BOOL operator<(
  [ref] const COARefTime &rt
);
```



## <a name="parameters"></a>參數

<dl> <dt>

*rt* \[裁判\]
</dt> <dd>

要比較的 **COARefTime** 物件參考。

</dd> </dl>

## <a name="return-value"></a>傳回值

如果此物件嚴格小於 *rt*，則傳回 **TRUE** 。 否則，會傳回 **FALSE**。
