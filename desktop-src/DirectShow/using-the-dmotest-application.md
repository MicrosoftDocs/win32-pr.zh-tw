# <a name="using-the-dmotest-application"></a>使用 DMOTest 應用程式

> [!Note]  
> SDK 中已不再提供 DMOTest 應用程式。

 

DMOTest 公用程式可協助確認 a 如果您開發的是，您應該在測試過程中包含 SQL-DMO 測試應用程式。

DMOTest 會安裝在 \\ SDK 根目錄下的 Bin DXUtils 目錄中。 若要使用 DMOTest，您必須先產生每個的測試資料。 SDK 包含適用于此用途的 DirectShow 篩選。 您可以在 DMOTest 應用程式說明中找到篩選器和 DMOTest 應用程式的詳細資料。 在應用程式中，從 [說明 **] 功能表選擇 [** **內容**]。 此說明包含產生測試資料和執行測試的逐步指示。
