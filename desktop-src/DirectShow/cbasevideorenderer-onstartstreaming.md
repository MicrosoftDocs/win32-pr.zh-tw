# <a name="cbasevideorendereronstartstreaming-method"></a>CBaseVideoRenderer. OnStartStreaming 方法

`OnStartStreaming`方法會重設控制串流處理的所有時間。

## <a name="syntax"></a>語法


```C++
HRESULT OnStartStreaming();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回 **HRESULT** 值。

## <a name="remarks"></a>備註

此成員函式會覆寫 [**CBaseRenderer：： OnStartStreaming**](cbaserenderer-onstartstreaming.md)。
