# <a name="dvdrect-object"></a>DVDRect 物件

> [!Note]  
> 此元件可在 Microsoft Windows 2000、Windows XP 和 Windows Server 2003 作業系統中使用。 它在後續版本中可能會變更或無法使用。

 

> [!Note]  
> 此 API 即將淘汰。 如需 DirectShow dvd 播放和流覽的相關資訊，請參閱[dvd 應用程式](dvd-applications.md)。

 

物件的屬性 `DVDRect` 可讓腳本應用程式在 [MSWebDVD](mswebdvd-object.md) 方法（例如 [**GetButtonRect**](getbuttonrect-method.md)、 [**GetClipVideoRect**](getclipvideorect-method.md)、 [**GetVideoSize**](getvideosize-method.md) 和 [**SetClipVideoRect**](setclipvideorect-method.md)）中指定及接收矩形座標。



| 屬性                        | 描述                                                            |
|-----------------------------------|------------------------------------------------------------------------|
| [**X**](x-property.md)           | 設定或抓取矩形左上角的 x 位置。 |
| [**y**](y-property.md)           | 設定或抓取矩形左上角的 y 位置。 |
| [**寬度**](width-property.md)   | 設定或抓取矩形的寬度（以圖元為單位）。               |
| [**高度**](height-property.md) | 設定或抓取矩形的高度（以圖元為單位）。              |
