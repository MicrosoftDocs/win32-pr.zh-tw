# <a name="avencvideodefaultupperfielddominant-property"></a>AVEncVideoDefaultUpperFieldDominant 屬性

指定先顯示的欄位。

這是可讀寫的屬性。

## <a name="data-type"></a>資料類型

**變異 \_BOOL** (**VT \_ bool**) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVEncVideoDefaultUpperFieldDominant**

## <a name="remarks"></a>備註

如果值為 **VARIANT \_ TRUE**，則會先顯示上方欄位。 否則，會先顯示較低的欄位。
