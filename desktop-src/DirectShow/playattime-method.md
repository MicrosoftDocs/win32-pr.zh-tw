# <a name="playattime-method"></a>PlayAtTime 方法

> [!Note]  
> 此元件可用於 Microsoft Windows 2000、Windows XP 及 Windows Server 2003 作業系統。 它在後續版本中可能會變更或無法使用。

 

方法會在 `PlayAtTime` 指定的時間開始在目前的標題中播放。

``` syntax
MSWebDVD.PlayAtTime(sTime)
```

## <a name="parameters"></a>參數

<dl> <dt>

<span id="sTime"></span><span id="stime"></span><span id="STIME"></span>*sTime*
</dt> <dd>

指定開始播放的時間（以字串表示）。 字串的格式必須是 "hh： mm： ss： ff" (指定小時、分鐘、秒、畫面格) 。

</dd> </dl>

## <a name="return-value"></a>傳回值

沒有傳回值。
