# <a name="cbasewindowgetwindowwidth-method"></a>CBaseWindow. GetWindowWidth 方法

方法會抓取 `GetWindowWidth` 視窗的目前寬度。

## <a name="syntax"></a>語法


```C++
LONG GetWindowWidth();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回視窗的寬度（以圖元為單位）。
