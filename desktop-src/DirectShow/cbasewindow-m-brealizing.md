# <a name="cbasewindowm_brealizing-member"></a>CBaseWindow：： m \_ bRealizing 成員

旗標，指定是否要實現新的調色板。 在「偵錯工具組建」中，當選擇區變更時，此值為 **TRUE** ，否則為 **FALSE** 。 零售組建中不會使用此變數。

## <a name="syntax"></a>Syntax


```C++
BYTE m_bRealizing;
```
