# <a name="pause-method"></a>Pause 方法

> [!Note]  
> 此元件可在 Microsoft Windows 2000、Windows XP 和 Windows Server 2003 作業系統中使用。 它在後續版本中可能會變更或無法使用。

 

`Pause`方法會在目前的位置暫停播放。

``` syntax
MSWebDVD.Pause()
```

## <a name="return-value"></a>傳回值

沒有傳回值。

## <a name="remarks"></a>備註

如果播放已經暫停，則這個方法不會執行任何動作。
