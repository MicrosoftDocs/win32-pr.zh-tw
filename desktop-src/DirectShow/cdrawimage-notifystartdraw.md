# <a name="cdrawimagenotifystartdraw-method"></a>CDrawImage. NotifyStartDraw 方法

`NotifyStartDraw`不支援此方法。

## <a name="syntax"></a>語法


```C++
void NotifyStartDraw();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。

## <a name="remarks"></a>備註

在舊版的基類程式庫中，此方法支援 [**CDrawImage**](cdrawimage.md) 物件的效能度量。
