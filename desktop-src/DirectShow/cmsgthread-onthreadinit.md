# <a name="cmsgthreadonthreadinit-method"></a>CMsgThread. OnThreadInit 方法

提供執行緒的初始化。

## <a name="syntax"></a>語法


```C++
virtual void OnThreadInit();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。

## <a name="remarks"></a>備註

如果您想要線上程啟動時進行自己的特定初始化，請覆寫此函數。
