# <a name="cbaseinputpinisflushing-method"></a>CBaseInputPin. IsFlushing 方法

`IsFlushing`方法會查詢篩選目前是否正在排清。

## <a name="syntax"></a>語法


```C++
BOOL IsFlushing();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回 [**CBaseInputPin：： m \_ bFlushing**](cbaseinputpin-m-bflushing.md) 成員變數。
