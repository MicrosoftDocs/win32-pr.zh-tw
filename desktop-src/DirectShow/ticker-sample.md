# <a name="ticker-sample"></a>報價範例

## <a name="description"></a>Description

此範例會使用影片混合轉譯器來 blend 影片和文字。 它會使用 [**IVMRMixerBitmap9**](/previous-versions/windows/desktop/api/Vmr9/nn-vmr9-ivmrmixerbitmap9) 介面，將文字 blend 至影片視窗的下半部。

應用程式會建立具有預設文字字串的點陣圖，並將其在畫面底部進行滾動。 若要變更文字，請從 [指示器 **] 功能表選擇 [** **設定文字字串 ...** ]。 若要設定字型，請從 [指示器 **] 功能表選擇 [** **設定字型 ...** ]。 如果您選取 [指示器 **] 功能表中的 [** **使用靜態影像**]，應用程式會從動態文字字串切換至靜態點陣圖影像。

## <a name="downloading-the-sample"></a>下載範例

若要下載 DirectShow SDK 範例，請安裝最新版本的 [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx)。
