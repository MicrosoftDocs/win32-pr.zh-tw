# <a name="shortcuts-to-c-dvd-reference-pages"></a>C + + DVD 參考頁面的快捷方式

下列介面是在 c + + 中撰寫 DVD 應用程式時通常使用的介面。



| 介面                                    | 目的                                                                                          |
|----------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**IAMLine21Decoder**](/previous-versions/windows/desktop/api/il21dec/nn-il21dec-iamline21decoder) | 控制項隱藏式輔助字幕顯示。                                                              |
| [**IDvdCmd**](/windows/desktop/api/Strmif/nn-strmif-idvdcmd)                   | 用於 advanced 命令同步處理技術，以控制命令物件。                 |
| [**IDvdControl2**](/windows/desktop/api/Strmif/nn-strmif-idvdcontrol2)         | )  ("set" 方法，將命令發出至 [DVD 流覽](dvd-navigator-filter.md) 器篩選器。     |
| [**IDvdGraphBuilder**](/windows/desktop/api/Strmif/nn-strmif-idvdgraphbuilder) | 建立 DVD 篩選圖形。                                                                    |
| [**IDvdInfo2**](/windows/desktop/api/Strmif/nn-strmif-idvdinfo2)               | 查詢 DVD 導覽器中的狀態資訊，以及)  ("get" 方法的光碟資訊。 |
| [**IDvdState**](/windows/desktop/api/Strmif/nn-strmif-idvdstate)               | 將資訊（例如書簽）儲存至光碟。                                                   |
| [**IVideoFrameStep**](/windows/desktop/api/Strmif/nn-strmif-ivideoframestep)   | 當解碼器支援時，控制「逐步執行框架」。                                            |
