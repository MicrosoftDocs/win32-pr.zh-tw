# <a name="cbasevideowindow-class"></a>CBaseVideoWindow 類別

![cbasevideowindow 類別階層](images/wctrl01.png)

類別會實 `CBaseVideoWindow` [**IVideoWindow**](/windows/desktop/api/Control/nn-control-ivideowindow)介面的 **IDispatch** 方法。 其餘的 **IVideoWindow** 方法會保留為純虛擬方法，而且必須由衍生類別來執行。

此類別中的 **IDispatch** 方法是標準的實作為，在這裡不會詳細描述。
