# <a name="cbasecontrolvideoonupdaterectangles-method"></a>CBaseControlVideo. OnUpdateRectangles 方法

`OnUpdateRectangles`當來源或目的矩形變更時，就會呼叫方法。

## <a name="syntax"></a>語法


```C++
virtual HRESULT OnUpdateRectangles();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回 >AAD-USERREADUSINGALTERNATIVESECURITYID-NOERROR。
