# <a name="cbaseallocatorm_lalignment-member"></a>CBaseAllocator：： m \_ lAlignment 成員

每個緩衝區的對齊方式。 每個緩衝區的位址都必須是此值的偶數倍數。 前置詞必須計算為對齊;請參閱 [**CBaseAllocator：： m \_ lPrefix**](cbaseallocator-m-lprefix.md)。

## <a name="syntax"></a>Syntax


```C++
long m_lAlignment;
```
