# <a name="ec_please_reopen"></a>EC \_ 請 \_ 重新開啟

原始程式檔已變更。 應用程式應該重建篩選圖形。

</dd> </dl>

## <a name="default-action"></a>預設動作

無。

## <a name="remarks"></a>備註

舊版[Windows 媒體來源](windows-media-source-filter.md)篩選傳送此事件。 新的篩選器不應傳送此事件。
