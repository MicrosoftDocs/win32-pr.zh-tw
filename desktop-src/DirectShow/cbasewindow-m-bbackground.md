# <a name="cbasewindowm_bbackground-member"></a>CBaseWindow：： m \_ bBackground 成員

旗標，指定調色板是否應為背景調色板。 若 **為 TRUE**，則會選取 [調色板] 作為背景調色板。

## <a name="syntax"></a>Syntax


```C++
BYTE m_bBackground;
```
