# <a name="ctransinplaceinputpinm_breadonly-member"></a>CTransInPlaceInputPin：： m \_ bReadOnly 成員

指定輸入資料流程是否為唯讀的旗標。 上游篩選器會在呼叫 [**NotifyAllocator**](ctransinplaceinputpin-notifyallocator.md) 方法時指定此資訊。 依預設，此值為 **FALSE**。

## <a name="syntax"></a>Syntax


```C++
BOOL m_bReadOnly;
```
