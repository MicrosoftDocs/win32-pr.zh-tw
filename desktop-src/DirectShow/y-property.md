# <a name="y-property"></a>y 屬性

> [!Note]  
> 此元件可用於 Microsoft Windows 2000、Windows XP 及 Windows Server 2003 作業系統。 它在後續版本中可能會變更或無法使用。

 

`y`屬性會設定或抓取矩形左上角的 y 位置。

``` syntax
[ iYPos = ] DVDRect.y
```

## <a name="return-value"></a>傳回值

傳回矩形左上角的 Y 位置作為整數。

## <a name="remarks"></a>備註

此屬性是讀取/寫入，沒有預設值。
