# <a name="dmo-demo-sample"></a>SQL-DMO 示範範例

## <a name="description"></a>Description

這個範例應用程式會示範如何在應用程式中使用 [DirectX 媒體物件](directx-media-objects.md) (的) 。 它會透過音訊效果 SQL-DMO 將音訊資料從 WAV 檔案串流至 DirectSound 緩衝區。

## <a name="usage"></a>使用方式

按一下 [ **音效檔** ] 按鈕以載入音效檔。 使用下拉式清單方塊來選取音訊效果。 按一下 [ **播放** ] 按鈕，聽聽播放音訊效果的音效檔。

## <a name="downloading-the-sample"></a>下載範例

若要下載 DirectShow SDK 範例，請安裝最新版本的 [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx)。

此範例安裝在下列路徑下： *\[ SDK \] 根* \\ 範例 \\ 多媒體 \\ DirectShow \\ \\ DMODemo。
