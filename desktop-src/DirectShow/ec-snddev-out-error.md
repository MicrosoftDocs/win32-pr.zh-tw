# <a name="ec_snddev_out_error"></a>EC \_ SNDDEV \_ 輸出 \_ 錯誤

音訊轉譯器篩選器中發生裝置錯誤。

## <a name="parameters"></a>參數

<dl> <dt>

<span id="lParam1"></span><span id="lparam1"></span><span id="LPARAM1"></span>*lParam1*
</dt> <dd>

[**SNDDEV \_ 錯誤**](/previous-versions/windows/desktop/api/audevcod/ne-audevcod-snddev_err)列舉類型中的 DWORD 值，指出發生失敗時如何存取裝置。

</dd> <dt>

<span id="lParam2"></span><span id="lparam2"></span><span id="LPARAM2"></span>*lParam2*
</dt> <dd>

DWORD 值，指出從聲音裝置呼叫傳回的錯誤。

</dd> </dl>

## <a name="default-action"></a>預設動作

無。
