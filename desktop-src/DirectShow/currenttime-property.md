# <a name="currenttime-property"></a>CurrentTime 屬性

> [!Note]  
> 此元件可用於 Microsoft Windows 2000、Windows XP 及 Windows Server 2003 作業系統。 它在後續版本中可能會變更或無法使用。

 

`CurrentTime`屬性會捕獲目前的播放時間。

``` syntax
[ sCurTime = ] MSWebDVD.CurrentTime
```

## <a name="return-value"></a>傳回值

傳回11個字元的 **字串** ，代表目前的播放時間，格式為 "*hh： mm： ss： ff*" (小時、分鐘、秒、畫面格) 。

## <a name="remarks"></a>備註

這個屬性是唯讀的，沒有預設值。
