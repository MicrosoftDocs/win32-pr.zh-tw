# <a name="defaultmenulanguage-property"></a>DefaultMenuLanguage 屬性

> [!Note]  
> 此元件可用於 Microsoft Windows 2000、Windows XP 及 Windows Server 2003 作業系統。 它在後續版本中可能會變更或無法使用。

 

屬性會抓取 `DefaultMenuLanguage` 預設的功能表語言。

``` syntax
[ iLang = ] MSWebDVD.DefaultMenuLanguage
```

## <a name="return-value"></a>傳回值

傳回 LCID 值，其中包含預設音訊語言的主要語言識別項。

## <a name="remarks"></a>備註

這個屬性是讀寫，沒有預設值。
