# <a name="wavdest-filter-sample"></a>WavDest 篩選範例

## <a name="description"></a>Description

WavDes 篩選會將音訊串流寫入至 WAV 檔案。 它會採用單一音訊串流作為輸入，而其輸出 pin 必須連接到檔案 [寫入](file-writer-filter.md) 器篩選器。 此範例篩選以 [**CTransformFilter**](ctransformfilter.md) 類別為基礎。

## <a name="downloading-the-sample"></a>下載範例

若要下載 DirectShow SDK 範例，請安裝最新版本的 [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx)。

此範例安裝在下列路徑下： *\[ SDK \] 根* \\ 範例 \\ 多媒體 \\ DirectShow \\ 篩選 \\ WavDest。
