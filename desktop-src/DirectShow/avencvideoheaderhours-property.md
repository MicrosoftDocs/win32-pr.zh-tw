# <a name="avencvideoheaderhours-property"></a>AVEncVideoHeaderHours 屬性

指定圖片群組中的開始時間數位 (GOP) 標頭。

這是可讀寫的屬性。

## <a name="data-type"></a>資料類型

**UINT32** (**VT \_ UI4**) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVEncVideoHeaderHours**

## <a name="property-value"></a>屬性值

這個屬性的有效範圍是0到23（含）。
