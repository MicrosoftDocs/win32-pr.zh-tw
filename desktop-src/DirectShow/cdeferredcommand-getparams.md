# <a name="cdeferredcommandgetparams-method"></a>CDeferredCommand. GetParams 方法

方法會將 `GetParams` **DISPPARAMS** 引數清單抓取至方法。

## <a name="syntax"></a>語法


```C++
DISPPARAMS* GetParams();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回 **DISPPARAMS** 引數清單。
