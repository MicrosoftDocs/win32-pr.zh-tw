# <a name="stream_time"></a>資料流程 \_ 時間

> [!Note]  
> 此 API 即將淘汰。 新的應用程式不應使用它。

 

資料流程 \_ 時間資料類型是用來表示 Microsoft DirectShow 多媒體串流的參考時間。 單位為100毫微秒。


```C++
typedef LONGLONG STREAM_TIME;
```



## <a name="remarks"></a>備註

這種資料類型相當於 DirectShow [**參考 \_ 時間**](reference-time.md)資料類型。
