# <a name="avdeccommonmeanbitrate-property"></a>AVDecCommonMeanBitRate 屬性

取得此解碼器目前的平均位元速率。

這是唯讀的屬性。

## <a name="data-type"></a>資料類型

**UINT32** (**VT \_ UI4**) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVDecCommonMeanBitRate**
