# <a name="avencvideoinputcolorlighting-property"></a>AVEncVideoInputColorLighting 屬性

指定用於觀看輸入影片的預期光源條件。

這是可讀寫的屬性。

## <a name="data-type"></a>資料類型

**UINT32** (**VT \_ UI4**) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVEncVideoInputColorLighting**

## <a name="property-value"></a>屬性值

這個屬性的值是 [**eAVEncVideoColorLighting**](/windows/desktop/api/codecapi/ne-codecapi-eavencvideocolorlighting) 列舉的成員。
