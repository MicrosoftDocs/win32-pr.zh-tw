# <a name="csourcegetpincount-method"></a>CSource. GetPinCount 方法

方法會抓取 `GetPinCount` 篩選上的釘選數目。 這個方法會實作為純虛擬 [**CBaseFilter：： GetPinCount**](cbasefilter-getpincount.md) 方法。

## <a name="syntax"></a>語法


```C++
int GetPinCount();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回此篩選器的釘選數目。
