# <a name="cdrawimagem_endsample-member"></a>CDrawImage：： m \_ EndSample 成員

`m_EndSample`成員變數會指定最新範例的停止時間。

## <a name="syntax"></a>語法


```C++
CRefTime m_EndSample;
```



## <a name="remarks"></a>備註

此值只在 [**CDrawImage：:D isplaysampletimes**](cdrawimage-displaysampletimes.md) 方法內有效。
