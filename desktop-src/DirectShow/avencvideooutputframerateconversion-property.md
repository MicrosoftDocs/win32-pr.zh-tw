# <a name="avencvideooutputframerateconversion-property"></a>AVEncVideoOutputFrameRateConversion 屬性

指定當輸出畫面播放速率不符合輸入畫面播放速率時，編碼器是否轉換畫面播放速率。

這是可讀寫的屬性。

## <a name="data-type"></a>資料類型

**UINT32** (**VT \_ UI4**) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVEncVideoOutputFrameRateConversion**

## <a name="property-value"></a>屬性值

這個屬性的值是 [**eAVEncVideoOutputFrameRateConversion**](/windows/win32/api/codecapi/ne-codecapi-eavencvideooutputframerateconversion) 列舉的成員。
