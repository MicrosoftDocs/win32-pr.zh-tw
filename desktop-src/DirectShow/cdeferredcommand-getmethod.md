# <a name="cdeferredcommandgetmethod-method"></a>CDeferredCommand. GetMethod 方法

`GetMethod`方法會抓取要執行之方法 (DISPID) 的分派識別碼。

## <a name="syntax"></a>語法


```C++
long GetMethod();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回要執行之方法的 DISPID。
