# <a name="cmediatypeisfixedsize-method"></a>CMediaType. IsFixedSize 方法

`IsFixedSize`方法會判斷樣本的大小是否固定或變數大小。

## <a name="syntax"></a>語法


```C++
BOOL IsFixedSize() const;
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回 **bFixedSizeSamples** 成員的值。
