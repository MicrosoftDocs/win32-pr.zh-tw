# <a name="cmediatypeformat-method"></a>CMediaType 格式方法

方法會抓取 `Format` 格式區塊的指標。

## <a name="syntax"></a>語法


```C++
BYTE* Format() const;
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回 **pbFormat** 成員。
