# <a name="avencvideoinputcolortransferfunction-property"></a>AVEncVideoInputColorTransferFunction 屬性

指定輸入影片從 RGB 到 R'G'B 的轉換函式。

這是可讀寫的屬性。

## <a name="data-type"></a>資料類型

**UINT32** (**VT \_ UI4**) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVEncVideoInputColorTransferFunction**

## <a name="property-value"></a>屬性值

這個屬性的值是 [**eAVEncVideoColorTransferFunction**](/windows/desktop/api/codecapi/ne-codecapi-eavencvideocolortransferfunction) 列舉的成員。
