# <a name="cmediatypeistemporalcompressed-method"></a>CMediaType. IsTemporalCompressed 方法

`IsTemporalCompressed`方法會判斷資料流程是否使用時態性壓縮。

## <a name="syntax"></a>語法


```C++
BOOL IsTemporalCompressed() const;
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回 **bTemporalCompression** 成員的值。
