# <a name="cdrawimagem_startsample-member"></a>CDrawImage：： m \_ StartSample 成員

**M \_ StartSample** 成員變數會指定最新範例的開始時間。

## <a name="syntax"></a>語法


```C++
CRefTime m_StartSample;
```



## <a name="remarks"></a>備註

此值只在 [**CDrawImage：:D isplaysampletimes**](cdrawimage-displaysampletimes.md) 方法內有效。
