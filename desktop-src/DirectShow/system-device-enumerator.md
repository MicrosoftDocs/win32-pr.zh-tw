# <a name="system-device-enumerator"></a>系統裝置列舉值

系統裝置列舉值會列舉安裝在系統上的篩選器和硬體裝置。 應用程式可以使用此元件找出指定類別中的篩選器和裝置。 藉由呼叫 **CoCreateInstance** 來建立此物件。



| 標籤 | 值 |
|------------------|------------------------------------------|
| 類別識別碼 | CLSID \_ SystemDeviceEnum                  |
| 介面       | [**ICreateDevEnum**](/windows/desktop/api/Strmif/nn-strmif-icreatedevenum) |
