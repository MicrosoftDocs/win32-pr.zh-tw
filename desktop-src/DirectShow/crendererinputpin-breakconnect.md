# <a name="crendererinputpinbreakconnect-method"></a>CRendererInputPin. BreakConnect 方法

**BreakConnect** 方法會從連接釋放 pin。 這個方法會覆寫 [**CBaseInputPin：： BreakConnect**](cbaseinputpin-breakconnect.md) 方法。

## <a name="syntax"></a>語法


```C++
HRESULT BreakConnect();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回 **HRESULT** 值。
