# <a name="avaudiochannelcount-property"></a>AVAudioChannelCount 屬性

取得音訊位資料流程中的通道數目。

這是唯讀的屬性。

## <a name="data-type"></a>資料類型

**UINT32** (**VT \_ UI4**) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVAudioChannelCount**

## <a name="remarks"></a>備註

通道數目包含 (LFE) 通道的低頻率效果（如果有的話）。
