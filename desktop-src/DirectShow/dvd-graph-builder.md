# <a name="dvd-graph-builder"></a>DVD Graph Builder

dvd Graph 產生器是用來建立 DVD 播放圖表的協助程式物件。 應用程式可以使用此元件來建立 DVD 播放和流覽的篩選圖形。 藉由呼叫 **CoCreateInstance** 來建立此物件。



| 標籤 | 值 |
|------------------|----------------------------------------------|
| 類別識別碼 | CLSID \_ DvdGraphBuilder                       |
| 介面       | [**IDvdGraphBuilder**](/windows/desktop/api/Strmif/nn-strmif-idvdgraphbuilder) |
