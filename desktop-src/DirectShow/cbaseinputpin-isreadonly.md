## <a name="syntax"></a>語法


```C++
BOOL IsReadOnly();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回 [**CBaseInputPin：： m \_ bReadOnly**](cbaseinputpin-m-breadonly.md) 成員變數的值。
