# <a name="ec_dvd_no_fp_pgc"></a>EC \_ DVD \_ NO \_ FP \_ .PGC

表示 DVD 光碟沒有 FP \_ .pgc (第一次播放程式鏈) ，而且 dvd 導覽器不會自動載入任何 .pgc 並開始播放。

## <a name="remarks"></a>備註

此事件是從第一個 Play 網域引發。
