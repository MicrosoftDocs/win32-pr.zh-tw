# <a name="cbasewindowgetwindowhdc-method"></a>CBaseWindow. GetWindowHDC 方法

`GetWindowHDC`方法會 (DC) 抓取視窗的裝置內容控制碼。

## <a name="syntax"></a>語法


```C++
HDC GetWindowHDC();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回 DC 的控制碼。
