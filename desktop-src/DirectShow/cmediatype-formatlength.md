# <a name="cmediatypeformatlength-method"></a>CMediaType. FormatLength 方法

方法會抓取 `FormatLength` 格式區塊的長度。

## <a name="syntax"></a>語法


```C++
ULONG FormatLength() const;
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回 **cbFormat** 成員。
