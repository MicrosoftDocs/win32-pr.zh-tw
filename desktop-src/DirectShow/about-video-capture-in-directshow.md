# <a name="about-video-capture-in-directshow"></a>關於 DirectShow 中的影片捕獲

本檔中使用的「 *影片*」一詞是指從硬體裝置接收影片的任何應用程式。 影片捕獲裝置不僅包括攝影機，也包括電視調諧器卡、視頻磁帶答錄機 (VTRs) 等等。 您可以將已捕獲的影片儲存到磁片或即時預覽。

本節說明使用 DirectShow 進行影片捕獲時，必須瞭解的一些概念：

-   [關於 Capture Graph Builder](about-the-capture-graph-builder.md)
-   [關於影片捕獲裝置](about-video-capture-devices.md)
-   [DirectShow 影片捕獲篩選](directshow-video-capture-filters.md)
