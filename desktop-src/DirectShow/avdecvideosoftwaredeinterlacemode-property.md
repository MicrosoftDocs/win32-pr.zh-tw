# <a name="avdecvideosoftwaredeinterlacemode-property"></a>AVDecVideoSoftwareDeinterlaceMode 屬性

指定解碼器的軟體逐行掃描模式。

這是可讀寫的屬性。

## <a name="data-type"></a>資料類型

**UINT32** (**VT \_ UI4**) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVDecVideoSoftwareDeinterlaceMode**

## <a name="property-value"></a>屬性值

這個屬性的值是 [**eAVDecVideoSoftwareDeinterlaceMode**](/windows/desktop/api/codecapi/ne-codecapi-eavdecvideosoftwaredeinterlacemode) 列舉的成員。
