# <a name="avencddcentredownmixlevel-property"></a>AVEncDDCentreDownMixLevel 屬性

在杜比數位音訊串流中指定中央 downmix 等級（以分貝 (dB) ）。 此屬性適用于杜比數位音訊編碼器。

這是可讀寫的屬性。

## <a name="data-type"></a>資料類型

**UINT32** (**VT \_ UI4**) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVEncDDCentreDownMixLevel**
