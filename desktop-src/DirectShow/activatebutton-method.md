# <a name="activatebutton-method"></a>ActivateButton 方法

> [!Note]  
> 此元件可在 Microsoft Windows 2000、Windows XP 和 Windows Server 2003 作業系統中使用。 它在後續版本中可能會變更或無法使用。

 

ActivateButton 方法會啟用選取的功能表按鈕。

``` syntax
        MSWebDVD.ActivateButton()
```

## <a name="return-value"></a>傳回值

沒有傳回值。

## <a name="remarks"></a>備註

將 [**DisableAutoMouseProcessing**](disableautomouseprocessing-property.md) 設為 **true** 之後，執行自訂滑鼠處理時，請使用這個方法。

您可以使用這個方法來啟動透過 [**SelectLeftButton**](selectleftbutton-method.md)、 [**SelectLowerButton**](selectlowerbutton-method.md)、 [**SelectUpperButton**](selectupperbutton-method.md)或 [**SelectRightButton**](selectrightbutton-method.md) 方法選取的按鈕。
