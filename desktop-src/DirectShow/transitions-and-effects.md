# <a name="transitions-and-effects"></a>轉換和效果

> [!Note]  
> \[廢棄。 此 API 可能會從 Windows 的未來版本中移除。\]

 

本節說明 DirectShow 編輯服務所提供的效果和轉換。



| 轉換                              | Description                                                                            |
|-----------------------------------------|----------------------------------------------------------------------------------------|
| [Compositor](compositor-transition.md) | 將前景延伸至指定的矩形，而不變更背景。 |
| [金鑰](key-transition.md)               | 根據色度、Alpha、色調或亮度來執行加密。                             |
| [SMPTE 抹除](smpte-wipe-transition.md) | 執行標準的 SMPTE 抹除。                                                     |



 



| 效果                                        | 描述                           |
|-----------------------------------------------|---------------------------------------|
| [Alpha Setter](alpha-setter-effect.md)       | 設定影片影像上的 Alpha 位。 |
| [音量信封](volume-envelope-effect.md) | 設定音訊串流上的磁片區。   |



 



| 來源物件                                | Description                                        |
|----------------------------------------------|----------------------------------------------------|
| [影片色彩來源](video-color-source.md) | 建立純色的連續影片影像。 |
