# <a name="whql-tests"></a>WHQL 測試

測試 PanScan/黑邊和704/720 寬影像的各種組合：

-   16x9 video (704 或 720 wide) 、16x9 子圖片 (720 wide) 
-   16x9 video (704 或 720 wide) 黑邊、4x3 子圖片
-   16x9 video (704 或 720 wide) PanScan、4x3 子圖片
-   4x3 video (704 或 720 wide) 、4x3 子圖片
