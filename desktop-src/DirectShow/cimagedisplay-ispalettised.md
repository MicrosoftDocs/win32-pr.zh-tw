# <a name="cimagedisplayispalettised-method"></a>CImageDisplay. IsPalettised 方法

`IsPalettised`方法會判斷目前的顯示格式是否為調色盤。

## <a name="syntax"></a>語法


```C++
BOOL IsPalettised();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

如果格式為調色盤，則傳回 **TRUE** ，否則傳回 **FALSE** 。
