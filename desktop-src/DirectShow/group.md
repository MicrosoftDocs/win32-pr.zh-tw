# <a name="group-directshow"></a>群組 (DirectShow) 

> [!Note]  
> \[廢棄。 此 API 可能會從 Windows 的未來版本中移除。\]

 

群組物件代表最上層的組合。 每個群組會在最後一個專案中定義一個影片或音訊串流。 若要建立這個物件，請呼叫 [**IAMTimeline：： CreateEmptyNode**](iamtimeline-createemptynode.md) 方法。

群組物件會公開下列介面：

-   [**IAMTimelineComp**](iamtimelinecomp.md)
-   [**IAMTimelineGroup**](iamtimelinegroup.md)
-   [**IAMTimelineObj**](iamtimelineobj.md)
