# <a name="cbasewindowm_bdogetdc-member"></a>CBaseWindow：： m \_ bDoGetDC 成員

旗標，指定是否要 (DC) 取得裝置內容。 若 **為 TRUE**，則物件會抓取視窗 DC 的控制碼，並建立相容的記憶體 DC。

## <a name="syntax"></a>Syntax


```C++
BOOL m_bDoGetDC;
```
