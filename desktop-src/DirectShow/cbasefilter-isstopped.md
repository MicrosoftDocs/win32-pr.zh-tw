# <a name="cbasefilterisstopped-method"></a>CBaseFilter. IsStopped 方法

`IsStopped`方法會判斷篩選目前是否已停止。

## <a name="syntax"></a>語法


```C++
BOOL IsStopped();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

如果篩選已停止，則傳回 **TRUE** ，否則傳回 **FALSE** 。
