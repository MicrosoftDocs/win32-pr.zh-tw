# <a name="src-attribute"></a>src 屬性

> [!Note]  
> \[廢棄。 此 API 可能會從 Windows 的未來版本中移除。\]

 

`src`屬性指定媒體檔案的路徑名稱。 檔案可以是 DirectShow 所支援的任何類型，也可以是靜止的影像。 靜止影像的支援格式為點陣圖、裝置獨立點陣圖、GIF 和 JPEG。

## <a name="possible-values"></a>可能的值

必須是指定路徑名稱的字串。

## <a name="applies-to"></a>套用至

[**clip**](clip-element.md)
