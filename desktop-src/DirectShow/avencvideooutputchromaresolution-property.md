# <a name="avencvideooutputchromaresolution-property"></a>AVEncVideoOutputChromaResolution 屬性

指定編碼影片的色度解析度。

這是可讀寫的屬性。

## <a name="data-type"></a>資料類型

**UINT32** (**VT \_ UI4**) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVEncVideoOutputChromaResolution**

## <a name="property-value"></a>屬性值

這個屬性的值是 [**eAVEncVideoChromaResolution**](/windows/win32/api/codecapi/ne-codecapi-eavencvideochromaresolution) 列舉的成員。
