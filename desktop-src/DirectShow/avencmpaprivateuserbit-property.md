# <a name="avencmpaprivateuserbit-property"></a>AVEncMPAPrivateUserBit 屬性

設定 MPEG-2 資料流程中私用使用者位的值。 這個屬性會套用至 MPEG 音訊編碼器。

這是可讀寫的屬性。

## <a name="data-type"></a>資料類型

**UINT32** (**VT \_ UI4**) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVEncMPAPrivateUserBit**

## <a name="property-value"></a>屬性值

這個屬性的值可以是0或1。
