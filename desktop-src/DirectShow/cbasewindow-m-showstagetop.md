# <a name="cbasewindowm_showstagetop-member"></a>CBaseWindow：： m \_ ShowStageTop 成員

將視窗樣式設定為 WS \_ EX 最上層的私用訊息 \_ 。

## <a name="syntax"></a>語法


```C++
UINT m_ShowStageTop;
```

如果影片轉譯器切換至全螢幕模式，則應該將此訊息傳送至視窗。
