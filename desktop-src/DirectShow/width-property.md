# <a name="width-property-directshow"></a> (DirectShow) 的寬度屬性

> [!Note]  
> 此元件可用於 Microsoft Windows 2000、Windows XP 及 Windows Server 2003 作業系統。 它在後續版本中可能會變更或無法使用。

 

`Width`屬性會設定或抓取矩形的寬度。

``` syntax
[ iWidth = ] DVDRect.Width
```

## <a name="return-value"></a>傳回值

傳回矩形的寬度（以圖元為單位），表示為整數。

## <a name="remarks"></a>備註

此屬性是讀取/寫入，沒有預設值。
