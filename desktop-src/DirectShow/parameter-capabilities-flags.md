# <a name="parameter-capabilities-flags"></a>參數功能旗標

這些旗標會指定媒體參數支援哪些信封曲線。



| 常數                                                                                                                                                                                      | 描述                                              |
|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------|
| <span id="MP_CAPS_CURVE_INVSQUARE"></span><span id="mp_caps_curve_invsquare"></span><dl> <dt>**MP \_ CAPS \_ 曲線 \_ INVSQUARE**</dt> </dl> | 反向正方形曲線。<br/>                         |
| <span id="MP_CAPS_CURVE_JUMP"></span><span id="mp_caps_curve_jump"></span><dl> <dt>**MP \_ CAPS \_ 曲線 \_ 跳躍**</dt> </dl>                | 跳到下一個無插補的點。<br/> |
| <span id="MP_CAPS_CURVE_LINEAR"></span><span id="mp_caps_curve_linear"></span><dl> <dt>**MP \_ CAPS \_ 曲線 \_ 線性**</dt> </dl>          | 線性插補。<br/>                         |
| <span id="MP_CAPS_CURVE_SINE"></span><span id="mp_caps_curve_sine"></span><dl> <dt>**MP \_ CAPS \_ 曲線 \_ 正弦**</dt> </dl>                | 正弦曲線。<br/>                                   |
| <span id="MP_CAPS_CURVE_SQUARE"></span><span id="mp_caps_curve_square"></span><dl> <dt>**MP \_ CAPS \_ 曲線 \_ 正方形**</dt> </dl>          | 拋物線曲線。<br/>                              |
