# <a name="x-property"></a>x 屬性

> [!Note]  
> 此元件可在 Microsoft Windows 2000、Windows XP 和 Windows Server 2003 作業系統中使用。 它在後續版本中可能會變更或無法使用。

 

`x`屬性會設定或抓取矩形左上角的 x 位置。

``` syntax
[ iXPos = ] DVDRect.x
```

## <a name="return-value"></a>傳回值

傳回矩形左上角的 X 位置作為整數。

## <a name="remarks"></a>備註

此屬性是讀取/寫入，沒有預設值。
