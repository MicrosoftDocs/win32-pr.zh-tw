# <a name="cbasewindowgetmemoryhdc-method"></a>CBaseWindow. GetMemoryHDC 方法

方法會抓取 `GetMemoryHDC` (DC) 的記憶體裝置內容控制碼。

## <a name="syntax"></a>語法


```C++
virtual HDC GetMemoryHDC();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回記憶體 DC 的控制碼。
