# <a name="backcolor-property-directshow"></a>背景屬性 (DirectShow) 

> [!Note]  
> 此元件可在 Microsoft Windows 2000、Windows XP 和 Windows Server 2003 作業系統中使用。 它在後續版本中可能會變更或無法使用。

 

`BackColor`當原生影片的長寬比與物件的顯示區域不同時，屬性會設定或抓取在影片矩形邊緣周圍出現的橫條色彩。

``` syntax
[ iBackColor = ] MSWebDVD.BackColor
```

## <a name="return-value"></a>傳回值

傳回整數值，表示背景色彩的 RGB 值。

## <a name="remarks"></a>備註

此屬性是讀取/寫入，預設值為 off-黑色 (0x100010) 。
