# <a name="directshow-objects"></a>DirectShow 物件

本主題包含與 DirectShow 所附的核心 COM 物件的參考專案，而不是篩選準則。 如需 DirectShow 隨附的篩選器清單，請參閱 [Directshow 篩選](directshow-filters.md)。



| Object                                                   | 描述                                                      |
|----------------------------------------------------------|------------------------------------------------------------------|
| [Capture Graph Builder](capture-graph-builder.md)       | 建立影片捕獲圖形。                                     |
| [DVD Graph Builder](dvd-graph-builder.md)               | 建立 DVD 播放圖形。                                      |
| [篩選圖形管理員](filter-graph-manager.md)         | 組建和控制項篩選圖形。                               |
| [篩選器對應程式](filter-mapper.md)                       | 在登錄中搜尋已註冊的篩選準則。                    |
| [媒體屬性包](media-property-bag.md)             | 設定或抓取 AVI 檔中的資訊和中的區塊。             |
| [記憶體配置器](memory-allocator.md)                 | 配置媒體範例的緩衝區。                             |
| [系統參考時鐘](system-reference-clock.md)     | 實行參考時鐘。                                    |
| [系統裝置列舉值](system-device-enumerator.md) | 列舉安裝在系統上的篩選器和硬體裝置。 |
