# <a name="playcap-sample"></a>PlayCap 範例

## <a name="description"></a>Description

簡易 capture 應用程式。

此應用程式會針對在使用者系統上找到的第一個影片捕獲裝置建立預覽視窗， (是否有任何) 。 它會示範使用 [**ICaptureGraphBuilder2**](/windows/desktop/api/Strmif/nn-strmif-icapturegraphbuilder2) 和 [**ICreateDevEnum**](/windows/desktop/api/Strmif/nn-strmif-icreatedevenum) 介面建立捕捉圖表的簡單範例。

如需支援更多功能的範例 capture 應用程式，請參閱 [AmCap 範例](amcap-sample.md)。

## <a name="downloading-the-sample"></a>下載範例

若要下載 DirectShow SDK 範例，請安裝最新版本的[Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx)。

此範例安裝在下列路徑下： *\[ SDK \] 根* \\ 範例 \\ 多媒體 \\ DirectShow \\ Capture \\ PlayCap。
