# <a name="countryregion-assignments"></a>國家/地區指派

國際電信聯集 (ITU) 標準國家/地區通話碼會傳遞至 [**IAMTuner：:p 的 \_ CountryCode**](/windows/desktop/api/Strmif/nf-strmif-iamtuner-put_countrycode) 方法，以選取預設的電視頻率對應。

例如，將非埃及的 ITU-T 呼叫程式碼傳遞給20，會選取埃及的預設電視頻率。

如需國家/地區的電話號碼清單，請參閱目前指派的國家/地區代碼的 ITU-T 建議。
