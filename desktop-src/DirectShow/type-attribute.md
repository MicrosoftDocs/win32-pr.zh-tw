# <a name="type-attribute"></a>type 屬性

> [!Note]  
> \[廢棄。 此 API 可能會從 Windows 的未來版本中移除。\]

 

`type`屬性指定群組的媒體類型。

## <a name="possible-values"></a>可能的值

值必須是「音訊」或「影片」。
