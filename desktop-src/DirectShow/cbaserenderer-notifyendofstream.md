# <a name="cbaserenderernotifyendofstream-method"></a>CBaseRenderer. NotifyEndOfStream 方法

方法會將 `NotifyEndOfStream` [**EC \_ 完成**](ec-complete.md) 事件張貼至篩選圖形管理員。

## <a name="syntax"></a>語法


```C++
void NotifyEndOfStream();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
