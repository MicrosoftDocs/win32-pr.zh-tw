# <a name="capture-method"></a>Capture 方法

> [!Note]  
> 此元件可用於 Microsoft Windows 2000、Windows XP 及 Windows Server 2003 作業系統。 它在後續版本中可能會變更或無法使用。

 

`Capture`當 MSWebDVD 物件處於無視窗模式時，方法會從影片框架中捕捉靜止影像。

``` syntax
MSWebDVD.Capture()
```

## <a name="remarks"></a>備註

這個方法會從 DVD-Video 影像中捕捉目前的畫面格，並將其貼入視窗中，讓使用者可以從中儲存或編輯影像。 MSWebDVD 物件必須處於無視窗模式，此方法才能成功。
