# <a name="avencaudioinputcontent-property"></a>AVEncAudioInputContent 屬性

指定音訊內容是否包含音樂或語音。

這是可讀寫的屬性。

## <a name="data-type"></a>資料類型

**UINT32** (**VT \_ UI4**) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVEncAudioInputContent**

## <a name="property-value"></a>屬性值

這個屬性的值是 [**eAVEncAudioInputContent**](/windows/win32/api/codecapi/ne-codecapi-eavencaudioinputcontent) 列舉的成員。
