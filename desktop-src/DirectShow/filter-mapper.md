# <a name="filter-mapper"></a>篩選器對應程式

篩選對應工具會在登錄中搜尋已註冊的篩選準則。 篩選 Graph 管理員會使用此元件來建立篩選圖形。 應用程式可以使用它來尋找符合指定搜尋條件的篩選準則。 藉由呼叫 **CoCreateInstance** 來建立此物件。
