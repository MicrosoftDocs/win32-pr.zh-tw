# <a name="cdynamicoutputpinstopusingoutputpin-method"></a>CDynamicOutputPin. StopUsingOutputPin 方法

在 `StopUsingOutputPin` 串流作業之後，方法會釋放對 pin 的存取權。

## <a name="syntax"></a>語法


```C++
virtual void StopUsingOutputPin();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
