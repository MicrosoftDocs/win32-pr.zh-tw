# <a name="oafilterstate"></a>OAFilterState

`OAFilterState`資料類型會定義篩選圖形的狀態。 此值為 [**篩選準則 \_ 狀態**](/windows/win32/api/strmif/ne-strmif-filter_state) 列舉的成員。 這種資料類型是針對 OLE Automation 用戶端所定義，例如 Microsoft Visual Basic 6.0。


```C++
typedef long OAFilterState;
```
