# <a name="creftimegetunits-method"></a>CRefTime. GetUnits 方法

`GetUnits`方法會以 100-毫微秒單位來捕獲參考時間。

## <a name="syntax"></a>語法


```C++
LONGLONG GetUnits();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回 [**CRefTime：： m \_ 時間**](creftime-m-time.md)的值。
