# <a name="installing-the-windows-region-change-support-components"></a>安裝 Windows 區域變更支援元件

應用程式開發人員可以利用內建 Windows 的 dvd 區域變更支援來變更 dvd 播放應用程式內的區域。

在 Windows 2000 和更新版本上，提供 DVD 區域變更支援的元件位於 DLL storprop.dll (可在% windir% \\ system32) 中使用。 預設會安裝此 DLL，而不需要額外的步驟，以確保此 DLL 可供使用。
