# <a name="currentangle-property"></a>CurrentAngle 屬性

> [!Note]  
> 此元件可用於 Microsoft Windows 2000、Windows XP 及 Windows Server 2003 作業系統。 它在後續版本中可能會變更或無法使用。

 

`CurrentAngle`屬性會設定或抓取角度區塊中的目前角度。

``` syntax
[ iCurAngle = ] MSWebDVD.CurrentAngle
```

## <a name="return-value"></a>傳回值

傳回代表角度的整數值。 必須是1到9之間的值。

## <a name="remarks"></a>備註

此屬性是讀取/寫入，沒有預設值。
