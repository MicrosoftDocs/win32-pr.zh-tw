# <a name="ctransformfilterstopstreaming-method"></a>CTransformFilter. StopStreaming 方法

`StopStreaming`當篩選參數切換至 [已停止] 狀態時，就會呼叫方法。

## <a name="syntax"></a>語法


```C++
virtual HRESULT StopStreaming();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回 S \_ OK。

## <a name="remarks"></a>備註

這個方法不會在基類中執行任何動作，但衍生類別可以覆寫它。
