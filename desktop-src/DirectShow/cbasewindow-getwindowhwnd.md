# <a name="cbasewindowgetwindowhwnd-method"></a>CBaseWindow. GetWindowHWND 方法

方法會抓取 `GetWindowHWND` 視窗的控制碼。

## <a name="syntax"></a>語法


```C++
HWND GetWindowHWND();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回視窗的控制碼。
