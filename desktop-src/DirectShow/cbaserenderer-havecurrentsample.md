# <a name="cbaserendererhavecurrentsample-method"></a>CBaseRenderer. HaveCurrentSample 方法

`HaveCurrentSample`方法會判斷篩選是否有範例。

## <a name="syntax"></a>語法


```C++
virtual BOOL HaveCurrentSample();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

如果篩選準則有樣本，則傳回 **TRUE** ，否則傳回 **FALSE** 。
