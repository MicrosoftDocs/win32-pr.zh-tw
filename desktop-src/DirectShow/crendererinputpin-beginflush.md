# <a name="crendererinputpinbeginflush-method"></a>CRendererInputPin. BeginFlush 方法

`BeginFlush`方法會開始清除作業。 這個方法會覆寫 [**CBaseInputPin：： BeginFlush**](cbaseinputpin-beginflush.md) 方法。

## <a name="syntax"></a>語法


```C++
HRESULT BeginFlush();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回 **HRESULT** 值。
