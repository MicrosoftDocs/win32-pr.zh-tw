# <a name="cbasewindowgetwindowheight-method"></a>CBaseWindow. GetWindowHeight 方法

方法會抓取 `GetWindowHeight` 視窗的目前高度。

## <a name="syntax"></a>語法


```C++
LONG GetWindowHeight();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回視窗的高度（以圖元為單位）。
