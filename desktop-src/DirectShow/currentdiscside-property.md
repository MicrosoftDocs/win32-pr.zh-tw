# <a name="currentdiscside-property"></a>CurrentDiscSide 屬性

> [!Note]  
> 此元件可用於 Microsoft Windows 2000、Windows XP 及 Windows Server 2003 作業系統。 它在後續版本中可能會變更或無法使用。

 

屬性會抓取 `CurrentDiscSide` 光碟目前的側邊。

``` syntax
[ iCurSide = ] MSWebDVD.CurrentDiscSide
```

## <a name="return-value"></a>傳回值

傳回代表目前端的整數值;側 A = 1 和側 B = 2。

## <a name="remarks"></a>備註

這個屬性是唯讀的，沒有預設值。
