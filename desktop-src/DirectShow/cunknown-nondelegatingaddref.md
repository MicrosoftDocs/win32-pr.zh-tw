# <a name="cunknownnondelegatingaddref-method"></a>CUnknown. NonDelegatingAddRef 方法

`NonDelegatingAddRef`方法會遞增物件的參考計數。 這個方法會實 **INonDelegatingUnknown：： NonDelegatingAddRef** 方法。

## <a name="syntax"></a>語法


```C++
ULONG NonDelegatingAddRef();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回參考計數。
