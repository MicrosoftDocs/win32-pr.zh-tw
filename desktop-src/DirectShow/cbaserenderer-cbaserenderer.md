# <a name="cbaserenderercbaserenderer-constructor"></a>CBaseRenderer. CBaseRenderer 函數

函式方法。

## <a name="syntax"></a>語法


```C++
CBaseRenderer(
   REFCLSID  RenderClass,
   TCHAR     *pName,
   LPUNKNOWN pUnk,
   HRESULT   *phr,
   UINT      TimerResolution = 1
);
```



## <a name="parameters"></a>參數

<dl> <dt>

*RenderClass* 
</dt> <dd>

轉譯器的類別識別碼。

</dd> <dt>

*pName* 
</dt> <dd>

包含篩選名稱之字串的指標，用於進行調試。

</dd> <dt>

*朋 克* 
</dt> <dd>

這個物件之擁有者的指標。 如果物件已匯總，請將指標傳遞至匯總物件的 **IUnknown** 介面。 否則，請將此參數設定為 **Null**。

</dd> <dt>

*phr* 
</dt> <dd>

接收 **HRESULT** 值。

</dd> <dt>

*TimerResolution* 
</dt> <dd>

計時器解析度。

</dd> </dl>
