# <a name="loading-and-previewing-a-project"></a>載入和預覽 Project

\[此 API 不受支援，而且可能會在未來變更或無法使用。\]

本主題說明如何載入和預覽具有[DirectShow 編輯服務](directshow-editing-services.md) (DES) 的專案檔。 它提供一個範例應用程式，可將 XML 專案檔載入空白的時間軸，並呈現用於預覽的時間軸。 您可以修改範例應用程式，以使用此 SDK 隨附的任何範例 XML 專案檔。

本文包含下列主題。

-   [載入 Project 檔案](loading-a-project-file.md)
-   [預覽 Project](previewing-the-project.md)
-   [預覽 Project：範例程式碼](previewing-a-project--example-code.md)
