# <a name="avencmpvscenedetection-property"></a>AVEncMPVSceneDetection 屬性

指定編碼器偵測到新場景時的行為。 這個屬性會套用至 MPEG 視頻編碼器。

這是可讀寫的屬性。

## <a name="data-type"></a>資料類型

**UINT32** (**VT \_ UI4**) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVEncMPVSceneDetection**

## <a name="property-value"></a>屬性值

這個屬性的值是 [**eAVEncMPVSceneDetection**](/windows/win32/api/codecapi/ne-codecapi-eavencmpvscenedetection) 列舉的成員。
