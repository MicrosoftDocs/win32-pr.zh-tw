# <a name="avencvideoforcesourcescantype-property"></a>AVEncVideoForceSourceScanType 屬性

指定輸入框架是漸進式或交錯式。

這是可讀寫的屬性。

## <a name="data-type"></a>資料類型

**UINT32** (**VT \_ UI4**) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVEncVideoForceSourceScanType**

## <a name="property-value"></a>屬性值

這個屬性的值是 [**eAVEncVideoSourceScanType**](/windows/desktop/api/codecapi/ne-codecapi-eavencvideosourcescantype) 列舉的成員。
