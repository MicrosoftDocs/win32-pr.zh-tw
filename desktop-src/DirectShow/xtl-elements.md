# <a name="xtl-elements"></a>XTL 元素

> [!Note]  
> \[廢棄。 此 API 可能會從 Windows 的未來版本中移除。\]

 

XTL 格式會定義下列元素。



| 元素                                  | 描述                                                                    |
|------------------------------------------|--------------------------------------------------------------------------------|
| [**在**](at-element.md)                 | 針對 **param** 元素指定以時間為基礎的設定。                        |
| [**clip**](clip-element.md)             | 指定來源檔案。                                                       |
| [**複合**](composite-element.md)   | 定義組合。                                                         |
| [**影響**](effect-element.md)         | 定義效果。                                                             |
| [**組**](group-element.md)           | 定義群組 (音訊或影片) 。                                              |
| [**線性**](linear-element.md)         | 針對 **param** 元素指定以時間為基礎的設定。                        |
| [**參數**](param-element.md)           | 指定轉換、效果或其他子物件上的屬性值。 |
| [**時程表**](timeline-element.md)     | 定義時間軸。                                                          |
| [**跟蹤**](track-element.md)           | 定義追蹤。                                                               |
| [**過渡**](transition-element.md) | 定義轉換。                                                          |
