# <a name="height-attribute"></a>height 屬性

> [!Note]  
> \[廢棄。 此 API 可能會從 Windows 的未來版本中移除。\]

 

`height`屬性會指定輸出影片的高度（以圖元為單位）。

## <a name="possible-values"></a>可能的值

浮點值。 值必須在小數點之前包含前置零。 例如，0.3，而不是. 3。 請勿使用超過7個小數位數。

## <a name="applies-to"></a>套用至

[**組**](group-element.md)

## <a name="remarks"></a>備註

只有當 **type** 屬性為時，才設定這個屬性 `video` 。
