# <a name="dmo-functions"></a>SQL-DMO 函數

本節說明 Microsoft DirectX Media 物件 (SQL-DMO) 函式。



| 函式                                             | 描述                                                   |
|------------------------------------------------------|---------------------------------------------------------------|
| [**DMOEnum**](/previous-versions/windows/desktop/api/Dmoreg/nf-dmoreg-dmoenum)                           | 列舉登錄中所列的 DMOs。                       |
| [**DMOGetName**](/previous-versions/windows/desktop/api/Dmoreg/nf-dmoreg-dmogetname)                     | 從登錄抓取 SQL-DMO 的名稱。                |
| [**DMOGetTypes**](/previous-versions/windows/desktop/api/Dmoreg/nf-dmoreg-dmogettypes)                   | 抓取註冊的媒體類型。               |
| [**DMORegister**](/previous-versions/windows/desktop/api/Dmoreg/nf-dmoreg-dmoregister)                   | 註冊 a。                                              |
| [**DMOUnregister**](/previous-versions/windows/desktop/api/Dmoreg/nf-dmoreg-dmounregister)               | 取消註冊。                                            |
| [**MoCopyMediaType**](/previous-versions/windows/desktop/api/Dmort/nf-dmort-mocopymediatype)           | 將一個媒體類型結構複製到另一個。                   |
| [**MoCreateMediaType**](/previous-versions/windows/desktop/api/Dmort/nf-dmort-mocreatemediatype)       | 配置新的媒體類型結構。                         |
| [**MoDeleteMediaType**](/previous-versions/windows/desktop/api/Dmort/nf-dmort-modeletemediatype)       | 刪除先前配置的媒體類型結構。 |
| [**MoDuplicateMediaType**](/previous-versions/windows/desktop/api/Dmort/nf-dmort-moduplicatemediatype) | 複製媒體類型結構。                            |
| [**MoFreeMediaType**](/previous-versions/windows/desktop/api/Dmort/nf-dmort-mofreemediatype)           | 釋出媒體類型結構的配置成員。        |
| [**MoInitMediaType**](/previous-versions/windows/desktop/api/Dmort/nf-dmort-moinitmediatype)           | 初始化媒體類型結構。                           |
