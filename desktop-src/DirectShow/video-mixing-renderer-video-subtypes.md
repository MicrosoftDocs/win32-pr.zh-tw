# <a name="video-mixing-renderer-video-subtypes"></a>影片混合轉譯器影片子類型

下列子類型定義為與影片混合轉譯器 (VMR) 篩選器搭配使用。

## <a name="vmr-7-subtypes"></a>VMR-7 子類型



| Subtype                              | 描述                                      |
|--------------------------------------|--------------------------------------------------|
| MEDIASUBTYPE \_ RGB32 \_ D3D \_ DX7 \_ RT    | 32位 RGB 轉譯目標。                        |
| MEDIASUBTYPE \_ RGB16 \_ D3D \_ DX7 \_ RT    | 16位 RGB 轉譯目標。                        |
| MEDIASUBTYPE \_ ARGB32 \_ D3D \_ DX7 \_ RT   | 32位 ARGB 轉譯目標。                       |
| MEDIASUBTYPE \_ ARGB4444 \_ D3D \_ DX7 \_ RT | ARGB4444 呈現目標。 適用于子圖片圖形。 |
| MEDIASUBTYPE \_ ARGB1555 \_ D3D \_ DX7 \_ RT | ARGB1555 呈現目標。 適用于子圖片圖形。 |



 

## <a name="vmr-9-subtypes"></a>VMR-9 子類型



| Subtype                              | 描述                                      |
|--------------------------------------|--------------------------------------------------|
| MEDIASUBTYPE \_ RGB32 \_ D3D \_ DX9 \_ RT    | 32位 RGB 轉譯目標。                        |
| MEDIASUBTYPE \_ RGB16 \_ D3D \_ DX9 \_ RT    | 16位 RGB 轉譯目標。                        |
| MEDIASUBTYPE \_ ARGB32 \_ D3D \_ DX9 \_ RT   | 32位 ARGB 轉譯目標。                       |
| MEDIASUBTYPE \_ ARGB4444 \_ D3D \_ DX9 \_ RT | ARGB4444 呈現目標。 適用于子圖片圖形。 |
| MEDIASUBTYPE \_ ARGB1555 \_ D3D \_ DX9 \_ RT | ARGB1555 呈現目標。 適用于子圖片圖形。 |
