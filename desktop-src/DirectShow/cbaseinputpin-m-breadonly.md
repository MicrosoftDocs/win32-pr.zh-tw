# <a name="cbaseinputpinm_breadonly-member"></a>CBaseInputPin：： m \_ bReadOnly 成員

指出配置器是否會產生唯讀媒體範例的旗標。 若 **為 TRUE**，配置器會產生唯讀範例。

## <a name="syntax"></a>Syntax


```C++
BYTE m_bReadOnly;
```
