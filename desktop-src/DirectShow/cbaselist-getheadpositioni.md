# <a name="cbaselistgetheadpositioni-method"></a>CBaseList. GetHeadPositionI 方法

方法會抓取 `GetHeadPositionI` 清單中第一個專案的位置。

## <a name="syntax"></a>語法


```C++
POSITION GetHeadPositionI();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回位置值，如果清單是空的，則 **為 Null** 。
