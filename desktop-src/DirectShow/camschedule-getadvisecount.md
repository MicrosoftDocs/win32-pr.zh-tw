# <a name="camschedulegetadvisecount-method"></a>CAMSchedule. GetAdviseCount 方法

方法會抓取暫止的 `GetAdviseCount` 建議要求數目。

## <a name="syntax"></a>語法


```C++
DWORD GetAdviseCount();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回暫止的建議要求數目。
