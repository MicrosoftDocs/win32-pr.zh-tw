# <a name="coutputqueuenotifythread-method"></a>COutputQueue. NotifyThread 方法

`NotifyThread`方法會通知執行緒佇列包含資料。

## <a name="syntax"></a>語法


```C++
void NotifyThread();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。

## <a name="remarks"></a>備註

在呼叫這個方法之前，請先保存重要區段。
