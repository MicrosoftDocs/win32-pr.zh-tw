# <a name="cbasebasicvideo-class"></a>CBaseBasicVideo 類別

![cbasebasicvideo 類別階層](images/wctrl02.png)

類別會實 `CBaseBasicVideo` [**IBasicVideo**](/windows/desktop/api/Control/nn-control-ibasicvideo)介面的 **IDispatch** 方法。 其餘的 **IBasicVideo** 方法會保留為純虛擬方法，而且必須由衍生類別來執行。

此類別中的 **IDispatch** 方法是標準的實作為，在這裡不會詳細描述。
