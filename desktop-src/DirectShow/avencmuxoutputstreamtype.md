# <a name="avencmuxoutputstreamtype-property"></a>AVEncMuxOutputStreamType 屬性

指定多工器所產生的輸出資料流程類型。

## <a name="data-type"></a>資料類型

**UINT32** (**VT \_ UI4**) 

## <a name="property-guid"></a>屬性 GUID

**CODECAPI \_ AVEncMuxOutputStreamType**

## <a name="property-value"></a>屬性值

這個屬性的值是 [**eAVEncMuxOutput**](/windows/win32/api/codecapi/ne-codecapi-eavencmuxoutput) 列舉的成員。
