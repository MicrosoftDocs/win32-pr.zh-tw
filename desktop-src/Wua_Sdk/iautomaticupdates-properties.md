# <a name="iautomaticupdates-properties"></a>IAutomaticUpdates 屬性

[**IAutomaticUpdates**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdates)介面會定義下列屬性。



| 屬性                                                   | 描述                                                                                                   |
|------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**ServiceEnabled**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-get_serviceenabled) | 取得布林值，指出自動更新所需的所有元件是否可用。 |
| [**設定**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-get_settings)             | 取得自動更新的設定。                                                        |
