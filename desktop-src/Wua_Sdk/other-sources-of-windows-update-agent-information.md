# <a name="other-sources-of-windows-update-agent-information"></a>Windows Update 代理程式資訊的其他來源

[Windows Update AGENT API](portal-client.md)SDK 包含 API 最完整且最新的描述。 這是適用于 WUA 之開發人員參考資料的主要來源。

## <a name="software-development-kit-sdk-for-windows-update-agent-wua"></a>Windows Update 代理程式 (SDK) 的軟體發展工具組 (WUA) 

下列資源說明如何部署和管理 Windows Server Update Services 和 WUA：

-   [Windows Server Update Services](/documentation/)

    您可以下載 Windows Server Update Services SDK，並查看說明如何安裝、部署及維護 Windows Update 服務的其他檔。

-   [MSDN 下載與程式碼中心](/powerapps/developer/common-data-service/org-service/subscribe-sdk-assembly-updates-using-nuget)

    您可以下載 WUA SDK 範例、標頭和程式庫的更新。 MSDN 下載和 Code Center 中的下載會依主題進行組織。 您也可以使用 [Microsoft 下載中心](https://go.microsoft.com/fwlink/p/?linkid=83468) 搜尋工具，在 MSDN 上找到下載 Microsoft 開發人員的網路。
