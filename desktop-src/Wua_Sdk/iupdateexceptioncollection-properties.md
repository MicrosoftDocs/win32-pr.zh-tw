|----------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateexceptioncollection-get__newenum) | 取得用來列舉集合的 [**IEnumVARIANT**](/windows/win32/api/oaidl/nn-oaidl-ienumvariant) 介面。 |
| [**計數**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateexceptioncollection-get_count)        | 取得集合中的項目數。                                                                       |
| [**項目**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateexceptioncollection-get_item)          | 取得集合中的 [**IUpdateException**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateexception) 介面。                                    |
