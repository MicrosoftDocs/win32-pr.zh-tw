# <a name="iinstallationprogress-properties"></a>IInstallationProgress 屬性

[**IInstallationProgress**](/windows/desktop/api/Wuapi/nn-wuapi-iinstallationprogress)介面會定義下列屬性。



| 屬性                                                                                   | 描述                                                                                                                                        |
|--------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CurrentUpdateIndex**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationprogress-get_currentupdateindex)                     | 取得以零為基底的索引值，這個值會指定在選取多個更新時，目前正在安裝或卸載的更新。 |
| [**CurrentUpdatePercentComplete**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationprogress-get_currentupdatepercentcomplete) | 取得目前更新之安裝或卸載程式的進度（以百分比表示）。                                    |
| [**PercentComplete**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationprogress-get_percentcomplete)                           | 取得整體安裝或卸載進程的進度（以百分比表示）。                                                   |
