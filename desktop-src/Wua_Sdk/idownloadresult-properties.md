# <a name="idownloadresult-properties"></a>IDownloadResult 屬性

[**IDownloadResult**](/windows/desktop/api/Wuapi/nn-wuapi-idownloadresult)介面會定義下列屬性。



| 屬性                                         | 描述                                                                                                     |
|--------------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| [**HResult**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadresult-get_hresult)       | 取得例外狀況代碼編號（如果在下載期間引發例外狀況代碼號碼）。                         |
| [**ResultCode**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadresult-get_resultcode) | 取得指定下載結果的 [**OperationResultCode**](/windows/win32/api/wuapi/ne-wuapi-operationresultcode) 列舉。 |
