# <a name="icategorycollection-properties"></a>ICategoryCollection 屬性

[**ICategoryCollection**](/windows/desktop/api/Wuapi/nn-wuapi-icategorycollection)介面會定義下列屬性。



| 屬性                                          | 描述                                                                                                                     |
|---------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Wuapi/nf-wuapi-icategorycollection-get__newenum) | 取得可用來列舉集合的 [IEnumVARIANT](/windows/win32/api/oaidl/nn-oaidl-ienumvariant) 介面。 |
| [**計數**](/windows/desktop/api/Wuapi/nf-wuapi-icategorycollection-get_count)        | 取得集合中的項目數。                                                                                  |
| [**項目**](/windows/desktop/api/Wuapi/nf-wuapi-icategorycollection-get_item)          | 從集合中取得 [**ICategory**](/windows/desktop/api/Wuapi/nn-wuapi-icategory) 介面。                                                           |
