# <a name="iupdatesession-properties"></a>IUpdateSession 屬性

[**IUpdateSession**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesession)介面會定義下列屬性。



| 屬性                                                          | 描述                                                                  |
|-------------------------------------------------------------------|------------------------------------------------------------------------------|
| [**ClientApplicationID**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesession-get_clientapplicationid) | 取得和設定目前的用戶端應用程式。                                |
| [**ReadOnly**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesession-get_readonly)                       | 取得布林值，這個值表示會話物件是否為唯讀。 |
| [**WebProxy**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesession-get_webproxy)                       | 取得和設定用來存取伺服器的 proxy 設定。         |
