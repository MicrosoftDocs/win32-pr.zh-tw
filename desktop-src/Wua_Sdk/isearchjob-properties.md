# <a name="isearchjob-properties"></a>ISearchJob 屬性

[**ISearchJob**](/windows/desktop/api/Wuapi/nn-wuapi-isearchjob)介面會定義下列屬性。



| 屬性                                      | 描述                                                                                                                                                 |
|-----------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AsyncState**](/windows/desktop/api/Wuapi/nf-wuapi-isearchjob-get_asyncstate)   | 取得傳遞至 [**IUpdateSearch. BeginSearch**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-beginsearch) 方法的呼叫端特定狀態物件。                         |
| [**IsCompleted**](/windows/desktop/api/Wuapi/nf-wuapi-isearchjob-get_iscompleted) | 取得布林值，指出是否已完整處理 [**IUpdateSearch. BeginSearch**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-beginsearch) 方法的呼叫。 |
