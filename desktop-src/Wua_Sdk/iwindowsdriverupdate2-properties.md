# <a name="iwindowsdriverupdate2-properties"></a>IWindowsDriverUpdate2 屬性

[**IWindowsDriverUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate)介面會定義下列屬性。



| 屬性                                                       | 描述                                                                                                             |
|----------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**CveIDs**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate2-get_cveids)                 | 包含與更新相關聯 (CVE) 識別碼的常見弱點和漏洞集合。 |
| [**IsPresent**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate2-get_ispresent)           | 取得布林值，這個值表示是否已在電腦上安裝更新。                                     |
| [**RebootRequired**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate2-get_rebootrequired) | 取得布林值，指出是否必須在安裝或卸載更新之後重新開機電腦。    |



 

如需此介面所繼承之成員的相關資訊，請參閱下列介面。

-   [**IWindowsDriverUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate)
