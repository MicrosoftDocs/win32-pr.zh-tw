# <a name="iautomaticupdatessettings2-properties"></a>IAutomaticUpdatesSettings2 屬性

[**IAutomaticUpdatesSettings2**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings2)介面會定義下列屬性。



| 屬性                                                                                  | 描述                                                                                                                                                         |
|-------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IncludeRecommendedUpdates**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatessettings2-get_includerecommendedupdates) | 取得和設定布林值，這個值會指出是否要在執行更新的搜尋及安裝更新時包含選擇性或建議的更新。 |



 

如需此介面所繼承之成員的相關資訊，請參閱下列介面。

-   [**IAutomaticUpdatesSettings**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings)
