# <a name="iupdateservicecollection-properties"></a>IUpdateServiceCollection 屬性

[**IUpdateServiceCollection**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservicecollection)介面會定義下列屬性。



| 屬性                                               | 描述                                                                                                          |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservicecollection-get__newenum) | 取得用來列舉集合的 [**IEnumVARIANT**](/windows/win32/api/oaidl/nn-oaidl-ienumvariant) 介面。 |
| [**計數**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservicecollection-get_count)        | 取得集合中的項目數。                                                                       |
| [**項目**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservicecollection-get_item)          | 取得和設定集合中的 [**IUpdateService**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservice) 介面。                               |
