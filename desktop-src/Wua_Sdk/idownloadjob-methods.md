# <a name="idownloadjob-methods"></a>IDownloadJob 方法

[**IDownloadJob**](/windows/desktop/api/Wuapi/nn-wuapi-idownloadjob)介面會定義下列方法。



| 方法                                            | 描述                                                                                                            |
|---------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [**清理**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadjob-cleanup)           | 等候非同步作業完成，並釋放所有回呼。                                        |
| [**GetProgress**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadjob-getprogress)   | 傳回描述下載目前進度的 [**IDownloadProgress**](/windows/desktop/api/Wuapi/nn-wuapi-idownloadprogress) 介面。 |
| [**RequestAbort**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadjob-requestabort) | 提出結束非同步下載的要求。                                                                       |
