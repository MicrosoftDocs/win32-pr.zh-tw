# <a name="iupdateinstaller-methods"></a>IUpdateInstaller 方法

[**IUpdateInstaller**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateinstaller)介面會定義下列方法。



| 方法                                                    | 描述                                                                          |
|-----------------------------------------------------------|--------------------------------------------------------------------------------------|
| [**BeginInstall**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-begininstall)     | 開始非同步安裝更新。                                  |
| [**BeginUninstall**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-beginuninstall) | 開始非同步卸載更新。                                |
| [**EndInstall**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-endinstall)         | 完成更新的非同步安裝。                               |
| [**EndUninstall**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-enduninstall)     | 完成更新的非同步卸載。                             |
| [**安裝**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-install)               | 啟動更新的同步安裝。                                    |
| [**RunWizard**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-runwizard)           | 啟動精靈，引導本機使用者完成安裝更新的步驟。 |
| [**解除安裝**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-uninstall)           | 啟動更新的同步卸載。                                  |
