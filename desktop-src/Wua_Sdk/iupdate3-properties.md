# <a name="iupdate3-properties"></a>IUpdate3 屬性

[**IUpdate3**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate3)介面會定義下列屬性。



| 屬性                                  | 描述                                                                                                             |
|-------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**BrowseOnly**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate3-get_browseonly) | 取得布林值，這個值表示是否只能透過流覽可用的更新來探索更新。 |



 

如需此介面所繼承之成員的相關資訊，請參閱下列介面：

-   [**IUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate2)
-   [**IUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate)
