# <a name="iautomaticupdatesresults-properties"></a>IAutomaticUpdatesResults 屬性

[**IAutomaticUpdatesResults**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatesresults)介面會定義下列屬性。



| 屬性                                                                                    | 描述                                                                                                                                             |
|---------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**LastInstallationSuccessDate**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatesresults-get_lastinstallationsuccessdate) | 取得自動更新成功安裝任何更新時的上次時間和國際標準時間 (UTC) 日期，即使發生某些失敗。 |
| [**LastSearchSuccessDate**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatesresults-get_lastsearchsuccessdate)             | 取得 AutomaticUpdates 成功搜尋更新時的上次時間和國際標準時間 (UTC) 日期。                                   |
