# <a name="updating-windows-update-agent-header-files"></a>更新 Windows Update 代理程式標頭檔

Windows Update 代理程式 Api 先前為個別下載。 它們現在包含在 Windows SDK 中。 若要下載 Windows SDK 移 [至此頁面](https://developer.microsoft.com/windows/downloads/windows-10-sdk)。

如需有關如何更新 WUA 的詳細資訊，請參閱 [更新 Windows Update 代理程式](updating-the-windows-update-agent.md)。
