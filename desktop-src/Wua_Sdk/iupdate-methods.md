# <a name="iupdate-methods"></a>IUpdate 方法

[**IUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate)介面會定義下列方法。



| 方法                                         | 描述                                                                                      |
|------------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**AcceptEula**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate-accepteula)       | 接受與 Windows Update 相關聯的 Microsoft 軟體授權條款。<br/> |
| [**CopyFromCache**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate-copyfromcache) | 將更新的內容複寫到指定的路徑。<br/>                                 |
