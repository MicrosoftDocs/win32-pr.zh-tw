# <a name="isysteminformation-properties"></a>ISystemInformation 屬性

[**ISystemInformation**](/windows/desktop/api/Wuapi/nn-wuapi-isysteminformation)介面會定義下列屬性。



| 屬性                                                                    | 描述                                                                                                                                     |
|-----------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------|
| [**OemHardwareSupportLink**](/windows/desktop/api/Wuapi/nf-wuapi-isysteminformation-get_oemhardwaresupportlink) | 取得 OEM 硬體技術支援資訊的超連結。                                                                             |
| [**RebootRequired**](/windows/desktop/api/Wuapi/nf-wuapi-isysteminformation-get_rebootrequired)                 | 取得布林值，指出是否需要重新開機系統，才能完成安裝或卸載一或多個更新。 |
