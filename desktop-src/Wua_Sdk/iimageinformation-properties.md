# <a name="iimageinformation-properties"></a>IImageInformation 屬性

[**IImageInformation**](/windows/desktop/api/Wuapi/nn-wuapi-iimageinformation)介面會定義下列屬性。



| 屬性                                     | 描述                              |
|----------------------------------------------|------------------------------------------|
| [**AltText**](/windows/desktop/api/Wuapi/nf-wuapi-iimageinformation-get_alttext) | 取得影像的替代文字。   |
| [**高度**](/windows/desktop/api/Wuapi/nf-wuapi-iimageinformation-get_height)   | 取得影像的高度（以圖元為單位）。 |
| [**來源**](/windows/desktop/api/Wuapi/nf-wuapi-iimageinformation-get_source)   | 取得影像的來源位置。   |
| [**寬度**](/windows/desktop/api/Wuapi/nf-wuapi-iimageinformation-get_width)     | 取得影像的寬度（以圖元為單位）。  |
