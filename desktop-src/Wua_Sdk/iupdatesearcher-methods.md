# <a name="iupdatesearcher-methods"></a>IUpdateSearcher 方法

[**IUpdateSearcher**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesearcher)介面會定義下列方法。



| 方法                                                              | 描述                                                                                                               |
|---------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------|
| [**BeginSearch**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-beginsearch)                   | 開始執行更新的非同步搜尋。 搜尋會使用目前設定的搜尋選項。 |
| [**EndSearch**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-endsearch)                       | 完成更新的非同步搜尋。                                                                             |
| [**EscapeString**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-escapestring)                 | 將字串轉換成可在搜尋準則字串中當做常值使用的字串。                          |
| [**GetTotalHistoryCount**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-gettotalhistorycount) | 傳回電腦上更新事件的數目。                                                                      |
| [**QueryHistory**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-queryhistory)                | 以同步方式查詢更新事件歷程記錄的電腦。                                                      |
| [**搜尋**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-search)                             | 執行更新的同步搜尋。 搜尋會使用目前設定的搜尋選項。              |
