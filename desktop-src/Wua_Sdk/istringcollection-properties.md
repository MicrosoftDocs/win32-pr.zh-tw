# <a name="istringcollection-properties"></a>IStringCollection 屬性

[**IStringCollection**](/windows/desktop/api/Wuapi/nn-wuapi-istringcollection)介面會定義下列屬性。



| 屬性                                        | 描述                                                                                                                     |
|-------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-get__newenum) | 取得可用來列舉集合的 [IEnumVARIANT](/windows/win32/api/oaidl/nn-oaidl-ienumvariant) 介面。 |
| [**計數**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-get_count)        | 取得集合中的項目數。                                                                                  |
| [**項目**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-get_item)          | 取得或設定集合中的字串。                                                                                        |
| [**ReadOnly**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-get_readonly)  | 取得布林值，指出集合是否為唯讀。                                                        |
