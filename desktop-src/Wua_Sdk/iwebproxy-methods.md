# <a name="iwebproxy-methods"></a>IWebProxy 方法

[**IWebProxy**](/windows/desktop/api/Wuapi/nn-wuapi-iwebproxy)介面會定義下列方法。



| 方法                                                                         | 描述                                                         |
|--------------------------------------------------------------------------------|---------------------------------------------------------------------|
| [**PromptForCredentials**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-promptforcredentials)                 | 提示使用者輸入用於 proxy 驗證的密碼。  |
| [**PromptForCredentialsFromHwnd**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-promptforcredentialsfromhwnd) | 提示使用者輸入用於 proxy 驗證的密碼。  |
| [**SetPassword**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-setpassword)                                   | 設定要提交給 proxy 伺服器進行驗證的密碼。 |
