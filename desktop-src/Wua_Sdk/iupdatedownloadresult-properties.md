# <a name="iupdatedownloadresult-properties"></a>IUpdateDownloadResult 屬性

[**IUpdateDownloadResult**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatedownloadresult)介面會定義下列屬性。



| 屬性                                               | 描述                                                                                                                          |
|--------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------|
| [**HResult**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadresult-get_hresult)       | 取得在更新作業期間引發的例外狀況 **HRESULT** 值（如果有的話）。                                       |
| [**ResultCode**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadresult-get_resultcode) | 取得 [**OperationResultCode**](/windows/win32/api/wuapi/ne-wuapi-operationresultcode) 列舉值，這個值會指定更新的作業結果。 |
