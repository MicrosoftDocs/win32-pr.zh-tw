# <a name="istringcollection-methods"></a>IStringCollection 方法

[**IStringCollection**](/windows/desktop/api/Wuapi/nn-wuapi-istringcollection)介面會定義下列方法。



| 方法                                         | 描述                                                    |
|------------------------------------------------|----------------------------------------------------------------|
| [**添加**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-add)           | 將項目新增至集合。                                |
| [**清楚**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-clear)       | 移除集合中的所有元素。                  |
| [**複製**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-copy)         | 建立集合的深度讀取/寫入複本。              |
| [**插入**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-insert)     | 將項目插入指定位置上的集合中。 |
| [**RemoveAt**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-removeat) | 從集合中移除指定索引處的項目。   |
