# <a name="iupdate2-properties"></a>IUpdate2 屬性

[**IUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate2)介面會定義下列屬性。



| 屬性                                          | 描述                                                                                                                                             |
|---------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CveIDs**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate2-get_cveids)                 | 取得與更新相關聯 (CVE) 識別碼的常見弱點和漏洞集合。                                                |
| [**IsPresent**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate2-get_ispresent)           | 取得布林值，指出電腦上是否有更新。                                                                         |
| [**RebootRequired**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate2-get_rebootrequired) | 取得布林值，指出電腦上是否需要重新開機系統，才能完成安裝或卸載更新。 |



 

如需此介面所繼承之成員的相關資訊，請參閱下列介面：

-   [**IUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate)
