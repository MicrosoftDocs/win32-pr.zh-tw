# <a name="iautomaticupdatessettings3-properties"></a>IAutomaticUpdatesSettings3 屬性

[**IAutomaticUpdatesSettings3**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings3)介面會定義下列屬性。



| 屬性                                                                                  | 描述                                                                                                                                     |
|-------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------|
| [**FeaturedUpdatesEnabled**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatessettings3-get_featuredupdatesenabled)       | 取得和設定布林值，這個值會指出是否要顯示精選更新的通知。                                             |
| [**NonAdministratorsElevated**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatessettings3-get_nonadministratorselevated) | 取得和設定布林值，指出非系統管理員是否可以執行某些更新相關的動作，而不需要系統管理員核准。 |



 

如需此介面所繼承之成員的相關資訊，請參閱下列介面。

-   [**IAutomaticUpdatesSettings**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings)
-   [**IAutomaticUpdatesSettings2**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings2)
