# <a name="iupdateservice2-properties"></a>IUpdateService2 屬性

**IUpdateService2** 介面會定義下列屬性。



| 屬性                                                         | 描述                                                                                                                                                                         |
|------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IsDefaultAUService**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservice2-get_isdefaultauservice) | 取得布林值，這個值會指出服務是否已向自動更新註冊，以及服務目前是否由自動更新當做預設服務使用。 |



 

如需此介面所繼承之成員的相關資訊，請參閱下列介面。

-   [**IUpdateService**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservice)
