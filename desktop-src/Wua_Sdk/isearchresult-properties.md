# <a name="isearchresult-properties"></a>ISearchResult 屬性

[**ISearchResult**](/windows/desktop/api/Wuapi/nn-wuapi-isearchresult)介面會定義下列屬性。



| 屬性                                               | 描述                                                                                                   |
|--------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**ResultCode**](/windows/desktop/api/Wuapi/nf-wuapi-isearchresult-get_resultcode)         | 取得指定搜尋結果的 [**OperationResultCode**](/windows/win32/api/wuapi/ne-wuapi-operationresultcode) 列舉。 |
| [**RootCategories**](/windows/desktop/api/Wuapi/nf-wuapi-isearchresult-get_rootcategories) | 取得電腦上目前可用之根類別的介面集合。             |
| [**更新**](/windows/desktop/api/Wuapi/nf-wuapi-isearchresult-get_updates)               | 取得搜尋所產生之更新的介面集合。                                        |
| [**警告**](/windows/desktop/api/Wuapi/nf-wuapi-isearchresult-get_warnings)             | 取得搜尋所產生的警告集合。                                                  |
