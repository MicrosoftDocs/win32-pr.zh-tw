# <a name="iupdatedownloader-methods"></a>IUpdateDownloader 方法

[**IUpdateDownloader**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatedownloader)介面會定義下列方法。



| 方法                                                   | 描述                                                                                |
|----------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [**BeginDownload**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloader-begindownload) | 開始非同步下載與更新相關聯的內容檔案。 |
| [**下載**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloader-download)           | 開始同步下載與更新相關聯的內容檔案。   |
| [**EndDownload**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloader-enddownload)     | 完成非同步下載。                                                        |
