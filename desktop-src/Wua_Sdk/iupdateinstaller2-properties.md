# <a name="iupdateinstaller2-properties"></a>IUpdateInstaller2 屬性

[**IUpdateInstaller2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateinstaller2)介面會定義下列屬性。



| 屬性                                           | 描述                                                                                                                       |
|----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| [**ForceQuiet**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller2-get_forcequiet) | 取得和設定布林值，這個值會指出是否強制 Windows Installer 在沒有使用者互動的情況下安裝更新。 |



 

如需此介面所繼承之成員的相關資訊，請參閱下列介面。

-   [**IUpdateInstaller**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateinstaller)
