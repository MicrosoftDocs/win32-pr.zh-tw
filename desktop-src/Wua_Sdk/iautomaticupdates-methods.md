# <a name="iautomaticupdates-methods"></a>IAutomaticUpdates 方法

[**IAutomaticUpdates**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdates)介面會定義下列方法。



| 方法                                                             | 描述                                                         |
|--------------------------------------------------------------------|---------------------------------------------------------------------|
| [**DetectNow**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-detectnow)                   | 開始自動更新（如果尚未啟動）。            |
| [**EnableService**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-enableservice)           | 啟用自動更新所需的所有元件。         |
| [**暫停**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-pause)                           | 暫停自動更新。                                          |
| [**繼續**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-resume)                         | 重新開機自動更新（如果已暫停）。                        |
| [**ShowSettingsDialog**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-showsettingsdialog) | 顯示對話方塊，其中包含自動更新的設定。 |
