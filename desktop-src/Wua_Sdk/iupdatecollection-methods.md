# <a name="iupdatecollection-methods"></a>IUpdateCollection 方法

[**IUpdateCollection**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatecollection)介面會定義下列方法。



| 方法                                         | 描述                                                    |
|------------------------------------------------|----------------------------------------------------------------|
| [**添加**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatecollection-add)           | 將項目新增至集合。                                |
| [**清楚**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatecollection-clear)       | 移除集合中的所有元素。                  |
| [**複製**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatecollection-copy)         | 建立集合的淺層讀取/寫入複本。           |
| [**插入**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatecollection-insert)     | 將項目插入指定位置上的集合中。 |
| [**RemoveAt**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatecollection-removeat) | 從集合中移除指定索引處的項目。   |
