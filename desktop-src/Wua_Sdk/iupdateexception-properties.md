# <a name="iupdateexception-properties"></a>IUpdateException 屬性

[**IUpdateException**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateexception)介面會定義下列屬性。



| 屬性                                    | 描述                                                |
|---------------------------------------------|------------------------------------------------------------|
| [**Context**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateexception-get_context) | 取得例外狀況的內容。                         |
| [**HResult**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateexception-get_hresult) | 取得例外狀況的以 Windows 為基礎的 **HRESULT** 程式碼。 |
| [**消息**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateexception-get_message) | 取得描述目前例外狀況的訊息。       |
