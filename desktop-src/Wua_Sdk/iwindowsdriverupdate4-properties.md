# <a name="iwindowsdriverupdate4-properties"></a>IWindowsDriverUpdate4 屬性

[**IWindowsDriverUpdate4**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate4)介面會定義下列屬性。



| 屬性                                                                               | 描述                                                                 |
|----------------------------------------------------------------------------------------|-----------------------------------------------------------------------------|
| [**PerUser**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate4-get_peruser)                                       | 取得布林值，這個值表示更新是否為每個使用者的更新。 |
| [**WindowsDriverUpdateEntries**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate4-get_windowsdriverupdateentries) | 取得適用于更新的驅動程式更新專案。          |



 

如需此介面所繼承之成員的相關資訊，請參閱下列介面。

-   [**IWindowsDriverUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate)
-   [**IWindowsDriverUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate2)
-   [**IWindowsDriverUpdate3**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate3)
