# <a name="symbol-handling"></a>符號處理

符號處理常式函式可讓應用程式輕鬆且可攜存取可執行映射的符號調試資訊。 這些函數是專門用來確保存取符號資訊。 這是必要的，因為這些函式會從符號格式隔離應用程式。

如需詳細資訊，請參閱下列主題：

-   [符號處理常式初始化](symbol-handler-initialization.md)
-   [符號路徑](symbol-paths.md)
-   [符號載入](symbol-loading.md)
-   [延遲的符號載入](deferred-symbol-loading.md)
-   [裝飾符號名稱](decorated-symbol-names.md)
-   [尋找符號](finding-symbols.md)
-   [Public、Global 和 Local 符號](public--global--and-local-symbols.md)
-   [符號處理常式清除](symbol-handler-cleanup.md)
