# <a name="error-handling-error-handling"></a>錯誤處理 (錯誤處理) 

撰寫完善的應用程式包括錯誤處理常式代碼，可讓它們正常地從非預期的錯誤中復原。 發生錯誤時，應用程式可能需要要求使用者介入，或者可以自行復原。 在極端情況下，應用程式可能會將使用者登出或關閉系統。

-   [關於錯誤處理](about-error-handling.md)
-   [使用錯誤處理](using-error-handling.md)
-   [錯誤處理參考](error-handling-reference.md)

如需例外狀況處理的詳細資訊，請參閱 [結構化例外狀況處理](structured-exception-handling.md)。
