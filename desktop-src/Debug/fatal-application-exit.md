# <a name="fatal-application-exit"></a>嚴重應用程式結束

**FatalAppExit** 函式會顯示訊息方塊，並在使用者關閉訊息方塊時終止應用程式。 這個函式只能用來做為最後的手段，因為它可能無法釋放應用程式所擁有的記憶體或檔案。
