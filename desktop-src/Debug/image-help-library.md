# <a name="image-help-library"></a>影像說明庫

此總覽描述 Imagehlp.dll DLL 所提供的函式集。 這些函式可讓您使用可移植的可執行檔 (PE) 映射。

-   [關於 Imagehlp.dll](about-imagehlp.md)
-   [Imagehlp.dll 參考](imagehlp-reference.md)

您無法轉散發作業系統隨附的 Imagehlp.dll DLL。 函式的子集包含在 [Debug Help Library](debug-help-library.md)中，也就是 restributable。
