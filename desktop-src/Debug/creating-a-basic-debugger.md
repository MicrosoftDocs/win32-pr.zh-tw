# <a name="creating-a-basic-debugger"></a>建立基本偵錯工具

[執行中進程的調試](debugging-a-running-process.md) 程式描述將偵錯工具附加至執行中的進程。 [撰寫偵錯工具的主要迴圈](writing-the-debugger-s-main-loop.md) 會描述偵錯工具的主要執行迴圈中使用的函式。

雖然大部分的偵錯工具都是用來建立偵錯工具，但有數個函式是設計用來在正在進行調試的進程中使用。 如需詳細資訊，請參閱 [使用偵錯工具進行通訊](communicating-with-the-debugger.md)。
