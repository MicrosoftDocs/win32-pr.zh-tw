# <a name="debug-support-from-process-thread-and-exception-functions"></a>從進程、執行緒和例外狀況函式的偵錯工具支援

某些需要進行偵錯工具的函式會實際分類為進程、執行緒或例外狀況處理函式。 如需如何啟動即將進行調試之進程的說明，請參閱 [處理](process-functions-for-debugging.md)函式。 如需如何檢查和操作執行緒內容和執行的詳細資訊，請參閱 [執行緒函數](thread-functions-for-debugging.md)。 如需偵錯工具如何處理例外狀況的說明，請參閱 [例外狀況處理](exception-handling-functions-for-debugging.md)函式。
