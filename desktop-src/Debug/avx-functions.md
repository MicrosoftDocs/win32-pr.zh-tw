# <a name="avx-functions"></a>AVX 函式

以下是 Intel AVX 函數。

## <a name="in-this-section"></a>本節內容



| 主題                                                                   | 描述                                                                                                                    |
|-------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| [**CopyCoNtext**](/windows/desktop/api/WinBase/nf-winbase-copycontext)<br/>                           | 複製來源內容結構 (包括任何 XState) 至初始化的目的地內容結構。<br/>         |
| [**GetEnabledXStateFeatures**](/windows/desktop/api/WinBase/nf-winbase-getenabledxstatefeatures)<br/> | 取得 x86 或 x64 處理器上已啟用 XState 功能的遮罩。<br/>                                                    |
| [**GetXStateFeaturesMask**](/windows/desktop/api/WinBase/nf-winbase-getxstatefeaturesmask)<br/>       | 傳回在 [**內容**](/windows/desktop/api/WinNT/ns-winnt-wow64_context) 結構中設定的 XState 功能遮罩。<br/>                        |
| [**InitializeCoNtext**](/windows/desktop/api/WinBase/nf-winbase-initializecontext)<br/>               | 使用必要的大小和對齊方式，初始化緩衝區內的 [**內容**](/windows/desktop/api/WinNT/ns-winnt-arm64_nt_context) 結構。<br/>       |
| [**LocateXStateFeature**](/windows/desktop/api/WinBase/nf-winbase-locatexstatefeature)<br/>           | 抓取 [**內容**](/windows/desktop/api/WinNT/ns-winnt-arm64_nt_context) 結構內 XState 功能的處理器狀態指標。<br/> |
| [**SetXStateFeaturesMask**](/windows/desktop/api/WinBase/nf-winbase-setxstatefeaturesmask)<br/>       | 設定在 [**內容**](/windows/desktop/api/WinNT/ns-winnt-arm64_nt_context) 結構中設定的 XState 功能遮罩。<br/>                             |
