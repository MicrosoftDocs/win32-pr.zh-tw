<span data-ttu-id="02e13-121">大部分的 Windows 程式都已經很容易使用，因為可以使用畫筆而非滑鼠，因此畫筆可以順暢地進行大部分的重要工作和互動，而且程式會回應筆勢。</span><span class="sxs-lookup"><span data-stu-id="02e13-121">Most Windows programs are already pen-friendly in that a pen can be used instead of a mouse, the pen works smoothly for most important tasks and interactions, and the program responds to gestures.</span></span> <span data-ttu-id="02e13-122">當程式協助手寫文字輸入時，就會變得更容易手寫。</span><span class="sxs-lookup"><span data-stu-id="02e13-122">A program becomes handwriting-friendly when it assists with handwritten text input.</span></span> <span data-ttu-id="02e13-123">當程式可以直接處理筆墨時，就會變成啟用筆墨，而不需要將筆刷筆觸轉譯成文字或對等的滑鼠移動。</span><span class="sxs-lookup"><span data-stu-id="02e13-123">A program becomes ink-enabled when it can handle ink directly, instead of requiring that pen strokes be translated to text or equivalent mouse movements.</span></span> <span data-ttu-id="02e13-124">這可讓使用者以自由流動、高品質的數位筆跡撰寫、繪製和新增批註。</span><span class="sxs-lookup"><span data-stu-id="02e13-124">This allows users to write, draw, and add comments in free-flowing, high-quality digital ink.</span></span> <span data-ttu-id="02e13-125">收集筆墨與收集滑鼠事件不同，因為筆墨需要較高的解析度和較高的取樣率，而且也可以增加壓力和傾斜的微妙。</span><span class="sxs-lookup"><span data-stu-id="02e13-125">Collecting ink is different than collecting mouse events, because ink requires higher resolution and a higher sample rate, and it can also add nuance with pressure and tilt.</span></span> <span data-ttu-id="02e13-126">如需建立手寫易記和啟用筆墨之程式的相關資訊，請參閱[使用畫筆](/previous-versions/windows/desktop/ms695501(v=vs.85))[整合筆跡](/previous-versions/windows/desktop/ms700674(v=vs.85))和文字輸入。</span><span class="sxs-lookup"><span data-stu-id="02e13-126">For information about creating handwriting friendly and ink-enabled programs, see [Integrating Ink](/previous-versions/windows/desktop/ms700674(v=vs.85)) and [Text Input Using the Pen](/previous-versions/windows/desktop/ms695501(v=vs.85)).</span></span>

<span data-ttu-id="02e13-127">在定位畫筆時，資料指標的需求較少，因為秘訣代表本身。</span><span class="sxs-lookup"><span data-stu-id="02e13-127">When positioning a pen, there is less need for a cursor because the tip represents itself.</span></span> <span data-ttu-id="02e13-128">不過，若要以協助作為目標，Windows 會提供一個小型的畫筆游標，指出目前的畫筆位置。</span><span class="sxs-lookup"><span data-stu-id="02e13-128">However, for targeting assistance, Windows provides a tiny pen cursor that indicates the current pen location.</span></span> <span data-ttu-id="02e13-129">不同于它所取代的滑鼠指標，除非畫筆接近顯示器，否則不需要畫筆游標，因此它會在閒置幾秒鐘後消失，以讓資訊看不見。</span><span class="sxs-lookup"><span data-stu-id="02e13-129">Unlike the mouse pointer it replaces, the pen cursor is not needed unless the pen is near the display, so it disappears after a few seconds of inactivity to allow an unobstructed view of information.</span></span>

<span data-ttu-id="02e13-130">大部分的畫筆易記程式都支援手勢。</span><span class="sxs-lookup"><span data-stu-id="02e13-130">Most pen-friendly programs support gestures.</span></span> <span data-ttu-id="02e13-131">手勢是在電腦上轉譯為命令，而不是滑鼠移動、書寫或繪製的螢幕上的畫筆快速移動。</span><span class="sxs-lookup"><span data-stu-id="02e13-131">A gesture is a quick movement of the pen on a screen that the computer interprets as a command, rather than as a mouse movement, writing, or drawing.</span></span> <span data-ttu-id="02e13-132">要執行的最快速且最簡單的筆勢之一是筆勢。</span><span class="sxs-lookup"><span data-stu-id="02e13-132">One of the quickest and easiest gestures to perform is a flick.</span></span> <span data-ttu-id="02e13-133">筆勢是一種簡單的手勢，會產生導覽或編輯命令。</span><span class="sxs-lookup"><span data-stu-id="02e13-133">A flick is a simple gesture that results in navigation or an editing command.</span></span> <span data-ttu-id="02e13-134">導覽筆觸包括向上拖曳、向下移動、向後移動和向前移動，而編輯筆觸包含複製、貼上、復原及刪除。</span><span class="sxs-lookup"><span data-stu-id="02e13-134">Navigational flicks include drag up, drag down, move back, and move forward, whereas editing flicks include copy, paste, undo, and delete.</span></span>

<span data-ttu-id="02e13-135">除了忙碌指標以外的所有指標都具有單一圖元熱點，可定義指標的確切畫面位置。</span><span class="sxs-lookup"><span data-stu-id="02e13-135">All pointers except the busy pointer have a single pixel hot spot that defines the exact screen location of the pointer.</span></span> <span data-ttu-id="02e13-136">作用點決定受互動影響的物件。</span><span class="sxs-lookup"><span data-stu-id="02e13-136">The hot spot determines which object is affected by the interaction.</span></span> <span data-ttu-id="02e13-137">物件會定義熱區域，也就是將作用點視為物件上方的區域。</span><span class="sxs-lookup"><span data-stu-id="02e13-137">Objects define a hot zone, which is the area where the hot spot is considered to be over the object.</span></span> <span data-ttu-id="02e13-138">通常，熱區域會與物件的框線一致，但可能會更大，讓互動更容易。</span><span class="sxs-lookup"><span data-stu-id="02e13-138">Typically, the hot zone coincides with the borders of an object, but it may be larger to make interaction easier.</span></span>

## <a name="design-concepts"></a><span data-ttu-id="02e13-142">設計概念</span><span class="sxs-lookup"><span data-stu-id="02e13-142">Design concepts</span></span>

<span data-ttu-id="02e13-143">使用畫筆進行輸入具有下列特性：</span><span class="sxs-lookup"><span data-stu-id="02e13-143">Using a pen for input has the following characteristics:</span></span>

-   <span data-ttu-id="02e13-144">**自然又直覺。**</span><span class="sxs-lookup"><span data-stu-id="02e13-144">**Natural and intuitive.**</span></span> <span data-ttu-id="02e13-145">每個人都知道如何使用畫筆來指向和點擊。</span><span class="sxs-lookup"><span data-stu-id="02e13-145">Everyone knows how to point and tap with a pen.</span></span> <span data-ttu-id="02e13-146">物件互動的設計目的是要與使用者以一致的方式與真實世界中的物件互動。</span><span class="sxs-lookup"><span data-stu-id="02e13-146">Object interactions are designed to correspond to how users interact with objects in the real world in a consistent manner.</span></span>
-   <span data-ttu-id="02e13-147">**表達。**</span><span class="sxs-lookup"><span data-stu-id="02e13-147">**Expressive.**</span></span> <span data-ttu-id="02e13-148">畫筆的筆觸很容易控制，因此撰寫、繪製、繪製、繪製和注釋的方式比使用滑鼠更容易。</span><span class="sxs-lookup"><span data-stu-id="02e13-148">Strokes of a pen are easy to control, making writing, drawing, sketching, painting, and annotating easier than doing so with a mouse.</span></span>
-   <span data-ttu-id="02e13-149">**更個人。**</span><span class="sxs-lookup"><span data-stu-id="02e13-149">**More personal.**</span></span> <span data-ttu-id="02e13-150">就像手寫筆記或簽章比輸入的還多，使用數位手寫筆記或簽章也會更個人化。</span><span class="sxs-lookup"><span data-stu-id="02e13-150">Just as a handwritten note or signature is more personal than a typed one, using a digitally handwritten note or signature is also more personal.</span></span>

-   <span data-ttu-id="02e13-157">**直接且吸引人。**</span><span class="sxs-lookup"><span data-stu-id="02e13-157">**Direct and engaging.**</span></span> <span data-ttu-id="02e13-158">使用畫筆可讓您覺得直接與畫面上的物件互動，而使用滑鼠或觸控板時，一律會要求您將手上的移動與個別的螢幕指標移動互動，而這些移動在比較上是間接的。</span><span class="sxs-lookup"><span data-stu-id="02e13-158">Using a pen makes you feel like you are directly interacting with the objects on the screen, whereas using a mouse or touchpad always requires you to coordinate hand movements with separate on-screen pointer movements which feels indirect by comparison.</span></span>

<span data-ttu-id="02e13-162">幸運的是，如果您的程式已經過妥善設計，而且易於接觸，則提供良好的畫筆支援很簡單。</span><span class="sxs-lookup"><span data-stu-id="02e13-162">Fortunately, if your program is already well designed and is touch-friendly, providing good pen support is easy to do.</span></span> <span data-ttu-id="02e13-163">基於這個目的，設計完善的程式：</span><span class="sxs-lookup"><span data-stu-id="02e13-163">For this purpose, a well-designed program:</span></span>

-   <span data-ttu-id="02e13-168">**具有良好的鍵盤支援。**</span><span class="sxs-lookup"><span data-stu-id="02e13-168">**Has good keyboard support.**</span></span> <span data-ttu-id="02e13-169">這項程式可提供標準的快速鍵指派，讓使用者更有效率，特別是可透過手勢產生的導覽和編輯命令。</span><span class="sxs-lookup"><span data-stu-id="02e13-169">The program makes users efficient by providing standard shortcut key assignments, especially for navigation and editing commands that can also be generated through gestures.</span></span>
