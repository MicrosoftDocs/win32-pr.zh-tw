## <a name="design-concepts"></a><span data-ttu-id="17203-136">設計概念</span><span class="sxs-lookup"><span data-stu-id="17203-136">Design concepts</span></span>

<span data-ttu-id="17203-137">**使用省略號**</span><span class="sxs-lookup"><span data-stu-id="17203-137">**Using ellipses**</span></span>

<span data-ttu-id="17203-138">當命令按鈕用於立即動作時，可能需要更多的資訊來執行動作。</span><span class="sxs-lookup"><span data-stu-id="17203-138">While command buttons are used for immediate actions, more information might be needed to perform the action.</span></span> <span data-ttu-id="17203-139">**指出需要額外資訊的命令 (包括確認) ，方法是在按鈕標籤的結尾加上省略號**。</span><span class="sxs-lookup"><span data-stu-id="17203-139">**Indicate a command that needs additional information (including confirmation) by adding an ellipsis at the end of the button label**.</span></span>

![<span data-ttu-id="17203-140">包含省略號之 [列印命令] 按鈕的螢幕擷取畫面</span><span class="sxs-lookup"><span data-stu-id="17203-140">screen shot of print command button with ellipses</span></span> ](images/ctrl-command-buttons-image4.png)

<span data-ttu-id="17203-141">在此範例中，Print .。。命令會顯示 [列印] 對話方塊，以收集詳細資訊。</span><span class="sxs-lookup"><span data-stu-id="17203-141">In this example, the Print... command displays a Print dialog box to gather more information.</span></span>

![<span data-ttu-id="17203-142">列印命令按鈕、沒有省略號的螢幕擷取畫面</span><span class="sxs-lookup"><span data-stu-id="17203-142">screen shot of print command button, no ellipses</span></span> ](images/ctrl-command-buttons-image5.png)

<span data-ttu-id="17203-143">相反地，在此範例中，Print 命令會將檔的單一複本列印到預設印表機，而不需要進一步的使用者互動。</span><span class="sxs-lookup"><span data-stu-id="17203-143">By contrast, in this example the Print command prints a single copy of a document to the default printer without any further user interaction.</span></span>

<span data-ttu-id="17203-144">**適當地使用省略號很重要，這表示使用者可以在執行動作之前做出進一步的選擇，或甚至完全取消動作**。</span><span class="sxs-lookup"><span data-stu-id="17203-144">**Proper use of ellipses is important to indicate that users can make further choices before performing the action, or even cancel the action entirely**.</span></span> <span data-ttu-id="17203-145">省略號提供的視覺提示可讓使用者探索您的軟體，而不需要擔心。</span><span class="sxs-lookup"><span data-stu-id="17203-145">The visual cue offered by an ellipsis allows users to explore your software without fear.</span></span>

<span data-ttu-id="17203-146">這並不表示每當動作只在執行動作需要額外的資訊時，才會 **顯示另一個視窗時，您應該使用省略號**。</span><span class="sxs-lookup"><span data-stu-id="17203-146">**This doesn't mean you should use an ellipsis whenever an action displays another window** only when additional information is required to perform the action.</span></span> <span data-ttu-id="17203-147">因此， **隱含動詞的任何命令按鈕都不會採用省略號**，例如使用 [關於]、[Advanced]、[說明] (，或任何其他連結至說明主題) 、選項、屬性或設定的命令。</span><span class="sxs-lookup"><span data-stu-id="17203-147">Consequently, **any command button whose implicit verb is to "show another window" doesn't take an ellipsis**, such as with the commands About, Advanced, Help (or any other command linking to a Help topic), Options, Properties, or Settings.</span></span>

<span data-ttu-id="17203-148">一般情況下，會在使用者介面中使用省略號來表示不完整性。</span><span class="sxs-lookup"><span data-stu-id="17203-148">Generally, ellipses are used in user interfaces to indicate incompleteness.</span></span> <span data-ttu-id="17203-149">顯示其他視窗的命令不完整，它們必須顯示另一個視窗，而且不需要額外的資訊來執行其動作。</span><span class="sxs-lookup"><span data-stu-id="17203-149">Commands that show other windows aren't incomplete they must display another window and additional information isn't needed to perform their action.</span></span> <span data-ttu-id="17203-150">這種方法可在橢圓形沒有值的情況下，消除螢幕雜亂。</span><span class="sxs-lookup"><span data-stu-id="17203-150">This approach eliminates screen clutter in situations where ellipses have little value.</span></span>

<span data-ttu-id="17203-151">**注意：** 判斷命令按鈕是否需要省略號時，請不要使用將 [許可權提升](winenv-uac.md) 為因素的需求。</span><span class="sxs-lookup"><span data-stu-id="17203-151">**Note:** When determining if a command button needs an ellipsis, don't use the need to [elevate privileges](winenv-uac.md) as a factor.</span></span> <span data-ttu-id="17203-152">提高許可權不是執行命令 (所需的資訊，而是用於許可權) ，而提高許可權的需求則會以安全性保護盾表示。</span><span class="sxs-lookup"><span data-stu-id="17203-152">Elevation isn't information needed to perform a command (rather, it's for permission) and the need to elevate is indicated with the security shield.</span></span>

<span data-ttu-id="17203-153">**如果您只做一件事 ...** 使用簡潔、明確、簡單易懂的標籤，清楚描述命令按鈕所執行的動作，並在適當的情況下使用省略號。</span><span class="sxs-lookup"><span data-stu-id="17203-153">**If you do only one thing...** Use a concise, specific, self-explanatory label that clearly describes the action that the command button performs, and use an ellipsis when appropriate.</span></span>
