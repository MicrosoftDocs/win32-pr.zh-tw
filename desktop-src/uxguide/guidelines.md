# <a name="guidelines"></a>指導方針

> [!NOTE]
> 此設計指南是針對 Windows 7 所建立，而且尚未針對較新的 Windows 版本進行更新。 大部分的指引仍然適用于準則，但展示和範例不會反映我們目前的 [設計指引](/windows/uwp/design/)。

這些章節包含 Windows 型桌面應用程式的詳細使用者體驗指導方針。

## <a name="in-this-section"></a>本節內容



| 主題                                                     | 描述                                                                                                                                                                                                                                                                                      |
|-----------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [控制項](controls.md)<br/>                       | 控制項是使用者在應用程式主視窗區域上與使用者互動的 UI 元素。 請參閱以 Windows 為基礎的桌面應用程式中控制項的視覺效果範例，並取得每個控制項的指導方針連結。<br/>                                                                                   |
| [命令](commands.md)<br/>                       | 命令是使用者在使用您的應用程式時可以採取的動作。 瞭解將命令新增至您的應用程式功能表、功能區和工具列的指導方針。<br/>                                                                                                                                        |
| [Text](text.md)<br/>                               | 文字包含使用者可在您的應用程式中看到的任何文字。 請參閱這些指導方針，以瞭解如何使用應用程式的 UI 文字、樣式和語氣。 <br/>                                                                                                                                                      |
| [訊息](messages.md)<br/>                       | 訊息是使用者在使用您的應用程式時，所需的任何訊息類型或想要查看的訊息。 瞭解如何在您的應用程式中呈現錯誤、警告、確認和通知。<br/>                                                                                                                  |
| [互動](interaction.md)<br/>                 | 互動是使用者與應用程式互動的各種方式，包括觸控、鍵盤、滑鼠等等。 使用這些指導方針來建立可針對觸控進行優化，但在輸入裝置間一致運作的直覺和獨特體驗。<br/>                           |
| [Windows](windows.md)<br/>                         | Windows 是桌面應用程式的主要「畫布」或 UI 介面，包括主視窗本身和快顯、對話方塊和嚮導。 當您決定要使用的介面和最佳使用方式時，請遵循這些指導方針。<br/>                                                         |
| [視覺效果](visuals.md)<br/>                         | 視覺效果包括控制項以外的視覺元素。 這些指導方針可協助您在應用程式中進行版面配置、字型、色彩、圖示等的決策。<br/>                                                                                                                       |
| [體驗](experiences.md)<br/>                 | 體驗是所有應用程式的常見體驗和使用案例，例如設定、首次執行和列印。 瞭解建立這些體驗的最佳作法，以及如何與您的應用程式品牌溝通。<br/>                                                                              |
| [Windows 環境](windows-environment.md)<br/> | Windows 環境是 Windows 所提供的螢幕上工作區域，類似于實體桌面，以及作業系統的核心擴充點。 瞭解如何利用應用程式的桌面、工作列、通知區域、控制台、說明和使用者帳戶控制。<br/> |
