# <a name="desktop"></a>桌面

> [!NOTE]
> 此設計指南是針對 Windows 7 所建立，而且尚未針對較新版本的 Windows 更新。 大部分的指引仍然適用于準則，但展示和範例不會反映我們目前的 [設計指引](/windows/uwp/design/)。

桌面是其程式的使用者工作區。 它無法提升您對程式或其品牌的認知。 不要濫用！

桌上型電腦是 Microsoft Windows 所提供的螢幕上的工作區，類似于實體桌面。 它是由 [工作區](glossary.md) 和 [工作列](winenv-taskbar.md)所組成。 工作區域可以橫跨多個監視器。

![一般 windows 桌面的螢幕擷取畫面 ](images/winenv-desktop-image1.png)

一般的 Windows 桌面。

作用中的監視器是使用中的程式正在執行的監視器。 預設監視器是具有 [開始] 功能表、工作列和通知區域的監視器。

## <a name="design-concepts"></a>設計概念

Windows 桌面具有下列程式存取點：

-   **工作區。** 使用者可以執行其工作的螢幕區域，以及儲存程式、檔和其快捷方式。 雖然在技術上，桌面包含工作列，但在大部分的情況下，它只會參考工作區域。
-   **[開始] 按鈕。** 所有程式和特殊 Windows 的存取點都將 (檔、圖片、音樂、遊戲、電腦主控台) 、使用「最近使用的」清單來快速存取最近使用的程式和檔。
-   **快速啟動。已從 Windows 7 移除。** 使用者選取之程式的直接存取點。
-   **任務。** 執行具有桌上型電腦狀態之程式的存取點。 雖然在技術上，工作列橫跨 [開始] 按鈕到通知區域的整個橫條，但是在大部分的內容中，工作列都指的是介於中間的區域，其中包含工作列按鈕。 此區域有時稱為 taskband。
-   **Deskbands.不建議使用。** 最小化功能、長時間執行的程式，例如語言列。 最小化至 deskbands 的程式不會在最小化時顯示工作列按鈕。
-   **通知區域。** 通知和狀態的短期來源，以及不存在於桌面上的系統和程式相關功能的存取點。

![[開始] 按鈕、工作列、縮圖的螢幕擷取畫面 ](images/winenv-desktop-image2.png)

Windows 桌面存取點包含 [開始] 按鈕、工作列和通知區域。 請注意工作列按鈕的縮圖功能。

**Windows 桌面是受限的共用資源，是使用者進入 Windows 的進入點。讓使用者保持控制。** 您應以預期的方式使用其區域，任何其他使用方式都應視為濫用。 絕對不要將它們視為提升您的程式或其 [品牌](exper-branding.md)認知的方式。

在 Windows 7 中，原始設備製造商 (Oem) 和獨立硬體廠商 (Ihv) 可以使用 Device Stage 來設計電腦和裝置的自訂、品牌化 UI，而不會讓使用者的桌上型電腦變得雜亂。 尤其是 Oem 可以使用 Device Stage PC，來提供自訂程式、服務供應專案及支援的功能。 如需詳細資訊，請參閱 [Microsoft 裝置體驗開發工具組](https://www.microsoft.com/whdc/device/DeviceExperience/Dev-Kit.mspx)。

**如果您只做一件事 .。。**

-   不要濫用桌面，讓使用者保持掌控。 如果您的目標使用者可能經常使用您的程式，請在安裝期間提供選項，將快捷方式放在桌面上（預設為未選取）。

## <a name="guidelines"></a>指導方針

-   **如果您的使用者很可能經常使用您的程式，請在安裝期間提供選項，將程式快捷方式放在桌面上。** 大部分的程式都不常使用，以保證提供此選項。
-   **預設會顯示未選取的選項。** 要求使用者選取此選項很重要，因為在桌上型電腦上不想要的圖示之後，許多使用者都不願意移除它們。 這可能會導致不必要的桌面雜亂。
-   **如果使用者選取選項，請只提供單一程式快捷方式。** 如果您的產品是由多個程式所組成，請只提供主要程式的快捷方式。
-   **只將程式快捷方式放在桌面上。** 請勿放置實際的程式或其他類型的檔案。

    **正確：**

    ![具有箭號重迭之快捷方式圖示的螢幕擷取畫面 ](images/winenv-desktop-image3.png)

    **不正確：**

    ![程式圖示的螢幕擷取畫面 ](images/winenv-desktop-image4.png)

    在不正確的範例中，會將程式（而不是快捷方式）複製到桌面。

-   **選擇可在不截斷的情況下顯示的標籤。** 使用者不應看見省略號。

    **正確：**

    ![具有完整名稱之快捷方式圖示的螢幕擷取畫面 ](images/winenv-desktop-image3.png)

    **不正確：**

    ![以截斷名稱顯示快捷方式圖示的螢幕擷取畫面 ](images/winenv-desktop-image5.png)

    在不正確的範例中，程式快捷方式標籤很長，因此會被截斷。
