# <a name="window-stations-and-desktops"></a>視窗工作站和桌上型電腦

Windows 提供三種主要的物件類別：使用者介面、圖形裝置介面 (GDI) 和核心。 [核心物件](/windows/desktop/SysInfo/kernel-objects) 是安全的，而 [使用者物件](/windows/desktop/SysInfo/user-objects) 和 [GDI 物件](/windows/desktop/SysInfo/gdi-objects) 則不是。 因此，為了提供額外的安全性，使用者介面物件是使用視窗工作站和桌面來管理，這本身就是安全物件。

如需詳細資訊，請參閱下列主題：

-   [關於視窗工作站和桌上型電腦](about-window-stations-and-desktops.md)
-   [視窗工作站和桌面參考](window-station-and-desktop-reference.md)
