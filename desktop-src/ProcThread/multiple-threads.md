# <a name="multiple-threads"></a>多個執行緒

*執行緒* 是可排程執行之進程內的實體。 進程的所有線程都會共用其虛擬位址空間和系統資源。 每個進程都是使用單一執行緒來啟動，但可以從其任何執行緒建立額外的執行緒。
