## <a name="installing-an-input-method-editor"></a>安裝輸入方法編輯器

下列各節說明如何安裝和使用輸入法編輯器 (Ime) ，以四種不同的東亞語言輸入複雜字元。 本文將討論每種語言特有的功能。

若要在應用程式中執行輸入方法編輯器 (IME) 功能，請參閱 [在遊戲中使用輸入法編輯器](/windows/desktop/DxTechArts/using-an-input-method-editor-in-a-game)。

根據預設，Microsoft Windows XP 系統上不會安裝 IME。 若要安裝，請完成下列步驟。

**安裝 IME**

1.  從主控台中，開啟 [地區及語言選項]。
2.  在 [語言] 索引標籤上，選取 [安裝東亞語言的檔案] 核取方塊。

    ![地區和語言選項的 [語言] 索引標籤](images/ime-1.png)

    [安裝補充語言支援] 對話方塊會出現，通知您語言檔案的儲存需求。

    ![安裝補充語言支援對話方塊](images/ime-install-lang-dialog.png)

3.  按一下 [確定] 關閉對話方塊。
4.  按一下 [語言] 索引標籤上的 [確定]。
5.  接著會出現另一個對話方塊，要求 Windows XP 安裝磁片或語言支援檔案所在的網路共用位置。 插入 Windows XP compact 磁片，或流覽至適當的網路位置，然後按一下 [確定]。 Microsoft Windows 會安裝必要的檔案，並提示您重新開機電腦。
6.  按一下 [是] 以重新啟動電腦。
7.  重新開機之後，再次開啟 [地區及語言選項] 控制台。
8.  在 [語言] 索引標籤上，按一下 [詳細資料]。 [文字服務和輸入語言] 視窗隨即出現。

    ![ext services 和輸入語言視窗](images/ime-2.png)

9.  在 [設定] 索引標籤上，按一下 [新增]。 [新增輸入語言] 視窗隨即出現。

    ![新增輸入語言視窗](images/ime-3.png)

10. 針對輸入語言選取中文 (臺灣) ，並針對鍵盤配置/IME 選取 Microsoft 新的語音輸入法2002a。
11. 按一下 [確定]。 現在您可以使用類似的方式來新增其他語言和 Ime。
12. 再按一次 [新增]，選取 [中文 (PRC]) 輸入語言和中文 (簡化) -Microsoft 針對鍵盤配置/輸入法3.0 的拼音輸入法，然後按一下 [確定]。
13. 按一下 [新增]，選取 [日文] 作為輸入語言，並選取 [Microsoft IME 標準 2002 ver]。 8.1 針對鍵盤配置/輸入法，然後按一下 [確定]。
14. 再按一次 [新增]，選取 [韓文] 作為 [輸入語言] 和 [韓文輸入系統] (IME 2002) 的鍵盤配置/輸入，然後按一下 [確定] 在 [文字服務和輸入語言] 視窗中，[已安裝的服務] 清單方塊現在應該包含四個新加入的 Ime。

    ![已安裝的服務清單方塊](images/ime-7.png)

15. 按一下 [確定] 以關閉 [文字服務和輸入語言] 視窗。
16. 按一下 [確定] 關閉 [地區及語言選項] 控制台。 Windows 的工作列現在應該包含以紅色圈起的輸入地區設定指標。 指標是否存在表示系統上已安裝一個以上的輸入語言。

    ![表示系統上已安裝一個以上輸入語言的指標](images/ime-8.png)

## <a name="simplified-chinese-ime"></a>簡體中文 IME

本節說明如何使用簡體中文 IME (拼音) 搭配 Microsoft 記事本來輸入幾個中文字元。

1.  啟動 [開始] 按鈕提供記事本 (，然後選取 [所有程式和附屬應用程式]) 。 在記事本中輸入一些字元。 這些字元將協助您在稍後更清楚地視覺化 IME 視窗。

    ![螢幕擷取畫面，它會顯示一些字元，可協助您在稍後針對簡體中文將 I M E 視窗視覺化。](images/ime-sc1.png)

2.  使用記事本作為作用中應用程式時，請按一下輸入地區設定指標，然後選取 [中文 (PRC]) 。 指標顯示會變更為 CH，以反映新的輸入語言是中文 (PRC) 。

    ![顯示輸入地區設定指標以選取中文 (P R C) 的螢幕擷取畫面。](images/ime-sc2.png)

3.  將游標放在記事本中。 按下鍵盤上的 [首頁]，讓游標位於行的開頭。 在鍵盤上輸入 "N"，然後輸入 "I"。 下圖顯示顯示的外觀。 小型水準矩形是讀取視窗，會顯示目前的讀取字串。 目前，由於輸入 "N" 和 "I"，讀取字串是 "ni"。

    ![顯示讀取字串為 ' n ' 和 ' i ' 的螢幕擷取畫面。](images/ime-sc3.png)

4.  輸入 "3"。 現在記事本有下列顯示。 因為 N + I + 3 是簡體中文拼音的完整發音，所以 IME 有足夠的資訊來預期使用者可能要輸入的字元。 因為您輸入了完整的發音，所以讀取視窗消失。 字元會顯示在記事本資料指標的上方。 此字元不是記事本的一部分，而是顯示在記事本頂端的另一個視窗中，並隱藏下記事本中的現有字元。 這個新視窗稱為組合視窗，其中的字串稱為組合字元串。 組合字元串會在顯示中加上底線。

    ![顯示組合視窗的螢幕擷取畫面，其中包含單一字元的組合字元串。](images/ime-sc4.png)

5.  現在輸入 "H"、"A"、"O"、"3" 來輸入另一個字元。 請注意，當輸入 "H" 時，讀取視窗會顯示，而且當輸入 "3" 時，會消失。 如下所示，組合字元串現在包含兩個字元。

    ![顯示組合視窗的螢幕擷取畫面，其中包含兩個字元的組合字元串。](images/ime-sc5.png)

6.  按下鍵盤上的向左鍵。 組合游標會在您輸入的第二個字元處向左移動一個字元。 視窗會出現在記事本的上方，如下所示。 此視窗稱為候選視窗。 它會顯示符合您所輸入發音的字元或片語清單。 您可以從候選清單中的專案選取所需的文字。 在此範例中，有兩個候選字元可用於相同的發音。

    ![有兩個可用於相同發音的候選字元](images/ime-sc6.png)

7.  輸入 "2" 來選取第二個專案。 候選視窗現在會關閉，然後以選取的字元更新撰寫字串。

    ![顯示以選取的字元更新的組合字元串的螢幕擷取畫面。](images/ime-sc7.png)

8.  按 ENTER 鍵。 這會告知 IME 已完成組合，而字串應該傳送至應用程式記事本在此範例中。 組合視窗會關閉，而且會透過 WM 字元將兩個字元傳送至記事本 \_ 。 底線會出現在下圖中，因為所顯示的兩個字元是記事本中文字的一部分。 記事本中的現有文字 "ABCDEFG" 已移至右側，因為已插入兩個以上的字元。 您現在已成功使用 IME 輸入兩個簡體中文字元。

    ![已成功使用 ime 輸入兩個簡體中文字元](images/ime-sc8.png)

    ![有三個可供相同發音使用的候選字元](images/ime-tc6.png)

## <a name="japanese-ime"></a>日文輸入法

本節逐步解說如何使用日文 IME 搭配記事本來輸入幾個日文字元。

1.  啟動 [記事本]。 在記事本中輸入一些字元。 這些字元將協助您在稍後更清楚地視覺化 IME 視窗。

    ![顯示字元的螢幕擷取畫面，可協助您在稍後針對日文將 I M E 視窗視覺化。](images/ime-j1.png)

2.  使用記事本作為使用中的應用程式，按一下輸入地區設定指標，然後選取 [日文]。 指標顯示會變更為 JP，以反映新的輸入語言是日文。

    ![輸入地區設定指標以選取日文](images/ime-j2.png)

3.  將游標放在記事本中。 按下鍵盤上的 [首頁]，讓游標位於行的開頭。 在鍵盤上輸入 "N"，然後輸入 "I"。 下圖顯示顯示的外觀。 因為 N + I 是日文的完整發音，所以 IME 有足夠的資訊來預期使用者可能要輸入的字元。 字元會顯示在記事本資料指標的上方。 此字元不是記事本的一部分，而是顯示在記事本頂端的另一個視窗中，並隱藏下記事本中的現有字元。 這個新視窗稱為組合視窗，其中的字串稱為組合字元串。 組合字元串會在顯示中加上底線。

    ![顯示組合字元串的螢幕擷取畫面，其中有一個日文字元加上底線。](images/ime-j3.png)

4.  現在輸入 "H"、"O"、"N"、"G" 和 "O" 以輸入兩個字元。 組合字元串現在包含四個字元，如下所示。

    ![顯示有四個日文字元的組合字元串的螢幕擷取畫面。](images/ime-j4.png)

5.  按下空格鍵。 這會指示 IME 將輸入的文字轉換成子句。 在下圖中，IME 將發音 "Nihongo" 轉換成以日文漢字撰寫的子句，也就是「日文語言」。

    ![ime 轉換日文發音](images/ime-j5.png)

6.  按下鍵盤上的向下箭號一次。 視窗會出現在記事本的上方，如下所示。 此視窗稱為候選視窗。 它會顯示符合您所輸入發音的子句清單。 您可以從候選清單中選取想要的字組。 在此範例中，有三個候選字元可用於相同的發音。 請注意，第二個專案已反白顯示，而且組合字元串會變更。 這是藉由輸入向下箭號所造成的，這會告知 IME 選取先前顯示的專案。

    ![顯示候選視窗的螢幕擷取畫面。](images/ime-j6.png)

7.  輸入 "2" 來選取第二個專案。 候選視窗現在會關閉，然後以選取的字元更新撰寫字串。

    ![顯示以選取的日文字元更新的組合字元串的螢幕擷取畫面。](images/ime-j7.png)

8.  按 ENTER 鍵。 這會告知 IME 已完成組合，而字串應該傳送至應用程式記事本在此範例中。 組合視窗會關閉，而且會透過 [**WM \_ 字元**](/windows/desktop/inputdev/wm-char)將兩個字元傳送至記事本。 底線會出現在下圖中，因為所顯示的兩個字元是記事本中文字的一部分。 記事本中的現有文字 "ABCDEFG" 已移至右側，因為已插入兩個以上的字元。 您現在已成功使用 IME 輸入幾個日文字元。

    ![使用 ime 成功輸入幾個日文字元](images/ime-j8.png)

## <a name="korean-ime"></a>韓文輸入法

本節說明如何搭配使用韓文 IME 與記事本來輸入幾個韓文字元。

1.  啟動 [記事本]。 在記事本中輸入一些字元。 這些字元將協助您在稍後更清楚地視覺化 IME 視窗。

    ![可協助您在稍後更方便視覺化 ime 視窗的字元](images/ime-k1.png)

2.  使用記事本作為作用中應用程式時，請按一下輸入地區設定指標，然後選取 [韓文]。 指標顯示會變更為 KO，以反映新的輸入語言是韓文。

    ![輸入地區設定指標以選取韓文](images/ime-k2.png)

3.  將游標放在記事本中。 按下鍵盤上的 [首頁]，讓游標位於行的開頭，然後輸入 "G"。 下圖顯示顯示的外觀。 對應到 "G" 的注音元素會出現在記事本上，並以區塊游標反白顯示。 此反白顯示的字元稱為組合字元串。 請注意，與其他語言的輸入法不同的是，撰寫字串會傳送給記事本，並在使用者輸入單一拼音專案時，立即插入現有文字的左邊。

    ![顯示組合字元串的螢幕擷取畫面，其中有一個字元插入至現有文字的左邊。](images/ime-k3.png)

4.  目前，組合字元串是由過渡字元所組成，因為使用者輸入的任何其他拼音元素都會就地變更撰寫字串。 現在輸入 "K"，然後輸入 "S"。 請注意，每次擊鍵都會變更中間字元。

    ![組合字元串](images/ime-k4.png)

5.  現在按下 CTRL 鍵。 出現的候選視窗會列出您可以針對輸入的發音（G + K + S）選取的中文字元。

    ![顯示候選視窗的螢幕擷取畫面，其中包含您可以在視窗底部選取的中文字元清單。](images/ime-k5.png)

6.  輸入數位 "1" 以選取清單中的第一個專案。 候選視窗會關閉，並使用選取的字元更新撰寫字串。

    ![以選取的字元更新的組合字元串](images/ime-k6.png)

7.  輸入 "R"、"N" 和 "R"。 然後按向右鍵，輸入另一個字元。

    ![候選視窗](images/ime-k7.png)

8.  輸入 "1" 以選取第一個專案。 您現在已成功使用 IME 輸入兩個韓文字元。 韓文字元已是記事本中文字字串的一部分。

    ![使用 ime 成功輸入兩個韓文字元](images/ime-k8.png)
