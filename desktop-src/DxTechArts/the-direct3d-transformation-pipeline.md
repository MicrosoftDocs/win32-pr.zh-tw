## <a name="overview"></a><span data-ttu-id="bb300-108">概觀</span><span class="sxs-lookup"><span data-stu-id="bb300-108">Overview</span></span>

<span data-ttu-id="bb300-109">Direct3D 使用三個轉換在像素座標（螢幕空間）中變更您的 3D 模型座標。</span><span class="sxs-lookup"><span data-stu-id="bb300-109">Direct3D uses three transformations to change your 3D model coordinates into pixel coordinates (screen space).</span></span> <span data-ttu-id="bb300-110">這些轉換為世界轉換、檢視轉換和投影轉換。</span><span class="sxs-lookup"><span data-stu-id="bb300-110">These transformations are world transform, view transform, and projection transform.</span></span>

<span data-ttu-id="bb300-111">「世界轉換」可控制如何將模型座標轉換成全局座標。</span><span class="sxs-lookup"><span data-stu-id="bb300-111">World transform controls how model coordinates are transformed into world coordinates.</span></span> <span data-ttu-id="bb300-112">全球轉換可以包含翻譯、旋轉和 scalings，但不適用於燈光。</span><span class="sxs-lookup"><span data-stu-id="bb300-112">World transform can include translations, rotations, and scalings, but it does not apply to lights.</span></span> <span data-ttu-id="bb300-113">如需有關使用世界轉換的詳細資訊，請參閱「 [世界轉換](/windows/desktop/direct3d9/world-transform)」。</span><span class="sxs-lookup"><span data-stu-id="bb300-113">For more information on working with world transforms, see [World Transform](/windows/desktop/direct3d9/world-transform).</span></span>

<span data-ttu-id="bb300-114">View 轉換控制從全局座標轉換成「相機空間」，以決定世界中的相機位置。</span><span class="sxs-lookup"><span data-stu-id="bb300-114">View transform controls the transition from world coordinates into "camera space," determining camera position in the world.</span></span> <span data-ttu-id="bb300-115">如需使用 view 轉換的範例，請參閱 [視圖轉換](/windows/desktop/direct3d9/view-transform)。</span><span class="sxs-lookup"><span data-stu-id="bb300-115">For an example of working with view transforms, see [View Transform](/windows/desktop/direct3d9/view-transform).</span></span>

<span data-ttu-id="bb300-116">投影轉換會將幾何從相機空間變更為「剪輯空間」，並套用透視扭曲。</span><span class="sxs-lookup"><span data-stu-id="bb300-116">Projection transform changes the geometry from camera space into "clip space" and applies perspective distortion.</span></span> <span data-ttu-id="bb300-117">「剪輯空間」一詞是指在此轉換期間將幾何裁剪到視圖磁片區的方式。</span><span class="sxs-lookup"><span data-stu-id="bb300-117">The term "clip space" refers to how the geometry is clipped to the view volume during this transform.</span></span> <span data-ttu-id="bb300-118">如需使用投影轉換的範例，請參閱 [投射轉換](/windows/desktop/direct3d9/projection-transform)。</span><span class="sxs-lookup"><span data-stu-id="bb300-118">For an example of working with projection transforms, see [Projection Transform](/windows/desktop/direct3d9/projection-transform).</span></span>

<span data-ttu-id="bb300-121">剪切和轉換頂點必須在同質空間中進行 (單純放置的空間、座標系統) 包含第四個元素的空間，但大部分應用程式的最終結果都必須是「螢幕空間」中所定義的非同質三維 (3D) 座標。</span><span class="sxs-lookup"><span data-stu-id="bb300-121">Clipping and transforming vertices must take place in homogenous space (simply put, space in which the coordinate system includes a fourth element), but the final result for most applications needs to be non-homogenous three-dimensional (3D) coordinates defined in "screen space."</span></span> <span data-ttu-id="bb300-122">這表示輸入頂點和裁剪磁片區必須轉譯成同質空間，才能執行裁剪，然後轉譯回非同質空間來顯示。</span><span class="sxs-lookup"><span data-stu-id="bb300-122">This means that both the input vertices and the clipping volume must be translated into homogenous space to perform the clipping and then translated back into non-homogenous space to be displayed.</span></span>
