# <a name="creating-a-private-sensor-pool"></a>建立私人感應器集區

私用感應器集區是保留給用戶端應用程式專用的生物識別單位集合。 私人集區支援專屬的驗證方法，並可讓用戶端應用程式使用廠商指定的控制命令來存取生物識別單位。
