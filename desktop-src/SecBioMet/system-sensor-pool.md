<span data-ttu-id="e8b1c-105">系統感應器集區是可共用的生物特徵辨識單位集合，可提供 Windows 驗證服務的存取權。</span><span class="sxs-lookup"><span data-stu-id="e8b1c-105">The system sensor pool is a collection of sharable biometric units that provide access to Windows authentication services.</span></span> <span data-ttu-id="e8b1c-106">此集區是由 Winlogon、UAC 和任何其他用戶端所使用，這些用戶端會將 SID 與特定生物特徵辨識範本產生關聯。</span><span class="sxs-lookup"><span data-stu-id="e8b1c-106">This pool is used by Winlogon, UAC, and any other client that associates a SID with a specific biometric template.</span></span> <span data-ttu-id="e8b1c-107">系統集區中的生物識別單位：</span><span class="sxs-lookup"><span data-stu-id="e8b1c-107">Biometric units in the system pool:</span></span>

-   <span data-ttu-id="e8b1c-108">可以由多個用戶端應用程式共用。</span><span class="sxs-lookup"><span data-stu-id="e8b1c-108">Can be shared by multiple client applications.</span></span>
-   <span data-ttu-id="e8b1c-109">將完成生物特徵辨識作業所產生的事件通知傳送給具有目前視窗焦點的應用程式。</span><span class="sxs-lookup"><span data-stu-id="e8b1c-109">Send event notices generated by the completion of biometric operations only to the application that has current window focus.</span></span>
-   <span data-ttu-id="e8b1c-110">使用帳戶 Sid 來代表範本身分識別。</span><span class="sxs-lookup"><span data-stu-id="e8b1c-110">Use account SIDs to represent the template identities.</span></span> <span data-ttu-id="e8b1c-111">所有與單一使用者帳戶相關聯的範本都會以指派給該帳戶的 SID 標記。</span><span class="sxs-lookup"><span data-stu-id="e8b1c-111">All of the templates associated with a single user account are tagged with the SID assigned to that account.</span></span>
-   <span data-ttu-id="e8b1c-112">取決於 Windows 生物特徵辨識服務所提供的可信任範本儲存體。</span><span class="sxs-lookup"><span data-stu-id="e8b1c-112">Depend on trustworthy template storage provided by the Windows Biometric Service.</span></span>

<span data-ttu-id="e8b1c-113">您可以在系統集區中包含生物識別單位（如果可能的話）：</span><span class="sxs-lookup"><span data-stu-id="e8b1c-113">A biometric unit can be included in the system pool if it can be:</span></span>

-   <span data-ttu-id="e8b1c-114">設定為在基本模式下運作，只做為生物特徵辨識捕獲裝置。</span><span class="sxs-lookup"><span data-stu-id="e8b1c-114">Configured to operate in basic mode and act only as a biometric capture device.</span></span>
-   <span data-ttu-id="e8b1c-115">設定為以 advanced 模式操作，但沒有上架的範本儲存體。</span><span class="sxs-lookup"><span data-stu-id="e8b1c-115">Configured to operate in advanced mode but has no onboard template storage.</span></span> <span data-ttu-id="e8b1c-116">也就是說，它必須使用 Microsoft 提供的儲存體介面卡和範本存放區。</span><span class="sxs-lookup"><span data-stu-id="e8b1c-116">That is, it must use the storage adapter and template store supplied by Microsoft.</span></span>
-   <span data-ttu-id="e8b1c-117">設定為以 advanced 模式運作、包含上架範本儲存體，而且可以產生必要的雜湊。</span><span class="sxs-lookup"><span data-stu-id="e8b1c-117">Configured to operate in advanced mode, contains onboard template storage, and can generate the required hashes.</span></span>

<span data-ttu-id="e8b1c-118">當新的感應器裝置插入電源時，Windows 生物特徵辨識服務會為其建立生物特徵辨識單位，並嘗試設定該單位以供系統感應器集區使用。</span><span class="sxs-lookup"><span data-stu-id="e8b1c-118">When a new sensor device is plugged in, the Windows Biometric Service creates a biometric unit for it and tries to configure that unit for use by the system sensor pool.</span></span> <span data-ttu-id="e8b1c-119">如果設定失敗，則會將生物特徵辨識單位放在未指派的感應器集區中。</span><span class="sxs-lookup"><span data-stu-id="e8b1c-119">If the configuration is not successful, the biometric unit is placed in the unassigned sensor pool.</span></span>
