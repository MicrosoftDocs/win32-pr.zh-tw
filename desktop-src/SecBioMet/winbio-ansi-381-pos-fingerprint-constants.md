## <a name="remarks"></a>備註

WINBIO \_ 手指 \_ 未指定的 \_ POS \_ \* 常數不是 ANSI 381 規格的一部分。 您可以使用這些常數來註冊每個帳戶 SID 的最多10個手指 (，) 沒有相關聯的位置資訊。

> [!IMPORTANT]
>
> 請勿嘗試驗證指紋常數值。 Windows 生物特徵辨識服務會先驗證提供的指紋常數值，然後才將它們傳遞給您的執行。
