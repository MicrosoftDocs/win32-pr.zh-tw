# <a name="winbio_component-constants"></a>WINBIO \_ 元件常數

呼叫 [**WinBioControlUnit**](/windows/desktop/api/Winbio/nf-winbio-winbiocontrolunit) 或 [**WinBioControlUnitPrivileged**](/windows/desktop/api/Winbio/nf-winbio-winbiocontrolunitprivileged) 來指定所使用的介面卡類型時，可以使用下列常數。



| 常數                                                                                                                                                                                        | 描述                             |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------|
| <span id="WINBIO_COMPONENT_SENSOR"></span><span id="winbio_component_sensor"></span><dl> <dt>**WINBIO \_ 元件 \_ 感應器**</dt> </dl>    | 指定感應器介面卡。<br/>  |
| <span id="WINBIO_COMPONENT_ENGINE"></span><span id="winbio_component_engine"></span><dl> <dt>**WINBIO \_ 元件 \_ 引擎**</dt> </dl>    | 指定引擎介面卡。<br/> |
| <span id="WINBIO_COMPONENT_STORAGE"></span><span id="winbio_component_storage"></span><dl> <dt>**WINBIO \_ 元件 \_ 儲存體**</dt> </dl> | 指定存放裝置介面卡。<br/> |

| 需求 | 值 |

| 最低支援的伺服器<br/> | Windows僅限 Server 2008 R2 \[ desktop 應用程式\]<br/>                                                       |
