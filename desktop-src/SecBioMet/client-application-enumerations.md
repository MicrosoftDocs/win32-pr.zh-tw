# <a name="client-application-enumerations"></a>用戶端應用程式列舉

Windows 生物特徵辨識架構 API 支援下列列舉類型。

## <a name="in-this-section"></a>本節內容



| 主題                                                                                      | 描述                                                                                                                                       |
|--------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------|
| [**WINBIO \_ 反 \_ 詐騙 \_ 原則 \_ 動作**](winbio-anti-spoof-policy-action.md)<br/>  | 指定您針對使用者的 lnk-antispoofing 原則所採取的動作類型。 <br/>                                                        |
| [**WINBIO \_ 非同步 \_ 通知 \_ 方法**](/windows/desktop/api/Winbio/ne-winbio-winbio_async_notification_method)<br/> | 定義常數，指定如何將非同步作業的完成通知傳遞至用戶端應用程式。<br/> |
| [**WINBIO \_ 認證 \_ 格式**](winbio-credential-format.md)<br/>                  | 定義可以用來指定使用者認證格式的旗標。<br/>                                                              |
| [**WINBIO \_ 認證 \_ 狀態**](winbio-credential-state.md)<br/>                    | 定義值，這個值會指定認證是否已與終端使用者的生物特徵辨識資料相關聯。<br/>                          |
| [**WINBIO \_ 認證 \_ 類型**](winbio-credential-type.md)<br/>                      | 定義可以用來篩選認證類型的旗標。<br/>                                                                       |
| [**WINBIO \_ 原則 \_ 來源**](winbio-policy-source.md)<br/>                          | 列出針對生物特徵辨識因素偵測詐騙的可能來源原則資訊。<br/>                                  |
