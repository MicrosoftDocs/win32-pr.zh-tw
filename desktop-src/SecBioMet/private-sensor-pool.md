# <a name="private-sensor-pool"></a>私人感應器集區

私用感應器集區是保留給用戶端應用程式專用的生物識別單位集合。 私人集區支援專屬的驗證方法，並可讓用戶端應用程式使用廠商指定的控制命令來存取生物識別單位。 私人感應器集區中的生物識別單位：

-   僅適用于建立集區的用戶端應用程式。
-   將完成生物特徵辨識作業產生的事件通知直接傳送至應用程式，而不考慮目前的視窗焦點。
-   使用 Guid 來識別生物特徵辨識範本。
-   共用單一的應用程式選取的範本資料庫。

如果用戶端應用程式有下列情況，則必須使用私用感應器集區：

-   管理使用特定應用程式範本資料庫的專用生物特徵辨識單位集合。 例如，請考慮員工出席應用程式，其中員工會在指紋辨識器上將其手指輕觸，以表示其在工作抵達的位置。 員工在執行應用程式的電腦上沒有 Windows 帳戶。 相反地，其指紋是由出席應用程式所管理的 Guid 所識別。
-   收集生物特徵辨識範例，而不只是將範例對應至 Sid。
-   將生物識別單元硬體置於維護模式，以更新固件。
-   將廠商定義的控制命令傳送到生物特徵辨識單位硬體或軟體。
-   嘗試使用上架儲存體來設定生物特徵辨識單位，以在 advanced 模式下操作，但該單元無法執行必要的雜湊作業。
-   從遠端桌面用戶端會話執行。

> [!Note]  
> 應用程式只能針對指紋生物識別建立私人感應器集區。 如果應用程式嘗試為其他任何人建立一個 (例如，臉部) ，要求將會失敗。
