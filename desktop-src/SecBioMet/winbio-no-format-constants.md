# <a name="winbio_no_format-constants"></a>WINBIO \_ 沒有 \_ 格式常數

如果 [**WINBIO \_ 註冊 \_ 格式**](winbio-registered-format.md) 結構的成員不可為零，您可以使用下列常數來檢查錯誤。



| 常數/值                                                                                                                                                                                                                                                                   | 描述                                                                                          |
|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------|
| <span id="WINBIO_NO_FORMAT_OWNER_AVAILABLE"></span><span id="winbio_no_format_owner_available"></span><dl> <dt>**WINBIO \_沒有 \_ 格式 \_ 擁有者 \_ 可用**</dt> <dt>0</dt> </dl> | 沒有任何 IBIA (國際生物特徵辨識產業關聯) 指派的擁有者值可供使用。<br/> |
| <span id="WINBIO_NO_FORMAT_TYPE_AVAILABLE"></span><span id="winbio_no_format_type_available"></span><dl> <dt>**WINBIO \_沒有 \_ 格式 \_ 類型 \_ 可用**</dt> <dt>0</dt> </dl>    | 沒有擁有者指派的格式類型可供使用。<br/>                                               |
