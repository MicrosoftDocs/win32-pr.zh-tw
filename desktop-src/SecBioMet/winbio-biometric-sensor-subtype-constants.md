# <a name="winbio_biometric_sensor_subtype-constants"></a>WINBIO \_ 生物特徵辨識 \_ 感應器 \_ 子類型常數

下列常數可用來當做 [**WINBIO \_ 單元 \_ 架構**](winbio-unit-schema.md)之 *功能* 參數的位元遮罩。 這些是指上架的感應器功能。



| 常數                                                                                                                                                                                                            | 描述                                        |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------|
| <span id="WINBIO_SENSOR_SUBTYPE_UNKNOWN"></span><span id="winbio_sensor_subtype_unknown"></span><dl> <dt>**WINBIO \_ 感應器 \_ 子類型 \_ 不明**</dt> </dl>     | 感應器子類型未知。<br/>      |
| <span id="WINBIO_FP_SENSOR_SUBTYPE_SWIPE"></span><span id="winbio_fp_sensor_subtype_swipe"></span><dl> <dt>**WINBIO \_ FP \_ 感應器 \_ 子類型 \_ 滑動**</dt> </dl> | 感應器支援指紋撥動。<br/> |
| <span id="WINBIO_FP_SENSOR_SUBTYPE_TOUCH"></span><span id="winbio_fp_sensor_subtype_touch"></span><dl> <dt>**WINBIO \_ FP \_ 感應器 \_ 子類型 \_ 觸控**</dt> </dl> | 感應器支援手指觸控。<br/>     |
