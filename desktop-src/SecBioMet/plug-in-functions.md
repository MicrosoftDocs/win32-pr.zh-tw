# <a name="plug-in-functions"></a>外掛程式函數

下列主題討論您可以用來建立組成生物識別單位之介面卡外掛程式的功能：

## <a name="in-this-section"></a>本節內容



| 主題                                                                 | 描述                                                                                                                                                                                                                                  |
|-----------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [引擎介面卡功能](engine-adapter-functions.md)<br/>   | 引擎介面卡會從已捕捉的範例產生生物特徵辨識範本、比對現有範本的範例和索引範本。<br/>                                                                                              |
| [感應器介面卡功能](sensor-adapter-functions.md)<br/>   | 感應器介面卡會包裝生物特徵辨識裝置，並提供標準介面，讓 Windows 生物特徵辨識服務設定感應器、捕獲範例，以及控制生物特徵辨識資料流程向處理引擎的流程。<br/> |
| [儲存體介面卡功能](storage-adapter-functions.md)<br/> | 儲存體介面卡會管理範本資料庫。<br/>                                                                                                                                                                                     |
