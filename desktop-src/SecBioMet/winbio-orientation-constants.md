下列常數指定感應器元件指定為強制的可能相機方向。



| 常數/值                                                                                                                                                                                                                                                           | Description                                                         |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------|
| <span id="WINBIO_ORIENTATION_UNSPECIFIED"></span><span id="winbio_orientation_unspecified"></span><dl> <dt>**WINBIO \_\_未指定方向**</dt> <dt>0</dt> </dl> | 未指定相機的強制方向。<br/> |
| <span id="WINBIO_ORIENTATION_LANDSCAPE"></span><span id="winbio_orientation_landscape"></span><dl> <dt>**WINBIO \_方向 \_ 橫向**</dt> <dt>1</dt> </dl>       | 相機需要橫向方向。<br/>    |
| <span id="WINBIO_ORIENTATION_PORTRAIT"></span><span id="winbio_orientation_portrait"></span><dl> <dt>**WINBIO \_方向 \_**</dt>直向 <dt>2</dt> </dl>          | 攝影機需要縱向方向。<br/>     |
| <span id="WINBIO_ORIENTATION_ANY"></span><span id="winbio_orientation_any"></span><dl> <dt>**WINBIO \_\_任意**</dt> <dt>3</dt>方向 </dl>                         | 相機允許任何方向。<br/>             |
