# <a name="framework-security"></a>架構安全性

Windows 生物特徵辨識服務可讓非特殊許可權的用戶端應用程式執行生物特徵辨識作業，而不需讓它們直接存取生物特徵辨識範例或硬體。 服務會在用戶端和生物特徵辨識元件之間建立安全性界限，如下圖所示。
