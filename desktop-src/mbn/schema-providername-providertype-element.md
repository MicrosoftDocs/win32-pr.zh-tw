# <a name="providername-providertype-element"></a>ProviderName (providerType) 元素

**ProviderName (providerType)** 元素包含行動電話通訊網路的名稱。

元素是最多20個字元的字串。

這是選擇性的項目。

``` syntax
<xs:element name="ProviderName"
    type="providerNameType"
 />
```

**ProviderName** 元素是由 [**providerType**](schema-providertype-complextype.md)複雜型別定義。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|-------------------------------------|---------------------------------------------------|
| 最低支援的用戶端<br/> | Windows 7 \[ 桌面應用程式 \| UWP 應用程式\]<br/> |
| 最低支援的伺服器<br/> | 都不支援<br/>                         |
