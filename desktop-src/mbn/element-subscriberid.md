# <a name="span-idwwan_profile_v4element_subscriberidspansubscriberid"></a><span id="WWAN_profile_v4.element_SubscriberID"></span>SubscriberID

識別設定檔的唯一識別碼。

若為 GSM 網路，這應該包含 SIM 的 IMSI (國際行動使用者身分識別) ，而針對 CDMA 裝置，則應該包含裝置的最小 (行動電話識別碼) 。

如需詳細資訊，請參閱 v1 [**SubscriberID**](./schema-subscriberid-mbnprofile-element.md) 元素的檔。

## <a name="element-hierarchy"></a>元素階層

[&lt;MBNProfileExt&gt;](element-mbnprofileext.md)  
**&lt;SubscriberID&gt;**

## <a name="syntax"></a>Syntax

``` syntax
<SubscriberID>

  subscriberIdType

</SubscriberID>
```
