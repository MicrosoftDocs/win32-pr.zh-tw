# <a name="mobile-broadband"></a>行動寬頻

## <a name="purpose"></a>目的

行動寬頻 API 是用來執行對行動電話通訊網路的連線能力。 應用程式不應該直接與這類行動寬頻裝置通訊。 相反地，他們必須使用 Mobile 寬頻 API。

## <a name="developer-audience"></a>開發人員對象

Mobile 寬頻 API 是針對 c + + 開發人員所設計。

此外，也有 .NET 開發人員使用 C 的介面 \# 。 如需詳細資訊，請閱讀這份 [白皮書](https://www.microsoft.com/whdc/connect/wireless/MB_ManagedCode.mspx)。

行動寬頻 API 可供需要控制和管理行動寬頻介面的協力廠商應用程式使用。

## <a name="run-time-requirements"></a>執行階段需求求

行動寬頻支援從 Windows 7 開始。
