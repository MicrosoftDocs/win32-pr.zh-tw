# <a name="name-mbnprofile-element"></a>MBNProfile) 專案的名稱 (

**名稱 (MBNProfile)** 元素會識別設定檔的名稱。

長度最多為255個字元的字串。

需要元素。

``` syntax
<xs:element name="Name"
    type="nameType"
 />
```

**Name** 元素是由 [**MBNProfile**](schema-mbnprofile-element.md)元素所定義。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|-------------------------------------|---------------------------------------------------|
| 最低支援的用戶端<br/> | Windows 7 \[ 桌面應用程式 \| UWP 應用程式\]<br/> |
| 最低支援的伺服器<br/> | 都不支援<br/>                         |
