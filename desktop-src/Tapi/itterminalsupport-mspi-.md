# <a name="itterminalsupport-mspi"></a>ITTerminalSupport (MSPI) 

如果有 MSP 存在，就會在 [位址物件](address-object.md)上公開 [**ITTerminalSupport**](/windows/win32/api/tapi3if/nn-tapi3if-itterminalsupport)介面。 此介面的方法可讓應用程式探索可用的終端機及/或建立一個，並取得所需 [終端機物件](terminal-object.md)的指標。

[**ITTerminalSupport**](/windows/win32/api/tapi3if/nn-tapi3if-itterminalsupport)介面是由 MSP 所執行，而且如果沒有與該位址相關聯的媒體服務提供者，就無法使用。 如需此介面的詳細資訊，請參閱 MSP 介面一節中的 **ITTerminalSupport** 。
