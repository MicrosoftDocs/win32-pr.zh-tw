# <a name="linecallcomplcond_-constants"></a>LINECALLCOMPLCOND \_ 常數

**LINECALLCOMPLCOND \_** 位旗標常數描述可以完成呼叫的條件。

<dl> <dt>

<span id="LINECALLCOMPLCOND_BUSY"></span><span id="linecallcomplcond_busy"></span>**LINECALLCOMPLCOND \_ 忙碌**
</dt> <dd> <dl> <dt>



在忙碌的情況下完成呼叫。


</dt> </dl> </dd> <dt>

<span id="LINECALLCOMPLCOND_NOANSWER"></span><span id="linecallcomplcond_noanswer"></span>**LINECALLCOMPLCOND \_ NOANSWER**
</dt> <dd> <dl> <dt>



在回電沒有任何答案條件下完成呼叫。


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>備註

無延伸。 所有32位都是保留的。
