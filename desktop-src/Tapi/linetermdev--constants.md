# <a name="linetermdev_-constants"></a>LINETERMDEV \_ 常數

**LINETERMDEV \_** 位旗標常數描述不同類型的終端機裝置。

<dl> <dt>

<span id="LINETERMDEV_HEADSET"></span><span id="linetermdev_headset"></span>**LINETERMDEV \_ 耳機**
</dt> <dd> <dl> <dt>



終端機是耳機。


</dt> </dl> </dd> <dt>

<span id="LINETERMDEV_PHONE"></span><span id="linetermdev_phone"></span>**LINETERMDEV \_ 電話**
</dt> <dd> <dl> <dt>



終端機是一組電話。


</dt> </dl> </dd> <dt>

<span id="LINETERMDEV_SPEAKER"></span><span id="linetermdev_speaker"></span>**LINETERMDEV \_ 喇叭**
</dt> <dd> <dl> <dt>



終端機是外部的喇叭和麥克風。


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>備註

無延伸。 所有32位都是保留的。

這些常數用來描述線條的終端機裝置，並協助應用程式判斷終端機裝置的本質。
