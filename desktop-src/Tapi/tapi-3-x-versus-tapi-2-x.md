# <a name="tapi-3x-vs-tapi-2x"></a>TAPI 3.x 與 TAPI 2。x

從 Windows XP 開始，可以使用以 COM 為基礎的電話語音 API，也就是 TAPI 3.x。 TAPI 3.x 為現代的通訊程式設計提供了大幅增強的開發工具，其中「呼叫」可能是以 IP 為基礎的網路上的影片串流，且會話中不會包含任何電話組。 一般來說，應用程式撰寫者會發現 TAPI 3.x 可提供更佳的支援來實行彈性的通訊控制。

不過，在某些情況下，TAPI 2.x 可能是正確的選擇。 如果繼承應用程式需要進行輕微修改，或有舊版平臺需要支援，則轉換為 COM API 可能不會符合成本效益。
