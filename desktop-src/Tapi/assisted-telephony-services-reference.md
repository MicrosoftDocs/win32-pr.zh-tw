# <a name="assisted-telephony-services-reference"></a>輔助電話語音服務參考

下表列出輔助電話語音功能。 輔助電話語音功能可讓任何應用程式建立語音通話和媒體通話。 如需背景資訊，請參閱「 [輔助電話語音](./assisted-telephony-overview.md) 」總覽。

## <a name="assisted-telephony-services-functions"></a>輔助電話語音服務功能



| 函式                                             | 描述                                                 |
|------------------------------------------------------|-------------------------------------------------------------|
| [**tapiRequestMakeCall**](/windows/desktop/api/Tapi/nf-tapi-tapirequestmakecall)   | 提交要求來撥打語音電話。                    |
| [**tapiRequestMediaCall**](tapirequestmediacall.md) | 已過時。 請勿使用。                                       |
| [**tapiRequestDrop**](/windows/desktop/api/Tapi/nf-tapi-tapirequestdrop)           | 已過時。 請勿使用。                                       |
| [**tapiGetLocationInfo**](/windows/desktop/api/Tapi/nf-tapi-tapigetlocationinfo)   | 傳回識別國家/地區和城市/區域的代碼。 |
