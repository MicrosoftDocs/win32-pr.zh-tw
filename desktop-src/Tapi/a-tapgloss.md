# <a name="a-telephony-api"></a> (的電話語音 API) 

[B](b-tapgloss.md) [C](c-tapgloss.md) [D](d-tapgloss.md) [E](e-tapgloss.md) F G [](u-tapgloss.md) [](v-tapgloss.md) [H](h-tapgloss.md) [I](i-tapgloss.md) J [K](k-tapgloss.md) [L](l-tapgloss.md) [M](m-tapgloss.md) N O [P](p-tapgloss.md) Q [R](r-tapgloss.md) [S](s-tapgloss.md) [T](t-tapgloss.md) [W](w-tapgloss.md) X Y Z

<dl> <dt>

<span id="tapi2.analog_display_services_interface_adsi__tapgloss"></span><span id="TAPI2.ANALOG_DISPLAY_SERVICES_INTERFACE_ADSI__TAPGLOSS"></span>**(ADSI) 的類比顯示服務介面**
</dt> <dd>

一種 Bellcore 標準，定義交換器、伺服器、語音信箱系統或服務局之間資訊流程的通訊協定，以及訂閱者的電話、電腦、資料終端機，或其他與螢幕通訊的裝置。 ADSI 可在電話連接期間，讓對話的文字版本顯示在畫面上。 例如，使用者撥打語音信箱的帳戶，並在螢幕以文字形式顯示功能表時，收到語音信箱功能的音訊功能表。

</dd> <dt>

<span id="tapi2.assisted_telephony_tapgloss"></span><span id="TAPI2.ASSISTED_TELEPHONY_TAPGLOSS"></span>**輔助電話語音**
</dt> <dd>

這組函式並不只是專門用來電話功能，而是設計用來建立可供任何應用程式使用的語音通話和媒體通話。 如需其他資訊，請參閱電話語音總覽中的 [輔助電話語音](./assisted-telephony-overview.md) 。 查看 [*基本電話語音*](b-tapgloss.md)、 [*擴充電話語音*](e-tapgloss.md)、 [*補充電話語音*](s-tapgloss.md)。

</dd> <dt>

<span id="tapi2.asynchronous_tapgloss"></span><span id="TAPI2.ASYNCHRONOUS_TAPGLOSS"></span>**非同步**
</dt> <dd>

資料傳輸方法，允許以不規則的間隔在每個字元前面加上一個開始位，然後以停止位來傳送字元。 查看 [*同步*](s-tapgloss.md)。

</dd> </dl>
