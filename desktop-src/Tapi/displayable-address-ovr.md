# <a name="displayable-address"></a>可顯示的位址

可顯示的位址是以表單形式呈現位址的字串，適合在使用者介面中顯示。

**TAPI 2.x：** 請參閱 [**lineGetCallInfo**](/windows/win32/api/tapi/nf-tapi-linegetcallinfo)、 **dwDisplayableAddressSize** 和 **dwDisplayableAddressOffset** 成員 [**LINECALLINFO**](/windows/win32/api/tapi/ns-tapi-linecallinfo)。

**TAPI 3.x：** 請 [**參閱 ITCallInfo：： get \_ CallInfoString**](/windows/desktop/api/tapi3if/nf-tapi3if-itcallinfo-get_callinfostring)或 [**ITCallInfo：:p 內容 \_ CallInfoString**](/windows/desktop/api/tapi3if/nf-tapi3if-itcallinfo-put_callinfostring)，並以 [**DISPLAYABLEADDRESS \_ 字串**](/windows/desktop/api/Tapi3if/ne-tapi3if-callinfo_string)的 **CIS \_ CALLINFO** 成員呼叫。
