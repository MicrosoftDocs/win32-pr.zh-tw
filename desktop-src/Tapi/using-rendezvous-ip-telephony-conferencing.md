# <a name="using-rendezvous-ip-telephony-conferencing"></a>使用會合 IP 電話語音會議

\[Windows Vista、Windows Server 2008 及後續版本的作業系統無法使用會合 IP 電話語音會議控制項和介面。 RTC 用戶端 API 提供類似的功能。\]

會合包含三組基本的介面：

-   [目錄控制項](directory-controls.md) 提供在 ILS 或 NTDS 伺服器上探索、建立或修改會議和使用者資訊的方法。
-   [會議 blob 控制項](conference-blob-controls.md) 允許提供者特定的會議修改。
-   [多播 COM 介面](multicast-com-interfaces.md) 提供了一種保留位址的方法，供多播會議使用。
