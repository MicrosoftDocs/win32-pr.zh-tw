# <a name="d-telephony-api"></a>D (電話語音 API) 

[](a-tapgloss.md) [B](b-tapgloss.md) [C](c-tapgloss.md) D [E](e-tapgloss.md) F [](u-tapgloss.md) [](v-tapgloss.md) G [H](h-tapgloss.md) [I](i-tapgloss.md) J [K](k-tapgloss.md) [L](l-tapgloss.md) [M](m-tapgloss.md) N O [P](p-tapgloss.md) Q [R](r-tapgloss.md) [S](s-tapgloss.md) [T](t-tapgloss.md) [W](w-tapgloss.md) X Y Z

<dl> <dt>

<span id="tapi2.d_channel_tapgloss"></span><span id="TAPI2.D_CHANNEL_TAPGLOSS"></span>**資料通道**
</dt> <dd>

ISDN 介面的資料通道。 這是基本費率 ISDN 線路上三個標準通道的其中一個。 在 BRI-ISDN 線路上，資料通道可以使用 16000 bps 的速率來傳送信號資訊和低速度封包資料。 在 PRI-ISDN 線路上，資料通道會以 64000 bps 執行。 請參閱 [*基本速率介面 (BRI-isdn)*](b-tapgloss.md)、 [*B 通道*](b-tapgloss.md)、 [*主要速率介面 (PRI-isdn)*](p-tapgloss.md)。

</dd> <dt>

<span id="tapi2.device_class_tapgloss"></span><span id="TAPI2.DEVICE_CLASS_TAPGLOSS"></span>**裝置類別**
</dt> <dd>

一組相關實體裝置或設備磁碟機，應用程式會透過這些裝置或設備磁碟機來傳送和接收進行呼叫的資訊或資料。 如需清單，請參閱 [TAPI 裝置類別](tapi-device-classes.md)。

</dd> <dt>

<span id="tapi2.did_tapgloss"></span><span id="TAPI2.DID_TAPGLOSS"></span>**我們**
</dt> <dd>

請參閱 *直接向內撥 ()*。

</dd> <dt>

<span id="tapi2.direct_inward_dialing_did__tapgloss"></span><span id="TAPI2.DIRECT_INWARD_DIALING_DID__TAPGLOSS"></span>**直接向撥 (的)**
</dt> <dd>

幾乎所有新式 Pbx 都提供的功能，可讓外部呼叫者藉由撥打標準的七位數電話號碼來達到內部擴充，而不需要通過操作員或來電。 請參閱 [*私用分支交換 (PBX)*](p-tapgloss.md)。

</dd> </dl>
