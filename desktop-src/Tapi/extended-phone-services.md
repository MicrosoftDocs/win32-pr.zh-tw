# <a name="extended-phone-services"></a>擴充的電話服務

擴充的電話服務 (或裝置特定的電話語音) 包含服務提供者所定義之 TSPI 的所有延伸模組。 TSPI 定義了一種機制，可讓服務提供者廠商使用裝置特定的擴充功能來擴充 TSPI。 TSPI 只會定義擴充機制，藉由這樣做可存取裝置專屬的延伸模組，其行為完全由服務提供者定義。

TSPI 包含純量和位旗標資料常數定義、資料結構、函數和回呼訊息。 定義的程式可讓廠商擴充大部分的，如本節中的主題所述。
