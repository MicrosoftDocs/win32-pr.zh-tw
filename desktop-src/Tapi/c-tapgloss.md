# <a name="c-telephony-api"></a>C (電話語音 API) 

[](a-tapgloss.md) [B](b-tapgloss.md) C [](u-tapgloss.md) [](v-tapgloss.md) [D](d-tapgloss.md) [E](e-tapgloss.md) F G [H](h-tapgloss.md) [I](i-tapgloss.md) J [K](k-tapgloss.md) [L](l-tapgloss.md) [M](m-tapgloss.md) N O [P](p-tapgloss.md) Q [R](r-tapgloss.md) [S](s-tapgloss.md) [T](t-tapgloss.md) [W](w-tapgloss.md) X Y Z

<dl> <dt>

<span id="tapi2.call_classification_tapgloss"></span><span id="TAPI2.CALL_CLASSIFICATION_TAPGLOSS"></span>**呼叫分類**
</dt> <dd>

此 TAPI 程式會報告媒體資料流程類型的變更， (語音、傳真、資料數據機等等) 參與的應用程式。

</dd> <dt>

<span id="tapi2.call_progress_monitoring_tapgloss"></span><span id="TAPI2.CALL_PROGRESS_MONITORING_TAPGLOSS"></span>**通話進度監視**
</dt> <dd>

接聽聲音音的程式，例如撥號音、特殊資訊語氣、忙碌信號，以及回電音，以判斷呼叫的狀態 (透過網路) 的進度。

</dd> <dt>

<span id="tapi2.call_redirection_tapgloss"></span><span id="TAPI2.CALL_REDIRECTION_TAPGLOSS"></span>**呼叫重新導向**
</dt> <dd>

允許應用程式在沒有先接聽來電的情況下，將供應專案來電轉接至另一個位址的函式。 呼叫重新導向與呼叫轉送不同，因為交換器不需要應用程式就能執行呼叫轉送;重新導向可依應用程式逐一呼叫來完成，例如，由呼叫者識別碼資訊所驅動。 這與傳送呼叫的來電轉接不同，需要先回答呼叫。 請參閱呼叫 *識別碼*、 *呼叫者識別碼*、重新導向 [*識別碼*](r-tapgloss.md)、重新導向 [*識別碼*](r-tapgloss.md)。

</dd> <dt>

<span id="tapi2.called_id_tapgloss"></span><span id="TAPI2.CALLED_ID_TAPGLOSS"></span>**呼叫識別碼**
</dt> <dd>

識別呼叫的原始目的地位址。 請參閱 *呼叫* 重新導向。

</dd> <dt>

<span id="tapi2.caller_id_tapgloss"></span><span id="TAPI2.CALLER_ID_TAPGLOSS"></span>**呼叫者識別碼**
</dt> <dd>

識別呼叫的原始位址。 請參閱 *呼叫* 重新導向。

</dd> <dt>

<span id="tapi2.central_office_tapgloss"></span><span id="TAPI2.CENTRAL_OFFICE_TAPGLOSS"></span>**中央辦公室**
</dt> <dd>

電話公司的當地設施，可在特定區域中提供電話語音。 通常，「訂閱者」的行會聯結至將其他「訂閱者」彼此連線、在本機和大量連接的設備。 也稱為「 *共同*」。

</dd> <dt>

<span id="tapi2.centrex_tapgloss"></span><span id="TAPI2.CENTREX_TAPGLOSS"></span>**Centrex**
</dt> <dd>

由當地電話公司從當地的總公司提供的公司電話語音。

</dd> <dt>

<span id="tapi2.computer_centric_connection_tapgloss"></span><span id="TAPI2.COMPUTER_CENTRIC_CONNECTION_TAPGLOSS"></span>**以電腦為中心的連接**
</dt> <dd>

使用連接至電話網絡與電話組之電腦增益集卡片或外部方塊的連接。 比較以 [*電話為中心的連接*](p-tapgloss.md)。

</dd> <dt>

<span id="tapi2.connected_id_tapgloss"></span><span id="TAPI2.CONNECTED_ID_TAPGLOSS"></span>**已連線-識別碼**
</dt> <dd>

呼叫實際連接之合作物件的識別碼。 如果呼叫轉向，這可能與被呼叫的合作物件不同。

</dd> </dl>
