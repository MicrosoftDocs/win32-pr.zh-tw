# <a name="track-controlled-playback"></a>Track-Controlled 播放

若要執行追蹤控制播放，請執行下列動作：

-   選取 [追蹤終端機到資料流程]。
-   建立檔案播放終端機。
-   設定屬性：儲存的檔案名和類型。
-   在檔案播放終端機上使用方法，以列舉檔案追蹤終端機。
-   列舉 TAPI 呼叫上的資料流程。
-   選取串流上的檔案追蹤終端機。
-   在檔案播放終端機上呼叫 [**start**](/windows/desktop/api/tapi3if/nf-tapi3if-itmediacontrol-start) 方法，開始將記錄的資料播放至 TAPI 串流。
