# <a name="extended-telephony-phone-functions"></a>擴充電話語音電話功能

 (或 Device-Specific 電話語音的外延電話語音) 包含服務提供者所定義之 TAPI 的所有延伸模組。 TAPI 定義了一種機制，可讓服務提供者廠商使用裝置專屬的延伸模組來擴充 TAPI。 TAPI 只定義擴充機制，藉由這樣做，可存取裝置專屬的擴充功能。 TAPI 不會定義其行為，而是由服務提供者完整定義。

TAPI 包含純量和位旗標資料常數定義、資料結構、函數和訊息。 定義的程式可讓廠商擴充大部分的，如下列主題所述。

-   [純量資料常數](scalar-data-constants.md)
-   [位旗標資料常數](bit-flag-data-constants.md)
-   [函數和訊息](functions-and-messages.md)
