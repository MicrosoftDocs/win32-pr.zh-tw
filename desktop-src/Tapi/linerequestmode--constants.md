# <a name="linerequestmode_-constants"></a>LINEREQUESTMODE \_ 常數

**LINEREQUESTMODE** 位旗標常數描述可從某個應用程式到另一個應用程式的不同類型電話語音要求。

<dl> <dt>

<span id="LINEREQUESTMODE_DROP"></span><span id="linerequestmode_drop"></span>**LINEREQUESTMODE \_ DROP**
</dt> <dd> <dl> <dt>



[TapiRequestDrop](/windows/desktop/api/Tapi/nf-tapi-tapirequestdrop)要求。


</dt> </dl> </dd> <dt>

<span id="LINEREQUESTMODE_MAKECALL"></span><span id="linerequestmode_makecall"></span>**LINEREQUESTMODE \_ MAKECALL**
</dt> <dd> <dl> <dt>



[TapiRequestMakeCall](/windows/desktop/api/Tapi/nf-tapi-tapirequestmakecall)要求。


</dt> </dl> </dd> <dt>

<span id="LINEREQUESTMODE_MEDIACALL"></span><span id="linerequestmode_mediacall"></span>**LINEREQUESTMODE \_ MEDIACALL**
</dt> <dd> <dl> <dt>



這個常數已過時。 請勿使用。


</dt> </dl> </dd> </dl>
