# <a name="ndis-proxy-tsp"></a>NDIS Proxy TSP

NDIS Proxy 是一種核心模式元件，它會將 NDIS miniport 公開為 TAPI 線路。 NDIS Proxy TSP (Ndptsp) 可讓 TAPI 與 NDIS Proxy 進行通訊。

NDIS Proxy TSP 安裝在 Windows Server 2003 作業系統、Windows XP、Windows 2000、Windows NT、Windows Millennium Edition、Windows 98 和 Windows 95。
