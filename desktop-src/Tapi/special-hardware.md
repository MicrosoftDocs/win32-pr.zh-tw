# <a name="special-hardware"></a>特殊硬體

TAPI 的一些較先進功能需要應用程式才能從電話取出資料。 大部分的電話無法直接連線到電腦來控制語音電話，因此目前無法支援在 POTS 中扮演的被動角色以外的電話語音功能。 未來，使用者將會安裝及設定電話組，就像其他週邊裝置一樣。 這些集合將會伴隨卡片，以控制電腦與電話之間的資訊流程。 用戶端/伺服器設定也可以讓使用者透過 LAN 連接到已安裝這類電路板和相關聯軟體的伺服器，以利用電話語音。
