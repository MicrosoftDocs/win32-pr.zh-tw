# <a name="lineproxystatus_-constants"></a>LINEPROXYSTATUS \_ 常數

**LINEPROXYSTATUS \_ 常數** 會在應用程式目前開啟的行上指出 proxy 的狀態。

如需所有可能的 proxy 要求值的清單和描述，請參閱 [**LINEPROXYREQUEST \_ 常數**](lineproxyrequest--constants.md) 。

<dl> <dt>

<span id="LINEPROXYSTATUS_ALLOPENFORACD"></span><span id="lineproxystatus_allopenforacd"></span>**LINEPROXYSTATUS \_ ALLOPENFORACD**
</dt> <dd> <dl> <dt>



這一行現在已針對 TAPI 3.0 版和更新版本 ACD 作業所需的所有 proxy 要求類型開啟 proxy。


</dt> </dl> </dd> <dt>

<span id="LINEPROXYSTATUS_CLOSE"></span><span id="lineproxystatus_close"></span>**LINEPROXYSTATUS \_ 關閉**
</dt> <dd> <dl> <dt>



Proxy 已關閉。


</dt> </dl> </dd> <dt>

<span id="LINEPROXYSTATUS_OPEN"></span><span id="lineproxystatus_open"></span>**LINEPROXYSTATUS \_ 開啟**
</dt> <dd> <dl> <dt>



已開啟新的 proxy。


</dt> </dl> </dd> </dl>
