# <a name="l-telephony-api"></a>L (電話語音 API) 

[](a-tapgloss.md) [B](b-tapgloss.md) [C](c-tapgloss.md) [D](d-tapgloss.md) [E](e-tapgloss.md) F [](u-tapgloss.md) [](v-tapgloss.md) G [H](h-tapgloss.md) [I](i-tapgloss.md) J [K](k-tapgloss.md) L [M](m-tapgloss.md) N O [P](p-tapgloss.md) Q [R](r-tapgloss.md) [S](s-tapgloss.md) [T](t-tapgloss.md) [W](w-tapgloss.md) X Y Z

<dl> <dt>

<span id="tapi2.line_device_tapgloss"></span><span id="TAPI2.LINE_DEVICE_TAPGLOSS"></span>**線路裝置**
</dt> <dd>

由電話線路和硬體所組成的實體通道，以控制它，例如傳真板、數據機或 ISDN 卡。 最簡單的線路裝置類型只對應到一條電話線。

</dd> <dt>

<span id="tapi2.line_device_class_tapgloss"></span><span id="TAPI2.LINE_DEVICE_CLASS_TAPGLOSS"></span>**線路裝置類別**
</dt> <dd>

與裝置無關的實體線路裝置（如數據機）標記法。

</dd> <dt>

<span id="tapi2.local_loop"></span><span id="TAPI2.LOCAL_LOOP"></span>**本機迴圈**
</dt> <dd>

電話與電話公司中央切換辦公室之間的電話網絡部分。 在此迴圈中，家庭電話的人類語音通常會以類比格式傳輸，且電腦的數位資料必須先由數據機轉換成類比。 數位網路會逐漸更換本機迴圈中的類比。

</dd> </dl>
