# <a name="tapi-object"></a>TAPI 物件

TAPI 物件代表本機電腦有權存取的所有電話語音資源，讓應用程式可以列舉所有的本機和遠端位址。 TAPI 3 應用程式必須建立 TAPI 物件的實例，然後將它初始化。

[初始化 tapi](initialize-tapi.md)程式碼範例顯示建立 TAPI 物件的範例。

請參閱 [Tapi 物件介面](tapi-object-interfaces.md) ，以取得與 TAPI 物件相關聯之所有介面的清單。
