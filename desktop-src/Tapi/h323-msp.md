# <a name="h323-msp"></a>H323 MSP

\[H323 MSP 無法在 Windows Vista、Windows Server 2008 和後續版本的作業系統中使用。 RTC 用戶端 API 提供類似的功能。\]

H323 MSP (H323msp.dll) 搭配其 TSP (H323. TSP) 時，可讓應用程式使用 IP 電話語音的 h.264 通訊協定來進行和接收影片和音訊通話。

Windows 2000 SP1 中的 H323 MSP 已增強，可啟用 H323 和 SDP 用戶端之間的通訊。 它現在會公開 [**IH323LineEx**](ih323lineex.md) 和 [**IKeyFrameControl**](ikeyframecontrol.md) 介面。

H323 MSP 與 Windows 2000 一起安裝。
