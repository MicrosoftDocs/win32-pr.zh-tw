# <a name="lineroammode_-constants"></a>LINEROAMMODE \_ 常數

**LINEROAMMODE \_** 位旗標常數描述線路裝置的漫遊狀態。

<dl> <dt>

<span id="LINEROAMMODE_HOME"></span><span id="lineroammode_home"></span>**LINEROAMMODE \_ 首頁**
</dt> <dd> <dl> <dt>



這一行連接到 [家用網路] 節點。


</dt> </dl> </dd> <dt>

<span id="LINEROAMMODE_ROAMA"></span><span id="lineroammode_roama"></span>**LINEROAMMODE \_ ROAMA**
</dt> <dd> <dl> <dt>



線路會連接到「漫遊」-「貨運公司」和通話會據以收費。


</dt> </dl> </dd> <dt>

<span id="LINEROAMMODE_ROAMB"></span><span id="lineroammode_roamb"></span>**LINEROAMMODE \_ ROAMB**
</dt> <dd> <dl> <dt>



線路會連接到「漫遊-B」貨運公司，並據此收費通話。


</dt> </dl> </dd> <dt>

<span id="LINEROAMMODE_UNAVAIL"></span><span id="lineroammode_unavail"></span>**LINEROAMMODE \_ UNAVAIL**
</dt> <dd> <dl> <dt>



漫遊模式無法使用，也不會知道。


</dt> </dl> </dd> <dt>

<span id="LINEROAMMODE_UNKNOWN"></span><span id="lineroammode_unknown"></span>**LINEROAMMODE \_ 不明**
</dt> <dd> <dl> <dt>



漫遊模式目前是未知的，但稍後可能會變成已知。


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>備註

無延伸。 所有32位都是保留的。
