# <a name="linecallinfostate_-constants"></a>LINECALLINFOSTATE \_ 常數

## <a name="remarks"></a>備註

無延伸。 所有32位都是保留的。

當此資料結構中發生變更時，會將 [**行 \_ CALLINFO**](line-callinfo.md) 訊息傳送至應用程式。 此訊息的參數是呼叫的控制碼，以及已變更之資訊專案的指示。 [**LINEADDRESSCAPS**](/windows/desktop/api/Tapi/ns-tapi-lineaddresscaps)資料結構也會指出這些呼叫資訊元素對位址上的每個呼叫都有效。
