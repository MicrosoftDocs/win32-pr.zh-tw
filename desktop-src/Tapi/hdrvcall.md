# <a name="hdrvcall"></a>HDRVCALL

[**HDRVCALL**](hdrvline.md)資料類型代表服務提供者的呼叫資料結構的不透明控制碼。 服務提供者必須將此類型的值解析為適當資料結構實例的參考。
