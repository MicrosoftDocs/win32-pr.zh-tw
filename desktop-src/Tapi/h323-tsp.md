# <a name="h323-tsp"></a>H323 TSP

\[H323 TSP 無法在 Windows Vista、Windows Server 2008 及後續版本的作業系統中使用。 RTC 用戶端 API 提供類似的功能。\]

H323 TSP () H323 與 [H323 MSP](./h323-msp.md) (H323msp.dll) 搭配使用時，可讓應用程式使用 IP 電話語音的 h. 通訊協定來進行和接收影片和音訊通話。

此 TSP 支援[](./lineaddresstype--constants.md) LINEADDRESSTYPE \_ PHONENUMBER、LINEADDRESSTYPE \_ IPADDRESS 和 LINEADDRESSTYPE DOMAINNAME 的網址類別型值 \_ 。 LINEADDRESSTYPE \_ DOMAINNAME 指的是電腦的功能變數名稱。

H323 MSP 已在 Windows 2000 SP1 中增強，以啟用 H323 和 SDP 用戶端之間的通訊。

H323 TSP 隨 Windows 2000 和更新版本一起安裝。
