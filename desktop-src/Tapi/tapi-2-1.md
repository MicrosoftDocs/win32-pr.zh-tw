# <a name="tapi-21"></a>TAPI 2。1

TAPI 2.1 提供上述 TAPI 2.0 規格的另一項：用戶端/伺服器功能。 使用 TAPI 2.1 和更新版本時，可以在可從網路上的任何電腦存取的電話語音伺服器上安裝電話語音硬體。 一組新的函式可讓您撰寫管理 Dll，以控制誰可以存取遠端資源。

**Windows Me/98/95：** TAPI 2.1 提供32位 Tsp 的支援。 當安裝支援時，TAPI 2.1 系統二進位檔：

-   32位 TAPI 1.3、TAPI 1.4 和 TAPI 2.x 應用程式
-   32位 TAPI 2.x Tsp
-   16位 TAPI 1.3 和 TAPI 1.4 應用程式
-   16位 TAPI 1.3 和 TAPI 1.4 Tsp
