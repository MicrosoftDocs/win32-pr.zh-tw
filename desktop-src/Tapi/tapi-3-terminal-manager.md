# <a name="tapi-3-terminal-manager"></a>TAPI 3 終端機管理員

TAPI 3 終端機管理員是由 Tapi3.dll 在 MSP 要求所建立，並由 MSP 用來進行動態終端機的建立和刪除。

MSP 動態終端機的精確執行詳細資料已超出本檔的範圍。 不過，許多作者可能會選擇使用媒體串流終端機 (MST) 的部分功能。 此終端機以 DirectShow 為基礎，而指定的通訊協定特定的篩選器可能會與 MST 一起撰寫和使用。
