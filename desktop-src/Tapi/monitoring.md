# <a name="monitoring"></a>監視

監視是服務提供者定義的功能，可偵測表示其他媒體的信號。 例如，一個應用程式可以在來電開始傳送傳真「來電」時，播放傳出的「離開訊息」語音訊息，並等候交握。 為了避免遺失傳真呼叫，本機應用程式會在播放語音訊息時監視此聲音。

監視包含 [媒體監視](/previous-versions/windows/desktop/legacy/ms725242(v=vs.85))、 [數位監視](/previous-versions/windows/desktop/legacy/ms725185(v=vs.85))和 [音訊監視](/previous-versions/windows/desktop/legacy/ms725520(v=vs.85))。
