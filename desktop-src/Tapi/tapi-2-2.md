# <a name="tapi-22"></a>TAPI 2。2

TAPI 2.2 提供特製化的電話中心管理功能。 在 TAPI 2.1 及更早的版本中，話務中心軟體需要謹慎地將線路和位址對應至撥置的函式。 TAPI 2.1 函式與 TAPI 3.x 的功能協調，可讓您更輕鬆地建立 ACD 系統。

**TAPI 2.x：** 如需詳細資訊，請參閱 [撥置控制](call-center-control.md)。

**TAPI 3.x：** 如需詳細資訊，請參閱 [撥置控制項](./about-call-center-controls.md)。
