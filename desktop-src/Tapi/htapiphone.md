# <a name="htapiphone"></a>HTAPIPHONE

**HTAPIPHONE** 資料類型代表 TAPI 的不透明控制碼，適用于手機資料結構。 TAPI 必須將此類型的值解析為適當資料結構實例的參考。 服務提供者不得嘗試透過這個方式來參考它，就像它是指標、對其值進行假設，或是以任何方式解讀其標記法，而不是在適當的時間將其值傳遞給 TAPI。
