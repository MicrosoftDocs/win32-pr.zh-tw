# <a name="b-telephony-api"></a>B (電話語音 API) 

[B](a-tapgloss.md) [](u-tapgloss.md) [](v-tapgloss.md) [C](c-tapgloss.md) [D](d-tapgloss.md) [E](e-tapgloss.md) F G [H](h-tapgloss.md) [I](i-tapgloss.md) J [K](k-tapgloss.md) [L](l-tapgloss.md) [M](m-tapgloss.md) N O [P](p-tapgloss.md) Q [R](r-tapgloss.md) [S](s-tapgloss.md) [T](t-tapgloss.md) [W](w-tapgloss.md) X Y Z

<dl> <dt>

<span id="tapi2.b_channel_tapgloss"></span><span id="TAPI2.B_CHANNEL_TAPGLOSS"></span>**B 頻道**
</dt> <dd>

作為 ISDN 介面基礎元件的 *持有* 人通道。 它每秒會有64000位 (bps) 雙向，是電路切換，而且能夠攜帶聲音或資料。 如需詳細資訊，請參閱 [持有人模式](./bearer-mode-ovr.md)。 請參閱 [*D channel*](d-tapgloss.md)、 [*整合式服務數位網路 (ISDN)*](i-tapgloss.md)。

</dd> <dt>

<span id="tapi2.band_tapgloss"></span><span id="TAPI2.BAND_TAPGLOSS"></span>**樂隊**
</dt> <dd>

兩個定義點之間的頻率範圍。

</dd> <dt>

<span id="tapi2.bandwidth_tapgloss"></span><span id="TAPI2.BANDWIDTH_TAPGLOSS"></span>**頻寬**
</dt> <dd>

在通訊中，指定範圍內的最高和最低頻率之間的差異。 例如，電話可以容納 3000 Hz 的頻寬、最低 (300 Hz) 和最高 (3300 Hz 之間的差異) 頻率。

</dd> <dt>

<span id="tapi2.bandwidth_allocation_control_protocol_bacp__tapgloss"></span><span id="TAPI2.BANDWIDTH_ALLOCATION_CONTROL_PROTOCOL_BACP__TAPGLOSS"></span>**頻寬配置控制通訊協定 (BACP)**
</dt> <dd>

一組規則，可透過動態多重連結的 PPP 連線來管理頻寬。 請參閱 [*多連結 ppp*](m-tapgloss.md)、 [*點對點通訊協定 (PPP)*](p-tapgloss.md)。

</dd> <dt>

<span id="tapi2.basic_rate_interface_bri_isdn__tapgloss"></span><span id="TAPI2.BASIC_RATE_INTERFACE_BRI_ISDN__TAPGLOSS"></span>**基本速率介面 (BRI-ISDN)**
</dt> <dd>

標準 ISDN 線路，由兩個 B 通道和一個 資料通道所組成。 請參閱 [*整合式服務數位網路 (ISDN)*](i-tapgloss.md)， [*主要速率介面 (PRI-ISDN)*](p-tapgloss.md)。

</dd> <dt>

<span id="tapi2.basic_telephony_tapgloss"></span><span id="TAPI2.BASIC_TELEPHONY_TAPGLOSS"></span>**基本電話語音**
</dt> <dd>

一種基本的電話語音服務，其中包含最少的功能子集，可提供一組最小的函式，以對應至單純舊的電話語音 (POTS) 。 需要 TAPI 服務提供者，才能支援所有基本的電話語音功能。 請參閱 [輔助電話語音](./assisted-telephony-overview.md)、延伸的 [*電話*](e-tapgloss.md)語音、 [*補充電話語音*](s-tapgloss.md)。

</dd> <dt>

<span id="tapi2.bri_isdn_tapgloss"></span><span id="TAPI2.BRI_ISDN_TAPGLOSS"></span>**BRI-ISDN**
</dt> <dd>

請參閱 *基本速率介面 (BRI-ISDN)*。

</dd> <dt>

<span id="tapi2.bps_tapgloss"></span><span id="TAPI2.BPS_TAPGLOSS"></span>**Bps**
</dt> <dd>

請參閱 *每秒位數 (bps)*。

</dd> <dt>

<span id="tapi2.bits_per_second_bps__tapgloss"></span><span id="TAPI2.BITS_PER_SECOND_BPS__TAPGLOSS"></span>**每秒位數 (bps)**
</dt> <dd>

傳輸速率測量，指出在一秒內透過電話線路透過數據機傳送多少位數的電腦資料。

</dd> </dl>
