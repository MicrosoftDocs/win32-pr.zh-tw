# <a name="w-telephony-api"></a>W (電話語音 API) 

[](a-tapgloss.md) [B](b-tapgloss.md) [C](c-tapgloss.md) [D](d-tapgloss.md) [E](e-tapgloss.md) F G [H](h-tapgloss.md) [I](i-tapgloss.md) J [K](k-tapgloss.md) [L](l-tapgloss.md) [M](m-tapgloss.md) N O [P](p-tapgloss.md) Q [R](r-tapgloss.md) [S](s-tapgloss.md) [T](t-tapgloss.md) [](u-tapgloss.md) [W X](v-tapgloss.md) Y Z

<dl> <dt>

<span id="tapi2.web_telephony_tapgloss"></span><span id="TAPI2.WEB_TELEPHONY_TAPGLOSS"></span>**網路電話語音**
</dt> <dd>

支援雙重存取網站 (可透過電話或網際網路瀏覽器存取的應用程式平臺) 以及 HTML 架構的 IVR 應用程式。 此平臺包含整合到伺服器的執行時間引擎、以 COM 為基礎的系統管理介面，用於安裝和設定 Web 電話語音應用程式，以及存取 TAPI 3.x 呼叫物件和腳本 API，以執行電話語音作業。

</dd> <dt>

<span id="tapi2.web_telephony_engine_webte__tapgloss"></span><span id="TAPI2.WEB_TELEPHONY_ENGINE_WEBTE__TAPGLOSS"></span>**(WTE) 的網路電話語音引擎**
</dt> <dd>

支援以 HTML 為基礎的 IVR 應用程式，以及雙重存取網站 (可透過電話存取的網站，以及透過圖形化瀏覽器) 。 請參閱 *網路電話語音*。

</dd> <dt>

<span id="tapi2.windows_open_services_architecture_wosa__tapgloss"></span><span id="TAPI2.WINDOWS_OPEN_SERVICES_ARCHITECTURE_WOSA__TAPGLOSS"></span>**Windows Open Services 架構 (WOSA)**
</dt> <dd>

提供單一系統層級介面以連接前端應用程式與後端服務的軟體架構。 Windows 電話語音是 WOSA 的一部分。

</dd> </dl>
