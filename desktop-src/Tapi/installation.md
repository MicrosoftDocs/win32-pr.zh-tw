# <a name="installation-telephony-api"></a>安裝 (電話語音 API) 

安裝程式的主要來源是目標作業系統、應用程式檔及服務提供者指示的資源套件。 針對 Microsoft 服務提供者，適當的資源套件包含相關指示。

安裝程式必須將「電話語音服務」列示為相依性。

如果應用程式或服務提供者使用登錄來儲存持續性私用資料，則此資訊不得放置在登錄的 TAPI 區段中。 此區段的格式不保證會在未來的版本中進行維護。
