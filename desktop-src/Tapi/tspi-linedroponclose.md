# <a name="tspi_linedroponclose"></a>TSPI \_ lineDropOnClose

**TSPI \_ lineDropOnClose** 函數已淘汰。 當唯一的擁有者關閉線路裝置的控制碼，以便在用戶端/伺服器環境中的多部用戶端電腦上容納呼叫的分散式控制時，TAPI 就不會再自動卸載通話。
