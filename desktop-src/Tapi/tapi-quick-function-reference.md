# <a name="tapi-quick-function-reference"></a>TAPI 快速函數參考

下列 TAPI 函式的超連結摘要會依作業類別來組織。 如需其他資訊，請參閱 [輔助電話語音](./assisted-telephony-overview.md) 和 [服務等級](./tapi-levels-of-service.md)的概述。



| 參考區段                                                                            | 描述                                                                                         |
|----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [基本電話語音服務參考](basic-telephony-services-reference.md)                 | 與 POTS 層級電話語音工作（例如 [**lineMakeCall**](/windows/desktop/api/Tapi/nf-tapi-linemakecall)）相關聯的函式。 |
| [補充電話語音服務參考](supplementary-telephony-services-reference.md) | 具有更先進功能的函式，例如呼叫重新導向。                                       |
| [擴充電話語音服務參考](extended-telephony-services-reference.md)           | 可以存取服務提供者特定功能的函式，而不是由 TAPI 直接定義。   |
| [輔助電話語音服務參考](assisted-telephony-services-reference.md)           | 用來建立語音電話和媒體通話的函式，例如使用電話撥號程式。              |
