# <a name="ipconf-msp"></a>IPConf MSP

\[IPConf MSP 無法在 Windows Vista、Windows Server 2008 和後續版本的作業系統中使用。 RTC 用戶端 API 提供類似的功能。\]

IPConf MSP (Confmsp.dll) 與其 TSP (IPconf. TSP) 配對，可提供會議存取網際網路通訊協定 (IP) 網路，例如網際網路和許多公司內部網路。 除了標準的 MSPI 方法之外，此 MSP 也會公開介面，以允許對會議中的個別參與者進行控制。

請參閱 [IPCONF Msp 介面](ipconf-msp-interfaces.md) ，以取得由此 MSP 公開之提供者特定介面的摘要。

Windows 2000 SP1 中的 IPConf MSP 已增強，可啟用 H323 和 SDP 用戶端之間的通訊。 它現在會另外公開 [**IMulticastControl**](imulticastcontrol.md) 介面。

IPConf MSP 與 Windows 2000 一起安裝。
