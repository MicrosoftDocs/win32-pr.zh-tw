# <a name="cmspstream-mspi-methods-implemented"></a>CMSPStream MSPI 方法實作為



| CMSPStream MSPI 方法                                   | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
|-----------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**取得 \_ 媒體媒體**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-get_mediatype)          | 取得資料流程的 [**媒體類型**](tapimediatype--constants.md)。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| [**取得 \_ 方向**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-get_direction)          | 取得資料流程的 [**終端方向**](/windows/desktop/api/Tapi3if/ne-tapi3if-terminal_direction)。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [**SelectTerminal**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-selectterminal)         | 選取 [**ITTerminal**](/windows/win32/api/tapi3if/nn-tapi3if-itterminal) 物件至資料流程。 取得函數中的鎖定。 這個基類方法只知道如何維護終端機清單;衍生的類別必須覆寫這個方法，以提供額外的功能 (例如，如果在資料流程執行時選取了終端機，就會停止並重新啟動資料流程) 。 基類的執行會檢查終端指標是否指向有效的記憶體、從終端機取得 [**ITTerminalControl**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalcontrol) 介面、從終端機取得位址控制碼，並檢查它是否符合 MSP 的位址控制碼、檢查終端機是否已存在於資料流程的清單中、將終端機新增至串流的終端機清單，以及 AddRefs 它。 如果有任何測試失敗，則會傳回 TAPI \_ E \_ INVALIDTERMINAL。 |
| [**UnselectTerminal**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-unselectterminal)     | 從資料流程取消將終端機取消。 取得函數中的鎖定。 在串流的終端機清單中尋找終端機，並將它從清單中移除，並加以釋放。 如果找不到終端機，它會傳回 TAPI \_ E \_ INVALIDTERMINAL。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**EnumerateTerminals**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-enumerateterminals) | 列舉在資料流程上選取的終端機。 它是終端機的快照集。 取得函數中的鎖定。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**取得 \_ 終端機**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-get_terminals)          | 建立與目前資料流程相關聯的終端機集合。 提供給 Automation 用戶端應用程式，例如以 Microsoft Visual Basic 撰寫的應用程式。 它是終端機的快照集。 取得函數中的鎖定。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**StartStream**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-startstream)               | 啟動資料流程。 呼叫篩選圖形上的 **IMediaControl：： Run** 。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| [**PauseStream**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-pausestream)               | 暫停資料流程。 在篩選圖形上呼叫 **IMediaControl：:P ause** 。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| [**StopStream**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-stopstream)                 | 停止資料流程。 呼叫篩選圖形上的 **IMediaControl：： Stop** 。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
