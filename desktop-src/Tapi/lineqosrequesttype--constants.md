# <a name="lineqosrequesttype_-constants"></a>LINEQOSREQUESTTYPE \_ 常數

TSP 會使用這些常數來識別 QoS (服務品質) 要求的類型。 目前只支援服務層級要求。 請參閱 [LINEQOSSERVICELEVEL \_ 常數](lineqosservicelevel--constants.md)。

<dl> <dt>

<span id="LINEQOSREQUESTTYPE_SERVICELEVEL"></span><span id="lineqosrequesttype_servicelevel"></span>**LINEQOSREQUESTTYPE \_ SERVICELEVEL**
</dt> <dd> <dl> <dt>

 0x00000001
</dt> <dt>



所提出的 QoS 要求是針對特定的服務層級。


</dt> </dl> </dd> </dl>
