# <a name="raw-channel-access-msp"></a>原始通道存取 MSP

原始通道存取 MSP (Rcsmsp.dll) 會透過 ksproxy 串流至 RCA，以允許透過以連接導向的網路驅動程式介面規格 (NDIS) 裝置（例如 ATM 和 ISDN 卡）進行串流處理。 RCA MSP 是由 ndptsp 所建立，也就是為 NDIS WAN 介面卡提供的 TSP。 在 Windows 2000 版本中，只會執行音訊串流處理。

原始通道存取 MSP 會隨 Windows 2000 安裝。
