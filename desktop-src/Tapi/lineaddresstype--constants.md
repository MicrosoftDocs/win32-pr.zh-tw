# <a name="lineaddresstype_-constants"></a>LINEADDRESSTYPE \_ 常數

網址類別型會識別位址格式，例如標準電話號碼或電子郵件地址。 只有協商 TAPI 3.0 版或更新版本的應用程式可以使用網址類別型。

</dt> <dd> <dl> <dt>

0x00000001
</dt> <dt>



網址類別型是標準的電話號碼。


</dt> </dl> </dd> <dt>

<span id="LINEADDRESSTYPE_SDP"></span><span id="lineaddresstype_sdp"></span>**LINEADDRESSTYPE \_ SDP**
</dt> <dd> <dl> <dt>

0x00000002
</dt> <dt>



網址類別型為 Session Description Protocol (SDP) 會議。


</dt> </dl> </dd> <dt>

</dt> </dl> </dd> <dt>

</dt> </dl> </dd> </dl>
