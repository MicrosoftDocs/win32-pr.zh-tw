# <a name="call-and-media-controls"></a>通話和媒體控制項

下列材質提供 TAPI 3 通話的一般評論，以及用來建立使用者和伺服器通訊應用程式的媒體控制物件、介面和方法。 [關於呼叫和媒體控制項](about-call-and-media-controls.md) 會描述主要物件，並且包含顯示物件和介面的超連結圖。 [使用 Call 和 Media 控制項](using-call-and-media-controls.md) 可在撰寫應用程式時提供更多有用的資訊，例如編譯所需的 .lib 檔案，以及一些程式碼範例。

如需詳細資訊，請參閱 [呼叫和媒體控制項參考](call-and-media-controls-reference.md)。
