# <a name="itstreamcontrol-mspi"></a>ITStreamControl (MSPI) 

如果有相關聯的 MSP，則會在呼叫物件上公開 [**ITStreamControl**](/windows/win32/api/tapi3if/nn-tapi3if-itstreamcontrol) 介面。 這個介面的方法可讓應用程式建立和移除資料流程。

[**ITStreamControl**](/windows/win32/api/tapi3if/nn-tapi3if-itstreamcontrol)介面是由 MSP 所執行。 如需此介面的詳細資訊，請參閱 MSP 介面一節中的 **ITStreamControl** 。
