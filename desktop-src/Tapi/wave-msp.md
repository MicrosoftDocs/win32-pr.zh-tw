# <a name="wave-msp"></a>Wave MSP

Wave MSP (Wavemsp.dll) 適用于將 Msp 引入 TAPI 3 的 MSP 串流架構之前所撰寫的舊版 Tsp。 某些舊版 Tsp 會公開 wave 裝置識別碼，這是應用程式用來控制 TAPI 2.1 下串流的方式。 Tapi3.dll 針對沒有自己的 MSP 但有 Wave 裝置的任何 TSP 使用 Wave MSP。

Wave MSP 與 Windows 2000 一起安裝。
