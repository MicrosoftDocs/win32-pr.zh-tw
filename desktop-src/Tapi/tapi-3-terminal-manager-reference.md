# <a name="tapi-3-terminal-manager-reference"></a>TAPI 3 終端機管理員參考



| 終端機管理員介面                      | Description                                                                                                                                                                            |
|--------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ITTerminalControl**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalcontrol)   | 提供方法來從篩選圖形連接和中斷終端機，並執行或停止轉譯器。                                                                                 |
| [**ITTerminalManager**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalmanager)   | 提供建立動態終端機的方法。                                                                                                                                          |
| [**ITTerminalManager2**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalmanager2) | 提供方法，以取得在目前系統中註冊之可插入終端機類別的相關資訊;衍生自 [**ITTerminalManager**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalmanager) 介面。 |
