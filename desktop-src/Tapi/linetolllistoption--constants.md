# <a name="linetolllistoption_-constants"></a>LINETOLLLISTOPTION \_ 常數

**LINETOLLLISTOPTION \_** 位旗標常數描述用於操作收費清單的選項。

<dl> <dt>

<span id="LINETOLLLISTOPTION_ADD"></span><span id="linetolllistoption_add"></span>**LINETOLLLISTOPTION \_ 新增**
</dt> <dd> <dl> <dt>



前置詞會新增至收費清單。


</dt> </dl> </dd> <dt>

<span id="LINETOLLLISTOPTION_REMOVE"></span><span id="linetolllistoption_remove"></span>**LINETOLLLISTOPTION \_ 移除**
</dt> <dd> <dl> <dt>



從收費清單中移除前置詞。


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>備註

無延伸。 所有32位都是保留的。
