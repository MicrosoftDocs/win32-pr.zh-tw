# <a name="tapi-levels-of-service"></a>服務的 TAPI 等級

TAPI 電話語音服務分為輔助電話語音服務以及完整電話語音 API 所提供的服務。 一般而言，完整的電話語音 API 是用來執行功能強大的電話應用程式，而輔助電話語音則是用來將最基本但有用的電話功能新增至非電話語音應用程式。 電話語音的服務分成下圖所示的群組：
