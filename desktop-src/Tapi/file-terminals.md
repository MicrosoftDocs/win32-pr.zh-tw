# <a name="file-terminals"></a>檔案終端機

在 TAPI 3.1 中，終端物件會展開以包含檔案終端機。 檔案終端機提供簡單的方法來執行一般作業，例如，將音訊串流或串流記錄到檔案，或後續播放錄製的音訊至資料流程。

使用五個基本物件來操作檔案終端機：

-   檔案播放終端機
-   檔案記錄終端機
-   檔案播放播放軌
-   檔案錄製播放軌
-   檔案事件

檔案終端機物件可視為儲存的抽象概念，通常是檔案。 追蹤會對應至該儲存體中的個別音訊資料流程，例如記錄語音的時間。

如需有關檔案終端機的詳細資訊，請參閱：

-   [檔案播放終端機和檔案記錄終端機](file-playback-terminal-and-file-recording-terminal.md)
-   [追蹤終端機](track-terminals.md)

如需與檔案終端機相關的詳細資訊和程式碼範例和工作清單，請參閱 [使用檔案終端](using-file-terminals.md)機。
