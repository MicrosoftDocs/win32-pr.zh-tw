# <a name="stringformat_-constants"></a>STRINGFORMAT \_ 常數

**STRINGFORMAT \_ 常數** 描述不同的字串格式。

<dl> <dt>

<span id="STRINGFORMAT_ASCII"></span><span id="stringformat_ascii"></span>**STRINGFORMAT \_ ASCII**
</dt> <dd> <dl> <dt>



指定每個字元使用一個位元組的標準 ASCII 字元格式。


</dt> </dl> </dd> <dt>

<span id="STRINGFORMAT_BINARY"></span><span id="stringformat_binary"></span>**STRINGFORMAT \_ 二進位檔**
</dt> <dd> <dl> <dt>



這是不帶正負號字元的陣列;可用於數值。


</dt> </dl> </dd> <dt>

<span id="STRINGFORMAT_DBCS"></span><span id="stringformat_dbcs"></span>**STRINGFORMAT \_ DBCS**
</dt> <dd> <dl> <dt>



指定使用每個字元一或兩個位元組的標準 DBCS 字元格式。


</dt> </dl> </dd> <dt>

<span id="STRINGFORMAT_UNICODE"></span><span id="stringformat_unicode"></span>**STRINGFORMAT \_ UNICODE**
</dt> <dd> <dl> <dt>



指定使用每個字元兩個位元組的標準 Unicode 字元格式。


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>備註

無延伸。 所有32位都是保留的。
