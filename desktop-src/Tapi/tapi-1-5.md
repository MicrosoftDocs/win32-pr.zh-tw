# <a name="tapi-15"></a>TAPI 1。5

TAPI 1.5 專為 Windows CE 版本1.0 所設計。 它是 TAPI 1.4 的子集，只能在 Windows CE 上提供，並記載于 Windows CE 開發人員檔中。
