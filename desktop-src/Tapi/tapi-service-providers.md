# <a name="tapi-service-providers"></a>TAPI 服務提供者

TAPI 服務提供者是 TAPI 應用程式與基礎硬體和傳輸通訊協定之間的抽象層。 [電話語音服務提供者](telephony-service-providers-start-page.md) (TSP) 提供呼叫控制和[媒體服務提供者](./media-service-providers-start-page.md) (MSP) 提供對通話媒體的詳細控制。 TSP/MSP 組可以執行應用程式所需的功能，範圍從基本的電話語音到 IP 多播多媒體會議。
