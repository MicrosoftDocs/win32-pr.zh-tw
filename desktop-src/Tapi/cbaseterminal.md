# <a name="cbaseterminal"></a>CBaseTerminal

**CBaseTerminal** 是適用于靜態和動態終端的終端基類。 它完全是泛型，因此任何的終端機都可以直接或間接衍生自這個類別。

定義于 Mspterm 中。
