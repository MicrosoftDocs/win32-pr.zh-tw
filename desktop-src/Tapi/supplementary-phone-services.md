# <a name="supplementary-phone-services"></a>補充電話語音

補充電話語音是一組除了基本電話語音以外的 advanced services （電話裝置支援）。 一般而言，此集合包含下列裝置功能： [hookswitch](/previous-versions/windows/desktop/legacy/ms725202(v=vs.85))、 [顯示](/previous-versions/windows/desktop/legacy/ms725186(v=vs.85))、獨特的 [環形](/previous-versions/windows/desktop/legacy/ms725264(v=vs.85))、 [按鈕](buttons.md)、獨特的 [燈](/previous-versions/windows/desktop/legacy/ms725210(v=vs.85))、 [資料區域](/previous-versions/windows/desktop/legacy/ms725182(v=vs.85))，以及檢查通話 [狀態](/previous-versions/windows/desktop/legacy/ms725508(v=vs.85))的功能。
