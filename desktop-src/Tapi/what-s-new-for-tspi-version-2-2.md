# <a name="whats-new-for-tspi-version-22"></a>TSPI 2.2 版的新功能

TSPI 版本2.2 變更主要包含一些具有其他元素的資料結構。 這些結構大多都是在 tapi 中定義，而 TAPI 應用程式和服務提供者會使用這些結構作為傳遞資訊的方式。

針對 TSPI 2.2 版新增或變更的函式和訊息如下所示：

-   [**LINE_AGENTSESSIONSTATUS**](./line-agentsessionstatus.md)
-   [**LINE_QUEUESTATUS**](./line-queuestatus.md)
-   [**LINE_AGENTSTATUSEX**](./line-agentstatusex.md)
-   [**LINE_GROUPSTATUS**](./line-groupstatus.md)
-   [**LINE_PROXYSTATUS**](./line-proxystatus.md)

[LINEADDRCAPFLAGS_ 常數](./lineaddrcapflags--constants.md)的 **LINEADDRCAPFLAGS_ACDGROUP** 成員

**LINEERR_INVALAGENTSESSIONSTATE** 和 **LINEERR_DISCONNECTED** [LINEERR_ 常數](./lineerr--constants.md)的成員。

-   [LINEAGENTSTATEEX_ 常數](./lineagentstateex--constants.md)
-   [LINEAGENTSTATUSEX_ 常數](./lineagentstatusex--constants.md)
-   [LINEAGENTSESSIONSTATE_ 常數](./lineagentsessionstate--constants.md)
-   [LINEAGENTSESSIONSTATUS_ 常數](./lineagentsessionstatus--constants.md)
-   [LINEQUEUESTATUS_ 常數](./linequeuestatus--constants.md)
-   [LINEGROUPSTATUS_ 常數](./linegroupstatus--constants.md)
-   [LINEPROXYSTATUS_ 常數](./lineproxystatus--constants.md)
-   [LINEPROXYREQUEST_ 常數](./lineproxyrequest--constants.md)
