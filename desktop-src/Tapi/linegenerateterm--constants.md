# <a name="linegenerateterm_-constants"></a>LINEGENERATETERM \_ 常數

**LINEGENERATETERM \_** 位旗標常數描述結束數位或音訊產生的條件。

<dl> <dt>

<span id="LINEGENERATETERM_CANCEL"></span><span id="linegenerateterm_cancel"></span>**LINEGENERATETERM \_ 取消**
</dt> <dd> <dl> <dt>



此應用程式、另一個應用程式已取消數位或語氣產生要求，或因為呼叫終止。 當數位或語氣產生因為服務提供者的內部失敗而無法完成時，也會傳回此值。


</dt> </dl> </dd> <dt>

<span id="LINEGENERATETERM_DONE"></span><span id="linegenerateterm_done"></span>**LINEGENERATETERM \_ 完成**
</dt> <dd> <dl> <dt>



要求的持續時間已產生要求的位數或要求的色調。


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>備註

無延伸。 所有32位都是保留的。
