# <a name="simple-record"></a>簡單記錄

若要執行資料流程的簡單記錄，請執行下列動作：

-   在呼叫層級選取檔案記錄終端機
-   建立檔案記錄終端機。
-   設定屬性：儲存的檔案名和類型。
-   選取呼叫上的檔案記錄終端機。
-   在檔案記錄終端機上呼叫 [**start**](/windows/desktop/api/tapi3if/nf-tapi3if-itmediacontrol-start) 方法，以開始記錄資料流程。
