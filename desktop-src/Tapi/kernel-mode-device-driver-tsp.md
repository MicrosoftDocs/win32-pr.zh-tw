# <a name="kernel-mode-device-driver-tsp"></a>Kernel-Mode 設備磁碟機 TSP

核心模式 TSP (Kmddsp，) 可讓應用程式處理舊版 WAN miniport (例如，將 NDIS 4.0 驅動程式) 為 TAPI 裝置。

核心模式 TSP 隨 Windows Server 2003 作業系統、Windows XP、Windows 2000、Windows NT Windows Millennium Edition、Windows 98 和 Windows 95 一起安裝。
