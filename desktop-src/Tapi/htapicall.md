# <a name="htapicall"></a>HTAPICALL

**HTAPICALL** 資料類型代表呼叫資料結構的 TAPI 不透明控制碼。 TAPI 必須將此類型的值解析為適當資料結構實例的參考。 服務提供者不得嘗試透過這個方式來參考它，就像它是指標、對其值進行假設，或是以任何方式解讀其標記法，而不是在適當的時間將其值傳遞給 TAPI。
