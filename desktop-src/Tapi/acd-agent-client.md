# <a name="acd-agent-client"></a>ACD 代理程式用戶端

撥接中心代理程式負責接收和處理來電，有時會對客戶或潛在客戶發出對外呼叫。 ACD 代理程式用戶端應用程式會提供與 [ACD Proxy](acd-proxy.md) 應用程式的通訊，並提供資訊，例如代理程式是否可供呼叫，以及代理程式可處理的呼叫類型。 下圖說明 TAPI 物件如何查看代理程式。
