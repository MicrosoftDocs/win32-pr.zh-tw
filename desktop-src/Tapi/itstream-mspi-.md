# <a name="itstream-mspi"></a>ITStream (MSPI) 

如果 MSP 與呼叫相關聯，則可以使用 [**ITStream**](/windows/win32/api/tapi3if/nn-tapi3if-itstream) 介面。 這個介面的方法可讓應用程式抓取資料流程的資訊;以啟動、暫停或停止資料流程;並選取或取消選取資料流程上的終端機。

[**ITStream**](/windows/win32/api/tapi3if/nn-tapi3if-itstream)介面是由 MSP 所執行。 如需此介面的詳細資訊，請參閱 MSP 介面一節中的 **ITStream** 。
