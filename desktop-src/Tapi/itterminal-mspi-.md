# <a name="itterminal-mspi"></a>ITTerminal (MSPI) 

[**ITTerminal**](/windows/win32/api/tapi3if/nn-tapi3if-itterminal)介面是 [終端物件](terminal-object.md)的基底介面。 它會提供方法來取得資訊，例如終端機類別和支援的媒體。

[**ITTerminal**](/windows/win32/api/tapi3if/nn-tapi3if-itterminal)介面是由 MSP 所執行，而且如果沒有與該位址相關聯的媒體服務提供者，就無法使用。 如需此介面的詳細資訊，請參閱 MSP 介面一節中的 **ITTerminal** 。
