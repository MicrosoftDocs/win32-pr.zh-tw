# <a name="network"></a>網路

網路會被視為傳輸機制，可將資料從某個點傳達給另一個點。 TAPI 服務提供者會處理執行作業所需的特定通訊協定，例如在指定的網路上建立通訊會話。 一般使用者或伺服器應用程式通常只需要非常一般的網路資訊，例如網址類別型。

一些常見的網路類型：

-   **(單純舊的電話語音) 的 POTS：** 語音和資料會在 *本機迴圈* 中以類比格式傳輸，並以數位方式傳送到其他位置。 一般來說，每個呼叫都會有一個媒體類型，每行一個通道。
-   **ISDN (整合式服務數位網路) ：** 經過數位傳送。 在基本速率介面上，最高可達 128 Kbps 的速度 (BRI-ISDN) 行，而主要速率介面 (PRI-ISDN) 行。 至少有三個通道以及最多32個通道，以同時獨立運作的語音和資料傳輸。 國際標準。
-   **T1/E1：** 經過數位傳送。 T1 是容量為 1.544 Mbps 的傳輸連結，通常用於跨遠端距離連接網路。 在歐洲聯集 T1 中稱為 E1。
-   **切換56：** 透過撥接電話線路以 56 Kbps 發出信號，但需要特殊設備。 受限於呼叫其他特殊配備的設備。
-   **CENTREX：** 透過一般電話線路和使用電話公司設備的集中式網路服務。 不需要特殊設備。
-   **數位 pbx (私用分支交換) 和關鍵系統：** 使用專屬介面在私人電話系統上傳輸語音和資料。
-   **IP 網路：** 使用網際網路通訊協定在網路上傳輸的語音和資料 (IP) ，例如網際網路本身或公司內部網路。

請注意這並不是完整的清單。 如果有適當的服務提供者，就可以支援任何網路傳輸機制。
