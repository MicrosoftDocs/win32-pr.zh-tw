# <a name="telephony-application-programming-interface-version-31"></a>電話語音應用程式設計介面版本3。1

## <a name="purpose"></a>目的

Microsoft 電話語音應用程式開發介面 (TAPI) 3.1 版是一種元件物件模型 (以 COM) 為基礎的 API，可合併傳統和 IP 電話語音。

## <a name="where-applicable"></a>適用時

可能的 TAPI 應用程式包括：

-   具有服務品質 (QOS) 的多播多媒體 IP 會議
-   使用 H. 323 通訊協定在網際網路上進行語音通話
-   能夠追蹤多個代理程式的電話中心應用程式
-   公用交換電話網絡上的基本語音電話 (PSTN) 
-   PBX 控制項
-   互動式語音回應 (IVR) 系統
-   語音信箱

## <a name="developer-audience"></a>開發人員對象

您可以用多種語言撰寫啟用 TAPI 的應用程式，包括 JAVA、Visual Basic 和 c/c + +。 需要熟悉 COM。 電信或其他電話語音應用程式的開發經驗很有説明，但並非必要。

## <a name="run-time-requirements"></a>執行階段需求求

TAPI 3.1 版可讓您開發 Windows Server 2003 作業系統、Windows XP 和 Windows 2000 的通訊應用程式。
