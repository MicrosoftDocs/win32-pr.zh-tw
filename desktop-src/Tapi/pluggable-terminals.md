# <a name="pluggable-terminals"></a>可插入的終端機

可插入的終端機可讓應用程式在執行時間建立終端機。 終端機通常是由電話語音服務提供者/媒體服務提供者 (TSP/MSP) 組來建立和控制，這仍然是操作終端機的最多功能和穩定方法。 不過，有些應用程式在適當的 TSP/MSP 組無法使用的情況下，需要使用簡單的終端機。 只有徹底熟悉 TSP/MSP 作業的應用程式寫入器才應使用可插入的終端機。

在 TAPI 3.1 版中，會將插入的終端機新增至 TAPI 模型。 可插入的終端機可讓協力廠商新增任何 MSP 可使用的新終端機物件。

如需詳細資訊，請參閱 [插入式終端機註冊](pluggable-terminal-registration.md) 和 [寫入插入式終端](writing-a-pluggable-terminal.md) 機。
