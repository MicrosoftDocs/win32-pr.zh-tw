# <a name="third-party-tsp"></a>協力廠商 TSP

專用通訊硬體（例如纜線數據機或電話語音卡）通常會使用一或多個裝置專用的 Tsp 和 Msp。 裝置的檔應該指出功能並提供程式設計指導方針。

如果舊版 TSP 沒有 MSP，但卻使用 wave 裝置，TAPI 會將 Wave MSP 與該 TSP 產生關聯。 未使用 wave 裝置的舊版 Tsp 將不會被指派相關聯的 MSP，也無法使用串流控制作業。

協力廠商 TSP 會隨其預定支援的裝置一起安裝。
