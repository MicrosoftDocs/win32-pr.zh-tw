## <a name="remarks"></a>備註

0xC0000000 至0xFFFFFFFF 的值適用于裝置專屬的延伸模組;0xBFFFFFFF 的值0x80000000 是保留的;以及使用0x00000000 到0x7FFFFFFF 作為要求識別碼。

如果應用程式收到錯誤訊息，指出它不會明確處理 (（例如裝置專屬延伸模組所定義的錯誤）) ，則應該將錯誤視為 PHONEERR \_ OPERATIONFAILED (，以找出未指定原因) 。
