# <a name="s-telephony-api"></a>S (電話語音 API) 

[](a-tapgloss.md) [B](b-tapgloss.md) [C](c-tapgloss.md) [D](d-tapgloss.md) [E](e-tapgloss.md) F [](u-tapgloss.md) [](v-tapgloss.md) G [H](h-tapgloss.md) [I](i-tapgloss.md) J [K](k-tapgloss.md) [L](l-tapgloss.md) [M](m-tapgloss.md) N O [P](p-tapgloss.md) Q [R](r-tapgloss.md) S [T](t-tapgloss.md) [W](w-tapgloss.md) X Y Z

<dl> <dt>

<span id="tapi2.service_provider_tapgloss"></span><span id="TAPI2.SERVICE_PROVIDER_TAPGLOSS"></span>**服務提供者**
</dt> <dd>

動態連結程式庫，可透過一組匯出的服務功能，支援透過電話網絡進行通訊。 服務提供者會執行透過電話網絡進行通訊所需的低層級工作，以回應由 TAPI 動態連結程式庫傳送的電話語音要求。 如此一來，服務提供者與 TAPI 結合，可防止應用程式與電話網絡通訊的服務和技術相關的詳細資料。

</dd> <dt>

<span id="tapi2.subaddressing_tapgloss"></span><span id="TAPI2.SUBADDRESSING_TAPGLOSS"></span>**subaddressing**
</dt> <dd>

在 ISDN 線路上提供的一項功能，可讓您比撥入時使用的單一電話號碼更多的資訊。

</dd> <dt>

<span id="tapi2.supplementary_telephony_tapgloss"></span><span id="TAPI2.SUPPLEMENTARY_TELEPHONY_TAPGLOSS"></span>**補充電話語音**
</dt> <dd>

提供 advanced switch 功能的服務層級，例如保存、傳輸等等。 所有補充服務都是選擇性的;也就是說，服務提供者不需要支援它們。 請參閱 [*輔助電話語音*](a-tapgloss.md)、 [*基本電話語音*](b-tapgloss.md)、 [*擴充電話語音*](e-tapgloss.md#tapi2.extended_telephony_tapgloss)。

</dd> <dt>

<span id="tapi2.switched_56_tapgloss"></span><span id="TAPI2.SWITCHED_56_TAPGLOSS"></span>**切換56**
</dt> <dd>

一種切換的資料服務，可讓使用者撥打電話給其他人，並以全雙工傳輸、數位同步 56 Kbps 來撥打通話的價格。

</dd> <dt>

<span id="tapi2.synchronous_tapgloss"></span><span id="TAPI2.SYNCHRONOUS_TAPGLOSS"></span>**同步**
</dt> <dd>

在連續的位、字元或事件之間有固定時間的資料傳輸方法。 時間是藉由共用單一時鐘來達成。 傳輸的每個結束都會與傳送的資料一起使用時鐘和資訊進行同步處理。 字元會以時間為單位，而不是依開始和停止位。 請參閱 [*非同步*](a-tapgloss.md)。

</dd> </dl>
