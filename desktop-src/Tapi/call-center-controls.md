# <a name="call-center-controls"></a>撥置中心控制項

撥接中心是在電話銀行撥打電話或撥入來電的地點。 例如，銀行或信用卡公司會使用撥入中心來處理帳戶查詢。

TAPI 3.x 可執行物件、介面和方法，協助程式設計人員建立撥接中心應用程式。 從程式設計的觀點來看，使用 TAPI 的電話中心作業會以下圖表示。

![使用 tapi 的電話語音中心作業](images/callcent.png)

有興趣建立撥接中心應用程式的程式設計人員也可能想要查閱可啟用部分撥接中心套件的 TAPI 2.2 功能。 請參閱 TAPI 2.2 [撥中心控制](./call-center-control.md)。
