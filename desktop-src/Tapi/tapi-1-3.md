# <a name="tapi-13"></a>TAPI 1。3

TAPI 1.3 (有時也稱為 TAPI 1.03) 是 TAPI 規格的第一個發行版本。 您可以在 TAPI 1.0 SDK 中找到 TAPI 1.3 系統二進位檔 (Tapi.dll 和其他) ，它只能在 Microsoft 開發人員 Network (MSDN) Level II 訂用帳戶中找到。 可轉散發套件目錄中的系統二進位檔 (可轉散發套件) 可轉散發，但設計成隻能在 Microsoft Windows 3.1 和 Windows 的工作組3.11 上執行。 它們不受支援，也無法在任何版本的32位 Windows 上運作。

用於開發 TAPI 1.3 應用程式的標頭和程式庫僅適用于 TAPI 1.0 SDK，而且僅支援16位開發。 事實上，TAPI 1.3 程式庫是唯一可用的16位 TAPI 程式庫，而且不論目標平臺或目標版本為何，都必須用於所有的16位應用程式。

TAPI 1.3 系統二進位檔只支援 Win16、TAPI 1.3 Tsp 和應用程式。
