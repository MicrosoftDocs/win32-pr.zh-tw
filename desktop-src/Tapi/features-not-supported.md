# <a name="features-not-supported"></a>不支援的功能

-   如果選擇這樣做，則衍生的 MSP 將會執行動態圖形處理。
-   如果選擇這樣做，則衍生的 MSP 將會執行子串流。
-   私用介面（例如參與者、編解碼器控制項等等）不會由基類所執行。
-   TSP-MSP 通訊將由衍生的 MSP 實行。
