<dl> <dt>

<span id="tapi2.integrated_services_digital_network_isdn__tapgloss"></span><span id="TAPI2.INTEGRATED_SERVICES_DIGITAL_NETWORK_ISDN__TAPGLOSS"></span>**整合式服務數位網路 (ISDN)**
</dt> <dd>

一種完全數位的電話/電信網路，可透過在基本速率介面上以高達 144000 bps 的速度傳送數位編碼信號，以高速的方式攜帶語音、資料、影像和影片，)  (的基本速率介面上的最高 bps， (PRI-ISDN) 行。 ISDN 是由至少三個通道所組成，而且最多為32個通道，以同時獨立運作的語音和資料傳輸。 請參閱 [*基本速率介面 (BRI-isdn)*](b-tapgloss.md)， [*主要速率介面 (PRI-isdn)*](p-tapgloss.md)。

</dd> <dt>

<span id="tapi2.interactive_voice_response_ivr_application_tapgloss"></span><span id="TAPI2.INTERACTIVE_VOICE_RESPONSE_IVR_APPLICATION_TAPGLOSS"></span>**互動式語音回應 (IVR) 應用程式**
</dt> <dd>

電話語音應用程式，可透過功能表階層引導電話來電者、提供語音回應、收集語音和資料輸入，以及代表來電者或程式贊助商執行其他作業。 請參閱 [*網路電話語音*](w-tapgloss.md)。

</dd> <dt>

<span id="tapi2.inverse_multiplexing_tapgloss"></span><span id="TAPI2.INVERSE_MULTIPLEXING_TAPGLOSS"></span>**反向多工**
</dt> <dd>

此程式可讓您在相同的時間範圍內，呼叫傳送更多資訊，而非透過單一通道傳輸之單一信號的資訊。 這是藉由將單一高速通道分割成多個信號來完成，將多個信號以比原始信號更低的速率運作，然後以原始的速率將個別傳輸的部分 recombining 至原始信號。 請參閱 [*頻寬*](b-tapgloss.md)。

</dd> <dt>

<span id="tapi2.ip_multicast_tapgloss"></span><span id="TAPI2.IP_MULTICAST_TAPGLOSS"></span>**IP 多播**
</dt> <dd>

IP 多播是一種將相同資料同時傳送至數部機器的配置。 啟用多播感知路由器的網路將能夠更快速地傳達這類資訊，只要在路由器之間傳送封包一次即可。 如需詳細資訊，請參閱 [關於會合 IP 電話語音會議](./about-rendezvous-ip-telephony-conferencing.md)。

</dd> <dt>

<span id="tapi2.isdn_tapgloss"></span><span id="TAPI2.ISDN_TAPGLOSS"></span>**Isdn**
</dt> <dd>

請參閱 *整合式服務數位網路 (ISDN)*。

</dd> </dl>
