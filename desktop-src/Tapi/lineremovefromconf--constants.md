# <a name="lineremovefromconf_-constants"></a>LINEREMOVEFROMCONF \_ 常數

**LINEREMOVEFROMCONF \_** 純量常數會說明參與會議通話的合作物件如何從會議電話中移除。

<dl> <dt>

<span id="LINEREMOVEFROMCONF_ANY"></span><span id="lineremovefromconf_any"></span>**LINEREMOVEFROMCONF \_ 任何**
</dt> <dd> <dl> <dt>



任何參與的合作物件都可以從會議電話中移除。


</dt> </dl> </dd> <dt>

<span id="LINEREMOVEFROMCONF_LAST"></span><span id="lineremovefromconf_last"></span>**LINEREMOVEFROMCONF \_ LAST**
</dt> <dd> <dl> <dt>



只有最近新增的合作物件可以從會議通話中移除


</dt> </dl> </dd> <dt>

<span id="LINEREMOVEFROMCONF_NONE"></span><span id="lineremovefromconf_none"></span>**LINEREMOVEFROMCONF \_ 無**
</dt> <dd> <dl> <dt>



無法從會議通話中移除合作物件。


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>備註

無延伸。 所有32位都是保留的。
