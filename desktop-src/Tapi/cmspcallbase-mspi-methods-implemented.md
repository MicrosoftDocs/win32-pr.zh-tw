# <a name="cmspcallbase-mspi-methods-implemented"></a>CMSPCallBase MSPI 方法實作為



| CMSPCallBase MSPI helper 方法                             | Description                                                                                                                                                                           |
|--------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CreateStream**](/windows/win32/api/tapi3if/nf-tapi3if-itstreamcontrol-createstream)         | 建立新呼叫的媒體串流。 取得函數中的鎖定。                                                                                                           |
| [**EnumerateStreams**](/windows/win32/api/tapi3if/nf-tapi3if-itstreamcontrol-enumeratestreams) | 列舉目前可用的資料流程。 取得函數中的鎖定。                                                                                                            |
| [**取得 \_ 資料流程**](/windows/win32/api/tapi3if/nf-tapi3if-itstreamcontrol-get_streams)          | 建立目前可用資料流程的集合。 提供給 Automation 用戶端應用程式，例如以 Microsoft Visual Basic 撰寫的應用程式。 取得函數中的鎖定。 |
