# <a name="tapi-31-requirements"></a>TAPI 3.1 需求

以下是 TAPI 3.x 的作業系統需求：

-   TAPI 3.0 版需要 Windows 2000 或更新版本。
-   TAPI 3.1 版需要 Windows Server 2003 作業系統或 Windows XP。

請注意，使用會合的 [IP 會議控制項](about-rendezvous-ip-telephony-conferencing.md) 至少需要存取一部 ILS 或 NTDS 伺服器。 如果有 NTDS 伺服器，就可以用它來尋找 ILS 伺服器。 如果沒有任何 NTDS 伺服器存在，終端使用者必須知道 ILS 伺服器的名稱。
