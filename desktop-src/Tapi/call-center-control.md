# <a name="call-center-control"></a>撥置中心控制項

您可以使用基本的 TAPI 2.2 來管理電話中心以及電話語音網路基礎結構 (的其他元素，例如透過協力廠商呼叫控制) 的 IVR 和語音信箱伺服器。 為了協助建立將在伺服器上執行的 ACD proxy，已將功能新增至 TAPI 2.2，以提供應用程式開發人員額外的協助。

下列主題說明您可以用來執行電話中心控制項的 TAPI 2.2 功能：

-   [撥打電話中心的模型](modeling-of-a-call-center.md)
-   [站](stations.md)
-   [預測撥號](predictive-dialing.md)
-   [呼叫佇列和路由點](call-queues-and-route-points.md)
-   [ACD 代理程式監視和控制](acd-agent-monitoring-and-control.md)
-   [站狀態控制](station-status-control.md)
-   [站狀態控制](station-status-control.md)
-   [撥號狀態計時器](call-state-timer.md)
-   [媒體事件計時器](media-event-timers.md)

TAPI 3.x 是撰寫 ACD 代理程式應用程式的慣用 API。 如需有關使用這些介面和方法的詳細資訊，請參閱「 [撥置中心控制項](./about-call-center-controls.md) 」一節。 不過，如有需要，TAPI 2.2 仍可供完整的 ACD 應用程式套件使用。
