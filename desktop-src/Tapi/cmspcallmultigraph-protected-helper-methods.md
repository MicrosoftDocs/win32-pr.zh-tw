# <a name="cmspcallmultigraph-protected-helper-methods"></a>CMSPCallMultiGraph 受保護的 Helper 方法



| CMSPCallMultiGraph 受保護的 helper 方法                             | Description                                                                               |
|-------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**RegisterWaitEvent**](/windows/desktop/api/Mspcall/nf-mspcall-cmspcallmultigraph-registerwaitevent)       | 將等候事件新增至執行緒集區。                                                     |
| [**UnregisterWaitEvent**](/windows/desktop/api/Mspcall/nf-mspcall-cmspcallmultigraph-unregisterwaitevent)   | 從執行緒集區中移除等候事件。                                                |
| [**InternalCreateStream**](/windows/desktop/api/Mspcall/nf-mspcall-cmspcallmultigraph-internalcreatestream) | 由 [**CreateStream**](/windows/win32/api/tapi3if/nf-tapi3if-itstreamcontrol-createstream) 呼叫以建立資料流程物件。 |
