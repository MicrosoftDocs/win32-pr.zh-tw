# <a name="functions-with-unicode-w-versions"></a>Unicode (W) 版本的函式

下列 TAPI 函式是以 Unicode (W) 和 ANSI () 版本來執行。 一般情況下，ANSI 版本的執行會呼叫 Unicode 版本，並執行從 Unicode 轉換為 Unicode 的 ANSI 參數和結構欄位所需的轉換;下表指出已轉換的參數。

明確呼叫泛型 ("W" 或 "A" 尾碼) 函式版本的應用程式將會執行 ANSI 版本，以與舊版 TAPI 相容。

> [!Note]  
> 整個電話語音服務提供者介面 (TSPI) 為2.0 版的 Unicode。

 

下表所列的是包含部分功能變數名稱之 TAPI 結構中的字串欄位參考。 例如， [**LINEFORWARD**](/windows/desktop/api/Tapi/ns-tapi-lineforward) 結構中的 "來電者 Address" 是由 **dwCallerAddressOffset** 欄位所指向，並以 **dwCallerAddressSize** 欄位分隔;在資料表中，此字串會識別為 **CallerAddress**。
