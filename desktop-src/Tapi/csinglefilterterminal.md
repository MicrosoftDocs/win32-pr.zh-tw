# <a name="csinglefilterterminal"></a>CSingleFilterTerminal

**CSingleFilterTerminal** 是另一個適用于靜態和動態終端機的終端基類。 它衍生自 [**CBaseTerminal**](cbaseterminal.md) ，並假設終端機具有單一的 DirectShow 篩選器，讓執行更為明確。 當符合這項條件時，從 **CSingleFilterTerminal** 衍生終端機會比從 **CBaseTerminal** 衍生的簡單得多。

定義于 MSPterm 中。
