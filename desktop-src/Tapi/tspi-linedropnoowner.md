# <a name="tspi_linedropnoowner"></a>TSPI \_ lineDropNoOwner

**TSPI \_ lineDropNoOwner** 函數已淘汰。 如果無法識別初始擁有者，則 TAPI 不會再自動卸載呼叫，以便在用戶端/伺服器環境中容納跨多個用戶端電腦之呼叫的分散式控制。
