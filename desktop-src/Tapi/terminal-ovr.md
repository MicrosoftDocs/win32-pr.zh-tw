# <a name="terminal"></a>終端

終端機是特定位址和媒體類型的來源或轉譯器。 終端機概念主要用於 TAPI 3，而且只對 TAPI 2 應用程式很重要稍微。 如需詳細資訊，請參閱 TAPI 3 [終端物件](terminal-object.md) 總覽。
