# <a name="p-telephony-api"></a>P (電話語音 API) 

[](a-tapgloss.md) [B](b-tapgloss.md) [C](c-tapgloss.md) [D](d-tapgloss.md) [E](e-tapgloss.md) F [](u-tapgloss.md) [](v-tapgloss.md) G [H](h-tapgloss.md) [I](i-tapgloss.md) J [K](k-tapgloss.md) [L](l-tapgloss.md) [M](m-tapgloss.md) N O P Q [R](r-tapgloss.md) [S](s-tapgloss.md) [T](t-tapgloss.md) [W](w-tapgloss.md) X Y Z

<dl> <dt>

<span id="tapi2.phone_centric_connection_tapgloss"></span><span id="TAPI2.PHONE_CENTRIC_CONNECTION_TAPGLOSS"></span>**以電話為主的連接**
</dt> <dd>

由單一的 POTS 線路所組成的連線，電腦會透過桌上型電腦電話組連線到交換器。 比較以 [*電腦為中心的連接*](c-tapgloss.md)。

</dd> <dt>

<span id="tapi2.plain_old_telephone_service_pots__tapgloss"></span><span id="TAPI2.PLAIN_OLD_TELEPHONE_SERVICE_POTS__TAPGLOSS"></span>**單純的舊電話語音 (POTS)**
</dt> <dd>

基本的電話語音服務，提供標準的單行電話、電話線路，以及存取公用的切換式網路。

</dd> <dt>

<span id="tapi2.point_to_point_protocol_ppp__tapgloss"></span><span id="TAPI2.POINT_TO_POINT_PROTOCOL_PPP__TAPGLOSS"></span>**點對點通訊協定 (PPP)**
</dt> <dd>

一種通訊協定，可讓電腦使用高速數據機透過標準撥號電話線連接到其他電腦。

</dd> <dt>

<span id="tapi2.pots_tapgloss"></span><span id="TAPI2.POTS_TAPGLOSS"></span>**盆**
</dt> <dd>

查看單純的 *舊電話語音 (的 POTS)*。

</dd> <dt>

<span id="tapi2.pri_isdn_tapgloss"></span><span id="TAPI2.PRI_ISDN_TAPGLOSS"></span>**PRI-ISDN**
</dt> <dd>

請參閱 *主要速率介面 (PRI-ISDN)*。

</dd> <dt>

<span id="tapi2.primary_rate_interface_pri_isdn__tapgloss"></span><span id="TAPI2.PRIMARY_RATE_INTERFACE_PRI_ISDN__TAPGLOSS"></span>**主要速率介面 (PRI-ISDN)**
</dt> <dd>

一種 ISDN 服務，以 64 Kbps 的速率提供 23 B 通道，64 Kbps (23B + D) 。 在歐盟，PRI 提供 30 B 通道和兩個 資料通道 (30B + 2D) 。 請參閱 [*基本速率介面 (BRI-ISDN)*](b-tapgloss.md)。

</dd> <dt>

<span id="tapi2.private_branch_exchange_pbx__tapgloss"></span><span id="TAPI2.PRIVATE_BRANCH_EXCHANGE_PBX__TAPGLOSS"></span>**私用分支交換 (PBX)**
</dt> <dd>

一種私人的電話切換系統，通常位於客戶的內部部署中（使用「附隨」主控台）。 PBX 連接到一或多個中央辦公室的一組通用行，以提供服務給一些個別電話，例如飯店、商務或政府辦公室。 Pbx 通常會使用專屬的數位線路通訊協定，雖然有些是以類比為基礎。

</dd> <dt>

<span id="tapi2.provisioning_tapgloss"></span><span id="TAPI2.PROVISIONING_TAPGLOSS"></span>**供應**
</dt> <dd>

組成 ISDN 線路的裝置和服務選項群組合。 當您訂購 ISDN 線路時，電話公司會布建這一行，也就是，它會根據交換器的實體功能以及您選擇的選項來設定您的 ISDN 服務。

</dd> <dt>

<span id="tapi2.public_switched_telephone_network_pstn__tapgloss"></span><span id="TAPI2.PUBLIC_SWITCHED_TELEPHONE_NETWORK_PSTN__TAPGLOSS"></span>**(PSTN) 的公用交換電話網絡**
</dt> <dd>

本機類比電話網絡，提供 3.1 kHz 語音等級的服務品質 (其持有人模式) 。

</dd> </dl>
