# <a name="storyboard-interfaces"></a>分鏡指令碼介面

本節包含支援分鏡腳本的 Windows 動畫管理員介面參考規格。

## <a name="in-this-section"></a>本節內容



| 主題                                                                                                 | 描述                                                                                                           |
|-------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**IUIAnimationStoryboard**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationstoryboard)<br/>                                   | 定義分鏡腳本，其中包含一組相對於彼此同步的轉換。<br/> |
| [**IUIAnimationStoryboard2**](/windows/win32/api/uianimation/nn-uianimation-iuianimationstoryboard2)<br/>                                 | 定義分鏡腳本，其中包含一組相對於彼此同步的轉換。<br/> |
| [**IUIAnimationStoryboardEventHandler**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationstoryboardeventhandler)<br/>           | 定義處理分鏡腳本狀態和更新事件的方法。<br/>                                    |
| [**IUIAnimationStoryboardEventHandler2**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationstoryboardeventhandler2)<br/>         | 定義處理分鏡腳本事件的方法。 <br/>                                                           |
| [**IUIAnimationLoopIterationChangeHandler2**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationloopiterationchangehandler2)<br/> | 定義處理分鏡腳本迴圈反復專案事件的方法。<br/>                                            |
| [**IUIAnimationPriorityComparison**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationprioritycomparison)<br/>                   | 定義優先順序比較的方法，動畫管理員會使用此方法來解決排程衝突。<br/>  |
| [**IUIAnimationPriorityComparison2**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationprioritycomparison2)<br/>                 | 定義方法，透過優先權比較來解決排程衝突。<br/>                           |
