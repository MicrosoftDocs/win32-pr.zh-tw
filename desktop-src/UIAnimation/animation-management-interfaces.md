# <a name="animation-management-interfaces"></a>動畫管理介面

本節包含支援動畫管理之 Windows 動畫管理員介面的參考規格。

## <a name="in-this-section"></a>本節內容



| 主題                                                                                   | 描述                                                                                                                                                              |
|-----------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IUIAnimationManager**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationmanager)<br/>                           | 定義動畫管理員，其提供用來建立和管理動畫的中央介面。<br/>                                                       |
| [**IUIAnimationManager2**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationmanager2)<br/>                         | 定義 [**動畫管理員**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationmanager2)，其提供用來建立及管理多個維度動畫的中央介面。<br/> |
| [**IUIAnimationManagerEventHandler**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationmanagereventhandler)<br/>   | 定義處理動畫管理員之狀態更新的方法。<br/>                                                                                         |
| [**IUIAnimationManagerEventHandler2**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationmanagereventhandler2)<br/> | 定義處理 [**動畫管理員**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationmanager2)更新的方法。<br/>                                                                |
