# <a name="windows-windows-and-messages"></a>Windows (視窗和訊息)

在以圖形化 Windows 為基礎的應用程式中，視窗是畫面的矩形區域，其中應用程式會顯示輸出並接收來自使用者的輸入。 因此，以圖形化 Windows 為基礎的應用程式的第一個工作就是建立一個視窗。

視窗會與其他視窗共用畫面，包括來自其他應用程式的視窗。 一次只有一個視窗可以接收來自使用者的輸入。 使用者可以使用滑鼠、鍵盤或其他輸入裝置，與此視窗及其擁有的應用程式互動。
