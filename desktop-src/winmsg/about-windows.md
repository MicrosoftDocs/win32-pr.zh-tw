## <a name="desktop-window"></a><span data-ttu-id="8cf3c-127">桌面視窗</span><span class="sxs-lookup"><span data-stu-id="8cf3c-127">Desktop Window</span></span>

<span data-ttu-id="8cf3c-130">桌面視窗使用點陣圖來繪製畫面的背景。</span><span class="sxs-lookup"><span data-stu-id="8cf3c-130">The desktop window uses a bitmap to paint the background of the screen.</span></span> <span data-ttu-id="8cf3c-131">點陣圖建立的模式稱為 *桌面壁紙*。</span><span class="sxs-lookup"><span data-stu-id="8cf3c-131">The pattern created by the bitmap is called the *desktop wallpaper*.</span></span> <span data-ttu-id="8cf3c-132">根據預設，桌面視窗會使用登錄中指定的 .bmp 檔中的點陣圖作為桌面壁紙。</span><span class="sxs-lookup"><span data-stu-id="8cf3c-132">By default, the desktop window uses the bitmap from a .bmp file specified in the registry as the desktop wallpaper.</span></span>
