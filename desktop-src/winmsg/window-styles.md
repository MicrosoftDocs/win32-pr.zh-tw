# <a name="window-styles"></a>視窗樣式

視窗樣式如下所示。 建立視窗之後，除非有注明，否則無法修改這些樣式。



| 常數/值                                                                                                                                                                                                                                                                                                                              | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="WS_BORDER"></span><span id="ws_border"></span><dl> <dt>**WS \_框線**</dt> <dt>0x00800000L</dt> </dl>                                                                                                                          | 視窗具有細線框線。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| <span id="WS_CAPTION"></span><span id="ws_caption"></span><dl> <dt>**WS \_標題**</dt> <dt>0x00C00000L</dt> </dl>                                                                                                                       | 此視窗具有標題列 (包含 **WS \_ 框線** 樣式) 。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| <span id="WS_CHILD"></span><span id="ws_child"></span><dl> <dt>**WS \_子**</dt> <dt>0x40000000L</dt> </dl>                                                                                                                             | 視窗是子視窗。 具有此樣式的視窗不可以有功能表列。 這個樣式無法與 WS 快顯 **樣式 \_** 搭配使用。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="WS_CHILDWINDOW"></span><span id="ws_childwindow"></span><dl> <dt>**WS \_CHILDWINDOW**</dt> <dt>0x40000000L</dt> </dl>                                                                                                           | 與 **WS \_ 子** 樣式相同。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="WS_CLIPCHILDREN"></span><span id="ws_clipchildren"></span><dl> <dt>**WS \_CLIPCHILDREN**</dt> <dt>0x02000000L</dt> </dl>                                                                                                        | 當繪圖出現在父視窗內時，排除子視窗所佔用的區域。 這是在建立父視窗時使用的樣式。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| <span id="WS_CLIPSIBLINGS"></span><span id="ws_clipsiblings"></span><dl> <dt>**WS \_CLIPSIBLINGS**</dt> <dt>0x04000000L</dt> </dl>                                                                                                        | 裁剪相對於彼此的子視窗;也就是說，當特定的子視窗收到 [**WM \_ 油漆**](../gdi/wm-paint.md) 訊息時， **WS \_ CLIPSIBLINGS** 樣式會將子視窗中所有其他重迭的子視窗剪下，以進行更新。 如果未指定 **WS \_ CLIPSIBLINGS** ，且子視窗重迭，則在子視窗的工作區中繪製時，可以在相鄰子視窗的工作區中繪製。<br/>                                                                                                                                                                                |
| <span id="WS_DISABLED"></span><span id="ws_disabled"></span><dl> <dt>**WS \_停用**</dt>的 <dt>0x08000000L</dt> </dl>                                                                                                                    | 視窗一開始已停用。 停用的視窗無法接收來自使用者的輸入。 若要在建立視窗之後變更此功能，請使用 [**EnableWindow**](/windows/win32/api/winuser/nf-winuser-enablewindow) 函式。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| <span id="WS_DLGFRAME"></span><span id="ws_dlgframe"></span><dl> <dt>**WS \_DLGFRAME**</dt> <dt>0x00400000L</dt> </dl>                                                                                                                    | 視窗具有一般與對話方塊搭配使用的樣式框線。 具有此樣式的視窗不能有標題列。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="WS_GROUP"></span><span id="ws_group"></span><dl> <dt>**WS \_群組**</dt> <dt>0x00020000L</dt> </dl>                                                                                                                             | 視窗是控制項群組的第一個控制項。 此群組是由第一個控制項和其後定義的所有控制項所組成，直到下一個具有 **WS \_ group** 樣式的控制項為止。 每個群組中的第一個控制項通常會有 **WS \_ TABSTOP** 樣式，讓使用者可以從群組移至群組。 使用者接著可以使用方向索引鍵，將群組中某個控制項的鍵盤焦點變更為群組中的下一個控制項。<br/> 您可以開啟和關閉此樣式來變更對話方塊導覽。 若要在建立視窗之後變更這個樣式，請使用 [**SetWindowLong**](/windows/win32/api/winuser/nf-winuser-setwindowlonga) 函數。<br/> |
| <span id="WS_HSCROLL"></span><span id="ws_hscroll"></span><dl> <dt>**WS \_HSCROLL**</dt> <dt>0x00100000L</dt> </dl>                                                                                                                       | 視窗具有水準捲軸。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| <span id="WS_ICONIC"></span><span id="ws_iconic"></span><dl> <dt>**WS \_ICONIC**</dt> <dt>0x20000000L</dt> </dl>                                                                                                                          | 視窗一開始最小化。 與 **WS \_ 最小化** 樣式相同。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="WS_MAXIMIZE"></span><span id="ws_maximize"></span><dl> <dt>**WS \_最大化**</dt> <dt>0x01000000L</dt> </dl>                                                                                                                    | 視窗一開始會最大化。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| <span id="WS_MAXIMIZEBOX"></span><span id="ws_maximizebox"></span><dl> <dt>**WS \_MAXIMIZEBOX**</dt> <dt>0x00010000L</dt> </dl>                                                                                                           | 視窗有最大化按鈕。 無法與 **WS \_ EX \_ CONTEXTHELP** 樣式結合。 您也必須指定 **WS \_ SYSMENU** 樣式。 <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| <span id="WS_MINIMIZE"></span><span id="ws_minimize"></span><dl> <dt>**WS \_最小化**</dt> <dt>0x20000000L</dt> </dl>                                                                                                                    | 視窗一開始最小化。 與 **WS \_ ICONIC** 樣式相同。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| <span id="WS_MINIMIZEBOX"></span><span id="ws_minimizebox"></span><dl> <dt>**WS \_MINIMIZEBOX**</dt> <dt>0x00020000L</dt> </dl>                                                                                                           | 視窗具有最小化按鈕。 無法與 **WS \_ EX \_ CONTEXTHELP** 樣式結合。 您也必須指定 **WS \_ SYSMENU** 樣式。 <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| <span id="WS_OVERLAPPED"></span><span id="ws_overlapped"></span><dl> <dt>**WS \_**</dt>重迭 <dt>0x00000000L</dt> </dl>                                                                                                              | 視窗是重迭的視窗。 重迭的視窗具有標題列和框線。 與 **WS \_ 磚** 樣式相同。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| <span id="WS_OVERLAPPEDWINDOW"></span><span id="ws_overlappedwindow"></span><dl> <dt>**WS \_OVERLAPPEDWINDOW**</dt> <dt> (ws 重迭 \_ \| ws \_ 標題 \| ws \_ SYSMENU \| ws \_ THICKFRAME \| ws \_ MINIMIZEBOX \| WS \_ MAXIMIZEBOX)</dt> </dl> | 視窗是重迭的視窗。 與 **WS \_ TILEDWINDOW** 樣式相同。 <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| <span id="WS_POPUP"></span><span id="ws_popup"></span><dl> <dt>**WS \_快顯視窗**</dt> <dt>0x80000000L</dt> </dl>                                                                                                                             | 視窗是一個快顯視窗。 這個樣式無法與 **WS \_ 子** 樣式搭配使用。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| <span id="WS_POPUPWINDOW"></span><span id="ws_popupwindow"></span><dl> <dt>**WS \_POPUPWINDOW**</dt> <dt> (WS \_ POPUP \| ws \_ BORDER \| ws \_ SYSMENU)</dt> </dl>                                                                              | 視窗是一個快顯視窗。 您必須結合 **ws \_ 標題** 和 **ws \_ POPUPWINDOW** 樣式，讓視窗功能表變成可見。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| <span id="WS_SIZEBOX"></span><span id="ws_sizebox"></span><dl> <dt>**WS \_SIZEBOX**</dt> <dt>0x00040000L</dt> </dl>                                                                                                                       | 視窗具有調整大小框線。 與 **WS \_ THICKFRAME** 樣式相同。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="WS_SYSMENU"></span><span id="ws_sysmenu"></span><dl> <dt>**WS \_SYSMENU**</dt> <dt>0x00080000L</dt> </dl>                                                                                                                       | 視窗的標題列上有視窗功能表。 您也必須指定 **WS \_ 標題** 樣式。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="WS_TABSTOP"></span><span id="ws_tabstop"></span><dl> <dt>**WS \_TABSTOP**</dt> <dt>0x00010000L</dt> </dl>                                                                                                                       | 視窗是使用者按下 TAB 鍵時，可以接收鍵盤焦點的控制項。 按下 TAB 鍵會將鍵盤焦點變更為具有 **WS \_ TABSTOP** 樣式的下一個控制項。<br/> 您可以開啟和關閉此樣式來變更對話方塊導覽。 若要在建立視窗之後變更這個樣式，請使用 [**SetWindowLong**](/windows/win32/api/winuser/nf-winuser-setwindowlonga) 函數。 若要讓使用者建立的視窗和非強制回應對話方塊使用索引標籤停止，請修改訊息迴圈來呼叫 [**IsDialogMessage**](/windows/win32/api/winuser/nf-winuser-isdialogmessagea) 函數。<br/>                                                                                           |
| <span id="WS_THICKFRAME"></span><span id="ws_thickframe"></span><dl> <dt>**WS \_THICKFRAME**</dt> <dt>0x00040000L</dt> </dl>                                                                                                              | 視窗具有調整大小框線。 與 **WS \_ SIZEBOX** 樣式相同。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
