# <a name="pointer-input-messages-and-notifications-glossary"></a>指標輸入訊息和通知詞彙

[指標輸入訊息和通知](messages-and-notifications-portal.md)的特定詞彙和定義。

<dl> <dt>

<span id="inputmsg.inputmsg_glossary_pointer"></span><span id="INPUTMSG.INPUTMSG_GLOSSARY_POINTER"></span>**指標**
</dt> <dd>

任何指標類型的輸入。

</dd> <dt>

<span id="inputmsg.inputmsg_glossary_pointer_identifier"></span><span id="INPUTMSG.INPUTMSG_GLOSSARY_POINTER_IDENTIFIER"></span>**指標 identfier**
</dt> <dd>

在其存留期內唯一識別指標的識別碼。 當第一次偵測到指標時，指標會變成存在狀態，並在它離開偵測範圍時結束其存在。

</dd> <dt>

<span id="inputmsg.inputmsg_glossary_pointer_frame"></span><span id="INPUTMSG.INPUTMSG_GLOSSARY_POINTER_FRAME"></span>**指標框架**
</dt> <dd>

與指定視窗相關聯之所有指標的單一更新。

</dd> </dl>
