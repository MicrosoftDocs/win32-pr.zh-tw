# <a name="pointer-input-messages-and-notifications-functions"></a>指標輸入訊息和通知函數

本節中的主題提供 [指標輸入訊息和通知](messages-and-notifications-portal.md) 函數的參考規格。

## <a name="in-this-section"></a>本節內容



| 主題                                                                                 | 描述                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
|---------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**EnableMouseInPointer**](/previous-versions/windows/desktop/api)<br/>                       | 讓滑鼠成為指標輸入裝置，並傳送 [WM_POINTER](messages.md) 訊息。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| [**GetPointerCursorId**](/previous-versions/windows/desktop/api)<br/>                           | 捕獲與指定指標相關聯的資料指標識別碼。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| [**GetPointerFrameInfo**](/previous-versions/windows/desktop/api)<br/>                         | 取得與目前訊息相關聯之指定指標的整個資訊框架。 <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| [**GetPointerFrameInfoHistory**](/previous-versions/windows/desktop/api)<br/>           | 取得資訊的整個框架 (包括針對與目前訊息相關聯的指定指標，將輸入框架合併) 。 <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| [**GetPointerFramePenInfo**](/previous-versions/windows/desktop/api)<br/>                   | 取得指定指標 (的整個畫面格資訊， [**PT_PEN**](/previous-versions/windows/desktop/api)) 與目前訊息相關聯的類型。 <br/>                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [**GetPointerFramePenInfoHistory**](/previous-versions/windows/desktop/api)<br/>     | 取得以畫筆為基礎之資訊的整個框架 (包括) 指定指標的結合輸入框架 ([**PT_PEN**](/previous-versions/windows/desktop/api)) 與目前訊息相關聯。 <br/>                                                                                                                                                                                                                                                                                                                                                                                        |
| [**GetPointerFrameTouchInfo**](/previous-versions/windows/desktop/api)<br/>               | 取得與目前訊息相關聯 [**PT_TOUCH**](/previous-versions/windows/desktop/api)) 類型之指定指標 (之觸控型資訊的整個框架。 <br/>                                                                                                                                                                                                                                                                                                                                                                                                                       |
| [**GetPointerFrameTouchInfoHistory**](/previous-versions/windows/desktop/api)<br/> | 取得以觸控為基礎之資訊的整個框架 (包括將輸入框架) 指定的指標 (類型 [**PT_TOUCH**](/previous-versions/windows/desktop/api)) 與目前的訊息相關聯。 <br/>                                                                                                                                                                                                                                                                                                                                                                                    |
| [**GetPointerInfo**](/previous-versions/windows/desktop/api)<br/>                                   | 取得與目前訊息相關聯之指定指標的資訊。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [**GetPointerInfoHistory**](/previous-versions/windows/desktop/api)<br/>                     | 取得與個別輸入相關聯的資訊（如果有的話），這些輸入已合併到指定指標的目前訊息中。 最新的輸入會包含在傳回的歷程記錄中，而且與 [**GetPointerInfo**](/previous-versions/windows/desktop/api) 函數所傳回的最新輸入相同。<br/>                                                                                                                                                                                                                                                                              |
| [**GetPointerInputTransform**](/previous-versions/windows/desktop/api)<br/>               | 取得與目前訊息相關聯之指標資訊座標的一或多個轉換。 <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**GetPointerPenInfo**](/previous-versions/windows/desktop/api)<br/>                             | 取得與目前訊息相關聯之類型 [**PT_PEN**](/previous-versions/windows/desktop/api)) 之指定指標 (的以畫筆為基礎的資訊。 <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [**GetPointerPenInfoHistory**](/previous-versions/windows/desktop/api)<br/>               | 取得與個別輸入相關聯的畫筆資訊（如果有的話），這些資訊已合併至 [**PT_PEN**](/previous-versions/windows/desktop/api)) 類型之指定指標 (的目前訊息。 最新的輸入會包含在傳回的歷程記錄中，而且與 [**GetPointerPenInfo**](/previous-versions/windows/desktop/api) 函數所傳回的最新輸入相同。<br/>                                                                                                                                                                                                         |
| [**GetPointerTouchInfo**](/previous-versions/windows/desktop/api)<br/>                         | 取得與目前訊息相關聯之類型 [**PT_TOUCH**](/previous-versions/windows/desktop/api)) 之指定指標 (的觸控式資訊。 <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**GetPointerTouchInfoHistory**](/previous-versions/windows/desktop/api)<br/>           | 取得與個別輸入相關聯的觸控式資訊（如果有的話），這些資訊已合併至 [**PT_TOUCH**](/previous-versions/windows/desktop/api)) 類型之指定指標 (的目前訊息。 最新的輸入會包含在傳回的歷程記錄中，而且與 [**GetPointerTouchInfo**](/previous-versions/windows/desktop/api) 函數所傳回的最新輸入相同。<br/>                                                                                                                                                                                                 |
| [**GetPointerType**](/previous-versions/windows/desktop/api)<br/>                                   | 抓取指定指標的指標類型。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [**GetUnpredictedMessagePos**](/previous-versions/windows/desktop/api)<br/>               | 在指標資料經過觸控預測處理之前取得指標資料。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**IsMouseInPointerEnabled**](/previous-versions/windows/desktop/api)<br/>                 | 指出是否為滑鼠設定 [**EnableMouseInPointer**](/previous-versions/windows/desktop/api) ，以作為指標輸入裝置，並傳送 [WM_POINTER](messages.md) 訊息。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                    |
| [**SkipPointerFrameMessages**](/previous-versions/windows/desktop/api)<br/>               | 判斷哪一個指標輸入框架針對指定的指標產生最近抓取的訊息，並捨棄從相同指標輸入框架產生的任何佇列 (unretrieved) 指標輸入訊息。 如果應用程式已使用 [**GetPointerFrameInfo**](/previous-versions/windows/desktop/api) 函式、 [**GetPointerFrameInfoHistory**](/previous-versions/windows/desktop/api) 函式或其中一個類型特定的 variant 來抓取整個框架的資訊，它就可以使用此函式來避免從該框架逐一抓取和捨棄剩餘的訊息。<br/> |
