# <a name="v-volume-shadow-copy-service"></a>V (磁碟區陰影複製服務)

<dl> <dt>

<span id="base.vssgloss_volume"></span><span id="BASE.VSSGLOSS_VOLUME"></span>**體積**
</dt> <dd>

邏輯儲存體單位。 磁片區是陰影複製建立作業的目標。

</dd> <dt>

<span id="base.vssgloss_volume_name"></span><span id="BASE.VSSGLOSS_VOLUME_NAME"></span>**磁片區 GUID 名稱**
</dt> <dd>

磁片區的唯一名稱。 磁片區 GUID 名稱是下列格式的字串：

\\\\?\\磁片區 {GUID}\\

 (記下尾端的 " \\ " ) 其中 GUID 是磁片區的全域唯一識別碼。 作業系統會在第一次遇到磁片區時指派磁片區名稱，例如在格式化或安裝期間。

請注意，磁片區可以有一個以上的磁片區 GUID 名稱。

</dd> </dl>
