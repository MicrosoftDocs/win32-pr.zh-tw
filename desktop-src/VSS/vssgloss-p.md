# <a name="p-volume-shadow-copy-service"></a>P (磁碟區陰影複製服務) 

[](vssgloss-a.md) [B](vssgloss-b.md) [C](vssgloss-c.md) [D](vssgloss-d.md) [E](vssgloss-e.md) [F](vssgloss-f.md) [G](vssgloss-g.md) [H](vssgloss-h.md) [I](vssgloss-i.md) J K [L](vssgloss-l.md) M [N](vssgloss-n.md) [O](vssgloss-o.md) P Q [R](vssgloss-r.md) [S](vssgloss-s.md) [T](vssgloss-t.md) [](vssgloss-v.md) [W](vssgloss-w.md) X Y Z

<dl> <dt>

<span id="base.vssgloss_partial_file_support"></span><span id="BASE.VSSGLOSS_PARTIAL_FILE_SUPPORT"></span>**部分檔案支援**
</dt> <dd>

藉由修改相關檔案的各小節來執行備份作業的容量，而不是使用整個檔案。 另請參閱 [*導向目標*](vssgloss-d.md)。

</dd> <dt>

<span id="base.vssgloss_persistent_shadow_copy"></span><span id="BASE.VSSGLOSS_PERSISTENT_SHADOW_COPY"></span>**持續陰影複製**
</dt> <dd>

當要求者釋放 [**>ivssbackupcomponents**](/windows/desktop/api/VsBackup/nl-vsbackup-ivssbackupcomponents) 物件或電腦重新開機時，不會自動刪除的陰影複製。 另請參閱 [*自動發行陰影複製*](vssgloss-a.md)、 [*陰影複製*](vssgloss-s.md)。

</dd> <dt>

<span id="base.vssgloss_plex"></span><span id="BASE.VSSGLOSS_PLEX"></span>**叢**
</dt> <dd>

一種特殊類型的陰影複製磁片區，完整且完全代表陰影複製磁片區，而不需要任何原始磁片區。 這通常也稱為分割鏡像，但本檔使用的是 Plex。

</dd> <dt>

<span id="base.vssgloss_postrestore_event"></span><span id="BASE.VSSGLOSS_POSTRESTORE_EVENT"></span>**PostRestore 事件**
</dt> <dd>

表示 VSS 還原已完成的 VSS 事件。

</dd> <dt>

<span id="base.vssgloss_postsnapshot_event"></span><span id="BASE.VSSGLOSS_POSTSNAPSHOT_EVENT"></span>**PostSnapshot 事件**
</dt> <dd>

表示陰影複製已完成的 VSS 事件。 另請參閱 [*陰影複製*](vssgloss-s.md)。

</dd> <dt>

<span id="base.vssgloss_preparebackup_event"></span><span id="BASE.VSSGLOSS_PREPAREBACKUP_EVENT"></span>**PrepareForBackup 事件**
</dt> <dd>

指出即將進行備份作業的 VSS 事件。

</dd> <dt>

<span id="base.vssgloss_preparesnapshot_event"></span><span id="BASE.VSSGLOSS_PREPARESNAPSHOT_EVENT"></span>**PrepareForSnapshot 事件**
</dt> <dd>

VSS 事件，指出寫入器應該採取步驟來準備即將進行的陰影複製操作。 另請參閱 [*陰影複製*](vssgloss-s.md)。

</dd> <dt>

<span id="base.vssgloss_prerestore_event"></span><span id="BASE.VSSGLOSS_PRERESTORE_EVENT"></span>**PreRestore 事件**
</dt> <dd>

VSS 事件，指出即將進行還原作業。

</dd> <dt>

<span id="base.vssgloss_provider"></span><span id="BASE.VSSGLOSS_PROVIDER"></span>**供應商**
</dt> <dd>

一種作業系統物件，它會攔截並管理 i/o 要求，以建立並代表檔案系統的磁片區陰影複製。 另請參閱 [*硬體提供者*](vssgloss-h.md)、 [*軟體提供者*](vssgloss-s.md)。

</dd> </dl>
