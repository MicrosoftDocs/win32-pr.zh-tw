# <a name="vss-security-issues"></a>VSS 安全性問題

VSS 基礎結構在內部是以 COM 為基礎，因此允許寫入器和要求者進行通訊，需要以 COM 為基礎的安全性。 COM 安全性提供了 VSS 開發人員應考慮的機制和指導方針：

-   [寫入器的安全性考慮](security-considerations-for-writers.md)
-   [要求者的安全性考慮](security-considerations-for-requestors.md)
