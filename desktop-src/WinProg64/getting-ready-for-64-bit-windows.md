# <a name="getting-ready-for-64-bit-windows"></a>準備開始64位 Windows

Windows 64 位版本的主要目標是讓開發人員輕鬆地針對其32位和64位應用程式使用單一原始程式碼基底。 本總覽說明如何讓您的原始程式碼同時支援32位和64位運算。 熟悉現有的 [Windows 資料類型](/windows/desktop/WinProg/windows-data-types) 將有所説明。
