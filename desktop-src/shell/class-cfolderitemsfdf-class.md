# <a name="cfolderitemsfdf-class"></a>CFolderItemsFDF 類別

**CFolderItemsFDF** 是 [**FolderItems**](folderitems.md)的集合。 它會實 [**IInsertItem**](/windows/desktop/api/Shobjidl/nn-shobjidl-iinsertitem) 介面。

## <a name="members"></a>成員

**CFolderItemsFDF** 類別繼承自 [**CFolderItems**](class-cfolderitems-class.md) ，但沒有其他成員。

## <a name="remarks"></a>備註

FDF 代表 *來自不同的資料夾*。

[**CFolderItems**](class-cfolderitems-class.md)類別會列舉多個資料夾中的專案，並使用 [**IInsertItem**](/windows/desktop/api/Shobjidl/nn-shobjidl-iinsertitem)介面將專案插入集合中。
