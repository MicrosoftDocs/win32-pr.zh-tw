# <a name="islibrarypinned-element-library-schema"></a>isLibraryPinned 元素 (程式庫架構) 

<isLibraryPinned>元素指定此程式庫是否釘選到 Windows 檔案總管的導覽窗格。 這個元素是選擇性的，且沒有任何子項目且沒有任何屬性。

## <a name="syntax"></a>Syntax


```
<!-- isLibraryPinned -->
    <xs:element name="isLibraryPinned" type="xs:boolean" default="false" minOccurs="0"/>
```



## <a name="element-information"></a>項目資訊



| Parent 項目                                                               | 子元素 |
|------------------------------------------------------------------------------|----------------|
| [libraryDescription 元素 (程式庫架構) ](schema-librarydescription.md) |                |



 

## <a name="remarks"></a>備註

若為 true，則會將程式庫釘選到 Windows 檔案總管流覽窗格。
