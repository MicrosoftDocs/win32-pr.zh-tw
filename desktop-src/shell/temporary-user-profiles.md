# <a name="temporary-user-profiles"></a>暫存使用者設定檔

每次發生錯誤情況時，就會發出暫時的使用者設定檔，以防止使用者的設定檔載入。 暫時設定檔會在每個會話結束時刪除，而使用者對其桌面設定和檔案所做的變更會在使用者登出時遺失。 只有在執行 Windows 2000 和更新版本的電腦上才能使用暫存設定檔。
