# <a name="fm_getselcount-message"></a>FM \_ GETSELCOUNT 訊息

由檔案管理員延伸模組所傳送，可在 [active 檔案管理員] 視窗中取出所選檔案的計數 (目錄視窗或 [搜尋結果] 視窗) 。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>必須為零。</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

傳回選取的檔案數目。
