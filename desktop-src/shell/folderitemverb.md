# <a name="folderitemverb-object"></a>FolderItemVerb 物件

表示專案可用的單一動詞。 此物件包含屬性和方法，可讓您取得動詞的相關資訊。

## <a name="members"></a>成員

**FolderItemVerb** 物件具有下列類型的成員：

-   [方法](#methods)
-   [屬性](#properties)

### <a name="methods"></a>方法

**FolderItemVerb** 物件有這些方法。



| 方法                              | 描述                                                                                  |
|:------------------------------------|:---------------------------------------------------------------------------------------------|
| [**Doit r**](folderitemverb-doit.md) | 在與動詞相關聯的 [**FolderItem**](folderitem.md) 上執行動詞。<br/> |



 

### <a name="properties"></a>屬性

**FolderItemVerb** 物件具有這些屬性。



| 屬性                                                     | 存取類型          | 描述                          |
|:-------------------------------------------------------------|:---------------------|:-------------------------------------|
| [**Application**](folderitemverb-application.md)<br/> | 唯讀<br/> | 未實作。<br/>          |
| [name - **](folderitemverb-name.md)<br/>               | 唯讀<br/> | 包含動詞的名稱。<br/> |
| [**父代**](folderitemverb-parent.md)<br/>           | 唯讀<br/> | 未實作。<br/>          |
