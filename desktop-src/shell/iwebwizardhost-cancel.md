# <a name="webwizardhostcancel-method"></a>WebWizardHost. Cancel 方法

模擬 [ **取消** ] 按鈕的按一下。

## <a name="syntax"></a>語法


```JScript
iRetVal = WebWizardHost.Cancel()
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="remarks"></a>備註

用戶端會藉由關閉嚮導，負責回應此方法的預期行為。
