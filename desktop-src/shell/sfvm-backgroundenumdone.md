# <a name="sfvm_backgroundenumdone-message"></a>SFVM \_ BACKGROUNDENUMDONE 訊息

通知回呼物件背景列舉已完成。 由 [**IShellFolderViewCB：： MessageSFVCB**](/windows/win32/api/shlobj_core/nf-shlobj_core-ishellfolderviewcb-messagesfvcb)使用。


```C++

                SFVM_BACKGROUNDENUMDONE

            
```



## <a name="parameters"></a>參數

此訊息沒有任何參數。
