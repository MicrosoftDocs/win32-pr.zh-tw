# <a name="ext_button-structure"></a>EXT \_ 按鈕結構

包含有關檔案管理員延伸模組 DLL 新增至 [檔案管理員] 工具列之按鈕的資訊。

## <a name="syntax"></a>語法


```C++
typedef struct tagEXT_BUTTON {
  WORD idCommand;
  WORD idsHelp;
  WORD fsStyle;
} EXT_BUTTON, *LPEXT_BUTTON;
```



## <a name="members"></a>成員

<dl> <dt>

**idCommand**
</dt> <dd>

類型： **WORD**

</dd> <dd>

按鈕的命令識別碼。

</dd> <dt>

**idsHelp**
</dt> <dd>

類型： **WORD**

</dd> <dd>

按鈕的說明字串識別碼。

</dd> <dt>

**fsStyle**
</dt> <dd>

類型： **WORD**

</dd> <dd>

按鈕的樣式。

</dd> </dl>
