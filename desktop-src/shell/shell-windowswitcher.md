# <a name="shellwindowswitcher-method"></a>Shell：： WindowSwitcher 方法

將開啟的視窗顯示在您可以切換的3D 堆疊中。

## <a name="syntax"></a>語法


```JScript
.WindowSwitcher()
```


```VB

Sub WindowSwitcher()
```





## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

### <a name="jscript"></a>JScript

這個方法不會傳回值。

### <a name="vb"></a>VB

這個方法不會傳回值。
