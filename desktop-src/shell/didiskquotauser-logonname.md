# <a name="didiskquotauserlogonname-property"></a>DIDiskQuotaUser. LogonName 屬性

取得使用者的登入帳戶名稱。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
LogonName = DIDiskQuotaUser.LogonName
```



## <a name="property-value"></a>屬性值

設定為使用者帳戶登入名稱的字串值。
