## <a name="remarks"></a>備註

**FOLDERTYPEID** 值為 guid，並在 Shlguid 中宣告。

在實務上，不論資料夾的內容為何，任何 **FOLDERTYPEID** 值都可以套用至任何資料夾。 **FOLDERTYPEID** 只是一個 view 範本，可針對預期的資料夾內容提供適當的版面配置和資料行。 針對 **FOLDERTYPEID** 值，不會對資料夾內容進行任何驗證，以確保相容性。

這些資料夾和其他資料夾類型都是在登錄的 **FolderTypes** 子機碼下定義，如下所示：

```
HKEY_LOCAL_MACHINE
   SOFTWARE
      Microsoft
         Windows
            CurrentVersion
               Explorer
                  FolderTypes
```
