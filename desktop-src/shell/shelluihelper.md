# <a name="shelluihelper-object"></a>ShellUIHelper 物件

由 shell 所執行，以協助編寫腳本和 Microsoft Visual Basic 開發人員使用 shell 中提供的一些功能。 **ShellUIHelper** 物件沒有任何屬性或事件。 提供方法以將專案新增至 Shell。

## <a name="members"></a>成員

**ShellUIHelper** 物件具有下列類型的成員：

-   [方法](#methods)
