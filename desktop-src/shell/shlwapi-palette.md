# <a name="shell-color-palette-handling-functions"></a>Shell 色彩調色板處理函式

本節說明 Windows Shell 色彩調色板處理函式。 本檔中所述的程式設計項目是由 Shlwapi.dll 匯出，並定義于 Shlwapi 和 Shlwapi 中。

## <a name="in-this-section"></a>本節內容



| 主題                                                           | 描述                                                                           |
|-----------------------------------------------------------------|---------------------------------------------------------------------------------------|
| [**ColorAdjustLuma**](/windows/desktop/api/Shlwapi/nf-shlwapi-coloradjustluma)<br/>           | 變更 RGB 值的亮度。 色調和飽和度不會受到影響。<br/> |
| [**ColorHLSToRGB**](/windows/desktop/api/Shlwapi/nf-shlwapi-colorhlstorgb)<br/>               | 將色彩從色調-亮度-飽和度 (HLS) 轉換為 RGB 格式。<br/>         |
| [**ColorRGBToHLS**](/windows/desktop/api/Shlwapi/nf-shlwapi-colorrgbtohls)<br/>               | 將色彩從 RGB 轉換為色調-亮度-飽和度 (HLS) 格式。<br/>         |
| [**SHCreateShellPalette**](/windows/desktop/api/Shlwapi/nf-shlwapi-shcreateshellpalette)<br/> | 建立指定之裝置內容的半色調調色板。<br/>               |
