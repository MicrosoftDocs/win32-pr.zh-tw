# <a name="didiskquotauserdisplayname-property"></a>DIDiskQuotaUser DisplayName 屬性

取得使用者的顯示名稱。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
DisplayName = DIDiskQuotaUser.DisplayName
```



## <a name="property-value"></a>屬性值

設定為使用者之顯示名稱的字串值。

## <a name="remarks"></a>備註

此屬性包含使用者的「易記名稱」。 不一定會定義其值。
