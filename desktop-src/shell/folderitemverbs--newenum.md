# <a name="folderitemverbs_newenum-method"></a>FolderItemVerbs。 \_NewEnum 方法

建立並傳回新的 [**FolderItemVerbs**](folderitemverbs.md) 物件，該物件為這個 FolderItemVerbs 物件的複本。

## <a name="syntax"></a>語法


```JScript
FolderItemVerbs._NewEnum()
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
