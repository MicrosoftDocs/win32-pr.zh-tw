# <a name="ishelllinkdual2-object"></a>IShellLinkDual2 物件

## <a name="members"></a>成員

**IShellLinkDual2** 物件具有下列類型的成員：

-   [屬性](#properties)

### <a name="properties"></a>屬性

**IShellLinkDual2** 物件具有這些屬性。



| 屬性                                            | 存取類型          | 描述                                   |
|:----------------------------------------------------|:---------------------|:----------------------------------------------|
| [**Target**](ishelllinkdual2-target.md)<br/> | 唯讀<br/> | 包含連結化物件的目標。<br/> |
