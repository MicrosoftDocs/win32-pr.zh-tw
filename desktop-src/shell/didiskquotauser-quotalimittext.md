# <a name="didiskquotauserquotalimittext-property"></a>DIDiskQuotaUser. QuotaLimitText 屬性

取得使用者目前的 [**配額限制**](diskquotacontrol-object.md) ，作為文字字串。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
QuotaLimitText = DIDiskQuotaUser.QuotaLimitText
```



## <a name="property-value"></a>屬性值

字串值，包含使用者目前的配額限制。
