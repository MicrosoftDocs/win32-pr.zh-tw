# <a name="sfvm_size-message"></a>SFVM \_ 大小訊息

通知回呼物件，資料夾檢視已調整大小。 由 [**IShellFolderViewCB：： MessageSFVCB**](/windows/win32/api/shlobj_core/nf-shlobj_core-ishellfolderviewcb-messagesfvcb)使用。


```C++
SFVM_SIZE
```



## <a name="parameters"></a>參數

此訊息沒有任何參數。
