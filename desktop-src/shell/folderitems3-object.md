# <a name="folderitems3-object"></a>FolderItems3 物件

擴充 [**FolderItems2**](folderitems2-object.md) 物件。 此物件支援其他方法和屬性。

## <a name="members"></a>成員

**FolderItems3** 物件具有下列類型的成員：

-   [方法](#methods)
-   [屬性](#properties)

### <a name="methods"></a>方法

**FolderItems3** 物件有這些方法。



| 方法                                | 描述                                                       |
|:--------------------------------------|:------------------------------------------------------------------|
| [**Filter**](folderitems3-filter.md) | 設定萬用字元篩選器以套用至傳回的專案。<br/> |



 

### <a name="properties"></a>屬性

**FolderItems3** 物件具有這些屬性。



| 屬性                                       | 存取類型          | 描述                                                       |
|:-----------------------------------------------|:---------------------|:------------------------------------------------------------------|
| [**動詞**](folderitems3-verbs.md)<br/> | 唯讀<br/> | 取得所有資料夾專案通用的動詞命令清單。<br/> |
