# <a name="folderitems2-object"></a>FolderItems2 物件

擴充 [**FolderItems**](folderitems.md) 物件。 它支援一個額外的方法。

## <a name="members"></a>成員

**FolderItems2** 物件具有下列類型的成員：

-   [方法](#methods)

### <a name="methods"></a>方法

**FolderItems2** 物件有這些方法。



| 方法                                            | 描述                                                                                                                                                                                                                                         |
|:--------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**InvokeVerbEx**](folderitems2-invokeverbex.md) | 在 [**FolderItem**](folderitem.md) 物件的集合上執行動詞。 這個方法是 [**InvokeVerb**](folderitem-invokeverb.md) 方法的擴充功能，可讓您透過一組旗標來進行作業的額外控制。<br/> |
