# <a name="windows-event-collector-enumerations"></a>Windows 事件收集器列舉

下表列出 Windows 事件收集器 SDK 的列舉。



| 列舉型別                                                                                               | 描述                                                                                                                             |
|-----------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**EC 訂用帳戶設定 \_ \_ \_ 模式**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_configuration_mode)                       | 指定變更訂用帳戶預設值的不同設定模式。                                            |
| [**EC \_ 訂用帳戶 \_ 內容 \_ 格式**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_content_format)                               | 指定在事件傳送到事件收集器電腦之前，要如何在傳送事件的電腦上呈現事件。 |
| [**EC \_ 訂用帳戶 \_ 認證 \_ 類型**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_credentials_type)                           | 指定與事件來源通訊時所要使用的認證類型。                                                         |
| [**EC \_ 訂閱 \_ 傳遞 \_ 模式**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_delivery_mode)                                 | 指定如何透過事件訂閱傳遞事件， (使用推播或提取模型) 。                                          |
| [**EC \_ 訂用帳戶 \_ 屬性 \_ 識別碼**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_property_id)                                     | 定義值，以識別用於訂用帳戶設定的事件訂閱屬性。                                           |
| [**EC 訂用帳戶執行時間狀態作用中 \_ \_ \_ \_ \_ 狀態**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_runtime_status_active_status) | 指定訂用帳戶的狀態或與訂用帳戶相關的事件來源。                                               |
| [**EC \_ 訂用帳戶 \_ 執行時間 \_ 狀態 \_ 資訊 \_ 識別碼**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_runtime_status_info_id)             | 指定值，這個值會識別事件來源或訂閱之執行時間狀態的屬性。                                |
| [**EC \_ 訂用帳戶 \_ 類型**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_type)                                                    | 指定要在來源起始或收集器起始的訂閱)  (使用的訂用帳戶類型。                                     |
| [**EC \_ 變異 \_ 類型**](/windows/desktop/api/Evcoll/ne-evcoll-ec_variant_type)                                                              | 定義值，這些值會指定 Windows 事件收集器函式中使用的資料類型。                                  |
