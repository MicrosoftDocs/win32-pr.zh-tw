# <a name="functions-windows-touch-gestures"></a>函數 (Windows Touch 手勢) 

本章節包含 Windows Touch 手勢的功能。

下列函式用於 Windows Touch 手勢。



| 函式                                                 | 描述                                                                                             |
|----------------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| [**CloseGestureInfoHandle**](/windows/desktop/api/winuser/nf-winuser-closegestureinfohandle) | 將與手動抓取之控制碼相關聯的資源關閉到手勢資訊結構。    |
| [**GetGestureExtraArgs**](/windows/desktop/api/winuser/nf-winuser-getgestureextraargs)       | 從 [**GESTUREINFO**](/windows/win32/api/winuser/ns-winuser-gestureinfo) 控制碼抓取手勢的其他相關資訊。    |
| [**GetGestureInfo**](/windows/desktop/api/winuser/nf-winuser-getgestureinfo)                 | 以手動方式抓取手勢資訊結構的控制碼。 |
| [**GetGestureConfig**](/windows/desktop/api/winuser/nf-winuser-getgestureconfig)             | 抓取視窗的手勢設定資訊。                                           |
| [**SetGestureConfig**](/windows/desktop/api/winuser/nf-winuser-setgestureconfig)             | 設定視窗的手勢設定。                                                            |
