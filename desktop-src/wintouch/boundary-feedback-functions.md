# <a name="functions-boundary-feedback"></a>函數 (界限意見反應) 

本節說明界限意見反應函式。



| 函式                                               | 描述                                                                                        |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------|
| [**BeginPanningFeedback**](/windows/win32/api/uxtheme/nf-uxtheme-beginpanningfeedback)     | 使用者會在使用界限意見反應之前叫用此函式。                               |
| [**EndPanningFeedback**](/windows/win32/api/uxtheme/nf-uxtheme-endpanningfeedback)     | 此函式會在邊界回饋完成時呼叫，並選擇性地觸發動畫。 |
| [**UpdatePanningFeedback**](/windows/win32/api/uxtheme/nf-uxtheme-updatepanningfeedback) | 在界限意見反應期間會呼叫此函式，並選擇性地觸發動畫。             |
