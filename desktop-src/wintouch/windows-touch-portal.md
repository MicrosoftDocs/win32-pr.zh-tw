# <a name="touch-input"></a>觸控輸入

## <a name="purpose"></a>目的

Windows 透過 Windows Touch 的端對端平臺和使用者介面架構所公開的互動模型，支援觸控輸入。

## <a name="developer-audience"></a>開發人員對象

Windows Touch 的功能是專為熟悉的 C/c + + 開發人員所設計，這些開發人員熟悉元件物件模型 (COM) 、UI 程式設計概念，以及一般觸控互動概念。

## <a name="run-time-requirements"></a>執行階段需求求

Windows 7
