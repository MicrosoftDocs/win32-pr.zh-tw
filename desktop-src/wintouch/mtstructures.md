# <a name="structures-windows-touch-gestures"></a>結構 (Windows Touch 手勢) 

本章節包含 Windows Touch 手勢的結構。

此區段中指定了下列結構。



| 結構                                          | Description                                                                              |
|----------------------------------------------------|------------------------------------------------------------------------------------------|
| [**GESTURECONFIG**](/windows/win32/api/winuser/ns-winuser-gestureconfig)             | 用來設定啟用或停用手勢訊息的設定。      |
| [**GESTUREINFO**](/windows/win32/api/winuser/ns-winuser-gestureinfo)                 | 包含筆勢的相關資訊。                                                    |
| [**GESTURENOTIFYSTRUCT**](/windows/win32/api/winuser/ns-winuser-gesturenotifystruct) | 包含有關 [**WM \_ GESTURENOTIFY**](wm-gesturenotify.md) 通知的資訊。 |
