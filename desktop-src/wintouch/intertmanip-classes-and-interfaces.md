# <a name="classes-and-interfaces"></a>類別和介面

本章節包含慣性和操作的類別和介面參考。

下列類別和介面可用於慣性和操作。



| 類別/介面                                          | Description                                                                  |
|----------------------------------------------------------|------------------------------------------------------------------------------|
| [**\_IManipulationEvents**](/windows/win32/api/manipulations/nn-manipulations-_imanipulationevents)    | 處理操作事件。                                                 |
| [**IInertiaProcessor**](/windows/desktop/api/manipulations/nn-manipulations-iinertiaprocessor)           | 處理 Windows Touch 之物件動作的相關計算。              |
| [**IManipulationProcessor**](/windows/desktop/api/manipulations/nn-manipulations-imanipulationprocessor) | 提供監視和回應 Windows Touch 輸入的功能。 |
