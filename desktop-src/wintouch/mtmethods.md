# <a name="methods-imanipulationprocessor"></a>方法 (IManipulationProcessor) 

本節包含 [**IManipulationProcessor**](/windows/desktop/api/manipulations/nn-manipulations-imanipulationprocessor) 介面的方法。

[**IManipulationProcessor**](/windows/desktop/api/manipulations/nn-manipulations-imanipulationprocessor)介面會公開下列方法。



| 方法                                                                      | 描述                                                                              |
|-----------------------------------------------------------------------------|------------------------------------------------------------------------------------------|
| [**CompleteManipulation**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-completemanipulation) | 當開發人員選擇結束操作時，就會呼叫這個方法。                |
| [**GetAngularVelocity**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-getangularvelocity)     | 計算目標物件正在移動的旋轉速度。                  |
| [**GetExpansionVelocity**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-getexpansionvelocity) | 計算目標物件正在擴充的速率。                              |
| [**GetVelocityX**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-getvelocityx)                 | 計算並傳回目標物件的水準速度。                    |
| [**GetVelocityY**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-getvelocityy)                 | 計算並傳回垂直速度。                                            |
| [**ProcessDown**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processdown)                   | 將資料摘要至與目標相關聯的操作處理器。                       |
| [**ProcessMove**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processmove)                   | 將資料摘要至與目標相關聯的操作處理器。                       |
| [**ProcessUp**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processup)                       | 將資料摘要至與目標相關聯的操作處理器。                       |
| [**ProcessDownWithTime**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processdownwithtime)   | 將資料（包括時間戳記）饋送至與目標相關聯的操作處理器。 |
| [**ProcessMoveWithTime**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processmovewithtime)   | 將資料（包括時間戳記）饋送至與目標相關聯的操作處理器。 |
| [**ProcessUpWithTime**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processupwithtime)       | 將資料（包括時間戳記）饋送至與目標相關聯的操作處理器。 |
