# <a name="messages-touch-input"></a> (觸控輸入的訊息) 

本節包含多點觸控輸入的訊息。

下列訊息用於多點觸控輸入。



| 訊息                           | 描述                                                                                                         |
|-----------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [**WM \_ 觸控**](wm-touchdown.md) | 指出一或多個觸控點（例如手指或畫筆）已觸及觸控式的數位化板表面。 |
