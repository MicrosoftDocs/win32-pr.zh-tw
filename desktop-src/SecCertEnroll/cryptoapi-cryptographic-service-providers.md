# <a name="cryptoapi-cryptographic-service-providers"></a>CryptoAPI 密碼編譯服務提供者

與密碼編譯 API ([*CryptoAPI*](/windows/desktop/SecGloss/c-gly)) 相關聯的提供者，在本檔中稱為加密服務提供者 (csp) 。 Csp 通常會執行密碼編譯演算法，並提供金鑰存放區。 另一方面，與 CNG 相關聯的提供者與金鑰儲存區中的演算法執行不同。 下列 Microsoft csp 隨 Windows Vista 和 Windows Server 2008 散發。

## <a name="microsoft-base-cryptographic-provider-v10"></a>Microsoft 基礎密碼編譯提供者 v1.0

會執行下列演算法來雜湊、簽署和加密內容。



| 名稱                                            | 使用          | 類型  | 金鑰大小 (預設/最小/最大)  |
|-------------------------------------------------|--------------|-------|----------------------------|
| 資料加密標準 (DES)                   | 加密   | 封鎖 | 56/56/56                   |
| 雜湊的訊息驗證總和檢查碼 (HMAC)    | 雜湊      | 任意   | 0/0/0                      |
|  (MAC) 的訊息驗證總和檢查碼           | 雜湊      | 任意   | 0/0/0                      |
| 訊息摘要 2 (MD2)                           | 雜湊      | 任意   | 128/128/128                |
| 訊息摘要 4 (MD4)                           | 雜湊      | 任意   | 128/128/128                |
| 訊息摘要 5 (MD5)                           | 雜湊      | 任意   | 128/128/128                |
| RSA Data Security 2 (RC2)                        | 加密   | 封鎖 | 40/40/56                   |
| RSA Data Security 4 (RC4)                        | 加密   | 封鎖 | 40/40/56                   |
| RSA 金鑰 Exchange                                | 金鑰交換 | RSA   | 512/384/1024               |
| RSA 簽章                                   | 簽署      | RSA   | 512/384/16384              |
|  (SHA1) 的安全雜湊演算法                    | 雜湊      | 任意   | 160/160/160                |
| 安全通訊端層 3 SHA 和 MD5 (SSL3 SHAMD5)  | 雜湊      | 任意   | 288/288/288                |
