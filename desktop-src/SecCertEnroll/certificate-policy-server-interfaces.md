# <a name="certificate-policy-server-interfaces"></a>憑證原則伺服器介面

下列介面可以用來以程式設計方式設定 (CEP) server 的憑證註冊原則。



| 介面                                                            | 描述                                                                                                   |
|----------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**IX509EnrollmentPolicyServer**](/windows/desktop/api/Certenroll/nn-certenroll-ix509enrollmentpolicyserver)   | 表示 CEP 伺服器。                                                                                      |
| [**IX509PolicyServerListManager**](/windows/desktop/api/CertEnroll/nn-certenroll-ix509policyserverlistmanager) | 管理 [**IX509PolicyServerUrl**](/windows/desktop/api/Certenroll/nn-certenroll-ix509policyserverurl) 物件的集合。                         |
| [**IX509PolicyServerUrl**](/windows/desktop/api/Certenroll/nn-certenroll-ix509policyserverurl)                 | 指定或抓取與 CEP 伺服器相關聯的屬性值，並更新相關聯的登錄值。 |
