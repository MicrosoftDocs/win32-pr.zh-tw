# <a name="cryptographic-provider-interfaces"></a>密碼編譯提供者介面

下列介面可以用來取得密碼編譯提供者的相關資訊。



| 介面                                    | 描述                                                                  |
|----------------------------------------------|------------------------------------------------------------------------------|
| [**ICspAlgorithm**](/windows/desktop/api/CertEnroll/nn-certenroll-icspalgorithm)       | 代表密碼編譯提供者所執行的演算法。             |
| [**ICspAlgorithms**](/windows/desktop/api/CertEnroll/nn-certenroll-icspalgorithms)     | 管理 [**ICspAlgorithm**](/windows/desktop/api/CertEnroll/nn-certenroll-ix509namevaluepair) 物件的集合。 |
| [**ICspInformation**](/windows/desktop/api/CertEnroll/nn-certenroll-icspinformation)   | 提供有關密碼編譯提供者之一般資訊的存取權。       |
| [**ICspInformations**](/windows/desktop/api/CertEnroll/nn-certenroll-icspinformations) | 管理 [**ICspInformation**](/windows/desktop/api/CertEnroll/nn-certenroll-icspinformation) 物件的集合。  |
| [**ICspStatus**](/windows/desktop/api/CertEnroll/nn-certenroll-icspstatus)             | 包含密碼編譯提供者/演算法配對的相關資訊。          |
| [**ICspStatuses**](/windows/desktop/api/CertEnroll/nn-certenroll-icspstatuses)         | 管理 [**ICspStatus**](/windows/desktop/api/CertEnroll/nn-certenroll-icspstatus) 物件的集合。            |
