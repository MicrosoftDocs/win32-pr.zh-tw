# <a name="boolean-certificate-enrollment-api"></a> (憑證註冊 API) 的布林值

布林值可以是 **TRUE** 或 **FALSE**。 **FALSE** 會編碼為 TLV 三個，其中 **值** 欄位為零 (0) 。 **TRUE** 會編碼為非零值。 **布林** 資料類型的 **標記** 為0x01。
