# <a name="trust-models"></a>信任模型

由於執行公開金鑰基礎結構的組織數量 (PKI) 會增加，而且這些組織中的使用者和電腦數目也會增加，與認證身分識別和建立信任相關聯的物流問題也會增加。 有兩個主要信任模型（憑證階層和交叉認證）已發展成有助於解決這些問題。 這兩種模型內的固有概念是憑證鏈的概念。 如需詳細資訊，請參閱下列主題：

-   [憑證階層](about-certificate-hierarchy.md)
-   [交叉認證](about-cross-certification.md)
-   [憑證鏈](about-certificate-chain.md)
