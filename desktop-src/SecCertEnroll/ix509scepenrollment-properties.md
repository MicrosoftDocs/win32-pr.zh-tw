# <a name="ix509scepenrollment-properties"></a>IX509SCEPEnrollment 屬性

[**IX509SCEPEnrollment**](/windows/desktop/api/Certenroll/nn-certenroll-ix509scepenrollment)介面會公開下列屬性。

## <a name="in-this-section"></a>本節內容



| 主題                                                                                              | 描述                                                                                                                                            |
|----------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Certificate 屬性**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_certificate)<br/>                         | 取得要求的憑證。<br/>                                                                                                       |
| [**Enrollment.certificatefriendlyname 屬性**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_certificatefriendlyname)<br/> | 取得或設定憑證的易記名稱。<br/>                                                                                         |
| [**FailInfo 屬性**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_failinfo)<br/>                               | 取得 [**ProcessResponseMessage**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-processresponsemessage) 方法偵測到失敗環境時的資訊。<br/> |
| [**OldCertificate 屬性**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_oldcertificate)<br/>                   | 取得或設定要求要取代的舊憑證。<br/>                                                                      |
| [**要求屬性**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_request)<br/>                                 | 取得內部 PKCS10 要求。<br/>                                                                                                              |
| [**ServerCapabilities 屬性**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-put_servercapabilities)<br/>           | 為要求設定慣用的雜湊和加密演算法。<br/>                                                                          |
| [**SignerCertificate 屬性**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_signercertificate)<br/>             | 取得或設定要求的簽署者憑證。<br/>                                                                                        |
| [**無訊息屬性**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-put_silent)<br/>                                   | 取得或設定是否要在要求期間允許 UI。<br/>                                                                                        |
| [**Status 屬性**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_status)<br/>                                   | 取得要求的狀態。<br/>                                                                                                             |
| [**TransactionId 屬性**](/windows/desktop/api/Certenroll/nf-certenroll-ix509scepenrollment-get_transactionid)<br/>                     | 取得或設定要求的交易識別碼。<br/>                                                                                            |
