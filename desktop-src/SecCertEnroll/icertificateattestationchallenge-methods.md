# <a name="icertificateattestationchallenge-methods"></a>ICertificateAttestationChallenge 方法

[**ICertificateAttestationChallenge**](/windows/desktop/api/Certenroll/nn-certenroll-icertificateattestationchallenge)介面支援下列方法。

## <a name="in-this-section"></a>本節內容



| 主題                                                                                           | 描述                                                                                                                                                                                                                                                    |
|-------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DecryptChallenge 方法**](/windows/desktop/api/Certenroll/nf-certenroll-icertificateattestationchallenge-decryptchallenge)<br/> | 將 [憑證管理](/windows/desktop/SecGloss/c-gly) 的挑戰從 CMS (CMC) 回應中解密，並建立要傳送給 CA 的重新加密回應。<br/> |
| [**Initialize 方法**](/windows/desktop/api/Certenroll/nf-certenroll-icertificateattestationchallenge-initialize)<br/>             | 使用透過 CMS 的完整 [憑證管理](/windows/desktop/SecGloss/c-gly) 來初始化 (CMC 從 CA 傳回的) 回應。<br/>                                        |
