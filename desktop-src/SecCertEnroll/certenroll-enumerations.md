# <a name="certenroll-enumerations"></a>CertEnroll 列舉

憑證註冊 API 支援下列列舉。

| 列舉型別                                                                              | 描述                                                                                                                                                                                                                                                                                                              |
|------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AlgorithmFlags**](/windows/desktop/api/CertEnroll/ne-certenroll-algorithmflags)                                            | 包含可用來精簡密碼編譯演算法搜尋的旗標。                                                                                                                                                                                                                                      |
| [**AlgorithmOperationFlags**](/windows/desktop/api/CertEnroll/ne-certenroll-algorithmoperationflags)                          | 指定演算法可執行檔作業。 這是用來根據這些功能來取得 CSP 演算法或 CSP 狀態資訊的作業功能。                                                                                                                                  |
| [**AlgorithmType**](/windows/desktop/api/CertEnroll/ne-certenroll-algorithmtype)                                              | 指定密碼編譯演算法的用途。                                                                                                                                                                                                                                                             |
| [**AlternativeNameType**](/windows/desktop/api/CertEnroll/ne-certenroll-alternativenametype)                                  | 指定建立 **AlternativeName** 延伸模組時可以指定的替代名稱類型。                                                                                                                                                                                                               |
| [**CERTENROLL \_ OBJECTID**](/windows/desktop/api/CertEnroll/ne-certenroll-certenroll_objectid)                                 | 包含預先定義的 [*物件識別碼*](/windows/desktop/SecGloss/o-gly) (oid) 支援憑證註冊 API。                                                                                                                                          |
| [**CERTENROLL \_ PROPERTYID**](/windows/desktop/api/CertEnroll/ne-certenroll-certenroll_propertyid)                             | 針對憑證註冊 API 所支援的外部屬性，包含預先定義的 Oid。                                                                                                                                                                                                                            |
| [**CommitTemplateFlags**](/windows/desktop/api/Certenroll/ne-certenroll-committemplateflags)                                       | 指定儲存和刪除範本的選項。                                                                                                                                                                                                                                                                     |
| [**Edi.encodingtype**](/windows/desktop/api/CertEnroll/ne-certenroll-encodingtype)                                                | 指定套用至位元組陣列以供顯示之用的編碼類型。                                                                                                                                                                                                                                             |
| [**EnrollmentCAProperty**](/windows/desktop/api/Certenroll/ne-certenroll-enrollmentcaproperty)                                     | 指定憑證授權單位單位屬性值。                                                                                                                                                                                                                                                                       |
| [**EnrollmentDisplayStatus**](/windows/desktop/api/CertEnroll/ne-certenroll-enrollmentdisplaystatus)                          | 指定是否要在使用者介面中顯示註冊狀態資訊。                                                                                                                                                                                                                                          |
| [**EnrollmentEnrollStatus**](/windows/desktop/api/CertEnroll/ne-certenroll-enrollmentenrollstatus)                            | 指定憑證要求的註冊狀態。                                                                                                                                                                                                                                                                |
| [**EnrollmentPolicyFlags**](/windows/desktop/api/Certenroll/ne-certenroll-enrollmentpolicyflags)                                   | 指定群組原則旗標。                                                                                                                                                                                                                                                                                            |
| [**EnrollmentPolicyServerPropertyFlags**](/windows/desktop/api/Certenroll/ne-certenroll-enrollmentpolicyserverpropertyflags)       | 指定預設原則伺服器。                                                                                                                                                                                                                                                                                     |
| [**EnrollmentSelectionStatus**](/windows/desktop/api/CertEnroll/ne-certenroll-enrollmentselectionstatus)                      | 指定是否會在註冊程式期間監視物件的註冊狀態。                                                                                                                                                                                                                    |
| [**EnrollmentTemplateProperty**](/windows/desktop/api/Certenroll/ne-certenroll-enrollmenttemplateproperty)                         | 包含指定範本的屬性值。                                                                                                                                                                                                                                                                           |
| [**InnerRequestLevel**](/windows/desktop/api/CertEnroll/ne-certenroll-innerrequestlevel)                                      | 指定 PKCS \# 7 或 CMC 要求內憑證要求的內含專案層級。                                                                                                                                                                                                                               |
| [**InstallResponseRestrictionFlags**](/windows/desktop/api/CertEnroll/ne-certenroll-installresponserestrictionflags)          | 包含旗標，這些旗標會識別在憑證鏈的本機安裝上所放置的限制。                                                                                                                                                                                                                   |
| [**KeyIdentifierHashAlgorithm**](/windows/desktop/api/CertEnroll/ne-certenroll-keyidentifierhashalgorithm)                    | 指定在憑證要求中用來雜湊公開金鑰的演算法。                                                                                                                                                                                                                                            |
| [**ObjectIdGroupId**](/windows/desktop/api/CertEnroll/ne-certenroll-objectidgroupid)                                          | 指定 OID 所屬的分類或群組。                                                                                                                                                                                                                                                                 |
| [**ObjectIdPublicKeyFlags**](/windows/desktop/api/CertEnroll/ne-certenroll-objectidpublickeyflags)                            | 指定是否使用公開金鑰演算法進行簽署或加密。                                                                                                                                                                                                                                          |
| [**PFXExportOptions**](/windows/desktop/api/CertEnroll/ne-certenroll-pfxexportoptions)                                        | 指定在建立個人資訊交換 (PFX) 訊息時，所包含的憑證鏈數量。                                                                                                                                                                                                       |
| [**Pkcs10AllowedSignatureTypes**](/windows/desktop/api/CertEnroll/ne-certenroll-pkcs10allowedsignaturetypes)                  | 指定簽署憑證要求時允許的簽章類型。                                                                                                                                                                                                                                            |
| [**PolicyQualifierType**](/windows/desktop/api/CertEnroll/ne-certenroll-policyqualifiertype)                                  | 指定套用至憑證原則的辨識符號類型。                                                                                                                                                                                                                                                         |
| [**PolicyServerUrlFlags**](/windows/desktop/api/Certenroll/ne-certenroll-policyserverurlflags)                                     | 包含 (CEP) 伺服器旗標的憑證註冊原則。                                                                                                                                                                                                                                                               |
| [**PolicyServerUrlPropertyID**](/windows/desktop/api/Certenroll/ne-certenroll-policyserverurlpropertyid)                           | 包含值，這些值會指定 [**GetStringProperty**](/windows/desktop/api/Certenroll/nf-certenroll-ix509policyserverurl-getstringproperty)方法所要傳回的屬性值類型，或由 [**IX509PolicyServerUrl**](/windows/desktop/api/Certenroll/nn-certenroll-ix509policyserverurl)介面上的 [**SetStringProperty**](/windows/desktop/api/Certenroll/nf-certenroll-ix509policyserverurl-setstringproperty)方法設定。 |
| [**RequestClientInfoClientId**](/windows/desktop/api/CertEnroll/ne-certenroll-requestclientinfoclientid)                      | 指定建立憑證要求的應用程式類型。                                                                                                                                                                                                                                                    |
| [**WebEnrollmentFlags**](/windows/desktop/api/Certenroll/ne-certenroll-webenrollmentflags)                                         | 指定網頁註冊行為。                                                                                                                                                                                                                                                                                       |
| [**WebSecurityLevel**](/windows/desktop/api/CertEnroll/ne-certenroll-websecuritylevel)                                        | 指定 web 的方法或屬性是否安全，以進行腳本處理。                                                                                                                                                                                                                                                |
| [**X500NameFlags**](/windows/desktop/api/CertEnroll/ne-certenroll-x500nameflags)                                              | 指定辨別名稱或相對辨別名稱的顯示和編碼特性。                                                                                                                                                                                                               |
| [**X509CertificateEnrollmentCoNtext**](/windows/desktop/api/CertEnroll/ne-certenroll-x509certificateenrollmentcontext)        | 指定要作為憑證用途之終端實體的本質。                                                                                                                                                                                                                                            |
| [**X509CertificateTemplateEnrollmentFlag**](/windows/desktop/api/CertEnroll/ne-certenroll-x509certificatetemplateenrollmentflag)   | 包含值，這些值會在註冊期間指定伺服器和用戶端動作。                                                                                                                                                                                                                                                |
| [**X509CertificateTemplateGeneralFlag**](/windows/desktop/api/CertEnroll/ne-certenroll-x509certificatetemplategeneralflag)         | 包含範本和相關聯憑證的使用和修改資訊。                                                                                                                                                                                                                                   |
| [**X509CertificateTemplatePrivateKeyFlag**](/windows/desktop/api/CertEnroll/ne-certenroll-x509certificatetemplateprivatekeyflag)   | 包含值，這些值會指定與私密金鑰相關的用戶端動作。                                                                                                                                                                                                                                                     |
| [**X509CertificateTemplateSubjectNameFlag**](/windows/desktop/api/CertEnroll/ne-certenroll-x509certificatetemplatesubjectnameflag) | 包含值，這些值會指定關於主體名稱的伺服器和用戶端動作。                                                                                                                                                                                                                                         |
| [**X509EnrollmentPolicyExportFlags**](/windows/desktop/api/Certenroll/ne-certenroll-x509enrollmentpolicyexportflags)               | 指定要從原則伺服器匯出的專案。                                                                                                                                                                                                                                                                   |
| [**X509EnrollmentPolicyLoadOption**](/windows/desktop/api/Certenroll/ne-certenroll-x509enrollmentpolicyloadoption)                 | 指定如何從原則伺服器取得原則。                                                                                                                                                                                                                                                                 |
| [**X509KeySpec**](/windows/desktop/api/CertEnroll/ne-certenroll-x509keyspec)                                                  | 指定用於舊版 CSP 的金鑰用途。                                                                                                                                                                                                                                                                    |
| [**X509KeyUsageFlags**](/windows/desktop/api/CertEnroll/ne-certenroll-x509keyusageflags)                                      | 指定憑證中包含之金鑰的用途。                                                                                                                                                                                                                                                               |
| [**X509PrivateKeyExportFlags**](/windows/desktop/api/CertEnroll/ne-certenroll-x509privatekeyexportflags)                           | 指定私密金鑰的匯出原則。                                                                                                                                                                                                                                                                           |
| [**X509PrivateKeyProtection**](/windows/desktop/api/CertEnroll/ne-certenroll-x509privatekeyprotection)                             | 指定 KSP 或 CSP 所支援的私密金鑰保護層級。                                                                                                                                                                                                                                                 |
| [**X509PrivateKeyUsageFlags**](/windows/desktop/api/CertEnroll/ne-certenroll-x509privatekeyusageflags)                             | 指定私用金鑰的允許用途。                                                                                                                                                                                                                                                                           |
| [**X509PrivateKeyVerify**](/windows/desktop/api/CertEnroll/ne-certenroll-x509privatekeyverify)                                     | 指定是否在私密金鑰驗證期間顯示使用者介面，以及如果 CSP 或 KSP 代表智慧卡，是否可以繼續驗證。                                                                                                                                                           |
| [**X509ProviderType**](/windows/desktop/api/CertEnroll/ne-certenroll-x509providertype)                                        | 指定密碼編譯服務提供者或金鑰服務提供者的類型。                                                                                                                                                                                                                                          |
| [**X509RequestInheritOptions**](/windows/desktop/api/CertEnroll/ne-certenroll-x509requestinheritoptions)                      | 指定從現有的憑證建立新的要求時，如何繼承索引鍵、延伸模組值和外部屬性。                                                                                                                                                                                  |
| [**X509RequestType**](/windows/desktop/api/CertEnroll/ne-certenroll-x509requesttype)                                          | 指定憑證要求類型。                                                                                                                                                                                                                                                                                  |
| [**X509SCEPDisposition**](/windows/desktop/api/certpol/ne-certpol-x509scepdisposition)                                       | 描述處理回應訊息之要求的產生處置。                                                                                                                                                                                                                                          |
| [**X509SCEPFailInfo**](/windows/desktop/api/certpol/ne-certpol-x509scepfailinfo)                                             | 描述 SCEP 憑證註冊失敗的本質。                                                                                                                                                                                                                                                          |
