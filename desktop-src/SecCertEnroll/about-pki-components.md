# <a name="pki-elements"></a>PKI 元素

 (PKI) 的一般 Microsoft 公開金鑰基礎結構包含下列元素。

| 元素                                                                     | 描述                                                                                                                                                                                              |
|-----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [憑證授權單位單位](about-certification-authorities.md)<br/> | 提供可驗證網路中個人、電腦和其他實體之身分識別的服務。 這同時包含根憑證授權單位和次級授權單位。<br/> |
| [憑證目錄](about-certificate-directory.md)<br/>         | 儲存憑證要求以及發行和撤銷憑證和憑證要求。<br/>                                                                                                      |
| [金鑰修復伺服器](about-key-recovery-server.md)<br/>             | 在憑證資料庫中儲存加密的私密金鑰，以在遺失之後復原。<br/>                                                                                                             |
