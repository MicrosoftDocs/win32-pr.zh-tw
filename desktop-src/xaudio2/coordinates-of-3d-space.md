# <a name="coordinates-of-3d-space"></a>3D 空間的座標

在3D 空間中，音效來源和接聽程式的位置、速度和方向會以笛卡兒座標表示，也就是三個軸上的值： X 軸、y 軸和 Z 軸。

軸是相對於應用程式所建立的觀點。 X 軸上的值從左至右、從上到右、從上到上、從最接近到 Z 軸向上增加。

**X3DAUDIO \_ 向量** 結構包含的值會描述三個軸上的位置、速度或方向。

傳統上，向量會以括弧括住的三個值來表示，並以逗號分隔，順序 (x，y，z) 。

針對 [位置]，這些值會在使用者定義的世界單位中。

針對速度，向量會描述每個軸在每秒的世界單位中移動的速率。

針對方向，這些值會以任意單位來表示，而它們彼此相對。 例如，如果3D 世界的基底視圖朝水準方向，而且接聽程式的方向是 (-1、0、1) ，則接聽程式會面向西北部。 因為向量中的值不是絕對單位，所以向量同樣可以表示為 (-5、0、5) 或 (-0.25、0、0.25) 。

3D 向量的運作方式很類似2D 向量，但在上下方向有額外的軸。 您可以藉由在圖形紙上繪製向量，查看2D 空間中向量的運作方式。 讓值從底部增加到紙張頂端，並從左至右。 從 (0、0) 到 (1、1) 從 (0、0) 到 (5，5) 所繪製的方向相同。 但是，第二行表示較遠的距離或速度。
