## <a name="remarks"></a>備註

**XAUDIO2 \_處理器** 的類型定義為32位值。

<div class="alert"><b>注意</b>  如果在呼叫 <a href="/windows/desktop/api/xaudio2/nf-xaudio2-xaudio2create">XAudio2Create</a>時設定了多個處理器位，系統會為每個處理器建立個別的背景工作執行緒。</div>



```
typedef UINT32 XAUDIO2_PROCESSOR;
```
