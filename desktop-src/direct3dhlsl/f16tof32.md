# <a name="f16tof32-function"></a>f16tof32 函式

將儲存在 uint 低半部的 float16 轉換為浮點數。

## <a name="syntax"></a>語法

``` syntax
float f16tof32(
  in uint value
);
```

## <a name="parameters"></a>參數

<dl> <dt>

*值* \[在\]
</dt> <dd>

類型： **uint**

輸入值。

</dd> </dl>

## <a name="return-value"></a>傳回值

類型： **float**

轉換的值。

## <a name="remarks"></a>備註

您也可以使用下列多載版本：

``` syntax
float2 f16tof32(uint2 value);
float3 f16tof32(uint3 value);
float4 f16tof32(uint4 value);
```

### <a name="minimum-shader-model"></a>最小著色器模型

下列著色器模型支援此函數。



| 著色器模型                                                        | 支援 |
|---------------------------------------------------------------------|-----------|
| [著色器模型 4](dx-graphics-hlsl-sm4.md) 和更高的著色器模型 | 是       |



 

雖然此函式適用于 [著色器模型 4](dx-graphics-hlsl-sm4.md) 和更新版本的著色器模型中，因為它是在4.0 和4.1 中進行模擬，所以在這些著色器模型上的 perfomant 比在 [著色器模型 5](d3d11-graphics-reference-sm5.md)上的還低。

下列著色器類型支援此函數：



| 頂點 | 船體 | 網域 | 幾何形狀 | 像素 | 計算 |
|--------|------|--------|----------|-------|---------|
| x      | x    | x      | x        | x     | x       |
