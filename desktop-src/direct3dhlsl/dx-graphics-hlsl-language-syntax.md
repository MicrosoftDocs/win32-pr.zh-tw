# <a name="language-syntax"></a>語言語法

HLSL 著色器是由變數和函式所組成，而函式是由語句所組成。 語言語法記載如何定義和宣告變數、加入流程式控制制，讓著色器可以根據變數來進行執行時間決策，以及撰寫自訂函數。
