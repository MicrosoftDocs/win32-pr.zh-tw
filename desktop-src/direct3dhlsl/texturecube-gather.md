# <a name="texturecubegather-methods"></a>TextureCube：：收集方法

傳回要在雙線性篩選作業中使用的四個材質值。

如需描述基礎 DXBC 指示的詳細資訊，請參閱 [gather4](./gather4--sm5---asm-.md) 上的檔。

### <a name="overload-list"></a>多載清單



| 方法                                                     | 描述                                                                                                              |
|:-----------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------|
| [**收集 (S，float)**](dx-graphics-hlsl-to-gather.md)      | 將材質取樣，並傳回四個樣本 (紅色元件僅) 用於雙線性插補。<br/> |
| [**收集 (S、float、uint)**](tcube-gather-s-float-uint-.md) | 取樣材質，並傳回所有四個元件以及作業的狀態。<br/>                      |
