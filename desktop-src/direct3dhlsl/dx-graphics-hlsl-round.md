# <a name="round"></a>round

將指定的值四捨五入到最接近的整數。 中間的案例會四捨五入至最接近的偶數。



| *ret* round (*x*)  |
|------------------|



 

## <a name="parameters"></a>參數



| Item                                                   | 描述                            |
|--------------------------------------------------------|----------------------------------------|
| <span id="x"></span><span id="X"></span>*X*<br/> | \[在 \] 指定的值中。<br/> |



 

## <a name="return-value"></a>傳回值

*X* 參數，四捨五入為浮點數類型內最接近的整數。
