# <a name="texture2darraygather-methods"></a>Texture2DArray：：收集方法

傳回 [**Texture2DArray**](sm5-object-texture2darray.md) 的四個材質值，這些值會在雙線性篩選作業中使用。

如需描述基礎 DXBC 指示的詳細資訊，請參閱 [gather4](./gather4--sm5---asm-.md) 上的檔。

### <a name="overload-list"></a>多載清單



| 方法                                                                | 描述                                                                                                               |
|:----------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------|
| [**收集 (S、float、int)**](sm5-object-texture2darray-gather.md)        | 傳回要在雙線性篩選作業中使用的四個材質值。<br/>                                 |
| [**收集 (S、float、int、uint)**](t2darray-gather-s-float-int-uint-.md)  | 傳回四個材質值，這些值會在雙線性篩選作業中使用，以及磚對應狀態。<br/> |
