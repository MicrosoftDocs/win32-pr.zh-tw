# <a name="store-function"></a>Store 函數

設定一個值。

## <a name="syntax"></a>語法

``` syntax
void Store(
  in uint address,
  in uint value
);
```

## <a name="parameters"></a>參數

<dl> <dt>

*位址* \[在\]
</dt> <dd>

類型： **uint**

輸入位址（以位元組為單位），必須是4的倍數。

</dd> <dt>

*值* \[在\]
</dt> <dd>

類型： **uint**

一個輸入值。

</dd> </dl>

## <a name="return-value"></a>傳回值

此函式不會傳回值。
