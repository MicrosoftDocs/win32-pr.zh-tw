## <a name="declaration-examples"></a><span data-ttu-id="a9b20-135">宣告範例</span><span class="sxs-lookup"><span data-stu-id="a9b20-135">Declaration Examples</span></span>

ps_3_0

; Declaring inputs
dcl_normal      v0.xyz

...
