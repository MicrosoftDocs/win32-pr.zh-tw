# <a name="nop---ps"></a>nop-ps

未執行任何操作。

## <a name="syntax"></a>Syntax



| nop |
|-----|



 

## <a name="remarks"></a>備註



| 圖元著色器版本 | 1\_1 | 1\_2 | 1 \_ 3 | 1\_4 | 2 \_ 0 | 2 \_ x | 2個 \_ sw | 3 \_ 0 | 3個 \_ sw |
|-----------------------|------|------|------|------|------|------|-------|------|-------|
| nop                   | x    | x    | x    | x    | x    | x    | x     | x    | x     |



 

此指令會執行無作業，或不執行任何操作。 呼叫的語法如下所示：


```
nop
```
