# <a name="break---ps"></a>中斷-ps

## <a name="syntax"></a>Syntax



| break |
|-------|



 

## <a name="remarks"></a>備註

| 圖元著色器版本 | 1\_1 | 1\_2 | 1 \_ 3 | 1\_4 | 2 \_ 0 | 2 \_ x | 2個 \_ sw | 3 \_ 0 | 3個 \_ sw |

| break                 |      |      |      |      |      | x    | x     | x    | x     |



 



|   需求                       | 值           |
|--------------------------|------------|
