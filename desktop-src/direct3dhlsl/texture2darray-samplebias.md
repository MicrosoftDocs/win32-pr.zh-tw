# <a name="texture2darraysamplebias-methods"></a>Texture2DArray：： SampleBias 方法

將偏差值套用至 mipmap 層級之後，將 [**Texture2DArray**](sm5-object-texture2darray.md)取樣。

### <a name="overload-list"></a>多載清單



| 方法                                                                                                | 描述                                                                                                                                                                                  |
|:------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleBias (S、float、float、int)**](dx-graphics-hlsl-to-samplebias.md)                               | 將偏差值套用至 mipmap 層級之後，將材質取樣。<br/>                                                                                                             |
| [**SampleBias (S、float、float、int、float)**](t2darray-samplebias-s-float-float-int-float-.md)           | 將偏差值套用至 mipmap 層級之後，將材質取樣，並使用選擇性的值來壓 (」 LOD) 值到的範例詳細層級。<br/>                                     |
| [**SampleBias (S、float、float、int、float、uint)**](t2darray-samplebias-s-float-float-int-float-uint-.md) | 將偏差值套用至 mipmap 層級之後，將材質取樣，並使用選擇性的值來壓 (」 LOD) 值到的範例詳細層級。 傳回操作的相關狀態。<br/> |
