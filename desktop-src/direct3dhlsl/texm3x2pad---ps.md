# <a name="texm3x2pad---ps"></a>texm3x2pad-ps

執行兩個數據列矩陣相乘的第一個資料列乘法。 此指令必須與 [texm3x2tex-ps](texm3x2tex---ps.md) 或 [texm3x2depth （ps](texm3x2depth---ps.md)）結合。 如需使用 texmpad 的詳細資訊，請參閱下列其中一項指示。

## <a name="syntax"></a>Syntax



| tex3x2pad dst、src |
|--------------------|



 

其中

-   dst 是目的地註冊。
-   src 是來源暫存器。

## <a name="remarks"></a>備註



| 圖元著色器版本 | 1\_1 | 1\_2 | 1 \_ 3 | 1\_4 | 2 \_ 0 | 2 \_ x | 2個 \_ sw | 3 \_ 0 | 3個 \_ sw |
|-----------------------|------|------|------|------|------|------|-------|------|-------|
| texm3x2pad            | x    | x    | x    |      |      |      |       |      |       |



 

此指令本身不能使用。
