# <a name="shader-model-51"></a>著色器模型5。1

本章節包含 HLSL 著色器模型5.1 的參考頁面（使用 D3D12 引進）。

著色器模型5.1 的功能與 [著色器模型 5](d3d11-graphics-reference-sm5.md)很類似，主要變更是在資源選取專案中提供更大的彈性，方法是允許從著色器內編制描述元陣列的索引。



| 功能                   | 功能                                                                |
|---------------------------|---------------------------------------------------------------------------|
| 指令集           | [**HLSL 內建函式**](dx-graphics-hlsl-intrinsic-functions.md)  |
| 頂點著色器最大值         | 沒有限制                                                            |
| 圖元著色器最大值          | 沒有限制                                                            |
| 新增著色器設定檔 | cs \_ 5 \_ 1、ds \_ 5 \_ 1、gs \_ 5 \_ 1、hs \_ 5 \_ 1、ps \_ 5 \_ 1、vs \_ 5 1、 \_ rootsig \_ 1 \_ 0 |
