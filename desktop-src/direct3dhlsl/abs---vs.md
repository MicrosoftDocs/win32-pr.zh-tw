# <a name="abs---vs"></a>abs-vs

計算絕對值。

## <a name="syntax"></a>Syntax

**abs dst、src**

其中

- dst 是目的地註冊。
- src 是來源暫存器。

## <a name="remarks"></a>備註



| 頂點著色器版本 | 1\_1 | 2 \_ 0 | 2 \_ x | 2個 \_ sw | 3 \_ 0 | 3個 \_ sw |
|------------------------|------|------|------|-------|------|-------|
| abs                    |      | x    | x    | x     | x    | x     |



 

此指示的運作方式如下所示。


```
dest.x = abs(src.x)
dest.y = abs(src.y)
dest.z = abs(src.z)
dest.w = abs(src.w)
```
