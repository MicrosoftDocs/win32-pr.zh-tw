# <a name="variables"></a>變數

HLSL 變數類似于 C 程式設計語言中所定義的變數。 類似于 C，變數有一些命名限制，具有相依于其宣告位置的範圍屬性，而且可以附加使用者中繼資料。 如同 C，有數種標準資料類型。 不同于 C，HLSL 也會定義其他資料類型，以協助將使用矩陣數學的4個元件向量的效能最大化，以在3D 圖形資料上運作。
