# <a name="buffer"></a>Buffer

存在於著色器模型4加上資源變數和緩衝區資訊中的緩衝區類型。



| 方法                                                   | 描述                         |
|----------------------------------------------------------|-------------------------------------|
| [**GetDimensions**](sm5-object-buffer-getdimensions.md) | 取得資源維度。       |
| [**載入**](buffer-load.md)                              | 讀取緩衝區資料。                  |
| [**運算子\[\]**](sm5-object-buffer-operatorindex.md)  | 取得唯讀的資源變數。 |



 

## <a name="minimum-shader-model"></a>最小著色器模型

下列著色器模型支援此物件。



| 著色器模型                                                                | 支援 |
|-----------------------------------------------------------------------------|-----------|
| [著色器模型 5](d3d11-graphics-reference-sm5.md) 和更新版本的著色器模型 | 是       |



 

下列著色器類型支援此物件：



| 頂點 | 船體 | 網域 | 幾何形狀 | 像素 | 計算 |
|--------|------|--------|----------|-------|---------|
| x      | x    | x      | x        | x     | x       |
