# <a name="discard-statement"></a>捨棄語句

請勿輸出目前圖元的結果。

丟掉



 

## <a name="parameters"></a>參數

None

## <a name="remarks"></a>備註

這個語句只能從圖元著色器呼叫;幾何著色器或頂點著色器中不支援此功能。
