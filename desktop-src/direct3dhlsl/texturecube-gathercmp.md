# <a name="texturecubegathercmp-methods"></a>TextureCube：： GatherCmp 方法

針對要在雙線性篩選作業中使用之 [**TextureCube**](texturecube.md) 的四個材質值，會傳回其與比較值的比較。

如需描述基礎 DXBC 指令的詳細資訊，請參閱 [gather4_c](./gather4-c--sm5---asm-.md) 的相關檔。

|:-----------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------|
| [**GatherCmp (S、float、float、uint)**](tcube-gathercmp-s-float-float-uint-.md) | 範例並比較材質，並傳回所有四個元件以及作業的狀態。<br/> |
