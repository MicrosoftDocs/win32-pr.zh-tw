# <a name="texturecubearraygather-methods"></a>TextureCubeArray：：收集方法

傳回四個材質值的紅色元件，這些值會在雙線性篩選作業中使用。

如需描述基礎 DXBC 指示的詳細資訊，請參閱 [gather4](./gather4--sm5---asm-.md) 上的檔。

### <a name="overload-list"></a>多載清單



| 方法                                                          | 描述                                                                                                                                |
|:----------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------|
| [**收集 (S，float)**](dx-graphics-hlsl-to-gather.md)           | 傳回四個材質值的紅色元件， (僅限紅色元件) 將用於雙線性篩選作業。 <br/>           |
| [**收集 (S、float、uint)**](tcubearray-gather-s-float-uint-.md) | 傳回四個材質值的紅色元件，這些值會在雙線性篩選作業中使用，以及磚對應狀態。<br/> |
