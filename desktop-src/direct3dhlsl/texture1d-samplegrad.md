# <a name="texture1dsamplegrad-methods"></a>Texture1D：： SampleGrad 方法

使用漸層來取樣 [**Texture1D**](sm5-object-texture1d.md) ，以影響範例位置的計算方式。

### <a name="overload-list"></a>多載清單



| 方法                                                                                                       | 描述                                                                                                                                                                                                        |
|:-------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleGrad (S、float、float、float、int)**](dx-graphics-hlsl-to-samplegrad.md)                                | 使用漸層來取樣材質，以影響範例位置的計算方式。<br/>                                                                                                             |
| [**SampleGrad (S、float、float、float、int、float)**](t1d-samplegrad-s-float-float-float-int-float-.md)           | 使用漸層來影響範例位置的計算方式，並使用選擇性值來將範例詳細資料 (」 LOD) 值到，以取樣材質。<br/>                                     |
| [**SampleGrad (S、float、float、float、int、float、uint)**](t1d-samplegrad-s-float-float-float-int-float-uint-.md) | 使用漸層來影響範例位置的計算方式，並使用選擇性值來將範例詳細資料 (」 LOD) 值到，以取樣材質。 傳回操作的相關狀態。<br/> |
