# <a name="statements"></a>陳述式

HLSL 語句是組成運算式的變數和函數的組合;HLSL 語句和句子一樣，都需要決定如何評估運算式的運算子。 您可以在著色器中建立個別的語句，或在稱為語句區塊的群組中建立數個語句。 若要建立 HLSL 語句，您必須知道撰寫運算式的語法，以及您將用來結合運算式的運算子。

-   [運算式](dx-graphics-hlsl-expressions.md)
-   [運算子](dx-graphics-hlsl-operators.md)
-   [語句區塊](dx-graphics-hlsl-statement-blocks.md)
