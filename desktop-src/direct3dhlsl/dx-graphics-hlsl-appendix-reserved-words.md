# <a name="reserved-words"></a>保留字

下列單字保留給 HLSL 語言使用。 請勿使用它們來命名您 HLSL 程式碼中的變數或函數。

:::column:::
    auto
    case, catch
    char, class, const\_cast
    default, delete, dynamic\_cast
    enum
    explicit
    friend
    goto
    long
    mutable
    new
    operator
    private, protected, public
    reinterpret\_cast
    short, signed, sizeof, static\_cast
    template, this, throw
    try, typename
    union, unsigned
    using
    virtual
:::column-end:::
