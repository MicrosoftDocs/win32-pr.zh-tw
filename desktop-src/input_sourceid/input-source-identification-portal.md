# <a name="input-source-identification"></a>輸入來源識別

## <a name="purpose"></a>目的

本節中的主題提供 Windows 8 中輸入來源識別的支援總覽。 輸入來源識別可讓應用程式精確且可靠地偵測輸入來源、識別輸入類型，以及提供最適合輸入類型的使用者體驗。

此處提供的主題集合描述您的應用程式如何取得這項資訊，並充分運用它來將客戶的使用者體驗優化。

輸入來源識別 Api 是針對建立 UI 架構的開發人員所設計，可在桌面應用程式之間提供一致的觸控優化使用者體驗。
