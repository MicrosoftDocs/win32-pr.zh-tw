# <a name="mobile-computing-configurations-for-sens"></a>適用于 SENS 的行動計算設定

連接功能和通知適用于設定如下的電腦：

-   在高頻寬網路上的銜接站中使用的行動電腦，有時可能會使用撥號連線。
-   單獨使用撥號連線的行動電腦。
-   以獨佔方式使用撥號連線的桌上型電腦。
-   連線到具有延遲問題之高頻寬網路的桌上型電腦。

在這些設定中，應用程式可以使用連接頻寬和延遲資訊來動態優化其網路可用性的作業。
