# <a name="runningtaskcurrentaction-property"></a>RunningTask. CurrentAction 屬性

針對腳本，取得執行中工作正在執行之目前動作的名稱。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
RunningTask.CurrentAction As String
```



## <a name="property-value"></a>屬性值

正在執行之工作正在執行之目前動作的名稱。
