# <a name="triggercollectionclear-method"></a>TriggerCollection Clear 方法

針對腳本，會清除集合中的所有觸發程式。

## <a name="syntax"></a>語法


```VB
TriggerCollection.Clear()
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
