# <a name="registeredtaskdefinition-property"></a>RegisteredTask 定義屬性

針對腳本，取得工作的定義。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
RegisteredTask.Definition As TaskDefinition
```



## <a name="property-value"></a>屬性值

工作的定義。
