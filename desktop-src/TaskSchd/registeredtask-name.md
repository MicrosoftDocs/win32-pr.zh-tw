# <a name="registeredtaskname-property"></a>RegisteredTask.Name 屬性

針對腳本，取得已註冊工作的名稱。

## <a name="syntax"></a>Syntax


```VB
RegisteredTask.Name As String
```



## <a name="property-value"></a>屬性值
