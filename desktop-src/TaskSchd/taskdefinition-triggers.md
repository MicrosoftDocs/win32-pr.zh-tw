# <a name="taskdefinitiontriggers-property"></a>TaskDefinition。 Trigger 屬性

針對腳本，取得或設定用來啟動工作的觸發程式集合。

## <a name="syntax"></a>Syntax


```VB
TaskDefinition.Triggers As TriggerCollection
```



## <a name="property-value"></a>屬性值

用來啟動工作的觸發程式集合。
