# <a name="registeredtaskenabled-property"></a>RegisteredTask 已啟用屬性

針對腳本，取得或設定布林值，這個值會指出是否已啟用已註冊的工作。

## <a name="syntax"></a>Syntax


```VB
RegisteredTask.Enabled As Boolean
```



## <a name="property-value"></a>屬性值

布林值，指出已註冊的工作是否已啟用。
