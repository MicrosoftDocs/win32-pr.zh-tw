# <a name="starting-an-executable-daily"></a>每天啟動可執行檔

撰寫每日啟動可執行檔的工作，是藉由定義每日觸發程式和可執行檔動作來完成。

## <a name="daily-triggers"></a>每日觸發程式

每日觸發程式會使用其開始界限來啟動觸發程式，並指定執行工作的當日時間。 啟動觸發程式之後，就會使用觸發程式間隔來指出工作每天、每天、每隔三天執行一次。
