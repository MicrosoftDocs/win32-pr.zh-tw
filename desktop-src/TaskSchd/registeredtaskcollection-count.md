# <a name="registeredtaskcollectioncount-property"></a>RegisteredTaskCollection Count 屬性

針對腳本，取得集合中已註冊工作的數目。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
RegisteredTaskCollection.Count As long
```



## <a name="property-value"></a>屬性值

集合中已註冊的工作數目。
