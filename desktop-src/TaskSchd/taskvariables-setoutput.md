# <a name="taskvariablessetoutput-method"></a>TaskVariables. SetOutput 方法

針對腳本，設定工作的輸出變數。 這個方法尚未實作。

## <a name="syntax"></a>語法


```VB
TaskVariables.SetOutput( _
  ByVal input _
)
```



## <a name="parameters"></a>參數

<dl> <dt>

*輸入* \[在\]
</dt> <dd>

工作的輸出變數。

</dd> </dl>

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
