# <a name="registrationinfoxmltext-property"></a>RegistrationInfo.XmlText 屬性

針對腳本，取得或設定工作的 XML 格式版本的註冊資訊。

## <a name="syntax"></a>Syntax


```VB
RegistrationInfo.XmlText As String
```



## <a name="property-value"></a>屬性值

XML 格式的工作註冊資訊版本。
