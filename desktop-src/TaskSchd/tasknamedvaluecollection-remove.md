# <a name="tasknamedvaluecollectionremove-method"></a>TaskNamedValueCollection 方法

針對腳本，會從集合中移除選取的名稱/值組。

## <a name="syntax"></a>語法


```VB
TaskNamedValueCollection.Remove( _
  ByVal index _
)
```



## <a name="parameters"></a>參數

<dl> <dt>

*索引* \[在\]
</dt> <dd>

要移除之名稱/值組的索引。

</dd> </dl>

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
