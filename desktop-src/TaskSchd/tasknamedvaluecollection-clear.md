# <a name="tasknamedvaluecollectionclear-method"></a>TaskNamedValueCollection Clear 方法

若為腳本，則會清除整個名稱/值配對的集合。

## <a name="syntax"></a>語法


```VB
TaskNamedValueCollection.Clear()
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
