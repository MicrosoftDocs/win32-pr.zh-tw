# <a name="task-scheduler-10-enumerated-types"></a>工作排程器1.0 列舉類型

\[\[此 API 可能會在後續版本的作業系統或產品中變更或無法使用。 請改用[工作排程器2.0 介面](task-scheduler-2-0-interfaces.md)或[工作排程器2.0 列舉類型](task-scheduler-2-0-enumerated-types.md)。 \]\]

描述列舉型別，這些列舉型別會定義工作排程器 1.0 Api 所使用的常數。


下列列舉類型是由工作排程器1.0 引進。



| 名稱                                             | 描述                                                                        |
|--------------------------------------------------|------------------------------------------------------------------------------------|
| [**工作 \_ 觸發程式 \_ 類型**](/windows/desktop/api/Mstask/ne-mstask-task_trigger_type) | 定義與工作相關聯的 [*觸發*](t.md) 程式類型。 |
| [**TASKPAGE**](/windows/desktop/api/Mstask/ne-mstask-taskpage)                     | 定義要取出的工作頁面類型。                                     |
