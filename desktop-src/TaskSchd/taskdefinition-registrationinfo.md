# <a name="taskdefinitionregistrationinfo-property"></a>TaskDefinition. RegistrationInfo 屬性

針對腳本，取得或設定用來描述工作的註冊資訊，例如工作的描述、工作的作者，以及工作的註冊日期。

## <a name="syntax"></a>Syntax


```VB
TaskDefinition.RegistrationInfo As RegistrationInfo
```



## <a name="property-value"></a>屬性值

用來描述工作的註冊資訊，例如工作的描述、工作的作者，以及工作的註冊日期。
