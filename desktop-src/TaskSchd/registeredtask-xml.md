# <a name="registeredtaskxml-property"></a>RegisteredTask.XML 屬性

針對腳本，取得已註冊工作的 XML 格式註冊資訊。

## <a name="syntax"></a>Syntax


```VB
RegisteredTask.XML As String
```



## <a name="property-value"></a>屬性值

已註冊工作的 XML 格式註冊資訊。
