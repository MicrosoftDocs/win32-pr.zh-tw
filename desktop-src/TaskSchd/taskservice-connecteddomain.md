# <a name="taskserviceconnecteddomain-property"></a>TaskService. ConnectedDomain 屬性

針對腳本，取得 [**TargetServer**](taskservice-targetserver.md) 電腦所連接之網域的名稱。

## <a name="syntax"></a>Syntax


```VB
TaskService.ConnectedDomain As String
```



## <a name="property-value"></a>屬性值

[**TargetServer**](taskservice-targetserver.md)電腦所連線的網域。
