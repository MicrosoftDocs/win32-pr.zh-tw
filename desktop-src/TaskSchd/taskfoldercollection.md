# <a name="taskfoldercollection-object"></a>TaskFolderCollection 物件

編寫物件的腳本物件，該物件會提供包含工作之資料夾集合的資訊和控制權。

## <a name="members"></a>成員

**TaskFolderCollection** 物件具有下列類型的成員：

-   [屬性](#properties)

### <a name="properties"></a>屬性

**TaskFolderCollection** 物件具有這些屬性。



| 屬性                                               | 描述                                                |
|:-------------------------------------------------------|:-----------------------------------------------------------|
| [**計數**](taskfoldercollection-count.md)<br/> | 取得集合中的資料夾數目。<br/>   |
| [**項目**](taskfoldercollection-item.md)<br/>   | 從集合中取得指定的資料夾。 <br/> |
