# <a name="taskvariablesgetinput-method"></a>TaskVariables. >getinput 方法

針對腳本，取得工作的輸入變數。 這個方法尚未實作。

## <a name="syntax"></a>語法


```VB
TaskVariables.GetInput( _
  ByRef input _
)
```



## <a name="parameters"></a>參數

<dl> <dt>

*輸入* \[擴展\]
</dt> <dd>

工作的輸入變數。

</dd> </dl>

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
