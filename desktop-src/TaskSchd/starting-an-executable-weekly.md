# <a name="starting-an-executable-weekly"></a>每週啟動可執行檔

撰寫每週啟動可執行檔的工作，是藉由定義每週觸發程式和可執行檔動作來完成。

## <a name="weekly-triggers"></a>每週觸發程式

每週觸發程式會使用其開始界限來啟動觸發程式，並指定執行工作的當日時間。 觸發程式啟動時，觸發程式也會指定在一周內執行工作的時間，以及工作每週執行一次、每週、每隔三周等等。
