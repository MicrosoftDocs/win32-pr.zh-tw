# <a name="registeredtasklasttaskresult-property"></a>RegisteredTask. LastTaskResult 屬性

針對腳本，取得上次執行註冊工作時所傳回的結果。

## <a name="syntax"></a>Syntax


```VB
RegisteredTask.LastTaskResult As String
```



## <a name="property-value"></a>屬性值

上次執行註冊工作時所傳回的結果。
