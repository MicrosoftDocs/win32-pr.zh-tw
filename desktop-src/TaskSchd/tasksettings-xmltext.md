# <a name="tasksettingsxmltext-property"></a>TaskSettings.XmlText 屬性

針對腳本，取得或設定工作設定的 XML 格式化定義。

這是可讀寫的屬性。

## <a name="syntax"></a>Syntax


```VB
TaskSettings.XmlText As String
```



## <a name="property-value"></a>屬性值

工作設定的 XML 格式化定義。
