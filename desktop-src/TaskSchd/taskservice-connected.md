# <a name="taskserviceconnected-property"></a>TaskService 連接屬性

針對腳本，會取得布林值，指出您是否連接到工作排程器服務。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
TaskService.Connected
```



## <a name="property-value"></a>屬性值

指出您是否連接到工作排程器服務的布林值。
