# <a name="taskdefinitionsettings-property"></a>TaskDefinition 設定屬性

針對腳本，取得或設定定義工作排程器服務執行工作方式的設定。

## <a name="syntax"></a>Syntax


```VB
TaskDefinition.Settings As TaskSettings
```



## <a name="property-value"></a>屬性值

定義工作排程器服務如何執行工作的設定。
