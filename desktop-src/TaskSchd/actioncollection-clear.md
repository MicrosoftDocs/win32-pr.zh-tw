# <a name="actioncollectionclear-method"></a>Actioncollection 動作 Clear 方法

針對腳本，會從集合中清除所有動作。

## <a name="syntax"></a>語法


```VB
ActionCollection.Clear()
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
