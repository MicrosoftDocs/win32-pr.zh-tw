# <a name="taskdefinitionxmltext-property"></a>TaskDefinition.XmlText 屬性

針對腳本，取得或設定工作的 XML 格式化定義。

## <a name="syntax"></a>Syntax


```VB
TaskDefinition.XmlText As String
```



## <a name="property-value"></a>屬性值

工作的 XML 格式化定義。

## <a name="remarks"></a>備註

工作的 XML 是由 [工作排程器架構](task-scheduler-schema.md)所定義。 如需工作 XML 的範例，請參閱 [ (XML) 的每日觸發程式範例 ](daily-trigger-example--xml-.md)。
