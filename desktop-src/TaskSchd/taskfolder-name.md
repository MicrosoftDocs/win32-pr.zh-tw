# <a name="taskfoldername-property"></a>TaskFolder.Name 屬性

針對腳本，取得用來識別包含工作之資料夾的名稱。

## <a name="syntax"></a>Syntax


```VB
TaskFolder.Name As String
```



## <a name="property-value"></a>屬性值

用來識別資料夾的名稱。
