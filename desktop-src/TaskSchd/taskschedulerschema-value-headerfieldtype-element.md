# <a name="value-headerfieldtype-element"></a>值 (headerFieldType) 元素

在電子郵件訊息中包含標頭欄位的值。

``` syntax
<xs:element name="Value"
    type="string"
 />
```

**Value** 元素是由 [**headerFieldType**](taskschedulerschema-headerfieldtype-complextype.md)複雜型別定義。
