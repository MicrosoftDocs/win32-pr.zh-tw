# <a name="monthlytriggerrunonlastdayofmonth-property"></a>MonthlyTrigger. RunOnLastDayOfMonth 屬性

針對腳本，取得或設定布林值，指出工作會在當月的最後一天執行。

## <a name="syntax"></a>Syntax


```VB
MonthlyTrigger.RunOnLastDayOfMonth As Boolean
```



## <a name="property-value"></a>屬性值

True 表示工作會在當月的最後一天執行，而不考慮當天的實際日期。否則為 False。
