# <a name="taskdefinitionprincipal-property"></a>TaskDefinition Principal 屬性

針對腳本，取得或設定工作的主體，此工作會提供工作的安全性認證。

這是可讀寫的屬性。

## <a name="syntax"></a>Syntax


```VB
TaskDefinition.Principal As Principal
```



## <a name="property-value"></a>屬性值

提供工作之安全性認證的工作主體。
