# <a name="monthlydowtriggerrunonlastweekofmonth-property"></a>MonthlyDOWTrigger. RunOnLastWeekOfMonth 屬性

針對腳本，取得或設定布林值，指出工作會在當月的最後一周執行。

## <a name="syntax"></a>Syntax


```VB
MonthlyDOWTrigger.RunOnLastWeekOfMonth As Boolean
```



## <a name="property-value"></a>屬性值

True 表示工作在當月的最後一周執行;否則為 False。
