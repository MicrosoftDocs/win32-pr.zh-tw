# <a name="taskvariablesgetcontext-method"></a>TaskVariables. GetCoNtext 方法

針對腳本，用來在相同作業實例中的不同步驟和工作之間共用內容。 這個方法尚未實作。

## <a name="syntax"></a>語法


```VB
TaskVariables.GetContext( _
  ByRef context _
)
```



## <a name="parameters"></a>參數

<dl> <dt>

*內容* \[擴展\]
</dt> <dd>

用來在相同作業實例中的不同步驟和工作之間共用內容的內容。

</dd> </dl>

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
