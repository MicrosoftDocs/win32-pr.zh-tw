# <a name="registeredtaskpath-property"></a>RegisteredTask 路徑屬性

針對腳本，取得已註冊工作儲存位置的路徑。

## <a name="syntax"></a>Syntax


```VB
RegisteredTask.Path As String
```



## <a name="property-value"></a>屬性值

註冊的工作儲存位置的路徑。
