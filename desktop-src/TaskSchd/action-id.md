# <a name="actionid-property"></a>Action.Id 屬性

針對腳本，取得或設定動作的識別碼。

## <a name="syntax"></a>Syntax


```VB
Action.Id As String
```



## <a name="property-value"></a>屬性值

動作的使用者定義識別碼。 工作排程器使用此識別碼來進行記錄。
