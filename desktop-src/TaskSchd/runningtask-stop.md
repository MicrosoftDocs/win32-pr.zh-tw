# <a name="runningtaskstop-method"></a>RunningTask. Stop 方法

針對腳本，會停止此工作的實例。

## <a name="syntax"></a>語法


```VB
RunningTask.Stop()
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
