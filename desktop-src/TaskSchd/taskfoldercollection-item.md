# <a name="taskfoldercollectionitem-property"></a>TaskFolderCollection 專案屬性

針對腳本，從集合取得指定的資料夾。

## <a name="syntax"></a>Syntax


```VB
TaskFolderCollection.Item( _
  ByVal Index _
) As TaskFolder
```



## <a name="property-value"></a>屬性值

代表所要求之資料夾的 [**TaskFolder**](taskfolder.md) 物件。
