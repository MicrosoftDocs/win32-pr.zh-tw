# <a name="registeredtasknumberofmissedruns-property"></a>RegisteredTask. NumberOfMissedRuns 屬性

針對腳本，取得已註冊的工作已錯過排程執行的次數。

## <a name="syntax"></a>Syntax


```VB
RegisteredTask.NumberOfMissedRuns As Integer
```



## <a name="property-value"></a>屬性值

註冊的工作錯過排程執行的次數。
