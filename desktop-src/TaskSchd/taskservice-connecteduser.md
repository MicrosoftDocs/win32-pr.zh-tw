# <a name="taskserviceconnecteduser-property"></a>TaskService. >connecteduser 屬性

針對腳本，取得連接到工作排程器服務之使用者的名稱。

## <a name="syntax"></a>Syntax


```VB
TaskService.ConnectedUser As String
```



## <a name="property-value"></a>屬性值

連接到工作排程器服務之使用者的名稱。
