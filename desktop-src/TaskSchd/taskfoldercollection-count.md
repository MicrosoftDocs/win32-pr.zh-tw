# <a name="taskfoldercollectioncount-property"></a>TaskFolderCollection Count 屬性

針對腳本，取得集合中的資料夾數目。

## <a name="syntax"></a>Syntax


```VB
TaskFolderCollection.Count As long
```



## <a name="property-value"></a>屬性值
