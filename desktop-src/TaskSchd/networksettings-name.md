# <a name="networksettingsname-property"></a>NetworkSettings.Name 屬性

針對腳本，取得或設定網路設定檔的名稱。 此名稱會用於顯示用途。

這是可讀寫的屬性。

## <a name="syntax"></a>Syntax


```VB
NetworkSettings.Name As String
```



## <a name="property-value"></a>屬性值

網路設定檔的名稱。
