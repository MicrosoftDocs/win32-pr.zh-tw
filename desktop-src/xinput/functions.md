# <a name="functions-programming-reference"></a>函數 (程式設計參考) 

可用的 XInput 函式。

## <a name="in-this-section"></a>本節內容

| 主題                                                                                 | 描述                                                                                                                                             |
|---------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XInputEnable**](/windows/desktop/api/XInput/nf-xinput-xinputenable)<br/>                                       | 設定 XInput 的報告狀態。<br/>                                                                                                          |
| [**XInputGetAudioDeviceIds**](/windows/desktop/api/XInput/nf-xinput-xinputgetaudiodeviceids)<br/>                 | 抓取與連接至指定控制器之耳機相關聯的音效轉譯和音效捕捉音訊裝置識別碼。<br/> |
| [**XInputGetBatteryInformation**](/windows/desktop/api/XInput/nf-xinput-xinputgetbatteryinformation)<br/>         | 抓取無線控制器的電池類型和充電狀態。<br/>                                                                       |
| [**XInputGetCapabilities**](/windows/desktop/api/XInput/nf-xinput-xinputgetcapabilities)<br/>                     | 抓取連接控制器的功能和功能。<br/>                                                                           |
| [**XInputGetDSoundAudioDeviceGuids**](/windows/desktop/api/XInput/nf-xinput-xinputgetdsoundaudiodeviceguids)<br/> | 取得與連接至指定控制器之耳機相關聯的音效轉譯和音效捕捉裝置 Guid。<br/>          |
| [**XInputGetKeystroke**](/windows/desktop/api/XInput/nf-xinput-xinputgetkeystroke)<br/>                           | 捕獲遊戲台輸入事件。<br/>                                                                                                             |
| [**XInputGetState**](/windows/desktop/api/XInput/nf-xinput-xinputgetstate)<br/>                                   | 抓取指定之控制器的目前狀態。<br/>                                                                                     |
| [**XInputSetState**](/windows/desktop/api/XInput/nf-xinput-xinputsetstate)<br/>                                   | 將資料傳送至連接的控制器。 這個函式是用來啟用控制器的震動函數。<br/>                              |
