# <a name="structures-xinput-game-controller-apis"></a>結構 (XInput 遊戲控制器 Api) 

可用的 XInput 結構

## <a name="in-this-section"></a>本節內容

| 主題                                                                         | 描述                                                                                                                                                                                        |
|-------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XINPUT \_ 電池 \_ 資訊**](/windows/desktop/api/XInput/ns-xinput-xinput_battery_information)<br/> | 包含電池類型和收費狀態的資訊。<br/>                                                                                                                                  |
| [**XINPUT \_ 功能**](/windows/desktop/api/XInput/ns-xinput-xinput_capabilities)<br/>                | 描述連接控制器的功能。 [**XInputGetCapabilities**](/windows/desktop/api/XInput/nf-xinput-xinputgetcapabilities)函數會傳回 [**XINPUT \_ 功能**](/windows/desktop/api/XInput/ns-xinput-xinput_capabilities)。 <br/> |
| [**XINPUT \_ 遊戲台**](/windows/desktop/api/XInput/ns-xinput-xinput_gamepad)<br/>                          | 描述 Xbox 控制器的目前狀態。<br/>                                                                                                                                 |
| [**XINPUT \_ 按鍵**](/windows/desktop/api/XInput/ns-xinput-xinput_keystroke)<br/>                      | 指定 [**XInputGetKeystroke**](/windows/desktop/api/XInput/nf-xinput-xinputgetkeystroke)傳回的按鍵資料。<br/>                                                                                                  |
| [**XINPUT \_ 狀態**](/windows/desktop/api/XInput/ns-xinput-xinput_state)<br/>                              | 代表控制器的狀態。<br/>                                                                                                                                                   |
| [**XINPUT \_ 振動**](/windows/desktop/api/XInput/ns-xinput-xinput_vibration)<br/>                      | 指定控制器震動功能的馬達速度層級。<br/>                                                                                                                |
