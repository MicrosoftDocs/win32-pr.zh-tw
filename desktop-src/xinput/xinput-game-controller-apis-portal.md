# <a name="xinput-game-controller-apis"></a>XInput 遊戲控制器 Api

## <a name="purpose"></a>目的

XInput 遊戲控制器 API 可讓應用程式從適用于 Windows 的 Xbox Controller 接收輸入。

| [程式設計參考](programming-reference.md)<br/> | XInput 函式和結構。<br/>                                                                                                             |

## <a name="developer-audience"></a>開發人員對象

XInput 遊戲控制器 Api 的設計目的，是要讓想要將 Xbox Controller 用於 Windows 應用程式的開發人員使用。
