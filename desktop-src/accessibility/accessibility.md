# <a name="windows-accessibility"></a>Windows 協助工具

本節包含的檔適用于設計可存取應用程式的 Windows 開發人員、輔助技術開發人員建立工具（例如螢幕閱讀程式和放大鏡），以及建立自動化腳本來測試 Windows 應用程式的軟體測試工程師。

您可以使用此處提供的資源，建立可供許多人使用的可存取 Windows 應用程式，包括殘障人士、個人喜好設定，以及特定工作樣式。
