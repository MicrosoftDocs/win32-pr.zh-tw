# <a name="legacy-accessibility-and-automation-technology---msaa-to-ui-automation"></a>舊版協助工具和自動化技術 - MSAA 至 UI 自動化

Windows 協助工具和自動化是由兩種技術所組成： Microsoft Active Accessibility (MSAA) 和 Microsoft 消費者介面自動化。 MSAA 是 Windows 95 引進的舊版技術，而消費者介面自動化是較新且更強大的技術，可解決 MSAA 的限制。

本節說明 MSAA 和消費者介面自動化 (之間的主要差異，以及如何轉換至消費者介面自動化) 。
