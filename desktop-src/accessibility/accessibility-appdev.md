# <a name="developing-accessible-applications-for-windows"></a>開發適用於 Windows 的無障礙應用程式

每個 Windows 應用程式架構都支援許多協助工具功能，讓殘障人士、個人喜好設定或特定工作樣式的使用者可以更輕鬆地使用他們選擇的任何 Windows 裝置。

本總覽描述每個架構中所支援的協助工具功能，以及您如何在建立應用程式時將其納入。
