# <a name="developing-assistive-technology-for-windows"></a>開發 Windows 的輔助技術

協力廠商輔助技術產品（例如螢幕讀者、放大鏡和專業導向硬體）是許多 Windows (和其他 Microsoft 產品) 的基本工具。

本教學課程說明如何建立與 Microsoft 產品相容的輔助技術工具，以供具有視覺、學習、機動/行動性及語言/通訊障礙或限制的人員使用。
