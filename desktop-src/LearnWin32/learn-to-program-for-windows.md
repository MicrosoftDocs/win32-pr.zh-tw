# <a name="get-started-with-win32-and-c"></a>開始使用 Win32 和 C++

本開始系列旨在告訴您如何在 c + + 中使用 Win32 和 COM Api 撰寫桌上型電腦程式。

在第一個課程模組中，您將學習如何建立及顯示視窗的逐步解說。 稍後的課程模組會介紹元件物件模型 (COM) 、圖形和文字，以及使用者輸入。

在本系列中，我們假設您對 c + + 程式設計有良好的使用知識。 未採用任何先前的 Windows 程式設計經驗。 如果您不熟悉 c + +，您可以在 [Visual C++ 開發人員中心](https://msdn.microsoft.com/vstudio//default.aspx)找到學習教材。  (此資源可能無法在某些語言及國家/地區使用。 )
