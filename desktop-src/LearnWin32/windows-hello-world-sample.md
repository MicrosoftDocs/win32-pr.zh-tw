# <a name="windows-hello-world-sample"></a>Windows Hello世界範例

這個範例應用程式會示範如何建立基本的 Windows 程式。

## <a name="description"></a>Description

Windows Hello World 範例應用程式會建立並顯示空白視窗，如下列螢幕擷取畫面所示。 課程模組1會討論此範例[。您的第一個 Windows 計畫](your-first-windows-program.md)。

![範例程式的螢幕擷取畫面。](images/window01.png)

## <a name="downloading-the-sample"></a>下載範例

此範例可在 [這裡](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/begin/LearnWin32/HelloWorld)取得。

若要下載，請移至 GitHub ([microsoft/Windows-傳統範例](https://github.com/microsoft/Windows-classic-samples/)) 上的範例存放庫根目錄，然後按一下 [**複製] 或 [下載**] 按鈕，將所有範例的 zip 檔案下載至您的電腦。 然後將資料夾解壓縮。

若要在 Visual Studio 中開啟範例，請選取 [檔案] **/[開啟/Project/Solution**]，然後流覽至您解壓縮資料夾的位置，然後 **Windows-傳統-範例-master/sample/Win7Samples/begin/LearnWin32/HelloWorld/cpp**。 開啟 **HelloWorld** 檔案。

載入範例之後，您必須將其更新為使用 Windows 10。 從 Visual Studio 的 [ **Project** ] 功能表中，選取 [**屬性**]。 將 **Windows SDK 版本** 更新為 Windows 10 SDK （例如10.0.17763.0 或更好的版本）。 然後將 **平臺工具** 組變更為 Visual Studio 2017 或更好。 現在您可以按 F5 來執行範例！
