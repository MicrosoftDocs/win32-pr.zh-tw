# <a name="webdav-api-structures"></a>WebDAV API 結構

WebDAV API 中會使用下列結構。



| 結構                                                   | Description                                                                                                                                                                                         |
|-------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DAV \_ 回呼 \_ 驗證 \_ BLOB**](/windows/desktop/api/Davclnt/ns-davclnt-dav_callback_auth_blob) | 此結構是用來儲存 [*DavAuthCallback*](/windows/desktop/api/Davclnt/nc-davclnt-pfndavauthcallback)回呼函式所取出的驗證 [*BLOB*](/windows/desktop/SecGloss/b-gly) 。 |
| [**DAV \_ 回呼 \_ 驗證 \_ UNP**](/windows/desktop/api/Davclnt/ns-davclnt-dav_callback_auth_unp)   | 此結構是用來儲存 [*DavAuthCallback*](/windows/desktop/api/Davclnt/nc-davclnt-pfndavauthcallback) 回呼函式所取出的使用者名稱和密碼資訊。                                               |
| [**DAV \_ 回呼 \_ 認證**](/windows/desktop/api/Davclnt/ns-davclnt-dav_callback_cred)            | [*DavAuthCallback*](/windows/desktop/api/Davclnt/nc-davclnt-pfndavauthcallback)回呼函式會使用此結構來儲存使用者認證資訊。                                                                               |
