# <a name="built-in-keying-module-identifiers"></a>內建的加密模組識別碼

內建于 Windows 篩選平台 (WFP) 之金鑰模組的識別碼，會由 GUID 表示。

這些識別碼的定義如下。

<dl> <dt>

<span id="FWPM_KEYING_MODULE_IKE"></span><span id="fwpm_keying_module_ike"></span>**FWPM \_ 加密 \_ 模組 \_ IKE**
</dt> <dd> <dl> <dt>



網際網路金鑰交換 (IKE) 的加密模組。


</dt> </dl> </dd> <dt>

<span id="FWPM_KEYING_MODULE_IKEV2"></span><span id="fwpm_keying_module_ikev2"></span>**FWPM \_ 加密 \_ 模組 \_ IKEV2**
</dt> <dd> <dl> <dt>



網際網路金鑰交換第2版 (IKEv2) 鍵控模組。

> [!Note]  
> 僅適用于 Windows Server 2008 R2 和 Windows 7。

 


</dt> </dl> </dd> <dt>

<span id="FWPM_KEYING_MODULE_AUTHIP"></span><span id="fwpm_keying_module_authip"></span>**FWPM \_ 加密 \_ 模組 \_ AUTHIP**
</dt> <dd> <dl> <dt>



AuthIP 金鑰模組。


</dt> </dl> </dd> </dl>
