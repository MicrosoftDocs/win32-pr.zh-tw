# <a name="built-in-provider-context-identifiers"></a>內建提供者內容識別碼

內建于 Windows 篩選平台 (WFP) 的提供者內容識別碼，都是由 GUID 表示。 這些內建提供者內容會識別要搭配安全通訊端使用的預設原則。

這些識別碼的定義如下。

<dl> <dt>

<span id="FWPM_PROVIDER_CONTEXT_SECURE_SOCKET_AUTHIP"></span><span id="fwpm_provider_context_secure_socket_authip"></span>**FWPM \_ 提供者 \_ 內容 \_ 安全 \_ 通訊端 \_ AUTHIP**
</dt> <dd> <dl> <dt>



驗證的網際網路通訊協定 (AuthIP) 安全通訊端的主要模式預設原則。


</dt> </dl> </dd> <dt>

<span id="FWPM_PROVIDER_CONTEXT_SECURE_SOCKET_IPSEC"></span><span id="fwpm_provider_context_secure_socket_ipsec"></span>**FWPM \_ 提供者 \_ 內容 \_ 安全 \_ 通訊端 \_ IPSEC**
</dt> <dd> <dl> <dt>



網際網路通訊協定安全性 (IPsec) 安全通訊端的快速模式預設原則。


</dt> </dl> </dd> </dl>
