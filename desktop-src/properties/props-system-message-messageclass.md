# <a name="systemmessagemessageclass"></a>MessageClass

Microsoft Outlook 訊息的類型 (會議、工作、郵件等) 。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7、Windows Vista 版本

```
propertyDescription
   name = System.Message.MessageClass
   shellPKey = PKEY_Message_MessageClass
   formatID = CD9ED458-08CE-418F-A70E-F912C7BB9C5C
   propID = 103
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
