# <a name="systemdeviceswinphone8cameraflags"></a>WinPhone8CameraFlags

WP8 攝影機裝置的旗標。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Devices.WinPhone8CameraFlags
   shellPKey = PKEY_Devices_WinPhone8CameraFlags
   formatID = B7B4D61C-5A64-4187-A52E-B1539F359099
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt32
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
