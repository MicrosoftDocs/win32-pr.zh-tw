# <a name="systemdevicesdevobjecttype"></a>DevObjectType

DevQuery 裝置物件類型

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Devices.DevObjectType
   shellPKey = PKEY_Devices_DevObjectType
   formatID = 13673F42-A3D6-49F6-B4DA-AE46E0C5237C
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt32
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
