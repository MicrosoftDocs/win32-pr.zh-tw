# <a name="systemdeviceslaunchdevicestagefromexplorer"></a>LaunchDeviceStageFromExplorer

指出是否啟動 Device Stage。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.Devices.LaunchDeviceStageFromExplorer
   shellPKey = PKEY_Devices_LaunchDeviceStageFromExplorer
   formatID = 78C34FC8-104A-4ACA-9EA4-524D52996E57
   propID = 77
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
