# <a name="systemcontactotheraddressstate"></a>OtherAddressState

連絡人的其他位址狀態名稱。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、version 1507、Windows 8.1、Windows 8、Windows 7、Windows Vista

```
propertyDescription
   name = System.Contact.OtherAddressState
   shellPKey = PKEY_Contact_OtherAddressState
   formatID = 71B377D6-E570-425F-A170-809FAE73E54E
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
