# <a name="systemdevicesaepservicebluetoothserviceguid"></a>AepService 藍牙. ServiceGuid

藍牙服務 GUID。

## <a name="windows-10-version-1703-windows-10-version-1607"></a>Windows 10，版本1703，Windows 10，版本1607

```
propertyDescription
   name = System.Devices.AepService.Bluetooth.ServiceGuid
   shellPKey = PKEY_Devices_AepService_Bluetooth_ServiceGuid
   formatID = A399AAC7-C265-474E-B073-FFCE57721716
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Guid
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
