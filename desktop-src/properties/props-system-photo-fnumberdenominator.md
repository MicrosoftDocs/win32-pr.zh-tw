# <a name="systemphotofnumberdenominator"></a>FNumberDenominator

PKEY \_ 相片 FNumber 的分母 \_ 。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7、Windows Vista 版本

```
propertyDescription
   name = System.Photo.FNumberDenominator
   shellPKey = PKEY_Photo_FNumberDenominator
   formatID = E92A2496-223B-4463-A4E3-30EABBA79D80
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt32
```

PKEY 值定義于 Propkey 中。
