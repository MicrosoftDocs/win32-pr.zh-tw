# <a name="systemdevicesdnssdnetworkadapterid"></a>Dnssd. NetworkAdapterId

要在其上搜尋服務之網路介面卡的 GUID。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Devices.Dnssd.NetworkAdapterId
   shellPKey = PKEY_Devices_Dnssd_NetworkAdapterId
   formatID = BF79C0AB-BB74-4CEE-B070-470B5AE202EA
   propID = 11
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Guid
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
