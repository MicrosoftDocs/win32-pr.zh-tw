# <a name="systemdeviceinterfaceprinterenumerationflag"></a>DeviceInterface. PrinterEnumerationFlag

印表機資訊印表機列舉旗標。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.DeviceInterface.PrinterEnumerationFlag
   shellPKey = PKEY_DeviceInterface_PrinterEnumerationFlag
   formatID = A00742A1-CD8C-4B37-95AB-70755587767A
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt32
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
