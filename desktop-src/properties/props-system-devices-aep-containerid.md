# <a name="systemdevicesaepcontainerid"></a>System.Devices.Aep.ContainerId

裝置關聯端點的父容器識別碼

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Devices.Aep.ContainerId
   shellPKey = PKEY_Devices_Aep_ContainerId
   formatID = E7C3FB29-CAA7-4F47-8C8B-BE59B330D4C5
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Guid
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
