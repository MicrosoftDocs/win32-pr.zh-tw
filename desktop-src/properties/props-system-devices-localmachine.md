# <a name="systemdeviceslocalmachine"></a>System.Devices.LocalMachine

若為 true，則表示有問題的裝置是電腦。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、version 1507、Windows 8.1、Windows 8

```
propertyDescription
   name = System.Devices.LocalMachine
   shellPKey = PKEY_Devices_LocalMachine
   formatID = 78C34FC8-104A-4ACA-9EA4-524D52996E57
   propID = 70
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="windows-7"></a>Windows 7

```
propertyDescription
   name = System.Devices.LocalMachine
   shellPKey = PKEY_Devices_IsLocalMachine
   formatID = 78C34FC8-104A-4ACA-9EA4-524D52996E57
   propID = 70
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
