# <a name="systemcontactbirthday"></a>System.object。生日

```
propertyDescription
   name = System.Contact.Birthday
   shellPKey = PKEY_Contact_Birthday
   formatID = 176DC63C-2688-4E89-8143-A347800F25E9
   propID = 47
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = DateTime
```
