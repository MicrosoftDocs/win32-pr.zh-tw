# <a name="systemdeviceinterfacebluetoothproductversion"></a>DeviceInterface。藍牙。ProductVersion

藍牙裝置產品版本。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版

```
propertyDescription
   name = System.DeviceInterface.Bluetooth.ProductVersion
   shellPKey = PKEY_DeviceInterface_Bluetooth_ProductVersion
   formatID = 2BD67D8B-8BEB-48D5-87E0-6CDA3428040A
   propID = 9
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
