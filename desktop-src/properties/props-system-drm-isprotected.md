# <a name="systemdrmisprotected"></a>IsProtected

指出檔案是否受 Rights Management (DRM) 的數位保護。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.DRM.IsProtected
   shellPKey = PKEY_DRM_IsProtected
   formatID = AEAC19E4-89AE-4508-B9B7-BB867ABEE2ED
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.DRM.IsProtected
   shellPKey = PKEY_DRM_IsProtected
   formatID = AEAC19E4-89AE-4508-B9B7-BB867ABEE2ED
   propID = 2
   SearchInfo
      IsColumn = true
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
