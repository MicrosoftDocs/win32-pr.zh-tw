# <a name="systemdevicesnotificationstore"></a>NotificationStore

裝置通知存放區。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.Devices.NotificationStore
   shellPKey = PKEY_Devices_NotificationStore
   formatID = 06704B0C-E830-4C81-9178-91E4E95A80A0
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Object
      IsInnate = true
```

PKEY 值定義于 Propkey 中。
