# <a name="systemmusiciscompilation"></a>IsCompilation

指出音樂檔案是否為編譯的一部分。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.Music.IsCompilation
   shellPKey = PKEY_Music_IsCompilation
   formatID = C449D5CB-9EA4-4809-82E8-AF9D59DED6D1
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Boolean
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
