# <a name="systemdevicesdefaulttooltip"></a>DefaultTooltip

預設狀態的工具提示。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.Devices.DefaultTooltip
   shellPKey = PKEY_Devices_DefaultTooltip
   formatID = 880F70A2-6082-47AC-8AAB-A739D1A300C3
   propID = 153
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
