# <a name="systemdevicesaepservicefriendlyname"></a>AepService FriendlyName

裝置關聯端點服務易記名稱

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Devices.AepService.FriendlyName
   shellPKey = PKEY_Devices_AepService_FriendlyName
   formatID = 71724756-3E74-4432-9B59-E7B2F668A593
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
