# <a name="systemstatusbarselecteditemcount"></a>System. StatusBarSelectedItemCount

在視圖中選取的專案計數和預估的總大小。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、version 1507、Windows 8.1、Windows 8

```
propertyDescription
   name = System.StatusBarSelectedItemCount
   shellPKey = PKEY_StatusBarSelectedItemCount
   formatID = 26DC287C-6E3D-4BD3-B2B0-6A26BA2E346D
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
