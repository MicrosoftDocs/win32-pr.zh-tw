# <a name="systemdeviceswifidirectisvisible"></a>WiFiDirect. IsVisible

指出 Wi-Fi 直接裝置的目前可見度

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Devices.WiFiDirect.IsVisible
   shellPKey = PKEY_Devices_WiFiDirect_IsVisible
   formatID = 1506935D-E3E7-450F-8637-82233EBE5F6E
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
