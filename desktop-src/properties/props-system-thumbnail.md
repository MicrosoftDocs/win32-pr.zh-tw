# <a name="systemthumbnail"></a>系統. 縮圖

表示以 VT CF 格式表示的縮圖 \_ 。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版、Windows 8。1

```
propertyDescription
   name = System.Thumbnail
   shellPKey = PKEY_Thumbnail
   formatID = F29F85E0-4FF9-1068-AB91-08002B27B3D9
   propID = 17
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Clipboard
      IsInnate = false
```

## <a name="windows-8-windows-7-windows-vista"></a>Windows 8、Windows 7、Windows Vista

```
propertyDescription
   name = System.Thumbnail
   shellPKey = PKEY_Thumbnail
   formatID = F29F85E0-4FF9-1068-AB91-08002B27B3D9
   propID = 17
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Clipboard
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
