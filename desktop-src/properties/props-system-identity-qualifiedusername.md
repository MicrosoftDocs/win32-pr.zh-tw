# <a name="systemidentityqualifiedusername"></a>QualifiedUserName

身分識別的限定使用者名稱。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、version 1507、Windows 8.1、Windows 8

```
propertyDescription
   name = System.Identity.QualifiedUserName
   shellPKey = PKEY_Identity_QualifiedUserName
   formatID = DA520E51-F4E9-4739-AC82-02E0A95C9030
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
