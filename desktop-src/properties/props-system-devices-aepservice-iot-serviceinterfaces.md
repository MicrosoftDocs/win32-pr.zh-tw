# <a name="systemdevicesaepserviceiotserviceinterfaces"></a>AepService. ServiceInterfaces。

適用于此服務的介面清單。

## <a name="windows-10-version-1703-windows-10-version-1607"></a>Windows 10，版本1703，Windows 10，版本1607

```
propertyDescription
   name = System.Devices.AepService.IoT.ServiceInterfaces
   shellPKey = PKEY_Devices_AepService_IoT_ServiceInterfaces
   formatID = 79D94E82-4D79-45AA-821A-74858B4E4CA6
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
