# <a name="systemlinktargetsfgaoflags"></a>TargetSFGAOFlags

連結目標的 [**IShellFolder：： GetAttributesOf**](/windows/win32/api/shobjidl_core/nf-shobjidl_core-ishellfolder-getattributesof) 旗標，其 SFGAO \_ PKEYSFGAOMASK 屬性已遮罩。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.Link.TargetSFGAOFlags
   shellPKey = PKEY_Link_TargetSFGAOFlags
   formatID = B9B4B3FC-2B51-4A42-B5D8-324146AFCF25
   propID = 8
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt32
      IsInnate = true
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.Link.TargetSFGAOFlags
   shellPKey = PKEY_Link_TargetSFGAOFlags
   formatID = B9B4B3FC-2B51-4A42-B5D8-324146AFCF25
   propID = 8
   SearchInfo
      IsColumn = true
   typeInfo
      type = UInt32
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
