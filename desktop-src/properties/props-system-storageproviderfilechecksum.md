# <a name="systemstorageproviderfilechecksum"></a>System. StorageProviderFileChecksum

檔案的儲存提供者所計算的總和檢查碼。 具有相同總和檢查碼值的檔案將會有相同的內容。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.StorageProviderFileChecksum
   shellPKey = PKEY_StorageProviderFileChecksum
   formatID = B2F9B9D6-FEC4-4DD5-94D7-8957488C807B
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Buffer
      IsInnate = false
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
