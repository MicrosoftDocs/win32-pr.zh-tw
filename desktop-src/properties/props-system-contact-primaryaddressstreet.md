# <a name="systemcontactprimaryaddressstreet"></a>PrimaryAddressStreet

連絡人的主要位址街道。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7、Windows Vista 版本

```
propertyDescription
   name = System.Contact.PrimaryAddressStreet
   shellPKey = PKEY_Contact_PrimaryAddressStreet
   formatID = 63C25B20-96BE-488F-8788-C09C407AD812
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
