# <a name="systemidentityuniqueid"></a>System.servicemodel

身分識別的唯一識別碼。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.Identity.UniqueID
   shellPKey = PKEY_Identity_UniqueID
   formatID = E55FC3B0-2B60-4220-918E-B21E8BF16016
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
