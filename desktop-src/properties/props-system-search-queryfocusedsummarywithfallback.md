# <a name="systemsearchqueryfocusedsummarywithfallback"></a>QueryFocusedSummaryWithFallback

檔的查詢焦點摘要。 如果沒有可用的，則會傳回 AutoSummary。 搜尋系統會自動建立這個屬性。 只有當取用應用程式中的值，而不是提供值給屬性處理常式時，才使用此屬性。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.Search.QueryFocusedSummaryWithFallback
   shellPKey = PKEY_Search_QueryFocusedSummaryWithFallback
   formatID = 560C36C0-503A-11CF-BAA1-00004C752A9A
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
