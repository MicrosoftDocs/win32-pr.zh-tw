# <a name="systemispinnedtonamespacetree"></a>System. IsPinnedToNameSpaceTree

識別 shell 資料夾是否釘選到流覽窗格。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.IsPinnedToNameSpaceTree
   shellPKey = PKEY_IsPinnedToNameSpaceTree
   formatID = 5D76B67F-9B3D-44BB-B6AE-25DA4F638A67
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
