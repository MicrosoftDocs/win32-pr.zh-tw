# <a name="systemitemparticipants"></a>System. ItemParticipants

和參與專案相關聯之人員的一般清單。 例如，這是電子郵件訊息寄件者清單、副本清單和寄件者的組合。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7、Windows Vista 版本

```
propertyDescription
   name = System.ItemParticipants
   shellPKey = PKEY_ItemParticipants
   formatID = D4D0AA16-9948-41A4-AA85-D97FF9646993
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
