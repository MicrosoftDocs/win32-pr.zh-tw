# <a name="systemdeviceswifidirectisconnected"></a>WiFiDirect. IsConnected

指出 Wi-Fi Direct 裝置的線上狀態

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Devices.WiFiDirect.IsConnected
   shellPKey = PKEY_Devices_WiFiDirect_IsConnected
   formatID = 1506935D-E3E7-450F-8637-82233EBE5F6E
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
