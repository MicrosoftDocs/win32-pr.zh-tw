# <a name="systemdevicescategoryids"></a>System.Devices.CategoryIds

表示實際的原始類別

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Devices.CategoryIds
   shellPKey = PKEY_Devices_CategoryIds
   formatID = 78C34FC8-104A-4ACA-9EA4-524D52996E57
   propID = 90
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
