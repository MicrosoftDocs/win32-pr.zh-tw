# <a name="systemdevicesaepservicebluetoothtargetdevice"></a>AepService。藍牙。TargetDevice

查詢的藍牙父裝置。 未快取的查詢所需。

## <a name="windows-10-version-1703-windows-10-version-1607"></a>Windows 10，版本1703，Windows 10，版本1607

```
propertyDescription
   name = System.Devices.AepService.Bluetooth.TargetDevice
   shellPKey = PKEY_Devices_AepService_Bluetooth_TargetDevice
   formatID = 9744311E-7951-4B2E-B6F0-ECB293CAC119
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt64
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
