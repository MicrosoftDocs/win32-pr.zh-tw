# <a name="systemthumbnailstream"></a>System. ThumbnailStream

以 VT 資料流程格式表示的縮圖 \_ ，windows gdi + 和 windows 編解碼器（例如 .jpg 和 .png）支援的資料。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版、Windows 8。1

```
propertyDescription
   name = System.ThumbnailStream
   shellPKey = PKEY_ThumbnailStream
   formatID = F29F85E0-4FF9-1068-AB91-08002B27B3D9
   propID = 27
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Stream
      IsInnate = false
```

## <a name="windows-8-windows-7-windows-vista"></a>Windows 8、Windows 7、Windows Vista

```
propertyDescription
   name = System.ThumbnailStream
   shellPKey = PKEY_ThumbnailStream
   formatID = F29F85E0-4FF9-1068-AB91-08002B27B3D9
   propID = 27
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Stream
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
