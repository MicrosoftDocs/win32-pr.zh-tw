# <a name="systemcontactcompanymaintelephone"></a>CompanyMainTelephone

指出連絡人公司的主要電話號碼。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7、Windows Vista 版本

```
propertyDescription
   name = System.Contact.CompanyMainTelephone
   shellPKey = PKEY_Contact_CompanyMainTelephone
   formatID = 8589E481-6040-473D-B171-7FA89C2708ED
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
