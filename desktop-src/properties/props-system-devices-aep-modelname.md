# <a name="systemdevicesaepmodelname"></a>System.Devices.Aep.ModelName

裝置關聯端點的模型名稱

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Devices.Aep.ModelName
   shellPKey = PKEY_Devices_Aep_ModelName
   formatID = A35996AB-11CF-4935-8B61-A6761081ECDF
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
