# <a name="systemdevicesaepcategory"></a>System.Devices.Aep.Category

裝置所屬的類別。 例如，印表機、相機等等。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Devices.Aep.Category
   shellPKey = PKEY_Devices_Aep_Category
   formatID = A35996AB-11CF-4935-8B61-A6761081ECDF
   propID = 17
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
