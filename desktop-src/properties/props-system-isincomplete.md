# <a name="systemisincomplete"></a>System. IsIncomplete

識別訊息是否已完全接收。 此值可用於某些錯誤情況。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.IsIncomplete
   shellPKey = PKEY_IsIncomplete
   formatID = 346C8BD1-2E6A-4C45-89A4-61B78E8E700F
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Boolean
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.IsIncomplete
   shellPKey = PKEY_IsIncomplete
   formatID = 346C8BD1-2E6A-4C45-89A4-61B78E8E700F
   propID = 100
   SearchInfo
      IsColumn = true
   typeInfo
      type = Boolean
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
