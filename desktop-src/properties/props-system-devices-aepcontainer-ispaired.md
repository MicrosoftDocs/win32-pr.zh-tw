# <a name="systemdevicesaepcontainerispaired"></a>System.Devices.AepContainer.IsPaired

其中一個子裝置關聯端點是否與系統配對。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Devices.AepContainer.IsPaired
   shellPKey = PKEY_Devices_AepContainer_IsPaired
   formatID = 0BBA1EDE-7566-4F47-90EC-25FC567CED2A
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
