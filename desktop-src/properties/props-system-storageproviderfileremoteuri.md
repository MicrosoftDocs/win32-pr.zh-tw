# <a name="systemstorageproviderfileremoteuri"></a>System. StorageProviderFileRemoteUri

此檔案的儲存提供者遠端 Uri。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版、Windows 8。1

```
propertyDescription
   name = System.StorageProviderFileRemoteUri
   shellPKey = PKEY_StorageProviderFileRemoteUri
   formatID = FCEFF153-E839-4CF3-A9E7-EA22832094B8
   propID = 112
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

PKEY 值定義于 Propkey 中。
