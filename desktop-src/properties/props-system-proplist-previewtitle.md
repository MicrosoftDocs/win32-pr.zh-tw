# <a name="systemproplistpreviewtitle"></a>PropList. PreviewTitle

要在預覽窗格標題區段中顯示的一或兩個屬性。 選用的第二個屬性會顯示為子標題。 在 "PreviewTitle" 的 regvalue 下註冊。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、version 1507、Windows 8.1、Windows 8、Windows 7、Windows Vista

```
propertyDescription
   name = System.PropList.PreviewTitle
   shellPKey = PKEY_PropList_PreviewTitle
   formatID = C9944A21-A406-48FE-8225-AEC7E24C211B
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
