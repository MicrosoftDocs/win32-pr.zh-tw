# <a name="systemdevicesaepbluetoothleisconnectable"></a>Aep。藍牙。Le. IsConnectable

藍牙 LE 裝置目前是否正在通告可連接的廣告。

## <a name="windows-10-version-1703"></a>Windows 10 (版本 1703)

```
propertyDescription
   name = System.Devices.Aep.Bluetooth.Le.IsConnectable
   shellPKey = PKEY_Devices_Aep_Bluetooth_Le_IsConnectable
   formatID = 995EF0B0-7EB3-4A8B-B9CE-068BB3F4AF69
   propID = 8
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
