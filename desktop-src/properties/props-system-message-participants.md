# <a name="systemmessageparticipants"></a>系統。參與者

參與通訊的參與者。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版、Windows 8。1

```
propertyDescription
   name = System.Message.Participants
   shellPKey = PKEY_Message_Participants
   formatID = 1A9BA605-8E7C-4D11-AD7D-A50ADA18BA1B
   propID = 2
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Multivalue String
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
