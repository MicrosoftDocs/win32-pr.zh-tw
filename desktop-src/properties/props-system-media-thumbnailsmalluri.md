# <a name="systemmediathumbnailsmalluri"></a>ThumbnailSmallUri

媒體專案的小型縮圖表示的 Uri。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Media.ThumbnailSmallUri
   shellPKey = PKEY_Media_ThumbnailSmallUri
   formatID = 64440492-4C8B-11D1-8B70-080036B11A03
   propID = 50
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
