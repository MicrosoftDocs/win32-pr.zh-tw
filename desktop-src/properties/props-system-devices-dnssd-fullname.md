# <a name="systemdevicesdnssdfullname"></a>Dnssd FullName

完整的 DNS-SD service 實例名稱，包括實例、服務和網域。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Devices.Dnssd.FullName
   shellPKey = PKEY_Devices_Dnssd_FullName
   formatID = BF79C0AB-BB74-4CEE-B070-470B5AE202EA
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
