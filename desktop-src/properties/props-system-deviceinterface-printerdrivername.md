# <a name="systemdeviceinterfaceprinterdrivername"></a>DeviceInterface. PrinterDriverName

印表機驅動程式檔案的名稱。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.DeviceInterface.PrinterDriverName
   shellPKey = PKEY_DeviceInterface_PrinterDriverName
   formatID = AFC47170-14F5-498C-8F30-B0D19BE449C6
   propID = 11
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
