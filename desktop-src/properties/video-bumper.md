# <a name="video-windows-property-system"></a>影片 (Windows 屬性系統) 

## <a name="in-this-section"></a>本節內容



| 主題                                                                                                | 描述                                                                                                                                |
|------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------|
| [系統 Video. 壓縮](./props-system-video-compression.md)<br/>                     | 指定影片壓縮格式。<br/>                                                                                         |
| [System.servicemodel](./props-system-video-director.md)<br/>                           | 表示導向影片的人員。 <br/>                                                                                   |
| [EncodingBitrate](./props-system-video-encodingbitrate.md)<br/>             | 表示影片串流的資料速率（以「每秒位數」為單位）。 "DataRate".<br/>                                                  |
| [FourCC](./props-system-video-fourcc.md)<br/>                               | 指定影片串流的 FOURCC 碼。<br/>                                                                                 |
| [FrameHeight](./props-system-video-frameheight.md)<br/>                     | 指出影片資料流程的框架高度。<br/>                                                                                |
| [System.string](./props-system-video-framerate.md)<br/>                         | 表示影片串流的畫面播放速率，以每1000秒的畫面格速率表示。<br/>                                                      |
| [FrameWidth](./props-system-video-framewidth.md)<br/>                       | 指出影片資料流程的框架寬度。<br/>                                                                                 |
| [HorizontalAspectRatio](./props-system-video-horizontalaspectratio.md)<br/> | 表示圖元外觀比例的水準部分。 XX： YY 的 X 部分。 例如，10是10:11 的 X 部分。<br/>  |
| [IsSpherical](https://www.bing.com/search?q=System.Video.IsSpherical)<br/>                           | 指出媒體檔案是否有球形視頻串流。<br/>                                                                  |
| [IsStereo](props-system-video-isstereo.md)<br/>                                  |                                                                                                                                            |
| [系統。方向](props-system-video-orientation.md)<br/>                            | 這是影片方向（以度為單位）。<br/>                                                                                       |
| [SampleSize](./props-system-video-samplesize.md)<br/>                       | 表示影片串流的樣本大小（以位為單位）。 "SampleSize"。<br/>                                                           |
| [StreamName](./props-system-video-streamname.md)<br/>                       | 指出影片資料流程的名稱。 "StreamName"。<br/>                                                                          |
| [StreamNumber](./props-system-video-streamnumber.md)<br/>                   | 指出所播放資料流程的序數。<br/>                                                                        |
| [TotalBitrate](./props-system-video-totalbitrate.md)<br/>                   | 表示所有影片和音訊串流的總數據速率（以「每秒位數」為單位）。<br/>                                             |
| [TranscodedForSync](./props-system-video-transcodedforsync.md)<br/>         | 表示外觀比例的垂直部分。<br/>                                                                             |
| [VerticalAspectRatio](./props-system-video-verticalaspectratio.md)<br/>     | 表示圖元外觀比例的水準部分。 XX： YY 的 Y 部分。 例如，11是10:11 的 Y 部分。<br/> |
