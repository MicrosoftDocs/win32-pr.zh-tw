# <a name="systemdevicesaepserviceprotocolid"></a>System.Devices.AepService.ProtocolId

探索到此裝置關聯端點服務的通訊協定身分識別

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Devices.AepService.ProtocolId
   shellPKey = PKEY_Devices_AepService_ProtocolId
   formatID = C9C141A9-1B4C-4F17-A9D1-F298538CADB8
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Guid
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
