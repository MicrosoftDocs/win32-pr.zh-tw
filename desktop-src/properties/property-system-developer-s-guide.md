# <a name="property-system-developers-guide"></a>屬性系統開發人員指南

在 Windows Vista 和更新版本中，中繼資料成為組織專案的方法，例如檔案、電子郵件或連絡人。 為了讓系統能夠根據其中繼資料來搜尋專案，以及使用者可以讀取或寫入中繼資料的位置，Windows Vista 引進了新的屬性系統。 這個系統中的中繼資料是由一組可擴充的屬性來表示。 隨著 Windows Vista 中的屬性，可將相片、音樂、檔、郵件、連絡人和檔案等專案的細節抽象化， (isv) 如果沒有任何現有的屬性符合需求，現在就可以將自己的屬性引進平臺。

本節說明 Windows 屬性系統內的下列開發案例：

-   [建立自訂屬性](./building-property-handlers-property-schemas.md)
-   [執行屬性處理常式](./building-property-handlers.md)
