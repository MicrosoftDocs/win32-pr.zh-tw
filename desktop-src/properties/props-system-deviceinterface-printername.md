# <a name="systemdeviceinterfaceprintername"></a>DeviceInterface. PrinterName

印表機的名稱。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.DeviceInterface.PrinterName
   shellPKey = PKEY_DeviceInterface_PrinterName
   formatID = 0A7B84EF-0C27-463F-84EF-06C5070001BE
   propID = 10
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
