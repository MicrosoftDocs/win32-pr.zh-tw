# <a name="systemmediadateencoded"></a>DateEncoded

代表檔案的編碼日期和時間。 DateTime 以 UTC (格式，而非檔案系統) 。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.Media.DateEncoded
   shellPKey = PKEY_Media_DateEncoded
   formatID = 2E4B640D-5019-46D8-8881-55414CC5CAA0
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = DateTime
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.Media.DateEncoded
   shellPKey = PKEY_Media_DateEncoded
   formatID = 2E4B640D-5019-46D8-8881-55414CC5CAA0
   propID = 100
   SearchInfo
      IsColumn = true
   typeInfo
      type = DateTime
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
