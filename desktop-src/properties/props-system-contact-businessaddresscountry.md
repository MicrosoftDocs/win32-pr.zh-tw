# <a name="systemcontactbusinessaddresscountry"></a>BusinessAddressCountry

連絡人的公司位址國家/地區。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7、Windows Vista 版本

```
propertyDescription
   name = System.Contact.BusinessAddressCountry
   shellPKey = PKEY_Contact_BusinessAddressCountry
   formatID = B0B87314-FCF6-4FEB-8DFF-A50DA6AF561C
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
