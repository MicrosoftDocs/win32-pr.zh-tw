# <a name="systemappzoneidentifier"></a>System. AppZoneIdentifier

應用程式容器的標記。 由檔案的最後寫入器所決定的區域識別碼。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.AppZoneIdentifier
   shellPKey = PKEY_AppZoneIdentifier
   formatID = 502CFEAB-47EB-459C-B960-E6D8728F7701
   propID = 102
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt32
      IsInnate = false
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
