# <a name="systemmediaframecount"></a>FrameCount

指出影像的框架計數。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.Media.FrameCount
   shellPKey = PKEY_Media_FrameCount
   formatID = 6444048F-4C8B-11D1-8B70-080036B11A03
   propID = 12
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt32
      IsInnate = true
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.Media.FrameCount
   shellPKey = PKEY_Media_FrameCount
   formatID = 6444048F-4C8B-11D1-8B70-080036B11A03
   propID = 12
   SearchInfo
      IsColumn = true
   typeInfo
      type = UInt32
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
