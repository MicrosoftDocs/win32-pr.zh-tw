# <a name="systemsearchgathertime"></a>GatherTime

日期時間值，表示 Windows Search 收集程式最後將此檔的屬性推送至 Windows Search 收集程式外掛程式的時間。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.Search.GatherTime
   shellPKey = PKEY_Search_GatherTime
   formatID = 0B63E350-9CCC-11D0-BCDB-00805FCCCE04
   propID = 8
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.Search.GatherTime
   shellPKey = PKEY_Search_GatherTime
   formatID = 0B63E350-9CCC-11D0-BCDB-00805FCCCE04
   propID = 8
   SearchInfo
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
