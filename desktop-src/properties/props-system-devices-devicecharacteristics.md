# <a name="systemdevicesdevicecharacteristics"></a>DeviceCharacteristics

裝置特性。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、version 1507、Windows 8.1、Windows 8

```
propertyDescription
   name = System.Devices.DeviceCharacteristics
   shellPKey = PKEY_Devices_DeviceCharacteristics
   formatID = A45C254E-DF1C-4EFD-8020-67D146A850E0
   propID = 29
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt32
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
