# <a name="systemcalendarremindertime"></a>ReminderTime

儲存使用者選擇要提醒事件的時間。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.Calendar.ReminderTime
   shellPKey = PKEY_Calendar_ReminderTime
   formatID = 72FC5BA4-24F9-4011-9F3F-ADD27AFAD818
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = DateTime
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.Calendar.ReminderTime
   shellPKey = PKEY_Calendar_ReminderTime
   formatID = 72FC5BA4-24F9-4011-9F3F-ADD27AFAD818
   propID = 100
   SearchInfo
      IsColumn = true
   typeInfo
      type = DateTime
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
