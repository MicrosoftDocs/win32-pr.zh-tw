# <a name="systemidentityproviderid"></a>System.servicemodel

這個屬性會指定提供者識別碼。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.Identity.ProviderID
   shellPKey = PKEY_Identity_ProviderID
   formatID = 74A7DE49-FA11-4D3D-A006-DB7E08675916
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Guid
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
