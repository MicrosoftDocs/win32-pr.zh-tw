# <a name="systemphotomaxaperture"></a>MaxAperture

相機錄製的鏡頭最大光圈，取自交換影像檔案 (EXIF) 資訊。 這個屬性是從 [MaxApertureNumerator](./props-system-photo-maxaperturenumerator.md) 和 [MaxApertureDenominator](./props-system-photo-maxaperturedenominator.md)計算而來。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、version 1507、Windows 8.1、Windows 8、Windows 7、Windows Vista

```
propertyDescription
   name = System.Photo.MaxAperture
   shellPKey = PKEY_Photo_MaxAperture
   formatID = 08F6D7C2-E3F2-44FC-AF1E-5AA5C81A2D3E
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
