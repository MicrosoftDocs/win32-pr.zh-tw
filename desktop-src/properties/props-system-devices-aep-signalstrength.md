# <a name="systemdevicesaepsignalstrength"></a>System.Devices.Aep.SignalStrength

裝置的信號強度。 僅適用于一些通訊協定。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Devices.Aep.SignalStrength
   shellPKey = PKEY_Devices_Aep_SignalStrength
   formatID = A35996AB-11CF-4935-8B61-A6761081ECDF
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Int32
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
