# <a name="systemdevicesaudiodevicerawprocessingsupported"></a>System.Devices.AudioDevice.RawProcessingSupported

音訊裝置的原始處理模式支援。 如果 VARIANT \_ TRUE，表示裝置支援原始處理模式。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版、Windows 8。1

```
propertyDescription
   name = System.Devices.AudioDevice.RawProcessingSupported
   shellPKey = PKEY_Devices_AudioDevice_RawProcessingSupported
   formatID = 8943B373-388C-4395-B557-BC6DBAFFAFDB
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
