# <a name="systemgpsspeed"></a>GPS. 速度

指出 GPS 接收器移動的速度。 從 PKEY \_ gps \_ SPEEDNUMERATOR 和 PKEY \_ gps SpeedDenominator 計算 \_ 。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7、Windows Vista 版本

```
propertyDescription
   name = System.GPS.Speed
   shellPKey = PKEY_GPS_Speed
   formatID = DA5D0862-6E76-4E1B-BABD-70021BD25494
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
