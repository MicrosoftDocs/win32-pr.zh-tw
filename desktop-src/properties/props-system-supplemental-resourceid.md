# <a name="systemsupplementalresourceid"></a>系統補充. ResourceId

包含遠端同步服務上專案的識別碼。 用來比較系統上的檔案與雲端中可用的檔案。

## <a name="windows-10-version-1703"></a>Windows 10 (版本 1703)

```
propertyDescription
   name = System.Supplemental.ResourceId
   shellPKey = PKEY_Supplemental_ResourceId
   formatID = 0C73B141-39D6-4653-A683-CAB291EAF95B
   propID = 3
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
