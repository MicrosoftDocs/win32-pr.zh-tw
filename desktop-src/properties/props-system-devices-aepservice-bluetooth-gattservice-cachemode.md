# <a name="systemdevicesaepservicebluetoothgattservicecachemode"></a>AepService. GattService. CacheMode

設定查詢的藍牙 Gatt 快取模式。

## <a name="windows-10-version-1511"></a>Windows 10 (版本 1511)

```
propertyDescription
   name = System.Devices.AepService.Bluetooth.GattService.CacheMode
   shellPKey = PKEY_Devices_AepService_Bluetooth_GattService_CacheMode
   formatID = 9744311E-7951-4B2E-B6F0-ECB293CAC119
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
