# <a name="systemdeviceswifidirectdeviceaddress"></a>WiFiDirect. DeviceAddress

Wi-Fi 直接裝置位址

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Devices.WiFiDirect.DeviceAddress
   shellPKey = PKEY_Devices_WiFiDirect_DeviceAddress
   formatID = 1506935D-E3E7-450F-8637-82233EBE5F6E
   propID = 13
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Buffer
      IsInnate = true
```
