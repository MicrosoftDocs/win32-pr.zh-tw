# <a name="systemdevicesaepcontainersupportsimages"></a>System.Devices.AepContainer.SupportsImages

指出裝置關聯端點容器是否支援影像轉換

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Devices.AepContainer.SupportsImages
   shellPKey = PKEY_Devices_AepContainer_SupportsImages
   formatID = 6AF55D45-38DB-4495-ACB0-D4728A3B8314
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
