# <a name="systemdevicesaudiodevicespeechprocessingsupported"></a>System.Devices.AudioDevice.SpeechProcessingSupported

音訊裝置的語音模式支援。 如果 VARIANT \_ TRUE，表示裝置支援語音模式。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Devices.AudioDevice.SpeechProcessingSupported
   shellPKey = PKEY_Devices_AudioDevice_SpeechProcessingSupported
   formatID = FB1DE864-E06D-47F4-82A6-8A0AEF44493C
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
