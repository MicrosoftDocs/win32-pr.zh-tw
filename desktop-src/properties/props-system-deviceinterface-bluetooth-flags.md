# <a name="systemdeviceinterfacebluetoothflags"></a>DeviceInterface。

藍牙裝置旗標。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版

```
propertyDescription
   name = System.DeviceInterface.Bluetooth.Flags
   shellPKey = PKEY_DeviceInterface_Bluetooth_Flags
   formatID = 2BD67D8B-8BEB-48D5-87E0-6CDA3428040A
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt32
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
