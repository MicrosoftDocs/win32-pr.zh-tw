# <a name="systemcreatoropenwithuioptions"></a>System. CreatorOpenWithUIOptions

提供的選項會影響 UI 控制項的行為，這些控制項會使用 CreatorAppId 中指定的應用程式啟動檔案。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.CreatorOpenWithUIOptions
   shellPKey = PKEY_CreatorOpenWithUIOptions
   formatID = C2EA046E-033C-4E91-BD5B-D4942F6BBE49
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt32
      IsInnate = false
      EnumeratedList
         UseValueForDefault = True
         enum
            name = Hidden
            value = 0
            text = Hidden
            defineToken = CREATOROPENWITHUIOPTION_HIDDEN
         enum
            name = Visible
            value = 1
            text = Visible
            defineToken = CREATOROPENWITHUIOPTION_VISIBLE
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
