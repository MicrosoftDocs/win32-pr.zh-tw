# <a name="systemvideoisspherical"></a>IsSpherical

指出媒體檔案是否有球形視頻串流。

## <a name="windows-10-version-1703"></a>Windows 10 (版本 1703)

```
propertyDescription
   name = System.Video.IsSpherical
   shellPKey = PKEY_Video_IsSpherical
   formatID = 64440491-4C8B-11D1-8B70-080036B11A03
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
