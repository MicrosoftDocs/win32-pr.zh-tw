# <a name="image"></a>Image

## <a name="in-this-section"></a>本節內容



| 主題                                                                                                                        | 描述                                                                                                                                                                                                                               |
|------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [BitDepth](./props-system-image-bitdepth.md)<br/>                                                   | 表示在影像的每個圖元中使用多少位數。  (通常是8、16、24或 32) 。<br/>                                                                                                                                       |
| [ColorSpace](./props-system-image-colorspace.md)<br/>                                               | 內嵌于影像中的 colorspace。 取自交換影像檔案 (EXIF) 資訊。<br/>                                                                                                                               |
| [CompressedBitsPerPixel](./props-system-image-compressedbitsperpixel.md)<br/>                       | 指出影像壓縮層級。 <br/>                                                                                                                                                                                        |
| [CompressedBitsPerPixelDenominator](./props-system-image-compressedbitsperpixeldenominator.md)<br/> | PKEY \_ 影像 CompressedBitsPerPixel 的分母 \_ 。<br/>                                                                                                                                                                        |
| [CompressedBitsPerPixelNumerator](./props-system-image-compressedbitsperpixelnumerator.md)<br/>     | PKEY \_ 影像 CompressedBitsPerPixel 的分子 \_ 。<br/>                                                                                                                                                                          |
| [系統映射壓縮](./props-system-image-compression.md)<br/>                                             | 用來壓縮影像的演算法。<br/>                                                                                                                                                                                      |
| [CompressionText](./props-system-image-compressiontext.md)<br/>                                     | 使用者易記的系統壓縮形式。 不打算以程式設計方式剖析。<br/>                                                                                                                                |
| [系統維度](./props-system-image-dimensions.md)<br/>                                               | 字串格式為的影像維度 `horizontal pixels x vertical pixels` 。 例如： `3080x2100` 。<br/>                                                                                                                      |
| [HorizontalResolution](./props-system-image-horizontalresolution.md)<br/>                           | 指出影像寬度中每個解析度單位的圖元數。<br/>                                                                                                                                                         |
| [HorizontalSize](./props-system-image-horizontalsize.md)<br/>                                       | 影像的水準大小（以圖元為單位）。<br/>                                                                                                                                                                                   |
| [ImageID](./props-system-image-imageid.md)<br/>                                                     |                                                                                                                                                                                                                                           |
| [ResolutionUnit](./props-system-image-resolutionunit.md)<br/>                                       | 指出解決單位。 用於具有非方形外觀比例，但沒有有意義的絕對維度的影像。 1 = 沒有任何絕對測量單位。 2 = 英寸。 3 = 公分。 預設值為 2 (英寸) 。<br/> |
| [VerticalResolution](./props-system-image-verticalresolution.md)<br/>                               | 指出影像高度中每個解析度單位的圖元數。<br/>                                                                                                                                                        |
| [VerticalSize](./props-system-image-verticalsize.md)<br/>                                           | 影像的垂直大小（以圖元為單位）。<br/>                                                                                                                                                                                     |
