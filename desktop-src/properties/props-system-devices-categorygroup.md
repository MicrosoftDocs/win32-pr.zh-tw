# <a name="systemdevicescategorygroup"></a>CategoryGroup

裝置類別的複數。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、version 1507、Windows 8.1、Windows 8

```
propertyDescription
   name = System.Devices.CategoryGroup
   shellPKey = PKEY_Devices_CategoryGroup
   formatID = 78C34FC8-104A-4ACA-9EA4-524D52996E57
   propID = 94
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="windows-7"></a>Windows 7

```
propertyDescription
   name = System.Devices.CategoryGroup
   shellPKey = PKEY_Devices_CategoryGroup_Desc
   formatID = 78C34FC8-104A-4ACA-9EA4-524D52996E57
   propID = 94
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

PKEY 值定義于 Propkey 中。
