# <a name="systemmusicalbumid"></a>AlbumID

這個屬性會區分具有不同演出者之相同標題的專輯。 它是 AlbumArtist 和 AlbumTitle 的串連方式。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.Music.AlbumID
   shellPKey = PKEY_Music_AlbumID
   formatID = 56A3372E-CE9C-11D2-9F0E-006097C686F6
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
