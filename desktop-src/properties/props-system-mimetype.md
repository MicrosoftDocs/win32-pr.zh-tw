# <a name="systemmimetype"></a>System. MIMEType

MIME 類型。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、version 1507、Windows 8.1、Windows 8、Windows 7、Windows Vista

```
propertyDescription
   name = System.MIMEType
   shellPKey = PKEY_MIMEType
   formatID = 0B63E350-9CCC-11D0-BCDB-00805FCCCE04
   propID = 5
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。

例如，一個 .EML 檔案的 [MIMEType]() 值為 `message/rfc822` 。

對於與多個 MIME 類型相容的媒體檔案，這個屬性可能會有一個偏差來指出適用于數位客廳網路聯盟 (DLNA) 的 MIME 類型。
