# <a name="systemitemdate"></a>System. ItemDate

專案的主要日期。 例如，在相片的情況下，此屬性會對應至 [DateTaken](./props-system-photo-datetaken.md)。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.ItemDate
   shellPKey = PKEY_ItemDate
   formatID = F7DB74B4-4287-4103-AFBA-F1B13DCD75CF
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.ItemDate
   shellPKey = PKEY_ItemDate
   formatID = F7DB74B4-4287-4103-AFBA-F1B13DCD75CF
   propID = 100
   SearchInfo
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
