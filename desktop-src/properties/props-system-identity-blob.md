# <a name="systemidentityblob"></a>System.string

用來匯入和匯出身分識別的 Blob。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.Identity.Blob
   shellPKey = PKEY_Identity_Blob
   formatID = 8C3B93A4-BAED-1A83-9A32-102EE313F6EB
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Blob
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
