# <a name="systemdevicesaepbluetoothcodservicestelephony"></a>Aep....。

裝置服務電話語音的藍牙類別。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版

```
propertyDescription
   name = System.Devices.Aep.Bluetooth.Cod.Services.Telephony
   shellPKey = PKEY_Devices_Aep_Bluetooth_Cod_Services_Telephony
   formatID = 5FBD34CD-561A-412E-BA98-478A6B0FEF1D
   propID = 11
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
