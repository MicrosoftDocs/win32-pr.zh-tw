# <a name="systemdevicesaepcontainersupportspositioning"></a>AepContainer. SupportsPositioning

指出裝置關聯端點容器是否支援定位

## <a name="windows-10-version-1703-windows-10-version-1607"></a>Windows 10，版本1703，Windows 10，版本1607

```
propertyDescription
   name = System.Devices.AepContainer.SupportsPositioning
   shellPKey = PKEY_Devices_AepContainer_SupportsPositioning
   formatID = 6AF55D45-38DB-4495-ACB0-D4728A3B8314
   propID = 8
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
