# <a name="systemisattachment"></a>System. IsAttachment

識別專案是否為附件。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.IsAttachment
   shellPKey = PKEY_IsAttachment
   formatID = F23F425C-71A1-4FA8-922F-678EA4A60408
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.IsAttachment
   shellPKey = PKEY_IsAttachment
   formatID = F23F425C-71A1-4FA8-922F-678EA4A60408
   propID = 100
   SearchInfo
      IsColumn = true
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
