# <a name="systemsecurityencryptionownersdisplay"></a>EncryptionOwnersDisplay

[檔案擁有權] 

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版

```
propertyDescription
   name = System.Security.EncryptionOwnersDisplay
   shellPKey = PKEY_Security_EncryptionOwnersDisplay
   formatID = DE621B8F-E125-43A3-A32D-5665446D632A
   propID = 25
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Multivalue String
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
