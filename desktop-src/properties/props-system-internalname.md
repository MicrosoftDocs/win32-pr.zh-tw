# <a name="systeminternalname"></a>System. InternalName

儲存在該檔案內資源區段中的 .exe 或 .dll 檔案的名稱。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、version 1507、Windows 8.1、Windows 8、Windows 7、Windows Vista

```
propertyDescription
   name = System.InternalName
   shellPKey = PKEY_InternalName
   formatID = 0CEF7D53-FA64-11D1-A203-0000F81FEDEE
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。

在過去，這項資訊通常會顯示在 Windows XPÂ **版本** 的屬性工作表頁面中。 在 Windows Vista 中，資訊已移至 [**詳細資料**] 索引標籤。不過，在 Service Pack 1 (SP1) 之前，Windows Vista 中並不會公開這個特定屬性。
