# <a name="systemsyncprogresspercentage"></a>System.componentmodel.progresschangedeventargs.progresspercentage

介於0與100之間的整數值，表示已完成的百分比。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.Sync.ProgressPercentage
   shellPKey = PKEY_Sync_ProgressPercentage
   formatID = 7BD5533E-AF15-44DB-B8C8-BD6624E1D032
   propID = 23
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt32
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
