# <a name="systemphototagviewaggregate"></a>TagViewAggregate

用於建立視圖的標記型屬性唯讀匯總。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.Photo.TagViewAggregate
   shellPKey = PKEY_Photo_TagViewAggregate
   formatID = B812F15D-C2D8-4BBF-BACD-79744346113F
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
