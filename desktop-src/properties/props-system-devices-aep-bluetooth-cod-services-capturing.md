# <a name="systemdevicesaepbluetoothcodservicescapturing"></a>Aep。藍牙。. 服務. 捕獲

裝置服務捕獲藍牙類別。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版

```
propertyDescription
   name = System.Devices.Aep.Bluetooth.Cod.Services.Capturing
   shellPKey = PKEY_Devices_Aep_Bluetooth_Cod_Services_Capturing
   formatID = 5FBD34CD-561A-412E-BA98-478A6B0FEF1D
   propID = 8
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

PKEY 值定義于 Propkey 中。
