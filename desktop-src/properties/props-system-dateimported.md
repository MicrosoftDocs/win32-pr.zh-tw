# <a name="systemdateimported"></a>System. DateImported

檔案匯入至私用應用程式資料庫的日期和時間。 例如，當相片匯入相片資料庫時，可以使用這個屬性。 這個屬性與 [DateAcquired](./props-system-dateacquired.md)不同。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.DateImported
   shellPKey = PKEY_DateImported
   formatID = 14B81DA1-0135-4D31-96D9-6CBFC9671A99
   propID = 18258
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = DateTime
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.DateImported
   shellPKey = PKEY_DateImported
   formatID = 14B81DA1-0135-4D31-96D9-6CBFC9671A99
   propID = 18258
   SearchInfo
      IsColumn = true
   typeInfo
      type = DateTime
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
