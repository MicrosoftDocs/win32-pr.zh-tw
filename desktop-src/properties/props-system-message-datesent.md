# <a name="systemmessagedatesent"></a>DateSent

傳送通訊的日期和時間。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.Message.DateSent
   shellPKey = PKEY_Message_DateSent
   formatID = E3E0584C-B788-4A5A-BB20-7F5A44C9ACDD
   propID = 19
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.Message.DateSent
   shellPKey = PKEY_Message_DateSent
   formatID = E3E0584C-B788-4A5A-BB20-7F5A44C9ACDD
   propID = 19
   SearchInfo
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
