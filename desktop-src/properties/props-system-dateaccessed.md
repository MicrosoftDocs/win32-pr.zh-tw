# <a name="systemdateaccessed"></a>System. DateAccessed

表示上次存取專案的時間。 索引服務易記名稱為 "access"。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.DateAccessed
   shellPKey = PKEY_DateAccessed
   formatID = B725F130-47EF-101A-A5F1-02608C9EEBAC
   propID = 16
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.DateAccessed
   shellPKey = PKEY_DateAccessed
   formatID = B725F130-47EF-101A-A5F1-02608C9EEBAC
   propID = 16
   SearchInfo
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
