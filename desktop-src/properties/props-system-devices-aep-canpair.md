# <a name="systemdevicesaepcanpair"></a>System.Devices.Aep.CanPair

裝置關聯端點是否可與系統配對

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Devices.Aep.CanPair
   shellPKey = PKEY_Devices_Aep_CanPair
   formatID = E7C3FB29-CAA7-4F47-8C8B-BE59B330D4C5
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
