# <a name="systemcontactaccountpicturesmall"></a>AccountPictureSmall

這是包含轉譯連絡人小型帳戶圖片所需資料的資料流程。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、version 1507、Windows 8.1、Windows 8

```
propertyDescription
   name = System.Contact.AccountPictureSmall
   shellPKey = PKEY_Contact_AccountPictureSmall
   formatID = 0B8BB018-2725-4B44-92BA-7933AEB2DDE7
   propID = 4
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Stream
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
