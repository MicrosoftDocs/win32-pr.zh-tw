# <a name="systemdevicesaepispaired"></a>System.Devices.Aep.IsPaired

裝置關聯端點是否與系統配對

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Devices.Aep.IsPaired
   shellPKey = PKEY_Devices_Aep_IsPaired
   formatID = A35996AB-11CF-4935-8B61-A6761081ECDF
   propID = 16
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
