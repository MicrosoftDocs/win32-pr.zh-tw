# <a name="systemmusicdiscnumber"></a>DiscNumber

## <a name="windows-10-version-1703-windows-10-version-1607"></a>Windows 10，版本1703，Windows 10，版本1607

```
propertyDescription
   name = System.Music.DiscNumber
   shellPKey = PKEY_Music_DiscNumber
   formatID = 6AFE7437-9BCD-49C7-80FE-4A5C65FA5874
   propID = 104
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt32
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
