# <a name="systemcreatorappid"></a>System. CreatorAppId

建立此檔案之應用程式的 AppId。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.CreatorAppId
   shellPKey = PKEY_CreatorAppId
   formatID = C2EA046E-033C-4E91-BD5B-D4942F6BBE49
   propID = 2
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = false
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
