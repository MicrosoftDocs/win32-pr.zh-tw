# <a name="systemsearchextendedproperties"></a>ExtendedProperties

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.Search.ExtendedProperties
   shellPKey = PKEY_Search_ExtendedProperties
   formatID = 7B03B546-FA4F-4A52-A2FE-03D5311E5865
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Blob
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
