# <a name="systemdevicesaepbluetoothcodminor"></a>Aep...。

裝置次要程式碼的藍牙類別。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版

```
propertyDescription
   name = System.Devices.Aep.Bluetooth.Cod.Minor
   shellPKey = PKEY_Devices_Aep_Bluetooth_Cod_Minor
   formatID = 5FBD34CD-561A-412E-BA98-478A6B0FEF1D
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
