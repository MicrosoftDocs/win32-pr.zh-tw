# <a name="systemsourcepackagefamilyname"></a>System. SourcePackageFamilyName

儲存體專案實例所產生之應用程式的套件系列名稱。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.SourcePackageFamilyName
   shellPKey = PKEY_SourcePackageFamilyName
   formatID = FFAE9DB7-1C8D-43FF-818C-84403AA3732D
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
