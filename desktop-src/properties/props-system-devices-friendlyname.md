# <a name="systemdevicesfriendlyname"></a>System.servicemodel

裝置的好記的名稱。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.Devices.FriendlyName
   shellPKey = PKEY_Devices_FriendlyName
   formatID = 656A3BB3-ECC0-43FD-8477-4AE0404A96CD
   propID = 12288
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
