# <a name="systemfulltext"></a>System. 全文檢索

這個屬性是用來指定應該在搜尋之資料來源)  (的所有有效屬性中盡可能廣泛地套用的搜尋字詞。 它不應該從資料來源發出。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.FullText
   shellPKey = PKEY_FullText
   formatID = 1E3EE840-BC2B-476C-8237-2ACD1A839B22
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
