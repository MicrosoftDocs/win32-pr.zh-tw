# <a name="systemdevicesaepdeviceaddress"></a>System.Devices.Aep.DeviceAddress

以裝置關聯端點的通訊協定為基礎的位址。 IP 裝置的 IP 位址、藍牙裝置的藍牙位址等等。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Devices.Aep.DeviceAddress
   shellPKey = PKEY_Devices_Aep_DeviceAddress
   formatID = A35996AB-11CF-4935-8B61-A6761081ECDF
   propID = 12
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
