# <a name="systemdevicesdiscoverymethod"></a>DiscoveryMethod

表示探索到的裝置的傳輸或實體連接。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.Devices.DiscoveryMethod
   shellPKey = PKEY_Devices_DiscoveryMethod
   formatID = 78C34FC8-104A-4ACA-9EA4-524D52996E57
   propID = 52
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
