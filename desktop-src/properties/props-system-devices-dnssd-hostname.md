# <a name="systemdevicesdnssdhostname"></a>Dnssd。主機名稱

裝置的 DNS 名稱正在裝載服務。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Devices.Dnssd.HostName
   shellPKey = PKEY_Devices_Dnssd_HostName
   formatID = BF79C0AB-BB74-4CEE-B070-470B5AE202EA
   propID = 7
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
