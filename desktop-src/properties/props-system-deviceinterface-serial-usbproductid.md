# <a name="systemdeviceinterfaceserialusbproductid"></a>DeviceInterface. UsbProductId

串列裝置 USB 產品識別碼

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.DeviceInterface.Serial.UsbProductId
   shellPKey = PKEY_DeviceInterface_Serial_UsbProductId
   formatID = 4C6BF15C-4C03-4AAC-91F5-64C0F852BCF4
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
