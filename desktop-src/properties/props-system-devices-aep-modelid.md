# <a name="systemdevicesaepmodelid"></a>System.Devices.Aep.ModelId

裝置關聯端點的模型識別碼

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Devices.Aep.ModelId
   shellPKey = PKEY_Devices_Aep_ModelId
   formatID = A35996AB-11CF-4935-8B61-A6761081ECDF
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Guid
      IsInnate = true
```
