# <a name="systemlayoutpatterncontentviewmodeforbrowse"></a>LayoutPattern. ContentViewModeForBrowse

識別內容視圖模式應該在流覽內容中套用此專案的版面配置模式。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.LayoutPattern.ContentViewModeForBrowse
   shellPKey = PKEY_LayoutPattern_ContentViewModeForBrowse
   formatID = C9944A21-A406-48FE-8225-AEC7E24C211B
   propID = 500
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      EnumeratedList
         UseValueForDefault = True
         enum
            name = Alpha
            value = alpha
            text
            defineToken = LAYOUTPATTERN_CVMFB_ALPHA
         enum
            name = Beta
            value = beta
            text
            defineToken = LAYOUTPATTERN_CVMFB_BETA
         enum
            name = Gamma
            value = gamma
            text
            defineToken = LAYOUTPATTERN_CVMFB_GAMMA
         enum
            name = Delta
            value = delta
            text
            defineToken = LAYOUTPATTERN_CVMFB_DELTA
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
