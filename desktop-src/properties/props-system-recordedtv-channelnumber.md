# <a name="systemrecordedtvchannelnumber"></a>RecordedTV. ChannelNumber

錄製的電視頻道。 例如，42、5、53。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.RecordedTV.ChannelNumber
   shellPKey = PKEY_RecordedTV_ChannelNumber
   formatID = 6D748DE2-8D38-4CC3-AC60-F009B057C557
   propID = 7
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt32
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.RecordedTV.ChannelNumber
   shellPKey = PKEY_RecordedTV_ChannelNumber
   formatID = 6D748DE2-8D38-4CC3-AC60-F009B057C557
   propID = 7
   SearchInfo
      IsColumn = true
   typeInfo
      type = UInt32
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
