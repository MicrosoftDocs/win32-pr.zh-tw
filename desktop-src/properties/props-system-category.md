# <a name="systemcategory"></a>System. Category

已取代。 可以指派給專案（例如檔或檔案）的類別。 這個屬性繼承自 OLE 檔案屬性，並已針對 Windows Vista 取代。 應改為使用關鍵字。 較舊的程式碼會將此屬性視為 VT \_ LPSTR。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、version 1507、Windows 8.1、Windows 8、Windows 7、Windows Vista

```
propertyDescription
   name = System.Category
   shellPKey = PKEY_Category
   formatID = D5CDD502-2E9C-101B-9397-08002B2CF9AE
   propID = 2
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Multivalue String
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
