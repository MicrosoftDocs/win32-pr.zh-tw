# <a name="systemnamespaceclsid"></a>System. NamespaceCLSID

專案的命名空間延伸的 CLSID，為此專案執行 IShellFolder 的物件。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.NamespaceCLSID
   shellPKey = PKEY_NamespaceCLSID
   formatID = 28636AA6-953D-11D2-B5D6-00C04FD918D0
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Guid
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
