# <a name="systemrecordedtvcredits"></a>RecordedTV 點數

以下列格式指出程式的信用額度。 "Actor1/Actor2/Actor3 ...;D irector1/Director2/Director3 ...;Host1/Host2.contoso.com/Host3 ...;GuestStar".

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、version 1507、Windows 8.1、Windows 8、Windows 7、Windows Vista

```
propertyDescription
   name = System.RecordedTV.Credits
   shellPKey = PKEY_RecordedTV_Credits
   formatID = 6D748DE2-8D38-4CC3-AC60-F009B057C557
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
