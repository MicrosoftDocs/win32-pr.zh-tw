# <a name="systemdocumentdatesaved"></a>System.Doc>ument。DateSaved

表示上次儲存檔的日期和時間。 舊名稱為 "DocLastSavedTm"。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.Document.DateSaved
   shellPKey = PKEY_Document_DateSaved
   formatID = F29F85E0-4FF9-1068-AB91-08002B27B3D9
   propID = 13
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.Document.DateSaved
   shellPKey = PKEY_Document_DateSaved
   formatID = F29F85E0-4FF9-1068-AB91-08002B27B3D9
   propID = 13
   SearchInfo
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
