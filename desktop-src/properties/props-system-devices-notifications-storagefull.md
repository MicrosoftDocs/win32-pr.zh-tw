# <a name="systemdevicesnotificationsstoragefull"></a>StorageFull。

裝置儲存體-完整通知。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、version 1507、Windows 8.1、Windows 8

```
propertyDescription
   name = System.Devices.Notifications.StorageFull
   shellPKey = PKEY_Devices_Notifications_StorageFull
   formatID = A0E00EE1-F0C7-4D41-B8E7-26A7BD8D38B0
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt64
      IsInnate = true
      EnumeratedList
         UseValueForDefault = True
         enumRange
            name = StorageFullNotification
            minValue = 0
            setValue = 0
            text = Your device is almost full. Delete files to create free space.
```

```
   name = System.Devices.Notifications.StorageFull
   shellPKey = PKEY_Devices_Notification_StorageFull

   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt64
      IsInnate = true
      EnumeratedList
         UseValueForDefault = True
         enumRange
            name = StorageFullNotification
            minValue = 0
            setValue = 0
            text = Your device is almost full. Delete files to create free space.
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
