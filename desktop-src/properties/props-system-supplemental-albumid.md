# <a name="systemsupplementalalbumid"></a>AlbumID

包含專案所屬專輯的識別碼。 可以與內容索引器 Api 中的專輯專案搭配使用，以通知其他應用程式建立的使用者或已建立的應用程式。

## <a name="windows-10-version-1703"></a>Windows 10 (版本 1703)

```
propertyDescription
   name = System.Supplemental.AlbumID
   shellPKey = PKEY_Supplemental_AlbumID
   formatID = 0C73B141-39D6-4653-A683-CAB291EAF95B
   propID = 2
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Multivalue String
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
