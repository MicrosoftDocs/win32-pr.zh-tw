# <a name="systemcontactotheraddresscountry"></a>OtherAddressCountry

連絡人的其他位址國家/地區名稱。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、version 1507、Windows 8.1、Windows 8、Windows 7、Windows Vista

```
propertyDescription
   name = System.Contact.OtherAddressCountry
   shellPKey = PKEY_Contact_OtherAddressCountry
   formatID = 8F167568-0AAE-4322-8ED9-6055B7B0E398
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
