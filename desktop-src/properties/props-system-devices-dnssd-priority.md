# <a name="systemdevicesdnssdpriority"></a>Dnssd. Priority

SRV 記錄優先順序欄位。 通常不會使用。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Devices.Dnssd.Priority
   shellPKey = PKEY_Devices_Dnssd_Priority
   formatID = BF79C0AB-BB74-4CEE-B070-470B5AE202EA
   propID = 9
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
