# <a name="systemproplistxpdetailspanel"></a>PropList. XPDetailsPanel

已過時。 要在 [XP web 視圖詳細資料] 面板中顯示的屬性清單。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、version 1507、Windows 8.1、Windows 8、Windows 7、Windows Vista

```
propertyDescription
   name = System.PropList.XPDetailsPanel
   shellPKey = PKEY_PropList_XPDetailsPanel
   formatID = F2275480-F782-4291-BD94-F13693513AEC
   propID = 0
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
