# <a name="systemdatearchived"></a>System. DateArchived

檔案專案上次的封存日期。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.DateArchived
   shellPKey = PKEY_DateArchived
   formatID = 43F8D7B7-A444-4F87-9383-52271C9B915C
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.DateArchived
   shellPKey = PKEY_DateArchived
   formatID = 43F8D7B7-A444-4F87-9383-52271C9B915C
   propID = 100
   SearchInfo
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
