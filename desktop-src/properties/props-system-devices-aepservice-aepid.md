# <a name="systemdevicesaepserviceaepid"></a>System.Devices.AepService.AepId

裝置關聯端點服務的父 AEP 識別碼

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Devices.AepService.AepId
   shellPKey = PKEY_Devices_AepService_AepId
   formatID = C9C141A9-1B4C-4F17-A9D1-F298538CADB8
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
