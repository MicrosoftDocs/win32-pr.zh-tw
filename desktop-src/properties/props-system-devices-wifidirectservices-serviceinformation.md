# <a name="systemdeviceswifidirectservicesserviceinformation"></a>WiFiDirectServices. ServiceInformation

Wi-Fi Direct Services 服務資訊

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Devices.WiFiDirectServices.ServiceInformation
   shellPKey = PKEY_Devices_WiFiDirectServices_ServiceInformation
   formatID = 31B37743-7C5E-4005-93E6-E953F92B82E9
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Buffer
      IsInnate = true
```

PKEY 值定義于 Propkey 中。
