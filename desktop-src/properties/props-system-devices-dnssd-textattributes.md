# <a name="systemdevicesdnssdtextattributes"></a>Dnssd. TextAttributes

與服務實例相關聯的文字資料。 每個字串通常是成對的索引鍵/值組，以 "=" 分隔。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Devices.Dnssd.TextAttributes
   shellPKey = PKEY_Devices_Dnssd_TextAttributes
   formatID = BF79C0AB-BB74-4CEE-B070-470B5AE202EA
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
