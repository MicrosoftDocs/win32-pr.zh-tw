# <a name="systemvideocompression"></a>系統 Video. 壓縮

指定影片壓縮格式。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7、Windows Vista 版本

```
propertyDescription
   name = System.Video.Compression
   shellPKey = PKEY_Video_Compression
   formatID = 64440491-4C8B-11D1-8B70-080036B11A03
   propID = 10
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>備註

這個屬性的值是 GUID 的字串標記法，其標準格式如下： "{xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx}"。 GUID 會指定影片格式。 如需可能的值，請參閱 [影片子類型 guid](../medfound/video-subtype-guids.md)。

PKEY 值定義于 Propkey 中。
