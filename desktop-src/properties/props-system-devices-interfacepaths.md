# <a name="systemdevicesinterfacepaths"></a>InterfacePaths

此裝置可用的介面。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.Devices.InterfacePaths
   shellPKey = PKEY_Devices_InterfacePaths
   formatID = D08DD4C0-3A9E-462E-8290-7B636B2576B9
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
