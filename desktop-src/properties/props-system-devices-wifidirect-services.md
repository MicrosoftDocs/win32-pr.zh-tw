# <a name="systemdeviceswifidirectservices"></a>WiFiDirect。

指出 Wi-Fi Direct 裝置所支援的服務

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Devices.WiFiDirect.Services
   shellPKey = PKEY_Devices_WiFiDirect_Services
   formatID = 1506935D-E3E7-450F-8637-82233EBE5F6E
   propID = 10
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
