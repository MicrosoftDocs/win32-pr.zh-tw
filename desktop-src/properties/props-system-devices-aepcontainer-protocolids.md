# <a name="systemdevicesaepcontainerprotocolids"></a>System.Devices.AepContainer.ProtocolIds

已貢獻用來建立裝置關聯端點容器的通訊協定識別碼清單

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Devices.AepContainer.ProtocolIds
   shellPKey = PKEY_Devices_AepContainer_ProtocolIds
   formatID = 0BBA1EDE-7566-4F47-90EC-25FC567CED2A
   propID = 13
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue Guid
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
