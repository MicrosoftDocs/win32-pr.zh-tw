# <a name="systemlastwriterpackagefamilyname"></a>System. LastWriterPackageFamilyName

應用程式容器的標記。 最後一個應用程式的套件系列名稱，用來編輯檔案的內容。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.LastWriterPackageFamilyName
   shellPKey = PKEY_LastWriterPackageFamilyName
   formatID = 502CFEAB-47EB-459C-B960-E6D8728F7701
   propID = 101
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
