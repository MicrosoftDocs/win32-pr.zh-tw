# <a name="systemdeviceinterfaceprinterdriverdirectory"></a>DeviceInterface. PrinterDriverDirectory

印表機驅動程式的目錄位置。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.DeviceInterface.PrinterDriverDirectory
   shellPKey = PKEY_DeviceInterface_PrinterDriverDirectory
   formatID = 847C66DE-B8D6-4AF9-ABC3-6F4F926BC039
   propID = 14
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
