# <a name="systemmusicdisplayartist"></a>DisplayArtist

這個屬性會根據[AlbumArtist](./props-system-music-albumartist.md)、IsCompilation 的[資訊，傳回](./props-system-music-artist.md)特定音樂檔案的專輯演出者最佳標記法，以及[](./props-system-music-iscompilation.md)的資訊。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.Music.DisplayArtist
   shellPKey = PKEY_Music_DisplayArtist
   formatID = FD122953-FA93-4EF7-92C3-04C946B2F7C8
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
