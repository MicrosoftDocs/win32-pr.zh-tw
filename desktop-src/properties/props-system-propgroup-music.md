# <a name="systempropgroupmusic"></a>PropGroup 音樂

屬性清單中所使用的屬性群組分隔符號，用來分隔音樂檔案與其他類型。 請勿使用這個屬性來取得或設定值。 它僅供標記之用。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7、Windows Vista 版本

```
propertyDescription
   name = System.PropGroup.Music
   shellPKey = PKEY_PropGroup_Music
   formatID = 68DD6094-7216-40F1-A029-43FE7127043F
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Null
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
