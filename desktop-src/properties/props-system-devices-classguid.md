# <a name="systemdevicesclassguid"></a>System.Devices.ClassGuid

裝置類別 Guid。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Devices.ClassGuid
   shellPKey = PKEY_Devices_ClassGuid
   formatID = A45C254E-DF1C-4EFD-8020-67D146A850E0
   propID = 10
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Guid
      IsInnate = true
```

PKEY 值定義于 Propkey 中。
