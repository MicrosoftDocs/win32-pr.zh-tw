# <a name="systemdevicesaepcontainerdialprotocolinstalledapplications"></a>AepContainer. DialProtocol. InstalledApplications

裝置關聯端點容器上支援撥號通訊協定的應用程式清單

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Devices.AepContainer.DialProtocol.InstalledApplications
   shellPKey = PKEY_Devices_AepContainer_DialProtocol_InstalledApplications
   formatID = 6AF55D45-38DB-4495-ACB0-D4728A3B8314
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
