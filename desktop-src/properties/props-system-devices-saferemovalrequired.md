# <a name="systemdevicessaferemovalrequired"></a>SafeRemovalRequired

指出裝置是否需要安全移除。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.Devices.SafeRemovalRequired
   shellPKey = PKEY_Devices_SafeRemovalRequired
   formatID = AFD97640-86A3-4210-B67C-289C41AABE55
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
