# <a name="systemdeviceinterfacewinusbusbsubclass"></a>DeviceInterface. WinUsb. UsbSubClass

來自 USB 裝置第一個 USB 介面描述項的子類別值。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版、Windows 8。1

```
propertyDescription
   name = System.DeviceInterface.WinUsb.UsbSubClass
   shellPKey = PKEY_DeviceInterface_WinUsb_UsbSubClass
   formatID = 95E127B5-79CC-4E83-9C9E-8422187B3E0E
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Byte
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
