# <a name="systemdevicescontainerid"></a>System.Devices.ContainerId

裝置容器識別碼。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.Devices.ContainerId
   shellPKey = PKEY_Devices_ContainerId
   formatID = 8C7ED206-3F8A-4827-B3AB-AE9E1FAEFC6C
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Guid
      IsInnate = true
```
