# <a name="systemdevicesaepaepid"></a>System.Devices.Aep.AepId

裝置關聯端點的身分識別

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.Devices.Aep.AepId
   shellPKey = PKEY_Devices_Aep_AepId
   formatID = 3B2CE006-5E61-4FDE-BAB8-9B8AAC9B26DF
   propID = 8
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

PKEY 值定義于 Propkey 中。
