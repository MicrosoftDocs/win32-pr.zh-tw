# <a name="systemcommunicationdateitemexpires"></a>DateItemExpires

專案因為保留原則而到期的日期。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、version 1507、Windows 8.1、Windows 8、Windows 7、Windows Vista

```
propertyDescription
   name = System.Communication.DateItemExpires
   shellPKey = PKEY_Communication_DateItemExpires
   formatID = 428040AC-A177-4C8A-9760-F6F761227F9A
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = DateTime
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
