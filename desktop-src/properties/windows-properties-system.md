# <a name="windows-property-system"></a>Windows 屬性系統

## <a name="purpose"></a>目的

Windows 屬性系統是資料定義的可擴充讀取/寫入系統，可提供統一的方式來表示 Shell 專案的中繼資料。 Windows Vista 和更新版本中的 Windows 屬性系統可讓您儲存和取得 Shell 專案的中繼資料。 Shell 專案是任何一段內容，例如檔案、資料夾、電子郵件或連絡人。 屬性是與 Shell 專案相關聯之中繼資料的個別部分。

## <a name="developer-audience"></a>開發人員對象

開始閱讀 Windows 屬性系統 SDK 檔之前，您應該先瞭解下列各項的基本概念：

-   元件物件模型 (COM)
-   Shell 命名空間程式設計

如需 COM 的簡介，請參閱 [Com 基本](../com/com-fundamentals.md)概念。 如需 Shell 命名空間程式設計的簡介，請參閱 [使用 Shell 命名空間開始使用](../shell/namespace-intro.md)。

如需使用 Windows 屬性系統的詳細說明，請參閱 [屬性系統總覽：開發案例](property-system-overview.md)。

## <a name="run-time-requirements"></a>執行階段需求求

使用 Windows 屬性系統的支援執行時間環境是 Windows Vista 或更新版本，而 Windows 軟體開發套件 (SDK) 。 Windows XP 和 Microsoft Windows 桌面搜尋 (WDS) 3.0 或更新版本也包含 Windows 屬性系統的子集。 若為 Windows 7 或更新版 Windows Vista SDK 下載，請參閱 [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx)。
