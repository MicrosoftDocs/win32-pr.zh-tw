# <a name="systemgpsaltitude"></a>GPS

根據 PKEY GPS AltitudeRef 中的參考指出 \_ 高度 \_ 。 從 PKEY \_ gps \_ ALTITUDENUMERATOR 和 PKEY \_ gps AltitudeDenominator 計算 \_ 。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、version 1507、Windows 8.1、Windows 8、Windows 7、Windows Vista

```
propertyDescription
   name = System.GPS.Altitude
   shellPKey = PKEY_GPS_Altitude
   formatID = 827EDB4F-5B73-44A7-891D-FDFFABEA35CA
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。

`label`針對 Windows Vista （含 Service Pack 1 (SP1) ）新增了特定間接字串參考的屬性（attribute **）的需求**。
