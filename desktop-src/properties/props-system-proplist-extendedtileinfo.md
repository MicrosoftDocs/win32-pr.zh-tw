# <a name="systemproplistextendedtileinfo"></a>PropList. ExtendedTileInfo

要在延伸磚上的 listview 中顯示的屬性清單。 在 "ExtendedTileInfo" 的 regvalue 下註冊。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、version 1507、Windows 8.1、Windows 8、Windows 7、Windows Vista

```
propertyDescription
   name = System.PropList.ExtendedTileInfo
   shellPKey = PKEY_PropList_ExtendedTileInfo
   formatID = C9944A21-A406-48FE-8225-AEC7E24C211B
   propID = 9
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
