# <a name="systemappusermodeltoastactivatorclsid"></a>AppUserModel. ToastActivatorCLSID

用來 CoCreate INotificationActivationCallback 介面，以通知快顯通知啟用。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.AppUserModel.ToastActivatorCLSID
   shellPKey = PKEY_AppUserModel_ToastActivatorCLSID
   formatID = 9F4C2855-9F79-4B39-A8D0-E1D42DE1D5F3
   propID = 26
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Guid
      IsInnate = false
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
