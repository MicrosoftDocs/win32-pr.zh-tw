# <a name="systemdevicesaepcontainersupportsinformation"></a>AepContainer. SupportsInformation

指出裝置關聯端點容器是否支援資訊

## <a name="windows-10-version-1703-windows-10-version-1607"></a>Windows 10，版本1703，Windows 10，版本1607

```
propertyDescription
   name = System.Devices.AepContainer.SupportsInformation
   shellPKey = PKEY_Devices_AepContainer_SupportsInformation
   formatID = 6AF55D45-38DB-4495-ACB0-D4728A3B8314
   propID = 14
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
