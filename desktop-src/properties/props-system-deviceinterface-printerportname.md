# <a name="systemdeviceinterfaceprinterportname"></a>DeviceInterface. PrinterPortName

印表機所在的埠。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10，1703、Windows 10、1607版、Windows 10、1511版、Windows 10、1507、Windows 8.1、Windows 8、Windows 7 版

```
propertyDescription
   name = System.DeviceInterface.PrinterPortName
   shellPKey = PKEY_DeviceInterface_PrinterPortName
   formatID = EEC7B761-6F94-41B1-949F-C729720DD13C
   propID = 12
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
