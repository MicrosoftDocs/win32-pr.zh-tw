# <a name="systemstorageproviderfileversionwaterline"></a>System. StorageProviderFileVersionWaterline

這個檔案的儲存提供者計算的檔案版本等高線。 此值可用來偵測檔案是否已變更。

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10，1703版、Windows 10、1607版、Windows 10、1511版、Windows 10、1507版

```
propertyDescription
   name = System.StorageProviderFileVersionWaterline
   shellPKey = PKEY_StorageProviderFileVersionWaterline
   formatID = B2F9B9D6-FEC4-4DD5-94D7-8957488C807B
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Buffer
      IsInnate = false
```

## <a name="remarks"></a>備註

PKEY 值定義于 Propkey 中。
