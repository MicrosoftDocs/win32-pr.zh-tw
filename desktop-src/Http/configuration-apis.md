# <a name="configuration-apis"></a>設定 API

HTTP 伺服器 Api 提供可讓系統管理員設定、查詢和刪除設定參數的功能。 設定 Api 會使用系統上的 HTTP 伺服器 API 來控制所有應用程式的設定。

下列各節包含設定的相關資訊。

> [!Note]  
> 這些都是要根據系統和/或應用程式需求套用的選擇性設定步驟。

 

[IP 接聽清單](ip-listen-list.md) ：說明如何設定 HTTP 以在特定的 IP 介面上運作。

[SSL 憑證](ssl-certificates.md) -詳述使用 HTTP 伺服器 API 來設定憑證的程式。

[命名空間保留、註冊和路由](namespace-reservations-registrations-and-routing.md) —概述使用 HTTP 伺服器 API 為電腦上的所有應用程式保留部分命名空間的程式。 此外，也會說明註冊和路由要求至特定應用程式的程式。

[HttpCfg.exe](httpcfg-exe.md) -使用設定 api 的範例程式。 此工具的編譯版本也會隨附在支援工具中的作業系統，以及說明文件的使用說明檔。
