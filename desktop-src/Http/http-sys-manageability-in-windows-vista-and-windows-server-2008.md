# <a name="httpsys-manageability-in-windows-vista-and-windows-server-2008"></a>Windows Vista 和 Windows Server 2008 中的 HTTP.sys 管理能力

在 Windows Vista 和 Windows Server 2008 中，除了 Windows Server 2003 中引進的 HTTP 記錄支援之外，IT 專業人員還會發現 HTTP 伺服器 API 元件的改良管理功能。 為了診斷和設定 HTTP 伺服器 API 元件，IT 專業人員可以使用三種工具：

-   新的 ETW 追蹤功能
-   效能計數器
-   Netsh 設定命令

HTTP 伺服器 API 可讓應用程式透過 HTTP 進行通訊，並包含 SSL 支援，讓應用程式可以透過安全的 HTTP 連線來交換資料。 Windows Server 2003 作業系統和 Windows XP SP2 支援 HTTP 伺服器 API。
