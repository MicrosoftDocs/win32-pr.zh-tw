# <a name="formatting-legend"></a>格式圖例

區段會使用下列格式設定慣例。

| 格式                                                                    | 意義                                                        |
|---------------------------------------------------------------------------|----------------------------------------------------------------|
| 斜體                                                                    | 使用者必須提供的資訊                          |
| 粗體                                                                      | 使用者必須依照顯示的方式鍵入的元素              |
| 省略符號 (...)                                                            | 命令列中可以重複多次的參數 |
| 介於括弧之間 (\[ \])                                                    | 選擇性項目                                                 |
| 介於大括弧 ({}) ; 以管道 () 分隔的選項 \| 。 範例： {偶數 \| 奇數} | 使用者必須從中選擇一項的選項集        |
| 新細明體字型                                                              | 程式碼或程式輸出                                         |
