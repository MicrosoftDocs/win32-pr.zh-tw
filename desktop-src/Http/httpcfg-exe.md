# <a name="httpcfgexe"></a>HttpCfg.exe

這項工具可協助您藉由包裝下列參數的 HTTP 設定 [api](configuration-apis.md) 來設定 HTTP 伺服器 API：

-   IP 接聽清單
-   SSL 憑證
-   命名空間保留

從 Windows Server 2003 開始，HttpCfg.exe 隨附于 Windows Server 作為二進位可執行檔，以及描述如何使用它的說明檔。 它可以從作業系統發佈磁片安裝為支援工具。

HttpCfg.exe 也可供下載;請參閱 [WINDOWS XP SP2 支援工具](https://www.microsoft.com/downloads/details.aspx?familyid=49AE8576-9BB9-4126-9761-BA8011FABF38)。

**Windows Vista 和 Windows Server 2008：** HttpCfg.exe 已淘汰，並且以 netsh.exe 中的類似功能取代。
