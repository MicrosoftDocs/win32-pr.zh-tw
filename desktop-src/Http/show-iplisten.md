# <a name="show-iplisten"></a>show iplisten

列出 IP 接聽清單中的所有 IP 位址。 IP 接聽清單可用來界定 HTTP 服務所繫結的位址清單範圍。 "0.0.0.0" 表示任何 IPv4 位址，"::" 表示任何 IPv6 位址。

``` syntax
show iplisten
 
```
