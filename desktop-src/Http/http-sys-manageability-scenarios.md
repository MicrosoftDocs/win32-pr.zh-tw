# <a name="httpsys-manageability-scenarios"></a>HTTP.sys 管理性案例

下列案例提供這些工具如何協助 IT 專業人員診斷和解決 (HTTP.sys) 元件的 HTTP 伺服器 API 相關問題的範例。 這些範例中使用的工具可在 Windows Vista 和 Windows Server 2008 中取得。

-   [案例1：使用 ETW 追蹤和 Netsh 命令的 HTTP Timeout 範例](scenario-1--http-timeout-example-using-etw-tracing-and-netsh-commands.md)
-   [案例2：使用 ETW 追蹤的剖析器範例](scenario-2--parser-example-using-etw-tracing.md)
-   [案例3：效能計數器](scenario-3--performance-counters.md)

本檔中的案例是使用 HTTP 伺服器 API 元件的管理能力工具的範例。 這些範例不是為了建議特定的設定變更。
