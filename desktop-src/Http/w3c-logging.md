# <a name="w3c-logging"></a><span data-ttu-id="f54af-103">W3C 記錄</span><span class="sxs-lookup"><span data-stu-id="f54af-103">W3C Logging</span></span>

<span data-ttu-id="f54af-104">W3C 擴充記錄是伺服器端記錄類型，可以在伺服器會話或 URL 群組上啟用。</span><span class="sxs-lookup"><span data-stu-id="f54af-104">W3C extended logging is type of server side logging that can be enabled on the server session or URL group.</span></span> <span data-ttu-id="f54af-105">在 URL 群組上啟用 W3C 記錄時，只會在路由至 URL 群組的要求上執行記錄。</span><span class="sxs-lookup"><span data-stu-id="f54af-105">When W3C logging is enabled on a URL group, logging is performed only on requests that are routed to the URL Group.</span></span> <span data-ttu-id="f54af-106">針對設定為啟用 W3C 記錄的每個 URL 群組，都會建立個別的記錄檔。</span><span class="sxs-lookup"><span data-stu-id="f54af-106">A separate log file is created for each URL group configured to enable W3C logging.</span></span>

<span data-ttu-id="f54af-107">當伺服器會話上啟用 W3C 記錄功能時，它會以集中式形式記錄伺服器會話下的所有 URL 群組。</span><span class="sxs-lookup"><span data-stu-id="f54af-107">When W3C logging is enabled on the server session it functions as centralized form of logging for all the URL groups under the server session.</span></span> <span data-ttu-id="f54af-108">伺服器會話中的所有 URL 群組都會保留單一記錄檔。</span><span class="sxs-lookup"><span data-stu-id="f54af-108">A single log file is maintained for all of the URL groups in the server session.</span></span>
