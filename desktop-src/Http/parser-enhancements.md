# <a name="parser-enhancements"></a>剖析器增強功能

在 Windows Server 2003 （含 Service Pack 1） (SP1) 中，HTTP 伺服器 API 允許來自 HTTP 用戶端的下列要求。

-   使用單行換行 (LF) 為行結束字元的要求。
-   包含線性泛空白字元的要求 (LWS 在 HTTP 要求行與標頭開頭之間) 。

預設會啟用這些行為，而且不受登錄設定控制。
