本章節包含如何使用下列 ESE API 元件的一般資訊。

  - [資料行](./columns.md)

  - [資料表中的索引編制](./indexing-in-the-table.md)

  - [建立資料庫](./creating-databases.md)

  - [交易](./transactions.md)

  - [ESE 錯誤](./extensible-storage-engine-errors.md)

  - [ESE 檔案](./extensible-storage-engine-files.md)

您的程式來源檔案應包含 esent 標頭檔，以存取可延伸儲存引擎 API 的函式原型和結構定義。 開發人員可使用 esent 程式庫檔案來建立使用可延伸儲存引擎 API 的應用程式。 應用程式會在執行時間連結至 esent.dll。
