# <a name="wchar"></a>WCHAR


_**適用于：** Windows |Windows伺服器_

## <a name="wchar"></a>WCHAR

WCHAR 資料類型包含16位的 Unicode 字元。

```cpp
    #if !defined(_NATIVE_WCHAR_T_DEFINED)
    typedef unsigned short WCHAR;
    #else
    typedef wchar_t WCHAR;
    #endif
```

### <a name="data-types"></a>資料類型

WCHAR

16位的 Unicode 字元。
