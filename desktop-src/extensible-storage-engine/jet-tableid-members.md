# <a name="jet_tableid-members"></a>JET_TABLEID 成員

包含受保護的成員  
包含繼承的成員  

JET_TABLEID 包含資料庫資料指標的控制碼，可用於呼叫 JET API。 資料指標只能搭配用來開啟該資料指標的會話使用。

[JET_TABLEID](./jet-tableid-structure.md)類型會公開下列成員。
