# <a name="jet_handle"></a>JET_HANDLE


_**適用于：** Windows |Windows Server_

## <a name="jet_handle"></a>JET_HANDLE

**JET_HANDLE** 資料類型包含泛型控制碼。

```cpp
    typedef JET_API_PTR JET_HANDLE;
```

### <a name="data-types"></a>資料類型

JET_HANDLE

**Null** 值表示不正確控制碼。
