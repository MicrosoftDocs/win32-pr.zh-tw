# <a name="instanceparameterspagetempdbmin-property"></a>InstanceParameters. PageTempDBMin 屬性

取得或設定暫存資料庫的初始大小。 大小是在資料庫頁面中。 大小為零表示應該使用一般資料庫的預設大小。 小型應用程式通常需要將暫存資料庫設定為盡可能小。 將此參數設定為 [PageTempDBSmallest](./systemparameters.pagetempdbsmallest-field.md) ，可以達到最小的暫存資料庫。

**命名空間：**  [Microsoft. Isam. Interop](./microsoft.isam.esent.interop-namespace.md)  
**元件：**  (在 Microsoft.Isam.Esent.Interop.dll) 中的 Microsoft。
