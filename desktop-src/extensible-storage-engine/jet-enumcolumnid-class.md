# <a name="jet_enumcolumnid-class"></a>JET_ENUMCOLUMNID 類別

當使用 JetEnumerateColumns 函式時，列舉一組特定的資料行，並選擇性地列舉這些資料行的一組特定的多重值。 JetEnumerateColumns 會選擇性地採用 JET_ENUMCOLUMNID 結構的陣列。
