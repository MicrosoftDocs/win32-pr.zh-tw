# <a name="jet_retinfoitagsequence-property"></a>JET_RETINFO itagSequence 屬性

取得或設定多重值資料行中值的序號。 值的陣列是以一為基礎。 第一個值是 sequence 1，不是0。 如果 [記錄] 資料行只有一個值，則應該以 itagSequence 的形式傳遞1。

**命名空間：**  [Microsoft. Isam. Interop](./microsoft.isam.esent.interop-namespace.md)  
**元件：**  (在 Microsoft.Isam.Esent.Interop.dll) 中的 Microsoft。
