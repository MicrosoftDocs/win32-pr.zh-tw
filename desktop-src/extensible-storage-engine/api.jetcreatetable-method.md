# <a name="apijetcreatetable-method"></a>JetCreateTable 方法

建立空的資料表。 新建立的資料表會以獨佔方式開啟。

**命名空間：**  [Microsoft. Isam. Interop](./microsoft.isam.esent.interop-namespace.md)  
**元件：**  (在 Microsoft.Isam.Esent.Interop.dll) 中的 Microsoft。

## <a name="syntax"></a>語法

``` vb
'Declaration
Public Shared Sub JetCreateTable ( _
    sesid As JET_SESID, _
    dbid As JET_DBID, _
    table As String, _
    pages As Integer, _
    density As Integer, _
    <OutAttribute> ByRef tableid As JET_TABLEID _
)
'Usage
Dim sesid As JET_SESID
Dim dbid As JET_DBID
Dim table As String
Dim pages As Integer
Dim density As Integer
Dim tableid As JET_TABLEIDApi.JetCreateTable(sesid, dbid, _
    table, pages, density, tableid)
```

``` csharp
public static void JetCreateTable(
    JET_SESID sesid,
    JET_DBID dbid,
    string table,
    int pages,
    int density,
    out JET_TABLEID tableid
)
```

#### <a name="parameters"></a>參數

  - sesid  
    類型： [Microsoft.Isam.Esent.Interop.JET_SESID](./jet-sesid-structure.md)  
    
    要使用的會話。

<!-- end list -->

  - dbid  
    類型： [Microsoft.Isam.Esent.Interop.JET_DBID](./jet-dbid-structure.md)  
    
    要在其中建立資料表的資料庫。

<!-- end list -->

  - table  
    類型： [system.string](/dotnet/api/system.string)  
    
    要建立之資料表的名稱。

<!-- end list -->

  - 頁面  
    類型： [system.object](/dotnet/api/system.int32)  
    
    資料表中的初始頁面數目。

<!-- end list -->

  - 密度 (density)  
    類型： [system.object](/dotnet/api/system.int32)  
    
    資料表的預設密度。 這是在執行連續插入時使用。

<!-- end list -->

  - tableid  
    類型： [Microsoft.Isam.Esent.Interop.JET_TABLEID](./jet-tableid-structure.md)  
    
    傳回新資料表的 tableid。
