# <a name="jet_dbid-members"></a>JET_DBID 成員

包含受保護的成員  
包含繼承的成員  

JET_DBID 包含資料庫的控制碼。 資料庫控制碼是用來管理資料庫的架構。 它也可以用來管理該資料庫內的資料表。

[JET_DBID](./jet-dbid-structure.md)類型會公開下列成員。
