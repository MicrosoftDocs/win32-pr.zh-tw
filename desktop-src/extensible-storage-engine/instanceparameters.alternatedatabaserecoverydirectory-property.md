# <a name="instanceparametersalternatedatabaserecoverydirectory-property"></a>InstanceParameters. AlternateDatabaseRecoveryDirectory 屬性

取得或設定資料夾的相對或絕對檔案系統路徑，其中損毀復原或還原作業可以在指定資料夾的交易記錄中找到參考的資料庫。

**命名空間：**  [Microsoft. Isam. Interop](./microsoft.isam.esent.interop-namespace.md)  
**元件：**  (在 Microsoft.Isam.Esent.Interop.dll) 中的 Microsoft。
