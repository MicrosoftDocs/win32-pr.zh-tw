# <a name="apitrysetindexrange-method"></a>TrySetIndexRange 方法

暫時限制索引項目的索引項目集合，其可使用 JetMove 從目前的索引項目目開始，然後在符合該資料指標中搜尋索引鍵所指定之搜尋準則的索引項目目和指定的系結準則中結束。 搜尋金鑰必須先前已使用 JetMakeKey 來建立。 如果索引範圍不是空的，則傳回 true，否則傳回 false。

**命名空間：**  [Microsoft. Isam. Interop](./microsoft.isam.esent.interop-namespace.md)  
**元件：**  (在 Microsoft.Isam.Esent.Interop.dll) 中的 Microsoft。
