# <a name="instanceparameterswaypointlatency-property"></a>InstanceParameters. WaypointLatency 屬性

取得或設定 esent 將延遲資料庫排清之記錄檔的數目。 如果失敗導致記錄檔遺失，這可以用來提高資料庫復原能力。 Windows 7 和更新的支援。 在 Windows XP、Windows Server 2003、Windows Vista 和 Windows Server 2008 上略過。
