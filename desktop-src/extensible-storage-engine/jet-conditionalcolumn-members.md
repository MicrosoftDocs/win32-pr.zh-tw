# <a name="jet_conditionalcolumn-members"></a>JET_CONDITIONALCOLUMN 成員

包含受保護的成員  
包含繼承的成員  

定義如何針對指定的索引執行條件式索引編制。 條件式索引僅包含符合指定條件之資料列的索引項目目。 不過，條件資料行不是索引索引鍵的一部分，而只會控制索引項目是否存在。
