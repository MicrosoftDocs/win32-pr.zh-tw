# <a name="jet_threadstats-members"></a>JET_THREADSTATS 成員

包含受保護的成員  
包含繼承的成員  

包含目前線程上資料庫引擎所執行之工作的累計統計資料。 這項資訊會透過 JetGetThreadStats 傳回。

[JET_THREADSTATS](./jet-threadstats-structure2.md)類型會公開下列成員。
