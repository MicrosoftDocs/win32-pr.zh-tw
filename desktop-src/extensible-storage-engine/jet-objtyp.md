# <a name="jet_objtyp"></a>JET_OBJTYP


_**適用于：** Windows |Windows伺服器_

## <a name="jet_objtyp"></a>JET_OBJTYP

**JET_OBJTYP** 的常數群組代表資料庫物件的型別。 目前只支援資料表。


| <p>常數/值</p> | <p>Description</p> | 
|-----------------------|--------------------|
| <p>JET_objtypNil<br />0</p> | <p>代表所有類型的物件。 目前只支援資料表。</p> | 
| <p>JET_objtypTable<br />1</p> | <p>表示資料表。</p> |
