# <a name="directwrite-glossary"></a>DirectWrite詞彙

<dl> <dt>

<span id="directwrite.directwrite_glossary_bidi__unicode_feature_"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_BIDI__UNICODE_FEATURE_"></span>**雙向 (Unicode 功能)**
</dt> <dd>

發音為「依彩色」。 包含不同語言之單字混合的文字，必須以不同的方向進行讀取。 通常需要特殊程式碼才能處理。 「阿拉伯文和希伯來文為雙向語言;文字會由右至左，但數位會由左至右。」

</dd> <dt>

<span id="directwrite.directwrite_glossary_cleartype"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_CLEARTYPE"></span>**清晰**
</dt> <dd>

一種字型顯示技術，可大幅增加字型顯示解析度，使電腦螢幕上顯示的字較平滑，而不會有鋸齒狀的感覺。 ClearType 利用數位介面，大幅增加彩色 LCD 監視器上文字的可讀性，例如膝上型電腦的 LCD 監視器及高品質平面桌上型顯示器。

</dd> <dt>

<span id="directwrite.directwrite_glossary_direct2d"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_DIRECT2D"></span>**Direct2D**
</dt> <dd>

硬體加速、即時模式、2D 圖形 API，可為2D 幾何、點陣圖和文字提供高效能和高品質的呈現。 Direct2D API 是設計來與使用 GDI、GDI+ 或 Direct3D 的現有程式碼交互操作。

</dd> <dt>

<span id="directwrite.directwrite_glossary_directwrite"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_DIRECTWRITE"></span>**DirectWrite**
</dt> <dd>

DirectX API，提供改良的高品質文字轉譯，以及與 GDI 和 Direct2D 的互通性。

</dd> <dt>

<span id="directwrite.directwrite_glossary_directx"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_DIRECTX"></span>**DirectX**
</dt> <dd>

Microsoft Windows 作業系統的延伸模組。 DirectX 技術可協助遊戲和其他程式使用硬體的先進多媒體功能。

</dd> <dt>

<span id="directwrite.directwrite_glossary_gdi"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_GDI"></span>**GDI**
</dt> <dd>

一種可執行程式，可從 Windows 應用程式處理圖形化函數呼叫，並將這些呼叫傳遞給適當的設備磁碟機，以執行產生輸出的硬體特定功能。 藉由在應用程式與輸出裝置之間做為緩衝區，GDI 會針對應用程式呈現與裝置無關的觀點，同時以裝置相依的格式與裝置互動。

</dd> <dt>

<span id="directwrite.directwrite_glossary_glyph"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_GLYPH"></span>**字形**
</dt> <dd>

指定字型中字元的實體標記法。 字元可能有許多字元，而系統上的每個字型可能會為該字元定義不同的圖像。

</dd> <dt>

<span id="directwrite.directwrite_glossary_glyph_composition"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_GLYPH_COMPOSITION"></span>**字元組合**
</dt> <dd>

將兩個或多個字元組合成單一字元。

</dd> <dt>

<span id="directwrite.directwrite_glossary_glyph_decomposition"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_GLYPH_DECOMPOSITION"></span>**符號分解**
</dt> <dd>

將單一圖像分割成多個字元。

</dd> <dt>

<span id="directwrite.directwrite_glossary_glyph_run"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_GLYPH_RUN"></span>**圖像執行**
</dt> <dd>

以特定順序排列的一組字元，具有相同的格式設定特性，例如字型、大小、粗細和樣式。

</dd> <dt>

<span id="directwrite.directwrite_glossary_hardware-accelerated_text"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_HARDWARE-ACCELERATED_TEXT"></span>**硬體加速文字**
</dt> <dd>

使用硬體加速來改善轉譯效能的技術所呈現的文字。

</dd> <dt>

<span id="directwrite.directwrite_glossary_interoperability"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_INTEROPERABILITY"></span>**互 操作 性**
</dt> <dd>

有兩個或多個 Api 可以使用和彼此之間傳輸資訊的能力。

</dd> <dt>

<span id="directwrite.directwrite_glossary_kerning"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_KERNING"></span>**字元**
</dt> <dd>

特定單字內特定字母組之間所套用的間距。

</dd> <dt>

<span id="directwrite.directwrite_glossary_letterspacing"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_LETTERSPACING"></span>**間距**
</dt> <dd>

調整兩個字元之間的間距，以建立偶數間距的外觀、將文字調整為指定的空間，以及調整換行字元。

</dd> <dt>

<span id="directwrite.directwrite_glossary_ligature"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_LIGATURE"></span>**結紮**
</dt> <dd>

結合兩個或多個字元來表示單一印刷字元。 新式拉丁腳本會使用一些。 其他腳本會使用許多與字型和樣式相依的連字號。 某些語言具有必要的連字，例如阿拉伯文。

</dd> <dt>

<span id="directwrite.directwrite_glossary_smart_pointers"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_SMART_POINTERS"></span>**智慧型指標**
</dt> <dd>

類別，這個類別會包裝會自動釋放介面物件的 COM 介面指標。

</dd> <dt>

<span id="directwrite.directwrite_glossary_swashes"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_SWASHES"></span>**字**
</dt> <dd>

Ornamental 新增至為更多樣式的字元所建立的字元。

</dd> <dt>

<span id="directwrite.directwrite_glossary_unicode"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_UNICODE"></span>**Unicode**
</dt> <dd>

Unicode 協會所開發的字元編碼標準，幾乎可表示世界上所有的書寫語言。 Unicode 字元集有多種表示形式，包括 UTF-8、UTF-16 和 UTF-32。 大部分的 Windows 介面是使用 UTF-16 的形式。  (GTMT 具有20個定義) 。

</dd> <dt>

<span id="directwrite.directwrite_glossary_windows_presentation_foundation__wpf_"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_WINDOWS_PRESENTATION_FOUNDATION__WPF_"></span>**Windows Presentation Foundation (WPF)**
</dt> <dd>

.net 微架構3.0、Windows Vista 和 Microsoft Silverlight 所使用的 GUI 架構。 .NET 微型 Framework GUI 類別是以 WPF 為基礎的鬆散。

</dd> </dl>
