# <a name="idwritenumbersubstitution-interface"></a>IDWriteNumberSubstitution 介面

針對指定的地區設定保存適當的數位和數位標點符號。

## <a name="members"></a>成員

**IDWriteNumberSubstitution** 介面繼承自 [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown)介面，但沒有其他成員。
