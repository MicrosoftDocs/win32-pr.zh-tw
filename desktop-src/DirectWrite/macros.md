# <a name="directwrite-macros"></a>DirectWrite 宏

DirectWrite 定義下列宏。

## <a name="in-this-section"></a>本節內容

| 主題 | 描述 |
|-|-|
| [**DWRITE_MAKE_FONT_AXIS_TAG**](/windows/win32/api/dwrite_3/nf-dwrite_3-dwrite_make_font_axis_tag) | 建立字型軸的 OpenType 標記。 |
| [**DWRITE \_ MAKE \_ OPENTYPE \_ 標記**](/windows/win32/api/dwrite/nf-dwrite-dwrite_make_opentype_tag) | 將 OpenType 標記建立為32位整數，使標籤中的第一個字元是最低的位元組 (最不重要的位元組由小到大架構) ，可用來與字型檔案中的標記進行比較。 這個宏與 [**DWRITE \_ 字型 \_ 功能 \_ 標記**](/windows/win32/api/dwrite/ne-dwrite-dwrite_font_feature_tag)相容。 |
