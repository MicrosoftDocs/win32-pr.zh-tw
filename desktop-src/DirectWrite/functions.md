# <a name="directwrite-functions"></a>DirectWrite 函式

DirectWrite 提供下列功能。 所有其他作業都是使用由下列函式所建立之 [**IDWriteFactory**](/windows/win32/api/dwrite/nn-dwrite-idwritefactory)物件所建立的 DirectWrite 介面物件來執行。

## <a name="in-this-section"></a>本節內容



| 主題                                                         | 描述                                                                                                             |
|---------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**DWriteCreateFactory**](/windows/win32/api/dwrite/nf-dwrite-dwritecreatefactory)<br/> | 建立用於後續建立個別 DirectWrite 物件的 DirectWrite factory 物件。<br/> |
| [**DWriteCoreCreateFactory (DWriteCore)**](/windows/windows-app-sdk/api/win32/dwrite_core/nf-dwrite_core-dwritecorecreatefactory)<br/> | 建立用於後續建立個別 DWriteCore 物件的 DirectWrite factory 物件。<br/> |
