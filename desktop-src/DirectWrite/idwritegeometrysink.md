# <a name="idwritegeometrysink-interface"></a>IDWriteGeometrySink 介面

**IDWriteGeometrySink** 是 [**ID2D1SimplifiedGeometrySink**](/windows/win32/api/d2d1/nn-d2d1-id2d1simplifiedgeometrysink)介面的 **typedef** 。 如需詳細資訊，請參閱 [**ID2D1SimplifiedGeometrySink**](/windows/win32/api/d2d1/nn-d2d1-id2d1simplifiedgeometrysink) 參考頁面。

## <a name="members"></a>成員

**IDWriteGeometrySink** 介面繼承自 [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown)介面，但沒有其他成員。
