# <a name="the-policy-object-type"></a>原則物件類型

[**原則**](policy-object.md)物件是容器物件。 下列物件類型可能位於這個容器內：

-   [**帳戶**](account-object.md) 物件
-   [**TrustedDomain**](trusteddomain-object.md) 物件
-   Secret 物件

只有一個 [**原則**](policy-object.md) 物件的實例。 它不能建立也不會損毀。
