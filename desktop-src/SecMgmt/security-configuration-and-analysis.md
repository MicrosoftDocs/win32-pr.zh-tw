# <a name="security-configuration-and-analysis"></a>安全性設定及分析

安全性設定和分析是一種獨立的嵌入式管理單元工具，可供使用者用來將一或多個已儲存的設定匯入至私用安全性資料庫。 匯入設定會建立儲存複合設定的電腦專用安全性資料庫。 您可以將此複合設定套用至電腦，並針對儲存在資料庫中的預存複合設定來分析目前的系統組態。

[安全性範本](security-templates.md)和安全性設定與分析組成安全性設定嵌入式管理單元。
