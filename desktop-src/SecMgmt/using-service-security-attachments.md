# <a name="using-service-security-attachments"></a>使用服務安全性附件

本節說明如何在 [安全性設定] 嵌入式管理單元中，建立支援服務特定設定所需的附件元件。您將需要執行並註冊兩個元件：延伸嵌入式管理單元，可顯示使用者介面和附件引擎來處理設定和分析要求。

如需詳細資訊，請參閱下列主題：

-   [註冊附件嵌入式管理單元延伸模組](registering-an-attachment-snap-in-extension.md)
-   [建立附件引擎](creating-an-attachment-engine.md)
-   [註冊附件引擎](registering-an-attachment-engine.md)
