# <a name="safer"></a>更安全

更安全的 API 中的函式提供任何應用程式，該應用程式會在啟動可執行檔之前，對外部來源的程式執行查詢安全性原則以進行核准的能力。 您可以在載入和執行可執行檔或活動內容之前呼叫這些函式。 更安全的 API 適用的應用程式類型，包括處理附件 (的應用程式，例如郵件用戶端和可以傳輸檔案) 和腳本解譯器的即時 messengers。 在更安全的 API 中支援函式開始 Windows XP。 如需詳細資訊，請參閱 [更安全](management-functions.md)的函式。
