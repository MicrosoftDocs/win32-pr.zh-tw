# <a name="trusteddomain-object-protection"></a>TrustedDomain 物件保護

建立 [**TrustedDomain**](trusteddomain-object.md) 物件時，它會被指派標準保護，如下所示：

-   全球本機群組會被授與一般的 \_ 執行存取權。
-   本機 \_ 系統管理員本機群組被授與 DELETE、generic \_ READ、generic \_ WRITE 和一般 \_ 執行存取權。
-   本機 \_ 系統管理員本機群組會被指派為物件的擁有者和主要群組。
