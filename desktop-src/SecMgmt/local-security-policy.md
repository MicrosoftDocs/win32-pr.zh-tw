# <a name="local-security-policy"></a>本機安全性原則

系統的本機安全性原則是一組有關本機電腦安全性性的資訊。

本機安全性原則資訊包括下列各項：

-   受信任的網域可驗證登入嘗試。
-   哪些使用者帳戶可以存取系統以及如何進行。 例如，以互動方式、透過網路或服務。
-   指派給帳戶的權利和 [*許可權*](/windows/desktop/SecGloss/p-gly) 。
-   安全性稽核原則。

「 [*本地安全機構*](/windows/desktop/SecGloss/l-gly) 」 (LSA) 將本機原則資訊儲存在一組 [LSA 原則物件](lsa-policy-objects.md)中。
