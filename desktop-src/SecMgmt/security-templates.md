# <a name="security-templates"></a>安全性範本

安全性範本是獨立的嵌入式管理單元工具，可供使用者用來定義與電腦無關的安全性設定。 這些設定會儲存為以文字為基礎的 .inf 檔案。

安全性範本和安全性設定 [與分析](security-configuration-and-analysis.md) 組成安全性設定嵌入式管理單元。
