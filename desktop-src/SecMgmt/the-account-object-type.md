# <a name="the-account-object-type"></a>帳戶物件類型

[**帳戶**](account-object.md) 物件是用來表示要在使用者或群組或本機群組成員上 bestowed 的 [*許可權*](/windows/desktop/SecGloss/p-gly)或系統存取權限。

[**帳戶**](account-object.md) 物件位於 [**Policy**](policy-object.md) 物件內。 這些物件中可能有零個以上的物件。

[**帳戶**](account-object.md)物件沒有物件特定的資料類型。
