# <a name="lsa-policy"></a>LSA 原則

 (LSA) 的「[*本地安全機構*](/windows/desktop/SecGloss/l-gly)」是受保護的 Windows 子系統，可維護系統上本機安全性的所有層面的相關資訊，統稱為系統的 [本機安全性原則](local-security-policy.md)。 除了提供原則資訊外，LSA 也提供在名稱和 [*安全識別碼*](/windows/desktop/SecGloss/s-gly) 之間轉譯的服務， (sid) 。
