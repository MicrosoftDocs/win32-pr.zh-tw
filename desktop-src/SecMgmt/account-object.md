# <a name="account-object"></a>Account 物件

**Account** 物件是用來將 [*許可權*](/windows/desktop/SecGloss/p-gly)、系統存取權和特殊配額指派給個別使用者或本機群組或群組的成員。

**帳戶** 物件中儲存的資訊包括：

-   使用者帳戶、群組帳戶或本機群組帳戶的 SID。
-   要指派給使用者或群組成員的許可權清單。 如果有數個 **帳戶** 物件指派給使用者的許可權，此清單就會包含所有指派的許可權。
