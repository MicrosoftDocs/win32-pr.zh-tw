# <a name="account-object-protection"></a>帳戶物件保護

[**帳戶**](account-object.md) 物件會以下列方式受到保護：

-   群組 **世界** 具有一般 \_ 執行。
-   群組 **本機系統 \_ 管理員** 具有 DELETE、GENERIC \_ READ、generic \_ WRITE、generic \_ EXECUTE 和 WRITE \_ DACL 存取權。
-   群組 **本機系統 \_ 管理員** 會被指派為帳戶物件的擁有者和主要群組。
