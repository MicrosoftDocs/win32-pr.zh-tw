# <a name="the-private-data-object-type"></a>私用資料物件類型

您可以使用私用資料物件來儲存加密形式的資料字串。 系統可以包含多個私用資料物件。 私用資料物件主要是用來支援伺服器帳戶密碼的儲存。 儲存在私用資料物件中的資訊，會使用產生的系統特定金鑰進行加密。
