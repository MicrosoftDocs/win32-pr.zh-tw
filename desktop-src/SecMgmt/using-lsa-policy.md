# <a name="using-lsa-policy"></a>使用 LSA 原則

下列主題描述如何使用 LSA 原則函數。

-   [使用 LSA Unicode 字串](using-lsa-unicode-strings.md) 說明如何將字串轉換為某些 LSA 原則函式所使用的結構格式。
-   [開啟原則物件控制碼](opening-a-policy-object-handle.md) 描述如何開啟本機 [**原則**](policy-object.md) 物件的控制碼。 許多 LSA 原則函式都需要這個控制碼做為輸入參數。
-   [管理原則資訊](managing-policy-information.md) 詳細資料如何設定及查詢本機系統和網域的全域原則資訊。
-   [接收原則變更事件](receiving-policy-change-events.md) 描述如何建立和註冊事件，以在原則資訊變更時接收通知。
-   [管理帳戶許可權](managing-account-permissions.md) 說明如何建立、刪除和列舉帳戶，以及如何新增和移除這些帳戶的許可權。
-   [管理受信任的網域資訊](managing-trusted-domain-information.md) 說明如何建立、列舉及刪除信任的網域，以及如何設定和取得受信任的網域資訊。
-   [在名稱和 Sid 之間](translating-between-names-and-sids.md) 轉譯會說明如何將 sid 對應到帳戶名稱，反之亦然。
-   [對應 Posix 識別碼](mapping-posix-identifiers.md) 說明如何將 sid 對應至32位 Posix 識別碼。
-   [儲存私用資料](storing-private-data.md) 會說明如何儲存和取出私用資料字串。

如需如何使用 LSA 登入和驗證使用者的詳細資訊，請參閱 [LSA 驗證](/windows/desktop/SecAuthN/lsa-authentication)。
