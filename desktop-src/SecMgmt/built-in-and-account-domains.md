# <a name="built-in-and-account-domains"></a>內建和帳戶網域

以 Windows 為基礎的系統上的網域分為下列兩個類別：

-   [非網域控制站的電腦](#computers-that-are-not-domain-controllers)
-   [屬於網域控制站的電腦](#computers-that-are-domain-controllers)

## <a name="computers-that-are-not-domain-controllers"></a>非網域控制站的電腦

安全性帳戶管理員 (SAM) 資料庫位於每部電腦上，而且會管理內建網域和帳戶網域的帳戶。



| 網域          | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
|-----------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 內建網域 | 此網域包含安裝作業系統時所建立的預設本機群組，例如系統管理員和使用者群組。 此網域的名稱是內建的當地語系化版本。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| 帳戶網域  | 帳戶網域可以包含使用者、群組和本機群組帳戶。 此網域中的系統管理員帳戶。 在工作站或成員伺服器的帳戶網域中定義的帳戶，限制為存取位於帳戶所在實體電腦上的資源。 在不屬於網路一部分，因此沒有 [主域](primary-and-trusted-domains.md)的系統上，會使用帳戶網域來存放提供電腦存取權的所有帳戶。 在屬於網路一部分的系統上，會使用電腦的帳戶網域來存放未存取網路資源的帳戶。 需要存取網路的帳戶必須定義在網域控制站的帳戶網域中。<br/> 系統會在系統安裝階段指派此網域的名稱，並由電腦名稱稱決定。 如果電腦名稱稱變更，則在重新開機電腦之前，將不會更新帳戶網域名。<br/> |



 

## <a name="computers-that-are-domain-controllers"></a>屬於網域控制站的電腦

網域控制站沒有內建或帳戶網域。 此外，這些系統會使用 Microsoft Active Directory Directory 服務來儲存帳戶存取訊號，而不是 SAM 資料庫。 如需詳細資訊，請參閱 [Active Directory](/windows/desktop/AD/active-directory-domain-services)。
