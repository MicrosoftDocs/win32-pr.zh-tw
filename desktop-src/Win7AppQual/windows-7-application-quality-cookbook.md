# <a name="windows-7-and-windows-server-2008-r2-application-quality-cookbook"></a>Windows 7 和 Windows Server 2008 R2 應用程式品質操作手冊

*Windows 7 和 Windows server 2008 R2 應用程式品質指南* 讓熟悉應用程式開發人員如何驗證其應用程式與新作業系統之間的相容性，並概述 Windows 7 和 Windows Server 2008 R2 中一些已知的應用程式相容性問題。 它也指出效能、可靠性和使用性方面的差異，並提供詳細白皮書和其他開發人員指南的連結。

## <a name="overview"></a>概觀

Windows 7 和 Windows Server 2008 R2 引進最新的作業系統技術和軟體發展平臺，供全球的應用程式開發人員和企業使用。 在進一步增強 Windows 的安全性、可靠性、效能和使用者體驗方面，已引進許多新功能，已改善現有功能，且已移除部分功能。

雖然 Windows 7 和 Windows server 2008 R2 與針對 Windows XP 撰寫的大部分個別應用程式高度相容、Windows server 2003、Windows Vista、Windows server 2008、Windows server 2008 R2 及其 service pack，但由於創新、安全性擴大和可靠性的緣故，某些相容性中斷是不可避免的。 整體來說，Windows 7 和 Windows Server 2008 R2 與現有應用程式的相容性很高。

本檔是以[Windows Vista 和 Windows Server 2008 應用程式相容性指南](/previous-versions/bb757005(v=msdn.10))中所述的概念為基礎。  (此資源可能無法在某些語言及國家/地區中使用。 ) 這份檔可讓您熟悉如何驗證應用程式與新作業系統之間的相容性，並概述 Windows 7 和 Windows Server 2008 R2 中一些已知的應用程式不相容問題。 但是，它也會指出效能、可靠性和使用性方面的差異，並提供詳細白皮書和其他開發人員指南的連結。

此外，Microsoft 正在投資數個新的增強功能和工具，讓您能夠建立更高品質的應用程式，並在 Windows 7 和 Windows Server 2008 R2 上的應用程式無法正常運作時進行疑難排解。 這些工具組括如何限定您的用戶端和伺服器應用程式，以參與 Windows 標誌計畫的指示。
