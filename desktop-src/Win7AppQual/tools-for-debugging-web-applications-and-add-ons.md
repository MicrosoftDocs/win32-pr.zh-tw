# <a name="tools-for-debugging-web-applications-and-add-ons"></a>用於調試 Web 應用程式和 Add-Ons 的工具

您可以藉由將程式碼寫入至最新的 web 標準，讓 web 應用程式或網站以原生方式在 Windows Internet Explorer 8 標準模式下運作，而不是[修正 web 應用程式和附加元件的相容性問題](remediating-web-applications-and-add-ons.md)。

若要完整評估您的應用程式或網站中的任何問題，您應該研究問題的根本原因。 例如，Internet Explorer 團隊使用開發人員工具判斷根本原因，並以邏輯方式減少可能的問題。 下列各節將說明數個可協助簡化偵錯工具的工具。



| 區段                                                                                                                    | 描述                                                                                                                                                                         |
|----------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Microsoft Expression Web SuperPreview](microsoft-expression-web-superpreview.md)                                         | 獨立的視覺化偵錯工具，可讓您更快速且更輕鬆地將您的網站從 Microsoft Internet Explorer 6 遷移至 Windows Internet Explorer 7 或 Internet Explorer 8。 |
| [Internet Explorer 相容性測試工具 (IECTT)](inernet-explorer-compatibility-test-tool--iectt-.md)                  | [Microsoft 應用程式相容性工具](/windows-hardware/get-started/adk-install)組的元件，可協助您診斷 web 應用程式中的問題。       |
| [Internet Explorer 8 開發人員工具](internet-explorer-8-developer-tools.md)                                             | Internet Explorer 8 中包含用於偵錯工具的開發人員工具。                                                                                                         |
| [Internet Explorer 8 相容性 Wizard (Aggiorno) ](inernet-explorer-8-compatibility-wizard--aggiorno-.md)              | 協力廠商工具可自動升級網站，以在 Internet Explorer 8 中正確轉譯，而不會中斷舊版瀏覽器的相容性。                                |
| [Fiddler Web 偵錯工具工具](fiddler-web-debugger-tool.md)                                                                 | 用來在網際網路與測試電腦之間捕獲網路流量的協力廠商工具。                                                                                           |
| [使用 Virtual PC 映射的應用程式相容性測試](application-compatibility-testing-using-virtual-pc-images.md) | 簡化相容性測試的 Virtual PC 映射。                                                                                                                              |
