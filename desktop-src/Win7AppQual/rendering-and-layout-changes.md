# <a name="rendering-and-layout-changes"></a>轉譯和版面配置變更

WindowsInternet Explorer 8 會更嚴格地解讀比舊版更多的 HTML 標籤，這項變更可能會導致網頁顯示錯誤。 例如，當網站在 Windows Internet Explorer 7 中正確顯示清單或資料表時，可能會在 Internet Explorer 8 中出現中斷。

您可以藉由減少網頁以找出格式錯誤的 HTML 來解決這些類型的問題。 如果您修正大部分的顯示問題，這些變更不會在舊版 Internet Explorer Windows 中造成問題，因為先前的版本只會忽略無關的 HTML。 如需這些變更的詳細資訊，請參閱 [網站相容性與 IE8](/archive/blogs/ie/site-compatibility-and-ie8) blog 文章。
