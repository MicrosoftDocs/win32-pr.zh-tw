# <a name="application-compatibility-testing-using-virtual-pc-images"></a>使用 Virtual PC 映射的應用程式相容性測試

若要開始測試 web 應用程式相容性，您必須建立測試環境。 為此工作建立及設定電腦的程式相當耗時，而且成本通常很高。

若要更輕鬆地進行相容性測試，您可以使用免費、預先建立的應用程式相容性 Virtual PC (VPC) 映射。 VPC 映射是預先設定的硬碟映射，您可以將其載入執行 [VIRTUAL PC 軟體](https://support.microsoft.com/help/14223/windows-xp-end-of-support)的現有電腦上。 這些映射可讓您在短時間內，以現有的應用程式徹底測試各種版本的 Microsoft Windows 和 Windows Internet Explorer。

所有目前支援的 Windows 版本都可使用 VPC 映射，並包含許多相容性測試資源和應用程式。 若要尋找測試映射，請參閱 [Internet Explorer 應用程式相容性 VPC 映射](https://www.microsoft.com/Downloads/details.aspx?FamilyID=21eabb90-958f-4b64-b5f1-73d0a413c8ef&amp;displaylang=en)。
