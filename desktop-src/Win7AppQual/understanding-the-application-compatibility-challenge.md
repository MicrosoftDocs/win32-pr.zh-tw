# <a name="understanding-the-application-compatibility-challenge"></a>瞭解應用程式相容性挑戰

Windows Internet Explorer 8 是 Windows 7 作業系統中的預設瀏覽器。 Internet Explorer 8 改進了許多功能，包括增強的平臺架構，其設計目的是為了讓使用者更安全且更快速地為使用者提供更好的設定和管理，以及為 網頁程式開發人員建立創新的內容。

其中有些功能使用新的轉譯和腳本引擎，也就是 Windows Internet Explorer 用來顯示 HTML 和 CSS 內容和執行腳本的核心元素。 架構變更和平臺增強功能帶來了應用程式相容性問題，您可能會在將應用程式遷移至舊版瀏覽器的 Internet Explorer 8 時遇到問題。

Internet Explorer 8 包含一些回溯相容性，但有一些變更 (包括支援 web 標準支援和安全性功能) 您可能需要升級為 Microsoft Internet Explorer 6 或舊版建立的 web 應用程式。

如需如何測試和解決 Internet Explorer 8 相容性問題的詳細資訊，請參閱 [遷移至 Internet Explorer 8 –資訊技術專業人員的資訊，以解決應用程式相容性](/previous-versions/windows/it-pro/windows-7/ff955640(v=ws.10))問題。
