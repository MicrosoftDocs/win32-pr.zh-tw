# <a name="microsoft-expression-web-superpreview"></a>Microsoft Expression Web SuperPreview

適用于 Internet Explorer 的 Microsoft Expression Web SuperPreview 是獨立的視覺化偵錯工具，可協助您對網站進行跨瀏覽器的相容性的偵錯工具。 Web SuperPreview 可確保您的網站在 Windows Internet Explorer 8 中正確運作，並維持與舊版 Windows Internet Explorer 的相容性。

Web SuperPreview 會根據您在電腦上安裝的版本，顯示您的網頁在 Microsoft Internet Explorer 6 以及 Windows Internet Explorer 7 或 Internet Explorer 8 中呈現的方式。 您可以並排顯示頁面，或做為重迭。 您可以使用尺規、輔助線和縮放和移動瀏覽，精確地識別版面配置中的差異。 您甚至可以比較您的頁面與目標瀏覽器呈現頁面的方式。

Web SuperPreview 也會識別專案的標記、大小、位置、套用的樣式，以及檔物件模型 (DOM) 樹狀結構中的位置，讓您可以快速修正錯誤。

> [!Note]  
> Web SuperPreview 是免費的應用程式，沒有任何期限，也不提供 Microsoft 的技術支援。

 

使用 Web SuperPreview 工具，您可以：

-   在開發電腦上的偵錯工具頁面。 Web SuperPreview 可直接與已安裝的 Microsoft Windows 瀏覽器搭配使用，並包含 Internet Explorer 6 轉譯引擎。 您可以搭配使用此引擎與其他版本的 Internet Explorer 或 Mozilla Firefox。
-   使用元素和 DOM 樹狀結構資訊，以更快速的方式進行偵錯工具。 Web SuperPreview 也會結合影像轉譯與 DOM 資料，因此您可以反白顯示影像內的特定 DOM 元素，並顯示不同瀏覽器之間的差異。 此工具會提供專案資訊（例如位置、維度和識別碼），以協助您找出導致相容性問題的元素。
-   支援 Windows 和 Macintosh 瀏覽器，包括 Internet Explorer 6 和更新版本、Firefox 3.6 和 Macintosh Safari。

如需此工具和其他開發人員工具的詳細資訊，請參閱 [Expression Studio 4 Web 總覽](https://www.microsoft.com/expression/products/StudioWebPro_Overview.aspx)、 [expression Web demo](https://msdn.microsoft.com/expression/ff723803.aspx)和 [運算式論壇](https://social.msdn.microsoft.com/forums/announce/threads)。
