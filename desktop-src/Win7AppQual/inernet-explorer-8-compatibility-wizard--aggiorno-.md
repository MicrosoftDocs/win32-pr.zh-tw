# <a name="internet-explorer-8-compatibility-wizard-aggiorno"></a>Internet Explorer 8 相容性 Wizard (Aggiorno) 

Windows Internet Explorer 8 中的相容性嚮導會自動升級網站，以在 Internet Explorer 8 中正確轉譯，而不會在舊版瀏覽器中中斷相容性。 Wizard 會將瀏覽器版本目標工作自動化，以維護頁面的外觀和功能。 如需此嚮導的詳細資訊，請參閱 [Internet Explorer 8 相容性](https://www.mobilize.net/)檢查程式快速入門。
