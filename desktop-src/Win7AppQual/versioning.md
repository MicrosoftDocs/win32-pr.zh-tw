# <a name="versioning-windows-7-and-windows-server-2008-r2-application-quality-cookbook"></a>Windows 7 和 Windows Server 2008 R2 應用程式品質操作手冊)  (版本控制

Windows Internet Explorer 最常見的應用程式相容性問題之一，就是使用者代理程式 (UA) 字串和版本向量。 Windows Internet Explorer 8 引進了新的 UA 字串，可協助應用程式偵測使用者正在執行的版本。 除了只增加與 UA 字串中 Internet Explorer 相關聯的 MSIE 值，Internet Explorer 8 還新增了 Trident/4.0 值，以協助您偵測 Internet Explorer 8 是否在相容性檢視模式中執行。 這些變更，加上硬式編碼版本檢查，可能會導致瀏覽器之間的相容性問題。
