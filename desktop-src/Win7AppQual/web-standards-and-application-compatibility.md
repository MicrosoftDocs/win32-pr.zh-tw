# <a name="web-standards-and-application-compatibility"></a>Web 標準和應用程式相容性

Web 標準提供了常見的技術基準，可讓使用者選取網頁瀏覽器，而不需要擔心使用所需的網站，讓 web 設計人員和開發人員能夠建立可供所有人使用的單一網站。

Web 瀏覽器的標準不斷演進，網頁瀏覽器會持續更新以採用這些標準。 因此，IT 專業人員和開發人員必須確保 web 應用程式與目前和新興的 web 標準和瀏覽器相容。 這項維護可確保 web 應用程式和網站相容于新版的瀏覽器，並協助防止許多可能發生的相容性問題。
