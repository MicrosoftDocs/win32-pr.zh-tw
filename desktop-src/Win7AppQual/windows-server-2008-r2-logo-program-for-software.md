# <a name="windows-server-2008-r2-logo-program-for-software"></a>適用于軟體的 Windows Server 2008 R2 標誌方案

## <a name="affected-platforms"></a>受影響的平臺

 **伺服器-** Windows Server 2008 R2  

### <a name="description"></a>Description

Windows Server 2008 R2 標誌程式已淘汰。

Windows Server 2008 R2 2012 測試提交不再接受 Microsoft 合作夥伴網路 (MPN) 專長認證或經認證的 Windows Server 標誌。

Windows Server 2012 R2 目前是「適用于 Windows Server 的認證」標誌的慣用平臺。 如需詳細資訊，請參閱 [企業雲端應用程式標誌認證方案](https://azure.microsoft.com/overview/commercial-marketplace/) 。
