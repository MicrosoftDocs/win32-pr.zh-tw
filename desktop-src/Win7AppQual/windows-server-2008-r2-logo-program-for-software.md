# <a name="windows-server-2008-r2-logo-program-for-software"></a>Windows適用于軟體的 Server 2008 R2 標誌方案

## <a name="affected-platforms"></a>受影響的平臺

 **伺服器-** Windows Server 2008 R2  

### <a name="description"></a>Description

Windows Server 2008 R2 標誌程式已淘汰。

WindowsMicrosoft 合作夥伴網路 (MPN) 專長認證或通過 Windows 伺服器標誌，即不再接受 Server 2008 R2 2012 測試提交。

Windows Server 2012R2 目前是認證 Windows 伺服器標誌的慣用平臺。 如需詳細資訊，請參閱[Enterprise 雲端應用程式標誌認證方案](https://azure.microsoft.com/overview/commercial-marketplace/)。
