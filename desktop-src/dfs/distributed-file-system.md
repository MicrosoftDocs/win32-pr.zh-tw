# <a name="distributed-file-system"></a>分散式檔案系統

分散式檔案系統 (DFS) 函式可讓您以邏輯方式將多部伺服器上的共用群組在一起，並以透明方式將共用連結至單一階層命名空間。 DFS 會在樹狀結構中組織網路上的共用資源。
