# <a name="reloadzone-method-of-the-microsoftdns_zone-class"></a>MicrosoftDNS 區域類別的 ReloadZone 方法 \_

**ReloadZone** 方法會從其資料庫重載 DNS 區域。

## <a name="syntax"></a>語法


```mof
void ReloadZone();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
