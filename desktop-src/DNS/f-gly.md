<span id="_dns_flat_name_space_gly"></span><span id="_DNS_FLAT_NAME_SPACE_GLY"></span>**一般命名空間**

可唯一識別 DNS 階層中主機的名稱，例如，在 Microsoft 的 products 階層中，名為 server1 的主機可能會有 server1.products.Microsoft.com 的 FQDN。
