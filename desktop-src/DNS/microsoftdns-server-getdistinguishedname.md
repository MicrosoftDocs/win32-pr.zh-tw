# <a name="getdistinguishedname-method-of-the-microsoftdns_server-class"></a>MicrosoftDNS 伺服器類別的 GetDistinguishedName 方法 \_

**GetDistinguishedName** 方法會抓取區域的 DS 分辨名稱。

## <a name="syntax"></a>語法


```mof
string GetDistinguishedName();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

傳回區域的 DS 分辨名稱。
