# <a name="dns-zones"></a>DNS 區域

DNS 區域是 (更精確的一組檔案或記錄，也就是對應至 DNS 階層式命名空間一部分的資源記錄專案) 的資料庫。 DNS 區域是用來說明哪些 DNS 伺服器負責 (的授權) ，以便解析 DNS 階層的指定區段的名稱解析查詢。 DNS 區域與網域結構的差異如下：區域可以由一或多個 DNS 網域組成。 Gadgets.widgets.microsoft.com 網域樹狀結構中的一個區域可能是小工具和 widget 網域的授權。 換句話說，DNS 區域不需要有與 DNS 網域之間的一對一關聯性。
