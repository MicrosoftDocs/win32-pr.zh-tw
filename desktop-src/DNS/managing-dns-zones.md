# <a name="managing-dns-zones"></a>管理 DNS 區域

DNS 區域是對應至 DNS 階層式命名空間一部分之 RR 專案的資料庫。 DNS 區域是用來界定哪些 DNS 伺服器有權解析 DNS 階層中指定區段的名稱解析查詢。

## <a name="administration-steps"></a>管理步驟

DNS WMI 提供者可從授權 DNS 伺服器本身或從遠端主機管理 DNS 區域。 使用 DNS WMI 提供者管理區域所需的一般步驟如下：

-   連接到 DNS WMI 提供者
-   取得區域實例
-   列出或修改區域屬性，或使用方法
