# <a name="name-resolution"></a>名稱解析

DNS 的目的是要提供名稱解析：比較主機名稱 (的程式，例如電腦名稱稱) 到 Rr 清單以及識別對應的 IP 位址。 反向名稱解析是將 IP 位址與 Rr 清單進行比較，並識別對應的主機名稱的程式。 當 DNS 伺服器收到查詢時，就會起始這些解析處理常式。
