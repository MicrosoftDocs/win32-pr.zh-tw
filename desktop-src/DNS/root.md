# <a name="root"></a>Root

階層命名空間中的最上層網域（例如 DNS 命名空間）稱為 [*根*](r-gly.md)。 根，定義為「 (沒有任何) 或更精確的」。 (點) ，則為必須起始整個網域樹系搜尋的基礎。
