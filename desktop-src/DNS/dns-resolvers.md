# <a name="dns-resolvers"></a>DNS 解析程式

[*解析程式*](r-gly.md)是隨附于 Windows 2000 傳輸控制通訊協定/網際網路通訊協定的軟體元件， (tcp/ip) 堆疊與 DNS 伺服器通訊，以建立並解析名稱解析查詢。 套件組合也適用于任何支援 DNS 的非 Microsoft 用戶端。 解析程式存在於 DNS 用戶端和 DNS 伺服器上。
