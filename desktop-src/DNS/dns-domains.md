# <a name="dns-domains"></a>DNS 網域

使用網際網路的大部分電腦專業人員都熟悉 DNS 網域;網域是 DNS 階層命名空間中的節點，而網域可進一步分割成子域。 例如，虛構 gadgets.widgets.microsoft.com 階層中的網域包括小工具、widget、microsoft 和 com 網域。
