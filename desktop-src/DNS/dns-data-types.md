# <a name="dns-data-types"></a>DNS 資料類型

以下是針對 DNS 定義的資料類型：


```C++
typedef DWORD IP4_ADDRESS;
```



<dl> <dt>

**IP4 \_ 位址**
</dt> <dd>

代表第四版網際網路協定 (IPv4) 位址的值。 例如，以 **0x7F000001** 的主機位元組順序表示的點十進位 IPv4 位址 **127.0.0.1**。

</dd> </dl>
