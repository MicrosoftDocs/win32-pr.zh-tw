# <a name="clearcache-method-of-the-microsoftdns_cache-class"></a>MicrosoftDNS Cache 類別的 ClearCache 方法 \_

**ClearCache** 方法會清除資源記錄的 DNS 伺服器快取。

## <a name="syntax"></a>語法


```mof
void ClearCache();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
