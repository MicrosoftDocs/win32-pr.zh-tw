# <a name="dns-wmi-provider-scripting-examples"></a>DNS WMI 提供者腳本範例

本節提供的腳本範例示範如何使用 DNS WMI 提供者編寫管理工作的腳本。

提供下列類型的腳本範例。



| 區段                                                                                     | 提供的範例                                                                                                                                                                          |
|---------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [管理 DNS 伺服器](dns-wmi-provider-samples-managing-a-dns-server.md)                 | 提供可讓您管理 DNS 伺服器的範例，包括停止、啟動或重新開機伺服器、列出區域、新增或刪除 IP 位址，以及其他。                |
| [管理 DNS 資源記錄](dns-wmi-provider-samples-managing-dns-resource-records.md) | 示範如何連接到 DNS WMI 提供者，以及如何執行與管理資源記錄相關聯的各種工作，包括新增、刪除、修改或列出資源記錄。 |
| [管理 DNS 區域](dns-wmi-provider-samples-managing-dns-zones.md)                       | 這裡的範例包括建立、修改和刪除區域，以及對 DNS 系統管理員造成的暫停、繼續、更新及其他區域工作。                                  |
