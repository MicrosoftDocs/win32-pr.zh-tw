# <a name="nomd-switch"></a>/nomd 參數

**/Nomd** 參數指定不會產生任何中繼資料 ( winmd) 檔。

``` syntax
midlrt /nomd
```

## <a name="switch-options"></a>切換選項

此參數沒有任何參數。

## <a name="remarks"></a>備註

## <a name="examples"></a>範例

**midlrt.exe 根據/nomd**
