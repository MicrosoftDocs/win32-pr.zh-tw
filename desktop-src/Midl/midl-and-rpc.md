# <a name="midl-and-rpc"></a>MIDL 和 RPC

當您開發 RPC 應用程式時，您可以使用文字編輯器來建立 IDL 檔案，以定義用戶端與伺服器之間的介面。 您可以使用 MIDL 編譯器來產生用戶端和伺服器程式所需的原始程式檔。 下列主題提供有關 RPC 應用程式使用 MIDL 編譯器的資訊：

-   [針對 RPC 介面產生的檔案](files-generated-for-an-rpc-interface.md)
-   [以特定32位或64位平臺為目標的存根](targeting-stubs-for-specific-32-bit-or-64-bit-platforms.md)

如需詳細資訊，請參閱 [建立 RPC 應用程式](/windows/desktop/Rpc/building-rpc-applications)。
