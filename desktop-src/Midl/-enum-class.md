# <a name="enum_class-switch"></a>/enum \_ 類別參數

**/Enum \_ 類別** 參數可讓您使用 c + +**列舉** 類別結構。

``` syntax
midlrt /enum_class 
```

## <a name="switch-options"></a>切換選項

此參數沒有任何參數。
