# <a name="helpcontext-attribute"></a><span data-ttu-id="3b3cc-104">helpcontext 屬性</span><span class="sxs-lookup"><span data-stu-id="3b3cc-104">helpcontext attribute</span></span>

## <a name="remarks"></a><span data-ttu-id="3b3cc-119">備註</span><span class="sxs-lookup"><span data-stu-id="3b3cc-119">Remarks</span></span>

<span data-ttu-id="3b3cc-120">\[ **HelpcoNtext** \] 屬性可以套用至下列元素： [**library**](library.md)、 \[ [**importlib**](importlib.md) \] 、 [**interface、interface**](interface.md) [**、**](dispinterface.md) [**module**](module.md)、 [**typedef**](typedef.md)、 **method**、 **property** 或 [**coclass**](coclass.md)。</span><span class="sxs-lookup"><span data-stu-id="3b3cc-120">The \[**helpcontext**\] attribute can be applied to the following elements: [**library**](library.md), \[[**importlib**](importlib.md)\], [**interface**](interface.md), [**dispinterface**](dispinterface.md), [**module**](module.md), [**typedef**](typedef.md), **method**, **property**, or [**coclass**](coclass.md).</span></span>
