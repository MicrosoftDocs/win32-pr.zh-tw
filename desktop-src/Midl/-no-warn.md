# <a name="no_warn-switch"></a>/no \_ 警告參數

**/No \_ 警告** 參數會指示 MIDL 編譯器隱藏警告訊息。

``` syntax
midl /no_warn
```

## <a name="switch-options"></a>切換選項

此參數沒有任何參數。

## <a name="remarks"></a>備註

使用 **/no \_ 警告** 參數相當於 **/W0**。

## <a name="examples"></a>範例

**midl/no \_ 警告檔案名 .idl**

**midl/W0 檔案名 .idl**
