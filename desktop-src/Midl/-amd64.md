# <a name="amd64-switch"></a>/amd64 參數

**/Amd64** 參數會指示 MIDL 編譯器為 Advanced 微型裝置64位環境產生存根檔案或類型程式庫檔案。

``` syntax
midl /amd64
```

## <a name="remarks"></a>備註

> [!Note]  
> MIDL.exe 命令列說明所描述的 **/x64** 參數已被取代。 應該改用 **/amd64** 。
