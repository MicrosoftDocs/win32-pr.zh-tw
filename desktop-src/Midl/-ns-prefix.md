# <a name="ns_prefix-switch"></a>/ns \_ 前置詞參數

**/Ns \_ 前置** 詞參數會在所有類型前面加上 "ABI" 前置詞。

``` syntax
midlrt /ns_prefix 
```

## <a name="switch-options"></a>切換選項

此參數沒有任何參數。
