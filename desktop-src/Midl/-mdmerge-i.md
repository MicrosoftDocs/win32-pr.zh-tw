# <a name="i-switch"></a>/i 參數

**/I** 參數會指定包含要合併之中繼資料檔案的目錄。

``` syntax
mdmerge /i metadata_directory
```

## <a name="switch-options"></a>切換選項

<dl> <dt>

*中繼資料 \_ 目錄* 
</dt> <dd>

指定包含要合併之中繼資料檔案的目錄。

</dd> </dl>
