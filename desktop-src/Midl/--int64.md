# <a name="__int64-attribute"></a>\_\_int64 屬性

關鍵字 **\_ \_ int64** 指定 MIDL 編譯器支援的有效整數。

``` syntax
__int64 identifier-name;
```

## <a name="parameters"></a>參數

<dl> <dt>

*識別碼-名稱* 
</dt> <dd>

指定有效的 MIDL 識別碼。 有效的 MIDL 識別碼最多包含31個英數位元和/或底線字元，而且開頭必須是字母或底線字元。

</dd> </dl>

如需如何使用 **\_ \_ int64** 的討論，請參閱 [**hyper-v**](hyper.md)。
