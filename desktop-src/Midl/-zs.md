# <a name="zs-switch"></a>/Zs 參數

**/Zs** 參數表示編譯器只會檢查語法，且不會產生輸出檔。

``` syntax
midl /Zs
midl /syntax_check
```

## <a name="switch-options"></a>切換選項

此參數沒有任何參數。

## <a name="remarks"></a>備註

此參數會覆寫所有其他參數，這些參數會指定輸出檔案的相關資訊。

您也可以使用 MIDL 編譯器參數 [**/syntax \_ 檢查**](-syntax-check.md)來指定語法檢查模式。

## <a name="examples"></a>範例

**midl/Zs 檔案名 .idl**

**midl/syntax \_ 檢查檔案名 .idl**
