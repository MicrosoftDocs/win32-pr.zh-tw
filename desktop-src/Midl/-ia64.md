# <a name="ia64-switch"></a>/ia64 參數

**/Ia64** 參數會指示 MIDL 編譯器為 Intel 架構64位環境產生存根檔案或類型程式庫檔案。

``` syntax
midl /ia64
```

## <a name="remarks"></a>備註

> [!Note]  
> 您無法使用 MIDL.exe 在 Windows 2000 上產生64位 tlb。
