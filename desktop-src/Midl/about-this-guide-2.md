# <a name="about-this-guide"></a>關於本指南

本節是 Microsoft 介面定義語言 (MIDL) 的參考，以及 MIDL 編譯器的使用者指南，它會從 IDL 檔案產生 C 語言檔案。

[使用 Midl 編譯器](using-the-midl-compiler-2.md)區段會列出 midl 必須交互操作的 C 編譯器和預處理器的需求。 本節也描述 MIDL 編譯器針對 RPC 存根、DCOM stub 和 OLE Automation 類型程式庫所產生的檔案。

[介面定義和類型程式庫](interface-definitions-and-type-libraries.md)一節提供有關如何建立 (idl) 和應用程式設定的介面定義 (ACF MIDL 的) 檔，包括 IDL 和 ACF 屬性的參考頁面。 另外還有關于如何匯入檔案和型別程式庫，以及匯入系統標頭檔的資訊。

[Midl Command-Line 參考](midl-command-line-reference.md)區段包含 midl 編譯器所辨識之每個命令列參數和切換選項的參考資訊。

[MIDL 語言參考](midl-language-reference.md)區段包含 Microsoft 介面定義語言中每個關鍵字的參考專案。

[Midl 編譯器錯誤和警告](midl-compiler-errors-and-warnings.md)區段會列出 MIDL 編譯器可以產生的錯誤和警告訊息。
