# <a name="partial-switch"></a>/partial 參數

**/Partial** 參數會指定針對 [**/metadata \_ dir**](-mdmerge-metadata-dir.md)參數所列出的中繼資料檔案解析未解析的類型。

``` syntax
mdmerge /partial 
```

## <a name="switch-options"></a>切換選項

此參數沒有任何參數。

## <a name="remarks"></a>備註

## <a name="examples"></a>範例

**mdmerge /partial**
