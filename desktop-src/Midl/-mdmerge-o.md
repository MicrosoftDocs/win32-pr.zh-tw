# <a name="o-switch-mdmerge"></a>/o switch (MDMERGE) 

**/O** 參數會指定用來組成中繼資料檔案的輸出目錄。

``` syntax
mdmerge /o output_directory
```

## <a name="switch-options"></a>切換選項

<dl> <dt>

*輸出 \_ 目錄* 
</dt> <dd>

指定包含合併中繼資料檔案的目錄。

</dd> </dl>
