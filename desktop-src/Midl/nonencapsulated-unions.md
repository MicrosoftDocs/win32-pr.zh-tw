# <a name="nonencapsulated-unions"></a>Nonencapsulated 聯集

Nonencapsulated 聯集是包含在結構中的聯集，其判別不包含在結構中。 Nonencapsulated 聯集的表示是因為 type 屬性 **\[ 參數 \_ 類型 \]** 的存在，而且欄位屬性 **\[ 參數 \_ 為 \]**。

如需 nonencapsulated 聯集的形式描述，請參閱聯 [**集**](union.md)。

等位的形狀在各平臺之間必須相同，以確保互連能力。

如需詳細資訊，請參閱 [封裝](encapsulated-unions.md)聯 [**集**](union.md)、等位、 **\[** [**參數 \_ 為**](switch-is.md) **\]** 和 **\[** [**參數 \_ 類型**](switch-type.md) **\]** 。
