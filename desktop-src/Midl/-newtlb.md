# <a name="newtlb-switch"></a>/newtlb 參數

這是選擇類型程式庫格式的預設設定。

``` syntax
midl /newtlb filename
```

## <a name="switch-options"></a>切換選項

此參數沒有任何參數。

## <a name="remarks"></a>備註

在 Windows 的最新版本上，此參數不會執行任何動作。 在不支援的 Windows 版本上，在 MIDL 命令列中指定這個參數將會產生錯誤。

## <a name="examples"></a>範例

**midl/newtlb 檔案名 .idl**

**midl 檔案名 .idl**
