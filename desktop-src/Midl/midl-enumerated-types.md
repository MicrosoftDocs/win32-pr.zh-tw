# <a name="midl-enumerated-types"></a>MIDL 列舉類型

MIDL 編譯器不會將 [**列舉**](enum.md)宣告轉譯為 **\# define** 語句，因為它是由某些 DCE 編譯器所組成。 它會在產生的標頭檔中重現為 C 語言 **列舉** 宣告。
