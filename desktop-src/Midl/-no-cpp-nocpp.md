# <a name="no_cpp-nocpp"></a>/no \_ cpp、/nocpp

**/No \_ cpp** 參數指定 MIDL 編譯器不會呼叫 C 預處理器來預先處理 IDL 檔案。

**/No \_ cpp** 參數優先于 [**/cpp \_ cmd**](-cpp-cmd.md)和 [**/cpp \_ opt**](-cpp-opt.md)參數。

## <a name="switch-options"></a>切換選項

``` syntax
midl /no_cpp
midl /nocpp
```

## <a name="example"></a>範例

**midl/no \_ cpp 檔案名 .idl**
