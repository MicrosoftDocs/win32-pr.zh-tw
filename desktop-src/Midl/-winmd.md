# <a name="winmd-switch"></a>/winmd 參數

[**/Winmd**](-metadata-dir.md)參數會將輸出中繼資料的名稱指定 ( winmd) 檔。

``` syntax
midlrt /winmd metadata_filename
```

## <a name="switch-options"></a>切換選項

<dl> <dt>

*中繼資料 \_ 檔案名* 
</dt> <dd>

指定中繼資料檔案的名稱。

</dd> </dl>
