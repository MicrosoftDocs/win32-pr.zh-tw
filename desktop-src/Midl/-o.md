# <a name="o-switch"></a>/o 參數

**/O** 參數會將輸出（例如錯誤訊息和警告）從標準輸出重新導向至指定的檔案。

``` syntax
midl /o outputfile
```

## <a name="switch-options"></a>切換選項

<dl> <dt>

*outputfile* 
</dt> <dd>

指定 MIDL 編譯器將會重新導向輸出的檔案名，例如錯誤訊息和警告。

</dd> </dl>
