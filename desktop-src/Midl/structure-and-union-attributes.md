# <a name="structure-and-union-attributes"></a>Structure 和 Union 屬性

使用 **switch \_** \* 屬性可指定遠端程序呼叫中聯集的特性。 您可以使用 [ [**略**](ignore.md) 過] 屬性，將某些結構或等位成員指定為用戶端應用程式的本機。



| 屬性                           | 使用方式                                                                                                                         |
|-------------------------------------|-------------------------------------------------------------------------------------------------------------------------------|
| [**開關**](switch.md)            | 選取封裝聯集的判別。                                                                           |
| [**切換 \_ 為**](switch-is.md)     | 識別 nonencapsulated 聯集的判別。                                                                      |
| [**切換 \_ 類型**](switch-type.md) | 識別 nonencapsulated 聯集的判別類型。                                                          |
| [**忽略**](ignore.md)            | 指定包含在結構或等位中的指標，以及指標所指出的物件不會傳送。 |



 

您也可以使用 [陣列和調整大小的指標屬性](array-and-sized-pointer-attributes.md) 來指定結構或等位成員的特性。
