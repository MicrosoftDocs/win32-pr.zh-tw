# <a name="no_format_opt-switch"></a>/no \_ 格式 \_ 選擇參數

**/No \_ 格式 \_ 選擇** 參數會變更 MIDL 優化類型和程式描述項的方式。

``` syntax
midl /no_format_opt
```

## <a name="switch-options"></a>切換選項

此參數沒有任何參數。

## <a name="remarks"></a>備註

根據預設，MIDL 會排除重複的類型和程式描述項，以縮減產生的 stub 程式碼大小。 使用 **/no \_ 格式 \_ 選擇** 參數會關閉此優化行為。

## <a name="examples"></a>範例

**midl/no \_ 格式 \_ opt mylean .idl**
