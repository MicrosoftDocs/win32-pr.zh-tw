# <a name="local-attribute"></a><span data-ttu-id="170c4-104">local 屬性</span><span class="sxs-lookup"><span data-stu-id="170c4-104">local attribute</span></span>

<span data-ttu-id="170c4-105">**\[ Local \]** 屬性指定對 MIDL 編譯器而言，介面或函式不是遠端的。</span><span class="sxs-lookup"><span data-stu-id="170c4-105">The **\[local\]** attribute specifies to the MIDL compiler that an interface or function is not remote.</span></span>

``` syntax
[ 
    local 
    [, interface-attribute-list] 
] 
interface interface-name
{
}

[ 
    object, 
    uuid(string-uuid), 
    local [, interface-attribute-list] 
]
interface interface-name
{
}

[ local [, function-attribute-list] ] function-declarator ;
```

## <a name="remarks"></a><span data-ttu-id="170c4-123">備註</span><span class="sxs-lookup"><span data-stu-id="170c4-123">Remarks</span></span>

<span data-ttu-id="170c4-124">**\[ 本機 \]** 屬性可以套用至個別的函式或整個介面。</span><span class="sxs-lookup"><span data-stu-id="170c4-124">The **\[local\]** attribute can be applied to individual functions or to the interface as a whole.</span></span>

<span data-ttu-id="170c4-125">在介面標頭中使用時， **\[ 本機 \]** 屬性可讓您使用 MIDL 編譯器作為標頭產生器。</span><span class="sxs-lookup"><span data-stu-id="170c4-125">When used in the interface header, the **\[local\]** attribute allows you to use the MIDL compiler as a header generator.</span></span> <span data-ttu-id="170c4-126">編譯器不會為任何函式產生存根，也不會確定可以傳送標頭。</span><span class="sxs-lookup"><span data-stu-id="170c4-126">The compiler does not generate stubs for any functions and does not ensure that the header can be transmitted.</span></span>

<span data-ttu-id="170c4-127">若為 RPC 介面， **\[ 本機 \]** 屬性不能與 **\[** [**uuid**](uuid.md) **\]** 屬性同時使用。</span><span class="sxs-lookup"><span data-stu-id="170c4-127">For an RPC interface, the **\[local\]** attribute cannot be used at the same time as the **\[**[**uuid**](uuid.md)**\]** attribute.</span></span> <span data-ttu-id="170c4-128">**\[ Uuid \]** 或 **\[ local \]** 都必須存在於介面標頭中，而您所選擇的必須只出現一次。</span><span class="sxs-lookup"><span data-stu-id="170c4-128">Either **\[uuid\]** or **\[local\]** must be present in the interface header, and the one you choose must occur exactly once.</span></span>

<span data-ttu-id="170c4-129">針對 (物件介面屬性識別的 COM 介面 **\[** [](object.md) **\]**) ，介面屬性清單可以包含 **\[ 本機 \]** 屬性，即使 **\[** [**uuid**](uuid.md) **\]** 屬性存在也一樣。</span><span class="sxs-lookup"><span data-stu-id="170c4-129">For a COM interface (identified by the **\[**[**object**](object.md)**\]** interface attribute), the interface attribute list can include the **\[local\]** attribute even though the **\[**[**uuid**](uuid.md)**\]** attribute is present.</span></span>

<span data-ttu-id="170c4-130">在個別函式中使用時， **\[ 區域 \]** 屬性會指定不會產生任何存根的本機程式。</span><span class="sxs-lookup"><span data-stu-id="170c4-130">When used in an individual function, the **\[local\]** attribute designates a local procedure for which no stubs are generated.</span></span> <span data-ttu-id="170c4-131">使用 **\[ local \]** 作為函式屬性是 DCE IDL 的 Microsoft 擴充功能。</span><span class="sxs-lookup"><span data-stu-id="170c4-131">Using **\[local\]** as a function attribute is a Microsoft extension to DCE IDL.</span></span> <span data-ttu-id="170c4-132">因此，當您使用 MIDL [**/osf**](-osf.md) 參數進行編譯時，就無法使用這個屬性。</span><span class="sxs-lookup"><span data-stu-id="170c4-132">Therefore this attribute is not available when you compile using the MIDL [**/osf**](-osf.md) switch.</span></span>

<span data-ttu-id="170c4-133">請注意，沒有屬性的介面可以匯入基底 IDL 檔案中。</span><span class="sxs-lookup"><span data-stu-id="170c4-133">Note that an interface without attributes can be imported into a base IDL file.</span></span> <span data-ttu-id="170c4-134">不過，介面必須只包含沒有任何程式的資料類型。</span><span class="sxs-lookup"><span data-stu-id="170c4-134">However, the interface must contain only datatypes with no procedures.</span></span> <span data-ttu-id="170c4-135">如果介面中包含一個程式，則必須指定 local 或 UUID 屬性。</span><span class="sxs-lookup"><span data-stu-id="170c4-135">If even one procedure is contained in the interface, a local or UUID attribute must be specified.</span></span>

## <a name="examples"></a><span data-ttu-id="170c4-136">範例</span><span class="sxs-lookup"><span data-stu-id="170c4-136">Examples</span></span>

``` syntax
/* IDL file #1 */ 
[
    local
] 
interface local_procs 
{ 
    void MyLocalProc(void);
} 
 
/* IDL file #2 */ 
[
    object, 
    uuid(12345678-1234-1234-123456789ABC), 
    local
] 
interface local_object_procs : IUnknown
{ 
    void MyLocalObjectProc(void);
} 
 
/* IDL file #3 */ 
[
    uuid(87654321-1234-1234-123456789ABC)
] 
interface mixed_procs 
{ 
    [local] void MyLocalProc(void); 
    HRESULT MyRemoteProc([in] short sParam); 
}
```
