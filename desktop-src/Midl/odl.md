# <a name="odl-attribute"></a>odl 屬性

MKTYPLIB.EXE 需要 ODL 介面上的 **\[ odl \]** 屬性。 MIDL 編譯器不需要 **\[ \] odl** 屬性，只會辨識出與舊版 **odl** 檔案的相容性。

> [!Note]  
> Mktyplib.exe 工具已淘汰。 請改用 MIDL 編譯器。
