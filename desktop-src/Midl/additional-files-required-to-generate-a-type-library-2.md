# <a name="additional-files-required-to-generate-a-type-library"></a>產生類型程式庫所需的其他檔案

若要編譯包含程式庫語句的 IDL 檔案，OLE 和 OLE Automation DLL 檔案必須在您的系統上。 系統會自動安裝這些檔案。
