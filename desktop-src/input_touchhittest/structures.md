# <a name="touch-hit-testing-structures"></a>觸控點擊測試結構

本節中的主題提供 [觸控點擊測試](touch-hit-testing-portal.md) 結構的參考規格。

## <a name="in-this-section"></a>本節內容

| 主題 | 描述 |
| --- | --- |
| [**TOUCH_HIT_TESTING_INPUT**](/windows/win32/api/winuser/ns-winuser-touch_hit_testing_input)<br/> | 包含觸控數位板所報告之觸控連絡人區域的相關資訊。<br/> |
| [**TOUCH_HIT_TESTING_PROXIMITY_EVALUATION**](/windows/win32/api/winuser/ns-winuser-touch_hit_testing_proximity_evaluation)<br/> | 包含點擊測試分數，指出物件是否為觸控連絡人區域的可能目標，相對於與觸控連絡人區域相交的其他物件。<br/> |
