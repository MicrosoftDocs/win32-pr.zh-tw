# <a name="system-restore-functions"></a>系統還原函式

下列函式會搭配系統還原使用。



| 函式                                             | 描述                                                                                                   |
|------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**SRRemoveRestorePoint**](/windows/desktop/api/SRRestorePtAPI/nf-srrestoreptapi-srremoverestorepoint) | 刪除指定的還原點。                                                                          |
| [**SRSetRestorePoint**](/windows/desktop/api/SRRestorePtAPI/nf-srrestoreptapi-srsetrestorepointa)       | 指定一組變更的開頭和結尾，使系統還原可以建立還原點。 |
