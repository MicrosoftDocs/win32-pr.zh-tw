# <a name="system-restore-wmi-classes"></a>系統還原 WMI 類別

下列 WMI 類別可搭配系統還原使用。



| WMI 類別                                          | Description                                                                                                                                |
|----------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------|
| [**SystemRestore**](systemrestore.md)             | 提供方法來停用和啟用監視、列出可用的還原點，以及在本機系統上起始還原。    |
| [**SystemRestoreConfig**](systemrestoreconfig.md) | 提供屬性來控制排程的還原點建立頻率，以及每個磁片磁碟機上耗用的磁碟空間量。 |
