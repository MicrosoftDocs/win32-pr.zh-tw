# <a name="directx-graphics-and-gaming"></a>DirectX 圖形和遊戲

此內容著重于在 Win32 應用程式中使用 DirectX。 如需在 UWP 應用程式中使用 DirectX 的詳細資訊，請參閱 [Windows 10 遊戲開發指南 (uwp) ](/windows/uwp/gaming/e2e)。

| 主題 | 描述 |
|-|-|
| [使用 DirectX 圖形開始使用](./getting-started-with-directx-graphics.md) | Microsoft DirectX 圖形提供一組 Api，可讓您用來建立遊戲和其他高效能的多媒體應用程式。 DirectX 圖形包含高效能的2D 和3D 圖形支援。 |
| [使用 COM 進行 DirectX 程式設計](prog-dx-with-com.md) | Microsoft 元件物件模型 (COM) 是數種技術所使用的物件導向程式設計模型，包括大量的 DirectX API 介面。 |
| [Direct2D](./direct2d/direct2d-portal.md) | Direct2D 是一種硬體加速的即時模式 2D 圖形 API，能夠以高效能和高品質來呈現 2D 幾何、點陣圖和文字。 |
| [Direct3D](./direct3d.md) | Direct3D 可讓您建立適用于遊戲和科學應用程式的立體圖形。 |

| [DirectXMath](./dxmath/directxmath-portal.md) | DirectXMath 提供最佳且可移植的介面，適用于單精確度浮點向量上的算術和線性代數運算 (2D、3D 和 4D) 或矩陣 (3 ×3和4× 4) 。 |
| [windowsnumerics .h Api](./numerics_h/windowsnumerics-h-apis-portal.md) | `windowsnumerics.h`標頭檔會定義 Windows 的 c + + 向量和矩陣類型。 |
