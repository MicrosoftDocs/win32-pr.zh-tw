# <a name="span-idvspixenginepixenginetexturedescriptorspanpixenginetexturedescriptor-structure"></a><span id="vspixengine.pixenginetexturedescriptor"></span>PixEngineTextureDescriptor 結構

代表材質資源的描述。

## <a name="syntax"></a>語法


```C++
} PixEngineTextureDescriptor;
```

## <a name="members"></a>成員

**formatDesc**  
描述材質的格式。

**mipmapCount**  
與材質相關聯的 mipmap 數目。

**arrayCount**  
材質是材質陣列時，即為數組中的材質數目。

**深度**  

**sampleCount**  
與材質相關聯的樣本數目。
