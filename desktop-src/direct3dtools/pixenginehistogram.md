# <a name="span-idvspixenginepixenginehistogramspanpixenginehistogram-structure"></a><span id="vspixengine.pixenginehistogram"></span>PixEngineHistogram 結構

代表材質的長條圖。

## <a name="syntax"></a>語法


```C++
} PixEngineHistogram;
```

## <a name="members"></a>成員

**horizontalMin**  
水準軸中每個 X、Y、Z 和 W 元件的最小值 (長條圖的網域) 。

**horizontalMax**  
水準軸中每個 X、Y、Z 和 W 元件的最大值 (長條圖的網域) 。

**verticalMin**  
垂直軸中每個 X、Y、Z 和 W 元件的最小值 (長條圖的範圍) 。

**verticalMax**  
垂直軸中每個 X、Y、Z 和 W 元件的最大值 (長條圖的範圍) 。

**dataLength**  
長條圖中所考慮的樣本數目。
