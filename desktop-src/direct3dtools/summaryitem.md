# <a name="span-idvspixenginesummaryitemspansummaryitem-structure"></a><span id="vspixengine.summaryitem"></span>SummaryItem 結構

表示事件的摘要資訊。

## <a name="syntax"></a>語法


```C++
} SummaryItem;
```

## <a name="members"></a>成員

**名稱**  
COM 字串，其中包含摘要事件的名稱。

**值**  
COM 字串，其中包含與事件相關的值。

**識別碼**  
事件的識別碼。

**ParentID**  
父事件的識別碼（如果有的話）。
