代表材質配量的索引

## <a name="syntax"></a>語法


```C++
} PixEngineTextureSliceIndex;
```

## <a name="members"></a>成員

**arrayIndex**  
當材質是材質陣列時，即為配量的陣列索引。

**mipmapLevel**  
配量的 mipmap 層級。

**sampleIndex**  
配量的範例索引。

**zSlice**  
配量的深度 (z) 。
