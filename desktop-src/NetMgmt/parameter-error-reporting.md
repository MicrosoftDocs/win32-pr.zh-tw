# <a name="parameter-error-reporting"></a>參數錯誤報表

Add 和 SetInfo 網路管理函式會傳回發生錯誤之參數的索引。 呼叫端可能會傳遞 *>parm \_ err* 參數的 **Null** 指標，指出該欄位不應由函式設定。
