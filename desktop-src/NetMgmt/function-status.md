# <a name="function-status"></a>函數狀態

網路管理功能在成功時傳回零;非零傳回碼表示錯誤。 因為網路管理函式使用 RPC，所以錯誤定義包含 RPC 錯誤碼。 如需詳細資訊，請參閱 [網路管理錯誤碼](network-management-error-codes.md)。
