# <a name="nls-support"></a>NLS 支援

網路管理函式會採用 Unicode 字串做為輸入，並在輸出上提供 Unicode 字串。 如果您的應用程式通常會使用 ANSI 字串，則必須在適當的情況下小心轉換成和自 Unicode。
