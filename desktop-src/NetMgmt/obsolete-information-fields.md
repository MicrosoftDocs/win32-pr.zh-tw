# <a name="obsolete-information-fields"></a>過時的資訊欄位

核心資訊結構中的許多資訊欄位將會過時。 這些欄位會保留在資訊結構中，以與16位版本的 Windows 相容，並且會在其他 Windows 系統上傳回智慧型預設值。
