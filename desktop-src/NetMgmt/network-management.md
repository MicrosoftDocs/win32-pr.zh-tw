# <a name="network-management"></a>網路管理

## <a name="purpose"></a>目的

網路管理功能提供管理使用者帳戶和網路資源的能力。 網路管理功能所提供的許多功能都不是由其他網路功能所提供。 但是，如果功能是由另一組函式提供，則網路管理功能的檔會將您參考到其他可用於相同工作的函式。

## <a name="developer-audience"></a>開發人員對象

網路管理函式是設計來供 C/c + + 程式設計人員使用。 程式設計人員也應該熟悉 Windows 網路概念

## <a name="run-time-requirements"></a>執行階段需求求

網路管理功能可以在所有 Windows 平臺上使用。 當網路管理平臺限制的特定執行或功能存在時，就會在檔中清楚注明它們。
