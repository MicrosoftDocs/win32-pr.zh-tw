# <a name="neterrorlogwrite"></a>NetErrorLogWrite

**NetErrorLogWrite** 函式已過時。 它隨附于與16位版本的 Windows 的相容性。 其他應用程式應該使用 [事件記錄](/windows/desktop/EventLog/event-logging)。
