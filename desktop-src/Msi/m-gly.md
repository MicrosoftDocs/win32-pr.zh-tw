# <a name="m-windows-installer"></a>M (Windows Installer) 

[](a-gly.md) [B](b-gly.md) [C](c-gly.md) [D](d-gly.md) [E](e-gly.md) [F](f-gly.md) [G](g-gly.md) H [I](i-gly.md) J K L M N [O](o-gly.md) [P](p-gly.md) [Q](q-gly.md) [R](r-gly.md) [S](s-gly.md) [T](t-gly.md) [](u-gly.md) [W X](v-gly.md) Y Z

<dl> <dt>

<span id="_msi_managed_application_gly"></span><span id="_MSI_MANAGED_APPLICATION_GLY"></span>**受控應用程式**
</dt> <dd>

系統許可權也稱為較高的許可權，用來安裝應用程式。 如果應用程式是 Windows 2000 上的每一電腦安裝，或是在 Windows 2000 上使用群組原則指派或發佈應用程式，則會在系統上管理應用程式。

</dd> <dt>

<span id="_msi_merge_using_windows_installer_gly"></span><span id="_MSI_MERGE_USING_WINDOWS_INSTALLER_GLY"></span>**合併**
</dt> <dd>

結合兩個安裝資料庫來形成單一資料庫。 如需詳細資訊，請參閱 [合併資料庫](merging-databases.md)。

</dd> <dt>

<span id="_msi_merge_module_gly"></span><span id="_MSI_MERGE_MODULE_GLY"></span>**merge 模組**
</dt> <dd>

資料庫與安裝套件合併，以將新元件傳遞至應用程式。 無法單獨安裝合併模組。 如需詳細資訊，請參閱 [合併模組](merge-modules.md)。

</dd> <dt>

<span id="_msi_microsoft_windows_installer_gly"></span><span id="_MSI_MICROSOFT_WINDOWS_INSTALLER_GLY"></span>**Microsoft Windows Installer**
</dt> <dd>

用於管理在32位平臺上安裝應用程式的用戶端安裝程式服務。 應用程式必須封裝 [*在封裝中。*](p-gly.md) 如需詳細資訊，請參閱[關於 Windows Installer](about-windows-installer.md)。

</dd> <dt>

<span id="_msi_.msi_file_gly"></span><span id="_MSI_.MSI_FILE_GLY"></span>**.msi 檔案**
</dt> <dd>

COM 結構化儲存體檔案，其中包含安裝應用程式所需的指示和資料。 每個 [*封裝*](p-gly.md) 至少包含一個 .msi 檔案。 .msi 檔案包含 [*安裝程式資料庫*](i-gly.md)、 [*摘要資訊資料流程*](s-gly.md)，以及可能有一或多個 [*轉換*](t-gly.md) 和 [*內部原始*](i-gly.md)程式檔。 如需詳細資訊，請參閱[Windows Installer 的副檔名](windows-installer-file-extensions.md)。

</dd> <dt>

<span id="_msi_.msm_file_gly"></span><span id="_MSI_.MSM_FILE_GLY"></span>**.msm 檔案**
</dt> <dd>

[Merge 模組](merge-modules.md)。 如需詳細資訊，請參閱[Windows Installer 的副檔名](windows-installer-file-extensions.md)。

</dd> <dt>

<span id="_msi_.msp_file"></span><span id="_MSI_.MSP_FILE"></span>**.msp 檔**
</dt> <dd>

[修補套件](patch-packages.md)。 如需詳細資訊，請參閱[Windows Installer 的副檔名](windows-installer-file-extensions.md)。

</dd> <dt>

<span id="_msi_.mst_file_gly"></span><span id="_MSI_.MST_FILE_GLY"></span>**.mst 檔**
</dt> <dd>

轉換套件。 如需詳細資訊，請參閱 [合併和轉換](merges-and-transforms.md)。 如需詳細資訊，請參閱[Windows Installer 的副檔名](windows-installer-file-extensions.md)。

</dd> </dl>
