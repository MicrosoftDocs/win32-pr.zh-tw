# <a name="admintoolsfolder-property"></a>AdminToolsFolder 屬性

**AdminToolsFolder** 屬性包含儲存系統管理工具之檔案系統目錄的完整路徑。 Microsoft Management Console (MMC) 將自訂的主控台儲存至此目錄，並與使用者漫遊。 如果已設定 [**ALLUSERS**](allusers.md) 屬性，此屬性會指向檔案系統目錄，其中包含電腦上所有使用者的系統管理工具。
