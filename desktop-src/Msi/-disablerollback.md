# <a name="disablerollback-property"></a>DISABLEROLLBACK 屬性

設定 **DISABLEROLLBACK** 屬性會停用目前設定的 [rollback](rollback-installation.md) 選項。 將此屬性設定為1，以防止安裝程式在安裝期間產生復原腳本，並儲存已刪除檔案的複本。

## <a name="remarks"></a>備註

預設會啟用復原。
