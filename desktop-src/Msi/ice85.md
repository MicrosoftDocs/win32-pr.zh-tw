# <a name="ice85"></a>ICE85

ICE85 會驗證 [MoveFile 資料表](movefile-table.md) 的 [未使用] 資料行是否為有效的完整檔案名。 此欄位可包含萬用字元 (\* 和？ ) 。

## <a name="result"></a>結果

ICE85 會張貼下列錯誤。



| ICE85 錯誤                                                      | Description                                                              |
|------------------------------------------------------------------|--------------------------------------------------------------------------|
| \[ \] 在 MoveFile 資料表中找到的「2」表格式不正確。 | MoveFile 資料表中的未通過欄位不是有效的完整檔案名 |
