# <a name="using-transforms"></a>使用轉換

轉換可讓您在不改變原始資料庫的情況下，將變更套用至資料庫，藉此提升應用程式安裝的彈性。 如需詳細資訊，請參閱 [資料庫轉換](database-transforms.md)。

[產生轉換](generating-a-transform.md)
