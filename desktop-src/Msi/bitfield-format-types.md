# <a name="bitfield-format-types"></a>位欄位格式類型

可設定資料的位格式類型可以用於整數資料庫欄位。 使用者應該從指定的子集選擇值。 這僅限慣例，Mergemod.dll 不會強制執行。 MsmConfigItemNonNullable 屬性是此資料格式的隱含，不需要設定。 Null 對此類型而言永遠不是有效的值。

下列位欄位格式類型存在。

[任意位欄位類型](arbitrary-bitfield-type.md)

位格式類型的可設定專案會用於整數資料行，而且一般可由任何整數取代。 使用者應該從指定的子集選擇值。 不過，這是根據慣例，Mergemod.dll 不會強制執行。 作者應該準備模組來處理任何值。
