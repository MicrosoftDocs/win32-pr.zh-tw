# <a name="editing-installer-shortcuts"></a>編輯安裝程式快捷方式

此 [快捷方式表](shortcut-table.md) 包含應用程式在使用者電腦上建立安裝程式快捷方式所需的資訊。 安裝程式快捷方式支援 [隨選安裝](installation-on-demand.md) 和 [公告](advertisement.md)。 請注意，在一般快速鍵的情況下，您無法編輯安裝程式快捷方式的屬性。

設定 [**DISABLEADVTSHORTCUTS**](disableadvtshortcuts.md) 屬性會停用產生的安裝程式快捷方式，並指定這些快速鍵應該改由一般快速鍵取代。
