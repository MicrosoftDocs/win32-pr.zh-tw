# <a name="e-windows-installer"></a>E (Windows Installer) 

[](a-gly.md) [B](b-gly.md) [C](c-gly.md) [D](d-gly.md) E [F](f-gly.md) [G](g-gly.md) H [I](i-gly.md) J K L [M](m-gly.md) N [O](o-gly.md) [P](p-gly.md) [Q](q-gly.md) [R](r-gly.md) [S](s-gly.md) [T](t-gly.md) [](u-gly.md) [W X](v-gly.md) Y Z

<dl> <dt>

<span id="_msi_elevated_gly"></span><span id="_MSI_ELEVATED_GLY"></span>**升高**
</dt> <dd>

以系統許可權執行的動作會以較高的許可權來呼叫。

</dd> <dt>

<span id="_msi_execution_phase_gly"></span><span id="_MSI_EXECUTION_PHASE_GLY"></span>**執行階段**
</dt> <dd>

當安裝程式執行安裝程式動作的腳本時。 在 Microsoft Windows 2000 中，此程式是由安裝程式服務所執行。 在受控應用程式中，腳本是以系統許可權執行。 如需詳細資訊，請參閱 [安裝機制](installation-mechanism.md)。

</dd> <dt>

<span id="_msi_execution_script_gly"></span><span id="_MSI_EXECUTION_SCRIPT_GLY"></span>**執行腳本**
</dt> <dd>

安裝的安裝程式動作。 執行腳本 [*會在安裝*](a-gly.md) 期間產生，並在 [*執行階段*](/windows)期間執行。 如需詳細資訊，請參閱 [安裝機制](installation-mechanism.md)。

</dd> <dt>

<span id="_msi_external_source_files_gly"></span><span id="_MSI_EXTERNAL_SOURCE_FILES_GLY"></span>**外部來源檔案**
</dt> <dd>

所安裝之應用程式的原始程式檔，儲存在 [*.msi*](m-gly.md)檔之外。 您可以將多個外部來源檔案壓縮並儲存在封裝 [*內含的*](c-gly.md) 封 [*包*](p-gly.md)檔中。

</dd> <dt>

<span id="_msi_external_user_interface_gly"></span><span id="_MSI_EXTERNAL_USER_INTERFACE_GLY"></span>**外部使用者介面**
</dt> <dd>

由安裝套件作者提供的 UI。 它不會使用安裝程式的內部 UI 功能。 如需詳細資訊，請參閱 [關於消費者介面](about-the-user-interface.md)。

</dd> </dl>
