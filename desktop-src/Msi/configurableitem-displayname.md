# <a name="configurableitemdisplayname-property"></a>ConfigurableItem DisplayName 屬性

[**ConfigurableItem**](configurableitem-object.md)物件的 **displayname** 屬性會從 [ModuleConfiguration 資料表](moduleconfiguration-table.md)的 displayname 資料行傳回值。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = ConfigurableItem.DisplayName
```



## <a name="property-value"></a>屬性值

## <a name="c"></a>C++

請參閱 [**取得 \_ DisplayName**](/windows/desktop/api/Mergemod/nf-mergemod-imsmconfigurableitem-get_displayname) 函數。
