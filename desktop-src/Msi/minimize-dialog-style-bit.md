# <a name="minimize-dialog-style-bit"></a>最小化對話方塊樣式位

如果設定此位，則可以將對話方塊最小化。 強制回應對話方塊會忽略此位，無法將其最小化。

## <a name="value"></a>值



| Decimal | 十六進位 | 常數                          |
|---------|-------------|-----------------------------------|
| 4       | 0x00000004  | **msidbDialogAttributesMinimize** |
