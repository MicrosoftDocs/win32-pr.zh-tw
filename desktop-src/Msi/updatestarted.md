# <a name="updatestarted-property"></a>UpdateStarted 屬性

當系統的變更開始進行此安裝時，安裝程式會設定 **UpdateStarted** 屬性，包括繼續暫止的安裝。

UI 條件陳述式使用此值來選取對話方塊。 如果設定此屬性，則會在發生錯誤時詢問使用者，並取消是否要還原或稍後再繼續。
