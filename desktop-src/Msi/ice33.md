# <a name="ice33"></a>ICE33

ICE33 會處理登錄 [表](registry-table.md) 中的專案，而且可能會針對註冊類別、副檔名、Progid、Shell 動詞、遠端伺服器 APPID、MIME 類型或 Typelibs 的每個資料表專案發出警告。 警告已過時，可予以忽略。
