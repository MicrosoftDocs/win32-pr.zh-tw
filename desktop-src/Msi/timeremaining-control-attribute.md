# <a name="timeremaining-control-attribute"></a>TimeRemaining 控制項屬性

這個屬性可讓文字控制項顯示安裝剩餘的大約分鐘數和秒數。 傳遞至文字控制項的記錄包含一個整數，代表剩餘的秒數。 文字控制項會在 [UIText 資料表](uitext-table.md) 中搜尋名為 TimeRemaining 的字串，並將整數格式化為適當的字串，以顯示分鐘和秒數。

## <a name="valid-controls"></a>有效的控制項

[Text](text-control.md)

## <a name="associated-bit-flags"></a>相關聯的位旗標

這個屬性不會使用位旗標。

## <a name="remarks"></a>備註

請參閱 [控制項屬性](control-attributes.md) 和您需要在 [控制項](controls.md)下建立的控制項。
