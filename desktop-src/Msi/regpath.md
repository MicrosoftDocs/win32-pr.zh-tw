# <a name="regpath"></a>RegPath

RegPath 資料類型是包含登錄路徑的文字字串。 登錄路徑可以包含屬性，如同 [格式化](formatted.md) 資料類型一樣。 RegPath 的開頭或結尾不能是反斜線 (\\) 。 檔案索引 \[ \# 鍵 \] 和 \[ $component 金鑰 \] 可以內嵌在路徑中，或以其他字元為開頭。
