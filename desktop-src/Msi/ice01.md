# <a name="ice01"></a>ICE01

ICE01 會驗證 ICE 機制是否正常運作。 這種 ICE 使用 [**time**](time.md) 屬性來取得時間，並傳回系統時間或無。

## <a name="result"></a>結果

ICE01 張貼訊息，提供安裝程式呼叫 ICE01 的時間。 這種 ICE 絕對不會傳回錯誤。
