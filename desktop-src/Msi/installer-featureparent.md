# <a name="installerfeatureparent-property"></a>FeatureParent 屬性

**FeatureParent** 屬性會指定功能的父項功能。 這個屬性的空字串表示功能沒有父系，而是根專案。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = Installer.FeatureParent
```



## <a name="property-value"></a>屬性值
