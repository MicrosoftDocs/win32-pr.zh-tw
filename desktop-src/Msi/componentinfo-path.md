# <a name="componentinfopath-property"></a>ComponentInfo 路徑屬性

元件的路徑。 這個屬性會呼叫 [**MsiGetComponentPathEx**](/windows/desktop/api/Msi/nf-msi-msigetcomponentpathexa) 函數。

**[Windows Installer 4.5 或更早版本](not-supported-in-windows-installer-4-5.md)：** 不支援。 從 Windows Installer 5.0 開始，可以使用這個屬性。
