# <a name="executable-files"></a>可執行檔

自訂動作可以啟動儲存在使用者電腦上，或包含在安裝套件中的可執行檔。

下列自訂動作類型會啟動可執行檔。



| 自訂動作類型                                 | Description                                         |
|----------------------------------------------------|-----------------------------------------------------|
| [自訂動作類型2](custom-action-type-2.md)   | 儲存在二進位資料表資料流程中的 EXE 檔案。           |
| [自訂動作類型18](custom-action-type-18.md) | 與產品一起安裝的 EXE 檔案。                  |
| [自訂動作類型50](custom-action-type-50.md) | EXE 檔案，其中包含由屬性值指定的路徑。 |
| [自訂動作類型34](custom-action-type-34.md) | 具有路徑參考目錄的 EXE 檔案。       |



 

請參閱 [所有自訂動作類型](summary-list-of-all-custom-action-types.md) 的摘要清單，以取得所有自訂動作類型的摘要，以及它們如何編碼至 [CustomAction 資料表](customaction-table.md)。
