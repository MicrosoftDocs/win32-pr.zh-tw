# <a name="ice96"></a>ICE96

ICE96 會確認已在[AdvtExecuteSequence 資料表](advtexecutesequence-table.md)中輸入[PublishFeatures 動作](publishfeatures-action.md)和[PublishProduct 動作](publishproduct-action.md)。 若沒有這些動作，就無法公告封裝。

## <a name="result"></a>結果

ICE96 會張貼下列警告。



| ICE96 警告                                                            | Description                                                                                                  |
|--------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------|
| PublishFeatures 動作在 AdvtExecuteSequence 資料表中是必要的。 | 若未在 AdvtExecuteSequence 資料表中包含 PublishFeatures 動作，就無法公告封裝。 |
| PublishProduct 動作在 AdvtExecuteSequence 資料表中是必要的。  | 若未在 AdvtExecuteSequence 資料表中包含 PublishProduct 動作，就無法公告封裝。  |
