# <a name="sequence-action"></a>順序動作

順序動作會排序並依序執行資料表中的動作。

[**SEQUENCE**](sequence.md)屬性定義此動作所使用的資料表。 資料表必須與 [InstallExecuteSequence 資料表](installexecutesequence-table.md)具有相同的架構。 資料表必須有 [動作]、[條件] 和 [順序] 資料行。

## <a name="sequence-restrictions"></a>順序限制

沒有任何順序限制。

## <a name="actiondata-messages"></a>ActionData 訊息

沒有任何 ActionData 訊息。
