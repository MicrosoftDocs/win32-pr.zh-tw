# <a name="restrictions-on-property-names"></a>屬性名稱的限制

所有屬性名稱都必須遵守這些限制。

-   屬性名稱是一種 [識別碼](identifier.md)，它是必須以字母或底線開頭的文字字串。 識別碼和屬性名稱可包含字母、數位、底線或句號;但開頭不能是數位或句號。
-   [公用屬性](public-properties.md) 名稱不能包含小寫字母。
-   [私用屬性](private-properties.md) 名稱必須包含一些小寫字母。
-   首碼為% 的屬性名稱代表系統和使用者環境變數。 這些都不會在 [屬性工作表](property-table.md)中輸入。 您只能使用 [環境資料表](environment-table.md)來修改環境變數的永久設定。
