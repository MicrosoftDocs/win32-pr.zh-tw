# <a name="last-printed-summary-property"></a>上次列印的摘要屬性

在安裝套件中，[ **上次列印的摘要** ] 屬性可以設定為 [系統管理安裝](administrative-installation.md) 期間的日期和時間，以在建立系統管理映射時錄製。 若為非系統管理安裝，此屬性與 [ [**建立時間/日期摘要**](create-time-date-summary.md) ] 屬性相同。

在轉換中，這個屬性應該是 null。

在 patch 封裝中，這個屬性應該是 null。
