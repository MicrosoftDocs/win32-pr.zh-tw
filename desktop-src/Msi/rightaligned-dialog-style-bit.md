# <a name="rightaligned-dialog-style-bit"></a>RightAligned 對話方塊樣式位

如果設定此樣式位，則會在對話方塊的右側對齊文字。

## <a name="value"></a>值



| Decimal | 十六進位 | 常數                              |
|---------|-------------|---------------------------------------|
| 256     | 0x00000100  | **msidbDialogAttributesRightAligned** |
