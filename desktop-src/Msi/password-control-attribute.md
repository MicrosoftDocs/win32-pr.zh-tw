# <a name="password-control-attribute"></a>密碼控制屬性

這個屬性會建立用於輸入密碼的編輯控制項。 控制項會將每個字元顯示為星號 (\*) ，因為它們是在控制項中輸入的。

設定 Password Control 屬性可防止安裝程式將與編輯控制項相關聯的屬性寫入記錄檔中。 如需詳細資訊，請參閱 [防止機密資訊寫入至記錄](preventing-confidential-information-from-being-written-into-the-log-file.md)檔。

## <a name="valid-controls"></a>有效的控制項

[編輯控制項](edit-control.md)

## <a name="associated-bit-flags"></a>相關聯的位旗標

這個屬性不會使用位旗標。

## <a name="remarks"></a>備註

如需詳細資訊，請參閱 [控制屬性](control-attributes.md) 和 [控制項](controls.md)。
