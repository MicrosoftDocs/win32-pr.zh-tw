# <a name="mergecloselog-method"></a>Merge. CloseLog 方法

[**Merge**](merge-object.md)物件的 **CloseLog** 方法會關閉目前的記錄檔。

## <a name="syntax"></a>語法


```JScript
Merge.CloseLog()
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。

## <a name="c"></a>C++

請參閱 [**CloseLog**](/windows/win32/api/mergemod/nf-mergemod-imsmmerge-closelog) 函式。
