# <a name="ice16"></a>ICE16

ICE16 會驗證 [屬性資料表](property-table.md)中 [**ProductName**](productname.md)屬性的值是否長度不超過63個字元。 DisplayName 的登錄機碼長度沒有任何限制。

## <a name="result"></a>結果

如果屬性工作表中設定的 ProductName 超過63個字元，ICE16 會張貼錯誤訊息。
