# <a name="transparent-control-attribute"></a>透明控制項屬性

如果透明控制項位是在文字控制項上設定，則會以透明的方式顯示控制項，並透過沒有任何字元的控制項顯示背景。 如果未設定此位，則文字控制項不是透明的。

## <a name="valid-controls"></a>有效的控制項

[Text](text-control.md)

## <a name="value"></a>值



| Decimal | 十六進位 | 常數                              |
|---------|-------------|---------------------------------------|
| 65536   | 0x00010000  | **msidbControlAttributesTransparent** |



 

## <a name="remarks"></a>備註

請參閱 [控制項屬性](control-attributes.md) 和您需要在 [控制項](controls.md)下建立的控制項。
