# <a name="ccp_drive-property"></a>CCP \_ 磁片磁碟機屬性

[ **CCP \_ 磁片磁碟機** ] 屬性會設定為 [RMCCPSearch](rmccpsearch-action.md)所要搜尋之抽取式磁碟區的根路徑。 RMCCPSearch 動作會使用檔案簽章來驗證合格產品是否已安裝在系統上，然後再執行升級安裝。

通常會透過使用者介面或自訂動作來設定這個屬性。 在執行 [RMCCPSearch](rmccpsearch-action.md) 動作之前，應該先設定這個屬性。
