# <a name="importing-and-exporting"></a>匯入和匯出

您可以使用安裝程式將文字封存匯入至使用中的資料庫，以及將資料庫檔案匯出至文字封存。 這對以文字為基礎的原始檔控制系統很有用。

若要將文字封存匯入作用中的資料庫，請呼叫 [**MsiDatabaseImport**](/windows/desktop/api/Msiquery/nf-msiquery-msidatabaseimporta) 函式。

若要將資料庫檔案匯出至文字封存，請呼叫 [**MsiDatabaseExport**](/windows/desktop/api/Msiquery/nf-msiquery-msidatabaseexporta) 函數。
