# <a name="authoring-merge-module-summary-information-streams"></a>撰寫合併模組摘要資訊資料流程

撰寫合併模組時，需要摘要資訊資料流程才能包含這些必要的摘要資訊屬性。

每個合併模組都必須有合併模組摘要資訊資料流程與下列摘要資訊屬性：

-   [**範本摘要**](template-summary.md)
-   [**修訂編號摘要**](revision-number-summary.md)
-   [**頁面計數摘要**](page-count-summary.md)
-   [**字數統計摘要**](word-count-summary.md)

如需合併模組中摘要資訊屬性的完整清單，請參閱 [合併模組摘要資訊資料流程參考](merge-module-summary-information-stream-reference.md)。
