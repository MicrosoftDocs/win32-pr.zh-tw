# <a name="recordfieldcount-property"></a>FieldCount 屬性

[**記錄**](record-object.md)物件的 **FieldCount** 屬性是唯讀屬性，會傳回記錄中的欄位數目。 超出此計數之欄位的讀取存取會傳回 Null 值。 寫入存取失敗。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = Record.FieldCount
```
