# <a name="object-model-for-mergemoddll-version-20"></a>Mergemod.dll 版本2.0 的物件模型

Mergemod.dll 版本2.0 的物件模型的組織方式如下所示。

![mergemod.dll 版本2.0 的物件模型](images/mergobj2.png)

[Merge 物件](merge-object.md)是模型的主要物件。 [**GetFiles 物件**](getfiles-object.md)是次要物件。 相關性是相依性 [**物件**](dependency-object.md)的集合。 錯誤是 [**錯誤物件**](error-object.md)的集合。 ConfigurableItems 是 [**ConfigurableItem 物件**](configurableitem-object.md)的集合。
