# <a name="ice89"></a>ICE89

ICE89 會驗證 progid 資料表中 Progid 父資料行中的值， \_ 是否為 progid 資料表中 progid 資料行的有效外鍵。 [](progid-table.md) 每個 ProgId 父代都應該在 ProgId 資料表中有一筆記錄。

## <a name="result"></a>結果

ICE89 會張貼下列錯誤。



| ICE89 錯誤                                                           | Description                                                                |
|-----------------------------------------------------------------------|----------------------------------------------------------------------------|
| \_Progid 資料表中的 progid 父項 ' \[ 1 \] ' 不是有效的 progid。 | ProgId 資料表中未列出指定的 ProgId 父代。 |
