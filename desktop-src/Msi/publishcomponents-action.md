# <a name="publishcomponents-action"></a>PublishComponents 動作

PublishComponents 動作會從 [PublishComponent](publishcomponent-table.md) 資料表管理元件的公告。

## <a name="sequence-restrictions"></a>順序限制

沒有任何順序限制。

## <a name="actiondata-messages"></a>ActionData 訊息



| 欄位 | 動作資料的描述                                   |
|-------|--------------------------------------------------------------|
| \[1\] | GUID，代表已公告功能的元件識別碼。 |
| \[2\] | 元件識別碼的辨識符號。                                   |



 

## <a name="remarks"></a>備註

PublishComponents 動作會從 PublishComponents 資料表中，發佈屬於針對公告或安裝所選取之功能的所有元件。
