# <a name="msiinstanceguid-property"></a>MSIINSTANCEGUID 屬性

**MSIINSTANCEGUID** 屬性的存在指出產品程式碼-變更轉換已註冊至產品。 **MSIINSTANCEGUID** 屬性的值會指定要用來安裝產品的哪一個實例。 **MSIINSTANCEGUID** 屬性只能參考已經隨實例轉換安裝的產品。

實例轉換是在 Windows Server 2003 或 Windows XP 上執行的安裝程式所提供的產品代碼變更轉換。 如需詳細資訊，請參閱 [安裝多個產品和修補程式實例](installing-multiple-instances-of-products-and-patches.md)。
