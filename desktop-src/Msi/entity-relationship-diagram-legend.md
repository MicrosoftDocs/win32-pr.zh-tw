# <a name="entity-relationship-diagram-legend"></a>實體關聯性圖表圖例

指向左菱形的黑色圓圈表示第一個資料表的主鍵與第二個數據表的外鍵之間有一對多關聯性。 如需此圖的範例，請參閱登錄 [資料表群組](registry-tables-group.md)。 Extension 是延伸模組資料表的主鍵和動詞資料表的外鍵。 因此，您可以註冊一個擴充功能來擁有數個動詞，但特定的動詞只能與一個擴充功能相關聯。
