# <a name="clientusersid-property"></a>UserSID 屬性

元件的使用者 SID。 這個屬性會呼叫 [**MsiEnumClientsEx**](/windows/desktop/api/Msi/nf-msi-msienumclientsexa) 函數。

**[Windows Installer 4.5 或更早版本](not-supported-in-windows-installer-4-5.md)：** 不支援。 從 Windows Installer 5.0 開始，可以使用這個屬性。
