# <a name="position-control-attribute"></a>Position 控制項屬性

這個屬性會決定控制項在對話方塊上的位置。 屬性的四個欄位包括：左方、頂端、寬度和高度。 將控制項左上角的控制項寬度、高度和座標，輸入控制項 [資料表](control-table.md)的 Width、Height、X 和 Y 資料行中。 將 [安裝程式單位](installer-units.md) 用於長度和距離的單位。

## <a name="valid-controls"></a>有效的控制項

所有現用控制項。

## <a name="associated-bit-flags"></a>相關聯的位旗標

這個屬性不會使用位旗標。

## <a name="remarks"></a>備註

請參閱 [控制項屬性](control-attributes.md) 和您需要在 [控制項](controls.md)下建立的控制項。

安裝程式單位等於10點 MS sans-serif 字型大小的一到第十二個高度。
