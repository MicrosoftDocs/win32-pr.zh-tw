# <a name="ice87"></a>ICE87

ICE87 會驗證 [屬性資料表](property-table.md)中未撰寫下列屬性。 您應改為在命令列上設定這些屬性。

-   [**ADDLOCAL 屬性**](addlocal.md)
-   [**REMOVE 屬性**](remove.md)
-   [**ADDSOURCE 屬性**](addsource.md)
-   [**ADDDEFAULT 屬性**](adddefault.md)
-   [**重新安裝屬性**](reinstall.md)
-   [**公告屬性**](advertise.md)
-   [**COMPADDLOCAL 屬性**](compaddlocal.md)
-   [**COMPADDSOURCE 屬性**](compaddsource.md)
-   [**FILEADDLOCAL 屬性**](fileaddlocal.md)
-   [**FILEADDSOURCE 屬性**](fileaddsource.md)
-   [**FILEADDDEFAULT 屬性**](fileadddefault.md)

## <a name="result"></a>結果

ICE87 會張貼下列警告。



| ICE87 警告                                                                                                                        | 描述                                                                                                                       |
|--------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| 屬性 ' \[ 1 \] ' 不應撰寫至屬性資料表。 這樣做可能會導致無法正確地卸載產品 | 無法在 [屬性工作表](property-table.md)中設定指定的屬性。 請改為在命令列上設定屬性。 |
