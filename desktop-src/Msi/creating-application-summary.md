# <a name="creating-application-summary-property"></a>建立應用程式摘要屬性

[ **建立應用程式摘要** ] 屬性會傳達建立安裝程式資料庫的應用程式。 一般而言，此摘要屬性的值是用來撰寫此資料庫的軟體名稱。
