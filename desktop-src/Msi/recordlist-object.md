# <a name="recordlist-object"></a>RecordList 物件

**RecordList** 物件是 [**記錄**](record-object.md)物件的集合。 您必須確認 **RecordList** 物件存在，而且在參考其屬性之前不是空的。

## <a name="members"></a>成員

**RecordList** 物件具有下列類型的成員：

-   [屬性](#properties)

### <a name="properties"></a>屬性

**RecordList** 物件具有這些屬性。



| 屬性                                     | 描述                                                          |
|:---------------------------------------------|:---------------------------------------------------------------------|
| [**計數**](recordlist-count.md)<br/> | 傳回 **RecordList** 物件中的專案數。<br/> |
| [**項目**](recordlist-item.md)<br/>   | 傳回 **RecordList** 物件集合中的記錄。<br/>   |
