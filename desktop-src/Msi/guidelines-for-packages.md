# <a name="guidelines-for-packages"></a><span data-ttu-id="91665-103">封裝的指導方針</span><span class="sxs-lookup"><span data-stu-id="91665-103">Guidelines for Packages</span></span>

<span data-ttu-id="91665-104">由於 [*使用者帳戶控制*](u-gly.md) (Windows Vista 中的 UAC) 在安裝期間會限制許可權，因此 Windows Installer 套件的開發人員不應假設其安裝一律可存取系統的所有部分。</span><span class="sxs-lookup"><span data-stu-id="91665-104">Because [*User Account Control*](u-gly.md) (UAC) in Windows Vista restricts privileges during an installation, developers of Windows Installer packages should not assume that their installation always has access to all parts of the system.</span></span>

-   <span data-ttu-id="91665-108">當您在 [InstallUISequence 資料表](installuisequence-table.md)中包含具有動作的 [*安裝內容*](i-gly.md)條件時，請根據具有特殊許可權的 [屬性](privileged.md)來使用條件陳述式。</span><span class="sxs-lookup"><span data-stu-id="91665-108">When including an [*installation context*](i-gly.md) condition with an action in the [InstallUISequence table](installuisequence-table.md), use a conditional statement based on the [Privileged property](privileged.md).</span></span> <span data-ttu-id="91665-109">請勿使用以 [**AdminUser**](adminuser.md) 屬性為基礎的條件。</span><span class="sxs-lookup"><span data-stu-id="91665-109">Do not use a condition based on the [**AdminUser**](adminuser.md) property.</span></span>
-   <span data-ttu-id="91665-110">包含安裝啟動條件的安裝內容時，請在[InstallExecuteSequence 資料表](installexecutesequence-table.md)中使用[自訂動作類型 19](custom-action-type-19.md) ，並在具有特殊[許可權的屬性](privileged.md)上進行自訂動作的條件。</span><span class="sxs-lookup"><span data-stu-id="91665-110">When including the installation context with the installation launch conditions, use a [Custom Action Type 19](custom-action-type-19.md) in the [InstallExecuteSequence table](installexecutesequence-table.md) and make the custom action conditional upon the [Privileged property](privileged.md).</span></span> <span data-ttu-id="91665-111">請勿將 [LaunchCondition 資料表](launchcondition-table.md) 中的動作與以 [AdminUser 屬性](adminuser.md) 或許可權屬性為基礎的條件一起使用。</span><span class="sxs-lookup"><span data-stu-id="91665-111">Do not use an action in the [LaunchCondition table](launchcondition-table.md) with a condition based on the [AdminUser property](adminuser.md) or Privileged property.</span></span>
-   <span data-ttu-id="91665-112">若要讀取或修改系統組態，請在[InstallExecuteSequence 資料表](installexecutesequence-table.md)中使用[延後執行自訂動作](deferred-execution-custom-actions.md)。</span><span class="sxs-lookup"><span data-stu-id="91665-112">To read or modify the system configuration, use a [deferred execution custom action](deferred-execution-custom-actions.md) in the [InstallExecuteSequence table](installexecutesequence-table.md).</span></span> <span data-ttu-id="91665-113">請勿使用 [InstallUISequence 資料表](installuisequence-table.md) 中的立即執行自訂動作來修改系統組態。</span><span class="sxs-lookup"><span data-stu-id="91665-113">Do not use immediate execution custom actions in the [InstallUISequence table](installuisequence-table.md) to modify the system configuration.</span></span>
-   <span data-ttu-id="91665-114">若要修改非使用者特定系統的元件，請在 [InstallExecuteSequence 資料表](installexecutesequence-table.md)中使用延遲的自訂動作。</span><span class="sxs-lookup"><span data-stu-id="91665-114">To modify parts of the system that are not user specific, use a deferred custom action in the [InstallExecuteSequence table](installexecutesequence-table.md).</span></span> <span data-ttu-id="91665-115">您應該在自訂動作類型中包含 msidbCustomActionTypeNoImpersonate 位。</span><span class="sxs-lookup"><span data-stu-id="91665-115">You should include the msidbCustomActionTypeNoImpersonate bit in the custom action type.</span></span>

-   <span data-ttu-id="91665-119">在原始封裝的 [MsiPatchCertificate](msipatchcertificate-table.md) 資料表中包含憑證，並使用相同的憑證簽署所有修補程式。</span><span class="sxs-lookup"><span data-stu-id="91665-119">Include a certificate in the [MsiPatchCertificate](msipatchcertificate-table.md) table of original package and sign all patches with the same certificate.</span></span>
