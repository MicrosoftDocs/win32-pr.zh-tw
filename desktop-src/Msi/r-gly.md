# <a name="r-windows-installer"></a>R (Windows Installer)

<dl> <dt>

<span id="_msi_reduced_ui_gly"></span><span id="_MSI_REDUCED_UI_GLY"></span>**縮小 UI**
</dt> <dd>

安裝程式的 [*內部使用者介面*](i-gly.md) 功能層級。 只顯示撰寫的非強制回應對話方塊。 如需詳細資訊，請參閱 [消費者介面層級](user-interface-levels.md)。

</dd> <dt>

<span id="_msi_reinstall_gly"></span><span id="_MSI_REINSTALL_GLY"></span>**安裝**
</dt> <dd>

已安裝應用程式的部分或完全重新安裝。 如果有任何檔案或登錄專案已損毀或遺失，通常會完成修復應用程式。 如需詳細資訊，請參閱 [重新安裝功能或應用程式](reinstalling-a-feature-or-application.md)。

</dd> <dt>

<span id="_msi_resiliency_gly"></span><span id="_MSI_RESILIENCY_GLY"></span>**彈性**
</dt> <dd>

從錯誤中順利復原的應用程式功能，而不會產生錯誤訊息。 如需詳細資訊，請參閱 [復原](resiliency.md)。

</dd> <dt>

<span id="_msi_rollback_gly"></span><span id="_MSI_ROLLBACK_GLY"></span>**回 滾**
</dt> <dd>

安裝失敗時，自動還原電腦的原始狀態。 如需詳細資訊，請參閱[復原安裝 (Windows Installer) ](rollback-installation.md)。

</dd> </dl>
