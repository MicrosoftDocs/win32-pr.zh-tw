# <a name="fatalerror-dialog"></a>FatalError 對話方塊

如果安裝因嚴重錯誤而終止，則會在安裝結束時顯示此強制回應對話方塊。 此對話方塊通常只包含一些文字和 **[確定]** 按鈕。 這個對話方塊的名稱必須出現在 [AdminUISequence 資料表](adminuisequence-table.md) 和 [InstallUISequence 資料表](installuisequence-table.md) 中，並以-3 作為序號。
