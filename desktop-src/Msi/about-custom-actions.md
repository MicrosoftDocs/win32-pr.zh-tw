# <a name="about-custom-actions"></a>關於自訂動作

雖然在大部分情況下，標準動作都足以執行安裝，但是自訂動作可讓安裝套件的作者藉由包含可執行檔、動態連結程式庫和腳本，來擴充標準動作的功能。 自訂動作不能用在用於公告、 [AdvtUISequence 資料表](advtuisequence-table.md) 和 [AdvtExecuteSequence 資料表](advtexecutesequence-table.md)的順序資料表中。
