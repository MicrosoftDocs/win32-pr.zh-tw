# <a name="displayed-error-messages"></a>顯示的錯誤訊息

[安裝程式](installer-function-reference.md)函式可能會顯示錯誤訊息對話方塊，並傳回下列任何錯誤。 只有當使用者介面層級為 [完整]、[已縮減] 或 [基本] 層級時，才會顯示錯誤方塊。 如需詳細資訊，請參閱 [消費者介面層級](user-interface-levels.md)。

安裝程式函數只會顯示下表中的錯誤訊息。 針對不在此清單中的錯誤，函式的呼叫端會負責處理錯誤訊息的顯示。



| 錯誤碼               | 錯誤                        |
|--------------------------|------------------------------|
| \_安裝 \_ 暫止時發生錯誤  | 安裝已暫停。   |
| \_安裝 \_ USEREXIT 時發生錯誤 | 使用者已結束安裝。 |
| \_安裝 \_ 失敗時發生錯誤  | 安裝失敗。              |
