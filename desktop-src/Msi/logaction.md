# <a name="logaction-property"></a>LOGACTION 屬性

**LOGACTION** 屬性是動作名稱的清單，只以分號分隔，而且沒有空格。 這個屬性中的動作名稱不區分大小寫，而且清單可以用分號結尾。

## <a name="remarks"></a>備註

安裝程式會記錄清單中動作的動作資料訊息。 屬性不會影響其他類型的記錄。
