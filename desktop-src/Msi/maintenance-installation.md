# <a name="maintenance-installation"></a>維護安裝

成功安裝應用程式之後，使用者可能會想要返回安裝以新增或移除功能。 這就是所謂的維護安裝。

套用至安裝的[轉換](transforms.md)必須在後續的維護安裝期間使用。
