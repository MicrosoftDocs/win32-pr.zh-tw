# <a name="indirectpropertyname-control-attribute"></a>IndirectPropertyName 控制項屬性

這個屬性是與控制項相關聯之間接屬性的名稱。 如果設定了 [間接](indirect-control-attribute.md) 位，控制項會顯示或變更具有這個名稱的屬性值，而這個名稱是 [控制資料表](control-table.md)的屬性資料行中所列屬性的值。

若為惰性控制項，這個屬性會傳回 Null。

## <a name="valid-controls"></a>有效的控制項

所有現用控制項。

## <a name="associated-bit-flags"></a>相關聯的位旗標

這個屬性不會使用位旗標。

## <a name="remarks"></a>備註

請參閱 [控制項屬性](control-attributes.md) 和您需要在 [控制項](controls.md)下建立的控制項。
