# <a name="updating-directory-structure-for-an-upgrade"></a>更新升級的目錄結構

使用 SDK 或其他編輯器提供的資料庫編輯器 Orca，在 MNP2001.msi 中開啟 [目錄資料表](directory-table.md) 。 範例升級的目錄結構與原始產品的目錄結構相同，而且在 [指定目錄結構](specifying-directory-structure.md)中所描述的原始目錄資料表中不需要進行任何變更。
