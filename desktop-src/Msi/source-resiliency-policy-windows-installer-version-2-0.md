# <a name="source-resiliency-policy"></a>來源復原原則

安裝程式會開始搜尋來源，方法是檢查來源清單中最近使用的來源位置、網路來源清單、媒體來源，以及最後的 URL 來源。 如需詳細資訊，請參閱 [來源復原](source-resiliency.md)。 如果搜尋失敗，安裝程式可能會顯示 [流覽對話方塊](browse-dialog.md) ，讓使用者可以手動搜尋來源。 如果 [使用者介面層級](user-interface-levels.md) 設定為 [無]，則無法顯示 [流覽] 對話方塊。 系統管理員藉由設定系統原則，來控制使用者的流覽對話方塊顯示。

使用 Windows Installer 時，根據預設，預設無法使用 [流覽] 對話方塊找出媒體上的受管理應用程式來源，而且無法安裝受控應用程式。 系統管理員可使用 [AllowLockdownBrowse](allowlockdownbrowse.md) 和 [AllowLockdownMedia](allowlockdownmedia.md) 原則，讓非管理員從媒體流覽或安裝受控應用程式。 系統管理員會使用 [DisAbleBrowse](disablebrowse.md) 和 [DisAbleMedia](disablemedia.md) 原則，停用從媒體流覽或安裝應用程式的功能。

下表摘要說明在 Windows Installer 中使用這些系統原則的方式。



| 系統原則                                  | 非管理員使用者非受管理應用程式                                                                                                             | 非管理員使用者受控應用程式                                                                                                                 | 系統管理員管理的應用程式非受管理應用程式                                                                                               |
|------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [AllowLockdownBrowse](allowlockdownbrowse.md) | 使用者可以使用位於媒體上的來源安裝未受管理的應用程式。 使用者可以流覽非受控應用程式來源的媒體。<br/>    | 使用者無法使用位於媒體上的來源來安裝受控應用程式。 使用者可以流覽媒體以取得受控應用程式的來源。<br/>      | 系統管理員可以使用位於媒體上的來源安裝應用程式。 系統管理員可以流覽媒體以取得應用程式的來源。<br/>    |
| [AllowLockdownMedia](allowlockdownmedia.md)   | 使用者可以使用位於媒體上的來源安裝未受管理的應用程式。 使用者可以流覽非受控應用程式來源的媒體。<br/>    | 使用者可以使用位於媒體上的來源來安裝受控應用程式。 使用者無法流覽受控應用程式來源的媒體。<br/>      | 系統管理員可以使用位於媒體上的來源安裝應用程式。 系統管理員可以流覽媒體以取得應用程式的來源。<br/>    |
| *預設值*                                      | 使用者可以使用位於媒體上的來源安裝未受管理的應用程式。 使用者可以流覽非受控應用程式來源的媒體。<br/>    | 使用者無法使用位於媒體上的來源來安裝受控應用程式。 使用者無法流覽受控應用程式來源的媒體。<br/>   | 系統管理員可以使用位於媒體上的來源安裝應用程式。 系統管理員可以流覽媒體以取得應用程式的來源。<br/>    |
| [DisAbleBrowse](disablebrowse.md)             | 使用者可以使用位於媒體上的來源安裝未受管理的應用程式。 使用者無法流覽未受管理應用程式來源的媒體。<br/> | 使用者無法使用位於媒體上的來源來安裝受控應用程式。 使用者無法流覽受控應用程式來源的媒體。 .<br/> | 系統管理員可以使用位於媒體上的來源安裝應用程式。 系統管理員無法流覽應用程式來源的媒體。<br/> |
| [DisAbleMedia](disablemedia.md)               | 使用者無法使用位於媒體上的來源安裝未受管理的應用程式。 使用者可以流覽非受控應用程式來源的媒體。<br/> | 使用者無法使用位於媒體上的來源來安裝受控應用程式。 使用者無法流覽受控應用程式來源的媒體。<br/>   | 系統管理員無法使用位於媒體上的來源安裝應用程式。 系統管理員可以流覽媒體以取得應用程式的來源。<br/> |
