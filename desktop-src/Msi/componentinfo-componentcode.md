# <a name="componentinfocomponentcode-property"></a>ComponentInfo. ComponentCode 屬性

有問題之元件的元件代碼。 這個屬性會呼叫 [**MsiGetComponentPathEx**](/windows/desktop/api/Msi/nf-msi-msigetcomponentpathexa) 函數。

**[Windows Installer 4.5 或更早版本](not-supported-in-windows-installer-4-5.md)：** 不支援。 從 Windows Installer 5.0 開始，可以使用這個屬性。
