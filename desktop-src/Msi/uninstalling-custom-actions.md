# <a name="uninstalling-custom-actions"></a>卸載自訂動作

使用自訂動作的安裝套件開發人員必須在每一種情況下決定如何條件讓自訂動作在套件卸載時正確運作。 封裝作者可決定是否要在卸載時執行自訂動作，而不是在卸載時執行，或在卸載時執行替代自訂動作。
