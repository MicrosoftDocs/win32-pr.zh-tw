# <a name="character-count-summary-property"></a>字元計數摘要屬性

[ **字元計數摘要** ] 屬性僅用於轉換。 摘要資訊資料流程的這個部分分成 2 16 位單字。 上方文字包含 [*轉換驗證旗標*](t-gly.md)。 下方的單字包含 [*轉換錯誤條件旗標*](t-gly.md)。

此屬性在安裝套件或修補程式套件中應為 Null。
