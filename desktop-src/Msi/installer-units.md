# <a name="installer-units"></a>安裝程式單位

Windows Installer 的使用者介面單位大約等於一到第十二個 (1/12) 10 點 MS sans-serif 字型大小的高度。 對話方塊、控制項和字型的相對大小可根據使用者介面中使用的字型大小而變更。 為了確保正確顯示使用者介面，安裝程式開發人員應該一律使用實際的字型大小來測試其應用程式。
