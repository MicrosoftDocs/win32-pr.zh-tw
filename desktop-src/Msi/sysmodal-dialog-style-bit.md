# <a name="sysmodal-dialog-style-bit"></a>SysModal 對話方塊樣式位

如果設定此樣式位，對話方塊將會停止所有其他應用程式，而其他應用程式則無法取得焦點。 這個狀態會一直保留到 SysModal 對話方塊關閉為止。

## <a name="value"></a>值



| Decimal | 十六進位 | 常數                          |
|---------|-------------|-----------------------------------|
| 8       | 0x00000008  | **msidbDialogAttributesSysModal** |
