# <a name="user-interface-wizard-behavior"></a>消費者介面 Wizard 行為

若要撰寫簡單的使用者介面，安裝套件的開發人員可以利用符合 interface wizard 行為的使用者介面設計。

「使用者介面」（user interface wizard）行為一詞表示序列中的每個對話方塊都包含 **下一個>>** 按鈕，使用者在輸入資料或在目前的對話方塊中設定資訊之後，使用者按一下此按鈕即可移至下一個對話方塊。 如果使用者決定返回並變更先前對話方塊中輸入的任何資訊，則每個對話方塊都包含使用者按一下以返回的 **<<上** 一個按鈕。 在嚮導順序的結尾，使用者按一下 **[完成]** 按鈕開始安裝程式。

UI 範例中所述的 UI 會遵循使用者介面 wizard 的行為。 請參閱 [安裝範例](an-installation-example.md)。
