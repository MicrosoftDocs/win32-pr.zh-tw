# <a name="windows-installer-file-name-extensions"></a>Windows Installer 副檔名

下列副檔名適用于 Windows Installer。



| 分機 | Description                                |
|-----------|--------------------------------------------|
| .msi      | Windows Installer 資料庫。                |
| .msm      | Windows Installer 合併模組。            |
| .msp      | Windows Installer 修補程式。                   |
| .mst      | Windows Installer 轉換。               |
| .idt      | 已匯出 Windows Installer 資料庫資料表。 |
| .cub      | 驗證模組。                         |
| . pcp      | Windows Installer 修補程式建立檔案      |
