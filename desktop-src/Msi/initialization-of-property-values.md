# <a name="initialization-of-property-values"></a>屬性值的初始化

建議您將安裝所使用的所有屬性都輸入至具有初始值的 [屬性資料表](property-table.md) 中。 安裝程式啟動時，安裝程式會將屬性設定為這些值。 空白是可接受的值的屬性，而內建于安裝程式中的屬性則不需要初始化。
