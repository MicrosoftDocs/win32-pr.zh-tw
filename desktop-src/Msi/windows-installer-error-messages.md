# <a name="windows-installer-error-messages-for-developers"></a>針對開發人員 Windows Installer 的錯誤訊息 () 

> [!NOTE]  
> 本文適用于使用 Windows Installer 為其應用程式建立安裝程式套件的軟體發展人員。 如果您是在安裝或卸載應用程式期間或之後遇到電腦問題的使用者，請洽詢客戶對您嘗試安裝或移除之軟體的支援。 如果您覺得您需要支援 Microsoft 產品，請前往我們的技術支援網站，網址為 [support.microsoft.com](https://support.microsoft.com)。

本主題詳述的錯誤碼是由 Windows Installer 傳回，錯誤碼為1000或更新版本。 編號1000到1999的錯誤碼會送出錯誤，而且必須撰寫到 [錯誤資料表](error-table.md)中。 編號大於2000的錯誤碼是內部錯誤，不會有撰寫的字串，但如果安裝套件未正確撰寫，則可能會發生這些錯誤。 如需 Windows Installer 函式特定的錯誤碼 **MsiExec.exe** 和 **InstMsi.exe**，請參閱 [MsiExec.exe 和 InstMsi.exe 錯誤訊息](error-codes.md)。 如需保留的錯誤碼清單，請參閱 [錯誤資料表](error-table.md)。 您可以搜尋網際網路或 [Microsoft 支援服務網站](https://support.microsoft.com/) ，以取得下表中許多訊息的解決方案。

安裝套件的開發人員也可以使用內部一致性評估工具來測試其套件的內部一致性。 如需詳細資訊，請參閱 [內部一致性評估工具-ices-003](internal-consistency-evaluators-ices.md)。
