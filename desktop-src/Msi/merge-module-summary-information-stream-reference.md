# <a name="merge-module-summary-information-stream-reference"></a>合併模組摘要資訊資料流程參考

下表識別合併模組之摘要資訊資料流程的屬性。 如需詳細資訊，請參閱 [摘要資訊資料流程](summary-information-stream.md)。



| 屬性                                                     | 識別碼                 | PID | 類型         | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|--------------------------------------------------------------|--------------------|-----|--------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**字碼頁**](codepage-summary.md)                         | PID \_ 字碼頁      | 1   | VT \_ I2       | 識別用來顯示摘要資訊的字碼頁。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| [**標題**](title-summary.md)                               | PID \_ 標題         | 2   | VT \_ LPSTR    | 「合併模組」。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| [**主體**](subject-summary.md)                           | PID \_ 主體       | 3   | VT \_ LPSTR    | [**ProductName**](productname.md) 屬性。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| [**作者**](author-summary.md)                             | PID \_ 作者        | 4   | VT \_ LPSTR    | [**製造商**](manufacturer.md) 屬性。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [**關鍵字**](keywords-summary.md)                         | PID \_ 關鍵字      | 5   | VT \_ LPSTR    | MergeModule、MSI、資料庫。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**註解**](comments-summary.md)                         | PID \_ 批註      | 6   | VT \_ LPSTR    | 描述合併模組及其元件。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [**範本**](template-summary.md)                         | PID \_ 範本      | 7   | VT \_ LPSTR    | 資料庫所支援的平臺和語言版本。 在每個合併模組中都是必要的。 如需詳細資訊，請參閱語法的 [**範本**](template-summary.md) 。 包含64位元件的模組必須設定 Intel64 或 x64。 如需詳細資訊，請參閱 [使用64位合併模組](using-64-bit-merge-modules.md)。<br/> 列出模組支援的所有語言的數位語言識別項。 清單中的第一種語言是模組的預設語言。 指定一個以上的語言會導致多語系合併。<br/> |
| [**上次儲存者**](last-saved-by-summary.md)               | PID \_ LASTAUTHOR    | 8   | VT \_ LPSTR    | 使用與 [**範本摘要**](template-summary.md) 屬性相同的語法，指定已修補資料庫的平臺和語言。                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| [**修訂編號**](revision-number-summary.md)           | PID \_ REVNUMBER     | 9   | VT \_ LPSTR    | 此合併模組的唯一 GUID。 在每個合併模組中都是必要的。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| [**前次列印時間**](last-printed-summary.md)                 | PID \_ LASTPRINTED   | 11  | VT \_ FILETIME | Null。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [**建立時間/日期**](create-time-date-summary.md)         | PID \_ 建立 \_ DTM   | 12  | VT \_ FILETIME | 建立安裝程式資料庫的時間和日期。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| [**上次儲存的時間/日期**](last-saved-time-date-summary.md) | PID \_ LASTSAVE \_ DTM | 13  | VT \_ FILETIME | 一開始是 null。 每次使用者變更安裝資料庫時，都會在儲存合併資料庫時，將值更新為目前的系統時間/日期。                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**頁面計數**](page-count-summary.md)                     | PID \_ PAGECOUNT     | 14  | VT \_ I4       | 必要安裝程式的最低版本。 以下列格式儲存為整數：主要 \* 100 + 次要。 在每個合併模組中都是必要的。                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**字數統計**](word-count-summary.md)                     | PID \_ WORDCOUNT     | 15  | VT \_ I4       | 為此屬性輸入 0 (零) 。 請注意，在合併模組中，無論這個屬性的值為何，檔案一律會在內嵌封包檔中。 在每個合併模組中都是必要的。                                                                                                                                                                                                                                                                                                                                                                                        |
| [**字元計數**](character-count-summary.md)           | PID \_ >CHARCOUNT     | 16  | VT \_ I4       | Null。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [**正在建立應用程式**](creating-application-summary.md) | PID \_ APPNAME       | 18  | VT \_ LPSTR    | 用來建立安裝程式資料庫的應用程式。 此值通常是用來撰寫此合併模組的軟體名稱。                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [**安全性**](security-summary.md)                         | PID \_ 安全性      | 19  | VT \_ I4       | "2"。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
