# <a name="icem01"></a>ICEM01

ICEM01 會驗證 ICE 機制是否正常運作。 這種 ICE 使用 [**time**](time.md) 屬性來取得時間，並傳回系統時間或無。

Merge module 會儲存在合併模組中，名為 Mergemod 的 .cub 檔中，而不是在包含用於封裝驗證的 Ices-003 的 .cub 檔案中。

## <a name="result"></a>結果

ICEM01 張貼訊息，提供安裝程式呼叫 ICEM01 的時間。 這種 ICE 絕對不會傳回錯誤。
