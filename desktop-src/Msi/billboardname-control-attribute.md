# <a name="billboardname-control-attribute"></a>BillboardName 控制項屬性

這個屬性會傳回目前執行之佈告欄的名稱，或設定並依名稱顯示佈告欄。 這個屬性會在預覽模式中使用。

## <a name="valid-controls"></a>有效的控制項

[廣告 牌](billboard-control.md)

## <a name="associated-bit-flags"></a>相關聯的位旗標

這個屬性不會使用位旗標。

## <a name="remarks"></a>備註

請參閱 [控制項屬性](control-attributes.md) 和您需要在 [控制項](controls.md)下建立的控制項。
