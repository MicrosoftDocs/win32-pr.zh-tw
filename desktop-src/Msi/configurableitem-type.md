# <a name="configurableitemtype-property"></a>ConfigurableItem。 Type 屬性

**Type** 屬性會傳回 ModuleConfiguration 資料表之 Type 資料行中的值。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = ConfigurableItem.Type
```
