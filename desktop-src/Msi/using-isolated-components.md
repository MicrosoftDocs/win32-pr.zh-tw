# <a name="using-isolated-components"></a>使用隔離的元件

藉由使用 [隔離的元件](isolated-components.md)，安裝套件的作者可以指定安裝程式將應用程式的共用檔案直接複製到應用程式的資料夾，而不是共用位置。 然後，應用程式會以獨佔方式使用這組私用的檔案。

如需詳細資訊，請參閱將 [com 元件安裝至私用位置](installing-a-com-component-to-a-private-location.md)、將 [非 com 元件安裝至私用位置](installing-a-non-com-component-to-a-private-location.md)、 [讓現有封裝中的 COM 元件成為私](make-a-com-component-in-an-existing-package-private.md) 用，或 [在現有的封裝中將非 COM 元件設為私](make-a-non-com-component-in-an-existing-package-private.md)用。
