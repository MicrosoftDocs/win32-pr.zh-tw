# <a name="author-summary-property"></a>作者摘要屬性

[ **作者摘要** ] 屬性會傳達安裝套件、轉換或修補套件的製造商。

-   將安裝套件中的 [ **作者摘要** ] 屬性設定為與 [ [**製造商**](manufacturer.md) ] 屬性相同的值。
-   將轉換中的 [ **作者摘要** ] 屬性設定為轉換的製造商名稱。
-   將修補程式套件中的 [ **作者摘要** ] 屬性設定為修補程式套件製造商的名稱。
