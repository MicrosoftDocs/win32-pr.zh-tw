# <a name="defaultdir"></a>DefaultDir

DefaultDir 資料類型是一種文字字串，其中包含有效的 [檔案名](filename.md) 或有效的 [識別碼](identifier.md)。 這只會用於 [目錄資料表](directory-table.md)中。 如果目錄是根目錄，則它必須是識別碼。 如果目錄是非根目錄，此值必須是檔案名或檔案名：檔案名組。 請注意，"." 可作為檔案名，並在目錄資料表中具有特殊意義。
