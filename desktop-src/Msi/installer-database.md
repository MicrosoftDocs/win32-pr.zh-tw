# <a name="installer-database"></a>安裝程式資料庫

本文主要是針對想要撰寫工具來建立新安裝程式套件的開發人員，以及需要安裝程式關係資料庫詳細資訊的開發人員。 應用程式開發人員採用完整的安裝程式套件，是管理應用程式安裝的有效方式。 對於想要使用自訂動作修改現有安裝程式套件和安裝程式的應用程式開發人員和系統管理員而言，您對安裝資料庫的知識也很有用。 如需 Windows Installer 與元件管理技術的一般資訊，請參閱[關於 Windows Installer](about-windows-installer.md)。

安裝程式資料庫包含安裝應用程式所需的所有資訊。 因此，開發人員若要建立安裝套件和開發人員，而不使用這類工具來建立安裝套件，則需要瞭解安裝程式資料庫。 以下主題提供安裝程式資料庫的主要功能區域：

[關於安裝程式資料庫](about-the-installer-database.md)

[使用安裝程式資料庫](using-the-installer-database.md)

[安裝程式資料庫參考](installer-database-reference.md)
