# <a name="dependencylanguage-property"></a>相依性。 Language 屬性

唯讀 **語言** 屬性會傳回所需模組的 **LANGID** 。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = Dependency.Language
```
