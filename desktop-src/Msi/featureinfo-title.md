# <a name="featureinfotitle-property"></a>FeatureInfo Title 屬性

[ **標題** ] 屬性是唯讀屬性，會在 [功能資料表](feature-table.md)的 [標題] 資料行中，傳回功能的標題。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = FeatureInfo.Title
```
