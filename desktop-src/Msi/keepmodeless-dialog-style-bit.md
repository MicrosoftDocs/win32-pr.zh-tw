# <a name="keepmodeless-dialog-style-bit"></a>KeepModeless 對話方塊樣式位

一般來說，如果未設定此位，而且會透過 Dataadapter.doaction 建立對話方塊，則所有其他 (通常會終結非模式) 對話方塊。 如果設定此位，則在建立此對話方塊時，其他對話方塊會保持運作狀態。

## <a name="value"></a>值



| Decimal | 十六進位 | 常數                              |
|---------|-------------|---------------------------------------|
| 16      | 0x00000010  | **msidbDialogAttributesKeepModeless** |
