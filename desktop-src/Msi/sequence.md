# <a name="sequence-property"></a>SEQUENCE 屬性

**SEQUENCE** 屬性指定的資料表具有與 InstallExecuteSequence 資料表相同的架構，也就是 Action、Condition 和 SEQUENCE 資料行。 [順序動作](sequence-action.md)會使用這個屬性。

## <a name="remarks"></a>備註

順序動作會依資料表的 [順序] 資料行所指定的順序，執行資料表中的動作。
