# <a name="windows-sdk-components-for-windows-installer-developers"></a>適用于 Windows Installer 開發人員的 Windows SDK 元件

Windows Installer 軟體發展工具組的元件包含在 Microsoft Windows 軟體開發套件 (SDK) 中。 Windows SDK 包括可轉散發元件、檔、安裝程式資料庫驗證工具、資料庫資料表編輯器、資料庫架構、開發工具、Visual Basic Scripting Edition (VBScript) 工具、範例產品和程式碼範例。

如需 Windows SDK 的下載頁面，請參閱 [Microsoft Windows 軟體開發套件 (SDK) ](https://developer.microsoft.com/windows/downloads/windows-10-sdk)。
