# <a name="ice09"></a>ICE09

ICE09 會驗證是否已針對標記為要安裝的每個元件將永久位設定為 SystemFolder。 ICE09 會張貼警告，因為您應避免將非永久的系統元件安裝至 SystemFolder。 若要避免這個警告，請在 [元件資料表](component-table.md) 的 [屬性] 資料行中，針對標記為要安裝的每個元件，將永久位設定為 [SystemFolder]。 如果資料庫沒有元件資料表，就不會進行任何驗證。
