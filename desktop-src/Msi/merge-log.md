# <a name="mergelog-method"></a>Merge. Log 方法

[**Merge**](merge-object.md)物件的 **Log** 方法會將文字字串寫入目前開啟的記錄檔。

## <a name="syntax"></a>語法


```JScript
Merge.Log(
  Message
)
```



## <a name="parameters"></a>參數

<dl> <dt>

*訊息* 
</dt> <dd>

要寫入記錄檔的文字字串。

</dd> </dl>

## <a name="return-value"></a>傳回值

這個方法不會傳回值。

## <a name="c"></a>C++

請參閱 [**Log**](/windows/win32/api/mergemod/nf-mergemod-imsmmerge-log) 函數。
