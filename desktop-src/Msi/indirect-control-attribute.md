# <a name="indirect-control-attribute"></a>間接控制屬性

間接控制屬性會指定是否要間接參考此控制項所顯示或變更的值。 如果設定此位，控制項會顯示或變更屬性的值，此屬性的識別碼列于 [控制資料表](control-table.md)的屬性資料行中。 如果未設定此位，控制項會在控制資料表的屬性資料行中顯示或變更屬性的值。

## <a name="valid-controls"></a>有效的控制項

所有現用控制項。

## <a name="value"></a>值



| Decimal | 十六進位 | 常數                           |
|---------|-------------|------------------------------------|
| 8       | 0x00000008  | **msidbControlAttributesIndirect** |



 

## <a name="remarks"></a>備註

請參閱 [控制項屬性](control-attributes.md) 和您需要在 [控制項](controls.md)下建立的控制項。
