## <a name="orca-documentation"></a>Orca 檔

[Orca](orca-exe.md) 是 Windows Installer SDK 提供的 GUI .msi 檔案編輯器。 它會提供 Windows Installer 資料庫資料表的完整存取權。

您可以透過 Orca 來提供一組說明檔。 若要取得 orca.exe 和 Orca 檔，您必須下載並安裝 Windows Installer SDK （以 Orca.msi 檔形式提供）。 當您安裝 [Microsoft Windows 軟體開發套件 (SDK) ](https://developer.microsoft.com/windows/downloads/windows-10-sdk/)之後，請按兩下 Orca.msi 檔案以安裝 Orca。

> [!Note]  
> 您無法另外下載 Orca。

 

雖然 Orca 可以存取 Windows Installer 的所有功能，但並不適合取代功能完整的封裝撰寫環境。 在許多情況下，您可以利用獨立軟體廠商提供的其中一種商用套件建立工具，更輕鬆地為應用程式建立 Windows Installer 安裝。

## <a name="windows-installer-software-vendors"></a>Windows Installer 軟體廠商
