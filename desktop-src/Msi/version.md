# <a name="version-windows-installer"></a>版本 (Windows Installer) 

版本資料類型是包含有效版本字串的文字字串。 版本字串的格式為

``` syntax
xxxxx.xxxxx.xxxxx.xxxxx
```

其中 *x* 是數位。

可接受的最大版本字串為65535.65535.65535.65535。

以下是有效版本字串的範例：

-   1
-   1.0
-   1.00
-   10.00
-   1.00.1
-   1.0.1
-   1.00.10
-   1.00.100
-   1.0.1000.0
