# <a name="identifier"></a>識別碼

識別碼資料類型是文字字串。 識別碼可以包含 ASCII 字元 a-z (a-z) 、數位、底線 (\_) 或句號 (. ) 。 不過，每個識別碼的開頭都必須是字母或底線。
