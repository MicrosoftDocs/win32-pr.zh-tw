<span data-ttu-id="767fb-104">RadioButtonGroup 控制項是一組選項按鈕。</span><span class="sxs-lookup"><span data-stu-id="767fb-104">The RadioButtonGroup control is a group of radio buttons.</span></span> <span data-ttu-id="767fb-105">它可讓使用者從預先決定的值清單中，選取屬性的字串或整數值。</span><span class="sxs-lookup"><span data-stu-id="767fb-105">It enables the user to select a string or integer value for a property from a predetermined list of values.</span></span> <span data-ttu-id="767fb-106">您可以藉由在 [控制項資料表](control-table.md)的屬性資料行中輸入屬性的名稱，將此控制項與屬性產生關聯。</span><span class="sxs-lookup"><span data-stu-id="767fb-106">You can associate this control with a property by entering the property's name into the Property column of the [Control table](control-table.md).</span></span> <span data-ttu-id="767fb-107">在 [選項按鈕資料表](radiobutton-table.md)的 [值] 資料行中，指定可能的選項值。</span><span class="sxs-lookup"><span data-stu-id="767fb-107">Specify the possible values for selection in the Value column of the [RadioButton table](radiobutton-table.md).</span></span> <span data-ttu-id="767fb-108">請注意，顯示的字串不一定與使用者選擇的值相同。</span><span class="sxs-lookup"><span data-stu-id="767fb-108">Note that the string displayed is not necessarily the same as the value that the user is choosing.</span></span>

<span data-ttu-id="767fb-109">每個 RadioButtonGroup 控制項都與屬性相關聯。</span><span class="sxs-lookup"><span data-stu-id="767fb-109">Every RadioButtonGroup control is associated with a property.</span></span> <span data-ttu-id="767fb-110">這個屬性的預設值必須在 [屬性工作表](property-table.md)中初始化。</span><span class="sxs-lookup"><span data-stu-id="767fb-110">The default value for this property must be initialized in the [Property table](property-table.md).</span></span> <span data-ttu-id="767fb-111">在 [選項按鈕資料表](radiobutton-table.md)中指定的每個 RadioButtonGroup 中，可能會有一個選項按鈕，其值欄位中的值符合這個屬性的預設值。</span><span class="sxs-lookup"><span data-stu-id="767fb-111">Within each RadioButtonGroup specified in the [RadioButton table](radiobutton-table.md), there may be one radio button that has a value in the Value field that matches the default value for this property.</span></span> <span data-ttu-id="767fb-112">這是 RadioButtonGroup 控制項的預設按鈕。</span><span class="sxs-lookup"><span data-stu-id="767fb-112">This is the default button for the RadioButtonGroup control.</span></span> <span data-ttu-id="767fb-113">**預設** 按鈕最初會在控制項中顯示為已選取。</span><span class="sxs-lookup"><span data-stu-id="767fb-113">The **Default** button is initially shown as selected in the control.</span></span>

<span data-ttu-id="767fb-116">RadioButtonGroup 控制項只會設定屬性值，而且不能用來傳送 [ControlEvent](control-events.md)。</span><span class="sxs-lookup"><span data-stu-id="767fb-116">RadioButtonGroup controls only set property values and cannot be used to send a [ControlEvent](control-events.md).</span></span>

<span data-ttu-id="767fb-117">執行會將整個群組視為一個控制項，因此無法隱藏或停用群組內的個別按鈕。</span><span class="sxs-lookup"><span data-stu-id="767fb-117">The implementation treats the whole group as one control, therefore it is not possible to hide or disable individual buttons within the group.</span></span> <span data-ttu-id="767fb-118">同樣地，所有按鈕都必須是相同的樣式，也就是所有的按鈕都有文字，或它們都有點陣圖 (或其他類似的功能) 。</span><span class="sxs-lookup"><span data-stu-id="767fb-118">Similarly all the buttons have to be of the same style, that is, either all of them have text or all of them have bitmaps (or other similar features).</span></span> <span data-ttu-id="767fb-119">按鈕的位置是相對於群組而撰寫的。</span><span class="sxs-lookup"><span data-stu-id="767fb-119">The position of the buttons is authored relative to the group.</span></span> <span data-ttu-id="767fb-120">如此一來，只要變更群組的座標，就可以移動整個群組，而不需要變更個別按鈕。</span><span class="sxs-lookup"><span data-stu-id="767fb-120">This way the entire group can be moved by changing only the coordinates of the group without changing the individual buttons.</span></span> <span data-ttu-id="767fb-121">在建立時，控制項會驗證個別按鈕不會超出群組的界限。</span><span class="sxs-lookup"><span data-stu-id="767fb-121">At creation the control verifies that the individual buttons do not extend beyond the boundaries of the group.</span></span>
