# <a name="windows-installer-features"></a>Windows安裝程式功能

功能是應用程式功能的一部分，可供使用者辨識，而且可能會決定個別安裝。 例如，某項功能可能是拼寫檢查、同義字或一組美工圖案。 父項和子功能的階層式關聯性通常存在，因此，如果已安裝子功能，也會自動安裝父項功能。

如需使用功能的相關資訊，請參閱下列內容：

-   [重新安裝功能或應用程式](reinstalling-a-feature-or-application.md)
-   [參考合併模組中的功能](referencing-features-in-merge-modules.md)
-   [使用功能和元件](working-with-features-and-components.md)
-   [檢查功能、元件、檔案的安裝](checking-the-installation-of-features-components-files.md)
-   [搜尋中斷的功能或元件](searching-for-a-broken-feature-or-component.md)
-   [發佈產品、功能和元件](publishing-products-features-and-components.md)
