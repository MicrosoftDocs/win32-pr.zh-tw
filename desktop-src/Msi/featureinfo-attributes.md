# <a name="featureinfoattributes-property"></a>FeatureInfo 屬性

**屬性**（attribute）屬性（attribute）是一種讀寫屬性，會在 [功能資料表](feature-table.md)的 [屬性] 資料行中，傳回此功能的值。

這是可讀寫的屬性。

## <a name="syntax"></a>Syntax


```JScript
propVal = FeatureInfo.Attributes
FeatureInfo.Attributes = propVal 
```
