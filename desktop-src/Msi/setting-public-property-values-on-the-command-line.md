# <a name="setting-public-property-values-on-the-command-line"></a>在命令列上設定公用屬性值

若要將公用屬性設定為常值字串值，請在引號之間加上字串。

PROPERTY = "string"

只有當字串包含空格時，才需要引號。 若要在命令列清除公用屬性 (將它設定為 Null) ，請將屬性的值設為空字串。 在此情況下，必須加上引號。

PROPERTY = ""。
