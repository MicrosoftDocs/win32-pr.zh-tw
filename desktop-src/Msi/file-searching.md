# <a name="file-searching"></a>檔案搜尋

在安裝期間，通常需要搜尋特定的檔案。 安裝程式有四個標準搜尋動作來提供此用途。 [AppSearch 動作](appsearch-action.md)會使用[AppSearch 資料表](appsearch-table.md)來判斷檔案是否存在於使用者的系統上。 在升級安裝期間， [CCPSearch 動作](ccpsearch-action.md) 和 [RMCCPSearch 動作](rmccpsearch-action.md) 都可以使用其對應的資料表來檢查使用者的系統是否有舊版產品。
