# <a name="databaseenableuipreview-method"></a>EnableUIPreview 方法

[**資料庫**](database-object.md)物件的 **EnableUIPreview** 方法可提供所需的支援來查看儲存在安裝程式資料庫中的使用者介面對話方塊，藉此協助撰寫對話方塊和公告欄。 方法會藉由傳回預覽 **資料庫** 物件來啟動預覽模式。 預覽模式會在預覽物件發行時結束。

## <a name="syntax"></a>語法


```JScript
Database.EnableUIPreview()
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
