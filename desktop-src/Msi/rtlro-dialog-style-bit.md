# <a name="rtlro-dialog-style-bit"></a>RTLRO 對話方塊樣式位

如果設定此樣式位，對話方塊中的文字會以由右至左閱讀的順序顯示。

## <a name="value"></a>值



| Decimal | 十六進位 | 常數                       |
|---------|-------------|--------------------------------|
| 128     | 0x00000080  | **msidbDialogAttributesRTLRO** |
