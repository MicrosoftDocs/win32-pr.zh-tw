# <a name="filesinuse-dialog"></a>FilesInUse 對話方塊

[FilesInUse] 對話方塊會警示使用者目前正在執行的檔案，這些檔案必須由安裝覆寫或刪除。 這讓使用者有機會關閉這些程式，並避免必須重新開機電腦，才能完成這些檔案的取代或刪除。 請參閱 [系統重新開機](system-reboots.md)。

此對話方塊將會依 [InstallValidate 動作](installvalidate-action.md)的要求建立。
