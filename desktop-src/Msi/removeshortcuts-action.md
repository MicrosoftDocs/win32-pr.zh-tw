# <a name="removeshortcuts-action"></a>RemoveShortcuts 動作

RemoveShortcuts 動作會管理移除已選取要卸載其功能的已公告快捷方式，或已選取要卸載其元件之 nonadvertised 快速鍵的移除。 如需詳細資訊，請參閱 [快速鍵對應表](shortcut-table.md)。

## <a name="sequence-restrictions"></a>順序限制

RemoveShortcuts 動作必須位於 [CreateShortcuts](createshortcuts-action.md) 動作之前。

## <a name="actiondata-messages"></a>ActionData 訊息



| 欄位 | 動作資料的描述      |
|-------|---------------------------------|
| \[1\] | 已移除快捷方式的識別碼。 |
