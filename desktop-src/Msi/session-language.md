# <a name="sessionlanguage-property"></a>Session。 Language 屬性

[**Session**](session-object.md)物件的 **Language** 屬性是唯讀屬性，代表目前安裝會話所使用的數位語言識別項。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = Session.Language
```
