# <a name="a-localization-example"></a>當地語系化範例

此範例說明如何將 [安裝範例](an-installation-example.md)中所述的 Windows Installer 封裝當地語系化。 原始安裝套件會從英文變更為法文語言版本。

當地語系化範例具有下列規格：

-   應用程式 MNP2000 的安裝 UI 應該從英文變更為法文。
-   法文版的 MNP2000 包含以法文語言撰寫 Readme.txt 和 Help.txt 檔案。
-   法文版本包含法國的旅遊專員清單，可提供給 Red 公園的機票。 這份清單 (以新檔案的形式提供，Fre.txt) 不是英文版的一部分。

當地語系化 [Windows Installer 套件](localizing-a-windows-installer-package.md)一節將說明當地語系化安裝的一般程式。

將英文版的 MNP2000.msi 以及 [規劃安裝](planning-the-installation.md) 中所述的所有來源檔案複製到新的資料夾中。 將資料夾中 MNP2000.msi 的名稱變更為 MNPFren.msi。 在下列各節中，您將修改此複本，以將應用程式當地語系化為法文。
