# <a name="merge-module-ice-reference"></a>Merge Module ICE 參考

下列清單提供每個 Merge Module ICE 的連結。



| ICEM                 | 合併模組 ICE 的簡短描述                                                                                                                                                                                                                    |
|----------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [ICEM01](icem01.md) | 驗證 ICE 機制是否正常運作。                                                                                                                                                                                                             |
| [ICEM02](icem02.md) | 驗證所有模組排除專案和相依性都與目前的模組相關。                                                                                                                                                                      |
| [ICEM03](icem03.md) | 驗證模組中的所有動作都是基本動作，或衍生自基底動作。                                                                                                                                                           |
| [ICEM04](icem04.md) | 確認合併模組的必要空白資料表確實為空白。                                                                                                                                                                                 |
| [ICEM05](icem05.md) | 檢查元件的關聯是否無效。                                                                                                                                                                                                         |
| [ICEM06](icem06.md) | 檢查模組是否有不正確功能參考。                                                                                                                                                                                                 |
| [ICEM07](icem07.md) | 驗證順序資料表和 [MergeModule.CABinet](mergemodule-cabinet.md) 檔中的檔案順序是否相符。                                                                                                                               |
| [ICEM08](icem08.md) | 確保模組不會排除其相依的內容。                                                                                                                                                                                          |
| [ICEM09](icem09.md) | 確認合併模組安全地處理預先定義的目錄。                                                                                                                                                                                    |
| [ICEM10](icem10.md) | 確認合併模組不包含 [屬性工作表](property-table.md)中不允許的屬性。                                                                                                                                         |
| [ICEM11](icem11.md) | 確認可設定的合併模組會列出模組的[ModuleIgnoreTable 資料表](moduleignoretable-table.md)中的[ModuleConfiguration 資料表](moduleconfiguration-table.md)和[ModuleSubstitution 資料表](modulesubstitution-table.md)。 |
| [ICEM12](icem12.md) | 確認在 ModuleSequence 資料表中，標準動作具有序號和自訂動作具有 BaseAction 和之後的值。                                                                                                                     |
| [ICEM13](icem13.md) | 確認發行者原則和設定元件不包含在合併模組中。                                                                                                                                                        |
| [ICEM14](icem14.md) | 驗證 [ModuleSubstitution 資料表](modulesubstitution-table.md)的值資料行。                                                                                                                                                              |
