# <a name="ice24"></a>ICE24

ICE24 會驗證 [屬性資料表](property-table.md)中的下列屬性：

-   [ [**ProductCode**](productcode.md) ] 屬性是有效的 [GUID](guid.md) 資料類型。
-   [**ProductVersion**](productversion.md)屬性是有效的產品版本。
-   [**ProductLanguage**](productlanguage.md)屬性是有效的 [語言](language.md)資料類型。

## <a name="result"></a>結果

如果其中有任何屬性的格式不是有效的資料類型，ICE24 會張貼錯誤訊息。
