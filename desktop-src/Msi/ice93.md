# <a name="ice93"></a>ICE93

如果自訂動作使用的名稱與標準動作相同，ICE93 會發出警告。 如需所有標準動作名稱的清單，請參閱 [標準動作參考](standard-actions-reference.md)。

## <a name="result"></a>結果

ICE93 會張貼下列警告。



| ICE93 警告                                                      | 描述                                                                                                                    |
|--------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| 自訂動作 ' \[ 1 \] ' 使用的名稱與標準動作相同。 | CustomAction 資料表的動作資料行中有一個識別碼，也就是 Windows Installer 標準動作的名稱。 |
