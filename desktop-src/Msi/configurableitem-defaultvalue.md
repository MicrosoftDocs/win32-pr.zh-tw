# <a name="configurableitemdefaultvalue-property"></a>ConfigurableItem DefaultValue 屬性

[**ConfigurableItem**](configurableitem-object.md)物件的 **defaultvalue** 屬性會從 [ModuleConfiguration 資料表](moduleconfiguration-table.md)的 defaultvalue 資料行傳回值。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = ConfigurableItem.DefaultValue
```



## <a name="property-value"></a>屬性值

## <a name="c"></a>C++

請參閱 [**取得 \_ DefaultValue**](/windows/desktop/api/Mergemod/nf-mergemod-imsmconfigurableitem-get_defaultvalue) 函數
