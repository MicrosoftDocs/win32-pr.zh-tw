# <a name="text-control"></a>Text Control

文字控制項會顯示靜態文字，而這可能會使用預先定義的樣式。

使用指定分行符號來顯示文字的建議方法是使用位於彼此下的多個單行文字控制項。 \\控制項文字欄位中的字元序列 n、 \\ r \\ n 或 \\ n \\ r 不會顯示為分行符號。 這些字元序列實際上是由控制項顯示。

## <a name="remarks"></a>備註

您可以使用 [**CreateWindowEx**](/windows/win32/api/winuser/nf-winuser-createwindowexa) 函數，從靜態類別建立這個控制項。 它有 **SS \_ LEFT**、 **WS \_ CHILD** 和 **ws \_ GROUP** 樣式。

請勿將透明文字控制項放在彩色點陣圖上方。 如果使用者變更顯示色彩配置，可能看不到文字。 例如，如果使用者針對協助工具原因設定高對比參數，文字可能會變成隱藏。
