# <a name="using-the-automation-interface"></a>使用 Automation 介面

若要存取 automation，必須自行註冊 DLL。 用來建立 [**安裝程式物件**](installer-object.md) 的 COM ProgId 是 **WindowsInstaller. 安裝** 程式。

如需自動化和 c + + 程式庫存取的相關資訊，請參閱 [關於 Automation 介面](about-the-automation-interface.md)。

如需 Windows Installer 物件的參考資料，請參閱[Automation 介面參考](automation-interface-reference.md)。
