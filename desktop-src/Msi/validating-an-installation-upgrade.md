# <a name="validating-an-installation-upgrade"></a>驗證安裝升級

每當對封裝進行任何變更時，升級套件的作者應該一律先在其封裝上執行驗證，再嘗試第一次安裝封裝並重新執行驗證。 在升級封裝上執行驗證的方式，與安裝套件的方式相同。 如需詳細資訊，請參閱 [驗證安裝資料庫](validating-an-installation-database.md)。

這會完成範例升級套件。 若要安裝升級，請先安裝 MNP2000.msi，然後再安裝 MNP2001.msi。 當您卸載 MNP2001 時，會從電腦移除原始和升級產品。
