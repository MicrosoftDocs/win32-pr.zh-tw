# <a name="codepage-summary-property"></a>字碼頁摘要屬性

[ **字碼頁摘要** ] 屬性是 ANSI 字碼頁的數值，用於儲存在摘要資訊中的任何字串。 請注意，這並不是安裝資料庫中字串的相同字碼頁。 呼叫 Unicode API 函式時，會使用 **字碼頁摘要** 屬性將摘要資訊中的字串轉譯成 Unicode。 在摘要資訊中設定任何字串屬性之前，必須先設定 **字碼頁的摘要** 屬性。
