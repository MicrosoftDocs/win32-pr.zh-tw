# <a name="leftunit-property"></a>LeftUnit 屬性

如果已設定 **LeftUnit** 屬性，則會將單位放在數位左邊，而不是一般右邊。 這可容納這是接受格式的語言。 UI 會使用這個屬性來顯示檔案大小以及不同磁片區上的可用空間。
