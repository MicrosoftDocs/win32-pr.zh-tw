# <a name="configurableitemhelpkeyword-property"></a>ConfigurableItem. HelpKeyword 屬性

**HelpKeyword** 屬性會傳回 [ModuleConfiguration 資料表](moduleconfiguration-table.md)中 HelpKeyword 資料行的值。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = ConfigurableItem.HelpKeyword
```



## <a name="property-value"></a>屬性值

## <a name="c"></a>C++

請參閱 [**get \_ HelpKeyword**](/windows/desktop/api/Mergemod/nf-mergemod-imsmconfigurableitem-get_helpkeyword) 函式。
