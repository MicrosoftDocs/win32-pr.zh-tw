# <a name="pidkey-property"></a>PIDKEY 屬性

**PIDKEY** 屬性包含使用者輸入的產品識別碼部分。 [ValidateProductID 動作](validateproductid-action.md)會藉由比較 **PIDKEY** 屬性與 [**PIDTemplate**](pidtemplate.md)屬性來驗證所輸入的產品識別碼。

您可以在命令列上或透過使用者介面來設定 **PIDKEY** 屬性。 這個屬性只包含使用者輸入的產品識別碼部分。

| 需求 | 值 |
|--------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
