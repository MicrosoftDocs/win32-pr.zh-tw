# <a name="using-windows-installer-with-wmi"></a>搭配使用 Windows Installer 與 WMI

Windows Installer 提供者可讓 Windows Management Instrumentation (WMI) 存取從 Windows Installer 應用程式收集而來的資訊。

[Windows Installer 提供者](/previous-versions/windows/desktop/msiprov/windows-installer-provider)是選用的 Windows 元件。 選擇性安裝 Windows Installer 提供者可確保回溯相容性，但不表示提供者的未來可用性。 如需有關 Windows Installer 提供者可用性的詳細資訊，請參閱[Windows Management Instrumentation (wmi](../wmisdk/wmi-start-page.md)中[Wmi 元件的作業系統可用性](../wmisdk/operating-system-availability-of-wmi-components.md)) 檔。

如需此 WMI 提供者的目前描述，請參閱主要[Windows Management Instrumentation (WMI) ](../wmisdk/wmi-start-page.md)檔中[Windows Installer 提供者](/previous-versions/windows/desktop/msiprov/windows-installer-provider)。 如需 Windows Installer 所支援的 WMI 類別清單，請參閱 [已安裝的應用程式類別](/previous-versions//aa390887(v=vs.85))。
