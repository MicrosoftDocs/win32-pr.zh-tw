# <a name="recordcleardata-method"></a>ClearData 方法

[**Record**](record-object.md)物件的 **ClearData** 方法會清除所有欄位中的資料，並將其設定為 Null。 儲存在欄位中的任何物件都會釋出。

## <a name="syntax"></a>語法


```JScript
Record.ClearData()
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
