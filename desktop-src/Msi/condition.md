# <a name="condition"></a>條件

Condition 資料類型是一種文字字串，其中包含可評估為 true 或 false 的有效條件陳述式。 如需條件陳述式語法的詳細資訊，請參閱 [條件陳述式語法](conditional-statement-syntax.md)。
