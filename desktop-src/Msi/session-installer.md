# <a name="sessioninstaller-property"></a>Session. Installer 屬性

[**Session**](session-object.md)物件的唯讀 **安裝程式** 屬性會傳回作用中的 **安裝程式** 物件。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = Session.Installer
```
