# <a name="mergeclosemodule-method"></a>Merge. CloseModule 方法

[**merge**](merge-object.md)物件的 **CloseModule** 方法會關閉目前開啟的 Windows Installer Merge 模組。

## <a name="syntax"></a>語法


```JScript
Merge.CloseModule()
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。

## <a name="remarks"></a>備註

關閉合併模組並不會影響任何未取出的錯誤。

### <a name="c"></a>C++

請參閱 [**CloseModule**](/windows/win32/api/mergemod/nf-mergemod-imsmmerge-closemodule) 函式。
