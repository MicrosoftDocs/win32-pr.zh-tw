# <a name="username-property"></a>USERNAME 屬性

**USERNAME** 屬性是執行安裝的使用者。

## <a name="default-value"></a>預設值

預設值可能會放在屬性工作表中。 如果未設定 **USERNAME** 屬性，則會使用登錄中的值自動設定。

## <a name="remarks"></a>備註

設定 [**NOUSERNAME**](nousername.md) 屬性以抑制使用者 **名稱** 的自動設定。
