# <a name="mergeopendatabase-method"></a>Merge. OpenDatabase 方法

[**Merge**](merge-object.md)物件的 **OpenDatabase** 方法會開啟位於指定路徑的 Windows Installer 安裝資料庫，以與模組合併。

## <a name="syntax"></a>語法


```JScript
Merge.OpenDatabase(
  Path
)
```



## <a name="parameters"></a>參數

<dl> <dt>

*路徑* 
</dt> <dd>

要開啟之資料庫的路徑。

</dd> </dl>

## <a name="return-value"></a>傳回值

這個方法不會傳回值。

## <a name="c"></a>C++

請參閱 [**OpenDatabase**](/windows/win32/api/mergemod/nf-mergemod-imsmmerge-opendatabase) 函式。
