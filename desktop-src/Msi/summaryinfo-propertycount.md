# <a name="summaryinfopropertycount-property"></a>SummaryInfo. PropertyCount 屬性

[**SummaryInfo**](summaryinfo-object.md)物件的 **PropertyCount** 屬性是唯讀屬性，表示摘要資訊物件中目前的屬性值數目。 它會將已新增、刪除或取代的屬性納入考慮。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = SummaryInfo.PropertyCount
```
