# <a name="disableloggingfrompackage"></a>DisableLoggingFromPackage

每部電腦 [系統原則](system-policy.md) 的值會設為 "1"，以停用電腦上所有使用者的 [**MsiLogging**](msilogging.md) 屬性為封裝指定的記錄。

[**MsiLogging**](msilogging.md)屬性和 DisableLoggingFromPackage 原則需要 Windows Installer 4.0。

## <a name="registry-key"></a>登錄金鑰

**HKEY \_本機 \_ 電腦** \\ **軟體** \\ **原則** \\ **Microsoft** \\ **Windows** \\ **安裝程式**

## <a name="data-type"></a>資料類型

**REG \_ DWORD**
