# <a name="uipreviewviewdialog-method"></a>UIPreview. ViewDialog 方法

[**UIPreview**](uipreview-object.md)物件的 **ViewDialog** 方法會顯示儲存在目前資料庫中的 [撰寫的使用者介面] 對話方塊。

## <a name="syntax"></a>語法


```JScript
UIPreview.ViewDialog(
  dialog
)
```



## <a name="parameters"></a>參數

<dl> <dt>

*對話 框* 
</dt> <dd>

對話方塊的必要名稱、區分大小寫，以及對話方塊資料庫資料表的主要索引鍵。

</dd> </dl>

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
