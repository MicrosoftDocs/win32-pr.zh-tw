# <a name="shortcut"></a>快速鍵

快速鍵資料類型通常用於 [快捷方式資料表](shortcut-table.md)的目標資料行。 如果 () 包含方括弧 \[ \] ，則會將快捷方式目標評估為格式化字串。 否則，會將快速鍵評估為 [識別碼](identifier.md) ，而且在 [功能資料表](feature-table.md)中必須是有效的外鍵。
