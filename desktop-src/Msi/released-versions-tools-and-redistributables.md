# <a name="released-versions-tools-and-redistributables"></a>已發行的版本、工具和可轉散發套件

下列主題提供 Windows Installer 的已發行版本、工具和可轉散發套件的詳細資訊。

-   [適用于 Windows Installer 開發人員的 Windows SDK 元件](platform-sdk-components-for-windows-installer-developers.md)會說明何處可以取得安裝程式可轉散發元件、檔、資料庫驗證工具、資料庫資料表編輯器、資料庫架構、開發工具、VBScript 工具、範例產品和程式碼範例。
-   [Windows Installer](windows-installer-redistributables.md)可轉散發套件提供安裝程式可轉散發元件的相關資訊。
-   [Windows Installer 的發行版本](released-versions-of-windows-installer.md)會列出隨附于不同作業系統之安裝程式的發行版本。
-   [Windows Installer 開發工具](windows-installer-development-tools.md)包含使用安裝程式開發工具的參考頁面。
-   [Instmsi.exe](instmsi-exe.md) 說明哪些 Instmsi 相容於何種作業系統。 InstMsi 是安裝安裝程式的可轉散發套件。
