# <a name="featureinfodescription-property"></a>FeatureInfo。 Description 屬性

**Description** 屬性是唯讀屬性，會在 [功能資料表](feature-table.md)的 [描述] 資料行中，傳回功能的描述。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = FeatureInfo.Description
```
