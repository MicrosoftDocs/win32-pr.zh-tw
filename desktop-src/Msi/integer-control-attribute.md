# <a name="integer-control-attribute"></a>整數控制項屬性

如果在控制項上設定此位，則在 [控制資料表](control-table.md) 的 [屬性] 資料行中指定的相關聯屬性為整數。 如果未設定此位，則屬性為字串值。

## <a name="valid-controls"></a>有效的控制項

[CheckBox](checkbox-control.md)

[ComboBox](combobox-control.md)

[編輯](edit-control.md)

[ListBox](listbox-control.md)

[RadioButtonGroup](radiobuttongroup-control.md)

## <a name="value"></a>值



| Decimal | 十六進位 | 常數                          |
|---------|-------------|-----------------------------------|
| 16      | 0x00000010  | **msidbControlAttributesInteger** |



 

## <a name="remarks"></a>備註

若要在控制項上設定此屬性，請在控制項 [資料表](control-table.md)之控制項記錄的 [屬性] 資料行中包含整數位。

請參閱 [控制項屬性](control-attributes.md) 和 [控制項](controls.md)。
