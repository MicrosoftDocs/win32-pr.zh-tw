# <a name="sessiondatabase-property"></a>Session 屬性

[**Session**](session-object.md)物件的 **database** 屬性是唯讀屬性，會將目前安裝會話的資料庫傳回做為 [**資料庫**](database-object.md)物件。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = Session.Database
```
