# <a name="text-control-attribute"></a>文字控制項屬性

這個屬性是顯示在控制項中的文字字串。 設定時，如果記錄的欄位0不是 null，則會使用 FormatText 來格式化記錄。 如果欄位0是 null，則記錄的第一個欄位會定義文字。 取得值時，一律會在第一個欄位中傳回。 有些控制項可能看不到這個文字。 在 Windows 控制項的快速鍵的定義方式是將 "&" 放在此字串中所需的字元前面。

## <a name="valid-controls"></a>有效的控制項

除了 [行控制項](line-control.md)以外的所有控制項。

## <a name="associated-bit-flags"></a>相關聯的位旗標

無。

## <a name="remarks"></a>備註

請參閱 [控制項屬性](control-attributes.md) 和您需要在 [控制項](controls.md)下建立的控制項。
