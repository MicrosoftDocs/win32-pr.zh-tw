# <a name="title-summary-property"></a>標題摘要屬性

[ **標題摘要** ] 屬性會簡短描述安裝程式封裝的類型。 這個屬性可能會使用「安裝資料庫」或「轉換」或「修補程式」等片語。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|--------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
