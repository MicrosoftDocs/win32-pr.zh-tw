# <a name="stringlist-object"></a>StringList 物件

**StringList** 物件是字串的集合。 用戶端必須確認 **StringList** 物件存在，而且在參考其屬性之前不是空的。

## <a name="members"></a>成員

**StringList** 物件具有下列類型的成員：

-   [屬性](#properties)

### <a name="properties"></a>屬性

**StringList** 物件具有這些屬性。



| 屬性                                     | 描述                                                          |
|:---------------------------------------------|:---------------------------------------------------------------------|
| [**計數**](stringlist-count.md)<br/> | 傳回 **StringList** 物件中的專案數。<br/> |
| [**項目**](stringlist-item.md)<br/>   | 傳回 **StringList** 物件集合中的字串。<br/> |
