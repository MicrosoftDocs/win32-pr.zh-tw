# <a name="nousername-property"></a>NOUSERNAME 屬性

將 **NOUSERNAME** 屬性設定為1，以抑制安裝程式從 [**USERNAME**](username.md) 屬性的設定。 這個屬性是供需要在第一次執行時收集使用者名稱的應用程式使用。

## <a name="default-value"></a>預設值

未設定。 安裝程式會使用登錄中的值，自動設定使用者 [**名稱**](username.md) 屬性。
