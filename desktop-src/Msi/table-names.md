# <a name="table-names"></a>資料表名稱

資料表名稱限制為31個字元。 如果資料表名稱包含句點 (允許但不建議) 的做法，則名稱必須以抑音符號括住 \` (ASCII 值 96) ，例如， \` filename。 \` 這個轉義語法可防止資料表名稱與保留字（例如 SQL 關鍵字）之間的衝突。 如需詳細資訊，請參閱[SQL 語法](sql-syntax.md)。
