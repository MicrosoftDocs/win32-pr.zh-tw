# <a name="ttcsupport-property"></a>TTCSupport 屬性

安裝程式會設定 **TTCSupport** 屬性，以指出作業系統是否支援 simsun18030.ttc (true 型別字型集合) 檔。

## <a name="remarks"></a>備註

如果安裝程式偵測到 Windows 2000 或下列任何作業系統，就會設定這個屬性：

-   日本-932
-   中國-936
-   臺灣-950
-   韓國-949
-   香港特別行政區-950

|--------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
