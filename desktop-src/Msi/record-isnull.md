# <a name="recordisnull-property"></a>Record 屬性

[**記錄**](record-object.md)物件的 **IsNull** 屬性是唯讀屬性，如果指定的欄位為 Null，則會傳回 True，如果欄位包含資料，則會傳回 False。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = Record.IsNull
```



## <a name="property-value"></a>屬性值

記錄中值的必要欄位編號（以1為基礎）。
