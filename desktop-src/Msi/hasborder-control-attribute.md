# <a name="hasborder-control-attribute"></a>HasBorder 控制項屬性

如果設定此位， [RadioButtonGroup](radiobuttongroup-control.md) 會在其周圍顯示文字和框線。

如果未設定樣式位，則不會顯示框線，而且群組上不會顯示任何文字。

## <a name="valid-controls"></a>有效的控制項

[RadioButtonGroup](radiobuttongroup-control.md)

## <a name="valid"></a>有效



| Decimal  | 十六進位 | 常數                            |
|----------|-------------|-------------------------------------|
| 16777216 | 0x01000000  | **msidbControlAttributesHasBorder** |



 

## <a name="remarks"></a>備註

若要在控制項上設定此屬性，請在控制項 [資料表](control-table.md)的 [屬性] 資料行中包含 HasBorder 位。

請參閱 [控制項屬性](control-attributes.md) 和您需要在 [控制項](controls.md)下建立的控制項。
