# <a name="userexit-dialog"></a>UserExit 對話方塊

這個強制回應對話方塊會在使用者結束的安裝結束時顯示。 此對話方塊通常只包含一些文字和 **[確定]** 按鈕。 這個對話方塊的名稱必須出現在 [AdminUISequence 資料表](adminuisequence-table.md) 和 [InstallUISequence 資料表](installuisequence-table.md) 中，並以-2 作為序號。
