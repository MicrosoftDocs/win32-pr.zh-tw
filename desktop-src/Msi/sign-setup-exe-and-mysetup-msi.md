# <a name="sign-setupexe-and-mysetupmsi"></a>簽署 Setup.exe 和 MySetup.msi

在 Web 服務器上放置 Setup.exe 和 MySetup.msi 之前，您應該使用 [SignTool](../seccrypto/signtool.md) 公用程式，使用您的數位憑證和私密金鑰 mycert.cer .Cer 和 mycert.cer 簽署檔案。 如需有關使用 SignTool 公用程式的詳細資訊，請參閱 Microsoft Windows 軟體開發套件 (SDK) 中的 [CryptoAPI 工具參考](../seccrypto/cryptoapi-tools-reference.md) 。
