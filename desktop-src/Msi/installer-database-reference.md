# <a name="installer-database-reference"></a>安裝程式資料庫參考

若要將安裝程式功能加入至您的應用程式，您可以使用 Windows Installer 資料庫資料表和安裝程式資料庫函數。 本節中的主題描述每個資料庫函數和資料庫資料表。

若要查看所有安裝資料庫資料表的清單，請參閱 [資料庫資料表](database-tables.md)。

若要查看在安裝資料庫中用來處理資訊的函數清單，請參閱 [資料庫](database-functions.md)函式。
