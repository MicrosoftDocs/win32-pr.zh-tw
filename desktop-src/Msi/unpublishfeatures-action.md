# <a name="unpublishfeatures-action"></a>UnpublishFeatures 動作

卸載整個產品時，UnpublishFeatures 動作會從系統登錄移除選取狀態和功能元件對應資訊。 此動作會查詢 [FeatureComponents 資料表](featurecomponents-table.md)。

## <a name="sequence-restrictions"></a>順序限制

沒有任何順序限制。

## <a name="actiondata-messages"></a>ActionData 訊息



| 欄位 | 動作資料的描述             |
|-------|----------------------------------------|
| \[1\] | Unadvertised 功能的識別碼。 |
