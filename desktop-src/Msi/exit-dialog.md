# <a name="exit-dialog"></a>結束對話

成功安裝結束時，會顯示 [結束] 對話方塊。 此對話方塊通常只包含一些文字和 **[確定]** 按鈕。 這個對話方塊的名稱必須出現在 [AdminUISequence 資料表](adminuisequence-table.md) 和 [InstallUISequence 資料表](installuisequence-table.md) 中，並以-1 作為序號。 這是強制回應對話方塊。
