# <a name="ice29"></a>ICE29

ICE29 會驗證截斷的資料流程名稱是否維持唯一。 任何具有二進位或物件資料行的資料表都會經過驗證。 請參閱 [Binary](binary.md) 資料行資料類型。

由 Win32 OLE 結構化儲存體執行的資料流程處理會限制資料流程名稱。 請參閱 [OLE 對資料流程的限制](ole-limitations-on-streams.md)。 安裝程式最多可壓縮長度為62個字元的資料流程名稱。 長度超過此名稱的名稱會被截斷。
