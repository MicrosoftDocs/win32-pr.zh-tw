# <a name="program-information-tables-group"></a>程式資訊資料表群組

「程式資訊資料表」群組包含在整個安裝過程中使用的重要資訊。

-   [屬性工作表](property-table.md)提供一個方法來指定安裝的所有[屬性](properties.md)。
-   [二進位資料表](binary-table.md)會保存點陣圖、動畫和圖示等專案的二進位資料。 二進位資料表也用來儲存自訂動作的資料。 此資料表可以包含佈告欄的點陣圖、程式的圖示，或自訂動作的可執行形式。
-   [錯誤資料表](error-table.md)是用來在處理錯誤並設定錯誤碼時，查詢錯誤訊息格式化範本。 這適用于未設定格式化範本的一般情況。 安裝程式有自己的錯誤處理機制。 錯誤會以記錄的形式傳遞。
-   [快速鍵表格](shortcut-table.md) ：快捷方式表包含應用程式在使用者電腦上建立快捷方式所需的資訊。
-   [ReserveCost 資料表](reservecost-table.md)包含每個元件正常運作所需的磁碟空間。
