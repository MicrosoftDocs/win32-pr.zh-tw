# <a name="ice-104"></a>ICE 104

ICE 104 只會驗證兩個數據表中的其中一個： [MsiLockPermissionsEx](msilockpermissionsex-table.md) 和 [LockPermissions](lockpermissions-table.md) 都存在於封裝中。

ICE 104 會驗證 [MsiLockPermissionsEx](msilockpermissionsex-table.md) 資料表的 LockObject、資料表和 SDDLText 欄位中所使用的語法。 ICE 104 不會驗證 MsiLockPermissionsEx 資料表中 SDDLText 欄位的值是否為有效的安全描述項。

**[Windows Installer 4.5 或更早版本](not-supported-in-windows-installer-4-5.md)：** 不支援。 從 Windows Installer 5.0 開始，可以使用這項冰。
