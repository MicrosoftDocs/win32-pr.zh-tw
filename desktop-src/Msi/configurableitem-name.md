# <a name="configurableitemname-property"></a>ConfigurableItem.Name 屬性

[**ConfigurableItem**](configurableitem-object.md)物件的 **name** 屬性會從 ModuleConfiguration 資料表的 name 資料行傳回值。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = ConfigurableItem.Name
```



## <a name="property-value"></a>屬性值

## <a name="c"></a>C++

請參閱 [**取得 \_ 名稱**](/windows/desktop/api/Mergemod/nf-mergemod-imsmconfigurableitem-get_name) 函數。
