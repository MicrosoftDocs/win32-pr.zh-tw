# <a name="installation-on-demand"></a>隨選安裝

使用傳統安裝技術時，必須結束應用程式並重新執行安裝程式，才能執行安裝工作。 當使用者想要在第一次執行安裝程式期間未選擇的功能或產品時，通常就會發生這種情況。 這通常會讓產品設定的程式效率不佳，因為它需要使用者先預期需要的功能，才能使用該產品。

隨選安裝讓您能夠在沒有檔案本身的情況下，為使用者和應用程式提供功能。 這個概念稱為「通告」。 Windows Installer 具有廣告功能的功能，並可讓您視需要進行應用程式功能或整個產品的安裝。 當使用者或應用程式啟用已公告的功能或產品時，安裝程式會繼續安裝所需的元件。 這會縮短設定程式，因為可以存取額外的功能，而不需要結束再重新執行另一個安裝程式。

當產品使用安裝程式時，使用者可以在安裝時選擇要安裝的功能或應用程式，以及要通告的應用程式。 然後，如果在執行應用程式時，使用者要求尚未安裝的已公告功能，應用程式就會呼叫安裝程式，以制定及時的功能層級安裝所需的檔案。 如果使用者啟用尚未安裝的已公告產品，作業系統會呼叫安裝程式來制定及時的產品層級安裝。

[公告](advertisement.md) 和隨選安裝也可以讓系統管理員針對不同的使用者群組，將應用程式指定為必要或選擇性，以協助進行系統管理。 有兩種類型的廣告稱為「指派」和「發佈」。 如果系統管理員將應用程式指派給群組，這些使用者就可以視需要安裝應用程式。 但是，如果系統管理員將應用程式發佈至群組，則不會對這些使用者顯示任何進入點，而且只有當另一個應用程式啟動已發佈的應用程式時，才會啟用隨選安裝。
