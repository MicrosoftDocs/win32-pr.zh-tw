# <a name="user-policies"></a>使用者原則

您可以在下列情況下設定下列使用者原則：

**HKEY \_目前的 \_ 使用者** \\ **軟體** \\ **原則** \\ **Microsoft** \\ **Windows** \\ **Installer**



| 值名稱                                                            | 值資料類型          | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
|-----------------------------------------------------------------------|---------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [AlwaysInstallElevated](alwaysinstallelevated.md)<br/>         | **REG \_ DWORD**<br/> | 如果這個值設定為 "1"，而且對應的電腦值也已設定，則安裝程式一律會以較高的許可權進行安裝。<br/> 否則，安裝程式會使用較高的許可權來安裝受控應用程式，並針對非受管理應用程式使用目前使用者的許可權層級。<br/>                                                                                                                                                                                                      |
| [DisableMedia](disablemedia.md)<br/>                           | **REG \_ DWORD**<br/> | 如果 [DisableMedia](disablemedia.md) 原則設定為 "1"，則執行某項產品維護安裝的使用者和系統管理員，將無法使用 [流覽] 對話方塊來流覽媒體來源（例如 cd-rom），以取得其他可安裝產品的來源。 不論安裝是否使用較高的許可權，都可以防止流覽其他產品。 如果使用者已正確標示媒體來源，則使用者仍然可以從媒體重新安裝產品。<br/> |
| [DisableRollback](disablerollback.md)<br/>                     | **REG \_ DWORD**<br/> | 如果這個值設定為 "1"，安裝程式將不會在安裝期間儲存復原檔案，而會停用安裝復原。 預設會啟用復原。 除非系統管理員絕對必要，否則建議不要使用此原則。<br/>                                                                                                                                                                                                                                                             |
| [SearchOrder](searchorder.md)<br/>                             | **REG \_ SZ**<br/>    | 安裝程式搜尋三種不同類型來源的順序：<br/> "n" –網路<br/> "m" –媒體 (CD-ROM 或 DVD) <br/> "u" – (統一資源定位器) 的 URL<br/> 例如，"nmu" 的值會指示安裝程式先搜尋網路來源、媒體來源第二個和 URL 來源。 省略字母會從搜尋中移除對應的磁片區類型。 此值沒有此值的預設順序為 network first，then 媒體接著 URL。<br/>          |
| [TransformsAtSource 原則](transformsatsource-policy.md)<br/> | **REG \_ DWORD**<br/> | 如果這個值存在且設為 "1";安裝程式會在產品的 [**sourcelist**](sourcelist.md) 中，于任何網路來源的根目錄中搜尋轉換檔案。 根據預設，轉換會儲存在使用者設定檔的 Application Data 資料夾中。<br/>                                                                                                                                                                                                                                             |
