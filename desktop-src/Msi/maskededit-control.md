# <a name="maskededit-control"></a>MaskedEdit 控制項

MaskedEdit 控制項是編輯欄位控制項，包含控制項文字欄位中的遮罩。 您可以藉由在 [控制項資料表](control-table.md)的屬性資料行中輸入屬性名稱，將控制項與字串值屬性產生關聯。

您可以使用 MaskedEdit 控制項來建立使用者輸入資訊的範本，例如電話號碼或產品識別碼代碼。 例如，使用者可以透過將 [**PIDTemplate**](pidtemplate.md)屬性設定為如下所指定之字串的 MaskedEdit 控制項，來輸入 [**PIDKEY**](pidkey.md)屬性：

12345<\#\#\# -%%%%%%%>@@@@@

字串會定義使用者的 [**PIDKEY**](pidkey.md) 屬性之專案的遮罩範本。 字串的可見區段會以一對括弧括住 (<>) 個字元。

下表指出遮罩的語法。



| 字元 | 意義                                                                                                                                                                                                                                                                                     |
|-----------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <      | 範本可見區段的左邊端。 此字元及其左邊的所有專案都會隱藏在使用者介面中。 範本中應該不能有一個以上的此字元實例。                                                                               |
| >      | 範本可見區段的右端。 此字元及其右邊的所有內容都會隱藏在使用者介面中。 在驗證期間，此字元會取代為虛線。 如果有可見區段以 < 開頭，則必須以相符的 > 結束。 |
| \#        | 此字元可以是數位 (數位。 )                                                                                                                                                                                                                                                     |
| %         | 此字元可以是替代數位 (數位) ，可讓遮罩控制自訂動作如何區分欄位。                                                                                                                                                          |
| @         | 此字元可以是 (數位的亂數字。 ) 此字元不應該出現在範本的可見部分中。                                                                                                                                                                       |
| &         | 此字元可以是任何字元。                                                                                                                                                                                                                                                        |
| ^         | 此字元可以是替代字元，此字元可讓遮罩控制自訂動作與欄位區別的方式。                                                                                                                                                                |
| ?         | 此字元可以是替代字元，此字元可讓遮罩控制自訂動作與欄位區別的方式。                                                                                                                                                                |
| \`        |以抑音符 \` 號 (ASCII 值 96) 可以表示替代字元，讓遮罩控制自訂動作區分欄位的方式。                                                                                                                                 |
| \_        | 此字元是常值底線字元。                                                                                                                                                                                                                                           |
| =         | 此字元是欄位結束字元。 這必須遵循 a \# 、%、^ 或 \` 。 這會建立一個與先前位置相同類型的輸入位置，然後以 '-' 分隔符號結束欄位。                                                                                 |



 

任何其他字元都會被視為常值常數。

針對可編輯的字元，控制項會針對相同類型的每個連續字元區塊，以一個視窗建立個別的編輯視窗。

## <a name="remarks"></a>備註

MaskedEdit 控制項會以 **BS \_ OWNERDRAW** 和 **WS \_ EX \_ CONTROLPARENT** 樣式，建立 **BUTTON** 類別的一個父視窗。 它會在此視窗中建立數個子視窗。

-   如果是常數文字部分，則會建立具有 **SS \_ LEFT** 和 **WS \_ 子** 樣式的靜態視窗。
-   針對可編輯的欄位，它會建立具有 **ws \_ 子** 系、 **ws \_ 框線** 和 **ws \_ TABSTOP** 樣式的編輯視窗。
-   若為數值欄位，視窗也會有 **ES \_ 數位** 樣式。

替代位數、% 和替代的英數位元、^、？和 \` 欄位可讓自訂動作以遮罩可控制的方式區分欄位，例如 ^ 可用於應為大寫的欄位。
