# <a name="32-bit-windows-installer-packages"></a>32位 Windows Installer 套件

32位封裝只由32位 [Windows Installer 元件](windows-installer-components.md) 所組成，而且必須在 [ [**範本摘要**](template-summary.md) ] 屬性的 [平臺] 欄位中輸入「Intel」值。

如需詳細資訊，請參閱 [64 位作業系統上的 Windows Installer](windows-installer-on-64-bit-operating-systems.md) 和 [64 位 Windows Installer 套件](64-bit-windows-installer-packages.md)。
