# <a name="custom-action-sources"></a>自訂動作來源

自訂動作來源可以存在於數種不同的表單中。 下表列出可以有各種來源形式的自訂動作類型。 如需詳細資訊，請參閱基本自訂動作類型的主題。



| 自訂動作來源                                           | 相關聯的自訂動作類型                                                                                                                                                                                 |
|----------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 二進位資料表中所儲存的自訂動作。                      | [自訂動作類型 1](custom-action-type-1.md)， [自訂動作類型 2](custom-action-type-2.md)， [自訂動作類型 5](custom-action-type-5.md)， [自訂動作類型 6](custom-action-type-6.md)         |
| 在安裝期間複製的自訂動作。                      | [自訂動作類型 17](custom-action-type-17.md)， [自訂動作類型 18](custom-action-type-18.md)， [自訂動作類型 21](custom-action-type-21.md)， [自訂動作類型 22](custom-action-type-22.md) |
| 參考目錄的自訂動作。                         | [自訂動作類型34](custom-action-type-34.md)                                                                                                                                                             |
| 參考屬性的自訂動作。                          | [自訂動作類型 50](custom-action-type-50.md)， [自訂動作類型 53](custom-action-type-53.md)， [自訂動作類型 54](custom-action-type-54.md)                                                     |
| 以常值腳本的形式儲存在資料庫資料表中的自訂動作。 | [自訂動作類型 37](custom-action-type-37.md)， [自訂動作類型 38](custom-action-type-38.md)                                                                                                         |
| 顯示錯誤訊息的自訂動作。                     | [自訂動作類型19](custom-action-type-19.md)                                                                                                                                                             |



 

查看 [所有自訂動作類型的摘要清單](summary-list-of-all-custom-action-types.md) ，以取得所有自訂動作類型的摘要，以及如何在 [CustomAction 資料表](customaction-table.md)中指定來源。
