# <a name="using-installer-functions"></a>使用安裝程式函數

若要將 Windows Installer 功能新增至您的應用程式，請使用安裝程式服務功能。 因為安裝程式是安裝公用程式和元件管理系統，所以使用安裝程式的應用程式必須將安裝程式服務函式併入正常操作的一部分。

下列主題和程式說明如何在您的應用程式中執行安裝程式功能：

-   [初始化應用程式](initializing-an-application.md)
-   [取得應用程式資訊](getting-application-information.md)
-   [要求功能](requesting-a-feature.md)
-   [安裝應用程式](installing-an-application.md)
-   [重新安裝功能或應用程式](reinstalling-a-feature-or-application.md)
-   [移除已公告的應用程式](removing-an-advertised-application.md)
-   [使用功能和元件](working-with-features-and-components.md)
-   [安裝遺失的元件](installing-a-missing-component.md)
-   [清查產品和修補程式](inventory-products-and-patches-.md)
-   [管理安裝來源](managing-installation-sources.md)
