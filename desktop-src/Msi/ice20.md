# <a name="ice20"></a>ICE20

ICE20 會驗證具有所撰寫使用者介面之安裝套件的標準對話方塊和對話需求。 具有撰寫使用者介面的安裝套件具有 [對話方塊資料表](dialog-table.md) ，且未設定 [**LIMITUI**](limitui.md) 屬性。

ICE20 會驗證下列資料表和需求。



| 標準對話方塊                            | 對話方塊需求                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
|--------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [FilesInUse 對話方塊](filesinuse-dialog.md) | [ListBox 資料表](listbox-table.md)。 [清單方塊控制項](listbox-control.md)，其[控制項資料表](control-table.md)的屬性資料行設定為 FileInUseProcess。<br/> 在[ControlEvent 資料表](controlevent-table.md)的事件資料行中輸入 EndDialog，並在 Argument 資料行中忽略的[按鈕控制項](pushbutton-control.md)。<br/> 在[ControlEvent 資料表](controlevent-table.md)的事件資料行中輸入 EndDialog，並在 Argument 資料行中結束的[按鈕控制項](pushbutton-control.md)。<br/> 在[ControlEvent 資料表](controlevent-table.md)的事件資料行中輸入 EndDialog 的[按鈕控制項](pushbutton-control.md)，然後在 Argument 資料行中重試。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| [錯誤對話方塊](error-dialog.md)           | [錯誤對話方塊樣式位](error-dialog-style-bit.md)必須設定為使用任何名稱來指定對話方塊，而這是錯誤對話方塊。 名為 ErrorText 的文字控制項。<br/> 在 \_ [對話方塊資料表](dialog-table.md)的 [控制] 第一個資料行中 ErrorText。<br/> 在 [ControlEvent 資料表](controlevent-table.md) 的事件資料行中具有 EndDialog 的 [按鈕] 控制項，以及 [引數] 資料行中的 ErrorAbort。<br/> 名為 C 的按鈕控制項，在 [ControlEvent 資料表](controlevent-table.md) 的事件資料行中具有 EndDialog，而 ErrorCancel 在 Argument 資料行中。<br/> 名為 I 的按鈕控制項，在 [ControlEvent 資料表](controlevent-table.md) 的事件資料行中具有 EndDialog，而 ErrorIgnore 在 Argument 資料行中。<br/> 名為 N 的按鈕控制項，在 [ControlEvent 資料表](controlevent-table.md) 的事件資料行中具有 EndDialog，而 ErrorNo 在 Argument 資料行中。<br/> [ControlEvent 資料表](controlevent-table.md)的事件資料行中有 EndDialog 的按鈕控制項，名為 O，在 Argument 資料行中 ErrorOk。<br/> 名為 R 的按鈕控制項，在 [ControlEvent 資料表](controlevent-table.md) 的事件資料行中有 EndDialog，而 ErrorRetry 在 Argument 資料行中。<br/> 名為 Y 的按鈕控制項，在 [ControlEvent 資料表](controlevent-table.md) 的事件資料行中具有 EndDialog，而 ErrorYes 在 Argument 資料行中。<br/> 所有按鈕的寬度、高度和 Y 座標都應該相同。 X 座標取決於安裝程式。<br/> 如果名為 ErrorIcon 的控制項存在，它必須在 [控制資料表](control-table.md)的類型資料行中有圖示。<br/> |
| [結束對話](exit-dialog.md)             | 在 [順序] 資料行中， [AdminUISequence 資料表](adminuisequence-table.md) 和 [InstallUISequence 資料表](installuisequence-table.md) 中必須有一個值為-1 的對話方塊或自訂動作。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| [UserExit 對話方塊](userexit-dialog.md)     | 在 [順序] 資料行中， [AdminUISequence 資料表](adminuisequence-table.md) 和 [InstallUISequence 資料表](installuisequence-table.md) 中必須有一個值為2的對話方塊或自訂動作。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| [FatalError 對話方塊](fatalerror-dialog.md) | 在 [順序] 資料行中， [AdminUISequence 資料表](adminuisequence-table.md) 和 [InstallUISequence 資料表](installuisequence-table.md) 中必須有一個對話方塊或自訂動作的值為3。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |



 

## <a name="result"></a>結果

如果 ICE20 無法驗證標準對話方塊和需求是否存在於安裝套件中，則會張貼錯誤訊息。
