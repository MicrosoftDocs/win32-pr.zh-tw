# <a name="installerpatchtransforms-property"></a>PatchTransforms 屬性

唯讀 **PatchTransforms** 屬性會傳回指定修補封裝中的轉換清單（以分號分隔），並套用至指定的產品。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = Installer.PatchTransforms
```



## <a name="property-value"></a>屬性值

|--------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
