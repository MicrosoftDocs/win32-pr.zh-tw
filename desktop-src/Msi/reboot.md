# <a name="reboot-property"></a>重新開機屬性

[ **重新開機** ] 屬性會抑制系統重新開機的特定提示。 系統管理員通常會使用此內容搭配一系列的安裝，在最後一次只重新開機時安裝數項產品。 如需詳細資訊，請參閱 [系統重新開機](system-reboots.md)。

[ForceReboot](forcereboot-action.md)和[ScheduleReboot](schedulereboot-action.md)動作會通知安裝程式提示使用者重新開機系統。 安裝程式也可以判斷序列中是否有任何 ForceReboot 或 ScheduleReboot 動作，需要重新開機。 例如，安裝程式在安裝期間需要取代任何使用中的檔案時，會自動提示重新開機。

您可以設定 [ **重新開機** ] 屬性來隱藏某些提示以重新開機，如下所示。



| 重新開機值   | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
|----------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Force          | 一律在安裝結束時提示您重新開機。 UI 一律會提示使用者在結束時重新開機選項。 如果沒有使用者介面，而且這不是 [多重封裝安裝](multiple-package-installations.md)，系統會在安裝結束時自動重新開機。 如果這是多套件安裝，系統就不會自動重新開機系統，且安裝程式會傳回錯誤 \_ 成功 \_ 重新開機 \_ 。 |
| 隱藏       | 抑制在安裝結束時重新開機的提示。 在安裝期間，安裝程式仍會提示使用者在遇到 [ForceReboot 動作](forcereboot-action.md)時重新開機。 如果沒有使用者介面，系統會在每個 ForceReboot 自動重新開機。 在安裝結束時重新開機 (例如，嘗試安裝使用中的檔案時所造成的) 已隱藏。                                    |
| ReallySuppress | 在安裝期間隱藏 ForceReboot 所起始的所有重新開機和重新開機提示。 在安裝結束時隱藏所有重新開機並重新啟動提示。 重新開機提示和重新開機本身都會隱藏。 例如，在安裝結束時重新開機，因為嘗試安裝使用中的檔案，所以會隱藏。                                                                                                                    |



 

## <a name="remarks"></a>備註

安裝程式只會評估 **重新開機** 屬性的第一個字元。
