# <a name="primaryvolumepath-property"></a>PrimaryVolumePath 屬性

安裝程式會將 **PrimaryVolumePath** 屬性的值設定為 [**PRIMARYFOLDER**](primaryfolder.md) 屬性所指定之磁片區的路徑。 例如，如果 **PRIMARYFOLDER** 所參考的資料夾解析為 "d： \\ ProgramFiles"， **PrimaryVolumePath** 就會設定為 "d："。
