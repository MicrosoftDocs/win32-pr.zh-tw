# <a name="configurableitemhelplocation-property"></a>ConfigurableItem. HelpLocation 屬性

**HelpLocation** 屬性會傳回 [ModuleConfiguration 資料表](moduleconfiguration-table.md)中 HelpLocation 資料行的值。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = ConfigurableItem.HelpLocation
```



## <a name="property-value"></a>屬性值

## <a name="c"></a>C++

請參閱 [**get \_ HelpLocation**](/windows/desktop/api/Mergemod/nf-mergemod-imsmconfigurableitem-get_helplocation) 函式。
