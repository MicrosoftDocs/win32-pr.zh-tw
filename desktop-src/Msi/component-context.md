# <a name="componentcontext-property"></a>Component 屬性

判斷為適用于問題元件的內容。 這個屬性會呼叫 [**MsiEnumComponentsEx**](/windows/desktop/api/Msi/nf-msi-msienumcomponentsexa) 函數。

**[Windows Installer 4.5 或更早版本](not-supported-in-windows-installer-4-5.md)：** 不支援。 從 Windows Installer 5.0 開始，可以使用這個屬性。
