# <a name="path"></a>路徑

路徑資料類型是包含有效完整路徑的文字字串。

字串也可以包含以方括弧括住的屬性名稱 \[ \] 。 在這種情況下，屬性（property）的值會在字串中取代屬性（property）的名稱（包括括弧）。

範例：

-   UNC 路徑： \\ \\ 伺服器 \\ 共用
-   本機磁片磁碟機： c： \\ temp
-   屬性名稱： \[ 磁片磁碟機 \] \\ temp
