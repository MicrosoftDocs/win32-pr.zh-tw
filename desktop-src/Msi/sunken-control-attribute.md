# <a name="sunken-control-attribute"></a>凹陷控制項屬性

如果設定此位，則會顯示具有凹陷、三維外觀的控制項。 在不同的 Windows 控制項和版本上，此樣式位的效果不同。 在某些控制項上，它沒有任何可見效果。 如果系統不支援凹陷控制項屬性，控制項會以預設視覺化樣式顯示。 如果未設定此位，則會顯示具有預設視覺化樣式的控制項。

## <a name="valid-controls"></a>有效的控制項

所有控制項。

## <a name="value"></a>值



| Decimal | 十六進位 | 常數                         |
|---------|-------------|----------------------------------|
| 4       | 0x00000004  | **msidbControlAttributesSunken** |



 

## <a name="remarks"></a>備註

若要在控制項上設定此屬性，請在控制項 [資料表](control-table.md)的 [屬性] 資料行中包含凹陷位。

請參閱 [控制項屬性](control-attributes.md) 和 [控制項](controls.md)。
