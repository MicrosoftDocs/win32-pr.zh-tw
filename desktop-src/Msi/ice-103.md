# <a name="ice-103"></a>ICE 103

ICE 103 會驗證 [MsiPrint](msiprint-controlevent.md) 和 [MsiLaunchApp](msilaunchapp-controlevent.md) 控制事件。

ICE 103 會驗證 [MsiPrint](msiprint-controlevent.md) 控制項事件僅用於具有 [ScrollableText](scrollabletext-control.md) 控制項的對話方塊。

**[Windows Installer 4.5 或更早版本](not-supported-in-windows-installer-4-5.md)：** 不支援。 從 Windows Installer 5.0 開始，可以使用這項冰。
