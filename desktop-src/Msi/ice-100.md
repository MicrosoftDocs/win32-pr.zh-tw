# <a name="ice-100"></a>ICE 100

ICE100 會檢查 [MsiEmbeddedUI](msiembeddedui-table.md) 資料表和 [MsiEmbeddedChainer](msiembeddedchainer-table.md) 資料表的撰寫。

## <a name="result"></a>結果

ICE100 會張貼下列錯誤。



| ICE100 錯誤                                                  | 描述                                                                                                                                                                                                                                             |
|---------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 標記為 UI 的 Dll 太多： \[ 1\]                             | [MsiEmbeddedUI](msiembeddedui-table.md)資料表的 Attributes 資料行包含太多標記為使用者介面 Dll 的 dll。 只有一個 DLL 應標示為使用者介面 DLL。                                                               |
| UI DLL 不可以有 null/零訊息篩選： \[ 1\]            | 在 [ [MsiEmbeddedUI](msiembeddedui-table.md) ] 資料表的 [屬性] 資料行中，標記為使用者介面 DLL 的 Dll 在 MessageFilter 欄位中有 null。 如果資料列參考使用者介面 DLL，則 [MessageFilter] 欄位中的值不應為 null。 |
| 資源 Dll 必須有 null/零訊息篩選： \[ 1\]      | 在 [MsiEmbeddedUI](msiembeddedui-table.md) 資料表中未標示為使用者介面 DLL 的 dll，在 MessageFilter 欄位中必須有 null。 如果資料列參考資源檔，而屬性的值為 null，則此欄位應為 null。            |
| 以這個檔案名標記的多個 Dll： \[ 2 \] (索引： \[ 1 \])  | [MsiEmbeddedUI](msiembeddedui-table.md)資料表中的多個 dll 具有相同的檔案名。 FileName 資料行不應包含重複的名稱。                                                                                                    |



 

ICE100 會張貼下列警告。



| ICE100 警告                                                                                          | 描述                                                                                                                                                                                                                                                                                                                         |
|---------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| MsiEmbeddedChainer 資料表中有多個 chainers。 請確定只有一個條件可執行。 | [MsiEmbeddedChainer](msiembeddedchainer-table.md)資料表中有多個專案。 只有一個內嵌 chainer 可以啟動。 如果有多個專案都是以條件方式執行，則不會定義要執行的專案。 如果使用者確定只有一個專案的條件可以解析為 true，則可以忽略此警告。 |
