# <a name="customsource"></a>CustomSource

CustomSource 資料類型是有效的 [識別碼](identifier.md) 或外部索引鍵，取決於 [CustomAction 資料表](customaction-table.md)的類型資料行。 它可以是 [二進位資料表](binary-table.md)、檔案 [資料表](file-table.md)、 [目錄資料表](directory-table.md)和 [屬性工作表](property-table.md)中的外部索引鍵。 請注意，屬性資料表中的外部索引鍵不會經過驗證，因為可以在執行時間加入屬性。
