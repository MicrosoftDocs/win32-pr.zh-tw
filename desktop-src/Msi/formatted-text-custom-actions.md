# <a name="formatted-text-custom-actions"></a>格式化的文本自訂動作

這種類型的自訂動作會從格式化的文字字串設定屬性或安裝目錄。 這個字串中的參數可以是屬性、環境變數、檔案路徑或元件的目錄路徑。 這適用于將資訊傳遞至後續的自訂動作。

下列基本自訂動作類型會設定屬性或目錄。 如需詳細資訊，請參閱特定的自訂動作類型。



| 自訂動作類型                                 | Description                                    |
|----------------------------------------------------|------------------------------------------------|
| [自訂動作類型51](custom-action-type-51.md) | 使用格式化的文字字串來設定屬性。  |
| [自訂動作類型35](custom-action-type-35.md) | 使用格式化的文字字串來設定目錄。 |



 

請參閱 [所有自訂動作類型](summary-list-of-all-custom-action-types.md) 的摘要清單，以取得所有自訂動作類型的摘要，以及它們如何編碼至 [CustomAction 資料表](customaction-table.md)。
