# <a name="dependencyversion-property"></a>Dependency。 Version 屬性

唯讀 **版本** 屬性會傳回目前合併所需的模組版本。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = Dependency.Version
```



## <a name="property-value"></a>屬性值

## <a name="c"></a>C++

請參閱 [**get \_ Version**](/windows/win32/api/mergemod/nf-mergemod-imsmdependency-get_version) 函數。
