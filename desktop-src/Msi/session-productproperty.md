# <a name="sessionproductproperty-property"></a>ProductProperty 屬性

[**Session**](session-object.md)物件的 **ProductProperty** 屬性是唯讀屬性，可從 product 資料庫抓取產品屬性。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = Session.ProductProperty
```



## <a name="property-value"></a>屬性值

需要區分大小寫的屬性名稱。
