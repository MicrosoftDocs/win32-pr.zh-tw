# <a name="system16folder-property"></a>System16Folder 屬性

Windows安裝程式會將 **System16Folder** 屬性設定為16位系統 dll 之資料夾的完整路徑。 請注意， **System16Folder** 屬性並未針對64位 Windows 定義，因為此系統不支援16位的應用程式或元件。
