# <a name="summary-information-stream"></a>摘要資訊串流

本節中的材質適用于撰寫自己的安裝程式的開發人員，他們想要深入瞭解 Windows Installer 資料庫資料表。 如需一般資訊，請參閱[關於 Windows Installer](about-windows-installer.md)。

安裝程式會使用摘要資訊資料流程進行兩個目的。 首先，它包含可透過 Microsoft Windows 檔案總管查看之封裝的相關資訊。 這項資訊可透過 **IStream** 介面進行存取。 其次，它包含安裝程式用來安裝封裝的屬性。 下列主題提供有關如何搭配安裝程式使用摘要資訊串流的相關資訊：

-   [關於摘要資訊資料流程](about-the-summary-information-stream.md)
-   [使用摘要資訊資料流程](using-the-summary-information-stream.md)
-   [摘要資訊資料流程參考](summary-information-stream-reference.md)
