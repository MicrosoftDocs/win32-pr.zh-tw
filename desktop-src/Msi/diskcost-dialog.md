# <a name="diskcost-dialog"></a>DiskCost 對話方塊

當設定 [**OutOfDiskSpace**](outofdiskspace.md) 屬性時，通常會出現 DiskCost 對話方塊。 這表示選取的安裝沒有足夠的磁碟空間。

您也可以從 [ [選取專案] 對話方塊](selection-dialog.md) 啟動此對話方塊，讓使用者瞭解安裝所需的磁碟空間。

[DiskCost] 對話方塊是一個強制回應對話方塊。 它通常會包含一個 [按鈕](pushbutton-control.md) ，讓使用者回到上一個對話方塊和一個 [VolumeCostList 控制項](volumecostlist-control.md)。
