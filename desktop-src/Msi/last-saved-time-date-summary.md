# <a name="last-saved-timedate-summary-property"></a>上次儲存的時間/日期摘要屬性

[ **上次儲存的時間/日期摘要** ] 屬性會傳達上次修改此安裝套件、轉換或修補套件的時間。

一開始，作者應該將 [ **上次儲存時間/日期摘要** ] 屬性的值設為 Null，表示尚未對封裝進行任何變更。 然後，作者應在每次儲存修改的安裝資料庫、轉換或修補套件時，將 **上次儲存的時間/日期摘要** 屬性更新為目前的系統時間/日期。
