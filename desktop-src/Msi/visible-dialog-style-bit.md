# <a name="visible-dialog-style-bit"></a>可見的對話方塊樣式位

如果設定此位，則預設會將對話建立為可見，否則就會隱藏起來。

## <a name="value"></a>值



| Decimal | 十六進位 | 常數                         |
|---------|-------------|----------------------------------|
| 1       | 0x00000001  | **msidbDialogAttributesVisible** |
