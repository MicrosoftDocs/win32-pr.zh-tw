# <a name="configurableitemdescription-property"></a>ConfigurableItem。 Description 屬性

**Description** 屬性會傳回 [ModuleConfiguration 資料表](moduleconfiguration-table.md)之 Description 資料行中的值。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = ConfigurableItem.Description
```



## <a name="property-value"></a>屬性值

## <a name="c"></a>C++

請參閱 [**get \_ Description**](/windows/desktop/api/Mergemod/nf-mergemod-imsmconfigurableitem-get_description) 函數
