# <a name="desktopfolder-property"></a>DesktopFolder 屬性

安裝程式會將 **DesktopFolder** 屬性設定為目前使用者的桌面資料夾的完整路徑。 如果有「所有使用者」設定檔存在，而且已設定 [**ALLUSERS**](allusers.md) 屬性，則這個屬性會設定為「所有使用者」設定檔中的資料夾。
