# <a name="word-count-summary-property"></a>字數統計摘要屬性

在安裝封裝的摘要資訊中，[ **字數統計** ] 屬性會指出來源檔案映射的類型。 如果這個屬性不存在，則預設為零 (0) 。

這個屬性是位欄位。 未來可能會加入新的位。 目前提供下列位。



| bit   | 值          | 描述                                                                                                                                                                                                                                      |
|-------|----------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 位0 | 0 1<br/> | 長檔名。 簡短檔案名。<br/>                                                                                                                                                                                                    |
| 位1 | 0 2<br/> | 來源未壓縮。 來源已壓縮。<br/>                                                                                                                                                                                         |
| 位2 | 0 4<br/> | 來源是原始媒體。 來源是系統管理安裝所建立的系統管理映射。<br/>                                                                                                                                 |
| 位3 | 0 8<br/> | 若要安裝此套件，可能需要較高的許可權。 安裝此套件不需要較高的許可權。<br/> 從 Windows Installer 4.0 版和 Windows Vista 或 Windows Server 2008 開始提供。<br/> |



 

這些會合並以提供 [ **字數統計** ] 屬性的下列其中一個值，表示來源檔案影像的類型。



| 值 | 類型                                                                                                                                                                                                                                                                                            |
|-------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 0     | 使用長檔名的原始來源。 符合 [目錄資料表](directory-table.md)中的樹狀結構。 若要安裝此套件，可能需要較高的許可權。                                                                                                                                     |
| 1     | 使用短檔案名的原始來源。 符合 [目錄資料表](directory-table.md)中的樹狀結構。 若要安裝此套件，可能需要較高的許可權。                                                                                                                                    |
| 2     | 使用長檔名的壓縮原始檔。 符合 [媒體資料表](media-table.md)中的封包和檔案。 若要安裝此套件，可能需要較高的許可權。                                                                                                                   |
| 3     | 使用短檔案名的壓縮原始檔。 符合 [媒體資料表](media-table.md)中的封包和檔案。 若要安裝此套件，可能需要較高的許可權。                                                                                                                  |
| 4     | 使用長檔名的系統管理映射。 符合 [目錄資料表](directory-table.md)中的樹狀結構。 若要安裝此套件，可能需要較高的許可權。                                                                                                                                |
| 5     | 使用簡短檔案名的系統管理映射。 符合 [目錄資料表](directory-table.md)中的樹狀結構。 若要安裝此套件，可能需要較高的許可權。                                                                                                                               |
| 8     | 安裝此套件不需要較高的許可權。 在[不使用 UAC 對話方塊的情況下撰寫套件](authoring-packages-without-the-uac-dialog-box.md)時，請使用此值。從 Windows Installer 4.0 版和 Windows Vista 或 Windows Server 2008 開始提供。<br/> |



 

請注意，如果封裝標示為壓縮 (位1設定) ，則 Windows Installer 只會安裝位於來源根目錄的檔案。 在此情況下，即使檔案 [資料表](file-table.md) 中標示為未壓縮的檔案，都必須位於要安裝的根目錄。 若要指定同時具有封包檔 (壓縮檔案的來源映射) 和與 [目錄資料表](directory-table.md)中的樹狀結構相符的未壓縮檔案，請在 [**字數統計] 摘要** 內容中保留位1取消設定 (值 =) 0，並在檔案 [資料表](file-table.md)的 [屬性] 資料行中為封包中的每個檔案設定 **msidbFileAttributesCompressed** (value = 16384) 。

在轉換中，[ **字數統計摘要** ] 屬性應該是 Null。

在 patch 封裝的摘要資訊資料流程中，[**字數統計**] 屬性會指出安裝修補程式所需的最小 Windows Installer 版本。



| 值 | 意義                                                                                                                                                                              |
|-------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 1     | 預設值，表示已使用 MSPATCH 來建立修補程式。                                                                                                        |
| 2     | 需要至少 Windows Installer 1.2，才能套用修補程式。 如果與1.2 之前的 Windows Installer 版本搭配使用，則字數為 "2" 的 patch 會立即失敗。 |
| 3     | 需要至少 Windows Installer 2.0，才能套用修補程式。 如果與2.0 之前的 Windows Installer 版本搭配使用，則字數為 "3" 的 patch 會立即失敗。 |
| 4     | 需要至少 Windows Installer 3.0，才能套用修補程式。 如果與3.0 之前的 Windows Installer 版本搭配使用，則字數為 "4" 的 patch 會失敗。             |
| 5     | 需要至少 Windows Installer 3.1，才能套用修補程式。                                                                                                               |



 

這是必要的摘要屬性。
