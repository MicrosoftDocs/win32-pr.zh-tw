# <a name="resume-property"></a>RESUME 屬性

繼續暫止的安裝時，會設定 **RESUME** 屬性。 這個屬性可決定要在 UI 中顯示的文字。 例如，當不在復原模式時，使用者可能會被詢問「您是否準備好安裝？」 在復原模式中，使用者可能會被詢問「您是否準備好完成安裝？」。

## <a name="remarks"></a>備註

為了與舊版相容，安裝程式也支援此屬性的名稱 **Resume** 。
