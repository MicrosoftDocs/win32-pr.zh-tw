# <a name="diskprompt-property"></a>DiskPrompt 屬性

**DiskPrompt** 屬性值會保存提示您輸入磁片的對話方塊所顯示的字串。 此字串應該包含產品的名稱，以及 CD-ROM 表面上所列印文字的預留位置。

## <a name="remarks"></a>備註

公告為產品屬性。
