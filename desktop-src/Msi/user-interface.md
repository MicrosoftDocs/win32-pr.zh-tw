# <a name="user-interface-windows-installer"></a>消費者介面 (Windows Installer) 

Windows Installer 為安裝應用程式或產品提供 (UI) 的完整使用者介面。 使用者介面會提供使用者可用來設定安裝的選項，並取得有關擱置安裝程式的使用者資訊。

[關於消費者介面](about-the-user-interface.md) 描述安裝程式使用者介面的功能。 [使用消費者介面](using-the-user-interface.md) 描述安裝程式內部 UI 的使用。

在 [安裝程式內部] 對話方塊和控制項樣式和選項的參考資訊，會顯示在 [消費者介面參考](user-interface-reference.md)中。
