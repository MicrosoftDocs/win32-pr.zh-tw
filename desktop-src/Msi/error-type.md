# <a name="errortype-property"></a>錯誤。類型屬性

唯讀 **類型** 屬性會傳回 msmErrorType 的其中一個值，表示此物件所代表的錯誤類型。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = Error.Type
```



## <a name="property-value"></a>屬性值

## <a name="c"></a>C++

請參閱 [**取得 \_ 型別函數**](/windows/win32/api/mergemod/nf-mergemod-imsmerror-get_type) 函式 (Error 物件) 。
