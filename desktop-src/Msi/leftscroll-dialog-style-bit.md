# <a name="leftscroll-dialog-style-bit"></a>LeftScroll 對話方塊樣式位

如果設定此樣式位，則捲軸位於對話方塊的左邊。

## <a name="value"></a>值



| Decimal | 十六進位 | 常數                            |
|---------|-------------|-------------------------------------|
| 512     | 0x00000200  | **msidbDialogAttributesLeftScroll** |
