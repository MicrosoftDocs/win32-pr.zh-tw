# <a name="applying-merge-modules"></a>套用合併模組

合併模組必須合併為使用合併工具的安裝套件。 最好的替代方法是取得自由散發的合併工具，或購買可從獨立軟體廠商取得的其中一個合併工具。 例如，您可以使用 Mergemod.dll 所提供的功能。

您可以使用不同的語言來開啟多語言合併模組，然後將其合併至目標，以產生不同的語言版本套件。 如需詳細資訊，請參閱

-   [使用 Automation 將合併模組合併至資料庫](using-automation-to-merge-a-merge-module-into-a-database.md)
-   [使用 API 將合併模組合併至資料庫](using-the-api-to-merge-a-merge-module-into-a-database.md)
-   [以特定語言開啟 Multiple-Language 合併模組](opening-a-multiple-language-merge-module-in-a-specific-language.md)
-   [將多個語言模組合併成相同的封裝多次](merging-a-multiple-language-module-into-the-same-package-multiple-times.md)
-   [將合併模組連接至多個功能](connecting-a-merge-module-to-multiple-features.md)
