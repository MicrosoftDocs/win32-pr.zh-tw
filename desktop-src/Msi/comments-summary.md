# <a name="comments-summary-property"></a>批註摘要屬性

[ **批註摘要** ] 屬性會傳達安裝套件、轉換或修補套件的一般用途。

安裝套件、轉換或修補套件的作者應該將 [ **批註摘要** ] 屬性的值設定為下列其中一個值：

<dl> 「此安裝程式資料庫包含安裝 <*產品*> 所需的邏輯與資料。」  
「此轉換包含安裝 <*產品*> 所需的邏輯與資料。」  
「此修補套裝程式含安裝 <*產品*> 所需的邏輯與資料。」  
</dl>

其中 <*product*> 是產品的名稱。
