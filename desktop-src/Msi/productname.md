# <a name="productname-property"></a>ProductName 屬性

**ProductName** 屬性包含要安裝之應用程式的名稱。 這僅供顯示之用。 公告為產品屬性。

此為必要屬性。

轉換可以變更此屬性。

## <a name="remarks"></a>備註

**ProductName** 屬性的長度不能超過63個字元。 DisplayName 的登錄機碼長度沒有任何限制。
