# <a name="suggested-advtuisequence"></a>建議的 AdvtUISequence

請勿編寫 AdvtUISequence 資料表。 安裝程式不會使用此資料表。 AdvtUISequence 資料表不應該存在於安裝資料庫中，也不應該保留空白。 在初始化期間，應該將要通告的應用程式名稱傳遞給安裝程式。
