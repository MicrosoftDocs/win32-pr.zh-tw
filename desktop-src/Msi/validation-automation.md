# <a name="validation-automation"></a>驗證自動化

您可以使用 Evalcom2.dll 來執行安裝封裝和合併模組的驗證作業。

本檔的下列各節包含在撰寫工具內使用 Windows Installer 驗證的相關資訊：

-   [使用 Evalcom2](using-evalcom2.md)
-   [EvalCom2 介面](evalcom2-interfaces.md)

建議使用 Windows Installer 來安裝 Evalcom2.dll。 適用于 Windows XP Service Pack 2 (SP2) 或更新版本的 Microsoft Windows 軟體開發套件 (SDK) 所提供的 Orca.msi 套件 Evalcom2.dll。 包含 COM 介面之元件的元件識別碼為 {53315473-B8D6-470A-9951-D9AE5C11FC91}。

如需取得 Windows SDK 的詳細資訊，請參閱 [Windows Installer 開發人員 Windows SDK 元件](platform-sdk-components-for-windows-installer-developers.md)。
