# <a name="dwm-messages"></a>DWM 訊息

本節包含桌面視窗管理員 (DWM) 訊息的相關資訊。

## <a name="in-this-section"></a>本節內容




| 主題 | 描述 | 
|-------|-------------|
| <a href="wm-dwmcolorizationcolorchanged.md"><strong>WM_DWMCOLORIZATIONCOLORCHANGED</strong></a><br /> | 通知所有最上層視窗的顏色標示色彩已變更。<br /> | 
| <a href="wm-dwmcompositionchanged.md"><strong>WM_DWMCOMPOSITIONCHANGED</strong></a><br /> | 通知所有最上層視窗的 DWM 組合已啟用或停用。 <br /><blockquote>[!Note]<br />從 Windows 8，一律會啟用 DWM 組合，因此不論影片模式變更為何，都不會傳送此訊息。</blockquote><br /> | 
| <a href="wm-dwmncrenderingchanged.md"><strong>WM_DWMNCRENDERINGCHANGED</strong></a><br /> | 當非工作區轉譯原則變更時傳送。<br /> | 
| <a href="wm-dwmsendiconiclivepreviewbitmap.md"><strong>WM_DWMSENDICONICLIVEPREVIEWBITMAP</strong></a><br /> | 指示視窗提供靜態點陣圖作為 <em>即時預覽</em> (也稱為該視窗的 <em>查看預覽</em>) 。<br /> | 
| <a href="wm-dwmsendiconicthumbnail.md"><strong>WM_DWMSENDICONICTHUMBNAIL</strong></a><br /> | 指示視窗提供靜態點陣圖，以用來做為該視窗的縮圖標記法。<br /> | 
| <a href="wm-dwmwindowmaximizedchange.md"><strong>WM_DWMWINDOWMAXIMIZEDCHANGE</strong></a><br /> | 當 DWM 組成的視窗最大化時傳送。<br /> |
