# <a name="dwm-structures"></a>DWM 結構

本節包含桌面視窗管理員 (DWM) 結構的相關資訊。

## <a name="in-this-section"></a>本節內容



| 主題                                                                     | 描述                                                                                                                                               |
|---------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DWM \_ BLURBEHIND**](/windows/desktop/api/Dwmapi/ns-dwmapi-dwm_blurbehind)<br/>                      | 指定 DWM 模糊後屬性。 由 [**DwmEnableBlurBehindWindow**](/windows/desktop/api/Dwmapi/nf-dwmapi-dwmenableblurbehindwindow) 函式使用。<br/>                     |
| [**DWM \_ 存在 \_ 參數**](/windows/desktop/api/Dwmapi/ns-dwmapi-dwm_present_parameters)<br/>     | 指定框架組合的 DWM 影片框架參數。 由 [**DwmSetPresentParameters**](/windows/desktop/api/Dwmapi/nf-dwmapi-dwmsetpresentparameters) 函式使用。<br/>   |
| [**DWM \_ 縮圖 \_ 屬性**](/windows/desktop/api/Dwmapi/ns-dwmapi-dwm_thumbnail_properties)<br/> | 指定 DWM 縮圖屬性。 由 [**DwmUpdateThumbnailProperties**](/windows/desktop/api/Dwmapi/nf-dwmapi-dwmupdatethumbnailproperties) 函式使用。<br/>                 |
| [**DWM \_ 計時 \_ 資訊**](/windows/desktop/api/Dwmapi/ns-dwmapi-dwm_timing_info)<br/>                   | 指定 DWM 組合計時資訊。 由 [**DwmGetCompositionTimingInfo**](/windows/desktop/api/Dwmapi/nf-dwmapi-dwmgetcompositiontiminginfo) 函式使用。<br/>         |
| [**MilMatrix3x2D**](/windows/desktop/api/Dwmapi/ns-dwmapi-milmatrix3x2d)<br/>                         | 指定描述轉換的3x2 矩陣。 <br/>                                                                                            |
| [**不帶正負號 \_ 比例**](/windows/desktop/api/Dwmapi/ns-dwmapi-unsigned_ratio)<br/>                      | 定義 DWM Api 所使用的資料類型。 它代表一般比例，而且即使是在單一 API 內，也會用於不同的用途和單位。<br/> |
