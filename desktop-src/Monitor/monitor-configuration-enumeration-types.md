# <a name="monitor-configuration-enumeration-types"></a>監視設定列舉類型

監視設定函數會使用下列列舉。



| 列舉類型                                                    | Description                                                                                      |
|---------------------------------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**MC \_ 色 \_ 溫度**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_color_temperature)              | 描述監視器的色溫。                                                         |
| [**MC \_ 顯示 \_ 技術 \_ 類型**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_display_technology_type) | 識別監視器顯示技術。                                                         |
| [**MC \_ 磁片磁碟機 \_ 類型**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_drive_type)                            | 指定是否要設定或取得監視器的紅色、綠色或藍色磁片磁碟機。                           |
| [**MC \_ 增益 \_ 類型**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_gain_type)                              | 指定是要取得或設定監視的紅色、綠色或藍色增益。                            |
| [**MC \_ 位置 \_ 類型**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_position_type)                      | 指定是否要取得或設定監視器顯示區域的垂直或水準位置。 |
| [**MC \_ 大小 \_ 類型**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_size_type)                              | 指定是否要取得或設定監視器顯示區域的寬度或高度。                 |
| [**MC \_ VCP 程式 \_ 代碼 \_ 類型**](/windows/desktop/api/LowLevelMonitorConfigurationAPI/ne-lowlevelmonitorconfigurationapi-mc_vcp_code_type)                     | 描述) 程式碼類型的虛擬主控台 (VCP。                                               |
