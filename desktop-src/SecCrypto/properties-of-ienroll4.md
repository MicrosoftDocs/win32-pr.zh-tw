# <a name="properties-of-ienroll4"></a>IEnroll4 的屬性

下列屬性是由 [**IEnroll4**](/windows/desktop/api/Xenroll/nn-xenroll-ienroll4) 介面所定義，或為 **IEnroll4** 介面繼承自 [**IEnroll**](/windows/desktop/api/Xenroll/nn-xenroll-ienroll) 和 [**IEnroll2**](/windows/desktop/api/Xenroll/nn-xenroll-ienroll2)的屬性。



| 屬性                                                                              | 描述                                                                                                                                                                                                                                                                                                             |
|---------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CAStoreFlags**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_castoreflags)                                         | 設定或抓取旗標，此旗標會在開啟時控制憑證存放區。                                                                                                                                                                                                                                         |
| [**CAStoreNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_castorenamewstr)                                   | 設定或抓取存放區的名稱，其中所有非「根」和非「我的」憑證都會保留下來。                                                                                                                                                                                                                        |
| [**CAStoreTypeWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_castoretypewstr)                                   | 設定或抓取要用於 [**CAStoreNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_castorenamewstr) 屬性所指定之存放區的存放區類型。                                                                                                                                                                                 |
| [**ClientId**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-get_clientid)                                                 | 設定或抓取用戶端識別碼要求屬性。                                                                                                                                                                                                                                                                      |
| [**ContainerNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_containernamewstr)                               | 設定或抓取要使用的 [*金鑰容器*](../secgloss/k-gly.md) 名稱。                                                                                                                                                                                     |
| [**DeleteRequestCert**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_deleterequestcert)                               | 設定或抓取布林指標，控制是否要刪除要求存放區中的虛擬憑證。                                                                                                                                                                                                        |
| [**EnableSMIMECapabilities**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll2-get_enablesmimecapabilities)                   | 設定或取得布林值，這個值會控制 PKCS \# 10 是否包含 [*安全/多用途網際網路郵件延伸*](../secgloss/s-gly.md) (S/MIME) 功能的已簽署屬性。 |
| [**EnableT61DNEncoding**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_enablet61dnencoding)                           | 設定或抓取布林值，這個值會控制要求中的辨別名稱是否編碼為 T61 字串，而不是 [*UNICODE*](../secgloss/u-gly.md) 字串。                                                                                                |
| [**GenKeyFlags**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_genkeyflags)                                           | 設定或抓取產生憑證要求時傳遞至 [**CryptGenKey**](/windows/desktop/api/Wincrypt/nf-wincrypt-cryptgenkey) 的值。                                                                                                                                                                                                    |
| [**HashAlgID**](/windows/win32/api/xenroll/nf-xenroll-ienroll2-get_hashalgid)                                               | 設定或抓取簽署 PKCS 10 憑證要求時使用的雜湊演算法 \# 。                                                                                                                                                                                                                                 |
| [**HashAlgorithmWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_hashalgorithmwstr)                               | 只設定或抓取用來簽署 PKCS 10 的簽章雜湊演算法 \# 。                                                                                                                                                                                                                                         |
| [**IncludeSubjectKeyID**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-get_includesubjectkeyid)                           | 指定是否將主體金鑰識別碼延伸新增至憑證要求。                                                                                                                                                                                                                                     |
| [**KeySpec**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_keyspec)                                                   | 設定或抓取產生的金鑰類型。                                                                                                                                                                                                                                                                            |
| [**LimitExchangeKeyToEncipherment**](/windows/win32/api/xenroll/nf-xenroll-ienroll2-get_limitexchangekeytoencipherment)     | 設定或抓取布林值，這個值會控制 AT \_ KEYEXCHANGE 要求是否包含 [*數位簽章*](../secgloss/d-gly.md) 和不可否認性的金鑰使用方式。                                                                                       |
| [**MyStoreFlags**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_mystoreflags)                                         | 設定或抓取用於 MY store 的登錄位置。                                                                                                                                                                                                                                                          |
| [**MyStoreNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_mystorenamewstr)                                   | 設定或抓取儲存連結私密金鑰之憑證的存放區名稱。                                                                                                                                                                                                                           |
| [**MyStoreTypeWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_mystoretypewstr)                                   | 設定或抓取 [**MyStoreNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_mystorenamewstr) 屬性所指定的存放區類型。                                                                                                                                                                                                      |
| [**ProviderFlags**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_providerflags)                                       | 設定或抓取 CSP 型別。                                                                                                                                                                                                                                                                                         |
| [**ProviderNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_providernamewstr)                                 | 設定或抓取要使用的 CSP 名稱。                                                                                                                                                                                                                                                                           |
| [**ProviderType**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_providertype)                                         | 設定或抓取提供者的類型。                                                                                                                                                                                                                                                                                 |
| [**PVKFileNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_pvkfilenamewstr)                                   | 設定或抓取將包含匯出索引鍵的檔案名。                                                                                                                                                                                                                                                 |
| [**RenewalCertificate**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_renewalcertificate)                             | 指定更新憑證的憑證內容。                                                                                                                                                                                                                                                          |
| [**RequestStoreFlags**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_requeststoreflags)                               | 設定或抓取要求存放區所使用的登錄位置。                                                                                                                                                                                                                                                     |
| [**RequestStoreNameWStr**](/windows/win32/api/xenroll/nf-xenroll-ienroll-get_requeststorenamewstr)                         | 設定或抓取包含虛擬憑證的存放區名稱。 此虛擬憑證和新增的私密金鑰會保留在要求存放區中，直到憑證授權單位單位處理要求並以 PKCS 7 回應為止 \# 。                                                                |
| [**RequestStoreTypeWStr**](/windows/win32/api/xenroll/nf-xenroll-ienroll-get_requeststoretypewstr)                         | 設定或抓取要用於 [**RequestStoreNameWStr**](/windows/win32/api/xenroll/nf-xenroll-ienroll-get_requeststorenamewstr) 屬性所指定之存放區的存放區類型。                                                                                                                                                                       |
| [**ReuseHardwareKeyIfUnableToGenNew**](/windows/win32/api/xenroll/nf-xenroll-ienroll2-get_reusehardwarekeyifunabletogennew) | 設定或取得布林值，這個值會決定當產生新的金鑰時遇到錯誤時，憑證註冊控制物件所採取的動作。                                                                                                                                                   |
| [**RootStoreFlags**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_rootstoreflags)                                     | 設定或抓取根存放區所使用的登錄位置。                                                                                                                                                                                                                                                        |
| [**RootStoreNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_rootstorenamewstr)                               | 設定或抓取根存放區的名稱，其中保留所有本質上受信任的自我簽署根憑證。                                                                                                                                                                                                    |
| [**RootStoreTypeWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_rootstoretypewstr)                               | 設定或抓取要用於 [**RootStoreNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_rootstorenamewstr) 屬性所指定之存放區的存放區類型。                                                                                                                                                                             |
| [**SPCFileNameWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_spcfilenamewstr)                                   | 設定或抓取檔案的名稱，以寫入產生的 base64 編碼 PKCS \# 7 （如憑證授權單位單位所傳回）。                                                                                                                                                                                     |
| [**ThumbPrintWStr**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll4-get_thumbprintwstr)                                     | 設定或抓取憑證資料的 [*雜湊*](../secgloss/h-gly.md) 。                                                                                                                                                                                                             |
| [**假如 useexistingkeyset**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_useexistingkeyset)                               | 設定或抓取布林值，指出是否應該使用現有的索引鍵。                                                                                                                                                                                                                              |
| [**WriteCertToCSP**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_writecerttocsp)                                     | 設定或抓取布林值，指出是否應將憑證寫入至 CSP。                                                                                                                                                                                                                    |
| [**WriteCertToUserDS**](/windows/desktop/api/Xenroll/nf-xenroll-ienroll-get_writecerttouserds)                               | 設定或抓取布林值，這個值會控制是否將憑證寫入使用者的 Active Directory 存放區。                                                                                                                                                                                                |
