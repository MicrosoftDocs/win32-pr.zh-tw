# <a name="error-checking-cryptography"></a>檢查 (密碼編譯) 時發生錯誤

針對 c + +，傳回值一律為 **HRESULT**，指出方法呼叫成功或失敗。 如果呼叫失敗，則傳回值會指出失敗的原因。
