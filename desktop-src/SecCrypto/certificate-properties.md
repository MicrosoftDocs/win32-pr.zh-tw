# <a name="certificate-properties"></a>憑證內容

憑證服務支援使用 [*x.509*](../secgloss/x-gly.md) 中所定義的憑證，ISO/IEC 9594-8 (也是 ISO/IEC) 。 以下是標準 x.509 憑證中包含的屬性。



| 欄位                                       | 描述                                                                                                                                                                                                     |
|---------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 版本                                     | 憑證格式的版本號碼。                                                                                                                                                                       |
| 序號                               | 憑證的序號。 此號碼是由簽發者所指派，在簽發的憑證的簽發者清單中是唯一的。                                                                          |
| 演算法識別碼和參數         | 簽章演算法和簽發者使用的任何參數。                                                                                                                                                      |
| Issuer                                      | 簽發憑證的 [*憑證授權單位*](../secgloss/c-gly.md) 單位名稱。                                               |
| 在 (日期之前)                            | 此日期之前的憑證無效。                                                                                                                                                                         |
|  (日期之後不)                             | 此日期之後的憑證無效。                                                                                                                                                                          |
| 主體名稱                                | 憑證簽發者或實體的名稱。 此欄位也可以包含憑證收件者的組織、組織單位、位置、州/省和國家/地區。 |
| 主體公開金鑰演算法和參數 | 用於主體 [*公開金鑰*](../secgloss/p-gly.md)的演算法和任何參數。                                                                       |
| 主體公開金鑰                          | 實際的公開金鑰 (位字串) 。                                                                                                                                                                           |
| 簽名                                   | 簽發者所提供的簽章。                                                                                                                                                                            |



 

根據憑證的 [*x.509*](../secgloss/x-gly.md) 版本而定，憑證可以包含下列專案。



| 選擇性欄位    | 描述                                                                                                                                                                                               |
|-------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 簽發者唯一識別碼  | 用來讓簽發者名稱明確地用於多個實體。 僅存在於 [*x.509*](../secgloss/x-gly.md) 2.0 或更新版本中。<br/> |
| 主體唯一識別碼 | 用來將主體名稱明確地用於一個以上的實體。 只存在於 x.509 2.0 或更新版本中。<br/>                                                                     |
| 延伸模組        | 用於指定任何所需的自訂屬性。 憑證中可以包含任意數目的延伸模組欄位。 只存在於3.0 版的 x.509 中。<br/>                                            |



 

> [!Note]  
> Microsoft 憑證服務會發出 x.509 第3版憑證。
