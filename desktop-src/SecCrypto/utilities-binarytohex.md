# <a name="utilitiesbinarytohex-method"></a>BinaryToHex 方法

\[**BinaryToHex** 方法可用於 [需求] 區段中指定的作業系統。\]

**BinaryToHex** 方法會將二進位壓縮的字串轉換成十六進位字串。

## <a name="syntax"></a>語法


```VB
Utilities.BinaryToHex( _
  ByVal BinaryString _
)
```



## <a name="parameters"></a>參數

<dl> <dt>

*BinaryString* \[在\]
</dt> <dd>

要轉換成十六進位字串的二進位封裝字串。

</dd> </dl>

## <a name="return-value"></a>傳回值

十六進位字串，相當於指定的二進位封裝字串。
