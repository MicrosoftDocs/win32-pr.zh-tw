# <a name="tools-to-sign-files-and-check-signatures"></a>用來簽署檔案和檢查簽章的工具

CryptoAPI 工具是下列工具，可用來以數位方式簽署檔案、確認檔案的簽章，以及建立類別目錄檔案。



| 工具                     | 備註                                                                                                                                                                                                                                                                            |
|--------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [MakeCat](makecat.md)   | 製作未簽署的類別目錄檔案。 此類別目錄檔案接著會用來比較類別目錄檔案中的 [*雜湊*](../secgloss/h-gly.md) 與個別檔案的雜湊，這有助於證明原始檔案不受篡改。 |
| [SetReg](setreg.md)     | 設定控制憑證驗證程式的登錄機碼。                                                                                                                                                                                                              |
| [SignTool](signtool.md) | 數位簽署檔案並確認檔案的數位簽章。                                                                                                                                                                                                                    |
