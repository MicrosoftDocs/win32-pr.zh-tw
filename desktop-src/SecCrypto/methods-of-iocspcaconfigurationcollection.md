# <a name="methods-of-iocspcaconfigurationcollection"></a>IOCSPCAConfigurationCollection 的方法

下列方法是由 [**IOCSPCAConfigurationCollection**](/windows/desktop/api/Certadm/nn-certadm-iocspcaconfigurationcollection) 介面所定義。 這裡不會顯示內容存取方法。 若要查看 **IOCSPCAConfigurationCollection** 的屬性，請參閱 [IOCSPCAConfigurationCollection 的屬性](properties-of-iocspcaconfigurationcollection.md)



| 方法                                                                                       | 描述                                                                                                                                                                                    |
|----------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CreateCAConfiguration**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-createcaconfiguration) | 建立新的 [*憑證授權單位*](../secgloss/c-gly.md) 單位 (CA) 設定，並將它新增至設定集。 |
| [**DeleteCAConfiguration**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-deletecaconfiguration) | 從設定集中移除已命名的 CA 設定。                                                                                                                                   |
