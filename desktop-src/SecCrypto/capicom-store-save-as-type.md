# <a name="capicom_store_save_as_type-enumeration"></a>CAPICOM \_ 存放區儲存 \_ \_ 為 \_ 類型列舉

**CAPICOM 存放區的「儲存 \_ \_ \_ 為 \_ 類型**」列舉類型表示 [*憑證存放區*](../secgloss/c-gly.md)的編碼方式。

## <a name="members"></a>成員



| member                                   | 描述                                         | 值 |
|------------------------------------------|-----------------------------------------------------|-------|
| **\_將 CAPICOM 儲存區儲存 \_ 為已 \_ \_ 序列化** | 存放區是以序列化格式儲存。<br/> | 0     |
| **\_將 CAPICOM 存放區 \_ 另存 \_ 為 \_ PKCS7**      | 存放區是以 PKCS \# 7 格式儲存。<br/>   | 1     |
