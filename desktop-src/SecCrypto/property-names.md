# <a name="property-names"></a>屬性名稱

[*憑證要求*](../secgloss/c-gly.md)屬性名稱可以是憑證服務所理解之其中一個通用屬性的「易記」名稱，也可以是 (OID) 的 [*物件識別碼*](../secgloss/o-gly.md)。 如需要求屬性名稱的詳細資訊，請參閱 [名稱屬性](name-properties.md)。
