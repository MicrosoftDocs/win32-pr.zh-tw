# <a name="utilitiesbytearraytobinarystring-method"></a>ByteArrayToBinaryString 方法

\[**ByteArrayToBinaryString** 方法可用於 [需求] 區段中指定的作業系統。\]

**ByteArrayToBinaryString** 方法會將位元組陣列轉換為二進位壓縮的字串。

## <a name="syntax"></a>語法


```VB
Utilities.ByteArrayToBinaryString( _
  ByVal varByteArray _
)
```



## <a name="parameters"></a>參數

<dl> <dt>

*varByteArray* \[在\]
</dt> <dd>

要轉換為二進位壓縮字串的位元組陣列。

</dd> </dl>

## <a name="return-value"></a>傳回值

二進位壓縮的字串，其中包含轉換的結果。
