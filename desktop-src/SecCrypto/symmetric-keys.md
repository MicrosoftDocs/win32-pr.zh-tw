# <a name="symmetric-keys"></a>對稱金鑰

[*對稱金鑰*](../secgloss/s-gly.md)是與 [*對稱式加密*](../secgloss/s-gly.md)配置的作業搭配使用的單一金鑰。 例如，這類金鑰可用於對稱式加密演算法，以進行加密和解密。

最知名的對稱演算法比最知名的 [*公開金鑰演算法*](../secgloss/p-gly.md)更快。 因此，在處理大量資料時，建議您這樣做。 某些較常見的對稱演算法為 [*RC4*](../secgloss/r-gly.md)、三重 DES ([*資料加密標準*](../secgloss/d-gly.md)) ，以及 [*進階加密標準*](../secgloss/a-gly.md) (AES) 。

因為涉及對稱金鑰的許多活動都與保存密碼相關，所以請務必盡可能將擁有特定對稱金鑰的人數保持在最小。 我們建議只有一或兩個人擁有任何一個對稱金鑰。
