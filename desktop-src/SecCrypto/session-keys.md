# <a name="session-keys"></a>工作階段金鑰

[*工作階段金鑰*](../secgloss/s-gly.md) 是為了在單一通訊會話中使用而產生的金鑰。 工作階段金鑰會經常變更，並在不再需要時捨棄。 通常會使用 [*對稱金鑰*](../secgloss/s-gly.md) 做為工作階段金鑰。

工作階段金鑰是暫時性的。 應用程式可以儲存這些金鑰以供稍後使用，或傳輸給其他使用者。 如需詳細資訊，請參閱[密碼編譯金鑰儲存體和 Exchange](cryptographic-key-storage-and-exchange.md)。
