# <a name="distinguished-name-fields"></a>辨別名稱欄位

[*憑證要求*](../secgloss/c-gly.md)包含的資訊應可唯一識別提出要求的人員。

\#憑證服務接受的 PKCS 10 格式憑證要求包含識別欄位，稱為辨別名稱 (DN) 欄位。 這些欄位將包含使用者在使用金鑰管理員、 [憑證註冊控制](certificate-enrollment-control.md)或其他方法建立憑證要求時所輸入的資訊。

以下是在憑證要求中完成 DN 欄位的一些指導方針。



| 欄位               | 描述                                                                                                                                                                                                                                                                                                                                                      |
|---------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 一般名稱         | 使用者憑證：您應輸入人員的完整名稱。 電腦憑證：您應輸入 *伺服器將在其上執行的 DNS 查閱中所使用的完整主機名稱 * **/** _路徑_ (例如， *hostname ***。**_範例_*_.com_*) 。<br/>                                                                                                  |
| 組織        | 您為 [組織] 欄位指定的名稱應該是您組織中已向適當城市、州或國家/地區授權單位註冊的法定名稱。 組織的法定名稱必須用在 [組織] 欄位中。                                                                                                      |
| 組織單位 | [組織單位] 欄位可以用來區別組織內的不同部門，例如「網際網路安全性單位」或「人力資源」。 您也可以使用此欄位來指定 DBA (執行商務為 .。。) 值。                                                                                           |
| 位置            | [位置] 欄位代表組織所在的城市。 如果組織只有本機的位置，藉由在麻塞諸塞州的州康橋城市的城市職員註冊商務授權，[位置] 欄位必須包含康橋。                                                                |
| 省/市   | [省份] 欄位會指定組織實際所在的位置。 如果您的組織已併入特拉華，但是有 DBA (將企業做為 .。。在加州內 ) ，請使用加州。 [省份] 欄位不應為縮寫欄位。 例如，"CA" 不是有效的狀態名稱。 「加州」是正確的狀態名稱。 |
| 國家/區域      | X.500 [*命名配置*](../secgloss/x-gly.md) 標準需要2個字元的國家/地區代碼。 美國的國家/地區代碼是 US;加拿大的國家/地區代碼為 CA。                                                                                                                          |
