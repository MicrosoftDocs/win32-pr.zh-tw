# <a name="using-cert2spc"></a>使用 Cert2spc.exe

下列命令會將 [*x.509*](../secgloss/x-gly.md)憑證（ *mycert.cer*）包裝到測試 PKCS \# 7 [*軟體 Publisher 憑證*](../secgloss/s-gly.md) (SPC) （稱為 *mycert.cer*）。 所建立的 SPC 僅供測試用途使用。 用來實際簽署要散發至公用之程式碼的 SPC，必須從 GTE、VeriSign、Inc. 或其他受信任的 [*憑證授權單位*](../secgloss/c-gly.md) 單位（ (CA) 取得。

**Cert2spc.exe** *mycert.cer * * * .cer* *  *mycert.cer * * * .spc**
