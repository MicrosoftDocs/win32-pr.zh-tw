# <a name="signed-data-additions"></a>新增簽署的資料

已簽署的資料功能已進行下列變更：

-   Nondata [*內部內容*](../secgloss/i-gly.md) 會封裝在八位字串中。
-   針對不同的訊息版本進行布建。
-   簽署者可以依簽發者和序號來識別，如 PKCS \# 7 1.5 或主體金鑰識別碼。
-   允許多個簽署人。
-   RSA、DSA 和 ECDSA 簽章都是允許的。
-   橢圓曲線密碼編譯 (ECC) 演算法和 AES 需要 Windows Vista 或更新版本。
