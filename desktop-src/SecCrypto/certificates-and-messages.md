# <a name="certificates-and-messages"></a>憑證和訊息

基底密碼編譯函式會使用 [*憑證*](../secgloss/c-gly.md)，而 [*低層級的訊息函式*](../secgloss/l-gly.md) 和 [*簡化的訊息*](../secgloss/s-gly.md) 函式也會使用這些功能。 若要傳送或接收併入憑證的訊息，請使用低層級的訊息功能。

如需使用憑證搭配低層級訊息功能的範例，請參閱 [範例 c 程式：編碼和解碼副署訊息](example-c-program-encoding-and-decoding-a-countersigned-message.md)， [範例 c 程式：編碼和解碼雜湊訊息](example-c-program-encoding-and-decoding-a-hashed-message.md)。

如需使用憑證搭配簡化訊息函式的範例，請參閱 [範例 C 程式：使用 CryptEncryptMessage 和 CryptDecryptMessage](example-c-program-using-cryptencryptmessage-and-cryptdecryptmessage.md)。

如需低層級訊息函式的詳細資訊，請參閱 [低層級的訊息](low-level-messages.md)。

如需簡化訊息函數的詳細資訊，請參閱 [簡化的訊息](simplified-messages.md)。
