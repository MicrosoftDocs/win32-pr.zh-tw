# <a name="active-directory"></a>Active Directory

憑證服務支援使用 Active Directory。 在企業 [*憑證授權單位*](../secgloss/c-gly.md) 單位 (CA) 的情況下，憑證服務需要使用 Active Directory。 企業 CA 會將使用者憑證和 [*憑證撤銷清單*](../secgloss/c-gly.md) 發佈 (CRL) 至 Active Directory。 您可以管理獨立 CA 來執行相同的動作，提供的 Active Directory 可供獨立 CA 使用，並使用自訂原則模組。
