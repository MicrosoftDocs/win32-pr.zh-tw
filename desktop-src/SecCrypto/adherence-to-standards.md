# <a name="adherence-to-standards"></a>遵循標準

憑證服務接受標準 PKCS \# 10 要求和問題 x.509 3.0 憑證。 憑證服務適用于非 Microsoft 用戶端和瀏覽器，以及非 Microsoft 網頁伺服器。
