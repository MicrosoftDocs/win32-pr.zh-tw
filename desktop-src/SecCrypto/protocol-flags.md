# <a name="protocol-flags"></a>通訊協定旗標

以下是密碼編譯通訊協定的預先定義常數。 這些值定義于 Wincrypt 中。



| 常數/值                                                                                                                                                                                                                            | Description                                                           |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------|
| <span id="CRYPT_FLAG_IPSEC"></span><span id="crypt_flag_ipsec"></span><dl> <dt>**CRYPT \_旗標 \_ IPSEC**</dt> <dt>0x0010</dt> </dl>       | 網際網路通訊協定安全性 (IPsec) 通訊協定。<br/>               |
| <span id="CRYPT_FLAG_PCT1"></span><span id="crypt_flag_pct1"></span><dl> <dt>**CRYPT \_旗標 \_ PCT1**</dt> <dt>0x0001</dt> </dl>          | 私人通訊傳輸 (PCT) 第1版通訊協定。<br/> |
| <span id="CRYPT_FLAG_SIGNING"></span><span id="crypt_flag_signing"></span><dl> <dt>**CRYPT \_旗標 \_ 簽署**</dt> <dt>0x0020</dt> </dl> | 簽署通訊協定。<br/>                                          |
| <span id="CRYPT_FLAG_SSL2"></span><span id="crypt_flag_ssl2"></span><dl> <dt>**CRYPT \_旗標 \_ SSL2**</dt> <dt>0x0002</dt> </dl>          | 安全通訊端層 (SSL) 第2版通訊協定。<br/>             |
| <span id="CRYPT_FLAG_SSL3"></span><span id="crypt_flag_ssl3"></span><dl> <dt>**CRYPT \_旗標 \_ SSL3**</dt> <dt>0x0004</dt> </dl>          | SSL 第3版通訊協定。<br/>                                    |
| <span id="CRYPT_FLAG_TLS1"></span><span id="crypt_flag_tls1"></span><dl> <dt>**CRYPT \_旗標 \_ TLS1**</dt> <dt>0x0008</dt> </dl>          | 傳輸層安全性 (TLS) 第1版通訊協定。<br/>         |
