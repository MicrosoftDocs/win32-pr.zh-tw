# <a name="capicom-versions"></a>CAPICOM 版本

\[CAPICOM 2.1.0.2 可用於下列作業系統： Windows Server 2008、Windows Vista 和 Windows XP。 相反地，請使用 .NET Framework 來執行安全性功能。 如需詳細資訊，請參閱 [使用 CAPICOM 的替代方案](alternatives-to-using-capicom.md)。\]

CAPICOM 1.0 提供下列功能：

-   以 PKCS \# 7 為基礎的數位簽章產生和驗證
-   使用憑證加密和解密資料
-   使用密碼加密和解密資料

CAPICOM 2.0 包含 CAPICOM 1.0 的功能，並新增下列功能：

-   Authenticode 數位簽章的產生與驗證
-   支援以檔案為基礎的認證
-   強大的憑證存放區搜尋
-   產生任意資料的雜湊
-   支援進階加密標準 (AES) 演算法
-   支援 advanced certificate 屬性，例如原則和範本

若要判斷各種平臺上支援的方法和屬性，請參閱 CAPICOM 物件的參考主題。 您可以從 Platform SDK 可轉散發套件 [： capicom](https://www.microsoft.com/download/details.aspx?id=25281)取得 capicom 2.0 可轉散發套件。
