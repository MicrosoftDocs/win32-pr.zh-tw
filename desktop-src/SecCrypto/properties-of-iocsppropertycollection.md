# <a name="properties-of-iocsppropertycollection"></a>IOCSPPropertyCollection 的屬性

下列屬性是由 [**IOCSPPropertyCollection**](/windows/desktop/api/Certadm/nn-certadm-iocsppropertycollection) 介面所定義。



| 屬性                                                        | 描述                                              |
|-----------------------------------------------------------------|----------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-get__newenum)    | 取得屬性集的列舉值。                   |
| [**計數**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-get_count)           | 取得屬性集內的屬性數目。         |
| [**項目**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-get_item)             | 取得屬性集中索引所識別的屬性。 |
| [**ItemByName**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-get_itembyname) | 取得屬性集內依名稱識別的屬性。  |
