# <a name="cryptxml"></a>CryptXML

CryptXML 提供一組低層級的原生 Api，可根據 XML 簽章語法和處理 (第二版) W3C 建議來執行 XML 數位簽章建立和驗證。 開發人員必須熟悉此標準，才能有效地使用 CryptXML 數位簽章 Api。

CryptXML 包含下列主題：

-   [XML 數位簽章總覽](xml-digital-signature-overview.md)
-   [XML 數位簽章 API 功能](xml-digital-signature-api-functionality.md)
-   [XML 數位簽章密碼編譯演算法](xml-digital-signature-cryptographic-algorithms.md)
-   [XML 數位簽章密碼編譯延伸模組](xml-digital-signature-cryptographic-extensions.md)
-   [XML 數位簽章安全性考慮](xml-digital-signature-security-considerations.md)
