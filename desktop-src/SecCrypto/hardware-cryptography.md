# <a name="hardware-cryptography"></a>硬體密碼編譯

憑證服務支援硬體相關的密碼編譯，例如發行儲存在 [*智慧卡*](../secgloss/s-gly.md)上的憑證。 企業 [*憑證授權單位*](../secgloss/c-gly.md) 單位 (CA) 支援在作業系統上發佈用來進行網域登入的智慧卡憑證。 此外，企業 CA 和獨立 CA 都支援發行其他類型的智慧卡憑證。

如需詳細資訊，請參閱產品檔。 如需智慧卡的開發人員資訊，請參閱 [智慧卡驗證](../secauthn/smart-card-authentication.md)。 如需允許系統管理員代表使用者要求智慧卡憑證之控制項的相關資訊，請參閱 [智慧卡註冊控制](/previous-versions/windows/desktop/secsmart/smart-card-enrollment-control-usage-scenario)。
