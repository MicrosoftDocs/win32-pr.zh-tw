# <a name="microsoft-dss-cryptographic-provider"></a>Microsoft DSS 密碼編譯提供者

Microsoft DSS 密碼編譯提供者支援使用 [*安全雜湊演算法*](../secgloss/s-gly.md) (SHA) 和 [*數位簽章標準*](../secgloss/d-gly.md) (DSS) 演算法進行雜湊、資料簽章和簽章驗證。 您可以將它匯出到美國和加拿大以外的地區。



|                   | 值               |
|-------------------|---------------------|
| **提供者類型** | >PROV \_ DSS           |
| **提供者名稱** | MS \_ DEF \_ DSS \_ >PROV  |
