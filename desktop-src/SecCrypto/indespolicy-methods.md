# <a name="indespolicy-methods"></a>INDESPolicy 方法

[**INDESPolicy**](/windows/desktop/api/certpol/nn-certpol-indespolicy)介面支援下列方法。

## <a name="in-this-section"></a>本節內容



| 主題                                                                        | 描述                                                                                       |
|------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------|
| [**GenerateChallenge 方法**](/windows/desktop/api/certpol/nf-certpol-indespolicy-generatechallenge)<br/> | 執行原則決策是否要對 SCEP 用戶端發出挑戰密碼。<br/> |
| [**Initialize 方法**](/windows/desktop/api/certpol/nf-certpol-indespolicy-initialize)<br/>               | 初始化 NDES 原則模組。<br/>                                                    |
| [**Notify 方法**](/windows/desktop/api/certpol/nf-certpol-indespolicy-notify)<br/>                       | 通知外掛程式有關 SCEP 憑證要求的交易狀態。<br/>        |
| [**Uninitialize 方法**](/windows/desktop/api/certpol/nf-certpol-indespolicy-uninitialize)<br/>           | 取消初始化 NDES 原則模組。<br/>                                                  |
| [**Verifyrequest 已方法**](/windows/desktop/api/certpol/nf-certpol-indespolicy-verifyrequest)<br/>         | 確認 NDES 憑證要求提交給 CA。<br/>                        |
