# <a name="certificate-and-trust-return-values"></a>憑證和信任傳回值

下表列出憑證和憑證信任傳回值。 這些值包含在標頭檔 Winerror.h 中。



| 名稱                            | 描述                                                                                                                    | 值      |
|---------------------------------|--------------------------------------------------------------------------------------------------------------------------------|------------|
| CERT \_ E \_ 危急               | 憑證包含標示為「重大」的未知延伸模組。                                                         | 0x800B0105 |
| CERT \_ E \_ 不正確 \_ 名稱          | 憑證的名稱無效。 名稱未包含在允許的清單中，或是已明確排除。 | 0x800B0114 |
| CERT \_ E \_ 不正確 \_ 原則        | 憑證有不正確原則。                                                                                | 0x800B0113 |
| CERT \_ E \_ ISSUERCHAINING         | 實際上，指定憑證的父系並未簽發該子憑證。                                                  | 0x800B0107 |
| CERT \_ E \_ 格式不正確              | 缺少憑證或重要欄位的空白值，例如主體或簽發者名稱。                       | 0x800B0108 |
| CERT \_ E \_ PATHLENCONST           | 已違反憑證鏈結中的路徑長度條件約束。                                                         | 0x800B0104 |
| CERT \_ E \_ UNTRUSTEDCA            | 憑證鏈已正確處理，但其中一個 CA 憑證不受原則提供者的信任。               | 0x800B0112 |
| CRYPT \_ E \_ NO \_ 撤銷 \_ 檢查 | 撤銷功能無法檢查憑證的撤銷。                                                    | 0x80092012 |
| 信任 \_ 電子 \_ 不良 \_ 摘要           | 物件的數位簽章未驗證。                                                                            | 0x80096010 |
| 信任 \_ E \_ 基本 \_ 限制    | 尚未發現憑證的基本條件約束擴充。                                                         | 0x80096019 |
| 信任電子憑證簽章 \_ \_ \_       | 無法驗證憑證的簽章。                                                                           | 0x80096004 |
| 信任 \_ 電子 \_ 計數器 \_ 簽署者       | 其中一個計數器簽章無效。                                                                                   | 0x80096003 |
| 信任 \_ E \_ 明確 \_ 信任    | 憑證已明確標示為未受使用者信任。                                                                | 0x800B0111 |
| 信任 \_ 電子的 \_ 財務 \_ 準則   | 憑證不符合或包含 Authenticode 財務延伸模組。                                                | 0x8009601E |
| 信任 \_ 電子 \_ 沒有 \_ 簽署者憑證 \_      | 訊息簽署者的憑證無效或找不到。                                                       | 0x80096002 |
| 信任 \_ 電子 \_ 系統 \_ 錯誤         | 驗證信任時發生系統層級的錯誤。                                                                           | 0x80096001 |
| 信任 \_ E \_ 時間 \_ 戳           | 時間戳記簽章或憑證無法驗證或不正確。                                                 | 0x80096005 |
