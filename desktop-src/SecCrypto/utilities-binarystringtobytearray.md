# <a name="utilitiesbinarystringtobytearray-method"></a>BinaryStringToByteArray 方法

\[**BinaryStringToByteArray** 方法可用於 [需求] 區段中指定的作業系統。\]

**BinaryStringToByteArray** 方法會將二進位壓縮的字串轉換成位元組陣列。

## <a name="syntax"></a>語法


```VB
Utilities.BinaryStringToByteArray( _
  ByVal BinaryString _
)
```



## <a name="parameters"></a>參數

<dl> <dt>

*BinaryString* \[在\]
</dt> <dd>

要轉換成位元組陣列的二進位壓縮字串。

</dd> </dl>

## <a name="return-value"></a>傳回值

包含轉換結果的陣列。
