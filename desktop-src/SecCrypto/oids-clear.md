# <a name="oidsclear-method"></a>Oid。 Clear 方法

\[**Clear** 方法可用於 [需求] 區段中指定的作業系統。 請改用 [**system.string 命名空間中的**](/dotnet/api/system.security.cryptography?view=dotnet-plat-ext-3.1&preserve-view=true) [**OidCollection 類別**](/dotnet/api/system.security.cryptography.oidcollection?view=netcore-3.1)。\]

**Clear** 方法會清除集合中的所有 [**OID**](oid.md)物件。

## <a name="syntax"></a>語法


```VB
OIDs.Clear()
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
