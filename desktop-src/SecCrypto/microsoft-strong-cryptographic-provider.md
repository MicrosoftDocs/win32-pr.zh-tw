# <a name="microsoft-strong-cryptographic-provider"></a>Microsoft 強式密碼編譯提供者

Microsoft 強式密碼編譯提供者可作為預設的 RSA 完整 [*密碼編譯服務提供者*](../secgloss/c-gly.md) ， (CSP) 。 它支援 Microsoft 增強型密碼編譯提供者的所有演算法，以及所有相同的 [*金鑰長度*](../secgloss/k-gly.md)。 如需預設金鑰長度和演算法的詳細資訊，請參閱 Microsoft 基礎密碼編譯提供者。 如需更長的金鑰長度和可用演算法的詳細資訊，請參閱使用 [Microsoft 增強型密碼編譯提供者](microsoft-enhanced-cryptographic-provider.md)的表格。



|                |                      |
|----------------|----------------------|
| 提供者類型： | **>PROV \_ RSA \_ FULL**  |
| 提供者名稱： | **MS \_ 強式 \_ >PROV** |
