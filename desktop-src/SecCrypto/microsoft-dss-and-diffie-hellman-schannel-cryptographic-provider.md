# <a name="microsoft-dss-and-diffie-hellmanschannel-cryptographic-provider"></a>Microsoft DSS 和 Diffie-hellman/Schannel 密碼編譯提供者

Microsoft DSS 和 [*diffie-hellman*](../secgloss/d-gly.md) / [*Schannel*](../secgloss/s-gly.md)密碼編譯提供者支援雜湊、資料以 DSS 簽署、產生 Diffie-Hellman (D-h) 金鑰、交換 d-h 金鑰，以及匯出 d-h 金鑰。 此 CSP 支援 SSL3 和 TLS1 通訊協定的金鑰衍生。 您可以將它匯出至其他國家/地區。



|                   | 值                       |
|-------------------|-----------------------------|
| **提供者類型** | >PROV \_ DH \_ SCHANNEL          |
| **提供者名稱** | MS \_ DEF \_ DH \_ SCHANNEL \_ >PROV |
