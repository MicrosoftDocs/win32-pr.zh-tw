# <a name="cryptography-cryptoapi-and-capicom"></a>加密、CryptoAPI 和 CAPICOM

CryptoAPI 和 CAPICOM 提供的服務可讓應用程式開發人員將資料加密/[*解密、使用數位憑證進行*](../secgloss/c-gly.md)驗證，以及編碼至 [*抽象語法標記法 (一)*](../secgloss/a-gly.md) (asn.1) 至其 Windows 型應用程式。 應用程式開發人員可以在 CryptoAPI 中使用這些函式，而不需要瞭解基礎執行的詳細資料，其方式與使用圖形程式庫的方式大致相同，不需要知道特定圖形硬體設定的任何內容。 CryptoAPI 適用于一些執行實際密碼編譯功能的 [密碼編譯服務提供者](microsoft-cryptographic-service-providers.md) 。
