# <a name="cryptography-constants"></a>密碼編譯常數

加密函式會使用下列常數，並在 Wincrypt 中定義。 密碼編譯常數會根據使用方式進行分類，如下所示。



| 常數                                                                                                           | 描述                                                                                                                                                                                                 |
|--------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [CryptEncodeObject 和 CryptDecodeObject 的常數](constants-for-cryptencodeobject-and-cryptdecodeobject.md) | 搭配 [*抽象語法標記法 (一)*](../secgloss/a-gly.md) 使用的預先定義常數 (asn.1) 編碼和解碼作業。 |
| [Netscape 延伸模組的常數](constants-for-netscape-extensions.md)                                         | 用於 Netscape 憑證延伸的預先定義常數。 本節提供有關使用這些延伸模組搭配 asn.1 編碼和解碼作業的詳細資料。                                      |
| [密碼編譯提供者名稱](cryptographic-provider-names.md)                                                   | 預先定義的字串常數，用於一般 [*密碼編譯服務提供者*](../secgloss/c-gly.md) 的名稱 (csp) 。   |
| [**MAXUIDLEN**](maxuidlen.md)                                                                                     | 數值常數，指定某些 Microsoft 密碼編譯提供者在取得使用者識別碼時將使用的最大字元數。                                                  |
| [**私密金鑰快取常數**](private-key-caching-constants.md)                                             | 常數，代表由 CSP 控制 [*私密金鑰*](../secgloss/p-gly.md) 快取的登錄專案。                                                |
| [通訊協定旗標](protocol-flags.md)                                                                               | 密碼編譯通訊協定的預先定義常數。                                                                                                                                                            |
