# <a name="additional-documentation-on-cryptography"></a>有關密碼編譯的其他檔

有關 [*密碼*](../secgloss/c-gly.md) 編譯和驗證的其他檔，以及相關聯的安全性問題，包括下列發行集：

-   Adams、Carlisle 和 Steve 羅依。 *瞭解 PKI：概念、標準和部署考慮*。 2d ed。 波士頓： Addison-Wesley Professional，2002。
-   Housley、Russ 和 Tim Polk。 *規劃 PKI：部署公開金鑰基礎結構的最佳作法指南*。 紐約： John Wiley & 兒子，Inc.，2001。
-   Komar、Brian。 *Microsoft Windows Server 2003 PKI 和憑證安全性*。 華盛頓州雷德蒙德： Microsoft 按下，2004。
-   Menezes、Alfred J.、Paul Van Oorschot 及 Scott A. Vanstone。 *應用程式加密手冊*。 在離散數學和其應用程式上的 CRC 按系列。 Boca Raton： CRC 按下，1996。
-   Raina、Kapil。 *適用于 Enterprise 的 PKI 安全性解決方案：解決 HIPAA、電子論文法案和其他合規性問題*。 Indianapolis： Wiley 發行，Inc.，2003。
-   Schneier、Bruce。 套用 *的密碼編譯： C 中的通訊協定、演算法和原始程式碼*。紐約： John Wiley & 兒子，1996。
-   Stinson，d. R。 *密碼編譯：理論與實務* (CRC 請在離散數學和其應用程式) 上按系列。 Boca Raton： CRC 按下，1995。

> [!Note]  
> 這些資源可能無法在某些語言及國家/地區中使用。
