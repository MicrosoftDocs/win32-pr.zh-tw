# <a name="hcryptprov_legacy"></a>HCRYPTPROV \_ 舊版

**HCRYPTPROV \_ 舊版** 資料類型是用來取代不再使用 **HCRYPTPROV** 資料類型的 [**HCRYPTPROV**](hcryptprov.md)資料類型。 型別為 **HCRYPTPROV \_ LEGACY** 的函式參數和資料結構應該設定為 **Null**。


```C++
typedef ULONG_PTR HCRYPTPROV_LEGACY;
```
