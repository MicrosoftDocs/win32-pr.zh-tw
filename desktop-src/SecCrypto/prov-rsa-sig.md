# <a name="prov_rsa_sig"></a>>PROV \_ RSA \_ SIG

>PROV \_ rsa \_ SIG 提供者類型是 >prov \_ RSA FULL 的子集 \_ 。 它僅支援 [*雜湊*](../secgloss/h-gly.md) 和 [*數位簽章*](../secgloss/d-gly.md)所需的函數和演算法。

## <a name="algorithms-supported"></a>支援的演算法

如需這些演算法的說明，請參閱詞彙。



| 目的      | 支援的演算法                                                                                                              |
|--------------|-----------------------------------------------------------------------------------------------------------------------------------|
| 金鑰交換 | 無                                                                                                                              |
| 簽名    | [*RSA*](../secgloss/r-gly.md)                                                                       |
| 加密   | 無                                                                                                                              |
| 雜湊      | [*MD5*](../secgloss/m-gly.md)[*SHA*](../secgloss/s-gly.md)<br/> |
