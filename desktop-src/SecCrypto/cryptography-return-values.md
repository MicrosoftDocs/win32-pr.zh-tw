# <a name="cryptography-return-values"></a>密碼編譯傳回值

加密函式會使用下列傳回值，並在 Wincrypt 中定義。 密碼編譯傳回值會根據使用方式進行分類，如下所示。



| 傳回值                                                                       | 描述                                                                                                |
|------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [ASN. 1 編碼/解碼傳回值](asn-1-encoding-decoding-return-values.md) | 可進行 asn.1 編碼或解碼的函式所傳回的預先定義常數。                    |
| [憑證和信任傳回值](certificate-and-trust-return-values.md)     | 當決定憑證的有效性和信任狀態時，所傳回的預先定義常數。 |



 

如需 CAPICOM 傳回值的清單，請 [**參閱 \_ capicom \_ 錯誤碼**](capicom-error-code.md)。
