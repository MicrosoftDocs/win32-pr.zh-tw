# <a name="methods-of-iocsppropertycollection"></a>IOCSPPropertyCollection 的方法

下列方法是由 [**IOCSPPropertyCollection**](/windows/desktop/api/Certadm/nn-certadm-iocsppropertycollection) 介面所定義。 這裡不會顯示內容存取方法。 若要查看 **IOCSPPropertyCollection** 的屬性，請參閱 [IOCSPPropertyCollection 的屬性](properties-of-iocsppropertycollection.md)



| 方法                                                                                      | 描述                                                                               |
|---------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**CreateProperty**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-createproperty)                     | 建立新的屬性，並將它加入至屬性集。                                    |
| [**DeleteProperty**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-deleteproperty)                     | 從屬性集移除已命名的屬性。                                             |
| [**GetAllProperties**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-getallproperties)                 | 取得屬性集中的所有屬性。                                                    |
| [**InitializeFromProperties**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-initializefromproperties) | 從現有伺服器設定所包含的屬性，建立屬性集。 |
