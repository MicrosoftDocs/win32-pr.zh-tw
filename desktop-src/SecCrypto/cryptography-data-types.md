# <a name="cryptography-data-types"></a>加密資料類型

加密函式、介面和物件會使用下列資料類型。



| 資料類型                                                                      | 描述                                                                                                                                                                                      |
|--------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ALG \_ 識別碼**](alg-id.md)                                                      | 指定演算法識別碼。                                                                                                                                                                 |
| [**HCERT \_ SERVER \_ OCSP \_ 回應**](hcert-server-ocsp-response.md)            | 表示與伺服器憑證鏈相關聯之 OCSP 回應的控制碼。                                                                                                              |
| [**HCRYPTHASH**](hcrypthash.md)                                               | 指定 [*雜湊物件*](../secgloss/h-gly.md)的控制碼。                                                                                      |
| [**HCRYPTKEY**](hcryptkey.md)                                                 | 指定密碼編譯金鑰的控制碼。                                                                                                                                                         |
| [**HCRYPTOIDFUNCADDR**](hcryptoidfuncaddr.md)                                 | 表示函式的控制碼，這個控制碼可使用 (OID) 的 [*物件識別碼*](../secgloss/o-gly.md) 來安裝。                 |
| [**HCRYPTOIDFUNCSET**](hcryptoidfuncset.md)                                   | 表示可以使用 OID 安裝的一組函式的控制碼。                                                                                                                 |
| [**HCRYPTPROV \_ 舊版**](hcryptprov-legacy.md)                                | 用來取代不再使用 **HCRYPTPROV** 資料類型的 [**HCRYPTPROV**](hcryptprov.md)資料類型。                                                                             |
| [**HCRYPTPROV \_ 或 \_ NCRYPT \_ 金鑰 \_ 控制碼**](hcryptprov-or-ncrypt-key-handle.md) | 指定 CryptoAPI [*密碼編譯服務提供者*](../secgloss/c-gly.md) (CSP) 或 CNG csp 的控制碼。 |
| [**HCRYPTPROV**](hcryptprov.md)                                               | 指定 CryptoAPI CSP 的控制碼。                                                                                                                                                           |
| [**KEYSVCC \_ 控制碼**](keysvcc-handle.md)                                      | 指定金鑰服務的控制碼。                                                                                                                                                            |
