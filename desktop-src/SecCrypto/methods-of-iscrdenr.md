# <a name="methods-of-iscrdenr"></a>ISCrdEnr 的方法

[**ISCrdEnr**](iscrdenr.md)所引進的方法如下表所示。



| 方法                                                                    | 描述                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
|---------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**註冊**](/previous-versions/windows/desktop/legacy/aa386564(v=vs.85))                                         | 代表使用者要求憑證，並將產生的 [*憑證*](../secgloss/c-gly.md) 儲存在使用者的 [*智慧卡*](../secgloss/s-gly.md)上。                                                                                                                                                                                                                                                    |
| [**enumCAName**](iscrdenr-enumcaname.md)                                 | 針對指定的憑證範本名稱，列舉 (Ca) 的 [*憑證授權單位*](../secgloss/c-gly.md) 單位名稱。                                                                                                                                                                                                                                                                                                           |
| [**enumCertTemplateName**](iscrdenr-enumcerttemplatename.md)             | 列舉憑證範本名稱。                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [**enumCSPName**](iscrdenr-enumcspname.md)                               | 列舉 (Csp) 的可用 [*密碼編譯服務提供者*](../secgloss/c-gly.md) 的名稱。                                                                                                                                                                                                                                                                                                                   |
| [**getCACount**](iscrdenr-getcacount.md)                                 | 傳回 (Ca 的 [*憑證授權單位*](../secgloss/c-gly.md) 單位數目) 願意根據指定的憑證範本頒發證書。                                                                                                                                                                                                                                                                            |
| [**getCAName**](iscrdenr-getcaname.md)                                   | 取得指定憑證範本的指定 CA 名稱。                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| [**getCertTemplateCount**](iscrdenr-getcerttemplatecount.md)             | 抓取憑證範本的數目。                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**getCertTemplateName**](iscrdenr-getcerttemplatename.md)               | 抓取憑證範本的名稱。                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| [**getCertTemplateSMIME**](iscrdenr-getcerttemplatesmime.md)             | 判斷憑證範本是否包含 szOID PKIX 的 \_ \_ \_ 電子郵件 \_ 保護金鑰使用方式。 如果此金鑰使用方式是憑證範本的一部分，憑證範本支援 [*安全/多用途網際網路郵件延伸*](../secgloss/s-gly.md) (S/MIME) 作業。                                                                                                         |
| [**getEnrolledCertificateName**](iscrdenr-getenrolledcertificatename.md) | 抓取 [**ISCrdEnr：：註冊**](/previous-versions/windows/desktop/legacy/aa386564(v=vs.85))的先前成功呼叫所產生的憑證名稱。 這個方法也可以用來在對話方塊中顯示憑證。 這個方法會呼叫 [*CryptoAPI*](../secgloss/c-gly.md) 函數 [**CertGetNameString**](/windows/desktop/api/Wincrypt/nf-wincrypt-certgetnamestringa)。                                                                                                                                        |
| [**getSigningCertificateName**](iscrdenr-getsigningcertificatename.md)   | 從簽署憑證抓取主體名稱。 這個方法也可以用來在對話方塊中顯示憑證。 這個方法會呼叫 [*CryptoAPI*](../secgloss/c-gly.md) 函數 [**CertGetNameString**](/windows/desktop/api/Wincrypt/nf-wincrypt-certgetnamestringa)。                                                                                                                                                                                                               |
| [**getUserName**](iscrdenr-getusername.md)                               | 抓取要作為憑證註冊目標之使用者的名稱。 在呼叫這個方法之前，您必須在 [**ISCrdEnr：： selectUserName**](iscrdenr-selectusername.md) 或 [**ISCrdEnr：： setUserName**](iscrdenr-setusername.md)的呼叫中指定使用者名稱。                                                                                                                                                                                                                         |
| [**resetUser**](iscrdenr-resetuser.md)                                   | 從智慧卡控制項清除使用者名稱。                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [**selectSigningCertificate**](iscrdenr-selectsigningcertificate.md)     | 顯示 [ **選取憑證** ] 對話方塊，允許簽署憑證 (也稱為 *註冊代理程式憑證*) 選取。 代表使用者註冊之前，您必須先選取簽署憑證。 與此簽署憑證相關聯的 [*私密金鑰*](../secgloss/p-gly.md) 會用來簽署 PKCS \# 7 要求。 PKCS \# 7 接著會包含使用者的 pkcs \# 10 要求 (使用) 的使用者私密金鑰進行簽署。 |
| [**selectUserName**](iscrdenr-selectusername.md)                         | 顯示 **選取的使用者** 介面，允許選取使用者名稱。 使用者名稱會套用至代表憑證註冊的使用者。                                                                                                                                                                                                                                                                                                                                         |
| [**setCAName**](iscrdenr-setcaname.md)                                   | 指定 CA 的名稱。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| [**setCertTemplateName**](iscrdenr-setcerttemplatename.md)               | 指定憑證範本的名稱。                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| [**setSigningCertificate**](iscrdenr-setsigningcertificate.md)           | 指定簽署憑證 (也稱為 *註冊代理程式憑證*) 。 代表使用者註冊之前，您必須選取或設定簽署憑證。 與此簽署憑證相關聯的 [*私密金鑰*](../secgloss/p-gly.md) 會用來簽署 PKCS \# 7 要求。 PKCS \# 7 接著會包含使用者的 pkcs \# 10 要求 (使用) 的使用者私密金鑰進行簽署。                                                      |
| [**setUserName**](iscrdenr-setusername.md)                               | 指定要在其上註冊憑證的使用者名稱。                                                                                                                                                                                                                                                                                                                                                                                                                       |
