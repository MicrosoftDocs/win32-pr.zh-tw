# <a name="prov_dh_schannel"></a>>PROV \_ DH \_ SCHANNEL

>PROV \_ DH \_ SCHANNEL 提供者類型支援 Diffie-Hellman 和 [*SCHANNEL*](../secgloss/s-gly.md) 通訊協定。

## <a name="algorithms-supported"></a>支援的演算法

如需這些演算法的說明，請參閱詞彙。



| 目的      | 支援的演算法                                                                                                                                   |
|--------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| 金鑰交換 | DH (暫時)                                                                                                                                          |
| 簽名    | [*Dss*](../secgloss/d-gly.md)                                                                                            |
| 加密   | [*Des*](../secgloss/d-gly.md)[*三重 des*](../secgloss/t-gly.md)<br/> |
| 雜湊      | [*MD5*](../secgloss/m-gly.md)[*SHA*](../secgloss/s-gly.md)<br/>                      |
