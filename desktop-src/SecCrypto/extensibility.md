# <a name="extensibility-cryptography"></a>擴充性 (密碼編譯) 

憑證服務支援 [*x.509*](../secgloss/x-gly.md) 憑證延伸模組。 原則模組可以新增或修改擴充功能，而結束模組可以捕獲這些擴充功能。

如需詳細資訊，請參閱 [擴充程式處理常式](extension-handlers.md) 。
