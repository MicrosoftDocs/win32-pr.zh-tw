# <a name="required-libraries"></a>必要的程式庫

連結 CryptoAPI 函數需要 Crypt32.dll 的可用性。 使用某些功能的程式可能也需要 Advapi32.dll .lib。 使用 Visual C++ 編譯器時，這些檔案應該會列在物件/程式庫模組中。
