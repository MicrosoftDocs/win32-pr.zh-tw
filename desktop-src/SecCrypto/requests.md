# <a name="requests"></a>Requests

憑證服務支援[](../secgloss/c-gly.md)以 PKCS \# 10 格式和 Keygen (Netscape) 格式為基礎的憑證要求。 憑證服務也支援 \# (CMS) 要求的 PKCS 7 更新要求和密碼編譯訊息語法。

憑證服務也會提供 [ICertRequest](/windows/desktop/api/Certcli/nn-certcli-icertrequest) 介面，其中包含提交憑證要求的方法，以及 (是否已核准接收所產生之已發行憑證的) 。

[憑證註冊控制項](certificate-enrollment-control.md)中提供其他安全性相關的介面。
