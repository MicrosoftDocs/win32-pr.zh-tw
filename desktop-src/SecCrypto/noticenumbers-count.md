# <a name="noticenumberscount-property"></a>NoticeNumbers Count 屬性

\[[ **計數** ] 屬性可用於 [需求] 區段中指定的作業系統。 如需詳細資訊，請參閱 [**限定詞**](qualifier.md)。\]

**Count** 屬性會抓取集合中的 [**擴充**](extension.md)物件數目。

## <a name="syntax"></a>Syntax


```VB
NoticeNumbers.Count As Long
```



## <a name="property-value"></a>屬性值

集合中的 [**擴充**](extension.md) 物件數目。
