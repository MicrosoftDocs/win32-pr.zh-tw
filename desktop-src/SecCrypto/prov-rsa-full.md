# <a name="prov_rsa_full"></a>>PROV \_ RSA \_ FULL

>PROV \_ RSA \_ 完整提供者類型支援 [*數位簽章*](../secgloss/d-gly.md) 和資料加密。 它被視為一般用途 CSP。 RSA 公開金鑰演算法可用於所有的公開金鑰作業。

## <a name="algorithms-supported"></a>支援的演算法

如需這些演算法的說明，請參閱詞彙。



| 目的      | 支援的演算法                                                                                                              |
|--------------|-----------------------------------------------------------------------------------------------------------------------------------|
| 金鑰交換 | [*RSA*](../secgloss/r-gly.md)                                                                       |
| 簽名    | RSA                                                                                                                               |
| 加密   | [*RC2*](../secgloss/r-gly.md)[*RC4*](../secgloss/r-gly.md)<br/> |
| 雜湊      | [*MD5*](../secgloss/m-gly.md)[*SHA*](../secgloss/s-gly.md)<br/> |
