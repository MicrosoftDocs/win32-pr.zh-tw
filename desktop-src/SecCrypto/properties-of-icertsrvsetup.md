# <a name="properties-of-icertsrvsetup"></a>ICertSrvSetup 的屬性

下列屬性是由 [**ICertSrvSetup**](/windows/desktop/api/Casetup/nn-casetup-icertsrvsetup) 介面所定義。



| 屬性                                             | 描述                                                                                                                                                                                                  |
|------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CAErrorId**](/windows/desktop/api/Casetup/nf-casetup-icertsrvsetup-get_caerrorid)         | 取得與失敗的 [*憑證授權單位*](../secgloss/c-gly.md) 單位相關的其他錯誤資訊 (CA) 規格的識別碼。 |
| [**CAErrorString**](/windows/desktop/api/Casetup/nf-casetup-icertsrvsetup-get_caerrorstring) | 取得與失敗的 CA 規格相關之其他錯誤資訊的字串資料。                                                                                                                  |
