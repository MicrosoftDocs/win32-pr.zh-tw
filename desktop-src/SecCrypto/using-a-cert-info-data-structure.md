# <a name="using-a-cert_info-data-structure"></a>使用 CERT \_ 資訊資料結構

[**CERT \_內容**](/windows/desktop/api/Wincrypt/ns-wincrypt-cert_context) 和 [**憑證 \_ 資訊**](/windows/desktop/api/Wincrypt/ns-wincrypt-cert_info) 是保存憑證中資訊的主要資料結構。 使用憑證時，可能需要將其解碼。 如需詳細資訊，請參閱 [解碼 CERT \_ 資訊結構](decoding-a-cert-info-structure.md)。 您可以將資訊加入已解碼的憑證 **\_ 資訊** 結構，然後再重新編碼結構。 如需詳細資訊，請參閱 [編碼憑證 \_ 資訊結構](encoding-a-cert-info-structure.md)。
