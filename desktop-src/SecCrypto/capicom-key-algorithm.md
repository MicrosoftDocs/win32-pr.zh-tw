| member                             | 描述                                | 值 |
|------------------------------------|--------------------------------------------|-------|
| **CAPICOM \_ 金鑰 \_ 演算法 \_ 其他** | 使用替代金鑰演算法。<br/> | 0     |
| **CAPICOM \_ 金鑰 \_ 演算法 \_ RSA**   | 使用 RSA 金鑰演算法。<br/>      | 1     |
| **CAPICOM \_ 金鑰 \_ 演算法 \_ DSS**   | 使用 DSS 金鑰演算法。<br/>      | 2     |
