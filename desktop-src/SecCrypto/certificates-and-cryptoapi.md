# <a name="certificates-and-cryptoapi"></a>憑證和 CryptoAPI

[*CryptoAPI*](../secgloss/c-gly.md) 支援使用在 [IETF RFC 3280](https://www.ietf.org/rfc/rfc3280.txt)中定義的 x.509 憑證。 本檔假設您使用的是 x.509 或可比較的 [*數位憑證*](../secgloss/c-gly.md)。

X.509 標準憑證包含下列資訊。



| 欄位                | 描述                                                                                                                                                                                                                |
|----------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 版本              | 憑證的版本號碼。                                                                                                                                                                                         |
| 序號        | 憑證的序號。                                                                                                                                                                                          |
| 演算法識別碼 | 憑證簽署者使用的簽章演算法。                                                                                                                                                                        |
| 簽發者名稱          | 憑證簽發者的名稱。                                                                                                                                                                                     |
| 生效時間           | 憑證無效之前的日期。                                                                                                                                                                            |
| 不晚於            | 憑證不正確日期。                                                                                                                                                                             |
| 主體名稱         | 憑證簽發者或實體的名稱。                                                                                                                                                      |
| 演算法            | 公開金鑰所使用的演算法。                                                                                                                                                                                         |
| 主體公開金鑰   | 實際公開金鑰 (位字串) 。                                                                                                                                                                                          |
| 簽發者唯一識別碼     | 選擇性欄位。 如果有的話，version 必須是第2版。                                                                                                                                                                     |
| 主體唯一識別碼    | 選擇性欄位。 如果有的話，version 必須是第2版。                                                                                                                                                                     |
| 延伸模組           | 選擇性欄位。 代表簽發者可能想要新增至憑證的其他資料，例如電子郵件地址或授權，以頒發證書。 如果有擴充功能，版本必須是第3版。<br/> |
