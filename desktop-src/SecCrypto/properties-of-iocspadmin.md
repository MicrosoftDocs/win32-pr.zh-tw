# <a name="properties-of-iocspadmin"></a>IOCSPAdmin 的屬性

下列屬性是由 [**IOCSPAdmin**](/windows/desktop/api/certadm/nn-certadm-iocspadmin) 介面所定義。



| 屬性                                                                                 | 描述                                                                                                                                                                                                                      |
|------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**OCSPCAConfigurationCollection**](/windows/desktop/api/Certadm/nf-certadm-iocspadmin-get_ocspcaconfigurationcollection) | 取得 [**OCSPCAConfigurationCollection**](/windows/desktop/api/Certadm/nf-certadm-iocspadmin-get_ocspcaconfigurationcollection) 物件的實例。 此物件代表回應程式服務可以處理狀態要求的一組憑證。 |
| [**OCSPServiceProperties**](/windows/desktop/api/Certadm/nf-certadm-iocspadmin-get_ocspserviceproperties)                 | 取得 [**OCSPPropertyCollection**](/windows/desktop/api/Certadm/nn-certadm-iocsppropertycollection) 物件的實例。 此物件代表回應程式服務的屬性。                                                                            |
