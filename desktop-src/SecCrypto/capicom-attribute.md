# <a name="capicom_attribute-enumeration"></a>CAPICOM \_ 屬性列舉

**CAPICOM \_ 屬性** 列舉型別會定義與簽章相關 [*聯的屬性類型。*](../secgloss/d-gly.md)

## <a name="members"></a>成員



| member                                                       | 描述                                                                | 值 |
|--------------------------------------------------------------|----------------------------------------------------------------------------|-------|
| **CAPICOM \_ 驗證 \_ 屬性 \_ 簽署 \_ 時間**         | 屬性包含建立簽章的時間。<br/> | 0     |
| **CAPICOM \_ 驗證的 \_ 屬性 \_ 檔 \_ 名稱**        | 屬性包含已簽署檔的名稱。<br/>         | 1     |
| **CAPICOM \_ 驗證的 \_ 屬性 \_ 檔 \_ 描述** | 屬性包含已簽署檔的描述。<br/>    | 2     |



## <a name="remarks"></a>備註

**CAPICOM \_ 屬性** 列舉類型是由 [**Attribute.Name**](attribute-name.md)屬性所使用。
