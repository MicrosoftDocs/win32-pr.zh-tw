# <a name="properties-of-iocspcaconfigurationcollection"></a>IOCSPCAConfigurationCollection 的屬性

下列屬性是由 [**IOCSPCAConfigurationCollection**](/windows/desktop/api/Certadm/nn-certadm-iocspcaconfigurationcollection) 介面所定義。



| 屬性                                                               | 描述                                                                                                                                                                              |
|------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-get__newenum)    | 取得設定集的列舉值。                                                                                                                                            |
| [**計數**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-get_count)           | 取得設定集中 (CA) 設定的 [*憑證授權單位*](../secgloss/c-gly.md) 單位數目。 |
| [**項目**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-get_item)             | 取得由設定集中索引所識別的 CA 設定。                                                                                                                    |
| [**ItemByName**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-get_itembyname) | 取得設定集中依名稱識別的 CA 設定。                                                                                                                     |
