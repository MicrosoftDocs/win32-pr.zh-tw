# <a name="certificate-and-message-encoding-types"></a>憑證和訊息編碼類型

許多功能都需要憑證或 [*訊息編碼類型*](../secgloss/m-gly.md)。 此編碼類型為 **DWORD**，可能包含憑證和訊息編碼類型。 憑證編碼類型會儲存在低序位字組中。 訊息編碼類型會儲存在高序位字組中。 某些函數或結構欄位只需要其中一種編碼類型，但一律可接受同時指定這兩種編碼類型。 如需同時指定編碼類型的範例，請參閱[ \# 包含和 \# 定義](-includes-and--defines.md)。

下列參數命名慣例可用來指出所需的編碼類型。



| 名稱                       | 註解                                                                                                                                                                                                                                                                                                                |
|----------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *dwMsgAndCertEncodingType* | 這兩種編碼類型都是必要的。                                                                                                                                                                                                                                                                                       |
| *dwMsgEncodingType*        | 只需要訊息編碼類型。                                                                                                                                                                                                                                                                             |
| *dwCertEncodingType*       | 只需要憑證編碼類型。                                                                                                                                                                                                                                                                         |
| *dwEncodingType*           | 需要訊息或憑證編碼類型。 如果包含憑證編碼類型的低序位字組為非零值，則會使用它。 否則，會使用包含訊息編碼類型的高序位字組。 如果同時指定這兩者，則會使用低序位字組中的憑證編碼類型。 |



 

下表顯示目前定義的編碼類型。



| 編碼類型          | 值      |
|------------------------|------------|
| CRYPT \_ ASN \_ 編碼   | 0x00000001 |
| X509 \_ ASN \_ 編碼    | 0x00000001 |
| PKCS \_ 7 \_ ASN \_ 編碼 | 0x00010000 |
