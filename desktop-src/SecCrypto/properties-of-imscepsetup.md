# <a name="properties-of-imscepsetup"></a>IMSCEPSetup 的屬性

下列屬性是由 [**IMSCEPSetup**](/windows/desktop/api/Casetup/nn-casetup-imscepsetup) 介面所定義。



| 屬性                                                 | 描述                                                                                                                                              |
|----------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MSCEPErrorId**](/windows/desktop/api/Casetup/nf-casetup-imscepsetup-get_msceperrorid)         | 取得與失敗 NDES 規格相關之其他錯誤資訊的識別碼。 父物件上的任何方法呼叫都會重設這個屬性。          |
| [**MSCEPErrorString**](/windows/desktop/api/Casetup/nf-casetup-imscepsetup-get_msceperrorstring) | 取得與失敗 NDES 規格相關之其他錯誤資訊的字串資料。 父物件上的任何方法呼叫都會重設這個屬性。 |
