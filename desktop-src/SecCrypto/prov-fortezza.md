# <a name="prov_fortezza"></a>>PROV \_ FORTEZZA

>PROV \_ FORTEZZA 提供者類型包含一組密碼編譯通訊協定和演算法，由 [*美國國家標準和技術*](../secgloss/n-gly.md) 所擁有， (NIST) 。

## <a name="algorithms-supported"></a>支援的演算法

如需這些演算法的說明，請參閱詞彙。



| 目的      | 支援的演算法                                                                                      |
|--------------|-----------------------------------------------------------------------------------------------------------|
| 金鑰交換 | [*KEA*](../secgloss/k-gly.md)         |
| 簽名    | [*Dss*](../secgloss/d-gly.md) |
| 加密   | [*Skipjack*](../secgloss/s-gly.md)                                |
| 雜湊      | [*SHA*](../secgloss/s-gly.md)           |
