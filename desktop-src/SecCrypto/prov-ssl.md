>PROV \_ ssl 提供者類型支援安全通訊端層 (ssl) 通訊協定。

如需這些演算法的說明，請參閱詞彙。



| 目的      | 支援的演算法                                        |
|--------------|-------------------------------------------------------------|
| 金鑰交換 | [*RSA*](../secgloss/r-gly.md) |
| 簽名    | RSA                                                         |
| 加密   | 不定                                                      |
| 雜湊      | 不定                                                      |
