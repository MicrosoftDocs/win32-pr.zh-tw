# <a name="prov_ms_exchange"></a>>PROV \_ MS \_ EXCHANGE

>PROV \_ MS \_ exchange 提供者類型是針對 microsoft EXCHANGE 郵件應用程式的密碼編譯需求，以及與 microsoft mail 相容的其他應用程式所設計。

## <a name="algorithms-supported"></a>支援的演算法

如需這些演算法的說明，請參閱詞彙。



| 目的      | 支援的演算法                                           |
|--------------|----------------------------------------------------------------|
| 金鑰交換 | [*RSA*](../secgloss/r-gly.md)    |
| 簽名    | RSA                                                            |
| 加密   | [*CAST*](../secgloss/c-gly.md) |
| 雜湊      | [*MD5*](../secgloss/m-gly.md)    |
