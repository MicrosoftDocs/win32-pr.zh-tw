# <a name="prov_dss"></a>>PROV \_ DSS

>PROV \_ DSS 提供者類型（例如 >prov \_ RSA \_ SIG）只支援 [*雜湊*](../secgloss/h-gly.md) 和 [*數位簽章*](../secgloss/d-gly.md)。 >PROV DSS 提供者類型所指定的簽名演算法 \_ 是 (DSA) 的 [*數位簽章演算法*](../secgloss/d-gly.md) 。

## <a name="algorithms-supported"></a>支援的演算法

如需這些演算法的說明，請參閱詞彙。



| 目的      | 支援的演算法                                                                                                              |
|--------------|-----------------------------------------------------------------------------------------------------------------------------------|
| 金鑰交換 | 無                                                                                                                              |
| 簽名    | [*Dss*](../secgloss/d-gly.md)                                                                       |
| 加密   | 無                                                                                                                              |
| 雜湊      | [*MD5*](../secgloss/m-gly.md)[*SHA*](../secgloss/s-gly.md)<br/> |
