# <a name="cryptography-essentials"></a>密碼編譯基本概念

Microsoft CryptoAPI、應用程式開發介面 (API) 和 CAPICOM 提供的服務，可讓開發人員根據 [*密碼*](../secgloss/c-gly.md) 編譯將安全性新增至應用程式。 CryptoAPI 包含編碼和解碼 [*抽象語法標記法 (一)*](../secgloss/a-gly.md) 的功能 (asn.1) 、加密和解密資料、使用 [*數位憑證*](../secgloss/d-gly.md)進行驗證，以及使用憑證存放區進行數位憑證管理。 CryptoAPI 和 CAPICOM 都支援 PKI 和對稱金鑰加密。

密碼編譯基本內容包含下列主題：

-   [加密、CryptoAPI 和 CAPICOM](cryptography--cryptoapi--and-capicom.md)
-   [密碼編譯金鑰](cryptographic-keys.md)
-   [編碼和解碼的資料](encoded-and-decoded-data.md)
-   [資料加密和解密](data-encryption-and-decryption.md)
-   [雜湊和數位簽章](hashes-and-digital-signatures.md)
-   [時間戳記 Authenticode 簽章](time-stamping-authenticode-signatures.md)
-   [數位憑證](digital-certificates.md)
-   [數位訊息](digital-messages.md)
