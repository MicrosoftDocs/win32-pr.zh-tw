# <a name="general-purpose-functions-cryptography"></a> (密碼編譯) 的一般目的函式

下列一般用途範例函式會在範例程式中執行一些輔助工作：



| 函式                               | 描述                                                              |
|----------------------------------------|--------------------------------------------------------------------------|
| [**ByteToStr**](bytetostr.md)         | 將位元組陣列轉換成字元字串。                  |
| [**GetSignerCert**](getsignercert.md) | 從具有金鑰簽章延伸的存放區抓取憑證。 |
| [**MyHandleError**](myhandleerror.md) | 寫入錯誤訊息並結束程式。                             |
