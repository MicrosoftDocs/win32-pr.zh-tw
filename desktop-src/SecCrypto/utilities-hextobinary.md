# <a name="utilitieshextobinary-method"></a>HexToBinary 方法

\[**HexToBinary** 方法可用於 [需求] 區段中指定的作業系統。\]

**HexToBinary** 方法會將十六進位字串轉換為二進位壓縮的字串。

## <a name="syntax"></a>語法


```VB
Utilities.HexToBinary( _
  ByVal HexString _
)
```



## <a name="parameters"></a>參數

<dl> <dt>

*HexString* \[在\]
</dt> <dd>

要轉換為二進位壓縮字串的十六進位字串。

</dd> </dl>

## <a name="return-value"></a>傳回值

二進位壓縮的字串，相當於指定的十六進位字串。
