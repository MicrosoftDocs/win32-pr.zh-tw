# <a name="noticenumbersitem-property"></a>NoticeNumbers 專案屬性

\[**專案** 屬性可用於 [需求] 區段中指定的作業系統。 如需詳細資訊，請參閱 [**限定詞**](qualifier.md)。\]

**Item** 屬性會取出 **Long** ，其代表集合的索引聲明編號。 這是預設屬性。

## <a name="syntax"></a>Syntax


```VB
NoticeNumbers.Item( _
  ByVal Index _
) As Variant
```



## <a name="property-value"></a>屬性值

代表集合之索引通知編號的 [**擴充**](extension.md) 物件。
