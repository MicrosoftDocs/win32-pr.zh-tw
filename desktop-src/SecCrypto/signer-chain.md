# <a name="signerchain-property"></a>簽署者。連鎖屬性

\[**Chain** 屬性可用於 [需求] 區段中指定的作業系統。 相反地，請使用 [**CmsSigner 類別**](/dotnet/api/system.security.cryptography.pkcs.cmssigner?view=dotnet-plat-ext-3.1&preserve-view=true)，以使用 [**system.servicemodel 命名空間。**](/dotnet/api/system.security.cryptography.pkcs?view=dotnet-plat-ext-3.1&preserve-view=true)\]

**Chain** 屬性會抓取簽署者的鏈。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
Signer.Chain As Chain
```



## <a name="property-value"></a>屬性值

簽署者的鏈。
