# <a name="properties-of-icertsrvsetupkeyinformationcollection"></a>ICertSrvSetupKeyInformationCollection 的屬性

下列屬性是由 [**ICertSrvSetupKeyInformationCollection**](/windows/desktop/api/Casetup/nn-casetup-icertsrvsetupkeyinformationcollection) 介面所定義。



| 屬性                                                            | 描述                                                                                                              |
|---------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Casetup/nf-casetup-icertsrvsetupkeyinformationcollection-get__newenum) | 取得此集合的列舉值。                                                                                   |
| [**計數**](/windows/desktop/api/Casetup/nf-casetup-icertsrvsetupkeyinformationcollection-get_count)        | 取得集合中的 [**ICertSrvSetupKeyInformation**](/windows/desktop/api/Casetup/nn-casetup-icertsrvsetupkeyinformation) 物件數目。         |
| [**項目**](/windows/desktop/api/Casetup/nf-casetup-icertsrvsetupkeyinformationcollection-get_item)          | 取得集合中索引所識別的 [**ICertSrvSetupKeyInformation**](/windows/desktop/api/Casetup/nn-casetup-icertsrvsetupkeyinformation) 物件。 |
