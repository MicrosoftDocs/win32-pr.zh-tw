# <a name="status-bar-styles"></a>狀態列樣式

除了由 *狀態列* 控制項支援的標準視窗樣式之外，本節還會列出樣式。



| 常數                                                                                                                                                         | 描述                                                                                                                                                                                                                           |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="SBARS_SIZEGRIP"></span><span id="sbars_sizegrip"></span><dl> <dt>**SBARS \_ SIZEGRIP**</dt> </dl> | 狀態列控制項將在狀態列的右端包含調整大小的底框。 調整大小的底框類似縮放框線；它是使用者可以按一下並拖曳調整父視窗大小的矩形區域。 <br/> |
| <span id="SBT_TOOLTIPS"></span><span id="sbt_tooltips"></span><dl> <dt>**SBT \_ 工具提示**</dt> </dl>       | [版本4.71。](common-control-versions.md)您可以使用此樣式來啟用工具提示。<br/>                                                                                                                                             |
| <span id="SBARS_TOOLTIPS"></span><span id="sbars_tooltips"></span><dl> <dt>**SBARS \_ 工具提示**</dt> </dl> | [版本5.80。](common-control-versions.md)與 SBT \_ 工具提示相同。 針對5.00 版或更新版本使用此旗標。 <br/>                                                                                                         |
