# <a name="about-common-controls"></a>關於通用控制項

本主題討論一般控制項，這是由通用控制項程式庫所執行的一組視窗，Comctl32.dll，也就是 Windows 作業系統隨附的 DLL。 如同其他控制項視窗，通用控制項是應用程式與另一個視窗一起使用的子視窗，以啟用與使用者的互動。
