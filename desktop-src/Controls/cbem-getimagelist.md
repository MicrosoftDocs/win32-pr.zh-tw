# <a name="cbem_getimagelist-message"></a>CBEM \_ GETIMAGELIST 訊息

取得指派給 ComboBoxEx 控制項之影像清單的控制碼。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>必須為零。</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

如果成功，則傳回指派給控制項之影像清單的控制碼; 否則傳回 **Null** 。

| 標頭<br/>                   | <dl> <dt>Commctrl。h</dt> </dl> |
