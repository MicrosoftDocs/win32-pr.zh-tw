# <a name="udm_getbase-message"></a>UDM \_ GETBASE 訊息

抓取目前的基本基底基底 (也就是上下按鈕控制項的基底10或 16) 。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>必須為零。</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

傳回值是目前的基底值。
