# <a name="about-buttons"></a>關於按鈕

按鈕是控制項，可讓使用者按一下以提供應用程式的輸入。

有數種類型的按鈕，以及一或多個按鈕樣式來區別相同類型的按鈕。 使用者使用滑鼠或鍵盤按一下按鈕。 按一下按鈕通常會將其視覺外觀和狀態 (從 [已檢查] 變更為 [已清除]，例如) 。 系統、按鈕和應用程式會共同變更按鈕的外觀和狀態。 按鈕可以將訊息傳送至其父視窗，而且父視窗可以將訊息傳送至按鈕。 有些按鈕是由系統所繪製，有些是由應用程式所繪製。 按鈕可單獨使用或在群組中，而且可以在標籤)  (的應用程式定義文字時出現。 它們屬於按鈕視窗類別。
