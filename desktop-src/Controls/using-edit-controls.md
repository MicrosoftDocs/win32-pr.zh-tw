# <a name="using-edit-controls"></a>使用編輯控制項

本節提供執行編輯控制項的資訊和範例程式碼。

編輯控制項通常用於對話方塊中，但您也可以在標準視窗的工作區中使用它們。 單行編輯控制項適合用來從使用者抓取單一字串。 多行編輯控制項可讓您在應用程式中執行簡單字處理器的大部分功能。

| [如何建立多行編輯控制項](use-a-multiline-edit-control.md)<br/>      | 本主題將示範如何藉由將多行編輯控制項新增至視窗的工作區，來執行簡單的單字處理器。 <br/> |
