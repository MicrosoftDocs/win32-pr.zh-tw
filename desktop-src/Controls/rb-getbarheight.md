# <a name="rb_getbarheight-message"></a>RB \_ GETBARHEIGHT 訊息

抓取 Rebar 控制項的高度。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>必須為零。</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

傳回表示控制項高度（以圖元為單位）的 **UINT** 值。
