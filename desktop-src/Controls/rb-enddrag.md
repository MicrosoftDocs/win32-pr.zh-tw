# <a name="rb_enddrag-message"></a>RB \_ ENDDRAG 訊息

終止 Rebar 控制項的拖放作業。 此訊息不會造成 [RBN \_ ENDDRAG](rbn-enddrag.md) 通知傳送。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>必須為零。</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

未使用此訊息的傳回值。
