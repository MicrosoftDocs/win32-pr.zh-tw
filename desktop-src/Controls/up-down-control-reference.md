# <a name="up-down-control"></a>Up-Down 控制項

本章節包含與上下按鈕控制項搭配使用之程式設計項目的相關資訊。

### <a name="overviews"></a>概觀



| 主題                                    | 目錄                                                                                                                                                                                                            |
|------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [上下控制項](up-down-controls.md) | 由上而下的控制項是一組箭號按鈕，使用者可以按一下來遞增或遞減值，例如在附屬控制項中顯示的捲軸位置或數位， (稱為「合作者視窗) 。<br/> |



 

### <a name="functions"></a>函式



| 主題                                              | 目錄                                                                                                                                                |
|----------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CreateUpDownControl**](/windows/desktop/api/Commctrl/nf-commctrl-createupdowncontrol) | 建立上下按鈕控制項。 注意：此函式已過時。 它是16位函式，而且無法處理範圍和位置的32位值。<br/> |



 

### <a name="messages"></a>訊息



| 主題                                                 | 目錄                                                                                                                                                                                                                               |
|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**UDM \_ GETACCEL**](udm-getaccel.md)                 | 抓取上下按鈕控制項的加速資訊。 <br/>                                                                                                                                                                 |
| [**UDM \_ GETBASE**](udm-getbase.md)                   | 抓取目前的基本基底基底 (也就是上下按鈕控制項的基底10或 16) 。 <br/>                                                                                                                                   |
| [**UDM \_ GETBUDDY**](udm-getbuddy.md)                 | 抓取目前好友視窗的控制碼。 <br/>                                                                                                                                                                          |
| [**UDM \_ GETPOS**](udm-getpos.md)                     | 抓取具有16位精確度之上下按鈕控制項的目前位置。 <br/>                                                                                                                                                |
| [**UDM \_ GETPOS32**](udm-getpos32.md)                 | 傳回上下按鈕控制項的32位位置。<br/>                                                                                                                                                                          |
| [**UDM \_ GETRANGE**](udm-getrange.md)                 | 抓取上下按鈕 (範圍) 的最小和最大位置。 <br/>                                                                                                                                                |
| [**UDM \_ GETRANGE32**](udm-getrange32.md)             | 抓取上下按鈕控制項的32位範圍。 <br/>                                                                                                                                                                          |
| [**UDM \_ GETUNICODEFORMAT**](udm-getunicodeformat.md) | 抓取控制項的 Unicode 字元格式旗標。 <br/>                                                                                                                                                               |
| [**UDM \_ SETACCEL**](udm-setaccel.md)                 | 設定上下按鈕控制項的加速。 <br/>                                                                                                                                                                              |
| [**UDM \_ SETBASE**](udm-setbase.md)                   | 設定上下按鈕控制項的基本基底。 基底值會決定合作者視窗是否以十進位或十六進位數字顯示數位。 十六進位數位一律不帶正負號，而且會簽署十進位數。 <br/> |
| [**UDM \_ SETBUDDY**](udm-setbuddy.md)                 | 設定上下按鈕控制項的合作者視窗。 <br/>                                                                                                                                                                              |
| [**UDM \_ SETPOS**](udm-setpos.md)                     | 設定具有16位精確度之上下按鈕控制項的目前位置。 <br/>                                                                                                                                                    |
| [**UDM \_ SETPOS32**](udm-setpos32.md)                 | 設定具有32位精確度的上下按鈕控制項的位置。<br/>                                                                                                                                                              |
| [**UDM \_ SETRANGE**](udm-setrange.md)                 | 設定上下按鈕 (範圍) 的最小和最大位置。<br/>                                                                                                                                                      |
| [**UDM \_ SETRANGE32**](udm-setrange32.md)             | 設定上下按鈕控制項的32位範圍。 <br/>                                                                                                                                                                               |
| [**UDM \_ SETUNICODEFORMAT**](udm-setunicodeformat.md) | 設定控制項的 Unicode 字元格式旗標。 此訊息可讓您變更控制項在執行時間所使用的字元集，而不需要重新建立控制項。 <br/>                                   |



 

### <a name="notifications"></a>通知



| 主題                                                            | 目錄                                                                                                                                                                                                                                                                                                                                                       |
|------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [NM \_ RELEASEDCAPTURE (上下) ](nm-releasedcapture-up-down-.md) | 通知上下控制項的父視窗，控制項正在釋放滑鼠捕捉。 此通知會以 [**WM \_ 通知**](wm-notify.md) 訊息的形式傳送。 <br/>                                                                                                                                                                       |
| [UDN \_ DELTAPOS](udn-deltapos.md)                                | 當控制項的位置即將變更時，作業系統會將其傳送至上下按鈕控制項的父視窗。 當使用者按下控制項的向上或向下箭號時，就會發生這種情況。 [UDN \_ DELTAPOS](udn-deltapos.md)訊息會以 [**WM \_ 通知**](wm-notify.md)訊息的形式傳送。 <br/> |



 

### <a name="structures"></a>結構

| [**UDACCEL**](/windows/desktop/api/Commctrl/ns-commctrl-udaccel)   | 包含上下按鈕控制項的加速資訊。 <br/>                                                                             |
