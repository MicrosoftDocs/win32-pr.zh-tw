# <a name="udm_setrange32-message"></a>UDM \_ SETRANGE32 訊息

設定上下按鈕控制項的32位範圍。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>

帶正負號的整數值，表示上下按鈕控制項範圍的新下限。

</dd> <dt>

*lParam* 
</dt> <dd>

帶正負號的整數值，表示上下按鈕控制項範圍的新上限。

</dd> </dl>

## <a name="return-value"></a>傳回值

未使用此訊息的傳回值。
