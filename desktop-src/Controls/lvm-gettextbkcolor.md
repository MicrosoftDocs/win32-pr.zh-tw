# <a name="lvm_gettextbkcolor-message"></a>LVM \_ GETTEXTBKCOLOR 訊息

抓取清單視圖控制項的文字背景色彩。 您可以明確地傳送此訊息，或使用 [**ListView \_ GetTextBkColor**](/windows/desktop/api/Commctrl/nf-commctrl-listview_gettextbkcolor) 宏來傳送。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>必須為零。</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

傳回文字的背景色彩。
