# <a name="reextendedregisterclass-function"></a>REExtendedRegisterClass 函式

\[適用于內部用途;不建議在應用程式中使用。 未來的版本可能不支援此功能。\]

註冊兩個類別名稱（REListBox20W 和 RECombobox20W），可用來建立 Rich Edit listbox 或 combobox 視窗。

## <a name="syntax"></a>語法


```C++
BOOL WINAPI REExtendedRegisterClass(void);
```



## <a name="parameters"></a>參數

此函式沒有參數。

## <a name="return-value"></a>傳回值

如果成功，則傳回 TRUE，否則傳回 FALSE。
