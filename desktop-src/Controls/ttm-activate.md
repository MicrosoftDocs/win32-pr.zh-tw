# <a name="ttm_activate-message"></a>TTM \_ 啟動訊息

啟用或停用工具提示控制項。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>

啟用旗標。 如果此參數為 **TRUE**，則會啟用工具提示控制項。 如果為 **FALSE**，則會停用工具提示控制項。

</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

沒有傳回值。
