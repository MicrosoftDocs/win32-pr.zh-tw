# <a name="rb_gettextcolor-message"></a>RB \_ GETTEXTCOLOR 訊息

抓取 Rebar 控制項的預設文字色彩。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>必須為零。</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

傳回代表目前預設文字色彩的 [**COLORREF**](/windows/desktop/gdi/colorref) 值。
