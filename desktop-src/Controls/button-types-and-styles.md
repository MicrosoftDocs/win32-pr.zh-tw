## <a name="radio-buttons"></a><span data-ttu-id="f9bd4-165">選項按鈕</span><span class="sxs-lookup"><span data-stu-id="f9bd4-165">Radio Buttons</span></span>

<span data-ttu-id="f9bd4-166">選項按鈕 (也稱為 *單選* 按鈕) 是由圓形按鈕和應用程式定義的標籤、圖示或點陣圖所組成，表示使用者可以藉由選取按鈕所做的選擇。</span><span class="sxs-lookup"><span data-stu-id="f9bd4-166">A *radio button* (also called option button) consists of a round button and an application-defined label, icon, or bitmap that indicates a choice the user can make by selecting the button.</span></span> <span data-ttu-id="f9bd4-167">應用程式通常會在群組方塊中使用選項按鈕，讓使用者可以選擇一組相關但互斥選項的其中一個。</span><span class="sxs-lookup"><span data-stu-id="f9bd4-167">An application typically uses radio buttons in a group box to enable the user to choose one of a set of related but mutually exclusive options.</span></span>

<span data-ttu-id="f9bd4-168">選項按鈕可以是下列兩種樣式之一： [標準] 或 [自動]，如 [樣式常數] [**BS \_ 選項按鈕**](button-styles.md) 和 [ [**BS \_ AUTORADIOBUTTON**](button-styles.md)] 所定義。</span><span class="sxs-lookup"><span data-stu-id="f9bd4-168">A radio button can be one of two styles: standard or automatic, as defined by the style constants [**BS\_RADIOBUTTON**](button-styles.md) and [**BS\_AUTORADIOBUTTON**](button-styles.md).</span></span> <span data-ttu-id="f9bd4-169">每個樣式都可以假設有兩個檢查狀態：已核取 (按鈕中的點) 或已清除 (按鈕) 中沒有任何點。</span><span class="sxs-lookup"><span data-stu-id="f9bd4-169">Each style can assume two check states: checked (a dot in the button) or cleared (no dot in the button).</span></span>

<span data-ttu-id="f9bd4-170">當使用者選取其中一個狀態時，選項按鈕會收到鍵盤焦點。</span><span class="sxs-lookup"><span data-stu-id="f9bd4-170">When the user selects either state, the radio button receives the keyboard focus.</span></span> <span data-ttu-id="f9bd4-171">系統會傳送包含 [BN \_ 點擊](bn-clicked.md)通知碼的 [**WM \_ 命令**](/windows/desktop/menurc/wm-command)訊息給按鈕的父視窗。</span><span class="sxs-lookup"><span data-stu-id="f9bd4-171">The system sends the button's parent window a [**WM\_COMMAND**](/windows/desktop/menurc/wm-command) message containing the [BN\_CLICKED](bn-clicked.md) notification code.</span></span> <span data-ttu-id="f9bd4-172">如果父視窗來自自動選項按鈕，就不需要處理此訊息，因為系統會自動設定該樣式的檢查狀態。</span><span class="sxs-lookup"><span data-stu-id="f9bd4-172">The parent window need not handle this message if it comes from an automatic radio button, because the system automatically sets the check state for that style.</span></span> <span data-ttu-id="f9bd4-173">但是，父視窗應該處理來自非自動選項按鈕的訊息，因為父視窗負責設定該樣式的檢查狀態。</span><span class="sxs-lookup"><span data-stu-id="f9bd4-173">But the parent window should handle the message if it comes from a non-automatic radio button, because the parent window is responsible for setting the check state for that style.</span></span> <span data-ttu-id="f9bd4-174">不論選項按鈕樣式為何，系統都會自動將按鈕重新繪製為其狀態變更。</span><span class="sxs-lookup"><span data-stu-id="f9bd4-174">Regardless of the radio button style, the system automatically repaints the button as its state changes.</span></span>

<span data-ttu-id="f9bd4-175">選項按鈕會以群組排列，而且可以隨時檢查群組中的一個按鈕。</span><span class="sxs-lookup"><span data-stu-id="f9bd4-175">Radio buttons are arranged in groups, and only one button in the group can be checked at any time.</span></span> <span data-ttu-id="f9bd4-176">如果已設定任何選項按鈕的 [**WS \_ 群組**](/windows/desktop/winmsg/window-styles) 旗標，該按鈕會是群組中的第一個按鈕，而緊接在定位順序中的所有按鈕會立即在定位順序中 (但不會有 **WS \_ 群組** 旗標) 屬於其群組的一部分。</span><span class="sxs-lookup"><span data-stu-id="f9bd4-176">If the [**WS\_GROUP**](/windows/desktop/winmsg/window-styles) flag is set for any radio button, that button is the first button in a group, and all buttons that follow it immediately in the tab order (but do not themselves have the **WS\_GROUP** flag) are part of its group.</span></span> <span data-ttu-id="f9bd4-177">如果沒有任何選項按鈕具有 **WS \_ 群組** 旗標，則對話方塊中的所有選項按鈕都會被視為單一群組。</span><span class="sxs-lookup"><span data-stu-id="f9bd4-177">If no radio buttons have the **WS\_GROUP** flag, all the radio buttons in the dialog box are treated as a single group.</span></span>

<span data-ttu-id="f9bd4-178">應用程式可以使用 [**IsDlgButtonChecked**](/windows/desktop/api/Winuser/nf-winuser-isdlgbuttonchecked) 函數來確定是否已核取選項按鈕。</span><span class="sxs-lookup"><span data-stu-id="f9bd4-178">The application can ascertain whether a radio button is checked by using the [**IsDlgButtonChecked**](/windows/desktop/api/Winuser/nf-winuser-isdlgbuttonchecked) function.</span></span>
