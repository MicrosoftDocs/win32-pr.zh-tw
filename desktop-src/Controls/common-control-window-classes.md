# <a name="window-classes-commctrlh"></a>視窗類別 (CommCtrl .h) 

此區段會列出通用控制項程式庫所提供的視窗類別名稱。



| 常數                                                                                                                                                                     | 描述                                                                                                                                                                                                                                                    |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="ANIMATE_CLASS"></span><span id="animate_class"></span><dl> <dt>**動畫 \_ 類別**</dt> </dl>                | 建立動畫控制項。 這些控制項會以無訊息模式顯示交錯 (AVI) 剪輯的音訊影片。<br/>                                                                                                                                                  |
| <span id="DATETIMEPICK_CLASS"></span><span id="datetimepick_class"></span><dl> <dt>**DATETIMEPICK \_ 類別**</dt> </dl> | 建立日期和時間選擇器控制項。 這些控制項提供簡單的介面，可與使用者交換日期和時間資訊。 <br/>                                                                                                                |
| <span id="HOTKEY_CLASS"></span><span id="hotkey_class"></span><dl> <dt>**快速鍵 \_ 類別**</dt> </dl>                   | 建立快速鍵控制項。 這些控制項可讓使用者輕鬆地定義快速鍵。 <br/>                                                                                                                                                             |
| <span id="LINK_CLASS"></span><span id="link_class"></span><dl> <dt>**LINK \_ 類別**</dt> </dl>                         | 建立 syslink 控制項。 這些控制項可搭配連結使用。 <br/>                                                                                                                                                                                      |
| <span id="MONTHCAL_CLASS"></span><span id="monthcal_class"></span><dl> <dt>**MONTHCAL \_ 類別**</dt> </dl>             | 建立月曆控制項。 這些控制項提供簡單的方法，讓使用者從熟悉的介面中選取日期。 <br/>                                                                                                                        |
| <span id="NATIVEFNTCTL_CLASS"></span><span id="nativefntctl_class"></span><dl> <dt>**NATIVEFNTCTL \_ 類別**</dt> </dl> | 建立原生字型控制項。 這些控制項是與原生字型一起使用。 <br/>                                                                                                                                                                           |
| <span id="PROGRESS_CLASS"></span><span id="progress_class"></span><dl> <dt>**進度 \_ 類別**</dt> </dl>             | 建立進度列。 這些控制項會指出冗長作業的進度。<br/>                                                                                                                                                                 |
| <span id="REBARCLASSNAME"></span><span id="rebarclassname"></span><dl> <dt>**REBARCLASSNAME**</dt> </dl>              | 建立 Rebar 控制項。 這些控制項會作為子視窗的容器。 <br/>                                                                                                                                                                       |
| <span id="STANDARD_CLASSES"></span><span id="standard_classes"></span><dl> <dt>**標準 \_ 類別**</dt> </dl>       | 建立內部的 User32 控制項，例如按鈕、編輯和捲軸。<br/>                                                                                                                                                                            |
| <span id="STATUSCLASSNAME"></span><span id="statusclassname"></span><dl> <dt>**STATUSCLASSNAME**</dt> </dl>           | 建立狀態視窗。 這些控制項會在水準視窗中顯示狀態資訊。<br/>                                                                                                                                                           |
| <span id="TOOLBARCLASSNAME"></span><span id="toolbarclassname"></span><dl> <dt>**TOOLBARCLASSNAME**</dt> </dl>        | 建立工具列。 這些控制項包含執行功能表命令的按鈕。 <br/>                                                                                                                                                                     |
| <span id="TOOLTIPS_CLASS"></span><span id="tooltips_class"></span><dl> <dt>**TOOLTIP \_ 類別**</dt> </dl>             | 建立工具提示控制項。 這些控制項會顯示一個小型的快顯視窗，其中包含描述應用程式中工具用途的文字行。<br/>                                                                                            |
| <span id="TRACKBAR_CLASS"></span><span id="trackbar_class"></span><dl> <dt>**[跟蹤] \_ 類別**</dt> </dl>             | 建立 trackbars。 這些控制項可讓使用者藉由移動滑杆，從某個範圍的值進行選取。 <br/>                                                                                                                                                   |
| <span id="UPDOWN_CLASS"></span><span id="updown_class"></span><dl> <dt>**上下按鈕 \_ 類別**</dt> </dl>                   | 建立上下按鈕控制項。 這些控制項結合了一對箭號與編輯控制項。 按一下箭號會遞增或遞減編輯控制項中的值。 <br/>                                                                                |
| <span id="WC_BUTTON"></span><span id="wc_button"></span><dl> <dt>**WC \_ 按鈕**</dt> </dl>                            | 建立按鈕控制項。 這些控制項可讓使用者起始動作。<br/>                                                                                                                                                                        |
| <span id="WC_COMBOBOX"></span><span id="wc_combobox"></span><dl> <dt>**WC \_ COMBOBOX**</dt> </dl>                      | 建立 combobox 控制項。 這些控制項包含可從中選取單一專案的下拉式清單。 <br/>                                                                                                                                       |
| <span id="WC_COMBOBOXEX"></span><span id="wc_comboboxex"></span><dl> <dt>**WC \_ COMBOBOXEX**</dt> </dl>                | 建立 ComboBoxEx 控制項。 這些控制項提供下拉式方塊控制項的延伸模組，可提供專案影像的原生支援。 <br/>                                                                                                            |
| <span id="WC_EDIT"></span><span id="wc_edit"></span><dl> <dt>**WC \_ 編輯**</dt> </dl>                                  | 建立編輯控制項。 這些控制項包含可編輯的文字。 <br/>                                                                                                                                                                                       |
| <span id="WC_HEADER"></span><span id="wc_header"></span><dl> <dt>**WC \_ 標頭**</dt> </dl>                            | 建立標題控制項。 這些控制項會在資訊的資料行頂端顯示標題，並讓使用者按一下標題來排序資訊。 <br/>                                                                                      |
| <span id="WC_LISTBOX"></span><span id="wc_listbox"></span><dl> <dt>**WC \_ LISTBOX**</dt> </dl>                         | 建立 listbox 控制項。 這些控制項包含專案清單。 <br/>                                                                                                                                                                                   |
| <span id="WC_IPADDRESS"></span><span id="wc_ipaddress"></span><dl> <dt>**WC \_ IPADDRESS**</dt> </dl>                   | 建立 IP 位址控制項。 這些控制項與編輯控制項類似，但可讓您輸入網際網路通訊協定中的數位位址 (IP) 格式。 <br/>                                                                                       |
| <span id="WC_LINK"></span><span id="wc_link"></span><dl> <dt>**WC \_ 連結**</dt> </dl>                                  | 建立 SysLink 控制項。 這些控制項包含超文字連結。<br/>                                                                                                                                                                                   |
| <span id="WC_LISTVIEW"></span><span id="wc_listview"></span><dl> <dt>**WC \_ LISTVIEW**</dt> </dl>                      | 建立清單視圖控制項。 這些控制項會顯示專案的集合，每個專案都包含一個圖示和一個標籤，並提供數種方式來排列專案。<br/>                                                                                    |
| <span id="WC_NATIVEFONTCTL"></span><span id="wc_nativefontctl"></span><dl> <dt>**WC \_ NATIVEFONTCTL**</dt> </dl>       | 建立原生字型控制項。 這些是在背景中運作的隱藏控制項，可讓對話方塊的預先定義控制項顯示目前的系統語言。<br/>                                                                         |
| <span id="WC_PAGESCROLLER"></span><span id="wc_pagescroller"></span><dl> <dt>**WC \_ PAGESCROLLER**</dt> </dl>          | 建立呼機控制項。 這些控制項可用來包含和滾動另一個視窗。 <br/>                                                                                                                                                              |
| <span id="WC_SCROLLBAR"></span><span id="wc_scrollbar"></span><dl> <dt>**WC \_ 捲軸**</dt> </dl>                   | 建立捲軸控制項。 這些控制項可讓使用者滾動視窗的內容。<br/>                                                                                                                                                     |
| <span id="WC_STATIC"></span><span id="wc_static"></span><dl> <dt>**WC \_ 靜態**</dt> </dl>                            | 建立靜態控制項。 這些控制項包含 noneditable 的文字。 <br/>                                                                                                                                                                                  |
| <span id="WC_TABCONTROL"></span><span id="wc_tabcontrol"></span><dl> <dt>**WC \_ TABCONTROL**</dt> </dl>                | 建立索引標籤控制項。 這些控制項會針對視窗或對話方塊的相同區域定義多個頁面。 每個頁面都包含一組資訊，或是當使用者選取對應的索引標籤時，應用程式所顯示的一組控制項。<br/> |
| <span id="WC_TREEVIEW"></span><span id="wc_treeview"></span><dl> <dt>**WC \_ TREEVIEW**</dt> </dl>                      | 建立樹狀檢視控制項。 這些控制項會顯示階層式專案清單。 每個專案都包含一個標籤和一個選擇性點陣圖。<br/>                                                                                                              |
