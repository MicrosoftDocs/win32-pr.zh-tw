# <a name="rb_getpalette-message"></a>RB \_ GETPALETTE 訊息

抓取 Rebar 控制項的目前調色板。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>必須為零。</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

傳回 **HPALETTE** ，指定 Rebar 控制項目前的調色板。
