# <a name="rb_settooltips-message"></a>RB \_ SETTOOLTIPS 訊息

將工具提示控制項與 Rebar 控制項產生關聯。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>

要設定之工具提示控制項的控制碼。

</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

未使用此訊息的傳回值。
