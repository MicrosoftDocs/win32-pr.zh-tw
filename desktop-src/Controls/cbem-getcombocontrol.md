# <a name="cbem_getcombocontrol-message"></a>CBEM \_ GETCOMBOCONTROL 訊息

取得子下拉式方塊控制項的控制碼。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>必須為零。</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

將控制碼傳回至 ComboBoxEx 控制項內的下拉式方塊控制項。
