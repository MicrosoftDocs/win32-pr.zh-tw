# <a name="format-values"></a>格式值

此區段會列出與 [**DrawThemeText**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemetext)和 [**GetThemeTextExtent**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemetextextent)函數的 *dwTextFlags* 參數搭配使用的值。 其中一個或多個值可用來指定文字字串的格式。 **DrawThemeText** 函式會在顯示矩形內（由 *pRect* 參數指定）繪製文字字串。 **GetThemeTextExtent** 函式可用來判斷指定長度和格式文字所需的周框。

## <a name="format-values"></a>格式值

下表列出格式值。



| 值                    | 描述                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
|--------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| DT \_ 底部               | 在顯示矩形底部呈現文字字串。 此值僅用於 DT \_ 單行值。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| DT \_ CALCRECT             | 決定顯示矩形的寬度和高度。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| DT \_ CENTER               | 將文字水準置中顯示矩形。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| DT \_ EDITCONTROL          | 複製多行編輯控制項的文字顯示特性。 具體而言，平均字元寬度的計算方式與編輯控制項的方式相同，且函式不會顯示部分可見的最後一行。                                                                                                                                                                                                                                                                                                                                                                                       |
| DT \_ END \_ 省略號        | 截斷比顯示矩形更寬的文字字串，並加入省略號以表示截斷。 除非指定了 DT MODIFYSTRING 旗標，否則不會修改字串 \_ 。                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| DT \_ EXPANDTABS           | 展開定位字元。 每個定位預設有八個字元。 DT \_ 文字 \_ 省略號、DT \_ PATH \_ 省略號和 dt \_ END \_ 省略號值不能搭配 dt \_ EXPANDTABS 值使用。                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| DT \_ EXTERNALLEADING      | 在行高中包含字型的外部前置。 通常，外部前導不會包含在文字行的高度中。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| DT \_ HIDEPREFIX           | 忽略文字中 & 的前置詞字元。 接下來的字母不會加上底線，但仍會處理其他前置詞字元。 例如：<br/> 輸入字串： "A&bc&&d"<br/> normal： "A<span class="underline">b</span>c&d"<br/> DT \_ HIDEPREFIX： "Abc&d"<br/>                                                                                                                                                                                                                                                                                                                                       |
| \_向左 DT                 | 將文字靠左對齊。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| DT \_ MODIFYSTRING         | 修改字串以符合顯示的文字。 除非 \_ \_ 指定了 DT END 省略號或 dt \_ 路徑 \_ 省略號，否則這個值不會有任何作用。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| DT \_ NOCLIP               | 繪製文字字串，而不裁剪顯示矩形。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| DT \_ NOFULLWIDTHCHARBREAK | 防止雙位元組字元集上的分行符號 (DBCS) ，如此一行中斷規則就相當於單一位元組字元集 (SBCS) 。 例如，這可以用來讓以韓文文字撰寫的圖示標籤更容易閱讀。 除非指定了 DT WORDBREAK，否則此值沒有任何作用 \_ 。                                                                                                                                                                                                                                                                                                                                   |
| DT \_ NOPREFIX             | 關閉前置字元的處理作業。 一般情況下， [**DrawThemeText**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemetext) 會將前置詞字元 & 解讀為指示詞，以底線顯示後面的字元，而前置詞字元 && 作為指示詞，以列印單一 &。 藉由指定 DT \_ NOPREFIX，此處理將會關閉。 例如：<br/> 輸入字串： "A&bc&&d"<br/> normal： "A<span class="underline">b</span>c&d"<br/> DT \_ NOPREFIX： "A&bc&&d"<br/>                                                                                                                                                            |
| DT \_ 路徑 \_ 省略號       | 以省略號取代文字中間的字元，讓結果符合顯示矩形。 如果字串包含反斜線 (\) 字元，DT \_ 路徑 \_ 省略號會盡可能保留最後一個反斜線之後的文字。 除非指定了 DT MODIFYSTRING 旗標，否則不會修改字串 \_ 。                                                                                                                                                                                                                                                                                                       |
| DT \_ PREFIXONLY           | 在 & 首碼字元後面的字元位置繪製底線。 一般來說， [**DrawThemeText**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemetext) 會將 & 解讀為指示詞，以將後面的字元加上底線，並將前置詞字元 && 為指示詞，以列印單一 &。 藉由指定 DT \_ PREFIXONLY，不會繪製任何字元，只會有底線。 空白字元會放置在通常會出現字元的位置。 例如： <br/> 輸入字串： "A&bc&&d"<br/> normal： "A<span class="underline">b</span>c&d<br/> DT \_ PREFIXONLY： " <span class="underline"></span> "<br/> |
| DT \_ 右邊                | 將文字對齊右邊。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| DT \_ RTLREADING           | 以由右至左的順序為雙向文字配置文字，例如希伯來文或阿拉伯文字型中的文字。 文字的預設方向是由左到右。                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| DT \_ 單行           | 在單一行上顯示文字。 換行字元和換行字元不會中斷。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| DT \_ TABSTOP              | 設定制表位。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| DT \_ TOP                  | 在顯示矩形的頂端呈現文字。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| DT \_ VCENTER              | 垂直對齊文字。 此值僅用於 DT \_ 單行值。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| DT \_ WORDBREAK            | 如果單字延伸超過顯示矩形的邊緣，則會在文字之間分隔線條。 換行字元/換行 (CR/LF) 序列也會中斷這一行。                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| DT \_ 文字 \_ 省略號       | 截斷不符合顯示矩形的任何單字，並加入省略號。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
