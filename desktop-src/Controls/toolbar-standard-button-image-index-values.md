## <a name="remarks"></a>備註

您可以使用這些值，在使用 [**TB \_ LOADIMAGES**](tb-loadimages.md) 訊息載入的標準映射清單中指定影像索引。 索引值會對應至通用控制項所使用之標準點陣圖內的影像。

下列每個標題都包含大型圖示的影像清單、小圖示的影像清單，以及將每個圖示的常數對應至其索引值的表格。
