# <a name="tb_getpressedimagelist-message"></a>TB \_ GETPRESSEDIMAGELIST 訊息

取得工具列控制項用來顯示處於已按下狀態之按鈕的影像清單。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>

必須為零。

</dd> <dt>

*lParam* 
</dt> <dd>

必須為零。

</dd> </dl>

## <a name="return-value"></a>傳回值

傳回影像清單的控制碼，如果未設定影像清單，則傳回 **Null** 。
