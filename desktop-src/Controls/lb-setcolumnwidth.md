# <a name="lb_setcolumnwidth-message"></a>LB \_ SETCOLUMNWIDTH 訊息

設定多資料行清單方塊中所有資料行的寬度（以圖元為單位）。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>

指定所有資料行的寬度（以圖元為單位）。

</dd> <dt>

*lParam* 
</dt> <dd>

不使用這個參數。

</dd> </dl>

## <a name="return-value"></a>傳回值

此訊息不會傳回值。
