# <a name="static-control-styles"></a>靜態控制項樣式

若要使用 [**CreateWindow**](/windows/desktop/api/winuser/nf-winuser-createwindowa) 或 [**CreateWindowEx**](/windows/desktop/api/winuser/nf-winuser-createwindowexa) 函式來建立靜態控制項，請指定靜態類別、適當的視窗樣式常數，以及下列靜態控制項樣式的組合。

| <span id="SS_CENTER"></span><span id="ss_center"></span><dl> <dt>**SS \_ CENTER**</dt> </dl>                            | 在矩形中將文字置中的簡單矩形。 文字會在顯示之前格式化。 延伸超過行結尾的單字會自動換行到下一個中央線的開頭。 長度超過控制項寬度的文字會被截斷。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |

## <a name="remarks"></a>備註

若要使用以多行文字的省略號來截斷單字或字串的樣式， [**DrawThemeText**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemetext) 或 [**DrawText**](/windows/desktop/api/winuser/nf-winuser-drawtext) 可以用於具有格式化旗標 **DT \_ WORDBREAK** 和 **dt \_ ENDELLIPSIS** 設定的擁有者繪製控制項。
