# <a name="ttm_settiptextcolor-message"></a>TTM \_ SETTIPTEXTCOLOR 訊息

設定工具提示視窗中的文字色彩。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>

新的文字色彩。

</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

未使用此訊息的傳回值。

## <a name="remarks"></a>備註

啟用視覺化樣式時，此訊息不會有任何作用。
