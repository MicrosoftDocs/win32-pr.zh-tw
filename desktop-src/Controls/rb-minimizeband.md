# <a name="rb_minimizeband-message"></a>RB \_ MINIMIZEBAND 訊息

將 Rebar 控制項中的寬線大小調整為最小的大小。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>

以零為基底的寬線索引，可將其最小化。

</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

未使用傳回值。

|-------------------------------------|---------------------------------------------------------------------------------------|

| 標頭<br/>                   | <dl> <dt>Commctrl。h</dt> </dl> |
