# <a name="hot-key"></a>快速鍵

本章節包含與快速鍵控制項搭配使用之程式設計項目的相關資訊。

### <a name="overviews"></a>概觀



| 主題                                    | 目錄                                                                                                                     |
|------------------------------------------|------------------------------------------------------------------------------------------------------------------------------|
| [快速鍵控制項](hot-key-controls.md) | 快速鍵控制項是一種視窗，可讓使用者輸入按鍵的組合，以做為快速鍵使用。<br/> |



 

### <a name="messages"></a>訊息



| 主題                                   | 目錄                                                                                                 |
|-----------------------------------------|----------------------------------------------------------------------------------------------------------|
| [**HKM \_ GETHOTKEY**](hkm-gethotkey.md) | 從熱鍵控制項取得快速鍵的虛擬按鍵碼和修飾詞旗標。<br/>             |
| [**HKM \_ SETHOTKEY**](hkm-sethotkey.md) | 設定快速鍵控制項的快速鍵組合。<br/>                                           |
| [**HKM \_ SETRULES**](hkm-setrules.md)   | 定義快速鍵控制項的無效組合和預設輔助按鍵組合。 <br/> |
