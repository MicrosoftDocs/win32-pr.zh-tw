# <a name="tbm_getthumblength-message"></a>TBM \_ GETTHUMBLENGTH 訊息

抓取捲軸中滑杆的長度。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>必須為零。</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

傳回滑杆的長度（以圖元為單位）。
