## <a name="remarks"></a>備註

> [!Note]  
> Comctl32.dll 第6版不是可轉散發套件，但包含在 Windows 中。 若要使用 Comctl32.dll 第6版，請在資訊清單中指定它。 如需資訊清單的詳細資訊，請參閱 [啟用視覺化樣式](cookbook-overview.md)。

 

當滑鼠停留在專案上方時，專案會變成 *熱* 。 如果啟用熱追蹤，則會反白顯示熱專案，但是使用者仍然必須按一下專案至少一次，才能選取該專案。 *啟用* 專案會使清單視圖控制項傳送 [LVN \_ ITEMACTI加值稅E](lvn-itemactivate.md) 通知。 當專案處於選取狀態時，就 *可以選取專案* 。

如果已設定 **lvs) \_ EX \_ ONECLICKACTI加值稅E** 樣式，則只需要按一下即可選取任何專案，因此可能會選取所有專案。 如果已設定 **lvs) \_ EX \_ TWOCLICKACTI加值稅E** 樣式，則必須按兩下滑鼠才能選取專案。 使用此樣式時，只有在按一下專案之後，才能選取該專案。

設定 **lvs) \_ EX \_ UNDERLINEHOT** 樣式會導致任何可以選取的熱專案的文字加上底線。 設定 **lvs) \_ EX \_ UNDERLINECOLD** 樣式會使非經常性存取（但可以選取）的專案文字加上底線。
