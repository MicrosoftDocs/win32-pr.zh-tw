# <a name="tb_gettextrows-message"></a>TB \_ GETTEXTROWS 訊息

抓取工具列按鈕上可顯示的文字資料列數目上限。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>必須為零。</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

傳回 INT 值，代表控制項將為按鈕顯示的文字資料列數目上限。
