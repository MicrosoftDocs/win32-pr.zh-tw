# <a name="tb_getstyle-message"></a>TB \_ GETSTYLE 訊息

抓取工具列控制項目前使用的樣式。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>必須為零。</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

傳回以 [工具列控制項樣式](toolbar-control-and-button-styles.md)組合的 **DWORD** 值。
