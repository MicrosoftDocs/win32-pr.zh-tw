# <a name="rb_deleteband-message"></a>RB \_ DELETEBAND 訊息

從 Rebar 控制項刪除頻外。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>

要刪除之帶狀線的以零為基底的索引。

</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

如果成功，則傳回非零，否則傳回零。
