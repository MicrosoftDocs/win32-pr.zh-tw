# <a name="tb_getimagelistcount-message"></a>TB \_ GETIMAGELISTCOUNT 訊息

取得與工具列相關聯的影像清單數目。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>必須為零。</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

傳回影像清單的數目。
