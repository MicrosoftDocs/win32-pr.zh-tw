# <a name="tb_getrows-message"></a>TB \_ GETROWS 訊息

使用 [**TBSTYLE \_ WRAPABLE**](toolbar-control-and-button-styles.md) 樣式來抓取工具列中的按鈕列數。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>必須為零。</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

傳回資料列的數目。
