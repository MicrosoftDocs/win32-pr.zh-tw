# <a name="rb_getbandcount-message"></a>RB \_ GETBANDCOUNT 訊息

抓取目前在 Rebar 控制項中的條紋計數。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>必須為零。</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

傳回代表指派給控制項之條紋數目的 **UINT** 值。
