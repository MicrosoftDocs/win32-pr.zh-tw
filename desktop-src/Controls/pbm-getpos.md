# <a name="pbm_getpos-message"></a>PBM \_ GETPOS 訊息

抓取進度列的目前位置。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>必須為零。</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

傳回代表進度列目前位置的 **UINT** 值。
