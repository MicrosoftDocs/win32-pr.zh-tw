# <a name="cbem_getextendedstyle-message"></a>CBEM \_ GETEXTENDEDSTYLE 訊息

取得用於 ComboBoxEx 控制項的延伸樣式。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>必須為零。</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

傳回 DWORD 值，包含用於控制項的 [ComboBoxEx 控制項延伸樣式](comboboxex-control-extended-styles.md) 。
