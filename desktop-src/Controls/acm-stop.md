# <a name="acm_stop-message"></a>未通過的 \_ 停止訊息

停止在動畫控制項中播放 AVI 剪輯。 您可以明確地傳送此訊息，或使用 [**動畫 \_ 停**](/windows/desktop/api/Commctrl/nf-commctrl-animate_stop) 用宏來傳送。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>必須為零。</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

如果成功，則傳回非零，否則傳回零。
