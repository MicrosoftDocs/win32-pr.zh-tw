# <a name="ipm_isblank-message"></a>IPM \_ ISBLANK 訊息

判斷 IP 位址控制項中的所有欄位是否為空白。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>必須為零。</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

如果所有欄位都是空白，則會傳回非零，否則會傳回零。
