# <a name="tb_setstyle-message"></a>TB \_ >setstyle 訊息

設定工具列控制項的樣式。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>必須為零。</dd> <dt>

*lParam* 
</dt> <dd>

值，指定要為控制項設定的樣式。 這個值可以是 [工具列控制項樣式](toolbar-control-and-button-styles.md)的組合。

</dd> </dl>

## <a name="return-value"></a>傳回值

沒有傳回值。
