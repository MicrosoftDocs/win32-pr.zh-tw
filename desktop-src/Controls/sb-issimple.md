# <a name="sb_issimple-message"></a>SB \_ ISSIMPLE 訊息

檢查狀態列控制項是否為簡單模式。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>必須為零。</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

如果狀態列控制項處於簡單模式，則會傳回非零，否則會傳回零。
