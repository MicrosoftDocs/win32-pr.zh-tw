# <a name="tb_setindent-message"></a>TB \_ SETINDENT 訊息

設定工具列控制項中第一個按鈕的縮排。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>

指定縮排的值（以圖元為單位）。

</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

如果成功，則傳回非零，否則傳回零。
