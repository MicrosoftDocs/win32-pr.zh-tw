# <a name="ttm_gettiptextcolor-message"></a>TTM \_ GETTIPTEXTCOLOR 訊息

抓取工具提示視窗中的文字色彩。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>必須為零。</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

傳回表示文字色彩的 **COLORREF** 值。
