# <a name="udm_setbuddy-message"></a>UDM \_ SETBUDDY 訊息

設定上下按鈕控制項的合作者視窗。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>

新好友視窗的控制碼。

</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

傳回值是上一個好友視窗的控制碼。
