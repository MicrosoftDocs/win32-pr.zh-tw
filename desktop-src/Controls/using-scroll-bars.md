# <a name="using-scroll-bars"></a>使用捲軸

本章節包含的主題會示範如何建立捲軸。

## <a name="in-this-section"></a>本節內容



| 主題                                                                                                                              | 描述                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
|------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [如何建立捲軸](create-scroll-bars.md)<br/>                                                                     | 建立重迭、快顯視窗或子視窗時，您可以使用 [**CreateWindowEx**](/windows/desktop/api/winuser/nf-winuser-createwindowexa) 函式，並指定 ws \_ HSCROLL、WS \_ VSCROLL 或這兩種樣式來新增標準捲軸。 <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| [如何滾動文字](scroll-text-in-scroll-bars.md)<br/>                                                                    | 本節說明您可以對應用程式的主視窗程式進行的變更，讓使用者能夠滾動文字。 本節中的範例會建立並顯示文字字串的陣列，並處理 [**wm \_ HSCROLL**](wm-hscroll.md) 和 [**WM \_ VSCROLL**](wm-vscroll.md) 捲軸訊息，讓使用者可以垂直和水準滾動文字。 <br/>                                                                                                                                                                                                                                                                 |
| [如何滾動點陣圖](scroll-a-bitmap-in-scroll-bars.md)<br/>                                                            | 本節說明您可以對應用程式的主視窗程式進行的變更，讓使用者能夠滾動點陣圖。 <br/> 此範例包含一個功能表項目，可將螢幕內容複寫到點陣圖，並在工作區中顯示點陣圖。 此範例也會處理由捲軸所產生的 [**wm \_ HSCROLL**](wm-hscroll.md) 和 [**wm \_ VSCROLL**](wm-vscroll.md) 訊息，讓使用者能夠以水準和垂直方式來移動點陣圖。 不同于滾動文字的範例，bitmap 範例會使用 [**BitBlt**](/windows/desktop/api/wingdi/nf-wingdi-bitblt) 函式來繪製工作區的無效部分。 <br/> |
| [如何建立標準捲軸的鍵盤介面](create-a-keyboard-interface-for-standard-scroll-bars.md)<br/> | 雖然捲軸控制項提供內建的鍵盤介面，但標準捲軸不是。 若要執行標準捲軸的鍵盤介面，視窗程式必須處理 [**WM 的 \_ KEYDOWN**](/windows/desktop/inputdev/wm-keydown) 訊息，並檢查 *wParam* 參數所指定的虛擬機器碼程式碼。 如果虛擬按鍵程式碼對應到箭號鍵，則視窗程式會將一組 [**wm \_ HSCROLL**](wm-hscroll.md) 或 [**wm \_ VSCROLL**](wm-vscroll.md) 訊息傳送給自己，並將 *wParam* 參數的低序位字組設定為適當的捲軸要求碼。 <br/>                                              |
