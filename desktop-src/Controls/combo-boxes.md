# <a name="combo-box-windows-controls"></a>下拉式方塊 (Windows 控制項) 

本章節包含與下拉式方塊搭配使用之程式設計項目的相關資訊。 下拉式方塊是 *下拉式* 方塊類別所定義的唯一控制項類型，結合了清單方塊和編輯控制項的大部分功能。
