# <a name="tb_autosize-message"></a>TB 自動 \_ 調整訊息

使工具列調整大小。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>必須為零。</dd> <dt>

*lParam* 
</dt> <dd>必須為零。</dd> </dl>

## <a name="return-value"></a>傳回值

沒有傳回值。

## <a name="remarks"></a>備註

應用程式會在透過設定按鈕或點陣圖大小或第一次加入字串，而導致工具列的大小變更時，傳送 TB 的自動 **\_ 調整** 訊息。
