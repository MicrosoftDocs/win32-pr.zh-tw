# <a name="toolbar-control-and-button-styles"></a>Toolbar 控制項和按鈕樣式

下列視窗樣式是工具列特有的樣式。 當建立工具列時，它們會與其他視窗樣式合併。

**注意** 針對通用控制項 [版本 6.00](common-control-versions.md)，如果 [視覺效果樣式](themes-overview.md) 與工具列搭配使用，不論樣式設定為何，按鈕一律是透明的。 否則，透明行為是正常的，如使用 TBSTYLE 的 \_ 平面或 TBSTYLE \_ 透明樣式所表示。

> [!Note]  
> Comctl32.dll 第6版不是可轉散發套件，但包含在 Windows 中。 若要使用 Comctl32.dll 第6版，請在資訊清單中指定它。 如需資訊清單的詳細資訊，請參閱 [啟用視覺化樣式](cookbook-overview.md)。

**注意：** 工具列按鈕可以有下列樣式的組合。 若要指定按鈕樣式，請在按鈕 [**TBBUTTON**](/windows/desktop/api/Commctrl/ns-commctrl-tbbutton)結構的 **fsStyle** 成員中設定適當的旗標。 並非所有樣式都可以合併。

針對 Shell 版本4.72 及更早版本，視窗和按鈕樣式旗標的格式為 TBSTYLE \_ XXX。 如果您要編譯 [4.72 版](common-control-versions.md) 或更早版本為 Commctrl 的應用程式，則必須使用 TBSTYLE \_ XXX 按鈕樣式旗標。 針對5.80 版和更新版本，所有按鈕樣式的格式都是 BTNS \_ XXX。 所有的 TBSTYLE \_ xxx 值都有相等 \_ 的 BTNS xxx 值，具有相同的意義和數值。 為了方便起見，下表提供這兩種表單。
