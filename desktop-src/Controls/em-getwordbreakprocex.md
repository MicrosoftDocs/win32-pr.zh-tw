# <a name="em_getwordbreakprocex-message"></a>EM \_ GETWORDBREAKPROCEX 訊息

抓取 rich edit 控制項目前註冊的延伸斷字程式位址。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>

未使用;必須為零。

</dd> <dt>

*lParam* 
</dt> <dd>

未使用;必須為零。

</dd> </dl>

## <a name="return-value"></a>傳回值

訊息會傳回目前程式的位址。
