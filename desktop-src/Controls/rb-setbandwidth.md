# <a name="rb_setbandwidth-message"></a>RB \_ SETBANDWIDTH 訊息

設定停駐區的寬度。

## <a name="parameters"></a>參數

<dl> <dt>

*wParam* 
</dt> <dd>

寬線的索引。

</dd> <dt>

*lParam* 
</dt> <dd>

新的寬度（以圖元為單位）。

</dd> </dl>

## <a name="return-value"></a>傳回值

如果已設定值，則傳回 **TRUE** ，否則傳回 **FALSE** 。
