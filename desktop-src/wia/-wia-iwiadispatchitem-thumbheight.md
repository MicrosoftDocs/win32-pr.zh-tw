# <a name="itemthumbheight-property"></a>ThumbHeight 屬性

縮圖影像的高度（以圖元為單位）。 如果這個專案不支援縮圖，這個屬性會傳回-1。 唯讀。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = Item.ThumbHeight
```



## <a name="property-value"></a>屬性值

接收縮圖高度的變數。
