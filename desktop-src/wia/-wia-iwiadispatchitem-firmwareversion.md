# <a name="itemfirmwareversion-property"></a>FirmwareVersion 屬性

捕獲裝置的固件版本。 這個屬性只適用于) 的裝置 (根專案類型專案。 唯讀。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = Item.FirmwareVersion
```



## <a name="property-value"></a>屬性值

接收固件版本的字串。
