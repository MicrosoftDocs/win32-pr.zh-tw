# <a name="using-wia-video"></a>使用 WIA 影片

WIA 提供 [**IWiaVideo**](/windows/desktop/api/Wiavideo/nn-wiavideo-iwiavideo) 介面，可讓應用程式在視窗中顯示串流影片、從影片串流捕捉靜止影像，並將這些影像儲存為 JPEG 檔案。

下列各節說明如何使用 Windows 映像取得 (WIA) 影片功能：

-   [預覽影片](-wia-previewing-video.md)
-   [從影片串流捕捉靜止影像](-wia-capturing-a-still-image-from-a-video-stream.md)

> [!Note]  
> WIA 不支援 Windows Server 2003、Windows Vista 或更新版本中的影片裝置。 針對這些版本的 Windows，請使用 [DirectShow](/previous-versions//ms783323(v=vs.85)) 從影片取得影像。
