# <a name="property-attributes-wia"></a> (WIA) 的屬性屬性

所有專案物件都有屬性。 屬性有屬性。 例如，屬性屬性會指出屬性是讀取、寫入或刪除。 它們也會指出有效的屬性值。 下列常數是有效的屬性屬性： 

| Property 屬性        | 意義                                                                                                  |
|---------------------------|----------------------------------------------------------------------------------------------------------|
| WIA 的可快取 \_ \_      | 裝置可以快取屬性的值。                                                               |
| WIA 的 [ \_ \_ 識別碼]           | 屬性具有合法旗標值的清單。 旗標值會使用位 **or** 運算來合併。 |
| WIA \_ 的 \_ 清單           | 屬性具有合法值的清單。                                                                 |
| WIA \_ \_ 內容無           | 屬性沒有任何與其相關聯的有效值。                                          |
| WIA \_ 的 \_ 範圍          | 屬性具有有效值的範圍。                                                                |
| 已 \_ 讀取的 WIA \_           | 應用程式可以讀取屬性的值。                                                           |
| WIA \_ 和 \_ RW             | 應用程式可以讀取和寫入屬性的值。                                                 |
| 需要 WIA 參數 \_ \_ 同步 \_ | 請勿使用。                                                                                              |
| WIA \_ \_ 目錄寫入          | 應用程式可以寫入屬性的值。                                                          |
