# <a name="itemname-property"></a>Item.Name 屬性

顯示在 UI 中的專案名稱。 唯讀。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = Item.Name
```



## <a name="property-value"></a>屬性值

接收名稱的字串。
