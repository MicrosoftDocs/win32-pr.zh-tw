# <a name="syntax-conventions"></a>語法慣例

下表列出 Windows 映像取得 (WIA) 腳本模型參考的語法語句中所使用的詞彙。 

| 詞彙                      | 描述                                                    |
|---------------------------|----------------------------------------------------------------|
| *檔案名稱*                | 檔案的路徑和名稱。                                   |
| *objDeviceInfo*           | [**DeviceInfo**](-wia-deviceinfo.md)物件。                |
| *objDeviceInfoCollection* | [**DeviceInfo**](-wia-deviceinfo.md)物件的集合。 |
| *objItem*                 | [**專案**](-wia-item.md)物件。                           |
| *objItemCollection*       | [**專案**](-wia-item.md)物件的集合。             |
| *objWia*                  | [**Wia**](-wia-wia.md)物件。                              |
| *PropID*                  | WIA 屬性常數。                                       |
| *無 功*                     | **變異** 值。                                           |
