# <a name="sti-compatibility"></a>STI 相容性

Windows 影像取得 (WIA) 架構是以 (STI) 的靜止映射架構所建立的基礎為基礎。 WIA 設備磁碟機將與 STI 的使用者模式驅動程式相容， (USD) 模型。 雖然原始的 USD 目的是要支援 TWAIN 資料來源和其他 Api，但 WIA 驅動程式支援新的 WIA 應用程式介面。
