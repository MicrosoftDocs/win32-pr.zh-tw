# <a name="itemfullname-property"></a>專案 FullName 屬性

抓取專案顯示在 UI 中的完整名稱。 唯讀。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = Item.FullName
```



## <a name="property-value"></a>屬性值

接收名稱的字串。
