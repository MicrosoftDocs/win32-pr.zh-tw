# <a name="deviceinfomanufacturer-property"></a>DeviceInfo。製造商屬性

抓取此硬體裝置的製造商名稱。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = DeviceInfo.Manufacturer
```



## <a name="property-value"></a>屬性值

接收名稱的字串。
