# <a name="deviceinfoname-property"></a>DeviceInfo.Name 屬性

Windows 映像取得 (WIA) 硬體裝置顯示在 UI 中的名稱。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = DeviceInfo.Name
```



## <a name="property-value"></a>屬性值

接收名稱的字串。
