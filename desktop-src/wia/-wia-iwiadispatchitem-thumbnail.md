# <a name="itemthumbnail-property"></a>專案。縮圖屬性

縮圖影像的路徑和檔案名。 如果專案不支援縮圖，或如果無法建立路徑，則此屬性為 **Null** 。 唯讀。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = Item.Thumbnail
```



## <a name="property-value"></a>屬性值

接收路徑和檔案名的字串。
