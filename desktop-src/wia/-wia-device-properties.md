# <a name="device-properties-wia"></a>裝置屬性 (WIA) 

裝置屬性是控制整部裝置之功能的屬性。 這些屬性只能透過代表裝置之樹狀結構的根專案來存取。 裝置屬性有三種類型：通用、掃描器和相機。

-   [**攝影機裝置屬性常數**](-wia-wiaitempropcameradevice.md)
-   [**掃描器裝置屬性常數**](-wia-wiaitempropscannerdevice.md)
