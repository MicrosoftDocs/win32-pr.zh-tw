# <a name="deviceinfoid-property"></a>DeviceInfo.Id 屬性

抓取 Windows 的映射取得識別碼 (WIA) 硬體裝置。 唯讀。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = DeviceInfo.Id
```



## <a name="property-value"></a>屬性值

接收識別碼的字串。
