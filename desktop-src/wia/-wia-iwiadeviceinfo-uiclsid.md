# <a name="deviceinfouiclsid-property"></a>DeviceInfo. UIClsid 屬性

針對此 Windows 映像取得 (WIA) 硬體裝置，抓取製造商提供之使用者介面的類別識別碼。 值是 GUID 的字串表示。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = DeviceInfo.UIClsid
```



## <a name="property-value"></a>屬性值

要接收類別識別碼的字串。
