# <a name="item-properties"></a>項目屬性

專案屬性是控制 Windows 映像取得 (WIA) 專案功能的屬性。 這些屬性是透過樹狀結構中代表裝置的任何專案來存取。 專案屬性有三種類型：通用、掃描器和相機。

如需這些屬性的參考資訊，請參閱：

-   [**常見的 WIA 專案屬性常數**](-wia-wiaitempropcommonitem.md)
-   [**攝影機 WIA 專案屬性常數**](-wia-wiaitempropcameraitem.md)
-   [**掃描器 WIA 專案屬性常數**](-wia-wiaitempropscanneritem.md)
