# <a name="itemheight-property"></a>Item. Height 屬性

專案的高度（以圖元為單位）。 唯讀。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = Item.Height
```



## <a name="property-value"></a>屬性值

接收高度的變數。

## <a name="remarks"></a>備註

如果專案是數位相機， **height** 屬性代表此攝影機所產生影像的高度。 如果專案是掃描器，這個屬性代表掃描床的高度。 如果專案為影像，則此屬性代表影像的實際高度。
