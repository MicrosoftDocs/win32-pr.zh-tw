# <a name="itemthumbwidth-property"></a>ThumbWidth 屬性

縮圖影像的寬度（以圖元為單位）。 如果這個專案不支援縮圖，這個屬性會傳回-1。 唯讀。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = Item.ThumbWidth
```



## <a name="property-value"></a>屬性值

接收寬度的變數。
