# <a name="wia-device-manager"></a>WIA 裝置管理員

Windows 的影像取得 (WIA) 裝置管理員是靜止映射 (事件監視器的延伸。 WIA 裝置管理員提供下列各項的物件、方法和介面：

-   安裝裝置
-   列舉裝置
-   查詢已安裝裝置的屬性
-   建立裝置物件
-   監視裝置事件
-   取得映射
-   註冊目的地應用程式。
