# <a name="iwiatransfercancel-method"></a>IWiaTransfer：： Cancel 方法

取消目前的傳送作業。

## <a name="syntax"></a>語法


```C++
HRESULT Cancel();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **HRESULT**

如果這個方法成功，它會傳回 **S \_ OK**。 否則，它會傳回 **HRESULT** 錯誤碼。
