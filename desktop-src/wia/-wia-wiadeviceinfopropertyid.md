# <a name="wiadeviceinfopropertyid"></a>WiaDeviceInfoPropertyId

裝置資訊屬性是描述裝置設定和安裝的屬性。 這些屬性可透過 [**IWiaDevMgr**](/windows/desktop/api/wia_xp/nn-wia_xp-iwiadevmgr) 介面，也可以透過根項目取得。

裝置資訊屬性在 \_ \_ C/c + + 中使用時，前面會加上 "WIA DIP" (裝置資訊屬性) ，而且 Windows 影像取得 (WIA) 提供。 針對腳本用途，這些常數會使用前置詞 "DeviceInfo"，而且是 WiaDeviceInfoPropertyId 列舉型別的一部分。 如需詳細資訊，請參閱 [**裝置資訊屬性常數**](-wia-wiadeviceinfoprop.md) 。
