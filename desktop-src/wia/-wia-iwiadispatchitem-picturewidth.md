# <a name="itempicturewidth-property"></a>PictureWidth 屬性

此數位相機所產生影像的寬度（以圖元為單位）。 僅適用于數位相機。 唯讀。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```JScript
propVal = Item.PictureWidth
```



## <a name="property-value"></a>屬性值

接收寬度的變數。
