# <a name="windows-networking-callbacks"></a>Windows網路回呼

Windows 的網路功能使用下列函式原型。



| 函式原型                                 | 描述                                                                                                           |
|----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**PFNGETPROFILEPATH**](/previous-versions/windows/desktop/legacy/aa385358(v=vs.85))     | 取得使用者設定檔路徑的函式原型。                                                            |
| [**PFNPROCESSPOLICIES**](/previous-versions/windows/desktop/legacy/aa385360(v=vs.85))   | 函式的原型，此函式會傳回與特定 Windows 進程相關聯的使用者帳戶原則資訊。 |
| [**PFNRECONCILEPROFILE**](/previous-versions/windows/desktop/legacy/aa385364(v=vs.85)) | 將本機設定檔與對應的全域設定檔協調的函式原型。                        |
