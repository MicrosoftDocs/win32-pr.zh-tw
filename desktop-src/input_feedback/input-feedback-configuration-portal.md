# <a name="input-feedback-configuration"></a>輸入意見設定

## <a name="purpose"></a>目的

本節中的主題提供 Windows 8 中輸入意見反應設定的總覽。 輸入回饋設定可讓您存取視覺效果的意見反應設定以進行使用者互動。

此處提供的主題集合說明如何使用輸入意見反應設定來優化應用程式的使用者體驗。 例如，在遊戲應用程式中，可能不需要或不想要使用預設的視覺效果回饋設定，因為干擾或與遊戲本身發生衝突。

輸入意見反應 ConfigurationAPIs 是針對建立 UI 架構的開發人員所設計，可在桌面應用程式之間提供一致的觸控優化使用者體驗。
