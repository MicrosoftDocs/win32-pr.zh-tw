# <a name="ui_pkey_morecolorslabel"></a>UI \_ PKEY \_ MoreColorsLabel

識別 UI \_ PKEY \_ MoreColorsLabel 屬性。

```
propertyDescription
   name = UI_PKEY_MoreColorsLabel
   shellPKey = UI_PKEY_MoreColorsLabel
   formatID = 00000408-7363-696e-8441798acf5aebb7
   propID = 408
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>備註

\_ \_ 應用程式會使用 UI PKEY MoreColorsLabel 來查詢與 [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md)的 [**更多色彩**] 按鈕相關聯的標籤值。

UI \_ PKEY \_ MoreColorsLabel 只適用于 [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md) ，其中 `ThemeColors` 或 `StandardColors` 指定為 **ColorTemplate** 屬性的值。
