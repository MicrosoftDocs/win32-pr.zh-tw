# <a name="ui_pkey_enabled"></a>UI \_ PKEY \_ 已啟用

識別 \_ 已啟用 UI PKEY 的 \_ 屬性。

```
propertyDescription
   name = UI_PKEY_Enabled
   shellPKey = UI_PKEY_Enabled
   formatID = 00000004-7363-696e-8441798acf5aebb7
   propID = 1
   typeInfo
      type = VT_BOOL
   booleanFormat
      formatAs = VARIANT_TRUE=-1, VARIANT_FALSE=0
```

## <a name="remarks"></a>備註

\_ \_ 應用程式會使用啟用的 UI PKEY 來查詢控制項是否已啟用，而且能夠回應使用者互動。

> [!Note]  
> 當應用程式模式變更時，此值不會自動設為0，而且會將命令的可見度切換為關閉。
