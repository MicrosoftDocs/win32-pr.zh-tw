# <a name="ui_pkey_standardcolorstooltips"></a>UI \_ PKEY \_ StandardColorsTooltips

識別 UI \_ PKEY \_ StandardColorsTooltips 屬性。

```
propertyDescription
   name = UI_PKEY_StandardColorsTooltips
   shellPKey = UI_PKEY_StandardColorsTooltips
   formatID = 00000412-7363-696e-8441798acf5aebb7
   propID = 412
   typeInfo
      type = VT_VECTOR | VT_LPWSTR
```

## <a name="remarks"></a>備註

\_ \_ 應用程式會使用 UI PKEY StandardColorsTooltips 來查詢 [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md)的色彩樣本工具提示。

屬性值是字串值的陣列。

無論針對 **ColorTemplate** 屬性指定的值為何，每個字串值都會對應至 [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md)中的色彩樣本的工具提示。
