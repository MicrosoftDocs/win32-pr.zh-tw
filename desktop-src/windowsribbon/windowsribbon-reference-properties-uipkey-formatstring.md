# <a name="ui_pkey_formatstring"></a>UI \_ PKEY \_ 格式格式

識別 UI \_ PKEY \_ 格式字串屬性。

```
propertyDescription
   name = UI_PKEY_FormatString
   shellPKey = UI_PKEY_FormatString
   formatID = 00000207-7363-696e-8441798acf5aebb7
   propID = 207
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>備註

UI \_ PKEY 格式 \_ 字串是由應用程式用來查詢 [微調](windowsribbon-controls-spinner.md) 控制項值的格式 (或測量單位) 。
