# <a name="ui_pkey_decimalvalue"></a>UI \_ PKEY \_ DecimalValue

識別 UI \_ PKEY \_ DecimalValue 屬性。

```
propertyDescription
   name = UI_PKEY_DecimalValue
   shellPKey = UI_PKEY_DecimalValue
   formatID = 00000201-7363-696e-8441798acf5aebb7
   propID = 201
   typeInfo
      type = VT_DECIMAL
```

## <a name="remarks"></a>備註

\_ \_ 應用程式會使用 UI PKEY DecimalValue 來查詢[微調](windowsribbon-controls-spinner.md)控制項的 [編輯] 欄位中的值。
