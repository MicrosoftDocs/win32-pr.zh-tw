# <a name="ui_pkey_representativestring"></a>UI \_ PKEY \_ RepresentativeString

識別 UI \_ PKEY \_ RepresentativeString 屬性。

```
propertyDescription
   name = UI_PKEY_RepresentativeString
   shellPKey = UI_PKEY_RepresentativeString
   formatID = 00000208-7363-696e-8441798acf5aebb7
   propID = 208
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>備註

\_ \_ 應用程式會使用 UI PKEY RepresentativeString 來查詢[微調](windowsribbon-controls-spinner.md)控制項的編輯欄位長度。

> [!Note]  
> \_ \_ 架構會使用 UI PKEY RepresentativeString 的值來計算[微調](windowsribbon-controls-spinner.md)控制項的寬度，而且永遠不會顯示。
