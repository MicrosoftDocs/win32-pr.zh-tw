# <a name="windows-ribbon-framework-glossary"></a>Windows 功能區架構詞彙

## <a name="g"></a>G

<dl> <dt>

**庫**
</dt> <dd>

豐富、可自訂的清單方塊，可用來依類別目錄來組織專案、將它們顯示在彈性的資料行和資料列型版面配置中，並以影像和文字來表示它們。 根據資源庫的類型，也支援即時預覽。

</dd> </dl>

## <a name="m"></a>M

<dl> <dt>

**浮動工具列**
</dt> <dd>

一組功能區命令，範圍為目前的內容，並顯示在指標附近的矩形容器中。

</dd> </dl>

## <a name="q"></a>Q

<dl> <dt>

**快速存取工具列 (QAT)**
</dt> <dd>

功能區 UI 上的小型可自訂工具列，可讓您按一下即可存取使用者選取的命令。

</dd> </dl>

## <a name="r"></a>R

<dl> <dt>

**功能區標記**
</dt> <dd>

標準的 XML 宣告式標記，指定功能區 UI 的命令、視覺效果資源、控制項版面配置和縮放行為。

</dd> </dl>

## <a name="t"></a>T

<dl> <dt>

**範本**
</dt> <dd>

一組版面配置提示，根據功能區大小指定功能區控制項的大小和組織。

</dd> </dl>

## <a name="v"></a>V

<dl> <dt>

**檢視**
</dt> <dd>

裝載 Windows 功能區 UI 控制項的最上層容器。

</dd> </dl>

## <a name="w"></a>W

<dl> <dt>

**Windows 功能區架構 (功能區)**
</dt> <dd>

豐富的命令呈現系統，可透過應用程式視窗頂端的命令列公開應用程式的主要功能。

</dd> </dl>
