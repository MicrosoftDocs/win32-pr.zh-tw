# <a name="constants-and-enumerations"></a>常數和列舉

Windows 功能區架構常數和列舉的參考檔集。

### <a name="constants-and-enumerations"></a>常數和列舉




| 常數 | 描述 | 
|----------|-------------|
| <a href="/windows/desktop/windowsribbon/windowsribbon-ui-all-commands"><strong>UI_ALL_COMMANDS</strong></a><br /> | 指定常數，識別標記資源檔中所宣告的命令集合。<br /> | 
| <a href="/windows/desktop/windowsribbon/windowsribbon-ui-collection-invalidindex"><strong>UI_COLLECTION_INVALIDINDEX</strong></a><br /> | 指定常數，識別集合中的無效索引。<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_collectionchange"><strong>UI_COLLECTIONCHANGE</strong></a><br /> | 指定可識別可對集合進行之變更類型的值。<br /> | 
| <a href="/previous-versions/windows/desktop/legacy/hh448736(v=vs.85)"><strong>UI_COMMANDCATEGORY</strong></a><br /> | 指定值，指出是否已在編譯時期或執行時間定義命令。<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_commandtype"><strong>UI_COMMANDTYPE</strong></a><br /> | 指定值，以識別與功能區控制項相關聯的命令類型。<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_contextavailability"><strong>UI_CONTEXTAVAILABILITY</strong></a><br /> | 指定識別內容索引標籤可用性的<a href="windowsribbon-element-ribbon-contextualtabs.md"><strong>值。</strong></a><br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_controldock"><strong>UI_CONTROLDOCK</strong></a><br /> | 指定值，這個值會識別快速存取工具列 (QAT) 的固定狀態。 <br /> | 
| <a href="/windows/desktop/api/Uiribbon/ne-uiribbon-ui_eventlocation"><strong>UI_EVENTLOCATION</strong></a><br /> | 識別與功能區控制項相關聯之事件的可能來源位置。<br /> | 
| <a href="/windows/desktop/api/Uiribbon/ne-uiribbon-ui_eventtype"><strong>UI_EVENTTYPE</strong></a><br /> | 識別與 <a href="windowsribbon-element-ribbon.md"><strong>功能區</strong></a>相關聯的事件種類。<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_executionverb"><strong>UI_EXECUTIONVERB</strong></a><br /> | 指定值，這個值會識別對應至使用者可在 <a href="windowsribbon-element-command.md"><strong>命令</strong></a>上起始之動作的執行識別碼。 <br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_fontdeltasize"><strong>UI_FONTDELTASIZE</strong></a><br /> | 指定值，指出反白顯示之文字執行的字型大小是否應遞增或遞減。<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_fontproperties"><strong>UI_FONTPROPERTIES</strong></a><br /> | 指定識別 <a href="windowsribbon-element-fontcontrol.md"><strong>FontControl</strong></a>之字型屬性狀態的值，例如 <strong>刪除線</strong>。<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_fontunderline"><strong>UI_FONTUNDERLINE</strong></a><br /> | 指定識別 <a href="windowsribbon-element-fontcontrol.md"><strong>FontControl</strong></a>之底線狀態的值。<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_fontverticalposition"><strong>UI_FONTVERTICALPOSITION</strong></a><br /> | 指定識別 <a href="windowsribbon-element-fontcontrol.md"><strong>FontControl</strong></a>垂直對齊狀態的值。<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_invalidations"><strong>UI_INVALIDATIONS</strong></a><br /> | 指定值，以識別命令的外觀使其失效。<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_ownership"><strong>UI_OWNERSHIP</strong></a><br /> | 指定值，這個值會識別功能區架構用來建立影像的擁有權條件。<br /> | 
| <a href="/windows/desktop/api/Uiribbon/ne-uiribbon-ui_swatchcolormode"><strong>UI_SWATCHCOLORMODE</strong></a><br /> | 指定樣本是否具有一般或單色模式。<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_swatchcolortype"><strong>UI_SWATCHCOLORTYPE</strong></a><br /> | 指定值，用來識別 <a href="windowsribbon-element-dropdowncolorpicker.md"><strong>DropDownColorPicker</strong></a> 或 <a href="windowsribbon-element-fontcontrol.md"><strong>FontControl</strong></a> 色彩選擇器中的色彩樣本如何 (的 <strong>文字色彩</strong> 或 <strong>文字反白顯示</strong>) 填滿。<br /><blockquote>[!Note]<br />這些只是建議。 應用程式可以將任何填滿類型與這些值產生關聯。</blockquote><br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_viewtype"><strong>UI_VIEWTYPE</strong></a><br /> | 指定可識別功能區架構視圖的值。<br /> | 
| <a href="/windows/desktop/api/uiribbon/ne-uiribbon-ui_viewverb"><strong>UI_VIEWVERB</strong></a><br /> | 指定值，識別要在功能區架構視圖上完成的動作類型。<br /> |
