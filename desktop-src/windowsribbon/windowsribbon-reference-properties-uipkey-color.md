# <a name="ui_pkey_color"></a>UI \_ PKEY \_ 色彩

識別 UI \_ PKEY \_ 色彩屬性。

```
propertyDescription
   name = UI_PKEY_Color
   shellPKey = UI_PKEY_Color
   formatID = 00000400-7363-696e-8441798acf5aebb7
   propID = 400
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>備註

UI \_ PKEY \_ 色彩是由應用程式用來查詢 [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md) 控制項的色彩值。

屬性值是 [COLORREF](../gdi/colorref.md) 值。

[COLORREF](../gdi/colorref.md)值的高序位位元組會被忽略。
