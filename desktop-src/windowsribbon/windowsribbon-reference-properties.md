# <a name="property-keys"></a>屬性索引鍵

Windows 功能區架構屬性索引鍵的參考檔，這是 **PROPERTYKEY** 和 **PROPVARIANT** 索引鍵/值組的集合，可用來描述功能區架構命令的狀態。

> [!Note]  
> 每個命令的集合中都必須有唯一的索引鍵。

 

## <a name="property-types"></a>屬性類型

[集合屬性](windowsribbon-reference-properties-collection.md)

功能區架構集合型控制項的屬性索引鍵參考檔。

[色彩選擇器屬性](windowsribbon-reference-properties-colorpicker.md)

功能區架構 [下拉式清單色彩選擇器](windowsribbon-controls-dropdowncolorpicker.md)屬性索引鍵的參考檔。

[字型控制項屬性](windowsribbon-reference-properties-fontcontrol.md)

功能區架構 [字型控制項](windowsribbon-controls-fontcontrol.md)的屬性索引鍵參考檔。

[架構屬性](windowsribbon-reference-properties-framework.md)

功能區架構之全域屬性索引鍵的參考檔。

[資源屬性](windowsribbon-reference-properties-resource.md)

功能區架構資源的屬性索引鍵參考檔。

[功能區屬性](windowsribbon-reference-properties-ribbon.md)

功能區架構功能區視圖的屬性索引鍵參考檔。

[狀態屬性](windowsribbon-reference-properties-state.md)

功能區架構控制項狀態或內容之屬性索引鍵的參考檔。
