# <a name="ui_pkey_nocolorlabel"></a>UI \_ PKEY \_ NoColorLabel

識別 UI \_ PKEY \_ NoColorLabel 屬性。

```
propertyDescription
   name = UI_PKEY_NoColorLabel
   shellPKey = UI_PKEY_NoColorLabel
   formatID = 00000407-7363-696e-8441798acf5aebb7
   propID = 407
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>備註

\_ \_ 應用程式會使用 UI PKEY NoColorLabel 來查詢與 [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md)的 [**無色彩**] 按鈕相關聯的標籤值。

屬性值是一個字串，表示與 [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md)的 [**無色彩**] 按鈕相關聯的標籤。

UI \_ PKEY \_ NoColorLabel 對所有 **ColorTemplate** 屬性值都是有效的。
