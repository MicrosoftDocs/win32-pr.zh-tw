# <a name="ui_pkey_categoryid"></a>UI \_ PKEY \_ 類別

識別 UI PKEY 的 [類別識別碼] \_ \_ 屬性。

```
propertyDescription
   name = UI_PKEY_CategoryId
   shellPKey = UI_PKEY_CategoryId
   formatID = 00000103-7363-696e-8441798acf5aebb7
   propID = 103
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>備註

\_ \_ 應用程式會使用 UI PKEY 類別目錄來查詢用來將資源庫控制項中的相關專案分組的類別識別碼。

屬性值是分類集合中專案的以零為基底的索引。

下列螢幕擷取畫面說明在 [ [**SplitButton**](windowsribbon-element-splitbutton.md) ] 功能表中使用分類的選項，其中的 [ **選取] 圖形** 識別碼為0，而 **選取選項** 的識別碼為1。

![顯示功能區圖庫中類別目錄的螢幕擷取畫面。](images/properties/ui-pkey-collection-categories2.png)
