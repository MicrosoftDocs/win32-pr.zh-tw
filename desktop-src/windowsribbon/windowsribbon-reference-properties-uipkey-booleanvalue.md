# <a name="ui_pkey_booleanvalue"></a>UI \_ PKEY \_ BooleanValue

識別 UI \_ PKEY \_ BooleanValue 屬性。

```
propertyDescription
   name = UI_PKEY_BooleanValue
   shellPKey = UI_PKEY_BooleanValue
   formatID = 00000200-7363-696e-8441798acf5aebb7
   propID = 200
   typeInfo
      type = VT_BOOL
   booleanFormat
      formatAs = VARIANT_TRUE=-1, VARIANT_FALSE=0
```

## <a name="remarks"></a>備註

\_ \_ 應用程式會使用 UI PKEY BooleanValue 來查詢 [核取方塊](windowsribbon-controls-checkbox.md)的切換狀態、[切換按鈕](windowsribbon-controls-togglebutton.md)，以及 [**SplitButtonGallery**](windowsribbon-element-splitbuttongallery.md)的按鈕控制項。
