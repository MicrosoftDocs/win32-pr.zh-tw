# <a name="ui_pkey_globalbackgroundcolor"></a>UI \_ PKEY \_ GlobalBackgroundColor

識別 UI \_ PKEY \_ GlobalBackgroundColor 屬性。

```
propertyDescription
   name = UI_PKEY_GlobalBackgroundColor
   shellPKey = UI_PKEY_GlobalBackgroundColor
   formatID = 00002000-7363-696e-8441798acf5aebb7
   propID = 2000
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>備註

\_ \_ 應用程式會使用 UI PKEY GlobalBackgroundColor 來查詢自訂功能區的全域背景色彩。

**在 Windows 8 中引進：** ui [ \_ PKEY \_ GlobalHighlightColor](windowsribbon-reference-properties-uipkey-globalhighlightcolor.md)無法獨立于 ui PKEY GlobalBackgroundColor 之外進行設定 \_ \_ 。
