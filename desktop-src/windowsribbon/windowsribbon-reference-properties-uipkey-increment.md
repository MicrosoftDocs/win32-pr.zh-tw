# <a name="ui_pkey_increment"></a>UI \_ PKEY \_ 增量

識別 UI \_ PKEY \_ 遞增屬性。

```
propertyDescription
   name = UI_PKEY_Increment
   shellPKey = UI_PKEY_Increment
   formatID = 00000205-7363-696e-8441798acf5aebb7
   propID = 205
   typeInfo
      type = VT_DECIMAL
```

## <a name="remarks"></a>備註

\_ \_ 應用程式會使用 UI PKEY 增量來查詢按一下 [遞增] 或 [遞減] 按鈕時，[微調](windowsribbon-controls-spinner.md)控制項的 [編輯] 欄位中的值會增加或減少的數量。
