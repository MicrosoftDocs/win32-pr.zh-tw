# <a name="ui_pkey_fontproperties_foregroundcolor"></a>UI \_ PKEY \_ FontProperties \_ ForegroundColor

識別 UI \_ PKEY \_ FontProperties \_ ForegroundColor 屬性。

```
propertyDescription
   name = UI_PKEY_FontProperties_ForegroundColor
   shellPKey = UI_PKEY_FontProperties_ForegroundColor
   formatID = 00000308-7363-696e-8441798acf5aebb7
   propID = 308
   typeInfo
      type = COLORREF
```

## <a name="remarks"></a>備註

UI \_ PKEY \_ FontProperties \_ ForegroundColor 是由應用程式搭配 [ui \_ PKEY \_ FontProperties \_ ForegroundColorType](windowsribbon-reference-properties-uipkey-fontproperties-foregroundcolortype.md)用來查詢 **文字顏色** 庫設定。

預設值是 `0x00000000`。
