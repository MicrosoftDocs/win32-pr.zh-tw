# <a name="ui_pkey_selecteditem"></a>UI \_ PKEY \_ SelectedItem

識別 UI \_ PKEY \_ SelectedItem 屬性。

```
propertyDescription
   name = UI_PKEY_SelectedItem
   shellPKey = UI_PKEY_SelectedItem
   formatID = 00000104-7363-696e-8441798acf5aebb7
   propID = 104
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>備註

\_ \_ 應用程式會使用 UI PKEY SelectedItem 來查詢資源庫控制項中所選取的專案。

屬性值是集合中所選取專案的索引。
