## <a name="remarks"></a><span data-ttu-id="cec8e-105">備註</span><span class="sxs-lookup"><span data-stu-id="cec8e-105">Remarks</span></span>

<span data-ttu-id="cec8e-106">UI \_ PKEY \_ 標籤是由應用程式用來查詢索引標籤、群組、按鈕、圖庫專案和其他功能區控制項的標籤文字。</span><span class="sxs-lookup"><span data-stu-id="cec8e-106">UI\_PKEY\_Label is used by an application to query the label text of tabs, groups, buttons, gallery items, and other Ribbon controls.</span></span>

> [!Note]  
> <span data-ttu-id="cec8e-107">Windows 8 和更新版本： [應用程式功能表](windowsribbon-controls-applicationmenu.md) 按鈕影像已變更為 Label： **File**。</span><span class="sxs-lookup"><span data-stu-id="cec8e-107">Windows 8 and newer: [Application Menu](windowsribbon-controls-applicationmenu.md) button image changed to label: **File**.</span></span> <span data-ttu-id="cec8e-108">建議您不要使用 [檔案] 作為任何您自己的索引標籤的標籤。</span><span class="sxs-lookup"><span data-stu-id="cec8e-108">We recommend that you do not use File as the label for any of your own tabs.</span></span>

 

<span data-ttu-id="cec8e-109">屬性值是限制為任何字元序列（包括空白字元和分行符號字元）的字串。</span><span class="sxs-lookup"><span data-stu-id="cec8e-109">The property value is a string constrained to any sequence of characters, including white space and line-break characters.</span></span>

> [!Note]  
> <span data-ttu-id="cec8e-110">使用通用字元集 (UCS) XML 字元參考 `&#xA;` 來指定分行符號。</span><span class="sxs-lookup"><span data-stu-id="cec8e-110">Use the Universal Character Set (UCS) XML character reference `&#xA;` to specify a line break.</span></span>

 

<span data-ttu-id="cec8e-111">不支援靠右對齊。</span><span class="sxs-lookup"><span data-stu-id="cec8e-111">Right alignment is not supported.</span></span>

<span data-ttu-id="cec8e-112">UI PKEY 標籤的最大長度 \_ \_ 為未系結。</span><span class="sxs-lookup"><span data-stu-id="cec8e-112">The maximum length of UI\_PKEY\_Label is unbounded.</span></span>

<span data-ttu-id="cec8e-113">如果命令是透過功能表項目公開，而且 [**LabelTitle**](windowsribbon-element-command-labeltitle.md) 或 UI \_ PKEY \_ 標籤的值包含前面加上連字號的字母，如下列範例所示，則會將這個字母視為使用者介面的快速鍵，以及功能區架構的鍵盤對應鍵。</span><span class="sxs-lookup"><span data-stu-id="cec8e-113">If a Command is exposed through a menu item and the value of [**Command.LabelTitle**](windowsribbon-element-command-labeltitle.md) or UI\_PKEY\_Label contains a letter preceded by an ampersand, as shown in the following example, this letter is treated as both a keytip and a keyboard accelerator for that Command by the Ribbon framework.</span></span>


```XML
<Command Name="cmdNew"
         Symbol="ID_FILE_NEW"
         Comment="New"
         Id="25001"
         LabelTitle="&amp;New"/>
```



<span data-ttu-id="cec8e-114">若要在標籤中顯示 & 符號，請將特殊字元指定換成雙符號 (`&&`) ，如下列範例所示。</span><span class="sxs-lookup"><span data-stu-id="cec8e-114">To display an ampersand in a label, escape the special character designation with a double ampersand (`&&`) as shown in the following example.</span></span>


```XML
<Command Name="cmdOpen"
         Symbol="ID_FILE_OPEN"
         Comment="Open"
         Id="25002"
         LabelTitle="&amp;&amp;Open"/>
```
