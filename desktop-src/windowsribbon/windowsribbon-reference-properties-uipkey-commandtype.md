# <a name="ui_pkey_commandtype"></a>UI \_ PKEY \_ CommandType

識別 UI \_ PKEY \_ CommandType 屬性。

```
propertyDescription
   name = UI_PKEY_CommandType
   shellPKey = UI_PKEY_CommandType
   formatID = 00000105-7363-696e-8441798acf5aebb7
   propID = 105
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>備註

UI \_ PKEY \_ CommandType 是由應用程式用來查詢資源庫控制項中的命令類型。

屬性值來自 [**UI \_ COMMANDTYPE**](/windows/desktop/api/uiribbon/ne-uiribbon-ui_commandtype) 列舉。
