# <a name="ui_pkey_minvalue"></a>UI \_ PKEY \_ MinValue

識別 UI \_ PKEY \_ MinValue 屬性。

```
propertyDescription
   name = UI_PKEY_MinValue
   shellPKey = UI_PKEY_MinValue
   formatID = 00000204-7363-696e-8441798acf5aebb7
   propID = 204
   typeInfo
      type = VT_DECIMAL
```

## <a name="remarks"></a>備註

UI \_ PKEY \_ MinValue 是由應用程式用來查詢 [微調](windowsribbon-controls-spinner.md) 控制項的最小值。
