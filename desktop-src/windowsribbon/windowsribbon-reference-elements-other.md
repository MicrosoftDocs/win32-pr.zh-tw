# <a name="others"></a>其他

Windows 功能區架構中補充標記元素的參考檔集。



| 主題                                                                              | 目錄                                                                                                                                                                                                       |
|------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**應用程式**](windowsribbon-element-application.md)                           | 表示功能區架構標記規格中的最上層元素。<br/>                                                                                                                      |
| [**應用程式。命令**](windowsribbon-element-application-commands.md)         | 表示應用程式所定義之所有命令的容器。<br/>                                                                                                                    |
| [**應用程式。 Views**](windowsribbon-element-application-views.md)               | 表示每個功能區架構視圖的容器，特別是 [**功能區**](windowsribbon-element-ribbon.md) 和 [**CoNtextPopup**](windowsribbon-element-contextpopup.md)。<br/>          |
| [**ColumnBreak**](windowsribbon-element-columnbreak.md)                           | 表示自訂 [**SizeDefinition**](windowsribbon-element-sizedefinition.md) 版面配置範本中 (可見或隱藏) 的垂直分隔符號。<br/>                                                      |
| [**ControlGroup**](windowsribbon-element-controlgroup.md)                         | 代表 [**SizeDefinition**](windowsribbon-element-sizedefinition.md) 版面配置範本中的一組控制項。<br/>                                                                                 |
| [**ControlNameDefinition**](windowsribbon-element-controlnamedefinition.md)       | 表示自訂 [**SizeDefinition**](windowsribbon-element-sizedefinition.md) 版面配置範本中的控制項名稱。<br/>                                                                          |
| [**ControlNameMap**](windowsribbon-element-controlnamemap.md)                     | 表示自訂 [**SizeDefinition**](windowsribbon-element-sizedefinition.md) 版面配置範本中控制項名稱的容器。<br/>                                                                |
| [**ControlSizeDefinition**](windowsribbon-element-controlsizedefinition.md)       | 表示自訂範本中控制項群組的版面配置樣式。<br/>                                                                                                                            |
| [**FlowMenuLayout**](windowsribbon-element-flowmenulayout.md)                     | 代表包含資源庫中專案之分行符號的水準配置。<br/>                                                                                                                             |
| [**GroupSizeDefinition**](windowsribbon-element-groupsizedefinition.md)           | 表示自訂範本中控制項群組的版面配置大小。<br/>                                                                                                                              |
| [**資料列**](windowsribbon-element-row.md)                                           | 代表自訂 SizeDefinition 版面配置範本中的一列控制項。<br/>                                                                                                                            |
| [**調整**](windowsribbon-element-scale.md)                                       | 透過 {**Group**， [**SizeDefinition**](windowsribbon-element-sizedefinition.md)} 配對，表示一 [**組**](windowsribbon-element-group.md)控制項的大小和配置喜好設定。<br/> |
| [**ScalingPolicy**](windowsribbon-element-scalingpolicy.md)                       | 表示用於調整規格的容器。<br/>                                                                                                                                                  |
| [**ScalingPolicy. IdealSizes**](windowsribbon-element-scalingpolicy-idealsizes.md) | 代表慣用 [**SizeDefinition**](windowsribbon-element-sizedefinition.md) 範本的縮放規格容器（以功能區大小為基礎）。<br/>                                   |
| [**SizeDefinition**](windowsribbon-element-sizedefinition.md)                     | 代表功能區控制項的自訂版面配置範本。 <br/>                                                                                                                                            |
| [**VerticalMenuLayout**](windowsribbon-element-verticalmenulayout.md)             | 代表資源庫中專案的垂直版面配置。<br/>                                                                                                                                                |
