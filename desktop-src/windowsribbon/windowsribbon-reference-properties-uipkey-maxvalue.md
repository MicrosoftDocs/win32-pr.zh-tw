# <a name="ui_pkey_maxvalue"></a>UI \_ PKEY \_

識別 UI \_ PKEY int32.maxvalue \_ 屬性。

```
propertyDescription
   name = UI_PKEY_MaxValue
   shellPKey = UI_PKEY_MaxValue
   formatID = 00000203-7363-696e-8441798acf5aebb7
   propID = 203
   typeInfo
      type = VT_DECIMAL
```

## <a name="remarks"></a>備註

UI \_ PKEY int32.maxvalue \_ 是由應用程式用來查詢 [微調](windowsribbon-controls-spinner.md) 控制項的最大值。
