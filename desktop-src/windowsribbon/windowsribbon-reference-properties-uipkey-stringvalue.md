# <a name="ui_pkey_stringvalue"></a>UI \_ PKEY \_ StringValue

識別 UI \_ PKEY \_ StringValue 屬性。

```
propertyDescription
   name = UI_PKEY_StringValue
   shellPKey = UI_PKEY_StringValue
   formatID = 00000202-7363-696e-8441798acf5aebb7
   propID = 202
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>備註

\_ \_ 應用程式會使用 UI PKEY StringValue 來查詢[下拉式列示方塊](windowsribbon-controls-combobox.md)控制項之 [編輯] 欄位中的值。
