# <a name="ui_pkey_itemimage"></a>UI \_ PKEY \_ ItemImage

識別 UI \_ PKEY \_ ItemImage 屬性。

```
propertyDescription
   name = UI_PKEY_ItemImage
   shellPKey = UI_PKEY_ItemImage
   formatID = 00000106-7363-696e-8441798acf5aebb7
   propID = 106
   typeInfo
      type = IUIImage
```

## <a name="remarks"></a>備註

\_ \_ 應用程式會使用 UI PKEY ItemImage 來查詢與資源庫控制項中的專案或命令相關聯的影像。

屬性值是 [**IUIImage**](/windows/desktop/api/uiribbon/nn-uiribbon-iuiimage) 物件。
