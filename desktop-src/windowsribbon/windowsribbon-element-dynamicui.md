# <a name="dynamicui-element"></a>DynamicUI 元素

代表功能區標記的容器。 此容器及其內容會在執行時間宣告。

## <a name="usage"></a>使用方式

``` syntax
<DynamicUI>
  child elements
</DynamicUI>
```

## <a name="attributes"></a>屬性

沒有任何屬性。

## <a name="child-elements"></a>子元素



| 元素                                                               | 描述                                     |
|-----------------------------------------------------------------------|-------------------------------------------------|
| [**CoNtextPopup**](windowsribbon-element-contextpopup.md)<br/> | 必須剛好發生一次 <br/> <br/> |
| [**索引標籤**](windowsribbon-element-tab.md)<br/>                   | 必須剛好發生一次 <br/> <br/> |
| [**TabGroup**](windowsribbon-element-tabgroup.md)<br/>         | 必須剛好發生一次 <br/> <br/> |



## <a name="parent-elements"></a>父元素

沒有父元素。

## <a name="element-information"></a>項目資訊



|                                     |                                                                         |
|-------------------------------------|-------------------------------------------------------------------------|
| 最低支援系統<br/> | Windows 7、windows Vista （含 SP2）和 Windows Vista 的平臺更新 |
| 可以是空的                        | 否                                                                      |
