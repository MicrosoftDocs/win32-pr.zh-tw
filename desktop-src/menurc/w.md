<dl> <dt>

<span id="tools.w_1_gly"></span><span id="TOOLS.W_1_GLY"></span>**警告： ASCII 字元不等於虛擬金鑰碼**
</dt> <dd>

[**快速鍵**](accelerators-resource.md)語句中存在不正確虛擬金鑰程式碼。 某些字元（例如 \* 、^ 或 &）的 ASCII 值不等於對應索引鍵的虛擬機器碼。 在星號 () 的情況下 \* ，虛擬金鑰程式碼相當於8的 ASCII 值，也就是相同索引鍵上的數值字元。

</dd> <dt>

<span id="tools.w_2_gly"></span><span id="TOOLS.W_2_GLY"></span>**警告：在沒有 VIRTKEY 的情況下使用 SHIFT 或 CONTROL**
</dt> <dd>

**ALT**、 **SHIFT** 和 [**CONTROL**](control-control.md)選項只適用于 [**快速鍵**](accelerators-resource.md)語句中的虛擬機器碼。 請確定 **VIRTKEY** 選項與其中一個其他選項搭配使用。

</dd> <dt>

<span id="tools.w_3_gly"></span><span id="TOOLS.W_3_GLY"></span>**寫入 resourcetypeimagelang： languagesize： size**
</dt> <dd>

 (V) Microsoft Windows 資源編譯器 (RC) 正在寫入指定的資源。 此 *類型* 為資源類型。 它可以是數位或字串。 *影像* 可以是數位或字串。 *語言* 是資源的國家語言。 *大小* 是資源的十進位大小（以位元組為單位）。

</dd> </dl>
