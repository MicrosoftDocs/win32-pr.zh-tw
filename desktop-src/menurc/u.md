# <a name="u-menus-and-other-resources"></a>U (功能表和其他資源) 

[](a.md) [B](b.md) [C](c.md) D [E](e.md) [F](f.md) G H [I](i.md) J K L M [N](n.md) [O](o.md) P Q [R](r.md) S [T](t.md) [](v.md) [W](w.md) X Y Z

<dl> <dt>

<span id="tools.u_1_gly"></span><span id="TOOLS.U_1_GLY"></span>**無法 createdestination**
</dt> <dd>

Microsoft Windows 資源編譯器 (RC) 無法建立目的地檔案。 請確定有足夠的磁碟空間。

</dd> <dt>

<span id="tools.u_2_gly"></span><span id="TOOLS.U_2_GLY"></span>**不對稱的括弧**
</dt> <dd>

請確定您已在 [**對話方塊**](dialog-resource.md) 語句中關閉每個左括弧。

</dd> <dt>

<span id="tools.u_3_gly"></span><span id="TOOLS.U_3_GLY"></span>**RCData 中有非預期的值**
</dt> <dd>

[**RCDATA**](rcdata-resource.md)語句中的 *raw 資料* 參數值必須是整數或字串（以逗號分隔）。 請確定您未在字串前後加上逗號或引號。

</dd> <dt>

<span id="tools.u_4_gly"></span><span id="TOOLS.U_4_GLY"></span>**值資料中有非預期的值**
</dt> <dd>

語句所包含的資訊格式或大小，與該參數的預期值不同。

</dd> <dt>

<span id="tools.u_5_gly"></span><span id="TOOLS.U_5_GLY"></span>**未知的 DIB 標頭格式**
</dt> <dd>

與裝置無關的點陣圖 (DIB) 標頭不是 [**BITMAPCOREHEADER**](/windows/win32/api/wingdi/ns-wingdi-bitmapcoreheader) 或 [**BITMAPINFOHEADER**](/previous-versions//dd183376(v=vs.85)) 結構。

</dd> <dt>

<span id="tools.u_6_gly"></span><span id="TOOLS.U_6_GLY"></span>**未知的功能表子類型**
</dt> <dd>

[**MENU**](menu-resource.md)語句的 *專案定義* 參數只能包含 [**MENUITEM**](menuitem-statement.md)和 [**POPUP**](popup-resource.md)語句。

</dd> <dt>

<span id="tools.u_7_gly"></span><span id="TOOLS.U_7_GLY"></span>**無法辨識的 VERSIONINFO 欄位;必須是 BEGIN 或逗點**
</dt> <dd>

[**VERSIONINFO**](versioninfo-resource.md)語句後面的資訊格式不正確。

</dd> </dl>
