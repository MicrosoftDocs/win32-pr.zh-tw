<span data-ttu-id="b4818-105">定義與 [**按鍵**](pushbutton-control.md)相同的推播方塊控制項，不同之處在于它不會顯示按鈕臉部或框架;只會顯示文字。</span><span class="sxs-lookup"><span data-stu-id="b4818-105">Defines a push-box control, which is identical to a [**PUSHBUTTON**](pushbutton-control.md), except that it does not display a button face or frame; only the text appears.</span></span>
