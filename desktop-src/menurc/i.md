# <a name="i-menus-and-other-resources"></a>我 (功能表和其他資源) 

[](a.md) [B](b.md) [C](c.md) D [E](e.md) [F](f.md) G H I [](u.md) [](v.md) J K L M [N](n.md) [O](o.md) P Q [R](r.md) S [T](t.md) [W](w.md) X Y Z

<dl> <dt>

<span id="tools.i_1_gly"></span><span id="TOOLS.I_1_GLY"></span>**不正確加速器**
</dt> <dd>

[**快速鍵**](accelerators-resource.md)語句中的 *事件* 參數無法辨識，或長度超過兩個字元。

</dd> <dt>

<span id="tools.i_2_gly"></span><span id="TOOLS.I_2_GLY"></span>**不正確加速器類型 (ASCII 或 VIRTKEY)**
</dt> <dd>

[**快速鍵**](accelerators-resource.md)語句中的 *型* 別參數必須包含 **ASCII** 或 **VIRTKEY** 值。

</dd> <dt>

<span id="tools.i_3_gly"></span><span id="TOOLS.I_3_GLY"></span>**不正確控制字元**
</dt> <dd>

[**快速鍵**](accelerators-resource.md)語句中的控制字元無效。 有效的控制字元是由插入號 (^) ，後面接著一個字母。

</dd> <dt>

<span id="tools.i_4_gly"></span><span id="TOOLS.I_4_GLY"></span>**不正確控制項類型**
</dt> <dd>

[**對話方塊**](dialog-resource.md)語句中的控制語句必須是下列其中之一： [**AUTO3STATE**](auto3state-control.md)、 [**AUTOCHECKBOX**](autocheckbox-control.md)、 [**AUTORADIOBUTTON**](autoradiobutton-control.md)、 [**CHECKBOX**](checkbox-control.md)、 [**COMBOBOX**](combobox-control.md)、 [**control**](control-control.md)、 [**CTEXT**](ctext-control.md)、 [**DEFPUSHBUTTON**](defpushbutton-control.md)、 [**EDITTEXT**](edittext-control.md)、[**分組**](groupbox-control.md)、[**圖示**](icon-control.md)、 [**LISTBOX**](listbox-control.md)、 [**LTEXT**](ltext-control.md)、 [**PUSHBOX**](pushbox-control.md)、[**按鈕**](pushbutton-control.md)、[**選項按鈕**](radiobutton-control.md)、 [**RTEXT**](rtext-control.md)、[**捲軸**](scrollbar-control.md)、 [**STATE3**](state3-control.md)。

</dd> <dt>

<span id="tools.i_5_gly"></span><span id="TOOLS.I_5_GLY"></span>**前置處理的 RC 檔案中有不正確指示詞**
</dt> <dd>

指定的檔案名包含內嵌的分行符號。

</dd> <dt>

<span id="tools.i_6_gly"></span><span id="TOOLS.I_6_GLY"></span>**無效。EXE 檔案**
</dt> <dd>

可執行檔 ( .exe) 檔無效。 請確定連結器已正確建立，且檔案存在。

</dd> <dt>

<span id="tools.i_7_gly"></span><span id="TOOLS.I_7_GLY"></span>**不正確參數，選項**
</dt> <dd>

使用的選項無效。 如需命令列選項的清單，請使用 **rc/？** 。

</dd> <dt>

<span id="tools.i_8_gly"></span><span id="TOOLS.I_8_GLY"></span>**不正確參數，太多 D 參數**
</dt> <dd>

指定了太多 **d** 個選項。 若要定義大量的符號，請使用 [**\# include**](-include.md)指示詞來包含定義的檔案。

</dd> <dt>

<span id="tools.i_9_gly"></span><span id="TOOLS.I_9_GLY"></span>**不正確類型**
</dt> <dd>

資源類型不同于 include 檔中定義的類型。

</dd> <dt>

<span id="tools.i_10_gly"></span><span id="TOOLS.I_10_GLY"></span>**使用方式無效。使用 RC-？尋求協助**
</dt> <dd>

請確定您至少有一個要使用的檔案名。 如需命令列選項的清單，請使用 **rc-？** 。

</dd> <dt>

<span id="tools.i_11_gly"></span><span id="TOOLS.I_11_GLY"></span>**讀取檔案時發生 i/o 錯誤**
</dt> <dd>

讀取失敗。 因為這是一般常式，所以不提供任何特定的檔案名。

</dd> <dt>

<span id="tools.i_12_gly"></span><span id="TOOLS.I_12_GLY"></span>**檔案中的 i/o 錯誤搜尋**
</dt> <dd>

在檔案中搜尋失敗。 因為這是一般常式，所以不提供任何特定的檔案名。

</dd> <dt>

<span id="tools.i_13_gly"></span><span id="TOOLS.I_13_GLY"></span>**寫入檔案時發生 i/o 錯誤**
</dt> <dd>

寫入失敗。 因為這是一般常式，所以不提供任何特定的檔案名。

</dd> </dl>
