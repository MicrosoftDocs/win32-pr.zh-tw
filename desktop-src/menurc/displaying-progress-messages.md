# <a name="displaying-progress-messages"></a>顯示進度訊息

RC 預設會進行編譯，而不會顯示進度訊息。 不過，您可以指定 RC 來顯示進度訊息。 若要這樣做，請使用 **/v** 選項。

下列範例會讓 RC 在編譯資源定義檔範例時回報其進度，並建立已編譯的資源檔範例 .res：
