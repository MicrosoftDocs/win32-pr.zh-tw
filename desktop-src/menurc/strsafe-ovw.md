# <a name="about-strsafeh"></a><span data-ttu-id="38b13-106">關於 >strsafe.h。h</span><span class="sxs-lookup"><span data-stu-id="38b13-106">About Strsafe.h</span></span>

<span data-ttu-id="38b13-107">不佳的緩衝區處理暗喻著在涉及緩衝區溢位的許多安全性問題中。</span><span class="sxs-lookup"><span data-stu-id="38b13-107">Poor buffer handling is implicated in many security issues that involve buffer overruns.</span></span> <span data-ttu-id="38b13-108">>strsafe.h 中定義的函式會在程式碼中為適當的緩衝區處理提供額外的處理。</span><span class="sxs-lookup"><span data-stu-id="38b13-108">The functions defined in Strsafe.h provide additional processing for proper buffer handling in your code.</span></span> <span data-ttu-id="38b13-109">基於這個理由，它們的用途是取代內建的 C/c + + 對應專案，以及特定的 Windows 執行。</span><span class="sxs-lookup"><span data-stu-id="38b13-109">For this reason, they are intended to replace their built-in C/C++ counterparts as well as specific Windows implementations.</span></span> <span data-ttu-id="38b13-110">從 Windows XP Service Pack 2 （含 Service Pack 2） (SP2) 開始 Windows SDK 中提供 >strsafe.h。</span><span class="sxs-lookup"><span data-stu-id="38b13-110">Strsafe.h is available in the Windows SDK starting with Windows XP with Service Pack 2 (SP2).</span></span>

<span data-ttu-id="38b13-111">>strsafe.h 函數的優點包括：</span><span class="sxs-lookup"><span data-stu-id="38b13-111">The advantages of the Strsafe functions include:</span></span>
