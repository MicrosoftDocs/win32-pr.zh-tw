# <a name="endif"></a>\#endif

**\# Endif** 指示詞會標示由 **\# ifdef** 指示詞所定義之條件式編譯區塊的結尾。 每個 **\# if**、 **\# ifdef** 或 **\# ifndef** 指示詞都需要一個 **\# endif** 。

``` syntax
#endif
```

這個指示詞沒有任何參數。
