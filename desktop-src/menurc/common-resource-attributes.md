# <a name="common-resource-attributes"></a>一般資源屬性

在16位 Windows 上支援的資源定義語句包含 *載入記憶體* 選項，可指定資源的載入和記憶體特性。 資源腳本中允許這些屬性以提供回溯相容性，但會忽略這些屬性。 載入對應的模組時，系統會載入 Windows 資源，並在卸載模組時將其釋出。

## <a name="load-attributes"></a>載入屬性

Load 屬性會指定資源的載入時間。 Load 參數必須是下列其中一個屬性。



| 屬性      | 描述                                                                  |
|----------------|------------------------------------------------------------------------------|
| **預 緊**    | 忽略。 在16位的 Windows 中，資源會與可執行檔一起載入。 |
| **LOADONCALL** | 忽略。 在16位的 Windows 中，會在呼叫時載入資源。              |



 

## <a name="memory-attributes"></a>記憶體屬性

Memory 屬性會指定資源為固定或可移動、是否為 discardable，以及是否為純虛擬。 Memory 參數可以是下列一或多個屬性。



| 屬性       | 描述                                                                                                                               |
|-----------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| **FIXED**       | 忽略。 在16位的 Windows 中，資源會保留在固定的記憶體位置。                                                              |
| **移動**    | 忽略。 在16位的 Windows 中，可在需要時移動資源以壓縮記憶體。                                                     |
| **DISCARDABLE** | 忽略。 在16位的 Windows 中，如果不再需要資源，則可以將它捨棄。                                                            |
| **純**        | 忽略。 已接受，可與現有的資源腳本相容。                                                                       |
| **IMPURE**      | 忽略。 已接受，可與現有的資源腳本相容。                                                                       |
| **共用**      | 忽略。 在16位的 Windows 中，一般模組會忽略 SHARED。 對於來自 ROM Windows 模組的資源，則會共用記憶體。        |
| **非共用**   | 忽略。 在16位的 Windows 中，一般模組會忽略非共用。 對於來自 ROM Windows 模組的資源，則不會共用記憶體。 |
