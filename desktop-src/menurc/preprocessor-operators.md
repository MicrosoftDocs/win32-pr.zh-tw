# <a name="preprocessor-operators"></a>前置處理器運算子

RC 支援在巨集定義中使用標準 C 前置處理運算子。 下表將說明這些運算子。



| 運算子 | 描述                                                 |
|----------|-------------------------------------------------------------|
| \#       | 將引數括在引號中。                            |
| \#@      | 以單引號括住引數。                     |
| \#\#     | 串連當做引數使用的標記，以形成其他標記。 |
