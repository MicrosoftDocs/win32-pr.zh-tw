# <a name="caption-statement"></a>CAPTION 語句

定義對話方塊的標題。 標題會出現在方塊的標題列 (如果有一個) 。

預設標題為空白。

``` syntax
CAPTION "captiontext"
```

<dl> <dt>

<span id="captiontext"></span><span id="CAPTIONTEXT"></span>*captioNtext*
</dt> <dd>

以雙引號括住的字元字串 ( ") 。

</dd> </dl>

## <a name="examples"></a>範例

下列範例示範 **CAPTION** 語句的用法：

``` syntax
CAPTION "Error!" 
```
