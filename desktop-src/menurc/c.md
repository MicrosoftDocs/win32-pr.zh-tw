<span id="tools.c_1_gly"></span><span id="TOOLS.C_1_GLY"></span>**無法重複使用字串常數**
</dt> <dd>

在 [**STRINGTABLE**](stringtable-resource.md) 語句中，您會使用相同的值兩次。 請確定您沒有混合重迭的十進位和十六進位值。

</dd> <dt>

<span id="tools.c_2_gly"></span><span id="TOOLS.C_2_GLY"></span>**控制字元超出範圍 \[ A？Z\]**
</dt> <dd>

[**快速鍵**](accelerators-resource.md)語句中的控制字元無效。 插入號 (^) 之後的字元必須在 A 至 Z 的範圍內。

</dd> <dt>

<span id="tools.c_3_gly"></span><span id="TOOLS.C_3_GLY"></span>**無法在-檔案名**
</dt> <dd>

Microsoft Windows 資源編譯器 (RC) 無法開啟指定的檔案。 請確定檔案存在，而且您輸入的檔案名正確無誤。

</dd> <dt>

<span id="tools.c_4_gly"></span><span id="TOOLS.C_4_GLY"></span>**無法 openresource-名稱**
</dt> <dd>

Microsoft Windows 資源編譯器 (RC) 無法開啟指定的檔案。 請確定檔案存在，而且您輸入的檔案名正確無誤。

</dd> <dt>

<span id="tools.c_5_gly"></span><span id="TOOLS.C_5_GLY"></span>**Creatingresource-名稱**
</dt> <dd>

 (V) Microsoft Windows 資源編譯器 (RC) 正在建立新的二進位資源 ( .res) 檔。

</dd> </dl>
