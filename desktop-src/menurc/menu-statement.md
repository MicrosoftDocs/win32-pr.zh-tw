# <a name="menu-statement"></a>MENU 語句

定義對話方塊的功能表。 如果未指定任何語句，則對話方塊沒有功能表。

``` syntax
MENU menuname
```

<dl> <dt>

<span id="menuname"></span><span id="MENUNAME"></span>*menuname*
</dt> <dd>

要使用之功能表的名稱或識別碼。

</dd> </dl>

## <a name="examples"></a>範例

下列範例示範如何使用 **功能表** 對話方塊語句：

``` syntax
MENU errmenu
```
