# <a name="pnrp-constants"></a>PNRP 常數

PNRP 命名空間提供者 API 會使用下列常數。



| 常數                                                 | 描述                                                |
|----------------------------------------------------------|------------------------------------------------------------|
| [**PNRP \_ 雲端 \_ 旗標**](/windows/desktop/api/Pnrpdef/ne-pnrpdef-pnrp_cloud_flags)           | 指定雲端名稱的有效性。                    |
| [**PNRP \_ 雲端 \_ 狀態**](/windows/desktop/api/Pnrpdef/ne-pnrpdef-pnrp_cloud_state)           | 指定 PNRP 雲端可以處於的不同狀態。     |
| [**PNRP \_ 解析 \_ 準則**](/windows/desktop/api/Pnrpdef/ne-pnrpdef-pnrp_resolve_criteria) | 指定 PNRP 用來解析搜尋的準則。 |
