# <a name="people-near-me-apis"></a>近端分享 Api

任何能夠 [近端分享](about-people-near-me.md) (PNM) 的應用程式都可以使用下列 Win32 api：

### <a name="contacts-api"></a>連絡人 API

連絡人 API 是用來存取已登入使用者的連絡人資訊、取出先前儲存之連絡人的相關資訊，或是儲存連絡人資訊和新連絡人的數位憑證。

### <a name="people-near-me-api"></a>近端分享 API

PNM API 可用來尋找附近的使用者、其公告的興趣、他們選擇要發佈的任意物件，以及公告的應用程式。

### <a name="publication-api"></a>發行集 API

發行集 API 是用來與遠端連絡人互動。 PNM 模組也會使用發行集管理員所使用的 [對等信號](windows-peer-components-for-people-near-me.md) 層來建立連接。

### <a name="invitation-api"></a>邀請 API

共同作業應用程式會使用邀請 API 來邀請特定的對等 () 到共同作業活動中。

### <a name="application-registration-api"></a>應用程式註冊 API

共同作業應用程式的安裝程式會使用應用程式註冊 API 來註冊應用程式。 具有 PNM 功能的應用程式會在安裝過程中提示使用者，以判斷是否應該將應用程式提供給子網的使用者使用。
