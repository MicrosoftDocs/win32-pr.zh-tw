# <a name="installing-the-peer-infrastructure"></a>安裝對等基礎結構

Windows 對等網路基礎結構是一種開發人員平臺，可為執行 Windows XP Service Pack 1 (SP1) with Advanced 網路套件、Windows XP Service Pack 2 (SP2) 、Windows Vista 和 Windows Server 2008 的電腦建立對等應用程式。

執行 Windows Vista 或 Windows Server 2008 的電腦已安裝 Windows 對等網路基礎結構。 對於執行 Windows XP （含 SP2）的電腦，必須執行下列步驟，才能安裝 Windows 對等網路：

-   按一下 [開始]、[控制台]，然後按一下 [新增或移除程式]。
-   按一下 [ **新增/移除 Windows 元件**]。
-   在 [ **元件**] 中，按一下 [ **網路服務** ] (但不要選取其核取方塊) ，然後按一下 [ **詳細資料**]。
-   選取 [ **對等** ] 核取方塊，然後按一下 **[確定]**。
-   按 [ **下一步**]，然後依照嚮導中的指示進行。

執行 Windows XP 含 SP1 的電腦需要安裝 [Advanced 網路套件](https://www.microsoft.com/downloads/details.aspx?FamilyID=e88cc382-8ce6-4739-97c0-1a52a6f005e4)（可免費下載），才能啟用對等網路。
