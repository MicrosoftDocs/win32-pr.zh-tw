# <a name="graphing-api-structures"></a>圖形 API 結構

圖形 API 會使用下列結構：



| 結構                                                                 | Description                                                                                                   |
|---------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**對等 \_ 事件 \_ 節點 \_ 變更 \_ 資料**](/windows/desktop/api/P2P/ns-p2p-peer_event_node_change_data)    | 如果觸發 **對等 \_ 圖形 \_ 事件 \_ 節點 \_ 變更** 事件，則包含資料的指標。                  |
| [**對等 \_ 事件同步處理的 \_ \_ 資料**](/windows/desktop/api/P2P/ns-p2p-peer_event_synchronized_data)   | 表示已同步處理的記錄類型。                                                      |
| [**對等 \_ 圖形 \_ 事件 \_ 資料**](/windows/desktop/api/P2P/ns-p2p-peer_graph_event_data)                 | 包含與對等事件相關聯的資料。                                                                   |
| [**對等 \_ 圖形 \_ 事件 \_ 註冊**](/windows/desktop/api/P2P/ns-p2p-peer_graph_event_registration) | 指定應用程式需要通知的對等事件。                                        |
| [**對等 \_ 圖形 \_ 屬性**](/windows/desktop/api/P2P/ns-p2p-peer_graph_properties)                  | 包含對等圖形、識別碼、範圍和其他資訊之原則的相關資料。                             |
| [**對等 \_ 節點 \_ 資訊**](/windows/desktop/api/P2P/ns-p2p-peer_node_info)                                | 包含對等圖形中特定節點的特定資訊。                                   |
| [**對等 \_ 安全性 \_ 介面**](/windows/desktop/api/P2P/ns-p2p-peer_security_interface)              | 指定對對等圖形 Api 呼叫的安全性介面，以用來驗證、保護和釋放記錄。 |
