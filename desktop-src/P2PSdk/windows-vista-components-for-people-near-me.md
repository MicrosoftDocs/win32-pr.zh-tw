# <a name="windows-vista-components-for-people-near-me"></a>Windows近端分享的 Vista 元件

[近端分享](about-people-near-me.md) (PNM) 會使用 Windows Vista 的下列元件：

### <a name="web-service-discovery"></a>Web 服務探索

Web 服務探索元件可讓您在本機子網上發行及探索服務。 在 PNM 中，使用者是在本機子網上公告的服務。 如需 Web 服務探索的詳細資訊，請參閱 [XML Web Service discovery](/previous-versions/dotnet/netframework-4.0/fxx6cfx2(v=vs.100))。

### <a name="schannel"></a>Schannel

Schannel 元件允許在兩個端點之間進行加密通訊。

### <a name="windows-sockets"></a>Windows Sockets

應用程式會使用 Windows 通訊端元件透過 tcp/ip 進行通訊。

### <a name="next-generation-tcpip-stack"></a>下一代 TCP/IP 堆疊

Windows Vista 中的新 tcp/ip 堆疊，預設已內建 IPv6 且已啟用。 如需詳細資訊，請參閱[Windows Vista 中的下一代 tcp/ip 堆疊](https://www.microsoft.com/technet/itsolutions/network/tcpip/default.mspx)。
