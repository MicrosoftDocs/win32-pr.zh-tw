# <a name="people-near-me-architecture"></a>近端分享架構

Windows Vista 中的[近端分享架構](about-people-near-me.md)提供了功能，可讓您發佈及探索姓名、興趣、應用程式、傳送和接受邀請、將探索到的對等新增為連絡人，以及保護與探索到的對等的通訊。

下列各節會根據下列元件集來描述近端分享架構：

-   [API](people-near-me-apis.md)
-   [商店](people-near-me-stores.md)
-   [Windows對等元件](windows-peer-components-for-people-near-me.md)
-   [WindowsVista 元件](windows-vista-components-for-people-near-me.md)
