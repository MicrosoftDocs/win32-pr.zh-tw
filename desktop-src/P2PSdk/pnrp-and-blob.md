# <a name="pnrp-and-blob"></a>PNRP 和 BLOB

PNRP 會使用 [**BLOB**](winsock-nsp-reference-links.md) 結構，在呼叫數個函式期間將資料傳遞至 **WSAQUERYSET** 結構。 若要搭配 PNRP 使用， **BLOB** 結構會指向 [**PNRPINFO**](/windows/desktop/api/Pnrpns/ns-pnrpns-pnrpinfo_v1) 結構或 [**PNRPCLOUDINFO**](/windows/desktop/api/Pnrpns/ns-pnrpns-pnrpcloudinfo) 結構。
