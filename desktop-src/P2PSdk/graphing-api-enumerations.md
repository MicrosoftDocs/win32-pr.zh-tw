# <a name="graphing-api-constants"></a>圖形 API 常數

圖形 API 會使用下列常數：



| 常數                                                          | 描述                                                       |
|-------------------------------------------------------------------|-------------------------------------------------------------------|
| [**對等 \_ 圖形 \_ 事件 \_ 類型**](/windows/desktop/api/P2P/ne-p2p-peer_graph_event_type)         | 指定要通知應用程式的對等事件種類。 |
| [**對等 \_ 圖形 \_ 屬性 \_ 旗標**](/windows/desktop/api/P2P/ne-p2p-peer_graph_property_flags) | 指定對等圖表的屬性。                             |
| [**對等 \_ 圖形 \_ 範圍**](/windows/desktop/api/P2P/ne-p2p-peer_graph_scope)                    | 指定對等圖形的網路範圍。                      |
| [**對等 \_ 圖形 \_ 狀態 \_ 旗標**](/windows/desktop/api/P2P/ne-p2p-peer_graph_status_flags)     | 指定節點在對等圖形中的目前狀態。     |
| [**對等 \_ 節點 \_ 變更 \_ 類型**](/windows/desktop/api/P2P/ne-p2p-peer_node_change_type)         | 指定對等節點圖形狀態變更的類型。            |
