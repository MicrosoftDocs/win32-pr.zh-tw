# <a name="common-structures"></a>通用結構

對等基礎結構會使用下列通用結構。



| 結構                                                                          | 描述                                                                                                                         |
|------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------|
| [**對等 \_ 位址**](/windows/desktop/api/P2P/ns-p2p-peer_address)                                              | 指定 IP 位址的相關資訊。                                                                                     |
| [**對等 \_ 連接 \_ 資訊**](/windows/desktop/api/P2P/ns-p2p-peer_connection_info)                             | 包含連接的相關資訊。 當您列舉對等繪圖或群組連接時，會傳回這個結構。 |
| [**對等 \_ 資料**](/windows/desktop/api/P2P/ns-p2p-peer_data)                                                    | 包含二進位資料。                                                                                                               |
| [**對等 \_ 事件 \_ 連接 \_ 變更 \_ 資料**](/windows/desktop/api/P2P/ns-p2p-peer_event_connection_change_data) | 包含更新的資訊，其中包括鄰近或直接連線的變更。                                              |
| [**對等 \_ 事件 \_ 傳入 \_ 資料**](/windows/desktop/api/P2P/ns-p2p-peer_event_incoming_data)                    | 包含更新的資訊，其中包含節點從鄰近或直接連接接收的資料變更。                       |
| [**對等 \_ 事件 \_ 記錄 \_ 變更 \_ 資料**](/windows/desktop/api/P2P/ns-p2p-peer_event_record_change_data)         | 包含應用程式要求對記錄或記錄類型進行資料變更的更新資訊。                              |
| [**對等 \_ 記錄**](/windows/desktop/api/P2P/ns-p2p-peer_record)                                                | 包含應用程式使用的記錄物件。                                                                                |
| [**對等 \_ 版本 \_ 資料**](/windows/desktop/api/P2P/ns-p2p-peer_version_data)                                   | 包含對等圖形和群組 Api 的版本資訊。                                                         |
