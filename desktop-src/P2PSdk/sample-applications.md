# <a name="sample-applications-peer-infrastructure"></a> (對等基礎結構) 的範例應用程式

下列範例應用程式包含在 Windows XP 對等 SDK 中。 當您使用對等基礎結構開發自己的對等應用程式時，這些範例可協助您。

-   [Graph聊天](#graph-chat)
-   [群組聊天](#group-chat)
-   [群組瀏覽器](#group-browser)

## <a name="graph-chat"></a>Graph聊天

Graph chat 範例應用程式是簡單的聊天應用程式，示範如何使用對等圖形 api 和對等名稱解析通訊協定， (PNRP) 命名空間提供者搭配 Winsock 2 API。 應用程式會示範下列工作：

-   建立圖形
-   連接至現有的圖形
-   中斷與現有圖形的連線
-   列舉對等實體
-   將記錄新增至圖形
-   使用圖表的直接連接
-   使用通知和事件基礎結構與圖形
-   使用 PNRP 註冊名稱
-   使用 PNRP 解析名稱
-   使用 PNRP 取消註冊名稱

## <a name="group-chat"></a>群組聊天

群組聊天範例應用程式是簡單的聊天應用程式，示範如何使用對等群組和 Identity Manager Api。 應用程式會示範下列工作：

-   建立身分識別
-   建立和取得身分識別資訊
-   列舉身分識別
-   列舉與身分識別相關聯的群組
-   建立群組
-   建立群組的邀請
-   連接至現有的群組
-   中斷與現有群組的連接
-   從群組屬性中解壓縮資訊
-   使用直接連接與群組
-   使用群組中的列舉函數
-   列舉群組成員
-   將記錄新增至群組
-   使用通知和事件基礎結構與群組

## <a name="group-browser"></a>群組瀏覽器

群組瀏覽器範例應用程式是簡單的對等群組管理工具，示範如何使用對等群組和 Identity Manager Api。 應用程式會示範下列工作：

-   列舉 PNRP 雲端
-   列舉身分識別
-   列舉與身分識別相關聯的群組
-   建立和刪除身分識別
-   建立群組，並將其與身分識別建立關聯
-   建立並儲存邀請
-   開啟邀請並使用它來加入群組
-   刪除身分識別和群組成員資格
-   連接至現有的群組
-   中斷與現有群組的連接
-   從群組屬性解壓縮資訊
-   使用群組中的列舉函數
-   列舉群組成員
-   使用通知和事件基礎結構與群組
