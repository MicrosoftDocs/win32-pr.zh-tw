# <a name="benefits-of-peer-networking"></a>對等網路的優點

對等基礎結構會提供完整的解決方案，讓您建立對等網路應用程式。 您可以使用對等圖形、群組、身分識別管理員和 PNRP 命名空間提供者 Api，來建立全新、令人興奮且強大的對等應用程式。 對等基礎結構提供穩固的網路基礎結構，讓您可以專注于 **開發** 應用程式，因為基礎結構是為您而開發。

## <a name="design-features"></a>設計功能

對等基礎結構提供下列功能：

| 功能            | 描述                                          |
|--------------------|------------------------------------------------------|
| **安全**         | 如果發生失敗或有導向的攻擊，就會有穩定性 |
| **可調整**       | 從簡單 Lan 到整個網際網路的擴充性  |
| **無伺服器**     | 沒有中央點失敗                          |
| **自動調整**    | 適應基礎結構內的變更          |
| **自我修復** | 自動修復和修正本身            |
| **共用**        | 啟用從邊緣網路端點共用       |



 

您可以使用對等基礎結構來建立下列類型的應用程式：

-   即時網路和通訊
-   共同作業
-   內容發佈
-   分散式處理協調
-   網際網路技術和通訊協定

## <a name="alternative-to-dns"></a>DNS 的替代方案

因為點對點實體會以臨機操作方式參與和通訊，對等基礎結構是 DNS 的有效替代方式，可進行名稱解析。 點對點基礎結構會使用 PNRP 命名空間提供者 API，以提供無伺服器機制來解析對等名稱。
