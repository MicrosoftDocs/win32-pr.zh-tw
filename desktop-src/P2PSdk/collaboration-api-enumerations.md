# <a name="collaboration-api-enumerations"></a>協同作業 API 列舉

下列列舉支援對等共同作業基礎結構函數。



| 列舉型別                                                                         | 描述                                                                                                                                                                                                                                                                   |
|-------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**對等 \_ 應用程式 \_ 註冊 \_ 類型**](/windows/desktop/api/P2P/ne-p2p-peer_application_registration_type) | 定義一組對等應用程式註冊旗標。                                                                                                                                                                                                                       |
| [**對等 \_ 變更 \_ 類型**](/windows/desktop/api/P2P/ne-p2p-peer_change_type)                                      | 定義可以在對等物件、端點或應用程式上執行的一組變更。                                                                                                                                                                                  |
| [**對等登入 \_ \_ 旗標**](/windows/desktop/api/P2P/ne-p2p-peer_signin_flags)                                    | 定義當對等登入對等共同作業網路時，可用的對等存在性發行集行為。                                                                                                                                                      |
| [**對等 \_ 監看 \_ 許可權**](/windows/desktop/api/P2P/ne-p2p-peer_watch_permission)                            | 定義一組許可權，這些許可權代表連絡人對監看對等的存在性。                                                                                                                                                                                        |
| [**對等 \_ COLLABORATION \_ 事件 \_ 類型**](/windows/desktop/api/P2P/ne-p2p-peer_collab_event_type)                         | 定義對等共同作業網路事件基礎結構可在對等上引發的事件集。                                                                                                                                                                |
| [**對 \_ 等狀態 \_**](/windows/desktop/api/P2P/ne-p2p-peer_presence_status)                              | 定義可供參與對等共同作業網路之對等的可能存在狀態設定集。 對等共同作業網路端點可以設定這些設定，以表示對等連絡人目前對其監看員的參與程度。 |
| [**對等 \_ 邀請 \_ 回應 \_ 類型**](/windows/desktop/api/P2P/ne-p2p-peer_invitation_response_type)           | 定義收到邀請以加入對等共同作業活動的回應集合。                                                                                                                                                                                 |
| [**對等 \_ 發行 \_ 領域**](/windows/desktop/api/P2P/ne-p2p-peer_publication_scope)                          | 定義發行對等物件或資料的範圍集合。                                                                                                                                                                                                        |
