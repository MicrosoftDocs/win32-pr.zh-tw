# <a name="about-people-near-me"></a>關於近端分享

近端分享 (PNM) 是一種方式，可讓人員和應用程式通告自己和他們的興趣，以輕鬆進行連線及共同作業。 近端分享中的「附近」是位於相同子網上的一組使用者。 PNM 是 windows Vista 內含的 [Windows 對等網路](what-is-peer-networking-.md) 的新功能。

本檔的這一節包含 PNM 的相關資訊，並包含下列主題：

-   [為何要使用近端分享？](why-use-people-near-me-.md)
-   [近端分享需求](people-near-me-requirements.md)

如需有關基本架構以及 PNM 所使用之特定 Api、存放區及元件的詳細資訊，請參閱 [近端分享架構檔](people-near-me-architecture.md)集。
