# <a name="peer-distribution"></a>對等分佈

對等散發 API （支援 Windows 7 和 Windows Server 2008 R2 中的分支快取功能）提供了一組平臺 api，可在從遠端辦公室存取時，增加集中式應用程式的網路回應能力，讓這些辦公室的使用者在區域網路上工作的體驗。 這組 Api 提供的功能也有助於降低整體廣域網路 (WAN) 使用率。

執行這些功能不需要任何既有的基礎結構。 藉由將 Windows 7 部署到用戶端電腦、將 Windows Server 2008 R2 部署到伺服器電腦，以及啟用分支快取功能，來改善遠端網路的效能。

對等散發 Api 與網路安全性技術（包括 SSL、SMB 簽署和端對端 IPsec）緊密搭配運作。 您可以使用此 API 來減少網路頻寬的使用率，並改善應用程式效能，即使內容已加密也一樣。
