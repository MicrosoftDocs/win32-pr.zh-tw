# <a name="about-the-peer-collaboration-api"></a>關於對等共同作業 API

對等共同作業基礎結構提供對等的網路架構，可用於共同作業無伺服器活動，例如網路遊戲配對、會議和其他互動式多參與者活動。

對等共同作業基礎結構所提供的 Api，可簡化應用程式在沒有伺服器的情況下追蹤對等狀態的程式、將邀請傳送給參與者、探索相同子網或 LAN 上的對等，以及管理連絡人。

此外，此基礎結構也可讓對等電腦探索相同子網或區域網路上的其他對等，並允許這些對等互連之間的互動。

無伺服器目前狀態是適用于所有 Windows Vista 應用程式的對等共同作業基礎結構功能。 任何使用共同作業基礎結構之無伺服器存在性元件的應用程式，都能正確地將使用者的狀態顯示到任何監看它的應用程式。 使用者也可以發行其目前狀態的詳細資訊，以及發佈向對等註冊的資料物件。
