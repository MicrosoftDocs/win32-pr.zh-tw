# <a name="grouping-api-enumerations"></a>將 API 列舉分組

群組 API 會使用下列列舉：



| 列舉型別                                                                        | 描述                                                                                                                                                                                                                                               |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**對等 \_ 群組 \_ 驗證 \_ 配置**](/windows/desktop/api/P2P/ne-p2p-peer_group_authentication_scheme)    | 定義一組可能的驗證配置，可用來驗證對等互連加入對等群組。                                                                                                                                           |
| [**對等 \_ 群組 \_ 事件 \_ 類型**](/windows/desktop/api/P2P/ne-p2p-peer_group_event_type)                          | 包含可在對等群組內發生的特定對等事件種類。                                                                                                                                                                                |
| [**對等 \_ 群組 \_ 問題 \_ 認證 \_ 旗標**](/windows/win32/api/p2p/ne-p2p-peer_group_issue_credential_flags) | 指定使用者認證是否儲存在群組中。                                                                                                                                                                                                  |
| [**對等 \_ 群組 \_ 屬性 \_ 旗標**](/windows/desktop/api/P2P/ne-p2p-peer_group_property_flags)                  | 指定對等群組成員資格設定。                                                                                                                                                                                                                 |
| [**對等 \_ 群組 \_ 狀態**](/windows/desktop/api/P2P/ne-p2p-peer_group_status)                                   | 指出對等群組是否有連線存在。                                                                                                                                                                                          |
| [**對等 \_ 成員 \_ 變更 \_ 類型**](/windows/desktop/api/P2P/ne-p2p-peer_member_change_type)                      | 定義對等的一組可能對等群組成員資格和狀態狀態。                                                                                                                                                                         |
| [**對等 \_ 成員 \_ 旗標**](/windows/desktop/api/P2P/ne-p2p-peer_member_flags)                                   | 允許應用程式指定在呼叫 [**PeerGroupEnumMembers**](/windows/desktop/api/P2P/nf-p2p-peergroupenummembers) 函式時，是否要列舉所有成員或只列舉其中的成員，或指出成員是否存在於對等群組中。 |
