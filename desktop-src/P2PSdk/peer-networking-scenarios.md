# <a name="peer-networking-scenarios"></a>對等網路案例

對等基礎結構可提供您許多選項來開發對等網路應用程式。 本主題包含下列對等網路案例的相關資訊：

-   共同作業應用程式
-   立即訊息或聊天應用程式
-   內容散發應用程式

## <a name="collaborative-applications"></a>共同作業應用程式

您可以使用對等基礎結構來建立共同作業應用程式，讓使用者不論是在相同的會議室或網際網路上，都能彼此互動地共用想法。 例如，您可以建立共用的類似白板應用程式。

> [!Note]  
> 對等基礎結構不支援音訊或影片串流。

 

## <a name="instant-messaging-or-chat-applications"></a>立即訊息或聊天應用程式

使用中央伺服器的立即訊息與聊天應用程式可以更新以使用點對點基礎結構。 例如，聊天應用程式可以使用對等基礎結構作為集中式伺服器的替代方案。 對等基礎結構可提升擴充性，並可讓使用者在沒有伺服器的情況下參與聊天和訊息活動。

## <a name="content-distribution-applications"></a>內容散發應用程式

您可以使用對等基礎結構作為內容的散發機制。 例如，您可以將電子報散發給訂閱服務的同事。 因為對等互連基礎結構中的資料會進行同步處理，所以即使發行者不在線上，訂閱者仍會收到電子報。
