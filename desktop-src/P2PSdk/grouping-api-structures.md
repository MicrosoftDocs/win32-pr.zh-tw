# <a name="grouping-api-structures"></a>將 API 結構分組

群組 API 會使用下列結構。



| 結構                                                                  | Description                                                                                                   |
|----------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**對等 \_ 認證 \_ 資訊**](/windows/desktop/api/P2P/ns-p2p-peer_credential_info)                     | 定義用來取得和發行對等安全性認證的資訊。                                   |
| [**對等 \_ 事件 \_ 成員 \_ 變更 \_ 資料**](/windows/desktop/api/P2P/ns-p2p-peer_event_member_change_data) | 包含描述對等群組成員狀態變更的資料。                                   |
| [**對等 \_ 群組 \_ 事件 \_ 資料**](/windows/win32/api/p2p/ns-p2p-peer_group_event_data-r1)                  | 包含發生之特定對等群組事件的相關資訊。                                     |
| [**對等 \_ 群組 \_ 事件 \_ 註冊**](/windows/desktop/api/P2P/ns-p2p-peer_group_event_registration)  | 定義成員可以註冊的特定對等群組事件。                                            |
| [**對等 \_ 群組 \_ 屬性**](/windows/desktop/api/P2P/ns-p2p-peer_group_properties)                   | 包含對等群組成員資格原則的相關資料。                                                    |
| [**對等 \_ 邀請 \_ 資訊**](/windows/desktop/api/P2P/ns-p2p-peer_invitation_info)                     | 定義加入對等群組之邀請的相關資訊。 邀請會以 Unicode 字串表示。 |
| [**對等 \_ 成員**](/windows/desktop/api/P2P/ns-p2p-peer_member)                                        | 包含描述對等群組成員的資訊。                                                 |
