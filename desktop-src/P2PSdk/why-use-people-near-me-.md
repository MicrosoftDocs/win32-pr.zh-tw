# <a name="why-use-people-near-me"></a>為何要使用近端分享？

以下是 [近端分享](about-people-near-me.md) (PNM) 案例的一些範例：

-   在會議期間開始與會議室中的人員交談。 使用聊天應用程式以動態方式探索房間內的其他使用者、選取特定使用者，然後開始聊天會話以共同作業或交換資訊。
-   PNM API 可讓您與會議室中的每個人共用檔。 雖然通用序列匯流排 (USB) 快閃磁片磁碟機 (Ufd) 讓您很容易就能在關閉的設定中的電腦之間傳輸檔（例如會議），但仍需要花費時間和精力將 UFD 插入每部電腦，並將檔案複製到本機資料夾。 有了具有 PNM 功能的應用程式，您可以在您想要共用或傳送檔的房間內選取使用者。
-   將電腦桌面上的資訊串流至同事的電腦，以進行資源分享。
-   在支援無線連線的機場終端機站大廈期間，開始與某人進行遊戲。 PNM API 可讓應用程式在玩遊戲時宣傳您的興趣、在機場上探索其他有興趣播放遊戲的人，以及開始玩遊戲的課程。
-   使用無線連線來公告產業會議的興趣，並探索其他人。 探索之後，使用者就能夠聊天或設定即席會議，以交換想法。
