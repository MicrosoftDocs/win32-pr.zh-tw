# <a name="about-system-shutdown"></a>關於系統關機

系統關機會讓系統能夠安全地關閉電腦。 所有的檔案系統緩衝區都會排清到磁片上，然後會顯示訊息方塊，通知使用者電腦可以關閉。 另外還有一個重新開機選項，將會重新開機電腦，而不會顯示 [系統關機] 訊息方塊。 如需詳細資訊，請參閱 [關機](shutting-down.md)。 如需範例，請參閱 [如何關機系統](how-to-shut-down-the-system.md)。

登出會停止所有與呼叫 exit 函式之進程的安全性內容相關聯的處理常式、將目前的使用者登出系統，並顯示 [登入] 對話方塊。 如需詳細資訊， [請參閱登出](logging-off.md)。 如需範例，請參閱 [如何登出目前的使用者](how-to-log-off-the-current-user.md)。

當您離開電腦時，鎖定工作站會防止未經授權的使用顯示。 若要解除鎖定工作站，您必須登入。 如需範例，請參閱 [如何鎖定工作站](how-to-lock-the-workstation.md)。

[關機事件追蹤](shutdown-event-tracker.md)器可讓使用者或應用程式記錄關閉或重新開機系統的原因。
