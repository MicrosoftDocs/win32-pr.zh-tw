# <a name="id3dx10fontgetdc-method"></a>ID3DX10Font：： GetDC 方法

傳回已將字型設定在其上的顯示裝置內容 (DC) 的控制碼。

## <a name="syntax"></a>語法


```C++
HDC GetDC();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **[ **HDC**](../winprog/windows-data-types.md)**

顯示 DC 的控制碼。
