# <a name="d3dxvector3_16f-structure"></a>D3DXVECTOR3 \_ 16F 結構

與 [**D3DXVECTOR3**](d3d10-d3dxvector3.md)相同，但它會使用 x、y 和 z 的16位浮點值。

## <a name="syntax"></a>語法


```C++
typedef struct D3DXVECTOR3_16F {
  FLOAT x;
  FLOAT y;
  FLOAT z;
} D3DXVECTOR3_16F, *LPD3DXVECTOR3_16F;
```

```

typedef struct D3DXVECTOR3_16F
{
#ifdef __cplusplus
public:
```
