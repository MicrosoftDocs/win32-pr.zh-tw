# <a name="shader-reference-direct3d-10-graphics"></a> (Direct3D 10 圖形) 的著色器參考

Direct3D API 定義了數個 API 元素，可協助您建立及管理可程式化著色器。

-   [著色器介面](d3d10-graphics-reference-d3d10-shader-interfaces.md)
-   [著色器函式](d3d10-graphics-reference-d3d10-shader-functions.md)
-   [著色器結構](d3d10-graphics-reference-d3d10-shader-structures.md)
-   [著色器列舉](d3d10-graphics-reference-d3d10-shader-enums.md)
-   [著色器常數](d3d10-graphics-reference-d3d10-shader-constants.md)

著色器是可執行檔程式，可控制頂點著色器、幾何著色器和圖元著色器 [管線階段](d3d10-graphics-programming-guide-pipeline-stages.md)。 著色器是使用 [HLSL](../direct3dhlsl/dx-graphics-hlsl-reference.md)專門設計的。
