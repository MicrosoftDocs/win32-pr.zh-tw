# <a name="effect-functions-direct3d-10"></a> (Direct3D 10) 的效果函數

本節包含下列效果函數的相關資訊：



| 函式                                                                      | Description                                                         |
|--------------------------------------------------------------------------------|---------------------------------------------------------------------|
| [**D3D10CompileEffectFromMemory**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10compileeffectfrommemory)           | 編譯效果。                                                  |
| [**D3D10CreateEffectFromMemory**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10createeffectfrommemory)             | 建立效果。                                                   |
| [**D3D10CreateEffectPoolFromMemory**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10createeffectpoolfrommemory)     | 建立效果集區。                                              |
| [**D3D10CreateStateBlock**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10createstateblock)                         | 建立狀態欄塊。                                               |
| [**D3D10DisassembleEffect**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10disassembleeffect)                       | 將已編譯的效果分解為著色器元件指示。    |
| [**D3D10StateBlockMaskDifference**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskdifference)         | 將兩個狀態欄塊遮罩與位 XOR 合併。                   |
| [**D3D10StateBlockMaskDisableAll**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskdisableall)         | 停用狀態捕捉。                                            |
| [**D3D10StateBlockMaskDisableCapture**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskdisablecapture) | 使用狀態欄塊遮罩來停用狀態捕捉。                    |
| [**D3D10StateBlockMaskEnableAll**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskenableall)           | 啟用狀態欄塊遮罩來捕捉和套用所有狀態變數。 |
| [**D3D10StateBlockMaskEnableCapture**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskenablecapture)   | 啟用狀態欄塊遮罩中的狀態值範圍。               |
| [**D3D10StateBlockMaskGetSetting**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskgetsetting)         | 取得狀態欄塊遮罩中的元素。                               |
| [**D3D10StateBlockMaskIntersect**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskintersect)           | 結合兩個狀態欄塊遮罩與位 AND。                   |
| [**D3D10StateBlockMaskUnion**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskunion)                   | 結合兩個狀態欄塊遮罩與位 OR。                    |
