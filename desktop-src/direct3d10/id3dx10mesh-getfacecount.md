# <a name="id3dx10meshgetfacecount-method"></a>ID3DX10Mesh：： GetFaceCount 方法

抓取網格中的臉部數目。

## <a name="syntax"></a>語法


```C++
UINT GetFaceCount();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **[ **UINT**](../winprog/windows-data-types.md)**

傳回網格中的臉部數目。
