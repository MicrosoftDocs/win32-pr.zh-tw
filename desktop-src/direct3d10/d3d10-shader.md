# <a name="d3d10_shader-constants"></a>D3D10 \_ 著色器常數

HLSL 編譯選項。



| \#定義                                        | 描述                                                                                                                                                                                                                                    |
|-------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| D3D10 \_ 著色器 \_ 避免 \_ 流量 \_ 控制             | 告知編譯器不允許流量控制 (可能的) 。                                                                                                                                                                                       |
| D3D10 \_ 著色器 \_ 調試                            | 插入 debug 檔案/行/類型/符號資訊。                                                                                                                                                                                                |
| D3D10 \_ 著色器 \_ 啟用 \_ 嚴謹度               | 根據預設，HLSL 編譯器會在已被取代的語法上停用嚴謹度。 指定此旗標可啟用可能不允許舊版語法的嚴謹度。                                                                                         |
| D3D10 \_ 著色器可以提供回溯 \_ \_ \_ 相容性 | 這可讓較舊的著色器編譯為 4 \_ 0 目標。                                                                                                                                                                                         |
| D3D10 \_ 著色器 \_ 強制 \_ VS \_ SOFTWARE \_ 無 \_ 選擇     | 為下一個最高的著色器設定檔編譯頂點著色器。 此選項會開啟 (的調試和) 的優化。                                                                                                                           |
| D3D10 \_ 著色器 \_ 強制 \_ PS \_ 軟體 \_ 無 \_ 選擇     | 為下一個最高的著色器設定檔編譯圖元著色器。 此選項會開啟 (的調試和) 的優化。                                                                                                                            |
| D3D10 \_ 著色器 \_ IEEE \_ 嚴謹度                 | 啟用 IEEE 嚴謹度。                                                                                                                                                                                                                       |
| D3D10 \_ 著色器 \_ 無 \_ PRESHADER                    | 停用 Preshaders。 使用這個旗標會導致編譯器不會提取靜態運算式以進行評估。                                                                                                                                 |
| D3D10 \_ 著色器 \_ 優化 \_ LEVEL0             | 最低優化層級。 可能會產生較慢的程式碼，但會更快完成。 這在高度反復的著色器開發週期中可能很有用。                                                                                             |
| D3D10 \_ 著色器 \_ 優化 \_ 級別1             | 第二個最低優化層級。                                                                                                                                                                                                              |
| D3D10 \_ 著色器 \_ 優化 \_ 級別2             | 第二個最高的優化層級。                                                                                                                                                                                                             |
| D3D10 \_ 著色器 \_ 優化（ \_ 3）             | 最高的優化層級。 會產生最佳的程式碼，但可能需要花費很長的時間才能完成。 這適用于應用程式的最終組建，其中效能是最重要的因素。                                 |
| D3D10 \_ 著色器 \_ 套件 \_ 矩陣的 \_ 主要資料列 \_         | 除非明確指定，否則矩陣將會在著色器的輸入和輸出上以資料列主要順序封裝。                                                                                                                                   |
| D3D10 \_ 著色器 \_ 套件 \_ 矩陣資料 \_ 行 \_ 主要      | 除非明確指定，否則矩陣將會在著色器的輸入和輸出上以資料行主要順序封裝。 這通常更有效率，因為它允許使用一連串的點積來執行向量矩陣乘法。 |
| D3D10 \_ 著色器 \_ 部分 \_ 精確度               | 強制所有計算都使用部分有效位數完成;這在某些硬體上的執行速度可能更快。                                                                                                                                                |
| D3D10 \_ 著色器 \_ 偏好 \_ 流程 \_ 控制            | 告知編譯器使用流程式控制制 (可能的) 。                                                                                                                                                                                             |
| D3D10 \_ 著色器 \_ 略過 \_ 優化               | 在程式碼產生期間略過優化;通常只建議進行 debug。                                                                                                                                                                |
| D3D10 \_ 著色器 \_ 略過 \_ 驗證                 | 請勿針對已知的功能和條件約束來驗證產生的程式碼。 請僅使用已在過去成功編譯的著色器。 著色器在設定為裝置之前，一律會經過 DirectX 驗證。         |
| D3D10 \_ 著色器 \_ 警告 \_ 為 \_ 錯誤            | 通知 HLSL 編譯器在編譯著色器程式碼時，將所有警告視為錯誤。 針對新的著色器程式碼，您應該使用此選項，以便解決所有警告，並確保最少可能難以找到的程式碼缺失。             |



 

這些常數會定義為 d3d10shader 中的宏。
