# <a name="id3dx10meshgetvertexbuffercount-method"></a>ID3DX10Mesh：： GetVertexBufferCount 方法

取得網格中的頂點緩衝區數目。

## <a name="syntax"></a>語法


```C++
UINT GetVertexBufferCount();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **[ **UINT**](../winprog/windows-data-types.md)**

網格中的頂點緩衝區數目。
