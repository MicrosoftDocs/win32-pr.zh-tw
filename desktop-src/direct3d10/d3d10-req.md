# <a name="d3d10_req"></a>D3D10 \_ 需求

資源限制常數。



| \#定義                                      | 值 | 描述                                                                                                                                                                                                                                                                                                                                                                                                                     |
|-----------------------------------------------|-------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| D3D10 \_ 需求 \_ MIP \_ 層級                       | 14    | 材質資源可能擁有的最大 mipmap 層級數目。                                                                                                                                                                                                                                                                                                                                                                    |
| D3D10 \_ 需求 \_ 資源 \_ 大小 \_ \_ （以 mb 為單位）     | 128   | 資源的最大可能大小（以 mb 為單位）。 應用程式可以建立大於某些圖形硬體上資源大小上限的資源。 不過，建議應用程式將資源保留在資源大小上限的範圍內，以取得圖形廠商之間的最大相容性數量。 執行時間只會保證所有 Direct3D 10 硬體都支援資源大小上限內的配置。 |
| D3D10 \_ 需求 \_ TEXTURE1D \_ 陣列 \_ 軸 \_ 維度 | 512   | 1D 材質陣列的陣列大小上限。                                                                                                                                                                                                                                                                                                                                                                                       |
| D3D10 \_ 需求 \_ TEXTURE2D \_ 陣列 \_ 軸 \_ 維度 | 512   | 2D 材質陣列的陣列大小上限。                                                                                                                                                                                                                                                                                                                                                                                       |
| D3D10 \_ 需求 \_ TEXTURE1D \_ U \_ 維度           | 8192  | 1D 材質的最大寬度。                                                                                                                                                                                                                                                                                                                                                                                                  |
| D3D10 \_ 需求 \_ TEXTURE2D \_ U \_ 或 \_ V \_ 維度    | 8192  | 2D 材質的最大寬度和高度。                                                                                                                                                                                                                                                                                                                                                                                       |
| D3D10 \_ 需求 \_ TEXTURE3D \_ U \_ V \_ 或 \_ W \_ 維度 | 2048  | 3D 紋理的最大寬度、高度和深度。                                                                                                                                                                                                                                                                                                                                                                               |
| D3D10 \_ 需求 \_ TEXTURECUBE \_ 維度            | 8192  | Cube 材質的大小上限。                                                                                                                                                                                                                                                                                                                                                                                                 |



 

常數定義于 D3D10 中。
