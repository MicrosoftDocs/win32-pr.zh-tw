# <a name="id3dx10meshgetflags-method"></a>ID3DX10Mesh：： GetFlags 方法

存取網格的建立旗標。

## <a name="syntax"></a>語法


```C++
UINT GetFlags();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **[ **UINT**](../winprog/windows-data-types.md)**

建立網格時，傳遞至 [**D3DX10CreateMesh**](d3d10-d3dx10createmesh.md) *options* 參數的建立旗標。 請參閱 [**D3DX10 \_ 網格**](d3dx10-mesh.md)。
