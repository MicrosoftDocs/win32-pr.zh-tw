# <a name="id3dx10threadpumpgetworkitemcount-method"></a>ID3DX10ThreadPump：： GetWorkItemCount 方法

取得目前線上程抽取中的工作專案數。

## <a name="syntax"></a>語法


```C++
UINT GetWorkItemCount();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **[ **UINT**](../winprog/windows-data-types.md)**

執行緒提取中目前排入佇列的工作專案數。
