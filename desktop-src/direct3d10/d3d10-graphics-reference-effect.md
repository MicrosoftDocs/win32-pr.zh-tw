# <a name="effect-reference-direct3d-10-graphics"></a> (Direct3D 10 圖形) 的效果參考

Direct3D API 定義了數個 API 元素，可協助您建立和管理效果系統。

-   [介面](d3d10-graphics-reference-effect-interfaces.md)
-   [函數](d3d10-graphics-reference-effect-functions.md)
-   [結構](d3d10-graphics-reference-effect-structures.md)
-   [常數](d3d10-graphics-reference-effect-constants.md)

效果系統會將狀態指派封裝在效果中。 每個效果都是管線狀態的集合，可使用狀態欄塊、資源和著色器細分為狀態指派。 根據效果的大小，您可以選擇在 ASCII 文字字串或效果檔案中執行一個（ ( fx) 。 若要在效果中組織資訊，請參閱 [效果格式](d3d10-effect-format.md)。

設計好效果之後，請使用效果 API 在轉譯期間設定裝置中的狀態。 同樣地，效果系統還支援一系列的反映 API 來取得和設定效果資料。 如需詳細資訊，請參閱 [效果系統介面 (Direct3D 10) ](d3d10-graphics-programming-guide-effects-interfaces.md)。
