# <a name="id3dx10meshbuffergetsize-method"></a>ID3DX10MeshBuffer：： GetSize 方法

取得網狀緩衝區的大小（以位元組為單位）。

## <a name="syntax"></a>語法


```C++
SIZE_T GetSize();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

類型： **[**大小 \_ T**](../winprog/windows-data-types.md)**

網狀緩衝區的大小（以位元組為單位）。
