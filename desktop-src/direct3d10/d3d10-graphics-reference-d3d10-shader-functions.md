# <a name="shader-functions-direct3d-10-graphics"></a> (Direct3D 10 圖形) 的著色器函式

本節包含下列著色器函式的相關資訊：



| 函式                                                                          | Description                                                                 |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------|
| [**D3D10CompileShader**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10compileshader)                                   | 編譯著色器。                                                           |
| [**D3D10DisassembleShader**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10disassembleshader)                           | 已取代。 請參閱 [**D3DX10DisassembleShader**](d3dx10disassembleshader.md)。 |
| [**D3D10GetGeometryShaderProfile**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getgeometryshaderprofile)             | 取得最適合指定裝置的幾何著色器設定檔。              |
| [**D3D10GetInputAndOutputSignatureBlob**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getinputandoutputsignatureblob) | 取得包含著色器簽章的緩衝區。                               |
| [**D3D10GetInputSignatureBlob**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getinputsignatureblob)                   | 取得包含著色器輸入特徵標記的緩衝區。                         |
| [**D3D10GetOutputSignatureBlob**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getoutputsignatureblob)                 | 取得包含著色器輸出特徵標記的緩衝區。                        |
| [**D3D10GetPixelShaderProfile**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getpixelshaderprofile)                   | 取得最適合指定裝置的圖元著色器設定檔。                 |
| [**D3D10GetShaderDebugInfo**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getshaderdebuginfo)                         | 從編譯的著色器取得著色器的調試資訊。                               |
| [**D3D10GetVertexShaderProfile**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getvertexshaderprofile)                 | 取得最適合指定裝置的頂點著色器設定檔。                |
| [**D3D10PreprocessShader**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10preprocessshader)                             | 產生包含著色器權杖資料流程的文字字串。               |
| [**D3D10ReflectShader**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10reflectshader)                                   | 已取代。 請參閱 [**D3DX10ReflectShader**](d3dx10reflectshader.md)。         |
