# <a name="d3dxcolor-structure-d3dx10mathh"></a>D3DXCOLOR 結構 (D3DX10Math .h) 

描述色彩值。

## <a name="syntax"></a>語法


```C++
typedef struct D3DXCOLOR {
  FLOAT r;
  FLOAT g;
  FLOAT b;
  FLOAT a;
} D3DXCOLOR, *LPD3DXCOLOR;
```
