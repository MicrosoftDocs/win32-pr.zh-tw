# <a name="d3d10_effect-constants"></a>D3D10 \_ 效果常數

這些常數是在建立效果來定義編譯行為或執行時間效果行為時使用。



| \#定義                                 | 值        | 描述                                                                                                                                                                                                                                                 |
|------------------------------------------|--------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| D3D10 \_ 效果 \_ 編譯 \_ 子 \_ 效果    | 1 << 0 | 將 fx 檔案編譯為子效果。 子效果沒有任何共用值的初始化，因為這些值會在效果集區中初始化。                                                                                                           |
| D3D10 \_ 效果 \_ 編譯 \_ 允許 \_ 緩慢的 \_ OPS | 1 << 1 | 預設會啟用效能模式。 效能模式防止非常值運算式出現在狀態物件定義中，因此不允許可變動狀態物件。 指定此旗標將會停用模式，並允許可變狀態物件。 |
| D3D10 \_ 效果 \_ 單個 \_ 執行緒          | 1 << 3 | 請勿嘗試與其他執行緒進行同步處理，將效果載入相同的集區。                                                                                                                                                                        |



 

這些常數會定義為 d3d10effect 中的宏。
