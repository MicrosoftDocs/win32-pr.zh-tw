# <a name="resource-enumerations-direct3d-10-graphics"></a> (Direct3D 10 圖形) 的資源列舉

下列列舉是用來指定在轉譯期間如何建立和存取資源的相關資訊。



| 列舉型別                                                     | 描述                                                                                                               |
|-----------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------|
| [**D3D10 系結 \_ \_ 旗標**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_bind_flag)                    | 識別管線將如何使用資源，並判斷資源可系結 () 的管線階段。 |
| [**D3D10 \_ CPU \_ 存取 \_ 旗標**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_cpu_access_flag)       | 指定資源允許的 CPU 存取類型。                                                                 |
| [**D3D10 \_ DSV \_ 維度**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_dsv_dimension)            | 指定如何存取深度範本視圖中使用的資源。                                                  |
| [**D3D10 \_ 對應**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_map)                                 | 識別要對應以供 CPU 讀取和寫入的資源。                                                    |
| [**D3D10 \_ 對應 \_ 旗標**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_map_flag)                      | 指定當 GPU 尚未完成資源時，CPU 應該如何回應任何對應方法。               |
| [**D3D10 \_ 資源 \_ 維度**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_resource_dimension)  | 識別使用中的資源類型。                                                                           |
| [**D3D10 \_ 資源 \_ 其他 \_ 旗標**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_resource_misc_flag) | 識別較不常見的資源建立選項。                                                                         |
| [**D3D10 \_ RTV \_ 維度**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_rtv_dimension)            | 指定如何存取呈現目標視圖中使用的資源。                                                          |
| [**D3D10 \_ 使用方式**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_usage)                             | 識別 CPU 和/或 GPU 應如何讀取和寫入資源。                                   |
