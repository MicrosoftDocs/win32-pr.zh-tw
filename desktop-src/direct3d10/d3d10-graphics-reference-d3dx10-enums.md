# <a name="d3dx-enumerations-direct3d-10-graphics"></a>D3DX (Direct3D 10 圖形) 的列舉

本節包含下列與 D3DX 搭配使用之列舉類型和旗標的相關資訊：



| 列舉型別                                                       | 描述                                                                                                                                                                            |
|-------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**D3DX10 \_ 通道 \_ 旗標**](d3dx10-channel-flag.md)              | 這些旗標會由函式使用，這些函式會在材質中的一或多個通道上運作。                                                                                                  |
| [**D3DX \_ CPU \_ 優化**](d3dx-cpu-optimization.md)          | 指定目前針對 D3DX 優化的指令集。                                                                                                                         |
| [**D3DX10 \_ 錯誤**](d3dx10-err.md)                                 | 錯誤會以負數值表示，而且無法合併。                                                                                                                      |
| [**D3DX10 \_ 篩選 \_ 旗標**](d3dx10-filter-flag.md)                | 材質篩選旗標。                                                                                                                                                               |
| [**D3DX10 \_ 圖像 \_ 檔案 \_ 格式**](d3dx10-image-file-format.md)   | 描述支援的影像檔案格式。                                                                                                                                            |
| [**D3DX10 \_ 網格**](d3dx10-mesh.md)                               | 用來指定網格之建立選項的旗標。                                                                                                                                     |
| [**D3DX10 \_ 網格 \_ 捨棄 \_ 旗標**](d3dx10-mesh-discard-flags.md) | 指定要從裝置捨棄哪些網格資料片段。 搭配 [**ID3DX10Mesh 使用：:D iscard**](id3dx10mesh-discard.md)。                                                         |
| [**D3DX10 \_ MESHOPT**](d3dx10-meshopt.md)                         | 指定要執行的網格優化類型。                                                                                                                               |
| [**D3DX10 \_ NORMALMAP \_ 旗標**](d3dx10-normalmap-flag.md)          | 這些旗標是用來控制 [**D3DX10ComputeNormalMap**](d3dx10computenormalmap.md) 產生一般對應的方式。 這些旗標的任意數目可能會以任何組合方式組合。 |
| [**D3DX10 \_ 儲存 \_ 材質 \_ 旗標**](d3dx10-save-texture-flag.md)   | 材質儲存選項。                                                                                                                                                                  |
