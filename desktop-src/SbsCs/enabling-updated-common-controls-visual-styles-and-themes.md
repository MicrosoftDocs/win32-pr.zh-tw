# <a name="enabling-updated-common-controls-visual-styles-and-themes"></a>啟用更新的通用控制項、視覺化樣式和主題

在 Windows XP 上執行的應用程式開發人員可以藉由包含描述應用程式對 COMCTL32.DLL 6.0 版相依性的資訊清單，將視覺樣式和主題新增至使用者介面 (UI) 。 應用程式需要已更新通用控制項的功能，才能使用主題和視覺化樣式。

如需使用視覺化樣式的詳細資訊，請參閱使用 Shell 程式設計[人員手冊](/previous-versions/windows/desktop/legacy/bb776778(v=vs.85))中的 Windows XP 視覺效果樣式。 包含 COMCTL32.DLL 版本6.0 的並存元件是 [支援的 Microsoft 並存元件](supported-microsoft-side-by-side-assemblies.md)之一。
