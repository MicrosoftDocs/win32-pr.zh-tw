# <a name="side-by-side-assembly-functions"></a>並存元件函數

並存元件 API 會使用下列函式來存取並存元件介面。



| 函式                                                     | 描述                                                                    |
|--------------------------------------------------------------|--------------------------------------------------------------------------------|
| [**CreateAssemblyCache**](/windows/desktop/api/Winsxs/nf-winsxs-createassemblycache)           | 取得 [**IAssemblyCache**](/windows/desktop/api/winsxs/nn-winsxs-iassemblycache) 介面的實例。 |
| [**CreateAssemblyNameObject**](/windows/desktop/api/Winsxs/nf-winsxs-createassemblynameobject) | 取得 [**IAssemblyName**](/windows/desktop/api/winsxs/nn-winsxs-iassemblyname) 介面的實例。   |
