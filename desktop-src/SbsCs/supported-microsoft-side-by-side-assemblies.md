# <a name="supported-microsoft-side-by-side-assemblies"></a>支援的 Microsoft 並存元件

以下是 Microsoft 支援的並存元件。 WindowsService Pack 可能包含這些元件的相容版本，以及安全性或錯誤修正。

您可以在 [Microsoft 下載中心](https://www.microsoft.com/Downloads/)搜尋可用的可轉散發套件。

## <a name="windows-server-2003"></a>Windows Server 2003

從 Windows Server 2003 開始，Microsoft 支援這些並存元件。

-   WinHTTP 5。1
-   Microsoft 隔離自動化元件，可讓您搭配腳本使用並存元件。
-   Shell 通用控制項 6.0 (Comctl32.dll 版本) 
-   GDI Plus 1.0 版 (GDIplus.dll) 
-   Visual C++ 執行時間程式庫版本6。0

## <a name="windows-xp"></a>Windows XP

從 Windows XP 開始，Microsoft 支援這些並存元件。

-   Shell 通用控制項 6.0 (Comctl32.dll 版本) 
-   GDI Plus 1.0 版 (GDIplus.dll) 
-   Visual C++ 執行時間程式庫版本6。0
