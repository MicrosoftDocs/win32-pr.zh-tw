# <a name="side-by-side-assembly-interfaces"></a>並存元件介面

並存元件 API 會使用下列介面。



| 介面                                        | 描述                                                                                                                                            |
|--------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IAssemblyCache**](/windows/desktop/api/winsxs/nn-winsxs-iassemblycache)         | 用來安裝、卸載或查詢並存元件。                                                                                          |
| [**IAssemblyCacheItem**](/windows/desktop/api/winsxs/nn-winsxs-iassemblycacheitem) | 用來將並存元件安裝到並存存放區，並使用以資料流程為基礎的安裝。                                                 |
| [**IAssemblyName**](/windows/desktop/api/winsxs/nn-winsxs-iassemblyname)           | 表示並存元件名稱。 並存元件名稱是由一組成對的名稱/值組所組成，描述並存元件。 |
