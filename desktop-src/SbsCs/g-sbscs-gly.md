# <a name="g-isolated-applications-and-side-by-side-assemblies"></a>G (隔離的應用程式和並存元件) 

[B C](a-sbscs-gly.md) [D](d-sbscs-gly.md) E F G H [I](i-sbscs-gly.md) J K L [M](m-sbscs-gly.md) N O [P](p-sbscs-gly.md) Q [R](r-sbscs-gly.md) [S](s-sbscs-gly.md) T W X Y Z

<dl> <dt>

<span id="_win32_global_application_configuration_gly"></span><span id="_WIN32_GLOBAL_APPLICATION_CONFIGURATION_GLY"></span>**全域應用程式設定**
</dt> <dd>

規格：系統上的所有應用程式都使用相同的並存元件版本。 全域設定是針對安裝在 Winsxs 資料夾中的全域組件資訊清單檔，以每個元件為基礎來指定。 當元件開發人員或系統管理員需要將系統上的所有應用程式設定為使用較新的元件版本時，可能會使用全域設定。 在此情況下，新的元件必須具有回溯相容性。 針對特定應用程式，每個應用程式的設定會覆寫預設或全域應用程式設定。

</dd> <dt>

<span id="_win32_global_assembly_manifest_gly"></span><span id="_WIN32_GLOBAL_ASSEMBLY_MANIFEST_GLY"></span>**全域組件資訊清單**
</dt> <dd>

定義並存元件之全域應用程式設定的檔案。 全域組件資訊清單檔會安裝到 Winsxs 資料夾中的系統元件存放區。

</dd> </dl>
