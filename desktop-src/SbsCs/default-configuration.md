# <a name="default-configuration"></a>預設組態

[元件資訊清單](assembly-manifests.md)和[應用程式資訊清單](application-manifests.md)中指定的元件版本，包含應用程式執行時應用程式應使用的預設設定。 應用程式和系統管理員可以在 Windows Server 2003 上安裝[發行者](publisher-configuration.md)設定或個別[應用程式](per-application-configuration.md)設定，以覆寫預設元件設定。
