# <a name="installing-side-by-side-assemblies"></a>安裝並存元件

您可以安裝並存元件作為 [共用元件](/windows/desktop/Msi/shared-assemblies) 或 [私用元件](/windows/desktop/Msi/private-assemblies)。 如需詳細資訊，請參閱將 [並存元件安裝為私用元件](installing-side-by-side-assemblies-as-private-assemblies.md) ，並 [安裝並存元件作為共用元件](installing-side-by-side-assemblies-as-shared-assemblies.md)。

請注意：

-   使用每個使用者[安裝內容](/windows/desktop/Msi/installation-context)安裝到全域組件快取的元件，不會受到 Windows 檔案保護的保護。
-   使用每部電腦[安裝內容](/windows/desktop/Msi/installation-context)安裝到全域組件快取的元件，會受到 Windows 檔案保護的保護。
