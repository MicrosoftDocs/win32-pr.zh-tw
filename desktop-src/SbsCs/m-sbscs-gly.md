# <a name="m-isolated-applications-and-side-by-side-assemblies"></a>M (隔離的應用程式和並存元件) 

[B C](a-sbscs-gly.md) [D](d-sbscs-gly.md) E F [G](g-sbscs-gly.md) H [I](i-sbscs-gly.md) J K L M N O [P](p-sbscs-gly.md) Q [R](r-sbscs-gly.md) [S](s-sbscs-gly.md) T W X Y Z

<dl> <dt>

<span id="_win32_manifest_file_gly"></span><span id="_WIN32_MANIFEST_FILE_GLY"></span>**資訊清單檔**
</dt> <dd>

描述並存元件或隔離應用程式的 XML 檔。 資訊清單檔案可能包含傳統儲存在登錄中的系結和啟用中繼資料，例如 COM 類別、介面和型別程式庫。 每個隔離的應用程式或並存元件都會伴隨資訊清單檔案。 應用程式資訊清單描述完整或部分隔離的應用程式，而組件資訊清單則描述並存元件。

</dd> <dt>

<span id="_win32_metadata_gly"></span><span id="_WIN32_METADATA_GLY"></span>**元**
</dt> <dd>

儲存在資訊清單檔中的資訊，有關隔離的應用程式或並存元件的系結或啟用。 例如，以傳統方式儲存在登錄中的 COM 類別、介面和類型程式庫的相關資訊，可能會以中繼資料的形式儲存在資訊清單中。 中繼資料會描述構成元件的專案，以及其對其他並存元件的相依性。 應用程式資訊清單和相依元件中指定的中繼資料，可用來填入啟用內容。

</dd> </dl>
