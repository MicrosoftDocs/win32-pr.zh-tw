# <a name="developer-license-apis"></a>開發人員授權 Api

使用這些 Api 來檢查、取得和移除開發人員授權。

## <a name="in-this-section"></a>本節內容



| 主題                                                                 | 描述                                             |
|-----------------------------------------------------------------------|---------------------------------------------------------|
| [**CheckDeveloperLicense**](/windows/desktop/api/wsdevlicensing/nf-wsdevlicensing-checkdeveloperlicense)<br/>     | 查看開發人員授權是否存在。<br/> |
| [**AcquireDeveloperLicense**](/windows/desktop/api/wsdevlicensing/nf-wsdevlicensing-acquiredeveloperlicense)<br/> | 取得開發人員授權。<br/>                |
| [**RemoveDeveloperLicense**](/windows/desktop/api/wsdevlicensing/nf-wsdevlicensing-removedeveloperlicense)<br/>   | 移除開發人員授權。<br/>                 |
