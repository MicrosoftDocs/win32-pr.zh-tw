# <a name="bluetooth-and-getpeername"></a>藍牙和 getpeername

[**Getpeername**](/windows/desktop/api/winsock/nf-winsock-getpeername)函式是用來取得對等藍牙裝置的藍牙位址。 **Getpeername** 函式僅適用于連接的藍牙通訊端。

如需詳細資訊，請參閱[Windows 通訊端](/windows/desktop/WinSock/windows-sockets-start-page-2)。
