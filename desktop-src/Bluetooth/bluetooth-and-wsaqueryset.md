# <a name="bluetooth-and-wsaqueryset"></a>藍牙和 WSAQUERYSET

[**WSAQUERYSET**](/windows/desktop/api/winsock2/ns-winsock2-wsaquerysetw)結構用於作業中，包括裝置查詢、服務查詢和設定服務。

下列主題將說明在每個作業中使用 WSAQUERYSET 結構的方式：

-   [適用于裝置查詢的藍牙和 WSAQUERYSET](bluetooth-and-wsaqueryset-for-device-inquiry.md)
-   [用於服務查詢的藍牙和 WSAQUERYSET](bluetooth-and-wsaqueryset-for-service-inquiry.md)
-   [適用于設定服務的藍牙和 WSAQUERYSET](bluetooth-and-wsaqueryset-for-set-service.md)

如需詳細資訊和完整的 [**WSAQUERYSET**](/windows/desktop/api/winsock2/ns-winsock2-wsaquerysetw) 參考，或其他 Windows 通訊端元素的詳細資訊，請參閱平臺軟體發展工具組 (SDK) 的 Windows 通訊端一節。
