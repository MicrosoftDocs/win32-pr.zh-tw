# <a name="bluetooth-and-shutdown"></a>藍牙和關機

藍牙使用 [**shutdown**](/windows/desktop/api/winsock/nf-winsock-shutdown)功能中斷與遠端無線電的連線。 藍牙沒有 TCP/IP 提供的相同正常關閉或半關閉的通訊協定層級支援。
