# <a name="bluetooth-and-read-or-write-operations"></a>藍牙和讀取或寫入作業

藍牙能夠執行 Windows 通訊端提供的所有讀取和寫入作業，這些作業目前支援其他的位址系列。

如需有關讀取和寫入作業的詳細資訊，請參閱 [Windows 通訊端](/windows/desktop/WinSock/windows-sockets-start-page-2)。
