# <a name="bluetooth-and-read-or-write-operations"></a>藍牙和讀取或寫入作業

藍牙能夠執行與其他位址系列目前支援的 Windows 通訊端所提供的所有讀取和寫入作業。

如需讀取和寫入作業的詳細資訊，請參閱[Windows 通訊端](/windows/desktop/WinSock/windows-sockets-start-page-2)。
