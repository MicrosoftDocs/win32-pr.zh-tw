# <a name="bluetooth"></a>Bluetooth

## <a name="purpose"></a>目的

藍牙是業界標準的通訊協定，可讓電腦、掌上型裝置、行動電話和其他裝置使用無線連線能力。

## <a name="where-applicable"></a>適用時

藍牙應用程式開發介面可讓開發人員使用現有的網路程式設計知識，快速開發或移植應用程式。

## <a name="developer-audience"></a>開發人員對象

藍牙是設計來供 C/c + + 程式設計人員使用。 Windows 通訊端提供某些藍牙功能。 需要熟悉 Microsoft Windows 網路和 Windows 通訊端程式設計。

## <a name="run-time-requirements"></a>執行階段需求求

Microsoft 藍牙支援從 Windows XP Service Pack 1 開始 (SP1) 。

Windows Vista SP2 和 Windows 7 提供藍牙2.1 的支援。
