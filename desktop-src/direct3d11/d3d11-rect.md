# <a name="d3d11_rect"></a>D3D11 \_ RECT

D3D11 \_ RECT 的宣告方式如下：

``` syntax
typedef RECT D3D11_RECT;
```

如需有關此 GDI 矩形結構的詳細資訊，請參閱 [**RECT**](/previous-versions//dd162897(v=vs.85))。
