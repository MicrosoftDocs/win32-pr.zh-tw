# <a name="d3d11_primitive_topology-enumeration"></a>D3D11 \_ 基本 \_ 拓撲列舉

管線如何解讀系結至輸入組合語言階段的頂點資料。 這些基本拓撲值會決定在螢幕上呈現頂點資料的方式。

## <a name="syntax"></a>Syntax


```C++
} D3D11_PRIMITIVE_TOPOLOGY;
```
