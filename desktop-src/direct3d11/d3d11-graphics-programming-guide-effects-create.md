# <a name="create-an-effect-direct3d-11"></a> (Direct3D 11) 建立效果

效果是藉由將編譯的效果位元組程式碼載入效果架構來建立。 不同于效果10，必須先編譯效果，才能建立效果。 您可以藉由呼叫 [**D3DX11CreateEffectFromMemory**](d3dx11createeffectfrommemory.md)來建立載入記憶體的效果。
