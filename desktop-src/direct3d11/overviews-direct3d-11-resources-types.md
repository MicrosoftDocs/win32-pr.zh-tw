# <a name="types-of-resources"></a>資源類型

資源是在記憶體中的區域，可由 Direct3D 管線存取，它們是您場景的構成要素。 資源包含許多類型的資料，例如 Direct3D 用來填入和轉譯場景的幾何、紋理或著色器資料。 本主題說明 Direct3D 10 中的資源類型，以及 Direct3D 11 中的新類型，包括結構化緩衝區和可寫入的材質和緩衝區。
