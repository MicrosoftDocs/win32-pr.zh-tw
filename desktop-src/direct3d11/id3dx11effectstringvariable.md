# <a name="id3dx11effectstringvariable-interface"></a>ID3DX11EffectStringVariable 介面

字串變數介面會存取字串變數。

## <a name="members"></a>成員

**ID3DX11EffectStringVariable** 介面繼承自 [**ID3DX11EffectVariable**](id3dx11effectvariable.md)。 **ID3DX11EffectStringVariable** 也有下列類型的成員：

-   [方法](#methods)

### <a name="methods"></a>方法

**ID3DX11EffectStringVariable** 介面具有這些方法。



| 方法                                                               | 描述                         |
|:---------------------------------------------------------------------|:------------------------------------|
| [**GetString**](id3dx11effectstringvariable-getstring.md)           | 取得字串。<br/>          |
| [**GetStringArray**](id3dx11effectstringvariable-getstringarray.md) | 取得字串的陣列。<br/> |



 

## <a name="remarks"></a>備註

> [!Note]  
> DirectX SDK 不提供任何已編譯的二進位檔來產生效果。 您必須使用效果11來源來建立效果類型的應用程式。 如需使用效果11來源的詳細資訊，請參閱 [效果的差異10和效果 11](d3d11-graphics-programming-guide-effects-differences.md)。
