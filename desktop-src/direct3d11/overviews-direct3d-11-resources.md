# <a name="resources"></a>資源

資源將資料提供給管線並定義您場景期間轉譯的項目。 從您的遊戲媒體載入或在執行階段動態建立資源。 通常，資源包括紋理資料、頂點資料和著色器資料。 大部分的 Direct3D 應用程式會在其存留期內廣泛地建立和終結資源。 本節說明 Direct3D 11 資源的各個層面。
