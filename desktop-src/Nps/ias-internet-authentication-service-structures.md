# <a name="nps-extensions-structures"></a>NPS 擴充功能結構

> [!Note]  
> 網際網路驗證服務 (IAS) 已重新命名網路原則伺服器 (NPS) 從 Windows Server 2008 開始。 本主題的內容適用于 IAS 和 NPS。

 

開發 NPS 擴充 Dll 時，請使用 [**radius \_ 屬性**](/windows/desktop/api/authif/ns-authif-radius_attribute) 結構來代表 radius 屬性或擴充屬性。

-   [**RADIUS \_ 屬性**](/windows/desktop/api/authif/ns-authif-radius_attribute)
-   [**RADIUS \_ 屬性 \_ 陣列**](/windows/desktop/api/authif/ns-authif-radius_attribute_array)
-   [**RADIUS \_ 延伸 \_ \_ 模組控制區塊**](/windows/desktop/api/authif/ns-authif-radius_extension_control_block)
-   [**RADIUS \_ VSA \_ 格式**](/windows/desktop/api/authif/ns-authif-radius_vsa_format)
