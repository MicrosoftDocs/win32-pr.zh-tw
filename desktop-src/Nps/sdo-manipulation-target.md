# <a name="manipulation-target"></a>Manipulation-Target

> [!Note]  
> 網際網路驗證服務 (IAS) 已重新命名網路原則伺服器 (NPS) 從 Windows Server 2008 開始。

 

套用屬性操作規則的目標資料。 屬性操作先前稱為「領域處理」。 如需詳細資訊，請參閱 [領域處理和屬性操作](/windows/desktop/Nps/sdo-realms-processing-and-attribute-manipulation)。

**LDAP 名稱：** msManipulationTarget

**識別碼：** 4113 = [ **IAS \_ 屬性 \_ 操作 \_ 目標**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**語法：** 枚舉 數

* * 值： * *

由網際網路指派的號碼授權單位定義 (IANA) 在 [RADIUS 類型](https://www.iana.org/assignments/radius-types) 檔中，以 [RFC 2865](https://www.ietf.org/rfc/rfc2865.txt)為基礎。

-   User-Name (1) 
-   被呼叫的工作站識別碼 (30) 
-   呼叫站-識別碼 (31)
