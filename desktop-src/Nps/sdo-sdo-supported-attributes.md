# <a name="sdo-supported-attributes"></a>SDO 支援的屬性

> [!Note]  
> 從 Windows Server 2008 開始， (IAS) 的網際網路驗證服務已重新命名為網路原則伺服器 (NPS) 。

 

NPS 伺服器資料物件具有 [**設定檔**](/windows/desktop/api/sdoias/ne-sdoias-profileproperties) 屬性，可提供屬性集合的存取權。 支援的屬性如下所示。

-   [帳戶提供者-名稱](/windows/desktop/Nps/sdo-acct-provider-name)
-   [帳戶提供者-類型](/windows/desktop/Nps/sdo-acct-provider-type)
-   [驗證提供者-名稱](/windows/desktop/Nps/sdo-auth-provider-name)
-   [驗證提供者-類型](/windows/desktop/Nps/sdo-auth-provider-type)
-   [忽略-使用者撥入-屬性](/windows/desktop/Nps/sdo-ignore-user-dialin-properties)
-   [操作-規則](/windows/desktop/Nps/sdo-manipulation-rule)
-   [操作-目標](/windows/desktop/Nps/sdo-manipulation-target)
-   [NP-允許-EAP-類型](/windows/desktop/Nps/sdo-np-allowed-eap-type)
-   [NP 允許-埠類型](/windows/desktop/Nps/sdo-np-allowed-port-types)
-   [NP-驗證-類型](/windows/desktop/Nps/sdo-np-authentication-type)
-   [NP-所謂的工作站識別碼](/windows/desktop/Nps/sdo-np-called-station-id)
