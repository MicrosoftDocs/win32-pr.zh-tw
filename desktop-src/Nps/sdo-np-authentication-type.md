# <a name="np-authentication-type"></a>NP-驗證-類型

允許連接的驗證類型。

**LDAP 名稱：** msNPAuthenticationType2

**識別碼：** 4105 = [ **IAS \_ 屬性 \_ NP \_ 驗證 \_ 類型**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**語法：** 整數

**值：  **

定義于 SdoIas .h 公用標頭檔中宣告的 [**驗證 \_ 類型**](/windows/desktop/api/SdoIas/ne-sdoias-authentication_type) 列舉類型。

-   **IAS \_驗證 \_ 無效** (0) 
-   **IAS \_AUTH \_ PAP** (1) 
-   **IAS \_AUTH \_ MD5CHAP** (2) 
-   **IAS \_AUTH \_ MSCHAP** (3) 
-   **IAS \_AUTH \_ MSCHAP2** (4) 
-   **IAS \_驗證 \_ EAP** (5) 
-   **IAS \_AUTH \_ ARAP** (6) 
-   **IAS \_AUTH \_ NONE** (7) 
-   **IAS \_驗證 \_ 自訂** (8) 
-   **IAS \_AUTH \_ MSCHAP \_ CPW** (9) 
-   **IAS \_AUTH \_ MSCHAP2 \_ CPW** (10) 
-   **IAS \_驗證 \_ PEAP** (11)
