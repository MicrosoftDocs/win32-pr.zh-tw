# <a name="server-data-objects-interfaces"></a>伺服器資料物件介面

伺服器資料物件 (SDO) API 會公開下列介面。




| 介面 | 目的 | 
|-----------|---------|
| <a href="/windows/desktop/api/sdoias/nn-sdoias-isdo"><strong>ISdo</strong></a> | 操作 SDO 物件。<br /> | 
| <a href="/windows/desktop/api/sdoias/nn-sdoias-isdocollection"><strong>ISdoCollection</strong></a> | 操作 SDO 物件的集合。<br /> | 
| <a href="/windows/desktop/api/sdoias/nn-sdoias-isdodictionaryold"><strong>ISdoDictionaryOld</strong></a> | 操作屬性字典。<br /> | 
| <a href="/windows/desktop/api/sdoias/nn-sdoias-isdomachine"><strong>ISdoMachine</strong></a> | 管理 SDO 電腦，並取出 SDO 物件。<br /> | 
| <a href="/windows/desktop/api/sdoias/nn-sdoias-isdoservicecontrol"><strong>ISdoServiceControl</strong></a> | 管理透過 SDO 管理的服務，也就是網際網路驗證服務 (IAS) 或遠端存取服務器 (RAS) 。<br /><blockquote>[!Note]<br />網際網路驗證服務 (IAS) 已重新命名網路原則伺服器 (NPS) 從 Windows Server 2008 開始。</blockquote><br /> |
