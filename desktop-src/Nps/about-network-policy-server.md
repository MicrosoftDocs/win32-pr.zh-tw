# <a name="about-network-policy-server"></a>關於網路原則伺服器

> [!Note]  
> 從 Windows Server 2008 開始， (IAS) 的網際網路驗證服務已重新命名為網路原則伺服器 (NPS) 。 本主題的內容適用于 IAS 和 NPS。

 

網路原則伺服器是 Microsoft 對 RADIUS 伺服器和 proxy 的實作為。 下列各節提供 NPS 和 RADIUS 的一般資訊。

-   [網際網路驗證服務和網路原則伺服器](internet-authentication-service-vs-network-policy-server.md)
-   [RADIUS 驗證、授權和帳戶處理](/windows/desktop/Nps/ias-radius-authentication-and-accounting)
-   [使用網路原則伺服器進行記錄](/windows/desktop/Nps/ias-radius-accounting-packets)
-   [使用狀態伺服器](/windows/desktop/Nps/ias-working-with-a-state-server)
