# <a name="servervalidation-eaptype-element-peap"></a>ServerValidation (EapType) 元素 (PEAP) 

**ServerValidation (EapType)** 元素包含如何執行伺服器驗證的相關資訊。

``` syntax
<xs:element name="ServerValidation"
    type="ServerValidationParameters"
 />
```

**ServerValidation** 元素是由 [**EapType**](mspeapconnectionpropertiesv1schema-eaptype-element.md)元素定義。

## <a name="remarks"></a>備註

**ServerValidation** 元素是選擇性的。
