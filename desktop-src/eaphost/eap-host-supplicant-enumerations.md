# <a name="eaphost-supplicant-enumerations"></a>EAPHost 要求者列舉

EAPHost 要求者 API 列舉如下所示。



| 列舉型別                                                                  | 描述                                                                                                        |
|------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| [**EAP \_ 互動式 \_ UI \_ 資料 \_ 類型**](/windows/desktop/api/eaptypes/ne-eaptypes-eap_interactive_ui_data_type)     | 指定提供給特定要求者 API 呼叫的互動式使用者介面內容資料類型集合。    |
| [**EAP \_ 方法 \_ 屬性 \_ 類型**](/windows/desktop/api/EapTypes/ne-eaptypes-eap_method_property_type)              | Windows 7 或更新版本：指定方法屬性類型。                                                            |
| [**EAP \_ 方法 \_ 屬性 \_ 值 \_ 類型**](/windows/desktop/api/EapTypes/ne-eaptypes-eap_method_property_value_type) | Windows 7 或更新版本：指定方法屬性值的資料類型。                                            |
| [**EAPHOST \_ AUTH \_ 狀態**](/windows/desktop/api/eaphostpeertypes/ne-eaphostpeertypes-eaphost_auth_status)                         | 定義一組可能的 EAP 驗證會話狀態值。                                              |
| [**EapHostPeerAuthParams**](/windows/win32/api/eaphostpeertypes/ne-eaphostpeertypes-eaphostpeerauthparams)                       | 定義可能的驗證參數值集。                                                       |
| [**EapHostPeerMethodResultReason**](/windows/win32/api/eaphostpeertypes/ne-eaphostpeertypes-eaphostpeermethodresultreason)       | 定義一組可能的原因，描述 EAP 方法傳回給要求者的結果。           |
| [**EapHostPeerResponseAction**](/windows/win32/api/eaphostpeertypes/ne-eaphostpeertypes-eaphostpeerresponseaction)               | 定義 EAP 驗證器或對等方法可在驗證期間向要求者指出的一組動作。 |
| [**隔離 \_ 狀態**](/windows/desktop/api/eaphostpeertypes/ne-eaphostpeertypes-isolation_state)                                  | 定義電腦的一組可能的隔離狀態值。                                                  |
