# <a name="eap-element-user-property"></a>Eap 元素 (使用者屬性) 

**Eap** 專案會捕捉選取的方法型別和方法特定的設定。

``` syntax
<xs:element name="Eap"
    type="BaseEapParameters"
 />
```

## <a name="remarks"></a>備註

方法可以定義 **Eap** 元素內的組成元素。 方法也會對 **Eap** 中的元素執行架構驗證。
