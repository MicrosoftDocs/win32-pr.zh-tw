# <a name="eaphost-peer-method-enumerations"></a>EAPHost 對等方法列舉

EAPHost 對等方法 API 列舉如下所示。



| 列舉型別                                                        | 描述                                                                                                                |
|--------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------|
| [**EapCredentialType**](/windows/desktop/api/eaptypes/ns-eaptypes-eapcredential)                         | 定義 EAP 認證類型和適當的認證。                                                          |
| [**EapPeerMethodResponseAction**](/windows/win32/api/eapauthenticatoractiondefine/ne-eapauthenticatoractiondefine-eappeermethodresponseaction) | 定義 EAP 驗證器可在驗證期間向要求者或 EAP 對等方法指出的動作集合。     |
| [**EapPeerMethodResultReason**](/windows/win32/api/eapauthenticatoractiondefine/ne-eapauthenticatoractiondefine-eappeermethodresultreason)     | 將 EAP 驗證器方法所傳回之 EAP 驗證會話的結果集定義為 EAP 對等方法。 |
