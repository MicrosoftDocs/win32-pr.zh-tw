# <a name="identity-element"></a>Identity 元素

**Identity** 元素會捕捉用於驗證的身分識別。

``` syntax
<xs:element name="Identity"
    type="string"
 />
```

## <a name="remarks"></a>備註

**標識** 元素是抽象的;每個方法都必須定義和使用實例檔中的衍生元素。
