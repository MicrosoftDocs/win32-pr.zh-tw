# <a name="type-baseeapparameters-element-for-user-properties"></a>輸入使用者屬性 (BaseEapParameters) 元素

**類型 (BaseEapParameters)** 專案會定義所選方法類型的預留位置元素，以及特定方法的認證。

``` syntax
<xs:element name="Type"
    type="integer"
 />
```

**Type** 元素是由 [**BaseEapParameters**](baseeapuserpropertiesv1schema-baseeapparameters-complextype.md)複雜型別定義。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|-------------------------------------|------------------------------------------------------|
| 最低支援的用戶端<br/> | \[僅限 Windows Vista 桌面應用程式\]<br/>       |
| 最低支援的伺服器<br/> | 僅限 Windows Server 2008 \[ desktop 應用程式\]<br/> |
