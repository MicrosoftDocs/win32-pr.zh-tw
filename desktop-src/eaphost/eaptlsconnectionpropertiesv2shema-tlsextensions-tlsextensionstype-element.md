# <a name="performservervalidation-element"></a>PerformServerValidation 元素

**PerformServerValidation (EapType)** 元素指出是否執行伺服器驗證。

``` syntax
<xs:element name="PerformServerValidation"
    type="xs:boolean"
 />
```

**PerformServerValidation** 元素是由 [**EapType**](eaptlsconnectionpropertiesv1schema-eaptype-element.md)元素定義。

## <a name="remarks"></a>備註

**PerformServerValidation** 元素是選擇性的。

## <a name="requirements"></a>規格需求



| 角色 | 最低 OS 版本 |
|------|--------------------|
| 用戶端<br/> | 僅 Windows 7 \[ 桌面應用程式\]<br/>              |
| 伺服器<br/> | Windows僅限 Server 2008 R2 \[ desktop 應用程式\]<br/> |
