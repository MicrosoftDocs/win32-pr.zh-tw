# <a name="eapmethod-eaphostusercredentials-element"></a>EapMethod (EapHostUserCredentials) 元素

**EapMethod (EapHostUserCredentials)** 元素會識別所參考的方法。

``` syntax
<xs:element name="EapMethod"
    type="EapMethodType"
 />
```

**EapMethod** 元素是由 [**EapHostUserCredentials**](eaphostusercredentialsschema-eaphostusercredentials-element.md)元素定義。
