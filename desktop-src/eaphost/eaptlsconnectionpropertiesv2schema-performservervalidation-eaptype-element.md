# <a name="tlsextensions-v2-schema"></a>TLSExtensions (v2 架構) 

**TLSExtensions (TLSExtensionsType)** 專案可讓架構的未來增強功能。

``` syntax
<xs:element name="TLSExtensions"
    type="TLSExtensionsType"
 />
```

## <a name="remarks"></a>備註

**TLSExtensions** 元素是選擇性的。
