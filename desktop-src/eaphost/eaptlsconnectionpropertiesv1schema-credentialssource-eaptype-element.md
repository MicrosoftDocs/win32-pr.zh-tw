# <a name="credentialssource-eaptype-element"></a>CredentialsSource (EapType) 元素

**CredentialsSource (EapType)** 元素包含 eap-tls 憑證位置的相關資訊。

``` syntax
<xs:element name="CredentialsSource"
    type="CredentialsSourceParameters"
 />
```

**CredentialsSource** 元素是由 [**EapType**](eaptlsconnectionpropertiesv1schema-eaptype-element.md)元素定義。

## <a name="remarks"></a>備註

**CredentialSource** 元素是選擇性的。
