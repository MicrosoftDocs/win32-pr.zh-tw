# <a name="sso-and-plap"></a>SSO 和 PLAP

單一 Sign-On (SSO) API 代表一組方法，用來以安全的方式取得網路使用者或電腦帳戶的 EAP 方法專屬認證，而不需要引發多個 UI 實例。

> [!Note]  
> 登入前存取提供者 (PLAP) 已淘汰。 如果在程式碼或檔中遇到「預先登入存取提供者」或「PLAP」，請分別以「單一登入」或「SSO」取代。
