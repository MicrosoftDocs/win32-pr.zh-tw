# <a name="baseeaptypeparameters-complex-type---connection-properties"></a>BaseEapTypeParameters 複雜類型-連接屬性

**BaseEapTypeParameters** 複雜型別是方法特定專案的基底元素，此元素會出現在 [BaseEapConnectionPropertiesV1 架構](baseeapconnectionpropertiesv1schema-schema.md)的衍生元素中。

``` syntax
<xs:complexType name="BaseEapTypeParameters"
    abstract="true"
 />
```

## <a name="remarks"></a>備註

**BaseEapTypeParameters** 是抽象元素。 **BaseEapTypeParameters** 的衍生元素必須在實例檔中使用。
