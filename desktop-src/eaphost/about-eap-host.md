# <a name="about-eaphost"></a>關於 EAPHost

根據 [RFC 3748](https://go.microsoft.com/fwlink/p/?linkid=84063)，EAPHost 會 (eap) 狀態電腦和 eap 通訊協定架構來實行可延伸的驗證通訊協定。

EAPHost 支援以 EAP 為基礎的 [網路存取保護](/windows/desktop/NAP/network-access-protection-start-page) (NAP) 。
