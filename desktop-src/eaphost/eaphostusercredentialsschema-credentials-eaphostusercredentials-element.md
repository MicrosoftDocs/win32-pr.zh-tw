# <a name="credentials-eaphostusercredentials-element"></a>認證 (EapHostUserCredentials) 元素

當方法設定為 XML 文字格式而非二進位 BLOB 時，會使用 **(EapHostUserCredentials)** 元素的認證。

``` syntax
<xs:element name="Credentials"
    type="BaseEapMethodUserCredentials"
 />
```

**認證** 元素是由 [**EapHostUserCredentials**](eaphostusercredentialsschema-eaphostusercredentials-element.md)元素定義。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|-------------------------------------|------------------------------------------------------|
| 最低支援的用戶端<br/> | Windows\[僅限 Vista desktop 應用程式\]<br/>       |
| 最低支援的伺服器<br/> | Windows\[僅限 Server 2008 desktop 應用程式\]<br/> |
