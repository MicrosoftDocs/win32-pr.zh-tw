# <a name="performservervalidation-peapextensionstype-element-v1-schema"></a>PerformServerValidation (PeapExtensionsType) 元素 (v1 架構) 

**PerformServerValidation (PeapExtensionsType)** 元素指出是否執行伺服器驗證。

``` syntax
<xs:element
    minOccurs="0"
    ref="extendedPeap:PerformServerValidation"
 />
```

元素是由 [**PeapExtensionsType**](mspeapconnectionpropertiesv1schema-peapextensionstype-complextype.md) 元素所定義。

## <a name="remarks"></a>備註

**PerformServerValidation** 元素是選擇性的。

## <a name="requirements"></a>規格需求



| 角色 | 最低支援作業系統版本 |
|------|------------------------------|
| 用戶端<br/> | \[僅限 Windows 7 桌面應用程式\]<br/>              |
| 伺服器<br/> | 僅限 Windows Server 2008 R2 \[ desktop 應用程式\]<br/> |
