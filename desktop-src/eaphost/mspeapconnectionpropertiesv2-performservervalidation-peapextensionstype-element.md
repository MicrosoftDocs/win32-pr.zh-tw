# <a name="performservervalidation-peapextensionstype-element-v2-schema"></a>PerformServerValidation (PeapExtensionsType) 元素 (V2 架構) 

**PerformServerValidation (PeapExtensionsType)** 元素指出是否執行伺服器驗證。

``` syntax
<xs:element name="PerformServerValidation"
    type="xs:boolean"
 />
```

**PerformServerValidation** 元素是由 [**PeapExtensionsType**](mspeapconnectionpropertiesv1schema-peapextensionstype-complextype.md)元素定義。

## <a name="remarks"></a>備註

**PerformServerValidation** 元素是選擇性的。

|------|--------------------|
