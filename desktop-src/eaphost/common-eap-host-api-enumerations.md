# <a name="common-eaphost-api-enumerations"></a>常見的 EAPHost API 列舉

下列列舉通用於所有 EAPHost API 技術。



| 列舉型別                                                                  | 描述                                                                                   |
|------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [**EAP \_ 屬性 \_ 類型**](/windows/desktop/api/eaptypes/ne-eaptypes-eap_attribute_type)                           | 定義可在驗證實體上使用的一組可能的 EAP 屬性類型。        |
| [**EAP \_ 設定 \_ 輸入 \_ 欄位 \_ 類型**](/windows/desktop/api/eaptypes/ne-eaptypes-eap_config_input_field_type)       | 定義當查詢使用者的認證時，可用的一組可能的輸入欄位類型。 |
| [**EAP \_ 方法 \_ 屬性 \_ 類型**](/windows/desktop/api/EapTypes/ne-eaptypes-eap_method_property_type)              | 定義一組可能的 EAP 方法屬性。                                            |
| [**EAP \_ 方法 \_ 屬性 \_ 值 \_ 類型**](/windows/desktop/api/EapTypes/ne-eaptypes-eap_method_property_value_type) | 為方法屬性定義一組可能的資料類型。                             |
| [**EapCode**](/windows/win32/api/eapmethodtypes/ne-eapmethodtypes-eapcode)                                                   | 定義一組 EAP 封包類型。                                                          |
