# <a name="peapextensions-eaptype-element-connection-property"></a>PeapExtensions (EapType) 元素 (連接屬性) 

**PeapExtensions (EapType)** 專案可讓架構的未來增強功能。

``` syntax
<xs:element name="PeapExtensions"
    type="PeapExtensionsType"
 />
```

**PeapExtensions** 元素是由 [**EapType**](mspeapconnectionpropertiesv1schema-eaptype-element.md)元素定義。

## <a name="remarks"></a>備註

**PeapExtensions** 元素是選擇性的。
