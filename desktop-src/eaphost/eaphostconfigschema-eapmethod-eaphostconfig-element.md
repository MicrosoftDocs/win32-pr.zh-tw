# <a name="eapmethod-eaphostconfig-element"></a>EapMethod (EapHostConfig) 元素

**EapMethod (EapHostConfig)** 元素會識別所參考的方法。

``` syntax
<xs:element name="EapMethod"
    type="EapMethodType"
 />
```

**EapMethod** 元素是由 [**EapHostConfig**](eaphostconfigschema-eaphostconfig-element.md)元素定義。

| 角色 | 最低 OS 版本 |
|------|--------------------|

| 伺服器<br/> | 僅限 Windows Server 2008 \[ desktop 應用程式\]<br/> |
