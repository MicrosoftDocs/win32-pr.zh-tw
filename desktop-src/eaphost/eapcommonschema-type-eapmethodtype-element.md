# <a name="type-eapmethodtype-element"></a>輸入 (EapMethodType) 元素

**類型 (EapMethodType)** 元素會參考 EAP 方法類型。

此類型是由網際網路指派的號碼授權單位所發出的唯一號碼 (IANA) 。

``` syntax
<xs:element name="Type"
    type="unsignedByte"
 />
```

**Type** 元素是由 [**EapMethodType**](eapcommonschema-eapmethodtype-complextype.md)複雜型別定義。
