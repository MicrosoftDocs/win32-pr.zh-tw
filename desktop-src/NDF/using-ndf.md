# <a name="using-ndf"></a>使用 NDF

網路診斷架構 (NDF) 隨附于 Windows Vista、Windows Server 2008 和更新版本中。 軟體發展人員可以使用 NDF 來診斷和修復可能導致應用程式失敗的網路問題。 開發人員可以使用 Microsoft 提供的 Api 將其應用程式連接至 NDF。 在某些情況下，開發人員也可以建立 NDF 協助程式類別延伸模組，以協助診斷其網路軟體的特定問題。
