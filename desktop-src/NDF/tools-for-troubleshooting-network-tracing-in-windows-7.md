# <a name="tools-for-troubleshooting-using-network-tracing-in-windows-7"></a>在 Windows 7 中使用網路追蹤進行疑難排解的工具

有一些工具可用來協助啟用和管理網路追蹤，以及查看產生的資料。 一些可能有用的工具組括 Netsh.exe、網路監視器、事件檢視器、 **netsh trace convert**、Tracerpt.exe 和 Xperf.exe。

在本節中，我們將探討如何使用 Netsh.exe 和網路監視器協助疑難排解網路追蹤。
