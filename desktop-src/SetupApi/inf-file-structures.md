# <a name="inf-file-structures"></a>INF 檔案結構

下列結構適用于 INF 檔案。



| 結構                                              | 描述                        |
|--------------------------------------------------------|------------------------------------|
| [**INFCONTEXT**](/windows/desktop/api/Setupapi/ns-setupapi-infcontext)                   | INF 檔案中的位置內容。 |
| [**SP \_ INF \_ 資訊**](/windows/desktop/api/Setupapi/ns-setupapi-sp_inf_information) | INF 檔案的相關資訊。    |



 

> [!Note]  
> 結構 INFCONTEXT 由安裝函式在內部使用，且不能由安裝程式的變更或參考。 這僅供資訊參考之用。
