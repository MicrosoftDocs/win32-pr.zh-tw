# <a name="mru-source-functions"></a>MRU 來源函數

下列函式會搭配 MRU 來源檔案清單使用。



| 函式                                                                 | 描述                                                                                       |
|--------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------|
| [**SetupAddToSourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupaddtosourcelista)                     | 將專案加入至來源清單。                                                                   |
| [**SetupCancelTemporarySourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupcanceltemporarysourcelist) | 取消任何暫時清單或不流覽的行為，並重新建立標準清單行為。        |
| [**SetupFreeSourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupfreesourcelista)                       | 釋出配置給來源清單的資源。                                                       |
| [**SetupQuerySourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupquerysourcelista)                     | 查詢目前的安裝來源清單。                                                 |
| [**SetupRemoveFromSourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupremovefromsourcelista)           | 從安裝來源清單中移除專案。                                                |
| [**SetupSetSourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupsetsourcelista)                         | 將安裝來源清單設定為系統 MRU 清單、使用者 MRU 清單或暫存清單。 |
