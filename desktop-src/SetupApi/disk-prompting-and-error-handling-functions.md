# <a name="disk-prompting-and-error-handling-functions"></a>磁片提示和錯誤處理函式

下列函數提供磁片提示和錯誤處理使用者介面。



| 函式                                         | 描述                                                                            |
|--------------------------------------------------|----------------------------------------------------------------------------------------|
| [**SetupCopyError**](/windows/desktop/api/Setupapi/nf-setupapi-setupcopyerrora)         | 產生對話方塊，通知使用者有複製錯誤。                          |
| [**SetupDeleteError**](/windows/desktop/api/Setupapi/nf-setupapi-setupdeleteerrora)     | 產生對話方塊，通知使用者刪除錯誤。                      |
| [**SetupPromptForDisk**](/windows/desktop/api/Setupapi/nf-setupapi-setuppromptfordiska) | 產生對話方塊，提示使用者輸入媒體磁片或原始程式檔位置。 |
| [**SetupRenameError**](/windows/desktop/api/Setupapi/nf-setupapi-setuprenameerrora)     | 產生對話方塊，通知使用者發生重新命名錯誤。                        |
