# <a name="using-inf-files"></a>使用 INF 檔案

-   [撰寫 INF 檔案](authoring-an-inf-file.md)
-   [開啟和關閉 INF 檔案](opening-and-closing-an-inf-file.md)
-   [從 INF 檔案中取出資訊](retrieving-information-from-an-inf-file.md)
-   [從 INF 檔案安裝](installing-from-an-inf-file.md)
-   [從 INF 檔案指定安全描述項](specifying-a-security-descriptor-from-an-inf-file.md)
