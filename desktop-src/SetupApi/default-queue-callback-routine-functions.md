# <a name="default-queue-callback-routine-functions"></a>預設佇列回呼常式函數

下列函式會搭配預設佇列回呼常式使用。



| 函式                                                                   | 描述                                                                                                                                                                            |
|----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SetupDefaultQueueCallback**](/windows/desktop/api/Setupapi/nf-setupapi-setupdefaultqueuecallbacka)             | 預設佇列回呼常式。                                                                                                                                                    |
| [**SetupInitDefaultQueueCallback**](/windows/desktop/api/Setupapi/nf-setupapi-setupinitdefaultqueuecallback)     | 初始化預設佇列回呼常式所需的內容資訊。                                                                                                          |
| [**SetupInitDefaultQueueCallbackEx**](/windows/desktop/api/Setupapi/nf-setupapi-setupinitdefaultqueuecallbackex) | 初始化預設佇列回呼常式所需的內容資訊，並指定替代視窗以顯示進度訊息。                                           |
| [**SetupTermDefaultQueueCallback**](/windows/desktop/api/Setupapi/nf-setupapi-setuptermdefaultqueuecallback)     | 釋放由 [**SetupInitDefaultQueueCallback**](/windows/desktop/api/Setupapi/nf-setupapi-setupinitdefaultqueuecallback) 和 [**SetupInitDefaultQueueCallbackEx**](/windows/desktop/api/Setupapi/nf-setupapi-setupinitdefaultqueuecallbackex)所配置的資源。 |
