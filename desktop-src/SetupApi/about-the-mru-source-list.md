# <a name="about-the-mru-source-list"></a>關於 MRU 來源清單

最近使用的 (MRU) 來源清單會保留在使用者的系統上。 安裝函式會在內部處理在使用者的電腦上建立新的來源清單及其位置的工作。 安裝程式函數會使用此清單來儲存先前安裝中所使用之來源路徑的相關資訊。

提示使用者輸入來源路徑時，您可以使用此資訊。 例如，您可以在先前的安裝中建立用來做為來源路徑之網路連線的下拉式清單。

根據您的許可權，您可以建立使用者清單、特定使用者專屬的使用者清單或系統清單（所有使用者都相同）。 除了系統和使用者來源清單之外，您還可以建立在安裝應用程式結束時捨棄的暫時來源清單。
