# <a name="about-the-file-log"></a>關於檔案記錄檔

檔案記錄檔會記錄在安裝期間複製到系統之檔案的相關資訊。 它可用於緊急修復或診斷工具。

用來記錄檔案安裝的檔案可以是系統記錄檔或您指定的任何其他檔案記錄檔。
