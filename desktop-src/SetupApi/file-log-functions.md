# <a name="file-log-functions"></a>檔案記錄檔函數

下列函式可用於檔案記錄檔。



| 函式                                                   | 描述                                 |
|------------------------------------------------------------|---------------------------------------------|
| [**SetupInitializeFileLog**](/windows/desktop/api/Setupapi/nf-setupapi-setupinitializefileloga)   | 初始化要使用的記錄檔。             |
| [**SetupLogFile**](/windows/desktop/api/Setupapi/nf-setupapi-setuplogfilea)                       | 將專案加入至記錄檔。              |
| [**SetupQueryFileLog**](/windows/desktop/api/Setupapi/nf-setupapi-setupqueryfileloga)             | 從記錄檔抓取資訊。      |
| [**SetupRemoveFileLogEntry**](/windows/desktop/api/Setupapi/nf-setupapi-setupremovefilelogentrya) | 從記錄檔中移除專案。           |
| [**SetupTerminateFileLog**](/windows/desktop/api/Setupapi/nf-setupapi-setupterminatefilelog)     | 釋放配置給記錄檔的資源。 |
