# <a name="file-queues"></a>檔案佇列

安裝函數包含檔案佇列功能。 檔案佇列是檔案複製、重新命名和刪除作業的清單。 您可以依任何順序將這些作業傳送至佇列。 認可佇列時，這些作業會以批次方式處理（依作業類型的順序）。

下列各節說明佇列是什麼，以及如何在建立安裝程式應用程式時使用它。 此外，也會討論在佇列認可時，排入佇列檔案作業的處理順序，以及佇列傳送至每個階段之回呼常式的通知。

-   [關於檔案佇列](about-file-queues.md)
-   [使用檔案佇列](using-file-queues.md)
-   [檔案佇列參考](file-queue-reference.md)
