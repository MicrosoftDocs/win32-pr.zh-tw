# <a name="gathering-setup-information-from-the-user"></a>正在收集使用者的安裝資訊

強烈建議您的安裝應用程式包含使用者介面，以取得使用者所需的任何資訊。 安裝函式不會提供這項功能，而且必須由您的安裝程式（通常是安裝精靈的形式）來執行。
