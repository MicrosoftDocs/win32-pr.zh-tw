# <a name="default-queue-callback-routine"></a>預設佇列回呼常式

包含在安裝函式中的預設回呼常式 [**SetupDefaultQueueCallback**](/windows/desktop/api/Setupapi/nf-setupapi-setupdefaultqueuecallbacka)，可讓您用來處理 [**SetupCommitFileQueue**](/windows/desktop/api/Setupapi/nf-setupapi-setupcommitfilequeuea)所傳回的通知。

下列各節將討論預設佇列回呼常式的格式、搭配使用預設佇列回呼常式與 [**SetupCommitFileQueue**](/windows/desktop/api/Setupapi/nf-setupapi-setupcommitfilequeuea)，以及如何建立以預設佇列回呼常式提供的功能為基礎的篩選回呼常式。

-   [使用預設佇列回呼常式](using-the-default-queue-callback-routine.md)
-   [預設佇列回呼常式參考](default-queue-callback-routine-reference.md)
