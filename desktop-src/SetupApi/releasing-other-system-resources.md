# <a name="releasing-other-system-resources"></a>釋放其他系統資源

如果使用來源清單或記錄檔，則應呼叫 [**SetupFreeSourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupfreesourcelista) 或 [**SetupTerminateFileLog**](/windows/desktop/api/Setupapi/nf-setupapi-setupterminatefilelog)來釋放資源。
