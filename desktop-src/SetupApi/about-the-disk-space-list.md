# <a name="about-the-disk-space-list"></a>關於 Disk-Space 清單

[磁碟空間] 清單可讓您計算完成安裝作業所需的磁碟空間。 建立磁碟空間清單並在其中新增檔案作業之後，您可以查詢清單，以判斷檔案操作所指定的目標磁片磁碟機，以及每個磁片磁碟機上所需的磁碟空間。

藉由比較目標磁片上可用空間的空間，您可以用程式設計的方式判斷是否有足夠的空間可用於目前的安裝。

您可以新增或移除磁碟空間清單中的檔案作業，並重新計算所需的磁碟空間。 例如，此功能可讓您撰寫與使用者互動的程式，讓他或她根據所需的磁碟空間，選擇要安裝的元件。

磁碟空間清單功能會自動檢查目標磁片上是否有預先存在的檔案複本。 如果找到舊版本的檔案，則磁碟空間清單函式會計算完成檔案作業所需的額外空間。

例如，如果複製作業指定將6000位元組檔案複製到 C 磁片磁碟機，而磁片磁碟機 C 上已經有2000位元組版本的檔案，則磁碟空間函數會傳回所需的4000位元組空間。 額外的空間是用來以較新的版本取代較舊版本的檔案。

磁碟空間清單會將檔案大小四捨五入到磁片叢集界限。
