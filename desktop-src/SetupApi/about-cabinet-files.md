# <a name="about-cabinet-files"></a>關於封包檔

封包是單一檔案，通常會加上尾碼。CAB，可將壓縮檔案儲存在檔案庫中。 壓縮檔案可以散佈在數個封包檔中。 在安裝期間，安裝應用程式會解壓縮儲存在封包中的檔案，並將它們複製到使用者的系統中。

如果使用封包檔，則安裝程式會使用封包的名稱作為其所在來源磁片的標記檔案。
