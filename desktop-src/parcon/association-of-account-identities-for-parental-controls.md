# <a name="association-of-account-identities-for-parental-controls"></a>家長監護帳戶身分識別的關聯

如同簡介中所述，Windows Vista 家長監護基礎結構所提供的設定、記錄和限制功能的有效使用，取決於要設定為標準使用者的 Windows 帳戶。 建議提供線上存取 (（例如 ISP 套件）的廠商) 為每個受控制的 Windows 帳戶建立和維護父系或守護者所允許的線上使用者名稱關聯。 如此一來，略過以雲端為基礎的家長監護，只是使用替代的線上身分識別。
