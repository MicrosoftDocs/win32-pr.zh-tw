# <a name="about-parental-controls"></a>關於家長監護

家長監護功能包含下列各項：

-   中央使用者介面功能與主控台整合。
-   選取的限制實施。
-   作業系統中的基礎結構，可提供整體原則的設定，以及公開目標擴充功能的已執行限制。
