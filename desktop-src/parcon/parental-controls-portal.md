# <a name="parental-controls"></a>家長監護

## <a name="purpose"></a>目的

在 Microsoft 家庭安全的工作中，Windows 提供家長監護功能來監視和限制所選電腦使用者暴露于線上的風險和不當內容。

Windows 中的「家長監護」技術旨在協助用心家長或監護人，以確保能以年齡或成熟度等級來存取其保護下的適當教材。 除了內建功能以外，它還提供了可擴充的基礎結構。

## <a name="developer-audience"></a>開發人員對象

獨立軟體廠商 (Isv) 的產品，這些產品可能會向使用者公開安全風險，或是目前致力於保護使用者免于遭受這類風險的使用者整合家長監護。

## <a name="run-time-requirements"></a>執行階段需求求

這項功能是在 Windows Vista 中引進的。 只有選取的取用者作業系統才能使用家長監護功能;它並非設計來與 Windows Server 版本搭配使用。 Windows 10 不支援這項功能。
