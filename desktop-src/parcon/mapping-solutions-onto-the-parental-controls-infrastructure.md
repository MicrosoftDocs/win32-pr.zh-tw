# <a name="mapping-solutions-onto-the-parental-controls-infrastructure"></a>將解決方案對應到家長監護基礎結構

一般 ISV 和 ISP/入口網站家長監護的五個類別，都很容易看出：

-   獨立用戶端應用程式。 例如，電子郵件用戶端和媒體播放機。 雖然媒體播放機具有網際網路對向的風險，有些則具有特定的服務，但 Windows 的家長監護主要是提高播放活動的記錄和限制，以將程式庫的影響降至最低，並為系統管理員記錄干擾。
-   用戶端/伺服器解決方案。 最顯著的範例是立即訊息解決方案。
-   ISP 套件。 這些是用戶端/伺服器解決方案和用戶端應用程式的集合，通常會在單一用戶端使用者介面中整合。 請注意，其中大部分也提供大部分或全部的功能，方法是使用瀏覽器存取，跨越至僅限網路的解決方案。
-   僅限 Web 的解決方案。 透過瀏覽器存取。 範例包括 web 型電子郵件和立即訊息功能。 您可以透過適當填入的網頁篩選器類別來封鎖對這些存取的存取。
-   類似防火牆的解決方案。 提供網路堆疊層級的篩選，以及可能的其他作業系統限制。
