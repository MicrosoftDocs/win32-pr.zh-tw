# <a name="bits-adsi-extension"></a>BITS ADSI 擴充功能

BITS 伺服器提供下列 ADSI 擴充介面，以啟用 BITS 上傳。



| 介面                                                        | 描述                                                                                                                                    |
|------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IBITSExtensionSetup**](/windows/desktop/api/Bitscfg/nn-bitscfg-ibitsextensionsetup)               | 用來啟用和停用 BITS 上傳至虛擬目錄。                                                                                 |
| [**IBITSExtensionSetupFactory**](/windows/desktop/api/Bitscfg/nn-bitscfg-ibitsextensionsetupfactory) | 用來在安裝 BITS 伺服器的安裝程式中，取得 [**IBITSExtensionSetup**](/windows/desktop/api/Bitscfg/nn-bitscfg-ibitsextensionsetup) 介面的指標。 |
