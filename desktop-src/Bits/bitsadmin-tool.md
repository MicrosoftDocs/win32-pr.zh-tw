# <a name="bitsadmin-tool"></a>BITSAdmin 工具

BITSAdmin 是一種命令列工具，可讓您用來建立下載或上傳作業，並監視其進度。 如需完整的工具及其所有命令的完整檔，請參閱 Windows IT 專業人員中心的 [bitsadmin](/windows-server/administration/windows-commands/bitsadmin) 和 [bitsadmin 範例](/windows-server/administration/windows-commands/bitsadmin-examples) 。

例如，若要列出所有 BITS 作業，您可以發出命令 `bitsadmin /list /verbose` 。 以下是該命令範例輸出的螢幕擷取畫面。
