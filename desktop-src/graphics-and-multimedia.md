# <a name="graphics-and-gaming"></a>圖形和遊戲

Windows 提供 api 和元件，可支援圖形、遊戲和影像處理。

## <a name="in-this-section"></a>本節內容

| 主題 | 描述 |
|-|-|
| [DirectX 圖形與遊戲](./directx.md)<br/> | DirectX 圖形提供一組 Api，可讓您用來建立遊戲和其他高效能的多媒體應用程式。 <br/> |
| [DirectComposition](./directcomp/directcomposition-portal.md)<br/> | DirectComposition 可透過轉換、效果和動畫來提供高效能點陣圖組合。 您可以使用 DirectComposition API 來建立以視覺方式吸引人的使用者介面，以從某個視覺效果到另一個視覺效果來呈現豐富且流暢的動畫轉換。<br/> |
| [遊戲模式](/previous-versions/windows/desktop/gamemode/game-mode-portal)<br/> | 通用 Windows 平台 (UWP) 的遊戲模式 API 可讓您利用 Windows 10 中的遊戲模式產生最佳化的遊戲體驗。<br/> |
| [遊戲裝置資訊](/previous-versions/windows/desktop/gamingdvcinfo/gaming-device-information-portal)<br/> | 遊戲裝置資訊 Api 可讓 UWP 遊戲開發人員判斷遊戲執行所在的主控台類型，以進行有關如何充分使用硬體的執行時間選項。<br/> |
| [Windows 影像處理元件 (WIC)](./wic/-wic-lh.md)<br/> | Windows 影像處理元件 (WIC) 是可延伸的平臺，可為數字影像提供低層級的 API。 WIC 支援標準 web 映射格式、高動態範圍影像和原始相機資料。<br/> |
| [Win2D (外部網站) ](https://github.com/Microsoft/Win2D)<br/> | Win2D 是簡單易用的 Windows 執行階段 API，適用于使用 GPU 加速轉譯的立即模式2d 圖形。 c \# 和 c + + 開發人員可以使用這項功能，為 Windows 8.1、Windows Phone 8.1 和 Windows 10 撰寫 Windows 應用程式。 它利用 Direct2D 的強大功能，並與 XAML 和 CoreWindow 無縫整合。<br/> |
| [Windows 存放區 (外部網站) 的角度](https://github.com/microsoft/angle/wiki)<br/> | Windows 存放區的角度是開放原始碼專案，可讓開發人員將 opengl es api 呼叫轉譯為 DirectX 11 api 呼叫，以在 Windows 上執行 opengl es 內容。 Windows 存放區的角度可支援 Windows 8.1、Windows Phone 8.1 和 Windows 10。<br/> |

> [!NOTE]
> 從 Windows 10 版本 1809 (10.0; 組建 17763) 開始，TruePlay 已從 Windows 中移除。 因此並未發佈 TruePlay 文件。
