# <a name="exposing-properties-through-idispatch"></a>透過 IDispatch 公開屬性

雖然大部分的控制項都有屬性，但不需要控制項來公開任何屬性，因此控制項不需要 **IDispatch**。 如果控制項有屬性，控制項就不會有控制項必須公開之屬性的需求。
