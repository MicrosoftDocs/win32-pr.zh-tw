# <a name="translating-to-java"></a>轉換成 JAVA

適用于 JAVA 的 Microsoft Visual j + + 開發系統提供兩種工具，可讓您使用 java 語法、JAVA 類型程式庫嚮導和 JActiveX 命令列工具來產生 COM 物件的型別程式庫資訊。 您可以使用上述任一種工具，產生 COM 物件的 JAVA 類別檔案。

在 Visual j + + 6.0 版中，JActiveX 工具會取代 JAVATLB 命令列工具。 如需詳細資訊，請參閱 Visual c + + 檔。

如果您使用的是 Visual j + + 以外的 JAVA 開發環境，請參閱該產品的檔，以取得 COM 轉換工具的相關資訊。

一旦為元件產生類別檔案之後，您就可以使用 **import** 語句將元件類別新增至您的 JAVA 應用程式。

然後，您可以在 JAVA 應用程式中建立及使用物件的類別。 如需 JAVA **import** 語句的詳細資訊，請參閱 java 程式設計語言的參考。

如需詳細資訊，請參閱下列主題：

-   [從 c + + 轉換成 JAVA](translating-to-java-from-c--.md)
-   [從 Visual Basic 轉換為 JAVA](translating-to-java-from-visual-basic.md)

-   [JAVA 類型程式庫嚮導](java-type-library-wizard.md)
