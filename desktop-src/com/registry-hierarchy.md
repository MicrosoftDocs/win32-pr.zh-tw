# <a name="registry-hierarchy"></a>登錄階層

登錄的結構是索引鍵、子機碼和命名值的階層，或是單一預設值。 子機碼可以有多個子機碼和值。 索引鍵會以反斜線分隔字串命名。 登錄中的每個索引鍵都可以有一或多個值，這些值可能包含字串、整數值或二進位資料。
