# <a name="extended-properties-events-and-methods"></a>擴充屬性、事件和方法

ActiveX 控制項容器不需要支援擴充控制項。 但是，如果控制項容器支援擴充屬性，則它必須支援下列最基本設定：

-   可見
-   父代
-   預設
-   取消

目前，擴充屬性、事件和方法沒有標準 dispid。
