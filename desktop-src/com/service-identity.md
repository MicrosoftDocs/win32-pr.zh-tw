# <a name="service-identity"></a>服務身分識別

選擇此身分識別會導致應用程式以服務的形式執行。 您可以從 **主控台** 選擇 [**服務**] 選項來設定服務帳戶。 如需詳細資訊，請參閱 [安裝為服務應用程式](installing-as-a-service-application.md)。
