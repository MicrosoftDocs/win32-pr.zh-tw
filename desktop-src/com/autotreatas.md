# <a name="autotreatas"></a>AutoTreatAs

自動將 [**TreatAs**](treatas.md) 索引鍵的 CLSID 設定為指定的值。

## <a name="registry-entry"></a>登錄項目

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes\CLSID
   {CLSID}
      AutoTreatAs = value
```

## <a name="remarks"></a>備註

這是指定類別識別碼的 **REG \_ SZ** 值。
