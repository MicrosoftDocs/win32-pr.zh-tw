# <a name="notifications-com"></a>COM)  (通知

通知是物件偵測到其名稱、狀態、資料或簡報變更時所產生的回呼。 容器和其他用戶端需要通知，才能適當地回應這些變更。 容器會將諮詢連接設定為感興趣的物件，以註冊接收通知。 其他感興趣的用戶端也可以這樣做。 容器也會建立接收通知的諮詢接收。 使用容器所建立的連線，有變更的物件會通知諮詢接收。 收到通知時，容器會採用已針對已發生變更類型定義的任何動作。

如需詳細資訊，請參閱下列主題：

-   [通知類型](types-of-notifications.md)
-   [通知的運作方式](how-notifications-work.md)
