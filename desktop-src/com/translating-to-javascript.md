# <a name="translating-to-javascript"></a>翻譯為 JavaScript

JavaScript 是 Netscape 所建立的指令碼語言。 它最初稱為 LiveScript。 儘管名稱和語法有相似之處，但 JavaScript 與 JAVA 程式設計語言無關。 JAVA 是 Sun Microsystems，Inc. 所開發的全功能程式設計語言。

JavaScript 是由各式各樣的網頁瀏覽器所支援，包括 Internet Explorer 和 Netscape Navigator。

除了 JavaScript 所提供的語言功能之外，腳本主機也可能會公開您的腳本可以存取的物件模型。 例如，Internet Explorer 會公開物件模型，讓腳本能夠與互動並以程式設計方式控制瀏覽器。 如需有關使用這類物件模型的詳細資訊，請參閱腳本主機上的檔。

如需 JavaScript 的詳細資訊，請參閱 JavaScript 參考。

下列主題描述當您將腳本從 VBScript 或 JScript 轉譯成 JavaScript 時，應考慮的問題：

-   [從 VBScript 轉換成 JavaScript](translating-to-javascript-from-vbscript.md)
-   [從 JScript 轉換成 JavaScript](translating-to-javascript-from-jscript.md)
