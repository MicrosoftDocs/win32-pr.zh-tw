# <a name="appid-clsid-key"></a>AppID (CLSID 金鑰) 

將 AppID 與 CLSID 相關聯。

## <a name="registry-entry"></a>登錄項目

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes\CLSID
   {CLSID}
      AppID = {AppID_GUID}
```

## <a name="remarks"></a>備註

這是 **REG \_ SZ** 值。
