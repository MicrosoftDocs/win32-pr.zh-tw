# <a name="turning-off-security"></a>關閉安全性

有幾個原因會讓您決定關閉 COM 提供的安全性。 開發應用程式時，先關閉安全性可能會很有説明，讓您能夠專注于應用程式內的其他功能。 然後，您可以在準備就緒時，將安全性功能加入至您的設計。 為了加速偵測，您可能會想要關閉安全性，然後一次新增一個安全性功能。 最後，有些應用程式不需要安全性，而且如果沒有執行任何安全性檢查，可能會更快執行。

下列主題提供逐步程式：

-   [關閉啟用安全性](turning-off-activation-security.md)
-   [關閉通話安全性](turning-off-call-security.md)
