# <a name="insertable-progid-key"></a>插入 (ProgID 金鑰) 

指出這個類別可在 OLE 2 容器中插入。

## <a name="registry-entry"></a>登錄項目

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes
   {ProgID}
      Insertable
```

## <a name="remarks"></a>備註

這是在 OLE 2 容器中可插入的物件所需的專案。
