# <a name="events-in-controls"></a>控制項中的事件

雖然大部分的控制項都有公開和引發數個事件，但不需要控制項來公開或引發任何事件，因此控制項不需要 [**IConnectionPointContainer**](/windows/desktop/api/OCIdl/nn-ocidl-iconnectionpointcontainer)。 如果控制項有任何事件，控制項就不會有控制項必須公開的事件需求。
