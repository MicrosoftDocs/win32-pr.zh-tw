# <a name="linking-and-embedding"></a>連結和內嵌

使用者可以建立兩種類型的複合檔案物件： *連結* 或 *內嵌*。 這兩種類型之間的差異在於儲存物件之來源資料的方式和位置。 物件所在的位置會影響物件的可攜性和啟動方法、如何執行資料更新，以及其容器的大小與結構。

-   [連結化物件](linked-objects.md)
-   [内嵌物件](embedded-objects.md)
