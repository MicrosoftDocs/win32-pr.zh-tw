# <a name="ws_group-and-ws_tabstop-flags-in-controls"></a>\_控制項中的 ws 群組和 ws \_ TABSTOP 旗標

控制項不應該在內部使用 WS \_ 群組和 ws \_ TABSTOP 旗標; 有些容器會依賴這些旗標來管理鍵盤處理。
