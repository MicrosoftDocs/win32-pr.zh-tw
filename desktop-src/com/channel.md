# <a name="channel-com"></a>通道 (COM) 

通道會負責在整個進程界限之間傳輸用戶端與物件之間的所有訊息。 通道的設計目的是要以透明方式使用不同的通道類型、與憑證的 DCE 標準 RPC 相容，並支援單一和多執行緒應用程式。
