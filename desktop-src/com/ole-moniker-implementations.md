# <a name="ole-moniker-implementations"></a>OLE 標記執行

OLE 針對不同的情況提供數個標記的執行，如下列主題所述：

-   [檔案名字](file-monikers.md)
-   [複合的名字](composite-monikers.md)
-   [專案的名字](item-monikers.md)
-   [反標記](anti-monikers.md)
-   [指標標記](pointer-monikers.md)
-   [類別名字標記](class-monikers.md)
-   [非同步名字](asynchronous-monikers.md)
-   [URL 的名字](url-monikers.md)

檔案、複合和專案的名字標記是最常使用的標記，因為它們可以用來在任何位置建立幾乎任何的物件。 反標記和指標的標記主要是在 OLE 內使用，但在執行自訂的標記時有一些應用程式。
