# <a name="linked-objects"></a>連結化物件

在複合檔案中插入物件的連結時，來源資料或 *連結來源* 會繼續存在於最初建立的位置，通常是在另一份檔中。 複合檔案只包含連結來源上儲存的實際資料的參考或 *連結*，以及如何將該資料呈現給使用者的相關資訊。

啟用連結會執行連結來源的伺服器應用程式，使用者需要此應用程式才能編輯或操作連結資料。 連結會將複合檔案的大小保持小。 當資料來源是由其他人維護，而且必須在許多使用者之間共用時，這也很有用。 如果維護連結來源的人員變更資料，則會在所有包含該資料連結的檔中自動更新變更。 除了建立簡單的連結之外，使用者還可以內嵌連結併合並連結和内嵌物件，以建立複雜的檔。
