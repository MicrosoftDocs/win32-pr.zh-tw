# <a name="linked-objects"></a><span data-ttu-id="cd877-103">連結化物件</span><span class="sxs-lookup"><span data-stu-id="cd877-103">Linked Objects</span></span>

<span data-ttu-id="cd877-104">在複合檔案中插入物件的連結時，來源資料或 *連結來源* 會繼續存在於最初建立的位置，通常是在另一份檔中。</span><span class="sxs-lookup"><span data-stu-id="cd877-104">When a link to an object is inserted in a compound document, the source data, or *link source*, continues to reside wherever it was initially created, usually in another document.</span></span> <span data-ttu-id="cd877-105">複合檔案只包含連結來源上儲存的實際資料的參考或 *連結*，以及如何將該資料呈現給使用者的相關資訊。</span><span class="sxs-lookup"><span data-stu-id="cd877-105">The compound document contains only a reference, or *link*, to the actual data stored at the link source, along with information about how to present that data to the user.</span></span>

<span data-ttu-id="cd877-106">啟用連結會執行連結來源的伺服器應用程式，使用者需要此應用程式才能編輯或操作連結資料。</span><span class="sxs-lookup"><span data-stu-id="cd877-106">Activating a link runs the link source's server application, which the user requires in order to edit or otherwise manipulate the link data.</span></span> <span data-ttu-id="cd877-107">連結會將複合檔案的大小保持小。</span><span class="sxs-lookup"><span data-stu-id="cd877-107">Linking keeps the size of a compound document small.</span></span> <span data-ttu-id="cd877-108">當資料來源是由其他人維護，而且必須在許多使用者之間共用時，這也很有用。</span><span class="sxs-lookup"><span data-stu-id="cd877-108">It is also useful when the data source is maintained by someone else and must be shared among many users.</span></span> <span data-ttu-id="cd877-109">如果維護連結來源的人員變更資料，則會在所有包含該資料連結的檔中自動更新變更。</span><span class="sxs-lookup"><span data-stu-id="cd877-109">If the person maintaining the link source changes the data, the change is automatically updated in all documents containing a link to that data.</span></span> <span data-ttu-id="cd877-110">除了建立簡單的連結之外，使用者還可以內嵌連結併合並連結和内嵌物件，以建立複雜的檔。</span><span class="sxs-lookup"><span data-stu-id="cd877-110">In addition to creating simple links, users can nest links and combine linked and embedded objects to create complex documents.</span></span>
