# <a name="changes-to-com-in-windows-vista"></a>Windows Vista 中的 COM 變更

Windows Vista 中的 COM 已進行下列變更：

-   COM 提高許可權的標記可讓在受限使用者帳戶下執行的應用程式 (LUA) ，以較高的功能啟動 COM 類別。 如需詳細資訊，請參閱 [COM 提高許可權的標記](the-com-elevation-moniker.md)。
-   當您安裝的應用程式會在 Windows Vista 電腦上進行 COM 呼叫，而您想要以系統管理員身分執行應用程式時，您必須安裝每台電腦的應用程式，而不是每位使用者的應用程式，否則將無法正常運作。
