# <a name="enhanced-metafiles"></a>增強的中繼檔

不令人驚訝的是，增強的中繼檔提供比標準中繼檔更多功能;使用增強的中繼檔通常可簡化呈現程式碼。 增強的中繼檔 DC 與標準中繼檔 DC 的使用方式完全相同。 OLE 支援增強的中繼檔，並包含與標準中繼檔的回溯相容性。

ActiveX 控制容器應使用增強的中繼檔，而不是標準的中繼檔。
