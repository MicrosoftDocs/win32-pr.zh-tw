# <a name="containers"></a>容器

ActiveX 控制容器是支援下列其他功能的 OLE 容器：

-   來自同進程伺服器的内嵌物件
-   就地啟用
-   OLEMISC \_ ACTI加值稅EWHENVISIBLE
-   事件處理

ActiveX控制容器必須提供所有這些功能的支援。
