# <a name="registration-and-licensing"></a>註冊和授權

控制項通常會以同進程伺服器 ( 形式提供。DLL) ，不過它也可以是本機或遠端伺服器 (。EXE) 。

控制項通常支援自我註冊，並且在具現化時，會建立一組登錄專案。 您也可以授權控制項來協助防止未經授權的使用。 如需自我註冊和授權的詳細資訊，請參閱 [元件物件模型](the-component-object-model.md) 。

如需詳細資訊，請參閱 [ActiveX 控制項登錄資訊](activex-controls-registry-information.md)。
