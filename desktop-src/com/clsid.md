# <a name="clsid"></a>CLSID

將 ProgID 與 CLSID 產生關聯。

## <a name="registry-entry"></a>登錄項目

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes
   {ProgID}
      CLSID = CLSID
```

## <a name="remarks"></a>備註

這是指定物件 CLSID 的 **REG \_ SZ** 值。
