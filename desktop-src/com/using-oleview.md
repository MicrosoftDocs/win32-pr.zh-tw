# <a name="using-oleview"></a>使用 OleView

雖然登錄編輯程式會顯示未處理的登錄視圖，而且對於學習登錄的組織很有説明，但使用登錄編輯程式可能相當耗時。

OleView 會提供登錄中所含資訊的更高層級視圖。 與其查看較長的 Clsid 清單，OleView 具有易記名稱的功能樹狀結構控制項。
