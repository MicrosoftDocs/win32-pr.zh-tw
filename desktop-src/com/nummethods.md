# <a name="nummethods"></a>NumMethods

包含相關聯介面中的方法數目，包括衍生介面的方法。

## <a name="registry-entry"></a>登錄項目

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes\Interface
   {IID}
      NumMethods = value
```

## <a name="remarks"></a>備註

這是 **REG \_ SZ** 值。
