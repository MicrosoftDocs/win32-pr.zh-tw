# <a name="translating-to-jscript"></a>翻譯為 JScript

JScript 是 Microsoft 在 ECMA 262 語言規格中的實作為。 Internet Explorer 和 Netscape Navigator 都受到支援。 JScript 是符合 ECMA 262 標準的第一種指令碼語言。

JScript 是由各式各樣的網頁瀏覽器所支援，包括 Internet Explorer 和導覽器。

除了 JScript 所提供的語言功能之外，腳本主機也可能會公開您的腳本可以存取的物件模型。 例如，Internet Explorer 會公開物件模型，讓腳本能夠與互動並以程式設計方式控制瀏覽器。 如需使用這類物件模型的詳細資訊，請參閱腳本主機的檔。

如需 JScript 的詳細資訊，請參閱 JScript 語言參考。 如需詳細資訊，請參閱 [ECMA 262 語言規格](https://www.ecma-international.org/publications/standards/Ecma-262.htm)。

下列主題描述將腳本從 JavaScript 或 VBScript 轉譯為 JScript 時應考慮的問題：

-   [從 VBScript 轉換成 JScript](translating-to-jscript-from-vbscript.md)
-   [從 JavaScript 翻譯為 JScript](translating-to-jscript-from-javascript.md)
