# <a name="appid-appid-key"></a>Appid (AppID 金鑰) 

識別對應至已命名可執行檔的 AppID GUID。

## <a name="registry-entry"></a>登錄項目

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes\AppID
   <Executable_name>
      AppID = {AppID_GUID}
```

## <a name="remarks"></a>備註

這是 **REG \_ SZ** 值。
