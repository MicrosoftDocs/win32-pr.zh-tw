# <a name="security-values"></a>安全性值

有幾個值會影響 COM 應用程式的安全性，包括驗證層級、模擬等級、掩蓋、應用程式識別、軟體限制原則設定和參考追蹤。

如需詳細資訊，請參閱下列主題：

-   [驗證層級](authentication-level.md) 描述如何協助控制 COM 物件之間通訊的保護量。
-   [委派和](delegation-and-impersonation.md) 模擬會說明在模擬期間，掩蓋會如何影響向伺服器出示的身分識別，以及用戶端如何授與伺服器授權給伺服器代表。
-   [應用程式識別](application-identity.md) 會識別並說明您的應用程式可以使用的身分識別。
-   [軟體限制原則](software-restriction-policy.md) 說明如何協助保護您的系統免于與執行不明程式碼相關的危險。
-   [參考追蹤](reference-tracking.md) 會說明如何確保物件不會提早釋出。
