# <a name="visual-basic-object-browser"></a>Visual Basic 物件瀏覽器

將 COM 物件加入至您的專案做為參考或元件之後，您可以使用 Visual Basic 的物件瀏覽器來查看 Visual Basic 中該物件的型別程式庫資訊。

若要在物件瀏覽器中查看 COM 物件：

-   按一下 [ **View** ] 功能表上的 [ **物件瀏覽器**]。

[物件瀏覽器] 視窗的左側會顯示所有加入至目前專案的參考和元件類別。 右側包含這些類別的方法、屬性和事件。

您可以從 [程式庫選取] 下拉式清單中選取物件程式庫，以篩選要顯示的 COM 物件類別。

如需詳細資訊，請參閱 Visual Basic 檔中的 [流覽 ActiveX 元件型別程式庫](/previous-versions/visualstudio/visual-basic-6/aa240782(v=vs.60)) 。

在某些情況下，Visual Basic 的物件瀏覽器不會顯示函數或屬性的傳回類型。 當傳回型別為預設型別 **Variant** 時，不會出現傳回型別。 如果物件瀏覽器的 [描述] 窗格將元素描述為函式或屬性，而且未顯示傳回型別，則傳回型別語句會是「As Variant」。
