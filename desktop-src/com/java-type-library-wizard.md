# <a name="java-type-library-wizard"></a>JAVA 類型程式庫嚮導

JAVA 類型程式庫 Wizard 是 Visual c + + 5.0 的元件。 此 wizard 會產生 COM 物件的類別檔案和 JAVA 類型程式庫資訊。

**產生 JAVA 類別檔案和型別程式庫資訊**

1.  按一下 [ **工具** ] 功能表上的 [ **JAVA 類型程式庫]**。 已安裝的 COM 元件清單隨即出現。
2.  按一下以選取您想要轉換之專案旁的核取方塊。
3.  按一下 [確定]  。 類型程式庫 wizard 會針對選取的元件產生類別檔案。 它會列印將每個元件插入 JAVA 應用程式或 applet 中所需的匯 **入** 語句，以及包含物件之 java 類型程式庫資訊的摘要文字檔。
