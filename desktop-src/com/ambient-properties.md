# <a name="ambient-properties"></a>環境屬性

ActiveX 控制容器至少必須支援使用標準 dispid 的下列環境屬性。



| 環境屬性             | Dispid          | 批註/條件                                                      |
|------------------------------|-----------------|--------------------------------------------------------------------------|
| LocaleID<br/>          | -705<br/> |   <br/>                                                          |
| UserMode<br/>          | -709<br/> | 適用于具有不同使用者和執行環境的容器。<br/> |
| DisplayAsDefault <br/> | -713<br/> | 針對預設按鈕相關的容器。<br/>      |
