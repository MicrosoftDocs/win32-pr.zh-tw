# <a name="translating-to-c"></a>轉換成 c + +

Visual C++ 提供數個工具，可協助您將 COM 物件新增至您的專案。 您可以在專案中加入 ActiveX 控制項、使用 MFC 類別 Wizard，或在 IDL 檔案上執行 MIDL 編譯器。

如果您使用 Visual C++ 以外的 c + + 開發環境，請參閱該產品的檔，以取得 COM 轉換工具的相關資訊。

當您將 COM 物件的 include 或 stub 檔案新增至專案之後，您的應用程式就可以使用 COM 物件的類別。

Visual C++ 也提供 OLE COM 物件檢視器，可讓您用來查看編譯的類型程式庫資訊。 物件檢視器會以 c + + 語法呈現 COM 類別。

如需詳細資訊，請參閱下列主題：

-   [從 Visual Basic 轉換成 c + +](translating-to-c---from-visual-basic.md)
-   [從 JAVA 轉換成 c + +](translating-to-c---from-java.md)
-   [將 ActiveX 控制項新增至 Visual C++ Project](adding-an-activex-control-to-a-visual-c---project.md)
-   [MIDL 編譯器](midl-compiler.md)
