# <a name="simple-data-binding"></a>簡單資料繫結

ActiveX controls 架構會定義資料系結機制，讓 ActiveX 控制項可以指定其中一或多個屬性是可系結的。 在大部分的情況下，資料繫結控制項不一定需要資料系結，因此可以插入不支援資料系結的容器中。 顯然，在這種情況下，可能會降低控制項的功能。
