# <a name="debugger"></a>偵錯工具

儲存慣用偵錯工具的名稱，以搭配 RPC 上的遠端 COM 偵錯工具使用。

## <a name="registry-entry"></a>登錄項目

```
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\DebugObjectRPCEnabled\AeDebug
   Debugger = value
```

## <a name="remarks"></a>備註

這是 **REG \_ SZ** 值。
