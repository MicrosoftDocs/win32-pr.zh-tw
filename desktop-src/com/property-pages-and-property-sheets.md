# <a name="property-pages-and-property-sheets"></a>屬性頁和屬性工作表

OLE 屬性頁可讓物件在稱為屬性工作表的索引標籤式對話方塊中顯示其屬性。 然後，終端使用者就可以查看和變更物件的屬性。 物件可以獨立于其用戶端來顯示其屬性頁，或用戶端可以從單一屬性工作表中的數個包含物件來管理屬性頁的顯示。 屬性頁也提供通知用戶端物件屬性變更的方法。

任何想要為其變更屬性提供使用者介面的物件，都可以使用這項技術。
