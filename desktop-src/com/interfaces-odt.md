# <a name="interfaces-ole-and-data-transfer"></a> (OLE 和資料傳輸) 的介面

下列介面可用來執行複合檔案，以及在應用程式之間執行資料傳輸。



| 介面                                                          | 描述                                                                                                                                                                                                                                                                                                                                               |
|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IAdviseSink**](/windows/desktop/api/ObjIdl/nn-objidl-iadvisesink)                                 | 可讓容器和其他物件接收資料變更的通知、查看變更，以及在感興趣的物件中發生的複合檔案變更。                                                                                                                                                                                              |
| [**IAdviseSink2**](/windows/desktop/api/ObjIdl/nn-objidl-iadvisesink2)                               | [**IAdviseSink**](/windows/desktop/api/ObjIdl/nn-objidl-iadvisesink)的延伸模組，可新增方法來處理連結化物件之標記的變更。                                                                                                                                                                                                                               |
| [**IAdviseSinkEx**](/windows/desktop/api/OCIdl/nn-ocidl-iadvisesinkex)                             | [**IAdviseSink**](/windows/desktop/api/ObjIdl/nn-objidl-iadvisesink)的延伸模組，可在物件的檢視狀態中新增通知接收變更的功能。                                                                                                                                                                                                                        |
| [**IDataAdviseHolder**](/windows/desktop/api/ObjIdl/nn-objidl-idataadviseholder)                     | 建立及管理資料物件與一或多個建議接收之間的諮詢連接。                                                                                                                                                                                                                                                              |
| [**IDataObject**](/windows/desktop/api/ObjIdl/nn-objidl-idataobject)                                 | 啟用資料傳輸以及資料變更的通知。                                                                                                                                                                                                                                                                                                |
| [**IDropSource**](/windows/desktop/api/OleIdl/nn-oleidl-idropsource)                                 | 您必須執行的其中一個介面，以提供應用程式中的拖放作業。                                                                                                                                                                                                                                                         |
| [**IDropSourceNotify**](/windows/desktop/api/OleIdl/nn-oleidl-idropsourcenotify)                     | 當使用者將滑鼠拖曳到潛在的放置目標視窗或從中移出時，就會收到通知。                                                                                                                                                                                                                                                         |
| [**IDropTarget**](/windows/desktop/api/OleIdl/nn-oleidl-idroptarget)                                 | 您必須執行的其中一個介面，以提供應用程式中的拖放作業。                                                                                                                                                                                                                                                         |
| [**IEnterpriseDropTarget**](/windows/desktop/api/OleIdl/nn-oleidl-ienterprisedroptarget)             | 由卸載目標應用程式執行時，此介面可讓 OLE 拖放引擎判斷放置目標應用程式是否打算評估企業保護原則，並提供 OLE 拖放引擎提供放置來源應用程式企業識別碼給放置目標應用程式的方式。 |
| [**IEnumFORMATETC**](/windows/desktop/api/ObjIdl/nn-objidl-ienumformatetc)                           | 列舉 [**FORMATETC**](/windows/win32/api/objidl/ns-objidl-formatetc) 結構，這些結構定義指定之資料物件所支援的格式和媒體。                                                                                                                                                                                                                              |
| [**IEnumMoniker**](/windows/desktop/api/ObjIdl/nn-objidl-ienummoniker)                               | 列舉標記資料表中的名字。                                                                                                                                                                                                                                                                                                           |
| [**IEnumOleDocumentViews**](/windows/desktop/api/DocObj/nn-docobj-ienumoledocumentviews)             | 列舉檔物件所支援的視圖。                                                                                                                                                                                                                                                                                                      |
| [**IEnumOleUndoUnits**](/windows/desktop/api/OCIdl/nn-ocidl-ienumoleundounits)                     | 列舉復原或重做堆疊上的復原單位。                                                                                                                                                                                                                                                                                                      |
| [**IEnumOLEVERB**](/windows/desktop/api/OleIdl/nn-oleidl-ienumoleverb)                               | 列舉物件可用的不同動詞，以遞增動詞數位的順序排列。                                                                                                                                                                                                                                                                 |
| [**IEnumSTATDATA**](/windows/desktop/api/ObjIdl/nn-objidl-ienumstatdata)                             | 列舉資料物件的諮詢連接資訊。                                                                                                                                                                                                                                                                                         |
| [**IObjectWithSite**](/windows/desktop/api/OCIdl/nn-ocidl-iobjectwithsite)                         | 提供簡單的方式來支援容器中物件與其網站之間的通訊。                                                                                                                                                                                                                                                           |
| [**IOleAdviseHolder**](/windows/desktop/api/OleIdl/nn-oleidl-ioleadviseholder)                       | 管理物件服務器中的諮詢連接和複合檔案通知。                                                                                                                                                                                                                                                                     |
| [**IOleCache**](/windows/desktop/api/OleIdl/nn-oleidl-iolecache)                                     | 提供在物件內快取之呈現資料的控制權。                                                                                                                                                                                                                                                                             |
| [**IOleCache2**](/windows/desktop/api/OleIdl/nn-oleidl-iolecache2)                                   | 可讓物件用戶端選擇性地更新使用 [**IOleCache：： cache**](/windows/desktop/api/OleIdl/nf-oleidl-iolecache-cache)建立的每個快取。                                                                                                                                                                                                                                |
| [**IOleCacheControl**](/windows/desktop/api/OleIdl/nn-oleidl-iolecachecontrol)                       | 提供適當的快取維護。                                                                                                                                                                                                                                                                                                                    |
| [**IOleClientSite**](/windows/desktop/api/OleIdl/nn-oleidl-ioleclientsite)                           | 提供内嵌物件，其中包含其顯示網站的位置和範圍、其標記、其使用者介面，以及其容器所提供的其他資源的相關資訊。                                                                                                                                                                           |
| [**IOleCommandTarget**](/windows/desktop/api/DocObj/nn-docobj-iolecommandtarget)                     | 可讓物件和其容器將命令分派給彼此。                                                                                                                                                                                                                                                                                  |
| [**IOleContainer**](/windows/desktop/api/OleIdl/nn-oleidl-iolecontainer)                             | 列舉複合檔案中的物件，或鎖定處於執行中狀態的容器。                                                                                                                                                                                                                                                                       |
| [**IOleDocument**](/windows/desktop/api/DocObj/nn-docobj-ioledocument)                               | 可讓檔物件與容器進行通訊，以建立其資料的資料檢視。                                                                                                                                                                                                                                                           |
| [**IOleDocumentSite**](/windows/desktop/api/DocObj/nn-docobj-ioledocumentsite)                       | 啟用已經實作為檔物件的檔，以略過就地使用中物件的一般啟用順序，以及直接指示其用戶端網站將其啟用為檔物件。                                                                                                                                      |
| [**IOleDocumentView**](/windows/desktop/api/DocObj/nn-docobj-ioledocumentview)                       | 讓容器與檔物件所支援的每個視圖進行通訊。                                                                                                                                                                                                                                                                         |
| [**IOleInPlaceActiveObject**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplaceactiveobject)         | 提供在就地物件與相關聯應用程式的外部畫面格視窗之間的直接通訊通道，以及包含内嵌物件之應用程式內的文件視窗。                                                                                                                                          |
| [**IOleInPlaceFrame**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplaceframe)                       | 控制容器的最上層框架視窗。                                                                                                                                                                                                                                                                                                          |
| [**IOleInPlaceObject**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplaceobject)                     | 管理就地物件的啟用和停用，並判斷應該可看見就地物件的多少部分。                                                                                                                                                                                                                            |
| [**IOleInPlaceObjectWindowless**](/windows/desktop/api/OCIdl/nn-ocidl-ioleinplaceobjectwindowless) | 啟用無視窗物件來處理視窗訊息，以及參與拖放作業。                                                                                                                                                                                                                                                       |
| [**IOleInPlaceSite**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplacesite)                         | 管理容器與物件之就地用戶端網站之間的互動。                                                                                                                                                                                                                                                                          |
| [**IOleInPlaceSiteEx**](/windows/desktop/api/OCIdl/nn-ocidl-ioleinplacesiteex)                     | 提供一組額外的啟用和停用通知方法，可讓物件在物件啟用和停用時，避免螢幕上不必要的閃爍。                                                                                                                                                            |
| [**IOleInPlaceSiteWindowless**](/windows/desktop/api/OCIdl/nn-ocidl-ioleinplacesitewindowless)     | 啟用無視窗物件來處理視窗訊息、參與拖放作業，以及執行繪圖作業。                                                                                                                                                                                                                       |
| [**IOleInPlaceUIWindow**](/windows/desktop/api/OleIdl/nn-oleidl-ioleinplaceuiwindow)                 | 在檔或框架視窗上協調框線空間。                                                                                                                                                                                                                                                                                                  |
| [**IOleLink**](/windows/desktop/api/OleIdl/nn-oleidl-iolelink)                                       | 啟用連結的物件，以提供其容器與連結相關的函式。                                                                                                                                                                                                                                                                    |
| [**IOleObject**](/windows/desktop/api/OleIdl/nn-oleidl-ioleobject)                                   | 可讓内嵌物件提供其容器的基本功能，並與其進行通訊。                                                                                                                                                                                                                                                       |
| [**IOleParentUndoUnit**](/windows/desktop/api/OCIdl/nn-ocidl-ioleparentundounit)                   | 讓復原單位包含子復原單位。                                                                                                                                                                                                                                                                                                           |
| [**IOleUILinkContainer**](/windows/desktop/api/OleDlg/nn-oledlg-ioleuilinkcontainera)                 | OLE 通用對話方塊使用此介面來管理容器連結的屬性。                                                                                                                                                                                                                                                               |
| [**IOleUILinkInfo**](/windows/desktop/api/OleDlg/nn-oledlg-ioleuilinkinfoa)                           | [**IOleUILinkContainer**](/windows/desktop/api/OleDlg/nn-oledlg-ioleuilinkcontainera)介面的延伸模組，可讓容器支援 **物件屬性** 對話方塊的 **連結** 頁面。                                                                                                                                                                            |
| [**IOleUIObjInfo**](/windows/desktop/api/OleDlg/nn-oledlg-ioleuiobjinfoa)                             | 提供 [**物件屬性**] 對話方塊的 [**一般**] 和 [**視圖**] 頁面所使用的資訊，顯示物件的大小、位置、類型和名稱的相關資訊。 它也可讓您透過 [ **轉換** ] 對話方塊來轉換物件。                                                                                          |
| [**IOleUndoManager**](/windows/desktop/api/OCIdl/nn-ocidl-ioleundomanager)                         | 可讓容器針對包含的控制項內發生的動作，執行多層的復原和重做作業。                                                                                                                                                                                                                                    |
| [**IOleUndoUnit**](/windows/desktop/api/OCIdl/nn-ocidl-ioleundounit)                               | 復原管理員會呼叫這個介面中的方法來執行復原動作，以及取得可在使用者介面中顯示的字串，以描述復原動作。                                                                                                                                                                                  |
| [**IOleWindow**](/windows/desktop/api/OleIdl/nn-oleidl-iolewindow)                                   | 讓應用程式能夠取得參與就地啟用之各種視窗的控制碼，也可進入和結束上下文相關的說明模式。                                                                                                                                                                                       |
| [**IViewObject**](/windows/desktop/api/OleIdl/nn-oleidl-iviewobject)                                 | 讓物件直接顯示，而不將資料物件傳遞給呼叫端。 此外，這個介面也可以建立和管理與「建議接收」的連接，讓呼叫端可以在 view 物件中收到變更的通知。                                                                                                                 |
| [**IViewObject2**](/windows/desktop/api/OleIdl/nn-oleidl-iviewobject2)                               | [**IViewObject**](/windows/desktop/api/OleIdl/nn-oleidl-iviewobject)介面的延伸模組，可針對物件的指定視圖傳回繪圖的大小。                                                                                                                                                                                                                      |
| [**IViewObjectEx**](/windows/desktop/api/OCIdl/nn-ocidl-iviewobjectex)                             | [**IViewObject2**](/windows/desktop/api/OleIdl/nn-oleidl-iviewobject2)介面的延伸模組，可提供無閃爍繪圖、點擊測試及控制項調整大小的支援。                                                                                                                                                                                                       |
