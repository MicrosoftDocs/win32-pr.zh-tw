# <a name="advantages"></a>優點

將您的應用程式實作為同進程伺服器的優點是加速和結合物件處理常式和本機伺服器的一些優點。 同進程伺服器的速度比本機伺服器快許多，原因很多。 首先，因為它們較小，而且是在容器應用程式的進程空間中執行，所以可以更快速地載入。 其次，它們經過優化，可執行某些工作。 第三，容器和伺服器之間的通訊不依賴遠端程序呼叫。
