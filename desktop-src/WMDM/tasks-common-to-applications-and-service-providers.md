# <a name="tasks-common-to-applications-and-service-providers"></a>應用程式和服務提供者的一般工作

本節說明下列工作，這些工作適用于 Windows Media 裝置管理員上建的應用程式和服務提供者。

-   如果您選擇不使用 SDK 隨附的標頭檔，則[編譯 SDK 所提供的](compiling-the-idl-files-supplied-with-the-sdk.md)idl 檔案會說明哪些 idl 檔案是用來建立標頭檔。
-   [啟用記錄](enabling-logging.md) 會描述 Windows Media 裝置管理員提供的記錄物件，以便從執行中的物件記錄資料。
-   [使用安全驗證通道](using-secure-authenticated-channels.md) 描述安全驗證通道，用於加密、解密、元件和訊息驗證。 任何建立在 Windows Media 上的裝置管理員都必須至少能夠執行元件驗證。
-   [處理受保護的內容](handling-protected-content.md) 會說明您的應用程式如何處理 Windows Media 數位版權管理 (DRM) 所保護的內容。
-   [取得和設定中繼資料和屬性](getting-and-setting-metadata-and-attributes.md) 會描述應用程式如何取得及設定儲存體或裝置的屬性和中繼資料。
