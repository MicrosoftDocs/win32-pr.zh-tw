# <a name="interfaces-for-service-providers-and-applications"></a>服務提供者和應用程式的介面

服務提供者和使用 Windows Media 裝置管理員 SDK 的應用程式都可以使用下列介面。



| 介面                                                | 描述                                                                               |
|----------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**IWMDMLogger**](/windows/desktop/api/wmdmlog/nn-wmdmlog-iwmdmlogger)                       | 記錄一般記錄檔中的專案。                                                        |
| [**IComponentAuthenticate**](/windows/desktop/api/mswmdm/nn-mswmdm-icomponentauthenticate) | 在 Windows 媒體裝置管理員的模組之間提供安全的加密通訊。 |
