# <a name="wmdm-glossary"></a>WMDM 詞彙

若要在詞彙中尋找字詞，請按一下您想要查閱的字詞中的第一個字母的字母。

您也可以在 [說明] 的文字中，按一下加底線的詞彙連結，以讀取詞彙中的詞彙。 按一下詞彙連結之後，詞彙和定義會出現在快顯視窗中。 若要關閉視窗，請按一下螢幕上的任何位置。

<dl> <dt>

<span id="projectid114.active_template_library__atl_"></span><span id="PROJECTID114.ACTIVE_TEMPLATE_LIBRARY__ATL_"></span>**Active Template Library (ATL)**
</dt> <dd>

Microsoft Visual C++ 的範本類別集合。 它是用來在撰寫 Windows Media 裝置管理員的服務提供者時建立 COM 伺服器。

</dd> <dt>

<span id="projectid114.atl"></span><span id="PROJECTID114.ATL"></span>**ATL**
</dt> <dd>

請參閱： [Active Template Library (ATL) ](/windows)的定義

</dd> <dt>

<span id="projectid114.gls_authentication"></span><span id="PROJECTID114.GLS_AUTHENTICATION"></span>**驗證**
</dt> <dd>

藉由驗證實體提供的認證，確認使用者、電腦、處理序或其他實體之識別的程序。 常見的認證形式包括數位簽章、智慧卡、生物識別資料，以及使用者名稱和密碼的組合。

</dd> <dt>

<span id="projectid114.authorization"></span><span id="PROJECTID114.AUTHORIZATION"></span>**授權**
</dt> <dd>

若要授與存取權給電腦系統或網路上的資源，通常是在實體驗證期間。 經過驗證的實體可能會被授權不同類型的存取或活動。

</dd> <dt>

<span id="projectid114.bit_rate"></span><span id="PROJECTID114.BIT_RATE"></span>**位元速率**
</dt> <dd>

每單位時間傳輸的位元數，通常會以每秒的位元數來表示。

</dd> <dt>

<span id="projectid114.buffer"></span><span id="PROJECTID114.BUFFER"></span>**緩衝區**
</dt> <dd>

在接收裝置或應用程式使用資料前，用於暫時保留資料所預留的記憶體區域。 緩衝可防止資料傳輸的中斷。

</dd> <dt>

<span id="projectid114.challenge"></span><span id="PROJECTID114.CHALLENGE"></span>**挑戰**
</dt> <dd>

授權的要求。 挑戰包含取用者電腦的相關資訊、要求的許可權清單以及內容的其他資訊，包括內容標頭和金鑰識別碼。 這項挑戰也可以包含第1版的挑戰，以防需要第1版授權。

</dd> <dt>

<span id="projectid114.clear_content"></span><span id="PROJECTID114.CLEAR_CONTENT"></span>**清除內容**
</dt> <dd>

未加密的 Windows Media 檔案。

</dd> <dt>

<span id="projectid114.client"></span><span id="PROJECTID114.CLIENT"></span>**用戶端**
</dt> <dd>

任何連接到其他電腦或程式或是要求其他電腦或程式提供服務的電腦或程式。 用戶端也可以參考啟用電腦或程式來建立連線的軟體。

</dd> <dt>

<span id="projectid114.content"></span><span id="PROJECTID114.CONTENT"></span>**內容**
</dt> <dd>

數位媒體檔或串流包含的音訊、視訊、影像、文字或任何其他資訊。

</dd> <dt>

<span id="projectid114.decrypt"></span><span id="PROJECTID114.DECRYPT"></span>**解密**
</dt> <dd>

將已加密的內容轉換回其原始格式。

</dd> <dt>

<span id="projectid114.digital_rights_management__drm_"></span><span id="PROJECTID114.DIGITAL_RIGHTS_MANAGEMENT__DRM_"></span>**數位版權管理 (DRM)**
</dt> <dd>

任何用來保護數位內容和服務擁有者之興趣的技術 (例如著作權擁有者) 。 一般而言，授權的收件者或使用者必須取得授權，才能使用該內容。

</dd> <dt>

<span id="projectid114.drm"></span><span id="PROJECTID114.DRM"></span>**DRM**
</dt> <dd>

請參閱：[數位版權管理 (DRM) ](/windows)的定義

</dd> <dt>

<span id="projectid114.encrypt"></span><span id="PROJECTID114.ENCRYPT"></span>**加密**
</dt> <dd>

以程式設計方式偽裝內容以隱藏其物質。

</dd> <dt>

<span id="projectid114.firewall"></span><span id="PROJECTID114.FIREWALL"></span>**防火牆**
</dt> <dd>

一個安全性解決方案，可將網路的其中一個部分從另一個部分單獨分開，根據流量篩選規則，僅允許授權的網路流量通過。

</dd> <dt>

<span id="projectid114.key"></span><span id="PROJECTID114.KEY"></span>**金鑰**
</dt> <dd>

將已封裝的 Windows Media 檔案解除鎖定所需的資料片段。 此金鑰包含在個別的授權中。

</dd> <dt>

<span id="projectid114.license"></span><span id="PROJECTID114.LICENSE"></span>**授權**
</dt> <dd>

附加至受保護內容的資料，這些資料會描述如何使用內容。

</dd> <dt>

<span id="projectid114.microsoft_secure_content_provider__scp_"></span><span id="PROJECTID114.MICROSOFT_SECURE_CONTENT_PROVIDER__SCP_"></span>**Microsoft Secure Content Provider (SCP)**
</dt> <dd>

適用于 Windows Media 裝置管理員的外掛程式，可提供 (DRM) 服務的數位版權管理。

</dd> <dt>

<span id="projectid114.microsoft_windows_media_player"></span><span id="PROJECTID114.MICROSOFT_WINDOWS_MEDIA_PLAYER"></span>**Microsoft Windows Media Player**
</dt> <dd>

從 Windows Media 伺服器或本機內容接收串流以進行播放的用戶端/控制項。 它可以做為獨立用戶端可執行程式來執行，或內嵌在 Web 網頁、c + + 程式或使用用戶端 ActiveX 控制項的 Microsoft Visual Basic 程式中。

</dd> <dt>

<span id="projectid114.object"></span><span id="PROJECTID114.OBJECT"></span>**物件**
</dt> <dd>

提供特定用途並視為一個單位之屬性和方法的集合。

</dd> <dt>

<span id="projectid114.player"></span><span id="PROJECTID114.PLAYER"></span>**播放機**
</dt> <dd>

一種用戶端程式或控制項，可接收從伺服器串流處理或從本機檔案播放的數位媒體內容。 Windows Media Player 是播放程式的範例。

</dd> <dt>

<span id="projectid114.plug_in"></span><span id="PROJECTID114.PLUG_IN"></span>**外掛程式**
</dt> <dd>

擴充或增強其他軟體功能的輔助軟體元件。

</dd> <dt>

<span id="projectid114.portable_device"></span><span id="PROJECTID114.PORTABLE_DEVICE"></span>**可攜式裝置**
</dt> <dd>

可與電腦或裝置交換檔案或其他資料的行動電子裝置。 可攜式裝置的範例包含掌上型 PC、可攜式數位音樂播放機及Smartphone。

</dd> <dt>

<span id="projectid114.portable_media_identifier"></span><span id="PROJECTID114.PORTABLE_MEDIA_IDENTIFIER"></span>**可移植媒體識別碼**
</dt> <dd>

卸除式存放裝置媒體的唯一識別碼，例如 CompactFlash 卡。 識別碼已硬式編碼到儲存媒體，而且無法變更。

</dd> <dt>

<span id="projectid114.protect"></span><span id="PROJECTID114.PROTECT"></span>**保護**
</dt> <dd>

使用金鑰加密檔案並新增資訊，例如授權取得 URL。

</dd> <dt>

<span id="projectid114.sdmi"></span><span id="PROJECTID114.SDMI"></span>**SDMI**
</dt> <dd>

請參閱的定義： [安全的數位音樂計畫 (SDMI) ](/windows)

</dd> <dt>

<span id="projectid114.secure_digital_music_initiative__sdmi_"></span><span id="PROJECTID114.SECURE_DIGITAL_MUSIC_INITIATIVE__SDMI_"></span>**(SDMI) 的安全數位音樂方案**
</dt> <dd>

設定安全數位音樂標準的組織。 SDMI 的主要目標之一，是要建立安全播放、儲存和散佈數位音樂的架構。

</dd> <dt>

<span id="projectid114.windows_media_encoder"></span><span id="PROJECTID114.WINDOWS_MEDIA_ENCODER"></span>**Windows Media 編碼器**
</dt> <dd>

一種技術，可將實況或預錄的音訊和影片壓縮成 Windows Media 串流，這些資料流程可以立即發佈或儲存為 Windows Media 檔案，以供稍後散發。

</dd> <dt>

<span id="projectid114.windows_media_file"></span><span id="PROJECTID114.WINDOWS_MEDIA_FILE"></span>**Windows Media** 檔案
</dt> <dd>

包含音訊、視訊或指令碼資料的檔案。 檔案的內容是以其中一種 Windows Media 編解碼器編碼。

</dd> <dt>

<span id="projectid114.windows_media_format"></span><span id="PROJECTID114.WINDOWS_MEDIA_FORMAT"></span>**Windows Media 格式**
</dt> <dd>

以 Windows Media 編解碼器編碼的數位媒體檔案或資料流程的格式。

</dd> <dt>

<span id="projectid114.windows_media_services"></span><span id="PROJECTID114.WINDOWS_MEDIA_SERVICES"></span>**Windows Media Services**
</dt> <dd>

用來透過網際網路或內部網路串流數位音訊和影片的服務。

</dd> </dl>
