# <a name="wmdm-reference"></a>WMDM 參考

Windows Media 裝置管理員 SDK 是由介面、結構、列舉和常數的集合所組成。 參考區段會根據使用這些介面的物件類型，將它們分割成群組。



| 區段                                                                                                    | 描述                                                                                                                                                                                         |
|------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [應用程式的介面](interfaces-for-applications.md)                                             | 描述僅供桌面應用程式、Windows Media Player 外掛程式或需要高階可攜式裝置存取的 COM 物件使用或執行的介面。                     |
| [服務提供者的介面](interfaces-for-service-providers.md)                                   | 描述只由服務提供者使用或執行的介面，此介面會處理與可攜式裝置的實際低層級通訊。                                                |
| [Windows Media DRM-Implemented 介面](windows-media-drm-implemented-interfaces.md)                   | 描述不打算由協力廠商服務提供者執行的介面，而是由 Windows Media DRM 和 Windows Media 裝置管理員在內部執行和呼叫的介面。 |
| [服務提供者和應用程式的介面](interfaces-for-service-providers-and-applications.md) | 描述應用程式或服務提供者可以使用的介面。                                                                                                               |
| [安全內容提供者的介面](interfaces-for-secure-content-providers.md)                     | 描述必須由裝置或應用程式所執行的介面，該裝置或應用程式將會使用受 DRM 保護的內容。                                                                                       |
| [結構](structures.md)                                                                               | 描述用於 Windows Media 裝置管理員方法的結構。                                                                                                                                 |
| [列舉類型](enumeration-types.md)                                                                 | 描述用於 Windows Media 裝置管理員方法的列舉。                                                                                                                               |
| [中繼資料常數](metadata-constants.md)                                                               | 描述 Windows Media 裝置管理員 SDK 所定義的常數，以針對裝置或裝置上的裝置或物件來設定或抓取不同的中繼資料屬性。                                 |
| [錯誤碼](error-codes.md)                                                                             | 描述 Windows Media 裝置管理員 SDK 方法可傳回的錯誤碼。                                                                                                         |
