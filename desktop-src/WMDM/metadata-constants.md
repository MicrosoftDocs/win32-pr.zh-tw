# <a name="metadata-constants"></a>中繼資料常數

中繼資料是有關從裝置傳送或傳出之內容的描述性資訊。 在 Windows Media 裝置管理員中，中繼資料是由屬性所組成，其中每個屬性都是成對的名稱/值。 名稱一律是寬字元字串。 值是 [**WMDM \_ 標記 \_ DATATYPE**](wmdm-tag-datatype.md) 列舉中所述的數個定義資料類型的其中一個。

Windows Media 裝置管理員具有預先定義的通用屬性，因此強烈建議您使用這些屬性來定義新的屬性。 針對每個預先定義的屬性，SDK 會定義包含名稱字串的全域常數。 您可以使用全域常數來減少印刷錯誤的機率。 例如，常數 g \_ wszWMDMTitle 會定義為 "title"，對應至 Title 屬性。

下列中繼資料屬性常數已定義為在此 SDK 中使用。 這些是在 mswmdm 中定義。
