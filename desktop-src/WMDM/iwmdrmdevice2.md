# <a name="iwmdrmdevice2-interface"></a>IWMDRMDevice2 介面

這個介面不是由服務提供者所執行，而是為了完成檔的目的而提供。

**IWMDRMDevice2** 介面會藉由提供取得授權狀態和部分同步處理清單的方法，來擴充 **IWMDRMDevice** 。

## <a name="members"></a>成員

**IWMDRMDevice2** 介面繼承自 [**IWMDRMDevice**](iwmdrmdevice.md)。 **IWMDRMDevice2** 也有下列類型的成員：

-   [方法](#methods)

### <a name="methods"></a>方法

**IWMDRMDevice2** 介面具有這些方法。



| 方法                                                         | 描述                                     |
|:---------------------------------------------------------------|:------------------------------------------------|
| [**GetLicenseState**](iwmdrmdevice2-getlicensestate.md)       | 取得授權狀態。<br/>              |
| [**GetPartialSyncList**](iwmdrmdevice2-getpartialsynclist.md) | 取得部分同步處理清單。<br/> |
