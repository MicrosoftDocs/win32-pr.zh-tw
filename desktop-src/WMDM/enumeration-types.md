# <a name="wmdm-enumerations"></a>WMDM 列舉

本主題說明 Windows Media 裝置管理員 SDK 的功能。 建議您將應用程式遷移至使用 Windows 可攜式裝置 API。 如需詳細資訊，請參閱 Windows 可攜式裝置 SDK。

Windows媒體裝置管理員定義下列列舉。



| 列舉類型                                                                  | 描述                                                                                     |
|-----------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [**WMDM \_ 列舉 \_ 的 \_ 有效 \_ 值 \_ 形式**](wmdm-enum-prop-valid-values-form.md) | 描述屬性的有效值可能形式。                                        |
| [**WMDM \_ 尋找 \_ 範圍**](wmdm-find-scope.md)                                      | 定義儲存物件的範圍。                                                        |
| [**WMDM \_ FORMATCODE**](wmdm-formatcode.md)                                       | 定義格式程式代碼清單，以描述從裝置傳輸的內容類型。 |
| [**WMDM \_ 會話 \_ 類型**](wmdm-session-type.md)                                  | 定義會話類型。                                                                       |
| [**WMDM \_ 儲存體 \_ 列舉 \_ 模式**](wmdm-storage-enum-mode.md)                       | 定義如何列舉儲存體上的內容。                                     |
| [**WMDM \_ 標記 \_ 資料類型**](wmdm-tag-datatype.md)                                  | 定義資料類型。                                                                            |
| [**WMDMMessage**](wmdmmessage.md)                                                | 定義訊息類型和狀態。                                                               |
