# <a name="privacy-statement-windows-media-device-manager-11-sdk"></a> (Windows Media 裝置管理員 11 SDK) 隱私權聲明

建立 Windows Media 裝置管理員應用程式的開發人員應該清楚地與他們的使用者溝通其隱私權的方式。 為了協助您建立終端使用者隱私權聲明，Microsoft 已將隱私權指導方針張貼在 [Windows Media 隱私權網頁](https://www.microsoft.com/windows/windowsmedia/privacy/10ssdk.aspx)上。
