# <a name="privacy-statement-windows-media-device-manager-11-sdk"></a>隱私權聲明 (Windows 媒體裝置管理員 11 SDK) 

建立 Windows 媒體裝置管理員應用程式的開發人員應該清楚地與他們的使用者傳達其隱私權的影響。 為了協助您建立終端使用者隱私權聲明，Microsoft 張貼了[Windows 媒體隱私權網頁](https://www.microsoft.com/windows/windowsmedia/privacy/10ssdk.aspx)的隱私權指導方針。
