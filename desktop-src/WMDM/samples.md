# <a name="wmdm-samples"></a>WMDM 範例

Windows Media 裝置管理員 SDK 包含兩個可編譯的範例：圖形化桌面應用程式範例，可讓您流覽連線的裝置、建立播放清單，以及將檔案拖曳至裝置或從中拖曳檔案;以及與電腦上的虛擬「裝置」通訊的範例服務提供者 (實際上是電腦上的硬碟機) 。 本檔說明如何編譯和執行這些專案，也提供其運作方式的相關資訊。

下列各節記載這兩個專案：

-   [桌面應用程式範例](sample-desktop-application.md)
-   [範例服務提供者](sample-service-provider.md)
