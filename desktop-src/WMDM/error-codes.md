# <a name="error-codes-windows-media-device-manager-11-sdk"></a> (Windows Media 裝置管理員 11 SDK) 的錯誤碼

Windows Media 裝置管理員方法可能會傳回各種不同的錯誤碼。 其中有些是 Windows Media 裝置管理員特有的，有些則是由其他 Sdk 或移植套件所定義，而這些套件可用來建立裝置，並向上升至 Windows Media 裝置管理員方法。 本節說明 Windows Media 技術可傳回的各種錯誤碼，並依來源組織。

**Windows Media 裝置管理員錯誤碼**

下列錯誤碼是由 Windows Media 裝置管理員 SDK 所定義。

服務提供者應避免傳回一般失敗代碼，例如，E \_ 失敗。 它應該盡可能以特定的錯誤碼傳回，因此，如果錯誤傳播至應用程式，應用程式就可以對使用者提供有意義的意見反應。

對於儲存體和裝置相關作業中的錯誤，服務提供者應該會採用 Win32 錯誤碼。 服務提供者可以 [使用 \_ 來自 \_ WIN32 的 hresult](/windows/win32/api/winerror/nf-winerror-hresult_from_win32) 將 Windows 錯誤碼轉換成 **hresult** 值，並使用它們做為傳回值。 這個宏和錯誤碼會定義在 winerror.h 中，這是 Platform SDK 的一部分。
