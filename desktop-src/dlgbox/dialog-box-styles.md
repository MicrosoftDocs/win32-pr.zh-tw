# <a name="dialog-box-styles"></a>對話方塊樣式

下表列出當您建立對話方塊時，可以指定的對話方塊樣式。 您可以使用這些樣式來呼叫 [**CreateWindow**](/windows/desktop/api/winuser/nf-winuser-createwindowa)和 [**CreateWindowEx**](/windows/desktop/api/winuser/nf-winuser-createwindowexa)函數、 [**DLGTEMPLATE**](/windows/desktop/api/Winuser/ns-winuser-dlgtemplate)和 [**DLGTEMPLATEEX**](dlgtemplateex.md)結構的 **style** 成員，以及資源檔中對話方塊定義的語句。



| 常數                                                                                                                                                               | 描述                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="DS_3DLOOK"></span><span id="ds_3dlook"></span><dl> <dt>**DS \_ 3DLOOK**</dt> </dl>                      | 已過時。 系統會自動將三維外觀套用至應用程式所建立的對話方塊。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| <span id="DS_ABSALIGN"></span><span id="ds_absalign"></span><dl> <dt>**DS \_ ABSALIGN**</dt> </dl>                | 指出對話方塊的座標為螢幕座標。 如果未指定此樣式，座標就是用戶端座標。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="DS_CENTER"></span><span id="ds_center"></span><dl> <dt>**DS \_ 中心**</dt> </dl>                      | 將對話方塊置中包含擁有者視窗的監視工作區域。 如果未指定任何擁有者視窗，此對話方塊會在系統所決定的監視工作區域中置中。 工作區域是工作列或任何透過像 appbars 未遮蔽的區域。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| <span id="DS_CENTERMOUSE"></span><span id="ds_centermouse"></span><dl> <dt>**DS \_ CENTERMOUSE**</dt> </dl>       | 將滑鼠游標上的對話方塊置中。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="DS_CONTEXTHELP"></span><span id="ds_contexthelp"></span><dl> <dt>**DS \_ CONTEXTHELP**</dt> </dl>       | 在對話方塊的標題列中包含問號。 當使用者按一下問號時，游標會變更為具有指標的問號。 如果使用者接著按一下對話方塊中的控制項，控制項就會收到 [**WM \_**](../shell/wm-help.md) 說明訊息。 控制項應將訊息傳遞至對話方塊程式，此程式應該使用 **help \_ WM \_ help** 命令呼叫函式。 Help 應用程式會顯示一個快顯視窗，其中通常包含控制項的說明。<br/> 請注意， **DS \_ CONTEXTHELP** 只是一個預留位置。 當對話方塊建立時，系統會檢查 **DS \_ CONTEXTHELP** ，如果有的話，會將 **WS \_ EX \_ CONTEXTHELP** 加入至對話方塊的延伸樣式。 **WS \_例如， \_ CONTEXTHELP** 不能與 **Ws \_ MAXIMIZEBOX** 或 **ws \_ MINIMIZEBOX** 樣式搭配使用。<br/>                                                                              |
| <span id="DS_CONTROL"></span><span id="ds_control"></span><dl> <dt>**DS \_ 控制項**</dt> </dl>                   | 建立一個對話方塊，此對話方塊可以做為另一個對話方塊的子視窗，與屬性工作表中的頁面很類似。 這種樣式可讓使用者在子對話方塊的控制項視窗之間進行索引標籤、使用其快速鍵等等。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="DS_FIXEDSYS"></span><span id="ds_fixedsys"></span><dl> <dt>**DS \_ FIXEDSYS**</dt> </dl>                | 讓對話方塊使用 **系統 \_ 固定 \_ 字型** ，而非預設 **系統 \_ 字型**。 這是與3.0 之前 Windows 之16位版本中的系統字型相容的等寬字型。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="DS_LOCALEDIT"></span><span id="ds_localedit"></span><dl> <dt>**DS \_ LOCALEDIT**</dt> </dl>             | 僅適用于16位應用程式。 此樣式會指示對話方塊中的 [編輯] 控制項，從應用程式的資料區段配置記憶體。 否則，編輯控制項會從全域記憶體物件配置儲存區。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| <span id="DS_MODALFRAME"></span><span id="ds_modalframe"></span><dl> <dt>**DS \_ MODALFRAME**</dt> </dl>          | 藉由指定 **WS \_ 標題** 和 **ws \_ SYSMENU** 樣式，建立具有強制回應對話方塊框架（可與標題列和視窗功能表結合）的對話方塊。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| <span id="DS_NOFAILCREATE"></span><span id="ds_nofailcreate"></span><dl> <dt>**DS \_ NOFAILCREATE**</dt> </dl>    | 即使發生錯誤（例如，如果無法建立子視窗，或系統無法建立編輯控制項的特殊資料區段）時，也會建立對話方塊。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| <span id="DS_NOIDLEMSG"></span><span id="ds_noidlemsg"></span><dl> <dt>**DS \_ NOIDLEMSG**</dt> </dl>             | 隱藏在對話方塊顯示時，系統會傳送給對話方塊擁有者的 [**WM \_ ENTERIDLE**](wm-enteridle.md) 訊息。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="DS_SETFONT"></span><span id="ds_setfont"></span><dl> <dt>**DS \_ SETFONT**</dt> </dl>                   | 指出對話方塊範本的標頭 ([標準] 或 [擴充]) 包含其他資料，以指定要用於工作區中文字的字型，以及對話方塊的控制項。 可能的話，系統會根據指定的字型資料來選取字型。 系統會透過傳送 [**WM \_ SETFONT**](/windows/desktop/winmsg/wm-setfont) 訊息的方式，將字型的控制碼傳遞至對話方塊和每個控制項。 如需此字型資料格式的說明，請參閱 [**DLGTEMPLATE**](/windows/desktop/api/Winuser/ns-winuser-dlgtemplate) 和 [**DLGTEMPLATEEX**](dlgtemplateex.md)。<br/> 如果未指定 **ds \_ SETFONT** 或 **ds \_ SHELLFONT** ，對話方塊範本就不會包含字型資料。<br/>                                                                                                                                                                                                                                 |
| <span id="DS_SETFOREGROUND"></span><span id="ds_setforeground"></span><dl> <dt>**DS \_ SETFOREGROUND**</dt> </dl> | 導致系統使用 [**SetForegroundWindow**](/windows/desktop/api/winuser/nf-winuser-setforegroundwindow) 函式，將對話方塊帶入前景。 這種樣式適用于需要從使用者立即注意的強制回應對話方塊，不論擁有者視窗是否為前景視窗。<br/> 系統會限制哪些進程可以設定前景視窗。 如需詳細資訊，請參閱[前景和背景 Windows](/windows/desktop/winmsg/window-features)。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| <span id="DS_SHELLFONT"></span><span id="ds_shellfont"></span><dl> <dt>**DS \_ SHELLFONT**</dt> </dl>             | 指出對話方塊應使用系統字型。 延伸對話方塊範本的 [ **字樣** ] 成員必須設定為 [MS Shell Dlg]。 否則，此樣式不會有任何作用。 也建議您使用 [DIALOGEX 資源](../menurc/dialogex-resource.md)，而不是 [對話方塊資源](../menurc/dialog-resource.md)。 如需詳細資訊，請參閱 [對話方塊](about-dialog-boxes.md)字型。<br/> 系統會使用 **dialog**、 **權數** 和 **斜體** 成員中指定的字型資料來選取字型。 系統會透過傳送 [**WM \_ SETFONT**](/windows/desktop/winmsg/wm-setfont) 訊息的方式，將字型的控制碼傳遞至對話方塊和每個控制項。 如需此字型資料格式的說明，請參閱 [**DLGTEMPLATEEX**](dlgtemplateex.md)。 <br/> 如果未指定 **ds \_ SHELLFONT** 或 **ds \_ SETFONT** ，擴充的對話方塊範本就不會包含字型資料。<br/> |
| <span id="DS_SYSMODAL"></span><span id="ds_sysmodal"></span><dl> <dt>**DS \_ SYSMODAL**</dt> </dl>                | 這種樣式已過時，而且包含在 Windows 的16位版本相容性中。 如果您指定此樣式，系統會建立具有 **WS \_ EX \_ 最上層** 樣式的對話方塊。 此樣式不會防止使用者存取桌面上的其他視窗。<br/> 請勿將此樣式與 **DS \_ 控制項** 樣式合併。<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
