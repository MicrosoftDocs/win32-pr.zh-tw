# <a name="iremotedesktopclienteventsondialogdismissed-method"></a>IRemoteDesktopClientEvents：： OnDialogDismissed 方法

## <a name="syntax"></a>語法


```C++
void OnDialogDismissed();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
