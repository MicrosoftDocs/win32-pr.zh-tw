# <a name="imsrdpdevicev2-properties"></a>IMsRdpDeviceV2 屬性

[**IMsRdpDeviceV2**](imsrdpdevicev2.md)介面會公開下列屬性。

## <a name="in-this-section"></a>本節內容

<dl> <dt>

[**CmClassGuid 屬性**](imsrdpdevicev2-cmclassguid.md)
</dt> <dd>

包含裝置的 configuration manager 安裝類別 GUID。

</dd> <dt>

[**CmDeviceInstance 屬性**](imsrdpdevicev2-cmdeviceinstance.md)
</dt> <dd>

包含裝置的 configuration manager 裝置實例。

</dd> <dt>

[**DeviceText 屬性**](imsrdpdevicev2-devicetext.md)
</dt> <dd>

包含裝置文字。

</dd> <dt>

[**DriveLetterBitmap 屬性**](imsrdpdevicev2-driveletterbitmap.md)
</dt> <dd>

包含代表裝置內所含之磁碟機號對應的位位。

</dd> <dt>

[**IsCompositeDevice 屬性**](imsrdpdevicev2-iscompositedevice.md)
</dt> <dd>

指定裝置是否為複合裝置。

</dd> <dt>

[**IsOptionalDevice 屬性**](imsrdpdevicev2-isoptionaldevice.md)
</dt> <dd>

指定裝置是否為 USB 重新導向的選用裝置。

</dd> <dt>

[**IsUSBDevice 屬性**](imsrdpdevicev2-isusbdevice.md)
</dt> <dd>

指定裝置是否用於 USB 重新導向。

</dd> </dl>
