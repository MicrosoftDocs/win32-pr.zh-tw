# <a name="remote-desktop-protocol-provider-enumerations"></a>遠端桌面通訊協定提供者列舉

遠端桌面通訊協定提供者 API 支援下列列舉類型。

## <a name="in-this-section"></a>本節內容

<dl> <dt>

[**WRDS \_ 連接 \_ 設定 \_ 層級**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wrds_connection_setting_level)
</dt> <dd>

指定 [**WRDS \_ 連接 \_ 設定**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_connection_settings)結構之 **WRdsConnectionSetting** 成員中包含的結構類型。

</dd> <dt>

[**WRDS 接聽程式 \_ \_ 設定 \_ 層級**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wrds_listener_setting_level)
</dt> <dd>

用來指定包含在 WRDS 接聽程式 [**\_ \_ 設定**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_listener_settings)結構之 **WRdsListenerSetting** 成員中的結構類型。

</dd> <dt>

[**WRDS \_ 設定 \_ 層級**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wrds_setting_level)
</dt> <dd>

指定包含在 [**WRDS \_ 設定**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_settings)結構之 **WRdsSetting** 成員中的結構類型。

</dd> <dt>

[**WRDS \_ 設定 \_ 狀態**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wrds_setting_status)
</dt> <dd>

針對 [**WRDS \_ SETTINGS \_ 1**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_settings_1) 結構的各種成員，指定原則設定的狀態。

</dd> <dt>

[**WRDS \_ 設定 \_ 類型**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wrds_setting_type)
</dt> <dd>

指定儲存在 [**WRDS \_ 設定**](/windows/desktop/api/Wtsdefs/ns-wtsdefs-wrds_settings) 結構中的設定類別。

</dd> <dt>

[**WTS \_憑證 \_ 類型**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wts_cert_type)
</dt> <dd>

包含值，這些值會指定用來取得授權的憑證類型。

</dd> <dt>

[**WTS \_登入 \_ 錯誤 \_ 重新導向程式 \_ 回應**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wts_logon_error_redirector_response)
</dt> <dd>

包含值，這些值會指定通訊協定對登入錯誤的慣用回應。

</dd> <dt>

[**WTS \_RCM \_ 清空 \_ 狀態**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wts_rcm_drain_state)
</dt> <dd>

包含遠端桌面工作階段主機 (RD 工作階段主機) server 的清空狀態相關資訊。

</dd> <dt>

[**WTS \_RCM \_ 服務 \_ 狀態**](/windows/desktop/api/Wtsdefs/ne-wtsdefs-wts_rcm_service_state)
</dt> <dd>

包含遠端桌面服務服務狀態的相關資訊。

</dd> </dl>
