# <a name="imstscaxeventsonwarning-method"></a>IMsTscAxEvents：： OnWarning 方法

當用戶端控制項遇到不嚴重的錯誤狀況時呼叫。

## <a name="syntax"></a>語法


```C++
void OnWarning(
  [in] long warningCode
);
```



## <a name="parameters"></a>參數

<dl> <dt>

*warningCode* \[在\]
</dt> <dd>

錯誤碼。

<dt>

<span id="1"></span>

<span id="1"></span>**1**


</dt> <dd>

點陣圖快取已損毀。

</dd> </dl> </dd> </dl>

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
