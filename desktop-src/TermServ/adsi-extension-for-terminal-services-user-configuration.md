# <a name="adsi-extension-for-remote-desktop-services-user-configuration"></a>遠端桌面服務使用者設定的 ADSI 擴充功能

Active Directory 服務介面 (ADSI) 延伸模組是一種特殊的 COM 物件，可延伸 COM 物件。 ADSI 擴充功能可以將功能加入至目錄物件，並簡化管理工作。

遠端桌面服務使用者設定的 ADSI 擴充功能延伸了 ADSI 物件，可讓網域系統管理員撰寫腳本，以自動維護遠端桌面服務特定的使用者屬性。

遠端桌面服務使用者設定的 ADSI 擴充功能包含在 Windows Server 中。

如需詳細資訊，請參閱下列主題：

-   [使用遠端桌面服務使用者設定的 ADSI 擴充功能](using-the-adsi-extension-for-terminal-services-user-configuration.md)
-   [遠端桌面服務使用者設定的 ADSI 擴充功能的參考](reference-for-the-adsi-extension-for-terminal-services-user-configuration.md)

如需 ADSI 的一般資訊，請參閱 [Adsi 擴充](/windows/desktop/ADSI/adsi-extensions)功能、 [adsi 屬性](/windows/desktop/ADSI/the-adsi-attribute-cache)快取和 [Active Directory 服務介面](/windows/desktop/ADSI/active-directory-service-interfaces-adsi)。
