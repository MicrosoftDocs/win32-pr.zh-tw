# <a name="imsrdpclientadvancedsettingsbitmappersistence-property"></a>IMsRdpClientAdvancedSettings：： BitmapPersistence 屬性

指定是否應該使用持續性點陣圖快取。 持續性快取可以改善效能，但需要額外的磁碟空間。

這是可讀寫的屬性。

## <a name="syntax"></a>Syntax


```C++
HRESULT put_BitmapPersistence(
  [in]  LONG bitmapPeristence
);

HRESULT get_BitmapPersistence(
  [out] LONG *pbitmapPeristence
);
```



## <a name="property-value"></a>屬性值

將此參數設定為0可停用快取，或設定為非零值以啟用快取。

> [!Note]  
> 參數名稱中的拼寫錯誤是在控制項的發行版本中。

 

## <a name="error-codes"></a>錯誤碼

如果成功，則傳回 **S \_ OK** 。
