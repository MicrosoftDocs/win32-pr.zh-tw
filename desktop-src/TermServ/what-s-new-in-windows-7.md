# <a name="whats-new-in-windows-7"></a>Windows 7 的新功能

Windows 7 為遠端桌面服務引進了下列新的程式設計項目。




| 需求 | 值 |
|--------|-------|
| 遠端桌面 ActiveX 控制介面<br /> | 下列介面 (與其相關聯的屬性) 已新增至<a href="remote-desktop-web-connection-reference.md">遠端桌面 ActiveX 控制項介面</a>。<br /><ul><li><a href="imsrdpclientadvancedsettings7.md"><strong>IMsRdpClientAdvancedSettings7 介面</strong></a></li></ul> | 
| 遠端桌面服務 Web 存取控制 API<br /> | 遠端桌面服務 Web 存取控制項會公開介面，以支援 RemoteApp 和桌面連線中的使用者特定資源的發行。<br /><ul><li><a href="/windows/desktop/TermServ/remote-desktop-services-web-access-control-interfaces">遠端桌面服務 Web 存取控制 API</a></li></ul> | 
| RemoteApp 和桌面連線執行時間 API<br /> | RemoteApp 和桌面連線執行時間 API 支援在 RemoteApp 和桌面連線中開發自訂用戶端。<br /><ul><li><a href="/windows/desktop/TermServ/remoteapp-and-desktop-connection-runtime-interfaces">RemoteApp 和桌面連線執行時間 API</a></li></ul> |
