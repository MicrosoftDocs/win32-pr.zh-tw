# <a name="imsrdpclientnonscriptable3showredirectionwarningdialog-property"></a>IMsRdpClientNonScriptable3：： ShowRedirectionWarningDialog 屬性

這是可讀寫的屬性。

## <a name="syntax"></a>Syntax


```C++
HRESULT put_ShowRedirectionWarningDialog(
  [in]  VARIANT_BOOL fShowRdrDlg
);

HRESULT get_ShowRedirectionWarningDialog(
  [out] VARIANT_BOOL *pfShowRdrDlg
);
```



## <a name="property-value"></a>屬性值

指定是否應顯示 [重新導向警告] 對話方塊。
