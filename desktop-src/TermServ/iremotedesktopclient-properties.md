# <a name="iremotedesktopclient-properties"></a>IRemoteDesktopClient 屬性

[**IRemoteDesktopClient**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclient)介面會公開下列屬性。

## <a name="in-this-section"></a>本節內容

<dl> <dt>

[**動作屬性**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-get_actions)
</dt> <dd>

抓取遠端桌面通訊協定 (RDP) 應用程式容器用戶端的動作物件。

</dd> <dt>

[**Settings 屬性**](iremotedesktopclient-settings.md)
</dt> <dd>

抓取遠端桌面通訊協定 (RDP) 應用程式容器用戶端的設定物件。

</dd> <dt>

[**TouchPointer 屬性**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclient-get_touchpointer)
</dt> <dd>

包含遠端桌面通訊協定 (RDP) 應用程式容器用戶端的 [**RemoteDesktopClientTouchPointer**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclienttouchpointer) 物件。

</dd> </dl>
