# <a name="imsrdpclientnonscriptable3drivecollection-property"></a>IMsRdpClientNonScriptable3：:D riveCollection 屬性

抓取要重新導向之磁片磁碟機物件的集合。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```C++
HRESULT get_DriveCollection(
  [out] IMsRdpDriveCollection **ppDriveCollection
);
```



## <a name="property-value"></a>屬性值

抓取 [**IMSRdpDrive**](imsrdpdrive.md)類型之磁片磁碟機物件的集合。
