# <a name="imsrdpclientadvancedsettingsminutestoidletimeout-property"></a>IMsRdpClientAdvancedSettings：： MinutesToIdleTimeout 屬性

指定用戶端在沒有使用者輸入的情況下，應維持連線的最大時間長度（以分鐘為單位）。 如果經過指定的時間，控制項會呼叫 [**IMsTscAxEvents：： OnIdleTimeoutNotification**](imstscaxevents-onidletimeoutnotification.md) 方法。

在需要中斷閒置會話連線的情況下，您可以使用此屬性。例如，在 kiosk 環境中。

這是可讀寫的屬性。

## <a name="syntax"></a>Syntax


```C++
HRESULT put_MinutesToIdleTimeout(
  [in]  LONG minutesToIdleTimeout
);

HRESULT get_MinutesToIdleTimeout(
  [out] LONG *pminutesToIdleTimeout
);
```



## <a name="property-value"></a>屬性值

新的時間（以分鐘為單位）。 預設值為零，它會停用此功能。 最大值為240，代表4小時。

## <a name="error-codes"></a>錯誤碼

如果成功，則傳回 **S \_ OK** 。
