# <a name="remote-desktop-services-audioendpoint-structures"></a>遠端桌面服務 AudioEndpoint 結構

下列結構會與 AudioEndpoint API 搭配使用。

## <a name="in-this-section"></a>本節內容

<dl> <dt>

[**AE \_ 目前 \_ 位置**](/windows/desktop/api/Audioengineendpoint/ns-audioengineendpoint-ae_current_position)
</dt> <dd>

將目前的畫面格位置從裝置報告到用戶端。

</dd> <dt>

[**APO \_ 連接 \_ 屬性**](/windows/desktop/api/Audioapotypes/ns-audioapotypes-apo_connection_property)
</dt> <dd>

包含動態變更的連接屬性。

</dd> </dl>

## <a name="remarks"></a>備註

遠端桌面服務 AudioEndpoint API 適用于遠端桌面案例;它不適用於用戶端應用程式。
