# <a name="imsrdpclienttransportsettingsgatewayhostname-property"></a>IMsRdpClientTransportSettings：： GatewayHostname 屬性

指定遠端桌面閘道 (RD 閘道) server 的主機名稱。

這是可讀寫的屬性。

## <a name="syntax"></a>Syntax


```C++
HRESULT put_GatewayHostname(
  [in]  BSTR newVal
);

HRESULT get_GatewayHostname(
  [out] BSTR *pProxyHostname
);
```



## <a name="property-value"></a>屬性值

包含 RD 閘道 server 主機名稱的字串。

## <a name="error-codes"></a>錯誤碼

如果成功，則傳回 **S \_ OK** 。
