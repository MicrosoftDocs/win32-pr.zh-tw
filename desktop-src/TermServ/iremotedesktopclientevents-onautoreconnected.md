# <a name="iremotedesktopclienteventsonautoreconnected-method"></a>IRemoteDesktopClientEvents：： OnAutoReconnected 方法

## <a name="syntax"></a>語法


```C++
void OnAutoReconnected();
```

## <a name="parameters"></a>參數

這個方法不會傳回值。
