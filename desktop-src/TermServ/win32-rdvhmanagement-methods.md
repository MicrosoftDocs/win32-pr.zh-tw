# <a name="win32_rdvhmanagement-methods"></a>Win32 \_ RdvhManagement 方法

[**Win32 \_ RdvhManagement 方法**](win32-rdvhmanagement.md)類別會公開下列方法。

## <a name="in-this-section"></a>本節內容

<dl> <dt>

[**RdvMigrateVmToDifferentHost 方法**](rdvmigratevmtodifferenthost-win32-rdvhmanagement.md)
</dt> <dd>

起始虛擬機器到指定主機的即時移轉。

</dd> <dt>

[**RdvCopyBaseVmLocally 方法**](rdvcopybasevmlocally-win32-rdvhmanagement.md)
</dt> <dd>

將本地基礎虛擬機器複製到指定的遠端桌面虛擬化 (RDV) 主機伺服器。

</dd> <dt>

[**RdvSetupVMPermissions 方法**](rdvsetupvmpermissions-win32-rdvhmanagement.md)
</dt> <dd>

在虛擬機器上設定目前使用者的許可權。

</dd> <dt>

[**RdvUndoVMPermissions 方法**](rdvundovmpermissions-win32-rdvhmanagement.md)
</dt> <dd>

在指定的虛擬機器上還原 [**RdvSetupVMPermissions**](rdvsetupvmpermissions-win32-rdvhmanagement.md) 所設定的許可權。

</dd> <dt>

[**RdvCreateUserDiskTemplate 方法**](rdvcreateuserdisktemplate-win32-rdvhmanagement.md)
</dt> <dd>

使用指定的最大大小，在指定的位置建立使用者磁片範本。

</dd> </dl>
