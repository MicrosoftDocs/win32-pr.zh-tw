# <a name="imsrdpdevicecollection2-interface"></a>IMsRdpDeviceCollection2 介面

代表裝置物件的集合。 這是 [**IMsRdpDeviceCollection**](imsrdpdevicecollection.md) 介面的增強功能。

## <a name="members"></a>成員

**IMsRdpDeviceCollection2** 介面繼承自 [**IMsRdpDeviceCollection**](imsrdpdevicecollection.md)。 **IMsRdpDeviceCollection2** 也有下列類型的成員：

-   [方法](#methods)

### <a name="methods"></a>方法

**IMsRdpDeviceCollection2** 介面具有這些方法。



| 方法                                                                         | 描述                                                                                                 |
|:-------------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------|
| [**AddDeviceByInstanceId**](imsrdpdevicecollection2-adddevicebyinstanceid.md) | 將未列出的裝置新增至裝置集合。<br/>                                                |
| [**RedirectNow**](imsrdpdevicecollection2-redirectnow.md)                     | 強制將從集合中選取或取消選取的裝置重新導向或移除。<br/> |
