# <a name="win32_terminalsetting-class"></a>Win32 \_ TerminalSetting 類別

**Win32 \_ TerminalSetting** WMI 類別代表可以套用至終端機的設定。

下列語法簡化自 MOF 程式碼，並依字母順序包含所有定義和繼承的屬性。

## <a name="syntax"></a>語法

``` syntax
[abstract, AMENDMENT]
class Win32_TerminalSetting : CIM_Setting
{
  string   Caption;
  string   Description;
  datetime InstallDate;
  string   Name;
  string   Status;
  string   TerminalName;
};
```
