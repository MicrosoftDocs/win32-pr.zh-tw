# <a name="iremotedesktopclienttouchpointer-properties"></a>IRemoteDesktopClientTouchPointer 屬性

[**IRemoteDesktopClientTouchPointer**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclienttouchpointer)介面會公開下列屬性。

## <a name="in-this-section"></a>本節內容

<dl> <dt>

[**Enabled 屬性**](iremotedesktopclienttouchpointer-enabled.md)
</dt> <dd>

RDP 應用程式容器用戶端控制項上是否啟用觸控指標功能。

</dd> <dt>

[**EventsEnabled 屬性**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclienttouchpointer-get_eventsenabled)
</dt> <dd>

是否針對 RDP 應用程式容器用戶端控制項啟用觸控指標事件通知。

</dd> <dt>

[**PointerSpeed 屬性**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclienttouchpointer-get_pointerspeed)
</dt> <dd>

觸控指標游標在虛擬桌面上的移動速度，相對於用戶端上的手勢速度有多快。

</dd> </dl>
