# <a name="scriptable-virtual-channels"></a>可編寫腳本的虛擬通道

如需使用可編寫腳本之虛擬通道的相關資訊，請參閱 [使用遠端桌面網頁連線執行可編寫腳本的虛擬通道](implementing-scriptable-virtual-channels-using-remote-desktop-web-connection.md)。

如需虛擬通道的相關資訊，請參閱 [遠端桌面服務虛擬通道](terminal-services-virtual-channels.md)。

**Windows xp：** Windows xp 包含遠端桌面網頁連線，也就是終端機服務進階用戶端 (TSAC) 的更新版本。 遠端桌面網頁連線支援 [遠端桌面通訊協定](remote-desktop-protocol.md) (rdp) 5.1，並與 rdp 5.0 回溯相容。
