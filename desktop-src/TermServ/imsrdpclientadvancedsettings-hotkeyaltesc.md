# <a name="imsrdpclientadvancedsettingshotkeyaltesc-property"></a>IMsRdpClientAdvancedSettings：： HotKeyAltEsc 屬性

指定要新增至 ALT 的虛擬按鍵碼，以判斷 ALT + ESC 的熱鍵取代。

只有在未啟用 [**KeyboardHookMode**](imsrdpclientsecuredsettings-keyboardhookmode.md) 屬性時，這個屬性才有效。

這是可讀寫的屬性。

## <a name="syntax"></a>Syntax


```C++
HRESULT put_HotKeyAltEsc(
  [in]  LONG hotKeyAltEsc
);

HRESULT get_HotKeyAltEsc(
  [out] LONG *photKeyAltEsc
);
```



## <a name="property-value"></a>屬性值

新的虛擬金鑰程式碼。 **VK \_INSERT** 是預設值，以 ALT + INSERT 作為產生的序列。

## <a name="error-codes"></a>錯誤碼

如果成功，則傳回 **S \_ OK** 。
