# <a name="imsrdpclientnonscriptable3devicecollection-property"></a>IMsRdpClientNonScriptable3：:D eviceCollection 屬性

抓取要重新導向的隨插即用 (PnP) 裝置物件的集合。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```C++
HRESULT get_DeviceCollection(
  [out] IMSRdpDeviceCollection **ppDeviceCollection
);
```



## <a name="property-value"></a>屬性值

抓取 [**IMSRdpDevice**](imsrdpdevice.md)類型之 PnP 裝置物件的集合。
