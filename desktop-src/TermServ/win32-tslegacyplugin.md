# <a name="win32_tslegacyplugin-class"></a>Win32 \_ TSLegacyPlugin 類別

代表內建的 RemoteApp 和桌面連線管理服務外掛程式將會查詢 RemoteApp 程式的遠端桌面伺服器。

下列語法已經過受管理物件格式 (MOF) 程式碼簡化，並包含所有已繼承的屬性。

從 Windows Server 2012 開始，這個類別已被取代。

## <a name="syntax"></a>語法

``` syntax
[DEPRECATED, dynamic, provider("Win32_TSCentralPublisher_Prov")]
class Win32_TSLegacyPlugin : CIM_LogicalElement
{
  string   Caption;
  string   Description;
  datetime InstallDate;
  string   Name;
  string   Status;
  sint32   Type;
};
```
