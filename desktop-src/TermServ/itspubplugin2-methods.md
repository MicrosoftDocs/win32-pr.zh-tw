# <a name="itspubplugin2-methods"></a>ItsPubPlugin2 方法

[**ItsPubPlugin2**](/windows/desktop/api/tspubplugin2com/nn-tspubplugin2com-itspubplugin2)介面會公開下列方法。

## <a name="in-this-section"></a>本節內容

<dl> <dt>

[**DeletePersonalDesktopAssignment 方法**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-deletepersonaldesktopassignment)
</dt> <dd>

呼叫以刪除指定的使用者與個人虛擬桌面集合中的虛擬機器之間的對應。

</dd> <dt>

[**GetResource2 方法**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-getresource2)
</dt> <dd>

這個方法是保留的，而且一律會傳回 **E \_ >notimpl**。

</dd> <dt>

[**GetResource2List 方法**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-getresource2list)
</dt> <dd>

抓取指派給指定使用者的資源清單。

</dd> <dt>

[**ResolvePersonalDesktop 方法**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-resolvepersonaldesktop)
</dt> <dd>

呼叫以解析指定的使用者與個人虛擬桌面集合中的虛擬機器之間的對應。

</dd> </dl>
