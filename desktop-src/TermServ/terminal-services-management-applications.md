# <a name="remote-desktop-services-management-applications"></a>遠端桌面服務管理應用程式

遠端桌面服務支援下列管理應用程式。

## <a name="in-this-section"></a>本節內容

<dl> <dt>

[遠端桌面服務使用者設定的 ADSI 擴充功能](adsi-extension-for-terminal-services-user-configuration.md)
</dt> <dd>

Active Directory 服務介面 (ADSI) 延伸模組是一種特殊的 COM 物件，可延伸 COM 物件。

</dd> <dt>

[遠端桌面服務許可權](terminal-services-permissions.md)
</dt> <dd>

您可以使用提供給遠端桌面服務的許可權，來控制使用者和群組存取伺服器的方式。

</dd> </dl>

[遠端桌面服務 Wmi 提供者](terminal-services-wmi-provider.md)會提供 WMI 存取權，讓系統管理員可以使用 [遠端桌面服務設定/連線] MICROSOFT MANAGEMENT CONSOLE (MMC) 嵌入式管理單元來設定這些資訊。

[遠端桌面服務使用者設定的 Adsi 擴充](adsi-extension-for-terminal-services-user-configuration.md)功能延伸了 adsi 物件，可讓網域系統管理員撰寫腳本，以自動維護遠端桌面服務特定的使用者屬性。

如需您可以設定的許可權清單以及許可權允許的工作， [遠端桌面服務許可權](terminal-services-permissions.md)。

如需詳細資訊，請參閱 [遠端桌面服務 WMI 提供者參考](terminal-services-wmi-provider-reference.md) ，以及 [使用遠端桌面服務使用者設定的 ADSI 擴充](using-the-adsi-extension-for-terminal-services-user-configuration.md)功能。
