# <a name="imsrdpclientadvancedsettingsshadowbitmap-property"></a>IMsRdpClientAdvancedSettings：： ShadowBitmap 屬性

\[不支援這個屬性。 從 Windows Server 2008 和 Windows 7 開始，對 **ShadowBitmap** 的呼叫一律會傳回 **\_ FALSE**。\]

指定是否應該使用陰影點陣圖。

在全螢幕模式中，一律會停用陰影點陣圖，因此當處於全螢幕模式時，這個屬性不會有任何作用。

這是可讀寫的屬性。

## <a name="syntax"></a>Syntax


```C++
HRESULT put_ShadowBitmap(
  [in]  LONG shadowBitmap
);

HRESULT get_ShadowBitmap(
  [out] LONG *pshadowBitmap
);
```



## <a name="property-value"></a>屬性值

將此參數設定為0，以停用此功能或非零值以啟用此功能。 停用這項功能通常可以改善效能，但可能會在繪製畫面時產生成品。
