# <a name="win32_tssessiondirectory-methods"></a>Win32 \_ TSSessionDirectory 方法

[**Win32 \_ TSSessionDirectory**](win32-tssessiondirectory.md)類別會公開下列方法。

## <a name="in-this-section"></a>本節內容

<dl> <dt>

[**CreateUserDiskTemplate 方法**](createuserdisktemplate-win32-tssessiondirectory.md)
</dt> <dd>

建立使用者磁片範本。

</dd> <dt>

[**DisableUserVhd 方法**](disableuservhd-win32-tssessiondirectory.md)
</dt> <dd>

停用使用者設定檔 VHD。

</dd> <dt>

[**EnableUserVhd 方法**](enableuservhd-win32-tssessiondirectory.md)
</dt> <dd>

在 RDSH 伺服器上啟用使用者設定檔 VHD。

</dd> <dt>

[**GetCurrentRedirectableAddresses 方法**](getcurrentredirectableaddresses-win32-tssessiondirectory.md)
</dt> <dd>

取得目前設定的 DNS 合格地址清單，可用於重新導向。

</dd> <dt>

[**GetRedirectableAddresses 方法**](getredirectableaddresses-win32-tssessiondirectory.md)
</dt> <dd>

取得 DNS 合格位址的完整清單。

</dd> <dt>

[**PingSessionDirectory 方法**](pingsessiondirectory-win32-tssessiondirectory.md)
</dt> <dd>

檢查遠端桌面連線代理人 (RD 連線代理人) 伺服器是否可供使用。

</dd> <dt>

[**SetCurrentRedirectableAddresses 方法**](setcurrentredirectableaddresses-win32-tssessiondirectory.md)
</dt> <dd>

設定可用於重新導向之 DNS 合格位址的已設定清單。

</dd> <dt>

[**SetLoadBalancingState 方法**](setloadbalancingstate-win32-tssessiondirectory.md)
</dt> <dd>

設定值，指出伺服器是否將參與 RD 連線代理人負載平衡。

</dd> <dt>

[**SetServerWeight 方法**](setserverweight-win32-tssessiondirectory.md)
</dt> <dd>

設定 RD 連線代理人負載平衡的伺服器加權值。

</dd> <dt>

[**SetSessionDirectoryActive 方法**](win32-tssessiondirectory-setsessiondirectoryactive.md)
</dt> <dd>

**SetSessionDirectoryActive** 方法會設定 **SessionDirectoryActive** 屬性。

</dd> <dt>

[**SetSessionDirectoryExposeServerIP 方法**](win32-tssessiondirectory-setsessiondirectoryexposeserverip.md)
</dt> <dd>

設定 **SessionDirectoryExposeServerIP** 屬性。

</dd> <dt>

[**SetSessionDirectoryProperty 方法**](win32-tssessiondirectory-setsessiondirectoryproperty.md)
</dt> <dd>

設定類別的 **SessionDirectoryLocation** 屬性或 **SessionDirectoryClusterName** 屬性。

</dd> <dt>

[**SetTSRedirectorMode 方法**](settsredirectormode-win32-tssessiondirectory.md)
</dt> <dd>

設定值，指出伺服器是否將作為遠端桌面服務的重新導向器。

</dd> </dl>
