# <a name="imsrdpclient7securedsettings3-property"></a>IMsRdpClient7：： SecuredSettings3 屬性

抓取支援 [**IMsRdpClientSecuredSettings2**](imsrdpclientsecuredsettings2.md) 介面的物件。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```C++
HRESULT get_SecuredSettings3(
  [out] IMsRdpClientSecuredSettings2 **ppSecuredSettings
);
```



## <a name="property-value"></a>屬性值

接收物件之 [**IMsRdpClientSecuredSettings2**](imsrdpclientsecuredsettings2.md) 介面指標的位址。
