# <a name="imsrdpclient7remoteprogram2-property"></a>IMsRdpClient7：： RemoteProgram2 屬性

抓取支援 [**ITSRemoteProgram2**](itsremoteprogram2.md) 介面的物件。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```C++
HRESULT get_RemoteProgram2(
  [out, retval] ITSRemoteProgram2 **ppRemoteProgram
);
```



## <a name="property-value"></a>屬性值

接收物件之 [**ITSRemoteProgram2**](itsremoteprogram2.md) 介面指標的位址。
