# <a name="imsrdpclientadvancedsettingsredirectsmartcards-property"></a>IMsRdpClientAdvancedSettings：： RedirectSmartCards 屬性

指定是否允許智慧卡的重新導向。

這是可讀寫的屬性。

## <a name="syntax"></a>Syntax


```C++
HRESULT put_RedirectSmartCards(
  [in]  VARIANT_BOOL redirectSmartCards
);

HRESULT get_RedirectSmartCards(
  [out] VARIANT_BOOL *predirectSmartCards
);
```



## <a name="property-value"></a>屬性值

請將此參數設定為 **VARIANT \_ TRUE** ，否則允許重新導向或 **VARIANT \_ FALSE** 。 **變異 \_TRUE** 會提示使用者確認在連接時允許重新導向（基於安全性目的）。

## <a name="error-codes"></a>錯誤碼

如果成功，則傳回 **S \_ OK** 。
