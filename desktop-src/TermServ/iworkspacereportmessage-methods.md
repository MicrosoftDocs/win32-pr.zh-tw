# <a name="iworkspacereportmessage-methods"></a>IWorkspaceReportMessage 方法

[**IWorkspaceReportMessage**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspacereportmessage)介面支援下列方法。

## <a name="in-this-section"></a>本節內容

<dl> <dt>

[**RegisterErrorLogMessage 方法**](/windows/desktop/api/workspaceruntime/nf-workspaceruntime-iworkspacereportmessage-registererrorlogmessage)
</dt> <dd>

註冊要在 UI 中使用的指定錯誤訊息。

</dd> <dt>

[**IsErrorMessageRegistered 方法**](/windows/desktop/api/workspaceruntime/nf-workspaceruntime-iworkspacereportmessage-iserrormessageregistered)
</dt> <dd>

判斷指定的錯誤訊息是否已在指定的工作區中註冊。

</dd> <dt>

[**RegisterErrorEvent 方法**](/windows/desktop/api/workspaceruntime/nf-workspaceruntime-iworkspacereportmessage-registererrorevent)
</dt> <dd>

註冊要在 UI 中使用的指定錯誤事件訊息。

</dd> </dl>
