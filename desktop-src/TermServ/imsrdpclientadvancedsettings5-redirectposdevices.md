# <a name="imsrdpclientadvancedsettings5redirectposdevices-property"></a>IMsRdpClientAdvancedSettings5：： RedirectPOSDevices 屬性

設定或抓取服務裝置重新導向點的設定。

這是可讀寫的屬性。

## <a name="syntax"></a>Syntax


```C++
HRESULT put_RedirectPOSDevices(
  [in]  VARIANT_BOOL fRedirectPOSDevices
);

HRESULT get_RedirectPOSDevices(
  [out] VARIANT_BOOL *pfRedirectPOSDevices
);
```



## <a name="property-value"></a>屬性值

將服務裝置重新導向模式的點設定為 **TRUE** 或 **FALSE**。 如果設定為 **TRUE**，則會啟用服務裝置重新導向模式的點。
