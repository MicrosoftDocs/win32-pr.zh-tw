# <a name="imsrdpclient5remoteprogram-property"></a>IMsRdpClient5：： RemoteProgram 屬性

抓取支援 [**ITSRemoteProgram**](itsremoteprogram.md) 介面的物件。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```C++
HRESULT get_RemoteProgram(
  [out] ITSRemoteProgram **ppRemoteProgram
);
```



## <a name="property-value"></a>屬性值
