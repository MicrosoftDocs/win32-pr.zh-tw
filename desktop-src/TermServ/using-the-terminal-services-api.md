# <a name="using-the-remote-desktop-services-api"></a>使用遠端桌面服務 API

本節說明如何使用遠端桌面服務 API，讓應用程式在遠端桌面服務環境中執行下列工作：

-   執行 [遠端桌面服務的管理](terminal-services-administration.md) 工作，例如列舉和管理網域中的遠端桌面工作階段主機 (RD 工作階段主機) 伺服器中的伺服器，或列舉和管理 RD 工作階段主機伺服器上的會話與處理常式。
-   強化遠端桌面服務環境中的用戶端/伺服器應用程式。
-   [使用遠端桌面服務的虛擬通道](using-terminal-services-virtual-channels.md) 在應用程式的用戶端和伺服器模組之間進行通訊。
-   設定和取出 [遠端桌面服務的使用者](terminal-services-user-configuration.md) 設定資訊。

如果您的應用程式是在不是遠端桌面服務環境的環境中執行，但是您想要使用遠端桌面服務 API 在遠端桌面服務環境中提供其他功能，請參閱 [執行時間連結至 Wtsapi32.dll](run-time-linking-to-wtsapi32-dll.md)。
