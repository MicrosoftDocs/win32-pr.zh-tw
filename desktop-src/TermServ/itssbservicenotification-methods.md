# <a name="itssbservicenotification-methods"></a>ITsSbServiceNotification 方法

[**ITsSbServiceNotification**](/windows/desktop/api/sbtsv/nn-sbtsv-itssbservicenotification)介面會公開下列方法。

## <a name="in-this-section"></a>本節內容

<dl> <dt>

[**NotifyServiceFailure 方法**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbservicenotification-notifyservicefailure)
</dt> <dd>

通知已登錄的外掛程式，遠端桌面連線代理人 (RD 連線代理人) 服務已停止。

</dd> <dt>

[**NotifyServiceSuccess 方法**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbservicenotification-notifyservicesuccess)
</dt> <dd>

通知已註冊的外掛程式，RD 連線代理人服務已開始。

</dd> </dl>
