# <a name="imsrdpclientadvancedsettings2maxreconnectattempts-property"></a>IMsRdpClientAdvancedSettings2：： MaxReconnectAttempts 屬性

指定在自動重新連接期間嘗試重新連接的次數。

這是可讀寫的屬性。

## <a name="syntax"></a>Syntax


```C++
HRESULT put_MaxReconnectAttempts(
  [in]  LONG MaxReconnectAttempts
);

HRESULT get_MaxReconnectAttempts(
  [out] LONG *pMaxReconnectAttempts
);
```



## <a name="property-value"></a>屬性值

新的重試次數。 有效的值為0到200（含）。

## <a name="error-codes"></a>錯誤碼

如果成功，則傳回 **S \_ OK** 。

## <a name="remarks"></a>備註

連接控制項時，無法設定這個屬性。

如需遠端桌面網頁連線的詳細資訊，請參閱 [遠端桌面網頁連線的需求](requirements-for-remote-desktop-web-connection.md)。
