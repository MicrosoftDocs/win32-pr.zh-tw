# <a name="itssbprovisioningpluginnotifysink-methods"></a>ITsSbProvisioningPluginNotifySink 方法

[**ITsSbProvisioningPluginNotifySink**](/windows/desktop/api/sbtsv/nn-sbtsv-itssbprovisioningpluginnotifysink)介面會公開下列方法。

## <a name="in-this-section"></a>本節內容

<dl> <dt>

[**LockVirtualMachine 方法**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-lockvirtualmachine)
</dt> <dd>

通知遠端桌面連線代理人 (RD 連線代理人) 虛擬機器已鎖定。

</dd> <dt>

[**OnJobCancelled 方法**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onjobcancelled)
</dt> <dd>

通知 RD 連線代理人工作已取消。

</dd> <dt>

[**OnJobCompleted 方法**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onjobcompleted)
</dt> <dd>

通知 RD 連線代理人作業已完成。

</dd> <dt>

[**OnJobCreated 方法**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onjobcreated)
</dt> <dd>

通知 RD 連線代理人已建立布建作業。

</dd> <dt>

[**OnVirtualMachineHostStatusChanged 方法**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onvirtualmachinehoststatuschanged)
</dt> <dd>

通知 RD 連線代理人虛擬機器主機的狀態已變更。

</dd> <dt>

[**OnVirtualMachineStatusChanged 方法**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioningpluginnotifysink-onvirtualmachinestatuschanged)
</dt> <dd>

通知 RD 連線代理人虛擬機器的狀態已變更。

</dd> </dl>
