# <a name="itssbtaskpluginnotifysink-methods"></a>ITsSbTaskPluginNotifySink 方法

[**ITsSbTaskPluginNotifySink**](/windows/desktop/api/sbtsv/nn-sbtsv-itssbtaskpluginnotifysink)介面會公開下列方法。

## <a name="in-this-section"></a>本節內容

<dl> <dt>

[**OnDeleteTaskTime 方法**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-ondeletetasktime)
</dt> <dd>

通知遠端桌面連線代理人 (RD 連線代理人) 已從佇列中移除工作。

</dd> <dt>

[**OnReportTasks 方法**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-onreporttasks)
</dt> <dd>

通知 RD 連線代理人新的工作報表。

</dd> <dt>

[**OnSetTaskTime 方法**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-onsettasktime)
</dt> <dd>

通知 RD 連線代理人已排程工作。

</dd> <dt>

[**OnUpdateTaskStatus 方法**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-onupdatetaskstatus)
</dt> <dd>

通知 RD 連線代理人工作的狀態已變更。

</dd> </dl>
