# <a name="requirements-for-remote-desktop-web-connection"></a>遠端桌面網頁連線的需求

遠端桌面網頁連線需要設定兩個系統，如下所示：

-   執行 Windows 和 IIS 4.0 或更新版本的主機伺服器。
-   可以下載並執行遠端桌面 ActiveX 控制項的用戶端。 任何執行 Microsoft Internet Explorer 4.01 (最好是 Service Pack 2) 或更新版本的用戶端，都可以執行遠端桌面 ActiveX 控制項。

安裝軟體的使用者必須有足夠的許可權才能安裝軟體。 例如，在 Windows XP 上，使用者必須是 Power Users 或 Administrators 群組的成員。
