# <a name="imsrdpdrivev2-interface"></a>IMsRdpDriveV2 介面

包含磁片磁碟機物件的相關資訊。 這是 [**IMsRdpDrive**](imsrdpdrive.md) 介面的增強功能。

## <a name="members"></a>成員

**IMsRdpDriveV2** 介面繼承自 [**IMsRdpDrive**](imsrdpdrive.md)。 **IMsRdpDriveV2** 也有下列類型的成員：

-   [屬性](#properties)

### <a name="properties"></a>屬性

**IMsRdpDriveV2** 介面具有這些屬性。



| 屬性                                                              | 存取類型          | Description                                                |
|:----------------------------------------------------------------------|:---------------------|:-----------------------------------------------------------|
| [**DriveLetterIndex**](imsrdpdrivev2-driveletterindex.md)<br/> | 唯讀<br/> | 包含磁片磁碟機的字母索引。<br/> |
