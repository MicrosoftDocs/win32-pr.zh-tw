# <a name="remote-desktop-virtualization-api-reference"></a>遠端桌面虛擬化 API 參考

遠端桌面虛擬化 API 支援介面和結構，可讓您用來建立自訂外掛程式，以覆寫遠端桌面連線代理人 (RD 連線代理人) 的標準重新導向邏輯。 自訂外掛程式可將連接重新導向至遠端桌面工作階段主機 (RD 工作階段主機) 伺服器，以及虛擬機器、虛擬桌面、blade 伺服器和實體桌面。
