# <a name="remote-desktop-services-web-access-control-interfaces"></a>遠端桌面服務 Web 存取控制項介面

遠端桌面服務 Web 存取控制項支援在 RemoteApp 和桌面連線中發佈使用者專屬的資源。 此控制項是遠端桌面連線用戶端 (MsTscAx.dll) 的包裝函式，以及 RemoteApp 和桌面連線執行時間 proxy (Tswbprxy.exe) 。

## <a name="in-this-section"></a>本節內容

<dl> <dt>

[**IMsRdpClientShell2**](imsrdpclientshell2.md)
</dt> <dd>

公開從遠端桌面 Web 存取 (RD Web 存取) 或從其他入口網站啟動遠端桌面連線用戶端的屬性。

</dd> <dt>

[**IMsRdpWorkspace**](imsrdpworkspace.md)
</dt> <dd>

公開管理 RemoteApp 和桌面連線認證和連接的方法。

</dd> <dt>

[**IMsRdpWorkspace2**](imsrdpworkspace2.md)
</dt> <dd>

公開將 RemoteApp 和桌面連線認證與連接產生關聯的方法。

</dd> </dl>
