# <a name="remote-desktop-services-reference"></a>遠端桌面服務參考

遠端桌面服務參考包含下列各節。

## <a name="in-this-section"></a>本節內容

<dl> <dt>

[遠端桌面服務使用者設定的 ADSI 擴充功能的參考](reference-for-the-adsi-extension-for-terminal-services-user-configuration.md)
</dt> <dd>

您可以用來檢查和設定遠端桌面服務使用者屬性的屬性方法。

</dd> <dt>

[遠端桌面閘道 API 參考](remote-desktop-gateway-api-reference.md)
</dt> <dd>

會將取代遠端桌面閘道 (RD 閘道) 之預設驗證和授權機制的外掛程式。

</dd> <dt>

[遠端桌面服務 API 參考](terminal-services-api-reference.md)
</dt> <dd>

您可以用來執行管理工作、管理使用者設定資訊、增強用戶端/伺服器應用程式，以及在遠端桌面服務環境中使用虛擬通道的資料類型、列舉類型、函式、訊息和結構。

</dd> <dt>

[遠端桌面服務 AudioEndpoint API 參考](terminal-services-audioendpoint-api-reference.md)
</dt> <dd>

支援用於音訊端點註冊和資料傳輸的介面。

</dd> <dt>

[遠端桌面服務 Web 存取控制項介面](remote-desktop-services-web-access-control-interfaces.md)
</dt> <dd>

支援在 RemoteApp 和桌面連線中發佈使用者專屬資源。 此控制項是遠端桌面連線用戶端 (MsTscAx.dll) 的包裝函式，以及 RemoteApp 和桌面連線執行時間 proxy (Tswbprxy.exe) 。

</dd> <dt>

[遠端桌面虛擬化 API 參考](terminal-services-virtualization-api-reference.md)
</dt> <dd>

支援可用來建立自訂外掛程式的介面和結構，這些外掛程式會覆寫遠端桌面連線代理人 (RD 連線代理人) 的標準重新導向邏輯。

</dd> <dt>

[RemoteApp 和桌面連線管理服務 API 參考](centralized-publishing-api-reference.md)
</dt> <dd>

支援介面和結構，以提供可供 RemoteApp 和桌面連線使用者使用的資源相關資訊。

</dd> <dt>

[RemoteApp 和桌面連線執行時間介面](remoteapp-and-desktop-connection-runtime-interfaces.md)
</dt> <dd>

支援允許在 RemoteApp 和桌面連線中開發自訂用戶端的介面。

</dd> </dl>
