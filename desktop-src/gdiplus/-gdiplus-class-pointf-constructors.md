# <a name="pointfpointf-constructors"></a>PointF. PointF 函數

本主題列出 [**PointF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) 類別的函式。 如需完整的類別清單，請參閱 **PointF 類別**。

### <a name="overload-list"></a>多載清單



| 建構函式                                                    | 描述                                                                                                                                                                       |
|:---------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**PointF (SizeF&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-pointf-pointf(inconstsizef_))   | 使用 [**SizeF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-sizef)物件來指定 **X** 和 **Y** 資料成員，以建立 [**PointF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf)物件。<br/> |
| [**PointF (PointF&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-pointf-pointf(inconstpointf_)) | 建立新的 [**PointF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) 物件，並從另一個 **PointF** 物件複製資料。<br/>                                                 |
| [**PointF (REAL，REAL)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-pointf-pointf(inreal_inreal)) | 使用兩個實數來建立 [**PointF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) 物件，以指定 **X** 和 **Y** 資料成員。<br/>                                     |
| [**PointF ()**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-pointf-pointf)              | 建立 [**PointF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) 物件，並將 **X** 和 **Y** 資料成員初始化為零。 這是預設建構函式。<br/>              |
