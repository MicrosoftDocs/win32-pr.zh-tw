# <a name="about-gdi"></a>關於 GDI+

Windows GDI+ 是 Windows XP 作業系統或 Windows Server 2003 作業系統的一部分，提供二維向量圖形、影像處理和印刷樣式。 GDI+ 透過新增功能以及將現有的功能優化，以 Windows 圖形裝置介面 (GDI)  (舊版 Windows) 中所包含的圖形裝置介面來改善。
