# <a name="regioncomplement-methods"></a>Region。補數方法

本主題列出 [**Region**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-region) 類別的補充方法。 如需 **region** 類別之方法的完整清單，請參閱 [區域方法](-gdiplus-class-region-methods.md)。

### <a name="overload-list"></a>多載清單



| 方法                                                                       | 描述                                                                                                                                                                                                    |
|:-----------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**補充 (Rect&)**](/previous-versions//ms534918(v=vs.85))     | [**Region：：補數**](/previous-versions//ms534918(v=vs.85))方法會將此區域更新為指定矩形內部的部分，而不會與此區域相交。<br/>  |
| [**補充 (RectF&)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-complement(inconstrectf_))   | [**Region：：補數**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-complement(inconstrectf_))方法會將此區域更新為指定矩形內部的部分，而不會與此區域相交。<br/> |
| [**(區域補充 \*)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-complement(inconstregion))     | [**Region：：補數**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-complement(inconstregion))方法會將此區域更新為另一個區域中未與此區域交集的部分。<br/>                         |
| [**補充 (GraphicsPath \*)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-complement(inconstgraphicspath)) | [**Region：：補數**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-complement(inconstgraphicspath))方法會將此區域更新為未與此區域交集之指定路徑內部的部分。<br/>            |
