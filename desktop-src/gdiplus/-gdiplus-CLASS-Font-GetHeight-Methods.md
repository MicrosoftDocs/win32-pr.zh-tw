# <a name="fontgetheight-methods"></a>GetHeight 方法

本主題列出 [**字型**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-font) 類別的 GetHeight 方法。 如需 **字型** 類別之方法的完整清單，請參閱 [字型方法](-gdiplus-class-font-methods.md)。

### <a name="overload-list"></a>多載清單



| 方法                                                                   | 描述                                                                                                                                                                                                                                                                                                                                                                                                                              |
|:-------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetHeight (REAL)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-font-getheight(inreal))            | [**Font-size：： GetHeight**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-font-getheight(inreal))方法會取得這個字型的行距（以圖元為單位）。 行距是兩連續文字行基準線間的垂直距離。 因此，行間距包括行之間的空格以及字元本身的高度。<br/>                                                                                      |
| [**GetHeight (圖形 \*)**](/previous-versions//ms536201(v=vs.85)) | [**Font-size：： GetHeight**](/previous-versions//ms536201(v=vs.85))方法會取得指定之 [**圖形**](/windows/win32/api/gdiplusgraphics/nl-gdiplusgraphics-graphics)物件目前單位中這個字型的行距。 行距是兩連續文字行基準線間的垂直距離。 因此，行間距包括行之間的空格以及字元本身的高度。<br/> |
