# <a name="services-services"></a>服務 (服務) 

*服務應用程式* 會符合服務控制管理員 (SCM) 的介面規則。 您可以在系統開機、使用者透過服務控制台小程式或使用 [服務功能](service-functions.md)的應用程式時，自動啟動它。 即使沒有任何使用者登入系統，也可以執行服務。

*驅動程式服務* 符合設備磁碟機的通訊協定。 它類似于服務應用程式，但不會與 SCM 互動。 為了簡單起見，本「 *服務* 」一詞是指本總覽中的 *服務應用程式* 。

觸發程式現在可以用來控制服務啟動。 如需詳細資訊，請參閱 [服務](service-configuration.md)設定。
