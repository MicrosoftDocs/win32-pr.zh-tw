# <a name="swbemservicessecurity_-property"></a>SWbemServices 安全性 \_ 屬性

[**SWbemServices**](swbemservices.md)物件的 **security \_** 屬性是用來取得或設定 **SWbemServices** 物件的安全性設定。 這個屬性是 [**SWbemSecurity**](swbemsecurity.md) 物件。

如需此語法的說明，請參閱 [腳本 API 的檔慣例](document-conventions-for-the-scripting-api.md)。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
SWbemServices.Security_ As Object
```
