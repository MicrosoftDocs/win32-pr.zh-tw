# <a name="swbemobjectsetsecurity_-property"></a>Swbemobjectset 搭配使用安全性 \_ 屬性

[**Swbemobjectset 搭配使用**](swbemobjectset.md)物件的 **security \_** 屬性是用來取得或設定 **swbemobjectset 搭配使用** 物件的安全性設定。 這個屬性是 [**SWbemSecurity**](swbemsecurity.md) 物件。

> [!Note]  
> 將 [**swbemobjectset 搭配使用**](swbemobjectset.md)物件的 [[**安全性 \_**](swbemobject-security-.md) ] 屬性設定為 **Null** ，會將無限存取權授與所有人。 如需無限制存取含意的詳細資訊，請參閱 [**SWbemSecurity**](swbemsecurity.md)。

 

如需此語法的說明，請參閱 [腳本 API 的檔慣例](document-conventions-for-the-scripting-api.md)。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
SWbemObjectSet.Security_ As Object
```
