# <a name="using-the-proper-active-directory-syntax"></a>使用適當的 Active Directory 語法

您可以在透過 WMI 呼叫 [) 提供者](/previous-versions/windows/desktop/dsprov/active-directory-provider) 時，提供輕量目錄存取通訊協定 (LDAP) 命名空間和 Active Directory 服務介面 (ADSI Active Directory 路徑。

下列主題描述 LDAP 命名空間和 ADSI 路徑：

-   [描述 LDAP 命名空間](describing-the-ldap-namespace.md)

    命名空間路徑描述遠端命名空間的位置。 使用命名空間路徑從遠端連線到 Active Directory 命名空間。

-   [描述 ADSI 路徑](describing-the-adsi-path.md)

    ADSI 路徑會使用 ADSI API 來描述物件的位置。 在提供者傳遞至 Active Directory 的參數中使用 ADSI 路徑。

> [!Note]  
> 如需有關在特定作業系統上支援和安裝此元件的詳細資訊，請參閱 [WMI 元件的作業系統可用性](operating-system-availability-of-wmi-components.md)。
