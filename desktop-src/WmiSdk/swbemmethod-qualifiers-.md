# <a name="swbemmethodqualifiers_-property"></a>SWbemMethod 的限定詞 \_ 屬性

[**SWbemMethod**](swbemmethod.md)物件的 **限定詞 \_** 屬性會傳回 [**SWbemQualifierSet**](swbemqualifierset.md)物件，此物件為這個方法的限定詞集合。 這是唯讀的屬性。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
SWbemMethod.Qualifiers_ As Object
```
