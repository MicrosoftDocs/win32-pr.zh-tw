# <a name="swbemnamedvalue-object"></a>SWbemNamedValue 物件

**SWbemNamedValue** 物件代表屬於 [**SWbemNamedValueSet**](swbemnamedvalueset.md)集合物件的單一命名值。 VBScript [**CreateObject**](creating-an-object-using-vbscript.md) 呼叫無法建立這個物件。

## <a name="members"></a>成員

**SWbemNamedValue** 物件具有下列類型的成員：

-   [屬性](#properties)

### <a name="properties"></a>屬性

**SWbemNamedValue** 物件具有這些屬性。



| 屬性                                          | 存取類型           | Description                                                                                    |
|:--------------------------------------------------|:----------------------|:-----------------------------------------------------------------------------------------------|
| [**Name**](swbemnamedvalue-name.md)<br/>   | 唯讀<br/>  | **SWbemNamedValue** 專案的名稱。<br/>                                               |
| [**值**](swbemnamedvalue-value.md)<br/> | 讀取/寫入<br/> | **SWbemNamedValue** 專案的值。 這是此物件的預設屬性。<br/> |
