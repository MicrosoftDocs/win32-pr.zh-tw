# <a name="wmi-command-line-tools"></a>WMI 命令列工具

WMI 有許多由系統管理員和 WMI 提供者寫入器使用的命令列工具。



| 區段                      | 描述                                                                                                                                                         |
|------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**mofcomp.exe**](mofcomp.md)   | 將受控物件格式 (MOF) 程式碼編譯成儲存在 WMI 儲存機制中的二進位格式。 針對 WMI 提供者建立或修改 MOF 檔案時使用。<br/> |
| [**smi2smir**](smi2smir.md) | 接受一個 SNMP 資訊模組作為輸入，並接受解析外部參考所需的任何其他模組。<br/>                               |
| [**winmgmt**](winmgmt.md)   | 執行 WMI 服務。 從 Windows Vista 開始，已加入數個新的交換器。<br/>                                                                |
| [**wmiadap**](wmiadap.md)   | 執行 ADAP 命令列工具，以剖析效能程式庫並將其傳輸至 WMI 存放庫。<br/>                                                       |
| [**wmic**](wmic.md)         | 在命令提示字元中執行 WMI 作業。<br/>                                                                                                               |
