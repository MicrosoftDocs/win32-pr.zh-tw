# <a name="qualifier-flavors"></a>限定詞類別

限定詞類別提供有關辨識符號的詳細資訊，例如衍生類別或實例是否可以覆寫辨識符號的原始值。

您可以使用下列語法，將限定詞類別新增至 MOF 檔案的頂端，使其可在整個定義中套用。 例如：

``` syntax
Qualifier Description : ToSubClass Amended;
```

**ToSubClass** 和 **修訂** 的類別適用于 MOF 中的所有 **描述** 限定詞。

下表列出辨識符號的類別。



| 限定詞類別    | 意義                                                                                                                                                                                                                                         |
|---------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **修訂**         | 基本類別定義中不需要限定詞，且可以移動到要進行當地語系化的增修條款。                                                                                                                                  |
| **DisableOverride** | 無法在衍生類別或執行個體中覆寫限定詞。 請注意，預設可覆寫傳播的限定詞。                                                                                                      |
| **EnableOverride**  | 傳播至衍生類別或實例時，限定詞的值可以覆寫。 設定 **EnableOverride** 是選擇性的，因為能夠覆寫辨識符號值是傳播限定詞的預設功能。 |
| **NotToInstance**   | 限定詞不會傳播至類別實例。                                                                                                                                                                                             |
| **NotToSubClass**   | 限定詞不會傳播到衍生類別。                                                                                                                                                                                             |
| **Restricted**      | 限定詞不會傳播至實例或衍生類別。                                                                                                                                                                                |
| **ToInstance**      | 限定詞會傳播到執行個體。                                                                                                                                                                                                       |
| **ToSubClass**      | 限定詞會傳播到衍生類別。 此類別僅適用于為類別定義的限定詞，無法附加至描述類別實例的辨識符號。                                                           |
