# <a name="preprocessor-commands"></a>預處理器命令

預處理器命令是放置在 MOF 檔案中的一行程式碼，可控制 MOF 編譯器編譯器代碼的方式。 編譯 MOF 檔案時使用兩個預處理器命令：

-   [\#包括](-include.md)
-   [\#pragma](-pragma.md)
