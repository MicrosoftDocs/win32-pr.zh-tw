# <a name="retrieving-amended-classes"></a>正在抓取修改過的類別

您可以藉由指定地區設定並設定旗標，指出您想要當地語系化的資訊，來抓取和修改修改過的類別。 用來取得當地語系化類別定義的程式，取決於您連接至 WMI 的方式。 如需詳細資訊，請參閱使用適用于 [wmi 的腳本 API 來抓取修改過的類別](retrieving-amended-classes-using-the-scripting-api-for-wmi.md) 以及 [使用適用于 WMI 的 COM API 來抓取修改](retrieving-amended-classes-using-the-com-api-for-wmi.md)過的類別。
