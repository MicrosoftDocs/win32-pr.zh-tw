# <a name="datetime"></a>DATETIME

DATETIME 資料類型是固定長度的 VT \_ BSTR 字串，表示特定的日期和時間或間隔。 如需日期時間的詳細資訊，請參閱 [日期和時間格式](date-and-time-format.md)。 如需 DATETIME 做為間隔的詳細資訊，請參閱 [間隔格式](interval-format.md)。
