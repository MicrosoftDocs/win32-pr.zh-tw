# <a name="swbemmethodsetcount-property"></a>SWbemMethodSet Count 屬性

[**SWbemMethodSet**](swbemmethodset.md)物件的 **Count** 屬性會定義集合中的專案數。 這是唯讀的屬性。

如需此語法的說明，請參閱 [腳本 API 的檔慣例](document-conventions-for-the-scripting-api.md)。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
SWbemMethodSet.Count As Integer
```
