描述 WMI SNMP 提供者錯誤1101到1110。

[嚴重錯誤1101](#fatal-error-1101)

## <a name="fatal-error-1101"></a>嚴重錯誤1101

<dl> <dt>

<span id="_1101__Fatal_____fileName__line____The_length_of_the_range_is_too_long_for_the_compiler__Does_not_fit_into_4_bytes__"></span><span id="_1101__fatal_____filename__line____the_length_of_the_range_is_too_long_for_the_compiler__does_not_fit_into_4_bytes__"></span><span id="_1101__FATAL_____FILENAME__LINE____THE_LENGTH_OF_THE_RANGE_IS_TOO_LONG_FOR_THE_COMPILER__DOES_NOT_FIT_INTO_4_BYTES__"></span>**<1101，嚴重>：「 <fileName><行 \#>：範圍長度太長，編譯器 (無法容納4個位元組) 」**
</dt> <dd>

範圍或大小規格中的模組語意錯誤，特定于 SNMPv1 或 SNMPv2C。 一組範圍的大小必須小於2³²。 一組範圍的大小會定義為最大值與集合中最小值之間的差異。

</dd> </dl>
