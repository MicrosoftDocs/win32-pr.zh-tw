# <a name="swbemprivilegesetcount-property"></a>SWbemPrivilegeSet Count 屬性

[**SWbemPrivilege**](swbemprivilege.md)物件的 **Count** 屬性包含 [**SWbemPrivilegeSet**](swbemprivilegeset.md)集合中的專案數。 這是唯讀的屬性。

如需此語法的說明，請參閱 [腳本 API 的檔慣例](document-conventions-for-the-scripting-api.md)。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
SWbemPrivilegeSet.Count As Integer
```
