# <a name="msft-classes"></a>MSFT 類別

MSFT 類別是 [*通用訊息模型*](gloss-c.md) (CIM) 的 Windows 執行所特有的。

物件可以分成數個主要類別：

-   [原則提供者類別](policy-provider-classes.md)
-   [WMI 疑難排解類別](wmi-troubleshooting-classes.md)

下列 MSFT 類別是在 System. mof 中定義，但不是由 WMI 所執行。

-   MSFT \_ WmiCoreEvent
-   MSFT \_ WmiCoreLogoffEvent
-   MSFT \_ WmiCoreLogonEvent
-   MSFT \_ WmiCoreObject
-   MSFT \_ WmiCoreStatus
-   MSFT \_ WmiCoreUser
-   MSFT \_ WmiTaskFailure
