# <a name="modifying-an-instance-property"></a>修改實例屬性

建立或尋找類別的實例之後，您可能會想要修改實例。 例如，您可能想要變更儲存在 [**Win32 \_**](/windows/desktop/CIMWin32Prov/win32-computersystem) 系統系統的本機實例中之系統名稱的名稱。 更新實例最常見的方式是更新整個實例。 如需詳細資訊，請參閱 [更新整個實例](updating-an-entire-instance.md)。 第二個選項是只更新實例的一部分。 如需詳細資訊，請參閱 [更新實例的一部分](updating-part-of-an-instance.md)。

如需使用類別的詳細資訊，請參閱 [操作類別和實例資訊](manipulating-class-and-instance-information.md)。
