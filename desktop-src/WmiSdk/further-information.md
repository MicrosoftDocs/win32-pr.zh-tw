WMI 工具組括：

-   WMI CIM Studio

    CIM Studio 工具可讓您使用 CIM 存放庫中的 WMI 類別和命名空間。 它也是 WMI 提供者開發的重要工具，因為它有助於在 CIM 存放庫中建立初始的 WMI 類別。

    WMI CIM Studio 提供下列功能：

    -   連線到所選的系統，並流覽任何可用命名空間中的 CIM 存放庫。
    -   依名稱、描述或屬性名稱搜尋類別。
    -   檢查與指定類別相關的屬性、方法和關聯。
    -   查看指定類別的可用實例。
    -   以 WQL 語言執行查詢。
    -   根據選取的類別產生 MOF 檔案。
    -   編譯要在 CIM 存放庫中載入的 MOF 檔案。

-   WMI 物件瀏覽器

    [物件瀏覽器] 可讓您查看電腦上存在的 WMI 物件。 階層的根目錄是代表您電腦系統的 [**Win32 \_**](/windows/desktop/CIMWin32Prov/win32-computersystem) 系統類型實例。 您可以使用 [物件瀏覽器] 來編輯屬性值和限定詞，以及執行方法。

-   WMI 事件註冊

    事件註冊工具可讓您針對指定的命名空間建立、顯示和修改事件取用者、篩選和計時器。 這是適用于開發人員和系統管理員的工具。

-   WMI 事件檢視器

    您可以使用事件檢視器來顯示您在 WMI 事件註冊中所設定的 WMI 事件出現次數。

您可以在中下載 PowerShell RC1 和檔 [https://www.microsoft.com/technet/scriptcenter/topics/msh/download.mspx](https://www.microsoft.com/technet/scriptcenter/topics/msh/download.mspx) 。
