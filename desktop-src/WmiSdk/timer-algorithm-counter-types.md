# <a name="timer-algorithm-counter-types"></a>計時器演算法計數器類型

計時器演算法計數器類型是根據取樣期間增加的效能物件使用量而定。 計數器資料是物件的總活動量量值，直到取樣發生的時間為止。 這兩個範例之間的差異指出物件在取樣期間內的使用時間總計。

除以取樣期間會導致物件在一段時間內處於作用中的時間比例。 除以內部輪詢間隔的數目，會決定輪詢範例之間的平均使用方式。

例如， [**Win32 \_ PerfRawData \_ PerfDisk \_ PhysicalDisk**](/previous-versions//aa394308(v=vs.85))類別中的 **AvgDiskSecPerRead** 屬性會使用效能 **\_ 平均 \_ 計時器** countertype。 它會計算從磁片讀取資料的平均時間（以秒為單位），而且需要基底屬性 **AvgDiskSecPerRead \_ 基底**。 與 **性能 \_ 計數器 \_ 計時器** 不同的是，平均計時器基底代表累積的作業數目，而計數器資料是運行時間值，這表示當除以時間底數時，它會產生所有作業的總時間（以秒為單位）。
