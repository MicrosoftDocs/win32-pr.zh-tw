# <a name="operating-system-availability-of-wmi-components"></a>WMI 元件的作業系統可用性

預設會在所有 Windows 桌面和伺服器平臺上安裝 WMI。 不過，根據作業系統版本和設定而定，某些 WMI 提供者可能會安裝或可能無法安裝。 例如，SNMP 提供者預設不會啟用，而且64位作業系統上預設不會安裝 Windows Installer 提供者 (MSI 提供者) 。 如需特定提供者可用性的詳細資訊，請參閱 [WMI 提供者](/windows/desktop/SrvNodes/wmi-mi-omi-providers)。
