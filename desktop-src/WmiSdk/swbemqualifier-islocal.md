# <a name="swbemqualifierislocal-property"></a>SWbemQualifier. IsLocal 屬性

[**SWbemQualifier**](swbemqualifier.md)物件的 **IsLocal** 屬性是布林值，指出此限定詞是否為本機。 這是唯讀的屬性。

如需此語法的說明，請參閱 [腳本 API 的檔慣例](document-conventions-for-the-scripting-api.md)。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
SWbemQualifier.IsLocal As Boolean
```
