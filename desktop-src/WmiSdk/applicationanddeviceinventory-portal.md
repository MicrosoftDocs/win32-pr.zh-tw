# <a name="application-inventory-classes"></a>應用程式清查類別

## <a name="purpose"></a>目的

應用程式清查的 WMI 類別可讓您探索已安裝的 Win32 應用程式，並 Windows 將應用程式儲存在 Windows 系統上。

## <a name="in-this-section"></a>本節內容



| 主題                                                                                  | 說明                                                                                                                                                                                                                                                    |
|----------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Win32 \_ InstalledWin32Program**](win32-installedwin32program.md)<br/>         | [**Win32 \_ InstalledWin32Program**](win32-installedwin32program.md)類別代表已安裝的 win32 應用程式。<br/>                                                                                                                            |
| [**Win32 \_ InstalledStoreProgram**](win32-installedstoreprogram.md)<br/>         | [**Win32 \_ InstalledStoreProgram**](win32-installedstoreprogram.md)類別代表安裝的 Windows 存放區應用程式。<br/>                                                                                                                    |
| [**Win32 \_ InstalledProgramFramework**](win32-installedprogramframework.md)<br/> | [**Win32 \_ InstalledProgramFramework**](win32-installedprogramframework.md)類別代表 [**win32 \_ InstalledWin32Program**](win32-installedwin32program.md)實例編譯或在執行時間使用的技術架構。<br/> |



 

## <a name="developer-audience"></a>開發人員對象

應用程式清查的 WMI 類別是設計用於 Enterprise 管理解決方案的開發人員，以及需要在系統上產生應用程式清查的本機或遠端系統管理工具的開發人員使用。
