# <a name="swbemmethodname-property"></a>SWbemMethod.Name 屬性

[**SWbemMethod**](swbemmethod.md)物件的 **name** 屬性（property）是描述這個方法名稱的字串。 這是唯讀的屬性。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
SWbemMethod.Name As String
```
