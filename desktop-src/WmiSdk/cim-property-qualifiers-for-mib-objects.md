# <a name="cim-property-qualifiers-for-mib-objects"></a>MIB 物件的 CIM 屬性限定詞

將 MIB 物件定義對應至 CIM 類別定義時，SNMP 提供者會使用下列 CIM 屬性限定詞。 SNMP 提供者必須要有必要的限定詞，才能完全解析 MIB 物件。 無法定義強制辨識符號會傳回錯誤。 指定不合法的辨識符號值也會傳回錯誤。

> [!Note]  
> 如需安裝提供者的詳細資訊，請參閱 [設定 WMI SNMP 環境](setting-up-the-wmi-snmp-environment.md)。

 



| Qualifier                          | 描述                                                                                                                                                                                            |
|------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Bits**                           | **字串** 如果 **文字 \_ 慣例** 辨識符號不等於位，則為強制性。<br/> 位之子類型定義的列舉值。<br/>                                        |
| **Cimtype**                        | **字串** 強制性<br/> 格式化基礎 CIM 通訊協定值的 CIM 文字表示。<br/>                                                                                    |
| **Defval**                         | **字串** 選<br/> 物件的預設值。<br/>                                                                                                                                       |
| **說明**                    | **字串** 選<br/> 物件的描述。<br/>                                                                                                                                    |
| **顯示 \_ 提示**                  | **字串** 選<br/> 物件的資料應該對使用者顯示的方式。<br/>                                                                                                    |
| **編碼方式**                       | **字串** 選<br/> 編碼 SNMPv1 和 SNMPv2C 通訊協定框架時使用的 SNMP 類型。<br/>                                                                                              |
| **列舉型別**                    | **字串** 如果 **文字 \_ 慣例** 辨識符號不等於 ENUMERATEDINTEGER，則為必要。<br/> 列舉整數子類型定義的列舉值。<br/>                  |
| **固定 \_ 長度**                  | **uint32** 選<br/> 固定長度值。<br/>                                                                                                                                           |
| [**答案**](standard-qualifiers.md) | **Bool** 選<br/> 類別定義的索引鍵屬性。<br/>                                                                                                                             |
| **按鍵 \_ 順序**                     | **uint32** 除非已指定索引 [**鍵**](standard-qualifiers.md) ，否則為選擇性。<br/> 類別定義中索引鍵屬性的順序。<br/>                                                   |
| **名稱**                           | **字串** 強制性<br/> 屬性的隱含名稱。 請注意，不會明確定義此限定詞。<br/>                                                                           |
| **物件 \_ 識別碼**             | **字串** 強制性<br/> 物件的 MIB 物件識別碼。<br/>                                                                                                                              |
| **物件 \_ 語法**                 | **字串** 選<br/> 物件的命名型別定義。<br/>                                                                                                                               |
| **讀取**                           | **Bool** 選擇性 (至少必須指定其中一個 **讀取** 或 **寫入**) <br/> 授與物件的讀取權限。<br/>                                                                     |
| **參考**                      | **字串** 選<br/> 指的是包含物件之詳細資訊的另一份檔。<br/>                                                                                   |
| **狀態**                         | **字串** 選<br/> 指出是否必須支援此物件。<br/>                                                                                                               |
| **文字 \_ 慣例**            | **字串** 強制性<br/> MIB [物件類型](object-type-macro.md) 宏語法子句的文字標記法。<br/>                                                           |
| **單位**                          | **字串** 選<br/> 物件代表的精確定義。<br/>                                                                                                             |
| **變數 \_ 長度**               | **字串** 選<br/> 與物件類型定義相關聯的最小值、最大值和固定長度值。<br/>                                                                       |
| **變數 \_ 值**                | **字串** 選<br/> 與物件的型別定義相關聯的範圍和固定值。<br/>                                                                                         |
| **虛擬機器 \_ 碼**                   | **Bool** 選<br/> 指出屬性的值應該以與類別定義中所有可存取的 MIB 物件相關聯之實例資訊的超集合為基礎。<br/> |
| **寫入**                          | **Bool** 選擇性 (至少必須指定其中一個 **讀取** 或 **寫入**) <br/> 授與物件的寫入權限。<br/>                                                                    |
