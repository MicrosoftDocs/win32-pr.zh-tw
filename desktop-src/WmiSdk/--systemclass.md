# <a name="__systemclass-class"></a>\_\_SystemClass 類別

**\_ \_ SystemClass** 抽象系統類別是用來衍生大部分系統類別的基類。 無法建立此類別的實例。

下列語法已從受管理物件格式 (MOF) 程式碼加以簡化，並包含所有繼承的屬性。 屬性會依字母順序列出，而不是依 MOF 順序列出。

## <a name="syntax"></a>語法

``` syntax
[abstract]
class __SystemClass
{
};
```

## <a name="members"></a>成員

**\_ \_ SystemClass** 類別不會定義任何成員。
