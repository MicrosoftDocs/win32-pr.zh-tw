# <a name="swbempropertyisarray-property"></a>Swbempropertyset. IsArray 屬性

[**Swbempropertyset**](swbemproperty.md)物件的 **IsArray** 屬性是一個布林值，可用來判斷這個屬性是否有陣列類型。 這是唯讀的屬性。

如需此語法的說明，請參閱 [腳本 API 的檔慣例](document-conventions-for-the-scripting-api.md)。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
SWbemProperty.IsArray As Boolean
```
