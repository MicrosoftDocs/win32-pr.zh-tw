# <a name="mapping-active-directory-to-wmi"></a>將 Active Directory 對應至 WMI

目錄服務提供者會將 Active Directory 中的類別和實例鏡像至 WMI 輕量型目錄存取協定 (LDAP) 命名空間。 由於這兩種環境的基本差異，您不能只是重新命名 Active Directory 的類別和實例，希望能夠正確地存取 WMI 中的類別和實例。 目錄服務提供者會改為遵循命名規則，以保留存在於 Active Directory 類別和實例之間的關聯性。

下列主題提供對應 Active Directory 類別和實例的相關資訊：

-   [對應 Active Directory 類別](mapping-active-directory-classes.md)
-   [Active Directory 實例的對應](mapping-active-directory-instances.md)

> [!Note]  
> 如需有關在特定作業系統上支援和安裝此元件的詳細資訊，請參閱 [WMI 元件的作業系統可用性](operating-system-availability-of-wmi-components.md)。
