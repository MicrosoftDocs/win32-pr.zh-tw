# <a name="__indicationrelated-class"></a>\_\_IndicationRelated 類別

**\_ \_ IndicationRelated** 抽象系統類別可作為所有事件相關類別的父類別。 類別不包含任何屬性。

下列語法已從受管理物件格式 (MOF) 程式碼加以簡化，並包含所有繼承的屬性。 屬性會依字母順序列出，而不是依 MOF 順序列出。

## <a name="syntax"></a>語法

``` syntax
[abstract]
class __IndicationRelated : __SystemClass
{
};
```

## <a name="members"></a>成員

**\_ \_ IndicationRelated** 類別不會定義任何成員。

## <a name="remarks"></a>備註

**\_ \_ IndicationRelated** 類別衍生自 [**\_ \_ SystemClass**](--systemclass.md)，但沒有屬性。
