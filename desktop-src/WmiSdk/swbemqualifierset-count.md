# <a name="swbemqualifiersetcount-property"></a>SWbemQualifierSet Count 屬性

[**SWbemQualifierSet**](swbemqualifierset.md)物件的 **Count** 屬性包含 **SWbemQualifierSet** 集合中的專案數。 這是唯讀的屬性。

如需此語法的說明，請參閱 [腳本 API 的檔慣例](document-conventions-for-the-scripting-api.md)。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
SWbemQualifierSet.Count As Integer
```
