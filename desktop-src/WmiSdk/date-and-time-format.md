# <a name="date-and-time-format"></a>日期和時間格式

WMI 會使用分散式管理工作強制 ([DMTF.org](https://www.dmtf.org/)) 通用訊息模型 (CIM) 規格所定義的日期和時間格式。

**CIM \_ datetime** 格式是在 MICROSOFT [受控物件格式 (Mof)](managed-object-format--mof-.md)的 [日期時間](datetime.md)MOF 資料類型中執行。 日期和時間格式也可以表示時間間隔。

如需 WMI 格式的詳細資訊，請參閱 [CIM \_ DATETIME](cim-datetime.md) 和 [Interval 格式](interval-format.md)。

如需 WMI DATETIME 格式轉換的詳細資訊，請參閱 TechNet ScriptCenter 上的 [**SWbemDateTime**](swbemdatetime.md) 和文章，以瞭解 [ (喔的時間，以及日期的)](https://www.microsoft.com/technet/technetmag/issues/2006/07/ScriptingGuy/default.aspx)。
