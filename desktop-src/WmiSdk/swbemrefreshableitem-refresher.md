# <a name="swbemrefreshableitemrefresher-property"></a>SWbemRefreshableItem 複習屬性

**SWbemRefreshableItem** 屬性代表包含 [**SWbemRefreshableItem**](swbemrefreshableitem.md)物件的父 [**SWbemRefresher**](swbemrefresher.md)物件。

如需此語法的說明，請參閱 [腳本 API 的檔慣例](document-conventions-for-the-scripting-api.md)。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
SWbemRefreshableItem.Refresher As Object
```
