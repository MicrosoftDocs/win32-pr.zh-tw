# <a name="swbemlasterrorproperties_-property"></a>SWbemLastError \_ 屬性

[**SWbemLastError**](swbemlasterror.md)物件的 **Properties \_** 屬性會傳回 [**內含**](swbempropertyset.md)物件，該物件為目前類別或實例之 WMI 屬性的集合。 這是唯讀的屬性。

如需此語法的說明，請參閱 [腳本 API 的檔慣例](document-conventions-for-the-scripting-api.md)。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
SWbemLastError.Properties_ As Object
```
