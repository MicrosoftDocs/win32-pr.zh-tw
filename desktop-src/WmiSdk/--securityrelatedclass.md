# <a name="__securityrelatedclass-class"></a>\_\_SecurityRelatedClass 類別

**\_ \_ SecurityRelatedClass** 抽象系統類別可作為所有型別安全性類別的父類別。 此類別的任何實例都不會建立，且所有安全性類別都必須衍生自這個基類。

下列語法已從受管理物件格式 (MOF) 程式碼加以簡化，並包含所有繼承的屬性。 屬性會依字母順序列出，而不是依 MOF 順序列出。

## <a name="syntax"></a>語法

``` syntax
[abstract]
class __SecurityRelatedClass
{
};
```

## <a name="members"></a>成員

**\_ \_ SecurityRelatedClass** 類別不會定義任何成員。
