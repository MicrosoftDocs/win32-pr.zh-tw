# <a name="m-wmi"></a>M (WMI)

<dl> <dt>

<span id="wmi.gloss_managed_object"></span><span id="WMI.GLOSS_MANAGED_OBJECT"></span>**受管理物件**
</dt> <dd>

表示為 WMI 類別實例的硬體或軟體系統元件。

</dd> <dt>

<span id="wmi.gloss_managed_object_format"></span><span id="WMI.GLOSS_MANAGED_OBJECT_FORMAT"></span>**受控物件格式 (MOF)**
</dt> <dd>

以介面定義語言為基礎的語言 (可描述管理資訊的 IDL) 。

</dd> <dt>

<span id="wmi.gloss_management_information_base"></span><span id="WMI.GLOSS_MANAGEMENT_INFORMATION_BASE"></span>**管理資訊基礎 (MIB)**
</dt> <dd>

儲存網路物件表示之集合的資料庫。

</dd> <dt>

<span id="wmi.gloss_metamodel"></span><span id="WMI.GLOSS_METAMODEL"></span>**元**
</dt> <dd>

CIM 元件，描述代表 *managed 物件* 的實體和關聯性。 例如，類別、實例和關聯都會包含在元模型中。

</dd> <dt>

<span id="wmi.gloss_metaschema"></span><span id="WMI.GLOSS_METASCHEMA"></span>**metaschema**
</dt> <dd>

*元模型* 的 [*架構*](gloss-s.md)。

</dd> <dt>

<span id="wmi.gloss_method_provider"></span><span id="WMI.GLOSS_METHOD_PROVIDER"></span>**方法提供者**
</dt> <dd>

提供者，可讓 WMI 存取類別的方法。 另請參閱 [撰寫方法提供者](writing-a-method-provider.md)。

</dd> <dt>

<span id="wmi.gloss_mib"></span><span id="WMI.GLOSS_MIB"></span>**MIB**
</dt> <dd>

請參閱 *管理資訊基底 (MIB)*。

</dd> <dt>

<span id="wmi.gloss_mof"></span><span id="WMI.GLOSS_MOF"></span>**MOF**
</dt> <dd>

請參閱 *受控物件格式 (MOF)*。

</dd> <dt>

<span id="wmi.gloss_mof_file"></span><span id="WMI.GLOSS_MOF_FILE"></span>**MOF 檔案**
</dt> <dd>

文字檔，其中包含使用 *受控物件格式 (MOF)* 語言所指定之類別和實例的定義。

</dd> </dl>
