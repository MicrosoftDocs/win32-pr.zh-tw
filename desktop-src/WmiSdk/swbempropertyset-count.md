# <a name="swbempropertysetcount-property"></a>內含 Count 屬性

使用 [**內含**](swbempropertyset.md)物件的 **Count** Automation 屬性，判斷 **內含** 集合中有多少個專案。 這是唯讀的屬性。

如需此語法的說明，請參閱 [腳本 API 的檔慣例](document-conventions-for-the-scripting-api.md)。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
SWbemPropertySet.Count As Integer
```
