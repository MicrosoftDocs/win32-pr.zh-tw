# <a name="creating-an-instance"></a>建立實例

實例是類別的案例，可描述您企業中的特定物件。 例如，如果您在桌面平臺中有兩個 CD-ROM 光碟機，則可以使用 [**Win32 \_ CDROMDrive**](/windows/desktop/CIMWin32Prov/win32-cdromdrive) 類別的兩個實例來描述這兩個磁片磁碟機。 因為實例代表您企業的真實屬性，您最有可能會建立比類別更多的實例。 建立實例的每個類別都有一或多個索引鍵屬性。 您無法使用超過256的索引鍵來建立實例。

如需建立實例的詳細資訊，請參閱下列主題：

-   [使用 c + + 建立和宣告實例](creating-and-declaring-an-instance-using-c-.md)
-   [使用 MOF 建立實例](creating-an-instance-using-mof.md)
-   [使用 VBScript 建立物件](creating-an-object-using-vbscript.md)
