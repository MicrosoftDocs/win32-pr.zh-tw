# <a name="wmi-providers"></a>WMI 提供者

下表列出作業系統 WMI 提供者。 其他提供者及其類別可能存在於 Microsoft 產品或協力廠商軟體所安裝的電腦上。 如需撰寫專屬提供者的詳細資訊，請參閱 [使用 WMI](using-wmi.md)。

WMI 提供者會將錯誤和警告訊息傳送至事件記錄檔。 如需詳細資訊，請參閱 [**WMI 提供者訊息**](wmi-event-messages.md)。
