# <a name="swbemmethodorigin-property"></a>SWbemMethod 屬性

[**SWbemMethod**](swbemmethod.md)物件的 **原始** 屬性會傳回這個方法引進的類別名稱。 針對具有深度繼承階層的類別，通常需要知道哪些方法已在哪些類別中宣告。 這是唯讀的屬性。

如需此語法的說明，請參閱 [腳本 API 的檔慣例](document-conventions-for-the-scripting-api.md)。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
SWbemMethod.Origin As String
```
