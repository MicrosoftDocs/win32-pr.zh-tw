# <a name="chstringfind-methods"></a>CHString：： Find 方法

\[[**CHString**](chstring.md)類別是 WMI 提供者架構的一部分，現在會被視為最終狀態，而且不會對影響這些程式庫的非安全性相關問題提供進一步的開發、增強功能或更新。 [MI api](/previous-versions/windows/desktop/wmi_v2/windows-management-infrastructure)應該用於所有新的開發。\]

**Find** 方法會在字串中搜尋子字串的第一個相符項。

### <a name="overload-list"></a>多載清單



| 方法                                          | 描述                                             |
|:------------------------------------------------|:--------------------------------------------------------|
| [**尋找 (WCHAR)**](/windows/win32/api/chstring/nf-chstring-chstring-find(lpcwstr))     | 搜尋此字串中的 **WSTR** 。<br/>    |
| [**尋找 (LPCWSTR)**](/windows/win32/api/chstring/nf-chstring-chstring-find(lpcwstr)) | 搜尋此字串中的 **LPCWSTR** 。<br/> |
