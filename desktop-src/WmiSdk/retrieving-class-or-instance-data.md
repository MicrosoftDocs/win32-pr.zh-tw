# <a name="retrieving-wmi-class-or-instance-data"></a>正在抓取 WMI 類別或實例資料

WMI 中的主要資料物件是類別和實例。 下表列出可用來從 WMI 類別或實例中取出資料的不同技術。



| 主題                                                                   | 描述                                                                       |
|-------------------------------------------------------------------------|-----------------------------------------------------------------------------------|
| [正在抓取 WMI 類別](retrieving-a-class.md)                        | 正在抓取物件一般參數的資訊。                    |
| [正在抓取 WMI 實例](retrieving-an-instance.md)                 | 抓取描述企業中唯一個別物件的資訊。 |
| [正在抓取 WMI 實例的一部分](retrieving-part-of-an-instance.md) | 抓取實例的一或多個部分，但不是整個實例。         |
| [存取 WMI 辨識符號](accessing-a-qualifier.md)                  | 存取包含在辨識符號中的詳細資訊。                              |



 

如需詳細資訊，請參閱 [操作類別和實例資訊](manipulating-class-and-instance-information.md)。
