# <a name="swbemobjectpathissingleton-property"></a>SWbemObjectPath. IsSingleton 屬性

[**SWbemObjectPath**](swbemobjectpath.md)物件的 **IsSingleton** 屬性是布林值，指出此路徑是否代表單一實例。 單一實例是永遠不會有一個以上的實例之類別的實例。 這是唯讀的屬性。

如需此語法的說明，請參閱 [腳本 API 的檔慣例](document-conventions-for-the-scripting-api.md)。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
SWbemObjectPath.IsSingleton As Boolean
```
