# <a name="swbemnamedvaluename-property"></a>SWbemNamedValue.Name 屬性

[**SWbemNamedValue**](swbemnamedvalue.md)物件的 **Name** 屬性是取得 **SWbemNamedValue** 專案時所使用的唯一不區分大小寫字串。 這是唯讀的屬性。

如需此語法的說明，請參閱 [腳本 API 的檔慣例](document-conventions-for-the-scripting-api.md)。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
SWbemNamedValue.Name As String
```
