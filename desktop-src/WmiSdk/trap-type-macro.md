# <a name="trap-type-macro"></a>TRAP 類型宏

為保持一致性，會先使用[RFC 1908](https://www.ietf.org/rfc/rfc1908.txt)中定義的對應，將每個「陷阱類型」宏的實例對應至「[通知類型](notification-type-macro.md)」宏的實例。

> [!Note]  
> 如需安裝提供者的詳細資訊，請參閱 [設定 WMI SNMP 環境](setting-up-the-wmi-snmp-environment.md)。
