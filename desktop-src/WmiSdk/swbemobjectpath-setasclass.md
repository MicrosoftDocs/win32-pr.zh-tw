# <a name="swbemobjectpathsetasclass-property"></a>SWbemObjectPath. SetAsClass 屬性

[**SWbemObjectPath**](swbemobjectpath.md)物件的 **SetAsClass** 方法會強制路徑定址 WMI 類別。

此屬性是唯寫的。

## <a name="syntax"></a>Syntax


```VB
SWbemObjectPath.SetAsClass
```



## <a name="property-value"></a>屬性值

## <a name="error-codes"></a>錯誤碼

取得或設定 **SetAsClass** 屬性之後， **Err** 物件可能會包含下列錯誤碼。

<dl> <dt>

**wbemErrFailed** -2147749889 (0x80041001) 
</dt> <dd>

未指定的錯誤。

</dd> </dl>
