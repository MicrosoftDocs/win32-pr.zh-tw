# <a name="swbemobjectpathclass-property"></a>SWbemObjectPath 類別屬性

[**SWbemObjectPath**](swbemobjectpath.md)物件的 **class** 屬性是物件路徑中類別的名稱。

如需此語法的說明，請參閱 [腳本 API 的檔慣例](document-conventions-for-the-scripting-api.md)。

這是可讀寫的屬性。

## <a name="syntax"></a>Syntax


