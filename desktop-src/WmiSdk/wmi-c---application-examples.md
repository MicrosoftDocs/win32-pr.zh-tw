# <a name="wmi-c-application-examples"></a>WMI c + + 應用程式範例

本節中的 WMI 應用程式範例是以 c + + 撰寫。 它們示範可以使用 WMI 元件完成的各種工作，並提供使用 Visual Basic 腳本的替代方案。 每個應用程式會以類似的方式分成一系列的步驟，讓來自不同範例的程式碼區段可以輕鬆地結合以形成自訂應用程式。

下表列出本節中的 c + + 範例。



| 範例                                                                                                                                  | 描述                                                                                                                                                |
|------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [範例：從本機電腦取得 WMI 資料](example--getting-wmi-data-from-the-local-computer.md)                               | 此範例會連接到本機電腦上的 WMI 命名空間，並從本機電腦上的查詢取得資料。 半同步方式會收集資料。 |
| [範例：以非同步方式從本機電腦取得 WMI 資料](example--getting-wmi-data-from-the-local-computer-asynchronously.md) | 此範例會連接到本機電腦上的 WMI 命名空間，並從本機電腦上的查詢取得資料。 資料會以非同步方式收集。    |
| [範例：從遠端電腦取得 WMI 資料](example--getting-wmi-data-from-a-remote-computer.md)                                 | 此範例會連接到遠端電腦上的 WMI 命名空間，並從遠端電腦上的查詢取得資料。 半同步方式會收集資料。 |
| [範例：呼叫提供者方法](example--calling-a-provider-method.md)                                                             | 此範例會連接到本機電腦上的 WMI 命名空間，並在 WMI 中呼叫方法。 方法會以同步方式執行。                          |
| [範例：透過 WMI 接收事件通知](example--receiving-event-notifications-through-wmi-.md)                            | 此範例會連接到本機電腦上的 WMI 命名空間，並接收來自本機電腦的事件。 半同步方式收到事件。   |
