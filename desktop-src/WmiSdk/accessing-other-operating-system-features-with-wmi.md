# <a name="accessing-other-operating-system-features-with-wmi"></a>使用 WMI 存取其他作業系統功能

WMI 具有延伸模組或提供者，可讓您存取其他作業系統功能，例如 Active Directory、ODBC 和 SNMP。 您也可以從設備磁碟機存取資料。

WMI 擴充功能無法在所有作業系統中使用。 如需詳細資訊，請參閱 [WMI 元件的作業系統可用性](operating-system-availability-of-wmi-components.md)。

下列主題提供存取其他作業系統功能的相關資訊：

-   [從設備磁碟機存取資料](accessing-data-from-device-drivers.md)
-   [存取 Active Directory](accessing-active-directory.md)
-   [存取 SNMP 裝置](accessing-snmp-devices.md)
