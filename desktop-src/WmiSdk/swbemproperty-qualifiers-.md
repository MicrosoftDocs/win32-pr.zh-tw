# <a name="swbempropertyqualifiers_-property"></a>Swbempropertyset 的限定詞 \_ 屬性

[**Swbempropertyset**](swbemproperty.md)物件的 **限定詞 \_** 屬性會傳回 [**SWbemQualifierSet**](swbemqualifierset.md)物件，該物件是此 WMI 屬性的限定詞集合。 這是唯讀的屬性。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
SWbemProperty.Qualifiers_ As Object
```
