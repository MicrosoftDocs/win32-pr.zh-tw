# <a name="wmi-specific-qualifiers"></a>WMI-Specific 限定詞

除了 CIM 限定詞之外，WMI 還提供一些額外的限定詞，以進一步精簡類別實例和作業的布建。

以下是 WMI 提供的其他限定詞清單：

-   [標準 WMI 限定詞](standard-wmi-qualifiers.md)
-   [SNMP 提供者專用的限定詞](qualifiers-specific-to-the-snmp-provider.md)
-   [視圖提供者專用的限定詞](qualifiers-specific-to-the-view-provider.md)
-   [WDM 提供者專用的限定詞](qualifiers-specific-to-the-wdm-provider.md)
-   [WMI 效能類別專用的限定詞](qualifiers-specific-to-wmi-performance-classes.md)

除了限定詞之外，描述限定詞行為的類別也可以套用至各種限定詞。 如需詳細資訊，請參閱 [限定詞](qualifier-flavors.md)類別。
