# <a name="policy-provider-classes"></a>原則提供者類別

WMI 原則提供者會透過類別提供群組原則的擴充功能，以允許在原則的應用程式中進行調整。 原則是使用 WMI 查詢語言 (WQL) 所設定，並位於 Active Directory 以方便部署。

下表列出 WMI 原則提供者的類別。



| 類別                                               | 描述                                                       |
|-----------------------------------------------------|-------------------------------------------------------------------|
| [**MSFT \_ 提供者**](/previous-versions/windows/desktop/wmisystemprov/msft-providers) | 公開與提供者實例相關的設定。             |
| [**MSFT \_ 規則**](/previous-versions/windows/desktop/policmanprov/msft-rule)                | 定義管理範圍 (SOM) 內的單一規則。         |
| [**MSFT \_ SomFilter**](/previous-versions/windows/desktop/policmanprov/msft-somfilter)      | 提供在目的電腦上評估的規則清單。 |
