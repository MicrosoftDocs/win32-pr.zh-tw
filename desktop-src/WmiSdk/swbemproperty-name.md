# <a name="swbempropertyname-property"></a>SWbemProperty.Name 屬性

[**Swbempropertyset**](swbemproperty.md)物件的 **name** 屬性會傳回這個 WMI 屬性的名稱。 這是唯讀的屬性。

如需此語法的說明，請參閱 [腳本 API 的檔慣例](document-conventions-for-the-scripting-api.md)。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
SWbemProperty.Name As String
```
