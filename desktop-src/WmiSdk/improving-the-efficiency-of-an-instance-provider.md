# <a name="improving-the-efficiency-of-an-instance-provider"></a>提升執行個體提供者的效率

WMI 高效能提供者會大幅增加 WMI 用戶端從 WMI 執行個體提供者取得資訊的速度。 主要的變更是，高效能提供者會以同進程伺服器的形式執行至用戶端應用程式或 WMI。 藉由將提供者放在用戶端進程內，您可以加速兩者之間的互動。

如需如何讓執行個體提供者成為高效能提供者的詳細資訊，請參閱 [讓執行個體提供者成為 High-Performance 提供](making-an-instance-provider-into-a-high-performance-provider.md)者。
