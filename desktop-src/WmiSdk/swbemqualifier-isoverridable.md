# <a name="swbemqualifierisoverridable-property"></a>SWbemQualifier. IsOverridable 屬性

[**SWbemQualifier**](swbemqualifier.md)物件的 **IsOverridable** 屬性是布林值，指出是否可在傳播時覆寫此限定詞。

如需此語法的說明，請參閱 [腳本 API 的檔慣例](document-conventions-for-the-scripting-api.md)。

這是可讀寫的屬性。

## <a name="syntax"></a>Syntax


```VB
SWbemQualifier.IsOverridable As Boolean
```
