# <a name="swbemqualifiername-property"></a>SWbemQualifier.Name 屬性

[**SWbemQualifier**](swbemqualifier.md)物件的 **name** 屬性會定義此限定詞的名稱。 這是唯讀的屬性。

如需此語法的說明，請參閱 [腳本 API 的檔慣例](document-conventions-for-the-scripting-api.md)。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
SWbemQualifier.Name As String
```
