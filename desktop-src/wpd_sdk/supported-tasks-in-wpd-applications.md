# <a name="supported-tasks-in-wpd-applications"></a>WPD 應用程式中支援的工作

WPD 應用程式介面支援下列工作：

-   連接到裝置
-   列舉附加的裝置
-   檢查裝置的功能
-   探索裝置的內容
-   將內容傳送至裝置，或在裝置上建立內容
-   編輯裝置上的內容
-   修改裝置上的物件
-   控制裝置
-   監視裝置事件

如需每項工作的詳細資訊，以及有關如何編寫特定工作之程式的詳細資訊，請參閱《 [Windows 可攜式裝置程式設計指南》](programming-guide.md)。
