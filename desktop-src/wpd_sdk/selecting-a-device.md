# <a name="selecting-a-device-wpd-api"></a>選取 (WPD API) 的裝置

當您的應用程式列舉連線的裝置之後，您可能會想要讓使用者能夠選取特定的裝置，並建立連線以進行進一步的作業。 在範例應用程式中， `ChooseDevice` DeviceEnumeration .cpp 模組中的函式會完成這些工作。

`ChooseDevice`函數會使用我們已看到的裝置列舉程式碼來列舉可用的裝置。  (請參閱 [列舉裝置](enumerating-devices.md) 一節以取得詳細資訊。 )
