# <a name="music-properties"></a>音樂屬性

Windows可攜式裝置支援下列音樂屬性。



| 屬性               | VarType        | 描述                                |
|------------------------|----------------|--------------------------------------------|
| **WPD \_ 音樂 \_ 專輯**  | **VT \_ LPWSTR** | 曲目所屬的專輯。            |
| **WPD \_ 音樂 \_ 歌詞** | **VT \_ LPWSTR** | 音樂的歌詞。                  |
| **WPD \_ 音樂的 \_ 情緒**   | **VT \_ LPWSTR** | 描述音樂情緒的字串。 |
| **WPD \_ 音樂 \_ 軌**  | **VT \_ UI4**    | 錄製的曲目編號。 |
