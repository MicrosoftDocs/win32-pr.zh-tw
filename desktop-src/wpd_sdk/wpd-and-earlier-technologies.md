# <a name="wpd-and-earlier-technologies"></a>WPD 和舊版技術

WPD 包含許多數位相機和便攜數位媒體播放機的增強功能，並且是在新版本的 Windows 中存取這些裝置的慣用方法。 現有的 Windows 映像取得 (WIA) 和 Windows 媒體裝置管理員 (中) WMDM Windows api 將會繼續受到支援，而且相容性層會提供 WPD 的相容性。 這些相容性層級可確保 WPD 裝置可以明確地公開至現有的 WIA 和 WMDM 應用程式，以做為 WIA 裝置或 WMDM 服務提供者。 不過，這些應用程式可能沒有與新的 WPD API 和新的 MTP 型裝置相同的功能範圍。

WPD MTP 驅動程式也支援 (PTP) 裝置的圖片傳送通訊協定。 WIA 中支援的 PTP 裝置會透過相容性層繼續在 WIA 應用程式中運作。 Microsoft 持續致力於確保在下一版的 Windows 中，應用程式和數位仍保持攝影機相容性，而這些新技術的設計目的是要在 Windows 中補充和增強數位的攝影機體驗。
