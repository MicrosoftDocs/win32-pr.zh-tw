# <a name="application-overview"></a>應用程式概觀

Windows可攜式裝置 (WPD) 提供基礎結構，可讓電腦存取儲存各種媒體和非媒體內容的多功能裝置。 此基礎結構可讓 convergent 的裝置，一開始會專注于智慧型存放裝置，例如便攜媒體播放機、數位攝影機和行動電話。

WPD 包含 (DDI) 的設備磁碟機介面，以及一組 (Api) 的應用程式開發介面。 本總覽說明如何在 Windows 應用程式中利用這些 api。

WPD 基礎結構提供高度可擴充的裝置標記法和控制機制。 這可讓垂直解決方案提供者使用 WPD API 來建立 WPD 標準化集以外的增強型使用者體驗。 其中的範例包括廠商提供的軟體，可執行固件更新或裝置診斷 (遠端支援) 。
