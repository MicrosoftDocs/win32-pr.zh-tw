# <a name="properties-and-attributes"></a>屬性和屬性

屬性會描述物件、屬性、資源和裝置。 屬性是一種特殊的屬性，可描述其他屬性或資源。

屬性和屬性會定義為具有兩個部分的 **PROPERTYKEY** 結構：類別目錄 GUID，以及該類別的唯一識別碼。 描述相關專案的屬性會共用通用的 GUID;此 GUID 是其屬性類別目錄。 類別可以描述一或多個屬性，每個屬性都有唯一的識別碼。 Microsoft 會定義一組屬性類別，您可以將這些屬性用於您自己的屬性，或自行建立新的類別。

接下來的主題列出 Windows 可攜式裝置所定義的屬性和屬性。
