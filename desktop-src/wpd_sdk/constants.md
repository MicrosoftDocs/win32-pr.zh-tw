# <a name="wpd-constants"></a>WPD 常數

Windows 可攜式裝置會定義下列常數類型。



| 常數類型                                              | Description                                                                                                                                                                                                                               |
|------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [一般常數](general-constants.md)                 | 裝置和應用程式的一般使用常數。                                                                                                                                                                                       |
| [錯誤常數](error-constants.md)                     | 方法所傳回的錯誤值。                                                                                                                                                                                                         |
| [事件常數](event-constants.md)                     | Windows 可攜式裝置所定義的事件種類。 事件種類包括加入的物件、移除的物件、物件更新、物件移動、物件複製、要求的物件傳輸、裝置重設、裝置已移除，以及裝置功能更新。 |
| [物件類型 Guid](object-type-guids.md)                 | Windows 可攜式裝置所定義的物件類型。 物件類型包括音訊、電子郵件和影像物件。                                                                                                                                  |
| [物件格式 Guid](object-format-guids.md)             | Windows 可攜式裝置所定義的檔案格式類型。 格式類型包括 WMA、WMV 和 JPEG。                                                                                                                                           |
| [資源類型 Guid](resource-type-guids.md)             | Windows 可攜式裝置所定義的資源類型。 資源類型包括音訊和縮圖影像資料。                                                                                                                                |
| [屬性和屬性](properties-and-attributes.md) | Windows 可攜式裝置所定義的屬性和屬性。                                                                                                                                                                            |
