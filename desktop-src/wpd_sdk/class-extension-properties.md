# <a name="class-extension-properties"></a>類別延伸模組屬性

Windows 可攜式裝置支援下列類別延伸模組屬性。



| 屬性                                                                      | VarType         | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
|-------------------------------------------------------------------------------|-----------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **WPD \_ 類別 \_ 延伸模組 \_ 選項 \_ 支援的 \_ 內容 \_ 類型**                 | **VT \_ 不明** | 值，指定驅動程式所支援之內容類型的 (超集合) 清單 (類似于呼叫 WPD \_ 命令功能，可 \_ \_ \_ \_ \_ 在 **WPD \_ 功能 \_ 類別目錄 \_ 所有**) 上取得支援的內容類型。                                                                                                                                                                                                                                                                                                                                             |
| **WPD \_ 類別 \_ 延伸模組選項不會 \_ \_ \_ 註冊 \_ WPD \_ 裝置 \_ 介面**    | **VT \_ BOOL**    | 值，指定呼叫端是否想要 WPD 類別擴充程式庫來註冊 WPD 裝置類別介面。 如果這個值為 true，呼叫端會負責註冊。<br/> 如果這個值為 false，則表示呼叫端需要類別延伸模組程式庫來執行註冊。<br/>大部分的驅動程式都應該允許類別延伸模組程式庫執行註冊，但類別延伸程式庫註冊 WPD 裝置類別介面的情況可能會造成不良的影響。 |
| **WPD \_ 類別 \_ 延伸模組 \_ 選項 \_ 註冊 \_ WPD 私用 \_ \_ 裝置 \_ 介面** | **VT \_ BOOL**    | 指出呼叫端想要 WPD 類別擴充程式庫註冊私用 WPD 裝置類別介面。 這不建議用於大部分的驅動程式。 只有當類別延伸程式庫註冊 WPD 裝置類別介面時，才應該使用它，將會造成不良的影響。 此選項通常會與 WPD 類別延伸模組選項搭配使用，而不會將 **\_ \_ \_ \_ \_ \_ WPD \_ 裝置 \_ 介面** 設為 **TRUE**                                                                                          |
| **WPD \_ 類別 \_ 延伸模組 \_ 選項 \_ 裝置 \_ 識別 \_ 值**            | **VT \_ 不明** | 這是包含裝置識別碼值的 [IPortableDeviceValues](iportabledevicevalues.md) ， (**WPD \_ 裝置 \_ 製造商**、 **WPD \_ 裝置 \_ 型號**、 **WPD \_ 裝置 \_ 固件 \_ 版本** ，以及 **WPD \_ 裝置 \_ 功能 \_ 唯一 \_ 識別碼**) 。 在初始化時將此包含在其他類別延伸模組選項中                                                                                                                                                                                                                               |
| **WPD \_ 類別 \_ 擴充功能 \_ 選項 \_ 傳輸 \_ 頻寬**                      | **VT \_ UI4**     | 指出傳輸的理論最大頻寬（以每秒千位）                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| **WPD \_ 類別 \_ 延伸模組 \_ 選項 \_ 裝置 \_ 識別 \_ 值**            | **VT \_ 不明** | 這是包含裝置識別碼值的 [IPortableDeviceValues](iportabledevicevalues.md) ， (**WPD \_ 裝置 \_ 製造商**、 **WPD \_ 裝置 \_ 型號**、 **WPD \_ 裝置 \_ 固件 \_ 版本** ，以及 **WPD \_ 裝置 \_ 功能 \_ 唯一 \_ 識別碼**) 。 在初始化時將此包含在其他類別延伸模組選項中。                                                                                                                                                                                                                              |
