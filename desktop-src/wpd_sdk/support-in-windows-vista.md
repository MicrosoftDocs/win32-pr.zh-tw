# <a name="support-in-windows-7-and-windows-vista"></a>Windows 7 和 Windows Vista 的支援

針對 Windows 7 和 Windows Vista 以外的 N 個 sku，WPD 執行時間會配套為作業系統的一部分。

### <a name="suported-api-elements"></a>支援 API 元素

下表指出 Windows 7、Windows Vista 和使用平臺更新 Windows vista 所支援的功能。



| 功能                        | Windows 7 | Windows Vista | WindowsVista 與平臺更新 |
|--------------------------------|-----------|---------------|----------------------------------------|
| 透過 USB 的 MTP                   | 是       | 是           | 是                                    |
| IP 上的 MTP                    | 是       | 是           | 是                                    |
| 藍牙的 MTP             | 是       | 否            | 是                                    |
| WPD 和 MTP 裝置服務    | 是       | 否            | 是                                    |
| WPD 自動化                 | 是       | 否            | 否                                     |
| 多函數/多重傳輸 | 是       | 否            | 否                                     |
| Device Stage                   | 是       | 否            | 否                                     |
| 裝置同步平臺           | 是       | 否            | 否                                     |



 

### <a name="the-platform-update-for-windows-vista"></a>Windows Vista 的平臺更新

Windows Vista 的平臺更新支援 Windows 可攜式裝置的 Windows 7 延伸模組 (WPD) 平臺。

### <a name="windows-vista-n-and-kn-versions-require-installation-of-windows-media-format-11-sdk"></a>WindowsVista N 和 KN 版本需要安裝 Windows 媒體格式 11 SDK

-   針對 Windows 7 N 和 KN 版本，您必須安裝 Windows 媒體格式 11 SDK，才能啟用 WPD 功能。
-   針對 Windows Vista N 和 KN 版本，您必須安裝 Windows 媒體格式 11 SDK，以啟用 WPD 功能。
