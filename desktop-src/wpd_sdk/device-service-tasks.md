# <a name="device-service-tasks"></a>裝置服務工作

WPD API 支援許多裝置服務工作。 這些工作包含下列各項：

-   正在列舉裝置服務。
-   開啟裝置服務。
-   列舉裝置服務的內容。
-   正在抓取支援的服務事件。
-   正在抓取支援的服務格式。
-   正在抓取支援的服務方法。
-   正在抓取支援的抽象服務。
-   正在抓取內容物件屬性。
-   撰寫內容物件屬性。
-   以同步方式叫用服務方法。
-   以非同步方式叫用服務方法。

這些作業將在下列各節中說明，並在 WpdServicesApiSample 應用程式中示範。 此外，WPD API 也支援傳送內容、移動內容，以及刪除裝置服務的內容。
