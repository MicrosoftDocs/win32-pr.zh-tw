# <a name="storage-properties"></a>儲存體屬性

Windows可攜式裝置支援下列存放裝置屬性。



| 屬性                                   | VarType        | 描述                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
|--------------------------------------------|----------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **WPD \_ 儲存體 \_ 容量**                 | **VT \_ UI8**    | 儲存體容量總計（以位元組為單位）。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| **WPD \_ \_ \_ 物件中的儲存體容量 \_**    | **VT \_ UI8**    | 表示物件中的總儲存容量;例如，SIM 卡上的可用插槽。                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| **WPD \_ 儲存體 \_ 描述**              | **VT \_ LPWSTR** | 人們看得懂的儲存體描述。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| **WPD \_ 儲存體 \_ 檔 \_ 系統 \_ 類型**       | **VT \_ LPWSTR** | 儲存體所使用之檔案系統的字串描述，例如「FAT32」、「NTFS」、「Contoso 檔案系統」等等。                                                                                                                                                                                                                                                                                                                                                                                                         |
| **WPD \_ 儲存體 \_ 可用 \_ 空間 \_ \_ （以位元組為單位）**   | **VT \_ UI8**    | 可用的儲存空間（以位元組為單位）。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| **WPD \_ \_ \_ \_ 物件中的儲存體可用空間 \_** | **VT \_ UI8**    | 可以寫入至裝置的其他物件數目。 例如，如果裝置只允許單一物件，則如果物件已經存在，就會是零。                                                                                                                                                                                                                                                                                                                                                          |
| **WPD \_ 儲存體 \_ 序號 \_**           | **VT \_ LPWSTR** | 儲存體的廠商特定序號。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| **WPD \_ 儲存體 \_ 最大 \_ 物件 \_ 大小**        | **VT \_ UI8**    | 指定可放置在此儲存體上之單一物件的大小上限（以位元組為單位）。                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| **WPD \_ 儲存體 \_ 類型**                     | **VT \_ UI4**    | 描述記憶體儲存媒體的實體類型。                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| **WPD \_ 儲存體 \_ 存取 \_ 功能**       | **VT \_ UI4**    | 識別全域影響此儲存體的任何寫入保護。 這優先于在個別物件上指定的存取權。 可能的值來自于 PortableDevice 中定義的 **WPD \_ 儲存體 \_ 存取 \_ 功能 \_ 值** 列舉。 例如，如果 WPD 儲存體類型 (是 **WPD \_ 存放裝置類型 \_ \_ 固定 \_ rom** 或 **WPD \_ 儲存體類型卸載式 \_ \_ \_ rom**) 的 **\_ 儲存體 \_ 類型**，則 **WPD \_ 存放裝置 \_ 存取 \_ 功能** 值必須是 **WPD \_ 儲存體 \_ 存取 \_ 功能， \_ \_ \_ 且不會 \_ \_ 刪除物件**。 |
