# <a name="support-in-windows-xp"></a>Windows XP 中的支援

若要讓您的應用程式在 Windows XP 中執行，使用者在電腦上需要有一份 Windows Media 格式執行時間。 您可以使用 WPD 應用程式重新發佈此執行時間。  (執行時間可在 Windows Media 格式 SDK 的可轉散發資料夾中找到。 )
