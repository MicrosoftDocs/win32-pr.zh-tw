# <a name="support-in-windows-xp"></a>Windows XP 中的支援

若要讓您的應用程式在 Windows XP 中執行，使用者將需要在其電腦上使用 Windows 媒體格式執行時間的複本。 您可以使用 WPD 應用程式重新發佈此執行時間。  (在 Windows 媒體格式 SDK 的可轉散發套件資料夾中找到執行時間。 )
