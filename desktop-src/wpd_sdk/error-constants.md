# <a name="error-constants"></a>錯誤常數

Windows 可攜式裝置方法可能會傳回各種 **HRESULT** 錯誤碼。 有些是 Windows 可攜式裝置專用的，有些則是由用來建立裝置的其他 Sdk 或移植套件所定義。 本節說明可傳回的各種錯誤碼，並依來源組織。

## <a name="media-transfer-protocol-error-codes"></a>媒體傳輸通訊協定錯誤碼

支援媒體傳輸通訊協定 (MTP) 的裝置可能會傳回下列錯誤碼。 這些錯誤碼大多表示 MTP 堆疊中的錯誤 (MTP 類別驅動程式與裝置) 之間的通訊，但對於診斷錯誤情況很有説明。
