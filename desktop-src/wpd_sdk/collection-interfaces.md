# <a name="collection-interfaces"></a>集合介面

下列介面可用來保存各種不同類型的值群組，包括 **PROPERTYKEY** 值、 **GUID** 值或其他集合介面。 用戶端應用程式和驅動程式都會使用這些介面來保存方法參數的資料。



| 介面                                                                            | 描述                                                            |
|--------------------------------------------------------------------------------------|------------------------------------------------------------------------|
| [**IPortableDeviceKeyCollection**](iportabledevicekeycollection.md)                 | 保留 **PROPERTYKEY** 值的集合。                          |
| [**IPortableDevicePropVariantCollection**](iportabledevicepropvariantcollection.md) | 保存相同 VARTYPE 的 **PROPVARIANT** 值集合。      |
| [**IPortableDeviceValues**](iportabledevicevalues.md)                               | 保留 **PROPERTYKEY** / **PROPVARIANT** 配對的集合。           |
| [**IPortableDeviceValuesCollection**](iportabledevicevaluescollection.md)           | 保存以零為基底的 **IPortableDeviceValues** 介面集合。 |
