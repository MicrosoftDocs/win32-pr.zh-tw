# <a name="operating-on-device-content"></a>操作裝置內容

WPD API 支援多個裝置上的作業。 這些作業包括：

-   將內容從電腦傳送至連線的裝置。
-   將內容從已連線的裝置傳送到電腦。
-   在連線的裝置上移動內容。
-   正在從已連線的裝置刪除內容。
-   將資源新增至連線裝置上的物件。

下面各節將會說明這些作業。
