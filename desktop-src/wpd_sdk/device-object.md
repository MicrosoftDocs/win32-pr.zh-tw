## <a name="commands"></a>命令

除了屬性之外，裝置也應支援 Windows 可攜式裝置所定義的一組特定命令。 物件或裝置支援的命令取決於其類型、功能和功能。

下表說明適用于裝置的命令類別（依功能）。 通常，裝置會落在數個類別之下，而且應該支援所有適用類別的命令。 例如，具有相機的行動電話會分成三個類別：所有裝置、SMS 裝置，以及靜止的影像捕獲裝置。 自訂驅動程式和用戶端應用程式可以支援您所定義的其他命令或屬性，但必須支援下列命令。 如需落在每個命令類別下之特定命令的說明，請參閱 [命令](commands.md)。



| Description                                                                                                                                                                                                                      | 命令類別目錄                                                                                                                                                                                                                                                                                                         |
|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 所有裝置。                                                                                                                                                                                                                     | **WPD \_ 類別 \_ CAPABILITIESWPD \_ 類別 \_ 通用**<br/> **WPD \_ 類別 \_ 物件 \_ 列舉**<br/> **WPD \_ 類別 \_ 物件 \_ 管理**<br/> **WPD \_ 類別 \_ 物件 \_ 屬性**<br/> **WPD \_ 類別目錄 \_ 物件 \_ 屬性 \_ 大量**<br/> **WPD \_ 類別 \_ 物件 \_ 資源**<br/> |
| 可以捕獲靜止影像（例如數位相機）的裝置。                                                                                                                                                                  | **WPD \_ 類別 \_ 仍為 \_ 影像 \_ 捕捉**                                                                                                                                                                                                                                                                                   |
| 可以傳送短訊息服務 (SMS) 訊息（例如行動電話）的裝置。 傳送 SMS 訊息通常稱為「文字訊息」。                                                                                      | **WPD \_ 類別 \_ SMS**                                                                                                                                                                                                                                                                                                     |
| 作為存放裝置運作的裝置。 這些包括外部磁片磁碟機。如果裝置支援格式化存放區，或將物件從某個位置移到另一個位置，則您的驅動程式應該支援此類別。<br/> | **WPD \_ 類別 \_ 儲存體**                                                                                                                                                                                                                                                                                                 |
