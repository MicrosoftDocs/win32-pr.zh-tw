# <a name="retrieving-device-information"></a>正在抓取裝置資訊

WPD API 提供一組豐富的介面和方法，可讓您用來取得指定裝置的資訊。 一般而言，裝置資料可以分成三個類別，如下表所述。



| 裝置資料                                                                             | Description                                                                               |
|-----------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [功能分類](retrieving-the-functional-categories-supported-by-a-device.md) | 功能類別描述裝置能夠執行的作業。 |
| [內容類型](retrieving-the-content-types-supported-by-a-device.md)                 | 內容類型代表裝置能夠轉譯的內容類型。         |
| [事件](retrieving-the-events-supported-by-a-device.md)                               | 某些裝置能夠引發您的應用程式可以追蹤和回應的事件。 |
