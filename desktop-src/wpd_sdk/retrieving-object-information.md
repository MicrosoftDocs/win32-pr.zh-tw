# <a name="retrieving-object-information"></a>正在抓取物件資訊

除了抓取裝置資訊，WPD API 還提供一組介面和方法，可讓您用來取得指定物件的資訊。 一般而言，物件資訊抓取可以分為兩個類別，如下表所述。



| 裝置資料                                                                                    | 描述                                                                               |
|------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [物件識別碼](retrieving-an-object-identifier-from-a-persistent-unique-identifier.md)   | 您可以取得對應至指定之持續性唯一識別碼的物件識別碼。 |
| [功能物件識別碼](retrieving-the-functional-object-identifiers-for-a-device.md) | 每個功能類別都支援一個或多個功能物件。                         |
