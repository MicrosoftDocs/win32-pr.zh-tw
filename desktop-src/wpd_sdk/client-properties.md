# <a name="client-properties"></a>用戶端屬性

Windows 可攜式裝置支援下列用戶端屬性。



| 屬性                                          | VarType                   | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
|---------------------------------------------------|---------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **WPD \_ 用戶端 \_ 所需的 \_ 存取**                  | **VT \_ UI4**               | 值，指定用戶端需要唯一的存取權限，例如泛型 \_ 唯讀。 如果未設定此屬性，WPD 會採用一般 \_ 讀取 \| 一般 \_ 寫入，這是建議的預設值。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **WPD \_ 用戶端 \_ 主要 \_ 版本**                   | **VT \_ UI4**               | 指定用戶端主要版本的數位。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| **WPD \_ 用戶端 \_ 次要 \_ 版本**                   | **VT \_ UI4**               | 指定用戶端次要版本的數位。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| **WPD \_ 用戶端 \_ 名稱**                             | **VT \_ LPWSTR**            | 用戶端應用程式的名稱字串。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| **WPD \_ 用戶端 \_ 修訂**                         | **VT \_ UI4**               | 指定用戶端 (或組建編號) 修訂的數位。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| **WPD \_ 客戶 \_ 端 \_ 安全性 \_ \_ 服務品質**   | **VT \_ UI4**               | 值，指定連接至驅動程式的安全性服務品質。 這與 **CreateFile** 的服務旗標安全性品質有關。<br/> 例如，這些旗標允許或不允許驅動程式模擬用戶端。 \_如果未提供此值，則會假設為安全性識別。 一般而言，應用程式應該將此值設定為安全性模擬 \_ ，因為某些驅動程式需要模擬才能運作。 不過，在具有已知裝置子集的嚴格安全性需求下執行的應用程式，可能會想要限制驅動程式識別或模擬呼叫者的能力。 請注意，將模擬等級減少為低於安全性模擬的任何 \_ 部分，可能會導致某些驅動程式功能降低。<br/>                                         |
| **WPD \_ 用戶端 \_ 共用 \_ 模式**                      | **VT \_ UI4**               | 值，指定用戶端從驅動程式要求的共用模式。 可能的值與 **CreateFile** 相同，例如，檔案 \_ 共用 \_ 讀取、檔案 \_ 共用 \_ 寫入等等。<br/> \_ \_ \| \_ \_ 如果用戶端未指定此參數，則會假設為檔案共用讀取檔案共用寫入。 這是建議的預設值。 但是，如果需要不同的共用模式，例如， \_ \_ 唯讀取要求讀取共用的檔案共用，則需要此參數。<br/>                                                                                                                                                                                                                                                                                                                                                          |
| **WPD \_ 用戶端 \_ WMDRM \_ 應用程式 \_ 私用 \_ 金鑰** | **VT \_ 向量 \_ \| vt \_ UI1** | 值，指定用戶端的 Windows Media DRM 應用程式私用金鑰。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **WPD \_ 用戶端 \_ WMDRM \_ 應用程式 \_ 憑證**  | **VT \_ 向量 \_ \| vt \_ UI1** | 值，指定用戶端的 Windows Media DRM 憑證。                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| **WPD \_ 用戶端 \_ 事件 \_ COOKIE**                    | **VT \_ LPWSTR**            | 用戶端提供的 cookie，是由這個用戶端發出的作業直接結果所張貼的事件中的驅動程式所傳回。 這是選擇性的屬性，驅動程式可以提供此屬性作為事件參數。 用戶端通常會使用此值來優化其事件處理。 例如，如果用戶端刪除物件，其事件處理常式可能會選擇忽略產生的事件，並瞭解它是否已採取適當的動作 (例如，在發出 delete) 的執行緒中更新視圖。                                                                                                                                                                                                                                                                                                                                                                |
| **WPD \_ 用戶端 \_ 最小 \_ 結果 \_ 緩衝區 \_ 大小**   | **VT \_ UI4**               | 指定傳送命令至驅動程式的緩衝區大小下限 (位元組) 。 用戶端可以使用這個選項，來建議用來將命令傳送至驅動程式的內部緩衝區大小下限。 這可能會導致用戶端的記憶體使用量較小，只會傳送具有小型資料大小的命令，例如取得或設定屬性。 驅動程式可以在其裝置參數區段中設定 **PortableDeviceResultsBufferMin** 來覆寫此緩衝區大小。 此緩衝區大小必須是1024或更新版本。 如果用戶端和驅動程式都沒有設定緩衝區大小，則預設為 1 MB。                                                                                                                                                                                                                                                  |
| **\_中斷連線時 WPD 用戶端 \_ 手動 \_ 關閉 \_ \_**    | **VT \_ BOOL**              | 用戶端的 advanced 選項，會針對每個已中斷連線的物件手動呼叫 **IPortableDevice：： close** 或 **IPortableDeviceService：： close** ，而不是依賴 API 來執行這項作業。 如果用戶端未指定此參數，則會假設為 FALSE。 如果用戶端未接聽 **WM \_ DEVICECHANGE** 訊息，則這是建議的預設值。 指定此選項的用戶端通常會接聽裝置的 **WM \_ DEVICECHANGE** 訊息，然後在裝置中斷連線時手動關閉 **IPortableDevice** 和 **IPortableDeviceService** 實例。 選擇此選項的用戶端，必須在裝置中斷連線時釋出這些物件，以謹慎地管理每個 **IPortableDevice** 和 **IPortableDeviceService** 實例的存留期。 若未這麼做，將會導致驅動程式的 WUDFHost.exe 進程流失。 |
