# <a name="media-properties"></a>媒體屬性

Windows 可攜式裝置支援下列媒體屬性。



| 屬性                                                                                                                 | VarType        | Description                                                                                                                                                                                                                                                                                                    |
|--------------------------------------------------------------------------------------------------------------------------|----------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **WPD \_ MEDIA \_ 專輯 \_ 演出者**                                                                                            | **VT \_ LPWSTR** | 識別執行媒體之演出者的字串。                                                                                                                                                                                                                                                      |
| **WPD \_ 媒體 \_ 音訊 \_ 編碼 \_ 設定檔**                                                                                 | VT \_ LPWSTR     | 指定編碼設定檔的字串。 媒體編解碼器可根據設定檔進行編碼，以定義特定的編碼演算法或優化程式。                                                                                                                                  |
| **WPD \_ 媒體 \_ 位元速率 \_ 類型**                                                                                            | **VT \_ UI4**    | [**WPD \_ 位元速率 \_ 類型**](wpd-bitrate-types.md)列舉，可進一步限定音訊或影片資料的位元速率。                                                                                                                                                                                     |
| **WPD \_ 媒體 \_ \_ 立即購買**                                                                                                 | **VT \_ BOOL**   | 布林值，指定使用者是否要在下一次進行網際網路連線時購買內容。                                                                                                                                                                                         |
| **WPD \_ 媒體 \_ 位元組 \_ 書簽**                                                                                           | **VT \_ UI8**    | 指定書簽 (以零為起始的位元組位移) ，其對應于在指定媒體上播放或觀看的最後一個位置。                                                                                                                                                                                     |
| **WPD \_ 媒體 \_ 編輯器**                                                                                                 | **VT \_ LPWSTR** | 編輯器，如果編輯器和執行者不同。                                                                                                                                                                                                                                                     |
| **WPD \_ 媒體 \_ 著作權**                                                                                                | **VT \_ LPWSTR** | 提供物件相關著作權資訊的字串。                                                                                                                                                                                                                                                 |
| <span id="wpd_media_description"></span><span id="WPD_MEDIA_DESCRIPTION"></span>**WPD \_ 媒體 \_ 描述**              | **VT \_ LPWSTR** | 指定之物件的描述。其中 [WPD \_ 媒體 \_ 子 \_ 描述](/windows) 屬性可以描述單一內容片段，而此屬性會描述相關媒體的集合。<br/>                                                                                 |
| **WPD \_ 媒體 \_ 目的地 \_ URL**                                                                                         | **VT \_ LPWSTR** | 以 null 終止、人們可讀取的字串，表示物件的 URL。 如果使用者按一下物件，這個 URL 會識別將出現的內容。**Mediacast** 物件通常會使用這個屬性來指定相關資訊的位置。<br/>                                          |
| **WPD \_ 媒體 \_ 持續時間**                                                                                                 | **VT \_ UI8**    | 內容的持續時間（以毫秒為單位）。                                                                                                                                                                                                                                                                  |
| **WPD \_ 媒體 \_ 有效 \_ 評等**                                                                                        | **VT \_ UI4**    | 介於1到100（含）之間的數位，可對內容進行分級，其中1是最小值。 這個值是根據使用者未明確設定的使用量統計資料。                                                                                                                                   |
| **WPD \_ 媒體 \_ 編碼 \_ 設定檔**                                                                                        | **VT \_ LPWSTR** | 設定檔名稱，如使用的特定編解碼器所定義。 如果編解碼器不需要特定的設定檔名稱，您應該改為在適當的屬性中指定音訊資訊。                                                                                                            |
| **WPD \_ 媒體內容類型 \_**                                                                                                    | **VT \_ LPWSTR** | 以 null 終止、人們可讀取的字串，指出此媒體所屬的類型。                                                                                                                                                                                                                     |
| **WPD \_ 媒體 \_ GUID**                                                                                                     | **VT \_ LPWSTR** | 指定128位全域唯一識別碼 (GUID) 。                                                                                                                                                                                                                                                         |
| <span id="wpd_media_height"></span><span id="WPD_MEDIA_HEIGHT"></span>**WPD \_ 媒體 \_ 高度**                             | **VT \_ UI4**    | 值，指定主資源 (通常是影片或影像) 的高度（以圖元為單位）。 請參閱 [WPD \_ 媒體 \_ 寬度](/windows)。                                                                                                                                                                    |
| **WPD \_ 媒體 \_ 上次 \_ 組建 \_ 日期**                                                                                        | **VT \_ 日期**   | 指定 mediacast 中的數列上次變更或編輯時間的日期。                                                                                                                                                                                                                                 |
| **WPD \_ 媒體 \_ 上次 \_ 訪問 \_ 時間**                                                                                     | **VT \_ 日期**   | 值，指定上一次在裝置上存取媒體的日期和時間。                                                                                                                                                                                                                       |
| **WPD \_ 媒體 \_ 管理 \_ 編輯器**                                                                                         | **VT \_ LPWSTR** | 以 null 終止、人們可讀取的字串，指出媒體管理編輯器的電子郵件地址。**Mediacast** 物件通常會使用這個屬性來指定 RSS 頻道或數列的管理編輯器。<br/>                                                                                 |
| **WPD \_ 媒體 \_ 中繼 \_ 類型**                                                                                              | **VT \_ UI4**    | 描述媒體類型的 [**WPD \_ 中繼 \_**](wpd-meta-genres.md) 內容列舉。                                                                                                                                                                                                                   |
| **WPD \_ 媒體 \_ 物件 \_ 書簽**                                                                                         | **VT \_ LPWSTR** | 指定 \_ \_ 所查看或播放最後一個物件的 WPD 物件識別碼。這個屬性會套用至播放清單和 mediacasts。<br/>                                                                                                                                                                               |
| **WPD \_ 媒體 \_ 擁有者**                                                                                                    | **VT \_ LPWSTR** | 以 null 終止、人們可讀取的字串，指出媒體擁有者的電子郵件地址。**Mediacast** 物件通常會使用這個屬性來指定 RSS 頻道或數列的擁有者。<br/>                                                                                                     |
| **WPD \_ 媒體 \_ 家長 \_ 評等**                                                                                         | **VT \_ LPWSTR** | 以 null 終止、人類看得懂的字串，指出媒體檔案的家長監護。 這通常是北美洲 (MPAA) 評等的移動圖片關聯。                                                                                                                                        |
| **WPD \_ 媒體 \_ 發行 \_ 日期**                                                                                            | **VT \_ 日期**   | 最初發行內容的日期。                                                                                                                                                                                                                                                             |
| **WPD \_ 媒體 \_ 取樣 \_ 率**                                                                                             | **VT \_ UI4**    | 內容的編碼範例速率（以每秒樣本數為單位）。                                                                                                                                                                                                                                              |
| **WPD \_ 媒體 \_ 略過 \_ 計數**                                                                                              | **VT \_ UI4**    | 此媒體已排入佇列等候播放，但使用者已明確略過的總次數。 這可用來自動計算使用者評等或修改內容更新規則。                                                                                                                  |
| **WPD \_ 媒體 \_ 來源 \_ URL**                                                                                              | **VT \_ LPWSTR** | 以 null 終止、人們可讀取的字串，表示指定媒體的來源 URL。**Mediacast** 物件通常會使用這個屬性來指定內容的來源位置。<br/>                                                                                                                   |
| **WPD \_ 媒體 \_ 星級 \_ 分級**                                                                                             | **VT \_ UI4**    | 介於0和5之間的數位，其中0是未分級的，1是最低評等，而5是最高評等。 使用者會明確輸入此值。                                                                                                                                        |
| <span id="wpd_media_sub_description"></span><span id="WPD_MEDIA_SUB_DESCRIPTION"></span>**WPD \_ 媒體 \_ 子 \_ 描述** | **VT \_ LPWSTR** | 給定物件的附屬描述。 這個屬性通常用來提供有關內容的其他資訊。其中 [WPD \_ 媒體 \_ DESCRIPTION](/windows) 屬性可以描述相關媒體的集合，而此屬性會描述單一的內容片段。<br/> |
| <span id="wpd_media_sub_title"></span><span id="WPD_MEDIA_SUB_TITLE"></span>**WPD \_ 媒體 \_ 子 \_ 標題**                   | **VT \_ LPWSTR** | 物件的子標題。 如果物件是數列中的資料集標題，這會是集標題，而 [WPD \_ 媒體 \_ 標題](/windows) 會是數列名稱。                                                                                                                                 |
| **WPD \_ 媒體 \_ 訂用帳戶 \_ 內容 \_ 識別碼**                                                                                | **VT \_ LPWSTR** | 用來識別線上訂閱服務內容片段的識別碼。 一般而言，這是由用戶端所設定，而不是由驅動程式或裝置設定。                                                                                                                                                    |
| **WPD \_ 媒體 \_ 時間 \_ 書簽**                                                                                           | **VT \_ UI8**    | 指定書簽 (（以毫秒為單位），) 對應至在指定媒體上播放或觀看的最後一個位置。                                                                                                                                                                                              |
| **WPD \_ 媒體 \_ 生存 \_ 時間 \_**                                                                                           | **VT \_ UI8**    | 以分鐘為單位指定下一次內容更新之前的時間 () 。                                                                                                                                                                                                                                                 |
| <span id="wpd_media_title"></span><span id="WPD_MEDIA_TITLE"></span>**WPD \_ 媒體 \_ 標題**                                | **VT \_ LPWSTR** | 內容的標題。 如果物件是數列中的資料集，這會是數列標題，而 WPD 的 [ \_ 媒體 \_ 子 \_ 標題](/windows) 會是集的名稱。                                                                                                                               |
| **WPD \_ 媒體 \_ 總 \_ 位元速率**                                                                                           | **VT \_ UI4**    | 物件的位元速率。                                                                                                                                                                                                                                                                                    |
| **WPD \_ 媒體 \_ 使用 \_ 計數**                                                                                               | **VT \_ UI4**    | 裝置上播放或查看此媒體的總次數。                                                                                                                                                                                                                                  |
| **WPD \_ 媒體 \_ 使用者 \_ 有效 \_ 評等**                                                                                  | **VT \_ UI4**    | 介於1到100之間的數位，可對此內容進行分級，1為最低。 使用者會明確設定此值。 預設值為0，表示尚未設定這個值。                                                                                                                    |
| **WPD \_ 媒體 \_ 管理員**                                                                                                | **VT \_ LPWSTR** | 以 null 終止、人們可讀取的字串，指出媒體管理員的電子郵件地址。**Mediacast** 物件通常會使用這個屬性來指定 RSS 頻道或數列的系統管理員。<br/>                                                                                             |
| <span id="wpd_media_width"></span><span id="WPD_MEDIA_WIDTH"></span>**WPD \_ 媒體 \_ 寬度**                                | **VT \_ UI4**    | 值，指定主要資源 (通常是影片或影像) 的寬度（以圖元為單位）。 請參閱 [WPD \_ 媒體 \_ 高度](/windows)。                                                                                                                                                                   |
| **WPD \_ MEDIA \_ 演出者**                                                                                                   | **VT \_ LPWSTR** | 識別媒體演出者的值。                                                                                                                                                                                                                                                                      |
