# <a name="wpd_section_data_units_values-enumeration"></a>WPD \_ 區段 \_ 資料 \_ 單位 \_ 值列舉

**WPD \_ 區段 \_ 資料 \_ 單位 \_ 值** 列舉指出參考資料區段的單位。

## <a name="syntax"></a>Syntax


```C++
typedef enum WPD_SECTION_DATA_UNITS_VALUES { 
  WPD_SECTION_DATA_UNITS_BYTES         = 0,
  WPD_SECTION_DATA_UNITS_MILLISECONDS  = 1
} ;
```
