# <a name="destinations"></a>目的地

路由表中的目的地是以網路位址和網路遮罩表示的網路專案。

路由表中的目的地專案包含：

-   位址，以網路位址和網路遮罩表示。
-   目的地的路由清單。
-   不透明指標位置的清單。
-   此目的地有效的視圖。 目的地包含每個包含下列資訊之視圖的結構：
    -   視圖的識別碼。
    -   此視圖中目的地的最佳路由指標。
    -   此視圖中最佳路線的擁有者。
    -   與此視圖中最佳路由相關聯的旗標。
    -   此視圖中處於按住狀態的任何路由的控制碼。
