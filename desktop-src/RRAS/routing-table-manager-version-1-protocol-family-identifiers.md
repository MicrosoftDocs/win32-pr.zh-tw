# <a name="routing-table-manager-version-1-protocol-family-identifiers"></a>路由表管理員第1版通訊協定系列識別碼

**Windows Server 2003：** 此 api 已由 [路由表管理員第2版](about-routing-table-manager-version-2.md)api 取代，而且將無法在 Windows Server 2003 之外使用。 新的應用程式應該使用路由表管理員第2版 API。

下列傳輸識別碼列在 Rtm 中。 將這些識別碼與 RTMv1 API 搭配使用。



| 常數                   | 描述                        |
|----------------------------|------------------------------------|
| RTM \_ 通訊協定 \_ 系列 \_ IPX | 識別 IPX 位址系列。 |
| RTM \_ 通訊協定 \_ 系列 \_ IP  | 識別 IP 位址系列。  |
