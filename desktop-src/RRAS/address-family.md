# <a name="address-family"></a>位址系列

位址系列包括網際網路封包 Exchange (IPX) 、網際網路通訊協定第4版 (IPv4) 和網際網路通訊協定第6版 (IPv6) 。 位址系列也可以稱為通訊協定系列。 這些位址系列是在 [RFC 1700](routing-protocols-request-for-comments.md)中定義。
