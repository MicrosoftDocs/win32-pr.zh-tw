# <a name="next-hop"></a>下一個躍點

下一個躍點是指向目的地的路徑下一個路由器。 來源的封包會依躍點逐一轉送至目的地。

路由中下一個躍點的路由器位址可用來唯一識別點對 multipoint 介面上的連結，其中所有連結都會共用相同的介面索引。
