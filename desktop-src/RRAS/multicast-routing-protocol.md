# <a name="multicast-routing-protocol"></a>多播路由通訊協定

多播路由通訊協定是一種服務提供者，可作為路由器架構架構內的用戶端。 路由架構是設計來由這類路由器用戶端模組擴充。

多播路由通訊協定會管理群組成員資格，並控制多播資料透過網路所採用的路徑。 多播路由通訊協定的範例包括：通訊協定獨立多播 (PIM) 、多播開放式最短路徑 First (MOSPF) ，以及距離向量多播路由通訊協定 (DVMRP) 。 網際網路群組管理通訊協定 (IGMP) 是一種特殊的多播路由通訊協定，可作為主機和路由器之間的媒介。
