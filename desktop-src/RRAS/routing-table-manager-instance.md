# <a name="routing-table-manager-instance"></a>路由表管理員實例

實例是另一個資料表，路由表管理員會使用此資料表來維護有關介面子集的路由資訊。 實例通常用來讓實體路由器作為一組虛擬路由器，也就是每個邏輯網路一個虛擬路由器。

路由表管理員目前只支援一個實例 (識別為零，預設) 。 用戶端可以向其他實例註冊，但是除了預設的路由器之外，路由器管理員可辨識或使用它以外的虛擬路由器。
