# <a name="change-notification-flags"></a>變更通知旗標



| 常數                         | 值      | 描述                                                                                                                                                           |
|----------------------------------|------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| RTM \_ 數目 \_ 變更 \_ 類型          | 3          | 指定路由表管理員目前使用的變更類型數目。                                                                            |
| RTM \_ 變更 \_ 類型 \_ 全部           | 0x0001     | 通知用戶端對目的地的任何變更。                                                                                                                   |
| RTM \_ 變更 \_ 類型 \_ 最佳          | 0x0002     | 通知用戶端最適合路由的變更，或當最佳路由變更時。                                                                                     |
| RTM \_ 變更 \_ 類型 \_ 轉送    | 0x0004     | 通知用戶端任何會影響轉送 (的最佳路由變更，例如下一個躍點變更) 。                                                                      |
| RTM \_ 通知 \_ 僅 \_ 標示為 \_ DESTS | 0x00010000 | 通知用戶端變更用戶端已標示的目的地。 如果未指定此旗標，則會傳送所有目的地的變更通知訊息。 |
