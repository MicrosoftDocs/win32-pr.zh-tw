# <a name="ip-information-types-for-router-information-blocks"></a>路由器資訊區塊的 IP 資訊類型

下列資訊類型列在 Ipinfoid 中。 執行 IP 傳輸時，請使用下列資訊類型搭配資訊標頭功能：

-   IP \_ 要求 \_ 撥號 \_ 篩選 \_ 資訊
-   IP \_ 篩選器 \_ 啟用 \_ 資訊
-   \_ \_ 篩選器資訊中的 IP \_
-   IP \_ 輸出 \_ 篩選 \_ 資訊
-   IP \_ 全域 \_ 資訊
-   IP \_ IFFILTER \_ 資訊
-   IP \_ 介面 \_ 狀態 \_ 資訊
-   IP \_ IPINIP \_ CFG \_ 資訊
-   IP \_ MCAST \_ 活動訊號 \_ 資訊
-   IP \_ MCAST \_ 界限 \_ 資訊
-   IP \_ MCAST \_ 限制 \_ 資訊
-   IP \_ PROT.10 \_ 優先順序 \_ 資訊
-   IP \_ 路由 \_ 資訊
-   IP \_ 路由器 \_ 光碟 \_ 資訊
