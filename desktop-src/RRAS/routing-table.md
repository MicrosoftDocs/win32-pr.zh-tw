# <a name="routing-table"></a>路由表

路由表管理員會為每個位址系列 (（例如 IPX 和 IPv4) ）維護不同的路由表。 RTMv1 API 支援 IPX 位址家族;RTMv2 API 支援 IPv4。

路由表是由目的地、路由和下一個躍點所組成。 這些專案會定義目的地網路的路由。
