# <a name="view-flags"></a>視圖旗標

使用視圖旗標來控制路由表的流覽。



| 常數                 | 值      | 描述                                                      |
|--------------------------|------------|------------------------------------------------------------------|
| RTM \_ \_ 位址 \_ 大小上限 | 16         | 位址系列的位址大小上限。                          |
| RTM \_ 最大 \_ 視圖          | 32         | 路由表中可處於作用中狀態的最大視圖數目。 |
| RTM \_ 視圖 \_ 識別碼 \_ UCAST     | 0          | 指定單播視圖。                                        |
| RTM \_ 視圖 \_ 識別碼 \_ MCAST     | 1          | 指定多播視圖。                                      |
| RTM \_ 視圖 \_ 遮罩 \_ 大小    | 0x20       | 指定可以設定的最大視圖數目。    |
| RTM \_ 視圖 \_ 遮罩 \_ 無    | 0x00000000 | 無論視圖為何，都會傳回信息。                      |
| RTM \_ 視圖 \_ 遮罩 \_ 任何     | 0x00000000 | 傳回所有視圖的目的地。 這是預設值。  |
| RTM \_ 視圖 \_ 遮罩 \_ UCAST   | 0x00000001 | 從單播視圖傳回目的地。                      |
| RTM \_ 視圖 \_ 遮罩 \_ MCAST   | 0x00000002 | 從多播視圖傳回目的地。                    |
| RTM \_ 視圖 \_ 遮罩 \_ 全部     | 0xFFFFFFFF | 傳回所有視圖的資訊。                              |
