# <a name="adapters"></a>配接器

介面卡代表連接到特定網路區段的硬體。 系結的區域網路卡是介面卡的其中一個範例。 同樣地，具有兩部數據機的電腦（每個都能夠連線到遠端網路）有兩張介面卡，分別代表每一台數據機。
