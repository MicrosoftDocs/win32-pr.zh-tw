# <a name="overview-of-mgm-callbacks"></a>MGM 回呼的總覽

MGM API 中有兩種類型的回呼：

-   [路由通訊協定回呼](routing-protocol-callbacks.md)
-   [IGMP 啟用和停用回呼](igmp-enable-and-disable-callbacks.md)

這些回呼結合了 MGM Api，可讓路由通訊協定、IGMP 和多播群組管理員之間進行通訊。
