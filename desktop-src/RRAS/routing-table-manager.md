# <a name="routing-table-manager"></a>路由表管理員

路由表管理員是路由資訊的中央存放庫，適用于在 [路由及遠端存取服務] (RRAS) 下運作的所有路由通訊協定。 它會在發生變更時通知用戶端，並允許用戶端交換私用資訊。

路由表管理員會提供路由資訊給所有感興趣的用戶端，例如路由通訊協定、管理程式和監視程式。 路由表管理員也會決定路由通訊協定已知的每個目的地網路的最佳路由。 路由表管理員會根據路由通訊協定優先順序以及與路由相關聯的計量來決定此路由。 管理路由器的人員可以使用 RRAS 管理主控台設定路由通訊協定優先順序。

路由表管理員會將最佳路由資訊傳遞至轉寄站 (每個位址系列，或每個介面) 和所有感興趣用戶端的轉寄站。

每個用戶端都會使用路由表管理員進行註冊，而傳回的會接收用戶端用來新增或刪除路由的控制碼。 用戶端可以取出儲存在路由表中的資訊。 此外，用戶端可以向路由表管理員註冊，以接收目的地最佳路由變更的通知。
