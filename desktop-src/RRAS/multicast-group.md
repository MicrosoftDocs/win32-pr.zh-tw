# <a name="multicast-group"></a>多播群組

多播群組是類別 D IP 位址 (也就是在 224.0.0.0-239.255.255.255) 的範圍內。 傳送到此範圍內位址的多播資料，不是以單一目標為目的地。 相反地，任何要求加入特定群組的主機都會收到此資料。 多播群組是由符號 G 或 G 所參考。
