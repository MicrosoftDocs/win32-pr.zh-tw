# <a name="finding-specific-information-in-the-routing-table"></a>尋找路由表中的特定資訊

用戶端必須能夠找出路由表中的特定資訊，而不需要列舉所有資料。 RTMv2 API 可讓您根據特定準則搜尋特定目的地、路由和下一個躍點。
