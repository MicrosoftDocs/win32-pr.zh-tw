# <a name="client"></a>用戶端

RTMv2 的用戶端是路由通訊協定的實例，或是使用 RTMv2 API 註冊和與路由表管理員進行互動的管理元件。 除非另有指定，否則路由表管理員用戶端的任何參考都會參考 RTMv2 的用戶端。

用戶端必須在與路由器管理員相同的進程中執行。
