# <a name="ipv6-information-types-for-router-information-blocks"></a>路由器資訊區塊的 IPv6 資訊類型

下列資訊類型列在 Ipinfoid 中。 執行 IP 傳輸時，請使用下列資訊類型搭配資訊標頭功能：

-   IP \_ 要求 \_ 撥號 \_ 篩選 \_ 資訊 \_ V6
-   IP \_ 篩選器 \_ 啟用 \_ 資訊 \_ V6
-   \_ \_ 篩選 \_ 資訊 V6 中 \_ 的 IP
-   IP \_ IFFILTER \_ 資訊 \_ V6
-   IP \_ 輸出 \_ 篩選 \_ 資訊 \_ V6
-   IPV6 \_ 全域 \_ 資訊
-   IPV6 \_ 路由 \_ 資訊
