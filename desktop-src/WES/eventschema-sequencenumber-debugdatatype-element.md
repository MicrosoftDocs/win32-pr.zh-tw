# <a name="sequencenumber-debugdatatype-element"></a>SequenceNumber (DebugDataType) 元素

追蹤訊息的本機或全域序號。

``` syntax
<xs:element name="SequenceNumber"
    type="unsignedInt"
    minOccurs="0"
 />
```

**SequenceNumber** 元素是由 [**DebugDataType**](eventschema-debugdatatype-complextype.md)複雜型別定義。
