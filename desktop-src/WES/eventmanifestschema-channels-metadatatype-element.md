# <a name="channels-metadatatype-element"></a>通道 (MetadataType) 元素

定義提供者可以記錄事件的通道清單。

``` syntax
<xs:element name="channels"
    type="ChannelListType"
 />
```

**通道** 元素是由 [**MetadataType**](eventmanifestschema-metadatatype-complextype.md)複雜型別定義。
