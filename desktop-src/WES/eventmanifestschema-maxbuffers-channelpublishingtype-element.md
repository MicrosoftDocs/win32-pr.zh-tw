# <a name="maxbuffers-channelpublishingtype-element"></a>maxBuffers (ChannelPublishingType) 元素

要配置給會話的緩衝區數目上限。

``` syntax
<xs:element name="maxBuffers"
    type="unsignedInt"
 />
```

**MaxBuffers** 元素是由 [**ChannelPublishingType**](eventmanifestschema-channelpublishingtype-complextype.md)複雜型別定義。
