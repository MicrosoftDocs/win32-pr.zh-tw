# <a name="minbuffers-channelpublishingtype-element"></a>minBuffers (ChannelPublishingType) 元素

要配置給會話的緩衝區數目下限。

``` syntax
<xs:element name="minBuffers"
    type="unsignedInt"
 />
```

**MinBuffers** 元素是由 [**ChannelPublishingType**](eventmanifestschema-channelpublishingtype-complextype.md)複雜型別定義。
