# <a name="valuemap-maptype-element"></a> (MapType) 元素的 valueMap

定義對應整數值和字串值的名稱/值組清單。

``` syntax
<xs:element name="valueMap"
    type="ValueMapType"
 />
```

**ValueMap** 元素是由 [**MapType**](eventmanifestschema-maptype-complextype.md)複雜型別定義。
