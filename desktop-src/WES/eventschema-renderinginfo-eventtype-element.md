# <a name="renderinginfo-eventtype-element"></a>RenderingInfo (的事件) 元素

包含事件的已轉譯訊息字串 (包含事件的訊息字串，以及任何事件屬性的訊息字串，例如層級、工作和 opcode) 。

``` syntax
<xs:element name="RenderingInfo"
    type="RenderingType"
 />
```

**RenderingInfo** 元素是由「型別 [**複雜型**](eventschema-eventtype-complextype.md)別」（element）所定義。
