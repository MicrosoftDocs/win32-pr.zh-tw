# <a name="task-definitiontype-element"></a>task (DefinitionType) 元素

\[從 Windows 7 版 Windows SDK 隨附的訊息編譯器開始，此專案已無法再使用。\]

定義您的提供者可以記錄的工作特定事件。

``` syntax
<xs:element name="task"
    type="TaskEventDefinitionType"
 />
```

**Task** 專案是由 [**DefinitionType**](eventmanifestschema-definitiontype-complextype.md)複雜型別定義。
