# <a name="data-structdefinitiontype-element"></a>資料 (StructDefinitionType) 元素

定義您想要包含在結構中的資料項目。

``` syntax
<xs:element name="data"
    type="DataDefinitionType"
 />
```

**資料** 元素是由 [**StructDefinitionType**](eventmanifestschema-structdefinitiontype-complextype.md)複雜型別定義。
