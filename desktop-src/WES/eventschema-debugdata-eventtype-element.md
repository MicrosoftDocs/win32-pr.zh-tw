# <a name="debugdata-eventtype-element"></a>DebugData (的事件) 元素

包含可針對 Windows 軟體追蹤預處理器 (WPP) 事件記錄的資料。

``` syntax
<xs:element name="DebugData"
    type="DebugDataType"
 />
```

**DebugData** 元素是由「型別 [**複雜型**](eventschema-eventtype-complextype.md)別」（element）所定義。
