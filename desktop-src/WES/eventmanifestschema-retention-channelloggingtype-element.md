# <a name="retention-channelloggingtype-element"></a>保留 (ChannelLoggingType) 元素

判斷記錄檔是否為連續或迴圈的記錄檔。

``` syntax
<xs:element name="retention"
    type="boolean"
 />
```

**保留** 專案是由 [**ChannelLoggingType**](eventmanifestschema-channelloggingtype-complextype.md)複雜型別定義。
