# <a name="eventmanifest-schema-simple-types"></a>EventManifest 架構簡單類型

以下是 EventManifest 架構定義的簡單類型。



| 簡單型別                                                       | 描述                                                                                                                                |
|-------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------|
| [**CountType**](eventmanifestschema-counttype-simpletype.md)     | 定義用來指定陣列中元素數目的計數類型。<br/>                                                |
| [**CSymbolType**](eventmanifestschema-csymboltype-simpletype.md) | 定義有效的 C/c + + 符號名稱。<br/>                                                                                              |
| [**filePath**](eventmanifestschema-filepath-simpletype.md)       | 定義包含檔案完整路徑的字串。<br/>                                                                |
| [**GUIDType**](eventmanifestschema-guidtype-simpletype.md)       | 以登錄格式定義全域唯一識別碼型別。<br/>                                                                  |
| [**HexInt8Type**](eventmanifestschema-hex8type-simpletype.md)    | 定義1位元組的十六進位型別。<br/>                                                                                              |
| [**HexInt16Type**](eventmanifestschema-hex16type-simpletype.md)  | 定義2個位元組的十六進位型別。<br/>                                                                                              |
| [**HexInt32Type**](eventmanifestschema-hex32type-simpletype.md)  | 定義4位元組的十六進位型別。<br/>                                                                                              |
| [**HexInt64Type**](eventmanifestschema-hex64type-simpletype.md)  | 定義8位元組的十六進位型別。<br/>                                                                                             |
| [**UInt8Type**](eventmanifestschema-hexint8type-simpletype.md)   | 定義不帶正負號的位元組類型。<br/>                                                                                                  |
| [**UInt16Type**](eventmanifestschema-hexint16type-simpletype.md) | 定義不帶正負號的短型別。<br/>                                                                                                 |
| [**UInt32Type**](eventmanifestschema-hexint32type-simpletype.md) | 定義不帶正負號的整數類型。<br/>                                                                                               |
| [**UInt64Type**](eventmanifestschema-hexint64type-simpletype.md) | 定義不帶正負號的 long 類型。<br/>                                                                                                  |
| [**LengthType**](eventmanifestschema-lengthtype-simpletype.md)   | 定義長度類型，這個類型會用來指定變數長度資料項目中的位元組或字元數目。<br/>                 |
| [**QNameList**](eventmanifestschema-qnamelist-simpletype.md)     | 定義限定名稱的清單。<br/>                                                                                              |
| [**strTableRef**](eventmanifestschema-strtableref-simpletype.md) | 定義字串，這個字串會參考資訊清單中的字串資料表中所定義的訊息字串，或是在 ( 的訊息中，) 檔案。<br/> |
