# <a name="provider-eventstype-element"></a>提供者 (EventsType) 元素

定義提供者及其用來定義其事件的中繼資料。

``` syntax
<xs:element name="provider"
    type="ProviderType"
 />
```

**Provider** 元素是由 [**EventsType**](eventmanifestschema-eventstype-complextype.md)複雜型別定義。
