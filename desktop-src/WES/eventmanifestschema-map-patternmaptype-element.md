# <a name="map-patternmaptype-element"></a>map (PatternMapType) 元素

定義用來在訊息字串中尋找相符字串的正則運算式，並加以修改。

``` syntax
<xs:element name="map"
    type="PatternMapValueType"
 />
```

**Map** 元素是由 [**PatternMapType**](eventmanifestschema-patternmaptype-complextype.md)複雜型別定義。
