# <a name="buffersize-channelpublishingtype-element"></a>bufferSize (ChannelPublishingType) 元素

包含配置給會話緩衝區的記憶體數量（以 kb 為單位）。

``` syntax
<xs:element name="bufferSize"
    type="unsignedInt"
 />
```

**BufferSize** 元素是由 [**ChannelPublishingType**](eventmanifestschema-channelpublishingtype-complextype.md)複雜型別定義。
