# <a name="data-templateitemtype-element"></a>資料 (TemplateItemType) 元素

定義您想要包含在事件中的資料項目。

``` syntax
<xs:element name="data"
    type="DataDefinitionType"
 />
```

**資料** 元素是由 [**TemplateItemType**](eventmanifestschema-templateitemtype-complextype.md)複雜型別定義。
