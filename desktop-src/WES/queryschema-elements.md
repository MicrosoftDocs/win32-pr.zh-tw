# <a name="query-schema-elements"></a>查詢架構元素

以下是查詢架構定義的元素。 此區段包含您要在查詢中使用的元素名稱;不過，若要取得每個專案的詳細資料，請參閱包含元素的複雜類型。



| 元素                                                                  | 描述                                                                                                                     |
|--------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------|
| [**Q u t**](queryschema-querylist-element.md)                       | 查詢的根節點。<br/>                                                                                          |
| [**查詢 (QueryListType)**](queryschema-query-querylisttype-element.md) | 定義一組選取器和 suppressors，用來在結果集中包含或排除事件。<br/> |
| [**選取 (QueryType)**](queryschema-select-querytype-element.md)       | XPath 查詢，識別要包含在查詢結果集中的事件。<br/>                                        |
| [**隱藏 (QueryType)**](queryschema-suppress-querytype-element.md)   | XPath 查詢，識別要從查詢結果集排除的事件。<br/>                                      |
