# <a name="function-debugdatatype-element"></a>函數 (DebugDataType) 元素

記錄追蹤訊息的函數名稱。

``` syntax
<xs:element name="Function"
    type="string"
    minOccurs="0"
 />
```

**Function** 元素是由 [**DebugDataType**](eventschema-debugdatatype-complextype.md)複雜型別定義。
