# <a name="event-schema-simple-types"></a>事件架構簡單類型

以下是事件架構定義的簡單類型。



| 簡單類型                                                 | 描述                                                               |
|-------------------------------------------------------------|---------------------------------------------------------------------------|
| [**GUIDType**](eventschema-guidtype-simpletype.md)         | 以登錄格式定義全域唯一識別碼型別。<br/> |
| [**HexInt32Type**](eventschema-hexint32type-simpletype.md) | 定義1位元組的十六進位型別。<br/>                             |
| [**HexInt64Type**](eventschema-hexint64type-simpletype.md) | 定義8位元組的十六進位型別。<br/>                            |
