# <a name="eventmanifest-schema-elements"></a>EventManifest 架構元素

以下是 EventManifest 架構所定義的元素。 此區段包含您在資訊清單中使用的元素名稱;不過，若要取得每個專案的詳細資料，請參閱包含元素的複雜類型。



| 元素                                                                                                                  | 描述                                                                                                                                                                                                                                                                          |
|--------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**autoBackup (ChannelLoggingType)**](eventmanifestschema-autobackup-channelloggingtype-element.md)                     | 決定當目前的記錄檔達到大小上限時，是否要建立新的記錄檔。<br/>                                                                                                                                                                           |
| [**二元 (TemplateItemType)**](eventmanifestschema-binary-templateitemtype-element.md)                                 | 包含 [事件記錄](/windows/desktop/EventLog/event-logging) 檔 API 所提供的二進位資料。<br/>                                                                                                                                                                                     |
| [**點陣圖 (MapType)**](eventmanifestschema-bitmap-maptype-element.md)                                                   | 定義對應位值和字串值的名稱/值組清單。<br/>                                                                                                                                                                                                 |
| [**bufferSize (ChannelPublishingType)**](eventmanifestschema-buffersize-channelpublishingtype-element.md)               | 包含配置給會話緩衝區的記憶體數量（以 kb 為單位）。<br/>                                                                                                                                                                                           |
| [**channel (ChannelListType)**](eventmanifestschema-channel-channellisttype-element.md)                                 | 定義可以記錄事件的通道。<br/>                                                                                                                                                                                                                          |
| [**通道 (MetadataType)**](eventmanifestschema-channels-metadatatype-element.md)                                     | 定義提供者可以記錄事件的通道清單。<br/>                                                                                                                                                                                                             |
| [**通道 (ProviderType)**](eventmanifestschema-channels-providertype-element.md)                                     | 定義提供者可以記錄事件的通道清單。<br/>                                                                                                                                                                                                             |
| [**clockType (ChannelPublishingType)**](eventmanifestschema-clocktype-channelpublishingtype-element.md)                 | 記錄每個事件的時間戳記時，所要使用的時鐘解析。<br/>                                                                                                                                                                                                   |
| [**controlGuid (ChannelPublishingType)**](eventmanifestschema-controlguid-channelpublishingtype-element.md)             | 識別包含 WPP 事件之 ETW 會話的會話 GUID。<br/>                                                                                                                                                                                                  |
| [**資料 (StructDefinitionType)**](eventmanifestschema-data-structdefinitiontype-element.md)                             | 定義您想要包含在結構中的資料項目。<br/>                                                                                                                                                                                                            |
| [**資料 (TemplateItemType)**](eventmanifestschema-data-templateitemtype-element.md)                                     | 定義您想要包含在事件中的資料項目。<br/>                                                                                                                                                                                                              |
| [**(InputType) 的描述**](eventmanifestschema-description-inputtype-element.md)                                     | 輸入類型的描述。<br/>                                                                                                                                                                                                                                          |
| [**事件 (DefinitionType)**](eventmanifestschema-event-definitiontype-element.md)                                       | 定義您的提供者可以記錄的事件。<br/>                                                                                                                                                                                                                              |
| [**事件 (作業碼)**](eventmanifestschema-event-opcode-element.md)                                                       | 無法使用。**Windows Server 2008 和 Windows Vista：** 定義工作特定作業碼的事件。 從 Windows 7 開始無法使用。<br/> <br/>                                                                                                                  |
| [**事件 (TaskEventDefinitionType)**](eventmanifestschema-event-taskeventdefinitiontype-element.md)                     | 無法使用。**Windows Server 2008 和 Windows Vista：** 定義工作特定事件的事件。 從 Windows 7 開始無法使用。<br/> <br/>                                                                                                                   |
| [**事件 (Instrumentationtype.event)**](eventmanifestschema-events-instrumentationtype-element.md)                           | 包含資訊清單中所定義之提供者的清單。<br/>                                                                                                                                                                                                            |
| [**事件 (ProviderType)**](eventmanifestschema-events-providertype-element.md)                                         | 包含提供者所定義之事件的清單。<br/>                                                                                                                                                                                                                  |
| [**importChannel (ChannelListType)**](eventmanifestschema-importchannel-channellisttype-element.md)                     | 識別已由另一個提供者定義的通道。<br/>                                                                                                                                                                                                           |
| [**instrumentationManifest**](eventmanifestschema-instrumentationmanifest-element.md)                                   | 資訊清單的根節點。<br/>                                                                                                                                                                                                                                            |
| [**檢測 (instrumentationManifest)**](eventmanifestschema-instrumentation-instrumentationmanifest-element.md) | 定義資訊清單包含的檢測類型 (例如事件或計數器檢測) 。<br/>                                                                                                                                                          |
| [**inType (InputTypeListType)**](eventmanifestschema-intype-inputtypelisttype-element.md)                               | 定義輸入資料類型。<br/>                                                                                                                                                                                                                                               |
| [**inTypes (TypeListType)**](eventmanifestschema-intypes-typelisttype-element.md)                                       | 包含輸入類型定義的清單。<br/>                                                                                                                                                                                                                                |
| [**關鍵字 (KeywordListType)**](eventmanifestschema-keyword-keywordlisttype-element.md)                                 | 定義可識別事件類別目錄的關鍵字。<br/>                                                                                                                                                                                                                   |
| [**關鍵字 (ChannelPublishingType)**](eventmanifestschema-keywords-channelpublishingtype-element.md)                   | 位元遮罩，可決定寫入通道的事件分類。<br/>                                                                                                                                                                                         |
| [**關鍵字 (MetadataType)**](eventmanifestschema-keywords-metadatatype-element.md)                                     | 定義分類事件的關鍵字清單。<br/>                                                                                                                                                                                                                        |
| [**關鍵字 (ProviderType)**](eventmanifestschema-keywords-providertype-element.md)                                     | 定義分類事件的關鍵字清單。<br/>                                                                                                                                                                                                                        |
| [**延遲 (ChannelPublishingType)**](eventmanifestschema-latency-channelpublishingtype-element.md)                     | 清除緩衝區之前等候的時間（以毫秒為單位）。<br/>                                                                                                                                                                                                            |
| [**層級 (ChannelPublishingType)**](eventmanifestschema-level-channelpublishingtype-element.md)                         | 要寫入通道之事件的嚴重性層級。<br/>                                                                                                                                                                                                                 |
| [**層級 (LevelListType)**](eventmanifestschema-level-levellisttype-element.md)                                         | 定義嚴重性值，以判斷記錄期間事件的詳細程度。<br/>                                                                                                                                                                                          |
| [**層級 (MetadataType)**](eventmanifestschema-levels-metadatatype-element.md)                                         | 定義指定事件嚴重性的層級清單。<br/>                                                                                                                                                                                                           |
| [**層級 (ProviderType)**](eventmanifestschema-levels-providertype-element.md)                                         | 定義指定事件嚴重性的層級清單。<br/>                                                                                                                                                                                                           |
| [**當地語系化 (instrumentationManifest)**](eventmanifestschema-localization-instrumentationmanifest-element.md)       | 定義您在資訊清單中參考的當地語系化資源群組。<br/>                                                                                                                                                                                               |
| [**記錄 (ChannelType)**](eventmanifestschema-logging-channeltype-element.md)                                         | 定義支援通道之記錄檔的屬性，例如其容量，以及是否為連續或迴圈。<br/>                                                                                                                                             |
| [**map (BitMapType)**](eventmanifestschema-map-bitmaptype-element.md)                                                   | 定義位值與字串值之間的對應。<br/>                                                                                                                                                                                                               |
| [**map (PatternMapType)**](eventmanifestschema-map-patternmaptype-element.md)                                           | 定義用來在訊息字串中尋找相符字串的正則運算式，並加以修改。<br/>                                                                                                                                                                        |
| [**map (ValueMapType)**](eventmanifestschema-map-valuemaptype-element.md)                                               | 定義整數值與字串值之間的對應。<br/>                                                                                                                                                                                                          |
| [**maps (ProviderType)**](eventmanifestschema-maps-providertype-element.md)                                             | 定義名稱/值組的清單，您可以在資訊清單的範本區段中參考。<br/>                                                                                                                                                                        |
| [**maxBuffers (ChannelPublishingType)**](eventmanifestschema-maxbuffers-channelpublishingtype-element.md)               | 要配置給會話的緩衝區數目上限。<br/>                                                                                                                                                                                                                |
| [**maxSize (ChannelLoggingType)**](eventmanifestschema-maxsize-channelloggingtype-element.md)                           | 記錄檔的大小上限（以位元組為單位）。<br/>                                                                                                                                                                                                                              |
| [**訊息 (訊息資料表)**](eventmanifestschema-message-messagetable-element.md)                                      | 包含資訊清單當地語系化區段中之字串的參考。<br/>                                                                                                                                                                                             |
| [**messageTable (EventsType) 元素**](eventmanifestschema-messagetable-instrumentationtype-element.md)                | 包含資訊清單當地語系化區段中之字串的參考。<br/>                                                                                                                                                                                             |
| [**中繼資料 (instrumentationManifest)**](eventmanifestschema-metadata-instrumentationmanifest-element.md)               | 包含您可以在其他資訊清單中參考的全域值。<br/>                                                                                                                                                                                                         |
| [**minBuffers (ChannelPublishingType)**](eventmanifestschema-minbuffers-channelpublishingtype-element.md)               | 要配置給會話的緩衝區數目下限。<br/>                                                                                                                                                                                                                |
| [**namedQueries (MetadataType)**](eventmanifestschema-namedqueries-metadatatype-element.md)                             | 未使用。 定義已命名查詢的清單，這些查詢會查詢事件訊息字串中的值，並執行指定的動作（如果找到的話）。<br/>                                                                                                                                        |
| [**namedQueries (ProviderType)**](eventmanifestschema-namedqueries-providertype-element.md)                             | 未使用。 定義已命名查詢的清單，這些查詢會查詢事件訊息字串中的值，並執行指定的動作（如果找到的話）。<br/>                                                                                                                                        |
| [**opcode (OpcodeListType)**](eventmanifestschema-opcode-opcodelisttype-element.md)                                     | 包含識別活動的數值，或在應用程式引發事件 (（例如初始化或關閉) ）時所執行之活動內的某個點。<br/>                                                                                |
| [**opcode (TaskEventDefinitionType)**](eventmanifestschema-opcode-taskeventdefinitiontype-element.md)                   | 無法使用。**Windows Server 2008 和 Windows Vista：** 包含事件的工作特定作業碼。 所有的 opcode 定義都假設為包含事件定義之工作的特定工作。 從 Windows 7 開始無法使用。<br/> <br/> |
| [**MetadataType)  (碼**](eventmanifestschema-opcodes-metadatatype-element.md)                                       | 定義可用於將工作中的事件群組在一起的作業碼清單。<br/>                                                                                                                                                                                                 |
| [**ProviderType)  (碼**](eventmanifestschema-opcodes-providertype-element.md)                                       | 定義可用於將工作中的事件群組在一起的作業碼清單。<br/>                                                                                                                                                                                                 |
| [**TaskType)  (碼**](eventmanifestschema-opcodes-tasktype-element.md)                                               | 定義工作特定作業碼的清單。<br/>                                                                                                                                                                                                                                  |
| [**outType (InputType)**](eventmanifestschema-outtype-inputtype-element.md)                                             | 決定如何呈現資料的輸出資料類型清單。<br/>                                                                                                                                                                                                      |
| [**patternMap (PatternMapListType)**](eventmanifestschema-patternmap-patternmaplisttype-element.md)                     | 定義兩個正則運算式之間的對應。 其中一個運算式可用來在訊息字串中尋找相符的字串，而另一個運算式則用來在服務將字串放回訊息字串之前變更字串。<br/>                                             |
| [**patternMaps (NamedQueryType)**](eventmanifestschema-patternmaps-namedquerytype-element.md)                           | 定義用來改變訊息字串的正則運算式組清單。<br/>                                                                                                                                                                                     |
| [**提供者 (EventsType)**](eventmanifestschema-provider-eventstype-element.md)                                         | 定義提供者及其用來定義其事件的中繼資料。<br/>                                                                                                                                                                                                    |
| [**發行 (ChannelType)**](eventmanifestschema-publishing-channeltype-element.md)                                   | 為通道所使用的會話定義記錄屬性。<br/>                                                                                                                                                                                                     |
| [**資源 (LocalizationType)**](eventmanifestschema-resources-localizationtype-element.md)                           | 定義字串資料表的群組，其中包含您在資訊清單中參考的當地語系化字串。<br/>                                                                                                                                                                  |
| [**保留 (ChannelLoggingType)**](eventmanifestschema-retention-channelloggingtype-element.md)                       | 判斷記錄檔是否為連續或迴圈的記錄檔。 <br/>                                                                                                                                                                                                    |
| [**sidType (ChannelPublishingType)**](eventmanifestschema-sidtype-channelpublishingtype-element.md)                     | 決定是否要包含主體的安全識別碼 (SID) ，並將每個事件寫入通道。<br/>                                                                                                                                                        |
| [**字串 (StringTableType)**](eventmanifestschema-string-stringtabletype-element.md)                                   | 定義當地語系化的字串。<br/>                                                                                                                                                                                                                                               |
| [**stringTable (LocalizationType)**](eventmanifestschema-stringtable-localizationtype-element.md)                       | 定義您可以在資訊清單中參考的當地語系化字串清單。<br/>                                                                                                                                                                                              |
| [**stringTable (資源)**](eventmanifestschema-stringtable-localizationtype-element.md)                              | 定義您可以在資訊清單中參考的當地語系化字串清單。<br/>                                                                                                                                                                                              |
| [**結構 (TemplateItemType)**](eventmanifestschema-struct-templateitemtype-element.md)                                 | 定義結構，其中包含一個或多個您想要包含在事件中的資料項目。<br/>                                                                                                                                                                         |
| [**task (DefinitionType)**](eventmanifestschema-task-definitiontype-element.md)                                         | 無法使用。**Windows Server 2008 和 Windows Vista：** 定義您的提供者可以記錄的工作特定事件。 從 Windows 7 開始無法使用。<br/> <br/>                                                                                                     |
| [**task (TaskListType)**](eventmanifestschema-task-tasklisttype-element.md)                                             | 定義應用程式的元件或子元件。<br/>                                                                                                                                                                                                                    |
| [**工作 (ProviderType)**](eventmanifestschema-tasks-providertype-element.md)                                           | 定義提供者可用來群組事件的工作清單。<br/>                                                                                                                                                                                                          |
| [**範本 (TemplateListType)**](eventmanifestschema-template-templatelisttype-element.md)                             | 此範本會定義要包含在事件中的資料。<br/>                                                                                                                                                                                                                |
| [**範本 (ProviderType)**](eventmanifestschema-templates-providertype-element.md)                                   | 定義範本清單，以指定要包含在事件中的資料。<br/>                                                                                                                                                                                             |
| [**類型 (MetadataType)**](eventmanifestschema-types-metadatatype-element.md)                                           | 定義 XML 類型的清單。<br/>                                                                                                                                                                                                                                              |
| [**UserData (TemplateItemType)**](eventmanifestschema-userdata-templateitemtype-element.md)                             | 用來呈現事件資料的 XML 片段。<br/>                                                                                                                                                                                                                    |
| [**MapType) 的 valueMap (**](eventmanifestschema-valuemap-maptype-element.md)                                               | 定義對應整數值和字串值的名稱/值組清單。<br/>                                                                                                                                                                                             |
| [**xmlType (XmlTypeListType)**](eventmanifestschema-xmltype-xmltypelisttype-element.md)                                 | 定義 XML 類型。<br/>                                                                                                                                                                                                                                                      |
| [**xmlTypes (TypeListType)**](eventmanifestschema-xmltypes-typelisttype-element.md)                                     | 定義輸出類型的清單，此清單可供服務用來判斷如何呈現輸入資料類型。<br/>                                                                                                                                                                       |
