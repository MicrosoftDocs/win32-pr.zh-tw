# <a name="localization-instrumentationmanifest-element"></a>當地語系化 (instrumentationManifest) 元素

定義您在資訊清單中參考的當地語系化資源群組。

``` syntax
<xs:element name="localization"
    type="LocalizationType"
 />
```

**當地語系化** 元素是由 [**instrumentationManifest**](eventmanifestschema-instrumentationmanifest-element.md)元素所定義。
