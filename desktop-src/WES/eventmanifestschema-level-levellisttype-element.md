# <a name="level-levellisttype-element"></a>層級 (LevelListType) 元素

定義嚴重性值，以判斷記錄期間事件的詳細程度。

``` syntax
<xs:element name="level"
    type="LevelType"
 />
```

**Level** 元素是由 [**LevelListType**](eventmanifestschema-levellisttype-complextype.md)複雜型別定義。
