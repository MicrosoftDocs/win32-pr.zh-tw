# <a name="importchannel-channellisttype-element"></a>importChannel (ChannelListType) 元素

識別已由另一個提供者定義的通道。

``` syntax

    type="ImportChannelType"
 />
```

**ImportChannel** 元素是由 [**ChannelListType**](eventmanifestschema-channellisttype-complextype.md)複雜型別定義。
