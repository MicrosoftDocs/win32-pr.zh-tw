# <a name="eventpayload-processingerrordatatype-element"></a>EventPayload (ProcessingErrorDataType) 元素

包含事件的二進位事件資料，此事件會在處理事件資料時造成錯誤。

``` syntax
<xs:element name="EventPayload"
    type="hexBinary"
 />
```

**EventPayload** 元素是由 [**ProcessingErrorDataType**](eventschema-processingerrordatatype-complextype.md)複雜型別定義。
