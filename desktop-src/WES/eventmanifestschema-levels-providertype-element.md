# <a name="levels-providertype-element"></a>層級 (ProviderType) 元素

定義指定事件嚴重性的層級清單。

``` syntax

    type="LevelListType"
 />
```

**層級** 元素是由 [**ProviderType**](eventmanifestschema-providertype-complextype.md)複雜型別定義。
