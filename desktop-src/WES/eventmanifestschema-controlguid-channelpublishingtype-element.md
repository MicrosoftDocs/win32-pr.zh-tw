# <a name="controlguid-channelpublishingtype-element"></a>controlGuid (ChannelPublishingType) 元素

識別包含 WPP 事件之 ETW 會話的會話 GUID。

``` syntax
<xs:element name="controlGuid"
    type="GUIDType"
 />
```

**ControlGuid** 元素是由 [**ChannelPublishingType**](eventmanifestschema-channelpublishingtype-complextype.md)複雜型別定義。
