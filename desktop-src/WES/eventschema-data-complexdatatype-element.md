# <a name="data-complexdatatype-element"></a>資料 (ComplexDataType) 元素

結構中資料項目的清單。 專案清單的順序與範本中所定義的順序相同。

``` syntax
<xs:element name="Data"
    type="DataType"
 />
```

**資料** 元素是由 [**ComplexDataType**](eventschema-complexdatatype-complextype.md)複雜型別定義。
