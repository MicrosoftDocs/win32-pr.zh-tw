# <a name="opcodes-metadatatype-element"></a>opcode (MetadataType) 元素

定義可用於將工作中的事件群組在一起的作業碼清單。

``` syntax
<xs:element name="opcodes"
    type="OpcodeListType"
 />
```

Opcode **元素是** 由 [**MetadataType**](eventmanifestschema-metadatatype-complextype.md) 複雜型別定義。
