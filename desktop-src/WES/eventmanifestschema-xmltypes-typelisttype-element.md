# <a name="xmltypes-typelisttype-element"></a>xmlTypes (TypeListType) 元素

定義輸出類型的清單，此清單可供服務用來判斷如何呈現輸入資料類型。

``` syntax
<xs:element name="xmlTypes"
    type="XmlTypeListType"
 />
```

**XmlTypes** 元素是由 [**TypeListType**](eventmanifestschema-typelisttype-complextype.md)複雜型別定義。
