# <a name="levels-metadatatype-element"></a>層級 (MetadataType) 元素

定義指定事件嚴重性的層級清單。

``` syntax
<xs:element name="levels"
    type="LevelListType"
 />
```

**層級** 元素是由 [**MetadataType**](eventmanifestschema-metadatatype-complextype.md)複雜型別定義。
