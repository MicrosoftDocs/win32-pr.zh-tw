# <a name="query-schema-complex-types"></a>查詢架構複雜類型

以下是查詢架構定義的複雜類型。



| 複雜類型                                                   | Description                                                                                                                                                         |
|----------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**QueryListType**](queryschema-querylisttype-complextype.md) | 定義查詢清單，其中可包含一組選取器和 suppressors，用來在結果集中包含或排除事件。 <br/> |
| [**QueryType**](queryschema-querytype-complextype.md)         | 定義一組選取器和抑制器查詢，用來在結果集中包含或排除事件。<br/>                               |
