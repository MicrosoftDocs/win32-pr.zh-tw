# <a name="processingerrordata-eventtype-element"></a>ProcessingErrorData (的事件) 元素

包含嘗試呈現事件時所發生之錯誤的詳細資料。

``` syntax
<xs:element name="ProcessingErrorData"
    type="ProcessingErrorDataType"

```

**ProcessingErrorData** 元素是由「型別 [**複雜型**](eventschema-eventtype-complextype.md)別」（element）所定義。
