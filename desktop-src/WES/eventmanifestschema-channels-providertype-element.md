# <a name="channels-providertype-element"></a>通道 (ProviderType) 元素

定義提供者可以記錄事件的通道清單。

``` syntax
<xs:element name="channels"
    type="ChannelListType"
 />
```

**通道** 元素是由 [**ProviderType**](eventmanifestschema-providertype-complextype.md)複雜型別定義。
