# <a name="bitmap-maptype-element"></a>點陣圖 (MapType) 元素

定義對應位值和字串值的名稱/值組清單。

``` syntax
<xs:element name="bitMap"
    type="BitMapType"
 />
```

**BitMap** 元素是由 [**MapType**](eventmanifestschema-maptype-complextype.md)複雜型別定義。
