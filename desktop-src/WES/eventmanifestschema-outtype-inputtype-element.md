# <a name="outtype-inputtype-element"></a>outType (InputType) 元素

決定如何呈現資料的輸出資料類型清單。

``` syntax
<xs:element name="outType"
    type="OutputType"
 />
```

**OutType** 元素是由 [**InputType**](eventmanifestschema-inputtype-complextype.md)複雜型別定義。
