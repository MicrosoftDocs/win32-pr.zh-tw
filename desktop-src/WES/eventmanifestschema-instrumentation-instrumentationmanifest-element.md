# <a name="instrumentation-instrumentationmanifest-element"></a>檢測 (instrumentationManifest) 元素

定義資訊清單包含的檢測類型 (例如事件或計數器檢測) 。

``` syntax
<xs:element name="instrumentation"
    type="InstrumentationType"
 />
```

**檢測** 元素是由 [**instrumentationManifest**](eventmanifestschema-instrumentationmanifest-element.md)元素所定義。
