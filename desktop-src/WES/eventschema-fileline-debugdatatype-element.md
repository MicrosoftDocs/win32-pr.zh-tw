# <a name="fileline-debugdatatype-element"></a>FileLine (DebugDataType) 元素

來源檔案的名稱，以及記錄追蹤訊息之原始程式檔內的行。

``` syntax
<xs:element name="FileLine"
    type="string"
 />
```

**FileLine** 元素是由 [**DebugDataType**](eventschema-debugdatatype-complextype.md)複雜型別定義。
