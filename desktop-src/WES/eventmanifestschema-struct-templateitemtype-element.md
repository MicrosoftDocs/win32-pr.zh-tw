# <a name="struct-templateitemtype-element"></a>結構 (TemplateItemType) 元素

定義結構，其中包含一個或多個您想要包含在事件中的資料項目。

``` syntax
<xs:element name="struct"
    type="StructDefinitionType"
 />
```

**Struct** 元素是由 [**TemplateItemType**](eventmanifestschema-templateitemtype-complextype.md)複雜型別定義。
