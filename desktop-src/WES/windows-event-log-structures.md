Windows事件記錄檔會定義下列結構。



| 結構                                | 描述                                                    |
|------------------------------------------|----------------------------------------------------------------|
| [**.EVT \_ RPC \_ 登入**](/windows/desktop/api/WinEvt/ns-winevt-evt_rpc_login) | 包含用來連接到遠端電腦的資訊。 |
| [**.EVT \_ 變異**](/windows/desktop/api/WinEvt/ns-winevt-evt_variant)      | 包含事件資料或屬性值。                        |
