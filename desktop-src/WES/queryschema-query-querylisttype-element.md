# <a name="query-querylisttype-element"></a>查詢 (QueryListType) 元素

定義一組選取器和 suppressors，用來在結果集中包含或排除事件。

``` syntax
<xs:element name="Query"
    type="QueryType"
 />
```

**查詢** 元素是由 [**QueryListType**](queryschema-querylisttype-complextype.md)複雜型別定義。
