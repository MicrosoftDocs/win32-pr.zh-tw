# <a name="patternmaps-namedquerytype-element"></a>patternMaps (NamedQueryType) 元素

定義用來改變訊息字串的正則運算式組清單。

``` syntax
<xs:element name="patternMaps"
    type="PatternMapListType"
 />
```

**PatternMaps** 元素是由 [**NamedQueryType**](eventmanifestschema-namedquerytype-complextype.md)複雜型別定義。
