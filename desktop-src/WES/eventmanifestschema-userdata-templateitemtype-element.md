# <a name="userdata-templateitemtype-element"></a>UserData (TemplateItemType) 元素

用來呈現事件資料的 XML 片段。

``` syntax
<xs:element name="UserData"
    type="XmlType"
 />
```

**UserData** 元素是由 [**TemplateItemType**](eventmanifestschema-templateitemtype-complextype.md)複雜型別定義。
