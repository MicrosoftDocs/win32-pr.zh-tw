# <a name="opcodes-providertype-element"></a>opcode (ProviderType) 元素

定義可用於將工作中的事件群組在一起的作業碼清單。

``` syntax
<xs:element name="opcodes"
    type="OpcodeListType"
 />
```

Opcode **元素是** 由 [**ProviderType**](eventmanifestschema-providertype-complextype.md) 複雜型別定義。
