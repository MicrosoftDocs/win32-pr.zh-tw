# <a name="event-definitiontype-element"></a>事件 (DefinitionType) 元素

定義您的提供者可以記錄的事件。

``` syntax
<xs:element name="event"
    type="EventDefinitionType"
 />
```

**事件** 元素是由 [**DefinitionType**](eventmanifestschema-definitiontype-complextype.md)複雜型別定義。
