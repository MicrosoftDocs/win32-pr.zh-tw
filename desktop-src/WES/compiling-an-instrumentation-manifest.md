# <a name="compiling-an-instrumentation-manifest"></a>編譯檢測資訊清單

[寫入資訊清單](writing-an-instrumentation-manifest.md)之後，請使用訊息編譯器來驗證資訊清單，並產生您在提供者中包含的資源和標頭檔。 如需使用編譯器的詳細資訊，請參閱 [**訊息編譯器**](message-compiler--mc-exe-.md)。 包含編譯器在專案中所產生的標頭和資源檔，其中包含提供者的其餘原始程式碼。 如需有關如何寫入資訊清單中所定義之事件的詳細資訊，請參閱 [開發提供者](developing-a-provider.md)。
