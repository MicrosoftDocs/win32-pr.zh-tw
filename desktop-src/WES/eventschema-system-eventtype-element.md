# <a name="system-eventtype-element"></a>系統 (的) 元素

包含識別提供者及其啟用方式、事件、寫入事件之通道的資訊，以及處理常式和執行緒識別碼之類的系統資訊。

``` syntax
<xs:element name="System"
    type="SystemPropertiesType"
 />
```

**系統** 專案是由「[**事件**](eventschema-eventtype-complextype.md)類型」複雜類型所定義。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|-------------------------------------|------------------------------------------------------|

| 最低支援的伺服器<br/> | Windows\[僅限 Server 2008 desktop 應用程式\]<br/> |
