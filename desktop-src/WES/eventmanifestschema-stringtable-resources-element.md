# <a name="stringtable-resources-element"></a>stringTable (資源) 元素

定義您可以在資訊清單中參考的當地語系化字串清單。

``` syntax
<xs:element name="stringTable"
    type="StringTableType"
 />
```

**StringTable** 元素是由 [**resources**](eventmanifestschema-resources-localizationtype-element.md)元素所定義。
