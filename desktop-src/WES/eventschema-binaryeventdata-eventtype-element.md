# <a name="binaryeventdata-eventtype-element"></a>BinaryEventData (的事件) 元素

以二進位 blob 的形式包含事件資料。

``` syntax
<xs:element name="BinaryEventData"
    type="hexBinary"
 />
```

**BinaryEventData** 元素是由「型別 [**複雜型**](eventschema-eventtype-complextype.md)別」（element）所定義。
