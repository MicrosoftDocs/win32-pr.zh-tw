# <a name="intypes-typelisttype-element"></a>inTypes (TypeListType) 元素

包含輸入類型定義的清單。

``` syntax
<xs:element name="inTypes"
    type="InputTypeListType"
 />
```

**InTypes** 元素是由 [**TypeListType**](eventmanifestschema-typelisttype-complextype.md)複雜型別定義。
