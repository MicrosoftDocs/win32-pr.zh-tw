# <a name="tasks-providertype-element"></a>工作 (ProviderType) 元素

定義提供者可用來群組事件的工作清單。

``` syntax
<xs:element name="tasks"
    type="TaskListType"
 />
```

**Tasks** 元素是由 [**ProviderType**](eventmanifestschema-providertype-complextype.md)複雜型別定義。
