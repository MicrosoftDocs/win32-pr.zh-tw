# <a name="autobackup-channelloggingtype-element"></a>autoBackup (ChannelLoggingType) 元素

決定當目前的記錄檔達到大小上限時，是否要建立新的記錄檔。

``` syntax
<xs:element name="autoBackup"
    type="boolean"
 />
```

**AutoBackup** 元素是由 [**ChannelLoggingType**](eventmanifestschema-channelloggingtype-complextype.md)複雜型別定義。

## <a name="requirements"></a>規格需求



| 需求 | 值 |
|-------------------------------------|------------------------------------------------------|
| 最低支援的用戶端<br/> | \[僅限 Windows Vista 桌面應用程式\]<br/>       |
| 最低支援的伺服器<br/> | 僅限 Windows Server 2008 \[ desktop 應用程式\]<br/> |
