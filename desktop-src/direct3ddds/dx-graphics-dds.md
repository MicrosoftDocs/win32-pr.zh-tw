# <a name="dds"></a>DDS

)  ( 的 DirectDraw 介面檔格式，會隨著 DirectX 7 引進，以儲存未壓縮和壓縮的 (DXTn) 紋理。 檔案格式支援 mipmap、cube 對應和磁片區對應。 [DirectXTex](https://github.com/Microsoft/DirectXTex)、 [DIRECTXTK](https://github.com/Microsoft/DirectXTK)、舊版 D3DX 和其他 DIRECTX 工具都支援 DDS 檔案格式。 從 Direct3D 10 開始，DDS 檔案支援材質陣列。

> [!Note]  
> D3DX (D3DX 9、D3DX 10 和[D3DX 11](/windows/desktop/direct3d11/d3d11-graphics-reference-d3dx11)) 公用程式庫已針對 Windows 8 淘汰，不支援 Windows Store 應用程式。 建議您利用 [DirectXTex](https://github.com/Microsoft/DirectXTex)、 [DirectXTK](https://github.com/Microsoft/DirectXTK)或兩者。
