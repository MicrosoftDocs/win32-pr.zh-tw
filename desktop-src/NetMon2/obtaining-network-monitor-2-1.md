# <a name="obtaining-network-monitor-21"></a>取得網路監視器2。1

網路監視器2.1 是以 Microsoft Windows Systems Management Server (SMS) 2003 或 Windows Server 2003 媒體上個別安裝的形式提供。

> [!Note]  
> 目前，網路監視器2.1 無法當作 Windows SDK 的一部分來使用;也可以在 Windows Vista 中使用。

 

## <a name="microsoft-windows-systems-management-server-2003"></a>Microsoft Windows Systems Management Server 2003

從 \\ \\ SMS 2003 產品 CD 上的 "NETMON I386" 資料夾執行 "NetmonSetup.exe"，以安裝網路監視器2.1。

如果您要從 SMS 2.0 進行升級，則必須先使用主控台中的 [ **新增或移除程式** ] 卸載網路監視器2.0。

## <a name="microsoft-windows-server-2003"></a>Microsoft Windows Server 2003

請遵循下列指示，從 Windows Server 2003 媒體安裝網路監視器2.1。

-   按一下 [ **開始**]，指向 [ **設定** ]，然後選取 [ **主控台** ] 功能表項目。
-   按兩下 [ **新增/移除程式**]。
-   按一下 [ **新增/移除 Windows 元件**]。
-   按一下 [ **管理和監視工具**]，然後按一下 [ **詳細資料**]。
-   按一下以選取 [ **網路監視器工具** ] 核取方塊，然後按一下 **[確定]**。
-   按一下 [下一步] 。
