# <a name="clipping-a-frame"></a>裁剪框架

您可以指定驅動程式裁剪框架。  (如果省略其他 capture 篩選區段，這可能是您的捕獲篩選) 的唯一功能。 如果 **nFrameBytesToCopy** 欄位不是零 (0) ，其值會指定要保留的每個框架的位元組數目。 如果欄位為零，則會保留整個框架。

> [!Note]  
> 您可以藉由設定 **nFrameBytesToCopy** 來裁剪任何通過捕捉篩選器其他部分的畫面格。
