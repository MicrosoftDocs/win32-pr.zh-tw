# <a name="d-network-monitor"></a>D (網路監視器) 

<dl> <dt>

<span id="_netmon_delayed_capture_gly"></span><span id="_NETMON_DELAYED_CAPTURE_GLY"></span>**延遲的捕獲**
</dt> <dd>

收集的網路流量會儲存在捕獲檔案中。 網路封包提供者 (NPP) 會執行延遲的捕獲。 剖析器和專家使用延遲的捕捉。 另請參閱 [*capture*](c.md)和 [*即時 capture*](/windows)

</dd> <dt>

<span id="_netmon_destination_address_gly"></span><span id="_NETMON_DESTINATION_ADDRESS_GLY"></span>**目的地位址**
</dt> <dd>

要將資料傳送至其中的電腦位址。

</dd> <dt>

<span id="_netmon_detail_pane_gly"></span><span id="_NETMON_DETAIL_PANE_GLY"></span>**詳細資料窗格**
</dt> <dd>

網路監視器框架視窗中的窗格，會顯示有關通訊協定屬性的詳細資訊。 依預設，此窗格會顯示在視窗的中央。

</dd> <dt>

<span id="_netmon_display_filter_gly"></span><span id="_NETMON_DISPLAY_FILTER_GLY"></span>**顯示篩選**
</dt> <dd>

用來限制 [框架檢視器] 視窗中所顯示之已捕捉資料量的篩選準則。 顯示篩選是物件的後置堆疊，可以是運算子、值或屬性。

</dd> </dl>
