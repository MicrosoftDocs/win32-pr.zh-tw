# <a name="configuring-experts"></a>設定專家

若要設定專家，請使用自訂對話方塊和 [**設定**](configure.md) 函數。 當您撰寫網路監視器專家時，您可以選擇。 您可以建立一組預設的設定專案，以在專家一開始執行時使用，或者您可以讓網路監視器的使用者在執行時間設定專家。 每個專家的設定都是由網路監視器儲存。 使用者可以選擇將專家設定重設回原始的預設設定。 請注意，您可以藉由開發將固定設定傳遞至網路監視器的專家，來略過 **設定** 的使用。
