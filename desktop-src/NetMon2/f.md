# <a name="f-network-monitor"></a>F (網路監視器) 

<dl> <dt>

<span id="_netmon_filter_blob_gly"></span><span id="_NETMON_FILTER_BLOB_GLY"></span>**篩選 BLOB**
</dt> <dd>

二進位大型物件 (BLOB) 用來限制 (NIC) 的網路介面卡搜尋。 篩選 BLOB 中的每個專案都必須符合 (NPP) BLOB （代表 NIC）的網路封包提供者中的對等專案。 例如，應用程式只能要求支援特定畫面播放速率的乙太網路卡或卡片。 您可以使用篩選 BLOB 來呼叫 **GetNPPBlobFromUI** 和 **GetNPPBlobTable** 函數。

</dd> <dt>

<span id="_netmon_follow_set_gly"></span><span id="_NETMON_FOLLOW_SET_GLY"></span>**後續設定**
</dt> <dd>

遵循通訊協定的通訊協定組。 如果剖析器無法從捕捉中的資料偵測到下一個通訊協定，網路監視器會使用下列一組通訊協定。 另請參閱 [*移交集*](h.md)

</dd> <dt>

<span id="_netmon_frame_gly"></span><span id="_NETMON_FRAME_GLY"></span>**框架**
</dt> <dd>

透過網路傳輸為一個單位的資料封包。 每個畫面格都會遵循相同的基本組織，並包含控制資訊，例如同步處理字元、站位址、錯誤檢查值和變動量的資料。 也稱為封包。

</dd> <dt>

<span id="_netmon_frame_viewer_window_gly"></span><span id="_NETMON_FRAME_VIEWER_WINDOW_GLY"></span>**框架檢視器視窗**
</dt> <dd>

顯示個別畫面格詳細資訊的網路監視器視窗。 您可以一次開啟數個框架檢視器視窗。

</dd> </dl>
