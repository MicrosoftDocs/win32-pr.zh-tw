# <a name="specifying-protocol-information"></a>指定通訊協定資訊

網路監視器驅動程式可以將資訊新增至框架表格，方法是指定找到的低層級通訊協定 (IP 或 IPX) ，以及通訊協定開頭的位移。 若要將此資訊新增至框架表格，請設定任何捕捉篩選。
