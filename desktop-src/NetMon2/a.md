# <a name="a-network-monitor"></a> (網路監視器) 

<dl> <dt>

<span id="_netmon_address_gly"></span><span id="_NETMON_ADDRESS_GLY"></span>**位址**
</dt> <dd>

網路節點的識別碼，可區分網路上的一個節點。 也稱為 *網路位址* 或 *MAC 位址*。 在 Microsoft Systems Management Server (SMS) 中，會使用位址來連接網站和網站系統。 傳送者會使用特定位址將指令和資料傳送至其他網站。

</dd> <dt>

<span id="_netmon_address_pair_gly"></span><span id="_NETMON_ADDRESS_PAIR_GLY"></span>**位址配對**
</dt> <dd>

資料傳輸的來源和目的地位址。 目的地可以是單一位址、多播或廣播。 在捕捉的資料中，來源位址是特定的位址。 在篩選準則規格中，來源位址可以是特定位址或任何位址，但不能是廣播或多播。

</dd> </dl>
