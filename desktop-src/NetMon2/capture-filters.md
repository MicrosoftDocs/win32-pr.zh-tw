# <a name="capture-filters"></a>捕獲篩選

網路監視器 SDK 包含 capture 篩選器，可讓您限制網路封包提供者 (NPP) 應用程式在其作業期間收到的框架數目。 藉由指定要讓自訂監視或 NPP 應用程式檢查的網路流量框架，您可以改善其處理效率。

本節包含下列主題：

-   [關於 Capture 篩選](about-capture-filters.md)
-   [**CAPTUREFILTER 結構**](the-capturefilter-structure.md)
