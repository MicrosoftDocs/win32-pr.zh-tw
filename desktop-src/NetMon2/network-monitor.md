# <a name="network-monitor"></a>網路監視器

## <a name="purpose"></a>目的

網路監視器會捕捉顯示和分析的網路流量。 它可讓您執行工作，例如分析先前在使用者定義方法中所捕獲的資料，以及從定義的通訊協定剖析器中提取資料。

## <a name="developer-audience"></a>開發人員對象

您可以使用 C/c + + 存取網路監視器所提供的所有 API 集。 也可以使用網路封 [*包提供者*](n.md) 的開發人員也必須熟悉 COM。

## <a name="run-time-requirements"></a>執行階段需求求

若要從網路監視器 API 呼叫，您必須在 Windows NT Server 4.0、Windows 2000 Server 或 Windows Server 2003 上執行，或已安裝 Microsoft Systems Management Server。

NPP 驅動程式和支援的功能包括所有版本的 Windows NT 4.0 和 Windows 2000 伺服器。

> [!Note]  
> Windows Vista 和更新版本不支援網路監視器2.1 及更早版本。
