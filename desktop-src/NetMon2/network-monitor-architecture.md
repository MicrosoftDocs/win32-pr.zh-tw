# <a name="network-monitor-architecture"></a>網路監視器架構

網路監視器的架構可分成兩組程式設計概念 [*，也就*](p.md)是與 [*專家*](e.md)和剖析器開發相關的程式，以及與 [*監視*](m.md)開發相關的概念。

雖然這兩個元件都使用相同的元件來捕捉網路流量，但它們各自使用不同的介面來捕捉資料和不同的元件來分析所捕獲的內容。
