# <a name="network-monitor-introduction"></a>網路監視器簡介

平臺軟體發展工具組 (SDK 的網路監視器部分) 藉由協助您建立自訂專家、通訊協定剖析器和網路封包提供者 (NPPs) ，來擴充網路監視器的功能。 本節包括下列檔：

-   建立專家和剖析器所需的程式設計主題。 此資訊也會說明如何撰寫 capture 篩選器，並使用 Blob 和 NPPs。
-   與專家、剖析器、Blob 和 NPPs 相關的 API 相關參考資訊。
-   網路監視器檔中使用的詞彙。

提供下列子主題以協助取得和安裝網路監視器。

-   [系統需求](system-requirements.md)
-   [取得網路監視器2。1](obtaining-network-monitor-2-1.md)
-   [安裝網路監視器檔案](installing-network-monitor-files.md)
