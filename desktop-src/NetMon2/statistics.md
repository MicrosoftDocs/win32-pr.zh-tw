# <a name="statistics-structure"></a><span data-ttu-id="dfcdd-104">統計資料結構</span><span class="sxs-lookup"><span data-stu-id="dfcdd-104">STATISTICS structure</span></span>

<span data-ttu-id="dfcdd-105">**統計資料** 結構會提供 capture 的統計資料。</span><span class="sxs-lookup"><span data-stu-id="dfcdd-105">The **STATISTICS** structure provides statistics for the capture.</span></span> <span data-ttu-id="dfcdd-106">其中某些統計資料是由網路監視器產生的，有些則是由 NPP 所連接的 NIC 所產生。</span><span class="sxs-lookup"><span data-stu-id="dfcdd-106">Some of these statistics are generated by Network Monitor, while others are generated by the NIC that the NPP is connected to.</span></span>

## <a name="syntax"></a><span data-ttu-id="dfcdd-107">語法</span><span class="sxs-lookup"><span data-stu-id="dfcdd-107">Syntax</span></span>


```C++
typedef struct _STATISTICS {
  __int64 TimeElapsed;
  DWORD   TotalFramesCaptured;
  DWORD   TotalBytesCaptured;
  DWORD   TotalFramesFiltered;
  DWORD   TotalBytesFiltered;
  DWORD   TotalMulticastsFiltered;
  DWORD   TotalBroadcastsFiltered;
  DWORD   TotalFramesSeen;
  DWORD   TotalBytesSeen;
  DWORD   TotalMulticastsReceived;
  DWORD   TotalBroadcastsReceived;
  DWORD   TotalFramesDropped;
  DWORD   TotalFramesDroppedFromBuffer;
  DWORD   MacFramesReceived;
  DWORD   MacCRCErrors;
  __int64 MacBytesReceivedEx;
  DWORD   MacFramesDropped_NoBuffers;
  DWORD   MacMulticastsReceived;
  DWORD   MacBroadcastsReceived;
  DWORD   MacFramesDropped_HwError;
} STATISTICS, *LPSTATISTICS;
```
