# <a name="system-requirements"></a>系統需求

電腦設定的最小值不同;針對網路監視器開發用途，建議開發人員擁有一部包含下列最低規格的電腦：

-   Pentium 處理器
-   32 MB RAM
-   30 MB 的可用硬碟空間
-   800 X 600 解析度的視頻監視器和視訊卡支援256色彩
-   網路連線
-   Microsoft Internet Explorer 4.01
-   Microsoft Visual C++ 版本5。0

測試使用網路監視器開發的應用程式需要具備下列最低規格的電腦：

-   486處理器至少執行 50 MHz
-   16 MB RAM
-   20 MB 可用硬碟空間
-   網路連線
