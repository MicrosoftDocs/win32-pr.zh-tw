# <a name="g-network-monitor"></a>G (網路監視器) 

<dl> <dt>

<span id="_netmon_generic_formatter_gly"></span><span id="_NETMON_GENERIC_FORMATTER_GLY"></span>**一般格式器**
</dt> <dd>

網路監視器提供的屬性資料格式器。 一般格式器可以格式化網路監視器 UI 的詳細資料窗格中所顯示的資料。 當您想要網路監視器使用一般格式器時，必須在屬性之 **PROPERTYINFO** 結構的 **InstanceData** 成員中指定 **FormatPropertyInstance** 函數。 另請參閱 [一般格式器輸出](generic-formatter-output.md)

</dd> </dl>
