# <a name="installing-network-monitor-files"></a>安裝網路監視器檔案

若要安裝並執行平臺軟體發展工具組 (SDK) 的網路監視器部分，請依照此程式進行操作。

**安裝和執行網路監視器 Platform SDK 檔案**

1.  若要在安裝 Platform SDK 期間安裝檔案，請選取網路監視器檔和範例程式碼。

    當安裝程式完成時，Platform SDK 安裝程式會建立批次檔 Setenv.bat。

2.  若要初始化必要的環境變數，請執行 Setenv.bat。
3.  若要編譯範例，請執行包含在 Microsoft Visual Studio 中的 **nmake**。
