# <a name="t-network-monitor"></a>T (網路監視器) 

<dl> <dt>

<span id="_netmon_total_statistics_gly"></span><span id="_NETMON_TOTAL_STATISTICS_GLY"></span>**統計資料總計**
</dt> <dd>

網路介面卡 (NIC) 報表和網路監視器產生的統計資料。 例如，網路畫面格的統計資料包括 NIC 所報告的框架數目、NIC 處理的畫面格數、透過 capture 篩選傳遞的框架數，以及儲存在緩衝區或 capture 檔中的框架數目。 如需這些統計資料的完整清單，請參閱 **統計資料** 結構。

</dd> </dl>
