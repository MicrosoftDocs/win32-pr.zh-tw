# <a name="monitor-interfaces"></a>監視介面

下列 COM 介面是用來開發監視器。



| 介面                              | 描述                                                                                                                                             |
|----------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [IMonitor](imonitor.md)               | 提供控制監視操作的方法。 此介面必須由監視器所執行。                                                     |
| [IMonitorEventer](imonitoreventer.md) | 提供用來傳送事件的方法。 此介面是由 [*監視器控制服務*](m.md) (MCSVC) 所提供。 |
