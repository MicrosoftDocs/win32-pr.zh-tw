# <a name="about-network-monitor-21"></a>關於網路監視器2。1

網路監視器是 Microsoft Systems Management Server (SMS) 的元件，可用來偵測執行 Microsoft 遠端存取服務器 (RAS) 的 Lan、Wan 和序列連結上的問題，並進行疑難排解。 網路監視器提供預先捕獲網路資料分析。

在後置分析中，會將網路流量儲存在專屬的捕獲檔案中，以便稍後分析已捕捉的資料。 在此情況下，分析可以 [*是通訊協定*](p.md) 剖析器挑選特定網路框架類型，並在網路監視器 UI 中顯示框架資料的形式;或分析的形式可以是檢查網路資料的 [*專家*](e.md) ，以及顯示報表 (專家也可以操控網路資料) 。

網路監視器提供下列類型的功能：

-   以即時或延遲模式捕捉網路資料。
-   在捕獲資料時提供篩選功能。
-   使用專家和剖析器進行詳細的後續捕獲分析。
