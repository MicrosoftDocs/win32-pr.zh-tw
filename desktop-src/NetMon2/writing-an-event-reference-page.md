# <a name="writing-an-event-reference-page"></a>撰寫事件參考頁面

使用事件參考頁面 (ERPs) ，以描述如何解決專家所偵測到的問題。 因為 ERPs 包含 HTML 文字，所以您可以使用任何 HTML 編輯器或文字編輯器來建立一個。

顯示 ERP 的網路監視器事件瀏覽器窗格，支援 Microsoft Internet Explorer 4.01 和更新版本的所有功能。 在您的 ERP 中內嵌任何支援的控制項或指令碼語言，以顯示資訊或從 ERP 啟動其他功能。 例如，您可以建立一個將電子郵件傳送到網路作業的控制項。

如需有關建立 ERP 的詳細資訊，請參閱下列主題：

-   [建立專家和監視事件參考頁面](creating-expert-and-monitor-event-reference-pages.md)
-   [命名事件參考頁面](naming-an-event-reference-page.md)
-   [測試事件參考頁面](testing-an-event-reference-page.md)
