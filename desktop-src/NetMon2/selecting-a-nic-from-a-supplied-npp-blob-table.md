# <a name="selecting-a-nic-from-a-supplied-npp-blob-table"></a>從提供的 NPP BLOB 資料表選取 NIC

若要從提供的 NPP BLOB 資料表中選取 (NIC) 的網路介面卡，請呼叫 [**SelectNPPBlobFromTable**](selectnppblobfromtable.md) 函數。 此函式會使用網路監視器 UI 來顯示 BLOB 資料表中所表示的 Nic，並傳回代表所選 NIC 的 NPP BLOB。

當您呼叫 [**SelectNPPBlobFromTable**](selectnppblobfromtable.md) 函數時，會出現 [ **選取網路** ] 對話方塊。 您可以在此查看 NPP BLOB 資料表中所代表 NPPs 的詳細資料、選取特定的 NIC，並查看其相關聯 NPP BLOB 結構的詳細資料。
