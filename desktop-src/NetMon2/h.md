# <a name="h-network-monitor"></a>H (網路監視器) 

<dl> <dt>

<span id="_netmon_handoff_set_gly"></span><span id="_NETMON_HANDOFF_SET_GLY"></span>**遞交集**
</dt> <dd>

遵循通訊協定的通訊協定組。 如果剖析器可以從捕捉中的資料偵測下一個通訊協定，則剖析器會使用通訊協定的遞交集。 另請參閱 [*下列集合*](f.md)

</dd> <dt>

<span id="_netmon_handoff_table_gly"></span><span id="_NETMON_HANDOFF_TABLE_GLY"></span>**提交資料表**
</dt> <dd>

包含剖析器遞交集的資料表。 您可以使用「提交」表格來存取「遞交集」資訊，而不需要存取剖析器定義的 INI 檔案。 這些遞交集資訊會儲存在每個剖析器的 INI 檔案中。 另請參閱 *移交集*

</dd> <dt>

<span id="_netmon_hex_pane_gly"></span><span id="_NETMON_HEX_PANE_GLY"></span>**十六進位窗格**
</dt> <dd>

[網路監視器框架檢視器] 視窗中的窗格，會顯示在捕捉中包含的框架中傳送的十六進位位元組。 根據預設，[十六進位] 窗格會出現在視窗底部。

</dd> </dl>
