# <a name="testing-a-monitor"></a>測試監視

安裝監視器之後，您可以使用網路作業的即時資料進行測試，也可以測試已知的事件集，方法是使用網路監視器來捕捉資料、儲存捕獲檔案，然後將資料傳輸到本機。  (如需有關如何在網路上傳輸 capture 檔案的詳細資訊，請參閱網路監視器線上說明。 )
