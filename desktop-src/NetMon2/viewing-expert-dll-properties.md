# <a name="viewing-expert-dll-properties"></a>查看專家 DLL 屬性

您可以使用下列程式來查看專家 DLL 屬性。

**若要查看 DLL 屬性**

1.  開啟 Windows 檔案總管並移至 \\ 網路監視器 (C： \\ Program Files NetMon2 專家的專家 \\ 子目錄 \\ ，例如) 。
2.  選取所需的 DLL，然後以滑鼠右鍵按一下該檔案。
3.  按一下 [ **屬性** 和 **版本**]。 下列 Myexpert.dll 屬性狀態視窗範例會顯示 DLL 屬性。
4.  按一下所需的 **專案名稱** ，即可在 [ **值** ] 窗格中顯示資訊。
5.  按一下 [確定] 以關閉 [內容] 對話方塊。
