# <a name="setclassidinblob-function"></a>SetClassIDInBlob 函式

**SetClassIDInBlob** 函式會設定 BLOB 的命名類別識別碼值。

## <a name="syntax"></a>語法


```C++
DWORD SetClassIDInBlob(
  _In_       HBLOB hBlob,
  _In_ const char  *pOwnerName,
  _In_ const char  *pCategoryName,
  _In_ const char  *pTagName,
  _In_ const CLSID *pClsID
);
```



## <a name="parameters"></a>參數

<dl> <dt>

*hBlob* \[在\]
</dt> <dd>

BLOB 的控制碼。

</dd> <dt>

*pOwnerName* \[在\]
</dt> <dd>

已設定之 BLOB **擁有** 者名稱的指標。

</dd> <dt>

*pCategoryName* \[在\]
</dt> <dd>

已設定之 BLOB **類別目錄** 名稱的指標。

</dd> <dt>

*pTagName* \[在\]
</dt> <dd>

已設定之 BLOB **標記** 名稱的指標。

</dd> <dt>

*pClsID* \[在\]
</dt> <dd>

已設定之 BLOB 的類別識別碼指標。

</dd> </dl>

## <a name="return-value"></a>傳回值

如果函式成功，則傳回值為 NMERR \_ SUCCESS。

如果函式不成功，則傳回值會是指出錯誤的 NMERR 值。
