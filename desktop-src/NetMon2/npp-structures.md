# <a name="npp-structures"></a>NPP 結構

本節說明 NPP 方法所使用的 NPP 結構。 這些結構是用來取出統計資料、提供系統狀態和統計資訊，以及指出哪些電腦正在執行網路監視器。 下表描述這些結構。



| NPP 結構                     | Description                                                                                                                      |
|------------------------------------|----------------------------------------------------------------------------------------------------------------------------------|
| [SESSIONSTATS](sessionstats.md)   | 提供抓取交談統計資料時的會話資訊。                                                         |
| [STATIONSTATS](stationstats.md)   | 提供特定 [*工作站*](s.md)的相關統計資料。                                                     |
| [統計](statistics.md)       | 提供在抓取統計資料和目前的捕獲暫停或停止時的網路統計資料。              |
| [透視](querytable.md)       | 指出哪些電腦正在使用網路監視器。                                                                             |
| [STATIONQUERY](stationquery.md)   | 提供使用網路監視器之電腦的相關資訊。 此 **結構是由 NPP 的** 工作方式結構所使用。 |
| [>NETWORKSTATUS](networkstatus.md) | 根據系統狀態提供目前的網路狀態。                                                                        |
