# <a name="providing-network-statistics-and-packet-information"></a>提供網路統計資料和封包資訊

網路封包提供者 (NPP) 是與網路監視器驅動程式進行通訊的 DLL，並提供網路統計資料和封包資料。

使用 NPP 的最常見方式包括：

-   網路統計資料。
-   即時傳遞至應用程式的畫面格。
-   只有在停用 capture 之後才可存取的封包。

本節包含下列主題：

-   [選取 NIC](selecting-a-nic-using-getnppblobfromui.md)
-   [使用 GetNPPBlobTable](using-getnppblobtable.md)
