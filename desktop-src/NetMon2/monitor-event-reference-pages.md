# <a name="monitor-event-reference-pages"></a>監視事件參考頁面

您可以使用 (ERPs) 的事件參考頁面，將您的監視器連接到網路監視器的 UI。 您可以使用它們來警示終端使用者偵測到的狀況和問題，或使用它們來提供建議的修正或說明資料。
