# <a name="installing-existing-erps-to-network-monitor-21"></a>將現有的 ERPs 安裝至網路監視器2。1

在您將事件參考頁面寫入和命名 (ERP) 之後，您必須加以安裝。

**安裝事件參考頁面**

1.  建立 ERP 目錄。 這些範例示範專家的目錄：

    ``` syntax
    C:\Program Files\NetMon2\Bok\Expert\
    ```

2.  為每個會顯示 ERPs 的新專家建立一個子目錄。 下列子目錄清單適用于專家：

    ``` syntax
    C:\Program Files\NetMon2\Bok\Experts\MyExpert
    ```

3.  將 ERP 檔案複製到相關聯的子目錄。 下列範例提供已安裝專家 ERPs 的完整路徑：

    ``` syntax
    C:\Program Files\NetMon2\Bok\Experts\MyExpert\MyExpert1.htm
    ```
