# <a name="installing-an-expert-to-network-monitor-21"></a>安裝專家來網路監視器2。1

若要安裝專家 DLL，請將它複製到網路監視器 \\ 專家子目錄 (例如，C： \\ Program Files \\ NetMon2 \\ 專家) 。 安裝之後，將會在下一次網路監視器開始時，辨識專家並可供使用。

下圖顯示從範例程式碼建立並複製到專家子目錄的 MyExpert DLL （MyExpert） \\ 。
