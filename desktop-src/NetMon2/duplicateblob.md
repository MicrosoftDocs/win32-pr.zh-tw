# <a name="duplicateblob-function"></a>DuplicateBlob 函式

**DuplicateBlob** 函式會複製特定的 BLOB。

## <a name="syntax"></a>語法


```C++
DWORD DuplicateBlob(
  _In_  HBLOB hSrcBlob,
  _Out_ HBLOB *hBlobThatWillBeCreated
);
```



## <a name="parameters"></a>參數

<dl> <dt>

*hSrcBlob* \[在\]
</dt> <dd>

複製之 BLOB 的控制碼。

</dd> <dt>

*hBlobThatWillBeCreated* \[擴展\]
</dt> <dd>

複製 BLOB 的控制碼。

</dd> </dl>

## <a name="return-value"></a>傳回值

如果函式成功，則傳回值為 NMERR \_ SUCCESS。

如果函式不成功，則傳回值為描述錯誤的 NMERR 值。
