# <a name="installing-a-monitor-to-network-monitor"></a>安裝監視以網路監視器

安裝監視 DLL 是一個簡單的程式。 首先，將 DLL 和相關聯的 HTML 設定表單複製到 NPP 監視器子目錄 (例如，C： \\ Windows \\ System32 \\ NPP \\ 監視器) 。 複製到目錄時，系統會在您下次啟動 [監視控制] 工具或 [監視控制服務 (MCSVC) 時，辨識並使用監視器。

例如，如果環境變數% SystemRoot% 是 C： \\ windows，則監視子目錄是 c： \\ windows \\ System32 \\ Npp \\ 監視器。
