# <a name="touch-injection"></a>觸控插入

## <a name="purpose"></a>目的

本節中的主題提供 Windows 應用程式之觸控插入的總覽。 [觸控式插入](/windows/win32/api/_input_touchinjection/)api 可讓 Windows 開發人員以程式設計方式模擬觸控輸入。
