# <a name="device-access-api"></a>裝置存取 API

## <a name="purpose"></a>目的

您可以使用裝置存取 API 來撰寫使用自訂驅動程式之特製化裝置的 Windows Store 裝置應用程式。 API 會提供方法來傳送控制程式代碼，以與裝置的自訂驅動程式進行通訊。

## <a name="developer-audience"></a>開發人員對象

裝置存取 API 是專為獨立硬體廠商 (IHV) 以及熟悉 c + + 和元件物件模型 (COM) 的 OEM 開發人員所設計。
