# <a name="windows-mixed-reality-com-interfaces"></a>Windows Mixed RealityCOM 介面

本節所包含的主題會提供 Windows Mixed Reality COM 介面的參考規格。

## <a name="in-this-section"></a>本節內容

| 主題 | 描述 |
|-|-|
| [**ISpatialInteractionManagerInterop**](/windows/desktop/api/spatialinteractionmanagerinterop/nn-spatialinteractionmanagerinterop-ispatialinteractionmanagerinterop) | 啟用與通用 Windows 平臺 (UWP) [SpatialInteractionManager](/uwp/api/Windows.UI.Input.Spatial.SpatialInteractionManager)物件之間的互通性，並提供存取 **SpatialInteractionManager** 成員的存取權，以從手、動作控制器和系統聲音命令存取使用者輸入。 |
| [**IHolographicSpaceInterop**](/windows/desktop/api/holographicspaceinterop/nn-holographicspaceinterop-iholographicspaceinterop) | 啟用與通用 Windows 平臺 (UWP) [HolographicSpace](/uwp/api/windows.graphics.holographic.holographicspace)物件之間的互通性，並提供 **HolographicSpace** 成員的存取權，以代表全像全像的場景。 |
