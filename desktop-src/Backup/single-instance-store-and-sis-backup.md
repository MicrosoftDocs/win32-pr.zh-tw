# <a name="single-instance-store-and-sis-backup"></a>Single-Instance 存放區和 SIS 備份

*單一實例存放區*（或 SIS）是一種設計來維護重複檔案的架構，最少磁片、快取和備份媒體的額外負荷。 *單一實例存放區備份*（或 SIS 備份）是備份應用程式呼叫以存取 SIS 架構的 API。
