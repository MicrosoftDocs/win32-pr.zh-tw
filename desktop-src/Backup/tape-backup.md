# <a name="tape-backup"></a>磁帶備份

*磁帶磁片* 區是由錄音媒體和其實體貨運公司所組成。 磁片區中的整個磁帶長度無法用來記錄資料。 在磁帶的開頭和結尾的簡短區段會保留，以將磁帶連接至電訊廠商的中樞。 可以記錄資料之磁帶上的第一個位置，就是所謂的 *開頭標記*，而最後一個位置則稱為「 *中結束記號*」。

每個磁帶磁片區都有一或多個磁碟分割。 磁碟分割是磁片區的一部分，其中包含其本身的開始和結束點，而不會與磁片區的任何其他部分重迭。 每個分割區都有三個預先定義的位置。 分割區中您可以記錄資料的第一個位置稱為資料 *分割開頭的標記*，而最後一個位置則稱為「 *分割區結束記號*」。 *早期警告標記* 位於資料分割結束記號的正上方。 早期警告位置會通知磁帶應用程式在到達磁碟分割結束記號之前，將緩衝的資料傳輸至磁帶。

分割區開始與結束點之間的區域通常 *會依標記* 或 *setmarks* 分割成區段。 標記和 setmarks 是不包含使用者資料的特殊記錄元素;它們只是將分割區分成較小的區域，以提供位址配置。 標記和 setmarks 有類似的用途，但 setmarks 在高容量磁帶機上提供更快速的定位。

一般而言，磁帶裝置支援的是標記和 setmarks。 支援兩者都能將磁帶資料格式化，以便 setmarks 不同磁片區的資料，並將檔案從不同的磁片區上的個別檔案分開。

另一個已錄製的元素，表示磁帶上的位置是 *清除間隙*、清除的磁帶區域或裝置無法辨識為標示或使用者資料的模式。

有三種類型的標記。 如果寫入作業是從分割區的開頭或較舊的長標記執行，則 *簡短* 的標記包含不能覆寫的簡短清除間距。 *長* 標記包含長清除間距，可讓應用程式將磁帶放置在標記的開頭，以及覆寫標記和清除間距。 *一般* 的標記不包含清除間距。 使用標記的磁帶裝置支援簡短和長的格式標記或一般的標記，但不支援全部三個。

Setmarks 或標記之間的分割區可以用來記錄資料。 寫入或讀取磁帶的資料單位稱為區塊。
