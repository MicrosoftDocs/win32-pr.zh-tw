# <a name="lsa-authentication"></a>LSA 驗證

LSA 驗證會描述 [*本地安全機構*](../secgloss/l-gly.md) (LSA) 的部分，應用程式可以使用這些元件來驗證本機系統，並將使用者登入本機系統。 它也會說明如何建立和呼叫驗證套件和安全性封裝。

LSA 驗證函式可讓您撰寫驗證套件、子驗證套件，或 (SSP/AP) 的結合安全性支援提供者/驗證套件。

下列主題提供有關 LSA 驗證的詳細資訊：

-   [LSA 驗證模型](lsa-authentication-model.md)
-   [LSA 登入會話](lsa-logon-sessions.md)
-   [LSA 使用者登入驗證](lsa-user-logon-authentication.md)
-   [驗證套件](authentication-packages.md)

如需 LSA 驗證函數的詳細資訊，請參閱 [驗證參考](authentication-reference.md)。

如需相關技術的詳細資訊，請參閱 [LSA 原則](../secmgmt/lsa-policy.md) 和 [Winlogon 和認證提供者](winlogon-and-credential-providers.md)。
