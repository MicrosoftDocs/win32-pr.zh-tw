# <a name="schannel-and-cryptoapi"></a>Schannel 和 CryptoAPI

Schannel 針對密碼編譯作業（例如儲存 [*公開/私密金鑰*](../secgloss/p-gly.md)）使用 [CryptoAPI](../seccrypto/cryptography-essentials.md) 。

下列主題提供有關 Schannel 如何利用 CryptoAPI 的詳細資訊。



| 主題                                                                   | 描述                                                                                                          |
|-------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [憑證存放區](certificate-stores.md)<br/>                 | 用戶端和伺服器憑證都必須儲存在憑證存放區中。<br/>                                |
| [CryptoAPI 2.0 私密金鑰](cryptoapi-2-0-private-keys.md)<br/> | Schannel 認證會在內部表示為憑證 [**\_ 內容**](/windows/win32/api/wincrypt/ns-wincrypt-cert_context) 結構。<br/> |
