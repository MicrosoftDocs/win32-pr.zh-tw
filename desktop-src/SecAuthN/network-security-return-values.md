# <a name="network-security-return-values"></a>網路安全性傳回值

網路提供者可以設定的傳回值包括下列錯誤碼。



| 錯誤碼         | Description                                                                                                                                                                                                                                                                                                                             |
|---------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| WN \_ 成功         | 此函數已成功。                                                                                                                                                                                                                                                                                                                 |
| \_不 \_ 支援 WN  | 此網路提供者不支援此函數。                                                                                                                                                                                                                                                                                 |
| WN \_ NET \_ 錯誤      | 發生未知的網路錯誤。                                                                                                                                                                                                                                                                                                       |
| WN \_ 更多 \_ 資料      | 傳入的資料緩衝區不足以包含所有可用的資料。                                                                                                                                                                                                                                                         |
| WN \_ 錯誤 \_ 指標    | 在函式呼叫期間指定了不正確指標。                                                                                                                                                                                                                                                                     |
| WN \_ 錯誤 \_ 值      | 在函式呼叫期間指定了不正確數值。                                                                                                                                                                                                                                                               |
| WN \_ 錯誤的 \_ 密碼   | 指定的密碼不正確。                                                                                                                                                                                                                                                                                                    |
| \_拒絕 WN 存取 \_  | 呼叫端沒有足夠的許可權可以完成此作業。                                                                                                                                                                                                                                                              |
| WN \_ 函數 \_ 忙碌中  | 此函式無法重新輸入，目前正在使用中，或提供者仍在初始化中，但尚未準備好進行呼叫。 只有在提供者即將可供使用時，才會傳回此錯誤碼。 MPR 會將這個傳回碼傳回到其呼叫端，而且可能會導致呼叫端重試。<br/> |
| WN \_ WINDOWS \_ 錯誤  | 必要的函式失敗。                                                                                                                                                                                                                                                                                                             |
| WN \_ 錯誤的 \_ 使用者       | 指定的使用者名稱無效。                                                                                                                                                                                                                                                                                            |
| WN \_ \_ 記憶體不足 \_ | 記憶體不足，無法完成操作。                                                                                                                                                                                                                                                                                 |
| WN \_ 未 \_ 連接  | 裝置不會重新導向。                                                                                                                                                                                                                                                                                                           |
| WN \_ 開啟的檔案 \_     | 無法取消連接，因為檔案仍處於開啟狀態。                                                                                                                                                                                                                                                                      |
| WN \_ 錯誤的 \_ 網路名稱    | 網路名稱無效。                                                                                                                                                                                                                                                                                                          |
