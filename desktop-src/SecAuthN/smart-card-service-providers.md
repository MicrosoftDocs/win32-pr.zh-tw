# <a name="smart-card-service-providers"></a>智慧卡服務提供者

智慧卡服務提供者 (SCSP) 提供 [*智慧卡*](../secgloss/s-gly.md) 功能的存取權。 他們可以提供單一功能的存取權，例如智慧卡 SDK 提供的基底服務提供者，也可以提供數個功能的存取權來完成更複雜的工作。

服務提供者提供透過 COM 介面的存取。 智慧卡 SDK 提供自己的基底服務提供者所使用的 COM 介面，以及開發自訂服務提供者時可使用的數個應用程式介面。



| 主題                                                                                   | 描述                                            |
|-----------------------------------------------------------------------------------------|--------------------------------------------------------|
| [基底服務提供者](base-service-providers.md)<br/>                         | 基底服務提供者。<br/>                     |
| [建立 ISO7816-4 APDU 命令](building-an-iso7816-4-apdu-command.md)<br/> | 將功能加入至服務提供者。<br/> |
| [廠商包裝函式服務提供者](vendor-wrapper-service-provider.md)<br/>       | 廠商包裝函式範例。<br/>                   |



 

如需 (基礎服務提供者和應用程式介面) 的智慧卡 SDK 所提供的所有 COM 介面的總覽，請參閱 [智慧卡介面](smart-card-interfaces.md)。
