# <a name="winlogon-and-credential-providers"></a>Winlogon 和認證提供者

[*Winlogon*](../secgloss/w-gly.md)是針對 [*登入會話*](../secgloss/l-gly.md)執行互動式登入的 Windows 模組。 您可以藉由執行和註冊認證提供者來自訂 Winlogon 行為。

如需有關如何執行認證提供者的詳細資訊，請參閱下列主題。



| 主題                                                                                                           | 描述                                                                                            |
|-----------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [認證提供者導向的 Windows 登入體驗](https://go.microsoft.com/fwlink/?LinkId=717287)<br/> | Winlogon 和認證提供者架構的總覽，以及範例認證提供者。<br/> |
| [Shell 介面](../shell/samples-usingthumbnailproviders.md)<br/>                                                                | 認證提供者介面參考。<br/>                                                    |
| [Windows 10 中的認證提供者](credential-providers-in-windows.md)<br/>                            | Windows 10 中的協力廠商認證提供者和系統認證提供者。<br/>             |



 

如需範例認證提供者的範例，請參閱位於 Windows SDK 安裝目錄中的範例 \\ \\ 安全性 \\ credentialprovider.h。

**Windows Server 2003 和 Windows XP：** 不支援認證提供者。 如需自訂 Winlogon 的詳細資訊，請參閱 [Winlogon 和 GINA](winlogon-and-gina.md)。
