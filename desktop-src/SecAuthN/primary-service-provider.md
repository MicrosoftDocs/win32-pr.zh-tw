# <a name="primary-service-provider"></a>主要服務提供者

系統已知的每個 [*智慧卡*](../secgloss/s-gly.md) 可能會有一個已註冊的 COM 介面列為其主要服務提供者。 當您透過 COM 存取 *智慧卡* 時，此 [*主要服務提供者*](../secgloss/p-gly.md) 會將控制項介面提供給該卡片。 這可讓您將卡片服務公開至各式各樣的程式設計環境，包括 JAVA 和 Microsoft Visual Basic 開發系統。
