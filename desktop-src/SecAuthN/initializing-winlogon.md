# <a name="initializing-winlogon"></a>正在初始化 Winlogon

當 [Winlogon](winlogon.md) 初始化時，它會向系統 (SAS) 註冊 CTRL + ALT + DEL [*安全的注意順序*](../secgloss/s-gly.md) ，然後在 WinSta0 視窗站內建立三部桌上型電腦。

註冊 CTRL + ALT + DEL 會使此初始化成為第一個進程，因此可確保沒有任何其他應用程式已與該金鑰順序連結。

WinSta0 是代表實體螢幕、鍵盤和滑鼠的視窗站物件名稱。 Winlogon 會在 WinSta0 物件中建立下列桌面。



| 桌面              | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
|----------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Winlogon 桌面     | 這是 Winlogon 和 [*GINA*](../secgloss/g-gly.md) 用於互動式識別和驗證以及其他安全對話方塊的桌面。 Winlogon 會在收到 SAS 事件通知時自動切換到此桌面。                                                                                                                                                                                                                                                                                                                                                                          |
| 應用程式桌面  | 每次使用者成功登入時，就會為該 [*登入會話*](../secgloss/l-gly.md)建立應用程式桌面。 應用程式桌面也稱為預設或使用者桌面。 此桌面是所有使用者活動發生的位置。 應用程式桌面受保護;只有系統和互動式登入會話可以存取它。 請注意，只有已登入使用者的特定實例可以存取桌面。 如果互動式使用者使用服務控制器啟動處理常式，該服務應用程式將無法存取應用程式桌面。 |
| 螢幕保護裝置桌面 | 當螢幕保護裝置正在執行時，這是目前的桌面。 如果使用者已登入，則系統和互動式登入會話都可以存取桌面。 否則，只有系統可以存取桌面。                                                                                                                                                                                                                                                                                                                                                                                                                                      |



 

Winlogon 是這些桌面的擁有者，可以將目前或可見的桌面切換至三部桌上型電腦，並允許 GINA 存取這項功能。 一般而言，GINA 開發人員不會變更目前的桌面，因為 Winlogon 會在與 GINA 通訊之前適當設定桌面。 每個 GINA 函式的描述會指出該呼叫的最新桌面。
