# <a name="tls-vs-ssl"></a>TLS 與 SSL 的比較

TLS 是與 SSL 3.0 密切相關的標準，有時也稱為「SSL 3.1」。 TLS 取代 SSL 2.0，應該用於新的開發。 從 Windows 10 開始，1607和 Windows Server 2016 版，SSL 2.0 已經移除，不再受到支援。

需要高層級互通性的應用程式應支援 SSL 3.0 和 TLS。 因為這兩個通訊協定之間的相似之處，所以不會在此檔中包含 SSL 詳細資料，除非它們與 TLS 不同。 以下是來自 [RFC 2246](https://www.ietf.org/rfc/rfc2246.txt)：

「此通訊協定和 SSL 3.0 之間的差異並不大，但它們的重要性足以讓 TLS 1.0 和 SSL 3.0 不會 (交互操作，不過 TLS 1.0 所併入的機制，可讓 TLS 的執行可回到 SSL 3.0) 。」
