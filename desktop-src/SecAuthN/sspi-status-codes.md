# <a name="sspi-status-codes"></a>SSPI 狀態碼

下列狀態碼適用于 SSPI 應用程式，並定義于 Winerror.h 中。



| 狀態碼                                        | 意義                                                                                                                                                                                                                                                                                                                                           |
|----------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| SEC \_ E \_ 演算法 \_ 不相符<br/>             | 用戶端和伺服器無法通訊，因為它們不具有一般演算法。<br/>                                                                                                                                                                                                                                               |
| SEC \_ E \_ 不良的系結 \_<br/>                   | 用戶端提供的 SSPI 通道系結不正確。<br/>                                                                                                                                                                                                                                                                        |
| SEC \_ E \_ 不良 \_ >PKGID<br/>                      | 要求的套件識別碼不存在。<br/>                                                                                                                                                                                                                                                                                       |
| SEC \_ E \_ 緩衝區 \_ 太 \_ 小<br/>              | 提供給函數的緩衝區不夠大，無法包含資訊。<br/>                                                                                                                                                                                                                                                  |
| 秒 \_ E \_ 無法 \_ 安裝<br/>                 | [*安全性封裝*](../secgloss/s-gly.md)無法成功初始化，因此不應該安裝。<br/>                                                                                                                                                                     |
| 秒 \_ E \_ 無法封 \_ 包<br/>                    | 封裝無法封裝內容。<br/>                                                                                                                                                                                                                                                                                             |
| SEC \_ E 憑證已 \_ \_ 過期<br/>                   | 收到的憑證已過期。<br/>                                                                                                                                                                                                                                                                                                  |
| SEC \_ E \_ CERT \_ 未知<br/>                   | 處理憑證時發生未知的錯誤。<br/>                                                                                                                                                                                                                                                                            |
| SEC \_ E \_ CERT \_ 錯誤的 \_ 使用方式<br/>              | 此憑證不適用於要求的使用方式。<br/>                                                                                                                                                                                                                                                                                  |
| SEC \_ E \_ 內容已 \_ 過期<br/>                | 應用程式參考的內容已關閉。 正確撰寫的應用程式應該不會收到此錯誤。<br/>                                                                                                                                                                                                   |
| SEC \_ E \_ CROSSREALM \_ 委派 \_ 失敗<br/> | 伺服器嘗試對伺服器領域外的目標進行 Kerberos 限制委派要求。<br/>                                                                                                                                                                                                                        |
| SEC \_ E \_ 加密 \_ 系統 \_ 無效<br/>         | 密碼編譯系統或總和檢查碼函數無效，因為必要的函式無法使用。<br/>                                                                                                                                                                                                                                 |
| 秒 \_ E \_ 解密 \_ 失敗<br/>                | 無法解密指定的資料。<br/>                                                                                                                                                                                                                                                                                             |
| \_需要 SEC E \_ 委派 \_<br/>            | 要求的作業無法完成。 電腦必須受信任以進行委派，而且必須將目前的使用者帳戶設定為允許委派。<br/>                                                                                                                                                                         |
| 偵測 \_ 到秒 E \_ 降級 \_<br/>             | 系統偵測到可能會嘗試危及安全性。 確認您可以聯繫已驗證的伺服器。<br/>                                                                                                                                                                                                         |
| SEC \_ E \_ 加密 \_ 失敗<br/>                | 無法加密指定的資料。<br/>                                                                                                                                                                                                                                                                                             |
| SEC \_ E 不 \_ 合法的 \_ 訊息<br/>                | 收到的訊息未預期或格式不正確。<br/>                                                                                                                                                                                                                                                                                |
| SEC \_ E \_ 未完成的 \_ 認證<br/>         | 提供的認證未完成，因此無法驗證。 無法初始化內容。<br/>                                                                                                                                                                                                                            |
| SEC \_ E \_ 未完成 \_ 訊息<br/>             | 提供的訊息不完整。 未驗證簽章。<br/>                                                                                                                                                                                                                                                                   |
| 每 \_ 秒 \_ 沒有足夠的 \_ 記憶體<br/>            | 沒有足夠的記憶體可完成要求。<br/>                                                                                                                                                                                                                                                                                |
| SEC \_ E \_ 內部 \_ 錯誤<br/>                 | 發生未對應到 SSPI 錯誤碼的錯誤。<br/>                                                                                                                                                                                                                                                                              |
| SEC \_ E \_ 不正確 \_ 控制碼<br/>                 | 傳遞給函數的控制碼無效。<br/>                                                                                                                                                                                                                                                                                        |
| SEC \_ E \_ 不正確 \_ 權杖<br/>                  | 傳遞給函數的 token 無效。<br/>                                                                                                                                                                                                                                                                                         |
| SEC \_ E \_ 簽發 \_ CA 未 \_ 受信任<br/>          | 處理用於驗證的智慧卡憑證時，偵測到未受信任的 [*憑證授權單位*](../secgloss/c-gly.md) 單位 (CA) 。<br/>                                                                                                            |
| 秒 \_ E \_ 發行 \_ CA 未 \_ 受信任的 \_ KDC<br/>     | 處理用於驗證的網域控制站憑證時，偵測到未受信任的 CA。 系統事件記錄檔包含額外的資訊。<br/>                                                                                                                                                                         |
| SEC \_ E \_ KDC 憑證已 \_ \_ 過期<br/>              | 用於智慧卡登入的網域控制站憑證已過期。<br/>                                                                                                                                                                                                                                                               |
| 秒 \_ E \_ KDC 憑證已 \_ \_ 撤銷<br/>              | 用於智慧卡登入的網域控制站憑證已撤銷。<br/>                                                                                                                                                                                                                                                          |
| SEC \_ E \_ KDC \_ 不正確 \_ 要求<br/>           | 傳送至 KDC 的要求無效。<br/>                                                                                                                                                                                                                                                                                       |
| SEC \_ E \_ KDC \_ 無法 \_ \_ 參考<br/>          | KDC 無法針對要求的服務產生參考。<br/>                                                                                                                                                                                                                                                                   |
| SEC \_ E \_ KDC \_ 未知 \_ ETYPE<br/>             | KDC 不支援要求的加密類型。<br/>                                                                                                                                                                                                                                                                             |
| 秒 \_ E \_ 登入 \_ 拒絕<br/>                   | 此狀態碼已過時。<br/>                                                                                                                                                                                                                                                                                                          |
| 秒 \_ 數 \_ 超過最大的 \_ 推薦數量 \_<br/>        | 超過票證參考的數目上限。<br/>                                                                                                                                                                                                                                                                              |
| 秒 \_ E \_ 修改的訊息 \_<br/>                | 提供用於驗證的訊息已經改變。<br/>                                                                                                                                                                                                                                                                                |
| 秒 \_ E \_ 多個 \_ 帳戶<br/>              | 收到的憑證已對應到多個帳戶。<br/>                                                                                                                                                                                                                                                                              |
| SEC \_ E \_ 必須 \_ 是 \_ KDC<br/>                   | 本機電腦必須是 (KDC) 的 Kerberos 網域控制站，但它並不是。<br/>                                                                                                                                                                                                                                                          |
| SEC \_ E \_ 無 \_ 驗證 \_ 授權單位<br/>   | 無法連絡任何授權單位進行驗證。<br/>                                                                                                                                                                                                                                                                                    |
| SEC \_ E \_ 無 \_ 認證<br/>                 | 沒有任何可用的認證。<br/>                                                                                                                                                                                                                                                                                                          |
| SEC \_ E \_ 無 \_ 模擬<br/>               | 此內容不允許任何模擬。<br/>                                                                                                                                                                                                                                                                                          |
| SEC \_ E \_ 沒有 \_ IP \_ 位址<br/>               | 無法完成要求的工作，因為本機電腦沒有任何 IP 位址。<br/>                                                                                                                                                                                                                                     |
| SEC \_ E \_ NO \_ KERB \_ KEY<br/>                   | 找不到任何 Kerberos 金鑰。<br/>                                                                                                                                                                                                                                                                                                             |
| SEC \_ E \_ 無 \_ PA \_ 資料<br/>                    | 需要原則管理員 (PA) 資料來判斷加密類型，但找不到。<br/>                                                                                                                                                                                                                                        |
| SEC \_ E \_ NO \_ S4U \_ prot.10 \_ 支援<br/>          | Kerberos 子系統發生錯誤。 針對不支援使用者服務的網域控制站，提出使用者通訊協定要求的服務。<br/>                                                                                                                                                                       |
| SEC \_ E \_ NO \_ TGT \_ REPLY<br/>                  | 用戶端正在嘗試協調內容，而伺服器需要使用者對使用者連線，但未傳送 TGT 回復。<br/>                                                                                                                                                                                                           |
| 秒 \_ E \_ 非 \_ 擁有者<br/>                      | 函數的呼叫端未擁有認證。<br/>                                                                                                                                                                                                                                                                               |
| 秒 \_ E \_ 不 \_ 支援<br/>                  | 不支援此要求。<br/>                                                                                                                                                                                                                                                                                                          |
| 秒 \_ E \_ 確定<br/>                              | 作業已成功完成。<br/>                                                                                                                                                                                                                                                                                                  |
| SEC \_ E \_ \_ \_ 順序<br/>               | 提供進行驗證的訊息不是順序。<br/>                                                                                                                                                                                                                                                                              |
| SEC \_ E \_ PKINIT \_ 用戶端 \_ 失敗<br/>         | 用於驗證的智慧卡憑證不受信任。<br/>                                                                                                                                                                                                                                                                     |
| SEC \_ E \_ PKINIT \_ 名稱 \_ 不符<br/>          | 用戶端憑證不包含有效的 UPN 或登入要求中的用戶端名稱不相符。<br/>                                                                                                                                                                                                                            |

| SEC \_ E \_ 撤銷 \_ 離線 \_ KDC<br/>        | 無法判斷用於智慧卡驗證之網域控制站憑證的撤銷狀態。 系統事件記錄檔包含額外的資訊。<br/>                                                                                                                                                           |
| \_ \_ \_ \_ 找不到 SEC E SECPKG<br/>              | 無法辨認安全性封裝。<br/>                                                                                                                                                                                                                                                                                               |

| 秒 \_ E \_ 關機 \_ \_ 進行中<br/>          | 系統關機正在進行中。<br/>                                                                                                                                                                                                                                                                                                      |
| 秒 \_ E \_ 智慧卡憑證已 \_ \_ 過期<br/>        | 用於驗證的智慧卡憑證已過期。<br/>                                                                                                                                                                                                                                                                        |
| SEC \_ E \_ 智慧卡憑證已 \_ \_ 撤銷<br/>        | 已撤銷用於驗證的智慧卡憑證。 事件記錄檔中可能會有其他資訊。<br/>                                                                                                                                                                                                                |
| \_需要秒 E \_ 智慧卡 \_ 登 \_ 入<br/>      | 需要智慧卡登入，但未使用。<br/>                                                                                                                                                                                                                                                                                         |
| \_ \_ \_ \_ 不 \_ 支援 SEC E 強式加密<br/>  | 安全性協調的另一端需要強式加密，但在本機電腦上則不支援。<br/>                                                                                                                                                                                                                  |

| 秒 \_ E \_ 時間 \_ 誤差<br/>                      | 用戶端和伺服器電腦上的時鐘不相符。<br/>                                                                                                                                                                                                                                                                            |
| 秒 \_ \_ 太 \_ 多 \_ 主體<br/>           | KDC 回復包含一個以上的主體名稱。<br/>                                                                                                                                                                                                                                                                                  |
