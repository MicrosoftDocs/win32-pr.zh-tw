# <a name="smart-card-readers"></a>智慧卡讀卡機

讀取器是 [*智慧卡*](../secgloss/s-gly.md) 系統中的標準裝置。 它們是透過驅動程式來控制，並且會透過隨插即用或透過主控台裝置專案，從系統引進和移除。

您必須定義每個讀取器，才能使用 [*智慧卡子系統*](../secgloss/s-gly.md)。 子系統對於未特別提供給它的任何讀取器不負任何責任。

智慧卡讀卡機可以分成稱為「讀者群組」的邏輯群組。 這些群組可以由子系統定義，也可以由系統管理員和使用者定義。 讀取器可以屬於一個以上的讀者群組

智慧卡子系統會定義下列群組。



| 群組                | 描述                                                                                                                                                                                            |
|----------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 捨棄 $ AllReaders     | 此群組包含系統中的所有讀取器。 提供給智慧卡子系統的新讀取器會自動包含在全系統的讀取器群組捨棄 $ AllReaders 中。                         |
| 捨棄 $ DefaultReaders | 此預設群組（每個 [*終端*](../secgloss/t-gly.md)機各一個）包含所有指派給終端機但未保留給特定用途的讀取器。 |
