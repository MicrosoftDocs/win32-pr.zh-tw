# <a name="smart-card-resource-manager-api"></a>智慧卡 Resource Manager API

[*資源管理員 API*](../secgloss/r-gly.md)提供下列功能。



| 主題                                                                                               | 描述                                                                                                                                                                                                                                                                 |
|-----------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [智慧卡資料庫查詢函數](smart-card-database-query-functions.md)<br/>           | 查詢智慧卡資料庫。<br/>                                                                                                                                                                                                                                   |
| [智慧卡資料庫管理功能](smart-card-database-management-functions.md)<br/> | 管理智慧卡資料庫。<br/>                                                                                                                                                                                                                                  |
| [Resource Manager 內容函數](resource-manager-context-functions.md)<br/>             | 管理 resource manager 資料庫作業的內容。<br/>                                                                                                                                                                                                |
| [Resource Manager 支援函式](resource-manager-support-function.md)<br/>               | 釋放已配置的記憶體。<br/>                                                                                                                                                                                                                                        |
| [智慧卡追蹤功能](smart-card-tracking-functions.md)<br/>                       | 追蹤讀者內的智慧卡。<br/>                                                                                                                                                                                                                                |
| [智慧卡與讀者存取功能](smart-card-and-reader-access-functions.md)<br/>     | 連線與智慧卡進行通訊，包括使用 [*T = 0*](../secgloss/t-gly.md)、 [*t = 1*](../secgloss/t-gly.md)和原始通訊協定傳輸資料。<br/> |
| [直接存取卡的功能](direct-card-access-functions.md)<br/>                         | 與可能不符合 ISO 7816 規格的卡片進行通訊。<br/>                                                                                                                                                                                      |
