# <a name="subauthentication-packages"></a>子驗證套件

Windows 提供的 [*驗證封裝*](../secgloss/a-gly.md)支援使用 [*子驗證套件*](../secgloss/s-gly.md)進行自訂。 子驗證封裝是一種 DLL，可補充或取代主要驗證套件所使用的驗證和驗證準則的一部分。 例如，特定的伺服器可能會提供子驗證套件，以使用不同的演算法驗證使用者的密碼，或指定不同格式的工作站限制。

如需子驗證套件的詳細資訊，請參閱使用平臺軟體發展工具組安裝的 SubAuth 範例 (SDK) 。
