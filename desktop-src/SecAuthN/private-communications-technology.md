# <a name="private-communications-technology"></a>私用通訊技術

私用通訊技術 (PCT) 已由 [*安全通訊端層*](../secgloss/s-gly.md) 3.0 和 [傳輸層安全性通訊協定](transport-layer-security-protocol.md)取代。

Schannel 支援 PCT 1.0 以提供回溯相容性;它不應該用於新的開發。 PCT 支援 SSL 2.0 所支援的大部分密碼套件。
