# <a name="authentication-reference"></a>驗證參考

驗證參考頁面包含 Microsoft 驗證函式、介面、物件、結構和其他程式設計項目的詳細說明。 這些主題提供用來建立使用者和服務身分識別之程式設計項目的相關資訊。 建立身分識別之後，就可以使用它來嘗試存取單一電腦或電腦網路上的資源。

參考頁面分為下列群組。



| 區段                                                          | 描述                                                                          |
|------------------------------------------------------------------|--------------------------------------------------------------------------------------|
| [驗證常數](authentication-constants.md)         | 驗證程式設計項目所使用的常數。<br/>                    |
| [驗證資料類型](authentication-data-types.md)       | 驗證程式設計項目所使用的資料類型。<br/>                   |
| [驗證列舉](authentication-enumerations.md)   | 驗證程式設計項目所使用的列舉。<br/>                 |
| [驗證函式](authentication-functions.md)         | 用於建立和驗證使用者身分識別的函式。<br/>  |
| [驗證介面](authentication-interfaces.md)       | 與使用者身分識別的建立和驗證搭配使用的介面。<br/> |
| [驗證登錄機碼](authentication-registry-keys.md) | 驗證程式設計項目所使用的登錄機碼。<br/>                |
| [驗證傳回值](authentication-return-values.md) | 驗證程式設計項目所使用的傳回值。<br/>                |
| [驗證結構](authentication-structures.md)       | 與驗證函式、介面和物件搭配使用的結構。<br/>   |
