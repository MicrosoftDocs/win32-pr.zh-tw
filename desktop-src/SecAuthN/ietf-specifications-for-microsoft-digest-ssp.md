# <a name="ietf-specifications-for-microsoft-digest-ssp"></a>Microsoft Digest SSP 的 IETF 規格

HTTP 驗證： (RFC 2617) 的基本和摘要式存取驗證可在中取得 [https://www.ietf.org/rfc/rfc2617.txt](https://www.ietf.org/rfc/rfc2617.txt) 。

您可以在中使用 Digest 作為 SASL 機制的網際網路草稿 [https://www.ietf.org/rfc/rfc2831.txt](https://www.ietf.org/rfc/rfc2831.txt) 。

如需有關 SASL 的詳細資訊，另請參閱中提供的簡單驗證及安全性階層規格 (RFC 2222) [https://www.ietf.org/rfc/rfc2222.txt](https://www.ietf.org/rfc/rfc2222.txt) 。
