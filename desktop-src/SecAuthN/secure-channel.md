# <a name="secure-channel"></a>安全通道

安全通道（也稱為 [*Schannel*](../secgloss/s-gly.md)）是 [*安全性支援提供者*](../secgloss/s-gly.md) ， (SSP) ，其中包含一組安全性通訊協定，可透過加密提供身分識別驗證和安全的私人通訊。

Schannel 主要用於需要安全超文字傳輸通訊協定 (HTTP) 通訊的網際網路應用程式。

-   [傳輸層安全性通訊協定](transport-layer-security-protocol.md)
-   [安全通訊端層通訊協定](secure-sockets-layer-protocol.md)
