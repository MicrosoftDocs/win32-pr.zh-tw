<span data-ttu-id="71800-105">加密套件是一組密碼編譯演算法。</span><span class="sxs-lookup"><span data-stu-id="71800-105">A cipher suite is a set of cryptographic algorithms.</span></span> <span data-ttu-id="71800-106">安全通道通訊協定會使用加密套件中的演算法來建立金鑰和加密資訊。</span><span class="sxs-lookup"><span data-stu-id="71800-106">Schannel protocols use algorithms from a cipher suite to create keys and encrypt information.</span></span> <span data-ttu-id="71800-107">加密套件會為下列每個工作指定一種演算法：</span><span class="sxs-lookup"><span data-stu-id="71800-107">A cipher suite specifies one algorithm for each of the following tasks:</span></span>

-   <span data-ttu-id="71800-108">金鑰交換</span><span class="sxs-lookup"><span data-stu-id="71800-108">Key exchange</span></span>
-   <span data-ttu-id="71800-109">大量加密</span><span class="sxs-lookup"><span data-stu-id="71800-109">Bulk encryption</span></span>
-   <span data-ttu-id="71800-110">訊息驗證</span><span class="sxs-lookup"><span data-stu-id="71800-110">Message authentication</span></span>

<span data-ttu-id="71800-111">[*金鑰交換演算法*](../secgloss/k-gly.md) 可保護建立共用金鑰所需的資訊。</span><span class="sxs-lookup"><span data-stu-id="71800-111">[*Key exchange algorithms*](../secgloss/k-gly.md) protect information required to create shared keys.</span></span> <span data-ttu-id="71800-112">這些演算法是非對稱的 ([*公開金鑰演算法*](../secgloss/p-gly.md)) ，適用于相對較少量的資料。</span><span class="sxs-lookup"><span data-stu-id="71800-112">These algorithms are asymmetric ([*public key algorithms*](../secgloss/p-gly.md)) and perform well for relatively small amounts of data.</span></span>

<span data-ttu-id="71800-113">大量加密演算法會加密在用戶端與伺服器之間交換的訊息。</span><span class="sxs-lookup"><span data-stu-id="71800-113">Bulk encryption algorithms encrypt messages exchanged between clients and servers.</span></span> <span data-ttu-id="71800-114">這些演算法是 [*對稱*](../secgloss/s-gly.md) 的，且適用于大量的資料。</span><span class="sxs-lookup"><span data-stu-id="71800-114">These algorithms are [*symmetric*](../secgloss/s-gly.md) and perform well for large amounts of data.</span></span>

<span data-ttu-id="71800-115">[訊息驗證](message-authentication-codes-in-schannel.md) 演算法會產生訊息 [*雜湊*](../secgloss/h-gly.md) 和簽章，以確保訊息的 [*完整性*](../secgloss/i-gly.md) 。</span><span class="sxs-lookup"><span data-stu-id="71800-115">[Message authentication](message-authentication-codes-in-schannel.md) algorithms generate message [*hashes*](../secgloss/h-gly.md) and signatures that ensure the [*integrity*](../secgloss/i-gly.md) of a message.</span></span>

<span data-ttu-id="71800-116">開發人員會使用 [**ALG \_ 識別碼**](../seccrypto/alg-id.md) 資料類型來指定這些元素。</span><span class="sxs-lookup"><span data-stu-id="71800-116">Developers specify these elements by using [**ALG\_ID**](../seccrypto/alg-id.md) data types.</span></span> <span data-ttu-id="71800-117">如需詳細資訊，請參閱 [指定 Schannel 密碼和加密強度](specifying-schannel-ciphers-and-cipher-strengths.md)。</span><span class="sxs-lookup"><span data-stu-id="71800-117">For more information, see [Specifying Schannel Ciphers and Cipher Strengths](specifying-schannel-ciphers-and-cipher-strengths.md).</span></span>
