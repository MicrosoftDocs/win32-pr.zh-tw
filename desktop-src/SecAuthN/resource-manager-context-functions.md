# <a name="resource-manager-context-functions"></a>Resource Manager 內容函數

下列函式會建立並釋放資料庫查詢和資料庫管理函式所使用的 [*資源管理員內容*](../secgloss/r-gly.md) 。



| 主題                                                  | 描述                                                                                                                                          |
|--------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SCardEstablishCoNtext**](/windows/desktop/api/Winscard/nf-winscard-scardestablishcontext) | 建立存取 [*智慧卡資料庫*](../secgloss/s-gly.md)的內容。 |
| [**SCardReleaseCoNtext**](/windows/desktop/api/Winscard/nf-winscard-scardreleasecontext)     | 關閉已建立的內容。                                                                                                                       |
