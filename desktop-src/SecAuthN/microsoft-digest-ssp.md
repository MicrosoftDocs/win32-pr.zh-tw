# <a name="microsoft-digest-ssp"></a>Microsoft Digest SSP

Microsoft Digest 是一種 [*安全性支援提供者*](../secgloss/s-gly.md) ， (SSP) 來執行摘要存取通訊協定，這是一種與超文字傳輸通訊協定相關之合作物件的輕量驗證通訊協定 (HTTP) 或簡單驗證安全性層 (以) 為基礎的通訊。

Microsoft Digest 提供簡單的挑戰回應機制來驗證用戶端。 這個 SSP 的目的是要讓用戶端/伺服器應用程式使用 HTTP 或 SASL 型通訊。

> [!IMPORTANT]
> 跨樹系不支援摘要式驗證。
