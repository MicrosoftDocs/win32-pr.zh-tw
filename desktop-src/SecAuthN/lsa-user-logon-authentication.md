# <a name="lsa-user-logon-authentication"></a>LSA 使用者登入驗證

LSA 處理本機電腦上的使用者登入和驗證，而且如果處理登入要求的驗證封裝支援傳遞驗證，LSA 也可以將使用者登入網路上的其他電腦。 LSA 可存取 (Ssp) 的安全性支援提供者的驗證套件。 執行下列其中一種驗證類型：

-   [互動式驗證](interactive-authentication.md)
-   [非互動式驗證](noninteractive-authentication.md)
