# <a name="winlogon-notification-package-reference"></a>Winlogon 通知套件參考

下列主題提供 [*Winlogon*](../secgloss/w-gly.md) 通知套件的參考資訊：

-   登錄專案會列出必須在系統登錄中輸入的[索引](registry-entries.md)鍵值和設定，以註冊通知封裝和其事件處理常式功能。
-   [事件處理常式](event-handler-function-prototype.md) 函式原型詳細說明事件處理常式函式的原型。 通知套件中的每個事件處理常式函式都必須符合此原型。
