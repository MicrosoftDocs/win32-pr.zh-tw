# <a name="administrative-functions"></a>系統管理功能

下列系統管理功能可讓網路提供者採取網路特定的動作，以顯示和操作網路特定目錄，換句話說，對應至非 Windows 通訊協定的目錄。



| 函式                                         | 描述                                                                                                                                                   |
|--------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**NPGetDirectoryType**](/windows/desktop/api/Npapi/nf-npapi-npgetdirectorytype) | 由檔案管理員呼叫以判斷網路目錄的類型。                                                                                          |
| [**NPDirectoryNotify**](/windows/desktop/api/Npapi/nf-npapi-npdirectorynotify)   | 由檔管理程式呼叫，以通知網路提供者目錄作業。 您可以使用此函式來執行特定目錄的特殊行為。 |
