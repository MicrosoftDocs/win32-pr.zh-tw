# <a name="querying-the-attributes-of-an-schannel-credential"></a>查詢 Schannel 認證的屬性

[**QueryCredentialsAttributes**](/windows/desktop/api/Sspi/nf-sspi-querycredentialsattributesa)函式會提供有關認證的 Schannel 特定資訊。 這項資訊最初是在建立認證時指定。 如需詳細資訊，請參閱 [取得 Schannel 認證](obtaining-schannel-credentials.md)。 此函數所報告的資訊適用于任何 ([*安全性*](../secgloss/s-gly.md) 內容) 使用指定的認證建立的連接。
