# <a name="resource-manager-support-function"></a>Resource Manager 支援函式

下列函式會使用捨棄 AUTOALLOCATE length 指示項來釋放配置的記憶體 \_ ，以簡化其他 [*resource manager*](../secgloss/r-gly.md) 函數的使用。



| 主題                                      | 描述                                                     |
|--------------------------------------------|-----------------------------------------------------------------|
| [**SCardFreeMemory**](/windows/desktop/api/Winscard/nf-winscard-scardfreememory) | 使用捨棄 AUTOALLOCATE 所傳回的釋放記憶體 \_ 。 |
