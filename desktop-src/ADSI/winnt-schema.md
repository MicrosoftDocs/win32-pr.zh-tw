# <a name="winnt-schema"></a>WinNT 架構

WinNT 架構會定義 Windows NT 4.0 網路的 ADSI 物件集合及其類別階層架構。 [WinNT 的 ADSI 物件](adsi-objects-of-winnt.md)包括使用者、電腦、網域、群組、資源、會話、服務及其他。 以下的 [WinNT 物件類別](winnt-object-class-hierarchy.md) 階層區段討論這些物件之間的階層式關聯性。

每個物件都會執行一組 ADSI 介面。 但是，不支援所有屬性和方法的已執行介面。 [Winnt 架構的強制和選擇性屬性](winnt-schemaampaposs-mandatory-and-optional-properties.md) 會列出 winnt 物件類別所支援的屬性。
