# <a name="interface-implementation-of-system-providers"></a>系統提供者的介面實作為

ADSI 併入了數個目錄服務提供者，亦即 LDAP 和 WinNT。 不同的提供者可能會選擇執行或不執行介面。 不同的目錄服務提供者支援不同的屬性和方法。 如需特定目錄服務提供者支援的詳細資訊，請參閱 [ADSI 系統提供](adsi-system-providers.md)者。
