# <a name="accessing-active-directory-using-visual-basic"></a>使用 Visual Basic 存取 Active Directory

microsoft Windows 2000 作業系統提供的其中一個最強大功能，就是 microsoft Active Directory 目錄服務。 當您登入 Windows 2000 網域時，Active Directory 會進入動作;若要在大樓中搜尋最接近的彩色印表機，您可以使用 Active Directory。 它可用於通訊錄查閱或搜尋大樓40中工作的使用者。 使用 Active Directory 時，您可以使用智慧卡登入 Windows 2000 網域。 您甚至可以將 Microsoft SQL Server 資料庫軟體和 Active Directory 的資料聯結。

許多其他的 microsoft 技術（例如 microsoft Message Queue Server (MSMQ) 、COM (類別存放區) 、網際網路通訊協定安全性 (IPsec) 、群組原則物件 (gpo) ，以及 Exchange 與 Active Directory 整合。

本節討論如何使用 Active Directory 服務介面 (ADSI) 來存取 Active Directory。 針對虛構公司（Fabrikam Corporation）使用案例，您將瞭解如何執行一些基本的系統管理工作。

當您進行此案例時，每個區段中的程式碼範例都會自行建立。

為了簡潔起見，所有的程式碼範例都是使用 Microsoft Visual Basic 開發系統來撰寫。 如需 c + + 轉換的詳細資訊，請參閱[將 ADSI Visual Basic 程式碼對應至 c + + 程式碼](mapping-adsi-visual-basic-code-to-c-code.md)。
