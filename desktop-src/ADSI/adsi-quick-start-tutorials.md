# <a name="active-directory-service-interfaces-quick-start-tutorials"></a>Active Directory 服務介面快速入門教學課程

Active Directory 服務介面 (ADSI) 快速入門教學課程是一系列的 ADSI 範例和支援的檔，其設計目的是要讓您熟悉用來存取 Active Directory 的程式設計模型。 每個範例都是簡短的，並以先前的範例為基礎。 教學課程包括：

-   [設定您的開發環境](setting-up-your-development-environment.md)
-   [使用 Visual Basic 存取 Active Directory](accessing-active-directory-using-visual-basic.md)
-   [ADSI 腳本教學課程](adsi-scripting-tutorial.md)
