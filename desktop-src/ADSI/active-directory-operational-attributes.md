# <a name="active-directory-operational-attributes"></a>Active Directory 操作屬性

操作屬性是目錄中儲存和使用的屬性，供系統管理之用。 例如，您可能會維護一個操作屬性，以記錄另一個屬性修改時的日期和時間。 您必須使用 [**GetInfoEx**](/windows/desktop/api/Iads/nf-iads-iads-getinfoex)的呼叫來明確地取出操作屬性。
