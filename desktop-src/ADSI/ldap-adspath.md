<span data-ttu-id="5577e-114">"DistinguishedName" 指定特定物件的分辨名稱。</span><span class="sxs-lookup"><span data-stu-id="5577e-114">The "DistinguishedName" specifies the distinguished name of a specific object.</span></span> <span data-ttu-id="5577e-115">指定之物件的辨別名稱一定是唯一的。</span><span class="sxs-lookup"><span data-stu-id="5577e-115">A distinguished name for a given object is guaranteed to be unique.</span></span>
