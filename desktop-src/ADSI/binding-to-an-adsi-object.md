# <a name="binding-to-an-adsi-object"></a>系結至 ADSI 物件

連接到目錄服務中的物件稱為「系結」（binding）。 系結至 ADSI 物件是與基礎目錄系統進行通訊的第一個步驟。 物件必須系結至，才能流覽其命名空間、搜尋資料、修改資料，或模擬使用者。 您也可以提供其他系結特性，例如使用者名稱、密碼、伺服器名稱、加密方法和安全驗證。 您可以使用的實際額外系結特性將視提供者而定。
