# <a name="objectcpp"></a>物件。Cpp

在範例提供者元件中，物件清單的程式碼範例（包括使用篩選）是在物件 .cpp 中。 下表列出支援的方法。



| Object                               | 描述                                 |
|--------------------------------------|---------------------------------------------|
| **ObjectTypeList::ObjectTypeList**   | Creator：使用篩選器建立清單。     |
| **ObjectTypeList：： ~ ObjectTypeList**  | 標準的函式。                        |
| **ObjectTypeList::GetCurrentObject** | 從清單中取出目前的物件。  |
| **ObjectTypeList：： Next**             | 增大索引。                             |
| **ObjectTypeList：： Reset**            | 重設索引。                            |
| **IsValidFilter**                    | 檢查篩選準則的完整性。          |
| **BuildDefaultObjectArray**          | 使用篩選器建立物件的陣列。 |
| **BuildObjectArray**                 | 建立物件陣列。                      |
