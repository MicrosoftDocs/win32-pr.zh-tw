# <a name="active-directory-service-interfaces-scripting"></a>Active Directory 服務介面腳本

Microsoft Active Directory 服務介面 (ADSI) 結合了像是 Microsoft Visual Basic Scripting Edition 或 Microsoft JScript 開發軟體的指令碼語言，為系統管理員提供管理使用者、服務、列印佇列，以及執行 Microsoft Windows NT 或 Windows 2000 作業系統之電腦網路其他部分的強大工具。 使用 ADSI 物件模型的腳本可以取代通常會使用系統管理工具或 Microsoft Management Console 完成的重複手動工作。 ADSI 腳本也可將一般的系統維護工作自動化，以進行免操作的作業，並提供簡單的介面來撰寫使用 Active Server Pages 的 web 系統管理工具。

如需使用 ADSI 編寫腳本的詳細資訊，請參閱腳本教學課程、範例和工具，請參閱：

-   Microsoft TechNet 網站上的[ADSI 腳本入門](https://www.microsoft.com/resources/documentation/windows/2000/server/scriptguide/sas_ads_overview.mspx)
-   Microsoft TechNet 網站上的[腳本中心](https://www.microsoft.com/technet/community/scriptcenter/default.mspx)
-   [錯誤和錯誤陷阱](errors-and-error-trapping.md)
