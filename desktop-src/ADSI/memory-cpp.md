# <a name="memorycpp"></a>記憶。Cpp

在範例提供者元件中，顯示記憶體配置和釋出的程式碼範例位於記憶體 .cpp 中。 下表列出支援的常式。



| 項目                | 描述                                                           |
|---------------------|-----------------------------------------------------------------------|
| **AllocProvMem**    | 配置指定的記憶體。                                            |
| **FreeProvMem**     | 指出的可用記憶體。                                                |
| **ReallocProvMem**  | 配置連續的記憶體。                                           |
| **AllocProvStr**    | 配置 LPWSTR 字串。                                            |
| **FreeProvStr**     | 如果尚未釋出，則為可用字串。                                     |
| **ReallocProvStr**  | 配置連續的記憶體。                                           |
| **ProvAllocString** | 驗證字串和第一個參數。 如果沒問題，則會執行配置。 |
