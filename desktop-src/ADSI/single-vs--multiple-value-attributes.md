<span data-ttu-id="665f5-108">單一值屬性的實例可以包含單一值。</span><span class="sxs-lookup"><span data-stu-id="665f5-108">An instance of a single-valued attribute can contain a single value.</span></span> <span data-ttu-id="665f5-109">多重值屬性的實例可以包含單一值或多個值。</span><span class="sxs-lookup"><span data-stu-id="665f5-109">An instance of a multivalued attribute can contain either a single value or multiple values.</span></span> <span data-ttu-id="665f5-110">Active Directory 不會建立具有空白值的屬性，因為屬性包含有效的值，或不存在於物件上。</span><span class="sxs-lookup"><span data-stu-id="665f5-110">Active Directory does not create attributes with empty values—either the attribute contains a valid value or it does not exist on the object.</span></span>
