# <a name="ldap-syntax-object"></a>LDAP 語法物件

LDAP 提供者會使用 LDAP 語法和 ADSI 語法之間的下列對應。



| LDAP 語法   | ADSI 語法 (Automation)             |
|---------------|-------------------------------------|
| ADsPath       | VT \_ BSTR                            |
| Boolean       | VT \_ BOOL                            |
| 計數器       | VT \_ I4                              |
| EmailAddress  | VT \_ BSTR                            |
| FaxNumber     | VT \_ BSTR                            |
| 整數       | VT \_ I4                              |
| 間隔      | VT \_ I4                              |
| List          | VT \_ VARIANT (vt \_ BSTR \| vt \_ 陣列)  |
| NetAddress    | VT \_ BSTR                            |
| OctetString   | VT \_ 變異                         |
| 路徑          | VT \_ BSTR                            |
| PhoneNumber   | VT \_ BSTR                            |
| PostalAddress | VT \_ BSTR                            |
| SmallInterval | VT \_ I4                              |
| String        | VT \_ BSTR                            |
| 時間          | VT \_ 日期                            |
