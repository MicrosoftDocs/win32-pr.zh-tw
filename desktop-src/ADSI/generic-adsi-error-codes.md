# <a name="generic-adsi-error-codes"></a>一般 ADSI 錯誤碼

任何 ADSI 系統提供者都可以傳回這些類型的錯誤碼，以表示 ADSI 特定的錯誤訊息。 它們具有已設定 **為 true** 或 **false** 之嚴重性位的設備代碼5。 當嚴重性位設定為 **true** 時，會導致表單0x80005xxx 的錯誤值和表單 **E \_ ADS \_ \*** 的錯誤訊息。 當嚴重性位設定為 **false** 時，就會產生0x00005xxx 形式的錯誤值，而錯誤訊息則是 **S \_ ADS \_ \*** 表單。

下表列出 Adserr 中定義的一般 ADSI 錯誤碼。



| 值       | 程式碼                                  | 描述                                                           | 更正動作                                                                                                                                                                                                                                                                                          |
|-------------|---------------------------------------|-----------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 0x00005011L | **S \_ ADS \_ ERRORSOCCURRED**            | 在查詢期間，發生一或多個錯誤。                          | 確認搜尋喜好設定可以設定為合法，如果是，則會正確設定。                                                                                                                                                                                                                  |
| 0x00005012L | **S \_ ADS \_ NOMORE 資料 \_ 列**              | 搜尋作業已達到最後一個資料列。                        | 繼續進行程式的其餘部分。                                                                                                                                                                                                                                                                        |
| 0x00005013L | **S \_ ADS \_ NOMORE 資料 \_ 行**           | 搜尋作業已達到目前資料列的最後一個資料行。 | 移至下一個資料列。                                                                                                                                                                                                                                                                                       |
| 0x80005000L | **E \_ ADS \_ 錯誤的 \_ 路徑名稱**             | 傳遞的 ADSI 路徑名稱無效。                                  | 確認物件存在於目錄伺服器上，並檢查路徑的印刷樣式錯誤。                                                                                                                                                                                                        |
| 0x80005001L | **E \_ ADS \_ 不正確 \_ 網域 \_ 物件**   | 要求了不明的 ADSI 網域物件。                          | 確認網域物件的路徑。                                                                                                                                                                                                                                                                      |
| 0x80005002L | **E \_ ADS \_ 不正確 \_ 使用者 \_ 物件**     | 要求了不明的 ADSI 使用者物件。                            | 確認使用者物件是否存在，檢查是否有錯誤的路徑和使用者存取權限。                                                                                                                                                                                                           |
| 0x80005003L | **E \_ ADS \_ 不正確 \_ 電腦 \_ 物件** | 要求了不明的 ADSI 電腦物件。                        | 確認電腦物件是否存在，檢查是否有錯誤的路徑和電腦存取權限。                                                                                                                                                                                                   |
| 0x80005004L | **E \_ ADS \_ 未知 \_ 物件**           | 要求了不明的 ADSI 物件。                                 | 驗證物件的名稱和存取權限。                                                                                                                                                                                                                                                    |
| 0x80005005L | **\_ \_ \_ 未設定 E ADS \_ 屬性**        | 未設定指定的 ADSI 屬性。                              |                                                                                                                                                                                                                                                                                                            |
| 0x80005006L | **\_ \_ \_ 不支援 E ADS \_ 屬性**  | 不支援指定的 ADSI 屬性。                         | 確認已設定正確的屬性。                                                                                                                                                                                                                                                                   |
| 0x80005007L | **E \_ ADS \_ 屬性 \_ 無效**         | 指定的 ADSI 屬性無效                                | 確認傳遞給方法呼叫的參數。                                                                                                                                                                                                                                                           |
| 0x80005008L | **E \_ ADS \_ 錯誤 \_ 參數**            | 一或多個輸入參數無效。                             |                                                                                                                                                                                                                                                                                                            |
| 0x80005009L | **E \_ ADS \_ 物件 \_ 未系結**           | 指定的 ADSI 物件未系結至遠端資源。          | 在呼叫 [**SetInfo**](/windows/desktop/api/Iads/nf-iads-iads-setinfo)之後，在新建立的物件上呼叫 [**GetInfo**](/windows/desktop/api/Iads/nf-iads-iads-getinfo) 。                                                                                                                                                                                      |
| 0x8000500AL | **\_ \_ \_ 未修改 E ADS \_ 屬性**   | 未修改指定的 ADSI 物件。                      |                                                                                                                                                                                                                                                                                                            |
| 0x8000500BL | **\_ \_ 已修改 E ADS 屬性 \_**        | 已修改指定的 ADSI 物件。                          |                                                                                                                                                                                                                                                                                                            |
| 0x8000500CL | **E \_ ADS \_ 無法 \_ 轉換 \_ 資料類型**   | 資料類型無法轉換成原生 DS 資料類型或從中轉換。      | 確認已使用正確的資料類型，以及（或）有足夠的架構資料可執行資料類型轉換。                                                                                                                                                                           |
| 0x8000500DL | **\_ \_ \_ \_ 找不到電子 ADS 屬性**      | 在快取中找不到屬性。                            | 確認已隱含或明確呼叫 [**GetInfo**](/windows/desktop/api/Iads/nf-iads-iads-getinfo) 。 如果屬性是操作屬性，則必須使用 [**GetInfoEx**](/windows/desktop/api/Iads/nf-iads-iads-getinfoex) （而非 **GetInfo**）進行明確的取出。 如果問題持續發生，就不會在伺服器上設定屬性。 |
| 0x8000500EL | **E \_ ADS \_ 物件 \_ 存在**            | ADSI 物件存在。                                               | 使用不同的名稱來建立物件。                                                                                                                                                                                                                                                                 |
| 0x8000500FL | **E \_ ADS \_ 架構 \_ 違規**         | 嘗試的動作違反目錄服務架構規則。     |                                                                                                                                                                                                                                                                                                            |
| 0x80005010L | **\_ \_ \_ 未設定 E ADS 資料行 \_**          | 未設定 ADSI 中的指定資料行。                         |                                                                                                                                                                                                                                                                                                            |
| 0x80005014L | **E \_ ADS \_ 無效 \_ 篩選**           | 指定的搜尋篩選準則無效。                               | 請使用目錄伺服器接受之篩選器的正確格式。                                                                                                                                                                                                                                     |
