# <a name="about-active-directory-service-interfaces"></a>關於 Active Directory 服務介面

 (ADSI) 的 Active Directory 服務介面，會將目錄服務的功能從分散式運算環境中的不同網路提供者抽象化，以提供一組用來管理網路資源的目錄服務介面。 系統管理員和開發人員可以使用 ADSI 服務來列舉和管理目錄服務中的資源，不論哪一個網路環境包含資源。

ADSI 可讓您更輕鬆地執行一般管理工作，例如新增使用者、管理印表機，以及尋找整個分散式運算環境中的資源。

ADSI 讓開發人員可以輕鬆地「目錄啟用」應用程式。 無論安裝哪一種目錄服務，系統管理員和開發人員都會處理一組目錄服務介面。
