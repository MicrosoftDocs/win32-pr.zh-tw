# <a name="libmaincpp"></a>LIBMAIN.Cpp

在範例提供者元件中，Adssmp.dll 標準 DLL 進入點的程式碼範例位於 Libmain .cpp 中。 下表列出支援的常式。



| 項目                                  | 描述                                                              |
|---------------------------------------|--------------------------------------------------------------------------|
| **DllGetClassObject**<br/>      | 用來尋找 class factory 的標準 DLL 進入點。<br/>        |
| **DllCanUnloadNow**<br/>        | 標準 DLL 進入點，用來判斷 DLL 是否可以卸載。<br/> |
| **LibMain**<br/>                | 標準 DLL 初始化進入點。<br/>                      |
| **IsCompatibleOleVersion**<br/> | 確認與 OLE 執行時間的相容性。<br/>                   |
| **DllMain**<br/>                | 大部分 Windows 2000 或 Windows NT 版本的進入點。<br/>     |
