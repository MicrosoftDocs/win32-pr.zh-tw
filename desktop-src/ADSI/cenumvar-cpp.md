| 方法                                          | 描述                                                                           |

| **CSampleDSEnumVariant：： Release**               | 標準 [**IUnknown：： Release**](/windows/win32/api/unknwn/nf-unknwn-iunknown-release) 實行。               |
| **CSampleDSEnumVariant：： Skip**                  | 標準 **IEnumXXXX：： Skip** 執行。                                          |
| **CSampleDSEnumVariant：： Reset**                 | 標準 **IEnumXXXX：： Reset** 實行。                                         |
| **CSampleDSEnumVariant：： Clone**                 | 標準 **IEnumXXXX：： Clone** 實行。                                         |
