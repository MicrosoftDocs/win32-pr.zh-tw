# <a name="propertycpp"></a>財產。Cpp

在範例提供者元件中，將不同資料類型儲存在 **VARIANT** 資料類型中的程式碼範例，以便將它們儲存為物件的屬性（property）。 下表列出支援的常式。



| 常式傳回的值                                                           | Description                  |
|-------------------------------------------------------------------|------------------------------|
| **取得 \_Bstr \_ 屬性** / **put \_ bstr \_ 屬性**                   | 適用于字串屬性。       |
| **取得 \_Long \_ 屬性** / **put \_ long \_ 屬性**                   | 適用于 long 整數屬性。 |
| **取得 \_Date \_ 屬性** / **put \_ date \_ 屬性**                   | 適用于日期格式屬性。  |
| **取得 \_Variant \_ bool \_ 屬性** / **put \_ variant \_ bool \_ 屬性** | 適用于布林值屬性。      |
| **取得 \_Variant \_ 屬性** / **put \_ variant \_ 屬性**             | 適用于 **VARIANT** 屬性。  |
