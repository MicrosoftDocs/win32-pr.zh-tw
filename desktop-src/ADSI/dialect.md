# <a name="dialect"></a>Dialect

用來搜尋目錄、ADO 和 OLE DB 的兩個 ADSI 介面，會使用搜尋字串來指定搜尋的參數。 搜尋參數包含搜尋的基底、搜尋篩選準則、要抓取的屬性，以及搜尋的深度。

您可以用下列其中一種方言指定搜尋字串：

-   [LDAP 方言](ldap-dialect.md)
-   [SQL 方言](sql-dialect.md)
