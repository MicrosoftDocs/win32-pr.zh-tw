# <a name="provider-minimum-requirements"></a>提供者的最低需求

若要撰寫 ADSI 提供者，請依照下列各節所述的方式來執行 Active Directory 服務介面。

-   [核心執行](core-implementation.md) 會簡要列出 ADSI 提供者必須支援的軟體元素。
-   [選擇性的執行](optional-implementation.md) 程式會簡短地列出很鼓勵 ADSI 提供者支援的軟體元素。
-   [自訂屬性](custom-properties.md) 是每個提供者所公開的屬性和介面，可提供目錄服務之原生功能的可程式性。
-   [雙重介面](dual-interfaces.md) 描述 ADSI 慣例，以提供可供自動化和非自動化相容控制器存取的介面。
