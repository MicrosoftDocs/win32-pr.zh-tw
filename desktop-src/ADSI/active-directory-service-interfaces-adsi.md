## <a name="purpose"></a><span data-ttu-id="87eac-107">目的</span><span class="sxs-lookup"><span data-stu-id="87eac-107">Purpose</span></span>

<span data-ttu-id="87eac-108">Active Directory 服務介面 (ADSI) 是一組 COM 介面，可用來從不同的網路提供者存取目錄服務的功能。</span><span class="sxs-lookup"><span data-stu-id="87eac-108">Active Directory Service Interfaces (ADSI) is a set of COM interfaces used to access the features of directory services from different network providers.</span></span> <span data-ttu-id="87eac-109">ADSI 用於分散式運算環境中，以呈現一組用於管理網路資源的目錄服務介面。</span><span class="sxs-lookup"><span data-stu-id="87eac-109">ADSI is used in a distributed computing environment to present a single set of directory service interfaces for managing network resources.</span></span> <span data-ttu-id="87eac-110">系統管理員和開發人員可以使用 ADSI 服務來列舉和管理目錄服務中的資源，不論哪一個網路環境包含資源。</span><span class="sxs-lookup"><span data-stu-id="87eac-110">Administrators and developers can use ADSI services to enumerate and manage the resources in a directory service, no matter which network environment contains the resource.</span></span>
