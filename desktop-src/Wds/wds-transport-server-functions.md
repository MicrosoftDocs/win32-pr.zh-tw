# <a name="wds-transport-server-functions"></a>WDS 傳輸伺服器功能

與內容提供者互動的多播伺服器所提供的功能。



| 函式                                                                         | 描述                                              |
|----------------------------------------------------------------------------------|----------------------------------------------------------|
| [**WdsTransportServerAllocateBuffer**](/windows/desktop/api/Wdstpdi/nf-wdstpdi-wdstransportserverallocatebuffer)     | 配置記憶體中的緩衝區。                            |
| [**WdsTransportServerCompleteRead**](/windows/desktop/api/Wdstpdi/nf-wdstpdi-wdstransportservercompleteread)         | 提供讀取操作的狀態。                       |
| [**WdsTransportServerFreeBuffer**](/windows/desktop/api/Wdstpdi/nf-wdstpdi-wdstransportserverfreebuffer)             | 釋放緩衝區所使用的記憶體。                        |
| [**WdsTransportServerRegisterCallback**](/windows/desktop/api/Wdstpdi/nf-wdstpdi-wdstransportserverregistercallback) | 向多播伺服器註冊提供者回呼。 |
| [**WdsTransportServerTrace**](/windows/desktop/api/Wdstpdi/nf-wdstpdi-wdstransportservertrace)                       | 傳送調試訊息。                               |
| [**WdsTransportServerTraceV**](/windows/desktop/api/Wdstpdi/nf-wdstpdi-wdstransportservertracev)                     | 傳送調試訊息。                               |
