# <a name="windows-machine-learning-interfaces"></a>Windows Machine Learning 介面

下列主題描述 Windows Machine Learning SDK 的 COM 介面。

## <a name="in-this-section"></a>本節內容



| 主題                                                                             | 描述                                                                                                                                                                                |
|-----------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IWinMLEvaluationCoNtext**](/windows/win32/api/winml/nn-winml-iwinmlevaluationcontext)<br/> | 代表將輸入和輸出系結至 WinML 模型的內容。<br/>                                                                                                             |
| [**IWinMLModel**](/windows/win32/api/winml/nn-winml-iwinmlmodel)<br/>                         | 表示具有對應中繼資料的 Windows Machine Learning 模型;包含模型描述 (名稱、作者、版本控制等 ) ，以及預期的輸入和輸出。 <br/> |
| [**IWinMLRuntime**](/windows/win32/api/winml/nn-winml-iwinmlruntime)<br/>                     | 代表載入和評估 WinML 模型的執行時間。<br/>                                                                                                                      |
| [**IWinMLRuntimeFactory**](/windows/win32/api/winml/nn-winml-iwinmlruntimefactory)<br/>       | 表示建立 WinML 執行時間以進行模型載入和評估的 factory。<br/>                                                                                         |
