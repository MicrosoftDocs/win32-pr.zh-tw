# <a name="about-teredo"></a>關於 Teredo

由於可用的 IPv4 位址數目有限，網際網路服務提供者 (ISP) 將需要開始轉換至 IPv6。 Windows Vista 將提供完整的 IPv6 支援，讓開發人員可以立即啟用 IPv6 支援，以確保其應用程式可供明天使用。 使用 Teredo 時，即使 ISP 網路只是 IPv4，Windows Vista 桌面還是能夠進行 IPv6 連線。

對等技術需要端對端連線能力，才能完成使用者體驗。 許多家裡都有網路位址轉譯 (NAT) 裝置來限制這些技術，而應用程式開發人員必須專注于找出解決這些問題的有效方法。 不過，Windows Vista 的 Teredo 支援藉由提供順暢的 NAT 遍歷，來降低應用程式開發人員的負擔。
