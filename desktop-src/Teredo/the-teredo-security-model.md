# <a name="the-teredo-security-model"></a>Teredo 安全性模型

[Teredo](about-teredo.md)技術與 Windows 安全性 Center 和[Windows 防火牆](/previous-versions/windows/desktop/ics/windows-firewall-start-page)結合時，可為終端使用者提供增強的安全性。 針對 Windows 防火牆，透過 Teredo 介面傳送和接收的 IPv6 流量，會與原生 IPv6 流量的處理方式相同。 Teredo 介面會被辨識為系統中的另一個迷你埠。

如需使用方式的詳細資訊，請參閱 [執行 Teredo 安全性模型](implementing-the-teredo-security-model.md)。
