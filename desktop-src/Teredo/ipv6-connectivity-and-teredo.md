# <a name="ipv6-connectivity-and-teredo"></a>IPv6 連線能力與 Teredo

大部分的網路都是僅限 IPv4 的，且不容易獲得端對端 IPv6 連線能力。 此外，大部分的 NAT 裝置甚至會限制端對端 IPv4 連線能力。 在 Windows Vista 中，可以在任何兩部主機之間進行 IPv6 端對端連線。 開發人員不必再擔心端對端連線能力，而是建立新的應用程式案例，讓他們能夠利用此連線能力。

Teredo 的目標是一種 IPv6 轉換技術，可讓您輕鬆地將端對端 IPv6 連線能力自動化。 在許多情況下，開發人員可能甚至不需要知道 Teredo 這類轉換技術存在於 Windows Vista 中。 WindowsVista 可以在背景中使用 Teredo，作為 IPv6 連線的流暢方法。
