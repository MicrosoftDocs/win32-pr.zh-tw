# <a name="using-teredo"></a>使用 Teredo

開發使用 [teredo](about-teredo.md)的應用程式時，目標是要達到 IPv6 連線能力，而不是專注于使用 teredo。 例如，Windows 會議空間和遠端協助是利用 IPv6 提供的端對端連線能力的兩個應用程式。 這些應用程式可以使用原生 IPv6、6to4 或 Teredo 位址來取得連線能力。 針對這些應用程式，以及任何其他可針對 IPv6 開發的應用程式，Teredo 的處理方式就像任何其他可進行連線的通訊協定。

-   [執行 Teredo 安全性模型](implementing-the-teredo-security-model.md)
-   [透過 Teredo 接收請求的流量](receiving-solicited-traffic-over-teredo.md)
-   [透過 Teredo 接收未經要求的流量](receiving-unsolicited-traffic-over-teredo.md)
-   [使用 Teredo 搭配 IP Helper](using-teredo-with-ip-helper.md)
-   [在 Windows XP 中使用 Teredo](using-teredo-in-windows-xp.md)
