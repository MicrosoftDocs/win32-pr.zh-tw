# <a name="systemmonitorgridcolor-property"></a>SystemMonitor. GridColor 屬性

抓取或設定圖形中所使用之格線的色彩。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
Property GridColor As stdole.OLE_COLOR
```



## <a name="property-value"></a>屬性值

如果選取了 [**ShowHorizontalGrid**](systemmonitor-showhorizontalgrid.md) 或 [**ShowVerticalGrid**](systemmonitor-showverticalgrid.md) 屬性，則圖形中所使用的水準和垂直格線色彩。
