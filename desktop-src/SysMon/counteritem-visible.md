# <a name="counteritemvisible-property"></a>CounterItem Visible 屬性

抓取或設定值，這個值會指出計數器是否為繪圖。

這是可讀寫的屬性。

## <a name="syntax"></a>Syntax


```VB
Property Visible As Boolean
```



## <a name="property-value"></a>屬性值

如果計數器是圖形化，則為 True;否則為 false。
