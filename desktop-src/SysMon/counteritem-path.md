# <a name="counteritempath-property"></a>CounterItem 路徑屬性

抓取計數器的路徑。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
Property Path As String
```



## <a name="property-value"></a>屬性值

計數器的路徑，例如「Processor (\_ Total) \\ % processor Time」。
