# <a name="systemmonitorshowhorizontalgrid-property"></a>SystemMonitor. ShowHorizontalGrid 屬性

抓取或設定值，這個值會決定是否要在圖形中顯示水準格線。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
Property ShowHorizontalGrid As Boolean
```



## <a name="property-value"></a>屬性值

True 表示水準格線會顯示在圖形中;否則為 false。 此屬性的預設值是 false。
