# <a name="counterscount-property"></a>計數器。 Count 屬性

抓取集合中 [**CounterItem**](counteritem.md) 實例的數目。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
Property Count As Long
```



## <a name="property-value"></a>屬性值

集合中的計數器數目。
