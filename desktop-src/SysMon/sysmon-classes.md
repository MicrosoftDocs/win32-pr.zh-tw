# <a name="sysmon-classes"></a>SYSMON 類別

您可以使用下列類別來設定系統監視器 (SYSMON) 控制項。



| 類別名稱                             | Description                                                                                                  |
|----------------------------------------|--------------------------------------------------------------------------------------------------------------|
| [**CounterItem**](counteritem.md)     | 提供計數器專案屬性資訊。                                                                  |
| [**計數器**](counters.md)           | 提供管理 [**CounterItem**](counteritem.md) 物件集合的方法。                |
| [**LogFileItem**](logfileitem.md)     | 提供屬性，以抓取系統監視器控制項所存取之單一記錄檔的路徑。     |
| [**LogFiles**](logfiles.md)           | 提供用來管理 [**LogFileItem**](logfileitem.md) 物件集合的方法和屬性。 |
| [**SystemMonitor**](systemmonitor.md) | 提供用來設定 [系統監視器] 控制項的方法和屬性。                            |
