# <a name="systemmonitorbackcolorctl-property"></a>SystemMonitor. BackColorCtl 屬性

抓取或設定控制項的背景色彩。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
Property BackColorCtl As stdole.OLE_COLOR
```



## <a name="property-value"></a>屬性值

控制項的背景色彩。
