# <a name="systemmonitorshowlegend-property"></a>SystemMonitor. ShowLegend 屬性

抓取或設定值，這個值會決定是否要顯示圖例。 圖例會列出計數器，並顯示用來繪製每個計數器的色彩。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
Property ShowLegend As Boolean
```



## <a name="property-value"></a>屬性值

True 表示圖例顯示在控制項中;否則為 false。 這個屬性的預設值是 True。
