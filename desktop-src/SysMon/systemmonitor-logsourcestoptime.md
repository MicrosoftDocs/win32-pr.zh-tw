# <a name="systemmonitorlogsourcestoptime-property"></a>SystemMonitor. LogSourceStopTime 屬性

從計數器集合中記錄在記錄檔中的計數器，抓取最新計數器值的時間戳記。

## <a name="syntax"></a>Syntax


```VB
Property LogSourceStopTime As Date
```



## <a name="property-value"></a>屬性值

在記錄檔中記錄的計數器集合中，計數器值最新計數器值的時間戳記。
