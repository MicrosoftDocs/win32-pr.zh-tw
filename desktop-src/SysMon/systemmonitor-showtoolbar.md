# <a name="systemmonitorshowtoolbar-property"></a>SystemMonitor. ShowToolbar 屬性

抓取或設定值，這個值會決定是否要在控制項上顯示工具列。

## <a name="syntax"></a>Syntax


```VB
Property ShowToolbar As Boolean
```



## <a name="property-value"></a>屬性值

True 表示工具列會顯示在控制項上;否則為 false。 這個屬性的預設值是 True。

## <a name="remarks"></a>備註

工具列可讓使用者新增或移除計數器、選擇用來顯示計數器資料的圖表類型，以及修改控制項的其他屬性。
