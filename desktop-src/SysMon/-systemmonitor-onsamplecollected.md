# <a name="systemmonitoronsamplecollected-event"></a>SystemMonitor. OnSampleCollected 事件

在收集到計數器的取樣值時通知您。

## <a name="syntax"></a>語法


```VB
SystemMonitor.OnSampleCollected()
```



## <a name="parameters"></a>參數

此事件沒有任何參數。

## <a name="return-value"></a>傳回值

此事件不會傳回值。
