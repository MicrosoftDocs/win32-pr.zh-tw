# <a name="systemmonitorlogsourcestarttime-property"></a>SystemMonitor. LogSourceStartTime 屬性

從已記錄在記錄檔中的計數器集合中的計數器，抓取最早計數器值的時間戳記。

## <a name="syntax"></a>Syntax


```VB
Property LogSourceStartTime As Date
```



## <a name="property-value"></a>屬性值

從計數器集合中記錄在記錄檔中的計數器最早計數器值的時間戳記。
