# <a name="systemmonitorcleardata-method"></a>SystemMonitor. ClearData 方法

清除控制項中的所有資料欄位。

## <a name="syntax"></a>語法


```VB
SystemMonitor.ClearData()
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
