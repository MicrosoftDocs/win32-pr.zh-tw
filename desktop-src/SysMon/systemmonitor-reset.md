# <a name="systemmonitorreset-method"></a>SystemMonitor：： Reset 方法

從 **計數器** 集合物件中移除所有 [**CounterItem**](counteritem.md)物件，並清除圖形和報表的視圖。

## <a name="syntax"></a>語法


```VB
Sub Reset()
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
