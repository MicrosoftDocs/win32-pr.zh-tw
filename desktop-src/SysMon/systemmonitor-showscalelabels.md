# <a name="systemmonitorshowscalelabels-property"></a>SystemMonitor. ShowScaleLabels 屬性

抓取或設定值，這個值會決定尺規標籤是否會顯示在圖形的垂直軸上。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
Property ShowScaleLabels As Boolean
```



## <a name="property-value"></a>屬性值

True 表示尺規標籤會顯示在圖形的垂直軸上;否則為 false。 這個屬性的預設值是 True。
