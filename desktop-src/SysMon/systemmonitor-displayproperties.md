# <a name="systemmonitordisplayproperties-method"></a>SystemMonitor：:D isplayProperties 方法

顯示 [ **Graph 屬性**] 對話方塊。

## <a name="syntax"></a>語法


```VB
Sub DisplayProperties()
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。

## <a name="remarks"></a>備註

此對話方塊可讓使用者修改系統監視器控制項的屬性。
