# <a name="systemmonitorbackcolor-property"></a>SystemMonitor 背景屬性

抓取或設定圖形和報表檢視的背景色彩。

這個屬性是唯讀的。

## <a name="syntax"></a>語法


```VB
Property BackColor As stdole.OLE_COLOR
```



## <a name="property-value"></a>屬性值

圖形和報表檢視的背景色彩。

## <a name="remarks"></a>備註

這是環境屬性。 這個屬性的值是由容器所決定。 設定這個屬性的值可能會影響控制項和容器的假像成為單一應用程式。
