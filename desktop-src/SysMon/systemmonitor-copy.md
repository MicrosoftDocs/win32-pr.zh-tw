# <a name="systemmonitorcopy-method"></a>SystemMonitor：： Copy 方法

將控制項的屬性設定、計數器清單和計數器資料複製到剪貼簿，以做為 HTML 物件。

## <a name="syntax"></a>語法


```VB
Sub Copy()
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。

## <a name="remarks"></a>備註

剪貼簿中只能有一個複本。
