# <a name="winhttp-versions"></a>WinHTTP 版本

Microsoft Windows HTTP Services (WinHTTP) 的使用者應使用最新版本的技術5.1 版。 不再支援5.0 版。

## <a name="version-51"></a>版本 5.1

WinHTTP 5.1 提供優於5.0 版的增強功能;如需新功能的詳細資訊，請參閱 [WinHTTP 5.1 中的新](what-s-new-in-winhttp-5-1.md)功能。

使用5.1 版時，WinHTTP 是下列作業系統的作業系統元件：

-   Windows 2000 Service Pack 3 和更新版本 (Datacenter Server 以外的) 
-   Windows XP Service Pack 1 (SP1) 和更新版本
-   Windows Server 2003 （含 Service Pack 1） (SP1) 和更新版本

若為 Windows Server 2003 （含 SP1），WinHTTP 是系統並存元件。 如需詳細資訊，請參閱 [並存元件](/windows/desktop/SbsCs/about-side-by-side-assemblies-)。

不支援在這些作業系統之外轉散發 WinHTTP 5.1。

## <a name="version-50"></a>版本 5.0

WinHTTP 5.0 可轉散發套件 "winhttp5.dll" 可在 Windows 2000 或更新版本上執行，或 Windows NT 4.0 與 Microsoft Internet Explorer 5.01 或更新版本。

> [!IMPORTANT]
> WinHTTP 5.0 下載已無法再使用。 自2004年10月1日起，Microsoft 已從 MSDN 移除 WinHTTP 5.0 SDK 下載，並已終止版本5.0 的產品支援。
