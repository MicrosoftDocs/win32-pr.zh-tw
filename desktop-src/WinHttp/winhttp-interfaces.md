# <a name="winhttp-interfaces"></a>WinHTTP 介面

[Microsoft Windows HTTP Services (WinHTTP) ](about-winhttp.md)會公開下列兩個介面：

-   [**IWinHttpRequest**](iwinhttprequest-interface.md) 會實施不牽涉到事件的 WinHTTP 方法。
-   [**IWinHttpRequestEvents**](iwinhttprequestevents-interface.md) 會實行 WinHTTP 事件和事件型方法。
