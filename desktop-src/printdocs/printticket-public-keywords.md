# <a name="printticket-public-keywords"></a>PrintTicket 公用關鍵字

本主題並非最新的。 如需最新資訊，請參閱 [列印架構規格](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip)。

下一節指定了可適用于 PrintTicket 的 PrintTicket 特定屬性描述和參數初始化運算式。

## <a name="printticket-specific-property-descriptions-usage"></a>PrintTicket 特定屬性描述使用方式

PrintTicket 特定屬性描述項代表可能出現在 PrintTicket 中的列印架構公用關鍵字。 其中包含應用程式或使用者可能想要在 PrintTicket 中指定的特定資訊。

## <a name="parameter-initializers-usage"></a>參數初始化運算式使用方式

參數初始化運算式採用 PrintTicket 形式的 ParameterInit 元素類型。 ParameterInit 元素類型包含參數的值，並在 PrintTicket 內初始化。 如需詳細資訊，請參閱 [參數結構](parameter-constructs.md) 一節。
