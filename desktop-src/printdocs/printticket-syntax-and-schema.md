# <a name="printticket-syntax-and-schema"></a>PrintTicket 語法和架構

本主題並非最新的。 如需最新資訊，請參閱 [列印架構規格](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip)。

PrintTickets 是以 XML 為基礎的列印架構架構語法來表示。 因此，讀取器應該檢查 [元素類型的摘要](summary-of-element-types.md) ，以取得必要的語法。

PrintTicket 架構會根據所使用的專案類型，以及在父元素和子項目所表示的一般結構，與列印架構架構緊密地平行。 請注意，在 PrintTicket 中不允許使用 ParameterDef 元素，而且只允許在 PrintTicket 中使用 ParameterInit 元素。
