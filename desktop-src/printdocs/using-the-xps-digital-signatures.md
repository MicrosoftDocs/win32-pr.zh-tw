# <a name="xps-om-digital-signatures"></a>XPS OM 數位簽章

本主題列出 XPS 檔 API 的數位簽章介面。

XPS OM 支援數位簽章;不過，XPS 檔中的數位簽章也可以直接存取，而不需要建立 XPS OM。 如需有關如何在不使用 XPS OM 的情況下存取 XPS 數位簽章的詳細資訊，請參閱 [Xps 數位簽章 API](xps-digital-signatures.md)。
