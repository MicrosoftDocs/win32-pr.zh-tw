# <a name="xps-document-api"></a>XPS 檔 API

XPS 檔 API 是原生的 Windows API，可讓程式建立、讀取、編輯、儲存和列印 XPS 檔。 Windows 7 和更新版本的 Windows 中支援 XPS 檔 API，並可用於使用者模式程式和 XPSDrv 印表機驅動程式。

XPS 檔 API 支援 XPS 和 OpenXPS，且是針對撰寫原生 Windows 應用程式的 c + + 程式設計人員所設計。 如需有關 Open XPS 規格的詳細資訊，請參閱 [open XML 檔規格 (OpenXPS) ](https://www.ecma-international.org/publications/standards/Ecma-388.htm)。
