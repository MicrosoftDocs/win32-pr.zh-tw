# <a name="overview-of-the-printcapabilities"></a>PrintCapabilities 總覽

本主題並非最新的。 如需最新資訊，請參閱 [列印架構規格](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip)。

PrintCapabilities 描述裝置上可用的設定或狀態。 PrintCapabilities 所描述的裝置通常可以放在許多不同的設定中。 在列印裝置的情況下，一般列印屬性包含雙工列印、解析度和媒體大小。 如果裝置支援這些屬性，它們就是該裝置設定的一部分。 使用者藉由指派特定的值給每個可用的屬性，來選取特定的設定;例如，雙工： OneSided、解析度： 600 DPI 和 MediaSize：法律聲明。

PrintCapabilities 建置於 Print Schema Framework 上，其定義可在 PrintCapabilities 中使用的元素。 Print Schema 關鍵字會定義常用的元素階層或關鍵字，以促進個別用戶端之間資訊和意圖的可攜性。 不過，Print Schema 關鍵字也允許私用延伸模組，以便您可以根據特定需求量身訂做 PrintCapabilities。
