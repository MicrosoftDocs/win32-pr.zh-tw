# <a name="print-processor"></a>列印處理器

列印多工緩衝處理器會監視目前的列印工作和目標印表機，以判斷列印工作的適當時間。 當多工緩衝處理器判斷出應該列印工作之後，就會呼叫列印處理器。 列印處理器是處理列印工作資料的外掛程式。

使用下列功能來處理列印處理器。



| 函式                                                           | 描述                                                          |
|--------------------------------------------------------------------|----------------------------------------------------------------------|
| [**AddPrintProcessor**](addprintprocessor.md)                     | 在指定的伺服器上安裝列印處理器。                    |
| [**DeletePrintProcessor**](deleteprintprocessor.md)               | 從指定的伺服器移除印表機處理器。                 |
| [**EnumPrintProcessorDatatypes**](enumprintprocessordatatypes.md) | 列舉指定的列印處理器支援的資料類型。 |
| [**EnumPrintProcessors**](enumprintprocessors.md)                 | 列舉安裝在指定伺服器上的列印處理器。     |
| [**GetPrintProcessorDirectory**](getprintprocessordirectory.md)   | 取得指定伺服器上列印處理器的路徑。  |
