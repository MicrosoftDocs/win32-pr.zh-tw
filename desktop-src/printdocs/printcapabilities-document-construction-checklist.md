    -   <span data-ttu-id="f10e3-113">針對列印架構關鍵字定義的功能實例，識別可設定此功能的每個可用狀態。</span><span class="sxs-lookup"><span data-stu-id="f10e3-113">For Print Schema Keywords-defined Feature instances, identify each of the available states that this Feature can be set to.</span></span> <span data-ttu-id="f10e3-114">每個狀態都對應至功能實例的選項。</span><span class="sxs-lookup"><span data-stu-id="f10e3-114">Each state corresponds to an Option of the Feature instance.</span></span> <span data-ttu-id="f10e3-115">判斷哪一個狀態對應至與這項功能相關聯之列印架構定義的選項實例，以及哪些狀態需要自訂的選項實例。</span><span class="sxs-lookup"><span data-stu-id="f10e3-115">Determine which of these states correspond to Print Schema-defined Option instances associated with this Feature, and which states require a customized Option instance.</span></span> <span data-ttu-id="f10e3-116">[選項定義](option-definitions.md)主題提供如何建立新選項實例的相關資訊，以及如何從現有的選項實例衍生新的選項實例。</span><span class="sxs-lookup"><span data-stu-id="f10e3-116">The [Option Definitions](option-definitions.md) topic presents information on how to construct new Option instances, and how to derive new Option instances from existing Option instances.</span></span>

    -   <span data-ttu-id="f10e3-117">針對非標準的功能實例，識別可用來區別某個選項與另一個選項的特性。</span><span class="sxs-lookup"><span data-stu-id="f10e3-117">For nonstandard Feature instances, identify the characteristics that can be used to distinguish one Option from another.</span></span> <span data-ttu-id="f10e3-118">代表 ScoredProperty 專案的每個這類特性，並在每個選項實例中，為每個 ScoredProperty 指派該選項的特定值。</span><span class="sxs-lookup"><span data-stu-id="f10e3-118">Represent each such characteristic by a ScoredProperty element, and in each Option instance, assign each ScoredProperty a Value that is specific to that Option.</span></span> <span data-ttu-id="f10e3-119">確定有足夠的 ScoredProperty 元素，讓指定功能的每個選項都是唯一的。</span><span class="sxs-lookup"><span data-stu-id="f10e3-119">Ensure that there are enough ScoredProperty elements so that each Option for a given Feature is unique.</span></span> <span data-ttu-id="f10e3-120">非標準的功能和選項實例的本質是 nonportable。</span><span class="sxs-lookup"><span data-stu-id="f10e3-120">Nonstandard Feature and Option instances are by their nature nonportable.</span></span> <span data-ttu-id="f10e3-121">也就是說，另一個驅動程式將無法找到任何對等的功能或選項，以符合您的驅動程式所建立之 PrintTicket 中指定的非標準功能或選項。</span><span class="sxs-lookup"><span data-stu-id="f10e3-121">That is, another driver will not be able to find any equivalent Feature or Option to match a nonstandard Feature or Option specified in the PrintTicket that your driver creates.</span></span>

<span data-ttu-id="f10e3-136">請注意，不論快照集為何，所報告的一組功能、選項和 ParameterDef 實例都不能變更。</span><span class="sxs-lookup"><span data-stu-id="f10e3-136">Note that the set of Feature, Option and ParameterDef instances reported must not change, regardless of the snapshot.</span></span> <span data-ttu-id="f10e3-137">組成每個選項實例以及指派給每個 ScoredProperty 元素之值的 ScoredProperty 實例也不得變更。</span><span class="sxs-lookup"><span data-stu-id="f10e3-137">The ScoredProperty instances that make up each Option instance and the Value assigned to each ScoredProperty element also must not change.</span></span> <span data-ttu-id="f10e3-138">這也適用于組成每個 ParameterDef 實例的屬性實例。</span><span class="sxs-lookup"><span data-stu-id="f10e3-138">The same holds true for the Property instances that make up each ParameterDef instance.</span></span>

<span data-ttu-id="f10e3-139">如需必須提供才能完整定義功能/選項結構和參數的其他屬性實例清單，請參閱 [ParameterDef](parameterdef.md) 和 [ParameterInit](parameterinit.md)。</span><span class="sxs-lookup"><span data-stu-id="f10e3-139">For a list of additional Property instances that must be supplied to fully define Feature/Option constructs and parameters, see [ParameterDef](parameterdef.md) and [ParameterInit](parameterinit.md).</span></span> <span data-ttu-id="f10e3-140">例如，每個功能都必須將其使用者介面指定 (UI) 的行為，特別是是否可以一次為每項功能選取一個或多個選項實例。</span><span class="sxs-lookup"><span data-stu-id="f10e3-140">For example, each Feature must specify its user interface (UI) behavior, specifically whether exactly one or multiple Option instances may be selected for each Feature at one time.</span></span> <span data-ttu-id="f10e3-141">列印架構關鍵字檔會定義這些屬性實例，其中必須出現在 PrintCapabilities 檔中，而且可以使用列印架構關鍵字中所定義的值實例。</span><span class="sxs-lookup"><span data-stu-id="f10e3-141">The Print Schema Keywords document defines these Property instances, where they must appear within the PrintCapabilities document, and which Value instances defined in the Print Schema Keywords are available.</span></span>
