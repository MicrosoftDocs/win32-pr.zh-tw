# <a name="addprinterdriver-function"></a><span data-ttu-id="ef95b-103">AddPrinterDriver 函式</span><span class="sxs-lookup"><span data-stu-id="ef95b-103">AddPrinterDriver function</span></span>

<span data-ttu-id="ef95b-104">**AddPrinterDriver** 函式會安裝本機或遠端印表機驅動程式，並產生設定、資料和驅動程式檔案的關聯。</span><span class="sxs-lookup"><span data-stu-id="ef95b-104">The **AddPrinterDriver** function installs a local or remote printer driver and associates the configuration, data, and driver files.</span></span>

> [!Note]  
> <span data-ttu-id="ef95b-106">不再建議您安裝不含驅動程式套件的印表機驅動程式。</span><span class="sxs-lookup"><span data-stu-id="ef95b-106">Installing a printer driver without a driver package is no longer recommended.</span></span> <span data-ttu-id="ef95b-107">請改用 [**InstallPrinterDriverFromPackage**](installprinterdriverfrompackage.md) 。</span><span class="sxs-lookup"><span data-stu-id="ef95b-107">Use [**InstallPrinterDriverFromPackage**](installprinterdriverfrompackage.md) instead.</span></span>
