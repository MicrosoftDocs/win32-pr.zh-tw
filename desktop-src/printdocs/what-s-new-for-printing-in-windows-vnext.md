# <a name="whats-new-for-printing"></a>列印功能的新功能

Windows 8 支援使用[JavaScript 和 HTML 來列印 Windows store 應用程式](/previous-versions/windows/apps/hh465225(v=win.10))，以及[使用 c #/VB/c + + 和 XAML Windows store 應用程式的列印](/previous-versions/windows/apps/hh465196(v=win.10))。

Windows 8 也包含新的 COM 型 API，可完整支援[OpenXPS 列印](app-support-for-openxps-printing.md)，以及存取 Windows 8 支援的新印表機驅動程式部分。 如需新 API 的詳細資訊，請參閱 [列印檔案封裝 API](tailored-app-printing-api.md)。

Windows 列印驅動程式收件匣程式可確保 Windows 8 包含高百分比的熱門印表機支援。
