# <a name="whats-new-for-printing"></a>列印功能的新功能

Windows 8 支援使用 [JavaScript 和 HTML 進行 Windows store 應用程式的列印](/previous-versions/windows/apps/hh465225(v=win.10)) ，以及 [使用 c #/VB/C + + 和 XAML 針對 windows store 應用程式進行列印](/previous-versions/windows/apps/hh465196(v=win.10))。

Windows 8 也包含新的 COM 型 API，可完整支援 [OpenXPS 列印](app-support-for-openxps-printing.md) ，以及存取 Windows 8 支援的新印表機驅動程式部分。 如需新 API 的詳細資訊，請參閱 [列印檔案封裝 API](tailored-app-printing-api.md)。

Windows 列印驅動程式收件匣程式可確保 Windows 8 包含高百分比的熱門印表機支援。

下列主題描述舊版 Windows 中所引進的列印和檔功能。

| [Windows Vista 中的列印功能新功能](./what-s-new-in-windows-vista.md)<br/> | 列出 (Api) Windows Vista 中引進的列印應用程式設計介面。<br/> |
