# <a name="print-spooler-api"></a>列印多工緩衝處理器 API

「列印多工緩衝處理器」 API 提供了一個介面，讓應用程式的「列印多工緩衝處理器」管理印表機和列印工作。

應用程式會使用「列印多工緩衝處理器」 API 作為其程式設計的一部分，而不是直接由使用者使用。

本節包含下列主題的相關資訊。



| 主題                                                                                             | 描述                                                                                                                                                                                                             |
|---------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [列印多工緩衝處理器 API 參考](printing-and-print-spooler-reference.md)<br/>                | 有關「列印多工緩衝處理器 API」的函式、結構和其他元素的詳細資訊。<br/>                                                                                                           |
| [非同步列印通知參考](asynchronous-printing-notification.md)<br/> | 描述應用程式和列印多工緩衝處理器裝載的元件（如印表機驅動程式和埠監視器）之間的非同步通訊所使用的函式、介面和列舉。<br/> |
| [印表機驅動程式安裝參考](printer-driver-installation-reference.md)<br/>     | 描述在電腦上安裝和設定印表機驅動程式的功能。<br/>                                                                                                                            |
