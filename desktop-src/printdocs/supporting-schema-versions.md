# <a name="supporting-schema-versions"></a>支援的架構版本

本主題並非最新的。 如需最新資訊，請參閱 [列印架構規格](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip)。

隨著新的需要，列印架構架構未來可能會變更。 這類變更預期會非常罕見，因為最常見的變更是引入新的實例名稱。 這項變更不會影響架構的結構，對用戶端和提供者而言應該是透明的。 對列印架構架構的結構和組織所做的任何變更，都會依其版本號碼的遞增來識別。 將不會識別列印架構關鍵字的新增專案。 PrintTicket 提供者必須能夠支援目前的 Print Schema Framework 版本以及任何先前的版本。 Print Schema Framework 版本是使用出現在 PrintTicket 根項目的 XML 屬性（Version）來識別。
