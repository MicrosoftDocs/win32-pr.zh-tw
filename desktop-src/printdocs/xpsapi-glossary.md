</dd> <dt>

<span id="xps.xpsapi_glossary_bidi_level"></span><span id="XPS.XPSAPI_GLOSSARY_BIDI_LEVEL"></span>**雙向層級**

</dd> <dt>

<span id="xps.xpsapi_glossary_bidi_text_level"></span><span id="XPS.XPSAPI_GLOSSARY_BIDI_TEXT_LEVEL"></span>**雙向文字層級**

</dd> <dt>

<span id="xps.xpsapi_glossary_bidirectional_nesting_level"></span><span id="XPS.XPSAPI_GLOSSARY_BIDIRECTIONAL_NESTING_LEVEL"></span>**雙向的嵌套層級**

</dd> <dt>

<span id="xps.xpsapi_glossary_bidirectional_text_level"></span><span id="XPS.XPSAPI_GLOSSARY_BIDIRECTIONAL_TEXT_LEVEL"></span>**雙向文字層級**

值，這個值會指定目前的文字方向。

</dd> <dt>

<span id="xps.xpsapi_glossary_bleed_box"></span><span id="XPS.XPSAPI_GLOSSARY_BLEED_BOX"></span>**出血方塊**
</dt> <dd>

包含裁剪標記並延伸在實體頁面外的區域。

</dd> <dt>

<span id="xps.xpsapi_glossary_color_channel"></span><span id="XPS.XPSAPI_GLOSSARY_COLOR_CHANNEL"></span>**色彩頻道**
</dt> <dd>

合併用來定義最終色彩值的數個獨立色彩之一。

</dd> <dt>

<span id="xps.xpsapi_glossary_color_interpolation"></span><span id="XPS.XPSAPI_GLOSSARY_COLOR_INTERPOLATION"></span>**色彩插補**
</dt> <dd>

在某個色彩系統中定義以用於另一個色彩系統的色彩值轉換。

</dd> <dt>

<span id="xps.xpsapi_glossary_deep_copy"></span><span id="XPS.XPSAPI_GLOSSARY_DEEP_COPY"></span>**深層複製**
</dt> <dd>

只複製物件的作業，以及物件所參考的所有內容。

</dd> <dt>

<span id="xps.xpsapi_glossary_device_font"></span><span id="XPS.XPSAPI_GLOSSARY_DEVICE_FONT"></span>**裝置字型**
</dt> <dd>

直接在裝置上支援的字型 (例如印表機) 。

</dd> <dt>

<span id="xps.xpsapi_glossary_digestmethod"></span><span id="XPS.XPSAPI_GLOSSARY_DIGESTMETHOD"></span>**DigestMethod**
</dt> <dd>

用來計算數位簽章的演算法。

</dd> <dt>

<span id="xps.xpsapi_glossary_dirty"></span><span id="XPS.XPSAPI_GLOSSARY_DIRTY"></span>**髒**
</dt> <dd>

有關在記憶體中有某些變更但變更尚未儲存到更永久儲存裝置（例如磁片）之條件的、相關或特性。

</dd> <dt>

<span id="xps.xpsapi_glossary_discardable_resource_parts"></span><span id="XPS.XPSAPI_GLOSSARY_DISCARDABLE_RESOURCE_PARTS"></span>**discardable 資源元件**
</dt> <dd>

DiscardControl 元件中列出的封裝元件。 根據 XML 檔規格，在 XPS 檔中，DiscardControl 元件包含可在資源受限的環境中捨棄的資源清單。

</dd> <dt>

<span id="xps.xpsapi_glossary_escape_sequence"></span><span id="XPS.XPSAPI_GLOSSARY_ESCAPE_SEQUENCE"></span>**escape 序列**
</dt> <dd>

可列印的字元序列，用來取代字串中的不可列印字元或保留字元。

</dd> <dt>

<span id="xps.xpsapi_glossary_escaped_name"></span><span id="XPS.XPSAPI_GLOSSARY_ESCAPED_NAME"></span>**轉義名稱**
</dt> <dd>

文字字串，其中每個保留或不可列印的字元都被稱為 escape 序列的字元序列所取代。

</dd> <dt>

<span id="xps.xpsapi_glossary_escaped_string"></span><span id="XPS.XPSAPI_GLOSSARY_ESCAPED_STRING"></span>**逸出字元串**
</dt> <dd>

字串，其中特殊和不可列印的字元已被 escape 序列字串取代。

</dd> <dt>

<span id="xps.xpsapi_glossary_fixed_document"></span><span id="XPS.XPSAPI_GLOSSARY_FIXED_DOCUMENT"></span>**固定檔**
</dt> <dd>

說明 FixedPage 部分順序的 XPS 檔部分。

</dd> <dt>

<span id="xps.xpsapi_glossary_fixed_document_sequence"></span><span id="XPS.XPSAPI_GLOSSARY_FIXED_DOCUMENT_SEQUENCE"></span>**固定檔順序**
</dt> <dd>

描述 FixedDocument 元件順序的 XPS 檔部分。

</dd> <dt>

<span id="xps.xpsapi_glossary_fixed_page"></span><span id="XPS.XPSAPI_GLOSSARY_FIXED_PAGE"></span>**固定頁面**
</dt> <dd>

XPS 檔元件，描述固定檔中的頁面內容。

</dd> <dt>

<span id="xps.xpsapi_glossary_font_collection"></span><span id="XPS.XPSAPI_GLOSSARY_FONT_COLLECTION"></span>**字型集合**
</dt> <dd>

XPS 檔中字型資源的集合。

</dd> <dt>

<span id="xps.xpsapi_glossary_font_embedding"></span><span id="XPS.XPSAPI_GLOSSARY_FONT_EMBEDDING"></span>**字型內嵌**
</dt> <dd>

在檔中包含字型資源的進程。

</dd> <dt>

<span id="xps.xpsapi_glossary_font_entry"></span><span id="XPS.XPSAPI_GLOSSARY_FONT_ENTRY"></span>**字型輸入**
</dt> <dd>

對於包含多個字型的字型資源，這是個別字型的索引。

</dd> <dt>

<span id="xps.xpsapi_glossary_font_resource"></span><span id="XPS.XPSAPI_GLOSSARY_FONT_RESOURCE"></span>**字型資源**
</dt> <dd>

XPS 檔中包含字型資訊的資源。

</dd> <dt>

<span id="xps.xpsapi_glossary_glyph_run"></span><span id="XPS.XPSAPI_GLOSSARY_GLYPH_RUN"></span>**圖像執行**
</dt> <dd>

連續執行共用一般格式的字元。

</dd> <dt>

<span id="xps.xpsapi_glossary_gradient_path"></span><span id="XPS.XPSAPI_GLOSSARY_GRADIENT_PATH"></span>**漸層路徑**
</dt> <dd>

線上性漸層中，從一個漸層點到另一個漸層的色彩變更垂直線條。 在放射狀漸層中，漸層原點與繪製漸層區域之橢圓形之間的線條。

</dd> <dt>

<span id="xps.xpsapi_glossary_gradient_region"></span><span id="XPS.XPSAPI_GLOSSARY_GRADIENT_REGION"></span>**漸層區域**
</dt> <dd>

在幾何內，定義漸層的區域。

</dd> <dt>

<span id="xps.xpsapi_glossary_gradient_stop"></span><span id="XPS.XPSAPI_GLOSSARY_GRADIENT_STOP"></span>**漸層停駐點**
</dt> <dd>

為漸層區域內的位置定義的特定色彩。 漸層停駐點是用來定義沿著漸層路徑的特定位置的色彩。

</dd> <dt>

<span id="xps.xpsapi_glossary_image_brush"></span><span id="XPS.XPSAPI_GLOSSARY_IMAGE_BRUSH"></span>**影像筆刷**
</dt> <dd>

使用影像填滿區域或圖形的筆刷。

</dd> <dt>

<span id="xps.xpsapi_glossary_image_resource"></span><span id="XPS.XPSAPI_GLOSSARY_IMAGE_RESOURCE"></span>**影像資源**

</dd> <dt>

<span id="xps.xpsapi_glossary_part_interleaving"></span><span id="XPS.XPSAPI_GLOSSARY_PART_INTERLEAVING"></span>**部分交錯**

</dd> <dt>

<span id="xps.xpsapi_glossary_path"></span><span id="XPS.XPSAPI_GLOSSARY_PATH"></span>**路徑**

<span id="xps.xpsapi_glossary_print_ticket"></span><span id="XPS.XPSAPI_GLOSSARY_PRINT_TICKET"></span>**列印票證**

</dd> <dt>

<span id="xps.xpsapi_glossary_printcapabilities"></span><span id="XPS.XPSAPI_GLOSSARY_PRINTCAPABILITIES"></span>**PrintCapabilities**

</dd> <dt>

<span id="xps.xpsapi_glossary_printticket"></span><span id="XPS.XPSAPI_GLOSSARY_PRINTTICKET"></span>**PrintTicket**

印表機功能的結構描述，以及用來列印檔案或部分檔（例如頁面）的設定。 PrintTicket 物件可以儲存為檔元件或記憶體資料流程。

</dd> <dt>

<span id="xps.xpsapi_glossary_printticket_part"></span><span id="XPS.XPSAPI_GLOSSARY_PRINTTICKET_PART"></span>**PrintTicket 部分**

</dd> <dt>

<span id="xps.xpsapi_glossary_signature_intent_string"></span><span id="XPS.XPSAPI_GLOSSARY_SIGNATURE_INTENT_STRING"></span>**簽名意圖字串**

</dd> <dt>

<span id="xps.xpsapi_glossary_signature_method"></span><span id="XPS.XPSAPI_GLOSSARY_SIGNATURE_METHOD"></span>**signature 方法**

</dd> <dt>

<span id="xps.xpsapi_glossary_signature_request"></span><span id="XPS.XPSAPI_GLOSSARY_SIGNATURE_REQUEST"></span>**簽署要求**

<span id="xps.xpsapi_glossary_signing_spot"></span><span id="XPS.XPSAPI_GLOSSARY_SIGNING_SPOT"></span>**簽署點**

<span id="xps.xpsapi_glossary_spread_area"></span><span id="XPS.XPSAPI_GLOSSARY_SPREAD_AREA"></span>**散佈區域**

在包含漸層的幾何中，位於漸層區域外之幾何內的區域。

</dd> <dt>

<span id="xps.xpsapi_glossary_unescaped_name"></span><span id="XPS.XPSAPI_GLOSSARY_UNESCAPED_NAME"></span>**非別名名稱**

以文字字串表示的名稱，在此字串中，無法列印和保留的字元 (例如空間和索引標籤) 尚未被 escape 序列取代。

</dd> <dt>

</dt> <dd>

符合 XML 紙張規格的 OPC 套件。 XPS 檔可以儲存為檔案或記憶體資料流程。

</dd> <dt>

<span id="xps.xpsapi_glossary_xps_object_model"></span><span id="XPS.XPSAPI_GLOSSARY_XPS_OBJECT_MODEL"></span>**XPS 物件模型**

</dd> <dt>

<span id="xps.xpsapi_glossary_xps_object_model_document_tree"></span><span id="XPS.XPSAPI_GLOSSARY_XPS_OBJECT_MODEL_DOCUMENT_TREE"></span>**XPS 物件模型檔樹狀結構**

</dd> <dt>

<span id="xps.xpsapi_glossary_xps_om"></span><span id="XPS.XPSAPI_GLOSSARY_XPS_OM"></span>**XPS OM**

其關聯性由 XPS 物件模型定義的程式設計物件。

</dd> <dt>

<span id="xps.xpsapi_glossary_xps_package"></span><span id="XPS.XPSAPI_GLOSSARY_XPS_PACKAGE"></span>**XPS 封裝**
</dt> <dd>

以 OPC 套件的形式儲存在檔案中的 XPS 檔，符合 XML 檔規格。

</dd> <dt>

<span id="xps.xpsapi_glossary_xpsdrv"></span><span id="XPS.XPSAPI_GLOSSARY_XPSDRV"></span>**XPSDrv**
