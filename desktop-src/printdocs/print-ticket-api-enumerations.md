# <a name="print-ticket-api-enumerations"></a>列印票證 API 列舉

列印票證 API 會使用下列列舉。

## <a name="in-this-section"></a>本節內容



| 列舉型別                                                   | 描述                                                                                                                                                             |
|---------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**EDefaultDevmodeType**](/windows/win32/api/prntvpt/ne-prntvpt-edefaultdevmodetype)<br/> | 當列印票證未指定所有可能的設定時，可讓使用者指定要使用哪個 [**DEVMODE**](/windows/win32/api/wingdi/ns-wingdi-devmodea) 作為預設值的來源。<br/> |
| [**EPrintTicketScope**](/windows/desktop/api/prntvpt/ne-prntvpt-eprintticketscope)<br/>     | 指定列印票證的範圍。<br/>                                                                                                                       |
