# <a name="processing-xps-documents-in-an-xpsdrv-filter"></a>在 XPSDrv 篩選中處理 XPS 檔

XPS OM 可以用來從 XPSDrv 印表機驅動程式篩選器中存取 XPS 檔的元件。

如需 XPS 篩選器和 XPSDrv 印表機驅動程式的詳細資訊，請參閱 Windows 驅動程式套件中的 [Xps 篩選](/windows-hardware/drivers/print/xps-filters) 。
