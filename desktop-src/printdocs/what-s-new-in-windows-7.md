# <a name="whats-new-for-printing-in-windows-7"></a>在 Windows 7 中列印的新功能

說明 Windows 7 中檔和列印功能的新功能。

## <a name="documents-and-printing"></a>文件和列印

已改進 Windows 7 的檔和列印功能，包括下列新功能和增強功能。



|                                                                                                                |                                                                                                                                                                                                                                                                                                                          |
|----------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [XPS 檔 API](/previous-versions/windows/desktop/dd316976(v=vs.85))<br/>                                                        | [Xps 檔 API](/previous-versions/windows/desktop/dd316976(v=vs.85))可讓您從 Win32 程式建立 XPS 檔，並將它們儲存為檔檔，或直接傳送到印表機。<br/> 您也可以使用 [XPS 檔 API](/previous-versions/windows/desktop/dd316976(v=vs.85)) 來修改 Win32 程式中的現有 xps 檔。<br/> |
| [XPS 數位簽章 API](/previous-versions/windows/desktop/ff819108(v=vs.85))<br/>                                      | [Xps 數位簽章 API](/previous-versions/windows/desktop/ff819108(v=vs.85))可讓您從 Win32 程式將數位簽章新增至 XPS 檔。<br/> 您也可以使用 XPS 數位簽章 API 來確認 XPS 檔是否為真實且未改變。<br/>                                          |
| [XPS 列印 API](xps-printing.md)<br/>                                                                   | [Xps 列印 API](xpsprint-api.md)可讓您從 Win32 程式列印 xps 檔。<br/>                                                                                                                                                                                                                   |
| [MICROSOFT XPS 檔轉換器 (MXDC) ](microsoft-xps-document-converter--mxdc-.md) 改進<br/> | [MXDC](microsoft-xps-document-converter--mxdc-.md)已經過改良，因此您可以使用現有的 GDI 功能來建立 xps 內容，並從 Win32 程式列印到 xps 列印路徑，而不是在 Windows Vista 中提供更好的圖形化精確度。<br/>                                   |
