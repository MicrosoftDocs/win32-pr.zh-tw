# <a name="print-spooler-api-enumerations"></a>列印多工緩衝處理器 API 列舉

本章節包含列印多工緩衝處理器 API 所使用之列舉的字母順序清單。

## <a name="in-this-section"></a>本節內容



| 列舉型別                                                             | 描述                                                                                                        |
|-------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| [**EPrintXPSJobOperation**](eprintxpsjoboperation.md)<br/>       | 指定 XPS 列印工作是否在幕後處理或轉譯階段。<br/>                           |
| [**EPrintXPSJobProgress**](eprintxpsjobprogress.md)<br/>         | 指定多工緩衝處理程式處理 XPS 列印工作時的目前執行內容。<br/>                         |
| [**列印 \_ 執行 \_ 內容**](print-execution-context.md)<br/> | 表示呼叫 [**GetPrintExecutionData**](getprintexecutiondata.md) 時的執行內容。<br/> |
| [**印表機 \_ 選項 \_ 旗標**](printer-option-flags.md)<br/>       | 指定以 [**OpenPrinter2**](openprinter2.md)開啟之印表機的控制碼快取。<br/>       |
