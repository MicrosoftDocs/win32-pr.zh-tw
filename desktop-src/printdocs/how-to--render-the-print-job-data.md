<span data-ttu-id="1686c-108">轉譯列印工作資料以進行列印時，需要取得程式專屬的資料，例如文字、影像和格式，以及將其轉換成與目的地印表機相容的格式。</span><span class="sxs-lookup"><span data-stu-id="1686c-108">Rendering print job data for printing involves taking the program-specific data, such as text, images, and formatting, and converting them into a format that is compatible with the destination printer.</span></span> <span data-ttu-id="1686c-109">將資料傳送至印表機的程式必須以印表機驅動程式所需的格式來傳送。</span><span class="sxs-lookup"><span data-stu-id="1686c-109">The program that sends the data to the printer must send it in the format that the printer driver requires.</span></span>
