# <a name="flush-method-of-the-control-class"></a>Control 類別的 Flush 方法

清除資料流程的轉送緩衝區，並將任何緩衝資料寫入設定檔。

## <a name="syntax"></a>語法


```mof
void Flush();
```



## <a name="parameters"></a>參數

這個方法沒有任何參數。

## <a name="return-value"></a>傳回值

這個方法不會傳回值。
